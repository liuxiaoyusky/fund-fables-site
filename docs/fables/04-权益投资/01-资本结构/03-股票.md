# 一、股票

## 🏺 寓言故事 —— 《五张船契与一个船队的命运》

闽江口有个叫陈阿海的船老大，一辈子在海上讨生活。五十岁那年，他把手下的五条商船合起来，成立了"海兴船行"，并给每个跟他出过海的兄弟发了一张船契。这船契不大，巴掌大小，却藏着五种说不清的魔力。

第一条魔力叫"收益"。每年腊月，船队从南洋运回胡椒和檀香，卖了钱，阿海便按船契的份数给大家分红。有一年风调雨顺，五条船都满载而归，每股分到二十两银子，比得上寻常人家两年的收成。老伙计赵大拿着分红，在镇上置了二亩地。但阿海也告诉大家，除了分红，船契还能在码头的"船契市"上买卖——行情好的时候，一张契能从十两涨到三十两，这涨出来的二十两，阿海叫它"资本利得"。

第二条魔力叫"风险"。阿海从不讳言这一点。第三年，一条船在琼州海峡遇上了台风，整船瓷器沉了底。那年没有分红，大家还要凑钱赔货主的损失。更惨的是第五年，朝廷突然禁海，船队被迫歇业，船契在市面上的价格跌到了三两一张。赵大急得病了一场，因为他把全部身家都押在了船契上。阿海拍着胸脯说："兄弟们放心，船行是'有限责任'，就算全赔光了，你们最多损失买契的钱，不会把家里的房子搭进去。"可赵大心里明白，那已经够他受的了。

第三条魔力叫"流动"。虽然船契不能退给船行换现银，但码头的船契市每天都有人叫卖。谁急用钱，挂牌子就能出手。只不过，卖价可能高于买价，也可能低于买价——这取决于行情，也取决于你肯不肯接受那个价。

第四条魔力叫"永久"。阿海说："只要海兴船行还在，这船契就有效。它没有到期日，不像钱庄的借据，三年五年要还本。"有人担心阿海百年之后怎么办，阿海笑道："船行是法人，我陈阿海死了，船行还在，船契照样分红。你们可以把契传给儿子、孙子。"

第五条魔力叫"参与"。每年端午，阿海在天后宫召开"船东大会"，持契人按份数投票，决定来年走哪条航线、换不换新船、要不要跟隔壁的李家船行合并。赵大持了十股，就有十票；钱二只持了一股，便只有一票。有一年，阿海想让自己的女婿当大副，赵大联合了几个老伙计投反对票，硬是把这事拦了下来。阿海虽然不高兴，但也认了——这就是规矩。

船行越做越大，阿海又发明了一种新船契，叫"优先契"。

"持优先契的兄弟，每年固定分五两银子，不管船行赚多赚少，这五两先给你们。要是船行散了伙，卖船的钱也是你们先拿。"阿海顿了顿，"但你们不能来船东大会投票，也不能决定航线。这是用投票权换的优先权。"

老周头买了优先契，图个安稳。可有一年船行亏损，优先契的银子没发下来。老周头去找阿海，阿海拿出两种方案："咱们这优先契分两种。一种是'累积'的——今年欠你的五两，明年赚了钱，连今年的补上，一共十两，补完才给普通契分红。另一种是'非累积'的——今年没赚，今年这五两就作废了，明年从头算。你当初买的是哪一种，看契上的戳。"

老周头低头一看，契上盖的是"累积"的印。他松了口气，却也不免唏嘘——同样是船契，普通契的兄弟们那年颗粒无收，而隔壁船行买了非累积优先契的老孙头，那年的五两银子就真的打了水漂。

又过了些年，阿海的儿子阿兴接班，把船行改成了两种股：A股和B股。A股一股顶B股十股的收益，但投票权却是一百比一。阿兴说："这样我能用少数股份控制船行，但你们该分的红一分不少。"老伙计们私下议论，说这世道变了，船契里的门道，比海上的暗礁还多。

赵大临终前，把那张泛黄的船契传给儿子，说了一句："记住，这五张契里，收益和风险是绑在一块的，流动和永久是矛盾的，参与权是你唯一的护身符。"

---

**📖 原文定义**

> 股票是股份有限公司发行的，用以证明投资者的股东身份，并据以获取股息和红利的凭证。

> 收益性是股票最基本的特征，它是指持有股票可以为持有人带来收益的特性。股票的收益主要有两类：一是股息和红利；二是资本利得。

> 风险性是指持有股票可能产生经济利益损失的特性。股票风险的内涵是预期收益的不确定性。

> 流动性是指股票可以依法自由地进行交易的特征。

> 永久性是指股票所载有权利的有效性是始终不变的，因为它是一种无期限的法律凭证。

> 参与性是指股票持有人有权参与公司重大决策的特性。

> 普通股股东享有收益权和表决权。表决权通常是每一股份的股东享有一份表决权，即所谓的"一股一票"。

> 优先股是一种相对普通股而言有某种优先权利的特殊股票。优先股的股息率是固定的。在公司盈利和剩余财产的分配顺序上，优先股股东先于普通股股东，但是优先股股东的权利是受限制的，一般无表决权。

> 累积优先股：任何未支付的股息可以累积起来，在普通股持有人收到股息前，用以后财会年度的盈利一起付清。非累积优先股：只能按当年盈利分取股息，未分的股息不能予以累积。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 船契代表对船行的所有权 | 股票是股东身份的凭证，代表对公司所有权的份额 |
| 年底按份分红 + 船契市价涨跌赚差价 | 收益性：股息红利 + 资本利得 |
| 台风沉船、禁海令导致亏损甚至血本无归 | 风险性：预期收益的不确定性 |
| 船契可在码头船契市买卖变现 | 流动性：股票可依法自由交易 |
| 船契无到期日，与船行共存亡 | 永久性：无期限法律凭证 |
| 船东大会按份数投票决定重大事项 | 参与性：按持股比例行使表决权 |
| 普通契持有者分红在优先契之后、有投票权 | 普通股的收益权和表决权 |
| 优先契固定分红、清偿优先但无投票权 | 优先股的固定股息、优先分配权、无表决权 |
| "累积"优先契欠发的股息以后补清 | 累积优先股：未支付股息可累积补发 |
| "非累积"优先契当年没发就作废 | 非累积优先股：当年未分股息不能累积 |
| A股收益权与B股不同、表决权比例悬殊 | 双重股权结构（如伯克希尔A类/B类股） |
