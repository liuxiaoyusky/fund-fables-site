# 6.2.5.3.3 执行客户的指示

## 🏺 寓言故事 —— 《那封电报》

那天早上,陈伯接到了一封电报。

电报是周老板从南洋拍来的。他随船运一批海味出去,半路遇上风浪,货受了潮,损失不小。他在南洋急需改保单——把原来那份货损险的受益范围扩大,否则这趟的损失全得自己扛。

电报上的字很短,只有一句:"陈伯,劳烦即刻替我办妥,改保,受益扩至水渍险。"

陈伯看完,把电报折好揣进口袋,对梁志明说:"志明,周老板要改保单。这事不能拖,我今天就得去保险公司办。"

梁志明点头,可心里有点打鼓。他刚接手不久,知道陈伯常说"客户的指示要准确及时",可这"准确"二字,他还没完全摸到门道。

"陈伯,"他试探着问,"电报上说'扩至水渍险',我直接照这句跟保险公司讲就行?"

陈伯停下手里的活,看着他。"志明,你想想——周老板在南洋,船上,风浪里。他拍这封电报,每一个字都付了电报费。他为什么不多写几个字?因为他急。可正因为急,他才指望我们这边把它办对、办准。"

他把电报从口袋里拿出来,摊在桌上。"你照着这句去讲,保险公司的办事员听不明白'受益扩至水渍险'是什么意思。你得先把它翻译成他们听得懂的话——保单号码多少、原来保什么、要加保什么、生效日期、受益人要不要变。一项项核对清楚,再去办。一个字办错,周老板这趟损失一分钱都拿不回来。"

梁志明愣了。他没料到一封急电报背后,还有这么多要核对的。

陈伯当天下午就去了保险公司。他把每一项核对清楚,办了加保手续。可办到一半,出了一件他没料到的事——保险公司说,加保水渍险,需要周老板本人或其授权代表签署一份补充协议。而周老板人在海上,签不了。

时间在走。陈伯站在保险公司的柜台前,想了一会儿。他没有硬等,也没有把这事闷在自己肚子里。他做了另一件事——当天就拍了一封回电报给周老板。

电报上写明:"加保已申请,惟需补签协议。阁下海上不便,预计将延误两日。请知悉,并示下是否授权代办。"

梁志明后来问陈伯:"您为什么不直接办完,或者直接告诉他办不了?"

陈伯把电报底稿夹进账本。"客户的指示,我们要准、要快。可真要是卡住了、延误了,最难的不是办不成,是闷着不说。客户在那头等,等的是个准信。你办成了告诉他,他安心;你办不成也告诉他,他好另想办法。最要命的是,他以为你办妥了,结果期限到了,一脚踏空——那才是对不起人家这封电报。"

三天后,周老板的船靠了岸,补签的协议也电报授权回来了。加保那天就生效,货损后来赔了下来,周老板没亏。

他回港后,特意来保险行一趟,没带别的,就带了句:"陈伯,电报钱不白花。"

陈伯笑了笑,没接话。

---

📖 原文定义

> 持牌保险代理人应采取合理步骤,准确及及时地执行客户的指示,如发生任何延误或未能执行该等指示,应在可行情况下尽快通知客户。

💡 对应点

| 故事元素 | 概念对应 |
|---------|---------|
| 陈伯说"办对、办准" | "准确"执行客户指示 |
| 陈伯当天就去办、不拖 | "及时"执行的要求 |
| 陈伯逐项核对保单号码、加保内容 | "采取合理步骤"确保准确 |
| 办到一半卡住、当天回电报 | 延误时"尽快通知客户" |
| 陈伯说"最要命的是他以为你办妥了" | 未通知导致客户"一脚踏空"的后果 |
