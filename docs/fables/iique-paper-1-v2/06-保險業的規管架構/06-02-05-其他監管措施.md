# 6.2.5 其他监管措施

## ⚖️ 寓言故事 —— 《官差上门》

那是个寻常的星期三上午,陈伯正在二楼教梁志明核保的规矩。楼梯口忽然上来两个生面孔。一个穿灰西装,中年,神色不怒不威;另一个年轻些,手里抱着一个公文包。

灰西装的人递过一张纸。林若兰接过来,是保监局签发的书面委任,上头白纸黑字写明这位先生获委任为查察员,依法前来查察本行有没有遵守《保险业条例》。

陈伯的脸色沉了一瞬,又恢复了。"请坐。要看什么,您说。"

查察员没坐下。他环顾一圈,开口:"陈先生,我要看三样东西。第一,你们行近一年所有的投保申请表;第二,你们向客户收取保费的收据存根;第三,你们有没有一套自己定的规矩,保证伙计卖保险时不糊弄客户。这三样,我现在就要。"

梁志明的脸一下白了。他上个月刚接手,保险柜里的单据他是按客户名字理的,可收据存根那部分他还没来得及对齐,有半抽屉是乱的。他偷偷看了陈伯一眼,手心全是汗。他心里飞快地盘算:要不趁查察员看别的东西时,把那半抽屉塞到最底下?

可他还没动,陈伯却像没事人一样,走到保险柜前,把三样东西一样一样取出来,摆在查察员面前——包括那半抽屉乱的存根,原封不动地端了出来。"都在这儿。您查。"

梁志明的心提到了嗓子眼。

查察员翻了那半抽屉乱的存根,皱了皱眉。他没发作,只是对年轻的随员说了一句:"记下来。这部分的存根,顺序有问题,需进一步核实。"

林若兰这时开口,语气很平:"先生,存根这部分,我们承认还没理齐。但所有单据都在,一张不少。如果您需要,我们现在就可以按月份重新排好,随时备查。"

查察员看了她一眼,点点头,没再多说。

陈伯在旁边没作声,可他心里清楚,他刚才那个"原封不动端出来"的动作,是这一上午最要紧的一下。他大可以把那半抽屉藏起来,先糊弄过去。可他做了一辈子账房,知道藏起来的东西,早晚是个把柄;摊开来的东西,哪怕乱,反而站得住。查察员要看的,不只是账齐不齐,更是你这个人肯不肯让他看。

可这事没完。三天后,陈伯接到一通电话。

另一头是那位查察员,语气比那天更沉:"陈先生,我们查到一笔。去年九月,有个客户投诉你们行的伙计,卖保单时把分红说得太满,客户事后发现跟条款对不上。这事,我们要立案调查。"

陈伯的手在听筒上紧了紧。他想起来了——那个伙计,半年前已经辞了。可那笔单子是他经手的,陈伯作为负责人,签字画押都在。

"先生,我配合。"陈伯的声音压得很低,"你们要什么,我给什么。"

电话那头停了一下。"还有一事,我得先告诉你。我们查证属实的话,后果不光是批评了事。轻则公开谴责、记过;重则,你的牌照可以撤销或暂时吊销,负责人资格也能一并撤销;也可以禁止你一段时间内再申请牌照、再当负责人;再严重,可以罚款——最高一千万,或者你因这事得利的三倍,哪个大按哪个罚。你心里要有数。"

陈伯握着听筒,半天没出声。一千两百万的生意,他半辈子的清白,都悬在这一通电话上。

林若兰从隔壁桌望过来,看见陈伯的背挺得笔直,可那只握听筒的手在微微发抖。

电话挂断后,陈伯在椅子上坐了很久。梁志明不敢出声,只默默给他倒了杯茶。

"志明。"陈伯终于开口,声音哑,"你以为牌照是拿到了就揣兜里的事?不是。拿了牌,人才刚刚站到刀口上。它随时能查你、随时能调你、随时能罚你。那个查察员,我不是不知道他会再来——我是没想到,他翻的是半年前一个已经走了的伙计的烂账。"

他顿了顿。"可这烂账,我签过字。签字那一刻,我就替他担了。你卖一张单时糊弄过去的那句话,三年后都能被人翻出来,变成撤你牌的刀。"

他把茶碗推开。"所以每一张单,都要经得起查。每一句跟客户说的话,都要当成对面坐的就是那个查察员。这才是拿了牌之后的日子。"

窗外,一辆电车叮叮地驶过。林若兰把刚才那张查察员的委任书复印件,夹进了那本蓝皮账的扉页——她要让它一直留在那儿,提醒这行里每一个人:刀口一直开着。

---

📖 原文定义

> 新颁布的《保险业条例》赋予保监局执行其法定职能的必要权力。保监局可书面指定一位人士作为查察员进行查察,以查明持牌保险中介人是否正在遵守或遵从《保险业条例》的条文;亦可委任调查员调查涉嫌违反条文或涉嫌欺诈等问题。如查证属实,保监局可行使纪律制裁权力,包括撤销或暂时吊销牌照、撤销或暂时吊销负责人认可、禁止申请牌照或获委任为负责人、公开或非公开谴责、以及命令缴付罚款(最高一千万港元,或因不当行为获取利润或避免损失数额的三倍,以较大者为准)。

💡 对应点

| 故事元素 | 概念对应 |
|---------|---------|
| 查察员出示书面委任 | 保监局书面指定查察员的权力 |
| 查察员当场索要三类文件 | 查察员的实地查察权 |
| 陈伯原封不动端出乱存根 | 接受查察、配合查证的态度 |
| 三天后的电话立案调查 | 由查察转调查的环节 |
| 查察员说的轻则谴责、重则撤牌 | 纪律制裁的层级 |
| 最高一千万或得利三倍 | 罚款上限的计算方式 |
| 陈伯"站到刀口上"的领悟 | 持牌后被持续监管的状态 |
