# 6.2.5.1.2 合规

## 📜 寓言故事 —— 《一封挂号信》

梁志明出事那天,是码头的事。

他在码头仓库帮一个客户清点货物时,跟人起了争执,推搡间对方摔了,报了警。最后梁志明被带到差馆,录了口供,控的是一项刑事伤人——虽然他自己一口咬定是对方先动的手。

从差馆出来已是深夜。他没回宿舍,径直上了保险行二楼。陈伯还没走,正借着台灯对账。看见梁志明这副模样,陈伯把老花镜摘下来,没问他去哪了,只问了句:"伤了没?"

梁志明摇头,然后一五一十把码头的事说了。说到最后,声音低下去:"陈伯,我被控了。刑事的。"

陈伯沉默了半晌。

"志明,"他终于开口,"我问你一件事。你是这家保险行的业务代表,你的牌照是保监局发的。你被控刑事罪这事,不是你一个人的私事。"

梁志明一愣。

"按规矩,你出了这种事,得做两件事,而且都不能拖。"陈伯从抽屉里抽出一张白纸,一支钢笔,放到他面前,"头一件,以书面的形式,报告保监局。不是打电话,不是托人带话,是写下来,签名,寄出去。第二件,报告你的委任机构——就是给你发牌卖保险的这家行。两件事,都得在'合理地切实可行的范围内尽快'。"

林若兰这时从隔壁走过来,听见后半段,补了一句:"志明,不光刑事。要是有别的三种情况,也都得这么报。"

梁志明抬头。

"清盘、被法院判破产——这是第一种。"林若兰一项项说,"被裁定刑事罪——这是第二种,就是你今天这个。被金融管理专员、证监会或积金局纪律处分——这是第三种。这三种里头,你撞上哪一种,都要第一时间,书面向保监局和委任机构两边都报。"

梁志明握着那支钢笔,手有点抖。"可这事……还没判呢。我只是被控。"

"被控也要报。"陈伯的语气没有商量的余地,"规矩要的就是'尽快'两个字。它不是等你判了才想知道,它要知道的是,你作为持牌的人,一出事就摊开来。藏着、拖着,比事情本身更要命。"

梁志明低头看着白纸。他想起陈伯常说的一句话——做这行,账要清,话要直,出了事要认。他咬了咬牙,把笔尖按了下去。

那晚,保险行的灯亮到很晚。一封写给保监局的挂号信,一封写给委任机构的报告,字迹一开始有些不稳,到后面越来越定。信封封好,贴上邮票,梁志明亲手投进了街角那个邮筒。

夜很深了。林若兰把灯关了,陈伯还站在窗边。他看着那个邮筒的方向,没说话。他心里清楚,这一封挂号信寄出去,志明作为持牌人的路还长;但要是他今晚没寄,这条路就先断了一截。

---

📖 原文定义

> 持牌保险代理人应遵守所有适用于该代理人的法律、保监局执行或发布并适用于该代理人的规则、规例、守则及指引,以及其他监管机构的规定。如持牌保险代理人:(i) 清盘或被香港或其他地方的法院裁定破产;(ii) 在香港或其他地方被裁定干犯刑事罪行(轻微罪行除外);或(iii) 受到金融管理专员、证监会或强积金局纪律处分,该代理人应在合理地切实可行的范围内尽快(i)以书面形式向保监局报告;及(ii)向其委任保险人或委任代理机构(如适用),以该保险人或代理机构指定的形式报告。

💡 对应点

| 故事元素 | 概念对应 |
|---------|---------|
| 梁志明被控刑事罪 | 三种须报告情形之一(被裁定干犯刑事罪) |
| 陈伯要他写书面报告 | "以书面形式"报告的要求 |
| 一封给保监局、一封给委任机构 | 两处报告对象 |
| 陈伯强调"合理地切实可行的范围内尽快" | 报告的时效要求 |
| 林若兰补的三种情形 | 清盘/破产、刑事、纪律处分三类触发事件 |
| 陈伯"藏着拖着更要命" | 主动、及时披露的合规精神 |
