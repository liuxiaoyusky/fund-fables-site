# 6.2.4 发牌

## 📜 寓言故事 —— 《五问再续》

上环那间保险行的二楼,门帘掀开,陈伯抖着雨伞进来。他把一个牛皮纸信封拍在林若兰桌上:"姑娘,帮我看看,这是不是要续牌的信?"

林若兰拆开,里面一张英文表格,抬头印着保监局的徽号。她一条条往下念,念到中间停住了。"陈伯,这回续牌,跟以前不同了。不是交钱盖章就了事。"

"怎么个不同?"陈伯把老花镜往鼻梁上推。

"它问五样东西。"林若兰的指尖点在表格上,"第一,你的学历、资历,做保险做了多少年;第二,你能不能称职、老实地做这行;第三,你的声誉、人品、靠不靠得住;第四,你的财政状况,有没有欠一屁股债还不上;第五,别的主管部门——证监会、积金局、或者境外类似机构——有没有罚过你、查着你。"

陈伯一愣。"我不是头一回做保险。三十年了。从前续牌,也就是个过场。"

"现在是过场不得。"林若兰把表格转过来对着他,"你瞧这行小字——这些要求不是交一次就完,是年年续牌都要再问一遍。今天你是好人,不代表三年后还是。它要确认你一直是这样的人。"

陈伯没出声。他端起茶碗,吹了吹,没喝。这五样里头,前两样他心里有底——三十年账房、十年保险,学历资历都摆在那。可第三样、第四样,他心里有一道坎。

那道坎,他谁都没提过。

二十年前,他和人合伙开过一间小行。生意做得不顺,合伙的那个朋友,背着他挪了一笔客户的款。事发时,陈伯自己也是蒙在鼓里的。后来那行被强制清盘,他作为合伙人,账上虽没动手脚,可名字挂在那张登记纸上,走不掉。他散了伙,回了上环,从此只做记账、做保险,再没跟人合伙。

这件事,街坊不知道,老客户不知道。可保监局那张表格上第三条、第四条问得明明白白:你曾任董事、合伙人的公司,在你任内或走后一年内被强制清盘、还不起债的,它都要查;你的控权人、合伙人有没有出过事,它也要看。

陈伯把茶碗放下。"姑娘,我问你一桩事。倘若有个人,二十年前合伙的行清了盘,他自己没动手脚,可名字挂在合伙纸上——这续牌,他过得了关么?"

林若兰抬起头,看着他。她听出了这话里的分量。"陈伯,这要看。它要查的,是那行清盘,跟你有没有关系——是同意、纵容,还是疏忽。如果你能说清,那笔账不是你动的,你当时不知情,事后也交代得清楚,它未必就卡你。"

"可它要查。"陈伯的声音低下来,"它翻开那笔旧账,问东问西,街坊会怎么看?"

林若兰没立刻答。她想了想,才开口:"陈伯,我跟你讲句实在的。它查,不是要揪你出来示众。它要的是一个'交代得清楚'。你藏着,它反而起疑;你自己摊开,把来龙去脉写明白,反倒站得稳。这张牌,它每年问你一遍,问的就是你今年还经不经得起被问。"

陈伯坐在椅背上,看着窗外的雨。雨打在瓦片上,很密。他想起了周老板前年铺子差点关门的事,想起自己这些年一笔一笔的账,想起年轻时那段没结果的合伙。三十年的清白,不是靠不说话撑出来的,是靠每一笔都记得、每一笔都经得起翻。

半晌,他回过头,语气平了下来。

"姑娘,那就照规矩办。二十年前的旧账,我一条一条写清楚,附上当年的清盘文件,随这份续牌表一起递上去。它要查,我让它查个明白。"

林若兰点点头,把表格收好。

陈伯走到门边,又停了一下。他没回头,只说了一句:"我这一辈子,最怕的就是'一直'两个字。做一回好人容易,一直做下去难。它这一问再问,倒像是替我守着这个'一直'。"

雨声落在窗沿上,一下一下,很稳。

---

📖 原文定义

> 新条例载列保监局发给牌照的条件,当中包括保险中介人牌照申请人必须符合的"适当人选"准则。该等"适当人选"准则必须持续遵守,并同时适用于续牌申请。在决定某人是否符合适当人选时,保监局会考虑该人的学历、资历或经验;是否有能力称职地、诚实地并公正地进行受规管活动;该人的信誉、品格、可靠程度和诚信;该人的财政状况或偿付能力;以及有否被其他主管当局或规管机构采取纪律行动等事宜。如申请人属商号,其控权人、合伙人和董事(如适用)也必须属于适当人选。

💡 对应点

| 故事元素 | 概念对应 |
|---------|---------|
| 续牌信上的五项询问 | "适当人选"准则的五个考量方面 |
| 林若兰强调"年年续牌都要再问一遍" | 适当人选是持续要求,非一次性 |
| 陈伯担心的二十年旧账 | 任内/走后一年内公司清盘会被翻查 |
| 林若兰说"交代得清楚就站得稳" | 适当人选认定看是否同意/纵容/疏忽 |
| 陈伯决定附上旧文件一起递 | 主动配合查证,持续合规 |
| 陈伯感慨的"一直" | 持续遵守——制度反复确认资格仍在 |
