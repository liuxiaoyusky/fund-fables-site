# 6.2.5.3.6 冷静期

## 🕯️ 寓言故事 —— 《留一道门》

那天下午,何师奶第三次来保险行。前两次都是问,这次是来签字的。

陈伯把申请表摊开,铅笔指到最底下那行签名格。何师奶的手刚要落笔,陈伯却把手按住了。

"师奶,您先别急。有一件事,签之前我得跟您说清楚。"

何师奶抬头。"什么事?"

"这份单子,带一个'冷静期'。"陈伯把条款上那一行小字指给她看,"从保单发出来的那天起算,有一段时间——具体几天,保单上会写明——您要是后悔了,可以退。退了,保费退给您,就当没买过。"

何师奶愣了一下。"还能这样?"

"能。但我得提醒您,这权利得您自己行使。您得在这段时间内,主动通知保险公司,说'我不要了'。过期不候。"陈伯顿了顿,"所以我今天不光让您签字,我还要您心里记着——这不是签下去就钉死了的事。您拿回去,慢慢看,看明白了再决定。"

何师奶松了口气,把名字签了。

可签字只是头一步。陈伯签完字,转头做了一件梁志明没料到的事。他没把申请表随手一塞,而是当天就整理好,当天就送去了保险公司。然后他盯着保单的进度——保单一旦批下来,他做的第一件事是拿到手,第二件事是马上送去何师奶手里。

梁志明不解。"陈伯,保单迟早要送,您急什么?何师奶又跑不了。"

陈伯把那叠保单用牛皮纸包好,在封皮上记下今天的日期。"志明,你想啊——冷静期是从保单发出来那天开始算的。我这边多耽搁一天,何师奶能考虑的日子就少一天。"

他指了指封皮上的日期。"我送得越快,她拿到手越早,她真正能拿来'冷静'的时间就越足。她要是在期限里觉得不合适,还有工夫退;要是等我拖到第十天才送过去,她还没想明白,期限就到了。那这道门,等于我替她关上了。"

梁志明怔住了。

陈伯把包裹往臂弯一夹,推门出去。雨刚停,街上还有水光。他要赶最后一班电车,今天之内送到何师奶手上。他心里清楚,签下去的那一笔,对何师奶是一个承诺;而他留出那段能反悔的时间,才是这承诺真正站得住的地方。

---

📖 原文定义

> 如保单载有冷静期条文,持牌保险代理人应遵守以下常规:(i) 在客户签署保单的申请表前,或(在申请表无须签署的情况下)在保单申请过程完成前,该代理人应告知客户在冷静期内可以取消保单的权利,并告知客户若其希望行使该权利,则应在冷静期内通知相关保险人;及(ii) 如该代理人有责任送交保单予客户,该代理人应于合理地切实可行的范围内尽快向客户送交保单(并保留送交日期的纪录),以便客户在冷静期届满前有足够时间审阅保单并考虑其购买的决定。

💡 对应点

| 故事元素 | 概念对应 |
|---------|---------|
| 何师奶落笔前陈伯按住手 | 签字前就要告知冷静期权利 |
| 陈伯解释"可退、需主动通知" | 告知取消权及行使方式(通知保险人) |
| 陈伯当天整理、当天送出 | "尽快递交"申请 |
| 陈伯盯进度、保单一到手马上送达 | "尽快送交保单"给客户 |
| 陈伯解释"我拖一天她考虑日子少一天" | 留给客户在冷静期内足够的审阅时间 |
| 封皮上记下送交日期 | 保留送交日期的纪录 |
