# 6.2.5.2.1 公平、公正及以客户的最佳利益行事

## ⚖️ 寓言故事 —— 《不是最贵那本》

何师奶来找陈伯那天,是为了一件她惦记了半个月的事。

她坐在保险行的硬木椅上,把一张宣传单摊开。那是别家公司的业务员塞给她的,上面印着一份储蓄寿险,保费一年三千二,分红画得很漂亮。

"陈伯,人家说这份比我自己家的强,分红高。您给我换换?"

陈伯把宣传单看了一遍,又看了一遍。他没急着说话,从抽屉里抽出一张表格——那是他这家保险行授权他能卖的几款产品清单。他拿铅笔在清单上一格格划过去。

"师奶,我跟您说实话。这份宣传单上的产品,我这儿没有授权卖。"他指着清单,"我现在能卖的,就这清单上的几样。我不能为了您一句'想换',就跑出去卖别家的——那是越了线。"

何师奶有点急。"那您这清单里,哪份对我最好?"

陈伯把清单上跟寿险、储蓄沾边的三份挑了出来,一字排开。他没说哪份最好,反而问起何师奶家里的事:茶餐厅的进项、家里有几个小孩要养、每月能匀出多少保费、这笔钱是想存着还是想留给后人。

何师奶一项项答,答到"每月能匀两百"时,自己顿了一下。

陈伯点了点其中一份。"师奶,照您说的,一年保费压在两千四上下最稳。这份分红不如宣传单上那份亮眼,可条款简单,期满保本,不需要您年年去操心。对一个要养铺子、又要养家的人,这份比那一份更对路。"

何师奶盯着他。"可是……分红少啊。"

"分红是画出来的,不是写死给您拿的。"陈伯把那份产品的条款翻到她面前,"您看这条,保证回报是多少,写得明明白白。我给您的意见,不能光挑数字大、好看的,得挑一个像您这样的客户,买了之后日子不会紧、出了事真能指望得上的。"

何师奶看了很久,最后把那张宣传单折了起来,收进包里。"陈伯,那我信您。"

陈伯没接她这句话,只把铅笔搁下。林若兰一直在隔壁桌打字,这时停了停手,看了他一眼。她知道陈伯刚才那番话里的分量——他不是在替自己行里卖最贵的产品,也不是在迎合何师奶想要"分红高"的心思。他做的是另一件更难的事:在他能卖的几样里,挑一个真的对她好的。

---

📖 原文定义

> 在进行关乎客户的受规管活动时,持牌保险代理人应:(i) 公平对待客户;及(ii) 考虑到客户的利益而向客户提供合适、公正及客观的意见。持牌保险代理人只应从该代理人的委任保险人或委任代理机构(如适用)所提供并获其授权进行推广的保险产品范围中,推荐最符合客户利益的保险产品。就此而言,最符合客户利益的保险产品应为一位合理持牌保险代理人基于客户情况认为是适合客户的产品。

💡 对应点

| 故事元素 | 概念对应 |
|---------|---------|
| 陈伯抽出的授权产品清单 | 只能从获授权推广的产品范围中推荐 |
| 陈伯问家底、问保费预算 | 基于客户情况作判断 |
| 没挑最贵、没挑分红最高 | 推荐"最符合客户利益"而非最高收益 |
| 陈伯指保证回报而非画饼 | 提供合适、公正、客观的意见 |
| 林若兰看懂的那一眼 | "公平对待客户"的实质 |
