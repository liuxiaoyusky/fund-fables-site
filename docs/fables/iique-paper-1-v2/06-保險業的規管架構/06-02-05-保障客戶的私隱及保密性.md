# 6.2.5.3.4 保障客户的私隐及保密性

## 📜 寓言故事 —— 《三道口》

那天傍晚,陈伯正在打烊前理账,一个西装笔挺的男人推门进来,自称是某家银行的职员。

"陈先生,我们行里在办一笔贷款,申请人叫何太。她说她上个月在您这儿保了一份住院险。我想跟您核实一下,她保的金额是多少,还有,她填的家底——物业、收入那些——您能不能给我看看?"

陈伯的手在账本上停住了。何师奶的事他是知道的。她怕意外连累家人,瞒着茶餐厅的伙计偷偷来保的,临走还叮嘱过一句:"陈伯,这事别跟人提。"

陈伯把账本合上,抬头看那男人。"先生,何太的事,我不能跟你说。"

男人眉头一皱。"我是银行来的,办正经贷款。这算什么机密?"

"您是办正经事,我信。"陈伯的语气不重,但很稳,"可何太的资料,是冲着买保险才交到我手上的。她没点头让我转给别人——不光没口头点头,连张白纸黑字都没签。这资料在我这儿,就是锁在抽屉里的事。"

男人还想说什么,陈伯摆了摆手。"我给您交个底,这资料我能说的情形,只有三种。"

他伸出三根手指。

"头一种,本来就是为了办她这份保险才用得上的。比方说保险公司要我转去核保、去理赔,这叫分内事,我照办。"

"第二种,她白纸黑字写下来,同意我给谁。口头不算,得签字。今天她没签,我就不能开口。"

"第三种,法律有规定,某个部门依法要我这资料,不交不行。您要是带了法院的令来,或者哪条法例明写着银行可调,我二话不说。"

男人沉默了。他掏了半天口袋,只摸出一张银行的名片,没有令,也没有何太的同意书。

陈伯把名片轻轻推回去。"先生,您回吧。您要何太的资料,得去找何太本人。"

男人走后,梁志明从里间探出头,小声问:"陈伯,刚才那位,看穿着是做大买卖的。您就不怕得罪他?"

陈伯把账本重新翻开。"志明,你记着,客户把家底交到你手上,是信你。这份信,不是让你拿去换人情的。她没点头、法律没开口,这资料在我嘴里就是块石头,沉到海底也漏不出来。"

梁志明没再说话,只默默把这件事记进了心里。

---

📖 原文定义

> 持牌保险代理人应视所有与客户有关的资料为机密资料,及不得使用或披露该等资料,唯下列情况除外:(i) 为进行受规管活动,而该等资料是为有关活动而提供的;(ii) 经客户书面同意;或(iii) 为遵守适用于该代理人并要求其作出披露的任何法律或规例。就持牌保险代理人在进行受规管活动的过程中收集的客户个人资料,有关代理人必须遵守《个人资料(私隐)条例》(第486章),并应遵从个人资料私隐专员就个人资料的收集、保存、使用及保安所发布的相关指引。

💡 对应点

| 故事元素 | 概念对应 |
|---------|---------|
| 何师奶叮嘱"别跟人提" | 客户资料视为机密资料 |
| 银行职员索要资料 | 非授权第三方的披露请求 |
| 陈伯伸出的三根手指 | 三种可披露的例外情形 |
| 核保/理赔转交资料 | 第(i)项:为进行受规管活动 |
| 陈伯强调"口头不算,得签字" | 第(ii)项:经客户书面同意 |
| 法院令或法定要求 | 第(iii)项:法律或规例要求 |
| 陈伯说"沉到海底也漏不出来" | 保密义务的刚性 |
