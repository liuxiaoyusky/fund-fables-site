# 一、指令驱动与报价驱动交易机制

## 🏺 寓言故事 —— 《三种交易方式》

镇上有三种做买卖的方式。

### 第一种：公开竞价（指令驱动）

镇中心有个大广场，买卖双方把报价写在纸条上扔进一个箱子。箱子按"价格优先、时间优先"撮合成交：
- **价格优先**：买的人出价越高越先成交，卖的人要价越低越先成交；
- **时间优先**：同样价格下，先报价的先成交。

这叫**指令驱动**（也叫订单驱动）。全球大多数证券交易所都采用这种方式。交易分两种：
- **集合竞价**：规定时间内接受的买卖申报一次性集中撮合（如开盘、收盘）；
- **连续竞价**：逐笔连续撮合。

交易指令分两类：
- **市价指令**：以当前市场价格立即成交。快速但面临价格波动风险；
- **随价指令**：设定某个价格才成交。包括：
  - **限价指令**：指定买入最高价/卖出最低价；
  - **止损指令**：价格达到某个阈值时自动执行，控制损失。

### 第二种：做市商报价（报价驱动）

镇上有个大商人，他手里总是有货，也随时愿意买货。他持续报出买入价和卖出价，买卖双方直接跟他交易。他的利润来自**买卖价差**。

这叫**做市商制度**（报价驱动）。做市商需要雄厚的资本和强大的定价能力，持续提供双边报价，维持市场流动性。

### 第三种：私下议价（询价交易）

买卖双方私下见面，一对一谈价格。这用于大宗交易、非标准化产品或流动性较低的证券。

我国银行间市场大部分交易通过询价完成，包括请求报价、做市报价、指示性报价等方式。

---

**📖 原文定义**

> 指令驱动，也称"订单驱动"（Order Driven），是一种竞价交易方式。市场价格由买卖双方提交的包含数量和价格的指令驱动，并通过竞价配对而产生。

> 报价驱动（Quote Driven）的核心是做市商制度。做市商持续向市场提供证券的买入价和卖出价，投资者直接与做市商进行交易。

> 询价交易是指交易的一方向一个或多个潜在交易对手询问和商议价格，然后选择最优报价进行交易的方式。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 广场公开竞价 | 指令驱动/订单驱动 |
| 价格优先时间优先 | 撮合成交原则 |
| 大商人持续报价 | 做市商制度/报价驱动 |
| 买卖价差 | 做市商利润来源 |
| 私下议价 | 询价交易 |
| 市价指令 | 按当前市场价格成交 |
| 限价指令 | 设定价格才成交 |
| 止损指令 | 价格触达自动执行 |
