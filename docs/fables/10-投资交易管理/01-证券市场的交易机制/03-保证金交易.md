# 三、保证金交易

## 🏺 寓言故事 —— 《借钱下注》

阿智有1万元，想赌股票涨跌。正常情况下只能买1万元的股票。但他听说赌坊可以**借钱给他**，让他以2万元下注——这就是保证金交易。

### 基本概念

保证金交易中，投资者向证券经纪商**借入资金或证券**，以少量自有资金撬动更大头寸。

- **融资（买空）**：借入资金买股票，看涨时使用；
- **融券（卖空）**：借入股票卖出，看跌时使用。

保证金率 = 自有资金 / 总投资额。例如1万元自有资金买2万元股票，保证金率 = 50%。

### 融资交易示例

阿智自有资金1.5万元，融资1.5万元（保证金率50%），以150元/股买入200股，总投资3万元。

**维持担保比例** = 股票市值 / 融资额。

- 股价跌至120元：维持担保比例 = (120×200)/15000 = **160%**；
- 股价跌至97.5元：维持担保比例 = **130%**（最低限额）。

若股价继续下跌，跌破130%，证券公司发出**保证金催缴**，要求2个交易日内追加担保至150%以上。否则**强制平仓**。

### 杠杆效应

融资交易的本质是**杠杆**：
- 股价涨20%（到180元），阿智收益率 = (180×200 - 30000 - 利息) / 15000 = **35.7%**；
- 股价跌30%（到105元），阿智收益率 = **-64.3%**。

收益和亏损都被放大了！即使股价不变，也要支付融资利息。

### 融券交易

融券是反向操作：借股票卖出，等股价跌了再买回来归还，赚取差价。同样需要维持担保比例不低于130%。

---

**📖 原文定义**

> 保证金交易（也称为杠杆交易）是指投资者从证券经纪商借入资金或证券，以较少自有资金撬动更大规模投资头寸的交易方式。

> 维持担保比例 = 股票市值 / 融资额。当维持担保比例小于130%时，投资者会接到保证金催缴通知。

> 融资交易将股票上涨时的利润扩大化，但也会同样放大损失率。
