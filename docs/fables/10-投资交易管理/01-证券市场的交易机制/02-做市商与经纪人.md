# 二、做市商与经纪人

## 🏺 寓言故事 —— 《商人 vs 中介》

镇上有两种帮人做买卖的角色，但本质完全不同。

### 做市商

做市商是**亲自下场做生意的商人**。他自己有货，自己出价买卖：
- 他报出一个**买入价**（他愿意出的买价，较低）；
- 他报出一个**卖出价**（他愿意卖的价，较高）；
- 买卖价差就是他的利润。

投资者直接和他交易，不和其他投资者打交道。做市商需要提供持续的双边报价，保持市场流动性。

### 经纪人

经纪人是**撮合买卖的中介**。他自己不买卖，只帮客户找对手方：
- 你找他买股票，他去市场上帮你找卖家；
- 你找他卖股票，他去市场上帮你找买家。

他的收入是**佣金**。他只撮合交易，不参与交易本身。

### 关键区别

| 对比维度 | 做市商 | 经纪人 |
|---------|-------|-------|
| 是否参与交易 | 是，直接与投资者交易 | 否，仅撮合 |
| 利润来源 | 买卖价差 | 佣金 |
| 提供流动性 | 是，持续双边报价 | 否 |
| 适合的品种 | 流动性较低、非标准化产品 | 标准化、高流动性产品 |

我国的货币经纪公司是银行间市场的经纪人，证券经纪商是交易所市场的经纪人。ETF的流动性服务商则兼具做市商的功能。

---

**📖 原文定义**

> 做市商通过与投资者进行买卖双向交易来提供市场流动性。做市商的利润主要来自证券买卖差价。

> 经纪人仅作为中介撮合交易或执行投资者的指令，并不直接参与到交易本身。经纪人的收入主要来源于为客户提供经纪服务的佣金。

> 对于流动性低的交易品种，做市商通常不会为其做市；指令驱动市场也不适合这类交易品种。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 亲自买卖的商人 | 做市商 |
| 买卖价差 | 做市商利润 |
| 帮找买家卖家的中介 | 经纪人 |
| 佣金 | 经纪人收入 |
| 流动性差的品种 | 需要做市商 |
| 标准化高流动性品种 | 指令驱动市场 |
