# 三、基金业绩评价应考虑的因素

## 🏺 寓言故事 —— 《六条评分规则》

评价基金业绩，不能只看收益率，要遵守六条规则。

### 规则一：看管理规模

规模大的基金单位份额固定费用低，分散投资效果更好。但规模过大可能带来流动性冲击和调仓困难。评价时要考虑规模效应和局限性。

### 规则二：看时间区间和市场环境

- 短期表现可能反映市场短期波动；
- 长期表现更能反映基金经理的真实能力；
- **同一类型基金的业绩必须在同一时段内比较**；
- 要关注基金在**市场下行期的抗风险能力**。

### 规则三：看风险调整后收益

投资管理的本质是风险管理。基金业绩评价的核心在于衡量其**产生风险调整后超额收益的能力**。正的风险调整后超额收益是主动管理型基金为投资者创造经济效益的重要体现。

### 规则四：防幸存者偏差

只考虑目前仍存续的基金，忽略了已经清盘的差基金，会导致对整体业绩的高估。评价时应包括已终止运作的基金。

### 规则五：看费用

费用直接影响净回报：
- **费前收益率**：扣除交易成本但未扣除管理费，反映投资活动本身的表现；
- **费后收益率**：扣除所有费用后的实际净回报。

比较时必须使用一致、可比的口径。

### 规则六：看投资风格一致性

基金实际风格是否偏离宣称的风格？风格漂移会导致风险收益特征变化，影响投资者的资产配置决策。

---

**📖 原文定义**

> 基金管理规模对其投资运作效率有着显著影响。

> 基金的表现既受到时间区间的影响，也与市场环境和经济周期密切相关。

> 基金业绩评价的核心在于衡量其风险调整后的收益能力。

> 幸存者偏差指的是在评估基金业绩时，只考虑当前仍然存续的基金，而忽略了那些由于表现不佳或其他原因而终止运作的基金。

> 基金的各项费用支出直接影响基金最终的净回报。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 规模太大调仓难 | 管理规模影响 |
| 同一时段比较 | 时间区间可比性 |
| 赚得多但风险也大 | 风险调整后收益 |
| 只看活下来的好学生 | 幸存者偏差 |
| 扣除各种费用后的净回报 | 费后收益率 |
| 实际风格是否偏离宣称 | 投资风格一致性 |
