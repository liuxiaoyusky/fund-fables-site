# 第四节 基金财务会计报告分析

## 二（五）基金份额变动分析

## 📜 寓言故事 —— 《十六铺码头人丁簿》

清明前后，十六铺码头的陆记米行刚忙过一阵南洋粳米的大单。账房先生孙掌柜把红皮账册递给老陆，账册末页附了一张"人丁簿"——记的是这一年里跟陆记做买卖的街坊，总共多少份子。

老陆翻开一看，眉头皱起来："孙掌柜，年初咱陆记一共挂着三百四十万零五千零六份半的份子；到了年尾巴，只剩二百九十二万四千五百四十三份半。整整少了四十八万零六百六十二份半。"

孙掌柜点头："东家，是这么个数。这一年下来，进来的街坊少，出去的街坊多，份子缩了。缩水一成四还多。"

老沈正好来喝茶，听见这话搁下杯子："陆兄，人丁簿缩水一成四，那可不是小数目。"

老陆叹气："可不是嘛。份子这么一来一回地折腾，咱这摊子可就被动了。"

老沈指着人丁簿的首页："份子大进大出，对东家有两桩不好。第一桩——今天进十个街坊，明天走八个；今天进五十个，明天走四十个。进来的人多，街上流言也多，进价要稳也稳不住；走的人多，又得忙着善后，把刚到手的米贱卖出去回钱。第二桩——进进出出的人多了，每天光顾着'接待新客、安抚走客'，哪还有心思琢磨下一批米该囤哪路？"

老陆越听越沉："难怪今年南洋粳米这一单，我举棋不定。"

"正是。"老沈说，"反过来，份子稳或者慢慢变大，对东家就好。街坊认可陆记的米、认陆记的价、认陆记的路，进的人多走的人少，或者一个不走，咱手头这摊货，就能安安稳稳地做长线。"

老冯从外头跑进来，递上一张红纸条："陆兄，城北郑老大刚派人来问，今年能不能在陆记多挂一份。他说街坊里几家大馆子都信咱的米，想一并合过来。"

老陆接过红条看了一眼，递给老沈："你瞧瞧，这就是街坊大客的认可。"

老沈点头："这就要看人丁簿上头一栏——'大客'那一栏。街坊里有大客、也有散客；大客一出手就是几百份，散客不过三五份。大客多了，说明陆记在外头的口碑硬，街面上的大买卖人家都认咱；散客多也不打紧，是街坊小门小户过日子。怕就怕大客也没几个，散客也跑得七零八落，那陆记这摊子就真撑不住了。"

孙掌柜把红条夹进人丁簿，插嘴道："东家，这一成四的缩水里头，我数了数，跑的大半是散客；城北郑老大和钱庄恒泰的几位大客，倒是一份没动。"

老沈击掌："那陆兄你心里该有数了。散客走了，份子一时难看，但大客稳着，说明陆记的根还在。把进出的口子收一收，今年稳住这几位大客，再把散客慢慢养回来，份子自然就回得来。"

老陆合上人丁簿，长出一口气："从前我只顾着看赚了多少钱，如今才知道，进的人多少、走的人多少、大客多不多——这本簿子，比账册还重要。"

老沈拱手："陆兄，开年稳住。"

老陆回礼："十六铺码头人丁簿，年年得细看。"

---

**📖 原文定义**

> 对于开放式基金来说，基金的份额会随申购、赎回活动而变动，基金定期报告中一般会披露基金份额的变动情况和基金持有人的结构。通过对基金份额变动情况和持有人结构的比较分析，可以了解投资者对该基金的认可程度。
>
> 一般来说，如果基金份额变动较大，则会对基金管理人的投资有不利影响；反之则有助于基金投资的稳定。
>
> 如果基金持有人中机构投资者较多，表明机构比较认可该基金的投资。
>
> [案例15-12] 报告期期初份额 3405206147.11 份，期末 2924543566.34 份，本期变动 -480662580.77 份，约占期初 14.12%。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 陆记米行"人丁簿"：跟陆记做买卖的街坊总份子 | 基金份额及持有人结构 |
| 年初 340.5 万份 → 年末 292.4 万份，少了 48 万份 | 报告期期初份额 3,405,206,147.11 份 → 期末 2,924,543,566.34 份 |
| 缩水一成四 | 本期变动 -480,662,580.77 份，约占期初 14.12% |
| 份子大进大出，对东家两桩不好：进价稳不住 + 忙于善后、无心长线 | 份额变动较大对管理人投资的不利影响 |
| 份子稳或慢慢变大，东家安安稳稳做长线 | 份额稳定或增加有助于基金投资稳定 |
| 街坊大客（郑老大、钱庄恒泰）一份没动 | 机构投资者持有稳定 |
| 人丁簿上"大客"一栏 | 持有人结构中的机构投资者占比 |
| 大客多 = 街坊大买卖人家认可陆记 | 机构投资者较多表明机构认可该基金 |
| 散客走了大客没走 = 陆记的根还在 | 关注份额变动的同时要分析持有人结构 |
