# 二、基金财务会计报告分析的主要内容

## 🎯 寓言故事 —— 《晋商票号的五张药方》

宣统二年的深秋，平遥城外的官道上黄叶纷飞。

查账师爷范澄背着旧皮箱，再度走进"汇通号"票号的大门。这一回不是受人所托，而是老掌柜李兆麟亲自写信请来的——上回范澄替票号查过账后，李兆麟一直惦记着他那句"季表比年表更早明白"。这大半年里，汇通号照范澄的建议，按季做了一份季表。如今四季过去，年关将至，李兆麟要把这四份季表连同去年那张年表，**一并请范澄再过一遍**。

后堂里，李兆麟、老账房孙禄、查账师爷范澄三人围炉而坐。炕桌上摊开五张表——四份季表、一份年表。

"范先生，"李兆麟亲手斟茶，"这大半年汇通号的进出，全在这五张表上。您上回说'查账有五桩事'，今儿还请再细细分说。"

范澄把茶盏推回，拿起第一份季表，眯起眼看了一会儿，开口道：

"**头一桩，瞧持仓。**"

他指着表上几行小字："李掌柜，您今春把八万两银子贷给了三家茶庄、两家绸缎庄，秋后又收回了多少？如今银库里实存的现银、库里押着的欠条、柜上留给存户备付的头寸——这三样各占多少？"

"现银三成五，欠条五成，备付一成五。"孙禄答得流利——他做了十九年账房，这些张口就来。

"这就叫**'持仓结构'**。"范澄点头，"银子压在欠条上还是压在现银上，是不同的生意。**现银多则稳，欠条多则利厚**。再者，您这欠条是茶庄的多还是绸缎庄的多——这便是'**行业分布**'，能看出您这票号的主业在哪一行。"

他又补一句："**持仓虽稳，亦要看天时**。哪怕您今春一两银子没动，可市面上的行情跌了，欠条上的抵押物缩水了，那您'持仓'的比例**也是会变的**。这便是我常说的一句——**占比的小动，未必是掌柜动了手脚，也可能是行情动了手脚**。"

李兆麟点头："记下了。"

范澄又拿起第二份季表，翻了几页，眉尖一挑：

"**第二桩，瞧盈利能力与分红。**"

他指着表中"本年利润"四个字："李掌柜，您这一年净赚了多少？"

"一万四千两。"孙禄答。

"隔壁'日升昌'呢？"

"听闻是一万六千两。"

"哦？咱比他少赚两千两。"李兆麟皱眉。

范澄摆手："**莫急**。您看他票号本钱多少？"

"八万两本金。"

"咱呢？"

"七万两本金。"

范澄笑了："**这便看出门道了**。日升昌赚一万六，本金八万——**赚头占本金两成**；您赚一万四，本金七万——**赚头占本金也是两成**。两家其实**盈利能力一样**。"

孙禄恍然："原来**看利润不能光看绝对数，要看占本金的比例**。"

"正是。"范澄又指着表上一行小字，"再看这行——'**本期已实现收益**'。什么叫'**已实现**'？是实打实进了口袋的银子，扣掉那些只是账面上浮动的。**浮盈不算赚，落袋才算赚**。"

他再翻到年表最后一栏："还有一桩——**分红**。您今年赚了一万四千两，可您打算分给存户多少？"

"约摸五千两。"

"那就要算**期末可供分配利润**——是**未分配利润**与**未分配利润中已实现部分**的**孰低数**。为何取'**孰低**'？是规矩——**未落袋的钱不能分**。您赚了浮盈一万，未实现的部分，**不能拿去分给存户**。"

李兆麟拊掌："妙！这五项算得明白，分红才能分得稳当。"

范澄端起茶盏呷了一口，又翻到第三份季表：

"**第三桩，瞧收入从哪来。**"

他指着"利息收入"、"投资收益"、"汇兑手续费"、"其他收入"四栏："您这票号一年进项，从这四口子里来。**利息**是放贷的收成，**投资收益**是入股丝茶行号的分红，**汇兑手续费**是替人汇银的手头费，**其他收入**是杂项。"

"这四口子的**比例**就看出您这票号的**路子**——**利息高的走的是放贷型**，**投资收益高的走的是投资型**，**汇兑手续费高的才是正经票号**。您这三季下来，**利息占七成、投资占两成、汇兑占一成**——您这票号，**是放贷多过汇兑**啊。"

李兆麟脸色微变。范澄笑了笑："**不是坏事，但要看您想往哪条道走**。票号之所以是票号，**汇兑才是本业**；放贷再多，**终究是钱庄的活儿**。"

"存户若不细看，单看总利润，定以为您这票号兴隆；可若细看收入结构，便知您**正渐渐从票号变成钱庄**。**这便是分析收入结构的好处**。"

范澄翻到第四份季表：

"**第四桩，瞧费用花在哪。**"

他指着"掌柜月俸"、"伙计月钱"、"库房租银"、"柜上杂费"、"税局厘金"五栏："赚了一万四千两，**花了多少**？"

"约摸三千两。"孙禄答。

"**掌柜月俸与伙计月钱占多少**？"

"约一半。"

"**柜上杂费与税局厘金占多少**？"

"约三成。"

"**剩下一成呢**？"

"库房租银。"

范澄点头："费用中有**人事费、营业费、税金、杂项**，每一项都该**逐年比一比**。**今年人事费比去年涨一成**，是加了人还是加了月俸？**今年杂费比去年涨半成**，是库房多租了还是柜上的笔墨纸张贵了？"

他顿了顿，又道："您可记得，**买卖股票的差价收入里，要扣一项'交易费'**？这交易费，**新规矩下不单列，全混在'投资收益'里**。从前的账是分开的，**如今是混的**。您若不细看，会以为交易费不见了——**其实没不见，只是藏进了投资收益的口袋**。"

李兆麟点头："**这便是细看费用的门道**。"

范澄最后翻到第五份季表。这一份表是孙禄特意新制的——专门记存户的进出。

"**第五桩，瞧存户的来与去。**"

"李掌柜，您今年存户是**多了**还是**少了**？"

"春上多了二百户，秋后又跑了一百五十户。"

"那您票号的银子是**多了**还是**少了**？"

"多了三万两。"

范澄眯起眼："**人少了，银却多了**——这说明什么？"

孙禄抢答："**大户来了，散户走了**。"

"对。"范澄抚掌，"**人多人少，看热闹；银多银少，看门道**。"

他又指着表中"机构存户"与"个人存户"两栏："您今年机构存户的银子涨了两成，**说明大商家越发信您**；个人存户的银子跌了一成，**说明小户有些动摇**。机构多则**生意稳**，个人多则**生意活**。您这票号，如今**机构占六成、个人占四成**——**机构太重，恐有'一户撤银、满盘皆动'的风险**。"

李兆麟听得后背发凉："**原来一份存户表，能读出这许多文章**。"

范澄合上五张表，缓缓道："李掌柜，**持仓、盈利、收入、费用、存户**——**这五张表便是票号的五张药方**。一张表诊一症，**五张表合诊，便知全身**。"

"从前您只看一张总表，**赚了多少、赔了多少**——这是看病只看一症。**如今分作五张细看，**方知这病是出在**持仓**上、还是出在**收入**上、还是出在**存户**上。**病不同，药便不同**。"

孙禄长叹："**从前我做账，是把五件事搅在一锅；今日查账，才知该分作五道看**。"

范澄点头："**会做账的是匠人，会查账的才是东家**。李掌柜，您这汇通号若再传三十年，靠的不是孙先生的算盘，**而是您这五张表**。"

窗外北风渐起，黄叶落尽。范澄把五张表叠好，亲手交还李兆麟。

"李掌柜，**年关将至，五张表合一份年表，记清楚这一年的来龙去脉**。明年开春，我再来。"

李兆麟起身相送，拱手道："范先生，**您这一席话，抵得上汇通号半年的进项**。"

范澄笑着摆摆手，背起旧皮箱，踏入风雪中。

## 📖 原文定义

> 基金作为一种进行证券投资的资产组合，与普通企业的财务会计报告的分析有很大的不同。不同类型的基金其分析的方法和内容也不一样。一般来说，对股票基金及混合基金的财务会计报告的分析包括五个方面：基金持仓结构分析、基金盈利能力和分红能力分析、基金收入情况分析、基金费用情况分析、基金份额变动分析。

> 基金定期报告的投资组合报告中要求披露基金资产组合情况，包括权益投资、固定收益投资、银行存款和结算备付金合计等项目占基金总资产的比例。

> 权益投资占基金总资产的比例如发生少量变动，并不意味着基金经理一定进行了增仓或减仓操作，因为市场波动也可能引起计算结果的变动。

> 本期利润反映了基金本期产生的净收益（或净亏损），是指本期利息收入、投资收益、公允价值变动收益、其他收入等收入项目扣除管理人报酬、托管费、利息支出等费用项目后的余额。

> 本期已实现收益反映了基金本期产生的已实现净收益（或净亏损），是指本期利润扣除公允价值变动收益后的余额。

> 期末可供分配利润反映了基金截至目前的可分红金额，是指期末未分配利润与未分配利润中已实现部分的孰低数。

> 基金收入包括利息收入、投资收益、公允价值变动损益和其他收入。

> 基金费用一般包括管理人报酬、托管费、销售服务费、利息支出、信用减值损失、税金及附加和其他费用。可以看到，基金费用中并没有单独列示交易费用，这是因为在新金融工具准则下，基金发生的交易费用都已经反映在了投资收益。

> 对于开放式基金来说，基金的份额会随申购、赎回活动而变动，基金定期报告中一般会披露基金份额的变动情况和基金持有人的结构。

## 💡 对应点

| 故事元素 | 概念对应 |
|---------|---------|
| 五张表：持仓、盈利、收入、费用、存户 | 股票基金/混合基金财务报告分析的五个方面 |
| 现银三成五、欠条五成、备付一成五 | 权益投资、固定收益投资、银行存款及结算备付金合计占比 |
| 欠条上茶庄多还是绸缎庄多 | 股票投资在各行业的分布，分析基金重点投资方向 |
| "占比的小动未必是掌柜动了手脚，也可能是行情动了手脚" | 持仓比例变动需注意：市场波动也可能引起计算结果变动 |
| 利润绝对数（日升昌 1.6 万 vs 汇通号 1.4 万） | 本期利润：反映净收益的绝对金额 |
| 赚头占本金两成 vs 两成 | 净值利润率：比较不同基金盈利能力的相对指标 |
| "浮盈不算赚，落袋才算赚" | 本期已实现收益：本期利润扣除公允价值变动收益 |
| "未落袋的钱不能分" | 期末可供分配利润：未分配利润与未实现部分的孰低数 |
| 利息占七成、投资两成、汇兑一成 | 收入结构分析：利息、投资、汇兑、其他四类收入的占比 |
| "票号若变了钱庄，从收入结构便能看出" | 通过收入结构分析可深入了解基金投资状况 |
| 人事费、营业费、税金、杂项逐年比 | 基金费用情况分析：管理人报酬、托管费、销售服务费等 |
| 交易费"藏进了投资收益的口袋" | 新金融工具准则下交易费用不单列，已反映在投资收益中 |
| 大户来了、散户走了 | 份额变动与持有人结构分析 |
| 人少银多 = 机构占比上升 | 机构投资者占比上升的意义 |
| "机构太重，恐有'一户撤银、满盘皆动'" | 份额变动较大对基金管理人的不利影响 |
| 五张药方合诊全身 | 五个方面综合分析基金财务报告 |
