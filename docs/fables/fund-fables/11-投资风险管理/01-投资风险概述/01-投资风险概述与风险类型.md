# 一、投资风险概述与风险类型

## 🏺 寓言故事 —— 《米铺这一年》

老冯在码头东头开了一爿米铺，开到第三个年头，生意刚稳下来。头一年他只卖本地粳米，第二年加了糯米和糙米，第三年又添了一间小仓房，专门囤远路来的香米。铺子对面是顾先生的茶馆，西边一墙之隔是老陆的杂货铺，再过去两间门面，便是老沈的木行。

这一年腊月三十晚上，老冯独自坐在米缸边算账。顾先生端着一壶酒推门进来，往米袋上一坐，笑了："老冯，你这铺子开了三年，我看你这大半年眉头没松过。"

老冯叹口气，把账本翻给他看。"顾先生，您是没见这一年的事。开春那会儿，公家在城门口贴了告示，说粳米不许再往北码头运——咱们铺子十个主顾里三个是北码头的船主，一纸告示下来，他们绕道去了南头。我这三成的生意，眼睁睁看着没了。"

顾先生点点头，没接话，只给他倒了杯酒。

老冯接着说："到了四月，本该是青黄不接、米价往上走的时节，谁知连着下了二十天雨，南边几县报上来都是好年景，米一船一船涌进码头，价钱一天比一天低。我那仓里压着三十石陈米，进价高、出价低，卖一石亏一石。"

"那您怎么熬过来的？"顾先生问。

"熬到五月底，米价稳住了，我才敢进货。"老冯说，"可六月里，朝廷突然加了一厘息，街面上借钱贵了三分。我手里压着一张去年发的糯米票子——当时是折价收来的，约定年底兑米。朝廷息钱一加，这票子在市面上转手就跌了价。原本说好了兑十石糯米，到七月底去兑，人家只肯兑八石。"

顾先生皱皱眉："那票子的事先放一放。后来呢？"

"后来更邪门。"老冯苦笑，"七月往后，街上银根紧，铜钱也毛了。我记得很清楚，正月里一两银子能买一石半好粳米，到九月一两银子只能买一石二。米是卖出去了，银子收回来，年底一算账，账面赚了三成，可实打实能买到的米，比正月还少。"

"还没完。"他给顾先生续了酒，"到了九月，老沈的木行要进一批南洋红木，托我借他二十石糯米做周转，说好三个月后还。我那会儿手头米多，就应了。哪想到十月里老沈的木船在江上翻了木料，本钱亏了大半，他人跑了。三个月后老沈没还米，我垫出去的那二十石，年底也没了踪影。"

老冯把杯子放在膝头："您说，这一年，我躲哪样了？"

顾先生没立刻答，往他米缸边走了两步，掀开缸盖。缸里只剩半缸米，米粒压得紧紧实实。顾先生回头看他："你缸里这些米，打算什么时候出手？"

"再等等。"老冯说，"年底米价翘一翘，过了年能卖个好价。"

"万一开春大家都要用钱，你急着兑米换现钱，街上却没人接呢？"顾先生问，"你这一缸米，压到正月里，谁来买？你想脱手，人家压你三成价；你不脱手，开春要付进货的款、付伙计的工钱，缸里这米又不顶现钱用。"

老冯愣住了。

顾先生把缸盖轻轻盖回去，站起来拍拍他肩："老冯，你这一年的难处，归起来其实就几样——公家改了规矩，市面冷了又热，朝廷加了息钱，铜钱毛了不值钱，借出去的米对方跑了，缸里米压着换不成现钱。哪一样都不是你做错了什么，可哪一样你都躲不开。"

"那我以后怎么办？"老冯问。

"看天吃饭的人，得知道天有几张脸。"顾先生说，"公家再改规矩，你得听得见；市面冷热，你得认得清；朝廷息钱一动，你得立刻回去翻一翻自己手里的票子；铜钱毛不毛，你得常去米市问问一两银子几斗米；借钱出去，看人要看准了；缸里压货，要给自己留一口活钱。这些事你做不全，可心里得有数。"

门外爆竹声响起来，码头上的年味已经飘到铺子里来了。老冯端着酒杯，看着自己那半缸压得紧紧实实的米，忽然明白——这一年哪一桩事他都没躲过去，可也正是这一桩一桩的事，让他学会用眼睛看天。

**📖 原文定义**

> 市场风险是指基金投资组合因宏观政治、经济、社会等环境因素影响而面临的风险。当基金投资于股票市场时，上市公司的股价不但受其自身业绩和所属行业的影响，还会受经济政策、经济周期、利率水平等宏观因素的影响，导致价格的不确定性；当基金投资于债券市场时，债券价格会随着利率的变动而波动：利率上升，债券价格下跌。因此，尽管基金通过分散投资来降低非系统性风险，但由于股票和债券等市场固有的系统性风险，基金仍不可避免地面临市场风险。

> 市场风险主要包括政策风险、经济周期性波动风险、利率风险、购买力风险、汇率风险等。

> 政策风险是指宏观政策变化对基金收益的影响。宏观政策，例如财政政策、产业政策、货币政策等，都会对金融市场产生影响，进而影响基金的收益水平。

> 经济发展有周期性，在不同的经济周期，不同资产也会呈现出不同的运作规律，进而影响基金投资业绩。

> 利率风险指的是因利率变化对资产价格造成不确定性的风险。利率变动主要受通货膨胀预期、中央银行的货币政策、经济周期和国际利率水平等影响。

> 购买力风险，通常也称为通货膨胀风险，是指由于通货膨胀等因素导致货币购买力下降，从而降低基金的实际收益，影响投资者的实际回报。

> 汇率风险指因汇率波动导致资产价格产生不确定性的风险。对于投资海外市场的基金，特别是合格境内机构投资者（QDII）基金，汇率波动对其资产价格的影响尤为显著。

> 信用风险源于债务人无法履行其财务义务的可能性。在基金投资中，信用风险主要涉及债券发行人以及回购交易和衍生品交易对手。

> 流动性是资产在短期内以接近公允价格快速完成市场交易的能力。流动性风险则是指在特定时间内无法以合理价格快速买卖资产的风险。

> 基金管理人管理的投资组合还面临资产价格的波动带来的投资风险。风险并非总是需要避免或消除。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 公家告示：粳米不许卖北码头 | 政策风险（财政/产业/货币政策变化） |
| 丰年米涌进码头、价钱跌 | 经济周期性波动风险 |
| 朝廷加了一厘息、糯米票子跌 | 利率风险（利率上升→债券/固定收益凭证价格下跌） |
| 一两银子从一石半米跌到一石二 | 购买力风险（通货膨胀） |
| 沈家木行借米跑路 | 信用风险（债务人无法履约） |
| 缸里米压紧、急用钱兑不出合理价 | 流动性风险（短期内无法以合理价格快速变现） |
| 老冯分散经营粳米/糯米/糙米/香米 | 分散投资降低非系统性风险 |
| 五种"天要塌"老冯哪样都躲不过 | 系统性风险不可避免、仍要面对市场风险 |
| 顾先生最后点出几件事都得有数 | 风险管理的本质——认识、衡量、并接受可承受的风险 |
