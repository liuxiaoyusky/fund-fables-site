# 一、投资风险概述与风险类型

## 🏺 寓言故事 —— 《三头怪兽》

投资世界里有三头怪兽，随时可能袭击你的资产。

### 第一头：市场风险

这头怪兽来自宏观环境的变化：
- **政策风险**：政府政策突然改变，如加息、限购、行业监管收紧；
- **经济周期风险**：经济从繁荣到衰退，不同资产表现截然不同；
- **利率风险**：利率上升，债券价格下跌；利率下降，债券价格上涨；
- **购买力风险（通胀风险）**：钱越来越不值钱，实际收益被侵蚀；
- **汇率风险**：投资海外资产时，汇率波动影响实际收益。

### 第二头：信用风险

这头怪兽来自交易对手可能违约：
- 债券发行人无法按期支付利息或本金；
- 债券评级下调导致价格下跌；
- 回购交易对手不履行回购义务。

管理信用风险的核心：选择高信用等级的债券，分散投资，避免对单一发行人过度集中。

### 第三头：流动性风险

这头怪兽让你"有钱花不出去"或"想卖卖不掉"：
- 资产端：市场流动性不足，无法按预期价格买卖资产；
- 负债端：开放式基金面临巨额赎回，被迫低价抛售资产。

流动性风险是综合性风险，市场、信用、操作等风险的缺陷都可能引发流动性危机。

### 风险管理的本质

风险不是要避免，而是要**管理**。风险管理的本质是在回报和可能的损失之间寻找平衡——确定可以接受的风险水平（**风险预算**），然后持续监测并调整，确保实际风险在预算范围内。

---

**📖 原文定义**

> 风险管理的本质并非预测未来，而是于不确定性中探寻方向。风险管理的目标也并非将风险暴露降至最低，而是在回报和可能的损失之间寻找平衡。

> 市场风险是指基金投资组合因宏观政治、经济、社会等环境因素影响而面临的风险。

> 信用风险源于债务人无法履行其财务义务的可能性。

> 流动性风险则是指在特定时间内无法以合理价格快速买卖资产的风险。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 三头怪兽 | 三类主要投资风险 |
| 政策、利率、通胀、汇率变化 | 市场风险 |
| 对手赖账 | 信用风险 |
| 想卖卖不掉 | 流动性风险 |
| 确定能承受多少损失 | 风险预算 |
| 在回报和损失间找平衡 | 风险管理本质 |
