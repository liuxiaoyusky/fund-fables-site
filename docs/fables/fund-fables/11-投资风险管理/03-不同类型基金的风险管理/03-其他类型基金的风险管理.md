# 三、其他类型基金的风险管理

## 🏺 寓言故事 —— 《四方米铺》

老陆的"粳米股票号"开张三个月，南码头又冒出三家米铺。老沈闻着味儿来串门："你这铺子前脚才立稳，后脚就跟了三个抢食的。"老陆把茶壶递过去："抢就抢呗，米缸多了码头才热闹。"门外"吱呀"一声，老冯推门进来，身后跟着个生面孔——四十几岁，灰布巾，自称姓吴，从北边来的。

四个人坐定，老陆便把这三家新铺的底细摆出来。第一家是"三合号"，掌柜姓杜，圆脸，眼睛滴溜溜转。他家铺子里米色最多——粳米、糯米、糙米三只缸并排摆着。老冯问："三只缸各占多少？"杜掌柜嘿嘿一笑："看行情。米山要涨，粳米就多摆点；要跌，糯米就压上去。配比是活的，三天一小调，半月一大调。"

第二家是"短票号"，掌柜姓葛，干瘦，眼睛亮得像铜铃。他家铺子小得出奇——柜台后面只放着一只木匣子，匣子里是几张盖了红印的票子。葛掌柜说，他只做"短票生意"——今天收一张明天到期的，最长也不超过四个月。老冯皱眉："这么短的票子能赚几个钱？"葛掌柜把匣子往桌上一拍："赚得少，但稳。票子到期不欠。"他补了一句，"匣子里的票子每天都得对一遍价。价对得上就卖，对不上就停一停。"

第三家最怪，叫"跟山号"，掌柜姓庄，戴细圆眼镜，话不多。他家铺子没柜台，只挂着一幅画——画的是南码头外的米山，山脚下标着二十家米铺的名字。庄掌柜指着画说："米山怎么走，我跟米山的步子怎么走。涨一钱跟一钱，跌一钱跟一钱。不挑水，不挑时辰。"老陆忍不住问："米山突然塌了，你也跟着塌？"庄掌柜推了推眼镜："对。我跟山的，不是挑米的。"

老冯正要问第四家，那位北边来的吴掌柜咳嗽一声："我就是第四家。我家在北边，铺子开在千里之外，专门从那边进货，运到南码头来卖。"老陆瞪大了眼："隔着千山万水，你怎么算账？"吴掌柜掏出本厚厚的账册："换算。咱这边用铜钱，那边用银饼。我赚的是银饼，拉回来要折成铜钱。两个钱之间有个比价，天天在变。比价一变，赚的就成了亏的。"他顿了顿，"更要紧的是，那边规矩和咱不一样。什么能卖什么不能卖、什么账能入什么账不能入——都得按那边的来。还有那位子上的事，今天这一家上台明天那一家下台，做买卖的法子也跟着换。"

老冯越听越糊涂，老沈也皱起了眉。正在这时，门帘一掀，顾先生拎着旱烟袋进来，往太师椅上一坐，把四个掌柜扫了一遍："都坐过来，把账本摊开。"

杜掌柜先摊开。顾先生扫了一眼："你三只缸都摆，灵活是灵活，但有一件事最怕——三只缸里的米都往一个方向跑，你以为分散了，其实没分散。股票那缸重时按股票那套称，债券那缸重时按债券那套查。"杜掌柜的笑僵了一下。

葛掌柜把匣子打开。顾先生用旱烟袋点了点那几张票子："匣子里的票子，到期日子平均摊下来不能超过四个月，最长不能超过八个月。要是头十位大买主占了你一半的货，还要再缩——平均不能超过两个月，最长四个月。还得留出三成银根。"葛掌柜直点头。

庄掌柜把那幅画展开。顾先生没看画，看的是画边上一行小字——"米山走了多少步、跟山号走了多少步"。顾先生说："这叫'步差'。步差大了就是你跟丢了米山，小了才跟得稳。分红你没分、要付脚力钱、柜上留着没动的米、抄名单时多写了一家少写了一家——每一样都让步差变大。步差要控死。"

吴掌柜最后摊开那本厚账册。顾先生翻到一半就合上："几道坎先点给你。第一，比价——一天一变，赚的钱经不起一次大跳水。第二，税——那边抽的、这边抽的、中间几道关卡抽的，万一突然加一道新税，连招呼都来不及收。第三，盘子——那位子上的事、那账本上的事、那招牌上的事，今天能走明天不一定能走。"

顾先生把四个账本摞在一起，站起来走到门口。米车一辆接一辆地过，他头也不回地说："老陆只管挑米，老沈只管盾，你四位要管的不只是米和盾——是米山外的米山，匣子里的票子，画边上的步子，千里外的比价。"

四个掌柜你看我我看你，最后杜掌柜先开口："您能不能把四家的账合起来再送一卷经？"顾先生笑了一声，把旱烟袋在门框上磕了磕："经是死的，铺子是活的。你们把今天这顿茶再喝一遍，比我送十卷经都管用。"

**📖 原文定义**

> 混合基金是指可同时投资于股票、债券和货币市场工具等资产的基金。其风险和预期收益通常低于股票基金，高于债券基金。混合基金股票仓位较高时，可参照股票型基金，对基金行业集中度、持股集中度等风险指标进行监控；在债券仓位较高时，可参照债券型基金，侧重对基金组合久期、持债集中度等风险指标进行监控。

> 货币市场基金投资组合的平均剩余期限不得超过120天，平均剩余存续期不得超过240天。当前10名份额持有人的持有份额合计超过基金总份额的50%时，平均剩余期限不得超过60天，平均剩余存续期不得超过120天；超过基金总份额的20%时，平均剩余期限不得超过90天，平均剩余存续期不得超过180天。

> 货币市场基金债券正回购的资金余额不得超过其净资产的20%。

> 影子定价确定的基金资产净值与摊余成本法计算的基金资产净值的负偏离度绝对值达到0.25%时，基金管理人应当在5个交易日内将负偏离度绝对值调整到0.25%以内。当正偏离度绝对值达到0.5%时，基金管理人应当暂停接受申购并在5个交易日内将正偏离度绝对值调整到0.5%以内。

> 指数基金所投资的标的指数成分股一般不得低于基金净资产的90%，其核心风险指标是跟踪误差，反映了基金收益与标的指数收益之间的偏离程度。

> ETF既可通过"一篮子股票"申购或赎回基金份额，也可像股票一样在二级市场交易。套利机制使得ETF价格与净值差异较小。ETF会承担所跟踪指数面临的系统性风险和跟踪误差，二级市场价格常常会产生溢价和折价。ETF还面临申购赎回清单出错风险。

> 跨境投资风险主要分为政治风险、汇率风险、税收风险、投资研究风险、交易和估值风险、合规风险等。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 三合号三只缸并排、配比可调 | 混合基金可同时投资股票、债券和货币市场工具，资产配置灵活 |
| 股票缸重按股票那套、债券缸重按债券那套 | 混合基金参照股票型/债券型基金相应风险指标监控 |
| 三只缸往一个方向跑、分散假象 | 不同资产类别之间可能高度相关性，需评估组合整体风险特征 |
| 短票号匣子里到期不超四个月的票子 | 货币市场基金投资组合平均剩余期限≤120天、平均存续期≤240天 |
| 头十位大买主占一半，票子再缩到两个月 | 持有人集中度>50%时平均剩余期限≤60天、存续期≤120天；>20%时≤90天、≤180天 |
| 票子每天对价、对不上就停一停 | 影子定价偏离度管理（负偏离0.25%调整、正偏离0.5%暂停申购） |
| 跟山号照着米山画、步差越小越稳 | 指数基金跟踪标的指数，跟踪误差是核心风险指标 |
| 米山塌了跟山号也塌 | 标的指数单边下跌时指数基金仍需跟踪、无法减仓 |
| 步差来自分红、费用、现金留存、权重偏离 | 跟踪误差主要来源及管理手段 |
| 千里外进货，银饼换铜钱，比价一天一变 | 跨境投资汇率风险（外币计价资产收益需换算回人民币） |
| 路上几道关卡抽税、加新税没商量 | 跨境投资税收风险（不同国家税收法律差异、追溯性修订） |
| 那边规矩不同、什么能卖什么不能卖 | 跨境投资合规风险（不同司法管辖区法律法规差异） |
| 位子上的人换来换去 | 跨境投资政治风险（地缘政治、监管政策、社会稳定） |
| 抄名单时多写了一家少写了一家 | 跟踪误差来源之一——成分股权重相对基准的偏离 |
| 顾先生最后让四家把账摊开 | 混合、货币、指数/ETF、跨境四类基金风险管理各有侧重 |
