# 三、混合基金的风险管理

## 🏺 寓言故事 —— 《三合号》

老陆的"粳米股票号"开张三个月, 南码头又冒出一家新铺子, 叫"三合号", 掌柜姓杜, 圆脸, 眼睛滴溜溜转。三合号铺子里米色最多 —— 粳米、糯米、糙米三只缸并排摆着。老冯串门时瞅见了, 便问: "杜掌柜, 你这三只缸各占多少?" 杜掌柜嘿嘿一笑: "看行情。米山要涨, 粳米就多摆点; 要跌, 糯米就压上去。配比是活的, 三天一小调, 半月一大调。"

这话传到了顾先生耳朵里。顾先生拎着旱烟袋过来串门, 进店就把杜掌柜招呼过来: "把你那本三合账册摊开。"

杜掌柜不敢怠慢, 把账册呈上。顾先生翻了几页, 用烟袋一指: "你这三只缸, 粳米缸占四成, 糯米缸占三成五, 糙米缸占两成五。表面看是分散了 ——"他顿了顿, "可你想过没有, 这三只缸里的米, 是从同一座米山下来的。"

杜掌柜愣住: "米山?"

顾先生把烟锅子敲了敲: "米山一涨, 三只缸一起涨; 米山一跌, 三只缸一起跌。你以为分了三个缸, 其实是一根绳上的三只蚂蚱。股票那头吃紧的时候, 你粳米缸里那几只大盘米, 跟着大盘一起晃; 糯米缸里那些跟信用有关的票子, 表面上不跟米山, 可米山跌狠了, 钱紧, 票子也吃紧; 糙米缸里的现银虽然稳, 可米山一涨, 现银一进一出又少赚了不少。三只缸, 跑的方向其实是一回事。"

杜掌柜脸上一红: "那该如何是好?"

顾先生摇头: "分缸的本事你已经会了, 但你少看了一样 —— 缸和缸之间, 是会一起跑的。这件事, 比单只缸的事更要看紧。"他朝账册上一指: "你粳米缸重的时候, 按股票那套查 —— 行业偏不偏、头几家占得多不多、跟风是跟得紧还是跟得松; 糯米缸重的时候, 按票子那套查 —— 票子多久到期、印票子的人靠不靠谱、急用钱时能不能脱手; 糙米缸重的时候, 又要单算那缸的变现本事。三只缸, 三套秤, 一套都不能少。"

他又补了一句: "这还不算。最怕的是你以为分了缸就安稳了, 结果三只缸里的米其实是同一家米铺进的货, 米铺一倒, 三只缸全塌。这种'假分缸', 才是真正坑人的。"

杜掌柜连夜重写了账册, 把每只缸里米的来源铺子都列清楚, 不能再让三家缸里的米是同一家铺子进的。他又按顾先生的指点, 每天记账时先看一眼: 今天三只缸是不是又在往一个方向跑? 一旦发现三只缸的步调高度一致, 就要在配比上做手脚, 把其中一两只缸减一减, 让步差拉大, 真分缸。

月底盘账的时候, 老冯又来串门: "杜掌柜, 听说你这三合号最近又稳又有赚?" 杜掌柜把新账册往桌上一摊: "这都是顾先生那三只秤的功劳 —— 股票那套、票子那套、糙米那套, 加上一根'分缸绳'。秤齐了, 绳也看住了, 三合号才真的合得起来。"

门外米车又一辆辆地过, 三合号柜上的那本账册封皮上, 多了两行字: "三只缸, 三套秤; 一根绳, 时刻察。"

---

**📖 原文定义**

> 混合基金是指可同时投资于股票、债券和货币市场工具等基金的基金。其风险和预期收益通常低于股票基金, 高于债券基金。根据资产投资比例及投资策略, 混合基金可进一步分为偏股型基金、偏债型基金、平衡型基金等。混合基金的一个显著优势在于其资产配置的灵活性, 这使得基金管理人能够根据市场变化及时调整投资策略。

> 混合基金股票仓位较高时, 可参照股票型基金, 对基金行业集中度、持股集中度等风险指标进行监控; 在债券仓位较高时, 可参照债券型基金, 侧重对基金组合久期、持债集中度等风险指标进行监控。

> 此外, 混合基金的风险管理还需要考虑不同资产类别之间的相关性。在某些市场环境下, 传统上被认为能够分散风险的资产可能会表现出高度相关性。因此, 基金管理人不仅要关注单个资产的风险, 还需要对整个组合的风险特征进行全面评估, 确保在各种市场条件下保持稳健的风险控制。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 三合号三只缸并排 (粳米/糯米/糙米) | 混合基金可同时投资股票、债券和货币市场工具 |
| 三天一小调、半月一大调 | 混合基金资产配置灵活, 可根据市场变化调整投资策略 |
| 股票缸重按股票那套查 (行业、头几家、跟风) | 股票仓位高时参照股票型基金指标 (行业集中度、持股集中度、β) |
| 票子缸重按票子那套查 (到期、印票人、脱手) | 债券仓位高时参照债券型基金指标 (久期、持债集中度、流动性) |
| 三只缸的米从同一座米山下来 | 资产类别之间可能高度相关, 传统分散可能失效 |
| 米山一涨三只缸一起涨 | 不同市场环境下分散资产可能表现出高度相关性 |
| 三家缸里的米是同一家铺子进的货 | 相关性导致的"假分散"风险 |
| 步调一致时减一两只缸拉大步差 | 对整个组合的风险特征进行全面评估 |
| 三套秤加一根"分缸绳" | 混合基金风险管理需多维度、动态的过程 |
