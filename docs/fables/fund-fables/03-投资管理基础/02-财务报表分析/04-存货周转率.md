# 04、存货周转率

## 🏺 寓言故事 —— 《阿昌的米仓》

1952 年初春，上海十六铺码头边上，米行一家挨着一家。沿江那间"昌记米行"是阿昌盘下来的，铺面不大，靠着黄浦江的船老大们吃饭。

阿昌盘店头一件事，是把仓里的米清了一遍。让他倒吸一口凉气的是：仓里堆着整整八千斤陈米，最早一批还是 1950 年秋天从西贡运来的。麻袋上落满灰尘，里头有的已经结块发黄。掌柜老金去世前两年，米行几乎只进不出，门可罗雀，米都睡在仓里。

阿昌接手头一个月，没急着开张。他把八千斤陈米分作三等：上层两成还能吃，按九折卖了；中层五成勉强能吃，掺了新米分批卖；下层三成彻底发霉，按四文钱一斤卖给了酱园做酱油。处理完一算，亏了整整一百五十块银元。

他站在空了大半的米仓里，愣了半天。转头问账房老周："咱家米行，一年里这米进仓出仓，能转几次？"

老周苦笑："老金在时，一年不到两次。进得多卖得少，仓里永远是满的。"

阿昌摇了摇头，把一本新的账本摆在桌上，写下四个字——**日进日清**。

他定下一条规矩：米仓常年只保持两千斤的量。船一靠岸，按批进货；零售按斤卖，食堂按袋发，赊账一概不收。每天打烊，账房老周就在账本上记一笔：今日进货多少、今日出货多少、仓里还剩多少。一个月下来一结，仓里两千斤米，已经在里头进进出出换了三茬。

头三个月，街坊都笑他："阿昌疯了，米行哪有日日进货的道理？船老大十天来一趟，你倒好，恨不得让他天天来。"

阿昌不答话，只顾低头算账。

到第六个月，码头上的船老大反倒主动把船靠到昌记门前。原来阿昌有个规矩：船到即卸、卸完即付，绝不赊欠，也不压船。老大们要赶下一趟水，自然愿意把最好的米先给他。昌记的米，比别家永远新一茬；街坊来买米，开口就是"昌记的米，下锅香"。

年底一结账，阿昌心里有了数。

他跟老周说："你算算，咱这米仓一年下来，那两千斤米，进进出出转了几回？"

老周拨了一通算盘，答："十二回。"

"那老金在时呢？"

"不到两回。"

阿昌把账本合上，慢慢说道："一样的仓、一样的码头、一样的两千斤地儿。老金让米睡在里头睡觉，一年睡一次，我让米天天在里头打滚，一年滚十二次。我这一年赚的钱，比他多出四倍不止。"

他顿了顿，又补了一句："米这东西，最怕'睡'。一睡就陈，一陈就贱，一贱就亏。一年转十二次，每一斤米都新鲜；一年只转两次，再好的米也得发霉。"

老周在账本空白页上，写下一行小楷：仓不在满，在于转；货不在多，在于活。转得越快，仓越活；仓越活，银子越活。

阿昌看着窗外的黄浦江，江水滔滔，米行的米仓门口，伙计正抬着一袋新到的松江粳米往里走，另一边街坊大娘拎着空米袋等着买米。进进出出，川流不息。

他笑了一下："这才像个米行的样子。"

**📖 原文定义**

> 存货周转率是公司在一年或者一个经营周期内存货的周转次数，是衡量公司存货流动程度的常用指标。其计算公式为：存货周转率 = 销售成本 / 平均存货。其中，销售成本指的是公司销售产品的成本（即营业成本），平均存货通常是期初存货与期末存货的算术平均数。存货周转率越大，说明存货变现速度越快，存货管理效率越高。

> 利用存货周转率，可算出存货周转一次平均要花多长时间，这就是存货周转天数，其计算公式为：存货周转天数 = 360 天 / 存货周转率。例如，如果一家公司年度报告的存货周转率是 4，这意味着这家公司平均每 90 天（360/4）存货周转一次。

> 存货周转率是衡量公司生产销售效率的指标，其决定因素包括业务模式、生产周期、供应链管理能力、生产组织能力和市场销售能力。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老金时期米仓常年八千斤，进得多卖得少，一年不到两次进出 | 存货周转率低——存货积压、变现速度慢 |
| 阿昌清仓时八千斤陈米发霉结块，按四文钱卖给酱园 | 存货积压导致损耗增加、变现价值下降 |
| 米仓常年只保持两千斤，新米压陈米 | 平均存货的概念——保持适度的合理库存 |
| "日进日清"、按批进货、每天记账 | 存货管理效率——加快进出仓的节奏 |
| 一年下来两千斤米"进了出、出了进"十二次 | 存货周转率 = 12 次/年 |
| 周转 12 次 vs 不到 2 次，赚的钱差出四倍 | 存货周转率越大，存货管理效率越高 |
| 一斤米一年转 12 次，每一斤都新鲜 | 周转一次所需时间短——存货周转天数 = 360/12 = 30 天 |
| 船到即卸、卸完即付、不赊欠 | 供应链管理与销售能力——周转率高的决定因素 |
| 米一"睡"就陈、一陈就贱、一贱就亏 | 存货周转率低的连锁后果：积压、减值、亏损 |
| "仓不在满，在于转；货不在多，在于活" | 存货周转率衡量存货流动程度的本质 |
