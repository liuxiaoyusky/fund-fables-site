# 08、销售利润率

## 🏺 寓言故事 —— 《两间米行的年账》

1952 年初春，上海十六铺码头一开江，米行街便热闹起来。沿江那两间米行——"丰泰"和"永昌"——隔着三间铺面，朝夕相望，街坊都说这两家东家是秤杆对秤杆，谁也不服谁。

丰泰的东家姓钱，四十出头，一脸福相。米行挂着金漆招牌，进门一排伙计齐齐整整，连记账先生都穿长衫戴礼帽。

永昌的东家姓林，瘦长条子，不爱说话。米行铺面跟丰泰一样大，伙计却只有丰泰一半，记账先生还是他自家兄弟。

这一年，两家米行的年账都贴在门外墙上。

丰泰账上写着：今年共卖出大米八万斤，进账八千块银元。

永昌账上也写着：今年共卖出大米八万斤，进账八千块银元。

卖米的钱，分文不差。街坊都议论，说丰泰赢定了——人家铺子气派，客人愿意进来。

可账本一张张往下念，议论声渐渐没了。

丰泰的账接着算。米是八千块进的货，搬运工、大船运费、麻袋损耗，就去掉了六百多块。再刨去伙计的工钱、记账先生的月俸，又去掉了八百多块。店租是最大一笔——铺子开在码头最当口的位置，一年光店租一千二百块。再加上请人写招牌、给老主顾送节礼、逢年过节摆酒请客，七七八八加在一起，又去了一千块。

八千块进账，一项项扣下来，最后账房先生念出——

"全年落到钱东家口袋里的：八百四十块银元。"

街坊们倒吸一口凉气。

轮到永昌。林东家没请账房，自家兄弟自己念。米是八千块进的，搬运靠自家兄弟和街坊帮衬，运费省了大半；伙计只有三人，工钱加起来不到丰泰一半；店租付的是码头稍偏的铺面，便宜三百多块；至于请客送礼、节礼排场，林东家一律不搞。

八千块扣下来，落到林东家口袋里的——

"两千六百块银元。"

街上一片哗然。卖米的钱一样多，可落进口袋的钱，差了将近三倍。

钱东家站在门口，脸上的笑挂不住了。他辩道："我铺子排场大，客人愿意进来；林东家小家子气，迟早要被街坊嫌弃。"

林东家没吭声，倒是他家兄弟开了口："客人是来买米，不是来买您家的排场。排场再大，米还是那个米，价还是那个价。"

旁边一位做过账房先生的老先生捻着胡子，慢悠悠说：

"卖米赚的八千块，扣掉米本身的进价，剩下的叫'加码'——那是你加价的本事。可光看加价没用，还得看这八千块里，店租花了多少、伙计花了多少、摆场面花了多少、零碎开销花了多少。把所有这些都扣完，最后落到东家口袋里的那一小块，才是一家米行真正'过日子'的本事。丰泰和永昌卖米的钱一样多，可落袋差了将近三倍——这差出来的，就是排场、闲人、空架子的代价。会卖米是本事，会花钱更是本事。一家米行赚不赚钱，不光看进账多少，要看扣完所有开销之后，还剩多少。"

钱东家听完，半天没说话。

后来街坊注意到，丰泰的伙计悄悄少了两个，店租也换了便宜些的位置。钱东家逢人便叹气："早知道这样，年初那笔气派钱就不该花。"

林东家还是不爱说话，米行还是那间不大的铺子。第二年，他用那两千六百块做本，又盘下隔壁一间铺面——依旧是素木招牌，依旧是三两个伙计，依旧不请客不摆排场。

有人问林东家："你赚的钱比钱东家多三倍，靠的是什么？"

林东家想了想，说："码头上的米一斤就那个价，客人进店来买米，付的就是那个价。可我一家老小吃穿用度、店租工钱、搬运损耗，七七八八加在一起，每一文钱都是从那八千块进账里抠出来的。抠得越省，剩下的越多。"

后来有人把两位东家的年账抄了下来，夹在一本讲"怎么才算真赚"的册子里，第一页写着：

> **"卖得多不如剩得多，进账大不如落袋稳。看一家米行会不会过日子，要看他把所有开销都扣完之后，那最后一小块还在不在。"**

---

**📖 原文定义**

> 销售利润率是指每单位营业收入所产生的利润，其计算公式为：销售利润率 = 净利润 / 营业收入 × 100%。例如，如果一家公司的净利润是10万元，营业收入是100万元，那么其销售利润率为 10/100 × 100% = 10%。
>
> 一般来说，其他条件不变时，销售利润率越高越好。然而需要注意的是，如果公司净利润中包含了大量与主营业务无关的收入（如投资收益、公允价值变动、资产处置收益等），则销售利润率的经济意义可能失真。为了更准确地反映主营业务的盈利能力，我们可以使用扣除非经常性损益后的净利润来计算调整后的销售利润率。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 两家米行卖米进账都是八千块 | 营业收入（销售收入）相同 |
| 米本身的进价 | 营业成本（直接成本） |
| 卖米进账扣掉米钱后剩下的"加码" | 毛利率——只扣直接成本 |
| 搬运费、运费、伙计工钱、店租、请客送礼、零碎开销 | 期间费用——运营中的所有开销 |
| 把所有开销都扣完，最后落到东家口袋的钱 | 净利润——扣完所有费用后的最终盈余 |
| 落进口袋的钱 ÷ 卖米进账 | 销售利润率 = 净利润 / 营业收入 |
| 丰泰落袋 840 块 vs 永昌落袋 2600 块，差出三倍 | 进账相同时，开销决定最终的盈利能力 |
| 丰泰"加码"不一定低，但开销大 | 毛利率与销售利润率的区别——后者扣的是所有费用 |
| 卖米赚的钱一样多，可落袋差三倍 | 销售利润率反映企业最终盈利能力 |
| 林东家"每一文钱都从进账里抠出来" | 销售利润率的经济本质：单位收入最终能转化为多少利润 |
| "看一家米行会不会过日子，要看扣完所有开销后那一小块还在不在" | 销售利润率越高越好，反映企业最终盈利水平 |
