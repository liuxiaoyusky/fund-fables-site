# 26、财务舞弊识别方法

## 🏺 寓言故事 —— 《老冯的米仓四道符》

1952 年初春，十六铺码头上米行林立，老冯的"冯记粮栈"是其中一户。冯掌柜原是钱庄出身，三年前辞了差事，攒了一笔钱盘下这间老栈，做起米粮生意。头两年他本本分分，赚得不算多，胜在稳当。可到了第三年，冯掌柜起了贪念——他想做十六铺最大的米行，要挤掉郑老大和老陆的铺子。银钱不够，他便动起了歪心思。

## 开场

开春那阵子，冯掌柜把账房老周叫到米仓后头的小屋。屋里堆着一摞摞麻袋，老周嗅了嗅，都是去年的陈米味儿。

"老周，咱们今年的账面要做得漂亮些。"冯掌柜开门见山，"我做这行三年，看明白了一个理——钱庄借钱看两样：账上家底多不多，账上赚得多不多。家底是银柜里的银子和仓里的米；赚头是账本上的利润。咱们要里外都撑得起来。"

老周为难："东家，银柜里多少就是多少，仓里米就这么多……"

"谁让你真去凑？"冯掌柜压低声音，"你就按我说的记，我自有办法。"

## 冲突

冯掌柜布置的第一桩，是把银柜里的现银"做大"。

"你去找恒泰钱庄的陈源丰，套套交情，让他把咱的存根上多写几个数。银柜里实打实只有一百两，存根上写一千两也无妨——反正外人只认存根。"

老周迟疑："可那利息……咱们一百两的家底，怎么能撑出一千两存银的利息来？"

冯掌柜冷笑："利息先欠着，等来年赚回来再补。账面上你只管写银柜一千两，谁会去数？"

就这样，冯记的存根上赫然写着一千两现银，是实际家底的十倍。冯掌柜拿着这张"满当当"的存根，去恒泰钱庄借出真金白银八百两，又在码头上租了三个大仓囤米。一时间，冯记的米仓堆得冒尖儿，路过的人都说冯掌柜发了大财。

布置的第二桩更妙。冯掌柜让老周在账上把"存银一千两"和"借银八百两"同时记下。账本上，左边写满仓满库的家底，右边写大大的欠条——这两本账各自都好看，可搁在一起就奇怪了。冯掌柜却不担心："账是各路人马各看各的。钱庄看左边，看咱家底厚不厚；同业看右边，看咱能借出多少银子撑门面。没人会把两边摊开一起算。"

## 转折

到了年关，商会要评"十六铺十大诚信米行"。冯掌柜想拿这个名头，便让老周把账上利润再添一添。利润一添，老周就发现一个绕不过去的怪事——

账上写今年赚了一千两，可钱柜里真进来的现银，连四百两都不到。一千两利润只有四百两现银，老周也算过：账上赚一块，钱柜实际进来不到四毛。

"东家，"老周手抖着把账本递给冯掌柜，"这账……净进现银和利润差得太远，传出去要出事的。"

冯掌柜没当回事："做买卖都有赊账，晚点回款正常。你在账上多写几笔赊账，权当客户下年才付清。"

老周又提一桩："东家，您和郑老大、林四姑他们去恒泰钱庄质押的股份，这几年押得也太狠了。冯记的股份您手里九成多都押了出去，要是今年账上不好看，恒泰催着补押，到那会儿可怎么办？"

冯掌柜摆摆手："押就押了，押出去的股份还能换银子回来救急。等今年拿到十大诚信米行的名号，股价一稳，咱们再赎回来就是。"

老周心里发凉——账上赚一块钱只进四毛现银、存银和借银都堆得老高、股份几乎都押了出去——这三样凑在一起，他想开口说"东家，怕是有人在背后戳脊梁骨"，可冯掌柜已经转身去码头应酬了。

## 结局

十大诚信米行的匾还没挂上去，出事的消息先传到了十六铺。

先是恒泰钱庄的陈源丰来找冯掌柜。陈源丰是冯掌柜的旧同事，平时酒桌上称兄道弟，这回却一脸严肃："老冯，商会有人来问你们冯记在我们这儿的实际存银，我照实说了——只有一百来两。你那存根上写的一千两，我只能说是你自个儿写的，与我们钱庄无关。"

接着是同业的老陆、老沈、孙掌柜，几家一碰头，也都觉得冯记的账奇怪——银柜满满却不付利息，借银高高却还要借，银柜里进钱比账上的利润少一半，东家的股份又几乎全押了出去。这四样同时冒出来，老陆摇着头说："老冯这账，跟戏台上的戏服似的，外头光鲜，里头全是竹架子。"

最后是周先生。周先生是十六铺的老账房，三十年没出过差错。他不要看冯掌柜的存根，也不要账本，只带着老周去冯记的米仓点数。仓里米堆得高，可一翻麻袋，下面都是陈米和碎米；再问码头的脚夫，这一年进米船比去年少了三成。

周先生把数字一一记下，对冯掌柜说："老冯，存根上写一千两银子，利息连二两都付不出；存银八百两，借银也是八百两，自个儿欠自个儿一肚子债；账上赚头一千两，钱柜只进四百两；你的股份九成多都押在外面。这四样里只要沾上一样，我就得来查；四样凑齐了——" 周先生顿了顿，"老冯，你这是把十六铺的米行当戏台来演。"

冯记的米仓最后被商会查封，冯掌柜锒铛入狱那天，老周站在米仓门口，看着衙役把一袋袋陈米搬出来验货，忽然想起那四样——

**仓里银子太多、利息却太少；仓里银子和外面欠债一样高；账上赚一块、钱柜进来不到五毛；东家股份几乎都押了出去——这四样只要凑齐一两样，米行就要小心；四样都冒出来，米行八成在作伪。**

后来冯记被新东家接手，老周也跟着留下来。有一天他从米仓后头翻出一本冯掌柜当年留下的旧账，扉页上写着一行字：

**"账本是衣裳，仓里是身子；衣裳可以缝，身子骗不了人。"**

老周把那行字抄在新账本的第一页，每年年关对账的时候都念一遍。

**📖 原文定义**

> 财务舞弊的识别方法：
> 1. 货币资金异常：货币资金占总资产比例远超行业平均、利息收入与货币资金规模不匹配。
> 2. "存贷双高"现象：银行存款和有息负债同时高企，违反正常财务管理逻辑。
> 3. 经营活动现金流与净利润不匹配：净现比持续低于 1，意味着每 1 元净利润实际流入现金不到 1 元。
> 4. 大股东高比例股权质押：可能成为操纵财务数据以维持股价的动机。
>
> [案例3-2] K公司：2014—2017年货币资金从99.85亿增至341.51亿，占总资产49.69%（远超行业15%）；存贷双高（340.45亿存款+239.77亿有息负债）；净现比0.18—0.50（远低于1）；控股股东质押91.91%。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 冯记存根写一千两，利息却只够付二两 | 货币资金异常：货币资金占总资产比例远超行业平均、利息收入与货币资金规模不匹配 |
| 账上存款一千两，借银八百两，两边都堆得老高 | "存贷双高"现象：银行存款和有息负债同时高企，违反正常财务管理逻辑 |
| 账上赚一千两，钱柜实际进来四百两，净现比远不到一 | 经营活动现金流与净利润不匹配：净现比持续低于 1，每 1 元净利润实际流入现金不到 1 元 |
| 冯掌柜手里股份九成多都押给了恒泰钱庄 | 大股东高比例股权质押：可能成为操纵财务数据以维持股价的动机 |
| 四样同时冒出来，米行八成在作伪 | 多个识别信号叠加出现 → 大概率存在财务舞弊 |
| 存根一千两与实际存银一百两不符（陈源丰当面对质） | 通过外部第三方（钱庄）独立验证账面货币资金的真实性 |
| 周先生不看账本，去米仓点数、问脚夫、查码头进出 | 结合非财务信息（实物、流水、外部观察）综合验证账面数据 |
| "账本是衣裳，仓里是身子" | 财务报表数据必须以真实经营事实为支撑，两者缺一不可 |
