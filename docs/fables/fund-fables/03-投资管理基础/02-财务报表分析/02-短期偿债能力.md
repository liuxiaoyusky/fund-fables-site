# 02、短期偿债能力

## 🏺 寓言故事 —— 《阿福的年关》

1952 年初春，十六铺码头边的米行街上，三家米行挨着开——瑞丰、永昌、德兴。街尾有个钱庄叫"同和"，开钱庄的老张头是几十年的老江湖，每年除夕前都要挨家挨户上门收赊出去的米款。

阿福是瑞丰米行的少东家，刚接父亲的班。腊月二十八，老张头带着账本先到了瑞丰。阿福请他上座，沏了壶龙井。

"老张伯，今年要收多少？"

"不多，三十二石米的款，腊月三十之前结清。"

阿福点点头，转身进了后堂，把米行这几天能动用的东西盘了一遍：柜台底下压着现银二十石米的钱，粮仓里还剩三十石陈米，另外有两张欠条——隔壁布庄赊了八石米的钱，说好年前还；城西李府赊了五石米的钱，开春再结。

他心里一合计：现银二十石，加两张欠条十三石，再把粮仓里的米卖掉，就是六十五石。而要付的款只有三十二石。手头可用的，是要付的两倍还多。

"老张伯，您放心，初一之前一分不差。"

老张头走后，阿福盘算：欠条得催一催，布庄的八石先收回来，李府那五石不急。他让伙计去布庄跑了一趟，下午钱就回来了。腊月三十，老张头来收账，阿福从柜台底下取出三十二石米的现银，一分不少。钱庄掌柜竖起大拇指："瑞丰的底子，硬。"

第二天，阿福去街上看同业。永昌米行的王掌柜可没这么轻松。永昌的铺面比瑞丰大一倍，粮仓里堆得满满当当，少说七八十石。可老张头要的是现银，仓里的米一时半会儿变不出钱。王掌柜急得像热锅上的蚂蚁，挨家挨户去借，初一前一天才凑齐。

"看着满坑满谷，可要拿现银出来，难呐。"王掌柜瘫在椅子上叹气。

阿福心里一动：原来"手头能用的"和"铺子里堆着的"，不是一回事。真正能立刻动用的，只有柜台底下的现银和布庄那八石——这才是"眼前能拿出来"的真东西。粮仓里的米呢？急着卖就要打折，跟王掌柜一样，米多却拿不出钱。

回到家，阿福把这几天的账理了一遍，贴在账房墙上：左边是"今天能拿出来的"，右边是"今天要付出去的"。左边列了现银、布庄已收回的欠条；右边列了钱庄的米款、伙计的工钱、年初要付的运费。差得越远，米行越稳当；差得越近，越要提心吊胆。

他又在"今天能拿出来的"那栏下面画了条虚线，把"粮仓里的米"和"李府那张欠条"单独列出——这两样要变成现银，得等一等、催一催，有时候还要打点折。

父亲从里屋出来，看见那张表，点点头："看一个米行能不能过得了年关，不光看它仓里有多少米，要看它'今天能拿出来的现银'够不够'今天要付出去的账'。能立刻兑现的越多，底气越足；堆在那里等着卖的越多，越要小心。"

阿福把那张表抄了一份，年年腊月都用。

后来十六铺的同行都知道：阿福看米行，跟别人不一样。他不先问"你有多少米"，先问"你今天能拿多少现银出来"。

**📖 原文定义**

> 短期偿债能力是指短期内公司在不致使财务状况恶化的前提下，利用手中持有的流动资产偿还短期负债的能力，其关注重点是公司的流动资产和流动负债。

> 公司的流动资产主要包括现金及现金等价物、应收票据、应收账款和存货等，这些资产能够在短期内快速变现，因而具有较强的流动性。流动负债是指公司需在一年或一个营业周期内偿付的各类短期债务。

> 速动资产是指流动资产中扣除变现能力相对较差的项目（主要是存货，有时也包括预付账款等），主要包括货币资金、交易性金融资产、应收票据、应收账款等。与流动资产相比，速动资产更具流动性。

> 流动比率反映公司利用流动资产偿还短期负债的能力。一般认为流动比率大于2表明公司具备较强的短期偿付能力。

> 速动比率更加保守地衡量公司短期偿债能力，因为它只考虑可以快速变现的资产，不包括存货等流动性相对较差的流动资产。通常，速动比率大于1被认为是健康的，表明公司具备良好的短期偿债能力。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老张头上门收三十二石米款 | 短期负债（流动负债）—— 一年内要付的债 |
| 阿福盘点的现银、欠条、仓中米 | 流动资产 —— 短期内能变钱的所有东西 |
| 阿福"手头可用的，是要付的两倍还多" | 流动比率大于2 —— 短期偿付能力较强 |
| 阿福催回布庄的欠条，李府的留到开春 | 应收账款 —— 短期可变现，但仍需催收 |
| 阿福把"今天能拿出来的现银"和"今天要付的账"列成两栏 | 速动资产对流动负债的覆盖率 —— 速动比率的思路 |
| 永昌米行米堆成山却拿不出现银 | 存货流动性差 —— 速动比率要剔除存货的原因 |
| 阿福在"立刻能拿的"下面画虚线，单列"仓中米"和"李府欠条" | 速动资产 vs 流动资产 —— 变现能力分层 |
| 仓中米急着卖就要打折 | 速动比率剔除存货的核心理由：存货变现时往往减值 |
| 阿福专问"今天能拿多少现银"而不是"你有多少米" | 速动比率"大于1才健康" —— 保守的偿债衡量 |
| 阿福年年腊月都重抄那张表 | 短期偿债能力需结合趋势分析，而非只看某一时点 |
