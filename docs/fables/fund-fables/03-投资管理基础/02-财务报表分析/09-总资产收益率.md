# 09、总资产收益率

## 🏺 寓言故事 —— 《三兄弟的家底》

1952 年初春，十六铺码头上陈老爷子归西，留下一笔不大不小的家业：两间铺面、一座仓栈、一处小码头，折算共值四千块银元。三个儿子分了——老大陈海接了米行，老二陈江接了油坊，老三陈湖接了酱园。

开春第一年过去，清明这天，三兄弟搬着账本回老宅给娘亲请安。

老大陈海先开口。他管米行，连门面带仓栈原值一千六百块，年底兜里揣回一百二十块利润。"娘，每百块家当，一年给我赚回七块半。"

老二陈江管油坊，家当两千四百块，一年赚了一百六十。"每百块家当，落到我手里是六块六。"

老三陈湖挠了挠头："娘，我才赚五十块……"

娘亲没接话，只问："那你们三家加在一起，一年共赚多少？"

"三百三十块。"

"咱陈家全部家当呢？"

"四千块整。"

娘亲把茶盏搁下："四千块的家当，一年回三百三十块——三百三十除以四千。"

陈海拨开算盘："八又四分之一。"

娘亲点头："这一笔账，叫'统共的赚头'。管你米行、油坊还是酱园，铺子大小本钱厚薄，都搁在一个锅里算，谁也别藏私。"

陈海脸微微一红。他心里清楚，自己那"七块半"是按自己本钱算的；陈江那"六块六"更低。兄弟俩单看的赚头，都被那"八块二"反超。

陈江也琢磨出味来："娘，那我和大哥的算法，是只算自己那一块的？"

娘亲说："是。你俩那是看自己那一份家当的赚头；我刚才那算法，是看陈家全副身家一年到底生出多少利。算法不同，看的角度不同。"

陈湖反倒笑了："娘，我虽然赚得少，可本钱也最薄啊。把我和大摆在一起看，我拖后腿；可要单看我自己那一千块一年回五十块，其实还行。"

娘亲拍了他一下："你倒想得开。"

陈海沉了沉："娘，那按您那'统共'算法，米行怎么把这八块二往上抬一抬？"

娘亲指了指窗外黄浦江："你米行有铺面有仓栈有码头，那是死的家当；一年到底赚多少，是活的。死的家当和活的赚头之间，差着一个'用'字。家当再多，一年只赚那一点，就是家当睡了觉；家当再少，一年进进出出滚了几茬，那才叫家当活了起来。"

三兄弟对视一眼，都没接话。

清明一过，三兄弟没再比谁赚得多。陈海把米行的仓栈重新理了一遍，借了一半空余给陈江堆油桶，租钱一年六十块；陈江又把压榨下来的豆渣折价卖给陈湖做酱。两家的家当，都活了起来。

年底再聚，账本一摊：米行一百八十，油坊二百一十，酱园八十，加上仓栈租金六十，合计五百三十块。陈家全副身家四千块，统共的赚头是五百三十除以四千。

陈海拨完算盘，抬头笑了："娘，今年每百块家当，一年回来十三块二。"

娘亲端起茶，抿了一口："这才像一家子的样子。"

---

**📖 原文定义**

> 总资产收益率，也称资产收益率，计算的是每单位资产能带来的利润。其计算公式为：总资产收益率 = 净利润 / 总资产 × 100%。例如，如果一家公司的净利润是 10 万元，总资产是 200 万元，那么其总资产收益率为 10/200×100% = 5%。

> 总资产收益率是衡量公司盈利能力的常用指标之一。总资产收益率高，表明公司利用资产创造利润的能力强。总资产收益率的问题在于，它使用的净利润仅是股东可以获得的利润，而总资产却包括少数股东权益和债权人权益。这可能导致对公司盈利能力的低估。为了更准确地衡量公司盈利对于股东的价值，通常结合使用净资产收益率。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 陈家全副身家四千块银元 | 总资产——分母，企业全部的家当 |
| 三家合计一年赚回三百三十块 | 净利润——分子，企业最终落袋的利润 |
| 娘亲的"统共的赚头"算法（330÷4000=8.25%） | 总资产收益率 = 净利润 / 总资产 |
| 陈海 7.5%、陈江 6.6% 都被 8.25% 反超 | 总资产收益率反映资产整体效率，不偏袒任何一块 |
| 借仓栈、卖豆渣，让死家当活起来 | 提高资产使用效率——提升总资产收益率的途径 |
| "家当睡了觉" vs "家当活了起来" | 资产整体效率——家当用得活，回报率就高 |
| 13.2% 抬升后三兄弟不再比谁赚得多 | 总资产收益率衡量"全副身家"的整体获利能力 |
| "这才是家当用起来了" | 总资产收益率高，表明企业利用全部资产创造利润的能力强 |
