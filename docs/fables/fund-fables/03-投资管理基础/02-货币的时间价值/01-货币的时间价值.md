# 01、货币的时间价值

## 🏺 寓言故事 —— 《老周的两笔账》

1952 年初春，上海十六铺码头边上，雨水刚停，米行一条街的青石板还泛着湿光。巷口的"德兴米行"掌柜老周，是这一带有名的明白人，可他的儿子小周却正为一件小事和他抬杠。

事情是这样的：码头上的船老大陈四，找老周借一百块银元去做一船东北大豆的买卖。陈四提了两笔账——"我今天拿走一百块，明年今天还你一百一十块"；又或者"我今天拿走一百块，半年后先还你五十，明年的今天再还你六十五块"。小周在旁边听着，悄悄算了一下：第二笔总共还一百一十五块，比第一笔多五块，挺划算，便冲父亲点头。

老周却笑了笑，提笔在账本上写下一行字："今日的一百，非明年的一百。"

小周不服气："爹，票子都是银元，怎么会不一样？"

老周没答话，而是从米行的钱柜里取出一百块银元，递给小周："你把这钱锁进柜子里，明年今天再拿出来，数目还是一百块。问题是——陈四要是今天拿走，他拿去进货、付船工、赚出钱来，半年后他能把五十块先还回来。这就叫'钱在今天，能生出明天的钱'。"

小周愣了一下。

老周接着说："再者，米行里的大米，今天一石米能换来一家人一个月的口粮；放到明年这时候，新米又下来了，陈米就卖不上价。你说，今年一石米和明年一石米，是不是同一石米？"

小周想了想，点头："不是。"

老周指着陈四的两笔账："所以，第一笔和第二笔，看着都是'多还几块'，可第二笔里头，五十块半年就回来了——回来的这五十块，又能进米、付工钱、接新船。第二笔钱比第一笔钱'值钱'，就在这里。"

小周这才明白：原来同样是借出去一百块银元，约定什么时候还、怎么还，比到底还多少更重要。今天能拿回来的钱，可以立刻去做事；明年才拿到的钱，就只能等到明年。

老周合上账本，对儿子说："做米行的人，最要紧的不是'赚多赚少'，是'钱回到手里多快'。回到得越快，钱就越活；越活，赚得就越多。"

窗外的黄浦江上，一艘货船正拉响汽笛，靠向十六铺的栈桥。

---

**📖 原文定义**

> 货币的时间价值（Time Value of Money，TVM）是指相同金额的货币在不同时间点上具有不同的价值。通常情况下，现在的一元比未来的一元更有价值，主要原因包括：1. 机会成本——当前持有的资金可以立即用于投资并产生未来收益；2. 通货膨胀——随着时间推移，货币购买力可能下降；3. 风险——未来的现金流存在不确定性，存在无法按预期收回的风险。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 今天的一百块 ≠ 明年的一百块 | 相同金额的货币在不同时间点具有不同价值 |
| 老周说"今日的一百，非明年的一百" | 现在的一元比未来的一元更有价值 |
| 陈四今天拿走钱去进货、付船工、赚出钱 | 机会成本——当前资金可立即投资并产生收益 |
| 半年后先还回的五十块又可进米、付工钱 | 早回收的资金能再投资，复用价值更高 |
| 今日一石米 ≠ 明年一石米（新米压陈米） | 通货膨胀——货币购买力随时间下降 |
| 陈四是否按时还钱、米价是否变动 | 未来现金流的不确定性（风险） |
| 第二笔账"早还五十"比第一笔更值钱 | 货币时间价值在投资决策中的体现 |
| "钱回到手里多快，比到底还多少更重要" | 资金回收速度影响其时间价值 |
