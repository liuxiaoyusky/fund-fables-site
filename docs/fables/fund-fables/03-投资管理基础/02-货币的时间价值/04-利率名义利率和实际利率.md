# 04、利率、名义利率和实际利率

## 🏺 寓言故事 —— 《钱庄墙上的两行数》

1952 年初春，十六铺米行街的青石板还带着前夜雨水的湿意。"德兴米行"隔壁新开了一家"通和钱庄"，开张当日就在白粉墙上刷出两行醒目的大字：本号放贷，年息五分。

老周的小儿子小周去码头看船回来，路过钱庄正巧撞见这一幕。"爹，通和钱庄的利息挺厚道，一百块存一年能拿五块利息。"他一路小跑回米行，把这事报给老周。

老周正在柜台后头拨算盘，闻言抬了抬眼皮："你怎么知道我一年后能拿五块？"

"墙上一尺大的字写着呢，年息五分。"

老周放下算盘，从柜台下摸出一张旧报纸，递给小周。报上头版登着市面物价的新闻——开春以来米价、油价、棉布价都比年前涨了一截。小周看完抬头："这是说，东西贵了？"

老周点头："东西贵了，就是你手里这一百块银元，今天能买一石米，过阵子可能只买得到八斗了。钱庄墙上的'五分利'，是按你今天这一百块算的；可等你明年今天把一百零五块取回来，米价已不是今天的米价，到米行来买米，会少买几升。"

小周眨眨眼："那我不是白赚了？"

"赚是赚了，但赚的没那么多。"老周用笔在账本上写下两行字，"钱庄墙上那一行'年息五分'，是明面上的价——你借出去一百块，明年收回一百零五块，这叫'标出来的利率'，钱庄伙计念它、你在街上听它、米行账房写它，都用这一行。"

"另一行呢？"小周凑过来。

"另一行在心里。"老周接着写，"米价一年涨了差不多两分。换句话说，你借出去的这一百块银元，明年要拿一百零五块回来，才能买到和今天一样多的米。账面上的赚头是五分，扣掉米价涨上去的两分，到手真正多买的米，只值三分。"

"那五分和三分，分别是啥？"

"墙上写的、嘴里念的、合同上按手印的——五分，那是'账面上的利率'，是钱庄认的数、米行也认的数，街坊邻居都认这一行。心里算的、扣掉米价上涨的——三分，那才是'你这一年下来，米缸里实实在在多出来的米'，才是你这一年真正赚到手的速度。"

小周这才有点转过弯来："所以墙上的数，是给街坊看的；心里的数，才是自个儿过日子的？"

老周点头："做米行的人，只盯墙上那行数，年头看年尾，总觉得'我明明赚了的'，可米缸一年比一年浅；若是把两行数对一对，心里那笔账才踏实。"

窗外的黄浦江上传来一声汽笛，码头上新一批东北大豆正在卸货，米行一条街又开始热闘起来。

---

**📖 原文定义**

> 利息率（Interest Rate），简称"利率"，是资金的增值与投入资金价值的比率，也是衡量资金增值量的基本单位。我们一般以年为计息周期，所说的年利率通常是指名义年利率。
>
> 名义利率（Nominal Interest Rate）是包含通货膨胀补偿的利率，反映了未扣除通货膨胀因素的资金增值率。
>
> 实际利率（Real Interest Rate）是指在扣除通货膨胀影响后的利率，反映了资金的实际购买力变化。换句话说，实际利率表示在剔除物价变动因素后，资金的真实增值程度。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 钱庄墙上"年息五分" | 名义利率——合同上标出来的、账面上的利率 |
| 钱庄放贷一百块、明年收回一百零五块 | 名义利率是"未扣除物价上涨"的资金增值率 |
| 小周说"一百块存一年能拿五块利息" | 名义利率的概念雏形 |
| 旧报上米价、油价、棉布价比年前涨了一截 | 通货膨胀——物价上涨侵蚀货币购买力 |
| 米价一年涨了差不多两分 | 通货膨胀率——名义利率需扣除的部分 |
| 账面上的赚头是五分，扣掉米价涨的两分 | 实际利率 ≈ 名义利率 − 通货膨胀率 |
| 到手真正多买的米，只值三分 | 实际利率——扣除通胀后资金的真实增值 |
| 心里那笔账、米缸里实实在在多出来的米 | 实际利率反映资金的实际购买力变化 |
| 钱庄认的数、米行认的数、街坊都认这一行 | 名义利率是市场上通用的报价利率 |
| 只盯墙上那行数，米缸一年比一年浅 | 不区分名义/实际利率会导致对投资效果的误判 |
| 标题《两行数》——墙上的一行与心里的一行 | 名义利率（账面）与实际利率（真实）的并置与差异 |
