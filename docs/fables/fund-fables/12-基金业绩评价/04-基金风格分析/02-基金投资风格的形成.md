# 二、基金投资风格的形成

## 🔍 寓言故事 —— 《米行三家的进货路》

1952 年初春,十六铺码头边上,开米行的有二十多家,可要论挑米挑得最有板有眼,街坊都点这三家:老陆的「陆永兴」、老沈的「沈裕和」、老冯的「冯记」。

老陆是太仓人,做米做了三十年。铺子不大,两间门脸,可缸里年年摆得满满的,只认一种米——苏北射阳的大粳米。粒大、色白、熬粥挑在筷子上不散,一斗能多出半斤饭。街坊笑他死脑筋,他只笑笑:「咱家就这一条路子,走通了比走宽了强。」他的主顾都是城里的老茶馆、老饭店,签的长约,按月送。铺子墙上挂着一块乌木牌子,刻着「专做苏北射阳粳米」,那是他爹留下的家训。

老沈不一样,他是绍兴人,精明得很。他的「沈裕和」米行开在码头最显眼的位置,门脸三间,缸一字排开十几只。街坊说,老沈挑米专挑那些个没人要的「丑米」——外省运来的、有点碎、有点陈、颜色发暗,可他总能跟乡下的米贩子压到最低价。压回来略略过筛,掺一掺新米,价钱比射阳粳米便宜三成。城里的糕团铺、酿酒的、做年糕的,排着队来他这儿拿货。老沈自己说过一句话:「我做的不是米,是便宜。」

老冯呢,夹在两家中间,铺子位置不上不下,门脸也不大不小。早年他在米行公会当过账房,账目清楚,人缘也好,可自己单干的时候却挑了一条别人没走过的路——他只做安徽芜湖来的新糯米。每年新糯米下来,他就亲自去芜湖盯田头,谷子一脱壳就装船往上海运。糯米色白、软糯,做汤圆、做粽子是一绝。每年端午、中秋前一两个月,他的缸就被糕饼铺订得满满的。老冯自己也说:「我做的是个'新'字,谷子一熟我就动,一过季我就停。」

街坊看得明白,三家的米缸里摆的不一样,进货的船不一样,签的约也不一样——这就是三家铺子的「活法」。可要说这活法怎么来的,各有各的根。

陆永兴的家训,是他爹陆老太爷在世时就定下的。陆老太爷在太仓乡下种了一辈子粳米,知道射阳那片地、那片水长出来的米最稳。临终前把老陆叫到床前,指着那块乌木牌子说:「铺子招牌可以换,这块牌子不能换。」老陆从二十岁接手,一干三十年,从未改过。这家训是写进「铺规」里的,伙计进门先背三条:只收苏北米、只做散粳米、只跟老主顾做生意。

沈裕和的路子,跟老沈自个儿的脾气分不开。老沈在钱庄恒泰当过学徒,会算账,知道什么叫「贱买贵卖」。他挑米行这行当,不是因为喜欢米,是因为「行当里头有差价可赚」。他铺子里也挂着家训,可那家训不是老沈他爹定的,是他自己给自己定的——挂在账房后头,伙计进进出出都能看见:「宁买贱三分,不买贵一厘。」这条规矩硬得很,可它不写要做什么米、跟谁做生意,只写一条:便宜。

老冯的路子,一半是铺子的家法,一半是老主顾的嘴。他的铺子原本也卖粳米、也卖糯米,可芜湖新糯米是五年前一个绍兴客商介绍给他的。那客商在十六铺开了家糕饼铺,年年端午要从他这儿拿糯米,头两年量小,第三年量一下翻了三倍。对方拍着老冯的肩说:「冯掌柜,你这新糯米给我供上,长约签五年。」老冯这才下了决心,把其他米全停了,一门心思做新糯米。这路子,既有铺子家法的底子,也有大主顾的嘴替他定了方向。

老陆听着三位老友各自说自家的来路,搁下茶碗在茶馆里说一句:「我家的活法,我爹定的;沈兄的活法,他自己定的;冯兄的活法,他大主顾替定的。三家路子不同,可都是被自家的家法、铺子的家规、主顾的偏好给框出来的。」老沈点头:「可不是嘛。活法不是凭空想出来的,是被逼出来、被教出来、被养出来的。」

到了这年开春,米行公会开年会,周先生坐在上席发话:「今年米行要分等挂牌,大铺子、大缸、签长约的,挂红牌;小铺子、走街串巷零卖的,挂蓝牌。各家要报一报自己的进货路子,公会立个档,主顾一翻档就知道哪家铺子做什么米。」老陆第一个站起来,报了「苏北射阳大粳米」;老沈第二个,报了「各省散米,以贱为主」;老冯第三个,报了「安徽芜湖新糯米」。周先生一一记下,又问:「你们三家,做的是大仓大行的大米,还是小街坊的小米?挑的是便宜的,还是挑最有盼头的新米?」老陆、老沈、老冯你看看我,我看看你,都笑了:「周先生,这一问,正好把咱三家的活法问出来了。」周先生提笔在档上又添两栏:「米色大小」与「挑米偏好」,自此三家铺子的活法在档上写得清清楚楚。街坊来买米,翻一翻档,心里就有数了。

**📖 原文定义**

> 基金投资风格是指基金在投资过程中所表现出的稳定且具有一致性的投资特征。基金的风格形成主要受以下因素影响:基金经理的投资理念和偏好;基金管理公司的文化和战略;基金合同和招募说明书中规定的投资范围和投资限制;客户和渠道的需求。
>
> 风格类型主要包括:
> - 规模(市值)风格:大盘 / 中盘 / 小盘
> - 价值 / 成长风格:价值型(被低估) / 成长型(高增长预期)
> - 投资风格还可能包括行业偏好、地理偏好、动量 / 反转等

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 米行三家的「活法」 | 基金投资风格 |
| 陆永兴只认苏北射阳大粳米 | 规模(市值)风格:大盘 |
| 老冯只做芜湖新糯米 | 规模(市值)风格:小盘(细分特色品类) |
| 老陆「做米做了三十年,认一条路走通」 | 基金经理的投资理念和偏好 |
| 陆老太爷临终定下的乌木牌子家训 | 基金管理公司的文化和战略 |
| 铺子墙上挂的「专做苏北射阳粳米」家训 | 基金合同 / 招募说明书中规定的投资范围 |
| 「宁买贱三分」的家训 | 投资限制(价值风格的内核) |
| 绍兴客商签下五年长约 | 客户和渠道的需求 |
| 老冯的铺子「既有家法,也有大主顾替他定了方向」 | 风格形成 = 内部家法 + 外部需求共同作用 |
| 陆家三十年不变 | 风格的稳定且具有一致性 |
| 米色大小(大仓大行 vs 小街坊) | 规模(市值)风格:大盘 / 中盘 / 小盘 |
| 挑便宜(老沈) vs 挑最有盼头的新米(老冯) | 价值型(被低估) vs 成长型(高增长预期) |
| 陆家专做苏北大粳米 vs 老沈做各省散米 | 行业 / 地理偏好也是风格的一部分 |
| 周先生立档分大铺子 / 小铺子,记「米色大小」「挑米偏好」 | 风格分析需要多维度归类,事前分析依据契约/档案 |
| 街坊翻档买米 | 事前分析:依据招募说明书 / 公开档案确定风格 |
