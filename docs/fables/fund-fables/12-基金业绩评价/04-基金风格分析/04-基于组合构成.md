# 二、基于组合构成的风格分析

## 🔍 寓言故事 —— 《米色两把尺》

一九五二年初春，十六铺码头的米行公会开完年会，周先生把档立了起来，老陆、老沈、老冯三家的「活法」白纸黑字记在档上。可档立起来没几天，麻烦事来了。

外头来的粮商越来越多，安徽的、江西的、湖南的、四川的，一船一船往上海涌。公会里头的米行也从二十多家涨到四十多家，每家铺子门口都摆着样品缸，缸上插着木牌——「苏北射阳粳米」「芜湖糯米」「江西早米」「湖南糙米」。主顾一进码头，眼睛就花了，分不清哪家是哪家。城里的老茶馆陈源丰掌柜就抱怨：「老陆家的米我吃了三十年，一认就认得出来；可新开的那十几家，缸里摆的米颜色差不多、样子差不多，我咋个挑？」

老冯听见了，把茶碗一搁，站起身来说：「陈掌柜，您这话说到了根子上。米行三十家，光看门口那块木牌不够，得把缸里的米摆出来看一看。」老沈一听也来了劲：「对对对，把米倒出来一摆，颜色、大小、整碎一眼就分得出。」老陆却摇头：「摆出来倒是能看，可四十家米行，一家摆一缸，谁看得过来？」

老冯笑了笑，从怀里掏出一张白纸，提笔画了一个方框，框里头横着画一道、竖着画一道，分成九格。「我有个笨法子——」他指着那九格，「第一把尺子，看米色大小。大仓大行运来的、粒大饱满的米，归一边；街坊坊间小打小闹、粒小些的米，归另一边；不大不小的放中间。三档。」他又用笔在框里竖着划了三格：「第二把尺子，看米色偏好。专挑便宜压价收回来的丑米、陈米，归一档；专挑谷子一熟就动、最有盼头的新米，归另一档；两样都做、不偏不倚的放中间。还是三档。」

老陆眯着眼看那九格，慢慢点头：「横竖两把尺子一交叉，九个格子，每家米行对号入座，缸里的米一摆出来就知道是哪种活法。」老沈也看明白了：「比如我自己，沈裕和，缸里摆的是各省运来的、有点碎有点陈的丑米——米色小、挑便宜，往格子里一对，正好对在左下那一格。」老冯补一句：「老陆家的陆永兴，缸里摆的是苏北射阳大粳米，粒大饱满，又专做一种三十年不变——米色大、专挑稳当的，正好对在右上那一格。我自家的冯记，缸里摆的是芜湖新糯米——按这个法子一摆，也对得上一格。」

周先生坐在上席，听了半晌，开口问：「老冯，你这法子倒清爽，可你咋个知道每家缸里摆的是啥米？总不能你一家一家去翻缸吧？」老冯愣了一下，摸了摸后脑勺：「这倒是个难题。各家铺子的缸，外人看不见，只有掌柜自己心里清楚。要么等公会派人去查账，要么等各家自己报。」周先生点点头：「查账这事，米行公会可以每季派人去核一核，可账目报到公会总要迟上几日；半年大核一次、年终再大核一次，最是齐全——可中间这段日子，主顾要是来问，公会也只能说『待查』。」

陈源丰掌柜叹口气：「要是能随时知道缸里摆的是啥米就好了。」老冯接话：「陈掌柜，这就跟您去茶馆喝茶一样，茶好茶坏，得等沏出来喝一口才知道。米行的缸也是，外头的人只能等公会查完账再翻档。」周先生把老冯那张九格的图收好，对众人说：「老冯这把尺子是好使，可有个要紧处——米行的账目要公开，公会要按时去核。这两样不齐，米色两把尺子就摆不出真章。」

老冯笑答：「周先生说得是。我这两把尺子，摆的是米行柜上实实在在摆出来的米色，公会核账核得勤，尺子才量得准；核得迟，尺子就量不准了。」周先生提笔在那九格图旁边批了一行字：「米色两把尺，九格分活法；核账若迟缓，尺子便失灵。」老陆、老沈、老冯三位老友相视而笑，公会里这一桩新规，就这么定下来了。

**📖 原文定义**

> 基于组合构成的风格分析方法是目前评估基金投资风格的常用方法。该方法是在某个时点上先将市场上的证券（如股票）按某些特征值（如市值、市盈率、账面值、账面市值比率等）进行风格分类（如大盘价值、小盘成长等），然后根据基金实际所持证券的各项特征值计算出该基金在每一项特征上的加权平均值（权重为各证券在投资组合中的市值比重），再对基金各项特征值进行综合以确定每只基金在该时点上的投资风格。
>
> 美国晨星（Morning Star）公司的基金投资风格箱从规模和成长性两个维度来划分股票风格，每个维度分为三类。股票按照规模风格分为大盘、中盘和小盘三类；按照价值-成长特性分为价值型、平衡型和成长型三类。如图12-1所示，晨星投资风格箱是一个 3×3 的矩阵，共有 9 种投资风格。
>
> 使用这种方法需要及时掌握基金投资组合持仓信息。对于外部评价机构，依据当前证券投资基金在季报中披露的前十大重仓股，在半年报和年报中披露全部持仓信息，且有一定延迟。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 米行柜上摆出来的米色 | 基金实际持仓的证券组合 |
| 老冯的两把尺子（米色大小 + 挑米偏好） | 风格分析的两个维度：规模（市值）+ 价值/成长 |
| 第一把尺子分「大 / 中 / 小」三档 | 规模风格：大盘 / 中盘 / 小盘 |
| 第二把尺子分「挑便宜 / 不偏不倚 / 挑最有盼头」三档 | 价值-成长特性：价值型 / 平衡型 / 成长型 |
| 横竖两把尺子交叉出九格 | 晨星投资风格箱：3×3 矩阵，9 种风格 |
| 老陆的陆永兴「粒大饱满 + 专做一种」对右上格 | 基金按组合特征值加权平均后归入某格 |
| 老沈的沈裕和「米色小 + 挑便宜」对左下格 | 风格分类取决于组合构成的实际特征 |
| 「米色大小」加权平均 | 按各证券在组合中的市值比重计算加权平均特征值 |
| 主顾不能随时知道缸里摆的米 | 外部评价需要基金的持仓数据 |
| 公会每季派人核账（季报） | 季报披露前十大重仓股 |
| 半年、年终大核（半年报、年报） | 半年报和年报披露全部持仓信息 |
| 核账总有迟几日 | 信息披露有一定延迟 |
| 「核账若迟缓，尺子便失灵」 | 基于组合构成分析方法的局限性 |
