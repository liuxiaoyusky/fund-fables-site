# 二、基金评价的原则

## 📜 寓言故事 —— 《评议堂的六条老规矩》

1952 年开春，米市重新热闹起来。十六铺的米行公会牵头把"评议堂"重新开了张——前两年打仗时评议堂停过一阵，街坊吃了不少亏：买米的人凭一张嘴、卖米的人凭一句"我家最好"，到头来亏的总是掏银子的那位。开张头一天，公会请来德高望重的顾先生坐堂，老陆、老沈、老冯、郑老大、林四姑、孙掌柜、苏伙计一干老米行人都到齐了，挤在堂里等听规矩。

顾先生一拱手，开门见山道："评议堂今天立堂，规矩是米行公会和镇上官面一道议出来的，一共六条。这六条不是写来好看的，是评议堂立身的根本，缺一条，堂就不叫堂，叫'乱堂'。"

老冯性子最急，张嘴就问："哪六条？"顾先生从袖中取出一张黄纸，念道："**第一，评米要长期。**一家铺子的米好不好，不是今天去称一回就说了算，要看一年、两年、三年里的稳当。**第二，评的人要公正。**坐在堂里拿秤的人，自己家里不许开米铺——自己又是裁判又是选手，评出来街坊谁信？**第三，评要全面。**一家铺子看三样：米色、配比、掌柜靠不靠谱，只看一样就下结论，是把街坊往沟里带。**第四，评要客观。**米好不好，按账面算、按斗秤称、按白纸黑字的进出账来定，不许拍脑袋、不许凭感觉、不许看掌柜今天请没请吃饭。**第五，评要前后一致。**去年用什么法子评，今年还要用什么法子评；今年这家铺子得了'稳'字，明年换了掌柜也得用同一把尺子量——**不能用一套尺子量去年、用另一套量今年**。**第六，评要公开。**怎么评的、什么标准、什么结果，都要贴到堂外的墙上让街坊看；哪家铺子有疑问，可以来问，可以来辩，评议堂不能把人挡在门外。"

念到第三条，郑老大拍了下大腿："这第三条我服！我老郑的铺子米色一等一的好，可配比掺过一点陈米，掌柜也换过两茬，要是评议堂光看米色给我贴个'头等'，街坊买回去照样要骂我。**米色、配比、掌柜**——三样都看，才叫评米。"

念到第五条，林四姑点了点头接话："这一条更要紧。我记得民国二十八年那年，评议堂评米用'出饭率'，到了二十九年忽然改成'看米色'，老客户来问'怎么今年这把尺子跟去年不一样'，堂里的人支支吾吾答不上来。从那以后，街坊就有一半不信这张榜了。评议堂要立得久，**尺子就不能一年一换**，换也得提前一年公告，让街坊心里有数。"

念到第六条，孙掌柜也开腔了："这一条顶要紧！我孙某人做米几十年，最怕的是评议堂关起门来评、关起门来定，街坊想买米连一张榜都看不到。我看州府里那些**把账本和规矩都摊在阳光下晒**的商号，哪家不是越做越稳？评议堂的墙就是一面镜子，自己立得正，街坊才敢照。"

老陆听完，站起来环顾一圈，缓缓道："我以前总以为评米就是看谁家米堆得高、谁家招牌响，今天听顾先生念完这六条，才明白评议堂评的不是一袋米，是一整条街的信用。**评米要长期、要公正、要全面、要客观、要前后一致、要公开**——这六条，哪一条丢了，街坊就要吃亏，整条米市就要乱。"

顾先生点头，把黄纸小心收进匣子里，对众人说："评议堂的堂号今天是头一回挂出去，这六条老规矩也是头一回念给街坊听。**米行评议堂立到今天，靠的不是评了多少铺子，是街坊信不信这张榜。**信了，评议堂才算立住了。"

---

**📖 原文定义**

> 根据《证券投资基金评价业务管理暂行办法（2020修订版）》规定，从事基金评价业务应遵循如下原则：
> - 长期性原则：对基金的评价应当注重长期表现，避免过度关注短期波动。
> - 公正性原则：评价机构应当保持独立、客观、公正的态度，不受外部干扰。
> - 全面性原则：评价应当从多个维度全面考察基金的业绩、风险、管理能力等。
> - 客观性原则：评价应当基于客观数据和事实，避免主观臆断。
> - 一致性原则：评价方法和标准应当保持前后一致。
> - 公开性原则：评价方法、标准和结果应当向投资者公开。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 评米要长期（看一年、两年、三年里的稳当） | 长期性原则 |
| 评的人自己家里不许开米铺——又是裁判又是选手 | 公正性原则（评价机构保持独立、不受外部干扰） |
| 米色、配比、掌柜三样都看 | 全面性原则（多个维度考察业绩、风险、管理能力） |
| 按账面算、按斗秤称、按白纸黑字的进出账来定 | 客观性原则（基于客观数据和事实，避免主观臆断） |
| 不能用一套尺子量去年、用另一套量今年 | 一致性原则（评价方法和标准保持前后一致） |
| 把账本和规矩都摊在阳光下晒，铺子有疑问可以来问来辩 | 公开性原则（评价方法、标准和结果向投资者公开） |
| 民国二十八年换尺子街坊不信的老故事 | 一致性原则的反面教训：标准不一则公信力受损 |
| 老陆悟到"评的是一整条街的信用" | 评价业务体系对维护市场公平性和透明度的根本意义 |
