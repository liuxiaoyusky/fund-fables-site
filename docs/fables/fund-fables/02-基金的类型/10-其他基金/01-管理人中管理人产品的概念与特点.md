# 01、管理人中管理人产品的概念与特点

## 🏺 寓言故事 —— 《码头上的总调度》

一九五二年的开春，黄浦江上的雾还没散透，十六铺码头的石阶上已经湿漉漉地印满了挑夫的脚印。鼎丰米铺的掌柜老邱坐在堂口，一筹莫展。

这一年春天，十六铺的米行比往年多了七八家，南来的船也挤得水泄不通。客户要的东西越来越细——有要苏北新粳的，有要湖广陈糯的，有要川米做糕饼的，有要洋米便宜顶账的。老邱米眼力再好，也挑不完这满江的米。

更要紧的是，他手下三个老伙计——老吴、老陈、老郑——跟了他十几年，米眼力个个不差，可脾气一个比一个倔。老吴只认镇江船头，老陈只跑芜湖仓栈，老郑呢，谁的账都不全听，只信自己那只磨得发亮的米斗。客户来找鼎丰进货，问一句"哪一路的米最好"，三个人能吵上半个时辰。

老邱叹了口气，把茶碗一推。

老朋友沈半仙拄着拐杖晃过来，端起茶碗吹浮沫："老邱，你这病根不在米上，也不在伙计上——在你非要自己当掌柜、又非要自己当调度。"

老邱一愣。

沈半仙慢条斯理地说下去："十六铺这二十几家米行，每家都有一位挑米的掌柜，一辈子就做这一件事——镇江的米归谁、芜湖的米归谁、湖广的米归谁，他们比谁都清楚。你何不把进货这件事，交给他们去办？"

"那我不成了甩手掌柜？"

"对，就是要你当甩手掌柜。"沈半仙笑了，"你一个人盯船、盯仓、盯价，开春以来你瘦了七八斤。结果呢？镇江那船的米到岸晚了三天，芜湖那仓的米受了潮，你说不清是船上的事、仓里的事、还是天的事。你一个人哪里管得过来？"

"依你之见呢？"

"把进货的本钱分成几份。找几位你信得过的米行掌柜，每位领一份，让他们用自己的眼光、自己的门路去挑米进货。镇江的归镇江好手，芜湖的归芜湖好手。你呢，就坐在店堂里，每隔几天翻翻账本——谁家的米好销，谁家的压了仓，谁家的价高了。好的下回继续用，不好的下回换人。"

老邱想了三天，把老吴、老陈、老郑叫来："从今往后，你们三个，不再跟着我的船跑。"

三个人面面相觑。

"我给你们每人一份本钱、一份门路。镇江归老吴，芜湖归老陈，湖广归老郑。赚了大家分，赔了共担。我不插手你们挑哪家的米，只看月底的账本。"

老吴迟疑："东家，那我们跟了您这么多年……"

"正因为跟了这么多年，我才放心。"老邱说，"我去挑米，挑不过那些挑了一辈子的掌柜；我去盯价，盯不过那些熬了半辈子的老行尊。可我会看人——你们三个，谁更适合跑镇江，谁更适合跑芜湖，我心里有数。挑掌柜，本来就是我该做的事。"

三个人都安静下来。

一年下来，鼎丰的米反而比往年好三成。十六铺的老客户惊奇地发现，鼎丰的粳米更糯、糯米更香、陈米更便宜——每一样都比别人家更对路。问老邱怎么做到的，老邱把茶碗一推：

"我没去挑米。我去挑了挑米的人。"

老吴、老陈、老郑也变了——他们不再是被东家指派的伙计，而是各管一摊的掌柜。镇江的米由老吴说了算，芜湖的米由老陈做主，湖广的糯米归老郑。各显其能，把各自的本事发挥到了极致。

多年后，十六铺的同行们说起鼎丰，都说这家米铺的东家最"懒"——从来不跟着船跑，从来不亲自验米。可鼎丰的生意，越做越大。

老邱晚年常坐在店堂里，对年轻人说一句话：

"一个人本事再大，也挑不完天下的米。会挑米的人，永远输给会挑人的人。"

---

**📖 原文定义**

> 管理人中管理人产品（简称MOM产品），是指证券期货经营机构（简称管理人）管理的同时符合以下特征的资产管理产品：（1）管理人按照要求委托两个或者两个以上符合条件的第三方资产管理机构（投资顾问）就资产管理产品的部分或者全部资产提供投资建议；（2）根据资产配置需要将资产管理产品的资产划分成两个或者两个以上资产单元，每一个资产单元按规定单独开立证券期货账户。其中，由基金管理人管理的公开募集的MOM产品即MOM基金。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老邱（鼎丰米铺东家） | MOM 产品的管理人 / 母管理人 |
| 十六铺二十几家米行各有挑米掌柜 | 市场上众多第三方资产管理机构（投资顾问） |
| 沈半仙建议老邱别亲自挑米 | MOM 的核心思路——母管理人挑选并委托子管理人 |
| 把本钱分成三份，交给老吴、老陈、老郑 | 委托两个或两个以上符合条件的第三方资产管理机构 |
| 镇江、芜湖、湖广各有专长的掌柜 | 长期跟踪研究各资产管理人的投资风格与专长 |
| 老邱不看过程，只看月底账本 | 管理人委托投资顾问提供投资建议，自身负责监督与考核 |
| 镇江的归镇江好手，芜湖的归芜湖好手 | 根据资产配置需要将资产划分成两个或两个以上资产单元 |
| 老吴、老陈、老郑各开各的账，各记各的账 | 每一个资产单元按规定单独开立账户 |
| 好的下回继续用，不好的换人 | 母管理人对子管理人的增减与配比调整 |
| 老邱从盯船、盯仓、盯价转为只看账本 | 母管理人核心能力是选人、配置、监督，而非亲自投资 |
| 鼎丰的米比往年好三成 | 通过多管理人协作实现风险分散、收益提升 |
| "会挑米的人，永远输给会挑人的人" | MOM 的本质——管理人的管理人，专业分工、协同作战 |
