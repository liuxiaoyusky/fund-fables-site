# 《庄记的告身与报帖》

## 一

庄掌柜的「庄记」众局，开到第四十五日，银子终于凑齐了。一千一百万元旧币，七位老主顾各出一般大小的份子，摞在「钱匣子」里，码得整整齐齐。

这日清早，庄掌柜把账房老吴唤来：「银子收齐了，下一步是立局。局子一立，往后有衙门的人来查、有同行的人来问、有老主顾的人来翻账——全凭这一纸'告身'。立局的日子、立局的时辰、立的什么局、立局的银子数，一笔一笔都得落地。」

老吴问：「掌柜的，独局和众局立的法子一样吗？」庄掌柜摇头：「独局和众局，立局的方式是两道。这里头有讲究——众局是银子一齐收齐，从那天起算，十日之内，我得在同业会馆的照壁上贴出'告身'，清清楚楚写明：'庄记某号某局，X年X月X日立，初始募集X元，主顾X人。'这是对外面讲的。独局是另一位说法——独局只一位金主，银两一入账，我不必贴告示，只消一封书面的信——盖上号子的戳——递给那一位金主，信里写明：'您的银子已入账，您的局已立妥。'这就完了。」

老吴把这两条记下来，又问：「那告身一贴，是不是就完事了？」

庄掌柜摆手：「贴告身是让外面的人知道。真正要紧的，是告身一贴、日子一算——从立局那天起，五日之内，我得把一摞'报帖'亲自送到'基金业协会'的衙门里去。」

## 二

「报帖？」老吴头一回听这个词。

庄掌柜把一摞文册从柜子里抱出来，指着最上面一份：「这就是报帖。报帖里头得有五样东西——」他一样一样数：

「头一样：备案报告。这是我庄掌柜写给协会的亲笔信，里头写清楚：'庄记某号某局，于X年X月X日立，请予备案。'」

「第二样：契约。也就是我和七位老主顾签的那一摞字据。哪一位出了多少、怎么分、怎么退、怎么担——白纸黑字，全在里头。」

「第三样：主顾名册。哪一位姓什么叫什么、家里住哪里、份子出了多少，列一张表。」

「第四样：缴款凭证。哪位主顾哪天交的钱、银行还是钱庄收的、给了什么字据——一张一张附上。」

「第五样：合规先生的审查意见。我们号子里的合规师爷得亲手写一份'这事没毛病'的字，盖上他的私章，附在最末一页。」

老吴倒吸一口气：「五样一样不能少？」

庄掌柜肃然道：「一样不能少。这报帖送到协会，协会的先生们一份一份地查——契约合不合、银子齐不齐、人数有没有超、主顾是不是合格、合规这关过没过。查过了、销了号，我这号子才算真的立住了。查不过——银子得退、契约得撤、主顾得挨个知会。」

## 三

老吴又问：「这五日里头要是赶上刮风下雨、过节放假，可否宽限几日？」

庄掌柜摇头：「一日都不宽限。五日是五日，第六日我若还没把报帖送到协会衙门，我这局子就属于'野局'——没人认、没人管、出了事没底。协会的先生会登报说我'庄记'未经备案、违规开张。我做了二十年的银子买卖，丢不起这个人。」

「那独局呢？」老吴又问。

「独局同样要备案，」庄掌柜说，「独局是立局后五日之内，我把同一摞报帖——契约、主顾名册、缴款凭证、合规意见——一样不少地送协会。独局只有一位主顾，所以名册上就一行；可契约、合规这两样，少一样协会都不收。」

老吴便问：「立了局、备了案，下一步呢？」

庄掌柜把茶盏往前一推：「下一步就是开张营业——找标的、谈买卖、给主顾派息、立账本。可这些都是后话。先头这两步——立局、备案——一步走错、后头全错。」

## 四

这日，庄掌柜亲自捧着那只装着五样文册的檀木匣子，喊了马车，往上海「基金业协会」衙门的方向驶去。匣子上贴着红签，写着四个字：「庄记报帖」。

老吴站在门口目送，回头对小柳说：「咱们这号子，告身是给外面人看的，报帖是给衙门看的。两张纸，一对外一对内——外头的人知道这号子立了，里头的人知道这号子守了规矩。」

小柳点头：「告身一亮，是立局；报帖一交，才叫真正立住。」

---

**📖 原文定义**

> 集合资产管理计划的募集金额缴足之日起 10 个工作日内，由证券期货经营机构公告资产管理计划成立；单一资产管理计划在受托资产入账后，由证券期货经营机构书面通知投资者资产管理计划成立。
>
> 证券期货经营机构应当在资产管理计划成立之日起 5 个工作日内，将备案报告、资产管理合同、投资者名单与认购金额、资产缴付证明和合规负责人审查意见等备案材料报中国证券投资基金业协会备案。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 「庄记某号某局」立局的整道手续 | 资产管理计划「成立」这一法律事实 |
| 众局十日内在会馆照壁贴「告身」 | 集合计划募集金额缴足之日起 10 个工作日内，由经营机构公告成立 |
| 独局不必贴告示，只递一封书面信给金主 | 单一计划在受托资产入账后，由经营机构书面通知投资者成立 |
| 立局起五日内送协会一摞文册 | 计划成立之日起 5 个工作日内向基金业协会备案 |
| 第一样「备案报告」 | 备案材料之一：备案报告 |
| 第二样「契约」 | 备案材料之二：资产管理合同 |
| 第三样「主顾名册」 | 备案材料之三：投资者名单与认购金额 |
| 第四样「缴款凭证」 | 备案材料之四：资产缴付证明 |
| 第五样「合规先生的审查意见」 | 备案材料之五：合规负责人审查意见 |
| 协会逐份查核、查过才销号 | 备案的实质审查与合规确认 |
| 告身对外、报帖对内 | 「成立」是面向投资者的法律事实，「备案」是面向监管的自律程序 |
| 告身→报帖一步接一步 | 「先成立、后备案」的程序顺序 |
