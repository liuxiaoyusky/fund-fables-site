# 同德号登册三日记

## 一

民国三年三月初八，苏州阊门外"同德号"绸缎庄的柜上一早便开了门。这一日，是同德号每季开放"新东家入伙、旧东家退伙"的正日子。

辰时刚过，街面上便热闹起来。西街的孙布商、东街的米行李掌柜、南浔来的周老太爷，都派了伙计递条子来。条子上写得清楚：愿入几股、愿退几股、份额转到何人名下。孙布商是来加买的，李掌柜是来退伙的，周老太爷则要把名下二十股转给新过门的孙媳妇。

柜上的账房老吴接过这一沓沓条子，逐一编号、登入"柜上流水簿"。他一边写一边对伙计们道："条子都收齐了，今夜关门后，我要一份不落地送到咱号里专管名册的管册先生手里。"

## 二

按同德号的老规矩——这一日便是"T日"。

当日闭店后，老吴把当日所有条子按"加买"、"退伙"、"过户"三类分好，汇总成一张总单，再连同各条子原件，亲自送到"管册先生"——永丰簿记行的孙先生手中。孙先生接过总单，在回执上盖了戳子："T日申时三刻，收到柜上汇总条子共四十七笔。"

孙先生回去后，便按规矩办那两件事：一是按当日的"份单价"（每份额值多少银子）折算；二是把折算结果记入同德号的"总东家名册"。他吩咐伙计："今夜务必办妥，明日便要发回各柜。"

## 三

三月初九——这便是"T+1日"。

辰时刚过，孙先生便把昨夜算好的结果分别抄成三份：一份发还同德号柜上的老吴，一份发到负责看守银柜的冯大官人，一份发到永丰簿记行自留的存根。老吴接到后，再由各柜上的伙计把消息分头通知到孙布商、李掌柜、周老太爷这些具体东家。

孙布商接到回执——"先生加买十股，份额已登入名册，从今日起生效"。他皱眉问道："昨夜我递条子时写错了'孙'字的偏旁，还能算数否？"老吴答："凡条子上写得不清不楚、对不上号、款子没到账的，登册那一步便要退回去。今日您这单子，簿记行已注明'字迹存疑、退回重填'，得重写一张才能算数。"

李掌柜则顺利收到"退伙十五股、款项三日内划出"的通知。周老太爷的过户回执也到了："名下二十股转孙媳妇名下，已登册生效。"

## 四

到这一步，T+1日的事便算完了。可同德号的业务并非只此一桩。

这日，柜上来了两位特殊的客人。一位是替南洋"瑞兴洋行"做事的林先生，递了条子要入伙；另一位是常熟人陈先生，他入伙的"份子"里头有一部分是替瑞兴洋行代持的——也就是"份子里套着洋行"。

老吴登记时，对这两位另作处理："林先生这一笔，是QDII类的，按规矩得T+2日才登册——T日是昨日，T+1是今日，那便要等到明日（T+2日）才算数。陈先生这一笔，份子里套着QDII，按规矩得T+3日。"

林先生急了："为何要慢一日？"老吴答："QDII的份子要等外头的洋行回了信儿才能定价；份子里套着QDII的，还要再等一日确认那洋行的份子落地，故而要T+3日。"

陈先生倒也爽快："不急不急，咱都是为了长久的买卖，多等一两日算啥。"

## 五

到三月初十——T+2日——林先生的QDII份子正式入册。

到三月十一——T+3日——陈先生那份"套着QDII"的份子也落了账。

从此同德号的规矩便在阊门外传开：T日收条子，T+1日登册下柜；若条子不合规，登记失败、退回重填；QDII的份子T+2日，套着QDII的T+3日。这便是那本"总东家名册"里，一笔一笔账落下来的日子。

---

**📖 原文定义**

> 基金份额登记过程实际上是基金注册登记机构通过注册登记系统对基金投资者所投资基金份额及其变动进行确认、记账的过程，与基金的申购和赎回过程一致。
>
> T日，投资者的申购和赎回申请信息通过销售机构网点传送至销售机构总部，由总部汇总后统一传送至注册登记机构。
>
> T+1日，注册登记机构根据T日各销售机构的申购和赎回申请数据及T日的基金份额净值统一进行确认处理，将确认的基金份额登记至投资者的账户，然后将确认后的申购和赎回数据信息下发至各销售机构，各销售机构再下发至各所属网点。同时，注册登记机构也将登记数据发送至基金托管人。如注册登记机构为非基金管理人，还应将登记数据发至基金管理人。信息不符合规定的，确认失败。
>
> QDII基金和一般基金中基金份额登记时间通常为T+2日；如果基金中基金的投资范围包含QDII基金，份额登记时间通常是T+3日。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 三月初八——"T日"收条子汇总 | T日销售机构汇总申购赎回申请传至注册登记机构 |
| 闭店后条子送到管册先生手中 | T日销售机构总部传送申请至注册登记机构 |
| 永丰簿记行孙先生管名册 | 基金注册登记机构 |
| 按当日"份单价"折算 | 按T日基金份额净值确认 |
| 三月初九——"T+1日"登册下柜 | T+1日注册登记机构确认并下发至销售机构、托管人 |
| 老吴再分头通知各东家 | 销售机构再下发至各所属网点 |
| 条子字迹存疑、退回重填 | 信息不符合规定则申购赎回失败 |
| 林先生QDII份子T+2日入册 | QDII基金份额登记时间通常为T+2日 |
| 陈先生份子套着QDII、T+3日 | 投资范围含QDII的FOF登记时间通常为T+3日 |
| 折算→记账→抄送各柜 | 注册登记机构对份额及其变动进行确认、记账 |
