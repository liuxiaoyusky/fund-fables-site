# 《老冯的招集帖》

## 开场

1952 年开春，十六铺码头老陆的「通济米号」做的是「合米」生意——街坊买米人把钱凑到米号，米号代为采米、囤仓、卖出，年终按份子分利。手头最大的份子，当数老冯。他一人就占了一成半的米，再加上另外几家大户联手，凑过两成不在话下。

平日里米号的大事小事，都由老陆和库房先生老沈两人说了算。照米号的老规矩，每年腊月要开一次「米东家会」，把一年的账摆到台面上讲清楚。可这一年入了冬，老陆忙着去松江收晚稻，老沈守着仓库不开腔，米东家们左等右等，眼看到了小雪，连一张会期告示都没贴出来。

## 冲突

老冯先去找老陆。老陆在码头上拍着胸脯说：「老冯，今年收成一般，米号里账还没拢齐，开会的事开春再说。」老冯追问：「开春是哪一月？议什么事？」老陆支支吾吾，只说「到时候再讲」。

老冯又去找库房先生老沈。老沈手里拨着算盘，头也不抬：「我只管库中存米、出入过秤，会议召集是陆掌柜的事，章程里没写我要过问。」

老冯回到自家铺子里，把十几家米东家请来一商量：账要讲，事要议，可掌柜不召集、库房也不召集，这会就这么黄了？他翻开米号当年的老契书，找到一条：「凡涉及迁铺、增减本米、亏损分摊等大事，若掌柜、库房均不召集，占米份子过两成之米东家，可自行招集，并报市廛备案。」

老冯一合计，自己这几家联手，份子稳稳过了两成。

## 转折

腊月初三，老冯掏腰包，请茶摊的小学徒抄了二十份帖子，挨家送到米东家们手上。帖子上写得明白：**腊月廿二午后，城南陈源丰茶楼二楼**，由老冯出面召集；议的事是「本年米号盈亏、库中存米、明年铺面是否迁址」；怎么讨论、怎么拍板，一笔一笔都列了出来。

帖子发完，老冯又把同样的内容写在大红纸上，贴在码头口的照壁上，落款盖了米号公认的那方小印。

街坊围过来看。林四姑问：「老冯，你这一帖发出去，老陆那边不恼？」老冯笑笑：「契书写得清楚，他不肯招，我招。」

孙掌柜端着茶碗从隔壁过来，眯眼一瞧，点了点头：「日子、形式、议的事、怎么议、怎么表决——五样写全，差一样都不算。」

老冯接口：「差一样，米东家们说了也不算。」

## 结局

腊月廿二那天，陈源丰茶楼坐得满满当当。老冯把一年的米账摊开：进了多少石、出了多少石、库中还存多少、亏了多少。一桩桩念完，米东家们七嘴八舌定下了明年不迁铺面、亏损按份子分摊两件事。

会一散，老沈从库房赶过来，开口就问：「老冯，会上的事，米号要贴一张文告跟外面讲一声吗？」老冯笑答：「老陆不在、老沈你也没开腔，这张文告就由我召集人来贴。」

他当晚就赶出一张告示，把会上议定的两件事白纸黑字贴在照壁上，请坊间街坊一同过目。旁边小字还写了一句：「如有疑问，可至城南陈源丰茶楼查阅议事录。」

钱庄恒泰的掌柜路过，看了看告示，对老冯拱了拱手：「老冯，你今年这场『米东家会』办得规矩——日子提前三十日贴出，议的事一件不漏，会后该讲的也都讲了，往后米号再起风浪，坊间都有凭据。」

老陆从松江回来，听说这事，沉默了半晌，也朝老冯拱了拱手：「老冯，往年是我小气了。」

老冯摆摆手：「掌柜也别往心里去。米号的事，米东家自己也有一份要写的。」

**📖 原文定义**

> 基金份额持有人的信息披露义务主要体现在与基金份额持有人大会相关的披露义务。
> 
> 根据《证券投资基金法》，当代表基金份额 10% 以上的基金份额持有人就同一事项要求召开持有人大会，而基金份额持有人大会的日常机构、基金管理人和托管人都不召集的，代表基金份额 10% 以上的基金份额持有人有权自行召集，并报中国证监会备案。
> 
> 该类基金份额持有人应至少提前 30 日公告基金份额持有人大会的召开时间、会议形式、审议事项、议事程序和表决方式等事项。基金份额持有人大会不得就未经公告的事项进行表决。
> 
> 会议召开后，基金管理人、基金托管人对基金份额持有人大会决定的事项不依法履行信息披露义务的，召集人应当履行相关信息披露义务。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老冯（占份子过两成的最大买米人） | 代表基金份额 10% 以上的持有人 |
| 老冯等联手凑齐两成米份子 | 10% 以上的份额门槛（多家联合达标） |
| 老契书上写明的「自行招集」条款 | 《证券投资基金法》赋予 10% 以上持有人的自行召集权 |
| 老冯去市廛备案 | 报中国证监会备案 |
| 老冯腊月初三发帖子、贴照壁 | 提前公告持有人大会 |
| 帖子上写明时间、地点、议程、讨论与表决办法 | 至少提前 30 日公告召开时间、会议形式、审议事项、议事程序和表决方式 |
| 老陆（掌柜）不肯召集 | 基金管理人未召集 |
| 老沈（库房先生）不肯召集 | 基金托管人未召集 |
| 老冯召集议事并定下迁铺、亏损分摊两事 | 持有人大会不得就未经公告的事项进行表决 |
| 会后老陆、老沈未出告示，老冯自己贴告示 | 基金管理人、托管人不履行信披义务时，由召集人履行 |
| 告示旁写「可至茶楼查阅议事录」 | 召集人保证所披露信息易得、可查 |
