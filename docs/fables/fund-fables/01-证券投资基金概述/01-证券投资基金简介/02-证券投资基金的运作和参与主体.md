# 02、证券投资基金的运作和参与主体

## 🏺 寓言故事 —— 《十六铺的那一晚》

一九五二年的春天，上海十六铺码头的米市照例红火。

老陆在街尾开"陆顺记"米行，做了二十年，三间门面。每天开铺前，他总要把"章法"在脑子里走一遍——进多少米，价钱怎么定，伙计怎么分派，账怎么记。他常说："漏了一环，前头挣的都要赔回去。"

老沈是街对面"沈裕兴"的老板，比陆顺记早开十年。他的米行最讲究"两头稳"——一头进米，一头出米，中间还有存米、运米、记账、对账，环环相扣。沈老板有句口头禅："米行不是一锤子买卖。米还在船上，事情就已经开始了。"

老冯年纪最轻，前年才接了他叔叔的铺子，叫"冯记米庄"。他读书多，做事讲规矩，街坊们说他"把米行做成了衙门"。冯老板不恼，反倒笑笑："衙门有衙门的门道。"

街角茶馆里，每天清晨坐着一位顾先生。早年是洋行的高级买办，如今改做账房和"看门人"——哪家铺子要立规矩、要分清"谁管钱、谁看钱"，都请他去指点。

这一年春天，米市出了件大事。三月初闽江发大水，福建的米船迟了半个月没来。米价像断了线的风筝，眼看要涨到天上去。各家铺子里留作周转的底子全被挤出来应急。整条街上人心惶惶。

老陆坐在账房里发愁。账上还有三百石米，可全压在柜上；伙计们一个月没发足工钱；老主顾上门问"能不能先赊一石"，他心里也清楚，这月若赊下月就难收回。他愁的不是"米够不够"，而是"米之外那一圈事，够不够用"。

正犯难，伙计来报：顾先生来了。

顾先生掀帘进来，也不坐，先在店里走了一圈，看看米仓、柜台、账本、伙计的脸。他回头对老陆说："陆老板，你这店，按老法子做，再做十年还是这样。要不要听我一句？"

老陆请他坐下，倒了杯茶。

顾先生说："米行这门生意，拆开看，是十一件事。"他竖起指头数："定章程、集钱、存钱、办事、登记、算账、报信、分钱、收摊、杂事、合朝廷规矩——从这一季要做什么米行开始，到没卖完的米折钱退回去、零零碎碎的官司和查账为止，每一件都不能省。"

老陆半天没说话。这十一件事，他做了二十年，居然一件不落，全做过了——只是没串成一根绳。

顾先生接着说："这十一件事，归到一个人身上做，累死也做不好。要分三摊人。"

第一摊，叫"叫卖"——让街坊知道你家米行、愿意来投钱的人越来越多，归伙计里嘴最甜的、手脚最勤快的，叫"前头那帮人"。第二摊，叫"办事"——拿着大家的钱去买米、验货、谈价，归最有眼力的，叫"中间那帮人"。第三摊，叫"守夜"——把进出记清楚、把账算明白、把仓里的米看住、把消息报给投钱的人，归店里最稳的几个老伙计，加上一家独立的银号，叫"后头那帮人"。

"这三摊人，加起来，才是一个完整的米行。"

老陆这一夜没睡着。他把顾先生的话在脑子里过了一遍又一遍，又起身翻出二十年来的老账本。他忽然明白：原来他做的那些事，老沈都做得比他更"分得开"——老沈家里，媳妇管账房，儿子跑船，弟弟守仓，三摊人早就各管一摊；而老冯那"衙门式"的做法，就是把顾先生这套章法，落到了纸面上。

第二天一早，老陆把老沈、老冯都请到了十六铺的茶馆。三人围着顾先生，听了一上午。最后，老陆站起来，朝顾先生深深一揖："顾先生，我做了二十年米行，到今天才把米行做全。"

顾先生摆摆手："不是做全了，是你终于看清楚，你本来就在做这一整套事。米行这门生意，从来不是一个人在做事，是一圈人、十一件事、前后照应着做。明白了这一圈，米行才真正立得住。"

茶馆外，十六铺码头又一艘运米船靠了岸。缆绳系紧的那一声"咣当"，和往日没什么两样。可坐在茶馆里的四个人，听着都觉得，这一"咣当"，比从前更沉了一些。

**📖 原文定义**

> 从基金产品生命周期看，根据《证券投资基金法》《私募投资基金监督管理条例》《公开募集证券投资基金运作管理办法》《私募证券投资基金运作指引》及《证券期货经营机构私募资产管理业务管理办法》，基金的运作环节一般包括基金产品的设计、基金的募集、基金财产的托管、基金的投资管理、基金份额的申赎、转让交易及登记、基金的估值与会计核算、基金的信息披露、基金的收益分配、基金的清算终止以及其他基金运作活动等所有环节。但不同的基金种类，其在运作环节上会存在一定的差异。

> 从基金管理人的角度看，基金运作管理活动可以分为基金的市场营销、基金的投资管理与基金的中后台管理三大部分。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老陆开铺前先把"章法"在脑中走一遍 | 基金产品设计 |
| 老沈那句"米还在船上就已经开始了" | 基金募集 |
| 把大家的钱单独存到独立银号 | 基金财产的托管 |
| 拿凑来的钱去闽北买米、跑船、谈价 | 基金的投资管理 |
| 新主顾加入、老主顾离开，登记名册 | 基金份额的申赎、转让交易及登记 |
| 日日算、月月结，库米值多少、该分多少 | 基金的估值与会计核算 |
| 把涨跌、动向告诉投钱进来的人 | 基金的信息披露 |
| 把赚来的钱按各人投的多少分下去 | 基金的收益分配 |
| 米行不做了，没卖完的米折钱退给各人 | 基金的清算终止 |
| 应对问询、查账、打官司等零碎事 | 其他基金运作活动 |
| 米行从生到死，每天的章法合乎朝廷行会 | 贯穿生命周期的法规与自律约束 |
| "前头那帮人"嘴甜脚勤、招揽主顾 | 基金的市场营销 |
| "中间那帮人"买米验货、谈价 | 基金的投资管理 |
| "后头那帮人"记账、守仓、报信 | 基金的中后台管理 |
| 顾先生把十一件事串成一根绳 | 基金运作环节的内在一体性 |
| 老陆做了二十年才看清自己本就在做这一整套 | 基金管理活动按三大类的视角切分 |
