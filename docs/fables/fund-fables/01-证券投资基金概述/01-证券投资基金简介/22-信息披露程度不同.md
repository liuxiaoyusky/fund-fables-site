# 22、信息披露程度不同

## 🏺 寓言故事 —— 《米行的账本》

一九五二年初春的上海，外滩钟声一响，十六铺码头的米行就开了门。

老陆在浦西开了家"陆记米行"，做了快三十年。一条黄浦江把米行分成两爿：江北岸是十六铺老码头，米铺货栈一字排开；江南岸是浦东高桥镇，有他家租下的三亩水田，秋收的稻子靠乌篷船过江入库。老陆识字不多，可账目清楚，有一本发黄的硬面账本，记着每天进了多少石米、卖出多少、钱庄存着多少银元。街坊说，老陆靠的就是这本账——清清楚楚，做人做事都一样。

镇东头住着王婶，丈夫三年前去了南洋做工，每月托人带钱回来。她钱一进门就锁进床头小木箱，米快吃完就上街买。街坊笑她："王婶，你这一箱子钱就这么放着？不怕老鼠啃？"王婶撇嘴："钱是我家的，放家里稳当，干嘛要让别人知道？"

老陆听了这话，摇摇头没接话。

开春后第三个月，浦东水田遇上倒春寒，秧苗冻死一半。老陆的二儿子陆小舟连夜撑船过江，跪在父亲面前："爹，田里减产已成定局，米行撑不到年底。"老陆合上账本，沉默半晌。

第二天，老陆在门口贴出红纸告示：

"各位街坊、乡亲：因浦东水田遇寒减产，陆记米行今春进货减少三成。米价由每石十二元提至十四元，原因是江北进货渠道也涨三成，已与码头报关行核实。库存米尚有八十石，预计可撑到新米下来。有疑问可来店里翻看账本。"

告示一贴，门口围满人。

有人喊："米涨价了，老陆要发国难财！"陆小舟急得要去撕，老陆一把按住他："不许撕。让大家看清楚。"

接下来三天，老陆搬张八仙桌到门口，把账本摊上。进货单、销货单、银元收付条一摞摞码齐。街坊围着看，识字的当场念给旁人。陆小舟站在旁边，手心攥出汗。

第三天傍晚，一位穿长衫的老先生挤到账本前，戴上老花镜翻看了一刻钟，点点头："老陆，你这账一笔笔对得上。涨价这事，码头我也问了，确实涨了。"

老先生转身对街坊说："人家把底都亮给你们看了。这样的主顾，我信得过。"

人群散去，陆小舟长长舒了口气。

老陆把账本收回铁皮柜，对儿子说："陆记三十年，靠的不是精明，是亮堂。街坊把钱交给我，我替他们管着；可我管得好不好、花到哪里去、还剩多少——这些事，我得让大家看得见。看不清的事，谁敢托付？"

陆小舟问："那王婶的钱……"

老陆笑了："王婶的钱是自家木箱，箱子是她的，钥匙是她的，米也是她一升升买。她不托付给谁，也就不需要让别人知道。咱们米行不一样——是替街坊管钱的。"

"所以咱们得定期把账亮出来？"

"对。"老陆扣好锁，"只要你替别人管钱，你就得让主顾知道钱去了哪里。这事不是面子，是规矩。"

初夏，账本又添新页。街坊路过米行，常停下看一眼那本硬面账。一天，王婶也来了，拎着半袋米："老陆，听说你账本常开？我这钱在床底放了好几年，能放你这儿吗？"

老陆愣了愣，笑着摆手："王嫂，你那钱是自家过日子的，放家里最方便。要是哪天想做点别的，咱们再聊。"

王婶拎着米走了，乌篷船的桨声在江上一荡一荡，远处的钟楼又敲响了一记。

---

**📖 原文定义**

> 基金管理人必须依法定期向投资者公布基金的投资运作情况以及披露各类临时信息；银行吸收存款之后，不需要向存款人披露资金的运用情况。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老陆替街坊管钱（米行的钱不全是自家的） | 受托管理他人资金 |
| 定期把账本摊在门口让人翻看 | 依法定期公布投资运作情况 |
| 红纸告示上写明米价上涨的原因 | 披露各类临时信息 |
| 进销货单据一笔笔对得上 | 披露要求：及时、准确、充分 |
| 王婶的床头小木箱（自己的钱自己管） | 银行吸存——存款是银行对存款人的负债 |
| 王婶不需要向街坊报告钱怎么用 | 银行吸收存款后不需向存款人披露资金运用情况 |
| 老陆说"我必须让大家看得见" | 信息披露的法定义务与监管要求 |
| 老陆"亮堂"才能让街坊继续托付 | 信息透明是受托管理关系的信任基础 |
| 王婶最后想把零散的钱也交给老陆 | 当资金从"自有"变为"受托管理"，披露义务即产生 |
