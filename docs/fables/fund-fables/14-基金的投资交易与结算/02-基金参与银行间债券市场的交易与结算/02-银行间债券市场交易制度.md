# 《码头的三条老规矩》

---

**【开场】**

1952 年初春，十六铺码头外滩沿江一带，老陆的米行正月初五刚开张，便遇上一桩棘手事。年初银根紧，朝廷财政部发行的「建设公债」利息虽高，可市面上能流通的借据零零散散，全靠各家钱庄、米行私下转手。江苏来的布商钱四爷扛着一袋子银元找老陆，说要买公债；浙江的茶商林四姑也上门，问能不能把手里的几张借据换成现银。

老陆挠头——他米行小，既不认识朝廷管银根的户部官员，也没本事天天挂出"收""卖"两牌子。他转身去找隔壁钱庄恒泰的老沈讨教。老沈把茶碗一推，慢悠悠道：「老陆啊，这码头上借据能转得开，靠的是三条老规矩。你不晓得，就只能干瞪眼。」

**【冲突】**

老陆急了：「你倒说清楚！」

老沈掰着指头讲：「头一条，朝廷在十六铺挑了四家大钱庄——恒泰、源丰、汇通、永丰——算是『朝廷指定的几家大钱庄』。户部要松银根、收银根，只跟这四家谈。咱们这等小铺子，连户部大门朝哪开都不知道。」

老陆更急：「那我这种小铺子，手里攥着几张借据想换现银，怎么办？」

「这就是第二条。」老沈继续讲，「码头上常年有几家能干的铺子，自家银钱足、库房也深，天天在门脸上挂出『收建设公债』『卖建设公债』两块牌子。你拿着借据去找他，他立马按牌价收走；你想买，他也立马卖给你。这种『一直挂价收/卖的挑夫头』，就是市场能转起来的枢纽。」

老陆追问：「那恒泰算不算？」

老沈苦笑：「恒泰算第二种，挑夫头。第三条规矩——还有些小铺子，连挑夫头都当不了，就只能委托恒泰这种大铺子代办。朝廷允许恒泰代其他铺子收账付账，办好过户手续，这叫『其他铺子靠这几家代办』。你那几张借据，过两天我让伙计替你跑一趟恒泰，把账清了。」

**【转折】**

钱四爷在旁边听完，插嘴道：「沈掌柜，照你这么说，朝廷指定的四家大钱庄岂不是最厉害？」

老沈摇头：「厉害归厉害，担子也最重。朝廷要借钱、要收回银根，恒泰、源丰这四家得硬着头皮接。你不接，朝廷就换人；接了，市价跌了你得自己扛。1950 年腊月银根一紧，恒泰账面上亏了两千块大洋，掌柜的差点把招牌摘了。」

林四姑也凑过来：「那挑夫头呢？」

老沈答：「挑夫头靠的是天天挂价、随时接单。牌价挂低了，自己亏；挂高了，没人找他转手。要是某一天他挂不出价，整个码头流动性就僵了——谁也买不进，谁也卖不出。去年四月源丰的伙计病了三天没挂牌，外滩小铺子们急得跳脚。」

钱四爷若有所思：「我明白了——朝廷指定的几家大钱庄是替朝廷调银根的；挑夫头是让市场转得动的；代办制度是让小铺子也能搭上车的。三条规矩，各管一摊。」

**【结局】**

老沈点头：「正是。朝廷要稳市价，靠头一条；要让买米人、卖米人随时找得到人出价、随时能脱手，靠第二条；小铺子没钱没库房又想参与，靠第三条。三条规矩拧成一股绳，码头上的借据才转得起来，老陆你那袋公债才卖得掉，林四姑的现银才换得回。」

老陆拱手谢过老沈，当天便托恒泰代办，把钱四爷的公债过了户，林四姑的几张借据也按当日牌价收走。十六铺江面上汽笛又响起来，借据的来去、银元的进出，在这三条老规矩的底下，又恢复了往日的秩序。

---

**📖 原文定义**

> 1. 公开市场一级交易商制度：央行遴选符合条件的债券二级市场参与者作为央行的对手方，配合货币政策目标实现。
>
> 2. 做市商制度：在债券市场上，由具有一定实力和信誉的市场参与者作为特许交易商，不断向投资者报出特定债券的买卖价格，并在该价位上接受投资者的买卖要求，以其自有资金和债券与投资者进行交易。
>
> 3. 结算代理制度：经央行批准可以开展结算代理业务的金融机构法人，受市场其他参与者的委托，为其办理债券结算业务。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 朝廷指定的几家大钱庄（恒泰、源丰、汇通、永丰） | 公开市场一级交易商 |
| 一直挂价收/卖的"挑夫头" | 做市商 |
| 其他铺子靠大钱庄代办收账付账 | 结算代理制度 |
| 朝廷调银根、稳市价 | 一级交易商的目的：配合货币政策 |
| 买米人、卖米人随时能成交 | 做市商的目的：提供市场流动性 |
| 小铺子也能搭上车参与借据转手 | 结算代理的目的：扩大参与面，便利结算 |
