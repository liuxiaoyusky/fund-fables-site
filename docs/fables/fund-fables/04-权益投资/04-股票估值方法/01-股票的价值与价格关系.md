# 一、股票的价值及其与价格的关系

## 💎 寓言故事 —— 《五种价值和两张皮》

### 股票的五种价值

**票面价值（面值）**：股票上印的金额。比如每股面值10元。发行价格高于面值叫"溢价发行"，超额部分计入资本公积。

**账面价值（每股净资产）**：公司财务报表上的价值。账面净值总额 ÷ 发行股数。比如账面净值10亿，发行5000万股，每股净值20元。

**清算价值**：公司解散清算时，每股能分到的剩余财产。通常低于账面价值（资产要低价甩卖，还要付清算费）。

**市场价值（市值）**：股票在市场上的交易价格。天天变，受各种因素影响。

**内在价值**：未来预期现金流的现值总和。这是"理论上应该值多少钱"，由基本面决定。

### 市场价格 vs 内在价值

市场价格由**两方面力量**决定：

**基本面因素**（长期）：
- 公司现状：收益、股息、资产、资本结构
- 未来前景：管理层素质、行业竞争力、宏观经济

**市场情绪因素**（短期）：
- 技术分析信号
- 短期新闻事件
- 投资者心理预期和情绪波动
- 市场操纵行为

**市场价格围绕内在价值波动**，但回归过程可能迟缓且波折。

价值投资者（如巴菲特）专注于寻找市场价格**暂时低于**内在价值的机会，耐心等待价值回归。

---

**📖 原文定义**

> 股票的票面价值（Par Value）即股票面值，是股票上标明的票面金额。

> 股票的账面价值（Book Value）又称"股票净值"或"每股净资产"，是每股股票所代表的实际资产的价值。

> 股票的清算价值（Liquidation Value）是指股份有限公司进行清算时，股票的每一股份所代表的实际价值。

> 股票的市场价值（Market Value），也称股票的市值，指股票在股票市场进行交易过程中所具有的价值。

> 股票的内在价值（Intrinsic Value）通常定义为对其未来预期产生的现金流以适当的贴现率进行折现后的现值总和。

> 股票的市场价格是由市场供求关系所决定的，不仅受资产内在价值与未来价值因素的影响，还可能受市场情绪的影响。

> 市场价格虽内在地受到其内在价值的牵引，但又时常因为市场情绪的喧嚣、信息的噪音或对基本面解读的差异而偏离航道。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 股票上印的金额 | 票面价值（面值） |
| 财务报表上的价值 | 账面价值（每股净资产） |
| 解散清算时每股分多少 | 清算价值 |
| 市场上的交易价格 | 市场价值（市值） |
| 未来现金流折现 | 内在价值 |
| 溢价发行 | 发行价格高于面值 |
| 公司现状+未来前景 | 基本面因素 |
| 技术信号+新闻+情绪 | 市场情绪因素 |
| 价格围绕价值波动 | 市场价格与内在价值的关系 |
| 巴菲特找价格低于价值的机会 | 价值投资理念 |
