# 第一节 基金投资者

## 📜 寓言故事 —— 《十六铺码头的门槛》

一九五二年初春，十六铺码头刚开市，孙掌柜就把一杆新秤立在了米铺门口。秤杆上头挂了一块小木牌，红漆写着"门槛"二字。

头一个到的是苏伙计的表弟——十六岁的小学徒，攥着三块铜钱，踮起脚往里瞅。孙掌柜拦下："小哥，散客一个，到对面老冯的铺子打一升米就成。"小表弟不服气："我兜里揣着钱，凭啥不卖我？"

孙掌柜一指木牌："咱这铺子卖的不是米，是替人打理米仓的'份子'。你一个学徒，兜里几个铜板，亏一宿就当丢了学徒钱，老冯那边的散米反倒合适你。"

小表弟气哼哼地走了。这是头一种——**散客**。

接着来的是周先生，怀里揣着一封钱庄恒泰的引荐信，肩上披着一件半旧的西式呢大衣。孙掌柜拱手："周先生是内行？"周先生点头："在上海滩看了十几年行情，涨涨跌跌，认得清账。"孙掌柜又问："仓里压得起？"周先生从怀里摸出一叠银票："五万两打底，跌了再补。"孙掌柜让进内堂。

这是第二种——**内行**。钱厚、懂行、扛得住亏，三样齐了。

郑老大也跟了进来，肥袖口上沾着油点子，手里攥着一沓盖着红章的"入会帖"——恒泰钱庄、源丰火险、还有一家纱厂。伙计一数，郑老大一口气要三百份"份子"。孙掌柜陪笑："郑爷，您这三家铺子联袂来，咱按'大宗客'的规矩办——份子价、章程单立、中间人佣金另谈。"郑老大把油手往腰上一抹："成交。"

这是第三种——**铺户**。开着字号，做着买卖，按公司户头入伙。

林四姑挎着竹篮，里头装着三十块银元，站在铺子外头犹豫。伙计问："四姑，要不要进来听章程？"林四姑搓着手："我是个寡妇拉扯三个娃，钱是有几个，可我不懂你们那些'净值''仓位'的弯弯绕……亏了心疼，睡不着。"伙计摇头："那您还是先到门口散米摊打米吧，等您家里再宽裕些、也想透了再讲。"

林四姑也走了。这算第四种——**钱够、但不扛事**。门槛不让她进内堂。

苏伙计看愣了，问孙掌柜："掌柜的，咱这一杆秤、四个门道，到底在分啥？"

孙掌柜拍了拍秤杆：

"头一层，散客还是铺户——一个是人来，一个是号来，这是'谁家来'的门道。"

"第二层，懂行不懂行、扛得住扛不住——周先生算内行，郑老大那三家铺子也算内行，林四姑钱够却不懂行，只能算外行。内行和外行之间，只要回头补足了'钱'或'懂'，就能换边站——咱这就叫'两边可转'。"

"第三层，钱够不够、能不能扛亏——林四姑就是钱够、却扛不住事的那种，按规矩只能让她在门外头转悠，等她补齐了再说。"

苏伙计挠头："那郑老大和周先生，怎么又算'内行'、又算'铺户'？"

孙掌柜笑："这就叫交叉——铺户里有内行也有外行；钱够、懂行、扛得住的，是里头最硬的那一拨。三道门、三杆秤，看的不是一张脸，是三样条件。"

孙掌柜又指了指那块红漆"门槛"二字的小木牌：

"码头上来买米的人，熙熙攘攘。卖米的人能挑客，监管的人能立门槛。门槛里头分人——按家里是人是号，按懂不懂行，按钱够不够、扛不扛事。分清楚了，卖的、买的、管的，三家都省心。"

苏伙计把木牌上的字一笔一划抄进了账簿里，嘀咕一句："回头我也照着这杆秤，画一张'分客图'。"

春风里，十六铺码头的吆喝声又热闹起来。那杆挂着"门槛"二字的新秤，在门头轻轻晃着。

**📖 原文定义**

> 投资者分类：
> （一）个人投资者与机构投资者
> 1. 个人投资者：以个人身份参与投资
> 2. 机构投资者：以法人身份参与投资，如商业银行、保险公司、证券公司、社保基金、企业年金等
>
> （二）专业投资者与普通投资者
> 1. 专业投资者：通常资金实力雄厚、具备专业投资知识和风险承受能力（如持牌金融机构、私募基金管理人、合格个人投资者等）
> 2. 普通投资者：除专业投资者之外的其他投资者
> 3. 普通投资者和专业投资者可相互转化
>
> （三）合格投资者与不特定社会公众
> 1. 合格投资者：具备相应风险识别和承担能力，金融资产不低于一定金额
> 2. 不特定社会公众：与合格投资者对应，普通大众
>
> （四）我国机构投资者的主要类别：商业银行及银行理财子公司、保险公司及保险资管公司、证券公司及其资管子公司、财务公司、全国社会保障基金理事会、中国投资有限责任公司、企业年金和职业年金

| 散客 / 散米摊 | 个人投资者这一大类在销售端的"散客"形态 |

| 恒泰钱庄、源丰火险、纱厂三类入会帖 | 机构投资者的主要类别：商业银行及理财子公司、保险公司及资管公司、企业等 |
| 周先生：看行情十几年、懂行、压得起五万两 | 专业投资者——资金雄厚 + 具备专业知识 + 风险承受能力 |

| 林四姑在门外散米摊打米 | 不特定社会公众——普通大众，未达到合格投资者的识别/承担标准 |

| 铺户里有内行也有外行 | 机构投资者与专业/普通投资者分类有交叉 |
| "卖米的、买的、管的，三家都省心" | 投资者适当性管理对销售机构、投资者、监管三方的共同意义 |
