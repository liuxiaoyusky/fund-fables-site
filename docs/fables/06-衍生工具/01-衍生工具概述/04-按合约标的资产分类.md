# 二、(三) 按合约标的资产分类

## 🏺 寓言故事 —— 《六座赌坊》

外乡人在镇上开了六座赌坊，每座赌坊赌的东西都不一样。

### 第一座：铜钱赌坊
赌的是铜钱和银两的兑换比例。今天一两银子换一千文，明天可能换九百文，也可能换一千一文。这赌坊赌的就是**汇率**的变化。这叫**货币衍生工具**——以货币或汇率为标的。

### 第二座：利钱赌坊
赌的是借钱利息的高低。钱庄的放款利率每天都在变，有人赌利率涨，有人赌利率跌。这叫**利率衍生工具**——以利率或与利率相关的金融工具为标的。

### 第三座：股票赌坊
赌的是各个商号的股价涨跌。王家绸缎铺今天一股十两，明天可能八两，也可能十二两。这叫**权益类衍生工具**——以股票或股票指数为标的。

### 第四座：信用赌坊
赌的是某人会不会赖账。比如李家少爷借了钱，大家赌他到期还不还得上。如果还不上，卖信用保护的人就要赔钱。这叫**信用衍生工具**——以信用风险为标的。

### 第五座：粮油盐赌坊
赌的是柴米油盐的价格。大米、菜油、食盐，这些都是大宗商品，价格波动很大。这叫**商品衍生工具**——以大宗商品为标的。

### 第六座：奇物赌坊
赌的是一些稀奇古怪的东西：明天会不会下雨（天气期货）、明年会不会发大水（巨灾衍生品）、甚至朝廷会不会换政策（政治期货）。这叫**其他衍生工具**——以非金融变量为基础。

六座赌坊，赌的东西五花八门，但规矩都是一样的：赌的是**标的资产**的变化，而不是标的资产本身。

---

**📖 原文定义**

> 货币衍生工具指以货币或汇率作为标的资产的金融衍生工具，包括货币远期合约、货币期货、货币期权、货币互换等。

> 利率衍生工具指以利率或与利率相关的金融工具（如债券）为标的资产的金融衍生工具，包括远期利率协议（FRA）、利率期货、利率期权、利率互换等。

> 权益类衍生工具是指以股票或股票指数为标的资产的金融衍生工具，主要包括股票期货、股票期权、股票指数期货、股票指数期权等。

> 信用衍生工具是指以基础产品所蕴含的信用风险为标的的金融衍生工具，用于转移或对冲信用风险。自20世纪90年代以来，信用衍生工具发展迅速，主要包括信用违约互换（CDS）、信用联结票据（CLN）等。

> 商品衍生工具是指以大宗商品（如能源、金属、农产品等）为标的的金融衍生工具，主要包括各种大宗商品的期货和期权。

> 除以上五类金融衍生工具外，还有相当数量金融衍生工具是在非金融变量的基础上开发的。例如，用于管理气候变化风险的天气期货、用于管理政治风险的政治期货、用于管理巨灾风险的巨灾衍生产品等。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 铜钱赌坊（赌汇率） | 货币衍生工具 |
| 利钱赌坊（赌利率） | 利率衍生工具 |
| 股票赌坊（赌股价） | 权益类衍生工具 |
| 信用赌坊（赌会不会赖账） | 信用衍生工具（CDS等） |
| 粮油盐赌坊（赌商品价格） | 商品衍生工具 |
| 奇物赌坊（赌天气、灾害等） | 其他衍生工具 |
