# 二、(二) 按产品形态分类

## 🏺 寓言故事 —— 《独立宝剑与嵌宝匕首》

镇上铁匠铺有两种卖法。

### 独立宝剑

第一种是**独立宝剑**——一整把剑就是一个完整的兵器，可以单独买、单独卖、单独定价。你走进铺子，看中哪把就直接买走，不依附于任何东西。这就是**独立衍生工具**：本身就是独立的金融合约，不依附于其他工具，能独立交易和定价。像期货合约、期权合约、互换合约这些，都是独立存在的。

### 嵌宝匕首

第二种是**嵌宝匕首**——匕首本身是个普通兵器，但刀柄上嵌了一颗宝石。这颗宝石让匕首变得不一般：如果敌人是红衣的，宝石就会发光；如果敌人是蓝衣的，宝石就暗淡。宝石的存在改变了匕首的"表现"，但你不能单独买这颗宝石，必须连匕首一起买。

这就是**嵌入式衍生工具**：嵌入在非衍生合约（主合约）中的衍生工具。它本身不能独立存在，依附于主合约，但会影响主合约的部分或全部现金流。比如公司发行的债券里附带了"赎回条款"——如果利率降到某个水平，发行方可以提前赎回债券。这个赎回条款就是嵌在债券里的衍生工具，它根据利率这个变量触发特定行为。

常见的嵌入式衍生工具还有：返售条款（投资者有权按约定价格卖回给发行方）、转股条款（债券可以转换成股票）、重设条款（利率或价格可以重新设定）等。

---

**📖 原文定义**

> 独立衍生工具指本身即为独立存在的金融合约，不依附于其他金融工具或合同，能够独立交易和定价。常见的独立衍生工具包括期权合约、期货合约或者互换合约等。

> 嵌入式衍生工具（Embedded Derivatives）指嵌入非衍生合约（即"主合约"）中的衍生工具。嵌入式衍生工具会影响主合约的部分或全部现金流，根据某些特定的变量（如利率、金融工具价格、汇率、价格指数、利率指数、信用等级或信用指数等）的变动而触发特定条款。常见的例子包括公司债券条款中包含的赎回条款、返售条款、转股条款、重设条款等。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 独立宝剑（可单独买卖） | 独立衍生工具 |
| 嵌宝匕首（宝石依附于匕首） | 嵌入式衍生工具 |
| 宝石根据敌人衣服变色 | 根据特定变量触发特定条款 |
| 不能单独买宝石 | 不能独立交易，依附于主合约 |
| 赎回条款 | 嵌入式衍生工具的常见例子 |
| 返售条款、转股条款、重设条款 | 其他常见嵌入式衍生工具 |
