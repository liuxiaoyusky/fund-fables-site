# 五、ETF联接基金

## 🏺 寓言故事 —— 《小镇的渡桥》

青河镇的集市上，有一家远近闻名的"百味铺"。

铺子的掌柜姓陈，人称陈掌柜。他有个绝活：能把天南海北的香料按一套秘方的比例配成"百味包"——花椒、八角、桂皮、丁香，样样都有，比例分毫不差。镇上的人都说，买一包百味包，就等于把天下好料都搬回了家。

百味包只在铺子里卖，而且有个规矩：一次至少买五十包。陈掌柜有他的道理——少了不够配，多了才划算。

这规矩拦住了一个人：阿秀。

阿秀是镇上织布的，每月手头紧，只想买两包百味包回家炖肉。她去找陈掌柜，陈掌柜摊开手说："姑娘，不是我不卖，是规矩在那儿。你凑够五十包再来。"

阿秀凑不起。她织一个月的布，也换不来五十包百味包的钱。

她想过找人合伙凑单，可合伙的人各有各的打算，今天凑明天散，麻烦得很。她也想过自己按配方去散买香料，可那配方复杂得很，她跑遍了全镇的香料摊，不是缺这就是少那，配出来的味道总是差一截。

阿秀站在百味铺门口，闻着里面飘出的香气，叹了口气。

这声叹息被一个人听见了——老周。

老周是镇上开杂货铺的，卖米卖油卖针线，做的是街坊生意。他看着阿秀，又看看百味铺，心里忽然亮了一下。

第二天，老周在杂货铺门口挂出一块新牌子："代售百味包，一包起卖。"

街坊们围上来问怎么回事。老周说："我一次从陈掌柜那儿进五十包百味包，存在我这儿。你们想买多少买多少，一包也行，两包也行。我按当天的市价卖给你们，不多收一文钱。"

阿秀第一个跑来，买了两包，喜滋滋地回家炖肉去了。

消息传开，来买的人越来越多。有人每天来买一包，攒着攒着就攒出了一锅好料；有人本来买米，顺手带一包百味包，方便得很。老周的杂货铺本来就有街坊们的信任，如今多了这项代售，生意更红火了。

有人问过老周："你图什么？"

老周笑着说："我图的是街坊方便。百味包我按批发价进，按市价卖，差价薄得很。但我杂货铺的人气旺了，买米买油的人也多了，这叫各得其所。"

陈掌柜也乐见其成。他的百味包还是按五十包起卖，但通过老周，那些原本够不上门槛的街坊，也间接尝到了百味包的滋味。他的配方没有变，规矩没有变，但吃到他百味包的人，多了十倍不止。

后来，青河镇的人给老周的杂货铺取了个外号，叫"渡桥铺"。

因为老周就像一座桥，一头连着陈掌柜的高门槛，一头连着阿秀这样的普通人。桥本身不产香料，但它让香料流到了该去的地方。

镇上有个教书先生，听说了这件事，在学堂里跟学生们说了一句话，后来成了青河镇的名言：

> "好东西若只能站在高台上，便只造福少数人；唯有搭一座桥，让它流到寻常巷陌，才算真正的好东西。"

---

**📖 原文定义**

> ETF联接基金是将绝大部分基金财产投资于跟踪同一标的指数的某一ETF（称为目标ETF）、紧密跟踪标的指数表现、可以在场外（银行渠道等）申购赎回的基金。根据中国证监会的规定，ETF联接基金投资于目标ETF的市值不得低于联接基金资产净值的90%，并且ETF联接基金的管理人不得对ETF联接基金财产中的ETF部分计提管理费。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 陈掌柜的"百味包" | ETF（交易型开放式指数基金），有最小申购份额门槛，只在场内交易 |
| 阿秀凑不起五十包 | 普通投资者资金量小，无法达到ETF一级市场申赎门槛 |
| 老周的杂货铺 | ETF联接基金，在场外（银行、第三方销售机构）销售 |
| 老周一次进五十包 | 联接基金将绝大部分资产（不低于90%）投资于目标ETF |
| 一包起卖 | 联接基金申购门槛低（通常1~1000元），方便普通投资者 |
| 老周不多收一文钱 | 联接基金持有的目标ETF部分不再重复计提管理费/托管费 |
| 每天来买一包、顺手带一包 | 联接基金支持定投、基金转换等多样化的场外投资方式 |
| "渡桥铺" | 联接基金是连接场外投资者与场内ETF的"桥梁" |
| 老周本身不产香料 | 联接基金不直接买股票，而是投资ETF，间接跟踪指数 |
