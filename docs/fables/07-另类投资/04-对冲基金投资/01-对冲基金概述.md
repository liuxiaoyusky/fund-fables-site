# 对冲基金概述

## 🏺 寓言故事 —— 《双面猎手沈默》

乾隆年间，江南有个商人叫沈默，经营丝绸生意二十年，攒下万贯家财。可他有个心病——生意越大，越怕风浪。有一年运河决堤，他的三船生丝沉了底，十年积蓄一夜归零。沈默在破庙躲雨时，遇见一个白发老者，老者只说了一句话："聪明人不是不赌，而是两面下注。"

沈默回到苏州，变卖了仅剩的老宅，在阊门外租了一间小铺子。他不再只做丝绸，而是同时做两件看似矛盾的事：低价收购被洪水毁坏的桑田，又高价卖出手中存粮。旁人不解，笑他疯了。沈默不语，只在账本上画下两道线——一道红，一道黑。

半年后，朝廷拨银修堤，桑田价格翻了三倍；而那年秋收歉收，粮价飞涨，他卖出的存粮又赚了一笔。红线和黑线交织，沈默竟比从前更富。人们这才明白，他不是在赌一个方向，而是在赌"变化"本身。

沈默的名声传到了京城。一位姓王的侯爷找上门来，说自己在北方有马场，在南方有盐田，可两头都怕天灾，想请沈默帮忙"对冲"。沈默想了想，提出一个条件："侯爷每年给我二百两银子作'管理费'，若赚了钱，再分我两成'业绩报酬'。"侯爷咬牙应了。

沈默的玩法越来越精。他不再只帮一两个人，而是把十几个富商的钱拢在一起，组成一个"基金"。有人想做丝绸，有人想做茶叶，有人想押海运。沈默不让他们各自为战，而是统筹安排：做丝绸的，同时做空棉纱；押海运的，同时买入陆路商队的份额。他对众人说："我不赌哪条路通，我只赌路不会全断。"

最精彩的一仗在嘉庆三年。那年白莲教起义，北方大乱，马场尽毁。王侯爷急得吐血，以为血本无归。可沈默早有布置——他在起义前一个月，悄悄在北方买了大量"空头"：借别人的马匹契约卖出，约定三个月后归还。马价暴跌时，他用极低的价格买回契约填上，一出一进，竟替侯爷赚回三倍。

有人问沈默："你怎么知道会起义？"沈默摇头："我不知道。但我知道，天下不会永远太平，也不会永远大乱。我同时押'太平时的马'和'大乱时的空'，这叫'市场中性'——不管风往哪边吹，我都在风里站着。"

沈默老了以后，收了十二个徒弟，每人专精一门：有的研究"事件驱动"，专等朝廷政令、商帮并购的消息；有的钻研"相对价值"，发现苏州的米比杭州贵一文，就低买高卖；还有的放眼"宏观"，看天象、读邸报，预判天下大势。十二个徒弟，十二种策略，合成一个"多策略"的庞大网络。

有个徒弟叫周小乙，天资聪颖却心高气傲。他嫌师父的"双面下注"太慢，偷偷把所有筹码押在"海运必兴"上。沈默发现后，没有责骂，只让他看账本：周小乙那笔押注若成了，能赚十倍；可若败了，要赔光全部。而沈默自己的账本，每一笔都有红有黑，十年间从未有一年亏损超过一成。

"小乙，"沈默合上账本，"你知道我为什么收'2%的管理费和20%的业绩报酬'吗？那2%，是我替你守夜的灯油钱；那20%，是我替你担惊受怕的辛苦钱。你想省灯油，就得自己守夜；你想不担惊，就得让我分润。这世上没有白吃的宴席。"

周小乙跪在地上，冷汗涔涔。他终于懂了：师父的"对冲"不是胆小，而是一种更高级的勇敢——敢于承认"我不知道"，然后用结构去对抗未知。

沈默死后，他的"基金"由十二个徒弟共同掌管。有人退出，有人加入，但那套"红线和黑线"的规矩传了下来。后来的人给它起了个名字，叫"对冲基金"。而沈默当年在破庙里听到的那个词——"两面下注"——成了这个行当最古老的信条。

---

**📖 原文定义**

> 对冲基金，最初是指运用对冲策略的基金。经过几十年的发展，对冲基金运用的策略大为拓展，不再局限于使用空头头寸进行对冲。广义的对冲基金已经成为一种金融资产类别。

> 对冲基金中基金（Fund of Hedge Funds）是持有对冲基金组合的基金，可以在投资策略和管理风格上实现多样化。其管理人在对冲基金研究上具有专业知识和经验，也能谈判出比个人投资者更好的条款。

> 根据HFR的数据，对冲基金曾经最常见的收费结构曾经是"2/20"，即2%的固定管理费和20%的业绩报酬。

> 市场中性策略的目的是在保持独立于市场风险的同时，从个别证券的变动中获利。管理人通过研究识别价值被低估和被高估的股票，做多被低估的股票，同时做空被高估的股票。理想情况下，投资组合的β应该在零附近。

| 故事元素 | 概念对应 |
|---------|---------|
| 沈默同时低价收桑田、高价卖存粮 | 对冲策略——通过多空双向操作降低风险 |
| 沈默把十几个富商的钱拢在一起统筹 | 对冲基金中基金（Fund of Hedge Funds）——组合多样化 |
| 每年二百两管理费+两成业绩报酬 | "2/20"收费结构——固定管理费+业绩报酬 |
| "同时押太平时的马和大乱时的空" | 市场中性策略——β在零附近，独立于市场风险 |
| 十二个徒弟各专精一门策略 | 多策略基金——股票对冲、事件驱动、相对价值、宏观等 |
| 周小乙单押海运 vs 沈默双面下注 | 对冲基金的核心特点——用结构对抗不确定性 |
