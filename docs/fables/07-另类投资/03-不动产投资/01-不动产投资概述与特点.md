# 不动产投资概述与特点

## 🏺 寓言故事 —— 《老周的三块地》

老周在县城东头有三块地。

第一块是祖上传下来的宅基地，紧挨着新规划的地铁线。三年前，开发商老王找上门，出价八百万，老周摆摆手："不卖。"老王加到了一千二百万，老周还是摇头。他不是贪心，是这块地太特殊了——位置、朝向、风水、邻里关系，全县城找不出第二块。老周心里清楚，这地和别人家的地根本不一样，就像世界上没有两片相同的叶子。这就是**异质性**，每一块不动产都是独一无二的，没法简单比较。

第二块地是他在城郊包下的五十亩工业用地。当年县里招商引资，老周咬咬牙，把全部积蓄砸了进去，还借了两百万。这块地没法拆成五十份卖给五十个人，只能整体持有。**不可分性**让老周的投资像一座山，沉甸甸地压在他的资产表上，占了他总身家的七成。

第三块地最让老周头疼。那是他在邻省某三线城市买下的一个商铺，本想收租养老，结果城市人口外流，商铺空置了两年。他想卖掉，中介告诉他："这地段现在没人要，真要出手，得打七折。"老周这才体会到什么叫**低流动性**——不动产不像股票，点一下鼠标就能买卖。产权过户、税费清算、买家寻找，动辄数月甚至数年。急卖？那就得割肉。

老周的独子周明在省城做基金经理，逢年过节回家，总劝父亲："爸，你把地换成REITs吧，流动性好，还能分红。"老周每次都骂："你懂什么？地拿在手里才是地，换成纸片子，心里空落落的。"

父子俩的僵持，在2023年春天被一场变故打破。

老周查出了早期肺癌，手术费加后续治疗，需要一大笔现金。他急得像热锅上的蚂蚁——宅基地不能卖，那是根；工业用地一时半会儿找不到买家；三线城市商铺更是无人问津。老周坐在医院走廊的长椅上，第一次感到自己引以为傲的"不动产王国"竟如此脆弱。

周明连夜赶回来，没有说"我早就告诉过你"，而是陪父亲坐了一整夜。天亮时，他说："爸，我有个办法。您把工业用地作价入股，加入一个房地产有限合伙项目。您做有限合伙人，只出钱，不参与管理，让专业的开发公司去运作。等项目建成出租，您每年拿分红。同时，您把商铺抵押出去，套出一部分现金应急。等身体好了，您再把部分资金配置到公募REITs里，那些是上市交易的，像股票一样买卖，流动性比直接持有不动产强得多。"

老周沉默了很久，问："那我的地，还是我的地吗？"

"地还是那块地，但您换了一种方式持有它。就像您把一篮子鸡蛋，分成了几个小篮子。REITs相当于把大块的、不可分割的不动产，切成了很多小份，让普通人也能参与。您不用自己当房东收租，专业机构帮您管，租金收益按比例分给您。而且，REITs在交易所上市，您想卖的时候，挂个单就能出手，不用等几个月找买家。"

老周手术后恢复得不错。他采纳了儿子的建议，把工业用地转入了一个房地产有限合伙，把部分资金配置了物流仓储REITs和产业园区REITs。三线城市商铺最终折价卖掉了，虽然亏了，但换来了救命钱。

一年后，老周在省城儿子家过年。他打开交易软件，看着自己持有的REITs份额，股息按时到账，心里踏实了不少。但他也明白了一件事：REITs虽然流动性好，但价格也会波动，遇到利率上涨或者市场不景气，照样会跌。而且，REITs的收益来自底层不动产的租金和增值，如果出租率下降或者租金收不上来，分红也会受影响。

"这世上没有完美的投资，"老周对周明说，"不动产直接投资像种树，扎根深，但挪不动；REITs像盆栽，搬得动，但根浅。各有各的好，各有各的险。关键是，得知道自己手里拿的是什么。"

周明点点头，给父亲倒了一杯茶。

窗外，城市的灯火次第亮起。老周想起自己的三块地——宅基地还在，工业用地正在有限合伙里生息，REITs份额在账户里静静躺着。他终于学会了在不动产的异质性、不可分性和低流动性之间，找到属于自己的平衡。

**📖 原文定义**

> 不动产是指土地以及建筑物等土地定着物，相对动产而言，强调财产和权利载体在地理位置上的相对固定性。不动产投资是指人们为实现某种预定的收益或效益目标，直接或间接地对不动产进行开发、经营、管理、服务等活动并投入资本的行为。

> 异质性：每一项不动产在地理位置、产权类型、用途和使用率等方面而言都是独特的，受法律法规、政策、宏观环境和货币环境等因素影响较多。这给不动产估值带来了困难。

> 不可分性：和传统的股票、债券等不同，直接的不动产投资金额大，且不容易拆分以卖给多个投资者。这意味着一项投资就可能占据投资组合的很大比例。

> 低流动性：由于前两项特性，直接的不动产投资流动性较差，产权交易时间长、费用高。若要快速转售，往往要承担较大的折价损失。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周的宅基地独一无二、无法比较 | 不动产的异质性 |
| 五十亩工业用地只能整体持有，无法拆分 | 不动产的不可分性 |
| 三线城市商铺空置两年、急售需打七折 | 不动产的低流动性 |
| 老周将工业用地作价入股，由专业开发公司运作，自己只出资不参与管理 | 房地产有限合伙（RELP） |
| 老周配置物流仓储REITs和产业园区REITs，像股票一样在交易所买卖 | 不动产投资信托（REITs）的流动性优势 |
| REITs价格会波动，收益受出租率、租金水平和利率影响 | REITs的风险收益特征 |
