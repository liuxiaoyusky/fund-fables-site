# 三、四、各类资产作用与投资策略

## 🏺 寓言故事 —— 《四种排兵布阵法》

### 各类资产的作用

**股票**：增长引擎，提供资本增值和股息收入，长期可抗通胀，但系统性风险无法通过个股分散消除。

**债券**：稳定器，与股票相关性低，提供规律性现金流，某些债券（如通胀挂钩债券）可对冲通胀。

**另类资产**：
- 私募资本：非公开市场机会，与传统资产相关性低；
- 实物资产（房地产、大宗商品）：租金收入+资产增值，通胀对冲能力强；
- 对冲基金：策略多样，追求不同市场环境下的超额收益。

### 核心—卫星策略

把组合分成两部分：
- **核心（60%~80%）**：被动管理，跟踪大盘指数，提供稳定的市场回报；
- **卫星（20%~40%）**：主动管理，投资特定机会或另类资产，追求超额收益。

优点：平衡风险与收益，核心稳定、卫星灵活。缺点：管理复杂，频繁再平衡增加成本。

### 风险平价策略

不按资金比例配置，而按**风险贡献度**配置。股票的波动大，所以配少一点；债券波动小，所以配多一点。目标是让每类资产对整体组合的风险贡献相等。

适合希望减少组合波动性的投资者。

### 全天候策略

桥水基金达利欧提出的策略：不预测经济环境，而是配置能应对各种环境的资产组合。

四种经济环境及对应的优质资产：
- 经济增长：股票、大宗商品；
- 经济衰退：债券、现金；
- 通胀上升：通胀保护证券、大宗商品；
- 通胀下降/通缩：股票、债券。

通过广泛分散降低对单一环境的依赖，在任何市场条件下都能获得合理回报。

### 多元资产基金

把多种资产类别组合在一个基金中，由专业基金经理动态管理。适合资产规模较小的投资者。养老目标基金就是典型例子。

---

**📖 原文定义**

> 核心—卫星策略的核心思想是将投资组合分为一个较大的核心部分和一个较小的卫星部分。核心部分一般采用被动管理方式，占比60%~80%；卫星部分占比20%~40%，主要采用主动管理方式。

> 风险平价策略是一种根据不同资产类别的风险贡献度来确定其在投资组合中的权重的策略，以实现风险均衡配置。

> 全天候策略的设计目标是在不同市场环境下都能获得稳定收益并有效控制风险。其核心理念是通过广泛分散投资，降低投资组合对单一经济环境的依赖。

> 多元资产基金将多种资产类别组合在一个投资组合中，既包括传统的股票和债券、指数基金、金融衍生品，也包括另类投资。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 主力部队 | 核心部分（被动管理） |
| 特种部队 | 卫星部分（主动管理） |
| 按风险而非资金比例配置 | 风险平价策略 |
| 不预测天气，准备各种衣服 | 全天候策略 |
| 四种经济环境对应不同资产 | 全天候策略的四个象限 |
| 一篮子买多种资产 | 多元资产基金 |
