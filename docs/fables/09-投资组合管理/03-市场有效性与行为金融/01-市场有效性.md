# 一、市场有效性

## 🏺 寓言故事 —— 《三种消息灵通程度》

镇上的赌坊有三种不同"消息灵通程度"的玩家。

### 弱有效市场

第一种玩家只看**历史价格走势**。他们研究过去的价格图表、成交量，试图预测未来。但如果市场是**弱有效**的，所有历史价格和成交量信息已经被完全反映在当前价格中。技术分析（看K线图）无法带来超额收益。

### 半强有效市场

第二种玩家研究**公开信息**：公司财报、行业新闻、政策公告、并购消息等。如果市场是**半强有效**的，所有这些公开信息已经被迅速反映到价格中。基本面分析（读财报、做估值）也无法持续获得超额收益。

注意：价格对新信息的反应不一定是瞬间完成的，可能需要一个短暂的调整过程。但如果调整后价格仍然长期偏离，说明市场还没达到半强有效。

### 强有效市场

第三种玩家甚至有**内部消息**：公司高管知道还没公开的业绩。如果市场是**强有效**的，连这些内部信息也已经被反映在价格中。即便是内部人也无法凭借信息优势获得超额收益——任何超额收益只能归因于运气。

### 竞争是市场效率的根源

市场之所以有效，是因为有大量投资者在不断收集、分析信息，试图寻找获利机会。一旦发现价格偏离，套利者就会介入，迅速纠正偏差。正是这种持续的信息竞争与套利机制，使价格趋向有效。

---

**📖 原文定义**

> 弱有效市场，是指证券价格能够充分反映所有历史价格与成交量信息。因此，纯粹依靠历史价格走势进行预测的技术分析将不再有效。

> 半强有效证券市场，是指证券价格不仅充分反映了历史价格信息，而且能迅速、充分地反映当前所有与公司证券相关的公开信息。

> 强有效证券市场是一种理论上的理想状态，指证券价格能够充分、及时地反映所有信息，包括公开发布的信息和未公开发布的内部信息。

> 证券价格之所以能趋向于反映所有相关信息，其根本原因是市场竞争。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 只看历史价格的玩家 | 技术分析 |
| 研究财报的玩家 | 基本面分析 |
| 有内部消息的玩家 | 内幕交易者 |
| 弱有效：历史信息已反映 | 技术分析无效 |
| 半强有效：公开信息已反映 | 基本面分析无效 |
| 强有效：所有信息已反映 | 任何人都无法持续超额收益 |
| 众人寻利纠正偏差 | 市场竞争与套利机制 |
