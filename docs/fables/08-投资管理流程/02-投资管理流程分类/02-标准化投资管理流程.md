# 标准化投资管理流程

## 🏺 寓言故事 —— 《陈掌柜的成衣铺》

陈掌柜在京城最热闹的街市上开了一家成衣铺，名叫"万顺祥"。与那些专给达官贵人量体裁衣的裁缝铺不同，陈掌柜的铺子里挂满了各式各样的成衣——有适合书生穿的长衫，有适合武人穿的劲装，有适合商人穿的绸缎袍子，还有给富贵人家预备的貂裘。任何人走进来，都能挑到一件合身的衣裳。

陈掌柜铺子里的衣裳，从来不是哪个客人说要什么他才做什么。相反，是他自己先琢磨：今年科举在即，读书人肯定多，得多备些素雅长衫；北边战事平息，武人回乡，劲装也该添些新款式；南方商路通了，商人袍子要用上好的苏杭绸缎。他还要考虑店里几个裁缝师傅各人的手艺——李师傅擅长刺绣，王师傅精于裁剪，张师傅做皮货是一绝。综合了市场需求、铺子定位和师傅特长，陈掌柜才定下这一季的货单。

有一天，一个姓赵的秀才走进万顺祥。他看中了架子上那件靛青色的长衫，料子是好棉布，袖口绣着淡雅的竹叶纹，标价五两银子。赵秀才摸了摸口袋，觉得价钱合适，款式也中意，便掏钱买了下来。

赵秀才穿上长衫，走了半条街，忽然觉得袖子略长了些。他折回铺子，对陈掌柜说："掌柜的，这袖子能改短一寸么？"

陈掌柜赔笑道："客官，这衣裳是成衣，尺码都是统一做的。您若嫌长，店里还有别的款式，您再挑挑？"

赵秀才有些不悦："我就喜欢这件，你帮我改改不行么？"

陈掌柜摇头："客官，万顺祥一天卖出几十件衣裳，要是每件都按客人的意思改，那还叫成衣么？您要是非要量身定做，街角刘裁缝铺倒是接这活儿。"

赵秀才悻悻地走了。他并不知道，陈掌柜不是不想改，而是改不得——万顺祥的衣裳，每一件都有固定的尺寸表、用料单和定价，这是铺子的规矩。若因一个客人破了例，其他客人也会纷纷要求改动，铺子就乱了套。真要改规矩，得召集所有买过衣裳的老主顾一起商议，那可不是一件容易的事。

这便是标准化投资管理流程与定制化的根本区别。

陈掌柜的铺子，就像一家公募基金公司。他不问赵秀才具体想要什么，而是先根据市场判断、铺子定位和师傅特长，设计出几款衣裳——也就是基金产品。每一款衣裳都有自己的"业绩比较基准"：长衫对标读书人的体面，劲装对标武人的飒爽，貂裘对标富贵人家的排场。这个基准是陈掌柜自己定的，不是赵秀才定的。

赵秀才买了长衫，就相当于认购了一只基金。他认同这件衣裳的款式和定价，便掏钱买入。但他无权要求陈掌柜为了他一个人，把竹叶纹改成梅花纹，或者把棉布换成绸缎。基金经理——也就是陈掌柜——只会按照既定的"基金合同"来管理这只"产品"，不会因为某个投资者的个别需求而改变策略。

更让陈掌柜头疼的是，成衣铺是"开放式"的——客人今天来买，明天来退，后天又来一个新人。他永远不知道明天架子上要备多少货。有时候一早上卖出十件长衫，下午又退了五件，他得随时调整库存，却不能因此改变长衫本身的款式和定位。这与资产管理计划中资金锁定的模式截然不同。

当然，规矩也不是绝对不能改。理论上，陈掌柜可以贴出告示，召集所有买过万顺祥衣裳的老主顾，商议修改某款衣裳的样式。但这过程极为繁琐——人要凑齐、意见要统一、新规矩要重写、还要报给商会备案。所以实际操作中，几乎没人这么干。真觉得某款衣裳不好卖了，陈掌柜宁可下架停产，另推新款，也不会去改老款的规矩。

几年下来，万顺祥的名气越来越大。因为衣裳是统一做的、统一卖的，价格透明、款式公开，任何人都能拿万顺祥的长衫去和别家铺子的长衫比较——哪家的料子好、哪家的针线密、哪家的价钱公道，一目了然。这便是标准化流程的好处：业绩公开，可与同类产品比较。

而街角刘裁缝铺呢？虽然能给每个客人量身定做，但做出来的衣裳各不相同，外人无从比较，价格也秘而不宣。各有各的好，各有各的客。

**📖 原文定义**

> 公募基金不同于私募基金和资产管理计划，其成立和策略选择并非源于特定投资者的需求，而是基于基金管理人的多方面综合考虑，包括产品线布局、市场关注点和基金经理特点等因素。在这种模式下，基金管理人承担了投资策略设计的责任，而投资者根据需求及风险承受能力选择产品，无法直接影响基金的投资策略。

> 当基金管理人确定了其拟发行基金的策略后，会进一步明确基金产品的各项关键要素，如投资范围、业绩比较基准、投资方法、费率结构等。

> 基金成立后，基金经理将严格按照基金合同执行投资管理。基金经理不会根据个别投资者的需求变化来改变投资策略，而是基于对市场的判断，在法规和基金合同框架内进行投资管理。此外，开放式基金的基金经理还需要应对资金进出不确定性的问题。

> 虽然理论上可以通过召开基金持有人大会来修改基金合同，但实际操作中这一过程较为复杂。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 陈掌柜先根据市场需求、铺子定位、师傅特长设计款式 | 基金管理人基于产品线布局、市场关注点和基金经理特点设计基金 |
| 成衣挂在架上，客人自行挑选 | 投资者根据自身需求及风险承受能力选择基金产品 |
| 赵秀才要求改袖子，陈掌柜拒绝 | 基金经理不会根据个别投资者需求变化改变投资策略 |
| 每件衣裳有固定的尺寸表、用料单、定价 | 基金合同明确投资范围、业绩比较基准、投资方法、费率结构 |
| 召集老主顾商议改规矩极为繁琐 | 修改基金合同需召开基金持有人大会，实际操作复杂 |
| 开放式铺子，客人随时来买退，库存难料 | 开放式基金需应对资金进出的不确定性 |
| 万顺祥衣裳价格透明、款式公开，可与别家比较 | 公募基金业绩公开，可与同类产品比较 |
| 刘裁缝铺量身定做但无从比较 | 私募基金无公开市场数据，无法与同类资产管理计划比较 |
| 陈掌柜宁可下架老款、另推新款，也不改老规矩 | 标准化流程下策略重大变化较难，不如发新产品 |
