# 定制化投资管理流程

## 🏺 寓言故事 —— 《老周的五十年航海图》

老周是江南一带最富有的丝绸商人，年过六旬，积累了二十万两白银的家业。他膝下只有一个体弱多病的独子，自己身体也每况愈下。一个深秋的傍晚，老周把城里最负盛名的船队管家——姓张的年轻人——请到了府上。

"张管家，"老周咳嗽了两声，"我这二十万两，想交给你管五年。五年后，我儿子要成亲，我要给他留一份体面的家业。"

张管家恭敬地坐下，从袖中取出纸笔："周老爷，出海之前，我得先问清楚三件事——您要去哪里、能冒多大的险、路上有什么忌讳。"

老周缓缓道来："我算过，每年能有三千五百两进项，就够家里开销和给儿子备婚事。再多我也不贪心。但有一条——船不能翻，我最多能承受一万两的损失，超过这个数，我全家就完了。还有，我不碰南洋的鸦片船，也不碰朝廷禁运的货物。五年内，这钱不能中途撤回。"

张管家一一记下。这就是**规划**的第一步——把老周的目标和限制，都量化了。

接下来，张管家花了半个月走访码头、询问老水手、查阅近十年的航海日志。他发现，来年季风偏弱，西洋航线会比往年安全；而东洋一带海盗出没，最好避开。他还注意到，瓷器在波斯的价格连续三年上涨，丝绸在扶桑却供过于求。这些判断，就是他形成的**资本市场预期**。

有了预期，张管家开始画航海图。他把二十万两分成几份：十二万两买瓷器走西洋长线，四万两囤生丝等来年涨价，两万两留作应急的现银，还有两万两——他犹豫了一下——决定暂时不动，观望风向。他给每份资产都设了上下浮动的界限，比如瓷器最多加到十四万两，最少不低于十万两。这便是**战略资产配置**。

最后，张管家把这些都写成了一份详细的《出海方略书》，交给老周过目。书中写明了收益目标、风险底线、航线选择、仓位比例、调仓规矩。老周看完后，郑重地按了手印。这就是**投资政策说明书**。

合同签订，船队出发，进入了**执行**阶段。

张管家首先**建仓**。他打听到，明年开春波斯王室要办大婚，对青花瓷的需求必涨；而扶桑那边，几个大织坊因火灾停产，生丝价格有望回升。于是他下令：七成瓷器仓位押注青花，两成生丝，剩下一成分散到药材和香料。在西洋航线上，他特意选择了七艘新造的大船——船体坚固、速度适中，正好能吃到季风转换期的"骑乘"之利。

船队出海两个月后，张管家收到消息：东洋某港口因瘟疫封城，当地囤积的茶叶价格暴跌。他判断这是短期乱象，迅速调拨五千两白银，低价收了一批上等茶叶。一个月后瘟疫解除，茶价反弹，他高价卖出，赚了一笔快钱。这种在既定方略框架内的灵活操作，就是**战术性调仓**。

半年后，风向变了。朝廷颁布新令，西洋航线加征重税，而东洋因签订通商条约，关税大减。张管家判断，这不是一时之风，而是长达数年的大势。他急忙修书一封，与老周商议。老周同意后，两人在原合同上补了一份协议：削减西洋瓷器仓位至零，把东洋生丝仓位加到十三万两。原本的战术调整，就此升格为新的**战略资产配置**。

日子一天天过去，船队有赚有赔。张管家每季度都要做一次**再平衡**——比如原本定好瓷器占六成，结果因为一批货暴利，仓位涨到了七成，他就会卖掉一部分，把比例拉回六成。这不是因为他不看好瓷器，而是他要确保整支船队的风险结构，始终符合老周当初的要求。

五年期满，张管家把账本摊在老周面前。他算了总收益、分析了每一笔盈亏的来龙去脉、还拿自己的成绩和当初定的目标——年化三千五百两——做了对比。这就是**业绩评估**中的业绩度量、业绩归因和绩效评估。

老周看着账本上稳稳当当的数字，长舒一口气。他知道，这五年，张管家不是在替他赌运气，而是在替他走一条事先画好的路——有目标、有边界、有预判、有调整、有回望。

**📖 原文定义**

> 规划侧重于确定投资方案所需的各种输入信息，包括客户需求和资本市场预期数据等；执行是资产配置和证券选择等方面的投资决策和实施过程；反馈则是对投资预期、投资目标和投资组合等方面变化的适应过程。

> 投资规划的首要任务是了解投资者需求，确认并量化投资者的投资目标和投资限制。投资目标是投资者对收益和风险要求的结合体，投资限制是对投资过程的进一步约束，如投资期限、流动性要求、监管规定及特殊要求等。

> 在战略资产配置中，投资管理人结合投资者投资目标、投资限制、资本市场预期，确定各类目标资产的配置比例，并设置各类资产权重的上限和下限，作为投资过程中风险管理的依据。

> 再平衡是指在资产价格发生变化后，投资经理将组合重新调整为最初设定的资产配置比例。再平衡的核心目的是确保投资组合的风险收益特征始终符合投资者的预期和要求。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周提出年化3500两收益、最多承受1万两损失、不碰禁运货物 | 确定并量化投资者的投资目标和投资限制 |
| 张管家走访码头、查阅航海日志、判断航线风险与机会 | 形成资本市场预期 |
| 二十万两分四份，各设上下浮动界限 | 建立战略资产配置 |
| 《出海方略书》写明目标、航线、仓位、规矩 | 制定投资政策说明书 |
| 船队出发，按既定比例买入瓷器、生丝、药材 | 建仓 |
| 瘟疫封城时低价收茶、一个月后高价卖出 | 战术性调仓 |
| 朝廷改税后与老周商议、补签协议调整航线 | 战略资产配置调整（需投资者同意） |
| 每季度把涨超比例的仓位卖出一部分、拉回原定比例 | 再平衡 |
| 五年期满算总收益、分析盈亏来源、与目标对比 | 业绩度量、业绩归因、绩效评估 |
| 整个流程分"问清楚—画好图—出海走—定期回望"四段 | 规划、执行、反馈三步骤 |
