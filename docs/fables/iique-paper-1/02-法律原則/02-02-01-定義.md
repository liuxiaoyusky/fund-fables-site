# 2.2.1 定义

## 故事

那年秋天，周太的母亲在中环看房时看中了一套二手房，交定金那天，周太临时要接孙子放学，走不开。

"我去不了，"周太对黄师傅说，"但我信你。你帮我签吧。"

黄师傅愣了一下："帮？你打算点帮法？"

"你去签字，律师楼那边就算你代我做代表。"周太说，"签了你的名字，但法律上等于是我签的。"

黄师傅觉得这事不简单，没敢贸然答应。他先去找了梁志明。梁志明听完，说："周太想让你做佢慨代理人——简单讲，就系你代她行事，结果由她承担。"

黄师傅说："那不是要我负法律责任？"

"不止你。"梁志明翻开笔记本，"代理关系里牵涉三层人：委托人（即周太）、代理人（即你）、还有第三者（即律师楼和卖方）。你签了字，周太受约束，但如果签的内容有问题，卖方也可以告你。"

林若兰那天刚好经过德辅道的街角茶餐厅，看见黄师傅和梁志明坐在里面，就走过来坐下了。听了几句后，她放下手里的文件夹，说："其实代理关系不一定需要有合约才能产生。"

"啥？"黄师傅不解。

"比如一个细路哥帮阿妈去士多买一包糖，"林若兰说，"他同阿妈之间没有签任何合约，但阿妈授权了他去买嘢，他买了糖回来，士多老板当然找阿妈收钱，而不是找那个细路哥。"

梁志明接话："这就系代理关系的关键——代理人代表委托人与第三者发生关系，但真正的权利义务发生在委托人与第三者之间，不是代理人与第三者之间。"

黄师傅想了很久，问："那如果周太事后说'我无叫过你签'，我怎么办？"

林若兰轻轻摇头："那就看周太事先有没有明确授权，或者事后有没有'追认'你的行为。如果没有授权、也没有追认，那你的签字就是你自己慨行为，周太可以唔认。"

黄师傅的脸色变了一变。他最后说："算了，我叫周太亲自去签。如果实在不行，让她写一张委托书给我，写明委托我做代理人，签了名我再去。"

梁志明点点头："这就对了。代理关系可以是明示的，也可以隐含的，但最稳阵慨做法系有一张书面委托。"

周太后来亲自去了律师楼。但那天黄师傅陪她一起，在等候区坐着等叫号。周太进去之前，黄师傅说了一句："如果律师讲得太快，你叫我，我进来帮你听。"

周太愣了一下，说："好。"

后来梁志明在笔记本上画了一张关系图：左边是委托人，中间是代理人，右边是第三者，三条线连在一起，写着"代理关系"四个字。他把这一页折了个角，留着以后给客人解释。

## 原文定义

代理关系是指存在于委托人与代理人之间的关系。代理人经常代表委托人与第三者安排协议或代为履行协议，因此代理法也须结合合约法一并考虑。代理关系不一定需要精确的委任程序才能产生，某些特定情况（如家事中的合理授权）亦可构成代理关系。代理关系涉及两类不同的合约：委托人与代理人之间的合约，以及委托人与第三者之间的合约。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周太让黄师傅帮她签字 | 代理人代表委托人与第三者（律师楼）行事 |
| 细路哥帮阿妈买糖 | 无代理合约仍可构成代理关系——关键在于授权 |
| 代理人签字，委托人受约束 | 代理关系的本质：权利义务落在委托人与第三者之间 |
| 委托书须写明授权范围 | 代理关系可通过明示协议产生 |
