# 2.2.5 委托人对代理人的责任（Duties Owed by Principal to Agent）

## 故事

梁志明在西环的保险行做了两年，街坊都叫他「后生哥」，但他不介意。湾仔装修判头黄师傅那年接了一个大单，需要帮手处理一批建材的保障文件，黄师傅图方便，问梁志明有没有辨法。

两个人在茶餐厅坐下，黄师傅把情况摊开：「有一批钢材，从葵涌码头运到观塘工厂，海上那段路我想有个兜底。志明，你帮我问问，几多钱？」梁志明花了三个下午，跑码头看货柜，又去几家保险行问价，最后替黄师傅选了一家，替他垫了验柜费和文件费，一共垫了三百二十块。

黄师傅接了文件，拍拍梁志明膊头：「做嘢，我心里有数。」

可是那一单完工之后，黄师傅迟迟没有把款项结清。梁志明去湾仔的工地找他，黄师傅总是说「工程尾数未到，过两日」；再去，又说「周太话要先留钱周转」；第三次去，黄师傅干脆躲了起来，让伙计阿标出来挡。

周太知道这件事后，在屋企和黄师傅吵了一顿：「人家帮了你，你迟迟不还，迟早被人闹上门，到时丢面的不仅是你。别人垫了钱，出了力，你拖着拖着，以后谁帮你？」

黄师傅坐在客厅，不说话。

几天后，梁志明在街上碰见了黄师傅，黄师傅主动走过来，把一个信封递给他：「志明，上次的事，迟了，唔好意思。这是垫的费用，加埋你帮我来来回回跑的手工费，一共四百，我给你五百，多的算利息。」

梁志明打开信封，点了点，抬头看着师傅：「Whitfield 律师楼的合约你拖过一次，林若兰替你催过一次，这次点解主动还？」

黄师傅说：「周太话：『点可以白嫖后生的心机。』我现在才知道，别人帮了你，唔俾钱，以后就冇人帮了。」

## 原文定义

委托人须按双方协定的金额或标准，于合理时间或指定期限内向代理人支付报酬。委托人亦须偿还代理人因代表其履行职务而合理招致的费用及开支。若委托人违反义务，代理人有权向委托人追讨。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明垫付验柜费和文件费共三百二十元 | 代理人因履行职务而合理招致的费用 |
| 黄师傅迟迟不还钱，称「工程尾数未到」 | 委托人未于合理时间内偿还款项 |
| 周太责怪黄师傅白嫖后生的心机 | 委托人不得以内部原因拒绝履行对代理人的义务 |
| 黄师傅主动连本带利归还五百元 | 委托人须按约支付报酬；迟延须承担利息或额外责任 |
