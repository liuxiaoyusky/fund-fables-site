# 2.1.2 合约的种类

## 故事

湾仔黄家，唐楼的二楼，厨房里周太正在算账。

黄师傅坐在客厅的小板凳上，脚边的水泥地上散落着几张分包商送来的催款单。事情是这样的：上个月有间本地洋行请黄师傅承包一栋旧楼的翻新工程，工期紧，价钱也谈得爽快，洋行老板当着周太的面拍胸口说"咱俩口头算数，你做我放心"，还亲自倒了杯茶递给黄师傅。

黄师傅高兴，觉得这是信任。签合同还要等律师楼排期，口头说定了更利索。

谁知道工程干到一半，洋行老板换了个人——原来的老板移民了，接手的是他儿子。新老板翻出账本，说旧老板口头答应的那些条款没有记录，不算数，要重新谈价钱。黄师傅傻了眼，人工已经进去一半，材料也订了，进退两难。

周太那天晚上在茶餐厅里当着街坊的面发了火："你次次都话握手就算数！上次灯坏了那个也是这样，说好免费帮你整，结果整完又要收钱，说口头承诺不算承诺！"

黄师傅铁黑着脸没反驳。街访介绍他去中环的保险行找梁志明，说阿明识英资律师楼的 Mr. Whitfield，或许有办法。

Mr. Whitfield 听完经过，翻了翻一份讲义，用略带口音的广东话慢慢解释："黄生，香港法律里合约分两种。一种叫简单合约——这种合约的订立非常简单，不需要特别手续。你们刚才说的握手、喝茶说好、甚至递一支烟，都可能构成简单合约。但这种合约，要有人反悔，打起官司来，你得有证据。"

"第二种呢？"黄师傅问。

"第二种叫契据，"Mr. Whitfield 指了指桌上一份厚重的烫金封面文件，"这种合约要求更严格——必须书面、签名、盖章、交付。少了任何一步都不行。好处是，这种合约就算没有实际'代价'，也可以强制执行。比如担保合约，如果做成契据形式，担保人就算没收到钱，债权人也能告他。"

他合上文件夹："但反过来，你今天签的这份口头协议，简单合约——没有白纸黑字，遇上对方不认，你就吃亏。"

黄师傅想起周太这些年每次都说"签纸才算数"，自己每次都说她太计较。现在才知道，香港法律承认两种合约，区别就在于有没有那一套"特殊手续"。

那天夜里回到家，周太没有再骂他。她只是从柜子里翻出一本旧账簿，在新的一页写下工程的条款，说："下次讲好数，我哋一齐去律师楼签纸。"

黄师傅望着她的背影，想起当年求婚时自己也是这样两手空空，只带了一句"我会照顾你"——而她真的照顾了这个家四十年。他没出声，只是把账簿接过来，在第一条旁边画了一个括号，等着填金额。

## 原文定义

合约可分为两类：(a) 简单合约（Simple contracts）：订立简单、容易，不依赖特殊手续，可为口头、书面或从行为中推断；(b) 以契据（Deeds）形式订立的合约：须经签署、盖章和交付，某些交易（如土地转让、担保）必须以契据达成，否则可能无法强制执行。绝大部分保险合约属于简单合约，实际上虽不必须有书面形式，但几乎一律采用书面。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 黄师傅与洋行老板握手喝茶说好价钱 | 口头简单合约——订立容易，但没有特殊手续保障 |
| 新老板否认口头承诺 | 简单合约若无书面证据，发生纠纷难以证明 |
| Mr. Whitfield 解释契据须签名盖章交付 | 契据合约有特殊形式要求，效力强于简单合约 |
| 担保必须做契据才能确保起诉权 | 契据可绕过"代价"要求而强制执行 |
