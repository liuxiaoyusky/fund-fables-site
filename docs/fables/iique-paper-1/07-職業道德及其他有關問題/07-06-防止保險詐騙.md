# 7.6 防止保险诈骗

## 故事

周老板的海味舖那年冻仓春节前被窃，损失不小。保险理陪下来之后，他请梁志明到店里吃开年饭，席间聊起街坊们的近况，忽然压低了声音。

「阿明，我有个相识，在元朗开车房，他的徒弟年前出了事——人在修车的时候被砸断了三根肋骨，住了两个月医院。」周老板摇摇头，「佢老婆走了之后带住个女，现在要养家，车房收入不够。」

「保险公司赔了没有？」梁志明问。

「赔是赔了，但唔够。」周老板把茶杯放下，「那个徒弟现在想再报一次，说之前有些费用『未计齐』。他问我，能不能找当时认识他情况的熟人，帮他『搞』一份新的医疗文件和损失清单。」

梁志明的手停在半空。

「周老板，这种事——」

「我知道。」周老板打断他，「我也跟他说了，唔好搞。但佢话，如果唔咁做，个女下学期就无钱交学费。」

那晚梁志明从海味舖出来，走到电车站的时候，林若兰正好从律师楼下夜班，远远看见他，问：「这么晚，一个人站着发呆？」

他把周老板说的事告诉她。林若兰静静听完，没有马上回答。

他们沿著德辅道西走了几步，林若兰才开口：「阿明，你记唔记得 Mr. Whitfield 去年在合规课上说的一句话？」

「边句？」

「他说，保险这个制度，基础系信任。投保的人信保险公司会在出事的时候赔钱，保险公司信投保的人如实告知真实情况。如果呢个基础没了，整个系统就会崩塌。」

梁志明点点头。

「那个车房师傅的处境当然可怜，」林若兰说，「但如果佢用假文件多拿一分钱，就等于从所有如实申报的人口袋里偷了一分钱。保险公司的损失最终会转嫁到所有正常缴费的人身上。」

「那怎么帮他？」

「先搞清楚他到底『未计齐』的是什么。」林若兰说，「有些合理的费用第一次可能确实漏报了——这种情况可以通过正规渠道补交证明申请覆核。但如果佢想的是夸大损失或者伪造文件，那就系另一回事了。」

她停了下来，看著梁志明的眼睛：「阿明，你系唔会帮他伪造文件的对吧？」

「梗系。」

「那就够了。」林若兰说，「你可以帮他找正常的补助途径，但唔可以踩过线。有些底线一旦踩了，就再也返唔到转头。」

第二天，梁志明请了半天假，亲自去了一趟元朗车房。他和那个断了肋骨的师傅谈了整整一个下午，最后帮他整理了一份真实的费用清单，递交给保险公司覆核。

保险公司最终追加了三千多元赔偿。唔够，但真实。

师傅接过钱的时候有点激动：「阿明，唔好意思，听人讲可以走捷径，一时起了贪念。」

「人之常情。」梁志明拍了拍他的肩，「但捷径的最后一站，往往比正路更远。」

那年圣诞，周老板送了一盒腊肠到保险行，附带一张字条：「车房师傅的女儿今年考上了IVE，谢谢你。」

梁志明把字条收进抽屉里，没有告诉任何人。

## 原文定义

诈骗是指「不诚实」或「作弊」。保险属于一个包含高度信赖成分的过程，有不少空间给不诚实的人有机可乘。诈骗可以发生在索偿层面（如虚假或夸大的索偿），也可以发生在投保层面（如刻意歪曲重要资料或明知而隐藏不利因素）。这些行为均属违反最高诚信原则。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 车房师傅想用假文件「再多拿」理赔 | 诈骗是透过不诚实行为寻求非法保险利益 |
| 师傅起初决定瞒报部分资料再报一次 | 投保层面的瞒报与歪曲事实同属诈骗 |
| 林若兰指出「系统的信任基础一旦崩塌」 | 保险的高度信赖成分使诈骗有机可乘 |
| 梁志明帮师傅正规覆核真实漏报的费用 | 合规渠道覆核是正当途径，唔系姑息瞒报 |
| 师傅最终承认一时起了贪念 | 违反最高诚信原则，属诈骗行为 |
