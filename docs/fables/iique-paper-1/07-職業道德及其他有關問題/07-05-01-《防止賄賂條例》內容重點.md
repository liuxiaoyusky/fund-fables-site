# 7.5.1 《防止贿赂条例》内容重点

## 故事

一九七九年十一月，周老板从干货铺拿了一盒金华火腿上保险行。

他跟了保险行二十年，跟梁志明的爸爸那辈人就认识。现在梁志明接了班，周老板的保单也转到了他手上。这次干货铺续保，梁志明帮他省了不少保费，周老板心里过意不去，非要送点东西。

梁志明看着那盒火腿，说："周老板，这个我不能收。"

周老板说："一盒火腿而已，又不是什么大钱。你爸以前收过我送的腊肠，你们街坊之间送点东西，有什么问题？"

"不是——"

"不是不给面子吧？"

梁志明正要再推辞，Mr. Whitfield从走廊那头走过来。Mr. Whitfield是这间英资律师楼的合规律师，平时主要管文件合规，今天刚从会议室出来，听到了最后几句。

"周老板，"Mr. Whitfield看了看那盒火腿，"这盒火腿，我先替阿明收下了。"

周老板脸色缓和了。梁志明愣在那里。

Mr. Whitfield把那盒火腿接过来，拍了拍手，对走廊那头喊了一声："若兰，麻烦你拿刀来，我们现在切了，请在场所有人喝茶。"

林若兰从办公室探出头来："Thompson先生不在，钥匙在他那里。"

"那就在这里切，"Mr. Whitfield说，"这盒火腿是好东西，大家一起尝尝。周老板，您也坐下来，我们一起吃。"

林若兰去隔壁借了刀，切了火腿，分给在场所有人。梁志明、周老板、陈伯、靓姐一人一片。

吃完之后，Mr. Whitfield对周老板说："周老板，这盒火腿是好东西，我替阿明收下，是因为您的心意不能糟蹋。但我也要跟您讲清楚，这东西以后不能再送了。"

周老板皱起眉头："这——"

"不是不给面子，"Mr. Whitfield说，"是因为香港有一条规矩，叫《防止贿赂条例》。阿明是我们保险行的职员，您是阿明的客户，您给他送东西，就叫提供利益。《条例》第9条写得很清楚，代理人未经主事人允许，不得接受任何利益。"

周老板这下听明白了："你是说，我送阿明这盒火腿，犯法？"

"严格来说是的。"

周老板愣了。林若兰把剩下的火腿包好，递给他："周老板，这盒您拿回去，下次不要再送了。"

周老板把火腿接过来，脸上的表情很复杂。他说："阿明，我不是想害你，我真的只是想——"

"我知道，"梁志明说，"但规矩是这样。我收了，我犯法；我不收，我也没不给面子——您看，Mr. Whitfield帮您收了，还切给大家一起吃了。"

Mr. Whitfield点了点头："周老板，《条例》里有一点您要记住：款待不算利益。就是说，您请阿明吃一顿饭，当面喝杯茶，这个不算。但送东西就不一样，金钱、馈赠、赏钱、利是、茶钱，全部都算。"

周老板把火腿盒抱在怀里，叹了口气："行，我记住了。下次我请你喝茶，请你吃饭，但不送东西。"

他走的时候，回头说了一句："阿明，你们这间保险行有规矩，我反而放心。说明您的保单将来赔得出钱。"

梁志明后来把这件事记在了笔记本上，在结尾写了一句话：好客户可以留，但规矩不能松。

林若兰经过他身边，看到了那句话，说："你怎么知道周老板是'好客户'？"

梁志明说："他要真想贿赂我，不会送一盒火腿。他送火腿，是因为他真把我们当街坊。街坊送礼是心意，不一样。"

林若兰想了想，没说话，把这句话也记在了自己的笔记本里。

## 原文定义

根据《防止贿赂条例》第9(1)条，代理人在处理其主事人的业务或事务时，未经主事人的允许，不得索取或接受任何利益。第9(2)条则规定，提供利益者同样违法。利益包括金钱、馈赠、贷款、佣金、赏钱、利是、茶钱等任何有价值的事物；只有当场的饮食款待不算利益。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板送火腿给梁志明，代理人收受利益 | 《条例》第9(1)条：代理人不得收受利益 |
| 周老板主动送礼 | 《条例》第9(2)条：提供利益者同样违法 |
| Mr. Whitfield当场切开分享，维持心意但不私自收下 | 合规处理客户馈赠的方式 |
| 请喝茶吃饭不算，送火腿就算 | 款待不算利益，馈赠算利益 |
| 周老板说送火腿是因为当他们是街坊 | 利益定义不含金额门槛，赏钱茶钱均算 |
