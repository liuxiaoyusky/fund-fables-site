# 7.4.6e 客户尽职审查

## 故事

周老板第一次坐在永亨保险行的柜台前时，带了一保温瓶茶，还带了一张手写的家庭地址。

那张纸是林若兰接过去的。她是中环英资律师楼的中英打字员，这天下班后帮永亨保险行校对英文文件，刚好碰上了。

"我的海味铺地址就是这个，"周老板指了指纸上那行字，"上环海边街，你们来查也行，我屋企人住这里也行。"

梁志明接过表格，扫了一眼职业那一栏：海味干货批发。然后他把表格翻过来，背面还空着。

"周老板，"他说，"你平时有做外汇或者汇款生意吗？"

周老板愣了一下："我卖花胶鲍鱼，哪来的外汇。"

"那有没有家人经常在海外？"

周老板想了想，说儿子在加拿大读书，每个月家里要汇钱过去。梁志明在表格背面记下了这一点，然后合上文件夹。

林若兰从旁边抬起头，说："周老板这种情况，算是低风险吧。没有政治敏感人物背景，没有大额现金来源不明，做的是正经干货批发生意，账户使用情况也稳定。"

梁志明点点头。

这就是指引3说的"简化尽职审查"。如果一个客户风险不高，背景清楚，日常活动符合他的生意性质，那么繁复的核查程序就可以简化。不是不查，是按比例来。

后来周老板再来续保费的时候，带了一张温哥华的汇款回执。回执上写着寄给儿子加币两千。梁志明没有追问那笔汇款的来源——因为之前已经问过了，已经记录在案了。

但有一次，情况不一样。

有一回一个叫陈仔的年轻人走进保险行，说要帮父亲买一份大额人寿险，受益人写的是一个陌生名字。陈仔说这是"中介费"，但又说不出是哪家中介。保费金额不大，但保额奇高，而且他要求把现金价值提走。

梁志明做完记录，没有马上接这张单子。

林若兰那天也在。她看了一眼受益人那栏，说："这个人名你查过没有？"

查过了。和已故某位商人的遗产案有关联。

梁志明按程序走了加强版尽职审查。他没有直接把单子退回去——那样做等于告诉对方你被盯上了。他只是请陈仔多等几天，等合规主任核完这单。

后来这张单子没有做成。不是因为拒绝，是因为陈仔自己没再来。

周老板后来跟别人说起永亨保险行，说那家店的伙计"问得多，但不麻烦"。他的意思是：梁志明问了很多问题，但问完之后该办的事照办，不像有些地方，要你跑断腿才能把事情说清楚。

梁志明后来跟林若兰说，这就是尽职审查的意义——不是为了刁难人，是把事情搞清楚，然后替真正需要保障的人把门守住。

## 原文定义

指引3规定，金融机构可按具体情况采取额外措施或采取简化的尽职审查措施，并赋予金融机构一定程度的酌情权，但须以风险为本，根据客户的实际风险状况决定审查的深度。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明问周老板"有没有海外家人汇款"并记录在案 | 尽职审查须了解客户的业务性质和正常活动模式 |
| 周老板背景清楚、无政治敏感身份，适用简化审查 | 低风险客户可适用简化尽职审查 |
| 陈仔投保大额寿险、受益人陌生、保额异常高 | 高风险情况须适用加强版尽职审查 |
| 梁志明请陈仔"等合规主任核完再通知"而非直接拒绝 | 加强审查须按程序进行，不向对方透露正在审查 |
| 林若兰说周老板"风险低"、陈仔受益人"要查" | 尽职审查以风险为本，审查深度因情况而异 |
