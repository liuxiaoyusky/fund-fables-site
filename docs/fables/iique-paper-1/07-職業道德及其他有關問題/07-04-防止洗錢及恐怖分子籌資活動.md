# 7.4 防止洗钱及恐怖分子筹资活动

## 故事

一九七九年七月的一个下午，梁志明在何师奶的茶餐厅里喝茶。

茶餐厅叫"利群"，在湾仔一条小巷子里。何师奶早上五点就起来开铺，晚上十点才收，十几年如一日。梁志明是这间铺的老客人了，读中学的时候就来这里吃早餐。

这天，何师奶在擦桌子的时候，邻桌两个中年男人在聊天。梁志明本来没在意，但听到其中一个人说"洗干净"三个字，他就把茶碗放下了。

"那笔钱是从外面弄进来的，换一层壳，变成正当生意收入，谁都查不出来。"一个人说。

另一个人笑了笑："哪有那么容易。"

"你不懂，现在有些行业，根本没人查那么细。"

何师奶走过来收他们用完的碗，顺手把那两个人瞪了一眼。那两个人识相地收了声，不一会儿就结账走了。

何师奶回来给梁志明添茶，低声说："阿明，刚才那两个人，你听到了？"

"听到一点。"

"他们说的是什么意思？什么叫洗干净？"

梁志明想了想，说："何师奶，我刚做保险这行不久，我也不算懂全。但我试着解释一下，您听听有没有道理。"

他把茶碗推到一边，认真地说："比如说，有人在外面做了一些不能见光的生意，赚了一笔钱，他想把这笔钱拿回家用。但这笔钱来路不明，如果直接拿出来花，会引起注意。所以他想个办法，让这笔钱过一个正当的渠道——比如一间正常的商号，一笔正常的生意——然后钱就变成了'正当收入'。这个过程，就叫洗钱。"

何师奶皱起眉头："那不是骗人吗？"

"是。但不只骗人。"梁志明说，"更麻烦的是，有些人拿保险来洗钱。他买一份保单，过几年退保，拿回来的钱就变成了正当收入。所以我们这行有规矩：收到钱、签保单、退保取钱，每一步都要问清楚钱从哪里来。"

何师奶明白了："所以你们要查客人的底细，不是因为不相信人家，是因为要确保进来的钱是干净的。"

"差不多是这个意思。"梁志明说，"恐怖分子筹资活动也差不多是这个路数——有些人想筹钱干坏事，他们会用各种看起来正常的渠道来收款。我们这行的人，要能认出那些不正常的交易。"

何师奶沉默了一会儿，然后说："我茶餐厅每个月也有几笔大的现金进账，是酒楼和商号的人来包席面的。那些钱干净不干净，我也搞不清楚。"

"何师奶，您这是收现金，不是过账，一般问题不大。但如果您哪天发现有人用您的铺头来转账，您得小心。"

何师奶点了点头。

梁志明喝完那杯茶，起身要走。何师奶叫住他，从柜台下面拿出一张纸，是利群茶餐厅的银行账号。她说："阿明，你帮我看看，以后有人要汇款进来，要怎么查才放心。"

梁志明把那串数字看了一遍，说："何师奶，这个您先留着。等我回去问过Mr. Whitfield，下回再来的时候，我给您讲清楚。"

他走的时候，雨刚好停了。何师奶站在门口，把那块写着银行账号的纸折好，放进了放钱的铁盒子里。

## 原文定义

金融机构有责任防止其服务被用来进行洗钱或恐怖分子资金筹集活动。保险业作为金融服务的一部分，需要对资金来源保持警觉，对异常交易进行核查。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 有人想把来路不明的钱通过正当生意"洗"成合法收入 | 洗钱的基本概念 |
| 用保险保单作为渠道，过几年退保取钱 | 通过保险产品洗钱的风险 |
| 恐怖分子筹资活动利用正当渠道收款 | 恐怖分子资金筹集活动的渠道 |
| 梁志明解释保险行每一步都要问清钱从哪里来 | 保险业的防止洗钱责任 |
| 何师奶请梁志明帮她核查汇款进来的钱是否干净 | 机构对客户资金来源的审查义务 |
