# 7.1.2 如果保险中介人是保险代理人

## 故事

周老板后来想加一份雇员补偿保险，经朋友介绍找了一个代理人。那代理人姓李，是英资保险公司的代理，穿着整齐，拿着一叠文件上门，把条款讲得很清楚。周老板投保三个月后，厨房伙计阿福工作时摔断了手，要申请理赔。

李先生帮周老板整理了文件，交到公司核赔部。核赔部认为阿福那天没穿防滑鞋，按条款属"违反安全操作规程"，要扣三成赔款。周老板觉得不合理，李先生也觉得条款有点严，但他是保险公司的代理，签的是保险公司的牌子，立场就有点尴尬了。

他把核赔部的决定如实告知周老板，又写了一封求情信给公司，说客人是新客，铺头一向管理尚可，伙计平时也有穿防护鞋的习惯，这次是意外。最后公司给了八五折赔偿，李先生把结果告知周老板，说："我帮你争到呢个结果，但保险公司最终决定权唔系我度。我只可以尽量反映情况。"

周老板有点不高兴："你系我揾嚟帮我攞赔偿，唔系帮公司解释。"

李先生说："周老板，我明你嘅感受。但我系保险公司的代理，唔系你请的。我有责任帮你，但我唔系你嘅代表，公司的决定我无法推翻。我能做嘅，就系如实讲清楚、尽力争合理结果。"

事后周老板跟梁志明提起这件事。梁志明想了想，说："你下次买嘢，如果想有人完全代表你立场，可以揾经纪。但代理人唔代表你，不代表对唔住你——佢只系讲清楚自己位置，两头都唔走埋。"

梁志明后来跟新人解释："做代理，你要记得你张牌系边个发嘅、你对边个负责。但你对客人的道德责任唔会因此消失，该讲清楚嘅要讲清楚，该做嘅要做——只不过立场同经纪唔一样。"

## 原文定义

保险代理人的委托人是保险人，而非被保险人。因此，代理人对保险公司负责，但并不获豁免履行法律及道德义务。保险代理人须遵守《保险业条例》相关条文、保险业监管局订立的规则及颁发的守则和指引。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 李先生说"公司的决定权唔系我度" | 保险代理人的委托人是保险人 |
| 李先生如实告知核赔结果并写求情信 | 对客户仍须履行道德义务 |
| 梁志明解释"经纪代表你，代理唔系" | 代理人与经纪人的根本区别 |
| 代理人两头都不走埋 | 代理人须明确立场但不获豁免义务 |
