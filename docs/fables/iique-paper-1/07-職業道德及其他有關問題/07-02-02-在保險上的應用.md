# 7.2.2 在保险上的应用

## 故事

周老板的海味铺开在上环水坑口街，三层唐楼，一楼卖鲍鱼花胶，二楼存货，三楼住人。那年夏天他想把铺头连存货一起投保，找的是梁志明所在的本地保险行。

梁志明二十出头，在保险行做了两年多，说话带街坊腔，看得懂英文保单。周老板信任他，因为梁志明的阿叔和周老板同一条街，周老板常说"自己人，信得过"。

梁志明拿了投保申请表上门，周老板在货仓里翻箱倒柜找营业执照，梁志明坐在阁仔填表。表格是总公司印的中英双语版本，印刷精美，密密麻麻。他填到第三页，忽然停住了。

第三页倒数第二栏写着：请列出投保人过去十年内的所有住院记录、手术名称、正在服用的药物，以及家族病史概述。

周老板拿了一叠银行存折过来，看见梁志明盯着那栏发呆，说："填啊，我前年割过胆，你写上去啰。"

梁志明抬起头："周老板，这份是火险连营业中断险，你开海味铺，怎么会要问你家族病史？"

周老板也愣了："是啊，我哪里知道，他们印我就照填啰。"

梁志明把表格收好，说要回去核对一下。

那天傍晚，他经过中环的英资律师楼，顺路去找林若兰。林若兰是律师楼的中英打字员，平时替好几家洋行和保险行打印文件，梁志明有几份英文保单条款看不懂，常拿去找她帮忙看。她坐在打字机前，桌上整齐堆着一叠叠待处理的函件，标签纸颜色各异。

"若兰，这份申请表是不是有问题？"梁志明把表格拿给她看。

林若兰看完，眉头微了一下。她没直接回答，而是从抽屉里抽出一本薄薄的小册子，封面上印着"个人资料（私隐）条例"几个字。

"梁仔，你知道吗？"她轻声说，"私隐条例有六项保障资料原则，其中第一项就说得很清楚：收集资料的目的要合法，收集的范围不能超过适度。你老板这份表，火险要问家族病史，这就是'超乎适度'。"

"那应该怎样？"

"比如周老板切除胆那次，如果他今天是因割胆来索偿医疗费用，那保险公司只需要知道那次手术和这次索偿有没有关系，不需要知道他十年前膝盖做过什么。"林若兰指着表格上那行字，"这条问题，等于把整间诊所的病历都复印了一遍。"

梁志明把那页折了个角。

回去之后，他找到保险行的负责人，说这份申请表的部分内容不符合私隐条例的要求。负责人姓陈，做了十几年保险，听完脸色不太好看，说："人家总公司统一印的表格，照填就是了。"

梁志明说："私隐条例去年已经生效。我查过指引，公署给保险业界发过一份专门文件，叫《妥善处理客户个人资料：给保险业界的指引》，里面说得很明白，收集的资料不能超乎适度。如果被查到，是我们保险行的责任，不是总公司的。"

负责人沉默了一阵，最后说："你改吧。"

梁志明把那份申请表重新整理了一遍，只保留与火灾及营业中断相关的合理问题，交还给周老板签字。周老板看完，问他："那个什么'不超乎适度'，我不懂，但我知道你没让我填一些莫名其妙的东西。"他签完名，从柜台下面摸出一罐可乐递给梁志明，"天气热。"

梁志明接过可乐，走出唐楼的时候，天色已经暗了。他把那本借来的私隐条例小册子放进包里，想着明天还给林若兰的时候，顺便问她借下一章来看。

## 原文定义

《妥善处理客户个人资料：给保险业界的指引》规定，收集客户资料须符合「不超乎适度」原则——收集的资料范围须与收集目的直接相关，不得收集超出实际需要的个人资料；同时，收集方式须属公平和合法，以欺诈或失实陈述取得资料属违规行为。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 火险申请表要求提供家族病史 | 收集资料「超乎适度」——资料范围与承保目的无关 |
| 梁志明对照私隐条例指引核查表格 | 保险人须依从《个人资料（私隐）条例》及其行业指引 |
| 林若兰以割胆手术为例说明「相关性」 | 资料是否相关须以具体索偿或申请内容为判断标准 |
| 负责人最终同意修改申请表 | 保险机构对从业员和代理的行为负有转承责任 |
