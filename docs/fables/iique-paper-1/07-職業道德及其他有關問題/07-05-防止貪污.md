# 7.5 防止贪污

## 故事

上环德辅道西的「联安保险行」门面不大，却在街坊里做了十几年。梁志明十六岁进去当学徒，如今二十六岁，能独当一面。保单、批文、核保、做文件，他都熟悉。街坊叫他「阿明」，客人信他。

那年夏天，来了个新客人——湾仔一间小型出入口行的周老板，想替店里十几个伙计各买一份人身保障。周老板精明，见过世面，倾了两次都说「再想想」。第三次来的时候，他掏出一个牛皮纸信封，压在保单草案下面，轻轻推过去。

「阿明，一点心意。」

梁志明手指碰到信封的时候就知道里面有东西。他打开一角——是钞票，不多，但够他两个月薪水。

周老板笑了笑：「规矩我懂，这笔钱不算在保费里。你替我『搞掂』那几份身体不好的伙计的核保，其他我不多问。」

梁志明的手悬在半空。

他认识那几个伙计：仓库的阿福有风湿，收银的阿莲血压高，按正常核保，这两个人不是加保费就是被除外。周老板的意思是，让他把这两个人的健康状况写成「良好」。

他想起进步夜校的法律课，Mr. Whitfield 曾在课上说：「《防止贿赂条例》管的不是钱多钱少，是你有没有滥用你代表的那个位置。」

那天夜里，梁志明把信封原封不动还给了周老板。

「周老板，您这笔钱我不能收。核保的事，我只能按实际情况报。」

周老板脸色一沉：「阿明，大家街坊——」

「正因为大家街坊，我才不能害您。」梁志明把保单递回去，「要是查出来，您那几位伙计的受益人一分钱都攞唔到。您的店也要俾人『钉牌』。」

周老板走了。梁志明在柜台上坐了很久。

三个月后，ICAC 收到匿名举报，指向港岛南区一间保险行的伙计。ICAC 执行处主任赵先生带人上门查文件的时候，梁志明坐在柜台后面，看见那人穿浅灰色西装，表情平静，一页一页地翻那些年的核保记录。

「联中」保险行不在名单上。但梁志明看见了名单上的名字——湾仔某出入口行，周老板。

ICAC 的人离开后，林若兰从律师楼打电话来核对一份英文合约的翻译。她在电话那头多问了一句：「听说附近有人俾ICAC拉了？」

「嗯。」

「你呢？」

「我没事。」梁志明放下电话，看著空空的柜台，忽然觉得那张拒绝过的牛皮纸信封，是这几年做得最稳的一笔生意。

## 原文定义

贪污一般是指受托人滥用职权而谋取私利。由廉政公署执行的《防止贿赂条例》协助商界维持有利于效率和公平竞争的营商环境，以保障不同持份者的利益，并保护主事人免受代理人以权谋私所损害。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明收到周老板的牛皮纸信封 | 受托人（保险从业员）被提供私利 |
| 周老板要求瞒报员工健康状况 | 滥用职权为特定人谋取不正当利益 |
| 梁志明拒绝并指出法律风险 | 保险从业员受《防止贿赂条例》监管 |
| ICAC 赵先生查阅核保记录 | 条例由廉政公署执行 |
| 周老板最终被举报查处 | 以权谋私终被条例追究 |
