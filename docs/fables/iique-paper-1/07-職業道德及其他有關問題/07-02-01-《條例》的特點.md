# 7.2.1 《条例》的特点

## 故事

林若兰在中环一间洋行做中英打字员，平时也帮梁志明的保险行录入投保文件。那些文件里有客人的姓名、身份证号码、地址、病历记录，每一份都装在牛皮纸信封里，从不折叠。

那年林若兰的上司Mr. Whitfield收到一封信，是一间做直邮推广的公司写来的，说可以免费帮保险行做客户回访推广，只需要提供一份客人名单和联系方式。林若兰看了一眼，觉得唔驶钱、又有客源，几好。她把这封信夹在日常文件里，等梁志明来取件时顺便给他看。

梁志明看完，眉头皱了起来。他想起陈伯讲过一句话："客人的资料唔系你嘅嘢，你只有使用权，无拥有权。"

梁志明去找Whitfield，说这事先唔得。"客人嚟投保，唔系嚟登记推销。我哋收佢哋个人资料时签过声明，讲明咗系用于边个目的。要转去直邮公司用，就系改变用途，客人未同意过。"

Whitfield想了想，说："咁如果我哋加一栏，喺投保书上注明资料可能用于推广，客人签咗就当同意？"

"要客人知道先算。唔系加一行细字就算数，要清楚写明、客人睇得明。"

Whitfield后来叫停了那个直邮合作。他说做直邮未尝不可，但必须先向客人发出一份说明，解释资料会点用、畀边个睇、客人有权拒绝。这样做才叫"合法和公平地收集个人资料"。

林若兰把那些客人资料锁进文件柜里，钥匙放在梁志明的抽屉。那间保险行后来订了一份内部规矩：所有含有客人个人资料的文件必须放进锁柜，不得用窗口式透明信封寄出，若要影印身份证，必须当场影印当场取回原件。

梁志明跟新人讲呢件事，只有一句："客人信你，先至会交资料。你负唔起呢个托付，就唔好收。"

## 原文定义

收集个人资料时须向资料当事人提供资料声明，说明收集资料的目的、资料可能转移给什么类别的人、不提供资料的后果及查阅和改正资料的权利。除非征得资料当事人同意，个人资料只能按收集时订明的目的使用。须采取适当保安措施确保个人资料不受未经授权的查阅、处理、删除或使用所影响。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 直邮公司要客人的联系方式 | 未经同意不得转移个人资料 |
| 梁志明说"客人授权用途才能用" | 收集目的限制原则 |
| 投保书须清楚说明资料用途 | 须向资料当事人提供收集个人资料声明 |
| 客人资料锁进文件柜、不得用透明信封寄出 | 个人资料保安措施要求 |
| "客人信你，先至会交资料" | 资料使用者对资料当事人的责任基础 |
