# 7.3.3 保险中的不公平歧视

## 故事

何师奶的表妹阿芳住在深水埗，三年前离了婚，带着一个五岁的女儿租住在唐楼里。离婚的时候前夫走得干净，没给赡养费，阿芳在制衣厂做车位工人，每月收入刚好够生活和交租。

阿芳想买一份家居保险，保的是火灾和爆窃——她住的那栋唐楼楼龄三十年，电线旧了，隔壁还曾经发生过火警。

她先去问了周老板。周老板和上环几家保险行都有往来，认识的人多。阿芳把事情说了，周老板帮她联系了一家保险公司的经纪。

第二天那个经纪回话了，说阿芳的申请"批不了"。

阿芳问为什么，对方说："我们公司有内部规定，单独抚养未成年子女的女性，属于高风险群体，不接受这类申请。"

阿芳当场愣住。她问："什么叫高风险？我又不是去做极限运动，我只是想保间屋。"

对方在电话里沉默了一会儿，然后说："这是公司的规定，我只是照做，您要不问问别家。"

阿芳把这件事告诉了何师奶。何师奶听完整件事，火大了。

"什么高风险，你估佢系保险顾问定系估佢系算命先生？"她对阿芳说，"你放心，我帮你揾人。"

何师奶找到了Mr. Whitfield。Whitfield是英资律师楼的合规律师，何师奶的儿子在酒楼做工，有一次帮Whitfield拎过文件，两人因此认识。Whitfield听何师奶把事情讲完，拿出一张纸记了几行。

"何女士，"他说，"这个案例涉及《性别歧视条例》和《家庭岗位歧视条例》。保险公司将'单独抚养未成年子女的女性'列为内部拒保类别，前提是对投保人个人情况没有任何评估，仅凭其家庭身份附加标签，这构成不公平歧视。"

"那应该怎样？"

"正确的做法应该是评估个人的实际风险：她的住址是哪里、房屋结构如何、是否有完善的消防设施、她的收入是否稳定。"Whitfield说，"而不是说'这类人一律不保'。"

何师奶问："那如果真的有统计数据显示，单亲家庭的索赔率更高呢？"

Whitfield想了想，说："就算是这样，也不能直接推论到每一个单亲家庭身上。如果保险公司想做差别处理，必须：第一，拿出具体而且经过验证的统计数据；第二，对每一个申请做个别评估；第三，给投保人提供上诉或解释的机会。三个条件缺一不可。"

阿芳后来找到了另一家愿意承保的公司——核保的过程看的是她住所的实际防火条件和她的财务稳定性，而不是她是不是单亲妈妈。

她投保成功那天，何师奶在茶餐厅里多加了一道菜，算是庆祝。

## 原文定义

不公平歧视是指无合理依据，仅凭偏见、刻板印象或个人身份特征（如性别、家庭状况）而对他人给予不公平对待的行为。《残疾歧视条例》或《性别歧视条例》均不允许基于主观偏见而非实际风险评估的差别待遇。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 保险公司以「单独抚养未成年子女的女性」为由直接拒保 | 无实际风险评估，仅凭家庭身份标签进行差别对待 |
| Whitfield指出正确做法应评估「住所实际防火条件」 | 差别对待须基于个人实际风险，而非群体标签 |
| 第二个经纪公司看的是「财务稳定性」而非「家庭身份」 | 合法的核保考虑因素须与实际风险直接相关 |
| 「三个条件缺一不可」 | 差别待遇若要合法，须满足数据基础、个别评估和申诉机制三个要求 |
