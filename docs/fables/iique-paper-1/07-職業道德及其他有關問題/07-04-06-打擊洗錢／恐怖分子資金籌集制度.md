# 7.4.6b 打击洗钱/恐怖分子资金筹集制度

## 故事

永亨保险行在二楼摆了二十年的保险柜、档案柜和一台老式英文打字机。梁志明刚入行的时候，二楼只有两张桌子，陈伯坐一张，经理坐一张。

一九七九年的某一天，Mr. Whitfield从律师楼过来，在二楼转了一圈，然后在经理那张桌子旁边加了一把椅子。

"这是合规主任的位置。"他说。

经理说："我们行就这么大，一个人做保险行不行？"

"不是一个人，"Mr. Whitfield说，"是两个人。一个合规主任，一个洗钱举报主任。"

梁志明当时在旁边。他以为Mr. Whitfield在说笑话——永亨保险行一共就三个职员，一张桌子都不够分，还要两把椅子？

但Mr. Whitfield没有笑。

他打开公文包，拿出一份文件，摊在桌上。那份文件是英文的，梁志明只能看懂一半，但里面的框架他记住了：高级管理层的监督、合规主任、洗钱举报主任、合规及审核职能、职员甄选和培训。

"这五条，"Mr. Whitfield指着文件说，"是每一家保险机构必须建立的内控制度。不是建议，是要求。"

经理看完文件，沉默了很久。然后他说："编制不够。"

"先设岗，"Mr. Whitfield说，"人可以从现有人员里挑。合规主任要懂英文、懂保险业务、也要懂这套制度。举报主任可以由合规主任兼任，但举报那条线要单独通到外面，不能走经理那条线。"

林若兰后来帮永亨保险行翻译了这份文件的中文版本。她把"合规主任"翻译成"负责把关的人"，把"洗钱举报主任"翻译成"负责说话的人"。

梁志明问她："为什么这么翻？"

"因为把关的人要对内，负责不让人从我们这里洗钱；说话的人要对外，负责在出事的时候把话说出去。"她说，"这两个角色有时候是同一个人，但说的话走的路不一样。"

永亨保险行后来真的设了这两个岗位。合规主任由陈伯兼任，因为他熟悉账目、了解每一位客户；洗钱举报主任由梁志明兼，因为他年轻、懂英文、也因为他是唯一一个和ICAC赵先生见过面聊过天的人。

陈伯一开始不太高兴。他觉得这是多出来的活。

但有一次，一笔来历不明的大额保费到期续保，陈伯在自己的账本里发现同一笔钱三个月前已经以另一个名字出现过一次。他把这件事告诉了梁志明，梁志明写了内部报告，直接送到了财富情报组。

那笔保费后来没有续成，但陈伯在永亨保险行的柜子里多放了一个抽屉，专门放这类记录。他说："这叫合规档案柜。"

二楼的格局后来一直没变：陈伯的桌子在左边，梁志明的桌子在右边，中间那把Mr. Whitfield搬来的椅子，一直放在靠窗的位置。没有人坐，但每次开合规培训会，那把椅子会被人坐上。

## 原文定义

根据指引3，金融机构必须设立涵盖以下范畴的管控措施：高级管理层的监督；委任一名合规主任及一名洗钱举报主任；合规及审核职能；职员甄选和培训。这些措施统称为"打击洗钱/恐怖分子资金筹集制度"。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| Mr. Whitfield在二楼加了一把椅子，设合规主任岗位 | 金融机构须委任合规主任 |
| "举报那条线要单独通到外面，不能走经理那条线" | 洗钱举报主任须有独立汇报渠道 |
| 林若兰把合规主任译为"把关的人"，举报主任译为"说话的人" | 合规主任对内防范风险；举报主任对外报告可疑情况 |
| 梁志明见过ICAC赵先生，被兼作举报主任 | 举报主任须熟悉外部举报渠道 |
| 陈伯发现同一笔钱以两个名字出现，写进合规档案柜 | 合规及审核职能须持续运行并留存记录 |
| 梁志明写了内部报告直接送财富情报组 | 发现可疑情况须通过内部系统向主管机关报告 |
