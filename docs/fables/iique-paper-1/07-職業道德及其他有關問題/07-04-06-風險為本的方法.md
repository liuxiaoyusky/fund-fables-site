# 7.4.6d 风险为本的方法

## 故事

周老板第三次从干货铺走上保险行的时候，手里多了一袋虾片，是干货铺卖剩的货，炸得太硬没人买，他索性带来给梁志明尝尝。

"阿明，我那间干货铺的保单，"他把虾片放在台面上，"你到底要查多久？"

梁志明把虾片收起来，说快了快了。然后他从抽屉里拿出一份文件，是干货铺的仓库地址文件。

"周老板，您这间干货铺，执照上的地址跟保单上的地址，差了半个门牌号。"

周老板伸过头来看了一眼："那不是同一间铺吗？楼上楼下都是我的。"

"可是楼上登记的是干货铺，楼下登记的是茶具批发。执照上写的是干货铺，但实际经营的位置——"

"都是我打理啦！"

梁志明停下手里的笔，抬头看着周老板。周老板也停下来了，语气软了一点："阿明，我不是想为难你，我知道你们有规矩。但那间铺子我跟了二十年，从来没出过事，你就当我保了吧，旺季一来我就忙不过来了。"

梁志明把文件和账本合上，说了一句话："周老板，您先坐一会儿。"

周老板愣了一下，然后坐下了。

过了一会儿，Mr. Whitfield从律师楼那边过来，梁志明把文件递给他。Mr. Whitfield翻了翻，皱起眉头。

"楼上做干货，楼下做茶具批发，这是两个不同的商业风险类别。"Mr. Whitfield说，"如果我们按干货铺的风险来承保楼下，实际经营的是茶具批发，这中间的风险差异——"

他停下来，看向周老板："您知道两个生意的风险有什么不同吗？"

周老板想了想："茶具批发比干货铺多了一样东西——瓷器，破了就没了。"

"对了。"Mr. Whitfield说，"瓷器破了要赔的钱，可能比干货铺遇到虫蛀还要多。如果您要保两层楼的生意，我们得按两个不同的风险类别来处理。"

周老板叹了口气："那要多久？"

"两个风险类别都要核查的话，"Mr. Whitfield说，"快的三天，慢的话要看仓库那边的情况。"

周老板站起来，正要抱怨，梁志明又补了一句："周老板，这样做完之后，将来回来看这份保单，每一个数字都经得起查。如果您当时说'算了就当干货铺保了'，将来真的出事，保险公司说我们没有如实告知，那个麻烦可比这三天大得多。"

周老板没说话。他看了看Mr. Whitfield，又看了看梁志明，最后看了看那袋虾片。

"那三天，"他说，"你们三天之内给我回复。"

他拿起虾片袋子，发现里面还有几块没碎的，愣了愣，又放下了。

梁志明送周老板下楼的时候，在楼梯口遇到了林若兰。林若兰问："周老板肯等三天了？"

"我让他坐下来，"梁志明说，"他坐下来之后，反而肯听了。"

林若兰问为什么，梁志明说："也许他明白了——我们多问几句，不是因为我们不想做他的生意，而是因为他将来要赔钱的时候，这份保单上的每一个字都得站得住脚。"

## 原文定义

指引3要求金融机构采取风险为本的方法来确定尽职审查及持续监察所需程度。如果客户经评估属于较高洗钱或恐怖分子资金筹集风险，应采取更严格的措施；如果属于较低风险，则可执行简化措施。其优点在于按优先次序以最具效益的方式分配资源，令最大的风险得到最高度的关注。

## 对应点

| 故事元素 | 概念对应 |

| 梁志明核对执照地址与实际经营情况，发现不一致 | 尽职审查核实客户真实情况 |
| Mr. Whitfield指出两层楼属于不同风险类别 | 高风险客户需更严格核查 |
| 干货铺与茶具批发风险不同，需要分别评估 | 风险差异影响承保决定 |
| 梁志明坚持核查完成后才出保单，而不是走过场 | 风险为本的方法实际执行 |
