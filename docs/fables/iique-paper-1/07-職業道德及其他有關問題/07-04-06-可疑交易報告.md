# 7.4.6j 可疑交易报告

## 故事

上环永亨保险行的二楼档案室里，梁志明已经对着同一叠保费收据看了三天。

那是周记海味铺的账目。周老板做干货生意三十年了，每次保费到期都准时，从不拖账。但这个月的情况不对——三张收据加起来，总额比往常多了将近一倍，而周老板那张老脸在柜台上笑得像没事人一样，只说"帮我入数，帮我入数"。

梁志明没问。他把收据锁进抽屉，下班去街角茶餐厅买了一碗面。

何师奶端面过来的时候，顺口说了一句："最近周老板来我这里少了，伙计算了，说老板转了钱去澳门。"

澳门。梁志明心里咯噔一下。

他想起指引3里那句话：了解你的客户，以及他们正常情况下会做什么。如果一个从未汇出过境的老海味铺，忽然多了那么多资金流动，而老板只字不提——这算什么？

他没去问周老板。他知道不能问。一旦问了，就等于告诉对方，有人注意到你的账目了。指引3管这个叫"通风报讯"——你不能给客户任何机会，让他知道有人在查他。

第二天早上，他把那几张收据的副本锁进自己的抽屉，然后去找林若兰。

林若兰当时在中环的英国律师楼做文件翻译，刚好经过永亨保险行的楼下。他截住她，把情况说了一遍，但没提周老板的名字。

"如果我现在去问他，"梁志明说，"他就知道我留意到了。他可能会转移，会销毁什么，或者干脆不做了——然后我们永远不知道那笔钱去了哪里。"

林若兰看了他一眼，说："那就按程序走。"

她说得平静，就像在说今天的茶凉了该怎么办。

梁志明后来才知道，按程序走的意思是——先不打草惊蛇，直接写内部报告，交给合规主任，再由合规主任转给财富情报组。在那之前，周老板不会收到任何通知，而他账户里的每一分钱都还在原来的地方。

ICAC的赵先生后来跟梁志明说过一句话："举报不是背叛。通风报讯才是。"

赵先生说这话的时候，在茶餐厅里，正对着何师奶端来的一杯凉茶。何师奶不知道他们在说什么，只知道这个穿西装的男人说话很慢，每个字都要想好了才出口。

"不告诉客户你在查他，"赵先生继续说，"不是因为要瞒着他，是因为一旦他知道了，整条线索就没了。举报不是卖人，是给整座城市留一条干净的水源。"

梁志明那天回家的路上，经过周记海味铺。周老板正在门口晒鱼肚，秋天的阳光照在他花白的头发上。梁志明没有停步。

他想：如果那笔钱真的干净，那就干净地过。如果不干净——他做了该做的事。

后来周老板那笔账果然查出了问题。梁志明没有自己去问他，这一条，他至今觉得做对了。

## 原文定义

指引3规定：一旦确立对洗钱或恐怖分子资金筹集的怀疑，即使金融机构没有进行交易，也须作出披露；须在切实可行范围内尽快披露；必须设有内部管控及制度，防止董事、职员向客户通风报讯。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明发现周老板账目异常后没有直接询问客户 | 发现可疑情况不得向客户通风报讯 |
| 梁志明先将情况写入内部报告而非直接举报 | 可疑交易报告须通过内部报告系统进行 |
| 梁志明说"一旦他知道了，线索就没了" | 防止通风报讯是保护调查完整性的必要措施 |
| 何师奶提及周老板"转钱去澳门" | 异常资金流动模式是识别可疑交易的依据 |
| 赵先生说"举报不是背叛，通风报讯才是" | 可疑交易报告是金融机构及其职员的法定责任 |
