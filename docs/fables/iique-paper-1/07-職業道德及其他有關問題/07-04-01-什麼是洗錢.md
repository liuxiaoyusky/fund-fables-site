# 7.4.1 什么是洗钱

## 故事

那年秋天，维港的夜雾特别早。下午五点多，码头上的灯已经亮起来，吊臂在雾里缓缓移动，像一些看不见的手。

周老板的海味铺有一批货要出——六箱干鲍鱼，买家是一个姓郑的台湾商人。郑先生通过一家贸易公司的名义下的单，付款方式是电汇，金额不小，而且郑先生说希望分三笔汇入，每笔隔两周。

周老板做海味生意二十年，什么客人都见过。他直觉这批货的付款方式有点奇怪：正经买家通常谈好价钱一次性付款，不会分三笔慢慢汇。但郑先生给的价钱比市价高了百分之八，周老板想了想，觉得多出来的部分或许就是"手续费"。

他把这件事告诉了梁志明。

梁志明听完，眉头皱了一下。他那段时间刚参加过公司组织的反洗钱培训，印象最深的不是那些法律条文，而是一个比喻：培训师说，洗钱就像有人把赃款扔进洗衣机里转了一圈，拿出来就变成干净的钱。

周老板的这批货，付款方式有点像那台洗衣机。

梁志明对周老板说："周老板，这单我不确定能不能接。您能不能给我两三天时间，我去查一查那间贸易公司的背景。"

周老板答应了。

梁志明去找了ICAC的赵先生。赵先生是廉政公署执行处的主任，平时会给保险和金融业做义务讲座，梁志明在一次培训会上见过他。赵先生听完事情经过，没有立刻表态，而是问了几个问题：那间贸易公司注册在哪个地方？郑先生有没有留下名片或身份证明？三笔汇款有没有指定汇入同一个账户？

梁志明说这些他都不清楚，当时只是觉得不对劲才来找赵先生。

赵先生从抽屉里拿出一本小册子，翻到其中一页，念了里面的一句话："洗钱是指使犯罪收益看起来并非犯罪收益的行为。"

他合上小册子，对梁志明说："你要记住三个关键：第一，钱的来源本身是犯罪所得；第二，经过一系列交易把它的来源掩盖；第三，结果是这些钱能够正常使用而不被怀疑。你周老板的这单生意，付款方式有异常，但没有足够证据说明它涉及上游犯罪。不过，你们保险行作为金融机构，有责任对可疑交易保持警觉。"

梁志明问："那应该怎样处理？"

赵先生说："向公司的合规部门报告，由公司决定是否向联合财富情报组提交可疑交易报告。你们保险行的人不是调查员，但有义务做第一道把关。"

周老板后来听了梁志明的建议，没有直接拒绝郑先生，而是以"货物周转需要时间"为由把发货时间推迟了。郑先生等不及，取消了订单。

周老板少赚了一笔钱，但他对梁志明说："算啰，唔见钱好过见官。"

## 原文定义

"洗钱"一词指出于以下意图的行为：使(a)属于干犯香港法律所订可公诉罪行的收益的任何财产，看似并非该等收益；或(b)全部或部分直接或间接代表该等收益的任何财产，看似不如此代表该等收益。即：通过交易处理，使犯罪所得资金的来源和性质被掩盖。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 郑先生分三笔、隔周汇入款项 | 洗钱的典型手法：通过多次交易掩盖资金来源 |
| 梁志明的直觉反应 | 保险从业人员须对异常交易模式保持警觉 |
| 赵先生解释洗钱的三个关键 | 上游犯罪收益 → 交易掩盖 → 资金正常使用 |
| 梁志明向合规部门报告而非直接举报 | 金融机构的合规报告是第一道防线，从业人员须按规定程序处理 |
| 周老板推迟发货导致对方取消订单 | 合理的风险规避行为，不直接拒绝亦不主动配合 |
