# 7.3.2 保险中的「公平」歧视

## 故事

何师奶有个老邻居，街坊都叫她吴婆婆。吴婆婆七十三岁，身体还算硬朗，每日早晨在佐敦公园打太极，平时帮儿子看看孙女。吴婆婆的丈夫去世得早，她靠一份早年买的人寿保单把儿子养大，所以一直相信"买保障"这件事。

那年吴婆婆想再添一份住院医疗险，托何师奶介绍，找到了梁志明。

梁志明收了申请表，拿去公司核保部审核。过了三天，核保部的回复来了：保费比标准费率高出百分之四十，理由是"年龄超过七十岁，风险评估较高"。

吴婆婆听到这个数字，筷子停在半空。

"我日日打太极，感冒都少有，凭什么贵四成？"她说，"你们保险公司是觉得我老所以呃我啰？"

何师奶在旁边听着，没插嘴。她认识梁志明，知道他不是那种呃人的人。

梁志明没有立刻解释。他约了吴婆婆和她儿子一起到茶餐厅，拿出几张纸。

"吴婆婆，我不是呃您。"他把纸摊开，"这是保险公司参考的一套精算数据，就是根据大量人口统计出来的风险模型。比如在七十岁以上的投保人里，住院频率、康复周期、长期病患的比例，都比年轻人高出一截。这些不是针对您一个人，是整群人放在一起看的结果。"

吴婆婆的儿子问："那为什么不直接拒保？"

"因为这些数据只是风险评估的一部分，不是唯一标准。"梁志明指着其中一行，"保险公司还会看个人的健康状况，比如吴婆婆您没有长期病，日常体检合格，所以公司愿意承保，只是风险溢价高一些。如果是完全不看个人情况的'一刀切'拒保，那才叫歧视。"

何师奶这时开口了："那如果有个年轻人成日食烟饮酒，风险也很高，保险公司收他几钱？"

梁志明说："一样会按实际风险调整保费。统计数据显示吸烟者的特定疾病发生率更高，核保时会加风险附加费。所以这不是看年纪，是看风险。"

吴婆婆听完，把那张纸拿过来仔细看了一遍，最后说："你讲得几清楚，我就信你。"

她签了投保书，保费是贵了四成，但她多添了一份保障。

梁志明收好文件，走的时候何师奶跟到门口，说："你刚才说的那些数据，是不是就是条例里说的'可合理依据的来源'？"

梁志明愣了一下，说："何师奶，您几时变得咁熟悉条例了？"

何师奶笑了笑，没答他，转身回了厨房。

## 原文定义

三部反歧视条例（性别、残疾、家庭岗位）均包含一项共同条文：若对某人的差别待遇符合以下两项条件，则不构成违法：(a) 该待遇参照了可合理依据的来源所得的精算数据或其他数据；(b) 以该等数据及任何其他有关因素而言，该待遇属于合理。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 核保部以「年龄超过七十岁」为由加收保费 | 差别待遇需要有精算数据作为可合理依据的来源 |
| 梁志明解释这是「整群人放在一起看的结果」 | 统计意义上的风险分类，而非针对个人的主观评判 |
| 吴婆婆因个人健康良好仍获承保 | 差别待遇须结合「任何其他有关因素」综合判断，不能一刀切 |
| 加费而非拒保 | 差别待遇须在合理范围内，方式须适当 |
