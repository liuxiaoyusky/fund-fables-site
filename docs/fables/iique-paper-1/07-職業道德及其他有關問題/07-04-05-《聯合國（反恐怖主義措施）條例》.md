# 7.4.5d 联合国（反恐怖主义措施）条例

## 故事

1980年，香港码头仍是世界上最繁忙的港口之一。每天有几十艘远洋船在维多利亚港卸货，船东、货主、保险公司、码头工人，每个环节都靠文件和习惯运转。梁志明有时候去中环的船东公司递文件，见过那些堆积如山的提单和保单，数字都是英文，来来回回没几个人真正看懂。

那年冬天，一间船东公司通过保险行安排了一组航运保险，附加了一个特别条款：受益人名单上有一家位于中东某地的子公司，名称很长，用阿拉伯文写的。梁志明把文件递上去，Mr. Whitfield一看，眉头就皱了。

他对梁志明说：「这张单子先停下来。」

梁志明问：「是文件不齐全，还是别的问题？」

Mr. Whitfield说：「中东那个地方，现在局势复杂，你知道。现在的问题不是这间公司有没有问题，而是这份保单的受益人架构，必须能经得起联合国安理会的审查。我们有义务确保这个架构里没有被制裁的个人或实体。」

他翻出文件，指著一页说：「《联合国（反恐怖主义措施）条例》规定，向恐怖分子或与恐怖分子有联系者提供财产及金融服务，均属违法。这个『提供』是广义的，包括透过保险产品转移资金或权益。你看这张单子——受益人是子公司，子公司背后是谁，中间有没有其他安排，我们没有办法完全看清楚。」

Mr. Whitfield建议公司向财富情报组做了一个初步查询，同时要求船东公司补充最终受益人的身份说明。船东起初不高兴，觉得被当成了嫌疑犯，但Mr. Whitfield说：「条例的目的不是怀疑任何人，而是确保金融系统不被利用。如果你们的安排是干净的，提交说明对你们只有好处。」

ICAC的赵先生后来来保险行做合规培训，特别提到反恐这一块。他说：「恐怖分子资金筹集跟普通洗钱不同，它不只是把脏钱洗干净，而是把干净的钱拿去做坏事。你们做的每一笔交易，都可能是这条链条上的一环。」

他说：「条例要求的是『知悉或怀疑就要报』，但更重要的是，在怀疑还没有成形的时候，就要建立一套机制，去识别那些看起来正常但经不起查的安排。」

梁志明那天把培训内容全部记了下来。后来他再去船东公司的时候，带了一份额外的表格，是公司合规部新设计的「高风险地区受益人核查清单」。他把清单交给船东公司的职员，说：「不是为难你们，是我们也要对上面有一个交代。」

对方接过去，说：「明白了。」

## 原文定义

《联合国（反恐怖主义措施）条例》规定，向恐怖分子或与恐怖分子有联系者提供或筹集财产及向他们提供任何财产或金融服务，均属违法。任何人如果知悉或怀疑任何财产为恐怖分子财产，而未能在合理范围内尽快作出披露，即属犯罪。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 中东子公司受益人的航运保单 | 跨境金融服务可能被恐怖分子资金筹集利用 |
| Mr. Whitfield要求识别最终受益人 | 反恐融资要求核查资金最终受益人身份 |
| 向财富情报组提交初步查询 | 以订明方式披露可疑情况以获法定免责 |
| 公司设计「高风险地区受益人核查清单」 | 金融机构需建立预防机制以减低恐怖分子资金筹集风险 |
