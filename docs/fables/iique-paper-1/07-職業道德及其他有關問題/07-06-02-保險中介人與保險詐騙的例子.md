# 7.6.2 保险中介人与保险诈骗的例子

## 故事

周老板的海味舖开在上环巷子里，招牌是手写的，门口常年放著一箩晒干的蚝豉。梁志明认识他三年，最初是替周老板的女儿买教育基金，后来慢慢替整间舖子大大小小的事都操心过。

那年冬天，周老板打电话来，说想加一份店舖财产险。「保安见财化水，前两年俾人爆过一次，损失唔少。」

梁志明上门做了实地查勘。店舖在旧式唐楼地下，没有自动洒水系统，电线有几处老化，仓储和营业区之间只隔了一块三合板。梁志明在记事簿上写写画画，心里已经有了数：这种情况，不是加保费，就是部分损失列入不保范围。

他把初步评估告诉周老板的时候，周老板沉默了一会儿。

「阿明，有件事我想问你。」他从柜台下拿出一个发黄的文件夹，「几年前我进过一批发菜，供应商拿了钱，供应商那边的人『走佬』了，结果那批货海关扣了。我损失了八万几。」

他把文件夹推过来。「这件事，当时的经手人知道。现在我要重新投保，这件事……需不需要如实填？」

梁志明翻开文件夹，看了两页。

「周老板，这是您个人的商业损失记录，不影响店舖财产险的核保——」

「不是这个意思。」周老板把文件夹按回去，「我是说，如果这间舖之前『出事』过，保险公司会不会……」

「会。」梁志明很慢地说，「而且如果他们查到您没如实申报，已经出的事可以变成拒赔理由。」

周老板的眉头皱了一下，没说话。

梁志明把文件夹合上，往回推了一点：「周老板，有件事我必须跟您说清楚。」

「您问我要不要申报，我的答案是：要。不是因为法律逼我这样说，是因为我真觉得瞒报最后『伤』的是您自己。保险这个野，系将来出事有人兜底；如果底层那张纸是假的，兜底的人可以走咗。」

林若兰那天晚上恰好经过上环，顺路来海味舖找梁志明核一份文件。她站在门口听了几句，没进去，只是轻轻敲了敲窗框。

梁志明回头，她用唇语说了三个字：「最高标。」

他懂了。

「周老板，」他重新开口，「这份申请表里的所有资料，我作为您的代理，必须按最高诚信标准填写。不是『方便就填、不方便就唔填』，是全部。您说的那笔旧账——如实申报。我会帮您准备一份说明，解释当年的情况，这样保险公司那边也不至于觉得您在『作古』隐瞒。」

周老板看了他一会儿，忽然笑了：「阿明，你这个人，几固执。」

「系，但系您信得过我。」

周老板最终如实申报了那笔旧账。保险公司确实要求他加保费、增加自付额，但最终给出的承保范围覆盖了唐楼最主要的几项风险。周老板在文件上签字的时候说：「我做了几十年生意，这是第一次觉得麻烦事前的说明，値。」

林若兰在一边整理文件，没抬头，但嘴角有一点弧度。

几个月后，周老板介绍了另一个街坊过来，说：「我唔识保险，但我识人。阿明这个人，介绍俾你。」

## 原文定义

根据法律及道德规范，保险中介人应在申报事宜上遵守最高诚信责任，无论这会给拟购保险带来什么实际后果。保险中介人须不协助诈骗，并举报可疑事件或提供证据。诈骗属于极度严重的事件，深入调查各宗索偿或保险金要求是保险人的责任，只有保险人能指称被诈骗。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明主动告知周老板须如实申报 | 保险中介人须遵守最高诚信责任 |
| 周老板想瞒报商业损失历史 | 漏报或失实陈述资料即属潜在诈骗 |
| 林若兰提示「最高标」 | 中介人须协助保险人对抗诈骗行为 |
| 周老板最终如实申报并获得承保 | 如实告知是理赔的前提 |
| 梁志明主动提出协助准备说明文件 | 中介人有责任协助却不掩饰 |
