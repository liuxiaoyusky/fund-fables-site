# 7.2 保护个人资料

## 故事

陈伯在保险行做了二十七年账房，从手写账本用到打字机，再从打字机用到早期的电子计算器。他有一本用了十五年的蓝色账簿，封面磨得发白，里面用工整的钢笔字记着每位客户的保费到期日、核保批注和赔款记录。

那天下午，梁志明从外面跑完单回来，看见陈伯坐在柜枱后面，正把一本厚厚的文件夹塞进抽屉里。

"陈伯，那是什么？"

"电脑部刚送来的。"陈伯说，"说是要把我这几年的客户记录全部转去电脑里面。"

梁志明探头看了一眼文件夹，是一份表格，标题写着"客户资料电脑化转移授权书"。

"要签字吗？"他问。

"还没决定。"陈伯把那本蓝色账簿翻到某一页，指着上面一串数字，"你看，二十年前有个客叫何先生，买了份人寿保单，第二年断供，我帮他申请了宽限期，后来他又续上了。这些东西电脑里有没有，我不知道。"

梁志明坐下来说："公司说转了电脑查找更方便，而且也符合私隐条例的要求。"

陈伯抬起头："私隐条例？"

梁志明把私隐条例的事简单讲了——电脑普及之后，个人资料的处理变得又快又广，一个人在银行开了户，在医院看了病，在保险行买了保单，这些信息如果全部串起来，等于把一个人的底细全部摊开在机器里。香港政府去年通过了《个人资料（私隐）条例》，就是为了防止这种事发生。

陈伯听完，把那本蓝色账簿合上。

"我明白，"他说，"就是不能让人家随便睇我哋嘢。"

"对。"梁志明说，"条例要求收集资料要有合法目的，不能多收，而且要告诉当事人资料用来做什么。"

陈伯把那份授权书推到一边，说："让电脑部的人先来跟我对一次，哪些要转，哪些不应该转，我自己先睇清楚。"

梁志明点点头。他知道陈伯不是在抗拒电脑，而是在守护那些他亲手记下的东西——每一个数字背后，都有一个具体的家庭，一次真实的投保决定，一段没有写在条款里但真实存在过的信任。

窗外，叮叮车从德辅道西缓缓驶过，车上的广告牌换成了新一季的信用卡宣传。陈伯把那本蓝色账簿放进抽屉锁好，然后把钥匙放进了贴身的口袋里。

## 原文定义

《个人资料（私隐）条例》（Personal Data (Privacy) Ordinance）是香港专门为保护个人资料制定的法例，针对电脑资讯科技带来的隐私风险。该条例的出台背景是"资讯科技的速度、效率及能力，使个人私隐严重受到影响"这一全球性关注议题。
