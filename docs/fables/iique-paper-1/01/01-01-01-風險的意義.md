# 1.1.1 风险(Risk)的意义

## 故事

上环海味街的夜总是最慢的。关了门的铺面外，街坊们喜欢在路边坐一坐，摇着葵扇，等暑气散干净。周老板坐在自家"周记海味"的门口台阶上，身旁是一箩还没入仓的花胶，闻起来有海水和胶原蛋白混在一起的味道。

他今年五十三岁，在这条街做了三十年生意。什么大风大浪没见过——台风浸过仓、火烛烧过隔壁、伙计偷过钱——他都没有亏过老本。他的道理很简单：凡是看得见的风险，都不算风险。

"看不见才危险。"他跟梁志明说这句话的时候，梁志明正好提着一兜凉茶走过。

梁志明二十出头，在隔壁巷子里一间本地保险公司做伙计。他个子不高，穿的白汗衫洗得有点泛黄，但特别干净。他跟周老板认识几年了——每年台风季前，他都会过来劝周老板考虑一下"周全点的安排"。

周老板每次都笑着摆摆手："志明啊，我做这么多年生意，靠的是眼睛和经验。你说的那个看不见的风险，我看不见，就当它不存在。"

梁志明没有反驳。他在铺子门口站了一会儿，视线落在那箩花胶上。他知道，花胶的价格随着产地和年份浮动巨大，周老板这批货如果赶在冬季婚礼旺季前出手，毛利起码能有一万多块。但他也知道，这批货只用了普通帆布箩盖着，没有任何额外的防潮措施。

"周老板，我给你讲个真事。"梁志明最终坐了下来，在台阶上接过周老板递来的茶。

去年圣诞前，码头那边有个姓梁的燕窝货主，从印尼运了一大货来港。他也相信"看得见才危险"的道理——燕窝在船舱里封得严严实实，他连续三年没出过事。那年冬天，他却碰上了麻烦：船在海上遇上风浪，有几箱燕窝受了潮，变了味。问题在于——他拿着提单去找船公司的时候，发现船东早已把货转了三手，谁也说不清那批受潮的燕窝到底是哪一批、谁该负责。相关方面说要有文件证明损失发生的时间和地点，货主拿不出来，最后只能认赔。

周老板没有说话，但他那双总是在算账的手停顿了一下。

梁志明说："那个货主，他看不见风险，不是不存在，是他没去想过。"

夜风从海味街的巷口吹过来，带着一点点维多利亚港的咸味。葵扇搁在周老板手边，没有再摇。

那年台风登陆得比往年早。信号球在半夜竖起了三号风球，周老板被雨声吵醒，窗外的街道已经积了半尺水。他翻身下床，抓起手电筒就往铺子跑——结果发现雨水顺着箩盖的缝隙滴进了花胶箩里，有三成受了潮，颜色从金黄转成了暗褐。

那一造冬季婚礼的订单，统统泡汤了。

周老板在铺子里站了很久。天光之后，他叫了一辆货车，把受潮的花胶拉到工场低价处理。损失不算最大，但足够他睡不好几个晚上——不只是钱的问题，而是那种"明明可以避免"的后悔感。

他主动找到了梁志明。

"志明，你帮我想想，有什么法子能让我睡个安稳觉。"他说这句话的时候，眼睛望着柜台上那本记满了来货日期和客户名字的旧账簿，声音比平时轻了很多。

梁志明没有急着拿出合约。他问周老板："你知道不知道，你这次睡不着，是输在了哪里？"

周老板想了一会儿："损失。"

"不是。"梁志明说，"是'不知道'。你不知道那天夜里会有风，你不知道帆布会漏，你不知道损失来了会有多大——这些'不知道'加在一起，才是你真正输的东西。"

周老板沉默了很久。窗外的街市开始嘈杂起来，第一架叮叮车从上环街市拐进了德辅道，发出那种旧旧的、拖长了的铃声。

"让我考虑一下。"他最终说。

梁志明站起来要走，又回头说了一句："周老板，看不见的，往往不是不存在，是我们还来不及去想。"

周老板点了点头，把那杯凉透了的茶一口喝完。

## 原文定义

风险（Risk）是指「与潜在损失有关的不确定性」——即一种无法确定是否会发生某类损失、或无法确定损失程度的处境。正是这种不确定性，连同潜藏在风险中令人忧虑的成分，促成了人们对保障的需求。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板看不见帆布会漏、雨会浸货 | 风险的不确定性：损失是否发生、程度如何，事先无法肯定 |
| 台风半夜来临、雨量超出预期 | 潜在损失客观存在，只是尚未显现 |
| 周老板事后"睡不着觉" | 情绪上的潜在损失：焦虑、遗憾等主观感受 |
| 受潮花胶低价处理，金钱损失可量化 | 财务上的潜在损失：金钱可以直接衡量 |
| 梁志明指出"不知道"才是真正的问题 | 风险的核心不是损失本身，而是损失发生的不确定性 |
