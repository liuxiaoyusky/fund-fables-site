# 1.2 保险的功能及好处

## 故事

油麻地茶餐厅的早晨，总比别处早半个钟头。

天还没亮透，何师奶就已经在灶台前站定了。酱油罐的铁盖子被她摸得发亮，茶杯碟子磕了角也舍不得换，墙上那张丈夫的旧照片用透明胶纸贴着，边角已经翘起来。街坊都说她抠，其实她只是怕。怕什么，她也说不清，只觉得口袋里的钱一旦拿出去了，就少了一层垫底。

隔壁海味铺的周老板也是如此。仓库里堆着发财叔寄卖的干鲍鱼，来来回回多少趟船，他觉到需要找人来看着。"出事就出事啰，命里有时终须有。"他最爱把这句话挂在嘴边。

那年夏天，梁志明上门来过两趟。他是街尾那间本地保险行的年轻伙计，穿一件洗得发白的蓝布短袖，文件袋却整整齐齐。何师奶听他说完，摇了摇头："三十文一个月，一年就三百六，我每日才挣多少？"她算了笔账，觉得不如存起来。

周老板更是连门都没开："我这行船走水靠祖宗，哪个要我出钱？"

那年冬天，腊月十七，深夜十一点。隔壁胶袋厂电表短路，火舌顺着风势卷了过来。何师奶的茶餐厅烧了半边，楼上的窗框都变了形。消防车从窝打老道一路呜呜开过来时，街坊们都站在远处看着，谁也不敢靠近。

出事那天，梁志明天没亮就来了。他手里拿着一个牛皮纸信封，薄薄的，递到何师奶手上的时候，她愣了半晌。

"多少？"

"不多。"梁志明声音很轻，"大家的保费凑出来的，志明一份心意。"

何师奶接过来，打开看了一眼。数目不大，但足够她在街市角租一档临时摊位，买几张新的胶凳，继续做午市。

"为什么？"她抬起头。

梁志明蹲下来，在瓦砾堆旁边拣了一块还算干净的石头坐下，说："你说，三十文一年，划不划算？"

她没答。

"你自己那份当然不够重建。但几十户人，每户出三十文，出事的时候，就有人有钱周转。"他停了停，"我们平日出少少，万一有事，不用自己一个硬啃。"

何师奶想起丈夫出事那年，她一个人在医院走廊坐了一夜，第二天醒来，积蓄已经用得七七八八。她那个时候就怕，怕再有第二次，怕身边的人也要经历那种"一分钱逼死英雄汉"的滋味。

周老板站在远处，没有走过来。他的海味铺没烧到，但第二天传来消息，仓库被雨水浸了——那年台风把屋顶吹开了口，干鲍鱼发了一地霉。他站在烂掉的木箱旁边发了好久的呆。

何师奶恢复营业那天，周老板特意过来，买了一碗双拼饭，坐了很久。临走时他问："你那份钱，够不够？"

"不够。"何师奶说，"但够我重新站起来。"

周老板沉默了一会："明年，我都要问下那份三十文的事。"

何师奶点了点头，没有笑，也没有劝。她知道，有些事说再多也没用，要自己碰过才知道疼。

那年年尾，何师奶和周老板都交了保费。梁志明把名字写在陈伯的旧账本上时，陈伯在旁边看着，轻轻说了句："佢终于明喇——一个人出三十文不够重建，但几十户人出三十文，就有人周转得过。"

梁志明收好账本，抬头看了一眼墙上那张微微发黄的旧照片。何师奶没说出口的那些话，他好像隐约都听见了。

## 原文定义

保险本质上是一种风险转移机制：通过支付保险费，把个人无法承受的重大财务损失转移给保险人；当灾害发生时，保险为被保险人提供财务补偿。保险的基本好处在于，在各种不同的投保事件中，出事时有人帮忙分担，而不是独自承受全部损失。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 何师奶认为"三十文一年太贵"，选择不投保 | 对风险转移机制的认知不足，未理解"小钱换大保障" |
| 火灾后何师奶的茶餐厅半毁，独自难以重建 | 未投保的个人承担全部潜在财务损失 |
| 梁志明送上"大家保费凑出来的"牛皮纸信封 | 保险的补偿功能：由多数人的小额保费形成资金池 |
| "一个人出三十文不够重建，但几十户人出，就有人周转得过" | 风险转移的核心逻辑：众人分担损失，减轻个体负担 |
| 周老板仓库被浸后，主动问起"那份三十文的事" | 亲身经历灾害后，主动认识保险的基本功能和好处 |
