# 2.2.3 代理人的权限

## 故事

那年冬天，黄师傅接了一单新活——观塘一间制衣厂升级设备，厂主陈先生想把整栋楼的财产保了。黄师傅自己跟保险公司不熟，就找梁志明帮忙。

梁志明说："可以，但我要先去现场核验，然后回公司出方案。"

"那你现在就系我的代理人？"黄师傅问。

"不完全是。"梁志明说，"你委托我去核验，这是授权。但我只能按这个授权行事，不能做超出授权的事。比如你叫我帮你谈判保费，我拿了授权，但我没有权力擅自答应一个你自己都不知道的价钱。"

林若兰在旁边解释得更清楚："代理人的权限分几种。第一种叫实际权限——委托人对代理人明示或隐含地表示同意，'你可以代我做呢样嘢'，这就是实际权限。"

"第二种呢？"黄师傅问。

"第二种叫表面权限。"林若兰说，"这种权限不来自委托人对代理人的直接授权，而是来自委托人对第三者表示——'某某是我的代理人'。第三者基于这个表示，相信代理人有权行事，即使实际上代理人没有得到充分授权，委托人也可能要受约束。"

Mr. Whitfield 那天也在场，补充了第三种："还有一种叫必要权限。比如你突然住院，无法联系外界，你邻居主动帮你续缴了家里的保险——这种情况，邻居就算没有得到你的明确授权，也可以构成代理人，因为他是在必要情况下为你采取了必要行动。"

黄师傅听得有点晕，说："等等——我请阿明帮我搞保险，你哋话他有三种权限，那我作为委托人，会不会被他'越权'？"

"会。"梁志明说，"比如我只得到你的授权去核验财产，但我私自替你答应了一个超出预算的保费——这就超出了我的实际权限。但如果你事先对陈先生说过'阿明代我行事，无问题'，那么陈先生基于你的话相信我，即使我越权，你可能也要受约束，因为我有'表面权限'。"

"那如果有人冒充我的代理人呢？"黄师傅问。

"那就是不容反悔的问题了。"林若兰说，"如果你自己对外表示或允许他人表示某人是你的代理人，你之后就不能对基于信任而与此人交易的第三者否认其权限。但这种情况下，你和这个人之间不一定存在真正的代理关系——除非你的行为过后被追认。"

黄师傅想了很久，最后说："我明白了。找代理人，第一要问清楚距有几大权限，第二要防止距越权，第三要小心自己讲过的话被别人信了。"

梁志明说："还有第四样——有时候没授权也可以产生权限，那就系必要权限。但这种权限只在极端情况下才成立，不要随便用。"

那天晚上离开律师楼时，天已经黑了。黄师傅在街上碰到周太，周太问他今天谈了什么。黄师傅想了想，说："三重锁——实际权限、表面权限、必要权限。开错一把，都会出问题。"

周太没听懂，但她没追问，只是递给他一把钥匙，说："屋企锁系第四种——永远唔会错的那一把。"

## 原文定义

代理人权限分为四类：(a) 实际权限（Actual authority）：由委托人对代理人明示或隐含表示同意而产生；(b) 表面权限（Apparent authority）：透过委托人对第三者表明同意而产生，第三者基于信任而与代理人交易，即使代理人实际无充分授权，委托人也可能受约束；(c) 必要权限（Authority of necessity）：在紧迫情况下，代理人为保护委托人的财产或权益而必要地采取行动，可不经明示授权而产生约束力；(d) 不容反悔的代理权（Agency by estoppel）：委托人若以言词或行为表示某人为其代理人，则不能对信赖此表示的第三者否认该代理人的权限。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 黄师傅委托梁志明代为核验 | 实际权限：委托人对代理人的明示授权 |
| 梁志明私自答应超出预算的保费 | 超出实际权限 = 越权行为，委托人不一定受约束 |
| 黄师傅曾对陈先生说"阿明代我行事" | 表面权限：委托人对第三者的表示使代理人获得超出实际授权的权力 |
| 邻居帮住院的黄师傅续缴保险 | 必要权限：紧迫情况下不经明示授权也可产生代理权限 |
| 黄师傅的表态被第三者信以为真 | 不容反悔：委托人的表示产生不可撤销的信赖 |
