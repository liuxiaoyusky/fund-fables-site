# 2.2.4 代理人对委托人的责任（Duties Owed by Agent to Principal）

## 故事

湾仔的夏天又闷又热，黄师傅站在一栋正在装修的唐楼天台上，心里盘算着下午要交代的事。

上环的英资律师楼 Mr. Whitfield 的办公室需要修补一批老文件柜，黄师傅接了这单活。事情本身不复杂，只是要将十二个文件柜从三楼搬到储物室，再换上新的门锁。黄师傅自己腰痛，去医师那边看中医的日子又偏偏撞了下午两点。

他想了想，对着正在递砖头的伙计阿标交代：「阿标，下午你去上环 Mr. Whitfield 律师楼，把文件柜搬完再换锁，款项回来再找我报。」阿标点点头，把口讯记在一张发黄的纸条背面。

下午三点，Mr. Whitfield 的办公室里，林若兰正在整理刚收到的英文信件，忽然听见走廊传来一声巨响。她走出来，只见阿标把最后一个文件柜放下的时候，旁边一个红木书架被撞斜了，书架上叠着的一叠旧合约散落一地。

Mr. Whitfield 闻声出来，看着满地散落的文件，脸色沉了下来。他对阿标说：「师傅交代你搬柜子，我有没有叫你撞书架？」阿标支支吾吾：「师傅说搬完柜子再说，我想着顺手帮你把书架也——」

Mr. Whitfield 摇摇头，在记事本上写了几行字，叫林若兰打一封正式通知寄给黄师傅，说损坏的红木书架要估价索偿。

黄师傅晚上回家的时候，周太正在数家用，一看见丈夫就黑了脸：「下午律师楼打电话来了，说书架撞花了大半，问我们赔不赔。」黄师傅愣了一下：「我明明只叫他搬柜子。」周太把手上的家用纸条拍在桌上：「你叫人去，你就要担。口头话，撞花了的赔的钱边个出？」

那一天晚上，黄师傅坐着想了很久。口头交代的事，出了岔子，责任还是在自己身上。他叫阿标去，阿标就是他；阿标答应了没搬好，烂摊子还得他收拾。

第二天，黄师傅亲自去了律师楼，把书架的事承了下来。他对 Mr. Whitfield 说：「Whitfield 先生，以后这类活我自己做，或者写清楚给谁做，不能再靠口信了。」Mr. Whitfield 点点头：「师傅，香港这行头，口头承诺也要当合约用。你叫人去，就要保证他做得到；他不靠谱，你就要承担。」

林若兰在一旁轻轻地把这句话记了下来，夹进一本蓝色文件夹里。

## 原文定义

代理人必须遵行委托人的合法指示，并以应有的谨慎及技巧履行职责。代理人不得将职责擅自转授予他人，除非获得明确授权。委托人既受代理人行为约束，也就有权就代理人的疏忽导致的损失向代理人索偿。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 黄师傅口头交代阿标，没写明范围 | 代理人必须遵从委托人指示 |
| 阿标自行决定顺手扶书架，导致损坏 | 代理人的「分代理」问题：无授权不得转授职责 |
| 黄师傅须为阿标的过失向律师楼负责 | 转承责任：委托人受代理人行为约束 |
| 黄师傅事后亲自收拾烂摊子 | 代理人须以应有谨慎履行职责，否则须赔偿 |
