# 2.1.3 合约元素和要点

## 故事

梁志明有一本蓝色封面的笔记本，每次去德辅道中 Mr. Whitfield 的洋行上培训课，就记满一页。他把笔记带回保险行，晚上关门后对着台灯反复看，在空白处画满了图。

第一页画的是一座六根柱子的房子。林若兰有次经过他工位，看见了笑了一下，说："阿明，你屋企系唔系姓黄？"

梁志明莫名其妙。林若兰摆摆手走开了。旁边工位的陈伯慢慢开口："她话你画嘢好认真，似黄师傅装修前先画设计图。"梁志明这才反应过来，不好意思地合上本子。

但那六根柱子不是他乱画的。那是他整理合约束六项要素的方式，每根柱子代表一个要求。六根柱子齐全，合约才能立起来；缺任何一根，要么倒塌，要么变成危楼。

第一根柱子，叫"要约"。梁志明在笔记本上写：有人开出条件，另一人接受，合约才能谈得上。比如投保人填投保单，是要约；保险公司发保单，是承约。

第二根柱子，叫"承约"。他特别在下面画了条横线：如果受约人想改条件，比如说加保费，这条改动本身就不是接受，而是反要约——原来的要约就作废了。

第三根柱子，叫"代价"。这是他最早弄混的概念。陈伯给他解释：你付保费，保险公司承诺赔你，这是双方的代价。你付出一样，我付出另一样，两边都有交换，合约才成立。纯粹送礼物性质的承诺，如果没有契据，香港法律上是可以反悔的。

第四根柱子，叫"订约行为能力"。未成年人、精神有问题的人，签的约可以推翻。公司也一样，不能超出营业执照范围行事。

第五根柱子，叫"合法性"。合约内容必须合法，走私货物的合同无效，杀人偿命的协议也不受保护。

第六根柱子，叫"订立法律关系的意向"。商业合约被法律推定有这项，但家庭或朋友之间的约定不算。

有一天黄师傅上门，说起湾仔有个街坊签了份"合作书"，结果对方说"我哋系朋友，唔关法律事"。梁志明翻出笔记本，说这类事最关键是区分：这是"街坊人情"还是"商业安排"。商业关系，法律默认双方有立约意向；家人之间的约定，法律不这么看。

黄师傅听到这里，拍了一下大腿："我就系朋友介绍的单，我以为大家讲心先，点知距走数走得快过神仙！"

周太刚好从厨房出来，听到了就说："叫你签纸你又唔签。"

黄师傅没再辩驳。他从口袋里掏出那张街坊介绍的单子，上面只有一行字和两个签名。他对着台灯看了很久，然后把单子小心地夹进梁志明那本蓝色笔记本里，说："阿明，帮我揾 Whitfield 写一份真正可以追的数字合约。"

梁志明翻开笔记本新的一页，在六根柱子旁边各画了一个勾。

## 原文定义

有效合约须具备六项元素：(a) 要约（Offer）；(b) 承约（Acceptance），内容须与要约完全一致，若有修改则构成反要约，使原要约失效；(c) 代价（Consideration），双方各为对方的承诺付出代价，若无代价则简单合约无效（契据除外）；(d) 订约行为能力；(e) 合法性；(f) 订立法律关系的意向，商业合约被推定具有此项，但社交或家庭安排不适用。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 六根柱子的比喻 | 合约须六项元素齐全，缺一则合约不成立或存在缺陷 |
| 保险公司发保单 = 承约 | 承约内容须与要约一致，修改条件构成反要约 |
| 黄师傅与街坊的合作书无代价支撑 | 无代价的简单合约无效（朋友间的"心意"不等于法律代价） |
| 街坊说"朋友间不关法律事" | 社交/家庭安排不推定具有订立法律关系的意向 |
| 周太要求签纸才能算数 | 有书面证据才能清晰证明合约的要素 |
