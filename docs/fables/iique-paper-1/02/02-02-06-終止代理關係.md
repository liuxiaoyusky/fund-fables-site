# 2.2.6 终止代理关系（Termination of Agency）

## 故事

那年冬天，黄师傅的身体开始走下坡。湾仔的工程做到一半，他腰痛得站不起来，去公立医院检查，医生说要立刻入院观察。

周太瞒着丈夫，把工程的事托给了伙计阿标，然后打电话给 Mr. Whitfield 的律师楼，说黄师傅短期内没法继续工程。电话是林若兰接的，她拿着记录去找 Mr. Whitfield，问这种情况合约怎么处理。

Mr. Whitfield 翻出之前黄师傅签的那份工程合约，指了指倒数第三段：「师傅入院了，如果是精神或身体上无法履行合约，代理关系可以终止。」林若兰问：「那律师楼之前的委托怎么算？师傅已经垫了材料费。」

Mr. Whitfield 说：「垫的费用要还，这是委托人的义务。但继续施工的义务，随着师傅的身体状况，已经没法履行了——不是他不想做，是身体不允许。这种情形，合约自动终止。」

周太后来亲自来了一趟律师楼，把黄师傅垫付的材料费清单交给林若兰。单子上有七张收据，每一张黄师傅都签了名。周太说：「师傅就这个脾气，几时都要有白纸黑字。他今次入医院，最不放心的不是自己身体，是这些单据有没有交代清楚。」

两个月后，黄师傅病情好转出院，亲自到律师楼续了一杯茶。他对 Mr. Whitfield 说：「Whitfield 先生，我之前那单工程停了，我老婆替我垫付的费用你们收到了吧？」Mr. Whitfield 点点头，把回执拿出来给他看。

黄师傅又说：「等我好番，我想重新开始，委托手续这次我想请梁志明帮我逐项列清楚，唔好再靠口头。」Mr. Whitfield 笑了一下：「师傅，这次你有优先权。」

梁志明后来把黄师傅的新合约从头到尾整理了一遍，每一项权限和义务都写得清清楚楚，交给林若兰打字。林若兰打好后，发现了一个小小的翻译问题，悄悄改正了过来。

## 原文定义

代理关系可通过多种途径终止：双方协商同意撤销；任何一方撤销协议；一方根本违约导致另一方终止合约并索偿；任何一方死亡或精神错乱；代理协议或其履行变得非法；以及协议期限届满。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 黄师傅因病无法继续履行工程合约 | 精神错乱或身体无法履行：代理关系自动终止 |
| 周太代替黄师傅通知律师楼工程暂停 | 委托人主动撤销或终止代理关系 |
| 黄师傅垫付的材料费由律师楼偿还 | 代理关系终止后，委托人仍须履行残留的财务义务 |
| 康复后黄师傅重新委托，这次要求白纸黑字 | 新的代理关系须重新订立，不能自动恢复 |
| 林若兰发现并悄悄改正了翻译问题 | 代理人须忠于职责，文件细节不容疏忽 |
