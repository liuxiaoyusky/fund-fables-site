# 2.2.2 代理关系如何产生

## 故事

十一月，香港刚过了台风季节，街上还有些被风刮断的树枝没人清。黄师傅的装修队接了一单大活——鸭脷洲一间新建的货仓要赶在年底前完成内部装修，货仓主人姓关，是个急性子，要求三周内完工。

关生的货仓需要投保工程险，但时间太紧，关生又不愿意等保险公司派人来核验。他找到黄师傅，说："你帮我搞掂，我信你。"

黄师傅说自己只做装修，不做保险。但关生说："你识保险行慨人，帮我签一份保单，费用我出，你帮我跑一趟。"

黄师傅硬着头皮答应了。他跑到梁志明的保险行，把情况一说。梁志明说："黄生，这里有个问题——你没有保险公司的授权，理论上你不能代表他们签任何东西。"

"但关生叫我签啊。"

"你签了，技术上无效。"梁志明说，"除非保险公司事后追认。"

"追认？"

林若兰那天也在，从文件夹里抬起头来，说："追认就是事后确认。意思是，你当时没拿到授权，但事后保险公司说'好，我认这笔单'，那你的行为就可以追溯到当初有效。"

黄师傅急了："那关生要我签了才开工，保险公司唔认怎么办？"

梁志明想了想，说："有两个做法。第一，让关生先和保险公司直接联系，让他们正式授权给你——这是通过协议产生代理关系。第二，你先不签，关生先口头说好，保险公司事后确认——这是追认。"

关生性子急，选了第二条。他直接打电话给保险公司的经纪，说明了情况，经纪说："你先开工，我这边等总公司在内部系统里确认。"

三周后，货仓如期完工。关生结算工程款那天，保险公司的批文也到了——追认了黄师傅当时签下的那份保单。

梁志明后来把这件事记在笔记本里，写道："追认有三个要点：第一，可以是书面、口头或行为；第二，追认后行为效力追溯到当初；第三，如果当初是越权行为，追认后越权部分一并生效。"

他在旁边画了个时间轴：起点写着"代理人行事（无授权）"，中间写着"委托人追认"，终点写着"追溯有效"。箭头从追认指向起点，打了个虚线圈。

黄师傅后来再也不敢随便代人签单了。他说："以后再有呢种情况，我叫距直接揾保险公司。"

周太听说了这件事，没有骂他，只是拿了一张白纸，在上面写了"委托书"三个字，放在客厅的抽屉里，以备不时之需。

## 原文定义

代理关系可以通过协议产生，该协议可以是明示的，也可以从各方的行为或处境中隐含而来。追认（Ratification）是指对某行为授予追溯权——即代理人行事时虽未取得授权，但委托人事后确认后便有效产生追溯性授权。追认可以通过书面、口头或行为三种方式进行。例如，保险代理人越权签单，保险公司事后接受投保并确认保障，则该合约的有效性可追溯至当初。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 关生叫黄师傅代签保单 | 代理人须有授权才能约束委托人 |
| 保险公司事后追认 | 追认使原本越权的行为获得追溯效力 |
| 关生打电话确认后再开工 | 通过协议（即使是口头）产生代理关系 |
| 追认后效力追溯到当初 | 追认的核心：事后确认，效力回溯 |
