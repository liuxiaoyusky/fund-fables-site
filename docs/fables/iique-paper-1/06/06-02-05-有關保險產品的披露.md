# 6.2.5.5.2 有关保险产品的披露

## 故事

梁志明有一个老客人姓梁太，先生走得早，她在上环经营一家小型干货铺。那年她想为自己和两个帮工各买一份保障，找到了梁志明。

梁志明拿出两份方案，一份来自他熟悉的英资公司，另一份来自一间他未曾合作过但在东南亚有稳健根基的保险公司。两份方案各有优劣。

"梁太，你听我讲。"他先把两份方案的相同之处列出来，"两份都包意外和医疗费用，这是基本。你问我边份好，我先讲清楚两点，再俾你自己决定。"

他一项一项比。

第一间英资公司的好处是历史悠久，在香港有大量理赔经验，文件流程稳定；但保费稍高，而且保障范围偏向企业客户，对小型铺头有些附加条款不适用。

第二间公司的好处是保费较低，保障范围对小型铺头更灵活；但公司在东南亚，要用英文文件沟通，理赔周期可能稍长。

梁太问："你叫我买边间？"

"我唔可以直接话你买边间，"梁志明说，"因为两份都是合法合规的保险公司，我讲完相同之处和不同之处，你要知这件事的决策权在你。你明白两份嘢的区别，边份适合你，由你自己决定。"

梁太想了很久，最后选了第二间。她说："我间铺细，悭啲钱，我宁愿自己辛苦啲跟进文件。"

一个月后，梁志明收到第二间公司的确认函，他把副本放进梁太的档案里。Mr. Whitfield 路过看了一眼，说："你有没有试过叫佢买第一间？"

"有，"梁志明说，"但我唔可以叫佢买我话好就好的那份。我只可以讲清楚两份嘢，边份边个位贵，边份边个位边个位灵活。决定权在客人。"

梁太后来铺头的伙计出了意外，顺利理赔。她没来找梁志明闹，只是在下次的茶餐厅账单里，多付了一碟腊味饭的钱。

## 原文定义

持牌保险代理人须向客户披露所推荐产品的保险公司名称、主要条款及细则、保费水平及支付期间、客户须付的费用及收费；比较产品时须充分解释相似性及差异，比较须准确且不误导。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明列出两份方案的相同之处 | 充分解释产品的相似性 |
| 英资公司历史悠久但保费较高；东南亚公司保费低但文件用英文 | 充分解释产品的差异 |
| 梁志明不直接说"买边份"，只讲清楚利弊 | 比较须准确不误导，决策权在客户 |
| 梁太自己选了第二间 | 客户须基于充分信息作出知情决定 |
