# 6.2.1 监管保险中介人的持牌制度的基本规定

## 故事

何师奶有个老朋友叫阿昌，以前在她茶餐厅隔壁开过一段时间的小电器铺。阿昌后来不做了，说去了一家保险公司当直销员，帮公司推销产品、上门见客、处理理赔。有时候阿昌路过茶餐厅，何师奶会叫他进来喝杯茶。

有一次，阿昌在店里遇到梁志明。两人聊了几句，阿昌听说梁志明是持牌保险经纪，有点意外："我们都是保险中介，你怎么还要专门持牌？我去保险公司报到，他们给我工牌，说可以出去见客了。"

梁志明愣了一下，问阿昌："你见客的时候，有没有向客人提供关于保险产品的意见？"

阿昌想了想："当然有，公司让我介绍产品给客人，有时候客人问起理赔怎么申请，我也会解释。"

梁志明说："这就是'受规管活动'。提供意见、安排合约、处理理赔——只要做了这些，就要先拿到牌照。"

阿昌不信，说公司都报到了，工牌也领了，怎么还要牌照？

梁志明回去问了Mr. Thompson和张小姐。张小姐说，旧制度下靠三家自律机构登记，中介人只要在机构有登记就能执业。新制度把这套废了，改成法定发牌制度：任何人只要进行受规管活动——洽谈或安排保险合约、提供受规管意见、处理索赔——就必须先取得保监局发出的牌照，没有例外。

张小姐补充了一句话："保险公司自己的员工也一样。你以为在保险公司工作就天然有资格？法律不这么看。如果你是保险公司的员工，以公司代表的身份向客人提供意见，你同样需要个人牌照。"

阿昌听说之后，第二天就去保监局网站查了。过了几个月，他告诉何师奶，他已经考完试、拿到牌照了。何师奶说："那就对了。规矩立了，就要照做。"

梁志明有一次和阿昌喝茶的时候，问他拿到牌照之后感觉怎么样。阿昌说："感觉被管得严了，但是客人反而更信任我。他们知道我是有牌的人，不是随便来敲门的。"

何师奶在一旁听到，点了点头。她说："街坊就是这样，看你有没有凭有据。有牌，就是凭据。"

## 原文定义

新的法定发牌制度规定，任何有意进行"受规管活动"（包括洽谈或安排保险合约、邀请或诱使他人订立合约、提供受规管意见、处理索赔等）的人，必须首先获得保监局发出的牌照；受规管活动还涵盖从香港以外地方积极向公众推广保险服务的行为；五类持牌人包括持牌个人保险代理、持牌业务代表（代理人）、持牌业务代表（经纪）、持牌保险代理机构及持牌保险经纪公司；豁免适用于大律师、律师、会计师、精算师、评估损失的理赔业务代表等特定职业。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 阿昌以为有公司工牌就能执业 | 旧自律规管制度已废除的转变 |
| 梁志明解释"提供意见、处理理赔都是受规管活动" | 四类受规管活动的定义范围 |
| 阿昌以公司代表身份向客人提供意见仍需牌照 | 保险公司雇员不因受雇而自动豁免 |
| 阿昌主动去考牌、拿牌 | 法定发牌制度的强制要求 |
| 客人看到牌照反而更信任阿昌 | 持牌制度对行业公信力的意义 |
| 豁免职业（律师、会计师等）不在故事中涉及 | 特定豁免群体的存在 |
