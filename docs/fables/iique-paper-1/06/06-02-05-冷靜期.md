# 6.2.5.3.6 冷静期

## 故事

签完申请表的那个傍晚，周老板在心里算了算帐，然后后悔了。

他走回梁志明的保险行，站在门口，抽了两支烟才进去。

「志明，唔好意思，我想取消。」

「吓？取消？」梁志明有点意外，「周老板，你系几时签的？」

「就今日下午。」

梁志明请他坐下，倒了一杯水。他没有急于劝说，只是说：「周老板，你记唔记得，签之前我同你讲过，你有一个叫做冷静期的权利？」

周老板皱眉：「冷静期？乜嘢嚟㗎？」

梁志明在心里快速回忆了一下说明的内容——那天他确实提过，但可能只是轻轻带过，没有详细解释。他决定这次把事情说清楚。

「冷静期，意思系你在签署申请表之后，仍然有一段时间可以改变主意，取消呢份保单。」他停了停，「系二十一日，或者你收到保单文件之后的二十一日，两个期限边个先到，以那个为准。」

周老板抬起头：「所以我系可以取消？」

「系，你可以。但我要提醒你一件事。」梁志明说，「呢个期限系用来俾你审阅保单文件、考虑清楚自己系咪真系需要呢份保障。如果你决定行使呢个权利，你要书面通知保险公司。」

「几时要通知？」

「越快越好。你今日提出，我明日就帮你递交。等你收到正式取消确认，咁就搞掂。」

周老板点了点头，最后还是没有取消。他把那杯水喝完，说：「我再谂清楚。」然后离开了保险行。

梁志明望着他的背影，知道自己今次做对了两件事：第一，告知了冷静期的存在；第二，没有催促他快速决定，也没有在他提出取消时表现出不悦。

几天后，周老板最终还是正式投保了。但梁志明在心里记住了这一天：如果他那天没有清楚地说明冷静期，周老板可能永远不知道自己有着这个权利。

而不知道自己有权利的人，往往也不会知道什么时候该行使它。

## 原文定义

如保单载有冷静期条文，持牌保险代理人应在客户签署申请表前告知其取消权利，并于合理切实可行的范围内尽快向客户送交保单，以便客户有足够时间审阅并考虑其购买决定。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明主动告知周老板冷静期权利 | 签署前的告知义务 |
| 周老板在提出取消时，梁志明没有劝说 | 尊重客户自主行使取消权 |
| 梁志明解释须书面通知保险公司 | 行使冷静期的方式要求 |
| 梁志明承诺尽快递交取消申请 | 代理人须尽快执行客户指示 |
