# 6.1 合宜性评估

## 故事

何师奶有个习惯：买任何东西之前，要问三家。何师奶的茶餐厅需要一份新的保障，梁志明按合宜性评估帮她梳理了情况之后，她问："你们公司有什么选择？还是只有这一款？"

梁志明老实说："不是只有我们公司。整个市场上有好几家保险公司，每家的强项不一样。有的赔付快，有的保障范围广，有的价钱便宜但限制多。"他说，他要做的不只是把自家产品卖出去，而是帮何师奶找到最适合她情况的方案。

张小姐在一旁听着，问他："那你帮她比较了几家？"

梁志明说，从充足范围的不同保险公司里寻找和选择，这是合宜性评估的一部分。他要看的不是哪家公司给的手续费高，而是哪家的产品真正适合何师奶：餐厅是租的，伙计有五个人，湾仔铺位的租金每年都在涨。

何师奶听完，想了想，说："你这样讲，我就明白了。你不是在卖东西给我，你是在帮我挑东西。"

过了几天，梁志明给何师奶整理了一份建议，里面列了不同保险公司的方案对比，每一栏都标明了保障范围和主要的限制条款。他告诉何师奶，其中一家的产品有一个限制条款和她餐厅的情况有关，他已经在备注里标出来了。

何师奶看了那行备注，说："你这个括号里的字，比正文还重要。你要是不写，我自己根本不会知道。"

梁志明说，这正是他应该做的。他提供给何师奶的意见，必须基于合理的依据，而不是因为哪家给他的回佣高。若非如此，他所提供的意见就不切合她的情况，甚至可能误导她。

那天下午，何师奶选了一个赔付条款清晰、限制条款她能接受的方案。梁志明说，行，明天我把完整的保单文件送过来，有任何不明白的地方，你随时问我。

何师奶说："我信你。但文件我还是要自己看一遍。"

梁志明笑了一下："应该的。"

## 原文定义

持牌保险代理人应从充足范围的不同保险公司处寻找及选择可供选择的保险产品，就客户情况考虑各种方案，在提供受规管意见时考虑到客户情况并基于合理依据提供意见；合宜性评估的程度应相称且合理，并顾及客户情况及保险产品类型等因素。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明告诉何师奶不止一家保险公司可选择 | 从充足范围的不同保险人处寻找选择 |
| 梁志明整理了不同公司的方案对比表格 | 合宜性评估中的比较与选择步骤 |
| 他在备注里标出了与何师奶情况相关的限制条款 | 提供意见须基于合理依据 |
| 何师奶说"文件我还是要自己看一遍" | 客户有权了解产品细节 |
| 方案选择的依据是适合何师奶，而非回佣高低 | 须基于合理依据提供意见的原则 |
