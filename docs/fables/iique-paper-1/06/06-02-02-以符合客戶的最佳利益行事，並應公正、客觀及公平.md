# 6.2.2.1 以符合客户的最佳利益行事，并应公正、客观及公平

## 故事

周老板的女儿要出嫁了，在酒楼定了三十桌喜酒，请了半个街坊。海味铺的老客户们都封了礼，消息传到梁志明耳朵里，他想打个电话说声恭喜，还没拨出去，周老板就推门进了保险行。

"阿明，我要给女儿买一份东西。"周老板坐下，语气很急，"你们有没有那种……一投保就能拿钱的？我想把保费当嫁妆。"

那天梁志明正好有点忙，前面还有两个客人在等。他看了看周老板，又看了看后面排队的人，本能想说"有，我们有这类产品"，然后顺手把单签了。但他的目光扫过周老板的脸——眼底发红，像是很久没睡好——他又把话咽了回去。

"周老板，您先坐下，我问您几个问题。"

张小姐那天也在公司，她听到梁志明问周老板：给女儿的这份钱，除了婚宴支出，家里还有没有其他负担？女儿嫁过去之后名下有没有财产或者要还的债？这几年生意怎么样？供款会不会影响铺头的日常周转？

周老板一个一个地答。说着说着，他叹了口气："其实我也不懂，我就是觉得应该买一份。街坊都买，我也不能让女儿嫁得比别人寒酸。"

梁志明听到这里，心里咯噔一下。他想起 Mr. Thompson 说过的一句话："阿明，如果一个客户说他想买什么，你就直接卖给他，你就没有尽到责任。客户的利益不是你以为的那个，是他自己真正需要、真正适合他的那个。"

梁志明后来花了几天时间，帮周老板仔仔细细地算了一笔账。如果买储蓄类产品，每个月要供款，供款期十年，万一铺头生意出了波动，能不能按时交保费？如果不买储蓄类，而是买一份消费型的保障，每年保费不多，保额足够覆盖意外，万一出事，女儿拿到的是一笔不用还债的钱，反而更实用。

他最终推荐的是后者。

周老板起初有些不高兴："怎么不是你推荐的那个？"梁志明把账算给他看，周老板算了算，点了头。他对梁志明说："好，听你的。你比我懂，我信你。"

## 原文定义

持牌保险经纪应把客户利益放在所有其他考虑之前，公平对待客户，并基于客户利益提供合适、公正及客观的意见。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明没有立刻答应周老板想买的产品 | 把客户利益放在所有其他考虑之前 |
| 梁志明询问家庭财务状况和还款能力 | 了解客户实际情况才能判断真正需要 |
| 梁志明推荐消费型保障而非储蓄型 | 基于客户具体情况给出适合的建议 |
| 周老板说"我信你" | 公正客观反而赢得更深的信任 |
