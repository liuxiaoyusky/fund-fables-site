# 6.2.3.5 保存记录

## 故事

张小姐是保险公司的会计兼合规助理，每天对着牌照清单和一堆文件打勾。那天她在仓库里翻旧文件，发现了一批早年客户的申请表和往来信函，有些纸已经发黄，字迹淡得像鬼影。其中一份文件的日期，是十年前的。

她把那份文件拿回办公桌，仔细辨认。是一份客户申请表，上面有签名，有核保的批注，但电脑系统里查不到这笔交易。更奇怪的是，纸上的核保批注写着"该客户有既往症，建议附加除外条款"，但这条记录在系统里完全没有。系统里只有一份简短的保单摘要，其他什么都没有。

张小姐把这件事报告了 Mr. Thompson。Mr. Thompson 放下手里的茶杯，把那份旧文件翻来覆去看了两遍，然后说："张小姐，这份文件，你打算怎么处理？"

"放进碎纸机吗？"她问。

"不。"Mr. Thompson 说，"你想想，为什么过了十年，我们还留着这些旧文件？"

张小姐想了想，说："因为要符合保存记录的规定？"

Mr. Thompson 没有直接回答。他从抽屉里拿出一本旧账簿，封面已经磨破了，里面密密麻麻记着几十年前的保费收取记录。他说："你看这本账，十几年前的事了，上面记的东西，电脑系统里早就没有了。但客户有事的时候，他需要我们拿得出记录来。"

张小姐明白他的意思了。这些旧文件不只是一叠纸，它们是这个行业信任的实体证明。客户把自己的情况告诉保险公司，把保费交给他们，把保障交托给他们，这些都是信任。这些信任不能只存在硬盘里，还要落在一张张看得见、摸得着的纸上。

她把那份旧文件重新归档，和 Mr. Thompson 商量之后，又建立了一套定期检查制度，确保纸质档案和电子档案的内容一致，不会再出现系统里有、纸质文件里没有，或者反过来纸质文件里有、系统里查不到的情况。

三年后，公司搬迁，在清理档案时发现有一批文件快要超过保存期限。张小姐提前做了数字化备份，避免了信息的丢失。

## 原文定义

持牌保险经纪公司应就其进行的受规管活动保存妥善的记录，以遵守所有适用于该经纪公司的法律、规则、规例、守则及指引在保存记录方面的规定。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 旧文件上有系统里没有的历史核保批注 | 纸质记录可能包含系统中缺失的重要信息 |
| Mr. Thompson 拿出旧账簿 | 保存记录是客户信任的实体证明 |
| 张小姐建立定期检查制度 | 合规保存记录需要制度和流程 |
| 提前数字化备份即将超期的文件 | 主动管理记录，避免信息丢失 |
