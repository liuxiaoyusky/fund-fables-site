# 6.2.7.3 避免让自身利益影响客户的决定

## 故事

湾仔茶餐厅的鱼蛋粉远近闻名。何师奶每天早上五点就起来熬鱼汤，汤底用的是她丈夫生前教她配的那几味——她丈夫以前在码头做工，后来出了意外走了，留下这间铺头和一个十二岁的女儿。

那年清明过后，梁志明路过湾仔，想起何师奶的丈夫以前也认识，便进去叫了一碗粉。何师奶端上鱼蛋粉，认出他来，笑着说："后生仔，好耐无见，你爸好吗？"

"几好，仲喺西环卖紧鱼。"梁志明说，"师奶你自己打理间铺，辛苦啦。"

何师奶苦笑了一下，说："唔话你知了。上个月有个客喺度食嘢，唔小心打烂咗一碟，碟碎片整亲只手，赔医药费赔咗我成千文。我就想，不如买份嘢，出事有人帮我埋单。"

梁志明记下她的需求，正要讲条款，何师奶忽然问："你自己有冇帮衬过我间铺？"

梁志明老实说："有嘅，我不时喺你度食早餐，鱼蛋粉同我阿爸卖嘅唔同，我细路仔阵时成日念住。"

何师奶笑了，说："咁就好。你帮我睇铺头，我间铺又帮衬你食早餐，大家都识对方，系唔系应该更用心帮我？"

梁志明想了一想，把钢笔放下，认真地说：

"师奶，我帮你系因为我识你，又因为呢份系正当嘅生意。我今日卖保险俾你，明日我都系卖保险。你铺头出事要赔钱，我唔会因为自己食过你一碗粉就收贵你，或者漏报你嘅情况。你信我，我一定帮你；但你信我之前，我想先讲清楚——我系保险行嘅人，我只做保险，你铺头嘅铺位险系我卖俾你嘅正经嘢，我唔会因为私交就随便报或者乱报。"

何师奶听完后，沉默了一会儿，然后说："好，你咁样讲我就收得安心。你讲清楚咗，我就信你。"

梁志明把条款逐条跟她过了一遍。何师奶签完名，低头喝汤，忽然说："你老公以前都系咁样做嘢——先讲清楚规矩，再做嘢。呢样嘢就系我信佢嘅原因。"她没再说下去，低头把汤喝完。

那天晚上，梁志明把这件事写进了合规备注。张小姐整理档案时看见这条记录，说："客户已确认了解代理关系性质，无异议。"她把文件夹进档案袋，那天晚上也去湾仔那间茶餐厅吃了一碗鱼蛋粉。

## 原文定义

持牌保险代理人拥有其他业务或职业时，应避免该等其他业务或职业的利益与客户的利益有任何冲突。若无法避免，应在可行情况下尽快向客户披露该等冲突，并对客户公平行事，将客户的利益置于代理人在其他业务或职业的利益之上。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明在茶餐厅吃早餐，同时做保险 | 代理人拥有其他业务（餐饮） |
| 何师奶问"你帮衬过我，我铺头出事你会唔会更用心" | 私人关系与商业委托之间的利益冲突 |
| 梁志明主动讲清楚"我只做保险，正经嘢正经报" | 应避免利益冲突的原则 |
| 梁志明把这件事写进合规记录，张小姐审阅 | 无法避免时须披露并留存记录 |
| 何师奶说"先讲清楚规矩，再做嘢" | 披露是信任的前提，不是障碍 |
