# 6.2.7.1 提供有关报酬的适当披露以避免潜在利益冲突

## 故事

上环海味舖老板周永昌这几天心里不踏实。

他的远房亲戚介绍了一间保险公司给他，说经纪阿强办事稳当。周老板约了阿强在公司附近茶餐厅见面，听了半个钟头，越听越糊涂。他不懂什么「终身寿险」和「住院现金」，也不懂为什么同一间公司推荐了三款不同的产品。周老板不傻，他心里嘀咕：这经纪口才好，样样说到心坎上，到底是产品真的好，还是介绍这款他有好处？

当晚他在自家舖头跟伙计聊起这事，伙计说：「老板，你知唔知佢卖呢份单，佣金有几多？」周老板说不知。伙计摇头：「你问都唔问，点知佢系帮你定帮自己？」

那句话刺进周老板心里。他做海味生意几十年，最讲究「来路分明」——燕窝是哪里来的，他要知道；鱼翅是哪个批发市场的，他也要问清楚。买保障这件事，比燕窝还复杂，佣金的事却一点都不透明，他心里实在过不去。

第二天，他打电话去保险公司，投诉这件事。接电话的是合规部主管Mr. Thompson，Thompson跟他约了时间，让他第三天下午来公司谈。

周老板按时到了。推门进去的时候，他见到Thompson的办公桌对面坐着一个年轻人，穿着白衬衫，打着领带，一看就是前线做业务的。那是梁志明，Thompson叫他也一起听。

Thompson说：「周老板，你投诉的事，我们查过了。你说得对——我们的经纪在安排那份保单之前，没有跟你说清楚佣金的事。这不符合我们的内部守则，也不符合监管机构的规定。」

周老板一怔。他没想到投诉效率那么高，更没想到对方直接认了。

Thompson继续说：「按照规定，经纪公司在安排保单之前，必须向客户充分说明报酬的内容。这不只是佣金，还包括我们公司的激励计划、各项费用。你作为客户，有权知道替你办事的人收了多少。」

梁志明接过话：「周老板，其实我今天也在——就是我负责的那个客户，也问过我同样的问题。他问我，你介绍这份计划，是因为计划适合我，还是因为你赚得多？」

周老板看着这个年轻人，心里有些触动。这年轻人敢把话说出来，说明他也有过类似的为难。

梁志明接着说：「我以前也觉得佣金的事不好提，怕客人多想。但后来我明白了一件事——客人不知道佣金，不是不想知道，是没有人告诉他。你替他想到这一步，主动说清楚，他反而信你。」

Thompson点点头，对周老板说：「你今天来投诉，是对的。以后你再签任何保单，如果经纪没有给你书面说明报酬细节，你可以直接找我们合规部。」

周老板站起来，说：「我不是找你麻烦。我只是想知道，你们收了钱，办事有没有对得住人。」

Thompson把一份文件递给他，说：「这是公司所有经纪费率表的目录，你拿回去看。以后凡是我们公司的经纪，介绍任何产品之前，都要把这个给你看。这是我们的规矩，也是监管的规定。」

周老板接过文件，点了点头。

梁志明送他到电梯口，周老板忽然说：「后生，你个客问你那个问题，你点答佢？」

梁志明想了想，说：「我话俾佢听，我收几多，点样收，佢明明白白，佢自然信我。」

周老板拍了拍他的肩膀，说：「好，做生意就系咁。」

## 原文定义

持牌保险经纪公司如有意为客户安排保单，并将收取由相关保险人支付的报酬，该经纪公司在安排保单前应就该报酬向客户作出充分披露。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板不知道经纪佣金多少，怀疑对方利益冲突 | 报酬资讯不透明损害客户利益 |
| Thompson承认经纪未作披露、违反规定 | 监管机构对报酬披露的要求 |
| 梁志明主动对客人说明佣金结构 | 「安排保单前」充分披露报酬的具体实践 |
| Thompson提供公司经纪费率表供周老板查阅 | 报酬披露须以书面形式、清晰说明 |
| 梁志明：「点样收，佢明明白白，佢自然信你」 | 充分披露报酬是建立客户信任、避免利益冲突的核心 |
