# 6.2.5.5.2 有关保险产品的披露

## 故事

周老板有个侄子叫阿安，在码头做物流，刚升了主管，想为自己和手下的货车队买一份保障。周老板说："你揾志明，佢帮我搞咗好多年，靠得住。"

梁志明约阿安到茶餐厅谈。阿安性子急，坐下来就说："志明哥，你唔使我慢慢拣，你帮我揾一个最划算的就得。"

梁志明说："我先问你几件事。"

他问了车队的规模、货物种类、车队里有没有外判司机、往年有没有出过事故。阿安一一答了。梁志明记满了一页纸，然后拿出两份方案。

"阿安，你听我讲。"他把两份方案并排放，"两份都系合法的，都包车队意外和货物损失。但两份有几点不同，你要知。"

第一份保费固定，但货物损失有上限，超出要额外申报；第二份保费稍高，但货物损失没有上限，而且包括外判司机，唔需要额外申报。

阿安问："你叫我拣边份？"

"我唔可以直接话你拣边份，"梁志明说，"因为两份都系好产品，拣边份睇你实际情况。如果你车队大多数系自己聘请的司机，外判少，第一份够用；如果你外判多，第二份更合适。我讲完，你自己决定。"

阿安想了很久。"我外判多，"他说，"第二份啱我。"

梁志明把第二份方案的每一项费用列出纸来，包括保费金额、支付时间、续保费用，一清二楚写在上面。"你慢慢睇，有问题随时问我。"

阿安接过纸，第二天签了字。梁志明把这份手写的费用清单复印一份放进档案里。Mr. Thompson 路过问："你做呢份嘢几费功夫，直接报个价唔得？"

"得，"梁志明说，"但我唔想阿安签完之后先发现有几项费用未讲清楚。到时候唔系费用的问题，系信任的问题。"

那一年，阿安的车队在台风后出了一次碰撞，顺利理赔。他没有特意多谢梁志明，只是介绍了一个同行朋友嚟。

梁志明接过那个新客，照例先问基本情况，再给方案，再逐项解释相似之处和不同之处。

## 原文定义

持牌保险代理人须向客户披露所推荐产品的保险公司名称、主要条款及细则、保费水平及支付期间、客户须付的费用及收费；比较产品时须充分解释产品的相似性及差异，任何比较均须准确且不误导。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明列出保费金额、支付时间、续保费用 | 须披露保费水平及支付期间 |
| 方案并列，逐项说明货物损失上限与外判司机覆盖差异 | 比较产品须充分说明相似性与差异 |
| 梁志明不直接说"拣边份"，只说明各自适用情形 | 比较须准确不误导，决策权在客户 |
| 手写费用清单复印存档 | 须保存相关记录 |
