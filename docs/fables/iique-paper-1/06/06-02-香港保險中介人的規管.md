# 6.2 香港保险中介人的规管

## 故事

梁志明入行那年，香港保险业还是"三行鼎立"的时代。保险代理登记委员会、香港保险顾问联会、香港专业保险经纪协会，各自管各自的人，规矩也是各家定各家审。梁志明登记在代理登记委员会，每年交年费，接受同行评议，觉得这就叫"规矩"了。

2019年秋天，他收到一封政府公函，说从9月23日起，旧的自律规管制度作废，所有保险中介人须向新成立的保险业监管局申领正式牌照。公函措辞正式，盖着官印，跟以前那些行业自律机构的通知完全不一样。

梁志明拿着信去找Mr. Whitfield。Whitfield是英资律师楼的合规律师，这几年一直在跟保险公司和经纪公司解释新条例。Whitfield把信看了一遍，说："从今天起，你唔再只系行会的人。你系持牌人，直接对政府负责。"

"区别大吗？"梁志明问。

"大。"Whitfield说，"以前你违规，行会内部处理，最严重咪除名。现在你违规，IA可以直接查你、罚你、吊销你张牌。以前行会系自律机构，靠同行约束；依家IA系政府法定机构，有法律效力。"

那几个月，梁志明忙着整理过去五年的客户记录、培训学时证明和所有签过的投保文件。他发现以前行会的要求和IA的要求一比，只能算"建议"——IA的才是硬性规定，缺一份文件就不受理。

Mr. Thompson在英资保险公司管合规，说得更直接："以前三行各自管，标准唔统一，有些行会松啲，有些严啲。现在IA统一标准，唔合格就唔发牌，无情讲。"

梁志明的新牌照批下来那天，他在中环的IA办事处排队，周围全是以前各行会的老面孔。有人抱怨手续麻烦，也有人说"终于有个统一标准，客人信得过"。

Whitfield说得对：这套新制度，不是找麻烦，是让整个行业有个政府背书的规矩，客人知道有地方投诉，有人负责到底。

## 原文定义

保险业监管局于2019年9月23日透过法定发牌制度，接手三个自律规管机构规管保险中介人的工作。保险业监管局自此直接向香港的保险中介人发牌及进行监管。

| 故事元素 | 概念对应 |

| IA的政府公函取代行业通知 | 法定发牌制度取代自律机构 |
