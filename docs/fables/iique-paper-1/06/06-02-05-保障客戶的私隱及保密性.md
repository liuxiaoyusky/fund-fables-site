# 6.2.5.3.4 保障客户的私隐及保密性

## 故事

那天下午，周老板走进梁志明的保险行，脸色不太好看。

「志明，我唔系嚟倾嘢嘅。」他把一个棕色的公文袋放在枱面上，「你帮我老婆份保单入面，佢身体边度有问题，你唔知咩？」

梁志明心头一沉。

三个月前，周太太确实确诊了一个需要长期控制的慢性病。梁志明在协助她填写申请表时，如实记录了她的健康状况，并如实向保险公司申报。那份保单最终顺利批出，周太太现在每月按时覆诊，病情稳定。

但周老板显然不这样理解。

「我老婆唔知自己系咁严重，佢一直以为只系小毛病。你估我系唔系想呃保险公司？我唔想！但系你叫我老婆知道咗，佢呢几个月喊咗几多次，你知唔知！」

梁志明一时间哑口无言。他从来没有想过，向保险公司如实申报，等于是向周老板揭露了一个他妻子不知道的秘密。

他请周老板坐下，倒了一杯茶。

「周老板，你老婆份申请表上，佢自己亲笔签了名，授权我代表她向保险公司申报健康情况。我冇权唔报，亦都冇权向其他人透露。」他停了一停，「但系我亦都冇主动告诉你。」

「但你明知我唔知！」

「我知。但我唔知道你哋屋企点样相处。你太太选择了如实申报，是因为她想得到保障，不是因为她想影响你哋之间的信任。」

周老板沉默了很久。茶凉了，他都没有喝。

后来，周老板还是走了。梁志明不知道他和妻子之间怎样谈了这个话题，但他知道自己在这个场面里，做对了一件事：他没有把客人的资料，当成可以用来解决问题的工具。

别人的病历，别人的财务，别人的家庭状况——那些别人写在表格里的信任，从来不属于他。

## 原文定义

持牌保险代理人须将所有与客户有关的资料视为机密资料，不得使用或披露，但以下情况除外：为进行受规管活动而提供相关资料；经客户书面同意；或为遵守适用法律或规例。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明如实申报周太太健康状况 | 为进行受规管活动而使用客户资料 |
| 梁志明没有主动告知周老板妻子的病情 | 不得随意披露客户机密资料 |
| 周太太亲笔签名授权申报 | 书面同意作为例外情形 |
| 梁志明不以客户资料作为解决问题的工具 | 不得将客户资料用于非授权目的 |
