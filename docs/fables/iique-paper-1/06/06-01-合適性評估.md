# 6.1 合宜性评估

## 故事

何师奶的茶餐厅在湾仔开了十几年。丈夫去世得早，她一个人撑着，舍不得吃穿，但对伙计从不抠门。台风天来，她第一个到店；伙计生病，她叫人送粥到诊所。

那年夏天，一个保险行的伙计来找她，说想帮她看看餐厅的保障够不够。何师奶听了半天，问了几个问题，最后说："你不先问我家里有几口人、一个月挣多少、伙计几个，直接就推荐这个那个？"那个伙计支支吾吾，说是公司统一培训的内容，所有客人都是这样讲的。何师奶说："你回去问问你老板。"

那个伙计是梁志明。他后来去问了张小姐，张小姐说，做合宜性评估，不是填表，是真的去了解客人。

过了一段时间，梁志明又去找何师奶。这次他坐在茶餐厅的长凳上，没有急着讲产品，而是问她：铺位是自己的还是租的？伙计有多少个？有没有人有长期病？家里有没有老人要照顾？何师奶一开始不耐烦，说你问这么多做什么。梁志明说："不问清楚，我推荐的东西不一定是适合你的。万一出事，赔不了，我对不起你，也对不起你那些伙计。"

何师奶放下茶杯，看了他一眼。

她告诉他，丈夫出事那年，餐厅差点开不下去。医院的账单、儿子的学费、伙计的工资，全部压在她一个人身上。她从那时候起就怕——怕的不是没钱，是不知道有什么事会让这个家一下子塌下来。

梁志明听完，没有立刻推荐任何东西。他说，如果何师奶不告诉他这些，他给她配的方案可能是错的：保额太低，或者根本不是她最需要的方向。他解释："我要先知道你怕什么，才能帮你找到合适的东西。"

那天，何师奶要了一杯凉茶，梁志明喝完了才走。

后来，何师奶的方案里有一份意外保障，保的是她自己和她的伙计。她没有多问细节，只说："你帮我安排就好。"梁志明知道，这是因为她信他。而他之所以能让她信，是因为他先花了时间去听。

那年茶餐厅装了新的招牌。何师奶让梁志明帮忙看看合约里有没有什么问题，他二话不说去了。

## 原文定义

持牌保险代理人应在提供受规管意见前，就客户情况进行适当的合宜性评估，以确保持牌保险代理人就客户情况获得足够资料，并据此提供意见；若客户未能提供所需资料，代理人应向客户说明，未获完整资料下所提供的意见未必切合客户情况；合宜性评估的程度应与产品类型和客户情况相称且合理。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明第一次没问清楚就直接推荐 | 缺乏合宜性评估的教训 |
| 第二次坐下了解何师奶的家庭、收入、雇员情况 | 采取合理步骤了解客户情况 |
| 梁志明解释"不知道你怕什么，配的东西可能是错的" | 合宜性评估的目的：获得足够资料，提供切合客户的意见 |
| 何师奶没有直接说担心家庭，而是聊起丈夫去世的经历 | 隐晦感情线：家庭责任作为保障动机的底色 |
| 旅游保险可作为申请过程的一部分进行评估 | 合宜性评估程度与产品类型相称的原则 |
