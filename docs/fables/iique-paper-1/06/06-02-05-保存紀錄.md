# 6.2.5.3.5 保存纪录

## 故事

台风吹袭过后，湾仔的街道满是塌下的树枝和碎玻璃。张小姐却比平常更早到了公司。

她是保险公司的合规助理，每天的工作就是对着一叠又一叠的文件——申请表、批核记录、客户通讯、变更通知单。她有自己的一套方法：颜色标签、编号归档、按日期排序。整齐得仿佛强台风都吹不乱。

这天早上，Mr. Thompson从英国总部打来电话，说监管机构可能会派人前来查察，要她把所有与持牌业务代表相关的记录整理好，随时可以调取。

张小姐挂上电话，打开了文件柜。她先确认了每一位业务代表的个人牌照状态，然后整理了他们的培训记录、委任文件、以及过去三年的客户来访记录。

文件很多，但她心里清楚：查察员要的不只是数量，而是看她有没有办法在合理的时间内，找出具体某一笔交易的来龙去脉。

她坐下来，开始做索引目录。

旁边的梁志明走过来，见她埋头做着重复性的工作，有点不解：「张小姐，呢啲文件放喺度就得啦，做索引做嚟做咩？」

张小姐头也不抬：「你去过律师楼未？他哋查嘢，唔会逐份文件慢慢揭。佢哋会问：『呢个客喺几时确实知道自己有冷静期？』然后我就要即刻拎出相应记录，证明我哋嘅代理人按时送了保单、告知了权利。如果我唔记得放喺边，就等于冇发生过。」

梁志明想起自己上个月处理的一份申请，似乎忘记了把客人的签收副本扫描归档。他皱了皱眉。

张小姐仿佛看穿了他的心思，说：「《保险业条例》要求我们的代理人妥善保存记录，并按时递交予委任保险人。你自己内部唔做好记录，到时唔单止影响你，仲会影响整间公司。」

窗外，救护车的鸣笛声划过维多利亚港的清晨。张小姐继续做着索引，仿佛这个城市的所有风暴都与她桌上那叠文件无关。

但她心里清楚：恰恰是这些看起来琐碎的纸张，构成了一个人能否持牌的底气。

## 原文定义

持牌保险代理人须遵从其委任保险人或委任代理机构的所有有关保存纪录的规定、政策及程序；如须递交记录，须在合理切实可行的范围内尽快递交。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 张小姐按规定的颜色标签、编号整理文件 | 妥善保存纪录的规定与程序 |
| 张小姐来不及找文件＝冇发生过 | 保存记录的可验证性要求 |
| 监管机构可能查察，需要即时调取 | 查察时须能提供完整记录 |
| 梁志明忘记归档签收副本 | 未妥善保存记录的风险 |
