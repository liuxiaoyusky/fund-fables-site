# 6.1.3b 保险投诉局提供服务职权范围

## 故事

油麻地果栏收档的时候，何师奶的茶餐厅才刚开始热闹起来。

街坊叫她何师奶，因为她从早忙到晚，身上那件白围裙从来没干净过。早上帮补茶水，中午收钱找赎，下午三点才能坐下来吃口饭。她丈夫五年前在码头卸货时被缆绳击中，当场没了。船上那笔赔偿拖了两年才下来，数目不大，但够她把茶餐厅撑到现在。

"阿明哥，进来饮杯茶。"何师奶把一杯凉透的柠檬茶推到梁志明面前。梁志明是隔壁街保险行的伙计，骑单车跑客户，偶尔经过她门口。

"师奶，我唔该。"他接过杯子，"那笔钱……怎么样了？"

何师奶没说话，从围裙口袋里掏出一封信。牛皮纸，边角已经起毛。信上盖着保险公司的红印章。

"话我唔系'指定受益人'，要我去律师楼攞多份文件。我老公走了五年了，我去边度揾律师楼——当年连正式注册都没有。"

信上写的是：受益人认定有争议，建议通过法律途径解决。

梁志明看完，把信折好："师奶，这间公司系投诉局的会员，你知唔知？"

"投诉局？"何师奶愣了一下，"我以为只有上环大公司先有。"

"投诉局系成个行业自己搞的，唔收钱，专帮街坊处理同保险公司拗数。你张保单系个人个账，赔多少钱？"

何师奶想了想："当时签的是十万人寿，加埋储x蓄部分……赔下来应该有九万多。"

"九万多？"梁志明拍了拍台面，"一百万以下，先过咗第一关。"

"第一关？"

"投诉局有几条规矩，几样嘢要啱，先会受理。"梁志明竖起手指，"第一，钱银性质——你这个系赔钱，啱。第二，唔超过一百万，啱。第三，保险公司要系投诉局会员，这间是，啱。第四，保单系个人保单，你那份是，啱。第五，投诉人要系保单持有人、受益人或者受保人——你是受益人，啱。"

何师奶听得入神，不自觉把柠檬茶往他那边推了推。

"还有，"梁志明说，"保险公司要出咗最终决定，先可以投诉。你这张信上写了佢哋的决定，呢样有。"

"但封信叫我打官司……"

"投诉局唔系打官司，唔洗你请律师，唔洗上法庭。但有一点——"他顿了顿，"投诉要在收到最终决定六个月内提出。你这封信几时嚟嘅？"

何师奶翻了翻抽屉，攞出一张皱巴巴的纸，上面有日期邮戳。"上个月中。"

"咁就来得及。"梁志明说，"不过有一项你要留意——投诉唔涉及商业保险，你呢份系个人人寿，唔关事。但投诉唔受理正在打紧官司嘅个案，你暂时唔告，净系去投诉局，先可以。"

何师奶沉默了很久。窗外的红色小巴轰隆驶过，震得玻璃柜嗡嗡响。

"阿明，我老公走𠮶阵，啲人叫我唔好追究，说街坊唔好打官司。"她低头擦着已经擦得发亮的台面，"但我唔系为咗呃钱。我系想知——点解我老公买嘅嘢，攞唔到手。"

梁志明没接话。他从口袋里掏出一支笔，在纸巾上写了投诉局的地址：中环德辅道中某号。

"师奶，投诉要书面。你写封信，将件事讲清楚，连埋那张保单副本、同埋保险公司封信，一齐寄过去。投诉局会派人睇，唔洗你到场，唔洗你请假，唔洗你使钱。"

何师奶接过纸巾，看了一眼地址，又看了一眼那杯已经彻底凉透的柠檬茶。

"你帮我写？"她问。

梁志明笑了笑："我写，你签名。这是你的事，不是我的事。"

三个月后，投诉局回了信。保险公司重新受理了她的理赔申请。何师奶没等到钱下来，但她等到了一个说法——保险公司承认受益人认定程序有误，愿意重新核算。

那年冬至，何师奶在茶餐厅门口贴了张红纸，写着"本铺已领保险投诉局服务认同"。街坊问她这是什么意思，她说："意思系——有事可以揾人，唔使自己孭。"

梁志明路过时看见那张纸，停了停，没进去喝茶。他赶着去跑下一单。

## 原文定义

保险投诉局受理投诉须符合以下条件：涉案个案的索偿金额不超过100万港元；涉案保险人须为投诉局会员；涉案保单须属个人保单；投诉人须为保单持有人、受益人、受保人或合法索偿人；保险公司须已作出最终决定；投诉须于接获最终决定后六个月内提出；投诉不得涉及商业保险，亦不得正处于法律程序或仲裁中。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 赔偿金额九万多 | 索偿金额不超过100万港元的上限条件 |
| 保险公司为投诉局会员 | 涉案保险人须为投诉局会员 |
| 何师奶为保单受益人 | 投诉人须为保单持有人、受益人、受保人或合法索偿人 |
| 保险公司出具"最终决定"信 | 须待保险人作出最终决定后方可投诉 |
| 投诉在六个月内提出 | 须于接获最终决定六个月内提出 |
| 投诉局不受理正在打官司的个案 | 投诉个案的非金钱条件之一：不得正在进行法律程序 |
| 何师奶被要求先撤诉才能投诉 | 不得正处于法律程序或仲裁 |
