# 6.2.3.7 有关保险索偿的协助

## 故事

那年夏天台风来得早，维港的风把周老板海味铺的后门灌进来的雨水泡了半仓库的干鲍鱼。损失不算小，周老板第一时间想到了梁志明。

他打电话到公司，张小姐接的，说梁志明出去见客了，让他下午再打。周老板发了脾气，说："我铺头俾人影紧，你叫我等？"

梁志明回来听张小姐说了，立刻回电。周老板在电话里急得声音都变了。梁志明说："周老板，你千其唔好自己搞住，先揾齐以下嘢——货物清单、存货照片、同埋上次盘点的数字。我听朝过嚟上环，帮你填索偿表格。"

第二天一早，梁志明真的来了。他带了空白的索偿表格，搬了张凳子在仓库门口坐下，一张一张地帮周老板核对清单。他把泡了水的鲍鱼列在一栏，没事的列在另一栏，又让周老板找出那天台风的新闻剪报，作为日期的证据。

表格填完，梁志明把副本留给周老板，自己把正本带回公司，当天就递给了保险公司。

张小姐问他："你自己跑上环帮他填表？这是他伙计的工作吧？"

梁志明说："佢唔识英文，又唔知填边几栏。我递上去的文件，我自然要确保内容系岩嘅。"

三个月后，周老板收到了赔款支票。他没说什么感谢的话，只是逢人就说："上环间保险行个后生仔，得嘞。"

## 原文定义

持牌保险经纪应在客户要求下，就其代表客户洽谈或安排的保单项下的任何索赔提供合理协助，并尽快向有关保险人转交收取自客户的有关该索赔的相关资料，且须适当谨慎地履行在客户协议中所述的所有相关义务。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明一回来就立刻回电周老板 | 经纪须在客户提出索赔时及时提供协助 |
| 梁志明带空白表格上门帮周老板填写 | 提供合理协助：包括代填表格、整理证据文件 |
| 要求客户提供货物清单、照片、新闻剪报 | 须向保险人转交与索赔相关的完整资料 |
| 核对副本后才递出正本，确保无误 | 适当谨慎地履行索赔义务，确保文件准确完整 |
