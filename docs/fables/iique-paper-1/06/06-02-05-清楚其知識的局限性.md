# 6.2.5.4.2 清楚其知识的局限性

## 故事

那天下午，一个从新加坡来的商人找到梁志明，说他在新加坡有一间食品加工厂，最近想在香港拓展业务，需要为工厂和出口货物同时投保，但他的大部分生产在新加坡，香港这边只是一个很小的包装车间。

梁志明听完，想了五分钟，然后说："陈先生，你讲嘅情况涉及跨境生产同出口货物，这部分我接触唔系好多。"

陈先生有点意外。在他的经验里，做这行的总爱当场答应，怕丢了客人。

梁志明继续说："我唔可以乱噏你。我对本地工厂的工伤和火险比较熟，但跨境货物运输和海外工厂资产这部分，我真系唔够经验。如果我乱噏你，到时候你的货物在新加坡出咗事，我帮唔到你。"

陈先生沉默了一会儿，问："咁你打算点？"

"我帮你问。"梁志明说，"我们公司有专做跨境业务的同事，我摇个电话，问清楚你的情况适合揾边个、点样处理，再回复你。"

那天晚上，梁志明留在公司打电话，先向 Mr. Thompson 请示，Mr. Thompson 又帮他联系了一位有跨境业务经验的同事。两天后，他给陈先生回了电话，介绍了一位在这方面更专业的同事跟进。

陈先生后来签了约，没有在梁志明这里投保。但他给梁志明介绍了一个本地客。

梁志明问点解，陈先生说："你去我问嘢，唔系乱噏，我又唔会因为你唔识就换人。呢种人，先信得过。"

梁志明把这件事记在本子的最后一页，不是当成绩，是提醒自己：识得话"我唔知"，比识得话"我知"更难，也更重要。

## 原文定义

持牌保险代理人不得就其缺乏进行有关受规管活动所需特定技巧或知识的事宜进行受规管活动；如有疑问，应寻求其委任保险人的适当人士或代理机构的负责人或高级管理层提供指引。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明说自己对跨境货物运输"接触唔系好多" | 清楚其知识局限性，不在缺乏特定知识的领域提供意见 |
| 他没有当场答应，而是说"我帮你问" | 如有疑问应寻求适当指引，而非凭估计行事 |
| 向 Mr. Thompson 请示并联系有跨境业务经验的同事 | 寻求委任保险人或代理机构适当人士提供指引 |
| 陈先生说"你唔识就换人，呢种人先信得过" | 诚实承认局限反而赢得信任 |
