# 6.2.7.1 透过适当披露主事人与代理人关系，避免潜在利益冲突并提高透明度

## 故事

上环的海味街一年四季都飘着鱿鱼干和蚝豉的气味。周老板在那儿开了间海味铺，柜台后面挂着发黄的账本，柜台前面堆着一箱箱准备发往英国和东南亚的货。每年台风季节过后，他都会听见同行说哪家货仓进了水、哪家商铺失了火，心里便发慌，想买一份保障。

那年夏天，周老板托街坊介绍，找到了一间挂着英资保险公司招牌的保险行。接待他的是个年轻伙计，叫梁志明。

梁志明带周老板进了小会议室，倒了杯茶。张小姐在门口探头看了一眼——她是合规部的，每天负责把牌照和披露文件按清单理好，今天轮到她在梁志明的这一单上做合规备注。

"周老板，我先跟你说清楚一件事。"梁志明说。他停了停，把钢笔放下，"我系帮人打工嘅，我代表的是这间英资保险公司。我只能卖这家公司的产品给你，没法帮你比较别家。"

周老板愣了一下，眉头皱了起来。"那你跟其他经纪有乜嘢唔同？我以前揾过嘅经纪话乜嘢都可以保晒，样样都识。"

梁志明笑了笑，说："船运码头您去过吧？船东请货运代理帮他揽货，代理就只替这一家船公司揽——他不会同时替五家船公司做代理，那样客人才真的不知道出了事该找边个。我也一样。"

周老板听了，想了一会儿，点点头："咁样讲我就明。你系边个公司嘅人，签单之前我已知，签完之后我都知揾边个。几好。"

梁志明拿出财产险和货运险的条款，逐条跟周老板过。张小姐在门外听见了，轻轻在笔记本上记了一笔：已披露代理关系。

签完字，周老板站起来，忽然说："我有个老友在澳门做药材生意，转头介绍俾你。"他拍了拍梁志明的肩，"不过你记得，你系边个公司嘅，唔好呃我。"

梁志明笑着送他出了门。那天放工时，张小姐把合规文件夹进档案袋，对他说："周老板签咗名，代理关系披露已入档。"梁志明把那份备注又看了一遍，想起周老板说的那句话——"你系边个公司嘅，签之前我就知"——原来做这行，第一件事不是说产品有几好，而是先把"我是谁"这三个字讲清楚。

他合上档案夹，把这天的记录放进柜子里。文件上的签名很稳，是周老板练了几十年毛笔字才有的那种笔力。

## 原文定义

持牌保险代理人的职分是其委任保险人或委任代理机构的代表。代理人应向客户披露：（i）该代理人是获委任保险人或代理机构委任，以推广该保险人或代理机构提供的保险产品，或就该保险产品提供意见或安排保单；及（ii）该代理人可以推广、提供意见或安排保单的保险产品仅限于其委任保险人或委任代理机构提供的产品。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明代表英资保险公司，只卖一家产品 | 代理人是委任保险人的代表 |
| 梁志明先说明身份，再说产品 | 须向客户披露代理关系 |
| 梁志明用船运代理作类比："只替一家船公司揽货" | 代理人仅能推广委任保险人的产品 |
| 周老板"签之前已知"是谁公司的人 | 透明度：客户知悉代理人身份 |
| 张小姐将披露记录入档 | 合规留存要求 |
