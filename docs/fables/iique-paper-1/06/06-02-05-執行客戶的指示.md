# 6.2.5.3.3 执行客户的指示

## 故事

那天下午三点，周老板打来电话，语气着急。

「志明，我老婆话我想加个受益人——我个女，佢今年考大学，我想趁我仲做得嘢，帮佢搞妥份教育基金。你帮我搞。」

「冇问题，我听日帮你递——」

「唔好等听日，」周老板打断他，「听日佢生日，我想今日搞定，惊唔记得。」

梁志明看了看表，三点十分。他手头上还有两份申请表要处理，都是当日要交的。保险公司的表格系统四点半关机，如果他要赶在今日完成周老板的申请，必须在四点前递交所有文件。

他深呼吸了一下，然后答应了周老板。

接下来的一个小时，梁志明几乎没有停过。他先打电话给保险公司的核保部确认加名表格的填写方式，然后一边打开周老板的现有保单档案，一边致电周老板的女儿确认她的身份资料。

四点十五分，他按时递交了申请。

四点三十分，他致电周老板：「搞掂了，你个女由今日起系受益人之一。你会收到保险公司的正式确认通知。」

周老板在那头沉默了一会，然后说：「志明，唔该。」

挂上电话，梁志明望着办公室窗外维多利亚港的船只，慢慢整理着桌面。他想起以前曾经有一位代理人，因为太忙而忘记了帮客人更新受益人资料，结果那位客人几年后去世，家属为此折腾了好几个月。

他庆幸自己今天没有把那通电话推掉。

「准确而且及时」——这是Mr. Thompson在合规培训上说的话。梁志明当时不以为然，现在他开始明白它的分量。

准确，是不要写错名字、填错资料。及时，是在客人需要的时候做到，而不是在你自己有空的时候才做。

客户的指示，不只是他们说了什么，也包括他们为什么这样说。

## 原文定义

持牌保险代理人应采取合理步骤，准确及时地执行客户的指示，如发生任何延误或未能执行，应在可行情况下尽快通知客户。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明赶在系统关机前递交申请 | 及时执行客户指示 |
| 梁志明确认周小姐的身份资料后才提交 | 准确执行客户指示 |
| 梁志明主动告知周老板完成进度 | 未能即时执行时主动沟通 |
| 提及其他代理人忘记更新受益人的往事 | 不及时执行可能造成严重后果 |
