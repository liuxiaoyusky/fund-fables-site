# 6.2.3.6 冷静期

## 故事

周老板在上环开了三十年海味铺，摸过的账本比梁志明看过的报纸还多。但他从没买过保险——不是舍不得钱，是觉得"写了字的东西，签了就没有回头路"。

那年夏天，周老板终于松口，托梁志明替他在一间英资保险公司投保铺面财产。梁志明是公司里最年轻的伙计，中英文文件都看得懂，周老板信他。

填完申请表那天，周老板拿起笔就要签。梁志明伸手拦了一下："周老板，唔好咁急。你望一望这份申请表最尾𠮶几行字。"

周老板皱眉："乜嘢嘑？"

"『冷静期』。"梁志明指着那行小字，"签完之后，你有三天时间慢慢谂清楚。唔钟意，随时可以取消，钱会退回比你。"

周老板放下笔，认真看了他一眼："你唔系想我走数呃？"

"梗系唔系。"梁志明笑了，"白纸黑字写清楚，你有权谂清楚再决定——呢个系对你好的。你谂多几日，睇清楚份保单，究竟赔边啲、不赔边啲，再决定留唔留。"

周老板没说话，把申请表收进公文袋，拍了拍梁志明的肩膀。

三天后，保单正式生效。周老板没有取消。他在茶餐厅请梁志明饮了杯奶茶，说："你肯等我，你靠谱。"

张小姐事后问梁志明："你真系等他三日？万一佢走数呢？"

梁志明说："规矩系保护客人的。客人信你，首先你唔好催人哋。"

## 原文定义

持牌保险经纪应在客户签署保单申请前告知客户冷静期权利，并在保单送交后确保客户有足够时间在冷静期届满前审阅保单及考虑购买决定。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明主动告知周老板冷静期权利 | 经纪须在签约前告知客户冷静期可取消保单的权利 |
| 周老板拿着申请表回家考虑三日 | 冷静期赋予客户充分时间审阅和考虑 |
| 梁志明没有催促，等保单正式生效 | 经纪须及时送交保单以便客户在冷静期内完成审阅 |
| 周老板因被尊重而信任梁志明 | 遵守冷静期规定有助建立客户信任 |
