# 6.2.5.1.1 准确的陈述及介绍

## 故事

周老板的海味舖在上环开了三十年，从干货到燕窝，什么名贵药材都经手过。八十年代初，香港做海味生意的商号都怕两件事：一是台风压港货受损，二是客户回头说"当初不是这样讲的"。周老板两样都经历过，所以格外信一样东西——白纸黑字。

那年春天，周老板的女婿进了洋行做经理，回来跟他说，不如趁舖头生意稳定，给老员工买份保障，算是人情。周老板觉得在理，叫女婿找个靠谱的人来谈谈。

来的是梁志明，穿一身深色西装，领口别着办事处的钢印徽章，说话不快不慢，把产品讲得清清楚楚。周老板听着觉得熨帖，便签了意向书。

签完第二天，周老板路过同业陈生的舖子，聊起来。陈生说："你那间公司的代理人前天也来过我这里，开口就说他们和三家大保险公司都有合作，市面上最好的产品都在他们手里。"周老板愣了一下——梁志明没说过这话。陈生还说对方递了张彩色产品介绍单，上面印着"获多家顶尖保险公司授权"。

周老板心里别扭，回去翻出梁志明那天给他的资料袋，里面是一份油印的条款摘要，末尾盖着保险公司的章，只字没提"市面上最好"这类话。周老板想了想，拿去给女婿看。女婿认英文，一行行译过来，跟他说："这份东西没问题，倒是你说的陈生那间，口气大得离谱。"

周老板当晚去找了陈生，问他那间公司的名字。陈生说："说是叫'环球保险顾问'，听着大公司气派。"周老板叫女婿去查，那间公司注册才半年，根本不是什么大保险公司旗下，名字也没有得到任何授权。

女婿说："爹，那份彩色单张是误导。周老板这次没中招，是运气好。但街坊里不知多少人看着那张单张就信了。"

这件事让周老板想起当年初入行，师傅教他三句话：话要说准、钱要数清、纸要留底。梁志明做的，正是师傅那套规矩。只不过现在这套规矩印成了规管文件，第6.2.5.1.1条，一条一条写得分明。

女婿后来跟梁志明说了这事。梁志明没说什么，只是把办事处的注册名称和商业名称重新核对了一遍，把名片上的措辞也改了——只留"授权代理"四个字，不多不少。

## 原文定义

持牌保险代理人不得误导或欺骗客户，并应确保就任何保险人、保险中介人或保险产品所作出的任何陈述或提供的资料为准确，并无误导或欺骗成分；在推广保险产品时，只应使用受委任保险人提供或认可的材料；不得使用可能欺骗、误导或混淆客户的名称。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明给的条款摘要只盖保险公司章，无夸大措辞 | 代理人陈述须准确，无误导成分 |
| 陈生那间公司的"获多家顶尖授权"单张 | 不准确、误导性陈述 |
| "环球保险顾问"名称未经任何保险公司授权 | 不得使用欺骗性或混淆性名称 |
| 梁志明重新核对办事处注册名称和商业名称 | 使用受委任保险人认可的材料 |
