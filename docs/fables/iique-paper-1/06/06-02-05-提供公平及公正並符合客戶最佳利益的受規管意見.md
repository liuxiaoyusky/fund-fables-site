# 6.2.5.2.2 提供公平及公正并符合客户最佳利益的受规管意见

## 故事

那年冬天，观塘一家制衣厂的老板娘蔡太找到梁志明。厂里有四十几个工人，接的都是欧洲订单。欧洲买家要求蔡太提供一份保障方案，否则不给新订单。

蔡太坐下就说："志明，我唔想听太多，我净系想知道，边只最平。"

梁志明拿出三份方案，每份来自不同保险公司。他本可以只给她看最便宜的那份，花十分钟收工。但他没有。

"蔡太，你要俾我啲时间。"他说，"我想去你厂里睇一次。"

蔡太有点意外，但还是答应了。

梁志明在制衣厂走了一圈。裁床旁边堆着布碎，走廊里电线拉得凌乱。他问蔡太：工人多大年纪？有没有人在厂里做了超过十年？厂里有没有小型锅炉？仓库离生产线有多远？

蔡太一一答了。梁志明又问：欧洲买家对工人保障有没有具体要求？蔡太想了想，说对方要求列明工伤和货物损坏两项。梁志明把这些全部记在本子上。

回公司的路上，Mr. Thompson 问他："志明，你点解唔直接报最平𠮶个？"

"最平𠮶个不包货物损坏，蔡太需要呢项。"梁志明说，"最贵𠮶个包晒，但蔡太嘅厂唔需要锅炉保障，客人多付钱买咗一堆用唔着嘢。"

他在白板上列出蔡太厂的实际风险，把三份方案逐项对照，每一项对应蔡太的实际需要。Mr. Thompson 看了一会儿，说："你记得自己的角色。你唔系帮边间公司卖嘢，你系帮蔡太揾一个适合佢嘅方案。"

梁志明把方案交给蔡太。蔡太看完，指着最便宜那份说："呢个唔包货物损失，我排除咗。但最贵嘅我唔需要咁多项。中间呢份，加一个货物损坏附加条款，正合适。"

她签了字。走出会议室时，她说："志明，你系第一个走去我厂里睇嘅人。"

那一年，蔡太的制衣厂拿到了欧洲订单，没有因为保障方案不足而被卡住。梁志明后来把那张制衣厂平面图收进了文件夹——不是留念，是提醒自己：给意见之前，先要真正了解你要给边个。

## 原文定义

持牌保险代理人在提供受规管意见前须作出合理查询以取得与客户相关的必要资料；须考虑自客户取得的资料，包括客户情况，作出有合理依据的意见；意见应为一位合理持牌保险代理人认为适合该客户的意见。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明去制衣厂实地查看、询问工人情况 | 提供意见前须作出合理查询以取得与客户相关的必要资料 |
| 梁志明列出三项风险与三份方案逐项对照 | 意见须基于对客户情况的充分了解 |
| Mr. Thompson 说"你是帮蔡太揾适合的方案" | 意见须符合客户最佳利益，而非推销某一产品 |
| 蔡太最终选了中间那份加附加条款 | 受规管意见须基于客户实际情况，而非单纯比较价格 |
