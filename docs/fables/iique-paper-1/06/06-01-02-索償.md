# 6.1.2c 索偿

## 故事

周老板的海味铺那年冬天出了一件事。一批干货在仓库里被白蚁蛀了，损失不小。他想起自己在梁志明公司买过一张财产保单，便带着保单和进货单据去理赔。

公司核赔人员看完材料，说："周老板，您这张保单条款里写明，储存环境须保持干燥通风。您这批货受了潮，按条款我们可以拒赔。"

周老板不服气："白蚁蛀和潮湿是两回事。那天是因为隔壁单位的水管爆了，水渗过来才潮湿的。"

梁志明把这件事报告给Mr. Thompson。Mr. Thompson调出条款，翻了几页，说："拒赔要有依据，不能说'按条款我们可以'就打发客人。你看这里——守则讲得很清楚，不合理拒赔的情形，包括'没有合理理由以未披露重要事实为由拒赔'，还有'违反保证但无欺诈且未导致损失'。"

他指着条款说："这批货的损失原因是白蚁，不是潮湿。水管爆裂是意外，不是周老板没如实申报。拒赔理由不成立。"

张小姐在旁边翻出处理时限记录，说："还有一条——理赔要公平、有效率、迅速。客人的合理查询要定期告知进度。拒绝的话，必须给出合理解释。"

最后公司派人实地查勘，确认水管爆裂为第三方责任，赔付了大半损失。张小姐在结案报告里写了一句："建议日后对条款中'拒赔'字眼作清晰标注，减少歧义。"

周老板拿到支票那天，专门走到公司柜台，把一张手写的收据交给梁志明。收据上写着"已收理赔款"。梁志明说不用这么正式，周老板说："要嘅，规矩啲好。"

那天下着雨，周老板没有带伞。梁志明从柜子里拿了一把公司备用的透明伞给他，他接过去，说："下次还你。"

他后来没有还。但那把伞一直挂在海味铺门口的钉子上，周老板也没把它当回事。

## 原文定义

索偿处理须公平、有效率及迅速；不得在未涉及欺诈且未导致实际损失的情况下，以违反保单保证为由拒赔；拒赔须有合理依据并向客户作出合理解释；索偿表格须以浅白语言表达，免费并迅速发放；第三方理赔人员须具备专业资格并合理行事。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 公司不得随意援引条款拒赔，须核实原因 | 不得无理拒赔（未导致实际损失则不可拒赔） |
| 理赔须迅速，不得拖延 | 公平、有效率、迅速处理原则 |
| 拒赔须向周老板解释水管爆裂属第三方责任 | 须作出合理解释 |
| 张小姐要求理赔人员具备资质并合理行事 | 第三方须具备专业资格 |
