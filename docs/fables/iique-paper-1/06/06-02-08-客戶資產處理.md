# 6.2.8.1 客户资产处理

## 故事

何师奶的茶餐厅开了十几年，靠的就是一个"信"字。街坊来喝茶，月底结账；伙计出粮发现金，从不拖到第三天。何师奶挂在嘴边的话是："钱过手，心要正。"

那年台风季，海味街的周老板要续保，周转不灵，先把三个月的保费现金交给梁志明，让他代为转交保险公司。梁志明是保险行的年轻伙计，接了钱，锁进抽屉，心想等公司出单再一起转。

谁知那几天账房陈伯不在，梁志明自己手头也紧。恰好公司要订一批新的保险文件，供应商说现金可以便宜两百块。梁志明犹豫了一下，把周老板那叠钞票先挪用了，心想月底对完账补回去就是了。

何师奶的伙计阿昌那天正好经过保险行门口，看见梁志明从抽屉里拿钱，顺口说了句："梁仔，那钱是谁的？"梁志明说没事，月底就还。阿昌回去跟何师奶提起，何师奶当晚就找上门来了。

她没骂人，只是坐在茶餐厅里，把那杯凉了的奶茶推到梁志明面前，说："阿昌跟我说你挪了钱。我知你唔系走数，但钱过手就系钱，唔系你嘅就唔系你嘅。你今日挪三百，下个月再挪三百，街坊就唔信你咯。"

何师奶从柜桶底翻出一个旧信封，里面是她丈夫出事那年收的赔偿金。"当年佢走咗，赔偿金系佢用命换来的，一毫子我都唔敢乱使。你做呢行，客人的钱就系咁样。"

梁志明把那三百块塞回抽屉，第二天一早就去银行开了个独立账户，把周老板的保费存了进去。他后来跟陈伯学做账，每个月末把客户账户的进出数目核对一遍，哪怕只有几百块也一笔笔对齐。陈伯说："做呢行最紧要个心正，其次先系账目靓。心正，账目自然靓。"

那间独立客户账户后来成了保险行的规矩。街坊来交保费，无论多少，当天必须进客户账户，从不混进公司的运营经费里。

## 原文定义

持牌保险经纪公司如收取或持有客户款项，须维持最少一个客户账户；客户账户须为以持牌保险经纪公司的名义在认可机构维持的账户，而该账户的名称有"客户"一词。持牌保险经纪公司须每公历月进行客户款项对账。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 何师奶说"钱过手，心要正" | 持牌保险经纪对客户款项的受信责任 |
| 周老板的保费单独锁进抽屉 | 独立的客户账户要求 |
| 陈伯要求每月末对账 | 每公历月进行客户款项对账 |
| 独立账户后来成为公司规矩 | 足够的管控及程序保障客户资产 |
