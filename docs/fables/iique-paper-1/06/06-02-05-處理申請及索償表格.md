# 6.2.5.3.2 处理申请及索偿表格

## 故事

周老板那年要给三个伙计各填一份意外险申请表。表是梁志明送来的，中英文对照，共四页。周老板翻了两下，说："这些格子我看得懂，但填起来怕写错。你帮我填？"

梁志明说："周老板，我可以帮您读题目，解释每一栏的意思，但最后那支笔，得您自己拿。"

周老板觉得奇怪，说："你们做这行的，不就是帮人跑腿填表吗？"梁志明说："跑腿可以，但表上每一个空格，最后都要客户自己确认。您不认，就是一张白纸；您签了字，这份表就跟您到老了。"

两人坐下来，一栏一栏过。健康状况那栏，周老板说伙计阿福去年摔过一次，膝盖动了手术。梁志明让他写清楚哪天伤的、在哪家诊所看的、医生开了什么诊断。阿福在旁边听着，说："这么麻烦，当初口头说一声不就行了？"梁志明说："口头说的，将来保险公司来查，查不到字据，就说不清楚了。"

填到声明栏的时候，周老板要签名。梁志明把表格推到他面前，说："您再看一遍，这几页内容都是您说的、您确认的，有没有漏掉的、填错的。确认无误再签。"

周老板仔细看了一遍，指出一处出生年份写错了一个数字。梁志明让他在旁边划掉，注上正确数字，签上姓名和日期，然后才正式填写。

签完字，周老板把复印件留了一份，说："我师傅以前教我，签了名字的东西，就是欠了的债，要收好。"梁志明点头。

那天走出海味舖，天已经黑了。梁志明想起Mr. Whitfield跟他说过的话："有些代理人帮客人填表，客人连签了什么都不知道，出了事就互相推诿。规矩写在那里，'须告知客户有责任确保资料准确完整'——这话不是讲给客人听的，是讲给代理人听的。"

## 原文定义

持牌保险代理人协助客户填写任何申请、索赔或其他表格时，应告知客户有责任确保表格内提供的资料及支持文件为准确完整；在未获客户同意及与客户确认内容完整准确前，不应填写、修改或向保险人递交任何表格；若知悉表格含有不准确资料，不应向保险人递交该表格。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明帮读题目、解释意思，但请周老板自己拿笔填 | 代理人可协助解释，但不能代填代签 |
| 梁志明要求周老板对健康状况栏如实填写，注明日期和诊所 | 须告知客户有责任确保资料准确完整 |
| 声明栏前梁志明把表推给周老板，请他再检查一遍 | 未获客户确认内容完整准确前不得递交 |
| 周老板发现出生年份写错，划掉更正后签名确认 | 知悉表格有误须更正，不得在有误状态下递交 |
| 梁志明保存复印件归档 | 代理人留存完整记录备查 |
