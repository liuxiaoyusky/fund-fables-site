# 6.2.7.2 披露代理人与其主事人之间条款与细则中的相关限制

## 故事

中环那间英资保险公司的会客室，窗外正对着一排灰白色的办公楼。梁志明第一次坐在那张长木桌旁，手心一直贴着裤线，像怕自己把文件弄皱。周老板坐在对面，手里夹着一封从英国总公司寄来的报价信，皱着眉看了半天。

周老板的海味铺想替几名伙计安排一份新的保障。他平日嘴硬，说自己做生意靠眼光，不靠洋文，可这回伙计阿成在码头搬货时摔伤，他才真的怕了。他来找梁志明，是因为信得过这个后生仔：说话不花哨，茶餐厅里见了面还会替他多叫一杯冻柠茶。

桌上摆着三只文件箱。第一只贴着蓝色标签，第二只贴着绿色标签，第三只却被张小姐轻轻推到了桌角，没打开。

周老板眼尖，立刻问："那只箱不用看？"

梁志明正要开口，Mr. Thompson 从门口走进来。他用手指敲了敲第三只箱，声音不重，却让房里安静下来。

"这只箱里也有适合小商号的安排，"他用慢慢的中文说，"但梁先生现在不能替这家公司介绍。"

周老板的脸色沉了下来。"不能介绍？那我怎么知道你给我的，是不是最合适？"

梁志明心里一紧。他想起上星期张小姐在档案柜前提醒他的话：有些委任文件表面只是几页纸，里面却写着一条限制，规定某段时间内不能替另一家公司推广某些产品。张小姐当时把铅笔停在那行小字旁，没有多说，只把复印件推给他。她那天还多留了一把伞在他桌边，像是怕他又急着赶去见客，连雨都忘了。

梁志明把第三只箱重新推回桌中央，却没有打开。他看着周老板说："周老板，这只箱我今天不能替你介绍，不是因为它不好，也不是因为我想藏起来。是我和其中一家委任公司的文件里有一条限制，限制我替另一家公司某类安排做介绍。"

周老板没有立刻说话。他的手指在桌面上敲了两下，像在算账，也像在衡量人心。

"那你现在跟我说，是想我怎样？"

"我可以把我能介绍的两份讲清楚，也可以告诉你这条限制会影响什么。"梁志明把那份复印件翻到相关段落，推到周老板面前，"如果你觉得需要比较第三家的安排，我可以建议你找另一位没有这个限制的人再问一次。你有权先知道我手上有什么边界，再决定信不信我的建议。"

窗外有叮叮车经过，铃声从皇后大道中一路荡进来。周老板看着那行英文小字，又看了看梁志明。过了好一会儿，他才把烟盒收回口袋。

"你们这些文件，真麻烦。"他说。

梁志明点点头，没有替文件辩解。

周老板又说："不过，麻烦归麻烦。你先讲清楚自己不能讲什么，我心里反而踏实一点。"

那天下午，梁志明只打开了蓝色和绿色两只箱。他没有把第三只箱说成坏东西，也没有假装它不存在。他只是把自己的限制摆在桌上，让周老板知道这张桌子上看得到的选择，为什么少了一只箱。

散会时，张小姐在门口递给梁志明一份重新打好的说明页。纸角夹着一枚小小的回形针，旁边多了一行她用铅笔写的字：给客人看的边界，也是在保护你自己。

梁志明把那页纸夹进档案。雨后的中环路面发亮，他没有多说什么，只在离开前回头看了一眼那间会客室。第三只箱仍在桌角，但这一次，它不是被藏起来的秘密，而是被说清楚的限制。

## 原文定义

如持牌个人保险代理或持牌保险代理机构代表超过一名获授权保险人行事，但与任何委任保险人之间的相关协议或委任条款限制其为另一名委任保险人的特定保险产品进行推广、提供意见或安排保单，该代理人或代理机构应向客户披露该限制，并解释在该限制下，其只会为该特定委任保险人的产品进行推广、提供意见或安排保单。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 桌上有三只文件箱，但第三只没有打开 | 代理人原本可能代表多家保险人，但某些产品受限制 |
| 梁志明说明自己不能介绍第三家公司某类安排 | 必须向客户披露委任协议或条款带来的限制 |
| 他把复印件翻到相关段落给周老板看 | 不只是口头带过，还要解释限制的含义 |
| 周老板可选择听现有两份建议，或另找没有限制的人比较 | 客户在知情后才能判断建议是否完整 |
| 张小姐写下“给客人看的边界，也是在保护你自己” | 披露限制既保护客户，也让代理行为更合规 |
