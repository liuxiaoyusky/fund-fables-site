# 3.3.4 保单对近因原则的更改

## 故事

陈伯的账房在上环楼梯街的尽头，靠着一扇能看见半个维港的窗。那天下午，他正在给一箱旧单据分类，冯船长推门进来，脸色不太好。

"陈伯，你帮我看一份单。"他把一份油渍斑斑的英文保单拍在桌上，"他们说不赔。"

陈伯把老花镜架正，一个字一个字地看。那是一份货运保单，专门保"顺发号"这趟船程的。条款里写着："直接或间接由碰撞、火灾、爆炸引起的损失，均在承保范围。"但紧接着又有一行小字："直接或间接由任何延误引起的损失，无论该延误是否由承保危险造成，均属除外。"

"上个月船在香港海域跟另一条船碰了，"冯船长说，"不算大事故，船壳破了点，补了三天。这三天里，我们运的那批日历错过了一月一日的交货期。买家说不要了，说一月一日之后这日历不值钱。"

陈伯抬起头："保单赔不赔这部分？"

"保险行说，延误是除外责任，'直接或间接'由延误引起的损失，不赔。"

陈伯把保单推了推，沉默了一会儿。

这时楼梯上又传来脚步声，是梁志明送文件路过。他看了一眼那份保单，说："'直接或间接'这几个字，在这里是把双刃剑。"

冯船长没听懂。

梁志明坐下来，指着那行小字说："如果保单写的是'由碰撞直接引起的损失'，那延误这部分可能还有得谈。但它写的是'直接或间接'——意思是不管碰撞离损害有多远，只要碰撞和损害之间有任何一层关系，这层关系就算在'间接'里，也被这条措词盖住了。"

陈伯接话："所以保险行引用这条，是说碰撞是受保危险没错，但碰撞引起了延误，延误又引起了市场损失，间接地，整件事都算在延误那条除外里头了。"

冯船长站起来，椅子腿在地上刮出一声闷响："那当初写'直接或间接'，不是骗人吗？以为什么都保，结果什么都除外？"

"不是骗人。"梁志明把保单翻到第一页，"这条措词的目的是：保险行想保船只本身在海上出事造成的直接损失，但不想保因为出事、延误、市场行情变化这类间接的钱。'直接或间接'在这里的意思是：从碰撞那一刻起，不管中间经过什么环节，只要是碰撞这条线上的，都算近因在碰撞。但延误本身又被单独拎出来，划成了除外。"

陈伯把那箱旧单据推到一边，慢慢说："打仗的时候，有个军官夜里在铁路上被火车撞死了。他的保单写明'直接或间接由战争引起的损失不赔'。战争只是个间接原因，火车才是直接撞死他的那个。但法院判了：保单措词里的'间接'，覆盖了战争和死亡之间的整条链条，间接原因也算近因，所以不赔。"

冯船长坐下来，把那份保单又拿起来，对着光线看那几个字。

"'直接或间接'。"他念了一遍，声音很轻。

陈伯说："下次签单之前，这些字要问清楚。'直接'和'主要地'其实是一样的意思，但加上了'间接'，整条就变了。"

窗外的渡轮汽笛响了一声。冯船长把保单折好，塞进内袋，站起来要走。走到门口，他停了一下，回头对陈伯说："那批日历，最后还是低价卖给了别家。亏了不少。"

陈伯没接话，只是把那箱旧单据重新拉回来，继续整理。账房的灯光昏黄，照在他那本发黄的账簿上。

## 原文定义

保单措词"直接或间接"：一个排除由某项特定危险"直接或间接"引致损失的除外条款，曾被法院解释为：即使除外危险的运作实际上只构成一个相对"近"而言较远的因，有关损失仍不获赔偿。即"间接"一词覆盖了从除外危险到损害的完整链条。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 保单写"直接或间接由碰撞引起"，但延误是除外 | "间接"一词扩大了近因的覆盖范围 |
| 碰撞→延误→市场损失，三层链条 | 间接原因也在"直接或间接"的覆盖范围内 |
| 陈伯引用军官案例 | 法院判例：间接原因构成近因时，仍适用除外条款 |
| 梁志明解释"间接"不是扩大保障，而是缩小 | "直接"和"主要地"含义相同；"间接"另有意图 |
