# 3.4.8 提供超额弥偿的保单条文

## 故事

周老板的海味铺那年也遭了殃。

不是火烧的，是贼。半夜有人撬开了后门的铁闸，把放干货的几口大木箱翻了个底朝天。鲍鱼、海参、干贝，被偷走了一半。

保险行来人查了现场，核算了损失，然后翻出一张旧保单，跟周老板说："按这张单子，赔你进货价。"

周老板指着货架说："进货价两年前的价。现在鲍鱼涨了三成，海参涨了四成，你们赔我两年前的价，我进货都进不回来。"

保险行的人说："保单写的是'实际价值'，实际价值就是扣除折旧后的价值，你用了两年，要扣损耗。"

周老板把那几口被翻乱的木箱踢了一脚，骂了一句。

后来还是梁志明出了主意。他把一份新的保单条款递过来，说："周老板，你听我说完。你这批货，下次可以保'重置保险'。重置的意思是：出了事，保险行赔你买新货的钱，不扣折旧。"

"那保费呢？"

"贵一点，但不多。"梁志明拿算盘打了一下，"你这批货，进货价十万，重置保费大概多一成。但要是出了事，赔你是十万零进货价的涨幅，不扣折旧。你自己算，哪头划算。"

陈伯在旁边插了一句："周老板以前做生意，总觉得买保障是花钱。现在你看，花在前面，省在后面。"

周老板把新条款看了一遍，又问："那有没有一种单子是，保的时候就定好一个价，出了事不管市场价怎么变，就按那个价赔？"

梁志明点头："有，叫'定值保单'。不过这种单子一般只用于价值高、不会折旧的东西，比如古董、字画、珠宝。你这批海味年年涨跌，不适合用定值单。"

"那我这批鲍鱼，用哪种？"

"'以新代旧'的那种。重置保险，专门针对你这种情况。"

周老板想了想，说："行。下次进货，按新价保。"

梁志明把这条记在单子上。

三个月后，又有人来撬铁闸。这次保险行按新条款赔了周老板，赔的是当时的市场价，一分没扣。

周老板拿到支票那天，在海味铺门口站了很久，看着伙计们把新进的货一箱箱搬进来。傍晚的日光照在鲍鱼的罐头上，反出一片金黄色。

陈伯路过，看见他站在门口，问他想什么。

他说："想明白一件事。买保障，不能只算眼前那点钱，要算长远。"

陈伯笑了笑，没接话，继续往楼梯街的方向走。身后海味铺的伙计们还在搬货，脚步声咚咚咚的，盖过了街上的电车声。

## 原文定义

提供超额弥偿的保单条文包括：(a) 重置保险：赔偿金额不因损耗和折旧而扣减；(b) "以新代旧"保障：同样不扣折旧，适用于家居和船体险；(c) 定值保单：保额基于专家估值，在全损时按约定价值赔偿，不考虑损失时的实际价值，适用于古董、珠宝等高价值不折旧物品。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 保险行说"按进货价，扣折旧" | 一般弥偿原则会扣除损耗和折旧 |
| 梁志明推荐"重置保险" | 重置保险：不扣折旧，赔买新货的钱 |
| "以新代旧" | 以新代旧：赔偿金额不因损耗折旧而扣减 |
| 周老板问"能不能按定好的价赔" | 定值保单：价值由双方预先约定，全损时按约定价值赔偿，不考虑市价变化 |
| 陈伯说"花在前面，省在后面" | 重置保险保费较高，但避免了折旧扣减的实际损失 |
