# 3.4.5 损余（Salvage）

## 故事

"顺发号"出事之后，冯船长在码头边上的茶餐厅坐了三天。

货舱里那批景德镇瓷器，十箱碎了六箱，剩下四箱磕磕碰碰，釉面上全是水渍和裂纹。瓷器行的张老板第一时间赶到码头，看了那四箱货，脸色比海面还难看。

"这四箱还能卖吗？"冯船长问。

张老板摇头："摆地摊或许有人要，三折都没人收。"

旁边保险行的核赔员插话："那我们按全损来算的话，这四箱也要算进去。"

冯船长正要说话，梁志明从人群中挤了过来。

"等等。"他把那四箱瓷器的发票翻出来，算了算，"四箱景德镇，进货价是六千块。四箱都泡了海水，釉面毁了，但瓷胎本身没裂。张老板，你最低愿意多少钱出手？"

张老板想了想："最多一千。搁在我仓库占地方，摆地摊还要搭人工，不划算。"

梁志明把数字报给核赔员："四箱的实际价值从六千跌到了一千，损失是五千。但一千块的残余价值，得从赔款里扣。"

冯船长听明白了，问："那就是说，赔我五千块，不是六千？"

核赔员翻开保单，指着其中一条："对。保单写的是'赔实际损失'，实际损失就是六千减一千。剩下这一千的东西归您处置，可以留着，也可以卖给我们。"

冯船长看了那四箱瓷器一眼。茶餐厅的灯光照在釉面上，水渍反着光，像碎掉的月亮。

陈伯在旁边说了一句："如果保单写的是另一种处理方式——保险行全额赔你六千，这四箱东西就归保险行了，他们自己处理。"

"那我怎么选？"冯船长问。

陈伯把算盘递给他："你自己算。拿五千，瓷器归你；拿六千，瓷器归保险行。你要是认识人能把瓷器卖个好价钱，就拿五千自己处理。你要是嫌麻烦，拿六千，让保险行去头疼。"

梁志明在旁边补了一句："不过冯船长，一般船东都选拿全额，把残货交给保险行。为什么？因为那四箱东西，在他们手上比在你手上值钱。"

冯船长低头看着那四箱瓷器。码头的海风把它们吹得轻轻晃动，叮当作响。

他想了一会儿，说："行，拿五千。"

张老板听到这句话，有些意外。旁边的人问他要不要接手那四箱，他想了想，摇了摇头。

冯船长叫人把四箱瓷器搬到船舱角落里，用油布盖好。临走的时候，他对梁志明说："下次出海，我还是少装点瓷器。"

梁志明笑了笑："船舱里放瓷器，近因怎么算都麻烦。"

船笛响了一声，冯船长上了"顺发号"，往维多利亚港的方向开去。日光落在甲板上，晒干了最后一点水渍。

## 原文定义

损余（Salvage）是指保险标的在受损后留下的剩余部分，若仍有经济价值，在计算弥偿金额时须将其价值扣除。处理方式有两种：(a) 从赔偿额中扣除损余价值，损余归被保险人所有；或(b) 保险人全额赔偿，损余由保险人收回并自行处理。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 四箱瓷器泡了海水，但瓷胎没裂，还有残余价值 | 损余是受损后仍具有经济价值的剩余部分 |
| 梁志明算出损失是五千（进货价六千减残值一千） | 计算弥偿时须从损失金额中扣除损余的残余价值 |
| 冯船长选择拿五千，瓷器归自己 | 方式(a)：从赔偿额中扣除损余，损余归被保险人 |
| 核赔员说"拿六千，瓷器归保险行" | 方式(b)：保险人全额赔偿，损余由保险人收回处理 |
