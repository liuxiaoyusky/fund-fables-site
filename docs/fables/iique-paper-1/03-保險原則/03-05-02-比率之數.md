# 3.5.2 比率之数

## 故事

观塘工业区的联安仓库前年换了老板，新东主是从事船运的冯船长。仓库不大，但码头工人多，火险一直是他心里的一块石头。

那年春节前，仓库底层的旧电箱短路，引发了一场小火。损失不大，五万块，但冯船长同时收到了两张保险公司的信件。

一封来自甲保险行，说：「我们的比率之数是四分之一，赔偿一万二千五百元。」

另一封来自乙保险行，说：「我们的比率之数是四分之三，赔偿三万七千五百元。」

冯船长拿著两封信去找梁志明：「点解佢话四分之一，佢话四分之三？边个话事？」

梁志明把两张保单和投保金额拿出来，在计算纸上画了一个简单的分数：「甲保险行保了你十五万，乙保险行保了你四十五万，总保额六十万。你出了五万蚊的损失，按比例——」

他在纸上写：15万/60万 = 25%，45万/60万 = 75%。

「所以甲赔四分之一，乙赔四分之三，加起来刚好是五万，一分不多一分不少。」

冯船长看著那个「1」字，忽然问：「如果三家保险行呢？」

梁志明笑了：「三家的比率之数加起来，还是等于一。」

陈伯那时正好从外面进来，听见这句话，接了一句：「整数好，账房最怕见到啲零散嘢，加埋唔岩就麻烦。」

冯船长拿著计算结果回仓库，路上经过码头，看见工人们正在卸货。他把那封信收好，没有张扬。五万蚊的损失，他拿了回来；但他心里多了一样以前没有想过的念头——原来一场损失，可以由几个人一齐找数，只不过几时边个出几多，系一早写好在张合约里面的。

## 原文定义

分担应用时，任何一位保险人对被保险人损失应负的最终部分，称为该保险人的「比率之数」。所有有关保险人的比率之数之和等于一（即损失的百分之一百）。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 甲、乙两家保险行各自计算份额 | 每个保险人有各自的比率之数 |
| 25% + 75% = 100%，刚好覆盖损失 | 所有比率之数之和等于一 |
| 梁志明用投保金额比例计算份额 | 比率之数由各家投保金额比例决定 |
| 冯船长最终拿了全部五万赔偿 | 不论分担方法如何，被保人最终获得全额弥偿 |
| 「三家保险行也一样加埋系一」 | 比率之数之和恒等于一（100%），与保险人数量无关 |
