# 3.5 分担（Contribution）

## 故事

湾仔茶餐厅里，三个老街坊坐在角落，各人出了一份火险，却烧了同一间屋。

陈伯先说：「我那份交了十五年，一分钱无呃过佢。」

周老板说：「我都系，十五年无出事，终于出事，佢就要走数？」

何师奶说：「三间保险行，三份保单，一间屋烧了，点算？」

三个人转头望向坐在旁边的梁志明。

梁志明喝了一口奶茶，说：「三个和尚没有水吃的故事，你们听过吧？」

三个人点头。

「三个和尚没水吃，是因为三个人都不想自己去打水。但分担这个原则，刚好相反——三个保险行都打了水，来到现场，发现原来每个人都只须打自己那份。」

梁志明拿出三张白纸，写了三个分数。

「屋烧了，损失是二十万。甲保险行保了十五万，乙保险行保了四十五万，丙保险行保了四十万，总共一百万。佢地三个，唔需要夹埋一百万，只需要夹埋赔二十万——按比例，甲出四分之一，乙出四分之三，丙补埋个零头。」

他在白纸上写：15/100 × 20 = 3万，45/100 × 20 = 9万，40/100 × 20 = 8万。

「三间公司，佢地各自话事，但赔的数加埋岩岩好系二十万。」

周老板拿著那张白纸看了很久，说：「原来分担的意思，唔系话几个人分担你的损失，而系几个人分担你一个人本来要追的数。」

陈伯笑著说：「咁就好啰，总好过你一个人走去追三间公司。」

何师奶说：「但系，如果其中一间话，我只赔我份，唔关另外两间事呢？」

梁志明说：「呢个就系衡平法则的用处。你唔使自己走去追，赔咗的那间，自然有权去向未赔的那间追番佢那份。」

三个人对望一眼，心里都踏实了。分担这个原则，说到底，就是不让被保险人做那个自己去追三间公司的可怜人。

## 原文定义

分担（Contribution）原则规定：当两份或以上保单同时对同一被保险人提供弥偿保障时，所有相关保险人按各自的比率之数分担损失，而非由单一保险人独自赔付。任何一个保险人履行弥偿后，有权向其他须负赔偿责任的保险人追讨分担。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 三个街坊各持一份火险，同一间屋烧了 | 重复保险：多份保单承保相同权益 |
| 三间保险行按比率分担二十万 | 分担原则：所有比率之数之和等于一（100%） |
| 周老板说「分担系几个人分担你一个人本来要追的数」 | 分担的核心：减轻被保险人向多方索偿的负担 |
| 赔了的那间可向未赔的那间追讨 | 衡平法则：已履行赔付的保险人有权追讨分担份额 |
| 三个街坊坐在茶餐厅倾偈 | 分担是日常商业社会的实际安排，非理论空谈 |
