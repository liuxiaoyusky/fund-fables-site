# 3.2.6 违反最高诚信的补救

## 故事

上环白记海味铺的周老板信奉一个道理：做生意靠街坊口碑，白纸黑字不过是走程序的过场。那年他盘下永安街一间旧铺位，顺道去洋行续保，开口只说"老规矩，跟去年一样"，却没提那铺子三年前着火燎过半边屋顶的事。

梁志明在保险行做了六年，什么客人没见过。他把保单草稿拿给周老板过目，周老板扫了一眼数目字，摇摇手说"得啦得啦"，在签名栏摁了拇指印。

转过年来，雨季一场雷暴，闪电劈中隔壁楼宇外墙，火舌顺着旧电线烧进白记的干货仓库。周老板第一时间冲到保险行拍桌子要索赔。梁志明翻开档案，发现了那处旧伤。

"周老板，你当年投保的时候，铺子有没有报过火警记录？"

周老板顿了顿，杯子在手里转了半圈："那间铺的事……那是很久以前的事啰，跟现在有什么相干？"

梁志明没答话，把档案送到理赔部。Mr. Thompson看完卷宗，只说了一句："这个情况，不止可以拒赔。"

他翻开保单第七页，指给周老板看一行小字：若被保险人或其代理人有任何不实陈述，本合约自始无效。

"意思就是，当作这份合约从来没有签过。"Thompson放下文件，"保费原则上可以退，但要看具体情况。"

周老板脸色变了。他以为最多就是赔不赔的问题，没想到连合约本身都可以不算数。

梁志明在一旁听着，想起师傅教他的话：保险是信人的行当，人家信你报了实话，你收了钱写了字，到出事的时候才发现有人藏了一角——这行就做不下去了。

"我不是故意呃你。"周老板放下茶杯，"我真的觉得那是旧账，不关新保单的事。"

"法律上不这么看。"Thompson合上卷宗，"未申报的重要事实，和故意隐瞒，在后果上对合约的杀伤力是一样的。"

后来周老板补交了说明材料。保险行退了保费，但没有理赔。Thompson告诉梁志明：如果周老板当年是故意隐瞒，那就连保费都未必退得回来，还可以另外追讨赔偿。幸好只是一时疏忽。

梁志明把这件事记在本子上。后来有客人问他"不说行不行"，他就把白记的故事讲一遍。

## 原文定义

若最高诚信责任遭违反（不论是欺诈性或疏忽性），受屈的保险人可自始推翻整个合约；若属欺诈性违反，还可额外提起侵权诉讼追讨损害赔偿。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板漏报旧火灾记录 | 违反最高诚信——未披露重要事实 |
| Thompson说"合约自始无效" | 补救(a)：撤销合约 |
| 疏忽漏报可退保费，故意隐瞒不退 | 欺诈性违反与疏忽性违反的区别对待 |
| Thompson说"还可追讨赔偿" | 补救(b)：侵权法损害赔偿诉讼 |
