# 3.6.2 如何产生

## 故事

观塘工业区有一间小型制衣厂，老板叫陈伯。说是老板，其实是从账房一步步熬上来的老文员，退休后接手了女婿留下的小厂。厂里专做来料加工，订单赶得急，机器三班转。

那年冬天，隔壁装修的电工用风焊烧穿了共用墙，火势蔓延过来，缝纫机、裁床和一整批赶给美国客人的冬装全部烧毁。

陈伯赶到的时候消防队刚收队。他在厂里做了三十年账，从没见过这么大的场面。保险公司的梁志明第二天就到了，验了现场，核对发票，十几天后赔了款。

但梁志明没有就此结案。他跟陈伯说："陈伯，这把火是隔壁电工疏忽引的，我们保险公司赔了你，按规矩，可以替你去找那个电工和负责装修的工程行追讨。"

陈伯想了想："隔壁那是租户，租约里写明了不准用明火，是他们违反合约。我这个业主本来就可以告他们。"

"对。"梁志明说，"这就是合约上的追偿权。我们赔了你，就可以用你的身份去追他们。"

陈伯又问："如果那个电工不是租户，而是我们厂的正式工人呢？"

"那就要看是不是在工作时间、是不是因为雇主的责任受的伤。"梁志明翻了翻文件，"按《雇员补偿条例》，雇员受伤，雇主得赔，雇主赔了之后，有权去找真正该负责的第三方。如果我们有替你付雇员补偿，也可以代雇主去追。"

陈伯点了点头，忽然想起女婿当年就是从这间厂的三楼窗口坠落受的伤。他没有再说话，只是把租约副本和工程单据都找了出来。

后来梁志明告诉他，那把火的追偿，走的是侵权法这条路：第三者疏忽引致损失，保险公司赔了被保险人，就有权以被保险人名义起诉那个疏忽的人。陈伯听完，慢慢地说："四条路，都是同一个理——有人赔了你，你赔的那个人，就该去找真正做错事的那一方。"

## 原文定义

代位权可通过以下四种方式产生：（a）侵权法：第三者疏忽引致属保单保障范围内的损失，保险人赔偿后以被保险人名义起诉；（b）合约：被保险人对造成损失的另一方有合约权利，保险人赔偿后代为行使；（c）法规：例如《雇员补偿条例》赋予雇主对造成雇员受伤的第三者的代位权，须移交予已付款的保险公司；（d）损余：保险人对保险标的的剩余部分享有代位权。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 隔壁电工疏忽引发火灾，保险公司赔陈伯后代位追讨 | 侵权法上的代位（第三者疏忽） |
| 陈伯作为业主根据租约追究租户违反禁止明火条款的责任 | 合约上的代位 |
| 雇员补偿条例下雇主将向第三方追偿的权利移交给保险公司 | 法规下的代位 |
| 保险公司保留对烧毁物资中残余部分的权利 | 损余中的代位权 |
