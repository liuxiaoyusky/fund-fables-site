# 3.2.3 重要事实（Material Facts）

## 故事

周老板坐在保险行的办公室里，对著投保单发愁。整页纸都是问题，他每填一个，心里就多一个疑问。

「我间仓库有自动洒水系统，」他指著投保单上某一行，「这个要申报吗？」

梁志明把投保单拿过来看了看。「这个——如果申报了，会降低保费。」

周老板不明白：「那岂不是应该报？我报了保费仲平，点解要纠结？」

梁志明笑了一下。「周老板，你听我解释。在我们的原则里，有一类事实，投保人就算不主动披露，也不算违反最高诚信。」

他翻开手册，找到一页：

「第一，常识范围内的事实。比如氢气会爆炸——任何一个正常人都知道，不需要特别说明。你投保单上不需要写『我间仓库有氢气』，因为这是众所周知的。」

周老板点头。

「第二，保险人已知或应被认为已知的事实。比如索马里的海盗问题——这是行内普遍知道的，保险人在评估航运风险的时候，已经把这个因素算进去了，唔需要你特意提起。」

周老板又点头，但眉头皱得更紧了。「那自动洒水系统呢？」

梁志明把投保单轻轻放在枱面上。「自动洒水系统是一项『可以降低风险』的事实。法律规定，如果某个事实一旦被披露，反而会让保险人觉得风险更低——也就是说，说了反而对投保人有利——那么投保人漏报这个事实，不构成违反最高诚信。」

周老板愣住了：「也就是说，我有洒水系统但漏报了，也没有问题？」

「严格来说是这样。但我建议你还是报，」梁志明说，「因为保险是一个双方共同建立信任的过程。如果你现在不报，以后出事，你就要花时间解释为什么明知有洒水系统但冇告知。麻烦。」

周老板想了想，把笔拿起来，在那一栏打了个剔号。

梁志明这才接着说：「另外，重要事实本身分为三类。第一类：影响保险人系唔系接纳你的投保——比如你间仓库上个月被海关查封过。第二类：只影响保费高低——比如你的仓库在工业区，保费本来就贵一些。第三类：两者均有影响——比如仓库本身的结构有问题。」

周老板把投保单填完，签了字。梁志明收了文件，站起来送他出门。

「梁生，」周老板临走前回头，「你刚才话有洒水系统不报也没问题——但你建议我报。我最欣赏就系你这种人，唔系照本宣科。」

梁志明笑着摆了摆手：「我只是不想你出事之后再解释。」

## 原文定义

(a) 重要事实：「在订定保费或决定是否承担风险方面，会影响一名审慎的保险人的判断的任何情况。」可分为三类：只影响是否接纳风险、只影响保费，以及两者均影响。(b) 以下三类事实无需主动披露：常识范围内的事实；保险人已知或应被认为已知的事实；以及减低风险状况的事实。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明解释三类无需披露的事实 | 重要事实的消极例外：常识已知、风险降低 |
| 自动洒水系统：漏报不算违反 | 减低风险的事实：即使影响保费，也不强制披露 |
| 仓库被海关查封 vs 仓库在工业区 | 三类重要事实：接纳风险决定、仅保费决定、两者兼具 |
| 梁志明建议主动报，但不强制 | 法律义务与商业信任的分别 |
