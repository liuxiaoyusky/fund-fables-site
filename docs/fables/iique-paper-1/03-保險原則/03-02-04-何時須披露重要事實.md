# 3.2.4 何时须披露重要事实

## 故事

去年，冯船长的儿子替他在一间本地保险行买了一份人身意外保险，保单于1981年11月1日生效。冯船长没怎么看条款，儿子说签就签了。

今年3月，公司安排船员做例行身体检查。验血报告出来那天，冯船长正在船上检查货仓。医疗船的姑娘打电话到船上，说超声波显示他的脾脏有一个阴影，需要进一步检查。

冯船长起初不以为意，觉得自己身体好得很。 后来姑娘的电话再打来，说专科建议做手术切除化验，他才开始担心。

他想起那份保险。

「如果我去做手术，」他对梁志明说，「这笔费用，保险赔不赔？」

梁志明翻开保单，看了很久。

「冯船长，」他说，「你的保单是去年11月1日生效的。你那个验血的结果，是今年3月才知道的。在普通法的原则下，合约订立之后，你才得知的重要事实，没有义务向保险人披露。」

冯船长愣了一下：「也就是说——我不用申报？」

「不用。这是一个关键的分界线。」梁志明说，「合约订立之前的披露是你的义务；合约订立之后，你发现了新情况，那是另一回事。」

「那手术费用呢？」

「这个要看你保单的条款，」梁志明如实说，「一般的人身意外保险，如果有『既有疾病』的除外责任——就是投保前已经存在的疾病不包的条款——那保险人可能会引用这一条拒赔。但这不是因为你违反了诚信，而是因为你的病情在投保前就存在，属于保单条款的范围。」

「那我应该点做？」

「如果你的手术和这个脾脏问题有关，你最好在续保之前想清楚，」梁志明说，「因为续保的时候，最高诚信的义务会重新出现——你要再次申报当前的健康状况。如果你瞒着不报，续保之后被发现，就算普通法下你有道理，保单条款也会令你处于被动。」

冯船长把这句话记住了。后来的手术化验结果出来，是良性的，他恢复得很快。续保的时候，他如实申报了那一段就医纪录。保险人看完，没有加价，也没有拒保，只是在保单上附了一个小小的批注：「既有疾病除外」。

冯船长把这个批注给梁志明看。梁志明说：「呢个就系制度的可爱之处——它唔系完美的，但它给了双方一个机会，把规矩摆在明面上。」

冯船长没有说话，只是把那张保单收好，继续跑他的船。

## 原文定义

(a) 在普通法下，投保人或其代理人须在保险合约订立之时披露重要事实；合约订立后才得悉的重要事实，无需披露。(b) 一些非人寿保单要求保险期内风险变更须予披露。(c) 保单续保时，须重新履行最高诚信义务（人寿保险不要求在接近保单周年日时再次主动申报健康状况）。(d) 合约更改时，最高诚信义务适用于更改事项。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 冯船长的体检结果在保单生效后才出来 | 普通法下的时间分界：合约订立前的披露是义务，成立后获悉的事实无需披露 |
| 手术费用可能因既有疾病条款被拒 | 保单条款中的「既有疾病」除外责任：不等同于违反诚信 |
| 续保时如实申报就医纪录 | 续保重新触发最高诚信义务 |
| Thompson在保单上附加除外责任批注 | 保险人的核保权利：知情后可附加条件 |
