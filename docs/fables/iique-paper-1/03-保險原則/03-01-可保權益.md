# 3.1 可保权益

## 故事

那年秋天，有个年轻人敲开了陈伯在上环的办公室门。他是阿明，在观塘一家制衣厂做师傅，太太刚生了第二个孩子，他想做一份人寿保障，但有些事想不通。

「陈伯，」阿明开门见山，「我太太为什么可以为我投保？我是她的丈夫，这够不够？」

陈伯把老花眼镜摘下来，慢慢擦了擦。

「『权益』呢个词，你不要想得太复杂，」他说，「在保险里面，它只有一个意思——你和那样野之间，有没有一种财务上的关系。」

他站起来，走到窗边，维多利亚港的船在远处鸣笛。

「比如你，你和太太之间当然有财务关系。你走了，她的收入就断了；你还在，她有你的薪水维持生活。这种『你出事她就会蚀钱』的关系，就是一种权益。」

阿明点点头，似乎懂了。

「再具体一些。你有一间屋，按揭给了银行。如果间屋烧了，银行就拿不回完整的抵押品，银行就会有损失。所以银行对那间屋有权益。船公司对船有权益，工厂对机器有权益，道理是一样的。」

陈伯走回桌边，拉开抽屉，最上面是一叠泛黄的航空邮简——伦敦寄来的。他看了一眼，把邮简放回原处。

「我再问你，」他对阿明说，「如果你是我的租客，你能不能帮我的辨公室投保火险？」

阿明想了想：「不能吧，我又没有那个单位。」

「正确，」陈伯说，「因为你和那个单位之间没有直接的财务关系。间屋烧了，你最多没地方做生意，你的损失是间接的，不是直接的财产损失。你没有资格为它投保，否则你就是拿别人的财产来博彩——出事之后，你损失了什么？什么都没有。」

阿明终于明白了：「所以保险不是赌博，道理就在呢度——你必须有野可以损失，才可以用保险去 cover 呢个损失。」

陈伯微微笑，把那叠文件合上。「你搞清楚呢个问题，后面所有的条款，你就知道点解要咁写了。」

阿明站起来，鞠了一个躬。走出门口的时候，他回头看了一眼——陈伯又打开了抽屉，把那张航空邮简拿出来，对著维港的暮色，看了很久。

## 原文定义

「权益」一词在这里是指一种与某物件或某人之间的财务关系。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 阿明作为丈夫可以为自己投保 | 夫妻之间有财务依赖关系，构成可保权益 |
| 陈伯解释银行对抵押房产有权益 | 债权人对抵押品有财务关系，构成可保权益 |
| 租客不能帮别人的单位投保火险 | 无直接财务关系，不能投保，否则构成赌博 |
| 陈伯看伦敦寄来的航空邮简 | 感情暗线：遗憾往事呼应「守护」的保险主题 |
