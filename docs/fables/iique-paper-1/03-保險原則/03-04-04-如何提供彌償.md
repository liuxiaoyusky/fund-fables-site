# 3.4.4 如何提供弥偿

## 故事

何师奶的茶餐厅在观塘开了十五年。那天她的雪柜突然坏了，压缩机烧了，冰箱里的冻肉全部报废。

她打给梁志明，问保险能赔多少。

梁志明来了一趟，看完现场，递给她一张理赔表。何师奶接过来一看，愣住了——表格上有四种赔法，她不知道选哪个。

"第一种，赔现金。保险公司算好你这批冻肉值多少钱，直接开支票给你。"

何师奶点头："这个我懂。"

"第二种，帮你修。压缩机修好，冰箱继续用，不用你出一分钱。"

"也挺好。"

"第三种，给你换一台新的雪柜。"梁志明顿了顿，"不是修修补补那种，是一台全新的。供应商给保险公司的价格比市价低，所以有时候换新的比赔现金还划算。"

何师奶想了想："那我这台旧的怎么办？"

"残值算进赔款里，你把旧机给保险公司也行，自己留着折价也行。"梁志明说，"第四种是恢复原状，就是让受损的东西回到出事之前一模一样状态——雪柜恢复成好的状态，不扣折旧。"

何师奶把四种方式来回看了三遍，最后选了第三种，换新机。

过了几天，周老板听说了这事。他在电话里问梁志明："那水险呢？跑船的也能选这四种？"

梁志明说："周老板，水险保单对这四种方式没有明文规定。船出事，保险人只能赔现金，不能说给你换一条船，或者帮你修。这是规矩。"

"为什么船不行？"周老板问。

"船比较复杂，每条船都不一样，没有现成的货可以换。而且有些船东自己有自己的修船厂，保险公司没办法统一安排。所以水险保单干脆写明：赔现金。"

周老板想了想："所以四种方式里头，有的保单让客人选，有的保单只准赔现金。"

"对。"梁志明说，"财产险的保单怎么写，你就怎么拿钱。水险保单没写的，一律折现。"

何师奶的新雪柜送来那天，她让伙计把旧的压缩机拆下来，装在一个纸箱里，放在后巷等人来收。梁志明路过看见了，没说什么，只是笑了笑。

她省下了买新机的差价，旧的压缩机还卖了三百块。

## 原文定义

财产保险单通常订明保险人可以按照现金支付、修理、更换、恢复原状四种方法中之任何一项偿付损失；但水险和非财产险保单对此没有明文规定，保险人有义务以现金方式结清有效索赔。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 何师奶选择换一台新雪柜 | 更换（Replacement）：给予替换物件 |
| 保险公司帮她修压缩机 | 修理（Repair）：支付修理费用 |
| 赔冻肉的现金 | 现金支付（Cash payment） |
| 恢复原状不扣折旧 | 恢复原状（Reinstatement）：恢复到受损前状态 |
| 水险只能折现 | 水险/非财产险保单对方式无规定，须以现金结清 |
| 旧压缩机算入赔款 | 残值处理：更换时残值需计入 |
