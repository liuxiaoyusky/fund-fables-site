# 3.4.9 实际执行弥偿时所面对的难题

## 故事

上环永安街的海味舖「广记」开了三十年，周老板在那一带街坊心里，就是「靠得住」三个字的化身。他卖的瑶柱、虾干、鲍鱼，个个客人摸得著、闻得出好坏。做生意靠的是信任，银货两讫，一把秤准头从不欺人。

那年冬天，腊味舖的电线走火，火舌顺著风向蔓过半条巷子，广记的后仓烧了个精光。周老板赶到时，只见满地焦黑的紫菜和发涨的瑶柱盒子，二十箱上等干鲍鱼没有一箱完整。

他站在灰烬里，点了一支烟，然后径直走去中环的英资保险行找Mr. Thompson。

「我那批货，」周老板把一叠入货单拍在桌上，「两礼拜前刚从澳门补回来，市价多少你们自己查。烧了这么多，你们说该怎么赔？」

Mr. Thompson推了推金丝眼镜，把入货单翻了三遍，然后慢条斯理地说：「周先生，按照合约，我们的赔偿金额是根据『实际现金价值』计算，不是市价，也不是入货价。」

「什么叫实际现金价值？」

「简单来说，就是物件烧掉时，市场上认为它值多少。一样东西放了几年，难免有折旧。」

周老板皱起眉头：「我的鲍鱼两礼拜前才购入，你说折旧？」

「我们需要评估师的报告。」

梁志明当时坐在角落做会议纪录，他是保险行的本地伙计，经常替Mr. Thompson把英文合约解释给街坊听。他见气氛僵了，便站出来：「周老板，不如我先带Mr. Thompson去看看货仓现场，大家当面倾，倾完再决定评估师的人选。」

评估师是保险行找的独立行头老行尊，验完货之后给了个数：二十箱受损鲍鱼，按折旧率算，现值是投保额的七成。

周老板在茶餐厅里对梁志明说：「他说折旧就折旧，我几十年人，烧了两礼拜的货，点解要当旧货计？」

梁志明说：「道理是这样的——你当年买这批货，是希望出事之后有人替你找回那笔钱，但制度设计上，它只认你『当下』失去了多少，不是你当初花了多少。」

周老板沉默了很久。

最后他拿了七成的赔偿，亲手把灰烬铲进铁桶。回家的路上，他没有坐车，沿著德辅道西走得很慢。三十年的海味舖，烧了两礼拜的货，到头来，制度只肯还他七成。他不怪任何人。他只是觉得，做了这么多年街坊生意，忽然间，连「值多少」都要跟一个外人解释，心里有点空。

梁志明后来把这件事记在记事本里，旁边写了一句：「补不完的，都是人心。」

## 原文定义

弥偿原则的执行面对以下难题：（a）全损时，赔偿金额是否以投保额为准；（b）当财产存在折旧，索偿人与理赔人员在折旧金额上容易产生分歧，而多数索偿人认为自己的财产并无或仅有轻微折旧。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板认为货物的价值等于入货价 | 索偿人倾向以原始成本计算损失 |
| Mr. Thompson 坚持按「实际现金价值」赔偿 | 保险人执行弥偿时须考虑折旧 |
| 独立评估师给出七成现值 | 折旧金额是常见争议点 |
| 周老板最终接受七成赔偿 | 弥偿原则要求被保人不能从损失中获利 |
