# 1.1.3 风险管理(Risk Management)

## 故事

上环的秋天，空气里混着咸鱼和虾酱的味道。永乐街上那块被台风打歪又扶正的老招牌，已经挂了三十多年。招牌下面，是周老板的合记海味舖。

周老板记得上一次仓库进水是哪一年。那是五十年代的事，台风把海水灌进了一楼。那之后，他在后巷修了条排水沟，在门口垫高了三寸门槛。二十多年了，他觉得自己已经摸透了这片海的脾气。

梁志明不一样。他在上环一间本地保险行做伙计，每天的工作就是和人谈"出事"这两个字。合记海味舖他来来回回走了六年，每次都带着新写的建议书进门，每次都被周老板那杯凉茶打发走。

"Uncle Chow，我今天不是来讲那个的。"梁志明在门口站定，没有掏文件，"我就想问你一件事。"

周老板把茶杯放下，等他说。

"去年九月那次台风，你还记得吧？"

怎么会不记得。那次台风不算大，但正好撞上天文大潮，海水从后巷的排水口往上涌。合记一楼浸了半尺水，损失了十几箱鲍鱼和瑶柱。周老板带着伙计抢救了三天，最后还是赔了三千多英镑——那是他当时账面上大半的流动资金。

"那次不是最坏的。"梁志明说，"我查过天文台的记录，如果潮位再高半尺，你那批顶级的南非鲍会全部报销。按你现在的库存，损失会超过五千英镑。"

五千。周老板心里咯噔了一下。他没跟任何人说过那次实际赔了多少。

"你以前是怎么过来的？"梁志明问。

"顶过去。"周老板说，"人和货，能救就救。赔了的，咬着牙再赚回来。"

"有没有想过，有些事情不一定非要自己咬着牙？"

梁志明从口袋里摸出一张折了几折的纸，展开，铺在柜台上。那是一张表，不是保单，是周老板自己的账——进货、出货、损耗、灾害记录，整整齐齐列了五年。

"你看这里。"梁志明指着其中一栏，"过去十年，你因为台风和水浸造成的损失，加起来有两万二千英镑。平均每年两千二。但中间有三年是平的，剩下七年基本没事。你觉得这是运气好，还是每年都在赌？"

周老板没说话。梁志明也没等他回答。

"风险管理，就是把这件事想清楚、做明白。不是买不买哪家的纸的问题。"他说，"首先是识别——你知道有风险，知道哪些风险。然后是量化——大概会损失多少，多久一次。接下来是应对，应对有五种方法。"

"五种？"周老板皱了皱眉。

"第一种，避免。不做某件事，就不会有某种风险。比如你不在一楼存贵价货，就不会有水浸的损失。"梁志明说，"第二种，防范。就是你说的那些排水沟、门槛，把损失发生的概率降下来。第三种，减损——万一出了事，怎么让损失小一点。"

"那第四种呢？"周老板问。他其实知道自己会听到什么。

"第四种叫转移。把损失的风险转给别人一部分。比如签一份合同，让船公司在货物上船之前负责。或者——"梁志明看了他一眼，"买一份保障，万一出了事，有人在钱上面帮你分担。"

"我不买那个。"周老板摆了摆手，"买那个就是把自己的命交给别人。"

"第五种是自己掏钱。"梁志明说，"但不是乱掏。是算清楚之后，每一年存一个数字，平摊掉那些意外带来的波动。这叫风险融资。"

周老板不说话了。梁志明也没再劝。

那天傍晚，何师奶的茶餐厅里，周老板跟何师奶提起这件事。何师奶在柜台后面擦着玻璃杯，听着，没插嘴。

"他跟我算了笔账。"周老板说，"说我不是运气好，是在赌。"

何师奶把玻璃杯放回架子，轻声说了一句："有些事情，想清楚了才去做，不是因为不相信自己。是因为下面还有一班人。"

梁志明回到铺子里，把建议书收进抽屉。那份建议书不只是一张纸——背后还有林若兰帮他整理的英文条款，每一条都标了注释，红笔的字迹很细。台风季还没来。有些事情，慢慢谈也不迟。

## 原文定义

风险管理的核心是对纯粹风险进行系统性管理，包含三个步骤：**识别**（Identify）风险来源、**量化**（Quantify）损失频率与幅度，以及**应对**（Respond）。应对工具主要有五种：风险**回避**（Avoidance）、损失**防范**（Loss Prevention）、损失**降低**（Loss Reduction）、风险**转移**（Risk Transfer）和风险**融资**（Risk Financing）。保险只是风险转移工具的一种，而非风险管理的全部。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 台风导致仓库水浸，货主遭受经济损失 | 纯粹风险（Pure Risk）：只会带来损失，不会有收益的不确定性 |
| 梁志明整理五年账目，量化灾害历史数据 | 量化（Quantify）：用历史数据估算损失频率与幅度 |
| 周老板在一楼存放贵价货，水浸风险始终存在 | 风险识别（Identify）：明确知道风险来源但未加以管理 |
| 不在一楼存贵价货 → 没有水浸损失 | 风险回避（Risk Avoidance）：以不从事某活动来彻底消除某风险 |
| 修排水沟、垫高门槛 → 降低水浸发生概率 | 损失防范（Loss Prevention）：降低损失发生的概率 |
| 转移给船公司的货物责任条款 | 风险转移（Risk Transfer）：通过合同将损失责任转嫁给对方 |
| 算清楚后每年存一笔钱，平摊意外冲击 | 风险融资（Risk Financing）：自保基金等工具应对残余风险 |
| 何师奶说"下面还有一班人" | 风险管理的动机：不是为了自己赌赢，而是为了不让身边的人受影响 |
