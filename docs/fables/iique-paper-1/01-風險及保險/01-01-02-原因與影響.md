# 1.1.2b 原因与影响

## 故事

上环干诺道西那排石头外墙的货仓，是周老板的父亲那一辈建的。周老板在这条街上做了二十年生意，靠的是看货、识人、准时发货。他信自己的眼睛，不信白纸黑字——尤其不信那种每年要交钱、出了事却要吵半天的文件。

梁志明是楼下那间本地保险行的年轻伙计，经常在这一带的货仓和商号之间走动。他个子不高，走路快，说话不快不慢。街坊叫他"志明"，他总是笑着应。这天下午，他又来到周老板的货仓，说是要例行检查。

"又来？"周老板站在二楼铁皮屋顶下，手里还攥着算盘。这批即将装船的干鲍鱼和海参，他刚跟几个南洋商行谈完价，正盘算着利润。

"例行看看。"梁志明打开手电筒，逐一检查通风口和墙角，"阿叔，最近有没有动过电线？"

"没有，没有。"周老板不耐烦地挥了挥手，"我做了二十年海味生意，老鼠咬电线这种事，唔使你教。"

梁志明没接话。他仔细绕完了一楼和二楼，货舱里干货堆放整齐，通风口也没什么问题。正要走，忽然停住脚。

"阿叔，你们那台货梯，上次年检是什么时候？"

周老板愣了一下。"货梯？"

"上次我来，没有看到维修记录。今天也没有。"梁志明的语气还是平平的，"如果货梯出了问题跌落去，伤了人或者坏了货，这笔账——"

"我买过保单了。"周老板打断他，"你们保险公司出的，字写得密密麻麻那一份。"

"那一份包的是楼宇结构火灾和第三者责任。"梁志明顿了顿，"货物本身和机器设备，要另外算。"

周老板的脸色变了。干货生意靠的是存货变现，一箱鲍鱼出了问题，整批货的资金就周转不过来。他压低声音问："那——我现在补一份，来得及吗？"

梁志明看着他，想了一会儿，说："来得及。机器设备单独加个条款，货舱这边加个水渍险。保费另算，但出了事，这两笔账都能追回来。"

周老板签了字。

那之后过了个把月，梁志明再来回访，周老板的货仓已经多了几项新记录：货梯的维修单、灭火器的更换标签、还有一份加厚版保单。周老板嘴上还是说保险麻烦，但见人就说志明那个后生"还算靠得住"。

那年八月，一宗意外还是发生了。深夜，隔壁货仓的电线短路，火苗窜进了公共走道。周老板的货仓被波及，一楼铁门烧得变了形，存货从门缝里冒出的烟熏坏了两百多箱干鲍。损失不小，但不多久，保险公司的人就到场了——梁志明拿着理赔单，和周老板一条条核对：楼宇结构一份，货物水渍一份，机器设备一份，账目清晰，赔款到账。

周老板拿到支票那天，在茶餐厅请陈伯和梁志明喝奶茶。

"还好买了。"他难得低声说了一句。

陈伯坐在旁边，拨了拨杯子。他在这条街上做了三十多年账房，见过太多人出事之后才后悔当初没做准备。

但他心里还有一个问题，一直没问出口。

那天喝完茶，梁志明正准备走，陈伯叫住了他。

"志明，有件事我想问。"陈伯压低声音，"那年——六六年的风灾，我还在账房学生意。整个上环码头全浸了，沿海的货仓有一半都冇了。那时候周老板的父亲，也买过保险，你说——那份保险赔不赔？"

梁志明站住了。

他没有立刻回答，而是在茶餐厅门外站了一会儿。外面的街道上，街坊们照常搬货、叫卖、推手推车。看起来一切都稳稳当当。

"陈伯，"他终于开口，"那年整个码头被淹，不止周老板一家，是整个区——几十间货仓、几百户人家、连修好的路都被冲断了。这种事，在保险的行业里，叫'基本风险'。"

"意思是——？"

"意思是，这件事没有'数'可以算。"梁志明的声音很平，"一件货烧了，赔两百箱；一台机器坏了，赔一台。这些损失，能数得清，能分得给每家每户按比例承担。但当整个社会都被卷进去——台风把整个海岸线摧毁，战争把整个港口炸平，饥荒让整个地区颗粒无收——这种损失，没有人能数得清，也没有哪家保险公司能赔得起。"

陈伯没说话。

"所以，保险公司能保的险，叫'特定风险'。"梁志明继续说，"一件事发生了，只影响个人，或者少数人；后果严重，但范围有限。这种风险，可以统计，可以定价，可以分摊——所以商业保险公司愿意接。"

他指了指周老板那间刚修好门的货仓："像这次电线走火，烧了周老板一间货仓，这就是特定风险。隔壁几家货仓都没事，范围有限，损失能算，赔得起。但六六年的那场风灾，烧的不是一间——是整个海岸线。这种事——"

"保险公司也救不了。"陈伯轻声接了一句。

"救不了。"梁志明点点头，"不是不想，是没有这个能力。账算不过来。"

茶餐厅里的收音机正在播天气预报：菲律宾以东有一个热带气旋正在形成，下周可能影响华南沿岸。街坊们照常忙碌，没有人特别在意。

陈伯把喝完的茶杯收进桶里，对梁志明说了一句："知道了。有些事，要靠老天；有些事，可以靠自己。你们保险公司——能做的，就是把'靠自己'这一段，弄清楚。"

梁志明愣了一下，然后笑了。

他没有接话，只是把当天的理赔文件收进公文包，起身穿过街市，往下一间货仓走去。

## 原文定义

风险按影响范围可分为特定风险和基本风险两类。特定风险（Particular risk）指后果仅影响个人或少数人、范围有限的风险，如单车意外、火灾等，商业保险公司可对其定价和承保。基本风险（Fundamental risk）指成因超逾个人或小群体所能控制、影响不计其数的人、甚至波及整个社会或人类的风险，如大规模洪灾、战争、饥荒等，商业保险公司在财务上无法以商业方式处理此类风险。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 电线走火波及周老板一间货仓 | 特定风险：范围有限，仅影响个人或少数人 |
| 两百箱干鲍损失可统计，保险公司按条款赔付 | 特定风险：后果可量化，可分摊，商业保险公司可承接 |
| 六六年整个上环海岸线被台风摧毁 | 基本风险：成因超逾个人所能控制，影响遍及全社会 |
| 台风导致整个海岸线损毁，保险公司无力赔偿所有人 | 基本风险：损失无法统计，不可分摊，商业方式不可行 |
| 梁志明解释"特定风险有'数'可算，基本风险没有" | 商业保险只承接可量化、可定价的特定风险 |
| 天气预报中提及的热带气旋正在形成 | 暗示基本风险的成因（自然灾害）超越个体掌控 |
