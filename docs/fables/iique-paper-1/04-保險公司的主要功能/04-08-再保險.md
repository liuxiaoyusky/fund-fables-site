# 4.8 再保险(Reinsurance)

## 故事

周老板在文咸西街做了三十年生意，什么风浪没见过。1982年的冬天，他却第一次觉得事情有些不对。

那年他接了一船发往伦敦的干瑶柱，总值六十多万港币——那是他上环整间铺子加上存货抵押才凑出来的数目。保单是梁志明经手的，梁志明替他在英资公司买的是全损险，周老板觉得安心。

船开出维港第三天，传来消息：新加坡海峡外有一艘货轮沉了，疑似是他那船货。周老板手里捧着电报，半晌说不出话。

六十多万。他全部身家。

梁志明那天刚从上环坐叮叮车赶到中环的总公司，手里的文件夹还带着街坊茶餐厅早餐的油烟味。他直接上了三楼，敲开Mr. Thompson的门。

"Thompson先生，周老板那船货可能出事了。"

Mr. Thompson坐在打字机前面，摘下老花眼镜，表情没有太大波动。他看了看桌上的索赔预估数字，然后平静地说："志明，坐。这个数目，我们不会赖账。"

梁志明坐下来，心里稍微松了一点。

"但是，"Thompson接着说，"你知不知道为什么我们敢接这张单？"

梁志明摇头。六十多万，这个数字对一间本地海味铺来说是天文数字，对一间保险公司来说也不算小。

Thompson拉开抽屉，拿出一份英文合约。"这张单，我们签出去之前，已经分了一部分出去。有一间伦敦的再保险人，已经替我们承担了这笔风险的一半以上。所以这笔赔偿，我们付得出，赔得出。"

他顿了顿，补充道："我们保险人，也需要有人替我们兜底。否则接一张单赔一张大单，公司再多钱也不够用——而且大部分钱都在投资里，短时间拿不出那么多现金。"

梁志明低头看那份合约，密密麻麻的英文条款，条款里清楚写明了超过某个数额的索赔，再保险人可以直接赔付给原保险人。

"所以，"Thompson把合约合上，"保险人把风险再转移出去，叫再保险。周老板的保单他照常索赔，我们照常赔付，但背后有另一间公司在撑住我们。这跟我们替客人做保障是一样的道理——只是这一次，保障的是我们自己。"

周老板后来顺利拿到了赔偿。他不知道背后有再保险人，也不知道那间伦敦公司的名字。他只知道出事之后，保险公司没有拖、没有赖、没有说资金周转困难。

梁志明也一直没有告诉他。

那张英文合约压在Thompson抽屉最底层，纸张已经有些发黄。周老板的六十多万，从账上划走的那一刻，公司里没有人需要慌忙套现投资，也没有人需要向上环的银行紧急融资。

再保险人已经把钱准备好了。

## 原文定义

再保险是指保险人根据一个或以上的保险合约所承担的风险，可以完全或部分地通过保险转移到其他保险人身上。购买再保险的最主要原因是为求稳健，另一个重要原因是为了提高保险人的承保能力——即使某些风险原本必须完全或部分拒保，再保险也能使保险人有能力承接。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板的保单背后有伦敦再保险人 | 再保险人承担原保险人的风险 |
| Thompson说"大部分钱在投资里，短时间拿不出" | 再保险避免保险人因大额索赔被迫紧急套现 |
| Thompson抽屉里的再保险合约 | 再保险合同可约定超额部分由再保险人直接赔付 |
| 周老板不知道有再保险人存在 | 再保险对保单持有人无直接影响，保险人须直接对保单持有人负全额责任 |
| 一张单分出去后公司仍赔得出 | 再保险提高保险人的承保能力 |
