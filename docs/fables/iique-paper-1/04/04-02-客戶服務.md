# 4.2 客户服务

## 故事

何师奶的茶餐厅开了十几年，什么样的客人都见过。但有一年夏天的三通电话，让她对"客户服务"这四个字有了新的理解。

第一通电话是她自己打过去的。店里伙计不小心把一份旧保单弄湿了，字迹模糊，何师奶想看看能不能补一份新的。电话打过去，接线的是保险公司客户服务部的一个年轻女职员，态度客气，听完何师奶的情况，说要查一下档案，让她等五分钟。

何师奶等了两分钟就有点不耐烦，正要挂电话，对方回过来了："何小姐，保单号码查到了，您可以过来取，也可以我们寄过去，寄的话要三个工作日。"何师奶选了自取，第二天一早就拿到了。

第二通电话是保险公司打给她的。台风季节快到了，客户服务部问她那份综合保单有没有加保财产险里的风灾项目，如果没有，可以考虑加。她接完电话，去找梁志明聊了聊，梁志明说加可以加，但要加保费。何师奶想了想，觉得有道理，第二年续保的时候就把风灾加上了。

第三通电话是她投诉用的。有一批货在仓库里受潮，她报损之后等了大半个月都没人来看现场。梁志明解释说是理赔部人手不够，客户服务部收到她的投诉之后，协调了两次，终于派人去了。事情处理完，何师奶在茶餐厅门口跟街坊说："投诉有用，但投诉完了要跟进，不能说完就算了。"

后来何师奶跟梁志明说："客户服务不是客客气气说话那么简单的。是人家找你的时候能找到，是你要办事的时候有人跟进，是你出了问题有人帮忙协调。"她顿了顿，"还有一样，就是人家觉得麻烦的事情，你提前一步想到了，台风季那个电话，就是这个。"

梁志明把这几句话记在心里，带回公司开会的时候说了一遍。理赔部的主管听完没说话，后来倒是把勘查的响应时间改成了五个工作日。

## 原文定义

客户服务（或称顾客服务）的核心职责包括：联络（处理各类查询并提供信息和指示）、公共关系（影响公司在公众心目中的形象）、文件提供（补发保单副本、更改内容、出具证明等）以及投诉处理（须公正迅速，可能涉及其他部门乃至高层管理和监管机构）。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 何师奶打电话补保单，职员两分钟内回电告知查询结果 | 联络：响应查询，提供信息与指示 |
| 台风季保险公司主动打电话提醒加保风灾 | 公共关系：主动维护客户关系，影响公司形象 |
| 何师奶来取补发的保单副本 | 文件提供：保单副本的补发与更改 |
| 等了大半个月没人勘查，何师奶投诉，客户服务部协调派人 | 投诉处理：公正迅速处理，可能须协调其他部门 |
