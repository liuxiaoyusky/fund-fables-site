# 4.12 管控要员

## 故事

梁志明在这间英资保险公司跑了三年前线，每天和客人解释保单条款，到月底把表格交到后台，就以为整间公司不过如此。那天 Mr. Thompson 叫他下午三点到会议室，说有东西要学。

会议室里只有张小姐和一份薄薄的名单。Mr. Thompson 把名单推到桌子中央，说："这是我们公司今年报给保险业监管局的管控要员名单，你看第一行。"

梁志明低头。名单上每行一个名字，后面跟着一个括号，里面写着"精算"，"财务控制"，"内部审计"，"合规"，"风险管理"这样的字眼。有一个名字后面同时写了两个词：精算是他认识的，但风险管理是什么意思？

"一间保险公司内部有很多看不见的手，" Mr. Thompson 说，"承保一张保单不是只有一个柜台在处理。前线接单，后台核算，再保部分要算出去，还有财务、内部审计、合规……每一块都要有人专门负责。这些人加在一起，才是整间公司的管治。"

张小姐接话："这些位置不能随便找个人挂着就算了。监管局要认定这些人选是合适的，才能列进名单。我们报上去三个人，其中一个被退回来，因为他的资历不够。"

梁志明想起上个月有个客人来投诉，说他的保单被某间公司的经纪拖了三个月不处理。那间公司后来被监管局点了名，说他们的中介人管理管控职能不到位。他当时不明白这句话的意思，现在有点明白了——一间公司派谁来管那些帮它卖保单的人，这本身就是一个专门的岗位，不是随便谁兼一下就行的。

"所以你们每天在做的，" 梁志明慢慢说，"也是在帮公司管好那些替公司接生意的人？"

Mr. Thompson 点点头："包括确保他们遵守《保险业条例》。我们公司有个专门的管控要员，负责中介人管理这一块。他要监督所有代理人的行为，还要确保他们转介过来的业务是合规的。如果有代理人不守规矩，是他签字报告上去。"

梁志明把那页名单看了很久。他从来只知道街坊找他买保障，不知道背后有这么多只手在维持整间机构的运转。每一只手都要经过认可，每一只手都有具体的职责。他忽然觉得前线那张柜台像是码头上的吊臂，看起来是自己在用力，实际背后有一整套钢铁结构在撑着。

散会时张小姐在整理文件，Mr. Thompson 提醒她下周一要把更新后的名单再递一次。监管局批得快，大概两周。

两周。梁志明想，这两周一过，名单上的名字才真正合法。公司管治，从来不是一句空话。

## 原文定义

保险公司可委任管控要员作为公司管治制度的一部分。管控要员是指单独或共同地负责履行保险公司一项或多项管控职能的个人，包括精算、财务控制、内部审计、合规、风险管理和中介人管理。保险公司在委任任何管控要员之前，必须先取得保险业监管局的认可。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| Mr. Thompson 与张小姐讨论管控要员名单 | 保险公司内部有专门部门负责管治 |
| 名单上不同人负责不同职能（精算、合规、风控等） | 管控要员职能包括多项专业分工 |
| 其中一人被监管局退回，资历不足 | 管控要员必须获监管局认可为适当人选 |
| 有专人负责监督保险代理人是否合规 | 中介人管理管控职能：管好替公司接洽业务的人 |
| 两周后监管局批复，名单才正式生效 | 认可程序是法定的管治要求 |
