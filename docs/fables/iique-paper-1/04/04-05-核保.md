# 4.5 核保（Underwriting）

## 故事

周老板的海味舖在上环开了快三十年。从花胶到鲍鱼翅，凡是晒得干的，他都收。有一年夏天，连续几个台风压港，他的货在码头仓库里闷了两周，等开仓的时候，花胶全发霉了。周老板那年亏了将近半年的本。

第二年春天，他走进中环一间英资保险行的门，找到一个叫梁志明的年轻伙计，说："我要投保，下次台风来，我不想再蚀这么大。"

梁志明把他填好的投保书递给坐在里间的Mr. Thompson。Thompson把表格放在台灯下，慢慢地一行行看。他的桌面上有几本厚册子，封面印着"UW Guidelines"，翻开来密密麻麻全是字。

"周老板，你这个仓库在哪里？"Thompson问。

"鸭脷洲码头。"

Thompson的手指在册子里划了几行，然后把投保书放回桌面，没有立刻说话。

周老板等不及了："怎么，不肯接单？我以前买过别家的，都没这么多事。"

Thompson站起来，把册子翻开给周老板看其中一页："你看，这里写得很清楚——码头仓库，尤其是存放干货的，如果不在二楼以上，我们叫它 target risk，但不是好的那种。台风一来，一楼被淹，损失就是整批货。"

梁志明在旁边听着，第一次意识到原来 Thompson 每天看的这些厚册子，不是为了为难客人，而是公司在计算"接这单划不划算"。他想起自己去年代理的一张普通家居保单，Thompson二话没说就批了；而周老板这张货仓单，他翻来覆去看了一个下午。

"那我的货怎么办？"周老板的声音有点急。

"我们可以接，"Thompson说，"但保费要高一点，而且你必须加装排水泵。我再给你一张清单，照着做，第二年续保的时候，我们可以重新看。"

周老板愣了一下。他以为"不接"就是"不接"，没想到还有条件可以谈。

梁志明后来问Thompson："为什么有些单当场批，有些要慢慢看？"

Thompson说："人寿保单，签了就不能取消，核保只看一次，错了就没有第二次机会。所以我们把核保权集中在总部，每一张单都要资深的人看过。general insurance不一样，保单每年续保，条件不对，下一年可以调。所以前线的人也可以有一些权限。"

他顿了顿："但不管哪种，最要紧的是知道什么叫target risk——有的业务是公司拼命要抢的，比如一个年轻、身体好的专业人士买人寿险；有的业务是要小心再小心的，比如码头上的干货铺。你知道为什么珠宝店在中环买盗窃险特别贵吗？因为太显眼了，小偷都知道那家店最值钱。"

梁志明点点头。他想起表姐嫁了人之后，姑妈逢人就说"趁年轻买一份，晚了就买不到了"，当时他还觉得姑妈在推销，现在才明白那是有道理的——核保员会看年纪、看身体，你越健康，对他们来说越合算。

周老板后来装了排水泵，缴了贵一点的保费。第二年续保的时候，Thompson把保费调低了一成，说："今年你那边没有出过事，条件可以松一点。"

周老板说："那我明年继续做你们的老客户。"

他不知道的是，Thompson的册子里，鸭脷洲码头那一条下面，有一行小字写着：如果连续三年无赔，可申请降费。他没有告诉周老板这行字，因为他希望周老板第三年还继续缴费，而不是为了省保费第二年就不续了。

## 原文定义

核保（Underwriting）是选择承保的风险及决定采用的保险条款的过程。在一般保险中，它是一个持续的过程，续保时可检讨条件甚至取消保单，因此核保权限可较分散；而人寿保险核保是一次性的，保单签发后不能单方面更改内容，故权限通常集中在总部。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| Thompson翻阅UW Guidelines厚册子，权衡是否接受周老板的货仓单 | 核保：选择风险、决定承保条件 |
| 人寿险核保一次性、权力集中；一般险每年续保可调整 | 人寿保险vs一般保险核保特点 |
| 鸭脷洲码头干货仓库被列为"高风险"；年轻健康专业人士被视为"优质业务" | Target risks（目标风险）的两种含义 |
| Thompson在续保时调整保费条件 | 一般保险核保的持续性与可调整性 |
