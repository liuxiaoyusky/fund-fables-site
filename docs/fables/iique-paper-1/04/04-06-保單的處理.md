# 4.6 保单的处理

## 故事

何师奶的茶餐厅开在油麻地，街坊都叫她"何太"。她的铺子不大，但门口常年挂着那块旧招牌，是她先生当年亲手写的。"何记茶餐厅"四个字已经有些斑驳，她始终舍不得换。

先生走后第四年，她终于去买了第一份意外保障。保险行的梁志明来铺子里跟她谈的，解释了很久，她才搞清楚自己买的是什么。"出事唔好连累屋企人"——这是她的原话，也是她买这份保障的原因。

保单签下来的那天，她把那张薄薄的纸仔细收进了铁盒里，和先生的旧照片放在一起。

两个月后，何师奶的伙计阿强在搬货的时候扭伤了腰，进了公立医院。何师奶想起保单，打电话给梁志明问理赔程序。梁志明接电话的时候，听到何师奶报出的保单号码，愣了一下。

他翻出那份保单的副本，仔细看了看，眉头皱了起来。

保单上写的是"何美施"，但何师奶身份证上的名字是"何美诗"。"施"和"诗"字形相近，打字的时候很容易混。梁志明想起保单制备的时候，是按何师奶口述写的，她又不太看英文单，一时没发现。

梁志明赶紧去找张小姐。张小姐是华英保险会计部的文员，专门负责保单制备和记录核对，每天对着打字机和数字，她比谁都仔细。

"呢个唔系大错，但系要出批单，"张小姐看完之后说，"一般保险出事唔一定要出示正本，但人寿保险就一定要。人寿保单的名字同身份证对唔上，理赔的时候好麻烦。"

梁志明问："但系何太买的是一般意外险，唔系人寿险，唔会出事卦？"

张小姐摇摇头："你知唔知一般保险同人寿保险的分别？一般保险出咗事，赔偿金额系睇损失有几多，保单本身唔需要一定出示。但人寿保险唔同，第一，人寿保险的合约唔可以被保险公司取消；第二，人寿保单可以转让出去，或者用来借钱的抵押，受让人系完全信任保单上的内容的。如果名字错了，受让人或者受益人可能要自己举证，麻烦得多。"

她停了停，继续说："而且何太买的是一般意外险，但佢当年投保的时候年龄已经偏大，如果当时有过往症没有申报，今次出事核查起来，保险公司有权拒绝赔偿。所以保单内容出错这件事，一定要尽快更正。"

梁志明把这些话在心里过了一遍，问张小姐应该点做。

张小姐说："出一张批单。批单系对原有保单的修改或补充，内容同正本一样有法律效力。以后每次续保，都会发新的批单，所有修改都要记录在案。"

梁志明打电话给何师奶，说保单有个小地方要改，让她带身份证过来签个字。何师奶心里有点慌："系唔系出咗乜嘢事？"梁志明说："唔系坏事，系帮你核对清楚资料，以后理赔唔使麻烦。"

何师奶来了之后，张小姐当面把批单的内容解释给她听。批单上写的是："原保单持证人姓名'何美施'更正为'何美诗'，身份证号码维持不变，其余条款维持原状。"何师奶听明白了，在上面签了字。

临走的时候，张小姐叫住她，说了一句话："何太，你记得，保费要准时交。一般保险的合约未收到保费都可以生效，但你买的是意外险，通常都要先付款，合同先算数。人寿保险更严格，唔收到第一期保费，合约都唔存在。"

何师奶愣了一下，问："唔系签咗名就已经算数吗？"

张小姐说："签了名系签了名，但合同生效同人寿保险的理赔权限集中有关，你呢份系意外险，暂时未需要担心。但如果你将来买人寿险，就一定要记住：保费未到，合同未算。"

何师奶点了点头，把那张批单和原来的保单一起收进铁盒里。

梁志明后来跟张小姐说，今次好在发现得早。张小姐说："保单制备这件事，最紧要就系核对。新的业务要核实事实准确性，错了要及时更正，出批单记录。何师奶今次系一般意外险，影响有限。但如果是人寿保单，一句名字错了，可能赔唔到钱。"

那天下午，何师奶茶餐厅的伙计给她送了一杯冻柠茶去保险行，说是何太叫送的。梁志明知道，那不是谢他帮填表，是谢他帮把事情核对清楚。

## 原文定义

保单处理涉及新业务的文件制备、签发、更正误差、批单及续保手续。一般保险的保单技术上来说并非必需，人寿保险则必须在提出保险金要求时出示保单文件，且人寿保单可以转让或用作贷款抵押，受让人依赖保单内容的真确性。人寿保险合约的存在与否，实务上通常取决于是否收到第一期保费。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 保单上"何美施"与何师奶身份证"何美诗"名字不符 | 保单内容准确性的重要性：人寿保单内容出错后果严重，因为可转让或作抵押 |
| 张小姐解释一般保险vs人寿保险的保单文件要求差异 | 一般保险：保单非必需；人寿保险：必须出示且可转让，受让人依赖真确性 |
| 张小姐发出批单，更正姓名错误并记录在案 | 批单的作用：对原保单的修改附页，具有法律效力，所有变更须记录 |
| 张小姐强调人寿保险"未收到第一期保费，合约不存在" | 保费缴纳的重要性：一般保险可先生效后天缴费，人寿保险以收到首期保费为合约前提 |
| 梁志明提前核对发现错误，避免理赔时出现问题 | 新业务程序：文件制备过程中核实事实准确性是最重要的环节 |
