# 6.2.5.4 客户协议及与客户订立的业务条款

## 故事

周老板的海味铺在上环开了三十年。从父亲那一辈起，街坊买燕窝蚝豉，习惯先记账，年底一起结。周老板觉得这是人情，做生意讲的就是"我信你，你也要对得住我"。

那年夏天，梁志明被公司派到上环驻点，背着一只旧公文包，挨家挨户解释新的业务条款。周老板是第三家。

"周老板，这次真的要签一份书面协议。"梁志明把文件摊在柜台上。密密麻麻的英文打头，中文翻译附在后面。周老板看了一眼，把文件推回去。

"志明，我同你阿叔识咗咁多年，你阿叔帮我阿爸出海带货，我帮你阿妈睇铺头，你估我信唔信你？签咁多纸，你当我新移民啊？"

梁志明站在那里，想说什么，又不知道怎么开口。他想起 Miss Cheung 交给他文件时说的话："白纸黑字不是不信任你，是万一出事了，谁都说不清。"他当时觉得这话冷冰冰的，现在站在周老板的海味铺里，看着满墙的蚝豉和干鲍，反而觉得这话有几分道理。

"周老板，"他换了个说法，"上个月西环做海味的陈叔，船在海上遇到风浪，货没到，货主问他赔不赔。陈叔说我们街坊不用讲这些，结果闹到要打官司，铺头都贴了封条。"

周老板没说话，点了根烟。

"这份文件不是用来绑住边个，是万一出事，大家睇白纸黑字就知道边个负责边样。保费几时交，交几多，出了事先揾边个——全部写清楚。你是我的客人，白纸黑字就是说，我对你有责任，你也知道自己有乜权利。"

周老板把烟掐了，拿起笔。

"志明，你阿叔当年帮我阿爸，真系没写过一张纸。我签，但你要记住——"他在签名处重重落下"周德厚"三个字，"签完这张纸，你唔好当我陌生客。"

梁志明把签好的文件小心收进公文包，说："签完你才有资格当我街坊。"

周老板愣了一下，笑出声来。

台风季节来之前，周老板的铺头办好了一切手续。那份协议被梁志明锁在柜里，但他说得清，每一条款对应的，正是周老板当年用街坊信用担保过的每一笔生意。

## 原文定义

持牌保险经纪公司须与客户订立书面协议，列明其为客户进行受规管活动的条款及细则，并应在合理切实可行的范围内尽快向客户提供协议副本并保存记录。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明要求周老板签书面协议 | 持牌保险经纪须与客户订立书面协议 |
| 协议列明保费支付方式、双方责任范围 | 书面列明受规管活动的条款及细则 |
| 梁志明锁好协议副本并说得清每一条款 | 保存客户协议的记录 |
| 周老板从"人情不需要文件"到愿意签字 | 书面协议并非不信任，而是明确责任 |
