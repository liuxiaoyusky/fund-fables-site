# 6.1.1b 股本要求

## 故事

1984年，上环，周老板的海味铺。

下午的阳光斜斜照入店铺，周老板坐在柜台后面，打算和来拜访的梁志明聊一聊。旁边坐著老陈——保险公司的老账房，今天特意过来帮忙解释一些财务问题。

"我哋常说保险公司要有本钱，"老陈放下手里的茶杯，缓缓说道，"但本钱有几种，实缴股本系最根本的一种。"

梁志明拿著笔和本子，准备记录。周老板的女儿若有所思地在旁边整理著鲍鱼罐头，一边听著。

"简单嚟讲，实缴股本就系股东实际投入公司、已经缴付的资金。"老陈说，"呢笔钱不同于借来的钱，公司倒了也不能拿走，系公司的家底的一部分。"

周老板马上明白了这个概念。"即系话，我自己出了十万本金，呢十万就系我的实缴股本？"

"正是。"老陈点头，"你用呢十万去周转、去备货、去交租。如果间铺倒了，这十万要先用来还债，还完有剩先可以分番俾股东。"

"那保险公司的实缴股本要求系几多？"梁志明问。

老陈从公事包里拿出一张表格，上面列著数字。"《保险业条例》规定了最低门槛。第一，只经营一般业务或长期业务，而非法定保险业务——最低一千万港元。"

周老板吹了吹口哨："一千万。"

"第二，"老陈继续说，"如果公司涉及法定保险业务，即系法律要求必须参保的业务，例如雇员补偿保险、汽车保险——不论是否兼营其他业务，最低要二千万港元。"

"为什么涉及法定业务就要加倍？"梁志明问。

老陈解释："法定保险业务的赔付系强制性的，保险公司唔可以拒绝投保，赔付金额也有法律规定。如果出了大事，例如一场大火的雇员补偿，赔付数字可以非常大。二千万的家底，系一个更稳妥的底线。"

周老板的女儿突然插嘴："咁同时做一般业务同长期业务呢？"

"问得好。"老陈说，"同时经营一般业务及长期业务，也是二千万港元。"

"专属自保保险人呢？"梁志明问。

"专属自保保险人系大企业自己设立的保险公司，专门为自己的集团提供保险。"老陈说，"佢哋的最低要求系二百万港元。点解低咁多？因为佢哋的业务主要系内部风险，唔对外公众经营，风险相对可控。"

周老板点了点头："明白了。做生意要有本钱，唔同的生意有不同的本钱要求。"

老陈赞同地说："但系记住，这些都系最低要求。香港的保险公司实际持有的实缴股本，往往远远超出最低门槛。"

"为什么？"梁志明问。

"因为最低门槛只系'可以营业'的水平，"老陈说，"真正想把生意做大、做稳的公司，需要更多的本钱支撑业务增长、应对突发情况、以及俾投资者和监管机构信心。"

梁志明把这些数字记在本子上。一千万、二千万、二千万、二百万——每个数字背后都有一个商业逻辑。

周老板站起身，走到窗边，看著街外人来人往。"我年轻时觉得有本钱就系有钱。后来先知，本钱大嘅意思唔系你有几有钱，而系你准备好承担几大的责任。"

老陈和梁志明都没有接话。但梁志明在心里记住了这句话。

## 原文定义

实缴股本最低要求为：一千万港元（只经营一般业务或长期非法定业务）；二千万港元（涉及法定保险业务）；二千万港元（同时经营一般及长期业务）；二百万港元（专属自保保险人）。上述数额仅为最低要求，香港保险人通常持有远超这些门槛的实缴股本。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 股东投入的十万本金，店倒了要先还债 | 实缴股本的概念：公司实际缴付的资金 |
| 一千万/二千万的差异 | 不同业务类别的最低股本要求 |
| 法定保险业务要加倍门槛 | 强制性赔付责任越大，股本要求越高 |
| 专属自保保险人二百万门槛 | 内部风险可控，最低要求相对较低 |
| 本钱大不是有钱，而是准备好承担更大的责任 | 股本要求的监管初衷 |
