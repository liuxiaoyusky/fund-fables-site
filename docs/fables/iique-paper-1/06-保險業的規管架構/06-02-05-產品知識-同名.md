# 6.2.5.5.1 产品知识

## 故事

周老板在海味行业做了三十年，什么人没见过。他信一条：找谁办事，就要知道这个人姓什么、是哪家公司的、拿什么牌头。街坊之间介绍生意，也要先报个名号，没名没姓的，他不敢信。

那年周老板要给舖头买一份火险保险，经女婿介绍，见了一个年轻的保险代理人。对方姓梁，说话客气，拿了一张名片递过来。周老板接过一看，上面印着"梁志明 保险顾问"六个字，下面一行小字写着"专业服务 诚信可靠"，最底下是一家叫"恒辉"的公司地址。

周老板把名片翻过来，没有背面。他问："梁生，你是替哪家公司做事的？牌照是多少？"梁志明说："我替恒辉保险做，恒辉是我挂靠的代理机构，牌照是——"他报了一串号码，又补了一句，"您要是觉得不够，我这边还有公司开的委托书副本。"

周老板看完副本，抬头说："你挂靠恒辉，那你自己是什么牌照类别？是个人代理牌照，还是机构代理牌照？"梁志明愣了一下，然后笑了笑，说："周老板您问得仔细，我这就给您看我的牌照原件。"他把一张塑封的个人代理牌照递给周老板，上面姓名、号码、类别、受恒辉委任白纸黑字，一样不少。

周老板看完，把牌照还给他，说："我做生意几十年，最怕找了个'无名氏'，出了事连人影都找不到。你这样做是对的。"

后来梁志明跟Mr. Thompson聊起这件事，说周老板问得太细，自己差点答不上来。Mr. Thompson说："问得对的才买。客人搞清楚你姓什么、代表谁，他才会信你；信了你，将来理赔时才不会觉得被骗。这不是刁难，是规矩。"

梁志明回去后，把名片重新印了一版，正面加上牌照号码和恒辉公司的全称，字体放大了两号。周老板第二次见面时看到了，说："这就好看了。"

## 原文定义

持牌保险代理人应向客户提供以下资料：姓名（注册名称及商业名称）、牌照号码、牌照类别（个人保险代理牌照、保险代理机构牌照或业务代表牌照）、受其委任的保险人或受其委任的代理机构名称；代理超过一名受委任保险人时，应就每项特定保险交易向客户明确说明代表哪一家；名片须显示身份证明上的姓名、牌照号码、牌照类别及受委任保险人或受委任代理机构名称。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板要求看梁志明的姓名和所属公司 | 须披露姓名（注册名称及商业名称） |
| 梁志明当场报出牌照号码 | 须披露牌照号码 |
| 周老板追问牌照类别是个人代理还是机构代理 | 须披露牌照类别 |
| 梁志明说明自己挂靠恒辉保险 | 须披露受委任保险人或代理机构名称 |
| 名片加印牌照号码和恒辉全称 | 名片须显示规定内容 |
