# 6.2.4.2 清楚其知识的局限性

## 故事

周老板有个朋友在新加坡，想把一批贵价燕窝从吉隆坡运到香港，问周老板有没有相熟的保险公司可以承保运输途中的风险。周老板把这件事告诉了梁志明。

梁志明接过资料，翻了几页，脸色有些为难。

"周老板，"他说，"呢单嘢我帮唔到你。"

周老板皱眉："你唔想接？"

"唔系唔想接，系我唔识。"梁志明把资料放下，"运输货物喺海上同码头之间𠮶段时间，风险评估同普通铺面火险完全唔同。货物值几钱、航线经过边啲海域、系咪需要再保险——呢啲我唔具备相关知识，唔应该乱讲。"

周老板看着他，问："你就咁推晒？"

"我唔系推，我系去问我老板。"梁志明拿起电话，"我请示Thompson先生，睇下公司有冇人熟悉货运险，再回复你。"

两天后，Mr. Thompson亲自约了周老板的朋友到公司谈。梁志明坐在旁边做记录，一句多余的话都没插。谈判结果是Thompson接手处理这单货运险，梁志明只负责跟进后续的文件往来。

事后周老板问他："你当时点解唔自己接？"

梁志明说："我自己接，乱噏一通，客人信咗我，最后赔唔到钱，责任算谁？唔识就讲唔识，然后揾识嘅人跟进——呢个先系对客人负责任。"

Thompson刚好走过，听到了，在门口站了一下，嘴角动了动，没说话，继续走远了。

## 原文定义

持牌业务代表（经纪）不得就其缺乏进行有关受规管活动所需特定技巧或知识的事宜进行受规管活动。如有疑问，应寻求其委任的持牌保险经纪公司负责人或高级管理层提供指引。持牌保险经纪应向客户提供准确及充足的资料，使客户得以作出知情的决定。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明明确表示自己不懂货运险的风险评估 | 持牌业务代表须清楚认识自身知识的局限性 |
| 梁志明主动向Thompson请示而非自己接单 | 有疑问时应向上级或高级管理层寻求指引 |
| 梁志明在谈判中只做记录，不乱插话 | 不具备相关知识时不应擅自行事 |
| Thompson提供准确信息后客人得到合适的承保方案 | 向客户提供准确充足资料，使其作出知情决定 |
