# 6.2.5.5.4 就他人转介的客户所需作出的披露

## 故事

周老板有个鱼丸铺的老邻居叫梁婶，儿子在内地做工，收入不错。梁婶想给儿子买一份保障，问周老板应该点样揾人。

周老板说："我识一个保险经纪叫梁志明，几靠谱的。你话我介绍你俾佢，佢会帮你搞掂。"

梁婶找到了梁志明。梁志明和她坐下来，先问她一些基本情况。聊了一会儿，梁婶说："周老弟讲，介绍费已经倾掂咗，我净系出一份钱，剩下嘅周老弟帮我搞。"

梁志明放下笔。

"梁婶，呢件事我要同你讲清楚。"他说，"你揾我，我系对你负责的人。周老弟是转介人，佢介绍你俾我，但将来保单出了事，你唔应该揾周老弟，因为保单唔系佢安排嘅。"

梁婶有点不解："但周老弟话佢熟悉呢间公司，帮我倾到好价钱。"

"佢可能真系熟悉，"梁志明说，"但保单嘅事，你只应该同我往来。保费交给谁？交给保险公司的账户，或者交俾我代收，唔好经周老弟。如果周老弟叫你把钱交给佢，你唔好照做。"

梁婶有点不高兴，觉得周老板一片好心，她唔好当面拒绝。

梁志明明白她的难处，换了个说法："你听我讲——周老弟对你嘅责任，就系介绍你俾一个靠谱的人。而家佢做到咗，佢对得住你。但保单签完之后，后续跟进、理赔、交钱，全部系我对你负责。周老弟唔参与，呢个系规矩，唔系我同佢有矛盾。"

他把这一段写在笔记本的边角，方便自己记住，也方便将来解释给其他客人听。

梁婶最终明白了，签了文件，交了保费直接付到梁志明公司的账户。后来理赔的时候，她第一时间揾梁志明，很快就搞掂了。佢没同周老板讲呢件事有几顺畅，因为有些嘢唔使讲。

## 原文定义

若客户由他人转介予持牌保险经纪，该经纪须告知客户：经纪负责安排保单，客户只应直接与经纪往来；转介人不代表经纪，亦不应参与安排保单；经纪就转介人对保单所给予客户的任何意见不承担责任；保费应直接支付予经纪或相关保险人，而非转介人。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁婶说"周老弟帮我搞掂" | 转介人不负责安排保单 |
| 梁志明说"出了事唔好揾周老弟，因为保单唔系佢安排" | 转介人不代表经纪，经纪对客户直接负责 |
| 梁志明说保费直接付到公司账户或经他转交，不要经周老弟 | 保费应直接支付予经纪或保险人，而非转介人 |
| 梁婶明白"周老弟的责任是介绍靠谱的人" | 转介人只负责转介，经纪对后续负直接责任 |
