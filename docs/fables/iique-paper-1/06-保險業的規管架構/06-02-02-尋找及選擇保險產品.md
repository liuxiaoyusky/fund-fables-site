# 6.2.2.2 寻找及选择保险产品

## 故事

张小姐在洋行负责处理文件和保险公司之间的往来。1980 年代初，香港的保险公司多了起来，每家的产品小册子厚厚一叠，推销信像雪片一样飞进写字楼。张小姐每天要拆几十封信，按公司分类，按产品归档，再送到负责的经纪手上。

那天周老板来，是要续一份很旧的保障计划。他从口袋里掏出一张皱巴巴的纸，说这是上一份合约的摘要，纸张已经发黄，上面印着一家老保险公司的名字。

张小姐接过来看了看，问他："周老板，您知道这份产品现在市场上有哪些类似的选择吗？"

周老板说："不需要挑了，我就买这个，习惯了。"

张小姐没有直接答应。她叫来了负责跟进周老板的梁志明，说："阿明，周老板续保这个产品之前，你要不要先看看现在市面上有没有更合适合的产品？老产品未必是最适合的。"

梁志明想了想，说："也是。"然后他花了两天时间，找了四家保险公司的产品资料，仔仔细细比较了保费、保障范围、赔付记录。他发现周老板现有的那份产品保费偏贵，保障范围却偏窄，有两家新保险公司的同类产品性价比更高。

他把比较结果整理成一张表格，标出每项的优劣，拿到周老板面前说："周老板，我看过了，市场上有更好的选择。这是我比较的结果，您看看。"

周老板看着表格，沉默了很久，最后说："阿明，我做了这么多年街坊生意，我知道有时候熟人生意做久了，双方都懒得改。但是今天你这么认真比给我看，我反而觉得，你是真的在替我想。以后你也这样做。"

## 原文定义

持牌保险经纪应从充足范围的不同保险人处，寻找及选择最符合客户利益的保险产品，且不应不合理地倚赖任何特定保险人以致损害客户的选择权。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明找四家保险公司的资料比较 | 从充足范围的不同保险人处寻找产品 |
| 张小姐提醒要先比较再续保 | 不应不合理倚赖特定保险人 |
| 梁志明整理表格对比优劣 | 基于客户具体情况判断何为最适合 |
| 周老板同意"以后也这样做" | 广泛比较赢得客户更深的信任 |
