# 6.2.3 持牌保险中介人的多重角色

## 故事

张小姐在公司做了三年会计，兼管合规文件，对牌照条例倒背如流。那年冬天，公司来了个新同事阿昌，说是保险代理机构的业务代表，来"学习交流"。

阿昌嘴很甜，见人就递名片。他和张小姐混熟之后，有天问她："张小姐，你们公司有没有客人想转保险公司？我手上有几间公司的产品，佣金比例比这边高，我可以帮你们介绍。"

张小姐愣了一下，没回答，转头去找了梁志明。

梁志明那天正好在Mr. Thompson的办公室里交文件，听完张小姐的话，脸色变了。

他对Thompson说："阿昌是保险代理机构的持牌业务代表（代理人），他不能同时替我们保险经纪公司拉客。这是牌照条例写得清清楚楚的限制。"

Thompson放下手里的钢笔，说："你去把Mr. Whitfield请过来。"

Mr. Whitfield是公司长期合作的法律顾问，熟悉《保险业条例》里的每一项细则。他听完事情经过，对阿昌说："条例对你这种持牌业务代表（代理人）有明确限制：你不能同时在两家机构任职，也不能同时为保险经纪公司服务。你来我们这里学习可以，但如果你替我们拉客，就违反了条例。"

阿昌脸色一阵红一阵白，收回名片，走了。

梁志明后来问张小姐："你当时为什么没有直接答应他？"

张小姐说："我做了三年合规，呢个界线一早就划定咗。一仆不侍二主——你替边个打工，就只能卖边间公司的嘢，唔可以掉转头卖第二间。呢个道理唔难明。"

Thompson在隔壁听到，说了句："张小姐，帮我check下阿昌𠮶间代理机构的牌照状态有无问题。"

张小姐应了一声，翻开牌照登记簿。

## 原文定义

持牌保险代理机构的人员、持牌保险经纪公司的人员、持牌业务代表（代理人）和持牌业务代表（经纪）均受到任职限制，不得同时兼任多个可能产生利益冲突的角色。例如持牌保险经纪公司的人员不得兼任持牌保险代理机构的经营人、董事或雇员。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 阿昌同时替代理机构和保险经纪公司拉客 | 持牌业务代表（代理人）不得兼任经纪公司服务角色 |
| 梁志明认出违规，主动向Thompson报告 | 持牌中介人有责任识别并制止违规兼职行为 |
| Mr. Whitfield援引条例细则告知阿昌边界 | 持牌中介人的角色限制由《保险业条例》明确规定 |
| 张小姐说"一仆不侍二主"是行业基本伦理 | 多重角色限制的核心目的是防止利益冲突 |
