# 6.2.8.1 保费处理

## 故事

周老板有个习惯，每个月月初都会把保费装进一个牛皮纸信封里，亲自送到梁志明的公司。信封上用圆珠笔写着自己的名字和保单号码，字迹有点潦草，但一笔一划都很认真。

这天，他又来了。梁志明在前台接过他递过来的信封，发现里面是一叠现金，全是一百元纸币，用橡皮筋扎得整整齐齐。

"周叔，这个月的保费。"周老板把信封往前一推，"下个月我可能会晚两天，因为铺子那边资金周转有点紧。"

梁志明把信封收下，正要给他开收据，忽然想起了什么。

"周叔，您怎么每次都用现金？"

"现金方便。"周老板说，"我不懂什么银行转账，每个月亲自送来，顺便看看你还在不在这里做事。"

梁志明笑了笑，但没有立刻把收据开给他。他看了看信封里的现金，又看了看周老板。

"周叔，按规定，我们公司对现金保费有限制。"他认真地说，"您每个月缴的金额，如果超过授权上限，我不能收现金。公司要求超过一定金额的保费，必须走银行转账。"

周老板皱起眉头："你们保险规矩真多。别人给我送钱都不管，我送钱给你们反而要被限制。"

"不是为难您。"梁志明把规定解释给他听，"现金保费的问题在于，没有银行记录，万一出了问题，双方都说不清楚。公司要求走转账，是为了保护客人——万一有人事后说没缴保费，转账记录就是凭证。"

周老板想了想："那我下个月去银行转账。但这个月的呢？已经拿来了。"

梁志明看了看这个月的金额，确认没有超过上限，然后把现金收下，从抽屉里拿出一本收据簿，认真地填写了一式两份的收据，把其中一份交给周老板。

"周叔，这是您的收据，您收好。以后每个月转账之后，把转账凭证保留好，和这份收据放在一起，万一将来有什么问题，有据可查。"

他把这一叠现金放进抽屉里的一个专用信封，这个信封和公司其他日常开支的现金是分开放的——这是Mr. Thompson在合规培训上特别强调过的。

周老板看着他把现金放进专用信封，有点惊讶："阿明，你们连钱都要分开放？"

"周叔，这是公司的规定，也是行规。"梁志明说，"保费是客人的钱，不是我的钱。我只是代收，然后按规定的时间和方式转交给保险公司。在这之前，这笔钱不能和我自己的钱混在一起。"

周老板沉默了一会儿。

"阿明，"他说，"你知道为什么我每个月都亲自来送钱吗？"

"方便？"梁志明猜测。

"不是为了方便。"周老板摇摇头，"是为了记住。我老婆走的那年，什么都没有留下，就留下一句话：家里要有一份保障。我每个月亲自来送钱，是为了让自己记住这件事——每个月都要供下去，每个月都要记得。"

梁志明把那张收据递给他，说："周叔，那您下个月就转账，把凭证给我就行。一样记得住。"

周老板接过收据，折好，放进西装内袋里，和保单放在一起。

那天傍晚，梁志明把现金信封封好，按公司规定的时间和方式，亲自送到了保险公司的收款窗口。窗口的职员核对了金额和保单号码，开了一张回执。梁志明把回执和留底收据订在一起，存进了客户档案里。

Miss Cheung从走廊那头走过来，看了一眼他桌上整齐排列的文件夹。

"周老板的保费？"她问。

"对，今天刚处理的。"梁志明说。

Miss Cheung点点头，没有多说什么。她知道梁志明的习惯——每一个客人的文件都整理得清清楚楚，每一笔保费都有去向记录。这是他在公司立身的基础，也是他让客户信任的原因。

周老板后来真的去银行办了转账。每个月初，他会把转账凭证夹进那本旧账本里，再拿着账本去找梁志明。账本上密密麻麻记着每一笔保费的支出，旁边还写着一些字："六月，转账，恒信，A计划，第三年。"

那本账本，他已经记了三年。

## 原文定义

持牌保险代理人只可在其委任保险人或代理机构授予的权限范围内收取保费的付款。如获授权收取保费款项，该代理人应：严格按照其委任保险人或代理机构的的规定、管控及时限去处理及交付款项；妥善保管所收取的保费，并避免将有关保费与该代理人的个人资金混合；以及备存所收取的保费的良好纪录。持牌保险代理人不应收取以现金方式支付的保费款项，除非：代理人不能在合理可行的情况下以其他方式接受付款；及其委任保险人或代理机构已授权该代理人收取现金款项，且金额在授权上限内，而该款项的处理亦将严格遵照有关授权的规定及管控。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明告诉周老板超过授权上限不能收现金，必须走转账 | 收取权限——代理人只能在授权范围内收取保费，超过上限须用其他方式 |
| 梁志明把周老板的现金放进"专用信封"，与个人资金分开 | 款项分开保管——保费不得与代理人个人资金混合，需单独管理 |
| 梁志明填写一式两份收据，并保存回执与转账记录 | 良好纪录——对所收取的每一笔保费须有完整记录，可供查核 |
| 周老板每月亲自送钱是为了"让自己记住" | 背景动机——保费按时缴付背后的真实驱动力是家庭责任 |
