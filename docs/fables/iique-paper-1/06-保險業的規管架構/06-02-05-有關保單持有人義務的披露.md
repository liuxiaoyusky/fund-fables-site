# 6.2.5.3 有关保单持有人义务的披露

## 故事

周老板有一个师弟姓梁，在澳门经营鱼栏。那年台风季节前，梁师弟想来香港为船队加一份保障，经周老板介绍找到梁志明。

梁志明约他在中环一间律师楼见面，Mr. Whitfield 在场。寒暄过后，梁志明拿出一张表，说："师弟，填呢份申请表之前，我要先同你讲清楚，你有几件事必须知道。"

他一条一条说。

第一，叫最高诚信。"你同人买保障，保障公司相信你说的每一个字。如果你有事瞒报，或者讲错咗，将来出事，保障公司可以拒绝赔偿。到时候，你出了钱，但拿唔到赔。"

第二，叫重要事实。"你话船队有几条船、近五年有冇出过事、船有没有改装过——呢啲全部要如实讲。你觉得唔重要的，可能恰恰系最影响保费和是否承保的关键。"

第三，叫声明。"呢张表填完之后你会签名。签名之前，你要睇清楚每一行，因为签了名，就代表你确认咗表上所讲的全部系事实。"

梁师弟听完，有点不高兴："志明，你系唔系觉得我呃你？"

"唔系觉得你呃我，"梁志明说，"系保障公司将来赔钱俾你𠮶阵，唔会因为你同我或者周老板系老友，就唔查你点解当初冇报𠮶条船改装过。我讲在前面，系因为你现在知道了，将来真出事，你唔会话'我唔知'。"

Mr. Whitfield 在旁边补了一句："在法律上，保障公司和你是互相信任的关系。你不披露，他们不赔。这是规矩，不是针对你。"

梁师弟沉默了一会儿，低头填表。填到"近五年有否出险"那一行，他停下来，想了很久，最后写了"有"。

后来他的船在台风中出了事，申请理赔。保险公司调查时发现他如实申报，没有拒赔。梁志明事后把那张填了"有"的表格复印件夹进自己的文件夹里——不是备案，是觉得那一下停顿里有比钱更重的东西。

## 原文定义

经纪协助客户提出保险申请时，须向客户解释最高诚信原则，提醒客户不披露重要事实或提供不正确资料可能导致保单失效或索赔遭拒；须向保险人披露客户提供的所有与保单相关的重要事实，以及经纪所知悉的所有相关重要事实。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明解释"最高诚信"原则 | 向客户解释最高诚信原则 |
| "有事瞒报出事时可以拒绝赔偿" | 不披露重要事实可致保单失效或索赔遭拒 |
| 梁志明说"签名之前要睇清楚每一行" | 让客户有机会在签署声明前进行检视 |
| 梁师弟在出险记录栏停下来如实填写 | 客户须向保险人披露重要事实 |
