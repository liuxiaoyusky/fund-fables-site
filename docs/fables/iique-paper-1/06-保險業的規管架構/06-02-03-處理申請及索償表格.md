# 6.2.3.2 处理申请及索偿表格

## 故事

周老板有个伙计叫阿强，在搬货的时候扭伤了腰。工伤不严重，但阿强担心医药费，周老板想起梁志明帮他买过意外险，就让他自己去报案。

阿强第二天来到梁志明的公司，手里攥着一叠医院收据，说："我唔识写字，你帮我在张表上写上我资料啦。"

梁志明把表格拿过来，扫了一遍，然后递给阿强一支笔。

"等阵，"阿强说，"你唔帮我写咩？"

"呢张表系关于你本人资料的，一定要你自己睇过、我先可以帮你填。"梁志明说，"你望一望第一栏：姓名、身份证号码、受伤日期、医生诊断——呢啲系你才知道的事实，我帮你写咗，万一有错，责任就唔知喺边。"

阿强拿起笔，一条一条填。填到"受伤经过"那一栏，他停下了，说："我唔知点样写得得体……"

梁志明说："你就照实写，几点喺铺头做乜嘢，点样整亲，几时去睇医生。写得清楚，赔得快。"

阿强填完，梁志明拿过来，和医院收据一条一条对了一遍，没有漏，没有多，然后让阿强在表格末端签名。

"你望下，"梁志明指着签名栏旁边的声明，"呢句的意思系，你确认表格内容系准确同完整。你签咗，就代表你负责。"

阿强签了名，把表格交给梁志明。

张小姐在旁边看着，等阿强走了，说："你真系够啰嗦。"

梁志明说："佢签落去，就要对内容负责。我帮佢填，就要确保我问过佢。呢个程序唔系啰嗦，系规矩。"

## 原文定义

持牌保险经纪代填或递交任何表格时，应告知客户其有责任确保资料及文件准确完整；在未得到客户授权及确认内容准确性前，不应填写或递交任何表格；若知悉表格含有不准确资料，不应向保险人递交。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明让阿强自己先看一遍表格内容 | 告知客户其有责任确保表格资料准确完整 |
| 受伤日期、医生诊断由阿强亲口陈述后填写 | 须在客户授权及确认内容完整性后才可代填 |
| 梁志明逐条对照收据核对内容 | 适当谨慎地确保所递交文件准确无误 |
| 签名前让阿强看声明，确认他知晓内容责任 | 客户签名即代表对表格内容准确完整负责 |
