# 6.2.4.1 产品知识

## 故事

张小姐每周有一个固定任务：把公司代理的所有保险产品说明书整理一遍，按类别放进不同的文件夹。她说这是Thompson定下的规矩。

梁志明刚入职时，觉得这件事枯燥。每次张小姐把新文件递给他，他都是随手放进抽屉里，不怎么看。

后来发生了一件事，改变了他的想法。

周老板的太太想给儿子存一笔教育基金，听街坊说有种"储蓄型"产品很好，就让周老板来问梁志明。梁志明想起公司文件夹里有类似的产品，正准备推荐——张小姐路过，看了一眼，说："呢只系投资相连的产品，回报唔保证，你客人想稳阵收息，唔系呢只嘢。"

梁志明当时愣在那里。他把文件夹拿出来，对照着看了半天，才发现自己差点把两类完全不同的产品搞混了。

那天收工后，他第一次主动去找张小姐要了那叠产品说明书，花了三个晚上，把每一种产品的保障范围、除外责任、缴费方式全部过了一遍。

他做了一张简表，夹在笔记本里：

"火险——赔物业；意外险——赔人身；货运险——赔运输途中；储蓄险——赔钱；医疗险——赔医药费。边只赔、边只唔赔，要背熟。"

张小姐经过，瞥了一眼，说："你终于开窍。"

梁志明说："客人信我，我总要知道我卖嘅系乜嘢。佢问我赔唔赔，我要答得出来。"

## 原文定义

持牌业务代表（经纪）应就其可能进行受规管活动的各类保险产品的性质及主要特点，以及所承保及其相关的风险有充分了解。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明差点把储蓄险和投资相连产品混淆 | 须了解每类产品的性质及主要特点 |
| 张小姐指出两类产品的根本区别（保证回报vs不保证） | 对产品特点认识不清会直接误导客户 |
| 梁志明花三晚通读所有产品说明书 | 充分了解产品性质是提供专业建议的基础 |
| 简表区分五类产品及各自赔付范围 | 须清楚所承保的风险类别及相应保障内容 |
