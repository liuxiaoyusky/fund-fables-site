# 6.1.2b 提供建议及销售的做法

## 故事

周老板坐在洋行会客室里，手里捏着一份产品说明书，翻了两页，抬头问梁志明："这个'续保须知'写的是什么？怎么每个字都认识，拼在一起我就不懂了？"

梁志明凑过去看。那段文字写的是："若被保险人于续保时未依约履行最高诚信原则，保险公司保留调整费率或终止合约之权利。"梁志明自己读了两遍，才明白大概意思是说：续保时如果发现当初有重要事实没有如实申报，保险公司可以加价或者不续。

林若兰正好端茶进来，顺手瞥了一眼那页纸，说："这种写法在香港街坊里，十个人有九个看不懂。要是在洋行给英国总部写报告，这样写早被打回来了。"

Mr. Thompson那天正好在，他接过说明书翻了翻，说："这就是守则要管的事情。"

他跟梁志明解释：《守则》对销售资料的要求是"最新、精确、浅白、不误导"。对投保书的要求是"以浅白语言表达，重要问题要清晰，最高诚信原则要解释清楚"。对保单条款的要求是"清晰易懂，续保时的如实告知义务要专门解释"。

"还有一条，"Mr. Thompson翻了翻文件，"行政管理方面，如果因为公司员工的错漏导致客户损失，客户不用承担。"

林若兰轻声说："那就是说，表格没设计好，员工填错了，最后还是公司赔。"

周老板听到这里，把说明书放下，说："那我这份保单，到时候理赔的时候，你们公司的人要是填错了数字，也得我去找他们算账吗？"

"不用。"梁志明说，"公司负责。"

周老板点点头，没再说话，但把那份说明书叠好放进了自己的布袋里。梁志明看见他没有丢，觉得这大概是一个街坊能给出的最高认可了。

那天下班，林若兰把那份说明书的副本放在Mr. Thompson桌上，旁边贴了一张便条："建议重写'续保须知'一段，照守则标准。"梁志明路过时没看见她的表情，只看见便条的字迹很整齐，和她平时打字的一样。

## 原文定义

销售资料须为最新、精确、以浅白语言叙述且不误导公众；投保书须以浅白语言表达并附明确指引，解释最高诚信原则，清晰提问重要事实；保单条款须清晰易懂，续保时须详尽解释如实告知义务；因保险公司员工错漏引致的损失，客户无须承担。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板看不懂"续保须知"的法律措辞 | 销售资料须以浅白语言表达 |
| 林若兰指出产品说明书格式不合标准 | 资料须精确、不误导 |
| 投保书须解释如实告知原则并清晰提问 | 最高诚信原则须在投保书中说明 |
| 员工错填导致客户损失由公司负责 | 行政管理：客户不因公司员工错漏而受损 |
