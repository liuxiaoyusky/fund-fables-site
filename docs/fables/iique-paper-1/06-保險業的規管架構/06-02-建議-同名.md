# 6.2 建议

## 故事

张小姐是公司的牌照和合规助理，每天对着一叠又一叠的文件打转。梁志明是她的年轻同事，入行不久，跟着一群街坊客人打交道。汤逊先生是合规部主管，管着一叠文件、巡查记录和牌照期限。

那年秋天，上环海味舖老板周老板的生意红火，打算扩大仓储和店面。周老板找到梁志明，问他有什么「可以保的」。梁志明拿着张小姐替他准备的资料清单，上门拜访了三次。

第一次，他坐下来问周老板：店舖是自置还是租用、库存主要有哪些种类、有多少个伙计、平时有没有借钱周转。张小姐事先提醒过他：「问清楚才知道什么叫『合适』，不是你自己觉得什么好就推什么。」

第二次，梁志明拿着清单上的资料，回去仔细研究。他算了一笔账：如果火烧了存货，周老板能撑多久；如果伙计在搬运时受伤，医药费从哪里来。最后他建议了一份综合保障方案：财产、火险和意外，三样绑在一起。

第三次，周老板打断他：「太贵了，我只要火险就可以了。」梁志明有点泄气，又解释了半个钟头。周老板的态度很坚定：「我做了几十年海味生意，火烧过一次我知道，水浸我知道，伙计搬花胶扭伤腰我也见过。但现在手头紧，先把最重要的火险保了，其他的迟些再说。」

梁志明没有再劝。他回去跟张小姐商量。张小姐翻出了汤逊先生那份合规文件，指着其中一页说：「你记录下来。」

梁志明不太情愿：「客人自己决定的，我记什么？」

张小姐说：「就是因为是他自己决定的，所以才要记。」

她示范给他看那份记录怎么写：

第一行：梁志明根据周老板的实际情况，向他提供了财产、火险和意外三位一体的综合保障建议。

第二行：周老板考虑后，决定只投保火险，原因为目前资金周转紧张，希望先行控制主要风险。

第三行：梁志明已详细说明只保火险的保障缺口，并表达了对该决定的保留意见，认为该决定未能全面覆盖周老板店舖所面临的主要风险。

第四行：以上决定由周老板在充分了解建议内容及可能后果的情况下，自行作出。

张小姐把草稿交还给他，叮嘱他去找周老板签字确认。梁志明拿着记录再一次去到上环，把四行字念给周老板听。周老板戴上老花眼镜，仔仔细细看了一遍，点点头说：「你们做事真麻烦。」然后签了字。

「麻烦是对你好。」梁志明说。

周老板抬起眼看了他一眼，没有再说什么。

几年后，周老板的店舖真的出了一次意外——不是火，是伙计在台风期间抢救存货时摔伤了腿。没有意外险，所有医药费都是周老板自己掏的。他打电话给梁志明，语气比当年更沉：「当年你说的那些话，现在我全明白了。」

梁志明把那份签了字的记录从档案里找出来，复印了一份寄给周老板。周老板留着，夹在自己店里的账本中间，偶尔拿出来翻一翻。

## 原文定义

持牌保险代理人向客户提供的受规管意见，应是合理的持牌保险代理人根据由客户取得的资料，包括客户情况，而认为适合客户的意见；如客户坚持作出与建议相反的关键决定，该代理人应记录并妥善备存该建议、客户的原因、该代理人的解释，以及该决定由客户自行作出之事实。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明三次上门收集周老板的情况（店舖、库存、伙计、借款） | 根据由客户取得的资料（包括客户情况）作出建议 |
| 梁志明提供「财产、火险、意外」三位一体的综合建议 | 认为适合客户的意见 |
| 周老板决定只保火险，梁志明记录并请他签字确认 | 记录并妥善备存建议、客户原因、代理人的解释及「客户自行作出之决定」 |
| 几年后周老板因意外自掏医药费，翻出那份记录 | 记录保护双方，证明程序合规，客户了解保障缺口 |
