# 6.1.1g 干预权力(Powers of Intervention)

## 故事

1985年，监管机构派人到中环某英资保险公司做了一次非例行巡查。

梁志明在前台见到几位陌生的官员进门，神情严肃，手里拿著文件夹。Mr. Thompson亲自出面接待，梁志明被安排去复印文件，没有听到太多内容。但从气氛判断，这不是一次普通的巡查。

晚饭时，Mr. Whitfield在茶餐厅坐下来，和梁志明聊起了这件事。

"Whitfield哥，公司出事？"梁志明压低声音问。

"未至于出事。"Mr. Whitfield说，"但系监管机构注意到了一些情况——这间公司的保费收入增长过快，业务扩张的速度超出了他们财务能够支撑的范围。监管机构担心这样下去，公司迟早应付不来赔付。"

"点解会咁？"梁志明问。

Mr. Whitfield解释："就好似一间茶餐厅，老板贪心，想一口气开三间分店，但系本钱唔够、师傅唔够、食材钱都紧张。短期内睇起身好风光，但系厨房周转唔过来，迟早出事。"

梁志明明白了。"监管机构想点？"

"佢哋打算干预。"Mr. Whitfield说，"《保险业条例》赋予监管机构一系列权力，当保险人出现问题的时候，可以采取行动。"

他从公事包里拿出一叠文件，上面列举了各类干预权力。

"第一，限制保费收入。"Mr. Whitfield说，"如果某间公司业务增长太快，监管机构可以要求佢哋暂停接收新业务，等财务状况稳定了再恢复。就好似茶餐厅，老板周转唔过来，银行就叫佢先停一停新订单，等周转过了再做。"

"第二，投资限制。"他继续说，"监管机构可以限制保险公司把资金投向边类资产、投向边个地区，防止佢哋把太多钱放喺高风险的地方。"

"第三，对新业务的限制。"Mr. Whitfield说，"限制公司订立或更改某些种类的合约，等于话俾佢哋知：呢类生意你暂时唔准再做。"

"第四，由认可受托人保管资产。"他说，"监管机构可以派人睇住保险公司的资产，确保资产唔会被随意转移或挪用。"

"第五，特定的精算调查。"Mr. Whitfield说，"如果监管机构怀疑保险公司的财务状况有问题，可以要求佢哋马上做一次详细的精算调查。"

"第六，委任经理。"他说，"如果情况比较严重，监管机构可以派一个经理入去接管间公司，直接管住盘数。"

"最严重的情况呢？"梁志明问。

"清盘。"Mr. Whitfield说，"如果保险公司真系周转唔过来，监管机构可以向法院提出呈请，要求将保险公司清盘。就系话，间公司执笠，破产清算，优先赔付俾香港的保单持有人。"

梁志明沉默了一会。"呢啲权力听起身好严厉。"

"系。"Mr. Whitfield说，"但系正因为有呢啲权力，监管机构先可以喺问题未去到最严重的地步之前及时处理，保障投保人的利益。"

他放下筷子，看著窗外的街景："我初做呢行的时候，觉得干预系一种惩罚。后来先明白，干预系一种保护——保护投保人，也保护行业本身。"

梁志明点点头。他想起小时候见过一间工厂倒闭，工人的遣散费拖了半年先收到。如果那间工厂一早被监管机构盯上、提早干预，工人的损失可能会少很多。

"所以巡查唔系麻烦。"梁志明说，"系一种提前的关心。"

Mr. Whitfield笑了笑，没有否认。

## 原文定义

监管机构在保险人出现问题时有权干预，措施包括：限制保费收入、投资限制、限制新业务、由认可受托人保管资产、进行特定精算调查、委任经理管理，以至向法院提出清盘呈请。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 保费收入增长过快，超出财务能支撑的范围 | 干预原因之一：业务增长过快可能导致财务困难 |
| 茶餐厅一口气开三间分店，周转不过来 | 比喻保险公司过度扩张的财务风险 |
| 限制接收新业务，等周转过了再恢复 | 干预权力之一：限制保费收入 |
| 派人睇住公司资产不被转移 | 干预权力之一：由认可受托人保管资产 |
| 情况严重派人接管 | 干预权力之一：委任经理管理 |
| 清盘呈请，间公司执笠 | 干预权力最后手段：向法院提出清盘 |
