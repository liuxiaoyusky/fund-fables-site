# 6.2.5.2.1 公平、公正及以客户的最佳利益行事

## 故事

那天早上，梁志明坐在茶水间，对着一叠产品建议书发呆。

他手上有两份人寿保障计划：A计划的佣金比率较高，公司近期主推；B计划的保障范围更阔，客户实际需要的其实是B。但B的佣金少，公司也不怎么宣传。

他打电话给Mr. Thompson，compliance部的主管。

「Thomson sir，我有个客人，三十几岁，已婚，有两个小朋友。B计划每月供款多三百，但保障范围明显适合佢。不过我哋近期主推A，我推B的话——」

「点解唔推？」Mr. Thompson反问。

「因为A佣金高。」

「佣金高唔系问题。」Mr. Thompson说，「问题在于你推A系因为佣金高，仲系因为你认为A适合𠮶个客。」

梁志明没有回答。

Mr. Thompson接着说：「条例规定，代理人在进行受规管活动时，必须公平对待客户，考虑客户的利益，提供合适、公正、客观的意见。你只可以从你获授权推广的产品范围内，推荐最符合客户利益的那一款。」

「但系，如果客人本身对产品唔了解，佢哋点知边个最好？」

「呢个就系你嘅工作。」Mr. Thompson的语气很平静，「你嘅价值唔系介绍边个产品佣金最高，而系帮客人理解边个最适合佢。如果你唔具备呢个判断能力，你就唔应该做呢个建议。」

梁志明挂上电话，回去见了那位客人。他详细比较了两份计划的保障范围、等候期和理赔条款，然后告诉客人：「B更适合你，但每个月贵三百。如果你想再考虑，冇问题。」

客人最后选了B。

走出客人屋企时，梁志明经过一间茶餐厅，看见玻璃窗上映着自己的倒影。他想，呢三百蚊，可以话系佢放低了某样野。

但佢知道客人最终选了真正适合自己的野。

## 原文定义

持牌保险代理人须公平对待客户，考虑客户利益，提供合适、公正及客观的意见；且只应在其获授权推广的产品范围内，推荐最符合客户利益的保险产品。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明起初倾向推A（佣金高但保障范围较窄） | 代理人利益与客户利益可能冲突 |
| 梁志明主动咨询Mr. Thompson | 寻求合规判断，公正客观地行事 |
| 详尽比较两份计划后再向客人解释 | 提供公正、客观、以客户利益为先的建议 |
| 客人最终自行选择了B | 代理人推荐，客人自主决定，代理人不得强迫 |
