# 6.2.5.1 与身份及职分相关的披露

## 故事

周老板有个老习惯：把重要的单据和信件都放在一个铁皮文件盒里，钥匙挂在腰间，走到哪里带到哪里。逢年过节整理东西的时候，他就拿出来翻一遍，数数今年签了多少合同、花了多少钱、谁是中间人。周老板说，这样做不是信不过谁，是信不过自己的记性。

那年周老板的海味舖要做一次全面的风险盘点，想找一家保险经纪公司帮忙比较市面上的火险产品。他让女婿去打听，女婿找了三家公司来谈。

第一家的代表姓马，来的时候西装笔挺，递了张名片，上面只印着"马先生"三个字和一行手机号，没有公司名称，没有牌照号码。周老板问："你是哪家公司的？"马先生说："我专门做商业险的，您不用管我在哪家公司，重要的是我能帮您拿到好价钱。"周老板听着不舒服，没再多问。

第二家来的是Mr. Whitfield。他先递了名片，正面清清楚楚印着名字、牌照号码、所属保险经纪公司"港汇经纪行"，还有一行字："代表客户与保险人往来"。坐下之后，Mr. Whitfield先给周老板看他的牌照原件，上面注明了牌照类别是保险经纪公司代表，然后他拿出一份表格，列出他所属港汇经纪行目前代理的保险公司名单，说："我代表您去找这些公司比较方案，最终选哪家，您自己决定。"

周老板问女婿："这个Mr. Whitfield讲得清楚吗？"女婿说："他讲了代表关系，就是说他是站在我们这边的，不是保险公司的推销员。而且他有牌照，有公司，信息都是透明的。"

第三家的代表迟到半小时，进来之后没带名片，也没带任何书面材料。周老板没让他坐下，直接说改天再约。

最后周老板选了Mr. Whitfield。签约那天，他把Mr. Whitfield的名片复印件也放进了那个铁皮文件盒里。

## 原文定义

持牌保险经纪人应向客户提供以下资料：姓名（注册名称及商业名称）、牌照号码、牌照类别、保险经纪人公司牌照或业务代表牌照、所属持牌保险经纪公司的名称、就购买保单事宜代表客户与保险人往来；代表超过一间保险经纪公司时须说明代表哪一家；名片须显示身份证明上的姓名、牌照号码、牌照类别及所属持牌保险经纪公司名称。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| Mr. Whitfield名片显示姓名、牌照号码、所属港汇经纪行 | 须披露姓名、牌照号码、牌照类别及所属公司 |
| 名片印有"代表客户与保险人往来"字样 | 须披露代表客户与保险人往来的事实 |
| Mr. Whitfield拿出保险公司名单，说最终由客户决定 | 须明确说明代表哪一家经纪公司 |
| 马先生名片无公司名、无牌照号，被周老板质疑 | 未披露规定内容，不符合要求 |
| 周老板把名片复印件归档保存 | 客户可核实经纪人身份的来源依据 |
