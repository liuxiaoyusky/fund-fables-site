# 6.2.1.3 骚扰、强迫或不当影响

## 故事

梁志明在上环的保险行做到第三年，街坊都认识他了。那天他接到一个电话，是老邻居陈叔打来的。陈叔在梁志明小时候常常给他买鱼蛋吃，在茶餐厅坐了十几年，如今六十多岁了，腿脚不如从前，但每天还是雷打不动地去茶餐厅开门做生意。

"阿明，我有件事想拜托你。"陈叔的声音有些不好意思，"我那个儿子阿全，你认识的，去年回来接手了茶餐厅。他对保险一窍不通，我跟他说，他不爱听。我想请你跟他讲讲，不用让他买，就是让他知道有这回事，将来万一有事，不至于两眼一抹黑。"

梁志明答应了。星期天中午，他在茶餐厅后门见到了阿全。阿全二十出头，皮肤晒得黑黑的，手臂上有烫伤的旧疤，说话直接，不拐弯。

"我爸叫你来的吧？"阿全擦了擦手，没有让座的意思，"不用讲了，保险我见多了，都是骗人的。"

梁志明站在原地，没有急着辩解。他问阿全："你最怕茶餐厅出什么事？"

阿全愣了一下，然后说："最怕客人吃完饭摔了跤，还有厨房起火。我爸上年就差点出过事，煤气炉的火舌蹿上来，差点烧到窗帘。"

梁志明听着，没有打断。他让阿全把话说完，然后慢慢讲，有什么方法可以让人在出事之后不用一个人扛。阿全听了一会儿，表情有些松动。

但当梁志明进一步解释意外保障的细节时，阿全突然拉下脸："你不要讲了，你讲这么多，不就是叫我买你的东西吗？"

梁志明感觉到气氛变了。他意识到，如果他现在顺着陈叔的请托，继续跟阿全讲下去，那就不是在提供建议，而是在用"老街坊的面子"推着阿全做一个他还没想清楚的重大决定。

他停下来，说："阿全，我不催你。今天先到这里。你想通了，随时来找我。"

阿全没有回答，但也没有起身离开。过了几秒，他说："你是不是不想做我生意？别人都是追着我说的。"

"负责任的建议不是追着人说的，"梁志明说，"是等人家想清楚了，自己来找你。"

阿全沉默了。那天傍晚，梁志明离开茶餐厅的时候，陈叔在门口抽着烟，表情有些失落，以为儿子又把事情搞砸了。

三年后，阿全主动打电话给梁志明，说茶餐厅要做大，该认真了解一下保险了。他第一句话就说："当年你说等我自己来找你，我今天来了。"

## 原文定义

持牌保险经纪不得骚扰、强迫或对客户施加不当影响，以诱使客户订立保险合约或作出关键决定。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 陈叔请梁志明劝阿全买保险 | 第三方请托不能成为强迫客户的理由 |
| 梁志明主动停下来，不再继续解释 | 发现客户有抗拒情绪时应停止推销 |
| 阿全说"别人都是追着我说的" | 持续施压属于不当影响 |
| 梁志明说"等人家想清楚了再来" | 负责任的建议尊重客户的自主决定权 |
