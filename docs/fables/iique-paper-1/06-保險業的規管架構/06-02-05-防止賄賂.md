# 6.2.5.1.4 防止贿赂

## 故事

何师奶的茶餐厅在油麻地做了十五年，街坊都知道她精明也讲信用。丈夫走后她一个人撑着，供女儿读完护士学校，又帮外甥阿明在保险公司找了一份跑腿的差事。阿明上进，有时跟着公司的人去见客，回来就跟姑妈讲外面的事。

那年冬天，有个自称是某保险公司业务主任的人来茶餐厅找到阿明，说："你姑妈这间舖子，还有她那些街坊朋友，都是熟客。你帮我们介绍几单生意，每介绍成一单，给你一个'利是'，一千块。"阿明算了算，觉得不错，正要答应，何师奶从后厨走出来，问了句："什么利是？"

业务主任说："何师奶，没什么大事，就是帮我们保险公司介绍客户，成功了有一点心意。"何师奶又问："这笔钱，是你私人给的，还是公司给的？"业务主任说："公司批的，但经我手给。"

何师奶没答应，也没拒绝，只是笑了笑，叫阿明送客。

那天晚上阿明回来，何师奶把这件事跟他说了："阿明，你出来做事，有些钱不能拿。不是因为数目大小，是因为拿了这笔钱，你将来介绍客户的时候，心里就有一只手在推你——推你去找最赚钱的，而不是最合适的。你姑爹当年就是吃了这种亏，帮朋友办事，朋友送了一笔钱，没多久就被人说闲话，闹到差点连舖子都开不下去。"

阿明去找了Mr. Thompson，把这件事原原本本说了。Mr. Thompson听完，跟他说："何师奶的直觉是对的。你是持牌保险代理人，你的主事人是你的派出机构。这笔钱，如果是你主事人没批过，你拿了，就是违规。"

阿明后来才知道，Mr. Thompson后来专程去那家保险公司走了一趟，确认那笔"利是"确实是公司行为，但程序上绕过了正常报批。Mr. Thompson让对方补了内部审批文件，才让阿明收下那笔钱——但前后拖了一个月，那段时间阿明一个客户都没介绍。

何师奶后来炖了一盅老火汤给阿明，说："慢一点没关系，路走得正就好。"那盅汤阿明喝到半夜。

## 原文定义

持牌保险代理人不得违反《防止贿赂条例》，未事先得到其受委任保险人或受委任代理机构许可，不得索取或接受任何人的利益作为就该受委任保险人或代理机构事务或业务采取行动的诱因或报酬；同样，不得在未获主事人许可下向代理人提供利益。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 业务主任提出"介绍成功就发利是"，不经正常报批 | 未获主事人许可不得索取或接受利益作为办事诱因 |
| 何师奶追问"私人给还是公司给" | 利益提供须有正当授权来源 |
| Mr. Thompson要求对方补完内部审批文件后才可发放 | 未获受委任保险人或代理机构许可不得接受利益 |
| 阿明在审批期间未介绍任何客户 | 合规程序优先，业务机会让步 |
| 何师奶讲丈夫当年"吃了亏"的故事 | 《防止贿赂条例》是保护客户和从业者的双方向规矩 |
