# 6.2.2 获授权保险公司与其代理人的关系

## 故事

上环的老街上，周老板的海味铺开了三十年，街坊邻居都认得。他每年在阿旺那里投保，阿旺是保险公司的代理人，穿着西装、夹着公文包，在上环一带的商号之间走了十几年。

周老板和阿旺很熟。阿旺每次来，都会在海味铺买两斤鲍鱼，说是帮衬街坊。周老板觉得，通过阿旺投保，就等于直接跟保险公司打交道。签个名，投个保，出事之后找阿旺，阿旺帮他们跟进，天经地义。

三年前的一个下午，阿旺突然不来海味铺了。周老板听别人说他跳槽了，去了一家新的保险公司。他没有收到任何通知，心里还惦记着那份还没到期的保单。后来听说阿旺已经不在原来的公司做了，就继续在阿旺那里续保。一年后，阿旺又回来了，说现在同时代理三家保险公司的产品，可以帮周老板挑选更合适的。

周老板续了约。

有一天晚上，阿旺拿了一瓶酒来海味铺，说："周老板，我跟你商量一件事。我现在同时代理三家保险公司，每家产品不一样，我可以帮你挑最划算的那家。"他当场拿出一份新的投保书，让周老板签名。周老板没怎么看，就在签名栏按了指印。

半年后，海味铺起了火。电线老化，烧了半间仓库，存货损失了差不多二十万。周老板报案，阿旺一开始接电话，后来就不接了。去了阿旺的办公室，人去楼空。周老板找到那家新的保险公司，客服说阿旺只是代理他们公司的某些产品，不包括周老板买的那款。那份新合约，根本不在阿旺的授权范围内。

周老板这才慌了。他找到阿旺理论，阿旺说："周老板，我已经不在那家公司做了，这个不是我负责的范围。"周老板坐在海味铺的门槛上，想不明白——阿旺明明穿着保险公司的制服，用的是保险公司的文件，为什么出事了，公司不认？

他把这件事投诉到其中一家授权保险公司。客服听了他的情况，说："周老板，根据《保险业条例》第68条的规定，如果代理人在授权范围内行事，授权保险公司是要承担责任的。但您这个案例比较复杂，涉及代理人同时代理多家公司，而且他卖给您的那款产品不在他对您的授权范围内。"

周老板的案子在公司内部引起了注意。授权保险公司承认阿旺是他们的代理人，但他同时代理多家保险公司，而向周老板推荐并签署的那款产品不在他对周老板的授权范围内。更重要的是，阿旺没有把这件事清楚地告诉周老板。最后，授权保险公司出于对客户负责的态度，对周老板的损失作了适当安排。

张小姐是公司的合规助理，她在处理周老板这个案例时，花了几个星期整理所有客户记录，确保没有其他客户遇到同样的问题。Mr. Thompson 在合规部开了一个内部会议，说："第68条的道理很简单——客户冲着保险公司的名字来，签的是保险公司的文件，公司就要对这个客户负责。阿旺的错，不能让客户一个人扛。"

周老板后来每次投保，都多了一个习惯：签名前，先打电话到保险公司确认代理人的身份，还要一份书面的代理人授权书确认函。他说："白纸黑字看过再签，安心一点。"

## 原文定义

《保险业条例》第68条规定，获授权保险公司须为其代理人在授权范围内的行为向客户承担法律责任，不论该行为是否在代理人的权限范围内。若代理人同时代理多家保险公司，授权保险公司须就其授权范围内的行为承担责任；若代理人超出授权范围行事，须视具体情况确定责任承担方。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板认为"找阿旺就等于找公司" | 客户信任的是保险公司的名字和授权 |
| 阿旺同时代理三家保险公司 | 代理人可获多家保险公司委任 |
| 阿旺卖周老板的产品不在授权范围 | 代理人超出授权范围时产生复杂责任问题 |
| 授权保险公司对周老板作出赔偿 | 保险公司须对代理人在授权范围内的行为负责 |
| 周老板后来要求书面确认函 | 客户有权知晓代理人是否在授权范围内行事 |
