# 6.2.2.3 提供公平及公正并符合客户最佳利益的受规管意见

## 故事

Mr. Whitfield 是英资律师楼的合规律师，管着好几家保险公司的合规文件，每个经纪的培训记录都要经他签字。那天下午，他从律师楼走到梁志明工作的保险行，带了一叠文件来开季度合规会。

会议开到一半，Mr. Whitfield 把一份投诉记录抽出来，放在台面上。记录很简单：一位客户投诉，经纪推荐了一个产品，客户买了，三个月后发现产品不符合自己的需要。

"阿明，这份投诉是你的。"Mr. Whitfield 声音平静，没有任何责备的意思，"我不是来追你责任的。今天叫你来，是想跟你理清楚一个流程。"

他翻开一份表格，说："经纪提供受规管意见之前，需要做什么？第一，充分了解客户的情况，包括他的家庭、收入、已有的保障。第二，在这个基础上，给出有合理依据的建议，而不是听说客户想买什么，就直接卖给他什么。第三，给客户讲清楚，你为什么推荐这个，而不是其他的。"

梁志明低着头听。他想起上一次周老板来的时候，他听完周老板说"我想买一份当嫁妆"，就给了一个推荐，前后不到十五分钟。那十五分钟里，他甚至没有问过周老板：家里还有没有别的负担，供款会不会影响生意周转。

"阿明，"Mr. Whitfield 放下文件，"经纪给的每一句话，都要经得起问。你的建议是根据客户的情况来的，你能讲清楚为什么，这个建议才是负责任的。"

三个月后，周老板再来公司的时候，梁志明没有立刻推荐产品。他请周老板坐下，倒了一杯茶，从头问起：家里有几口人，平时铺头有没有贷款，女儿嫁过去之后住在哪里。那天他们在茶餐厅谈了很久，周老板把他这几十年做生意的风风雨雨都说了一遍。

梁志明把谈话内容记在本子上，花了几天时间，找了两家保险公司的产品，仔细比较，最后给了周老板一份手写的建议书。上面写清楚了每项选择的利弊，为什么这个比那个更适合周老板的家庭情况。周老板看完，说："阿明，你终于像个做这行的人了。"

## 原文定义

持牌保险经纪提供受规管意见前须作出合理查询以充分了解客户情况；给出的意见须有合理依据，并能帮助客户作出知情的决定。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| Mr. Whitfield 指出投诉的流程问题 | 提供意见前须充分了解客户情况 |
| "你为什么推荐这个" | 建议须有合理依据，经得起追问 |
| 梁志明用十五分钟听完就推荐 | 未充分了解情况就给建议属于失职 |
| 梁志明用几天时间比较两家产品 | 基于客户具体情况，公正客观地推荐 |
| 手写建议书列明利弊 | 向客户提供充分资料以助其作出知情决定 |
