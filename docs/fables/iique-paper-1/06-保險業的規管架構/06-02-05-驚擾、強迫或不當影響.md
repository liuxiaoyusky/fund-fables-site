# 6.2.5.1.3 惊扰、强迫或不当影响

## 故事

湾仔的早晨，叮叮车从轩尼诗道哐当哐当地驶过。何师奶的茶餐厅刚开门，伙计们在水吧后面忙着冲奶茶，门口蒸笼里的叉烧包冒出一股白气。

梁志明在这家茶餐厅对面的一家小保险行做事，刚入行半年。这天早上，他被师傅老刘叫到一边。

"对面何师奶，上次说考虑一下那个住院保障，今天你去签单。"老刘把一张已经填好一半的表格推过来，"她女儿下个月要生孩子了，这档口最合适。你去跟她说说，催一催。"

梁志明接过表格，有点犹豫："刘叔，她上次说想和女儿商量商量……"

"商量什么？商量来商量去生意都跑了。"老刘压低声音，"你去的时候，语气硬一点，就说这是公司这个月的主打产品，过了这村没这店。她一个开茶餐厅的，哪懂那么多，让她签就是了。"

梁志明站在原地，捏着那张表格，没有说话。

他走进茶餐厅的时候，何师奶正在柜台后面算账。她认得梁志明，招呼他坐下，让伙计端了一杯奶茶过来。

"何师奶，上次那个住院保障的事，您考虑得怎么样了？"梁志明问。

"我正要跟我女儿说呢，"何师奶放下笔，"她下个月生小孩子，我想等她确定了，再一起看看那款产品适不适合。"

梁志明想起老刘的话，顿了一下。但他的目光落在何师奶柜台上——那里放着一张老照片，照片里的男人穿着旧式西装，站在茶餐厅门口。何师奶的手指偶尔会无意识地碰一碰那张照片的边框。

那是她已经去世的丈夫。

"何师奶，这个产品确实不错，不过您和女儿一起看会更清楚。"梁志明把表格收起来，"您不用急，等您和女儿商量好了再找我，我帮你们详细解释。"

何师奶抬起头，有些意外。

"老刘说这个月必须签的。"她说。

"那是他的想法，不是您的。"梁志明站起来，"出事的时候赔钱的不是老刘，是您和您的家人。保障这件事，应该您和女儿一起决定，不是别人催出来的。"

何师奶沉默了几秒，然后轻轻点了点头。

"阿明，你和别的保险人不一样。"她把那张照片重新摆正，"我老公以前说过，做生意要老实。这杯奶茶算我请的。"

梁志明走的时候，老刘在保险行门口等着，脸色不太好看。

"没签？"老刘问。

"她要和女儿一起看。"梁志明说，"催她也没用。"

"你这个人怎么——"

"刘叔，"梁志明打断他，"她先生走了，茶餐厅就她一个人撑着。她要的是一份保障，不是一份被人催着签的字据。她信我了才会找我，信我不是因为催得紧，是因为我让她自己想清楚。"

老刘张了张嘴，最终没有再说什么。

那天傍晚，梁志明绕路经过茶餐厅，看见何师奶正和一个大肚子的年轻女人坐在里面，两人对着桌上的几张纸在商量什么。何师奶抬头看见他，招了招手。

"阿明，进来坐。我女儿来了，我们想约你明天详细介绍那款产品。"

梁志明笑了笑，点了点头。

他没有硬销一份合约，却赢得了一张信任。

## 原文定义

持牌保险代理人不得骚扰、强迫或对客户施加不当影响以诱使客户订立保险合约或作出关键决定。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 老刘要求梁志明"语气硬一点""催一催" | 强迫——以压力手段催促客户快速决定 |
| 梁志明把表格收起来，让何师奶和女儿商量 | 不得施加不当影响——尊重客户自主决定的权利 |
| 何师奶柜台上的老照片 | 背景动机——她需要的不是被催促，而是一份真正适合的保障 |
