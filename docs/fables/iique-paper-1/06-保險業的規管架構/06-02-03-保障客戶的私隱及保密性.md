# 6.2.3.4 保障客户的私隐及保密性

## 故事

梁志明在保险行做到第五年，周老板已经成了他最熟的老客户之一。海味铺逢年过节会送鲍鱼礼包，梁志明去上环办事的时候会顺路坐一坐。两人喝茶聊天，周老板什么都讲：铺头进了新货，儿子闹了婚变，老太太去年住了三个月医院。这些话，周老板在街坊面前不会说，但在梁志明面前说得自然，因为他是投保人，这些事跟保单有关，说了才有人帮他评估风险。

有一天，梁志明在茶餐厅吃早饭，听到邻桌有人在聊天。是张小姐和另一家保险公司的经纪。那位经纪的声音不大，但在安静的茶餐厅里，每个字都听得很清楚。

"你们那个周老板，最近怎么样？"那个经纪问。

"还好，"张小姐说，"我听说他儿子刚结完婚，摆了几十桌，花了不少钱。这几年海味生意不错，他应该赚了点。"

梁志明手里的筷子停住了。他认得周老板的声音轮廓描述，那分明就是他们公司张小姐在跟别人说周老板家里的事情。

他站起身，走到张小姐那一桌，低声说："Miss Cheung，这里是公共场合，你们在聊客人的私事。"

张小姐抬头，看见是梁志明，愣了一下，然后脸色变了。她压低声音，说："阿明，我不是有意的。这位是新来的经纪，不懂规矩，我就是顺嘴说两句。"

"顺嘴也不行。"梁志明的声音很平静，但语气没有退让，"你把他的家庭情况、工作情况、经济状况都说出来了。虽然没有说姓名，但这些细节拼在一起，他很容易被认出来。客户把这些事告诉我们，是因为他相信我们会保密。"

张小姐低下了头。那个新来的经纪也满脸通红，收拾东西先走了。

后来梁志明跟张小姐说："我们这行，接触到的都是客户的私事。隔墙有耳，在茶餐厅说，在电梯里说，在聚会上说，都可能被不该听到的人听到。我们能做的不多，就是自己管好自己的嘴。"

张小姐点点头，说："阿明，我懂了。"

从那以后，梁志明多了一个习惯：每次离开座位，都把客户文件收进抽屉；每次下班，都确认文件柜的锁是关好的。他不在公共场合谈论任何客户的事情，逢年过节跟朋友聚会，有人问起保险行业的事，他只讲"这个行业有意思的地方"，不讲具体是哪个客人、遇到了什么事。

这些事情很小，小到没有人会在意。但正是这些小事，守住了客户把最私人的事情交给他们时的那一份信任。

## 原文定义

持牌保险经纪须视所有与客户有关的资料为机密资料，不得使用或披露该等资料，唯为进行受规管活动、经客户书面同意或为遵守法律规定之目的除外。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 张小姐在茶餐厅谈论周老板的家庭情况 | 将客户资料用于非授权场合属违规 |
| 梁志明指出"客户把事告诉我们是信我们" | 客户信息是建立在信任基础上的机密 |
| 张小姐说"顺嘴说两句" | 日常闲聊也可能泄露客户隐私 |
| 梁志明离开座位就收文件 | 妥善保管客户资料是日常责任 |
| 梁志明不在聚会提及具体客户 | 客户信息不得用于授权范围以外的用途 |
