# 6.2.5 其他监管措施

## 故事

整间会议室安静得可以听见冷气机滴水的声音。

Mr. Whitfield坐在长桌一端，面前放着一叠文件和一支笔。梁志明坐在他旁边，手心有点出汗。会议室的另一端坐着两位陌生人——一位穿着西装的中年男士，和一位拿着公事包的年轻女子。他们没有自我介绍，只是点了点头。

「两位是保监局的查察员。」Mr. Thompson在门口低声对梁志明说，然后走进来坐在他另一侧。

Mr. Whitfield礼貌地问：「两位今次前来，希望查察哪些范围？」

那位男士打开公事包，取出一叠文件：「我们希望查证贵公司就持牌业务代表的投诉处理程序，以及相关的内部监控是否足够。」

接下来的整个下午，查察员查阅了大量的内部文件——培训记录、投诉处理表格、负责人的委任文件、合规自查报告。张小姐在一旁负责调取文件，每一次查察员提出要求，她都能在十五分钟内递上相关资料。

查察结束后，Mr. Whitfield送查察员离开。回到会议室，Mr. Thompson对梁志明说：「今日算系顺利。查察员主要睇你有冇设立有效的内部监控，以及你哋嘅持牌业务代表系咪仍然系适当人选。」

「如果发现问题呢？」梁志明问。

「保监局可以撤销或暂时吊销牌照，可以公开或非公开谴责，可以罚款，最高一千万，或者视乎情况罚三倍于所涉及的利润或避免的损失。」Mr. Thompson的语气很平静，但每一个字都像石头落在桌面，「亦都可以禁止一个人喺某段期间内申请牌照。」

「保监局点解有咁大权力？」

Mr. Whitfield把文件合上，说：「《保险业条例》授权保监局订立规则、颁布准则及指引，并可对不遵守的持牌中介人采取纪律行动。权力大，因为影响的唔系只有公司，而系所有保单持有人的利益。」

梁志明望着窗外维多利亚港的船只，想起那天周老板问他的一句话：「你哋保险公司，有人管你哋㗎？」

他现在知道答案了。不只是有人管，而且管得很仔细。

## 原文定义

根据《保险业条例》，保监局有权进行查察及调查，并可对持牌保险中介人施加纪律制裁，包括撤销或暂时吊销牌照、公开或非公开谴责，以及罚款最高一千万港元等。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 查察员来自保监局查察持牌机构 | 监管机构的查察权力 |
| 张小姐能迅速调取完整的内部记录 | 机构须设立有效的内部监控与记录保存 |
| Mr. Thompson介绍罚款、吊牌等制裁选项 | 保监局可施加的纪律制裁种类 |
| 梁志明理解持牌人必须持续合规 | 不当行为可导致持牌人失去「适当人选」资格 |
