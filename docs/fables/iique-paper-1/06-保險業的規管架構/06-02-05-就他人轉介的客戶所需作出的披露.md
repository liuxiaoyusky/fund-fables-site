# 6.2.5.5 就他人转介的客户所需作出的披露

## 故事

周老板的侄女阿萍要从广州来香港开服装店，周老板说："志明，我侄女信我，我话你靠得住，你帮她搞掂间铺的保障。"

梁志明说好，约了阿萍在茶餐厅见面。阿萍带了周老板的介绍信，还有一袋她妈妈从广州寄来的干货样品，说顺道帮他尝尝新到的瑶柱。

聊到正事，阿萍问："周叔话有个朋友叫阿强，说这间保险公司是他熟悉的，阿强可以帮我安排，费用佢帮我倾掂。我系唔系直接揾阿强就得？"

梁志明放下筷子，想了想。他想起 Mr. Whitfield 那天在文件里画了红线的部分。

"阿萍，我同你讲清楚，"他说，"周老板转介你嚟揾我，责任就系我对你。你揾阿强，得；你听阿强的意见，出了问题，阿强对你没责任。但你揾我，我就要对你负责。"

阿萍有点糊涂："但阿强话佢认识保险公司的人，倾嘢方便。"

"系咁嘅——"梁志明把话说慢了点，"阿强系转介人，佢唔代表我，亦都唔应该帮你安排保单。保单嘅事，你只应该同我倾。保费直接交俾我，或者交俾保险公司，唔好经阿强。"

阿萍想了很久，说："即系周叔介绍我嚟，你系我嘅人，但阿强唔系？"

"你可以咁理解。"梁志明说，"周叔介绍你嚟，我接咗你，就系我对你负责。将来有事，你揾我，我帮你跟进到底。但如果你绕过我揾阿强，到时出咗事，唔知有几多人会企出嚟。"

阿萍低头喝了口奶茶。过了一会儿，她说："志明，咁我正式俾你做。你话保费交几多、几时交，我照办。"

梁志明把那袋瑶柱放回她手里，说："你妈咪寄嚟嘅嘢，你拎返去。我这边搞完，请你食双皮奶。"

## 原文定义

若客户由他人转介，持牌保险经纪须告知客户：经纪负责安排保单，客户只应直接与经纪往来；转介人不代表经纪，亦不参与安排保单；经纪就转介人对保单所给予客户的任何意见不承担责任；保费应直接支付予经纪或相关保险人，而非转介人。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 阿萍问是否直接揾转介人阿强 | 转介人不代表保险经纪 |
| 梁志明说保单的事只应与他直接往来 | 客户只应与经纪往来，不与转介人往来 |
| 梁志明指出保费不能经阿强转交 | 保费应直接支付予经纪或保险人，而非转介人 |
| 梁志明说"我接咗你，就系我对你负责" | 经纪对客户承担直接责任 |
