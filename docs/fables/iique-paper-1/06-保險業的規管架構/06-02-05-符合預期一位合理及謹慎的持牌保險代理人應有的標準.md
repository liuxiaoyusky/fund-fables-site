# 6.2.5.3.1 符合预期一位合理及谨慎的持牌保险代理人应有的标准

## 故事

梁志明入行第三年，Mr. Whitfield带他去过一次老字号金舖。那家金舖的东主姓关，在上环做了四十年，来谈的是一份财产综合险。关伯的女儿帮他整理账目，漏报了一批放在仓库里的陈年饰金，总值不高，但漏报了就是漏报了。

签完约过了大半年，仓库出了一次小火，抢救及时，没烧掉什么，但有几箱货物受了潮。保险公司派人来查，说投保金额和实际库存有出入，要核减赔付。关伯不服，说那批漏报的东西根本不值几个钱，火也不是那批货烧的。保险公司说，综合险讲究整体准确，漏报的部分不在保障范围。

Mr. Whitfield带着梁志明去处理这件事。在保险公司办公室里，Mr. Whitfield把理赔报告一条条核对，指出了两处可以通融的地方，帮关伯争取到八成赔付。走出保险公司大门，关伯说："Mr. Whitfield，多谢你。这次是我女儿不懂，漏报了东西，我自己也有责任。"

回去的路上，梁志明问Mr. Whitfield："这次的事，您能帮到关伯，是因为您细心，还是因为您懂规则？"Mr. Whitfield说："两个都是。但更重要的是，在签合同之前，我没有帮他把表格做完就完事——我核过他的仓库清单，发现数目对不上，让他补报了。那次他嫌我啰嗦，说几箱旧货不值几个钱，不用报。我说，写进合同里的数字，一个都不能少。"

梁志明想起自己入行时，师傅教他的那句话："做这行，手要稳，眼要准，该问的要问清楚，该记的要记清楚。"他当时不懂，现在懂了——那个"该"字，就是合理的谨慎标准。不是过度谨慎，也不是马虎了事，而是像师傅那把旧木尺，该量几寸就量几寸，不多不少。

后来梁志明带新人，第一件事就是教他们：帮客户填表之前，先把事实核对一遍；表填完了，再把数字复核一遍；递上去之前，把副本留一份在档案里。他说："这些动作加在一起，就是一个谨慎的代理人该有的标准。"

## 原文定义

持牌保险代理人应一直以合理标准的谨慎、技巧和努力进行受规管活动；合理的谨慎标准是预期一位进行受规管活动的谨慎专业保险代理人的标准。若代理人聘用他人提供支援，须确保该人具有诚信和胜任能力，并尽职监督。
