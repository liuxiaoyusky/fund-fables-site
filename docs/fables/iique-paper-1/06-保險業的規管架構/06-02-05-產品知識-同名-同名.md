# 6.2.5.4.1 产品知识

## 故事

陈伯在银行做了三十七年账房，看过无数张保单，替自己的子女也买过分红险和医疗险。他有一个习惯——拿到任何一份合约，先翻到"除外责任"那一页，从后往前看。他说，正常的条款告诉你赔什么，倒数第二页告诉你不赔什么，那才是这份合约真正的面目。

那年陈伯的外孙要结婚了，他想把一份储蓄型的保单转给外孙作贺礼。洋行的保险代理人来了两趟，第一趟讲产品好，第二趟讲收益高。陈伯听着，觉得哪里不对，便问他："这份保单有没有等待期？有没有退保费用？"代理人愣了一愣，说："这个我回去确认一下。"

陈伯没等他确认，自己翻出产品手册，一页页看下去，发现这份储蓄险的前五年退保，现金价值只有已缴保费的四成。他又问："那意外保障呢？有没有独立的意外附加险？"代理人答不上来，说公司产品很多，不一定每款都熟悉。

陈伯后来找来了梁志明。梁志明把产品手册摊开，一样样解释：这份是储蓄险，主要功能是积累现金价值，不是提供即时意外保障；如果要加意外险，需要另外附加，而且附加险有等待期，在等待期内因意外产生的医疗费用不在赔付范围。

陈伯听完，说："我外孙要的是一份能看得清楚的东西，不是数字游戏。"他最后没买那份储蓄险，而是换了一份定期寿险加一个意外险的组合。梁志明帮他讲清楚了每一个险种的保障范围和除外责任，陈伯签完字，把条款复印件放进了一个牛皮纸信封里。

那天傍晚陈伯把这件事写进了日记，他说：买东西要看清楚，听人讲也要看清楚。两样都清楚了，才不会后悔。

## 原文定义

持牌个人保险代理及持牌业务代表应就其可能进行受规管活动的各类保险产品的性质及主要特点，以及所承保及相关风险有充分了解。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 陈伯看保单先翻"除外责任"页 | 须了解产品的性质和主要特点 |
| 梁志明解释储蓄险的现金价值和退保费用 | 须了解产品的具体特点和风险 |
| 梁志明说明储蓄险不含意外保障，需另外附加 | 须了解各类产品的不同保障范围 |
| 陈伯要求分清楚各险种的赔付范围和除外责任 | 须对产品所承保的风险有充分了解 |
