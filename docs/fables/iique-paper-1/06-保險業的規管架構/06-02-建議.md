# 6.2 建议

## 故事

那个礼拜，梁志明心事重重。

张小姐在午饭时间的茶餐厅里坐到他对面，开门见山问他是不是出了什么事。他端着那杯已经放凉的奶茶，想了想，最终还是说了。

原来是他手上正在跟进的周老板。周老板的生意最近红火，想扩大仓储和库存覆盖范围，张小姐已经查出他的店舖目前只有火险，财产保障和意外保障统统不够。张小姐帮梁志明整理了一份详细的建议清单，梁志明也已经跟周老板见过两次面，周老板原则上接受了他建议的保障方案。

但问题出在汤逊先生那里。

汤逊先生在下午把梁志明叫进了办公室，语气不重，但意思很清楚：这个月公司有一款理财产品，配额相当紧俏，其他代理都盯着，汤逊先生希望梁志明把自己手上的那份配额让出来，匀给另一个有长期大客关系的老客户。周老板的单子，下个月再说。

梁志明知道这款理财产品是汤逊先生部门今年的主推项目，推成了，他的季度考评会好看，公司也会满意。但周老板的保障不能等——上环那些潮湿的库存，台风季节随时可能出事，张小姐的建议是有道理的。

张小姐放下筷子，问他：「汤逊先生有没有说，这款理财产品跟他有什么关系？」

梁志明想了想，说：「他说了，公司今年有压力要推这款产品。」

「那就是冲突了。」张小姐说得很平静，「公司要推这款产品，他的部门有考评压力；他让你让出配额，他的关系户拿到理财产品，他自己得益。但周老板现在需要的是财产和意外保障，不是理财产品。这两件事搁在一起，你怎么选？」

梁志明没有立刻回答。

张小姐接着说：「总原则只有一条：有冲突，就让对方知道。自己想办法不要让，让不了，就说出来。」

当天下午，梁志明去找汤逊先生。他没有拒绝，也没有立刻答应，而是把他从张小姐那里得知的情况说了出来：公司今年有这款理财产品的推广压力，汤逊先生让他让出配额，这个安排可能会影响他对周老板的建议独立性。

汤逊先生的脸色有些不好看，但梁志明接着说：「我打算把这个情况如实告诉周老板——公司的关系、您的安排、还有我自己的想法，都跟他说清楚。他自己判断要不要接受这款理财产品。」

汤逊先生沉默了一会儿，说：「你考虑清楚了？」

「考虑清楚了。」梁志明说，「这样对大家都清楚。」

汤逊先生没有再说什么，只是摆了摆手，让梁志明出去了。

那个星期六，梁志明和张小姐一起去上环找周老板。张小姐把一份文件递给梁志明，让他自己念给周老板听。文件上写着：

「本公司与上述理财产品存在业务关系。本公司管理层对该产品的销售设有考核目标。汤逊先生作为合规部主管，曾就此向梁志明提出建议。梁志明确认，他将根据周老板的实际情况独立评估，并提出符合周老板利益的建议。」

周老板把文件翻来覆去看了两遍，没有生气。他摘下老花眼镜，说：「我做了几十年海味生意，我知道每行都有自己的规矩。你直接跟我说了，我反而知道你这个人是能说真话的。」

他重新坐回椅子上，把那份保障方案拿出来，跟梁志明确认了每一项的细节，最后签了字。

汤逊先生后来经过看见周老板的签字，皱了皱眉，但什么都没说。

张小姐事后对梁志明说：「你看，坦诚没有那么难。不是让你牺牲什么，是让你少一点瞒着瞒着的精力，大家反而信得过你。」

梁志明把那张披露文件的副本夹在周老板的档案里，继续做着每天的日常工作。

## 原文定义

持牌保险经纪应尽其最大努力避免利益冲突，而在无法避免该等冲突的情况下，该经纪应透过适当披露管理该些冲突，以确保客户在任何时间均受到公平对待。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 汤逊先生要求梁志明让出配额，同时推销公司理财产品 | 利益冲突：经纪的建议可能受到公司考核目标的影响 |
| 梁志明主动向汤逊先生说明情况，表明这可能影响其建议独立性 | 最大努力避免利益冲突 |
| 梁志明和张小姐主动向周老板出示文件，披露公司、理财产品、汤逊先生安排之间的关系 | 透过适当披露管理冲突 |
| 周老板看过披露文件后仍然决定接受梁志明的建议并签字 | 确保客户在任何时间均受到公平对待 |
| 汤逊先生皱眉但没有干预，文件正式存档 | 适当披露使各方关系透明，冲突得以被管理而非被掩盖 |
