# 6.2.7.3 避免让自身利益影响客户的决定

## 故事

梁志明从保险行下班后，会去西环码头那边帮父亲看一阵鱼档。鱼档不是他的主业，但他每天傍晚收工前都会过去理货、记账，顺道把第二天的报价单交给父亲。那是他从小走到大的路，也是他最早学会算账的地方。

那年冬天，周老板想把海味铺和家里住的那栋旧楼一起投保。他想了想，与其分开买，不如找同一个经纪，省得出事了要跟两间公司打交道。他打听到梁志明，便约了他到店里谈。

梁志明把条款逐一讲完，周老板问："你做保险，咁你爸咪系你阿爸，你帮我投保，你阿爸间铺你帮唔帮衬？"

梁志明愣了一下，笑着说："帮嘅，我爸档口都系我帮手打理。"

周老板眉头一皱："咁你帮我买果份嘢，会唔会因为想抽你阿爸笔钱，净系叫我买啲便嘅？你阿爸果份你又会不会叫我买贵啲？"

梁志明知道这不是在质疑他的人品，而是在问一个很实在的问题。他把茶杯放下，认真地想了想，然后说：

"周老板，你问得好。我帮我爸打理鱼档系我屋企事，同我卖保险系两码嘢。货物险系睇你铺头嘅货值同风险，住宅火险系睇栋楼嘅结构同位置，两者唔可以绑在一齐卖。我可以讲清楚，我阿爸间档口我系自己睇住，唔会因为我卖保险俾你而有任何分别。"

他停了停，又说："但系如果你觉得我同时打理自家铺头会影响你嘅判断，你可以揾其他经纪。我唔会因为你走咗呢单就摆我阿爸喺度。"

周老板听完，点了点头。"好，你讲得清楚，我就信你。不过我有个条件——你以后有任何其他兼职要做，记得先讲我知，我唔怕你做嘢，就怕你闷声唔出气。"

梁志明答应了。那天夜里，他把这件事写进了自己的合规记录本，注明客户已就兼职情况知悉，且同意继续委托。后来 Mr. Thompson 翻查他那一季的档案，看见这条备注，点了点头，没有多说什么。

梁志明合上档案夹，想起周老板那句"闷声唔出气"。他懂那意思——做生意最怕的不是对方同时做几样，而是对方做了几样却不让客人知道。

## 原文定义

持牌保险经纪拥有其他业务或职业时，应避免该等其他业务或职业的利益与客户的利益有任何冲突。若无法避免，应在可行情况下尽快向客户披露该等冲突，并对客户公平行事，将客户的利益置于经纪在其他业务或职业的利益之上。此外，持牌保险经纪如与同属一公司集团的关联公司就某保单提供服务，应采取措施透过披露等方式避免或管理潜在利益冲突，确保客户在任何时间均受到公平对待。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明同时打理父亲的鱼档和保险行 | 持牌经纪拥有其他业务/职业 |
| 周老板问"会不会叫我买便啲俾阿爸" | 潜在利益冲突的具体形态 |
| 梁志明明确区分两件事：家庭事务 vs 保险委托 | 应避免利益冲突的原则 |
| 梁志明主动披露兼职情况，写入合规记录 | 无法避免时须披露冲突 |
| 周老板说"有任何兼职记得先讲我知" | 透明披露是客户信任的基础 |
