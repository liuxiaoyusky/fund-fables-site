# 6.2.3.3 执行客户的指示

## 故事

那年秋天，周老板决定把海味铺的投保额提高三成。他让梁志明去办这件事，说完就把电话挂了。

梁志明接完电话，回到自己的卡位，开始填表格。他把那间英资保险公司的抬头纸理好，提笔准备写信——然后发现一个问题：周老板在电话里说要加三成，但他没说加哪一部分。铺面结构？存货？还是铺内设备？

他看了看手表，已经下午四点半。周老板的铺子五点收档，这个时间打过去，大概率没人接。

梁志明还是拨了电话，果然没人接。他放下听筒，提起笔，给周老板写了张便条：

"周老板台鉴：您要求调整保额，我已收到，唯细节待确认。为确保指示准确执行，请您明早来电补充。以下为待确认事项……"

他用公司信笺把这张便条折好，套进信封，第二天一早让信差送去上环。

周老板第二天早上开门，看到信封，先骂了一句"呢个后生仔真系多事"，然后还是拨了电话，把细节讲清楚了。

梁志明当日就把申请递交上去，没有拖延。

Mr. Thompson后来审他的文件，问他："你为什么不直接按三成去填？客户说三成，你就填三成，简便得多。"

梁志明说："指示不清晰，我猜住填，万一猜错，客人受损。宁可慢半日，不要错一次。"

Thompson没有再说什么，在文件上盖了章。

## 原文定义

持牌保险经纪应采取合理步骤，准确及时地执行客户指示。如发生任何延误或未能执行该等指示，应在可行情况下尽快通知客户。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明发现指示细节不清，没有猜着填 | 准确执行：须确保充分理解客户指示再行动 |
| 周老板未接电话，梁志明当日发便条补位 | 及时执行：即使无法立刻执行，也要推进到位 |
| 确认细节后才递交申请，没有延误 | 准确及时：执行前核对清楚，避免错误 |
| 梁志明向Thompson解释：宁可慢半日不要错一次 | 延误或无法执行时应通知客户，而非不了了之 |
