# 6.2.3.1 符合预期一位合理及谨慎的持牌保险经纪应有的标准

## 故事

梁志明入行头三个月，Mr. Thompson交代他一件事：每份递出去的文件，出门前自己要复查一遍。

他第一次被退回来，是因为一张续保通知信。他按模板打印出来，字迹工整，自己觉得没什么问题，交给了张小姐发出。

三天后，保险公司来电话，说这份通知信的受益人名称写错了。周老板的铺子登记名字是"周记海味有限公司"，梁志明写成了"周记海味号"。

Thompson把他叫进办公室，把那封退回来的信放在他面前。

"你知不知错？"Thompson问。

"我……我打漏咗个字。"梁志明低着头。

"问题唔系漏字。"Thompson说，"问题系你寄出之前，冇核对过文件内容。一间谨慎的保险经纪行，发出去的文件代表公司嘅信用。客人凭呢份文件知自己的权利，保险公司凭呢份文件知自己的责任。你写错一个名，客人的权利同责任都变得模糊。"

梁志明把那封信带回卡位，对着营业执照把受益人名称重新抄了一遍，再让张小姐发了一次。这次他检查了三遍。

之后每次递文件前，他都先问自己：这份文件如果出了问题，谁会受损？

张小姐后来说："你最近递嘢干净利落咗好多。"

梁志明说："吃亏食喺眼前，学一次够晒。"

## 原文定义

持牌保险经纪应一直以合理标准的谨慎、技巧和努力进行受规管活动。合理的谨慎标准是预期一位进行受规管活动的谨慎专业保险经纪的标准。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明漏写公司全名，导致受益人名称错误 | 未达合理谨慎标准：文件发出前未核对关键信息 |
| Thompson指出问题核心是"没核对"而非"漏字" | 合理的谨慎标准要求经纪对每个细节尽专业注意义务 |
| 受益人名称错误导致权利责任模糊 | 不谨慎的文件处理可能直接损害客户利益 |
| 梁志明之后每次递文件前自问"谁会受损" | 谨慎专业保险经纪的标准是行事的内在参照 |
