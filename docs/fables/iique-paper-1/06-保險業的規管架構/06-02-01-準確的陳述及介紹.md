# 6.2.1.1 准确的陈述及介绍

## 故事

张小姐做合规助理这几年，最怕的不是文件多，是梁志明写营销文案。

那年公司打算推一个新的医疗计划，目标客户是街坊小店的老顾客。梁志明花了两天写了一份宣传单，张小姐对稿子的时候，发现了三处问题。

第一处：梁志明在文案里写了"本计划的赔偿范围比市场上大多数产品更全面"。张小姐指着这行字说："你比较的是哪几家？有没有具体数据？没有数据就是笼统说法，不够准确。"

第二处：梁志明写了一个假设案例，说某位客人"选择了我们的计划后，顺利获得了赔偿"，但这个案例没有注明是真实个案还是假设情景。张小姐说："如果是真实的，要客户同意才能用；如果是假设的，要说清楚是假设，否则客人可能误以为是真实记录。"

第三处最要命：文案里有一句"本计划完全涵盖所有住院费用"，张小姐把它划掉了，换成了"在计划条款范围内报销合理医疗开支"。她说："你写'完全涵盖'，真到理赔的时候，客人拿这句话来找你，你怎么办？"

梁志明看了看那三处修改，说："我下次会注意。"

张小姐说："不是下次注意的问题。规则很清楚：持牌保险经纪人不得误导或欺骗客户，所有陈述必须是准确的，无误导或欺骗成分。你想跟客人说清楚你的产品好，没问题，但要讲事实，不要讲感觉。"

那天晚上，梁志明把文案改好了，交给张小姐过目。张小姐又看了一遍，在最后加了一句："以上资料仅供参考，详细条款以保单文件为准。"梁志明问："这句话每次都要加？"张小姐说："你自己看看规则，规则要求的就是准确、完整、不误导。"

梁志明把那句话说了一遍，点了点头。

周老板后来看到这份宣传单，说："写得比上次清楚多了。上次那句'最全面'，我看了半天也不知道保什么。"

梁志明说："这是张小姐帮我改的。"

周老板笑了一声："那就对了。她管合规的，眼睛尖。"

## 原文定义

持牌保险经纪人于进行受规管活动时，应对客户本着真诚行事；不得误导或欺骗客户，并应确保所作任何陈述或提供的资料均为准确，无误导或欺骗成分；不得作出不准确、误导或欺骗性的陈述或比较，以诱使客户订立保单或以另一保单取代现有保单；自制的广告或营销材料须含有准确资料，持牌业务代表只应使用公司提供或认可的材料。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明写"比大多数产品更全面"但没有具体数据 | 无具体依据的笼统比较构成误导性陈述 |
| 真实案例未注明客户同意或未说明是假设 | 案例使用须符合准确性要求 |
| "完全涵盖所有住院费用"被改为"在条款范围内报销" | 绝对化表述不符合准确性原则 |
| 文案最后加"资料仅供参考，以保单文件为准" | 准确、完整、不误导的要求 |
| 只能使用公司提供或认可的营销材料 | 对营销材料的合规要求 |
