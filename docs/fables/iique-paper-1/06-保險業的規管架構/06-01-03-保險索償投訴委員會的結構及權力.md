# 6.1.3a 保险索偿投诉委员会的结构及权力

## 故事

湾仔的电车总站旁边，周德记茶餐厅已经开了二十年。周老板站在柜台后面算账，脸色不太好。上个月伙计阿昌在搬货的时候摔伤了腿，保险公司说不在赔付范围内，拒了。周老板给阿昌垫了医药费，又去理论，保险公司还是那四个字：不在范围。

周老板不懂什么条款，他只知道阿昌做了十多年，摔伤那天是为了搬餐厅的货。周志明是上环一间保险行的伙计，经人介绍来找周老板，说可以帮他把这件事交到一个公平的地方去评评理。

"真的不用钱？"周老板问。他听过太多"不用钱"结果被层层收费的故事。

"不用。投诉人不用付任何费用。"周志明说，"但你要先知道，这个委员会只处理涉及赔钱的投诉，而且最多判一百万。"

周老板点了点头。

投诉交上去之后，周志明陪他去见了委员会的调解员。调解员是个退休的洋行经理，不偏帮保险公司，也不偏帮茶餐厅老板。他仔细看了保单条款，问了几个关键问题：阿昌那天在做什么、餐厅的货是自己公司的还是客人的、保险公司的合同里有没有写清楚这类情况不赔。

周志明这才明白，委员会处理投诉，不是简单地看合同字面，而是要看"这样做对投诉人公道不公道"。委员会参照的是行业里处理赔钱的公认做法，核心原则是"又快又公道"。保险公司拿出来的内部指引写的是"迅速、快捷及公道地处理索偿"，但阿昌的案子拖了快两个月才出结果，中间连一封正式通知都没有。

委员会最后裁定：保险公司的拒赔理由不充分，要补发赔付。

周老板问："他们会不会上诉？"

"不会。"周志明说，"但如果你不满意，还可以走法律途径。"

周老板没有走法律途径。他说，能有人愿意听完两边的说法，对着他把道理讲清楚，这已经是他想了很久的事情。

那天晚上，他给阿昌包了个红包，说是"营养费"。阿昌不要，说老板已经垫了医药费。周老板说："拿着。你帮我搬货受伤，我有责任。委员会说你该拿的钱，就该你拿。"

他没有讲什么大道理。茶餐厅的灯光暖黄，街外人来人往，招牌上"周德记"三个字还用的是旧式写法。

## 原文定义

保险索偿投诉委员会（ICCP）由一位独立人士担任主席（须经财经事务及库务局局长认可），其余四位委员中两位来自保险业，两位为业外人士；委员会裁定赔偿金额最高一百万港元，保险人无权就裁决上诉，投诉人无需缴付任何费用；委员会须参照保险合约条款与"优良保险惯例"原则进行处理，核心守则为"迅速、快捷及公道地处理索偿"。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 委员会由独立人士主席、业外人士占多数 | ICCP 独立性与公正性 |
| 周老板投诉不用付任何费用 | 投诉人无需缴费 |
| 最高赔一百万，保险人不能上诉 | 裁定限额与约束力 |
| 委员会不仅看合同字面，也讲"公道" | 优良保险惯例原则 |
| 投诉人被拒赔，委员会判补发赔付 | ICCP 实际裁决功能 |
