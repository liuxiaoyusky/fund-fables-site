# 6.2.5.5.3 产品知识

## 故事

何师奶的茶餐厅在油麻地开了十五年，丈夫走后她一个人撑着。伙计摔伤那次，住院费花了她三个月流水，她没吭声，咬牙付了。后来她想通了——铺头虽小，也得有人兜底。

她找梁志明，想给三个伙计各买一份意外保障。梁志明来了两次，第二次带了一张申请表，说填完就可以递上去。何师奶拿过表格，上面要填健康状况那一栏，她看了看伙计阿强——阿强上个月骑单车摔过一次，手臂缝了四针，这事她知道。

"这个要紧吗？"何师奶指着健康状况那栏问。

梁志明放下茶杯，说："师奶，我今天来，就是要跟你讲清楚这件事。"

他没急着填表，而是拿出一张白纸，在上面写了四个字：最高诚信。然后他一条条解释——买这份保障，等于是跟保险公司立约，铺头这边要把底牌亮出来，哪些事有过，哪些地方不妥当，都要说清楚。如果不说，或者说了不准确的，将来真出了事，保险公司查出来，可以不赔，而且保单可能直接作废。

何师奶听完没出声。她想起当年丈夫出事时，有亲戚说她"早知道就报案了"，那句话她记到现在——不是因为没报案，是因为有些事没讲清楚，后来反而说不明白。

"阿强上个月摔过一次，手臂缝过针。"何师奶说。

"这个要填进去，"梁志明说，"而且要附上那次就诊的病历记录。如果当时没留，也要写清楚哪天、在哪里看过医生。"

何师奶让阿强找出那次就诊的单据。梁志明帮她核对日期、病名、医生签名，一项项填进表格。最后他在声明栏停下，让何师奶自己看一遍，再由她自己签字。

"你自己签，"他说，"签了名字，就要对里面的内容负责。我可以帮你填，但不能替你签。"

何师奶签完那天，何师奶的外甥女正好来吃饭，听说了这事，说："姑妈，你找的这个代理人倒是讲规矩的。我听别人说，有的中介帮人填表，签个名就完事，真出事，客户连表格写了什么都不清楚。"

何师奶没接话，把那张签完字的表格复印件夹进了舖头的账本里。

## 原文定义

持牌保险代理人在协助客户提出保险申请时，应向客户解释最高诚信原则，并提醒客户不披露重要事实或向保险人提供不正确资料可能会导致保单失效或索赔遭保险人拒绝；应告知客户须向保险人披露的重要事实种类；应让客户有机会在签署声明前检视其所载内容。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明解释"最高诚信"原则 | 向客户解释最高诚信原则 |
| 告知不披露或提供不实资料会导致保单失效、索赔被拒 | 提醒客户不如实披露的后果 |
| 阿强摔伤就医须如实填入健康状况栏 | 告知须披露的重要事实种类 |
| 声明栏让何师奶自己看、自己签，不代签 | 客户有机会在签署声明前检视其内容 |
