# 3.4.6 委付（Abandonment）

## 故事

"顺发号"第三次进船坞的时候，冯船长做了一个决定。

那趟从汕头回来的航程，船底被不明物体撞了个大窟窿。海水灌进来，把引擎舱淹了个透。船虽然没沉，但修的话，要把整个底舱拉上来换板，修理费算下来，比船本身的价值还高。

造船厂的老师傅来看了一眼，把扳手往地上一扔，说："船长，这船不值得修了。"

冯船长蹲在船坞边上抽烟，一根接一根。船在干坞里斜躺着，露出底舱那块被撞凹进去的铁皮，像一块淤青。

他想：修，比船还贵。不修，就拿不回多少钱。

陈伯那天恰好来船坞办事，看见他在船边发呆，走过来问了情况。然后陈伯把他拉到一边，说了一句话："你可以试试委付。"

"委付？"冯船长没听过这个词。

陈伯说："就是把你的船交给保险行，然后拿全损的赔偿。"

梁志明后来也来了，把保单翻到那一页，解释给冯船长听："委付的条件是：你的船受损很严重，严重到修理费接近或超过了船本身的价值。这种情况下，你可以向保险行提出，把船的残骸——也就是船本身——全部交出来，换取全额赔偿。"

"那残骸怎么处理？"冯船长问。

"归保险行。他们可以自己卖，也可以拆了卖零件。"

冯船长站起来，走到船边，用手摸了摸那块被撞凹的铁皮。这条船跟了他十五年，从一条小机帆船换成现在这艘，他给它起名字、上油漆、装新引擎，每一块铁皮他都熟悉。

"委付之后，这船就不是我的了？"他问。

"不是。"梁志明说，"你要想清楚。委付是把整艘船交出去，不是只交那块破了的地方。你拿了钱，船和船上所有东西都归保险行处置。"

陈伯在旁边补了一句："不过对你来说，与其花比船还贵的修理费把船修好，然后继续用一条老船，不如拿一笔钱重新开始。"

冯船长没说话。他在船坞边上站了很久，一直站到傍晚，船厂的灯亮起来，把那条斜躺着的船照出一层铁锈色的光。

最后他回去了，跟梁志明说："委付。"

梁志明帮他准备文件。委付通知递交的那天，冯船长没有去码头。他在上环的茶餐厅里坐了一天，喝了三壶普洱，看着窗外的电车慢慢驶过。

几天后，保险行的赔付到了。全额赔偿，不多不少。

船厂的人后来告诉他，那艘船被拖到昂船洲，拆了卖废铁，卖了三千多块。

冯船长听到这个数字，笑了笑，没说话。

## 原文定义

委付（Abandonment）是指在某些情况下，向保险人交出保险标的，作为索取全损赔偿金额的条件。委付后，保险标的或其残余部分须完全移交给保险人，使其可从中获取利益。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 修理费比船本身价值还高 | 委付适用于修理费接近或超过标的实际价值的情况 |
| 冯船长把整艘船交出去，换取全额赔偿 | 委付是将保险标的全部移交给保险人，以换取全损赔偿 |
| 保险行获得残骸处置权，拆了卖废铁 | 委付后标的归保险人所有，保险人可从中获益（这点与代位有关） |
| 冯船长拿钱重新开始 | 委付是被保险人在全损情况下的选择：拿全额，放弃残骸 |
