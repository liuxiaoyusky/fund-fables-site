# 3.3.2 危险的种类

## 故事

何师奶的茶餐厅在旺角经营多年，她给店子保了一份火险，觉得这下可以睡安稳觉了。那年夏天，一伙贼深夜撬门进来，撬开收银机，把当天营业款全部拿走。第二天早上何师奶报了警，也报了保险。

梁志明来查勘的时候，翻了翻保单，说了一句何师奶当时没听懂的话："这个损失不在受保范围。"

"火险唔保盗窃？"何师奶愣住了。

"火险保火灾相关的情况。盗窃是另一类危险，我们叫它'不保危险'——不是受保的，但也不在除外范围。简单说，就是不赔。"

何师奶急了："那我的钱就白丢了？"

"不是白丢，是这张保单不保这一项。"梁志明把保单翻到第二页，指给她看，"你看，这里写了受保范围：火灾、雷击、爆炸。盗窃不在里面。除非你另外买盗窃险。"

陈伯在旁边喝茶，插了一句："就像我那艘船的保单，保搁浅和碰撞，但不保天气——天气好的时候不赔，天气坏的时候也不赔，因为根本不在单子里。"

何师奶问："那什么是'除外'的？"

"除外的，就是本来应该赔，但你跟我说了某种情况，这种情况就不赔了。"梁志明说，"比如战争。打仗引起的火灾，火险不赔——因为火灾是受保的，但引起火灾的原因是战争，战争是除外危险。"

何师奶总算弄明白了。她后来加了一份盗窃险，说："下次我要问清楚，哪些事'包'，哪些事'唔包'，哪些事'讲定先'。"

梁志明笑着说："阿婆，你学得快。危险分三类，不是一句'包'或'唔包'能说完的。"

## 原文定义

危险（peril）分三类：受保危险（insured peril）为保单明确承保的危险；除外危险（excepted peril）本应受保但被除外条款豁免；不保危险（uninsured peril）既非受保也非除外，相应损失一般不获赔偿。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 火险不赔盗窃损失 | 盗窃属于不保危险 |
| 陈伯的船险不保天气 | 不保危险的特征：不属受保亦非除外 |
| 梁志明解释"战争引起的火灾不赔" | 除外危险：火灾受保，但战争引起则除外 |
| 何师奶后来加购盗窃险 | 不保危险可通过单独附加险转为受保 |
