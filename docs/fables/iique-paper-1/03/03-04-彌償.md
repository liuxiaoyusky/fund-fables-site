# 3.4 弥偿（Indemnity）

## 故事

梁志明刚进保险行那年，师父陈伯把一本泛黄的练习本交给他，说：「慢慢睇，急唔得。」

练习本里记的不是公式，而是一个又一个街坊的故事。陈伯最常提起的是三十年前观塘一间山寨制衣厂的老板，赶圣诞订单时工场起了火，损失了十几万。那老板事后四处找熟人借钱，见人就说：「算我倒霉。」

陈伯那时刚从英资洋行转过来，他跟老板说：「你点解唔一早揾人写张保单？」

老板苦笑：「写张保单几钱，你知道吗？再说，我请了你，你也要赚我钱，最后还不是我蚀。」

陈伯没有回答。

三年后，老板第二次遭了火，烧得比上一次更狠。这一次，他的厂彻底没了。街坊凑了几千块给他老婆，远远不够。他站在废墟前，对著陈伯说了句：「当年你叫我买，我以为你想揾我笨。」

「我那时也不知道，」陈伯对梁志明说，「我只知道，唔能够因为有人想揾你笨，你就好似唔记得自己会有损失。」

梁志明后来在实际工作中慢慢明白这句话的意思。他见过有街坊出了事，拿著保单去闹，说「我明明保了二十万，你就赔十万？」；也见过有客人同时买了三份火险，出事之后想著「终于发达了」。两种人，陈伯都摇头。

梁志明跟周老板解释的时候，尽量把话说得简单：「保险这张纸，是一张借据，不是发财票。你借了银行的钱，最后要还；你借了保险的承诺，出事就按你真正损失了多少还你。不是越多越好，是刚刚好。」

周老板点点头，没有说话，但梁志明知道他听进去了。

## 原文定义

弥偿（Indemnity）是一项保险基本原则，指被保险人因保险事故所遭受的损失，可由保险人给予弥补，但不能从中获利。其核心含义是：被保险人只能索回其所损失的东西。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 山寨厂老板第一次失火后四处借钱 | 没有保障时，损失由本人完全承担 |
| 老板以为保险行的人想揾他笨 | 街坊对保险制度的信任问题 |
| 第二次大火后工厂彻底倒闭 | 未做好风险转移的严重后果 |
| 梁志明把保障比作「借据」 | 弥偿原则的核心：填补损失，而非获利 |
| 周老板最后点头接受 | 理解「不补偿」是制度的公平机制 |
