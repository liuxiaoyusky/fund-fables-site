# 3.5.1 有关分担的衡平法则

## 故事

周老板和周师奶结婚二十五年，在永安街住了二十年，两个人都是做海味生意的，脾气硬，话少，但心细。周师奶不喜欢把全部身家放在同一个地方，她跟丈夫说：「我再去揾间保险行，多保一间，稳阵啲。」

周老板不以为意：「一份够了，边有咁多事。」

师奶没有理他，自己去买了一份火险，受益人写的是丈夫的名字。她想的是：「他成日去南中国收货，家里万一有事，间中银纸起码可以重新开始。」

周老板不知道这件事，他自己那一份火险，一直按时缴费。

那年夏天，一场大火烧了半条巷子，广记的后仓和前铺都受了灾。二十万的损失，周老板拿著两份保单去找保险行，他以为这下好了，两份都赔，说不定还有得找。

梁志明在中环的洋行里听到这个消息，皱起了眉头。

他打电话给周师奶，师奶在电话里有点不好意思：「我以为他那份够晒，咪再买一份，边知佢自己又有一份……」

梁志明跟Mr. Thompson研究完两份保单，给了周老板一个数：「两份加起来，你只能拿到二十万。不是四十万。」

周老板坐在茶餐厅里，半支烟没有点完。他对梁志明说：「她那份是她买的，我这份是我的，点解加起来得一份？」

梁志明想了想，说：「周老板，你听我讲。两个人都担心同一间屋出事，都买了保障，这是你们的心意。但制度设计的原则是——间屋烧了，你只能重新起一次，唔可以起两次。两份保险看起来是两份安心，但赔的时候，只认你烧了几多，不认你买了几份。」

周师奶坐在旁边，一直没有说话。后来她轻声说了句：「我买的时候，谂住万一出事，佢起码有笔钱。唔系话想发达。」

梁志明说：「我知。」

最终，二十万由两间保险行按比率分担，周老板足额拿到了损失的赔偿。走出洋行时，周师奶轻轻挽了挽丈夫的手臂，什么也没有说。两份保单，两份心意，最后只换来一份安心。但那份安心是真的。

## 原文定义

重复保险（double insurance）指两份或以上保单同时承保相同或部分相同的权益，且总保额超过法律允许的弥偿程度。衡平法则规定：任何一个保险人履行弥偿后，有权要求其他对同一被保险人须负赔偿责任的保险人分担已赔付的金额。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周师奶瞒著丈夫多买了一份火险 | 重复保险的发生：被保人自己或他人代为投保 |
| 两份保险都想赔同一场火 | 两份保单承保了相同权益 |
| 两份加起来只能赔二十万 | 弥偿原则限制：被保人不得从损失中获利 |
| 两间保险行按比率分担二十万 | 衡平法则：保险人有权互相追讨分担 |
| 周师奶说「唔系想发达」 | 区分善意重复保险与故意双重索偿 |
