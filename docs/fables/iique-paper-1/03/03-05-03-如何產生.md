# 3.5.3 如何产生

## 故事

维港码头边的联安仓库，平常由冯船长打理，有时候也替街坊存放货物。上环海味舖广记的周老板，便常年租了仓库一角，放他的干货。周老板怕火，每年的火险年年续。

冯船长见周老板年年缴保费，便想：我的仓库也要有个保障。于是，他另外买了一份火险，把整间仓库连同周老板那批存货一起保了。他想的是：「我的仓库我当然要保，那批货放在我地方，万一著火，我也要负责。」

那年夏天，仓库底层一批被海水汽潮了的麻袋起火，烧了周老板存在那里的十几箱干鲍鱼。周老板损失惨重，拿著自己的火险单去找保险行索偿。同时，冯船长也拿著他那一份火险单，要求他自己的保险行赔偿。

两间保险行各自做了理算，发现问题来了——同一场火，烧了同一批货，两张保单都想赔，于是两间公司坐下来讨论分担。

会议上，Mr. Thompson代表冯船长那间保险行，他说：「既然都保了同一个仓库，损失又相同，应该按比率分担。」

梁志明那边代表周老板的保险行，他想了想，没有直接否认，而是问：「甲乙两份保单，各自由边个出钱买的？」

「周老板自己买了一份，冯船长也自己买了一份。」

「那就好。两份保单都对这批货提供损失赔偿，对吧？」

「对。」

「两份都承保了火灾这个成因？」

「对。」

「两份都覆盖了这批受损的干鲍鱼？」

「对。」

梁志明停了一下，问出最后一个问题：「但是——周老板那份保单，承保的是他自己的『物主权益』；冯船长那份，承保的是他自己的『仓库经营人权益』。同一场火，两份保单保护的东西一样吗？」

Mr. Thompson一时语塞。

梁志明继续说：「分担的原则是——只有在两份保单保护的是同一种权益时，才需要分担。周老板怕自己的货烧了，冯船长怕自己的仓库出事，他们两个人都紧张同一批货，但他们紧张的理由不同。这两张保单，表面上都在赔那场火，但实际上，甲赔的是老板的损失，乙赔的是船长的损失。权益不同，所以不存在分担。」

Mr. Thompson最终同意，梁志明的保险行全额赔偿了周老板，冯船长的保险行则另案处理了仓库本身的损失。

走出洋行的时候，冯船长对梁志明说：「本来以为分担好简单，原来要过晒五关先计得埋。」

梁志明说：「系呀，缺任何一关，都唔会计到分担。」

## 原文定义

分担的产生须满足五个条件：（a）每张保单都对有关损失提供弥偿保障；（b）每张保单都承保了受影响的权益；（c）每张保单都承保了引致损失的危险（成因）；（d）每张保单都承保了受影响的保险标的；及（e）每张保单对该损失均负赔偿责任。条件（b）的关键在于，两份保单必须承保「同一权益」——若一份保单承保物主权益，另一份承保保管人权益，则不构成「相同权益」，不产生分担。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 同一场火烧了同一批货 | 同一危险（成因）引致同一损失 |
| 周老板和冯船长各持一份火险单 | 两份保单均提供弥偿保障（条件a） |
| 两张保单都覆盖了这批干鲍鱼 | 两张保单均承保了受影响的保险标的（条件d） |
| 周老板保的是物主权益，冯船长保的是仓库经营人权益 | 条件(b)关键：必须是同一权益才产生分担 |
| 两份保单保护的「理由」不同 | 权益不同，不满足条件(b)，不产生分担 |
| 两间保险行各自独立赔付 | 分担只在条件全部满足时才发生 |
