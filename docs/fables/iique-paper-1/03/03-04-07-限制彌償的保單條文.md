# 3.4.7 限制弥偿的保单条文

## 故事

何师奶的茶餐厅在观塘工业区开了十二年，那天半夜，电线走火，把整个后厨烧了个精光。

她站在街边，看着消防员把火扑灭，厨房已经剩下一个焦黑的壳。冰箱里的肉、碗碟、炉头、风扇——什么都没剩。

梁志明第二天一早赶到现场。他是保险行的伙计，何师奶这份火险单是他三年前帮忙办的。他把保单翻出来，算了算，开始冒冷汗。

保额写的是十二万。但何师奶当年投保的时候，茶餐厅刚开张，装修加设备，算下来值十二万。十二年过去了，观塘的铺租涨了三倍，同一条街、同等面积的店，装修加设备，没有二十万下不来。

"何师奶，这份单……保额不够。"他把单子递过去。

何师奶正在擦那张旧餐桌，听到这话，手停了一下。

"不够是什么意思？"

"你的实际财产值二十万，但你只保了十二万。这叫不足额投保。出了全损，保险行只赔十二万，不是二十万。"

何师奶把抹布摔在桌上："那我亏八万？"

"按比例算，是的。你只保了六成，损失就只赔六成。"

周老板那天碰巧也在观塘进货，听说茶餐厅出事，也赶了过来。他看了一眼梁志明手里的保单，说："有没有办法不那么赔？"

梁志明说："有一种叫'比例分摊'的条款，就是这个效果。不足额投保，赔偿按比例递减。所以中介人要确保客户按全额投保，很重要。"

何师奶靠在墙上，深吸了一口气，说："当年办单的时候，我想省点保费，就少报了一点。没想到……"

她没把话说完。街边的消防车响着警笛开走了，声音越来越远。

周老板忽然想起什么："你那保单有没有'免赔额'？我们酒楼上次理赔，保险公司扣了头几千块，说免赔额以下的不赔。"

梁志明翻了一页，说："有，四千块免赔额。这次损失是全损，免赔额不影响。但如果是小意外，只烧了一个炉头，修理费三千块，那就全由你自己出了。"

何师奶点点头，沉默了一会儿，然后问："那我这次能赔多少？"

"全损二十万，但你保了十二万，比例是六成，所以保险行赔十二万。扣掉四千免赔额，实付十一万六。"

何师奶重新拿起那块抹布，继续擦那张烧不坏的旧餐桌。

周老板临走的时候，在门口跟梁志明说："你帮我看看我那份海味铺的保单。我那年也是按账面价值保的，不知道现在够不够。"

梁志明把何师奶那份单子收好，说："行，不过你要做好准备，账面的数字不一定等于实际的价值。"

街对面的大厦正在拆架子，铁锤敲在水泥上，咚咚作响。何师奶的茶餐厅黑着灯，旧招牌还挂在墙上，"何记"两个字被烟熏得有些发黄。

## 原文定义

限制弥偿的保单条文包括：(a) 比例分摊：不足额投保时，赔偿按比例递减；(b) 免赔额：低于指定金额的损失不获赔偿；(c) 起赔额：损失达到或超过指定金额时全额赔偿，未达则完全不赔；(d) 保单限额：赔偿以保单约定的最高金额为限。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 保额十二万，实际值二十万，只赔十二万 | 比例分摊：不足额投保，赔偿按投保比例递减 |
| 梁志明说"确保全额投保很重要" | 比例分摊对不足额投保的惩罚，提醒中介人需确保足额投保 |
| 保单有四千块免赔额，全损时扣四千 | 免赔额：低于指定金额的损失由被保险人自行承担 |
| 周老板问"有没有起赔额" | 起赔额：达到或超逾指定金额时全额赔偿，未达则完全不赔 |
| 保单限额：超过最高金额的损失不赔 | 保单限额：任何超过约定最高赔偿额的部分均不获赔偿 |
