# 3.3.3 原则的应用

## 故事

维港的夜班渡轮汽笛低鸣，冯船长靠在"顺发号"货舱壁上，手里那盏煤油灯在海风里晃。舱里堆着四家商号从广州运来的货——有周老板的海味罐头，有制衣厂的面料，有化工行的颜料桶，还有一箱景德镇瓷器。

凌晨四点，值班水手喊了一声："船长，前方有船！"

碰撞来得毫无预兆。"顺发号"船身一震，瓷器箱轰然倒塌，砸中了隔壁堆放的颜料桶。颜料桶破裂，液体流淌，正好流进厨房灶台底下——那灶台有一盏忘记关的煤油灯。

火，从那里烧了起来。

灭火的水把甲板淹了个透。等火灭了，船舱里积了半尺深的海水，所有货物都被泡坏了。周老板、何师奶派来的伙计、颜料行的账房，围在码头上，等着冯船长给他们一个说法。

冯船长先找的是梁志明。梁志明是保险行的伙计，广州那边就有朋友做这行，几张单子都是他在香港这边帮忙安排衔接的。他连夜赶到码头，把四张保单摊在货箱上：

第一张，只保碰撞；第二张，只保火灾；第三张，只保爆炸；第四张，只保海水。

账房先开口："船长疏忽引起碰撞，碰撞引起火灾，火引起爆炸，爆炸让船壳裂开，海水才进来。按道理，船长疏忽才是根本原因，这四张单子都不赔。"

冯船长脸色铁青。他从口袋里摸出一根烟，没点着，又塞了回去。

梁志明却把四张单子重新理了一遍，指着第三张和第四张说："你再看看，这些危险是一个接一个、不被干扰地发生的。碰撞→火灾→爆炸→入水，一气呵成。火险单要看的是，火引出了什么；海险单要看的是，海水进来之后做了什么。第四张单子只问：海水进来了没有？进了，就赔。"

他顿了顿，又说："第三张单子也一样。爆炸是受保危险，它引出的连锁反应，保单只认最后一个近因。"

账房把算盘拨了一遍，抬起头："也就是说，每张单子各自赔自己那段？"

梁志明点头。

冯船长把没点着的烟重新叼回嘴里，这次他点着了，深吸一口。码头的海风把那口烟吹散，他透过烟雾看着那些被海水泡坏的木箱，说："那第一张单子呢？"

"第一张只保碰撞，碰撞是受保危险。碰撞直接造成的那部分损失，当然赔。"

码头的风又大了一些。冯船长把烟头摁灭在铁栏杆上，跟梁志明说："你帮我把这话讲给周老板听。他精明，要是他问我'为什么不全赔'，我答不上来。"

梁志明把保单收进公文包，说："周老板的货这次受损，按近因来分析，碰撞是近因，但碰撞是受保危险，所以赔。火烧起来之后，近因变成了火，火也是受保危险。爆炸同理。只有跟海味单没关系的那一段，才不在这张单子范围内。"

他走之前，回头说了一句："不过船长，以后装货，这几种货物得分开堆。单一成因才简单，一堆成因搅在一起，赔起来麻烦，查起来更麻烦。"

冯船长望着他走远的背影，没说话。码头的灯柱在夜雾里光晕模糊，海风把腥味和烟味一起吹过来。

他低头看了一眼那四箱泡坏的货，心想：以后，还是听保险行的。

## 原文定义

近因原则在实际应用中的规则：(b) 如果只有单一成因，事情处理就直截了当：假如该成因是受保危险，所引起的损失是受保的；(c) 如果涉及一项以上的危险，须引用近因法则；(i) 由受保危险直接引发的不保危险，所引致的损失可获赔偿。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 碰撞→火灾→爆炸→入水，一气呵成的连锁事件 | 多个危险以一连串事故形式发生，近因依次传递 |
| 第一张单子保碰撞，第二张保火灾 | 不同保单只赔自己那份受保危险引发的损失 |
| 梁志明说"火烧起来之后，近因变成了火" | 近因不一定是最初的原因，而是直接引发损害的那个 |
| 账房说"船长疏忽才是根本原因" | 首个成因不一定构成近因，需按事件链条逐段分析 |
