# 3.1.4 如何产生

## 故事

湾仔鹤园街有一排老旧的唐楼，楼顶加建的铁皮屋住了七户人家，黄师傅和周太也是其中之一。

那年夏天，台风温黛来袭，唐楼的屋顶被吹翻了两块铁皮，雨水直接灌进了二楼陈伯的单位。陈伯的脱床、被子和一箱脱票都泡了汤。后来才知道，楼宇的公用地方没有人买过火险，所有损失只能由各户自己承担。

黄师傅这次没有躲。他开始在屋顶上忙着找师傅修补，周太在旁边递工具。修补期间，黄师傅对周太说：「我打算找梁志明帮我查清楚，我们这些唐楼，要怎样才能有个兜底。」

周太问：「你说的兜底，是保边个？」

黄师傅说：「三个。第一，我自己做的装修工程出了岔子，伤到路人，我要对别人负责，这是第一层。第二，我们住的这栋唐楼，万一火烧了，楼垮了，我要对其他住户负责，这是第二层。第三，如果我在工程里伤了别人，别人告我，我要有人帮我孭这笔钱，这也是一层。」

梁志明听完黄师傅的叙述，在蓝色文件夹里翻了一会，找出了三份不同的说明材料。他对黄师傅说：「师傅，你说的三层，分别对应不同的可保权益。第一，你自己作为施工人员，万一出了意外伤到工人或路人，你对自己的作为是有可保权益的，这叫责任保险。第二，你住的唐楼，你虽然不是业主，但你们各户联合起来，也可以作为委托人，让大厦管理处统一投保火险——这叫财产保险。第三，你作为替业主做事的判头，万一有人告你，你可以用责任保险把自己的法律责任转出去。」

黄师傅又问：「那陈伯呢？他不是做工程的，他这些年的损失怎么算？」

梁志明说：「陈伯有另一种。他的脱床和文件被水泡了，他对这些财物有所有权，这种所有权本身就是可保权益。」他顿了顿，「而且陈伯上年跟我说过，他担心楼宇维修期间收不到租金，他作为实际上的使用者，对这个潜在的租金损失也是有可保权益的——这叫法律权利保险。」

周太在旁边默默记了下来。后来她对黄师傅说：「原来一样住在一栋楼里面，每个人可以保的东西都不一样，有的保命，有的保楼，有的保收入，有的保别人告你。关键是要搞清楚——边个有权保，保的是边个。」

台风过后，唐楼的屋顶修好了。黄师傅去梁志明的保险行，把三份投保书都填好了。林若兰帮他打字，一份一份核对，发现黄师傅在「受益人」那一栏都填了「周太」。

她问：「师傅，你不写自己？」

黄师傅说：「我还活着，我就在。她不在了，我才需要那张纸。」

## 原文定义

可保权益可在以下情况产生：人身保险——每人对自己的生命、配偶的生命享有可保权益；财产保险——所有人对其财产享有可保权益，遗嘱执行人、按揭持有人、保管人等也可投保；责任保险——每人对自身的法律责任可购买保险；法律权利保险——面临合法权利受侵害或收入损失风险的人可就此投保。代理人经委托人授权投保的，享有与委托人同等程度和范围的可保权益。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 黄师傅投保工程责任险，万一伤到路人可获保障 | 责任保险：每人对自身作为或不作为的法律责任具有可保权益 |
| 唐楼各户联合投保火险，保障楼宇财产 | 财产保险：财产所有人对其财产具有可保权益 |
| 陈伯担心维修期间租金损失，为租金投保 | 法律权利保险：面临合法权利受损或收入损失风险的人可投保 |
| 林若兰在受益人栏看到「周太」 | 受益人的安排属人身保险中的可保权益延伸 |
| 梁志明解释三层保障，分别对应不同投保类型 | 可保权益依据人身、财产、责任、权利等不同基础而产生 |
