# 3.2.5 违反最高诚信（Breach of Utmost Good Faith）的类别

## 故事

那年圣诞前，Mr. Thompson的桌上堆满了理赔文件。他在湾仔的英资保险公司工作了二十年，圣诞前的这个月，是理赔高峰期。

那天他要处理四个案子，每个都涉及不同程度的瞒报或失实陈述。

第一个案子是观塘一家制衣厂的火险。厂家在投保前就知道厂房的旧电线没有更换，仍然填报称「设备状况良好」。Mr. Thompson把文件合上，说：「呢个唔系无心之失。佢明知电线有问题，但仍然申报话冇问题，明显系想呃保费。」

梁志明在一旁整理文件，轻声说：「欺诈性失实陈述？」

「正确，」Thompson说，「投保人明知故犯，提供假的资料误导保险人。」

第二个案子是一艘内河货船的沉没索赔。船东在投保时说船况「正常」，但保险公司调查后发现，船东只是「忘记了」申报三年前的一次搁浅事故——船底曾被撞击，之后修补过，但船东说他以为「小事，唔需要报」。Thompson摇头：「呢个唔系故意的，但系疏忽大意。这是非欺诈性失实陈述。」

第三个案子是一张商舖的火险单。店主没有主动告知保险公司舖内存放了大量高度易燃的漆油溶剂，因为「投保单上冇问我存什么货」。Thompson说：「投保单有一般性问题问舖内存放物，佢冇如实答复，而且系故意的——但佢的目的不是呃更低保费，而系觉得麻烦，唔想解释。这种算是欺诈性不披露。」

第四个案子最复杂。一个茶餐厅老板娘的伙计在工作时受伤，申请意外索赔。保险公司发现她在续保时没有主动告知有两名伙计曾经因工受伤——但她自己解释说，以为续保只是「延续旧单，唔需要重新申报所有野」。Thompson看完笔录，在文件上写：「非欺诈性不披露。佢并非故意，只是对续保的义务理解有偏差。」

梁志明把四份文件整理好，看着标签纸上的分类：欺诈性失实陈述、非欺济性失实陈述、欺诈性不披露、非欺诈性不披露。

「Thompson先生，这四种，处理的原则一样吗？」

Thompson摇头。「完全不一样。欺诈性的，无论是失实陈述还是不披露，保险人可以拒绝赔偿，而且受保人还可能面临法律追究。非欺诈性的，具体后果要视乎情况——有时候保险人可以调整保费或者拒绝赔偿，但受保人没有欺诈的恶意，处理的空间更大。」

他拿起第一份文件，在右上角画了一个大大的交叉：「这个，坚决拒。」

又拿起第二份：「这个，酌情处理，可以唔赔，但唔告。」

第三份：「呢个麻烦，先放在一边，慢慢倾。」

第四份：「呢个，教育为主，酌情理赔。」

梁志明把文件收好，心里记住了这四种分法。走出办公室的时候，他路过林若兰的座位，她正在打字，看也没抬头，只是把一叠翻译好的文件推到桌面边缘，刚好在他经过的地方。梁志明伸手拿了，没有停步。

## 原文定义

违反最高诚信有两种形式：失实陈述（提供虚假资料）和不披露（没有披露重要资料）。每种形式依动机分为欺诈性与非欺诈性（无意或疏忽），共四类：(a) 欺诈性失实陈述；(b) 非欺诈性失实陈述；(c) 欺诈性不披露；(d) 非欺诈性不披露。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 制衣厂明知电线有问题仍申报良好 | 欺诈性失实陈述：故意提供虚假重要事实 |
| 船东忘记申报三年前的搁浅事故 | 非欺诈性失实陈述：无意或疏忽地提供虚假事实 |
| 商舖店主明知存放危险品但不主动说 | 欺诈性不披露：故意不披露重要事实 |
| 茶餐厅老板娘以为续保只是延续旧单，冇主动申报 | 非欺诈性不披露：无意或疏忽地漏了披露 |
| Thompson对四案的不同处理态度 | 四种分类的实际后果差异：欺诈性后果最严重，非欺诈性有酌情空间 |
