# 3.4.2 含义

## 故事

周老板的干货铺开在上环楼梯街，铺子不大，但鲍鱼、花胶、蚝豉摆满三层木架。那天下午，他坐在柜面后头，眉头拧成一道绳——不是为生意，是为侄子阿荣。

阿荣今年二十三，刚在码头货仓找了份搬运的活。周老板想起自己年轻时在仓库摔断过腰，躺在床上三个月没收入，就让阿荣去问问有没有保单可以买。阿荣跑了一趟保险公司，带回来的消息却是：这种险，按手指赔。

"断一只赔多少，断两只赔多少，"阿荣比划着，"不是赔你损失，是赔一个数。"

周老板没听懂。什么叫"赔一个数"？他做了几十年干货生意，什么都讲究"值多少赔多少"——货烧了赔货值，铺子被偷赔进货价，这是天经地义的道理。手指断了，怎么能用一个数来算？

他想起陈伯。陈伯年轻时候在船公司当文员，现在早退休了，但脑子比账本还清楚。周老板泡了壶普洱，把阿荣说的事讲给陈伯听。

陈伯听完，叹了口气："保险公司不赔你的损失，赔一个他们预先答应的数。手指值多少钱，没人算得出来。"

"算不出来就不赔了？"周老板急了。

"不是不赔，是换个赔法。"陈伯说，"这种险叫'利益险'——你在保单里跟他们约好，万一出事，他们按约定金额给你。不管你实际损失多少，不管你断了多少根手指，就按你们当初写好的那个数来。"

他顿了顿，又说："普通的财产险、火险、车险，用的是另一套做法——你要先证明自己到底损失了多少，他们才能赔你。赔到手的钱，不会比你实际丢掉的多。这叫'弥偿'，就是把你恢复到出事之前的样子。"

"那阿荣的手指呢？"

陈伯摇了摇头："手指断了，怎么'恢复'？海味烧了可以重新进货，手指断了接不回去。所以这种险干脆不跟你算损失，直接约定一个数，写进保单里，双方同意，将来就按这个数来。"

周老板沉默了半晌。

他想起干货铺的账——每年火险、车险都按实际货值投保，出了事保险行就是照单赔钱，从不多给，也从不少给。他一直以为天下保险都是这样。

原来不是。

阿荣后来还是买了那份意外险。周老板没再说什么，只是叮嘱他上工时候多加小心。

陈伯把普洱喝完，起身要走，顺口说了一句："有些东西，不出事的时候觉得没用；出了事，拿到的钱又不是照损失算的——人嘛，都是这样。"

周老板没接话。他把账本翻到那一页，在阿荣的名字旁边，用铅笔轻轻写了个"已购"。

## 原文定义

弥偿原则不能应用于一切保险类别，人寿保险及人身意外保险一般不适用弥偿原则，因为人的伤亡无法用金钱精确量度，这类保单提供的是按照约定数额支付的利益，而非金钱弥偿。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板一贯用"货值"思维理解赔偿 | 普通财产保险适用弥偿原则 |
| 阿荣的意外险"按手指赔一个数" | 人寿保险/意外险不适用弥偿，提供约定利益 |
| 陈伯说"手指断了怎么恢复" | 人身伤亡无法用金钱精确量度，是弥偿原则的适用边界 |
| 保单里预先写好的金额 | 约定利益（Benefit）而非弥偿（Indemnity） |
