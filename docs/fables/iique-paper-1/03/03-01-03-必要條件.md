# 3.1.3 必要条件

## 故事

陈伯在湾仔一间老牌银号做了三十七年胀房，湾仔的街坊都叫他「胀房伯」。他有一个老朋友叫梁宽，是做中巴客运生意的，十年前跟陈伯借了五万块钱周转，写了借据，约定每年还息，本金慢慢还。

那年秋天，梁宽的身体忽然差了起来，咳了几个月不见好，去医院一查，已经是晚期。陈伯去探望他的时候，梁宽握着他的手说：「陈伯，我走之后，家里那部中巴，你拿着。还有，我走之后，老婆和一对子女……」

陈伯点点头，心里却已经有了一个念头。他想起前几天在茶餐厅听见的一个词——「可保权益」。梁宽欠他五万，如果他能买一份以梁宽生命为标的的保险，将来梁宽走了，保险赔偿就能抵债。

他找到了梁志明，说了自己的想法，递上了那张十年前的借据。梁志明看了又看，然后轻声说：「陈伯，这笔借贷关系是有的，但它是对梁宽的债权，不是对他生命的法律认可关系。你想用借贷去保障他的命，这在法律上是不被承认的。」

陈伯有点不服气：「我借了钱给他，他死了我就收不回来，我点解不能保？」

梁志明从文件夹里抽出三张白纸，递给陈伯：「陈伯，你听我讲完。可保权益要过三关：第一，要有可以被保的标的，梁宽的生命符合条件，这关过了；第二，标的要作为保险的标的，这关也过了；第三，也是最紧要的——你和标的之间必须有法律认可的关系。」

他接着说：「你对梁宽的借贷，是财务关系，但你没有办法只靠这个财务关系，去保他的命。你更不能因为他欠你钱，就去保他的财产。除非梁宽把中巴按给你，抵押手续做完了，你对那部车才可能有可保权益。」

陈伯默默地把那三张白纸收了起来。后来他告诉林若兰：「若兰，我这把年纪，以为借了钱就能保，结果三关都过不了。原来法律睇的，不是有没有钱的问题，而是有没有正当的关系。」

梁志明后来想了想，帮陈伯找了一个合规的方案：让梁宽的遗嘱执行人把还款安排写清楚，并以中巴的遗产份额作为抵押，陈伯作为债权人对抵押财产享有可保权益。三个信封，三关，慢慢凑齐。

## 原文定义

可保权益必须同时符合三个条件：存在可被保的人身、财产、法律责任或法律权利；该标的为保险标的；投保人与标的之间存在法律认可的关系，且该关系能令投保人因保险事故发生而遭受财务损失。单纯的财务关系不足以产生可保权益。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁宽的生命可被投保 | 第一条件：存在可被保的人身（生命） |
| 陈伯想投保以梁宽生命为标的的保险 | 第二条件：人身等为保险标的 |
| 陈伯只有借贷关系，不能投保梁宽的生命 | 第三条件：须有法律认可的关系，借贷债权不足以产生对债务人生命的可保权益 |
| 梁志明指出须以抵押财产才能产生可保权益 | 可保权益须与具体标的之间有法律认可的联结 |
| 最后以遗产份额抵押，陈伯才获得可保权益 | 债权人须以抵押财产产生对应的可保权益 |
