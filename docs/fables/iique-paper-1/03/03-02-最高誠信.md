# 3.2 最高诚信

## 故事

何师奶在旺角经营茶餐厅十几年，招牌上的字都褪了两层色，她只肯重新描一遍，不肯换新的。伙计们背后叫她"孤寒师奶"，但街坊都知道，她对自己人其实不差——那年大厨摔伤腿，她二话不说垫了全部医药费。

何师奶去保险行的次数不多，但每次去都是认真准备过的。这回她想给茶餐厅续一份火险，顺手也把自己的寿险一起办了。

梁志明接待她，照例问了一串问题。何师奶一一答了，末了补了一句："对了，我三年前做过一次手术，胆囊拿掉了。住了三天就能落床，算小事啦，没写在病历里。"

林若兰正好在旁边整理文件，听见这话，抬头看了何师奶一眼。

"师奶，这个你要不要写进去？"梁志明问。

"写了会点啊？"

"那就要看保险行怎么批。"梁志明把笔放下，"胆囊手术不算小事，投保的时候要如实讲。这是规定。"

"我以为住三天就能出院就算小事……"何师奶声音低了下去。

林若兰没说话，低头继续打字，但她把这一页文件的边角折了个角做记号。

后来何师奶的事情批下来了，寿险加了小半成的保费，但没有拒保。梁志明解释说：如果当初没讲这笔，保险行将来发现，是可以当作合约自始无效的，到时候一分钱都赔不到。

何师奶临走时说："我明白啰，揾食艰难，唔到你呃我，唔到我呃你。"

梁志明笑了笑："阿婆，你讲的是街坊买卖。保险行唔讲人情，讲白纸黑字——但白纸黑字的前提，是你我都把话说清楚。"

林若兰把那份折了角的文件收进档案柜，锁好。

## 原文定义

保险合同双方均须以最高诚信（utmost good faith）行事，被保险人须主动披露一切重要事实，保险人亦须如实告知保单条款内容。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 何师奶主动补报胆囊手术 | 被保险人的主动披露义务 |
| 梁志明解释"三天能出院也要讲" | 重要事实不以当事人主观判断为准 |
| 保险行要求补充申报才能批单 | 最高诚信是双向义务 |
| 若当初不报，保险行可宣告合约无效 | 违反最高诚信的后果 |
