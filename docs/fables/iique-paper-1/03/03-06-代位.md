# 3.6 代位（Subrogation）

## 故事

梁志明做了十年保险行伙计，最怕客人问一句话："出事之后呢？"

那年周老板的仓库烧了，何师奶的伙计摔伤了腿，冯船长的船被人撞沉，每一个人拿到赔偿之后都会问同一件事："拿了你们的钱，我还能再找那个做错事的人要钱吗？"

梁志明后来学会了用一个比喻：钱是水，被保险人是容器，保险公司是先把水倒满的那个人。满了之后，水不能溢出来，但保险公司有权去找到底是谁把容器打翻的。

这个比喻不精确，但街坊们听得懂。

周老板问的是："那他们要是追回来的钱比赔给我的还多呢？"梁志明答不上来，回去翻了半天的文件，第二天才去找周老板解释免赔额和责任限额的事。

何师奶问的是："伙计出事，我赔了钱，能不能去找那家没有围栏的工程公司？"答案是能，只要何师奶的雇主责任险赔了，保险公司就站到她位置上去了。

冯船长问的是："我老婆的事，人没了，怎么算？"梁志明想了很久，说："人命没有价，赔的是情分，不是债，所以不能代位。"

陈伯在旁边听他们说话，想起自己年轻时在船公司做账房的日子，那时候保险两个字还不那么流行，街坊出事，要么靠积蓄，要么靠众筹，要么自认倒霉。现在有了白纸黑字，有了代位的规矩，至少有人赔完之后，会认真去找那个真正该负责的人。

梁志明把这整套规矩叫做"代位"。他说得简单："你们赔了多少，我们就去找那个出错的人要多少。你们不能拿两次，我们也不会少要一分。"

周老板后来跟新来的客人介绍保险时，总爱说："买保险就像请了个账房先生，出了事他来赔，赔完之后他去找真正欠你钱的人要账。这就是规矩。"

## 原文定义

代位是弥偿原则的延伸。当保险人对被保险人作出弥偿后，保险人即取代被保险人的位置，取得对造成损失的第三者的追偿权利，并以被保险人的名义行使该权利。代位的目的是确保被保险人不会因同一损失获得双重赔偿，从而维护保险法中弥偿原则的完整性。
