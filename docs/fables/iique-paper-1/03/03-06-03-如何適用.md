# 3.6.3 如何适用

## 故事

冯船长跑了二十多年船，识得维港的每一条水道。这年他给船买了综合保险，投保时还特意问了梁志明一句："我要是出事，你们赔不赔？"

"赔。"梁志明说，"只要是保单里写明的损失，都会赔。"

后来冯船长的船在维修期间被一辆疏忽驾驶的趸船撞沉，保险公司按保单赔了船损。梁志明来找冯船长办手续的时候，顺便问了一句："那个趸船是你在维修期间停在那里的？按理说他们的船撞了你，你可以另外再找他们赔。"

冯船长摆摆手："不用追了，赔够了。"

过了几年，冯船长的老朋友陈伯来码头找他喝酒，两人聊起保险的事。陈伯说："老冯，人寿保险不一样，你买了人寿保单，受益人是你老婆，结果被一个摩托车司机撞死了，保险公司赔了钱，不能再去找那个司机追。"

冯船长放下酒杯："为什么？我船被撞了他们能追，怎么人没了反而不能追？"

陈伯慢悠悠地说："船有价，赔了多少就是多少，保险公司赔了你，就站到你的位置上去找对方，这叫弥偿。人命无价，人寿保险赔的那笔钱，不是'补偿你损失了多少'，而是'给你家人一点安排'。没有弥偿，就没有代位。"

冯船长想了很久，想起自己老婆早逝，没留下什么话。他沉默了一会儿，说："原来如此。赔的是情分，不是债。"

那杯酒他喝得很慢。

## 原文定义

代位的适用前提是弥偿原则的存在。若损失不属于弥偿性质，则不产生代位权。例如人寿保险的赔付不属于弥偿（因为人的生命无法以金钱精确衡量），因此人寿保险人在赔付后不能取得对第三者的代位权。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 冯船长的船被趸船撞沉，保险公司赔了后可代位追趸船 | 财产保险：存在弥偿，代位适用 |
| 摩托车司机撞死人，人寿保险人赔了不能再追 | 人寿保险：无弥偿性质，代位不适用 |
| 陈伯解释"人命无价，赔的是情分" | 人寿保险赔付的非弥偿性质 |
| 冯船长说"船有价，人无价" | 弥偿原则与代位权的适用前提 |
