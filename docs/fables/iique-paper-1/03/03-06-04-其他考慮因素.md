# 3.6.4 其他考虑因素

## 故事

永成海味铺那场火之后，周老板和梁志明渐渐熟了起来。理赔结案了，但两笔账还没算完，周老板心里一直有疑问。

第一笔是时间上的。周老板记得，保险公司的赔偿到账之前，梁志明就已经在整理材料准备追那辆送货的车了。他忍不住问："你们还没赔给我，怎么已经在追了？"

梁志明解释："按普通法，确实要赔了才能代位。不过我们这份不是水险保单，保单里写明了，就算还没实际付款，只要在处理理赔的过程中，我们也有权先开展追偿工作。这样快一些，对大家都有好处。"

周老板嗯了一声，没再追问。

第二笔账就没那么好打发了。那批被火烧掉的干鲍鱼，投保时按当时市价算的，赔的时候市价已经涨了。保险公司按旧价赔了十万元，但后来梁志明他们追回了十二万元，多出来的两万元怎么办？

梁志明把一张计算表递给周老板："这两万块要分。你自己要承担一部分。"

"为什么？"

"你还记得保单里有一万块的免赔额吗？那就是说，出事的时候你自己要先扛一万块，我们赔你九万。现在我们追回了十二万，超过了你承担的免赔额部分，所以超出的收益要分。"

周老板低头看那张表。免赔额承担一万，责任限额承担若干，还有一笔按保额比例算出来的共同保险份额，全部列得清清楚楚。他看了半天，把表放下，说："道理我懂了。分多少按合同来，但我那批货真的值那个价。"

梁志明点了点头，把表收回文件夹："周老板，还有一件事——如果当初保额不足，你只保了市价的八成，那追回来的钱，你也要按两成来分。就像合资做生意，你出了多少本钱，就拿多少分红。"

周老板叹了口气，拿起茶杯，说："做生意的账，还是不能省。"

## 原文定义

普通法规定代位权须在作出弥偿后方可取得，非水险保单通常另行约定允许提前行使代位权。代位追偿的收益以保险人已支付的弥偿金额为上限，超出部分归被保险人。若存在免赔额、责任限额或保额不足的共同保险安排，被保险人须按相应比例分享代位收益。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 保单约定可在实际付款前开展追偿 | 非水险保单可约定提前行使代位权 |
| 追回十二万，但周老板须按免赔额比例分享收益 | 免赔额安排下的代位收益分配 |
| 追回款超过已赔付金额，多出部分归周老板 | 代位收益以已支付弥偿金额为上限 |
| 保额只保了八成，周老板按两成分享追偿收益 | 比例分摊下的代位收益分享 |
