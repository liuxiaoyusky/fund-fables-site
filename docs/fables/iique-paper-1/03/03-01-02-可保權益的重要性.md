# 3.1.2 可保权益的重要性

## 故事

何师奶在油麻地开茶餐厅已经十七年了。丈夫走了七年，她靠着一档咖央多士和菠萝油，把两个女儿拉拔大了。屋企里，她最值钱的东西是那台用了十二年的火水雪柜，和柜台后面那个装茶叶的旧铁罐。

那年冬天特别冷，临近春节，工厂都赶着交单，何师奶的茶餐厅也是日夜连轴转。腊月二十八的凌晨，厨房电箱短路，走电的火舌沿着油腻的墙壁窜上去，眨眼间整个厨房就烧黑了。

庆幸的是火被街坊和消防员合力扑灭，没有伤到人。但厨房烧得一塌糊涂，年夜饭的食材全没了，连雪柜都变了形。何师奶站在烧焦的厨房门口，没有哭，只是轻轻地说了一句：「完了。」

春节过后，梁志明路过油麻地，听见街坊说起这件事，主动去找何师奶。他问：「师奶，你有没有买过火险？」何师奶打开柜子，翻出一叠泛黄的单据，最后抽出来一张红色收据，上面写着「火险保费」，但仔细一看，有效期截止到三年前。

「这个……」何师奶拿着收据，声音有些发抖，「上年我以为自动续期的，没有理……」

梁志明摇摇头：「师奶，过期了，这个等于没有买。而且……」他指着收据上的名字，「当时投保的是谁？」何师奶说：「是我先生的名字。」

梁志明沉默了一会，轻声说：「师奶，你先生走了超过两年，这张保单本来就没有可以续的基础了。」

何师奶把那张红收据放回抽屉，动作很轻，像是不想弄出声响。

梁志明后来自掏腰包，发动了几个街坊给何师奶凑了一笔周转资金，够她先把厨房收拾干净。但那台十二年的雪柜，何师奶最终没有换新的。她说：「留着也好，提醒我，有些事情唔可以拖。」

## 原文定义

缺乏可保权益，保险合同即属无效，退还保费的相关规则因所属险种及《保险业条例》《海上保险条例》的相关条文而异。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 保单过期三年未续 | 缺乏有效可保权益的合同属无效 |
| 投保人是何师奶已故丈夫的名字 | 可保权益须存在于投保人与标的之间，丈夫离世后关系消灭 |
| 何师奶以为自动续期，没有主动确认 | 可保权益须持续存在，否则合同失效 |
| 街坊集资资助周转而非保险理赔 | 无可保权益即无从获得合同保障，只能依靠其他救济 |
