# 3.5.5 如何被保单条件更改

## 故事

周老板那年重新投保，找了两间保险行同时报价。一间是本地行，费率平，条款简单；另一间是中环的英资洋行，费率高一些，但文书写得清清楚楚。梁志明帮他比较，选了英资洋行那份，理由是「条款清楚，你知自己做紧乜。」

一个月后，联安仓库又出了事，这次烧的是另一批放在周老板货架旁等待出货的干制食品。损失二十万，周老板翻出两份火险单，分别来自本地行和英资洋行——上次投保他两份都保留了，图个安心。

两间保险行接到索偿，先核查条款。

英资洋行的保单里有一项「比率份额条款」，白纸黑字写明：当有重复保险出现时，本公司只承担自身比率份额内的赔付，超出部分由被保险人自行向其他保险人追讨。

本地行的保单则没有这项条款，按正常程序直接认赔。

梁志明把两份保单摆在周老板面前，说：「你先看看。」

英资洋行那份，总保额较高，比率份额是四分之三，理论上应赔十五万；但条款写明它只赔自己那份，所以最终赔了十五万。

本地行那份比率份额是四分之一，赔了五万。

加起来刚好二十万，周老板一分钱没有少拿。

周老板却不高兴：「麻鬼烦，点解要我自己走去追？」

梁志明说：「因为你买的时候选了这份保单，它的条款就是这样写的。你要拿足二十万，就要亲自去追那间迟迟未赔的本地行。」

周老板拿著那张本地行的赔付支票，心里有点后悔：「早知就只留一份，唔使自己跑多一趟。」

过了几天，他又想起另一件事——上次的海上集装箱货轮，船上有一批客人的相机同时被家居保险和专门的全险保单覆盖了，当时因为有「免分担条款」，家居保险直接拒赔，全部由那张「更具体」的专门保单独自赔付。他想起自己帮那位客人解释过条款，心里明白了：「原来条款设计，决定了边个出钱、边个唔使出钱。」

他自言自语：「买保险，唔系睇边间平，系睇张合约点样写。」

## 原文定义

分担的衡平法则可被三种保单条款修改：（a）比率份额条款将保险人的责任限于其比率份额，被保险人须自行向其他保险人追讨余额；（b）免分担条款使本保单于重复保险出现时完全免除赔偿责任，由其他更具体的保单独自承担；（c）部分分担条款对特定情况下的分担范围作出限制，例如火险保单中的「海上条款」规定，若损失已由水险覆盖，火险不分担。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 英资洋行保单含比率份额条款 | 比率份额条款：保险人只赔自身比率份额 |
| 周老板须自行向本地行追讨余额 | 比比率份额条款将「向其他保险人追讨」的责任转移给被保险人 |
| 相机同时被两份保单覆盖，家居保险因免分担条款拒赔 | 免分担条款：更具体的保单独自承担赔偿 |
| 火险和水险约定了谁先赔、赔多少 | 部分分担条款：对特定重叠情况作出限制 |
| 周老板后悔「点解要自己跑多一趟」 | 保单条款选择的实际后果 |
