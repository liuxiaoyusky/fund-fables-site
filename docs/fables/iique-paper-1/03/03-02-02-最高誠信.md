# 3.2.2 最高诚信（Utmost Good Faith）

## 故事

那年冬天，周老板运一批花胶从日本经香港转口。保险行的人来了几次，每次递一份投保单给他填。他看完问题栏，照直答了交回去。

梁志明收到投保单的时候，皱了眉头。

「周老板，」他打电话过去，「你仓库旧年有一次火警纪录，你知道吧？」

「知道，但投保单上冇问我呢个问题喔，」周老板说，「佢问的是现有的设施和船期，旧年那单系意外，已经搞掂了，唔关今次的事。」

梁志明把那个下午把这个问题想了很久，然后去找林若兰。

林若兰正在替Mr. Thompson整理理赔文件，抬起头说：「你问的是 disclosure 的问题？」

「是。周老板的投保单上，没有具体问题问旧年火警，但他知道这件事。我在想，这是不是我们叫做『最高诚信』的那个范畴。」

林若兰放下手里的文件。「我去问问Thompson先生。」

那天傍晚，林若兰把回复转告梁志明：「Thompson先生说，在保险——特别是水险和火险——诚信的标准比普通合约要高。普通合约是你问我答，但保险是你就算没有问题，你也要主动披露所有可能影响保险人判断的事实。」

梁志明立刻明白了：「也就是说，就算投保单上冇列出『你旧年有冇火警』这一条，周老板也有义务主动告知？」

「正确。而且这个『重要事实』的定义，唔系以周老板自己认为重要与否来衡量，而是以一个审慎的保险人的角度去看——如果佢知道这件事，会不会影响佢决定接唔接这单生意，或者收几多保费。」

梁志明立刻致电周老板。周老板听完，愣了一会：「那我之前没有申报，算不算犯法？」

「还未构成犯法，」梁志明说，「但你现在补报，我们可以在保单正式生效之前把这个事实加上去。如果等到出了事再发现你冇申报，到时就大件事了——保险人可以以违反这个最高诚信原则为由，拒绝赔偿。」

周老板第二天亲自来了保险行，填了一份补充申报书。梁志明和林若兰把文件翻译成英文，交给Mr. Thompson。 Thompson看后，在申报书上盖了章，备注：「已悉，收悉。」

林若兰翻译完文件，顺手把一份副本夹进一个旧信封，没有交给周老板，而是递给了梁志明。梁志明打开一看，上面写著：「下次，记得问清楚客人有冇遗漏。」

他没有说谢谢，只是点了点头，把信封收好。

## 原文定义

保险受到一种更为严格、在普通法中常被称为最高诚信的原则所约束。它的意思是不论另一方是否要求有关的资料，各方必须向另一方披露所有重要的资料（称为「重要事实」）。例如，即使投保单上并没有相关的问题，火险投保人仍须向保险人披露有关过往损失的纪录。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 投保单未问火警纪录，周老板认为无需主动告知 | 一般合约诚信：未询问的事实无需披露 |
| 梁志明指出保险有更高标准，必须主动告知 | 最高诚信：未询问的重要事实也须主动披露 |
| Thompson先生从审慎保险人角度确认标准 | 重要事实的判断标准：客观审慎保险人的角度，非当事人主观认知 |
| 周老板补报后保单才正式生效 | 未申报的重要事实可在生效前补充，否则事后可构成拒赔理由 |
| 林若兰主动核查并提醒梁志明 | 隐晦感情线：工作细节中的互相照应 |
