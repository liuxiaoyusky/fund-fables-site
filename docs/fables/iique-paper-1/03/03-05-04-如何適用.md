# 3.5.4 如何适用

## 故事

何师奶的茶餐厅在湾仔开了十五年，她丈夫走得早，留下两个女儿和一个破旧的厨房。何师奶嘴上不说，但心里最怕的就是「一个崩 another一个」。她不懂什么大道理，只知道家里不能没有收入。

茶餐厅有个伙计叫阿宝，二十岁出头，是家里唯一的经济支柱。何师奶见他日日早出晚归，心里有点慌，便瞒著他买了两份人寿保险，心想：「一份得，两份稳阵啲。」

那年端午节过后不久，阿宝在送货路上出了车祸，人没了。

何师奶听到消息时，手里的切肉刀还没有放下。她站在厨房里，很久没有动。然后她打电话给梁志明。

两间保险行先后做了理算。一个星期后，两张支票先后送到何师奶手里，金额加起来，正好是两份保单约定的死亡给付之和。

何师奶有点吃惊。她问梁志明：「两份都赔？唔系话买多咗就只赔一份呢？」

梁志明摇头：「何师奶，火险、水险要分担，因为那些是弥偿性质——烧了你的屋，最多赔你起一回的钱。但人寿保险不同，它不是看你损失了多少，而是按约定金额赔付。你买了两份，就是两份都算。」

何师奶沉默了很长一段时间。后来她轻声说：「我只是不想阿宝走了，他屋企人一个仙都没有。」

两份人寿保险的赔付，最后全数交到了阿宝父母手里。何师奶站在茶餐厅门口，看著街上的人来人往，没有说什么。她转身回厨房，继续切她的肉。

梁志明走出茶餐厅时，心里记下了师奶那句话：「一个崩 another一个。」——这不是理算规则，但它是人心。

## 原文定义

只有当弥偿出现时，分担才会适用。如果被保险人购买了两份或以上独立的人寿保险，由于人寿保险不适用弥偿原则，每份保单均须全额赔付。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 何师奶同时购买了两份人寿保险 | 人寿保险的给付以约定金额为准，不计算实际损失 |
| 两份人寿保险都全额赔付了 | 人寿保险不适用弥偿，故不产生分担 |
| 火险、水险才需要分担 | 分担只适用于弥偿性质的险种（财产险、火险、水险等） |
| 何师奶说「我只是不想阿宝走了他屋企人一个仙都没有」 | 人寿保险的实际意义：家庭经济保障，而非财产价值填补 |
