# 3.4.3 与可保权益的关联

## 故事

那年冬天，周老板的海味铺失了火。

火是从隔壁文具店烧过来的，铺里的干货、花胶、鲍鱼全部报销。周老板站在街上，看着伙计们往外搬烧焦的木箱，半晌说不出话。

他想起梁志明年初跟他讲过一句话："周老板，你这份保单，保的是铺子里的货，赔的时候要看你的货到底值多少。"

当时他没太在意。

理赔员来了，是个戴金丝眼镜的年轻人，姓吴。吴先生翻了翻保单，又看了看周老板仓库的进货单据，开口说："周老板，你投保的金额是八十万，但你这批货的实际价值是一百万。"

"意思是我赔少了？"周老板声音大了。

吴先生没急，从公文包里抽出一张表："保单赔的是你的权益。你对这批货有多少'份'，就赔你多少。你只保了八成，赔的时候也按八成来。"

周老板不服气，转头去找陈伯。陈伯听完，拿出一张旧账簿指给他看："你看，你当年在船公司跑船，你老婆给你买过一份寿险，那份险的赔法就跟这批干货不一样。"

"哪里不一样？"

"你老婆买那份寿险，是因为她跟你有切身关系，你出事她最受影响。这叫可保权益——她跟你之间有一条线，这条线断了，她就受损失。"陈伯说，"但你老婆没办法证明她'损失了多少'。你值多少钱？感情怎么折算？孩子以后少一个人养，这个数怎么定？所以寿险干脆不定'损失'，直接约定一个数。"

周老板低头看着自己烧焦的干货："所以我这批货……"

"你买这份火险的时候，权益是清清楚楚的——这批货进货花了多少钱，市场价多少，账面上都有数。你损失了多少，赔多少，天经地义。"陈伯说，"这就是为什么财产险要按实际价值来赔。赔到手的钱，不会比你实际丢掉的多。"

吴先生后来按八十万的全额赔了。周老板补进了二十万的货，铺子重新开张。

那天收铺之后，陈伯一个人坐在天后庙前的石凳上，看了很久海。他想起年轻时候未婚妻离开香港去英国，没有带走任何东西，也没有拿到任何赔偿。有些东西，没法按数目来清点。

周老板后来每年续保，都主动把货值重新估一遍，再也没有少报过。

## 原文定义

可保权益是对某项标的之财务「权益」，其价值多少，就应在全损的情况下赔付多少钱，才能让保单持有人得到完全补偿；然而寿险及人身意外保险因涉及价值无限之可保权益，一般不适用弥偿原则。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板只保了货值的八成，赔款也按八成 | 可保权益决定弥偿的边界：权益多少，赔多少 |
| 花胶鲍鱼的价值可以算出来 | 财产保险的权益可精确量度，按实际价值弥偿 |
| 陈伯说老婆买寿险"无法折算损失" | 人寿保险涉及价值无限的可保权益，不适用弥偿 |
| 寿险直接约定一个数 | 寿险/意外险以约定利益代替弥偿 |
