# 3.1.6 转让（Assignment）

## 故事

周老板那年夏天决定把生意收了，搬到西贡住。那批存在仓库里的干鲍鱼和花胶，他想全部转给接手生意的阿坚。问题来了：那张水险保单怎么办？

他去找梁志明。

梁志明说：「有两种做法，后果完全不同。」

第一种是把整份保险合约转让给阿坚。这样的话，保单从周老板身上转到阿坚身上，损失发生时保险公司赔偿的是阿坚的损失，不是周老板的。这种转让，转让的那一刻，双方都必须对那批货享有可保权益——周老板有，因为货本来是他的；阿坚也有，因为转让之后货就是他的。寿险的转让则不同，即使转让了合约，受保生命也不会换人。

第二种是只把出事后收钱的权利转给阿坚。保单本身仍然承保周老板的损失，但出事后有权向保险公司提起诉讼拿钱的是阿坚，不是周老板。这种转让叫「转让收取保险金的权利」。它的特点是：阿坚不需要对那批货有可保权益——实际上，这种转让可以构成馈赠，纯粹是把收钱的资格转让出去。

周老板问：「那要不要通知保险公司？」

梁志明说：「第二种转让——转让收取保险金的权利——不需要保险公司同意。但第一种转让——合约本身的转让——就要看情况了。水险单和寿险单有个好处：它们可以不经保险公司同意而转让。」

周老板最后选择了第一种，把保险合约连同货一起转给阿坚，通知了Mr. Thompson，办了正式转让手续。转让书上清楚写着一句：转让的是权益，不是责任。阿坚接手之后，货出了事由他索赔，但转让之前欠下的保费仍然由周老板负责。

阿坚接过文件的时候，对周老板说：「接手之后的事，我来孭；接手之前的事，你来孭。」

周老板点点头。他知道这就是转让的本质：权益可以移转，义务仍然留在原地。

## 原文定义

(a) 转让保险合约：保单从转让人转到承让人身上，承让人须有可保权益，事故发生时保险人向承让人的损失作出赔付。寿保险转让不换入新的受保生命。(b) 转让收取保险金的权利：保单仍承保转让人的损失，但承让人有权起诉保险人索赔；承让人无需有可保权益，可构成馈赠。(c) 转让保险合约时双方均须有可保权益。(d) 转让收取保险金的权利无需保险人同意；寿险单和海上货物保险单可不经保险人同意而转让。(e) 转让的是权益而非责任。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板把整份保险合约连货转给阿坚 | 保险合约转让：权益和风险一同转移，双方须有可保权益 |
| 转让之后出事，保险公司赔给阿坚 | 合约转让的效果：赔付与权益同步转移 |
| 阿坚只接手之后的责任，之前的保费周老板负责 | 转让的是权益，不是责任 |
| 把收钱的权利转让给人不需要可保权益 | 收取保险金权利的转让：承让人无需可保权益，可构成馈赠 |
