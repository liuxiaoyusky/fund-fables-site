# 7.4.3 保险业所面对的风险

## 故事

上环的海味舖「广栈」在周老板手里做了三十多年。儿子周家俊前年结婚，媳妇娘家在泰国，周老板想把一部分积蓄转到海外，问梁志明有没有辨法。

梁志明在保险行做了六年，对周老板一家人很熟悉。周老板的女儿早年嫁去英国，儿子留在香港帮手打理舖头，周老板手上有几张人寿保单，都是早年买的，现金价值不高，也不是什么大额保费。

「整付保费。」梁志明说，「一次过付清那种，现金价值累积快，临时要用钱也可以周转。」

周老板拿出泰国那边银行的汇票副本，说媳妇娘家在曼谷有些项目，资金迟早要转回来，想先在保险里放一段时间，「起码有张单据，来路清楚。」

林若兰在洋行见过太多这类操作。她拿著那张整付保费建议书翻了很久，对梁志明说：「整付、储蓄型、可以改受益人——你自己说说，这几个加起来是什么。」

梁志明不笨。他知道保险单这种东西，付了钱，过几年退保或者到期，钱就回来了，中间谁拿钱、拿多少，全看保单上写的是谁。周老板要改受益人，说不定过几天就改成了媳妇在泰国的某个亲戚。资金来路不明，退保金额却干干净净，这正好是整付保费保险单最容易被拿来用的地方。

他去找Mr. Whitfield。律师楼在花园道，办公室外面是港督府的山坡，空气里有英国烟草的味道。Mr. Whitfield把建议书放在桌上，说：「保险公司卖的是长期承诺，别人看中的也正是这个长期。现金价值累积快，保单又可以抵押或转让——这些在正当用途上是优点，在错误用途上也是弱点。」

他说的不是「你要拒绝这个客户」，而是「你要想清楚，万一这笔钱的来路有问题，你们公司就变成了帮凶」。不是不做生意，而是看清楚是什么生意。

梁志明回去把情况向公司合规部报告，公司决定在核保之前，要求周老板提供资金来源说明。周老板起初不高兴，觉得被当成了可疑人物。梁志明上环跑了一趟又一趟，陪他整理泰国那边的汇款证明，把来路一步步说清楚。周老板的女儿从英国写了封解释信，连同泰国银行的对帐单一起交了上去。

单子最终还是做了。周老板的资金来源确实没有问题，只是儿媳两家的资产转移确实复杂，多解释几步对大家都好。

林若兰把那份资金来源说明复印了一份存档。她对梁志明说：「不是不信任周老板，是这一行里，有些人专门找看起来正当的东西来藏东西。」

## 原文定义

保险业面对洗钱和恐怖分子资金筹集活动的风险。容易被利用作为进行洗钱或恐怖分子筹资活动工具的长期保险合约包括：整付保费合约、可储存现金价值的整付保费人寿保险单、固定及变动年金，以及二手储蓄人寿保险单。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板想把资金先放进保险单再转出去 | 长期保险合约可用作资金转移和受益人变更的工具 |
| 整付保费、储蓄型、可以改受益人 | 容易被利用的长期保险产品特征 |
| 梁志明上报合规部、要求资金来源说明 | 保险中介人肩负防止洗钱的职责 |
| 泰国汇款证明和英国女儿的解释信 | 客户尽职审查的具体要求 |
