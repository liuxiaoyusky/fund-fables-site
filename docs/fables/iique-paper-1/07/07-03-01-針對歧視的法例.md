# 7.3.1 针对歧视的法例

## 故事

何师奶的茶餐厅开在佐敦吴松街，铺名就叫"何记"，招牌是旧式霓虹灯管，红色的"何"字夜里亮得很。她守了这间铺头十二年，丈夫走的时候儿子七岁，如今儿子已经中学毕业，在酒楼做帮厨。

那年夏天，差馆里来了个人，说是平等机会委员会的调解主任，姓林，专门处理街坊投诉。何师奶以为是来查牌的，后来才知道不是。

事情要从上个月说起。何师奶的茶餐厅旁边新开了一间烧腊店，老板姓马，两人平时见面会点头打招呼。马老板的烧腊店需要扩大投保范围，找了梁志明所在的保险行，承保之前需要做一次店铺风险评估。

评估报告里有一条写着：建议对店铺结构进行加固，特别是电气线路。

梁志明把报告交给马老板的时候，顺口说了一句："报告里还提到，如果投保人是女性且独自经营，需要额外提交一名紧急联络人作为担保。"

马老板把报告拿回去，过了一个星期没有消息。梁志明打电话去问，马老板说："我不需要加担保，你帮我把那条删掉就行了。"

梁志明查了查公司的核保手册，找不到这条规定。他又去问负责人，负责人说可能是评估员自己加的，让他改掉就是了。

后来那条被删了，马老板的保单也顺利出了。

但何师奶听说这件事之后，心里一直不舒服。她想：如果换成是我，他们会不会也要求我加担保人？

她没去找保险行，而是直接走到了平等机会委员会的办事处。接待她的是林主任，四十岁上下，穿浅灰色西装，说话不快但很清楚。何师奶把事情说了，林主任听完，在笔记本上记了几行字。

"何女士，"他说，"香港现在有四部条例专门管这件事。《性别歧视条例》、《残疾歧视条例》、《家庭岗位歧视条例》和《种族歧视条例》，分别针对不同类型的歧视，由平等机会委员会负责执行。"

"那保险业会不会也有这种事？"何师奶问。

"保险业务是这些条例的适用范围之一。"林主任说，"但我要告诉你，条例里也有规定，在某些合理情况下，保险人可以对不同投保人给予不同对待——前提是有合理的统计数据支持，而且处理方式是适当的。这叫'合法差别待遇'。"

何师奶听完，想了想，说："那个评估员自己加上去的担保要求，是不是就是不合理的那种？"

林主任没有直接回答，只是把委员会的宣传单张递给她，说："你问得很对。任何机构在处理投保申请的时候，都不能凭印象或者偏见来设限，要看有没有实际的合理依据。"

何师奶把那叠单张带回茶餐厅，放在柜台后面，留给伙计们看。

第二天梁志明来喝茶的时候，何师奶叫住他，把单张给他看。梁志明看完没说话，低头喝茶。何师奶说："你们做保险的，要学的东西真多。"

梁志明放下茶杯说："何师奶，您说得对。我回头查一查，看公司还有没有其他地方也有这种问题。"

## 原文定义

平等机会委员会（Equal Opportunities Commission, EOC）负责执行四部反歧视条例，分别针对：性别与婚姻状况（《性别歧视条例》）、残疾（《残疾歧视条例》）、家庭岗位（《家庭岗位歧视条例》）以及种族（《种族歧视条例》）。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 评估员对独自经营的女性要求额外担保人 | 性别歧视：无合理依据对女性投保人附加额外条件 |
| 何师奶向平等机会委员会咨询 | EOC是执行四部反歧视条例的法定机构 |
| 林主任提到"合法差别待遇"须有合理依据 | 条例允许在特定条件下差别对待，但须符合法定标准 |
| 梁志明事后核查公司其他做法 | 保险机构须定期审视自身是否存在无根据的歧视性要求 |
