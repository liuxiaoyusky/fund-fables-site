# 7.4.5a 打击洗钱及恐怖分子资金筹集（金融机构）条例

## 故事

梁志明那年二十五岁，在保险行里是最新进的营业员，公司让他跟著前辈学核保。他师父姓梁，说话不多，但做事仔细，在公司做了十七年，人称「老梁」。

那年年底，公司内部查帐，发现有一笔整付保费的资金说明不完整，客人提供了文件，但核保部门处理得比较赶，没有来得及向合规部正式请示，就先把保单生效了。后来合规部追那份文件，发现档案里根本没有副本。

公司自查，写了一份内部报告，上交金管局。报告里说，情况是「流程延误」，不是蓄意违规。

Mr. Whitfield看了那份报告，在空白处用蓝色墨水笔批了几行字：「流程延误不等于没有问题。如果资金真的涉及非法来源，这份延误本身就可能让金融机构变成被利用的工具。」

他约了公司几个主管开会，第一句话就说：「你们看看这份刑事罚则，最高两年监禁加一百万罚款，这不是吓人的数字，是真的有人会坐的。」他翻到另一页，「如果是出于欺诈意图，七年监禁。你们觉得自己的机构不会被人怀疑欺诈？」

老梁坐在角落里，没说话。他想起那年七月，有个客人拿了一张泰国银行的本票来投保，金额不小，护照复印本模糊，当时他催得急，没有再追一份清晰的副本就直接递上去了。现在想来，那是整个流程里最大的漏洞。

ICAC的赵先生后来也来了保险行，做例行走访，没有带搜查令，只是来看合规制度。他翻了几份记录，问老梁：「你有没有怀疑过那笔款项？」老梁说：「那时候只觉得客人的文件不够齐，没往深处想。」

赵先生说：「条例说的，不是你要确定他一定有问题，而是你有合理理由怀疑，就应该报告。你没有这个义务吗？」他合上文件夹，说：「机构有机构的责任，个人也有个人的。」

老梁后来写了一份详细的说明，解释当时的情况，解释资金来源的后续核查结果。Mr. Whitfield帮他看了几遍，纠正了一些法律措辞。说明最终提交，纪律行动是内部谴责，罚款没有了，没有上升到刑事层面。

但老梁在保险行门口站了很久。他想起进公司第一天，前辈跟他说：「做保险，做的就是别人的信任。」他现在才真正明白这句话的意思。

## 原文定义

根据打击洗钱条例，金融机构如果明知而违反该条例指明的条文，即属犯罪，最高可被判监禁两年及罚款一百万港元。如果出于欺诈有关当局的意图而违反，最高可被判监禁七年及罚款一百万港元。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 保险公司未完整核查资金来源就出单 | 金融机构对客户尽职审查的法定义务 |
| 延误提交内部报告 | 「明知而违反」可能构成刑事责任 |
| 老梁未主动上报可疑款项 | 机构及个人均有举报责任 |
| 纪律行动：谴责及纠正程序 | 违反条例的纪律处分形式 |
