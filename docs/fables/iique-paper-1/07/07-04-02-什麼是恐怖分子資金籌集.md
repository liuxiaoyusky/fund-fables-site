# 7.4.2 什么是恐怖分子资金筹集

## 故事

林若兰在英资律师楼做了五年打字员，办公室在中环一栋写字楼里，窗外看得见维港。那年九月，台风来袭，学校停课，写字楼里的人走得七七八八，林若兰留下来把几份紧急文件打完。

晚上八点多，她正准备收拾东西走人，办公桌上的电话响了。

电话那头是一个男人的声音，没有报姓名，说话很慢："林小姐，我有一件事想请你帮忙。我人在海外，有一些款项想通过你们律师楼的账户转进来，然后我再叫你们帮我做一笔海外投资。具体操作，你们律师楼的Mr. Whitfield会处理。"

林若兰握着话筒，手心微微发凉。

她问："先生，请问您贵姓？"

对方没有回答，只是说："具体的，到时再说。"

电话挂断了。

林若兰站在原地，盯着电话机看了很久。窗外风雨大作，维港的灯火一片模糊。

她走到隔壁房间，敲了敲Mr. Whitfield的门。Whitfield还在，正对着一叠合规文件皱眉。林若兰把刚才的电话原原本本说了一遍。

Whitfield听完，放下手里的文件。

"若兰，你做得对。"他说，"这件事有几处非常可疑：第一，一个陌生人打电话来，要求借用律师楼的账户转账；第二，对方指定了我，说'Mr. Whitfield会处理'，这意味着他知道我们律师楼的运作方式，甚至可能知道内部人员分工；第三，他说款项来自海外，要做海外投资——这个模式跟恐怖分子资金筹集的典型路径高度吻合。"

林若兰问："恐怖分子资金筹集具体是指什么？"

Whitfield说："《联合国（反恐怖主义措施）条例》里有定义。简单来说，有三种情况会触犯条例：第一，明知或罔顾地，为恐怖主义行为提供或筹集资金；第二，明知或罔顾地，向恐怖分子或与恐怖分子有联系者提供财产或金融服务；第三，以上述方式为恐怖分子的利益筹集财产或寻求服务。"

他站起身，走到窗边，台风正从维港正面吹过。

"你说那个人说'人在海外'，又说'款项从海外转进来'，如果他真的是恐怖分子或者与恐怖分子有联系，那这通电话本身就构成了'向恐怖分子提供金融服务'的行为。即使律师楼最终没有实际转账，知道这件事并保持沉默，在法律上也有问题。"

林若兰问："那应该怎样？"

Whitfield说："立刻向联合财富情报组提交可疑交易报告。如果必要，我会亲自处理这件事。"

第二天一早，Whitfield写好了报告，签了字。林若兰坐在打字机前，把报告打了两份，一份存档，一份由Whitfield亲自送到相关部门。

台风过了，维港恢复了往日的平静。林若兰下班的时候，走过律师楼大堂，看见Whitfield站在门边看雨后的街道。

"Mr. Whitfield，"她轻声说，"那个人可能只是个骗子，未必真的是恐怖分子。"

Whitfield说："我知道。但在搞清楚之前，每一个环节都必须按规定处理。不是因为我们确定对方是什么人，而是因为我们不确定对方是什么人。"

林若兰点点头，走进雨后的街道，霓虹灯在水洼里碎成了很多颜色。

## 原文定义

"恐怖分子资金筹集"是指：(a) 在怀有将财产用于恐怖主义行为的意图下，或明知财产将用于该目的下，提供或筹集财产；(b) 明知或罔顾某人是否为恐怖分子或与恐怖分子有联系者，向其直接或间接提供财产或金融服务；(c) 明知或罔顾某人是否为恐怖分子或与恐怖分子有联系者，为其利益筹集财产或寻求金融服务。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 陌生人要求借用律师楼账户转款 | 以提供金融服务的方式参与恐怖分子资金筹集 |
| 对方知道内部人员分工 | 可疑人物掌握机构运作信息，属于预警信号之一 |
| Whitfield说"人在海外+海外汇款+指定律师处理" | 恐怖分子资金筹集的典型路径特征 |
| 向联合财富情报组提交可疑交易报告 | 金融机构及其从业人员的法定合规义务 |
| "不是因为确定对方是什么人，而是因为不确定" | 恐怖分子资金筹集的认定不要求实际证明对方是恐怖分子 |
