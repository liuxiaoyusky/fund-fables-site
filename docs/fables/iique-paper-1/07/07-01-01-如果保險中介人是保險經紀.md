# 7.1.1 如果保险中介人是保险经纪

## 故事

周老板要帮海味铺买保险，找了两家保险公司询价，又托朋友介绍了一个保险经纪。朋友说："你揾经纪好，经纪帮客人揾，唔系帮边家保险公司揾。"

经纪就是梁志明。他帮周老板盘了一下：铺面财产要保、存货要保、伙计工伤要保、老板本人万一出事铺头点算也要保。周老板问："你推荐边家？"

梁志明拿出三间公司的报价单，一间一间比：保障范围、赔偿比例、免赔额、保费。他说："我帮你比较过，你觉得边份适合你就签边份。"

周老板翻了翻，发现三份报价里有一间保费最便宜，但赔偿条款里有一条写明"地震、海啸等不可抗力免赔"。另一间贵一成，但赔付范围广得多。周老板问："你觉得呢？"

梁志明说："我唔帮你拣，边个都唔帮。我帮你睇清楚，拣边份你自己决定。但你拣完之后，我就代表你去跟进，日后理赔有任何问题，你揾我。"

周老板的老账房陈伯在旁边听了，问："你哋经纪系两边都帮？"

"唔系。"梁志明答得干脆，"我收客人钱，帮客人揾嘢。我唔系保险公司的代理，唔会偏帮边间。保险公司出货，我帮客人把关；客人出事，我帮客人追赔。"

后来周老板的仓库在台风天浸了水，存货损失不少。梁志明陪他整理索赔文件，发现保险公司的评估报告漏算了一笔货款。他写了封信，逐条列出差额，附上原始单据。理赔最终比预期多了三成。

周老板说："当初以为揾边间保险公司买就揾边间，依家先知揾经纪系揾对人。"

## 原文定义

保险经纪被视为保险业的专家，并须独立于任何一个保险人。保险经纪是保单持有人的代理人，对保单持有人负有代理法下的法律义务，必须以客户的最佳利益行事，将客户利益置于首要地位。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明拿出三间公司的报价单比较 | 保险经纪须独立于任何保险人 |
| "我帮你睇清楚，拣边份你自己决定" | 以客户最佳利益行事的建议义务 |
| 漏水理赔时经纪帮客人追赔差额 | 保险经纪是保单持有人的代理人 |
| 梁志明不代表任何一间保险公司 | 独立性的核心要求 |
