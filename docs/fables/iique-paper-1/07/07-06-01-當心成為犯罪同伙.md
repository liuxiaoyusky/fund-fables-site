# 7.6.1 当心成为犯罪同伙

## 故事

那年十一月，联安保险行来了个新的兼职伙计，叫阿昌。

阿昌是坏了一条腿的年轻人，走路有点跛，据说早年在大学读过书，后来因事辍学。梁志明没多问，每个人都有自己的故事。经理说让阿昌帮忙接电话、整理客户卡片，做些轻巧的事。试用期三个月。

阿昌上手很快，英文好，打字快，还能帮林若兰分担一些来往律师楼的文件翻译。林若兰对这个新来的伙子颇有好感——他做事仔细，有分寸，不像有些后生仔那样得寸进尺。

转正前两周，出了件事。

那天梁志明外勤，回行的时候看见阿昌坐在他的工位前，对著一份汽车保单的副本发呆。旁边站著一个西装笔挺的中年男人，正在低声说话。梁志明放轻脚步，停在门口。

「……你帮我签这份申请表，第三页那栏，如实填写就『甩唔少保费』。」中年男人声音压得很低，「你们公司的人不会查出来。」

阿昌没说话，手指在那份文件上轻轻敲了两下。

梁志明立刻走进去，咳了一声。

阿昌回头，表情平静：「梁哥，这位是黄先生，黄生。」

黄先生转过身，笑容很职业：「梁先生，久仰。我和黄昌是旧相识，今天路过，上来打个招呼。」

梁志明看了阿昌一眼，又看了那份文件一眼。文件上有几处填好的数字——保额那栏，金额高得不正常。

「黄生，阿昌是我们行里的人，」梁志明的语气很慢，「他的工作范围不包括代客人填申请表。」

黄生脸上那层笑容收了一收，又很快回来：「梁先生误会了，我只是——」

「我知道。」梁志明把那份副本从阿昌手下抽出来，「阿昌，麻烦你帮我冲杯茶。」

黄生走了以后，梁志明把那份副本收进柜子，锁好。

那晚他和阿昌一起下班，走在德辅道西的路上，梁志明终于忍不住。

「阿昌，他叫你签那份表的时候，你知道他想做什么吧？」

阿昌沉默了一会儿：「……知道。」

「那他给你什么？」

「事成之后一千。」阿昌声音很轻，「我没答应，也没拒绝。」

梁志明停下脚步。这一刻他忽然想起 Mr. Whitfield 在夜校说的那句话：「从犯和主犯的法律责任是一样的。不是你不动手就没事——有权阻止却选择袖手旁观，在法律上一样系罪。」

「阿昌，」他说，「下次他再来，你马上叫我。」

「……你不打算举报他？」

「我打算你先学会保护自己。」梁志明拍了拍阿昌的肩，「黄生那种人，今天不走，明天也会再来。ICAC 赵先生他们那边，我会请林若兰帮忙留意一下。」

阿昌没说话，但他们继续往电车站走的时候，梁志明注意到那个年轻人的脚步比平常轻了一些。

第二个礼拜，黄生果然又出现了。这一次梁志明不在，林若兰在律师楼直接打了 ICAC 的举报热线。

后来黄生被起诉的时候，阿昌作为证人出庭。法官在判词里特别提到：「……第三被告声称自己并未实际参与填写虚假文件，但其放任态度构成法律意义上的协助及教唆。」

阿昌最后没被起诉，但他在联安保险行的试用期被延长了三个月。梁志明请他喝了杯鸳鸯，告诉他：「记住这次。以后任何时候，有人叫你『帮衬』他做你明知不对的事，你要做的第一件事不是考虑后果——是立刻拒绝。」

阿昌点点头。那杯鸳鸯喝到一半，他忽然说：「梁哥，当时你是不是想过，黄生可能是你上线经理的人？」

梁志明没回答。有些问题，不需要答案。

## 原文定义

一项刑事罪的从犯是指一名协助、教唆、怂使或促致犯该罪行的人。从犯行为几乎必然仅仅是协助或鼓励犯该罪行即可。控方必须证明被告在进行协助行为时怀有所需犯罪心意——即被告知悉他的行为能协助或鼓励干犯该刑事罪行，而不需证明被告有从犯罪中获益的意图。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 黄生要求阿昌代填虚假申请表 | 主犯从事犯罪行为 |
| 阿昌「没答应，也没拒绝」的态度 | 不作为的参与同样有罪 |
| 阿昌有权控制局面却未阻止 | 蓄意不行使权力可能构成积极鼓励 |
| 梁志明立即干预、并请林若兰举报 | 辨认刑事活动并远离、主动协助举报 |
| 黄生被起诉、阿昌作为证人 | 从犯与主犯法律责任相同（控方已证明犯意） |
