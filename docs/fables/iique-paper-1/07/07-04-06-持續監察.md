# 7.4.6f 持续监察

## 故事

周老板在永亨保险行投保了三十年。

陈伯看着他从三十岁的中年男人变成六十岁的老头子。周老板的保费从每月几十块慢慢涨到每月几百块，保单也从一份变成三份。中间有几年他没续保，有几年他加了保额，陈伯都记在账本里。

所以当周老板最后一次走进永亨保险行，要退掉其中一份保单、把所有现金价值一次过取走的时候，陈伯第一个注意到了。

三十年来，周老板从没退过保。

"周老板，"陈伯把眼镜扶上去，看着他，"这份保还有两年就供完了，现在退，不划算的。"

"我知道。"周老板说，"但我急用钱。"

"急用钱可以贷款，保单贷款，利息比外面低。"

"不了，"周老板摆摆手，"这次要用的是大数目。"

陈伯没再问。他把周老板的退保申请收下，然后在账本那一页折了一个角。

下午梁志明回来，陈伯把账本给他看。

"三十年，十张单，从来没退过保。"陈伯指着那个折角，"这个角，是提醒我，这笔账要盯紧。"

梁志明翻了翻账本。他注意到周老板最近三个月的保费都是提前交的，现金存款也比往常多。

"他儿子在加拿大。"梁志明说，"上个月刚说汇了两千加币过去。"

"对。"陈伯说，"但他现在要取的数目，比他儿子的学费多很多。"

梁志明没把这件事直接和洗钱联系在一起。他只是觉得：一个人三十年没变过的习惯，忽然变了。这本身就是一种信号。

指引3说的持续监察，不是让你盯着一个人一次，而是让你用时间来观察一个人。

梁志明后来请周老板多等几天，说要等合规主任核完这份退保申请。周老板等了两天，然后自己打来电话，说"不用了"，退保的事以后再说。

那笔保费后来还是存进了周老板的账户。梁志明在记录里写了四个字："继续观察。"

陈伯后来在账本上补了一句："三十年看一个人，不难。难的是三十年一直在看。"

## 原文定义

指引3强调，有效的持续监察措施对于了解客户活动至为重要。它不但是有效的打击洗钱/恐怖分子资金筹集系统中不可缺少的部分，也有助于金融机构了解客户及侦测异常或可疑活动。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 陈伯看着周老板投保三十年，了解他从未退保的习惯 | 持续监察的基础是长期了解客户的正常活动模式 |
| 周老板三十年第一次要退保取走现金价值 | 客户行为模式的重大变化本身即是需要警觉的信号 |
| 梁志明注意到保费提前交、存款比往常多 | 持续监察须关注客户账户的资金流动异常 |
| 梁志明请周老板"等合规主任核完"而非直接拒绝 | 持续监察发现异常后须按程序处理，而非直接拒绝服务 |
| 梁志明记录"继续观察" | 监察结果须记录在案 |
| 陈伯说"三十年一直在看" | 持续监察是长期、持续的行为，不是单次核查 |
