# 7.4.5b 贩毒（追讨得益）条例

## 故事

周老板有个老朋友姓陈，在码头做了二十多年的出入口。1979年入冬的时候，陈叔的女儿要结婚，陈叔走进保险行，说要给女儿买一份储蓄型人寿保单，受益人写女儿的名字，「嫁妆的一部分」。

保费是五十张现钞，用橡皮筋扎成一叠。

梁志明接下来，点清楚了钞票，填了收据，按程序请陈叔提供身份证明和地址证明。陈叔的文件都很完整，交了身份证、住了二十年的地址证明、还有公司的在职证明。梁志明把文件存档，准备递交核保。

林若兰路过，看见桌面上那叠钞票，停了停，没说什么。

第二天，Mr. Whitfield把梁志明叫到一旁，说：「昨晚我看了你的单子。有几个地方我想跟你核实一下。」

他说的不是「这单有问题」，而是：「陈先生的公司是做正规出入口的，对吗？他平时的保费规模是这个量级吗？」

梁志明去查了公司的业务记录，发现陈叔过去的保费从来没有超过五位数，这一次突然跳到六位数，而且全部是现钞，没有支票，也没有转帐。

Mr. Whitfield说：「《贩毒（追讨得益）条例》说，知道或有合理理由相信任何财产代表任何人的贩毒得益，而仍然处理该财产，即属犯罪。你不需要有确实证据，有合理理由怀疑就已经足够了。」

他停了一停，说：「当然，现钞本身不代表任何事，很多正当生意也用现钞。但是一个客人突然拿出一笔远超平时量级的现钞，这就值得问几个问题了。」

梁志明去找周老板，请他帮忙侧面了解一下。周老板回来说，陈叔前阵子确实在外面借了一笔钱周转，具体做什么没有多说，但那笔借款是问一个在澳门有背景的人借的，利息不低。

Mr. Whitfield说：「这个信息本身不能证明任何事，但结合现钞的量和频率，已经足够让我们有一个『合理怀疑』的基础。」他指示公司向财富情报组提交披露，同时在内部记录上如实说明了核查经过。

陈叔的保单最终没有做成。公司以「核保标准未达要求」为由婉拒，没有透露具体原因。林若兰事后跟梁志明说：「不是我们在怀疑他，是这个情形本身需要有一个正式的交代。」

梁志明把整份文件订好，存进了归档柜的最里层。

## 原文定义

根据《贩毒（追讨得益）条例》，如果有人知道或有合理理由相信任何财产代表任何人的贩毒得益而仍处理该财产，即属犯罪，经定罪后的最高刑罚为监禁十四年及罚款五百万港元。以订明方式向财富情报组提交报告，可获得法定免责辩护。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 现钞缴付远超平常量级的保费 | 可疑交易的识别信号 |
| 周老板侧面得知借款人背景 | 侧面核查作为合理怀疑的依据之一 |
| 公司向财富情报组正式披露 | 以订明方式提交可疑交易报告以获法定免责 |
| 保单因核保标准未达要求而婉拒 | 机构对可疑交易的主动拒绝机制 |
