# 7.4.6h 恐怖分子资金筹集及扩散资金筹集

## 故事

事情是从一碗鱼翅开始的。

周老板那天拿了一盒干鲍给梁志明，说是一个新客户送的。那客户姓马，说是在东南亚做干货生意，想通过周老板的渠道进一批货，货款用港币结算。

周老板说："人家出手大方，你帮我看看这单能不能接。"

梁志明没接那盒鲍鱼。他回去查了那个马先生的名字，然后把名字递给Mr. Whitfield。

Mr. Whitfield花了两个晚上，查出了一串东西。

"这个名字，"他把一张纸推过桌面，纸上只有几行字，"在联合国制裁名单上出现过。虽然后来撤下来了，但中间那几年，他和某些组织的关系没有完全断干净。"

梁志明问："那他现在能不能开户？"

"不能。"Mr. Whitfield说得很清楚，"《联合国制裁条例》授权保安局局长冻结怀疑是恐怖分子的财产。任何人不得直接或间接向恐怖分子或与其有关联者提供任何财产或金融服务。违者最高可判七年监禁。"

"那如果有人替这个姓马的来买保险呢？"

"一样。"Mr. Whitfield把文件合上，"替他买，替他受益，或者用他的钱，无论哪种形式，都违法。十四年。"

梁志明当时吃了一惊："十四年？"

"是的。"Mr. Whitfield的声音很平，"如果证明是主动参与恐怖分子资金筹集，最高十四年监禁，罚金由法庭定，不封顶。"

后来的事情和梁志明想的不太一样。那个姓马的客户没有再来。周老板也收回了那盒干鲍，说"算了，不做这笔生意了"。他不知道中间查出了什么，但街坊的直觉告诉他，有些钱是不能赚的。

ICAC的赵先生后来来永亨保险行做培训。他带了一张名单，让在场所有人传阅。名单上的人名有英文的，有中文的，也有阿拉伯文的。赵先生说："这张名单，记住它。每一次接触新客户，每一笔大额交易之前，查它。"

"如果名字在上面呢？"有人问。

"冻结账户，停止交易，向财富情报组报告。"赵先生说，"不要问为什么，不要通知对方，不要给自己找借口。"

那天下课后，梁志明留在会议室，把那份名单抄了一遍。他没有抄全部，只抄了最常见的几个姓氏。他把那几个字写在办公桌玻璃板底下，压在一叠保费收据下面。

何师奶有回去保险行喝茶，看见了那几个字，问他这是什么。

"查账用的。"他说。

"查什么账？"

"查不应该进来的钱。"

何师奶似懂非懂，但没再问。

那盒鲍鱼后来还给了周老板。周老板当晚拿回家炖了汤，没有分给梁志明。

## 原文定义

根据《联合国制裁条例》，保安局局长可冻结怀疑是恐怖分子的财产；任何人不得直接或间接向恐怖分子提供财产或金融服务，违者最高可判七年监禁；主动为恐怖分子筹集资金者，最高可判十四年监禁及无上限罚金；扩散资金筹集方面，若提供服务者合理怀疑服务可能与大规模毁灭武器扩散有关，即属犯罪。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| Mr. Whitfield查出"马先生"在联合国制裁名单上有过记录 | 金融机构有义务核查客户是否在制裁名单上 |
| "替他买，替他受益，用他的钱，无论哪种形式都违法" | 禁止以任何形式向恐怖分子提供财产或金融服务 |
| 最高十四年监禁、无上限罚金 | 恐怖分子资金筹集的刑罚极为严厉 |
| 赵先生说"不要通知对方，不要找借口" | 发现疑似恐怖分子关联须立即冻结并报告，不得通风报讯 |
| 梁志明把名单上的姓氏写在玻璃板底下 | 制裁名单须便于职员日常查阅核验 |
| 周老板直觉判断"有些钱不能赚" | 合规文化需要机构自上而下建立风险意识 |
