# 7.4.6c 适用于香港以外进行业务的集团的打击洗钱/恐怖分子资金筹集政策

## 故事

永亨保险行的总公司在一九七九年收到新加坡分行的季度报告时，经理把报告递给梁志明，说："你来写回复。"

"我不懂新加坡的情况。"梁志明说。

"所以要你去学。"经理说，"总公司在香港，但我们在新加坡有合作机构，在曼谷有一家附属公司。如果那边的合规制度不完善，香港这边也要负责任。"

梁志明花了三个晚上，把指引3里关于"集团政策"的那几段英文翻译了出来。

指引3的意思很明确：如果你在香港成立，在外地有分行或附属企业，你就要确保那些地方有和香港一样的合规制度。如果当地法律不允许做同样的合规审查，那就必须做两件事：第一，通知有关当局；第二，采取额外措施把风险压下去。

梁志明后来和林若兰一起起草了一份通知函，把香港这边的合规要求翻译成三种不同的版本：一份英文的，发给新加坡的合作机构；一份泰文的，发给曼谷的附属公司；一份中文的，留档备用。

Mr. Whitfield看了通知函的英文版本，说："措辞可以，但你们要加一条——如果当地有任何情况导致无法按照要求执行，必须在三十天内报告香港总部。"

"为什么要三十天？"梁志明问。

"因为如果你不知道那边出了问题，而那边出了问题，香港这边要背锅。"Mr. Whitfield说，"不是推卸，是整个集团共用一个名字，共用一个声誉。一旦有一个地方出事，所有地方都要承担后果。"

曼谷那边后来回了一封信，说当地对客户身份的了解要求比香港低，但曼谷的合作机构已经同意提高审查标准，新标准从下个季度开始执行。信是泰文写的，附了一份英文摘要。

林若兰把摘要翻译成中文，夹在那封泰文信里，一起锁进了档案柜。

梁志明后来去新加坡出差，在那边见了合作机构的负责人。对方是一个四十多岁的华人，说一口流利的广东话和英文，但当他听说永亨保险行有合规主任、有举报主任、定期更新制裁名单的时候，他沉默了一会儿，然后说："我们这边没有。"

"那你们有什么？"梁志明问。

"我们有老规矩，"对方说，"我认识我的每一个客户，做了三十年，我知道谁的钱干净。"

梁志明没有和他争论。他把指引3的中文版本和英文版本都留给了他，然后写了一封很长的内部报告，详细记录了这次会面。

报告里有一句话："新加坡合作机构的合规基础薄弱，总公司须考虑以合同条款约束对方的合规义务。"

经理看了报告，签了字，附在了下一季度给监管机构的备案文件里。

后来梁志明每次经过新加坡的写字楼，都会想起那个四十多岁的中年人，和他那句"我知道谁的钱干净"。

他不确定那个人有没有道理。他只知道，指引3不是靠"知道"来运行的。它靠的是记录、名单、系统，和一份书面通知函就能找到人的能力。

## 原文定义

指引3规定，在香港成立为法团的金融机构须确保外地分行或附属企业设有集团打击洗钱/恐怖分子资金筹集政策及程序；若当地法律不允许完全合规，须通知有关当局，并采取额外措施减低风险；合规依据须记录在案。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 经理说"新加坡和曼谷的合规不完善，香港也要负责任" | 集团须对所有境外分行及附属企业的合规承担整体责任 |
| 指引3要求"通知有关当局并采取额外措施" | 当地法律限制导致无法完全合规时，须报告并额外减损 |
| 林若兰把合规要求译成英文、泰文、中文三种版本 | 集团政策须通知到所有外地分行及附属企业 |
| Mr. Whitfield说"整个集团共用一个名字，一个声誉" | 集团合规制度须具有一致性，否则一处违规影响全局 |
| 梁志明记录新加坡机构"合规基础薄弱"并写入内部报告 | 发现外地机构合规不足须记录在案，并向总部报告 |
| 新加坡负责人说"我认识我的每一个客户" | 以个人关系代替系统化合规审查，不符合指引3要求 |
| 梁志明建议"以合同条款约束对方的合规义务" | 须以书面协议确保外地合作方承担相应合规责任 |
