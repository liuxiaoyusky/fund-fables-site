# 7.4.6a 引言

## 故事

一九七九年秋天，中环汇丰大楼十五楼，Mr. Whitfield把一份薄薄的文件递给林若兰。

"译成中文，"他说，"这是给你们保险行的，不是给我们律师楼的。"

林若兰接过那叠纸，是英文打出来的，油墨还很新。她看了抬头：《打击洗钱及恐怖分子资金筹集指引（草案）》。下面的括号里写着"指引3"。

当时香港还没有正式立法，但英国律师楼已经在为各大保险机构准备合规文件了。Mr. Whitfield的意思很明确：法律会来的，与其到时候手忙脚乱，不如现在就开始准备。

林若兰把文件带回打字室，一个字一个字敲。敲到第二页的时候，她停了一下，抬头看了看窗外。

窗外是中环的傍晚，叮叮车从毕打街叮过去，铃铛的声音隔着玻璃传进来，很远很轻。

她想起梁志明前两天问她的一个问题：以后客户来买保险，我们要查他是不是坏人吗？

她当时没答上来。现在她知道了。

指引3的目的不是让保险行变成警察局。它是两件事：第一，告诉保险机构，你们有责任认识自己的客户，知道他们的钱从哪里来；第二，告诉保险机构，如果你们发现有人用保险渠道洗钱或者资助恐怖分子，你们有义务报告，而不是假装没看见。

Mr. Whitfield后来在一次内部会上说了一句让她印象很深的话："这条指引不是用来抓人的。是用来守门的。门守住了，整栋楼的人才住得安心。"

他说的是法律，但他望着窗外中环夜景的眼神，让林若兰觉得，他说的不只是法律。

梁志明后来也看了这份文件的译稿。他问："如果不遵守呢？"

"不遵守本身不会被起诉，"林若兰说，"但如果出事了，在法庭上，这份指引会成为证据。法官会觉得，既然你们有能力照着做却没有做，那你们是有责任的。"

"还有呢？"

"还有，"她顿了顿，"董事和控权人的'适当人选'资格可能会受影响。"

梁志明沉默了。那天回家的路上，他经过周记海味铺，看见周老板在收铺，两个人隔着街点了点头。周老板不知道，这份即将到来的新规，和他每一次按时缴保费的习惯，有着隐秘的联系。

一九七九年的香港还没有智能手机，街上的人还用现金买东西。但有些人已经在为未来做准备。梁志明回到永亨保险行的二楼，把那份文件的副本锁进了文件柜的最里层。

三年后，香港通过了相关条例。但在那之前，梁志明已经知道门在哪里了。

## 原文定义

指引3由保险业监管局发出，于一九七九年发布，为获授权保险人、再保险公司、持牌个人保险代理、保险代理机构及保险经纪公司（统称"保险机构"）提供打击洗钱及恐怖分子资金筹集的实务导引；不遵守指引本身不会被起诉，但指引在法庭上可被接纳为证据。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| Mr. Whitfield提前翻译指引文件供保险行参考 | 指引3的发布是为了让保险机构提前准备合规制度 |
| 林若兰问"我们要查客户是不是坏人吗" | 指引3的核心目的之一是认识你的客户 |
| Mr. Whitfield说"这条指引是用来守门的" | 打击洗钱制度是防范系统性风险的屏障 |
| 梁志明问"如果不遵守会怎样" | 不遵守本身不构成起诉，但会在法律程序中成为不利证据 |
| 梁志明把文件副本锁进最里层文件柜 | 保险机构须将合规依据记录在案 |
| 梁志明三年后说"已经知道门在哪里" | 提前准备合规制度，在法律正式实施前已具备应对能力 |
