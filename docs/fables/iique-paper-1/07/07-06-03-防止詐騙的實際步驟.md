# 7.6.3 防止诈骗的实际步骤

## 故事

何师奶的茶餐厅在油麻地，叫「明记」，招牌是她的手写大字，门口常年有两张塑料凳，夏天放一桶冰镇的咸柑橘茶。

联安保险行和她合作了六年。最初是何师奶主动找上门的——她要给伙计买意外险，理由很简单：「我老公走的时候，乜都冇留低。我唔想我啲伙计出事之后，屋企人一分钱都攞唔到。」

梁志明每年续保的时候都会上门坐坐，核对人名和受益人资料。何师奶每次都给他留一碗例汤，话不多，但每句都问在要害上。

那年续保前半个月，何师奶打电话到保险行，声音有点急：「阿明，你上来一趟。」

梁志明第二天下班去到明记。何师奶把一叠文件放在餐桌上，都是理赔申请表，填好了一半。

「我想替大徒弟阿标申请理赔，他上个月喺后巷整雪柜，整条腿俎受了伤。」何师奶指了指其中一栏，「这里要填『出事经过』，我想问你——」

她压低了声音：「阿标受伤那天，有个自称系你哋公司的人打电话来，问咗阿标的情况，然后叫我们先别声张，等他帮我们『搞掂』，理赔偿款可以多一倍。」

梁志明的眉心跳了一下。

「那人叫什么名字？」

「佢话姓李，叫李生。说是保险公司理赔部的。」

「他有没有给您名片？」

「没有，只留了个电话。」

梁志明把那份文件拿过来，翻到最后一页——理赔部的直属电话和地址，与那人提供的完全不一样。

「何师奶，这份表您先停下来。」他把文件轻轻合上，「您接到的电话，很可能是『食价』的。有人想借您这张单，呃保险公司，也呃您。」

何师奶的脸沈下来：「点解会有人知我哋件事？」

「我也不确定。」梁志明说，「但现在最重要的是，您千祈唔好再联络那个人，也不要按他说的『搞掂』。理赔的事，直接经我。」

他现场打了个电话给林若兰。林若兰在律师楼负责来往保险公司的文件，她在电话里说了两句：要核实那份申请表里的签名和日期记号，看看有没有被复印过的痕迹。

那个「李生」再也没有打来。

两周后，阿标的理陪款直接由保险公司结算，金额合理，款项清晰。何师奶在确认书上签字的时候，对梁志明说：「阿明，你们那个行业，我唔识，但我知边个靠得住。」

梁志明走的时候，何师奶往他的自行车篮里塞了一袋自家包的粽子。路上他打开一看——五只，恰好够他和林若兰分。

回到保险行，他把其中三只送到律师楼前台，说：「若兰姐，若兰说话算数。」

前台小姐笑了，没再多问。

## 原文定义

防止诈骗的最重要原则是首先意识到它可能发生。具体步骤包括：保持警觉（留意可疑情况）、勤勉（迅速更新记录）、沟通（与保险人保持紧密联系）、诚信（保持最高道德标准）。保险中介人不论代表被保险人或保险人，均须时刻警觉。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 何师奶察觉「李生」有异常，主动联络梁志明 | 首先意识到诈骗可能发生 |
| 梁志明核实理赔申请表和官方信息不一致 | 保持警觉，识别可疑情况 |
| 梁志明马上联络林若兰核实文件、马上停止与「李生」来往 | 勤勉、迅速行动、不延误 |
| 梁志明绕过「李生」直接与保险公司沟通 | 与保险人保持紧密联系 |
| 何师奶最终通过正规渠道获得理赔 | 诚信对抗诈骗，正规渠道终有保障 |
