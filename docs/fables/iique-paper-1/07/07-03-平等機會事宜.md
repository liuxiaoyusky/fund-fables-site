# 7.3 平等机会事宜

## 故事

何师奶的茶餐厅开在油麻地，招牌是旧式铁皮做的，"何记"两个字被油烟熏得有些发黄，但街坊都认这块老招牌。丈夫走了五年，她一个人撑着铺头，从早忙到晚。伙计都是跟了她好些年的老员工，她不想他们出事时还得自己掏腰包。

那天下午，她把铺头交给阿英看，走路到上环的保险公司填表。接待她的是个年轻伙计，叫梁志明，街坊介绍来的，说他"醒目、靠得住"。

梁志明看何师奶的申请表，看到"婚姻状况"那一栏填了"丧偶"，又看到"供养人数"三个人，便客气地说："何师奶，您这份申请我要呈上去批核，估计要两三天。"

三天后，何师奶再来。梁志明有些为难地把一张条款递给她，说核保部回复是"不接受投保"。何师奶以为是自己的身体出了问题，问是不是要验身，或者加钱。梁志明摇摇头，压低声音说了一句："上面说……单身女士、没先生的那种，拒保是惯例。"

何师奶愣住了。

她站在柜台上，手里攥着那张纸，在原地站了好一会儿。街上的人来来往往，没人注意到她。她忽然觉得有些好笑——她供三个伙计吃饭，五年没欠过一笔账，而保险公司的理由是她没嫁过人。

那天傍晚，她在茶餐厅收铺时，林若兰正好路过。何师奶有时会替林若兰留一份双皮奶，两人相识多年。林若兰见她坐在门口发呆，便问她怎么了。何师奶把事情原原本本说了一遍。林若兰听完，没立刻说话，低头看了看那张拒绝投保的通知。

"这不是精算数据的问题。"林若兰慢慢地说，"他们说这是'风险评估'，但他们连你的店都没来看过，没查过你伙计的工伤记录，没问过你的铺头做了多少年。他们只是看到'丧偶'两个字，就把你放进了另一栏。"

林若兰带何师奶去找了一个人——Mr. Whitfield，中环英资律师楼的合规律师，专门替商号处理这类文件。Mr. Whitfield听完来龙去脉，把那张通知翻来覆去看了一遍，然后从书架上取下一本灰扑扑的《性别歧视条例》。

"保险公司可以说女性的平均寿命比男性长，所以保费不同——这是公平歧视，因为它有精算数据支撑。"Mr. Whitfield说，"但拒绝接受任何丧偶或单身女性的投保，而不去看这个人实际的健康记录、经营记录、风险状况——这不是风险评估，这是用性别和婚姻状况做挡箭牌。法律允许合理的区分，但不允许无根据的偏见。"

何师奶后来把这件事写成投诉信，交到了平等机会委员会。几个月后，保险公司的人重新上门，这次他们真的派人来看了茶餐厅的厨房和走火通道，翻了她伙计的工伤记录，最后按实际风险给了她一份保费合理的保单。

铺头重新签完合同那天，何师奶把那张旧铁皮招牌擦了擦，照样挂在门口。阿英问她干嘛不换个新的。她说："旧的还能用，换什么。"

林若兰后来把这件事讲给梁志明听。梁志明沉默了很久，说："当初那张表……我确实觉得哪里不对，但没想到不对在哪里。"林若兰没接话，只是把他桌上另一份申请表里"婚姻状况"那一栏用铅笔轻轻圈了出来。

## 原文定义

香港《性别歧视条例》等反歧视法例规定，就保险而言，如果对某人的待遇能符合以下两项条件则不违法：(a)有关待遇是参照可合理依据的来源所得的精算数据或其他数据而给予的；及(b)以该等数据及任何其他有关因素而言，有关待遇属于合理。不公平的歧视（例如仅因偏见而拒绝某性别投保，或拒绝为离婚妇女提供家居保险）则属于违法行为。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 何师奶因"丧偶单身"被拒保，未经实际风险审查 | 不公平歧视：以婚姻状况和性别偏见代替精算数据 |
| Mr. Whitfield解释"精算数据支撑的保费差异" | 公平歧视：基于可合理依据的精算数据，且经实际因素核实 |
| 保险公司重审时派人查厨房、翻工伤记录，按实际风险定价 | 合规做法：须参照真实数据并结合其他有关因素评估合理性 |
| 林若兰指出"他们没来铺头看过，没查过记录" | 不公平歧视的本质：不经实际审查，凭类别标签直接下结论 |
