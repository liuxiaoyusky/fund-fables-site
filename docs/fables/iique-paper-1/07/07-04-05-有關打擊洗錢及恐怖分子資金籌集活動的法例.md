# 7.4.5 有关打击洗钱及恐怖分子资金筹集活动的法例

## 故事

那年Mr. Whitfield从律师楼拿出四份不同的法例文本，全部摊在会议室的长桌上，对著梁志明和林若兰说：「你们先不要急著看内容，先把这四份法律的层次弄清楚。」

他从上环的一个案例讲起：一家小型保险代理行的职员，帮一个客户把一笔来路不明的款项分批存入不同帐户，再分别购买保险产品，累积了一段时间之后退保，把钱全部转到海外。这笔资金后来被查出与跨境贩毒有关，职员被捕，保险公司被罚款。

Mr. Whitfield说：「你们看这件事，触犯了起码三套法律。首先，《贩毒（追讨得益）条例》——处理了来自贩毒的款项。其次，《有组织及严重罪行条例》——资金后来被认定涉及有组织的严重犯罪。第三，《打击洗钱条例》——保险公司作为金融机构，没有尽到客户审查的责任。」

林若兰问：「反恐呢？」

Mr. Whitfield说：「反恐不是每一个案件都涉及，但如果资金最后流向了指定的恐怖分子或组织，那就进入了第四套法律——《联合国（反恐怖主义措施）条例》的范围。香港是国际金融中心，资金跨境流动频繁，这四套法律构筑的不是四道独立的墙，而是一个互相咬合的系统，堵住任何一个缺口都不够，要全部一起守住。」

他指了指桌上的文本，说：「《打击洗钱条例》是地基，适用于所有金融机构，订明了客户审查和备存纪录的基本要求。上面三套是支柱：《贩毒条例》针对毒品犯罪收益，《有组织及严重罪行条例》扩展到所有可公诉罪行的收益，反恐条例则是2001年之后新增的，专门针对恐怖分子资金筹集。支柱之间有重叠，但各有各的射程范围，补在一起，才是一个完整的覆盖。」

ICAC的赵先生在培训课上补充了实务层面。他说：「你们在保险行做事，日常接触的可能是贩毒受害者家属的理赔款、可能是某个商人的储蓄保单、可能是船东的航运险——场景各异，但背后的资金流向只有一个方向：正常，还是不正常。你们的责任，就是看清楚这个方向。」

梁志明后来把四份法例的核心条款摘录在一张卡片上，放在工作证背面，随时可以拿出来核对。他跟林若兰说：「四套法律，四个方向，但有一个原则始终不变：知悉或怀疑，就要报告，尽快，按订明的方式。」

林若兰把那句话写在了记事本上。

## 原文定义

在香港，与洗钱及恐怖分子资金筹集有关的四项主要法例为：《打击洗钱及恐怖分子资金筹集（金融机构）条例》、《贩毒（追讨得益）条例》、《有组织及严重罪行条例》及《联合国（反恐怖主义措施）条例》。金融机构及其主管人员和职员均须充分了解他们在不同法例之下的各种责任。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 《打击洗钱条例》作为地基 | 四法例中的基础性规定，适用于所有金融机构 |
| 《贩毒条例》针对毒品犯罪收益 | 特定犯罪收益的追讨与没收 |
| 《有组织及严重罪行条例》涵盖所有可公诉罪行 | 扩展洗钱罪至更广范围的犯罪收益 |
| 《联合国反恐条例》针对恐怖分子资金筹集 | 2001年后新增，呼应联合国安理会决议 |
| 四套法律构成互相咬合的系统 | 金融机构需全面了解不同法例下的多重责任 |
