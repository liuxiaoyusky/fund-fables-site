# 7.5.2 举报贪污

## 故事

那天傍晚，何师奶的茶餐厅快打烊了。

梁志明来喝茶，在角落里坐了一会儿。他手里捏着一张纸，是下午整理保单文件时发现的——周老板的干货铺续保申请表上，有一行数字被改过，改动的地方没有周老板的签字确认。

他清楚地记得自己没有改过这份文件。林若兰也没有。那么是谁改的？

何师奶擦完最后一张桌子，把抹布搭在肩上，走过来给他添茶。她看到梁志明的表情，没问什么，只是在对面坐了下来。

梁志明沉默了很久，才开口："何师奶，我有一件事不知道该不该说。"

何师奶说："你先说，我听。"

他把那张纸的事讲了一遍。他说得很慢，中间停了好几次，好像在每一个停顿里重新考虑一次要不要继续说下去。

何师奶听完，没有急着给意见。她给他又倒了一杯茶，然后说了一个故事。

"我先生以前在码头做工，"她说，"有一天他跟我说，有个人叫他帮忙带一样东西上船，他说不知道是什么，但那人给了三百块。他当时觉得不对，没有带。过了几天，那人被海关抓了，先生说他当时要是带了，三百块换几个月坐牢，不划算。"

她放下茶壶，看着梁志明："他没有带，因为觉得不对。但当时他想举报，不知道找谁。码头的人说这种事不要多管，管了会被人记恨。他就憋在心里，一直到出事前都没跟任何人说。"

梁志明知道何师奶的先生早年出了意外去世了。他没问细节，但他隐约知道那次意外跟码头有些关系。

"所以，"何师奶说，"你觉得不对的事，你要有地方说。憋在心里没有用。"

梁志明说："可是我不知道这算不算'不对'。也许是周老板自己改的，也许是有人不小心，也许根本不是故意的——"

"你问自己一个问题，"何师奶打断他，"如果将来周老板的铺子真的出事，这份申请表被人查出来，你知道有人改过但没有说出来，你会怎么想？"

梁志明想了一想，说："我会后悔。"

何师奶点了点头。

她站起来，从柜台下面翻出一张旧报纸，在社会新闻版指了指一行字："廉政公署，二十四小时举报热线。"

"我存了好几年了，"她说，"一直没机会用，但我觉得应该留着。"

梁志明把那个号码抄在笔记本上。2526 6366。他抄完之后，把那张纸折好，放进了贴身的口袋里。

何师奶把那张旧报纸收回去，说："阿明，你今晚回去想清楚。想清楚了，明天去举报也行，不用急在这一刻。"

梁志明把茶喝完，说："何师奶，有一件事我跟你说实话。"

"你说。"

"我想去举报，不是因为我想做英雄，是因为我不想有一天被问起来的时候，我说不清楚。"

何师奶笑了笑："这就是实话。实话就够了。"

她把门板一块一块装上的时候，梁志明还坐在角落里。何师奶说："你再坐一会儿，那杯凉茶还没凉透。"

梁志明把那张纸从口袋里拿出来，又看了一遍，然后在背面写了四个字：如实举报。

他走的时候，回头说了一声谢谢。何师奶没有回答，只是在门缝里向他挥了挥手。

后来他去了廉政公署。不是为了做英雄，是为了有一天续保的时候，能抬起头说一句：我没有做错。

## 原文定义

保险中介人必须保持高度诚信，拒绝贪污诱惑，提高警觉以避免触犯《防止贿赂条例》或其他罪行。如遇贪污事件，应立刻向廉政公署举报。廉政公署二十四小时举报热线：2526 6366，所有举报内容绝对保密。
