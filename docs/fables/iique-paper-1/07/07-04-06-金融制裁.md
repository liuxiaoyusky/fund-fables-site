# 7.4.6g 金融制裁

## 故事

一九七九年六月，台风季快到了。

上环一间老牌保险行的办公室里，梁志明正埋头核对着周老板的货运保单。周老板在隔壁开着海味铺，这次要运一货柜干鲍鱼去台湾，收货方催得紧，说台风季一到，货在海上多等一天就多一分风险。

账房陈伯把一张银行汇款单递过来，说周老板那边要先付一笔定金给台湾的船公司，款要从保险行这边过一下账。梁志明看了看金额，又看了看汇款单上的受益人名称——台湾某船务公司，在维港跑船的谁都知道这个名字。

他把汇款单拿给林若兰看。林若兰在英资洋行做打字员已经两年了，文件一看就仔细，这次正好在保险行帮Mr. Thompson整理旧档案。

"这个公司名，"林若兰指着汇款单上的受益人，"我在政府宪报上见过类似的。"

梁志明心里一沉。政府宪报他平时很少翻，但林若兰既然提了，他就不能当没听见。他去柜子里翻出三周前送来的一大叠政府通告，其中一份封面印着《联合国制裁条例》的标题。

他找到名单，一行一行地核对。林若兰凑过来，两人把汇款单和名单并排放着，一字一字地比。周老板在门外等着，电话响了好几次，是台湾那边的经纪在催。

"是这个公司。"林若兰的声音很轻，但很确定。

梁志明站起来，走到Mr. Thompson的办公室。Thompson先生是英资洋行的经理，这间保险行在同一个屋檐下办公，两家共用一堵墙。梁志明把汇款单和宪报放在他桌上，什么都没说。

Thompson看了很久，然后把宪报合上。"先别动这笔账。"他说，"我要打一个电话给伦敦。"

周老板听说付款要暂停，急得额头都出汗了。那批鲍鱼就在葵涌码头，台风再来，货柜泡了汤他就什么都没有了。梁志明听着周老板在电话那头发火，心里也不好受。

但Thompson先生那天傍晚给出了一个理由："汇款单上的那笔钱，不是周老板的钱，也不是我们的钱。那是一家名列联合国制裁名单的公司的钱。在香港，没有行政长官特批，谁都不能动。"

"可是他们只是名字像而已，"周老板说，"他们根本不是同一家公司。"

"名字像不算，"Thompson先生说，"政府宪报上的名单不会写错。"

两个月后，真相大白。那家台湾船务公司确实和制裁名单上的实体有关系，已经通过另一间银行收到了类似的款项，结果那间银行被伦敦的总行调查，整条业务线都被冻结了。周老板的货最终通过一间持牌汇款公司收到了款——过程慢了三周，但货物安全到了台湾，台风也没有来。

梁志明后来把Thompson先生桌上那张宪报借回去看了一个晚上。他没有全看懂，但记住了其中一句：禁止直接或间接为名列指定名单的人士提供任何资金或经济资源。他把那页折了个角，放进了自己的笔记本里。

## 原文定义

《联合国制裁条例》授权行政长官订立规例，执行联合国安全理事会的制裁决定。制裁通常禁止直接或间接为指定人士提供任何资金或经济资源，或处理属于该指定人士的任何资金或经济资源。违反者可被检控。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 汇款单上的受益人名称与政府宪报名单上的名字一致 | 金融制裁指定名单 |
| Thompson先生要求暂停付款，要求伦敦确认 | 执行制裁前需确认指定身份 |
| 未经特批不得向名单内人士付款 | 行政长官特批机制 |
| 另一间银行被调查整条业务线被冻结 | 违规处理制裁资金的严重后果 |
