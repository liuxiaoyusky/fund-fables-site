# 5.1.4 再保险(Reinsurance)

## 故事

冯船长在维港跑了二十年船，什么风浪都见过。二十年来他只信三样东西：自己的舵、老妈祖，和一份写得清清楚楚的保单。

那年春天，他刚从广州黄埔船厂接了一艘新铁壳船回来，跑内河货运。从广州装货，落香港仔码头卸货，航程不远，但维港风浪说变就变。冯船长不赌运气——他在上环一间老牌保险行给自己和船都投了保，代理人是认识多年的梁志明。

梁志明帮他安排好了保障：船体、机器、货物，都在单子里。白纸黑字写得清楚。冯船长看过一遍，点了点头，把保费交给梁志明，转身又要回船上。

"冯船，文件里还有一样东西，我帮你加进去了。"梁志明说，"出事的话，不只我们赔你，我们公司也会有人一起扛。"

"一起扛？"冯船长没听懂。

梁志明解释说，保险公司接了太多同类风险，会把其中一部分再分出去，让别的保险人接手。这样万一真的出事，大家一起赔，单一一家公司的压力就没那么大。

冯船长没再问。在他看来，保单就是一张纸，写了什么就是什么。"一起扛"三个字，他没有放在心上。

那年六月，一批从广州运往香港的干货在维港外海遇上了突发雷雨。风来得急，浪涌得高，货舱进了水，三十多箱货物泡了汤。冯船长第一时间给梁志明打了电话。

梁志明赶到码头时，雨已经停了，但甲板上还残留着海水痕迹。三十多箱干货全部需要定损。梁志明逐一拍照记录，填了报案表，递给公司核赔部门。

几天后，冯船长拿到了赔款。他以为这件事就此了结，直到梁志明在茶餐厅请他喝茶时才告诉他另一件事。

"冯船，这次我们赔你赔得很痛快，但其实我们自己也捏了一把汗。那批货的数量大，如果再多淹几箱，我们公司那季的账就不好看了。"

"你们不是说过大家一起扛吗？"冯船长这才想起那句话。

梁志明点头。那天他告诉冯船长，保险公司接了太多同类风险之后，会主动把其中一部分再分出去——找别的保险人接手，叫"分出再保险"。反过来，如果一间保险公司专门做这种接风险的生意，帮别人的保险公司分担损失，那就叫"专业再保险人"。他们不做直接面对船东和商人的生意，只做保险公司之间的生意。

"所以那天你说'有人一起扛'，不是客套话，是真的有人。"冯船长终于明白了。

梁志明说是。"所以冯船你买的那张保单，背后不只是一间公司在撑着，是一个圈子在兜着。我们赔你，我们的上家也会分一部分。这样大家都能扛得住。"

冯船长沉默了一会儿。他想起自己二十年来只信三样东西：舵、老妈祖，和那张写得清清楚楚的保单。现在他知道了，第四样东西叫做"有人替你分担"——而分担这件事本身，也需要有人来承接。

那天下午，冯船长把他那张旧保单重新翻了出来。他叫梁志明帮他把每一条都再解释一遍。这一次，他没有急着走。

## 原文定义

再保险人向保险人提供保险保障。再保险分为：（a）分出再保险：保险人向其他保险人购买保险；或（b）分入再保险：保险人担任再保险人角色，承保已被其他保险人承保的风险。专业再保险人指只经营再保险业务的保险人。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 冯船长向梁志明的保险公司投保 | 保险人直接承保风险 |
| 保险公司将部分风险转给其他保险人 | 分出再保险（Outwards reinsurance） |
| 专业承接其他保险公司风险的机构 | 专业再保险人（Professional Reinsurers） |
| 冯船长最终拿到赔款 | 保险公司与再保险人共同承担赔付 |
