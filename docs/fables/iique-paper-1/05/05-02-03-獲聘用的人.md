# 5.2.3 获聘用的人

## 故事

这是梁志明入行的第三年。

他在上环一间本地保险行做伙计，每月固定跟师父落区见客。这天黄昏，他在观塘一间制衣厂办公室外面等黄师傅。工厂放了工，机器停了，只有风扇转动的声音。

黄师傅穿着油迹斑斑的工作服走出来，见到梁志明，笑着递过一支平安硬包：「阿明，你又来捞世界？」

「黄师傅，今日老板叫我过来，问吓有冇需要加多份意外保障。」

黄师傅把梁志明带到工厂的空地，几个师傅正蹲在角落食烟。黄师傅指着那班人说：「你睇，我手下七八个师傅，加上判头和后勤，整间厂少说都有二十几三十人。每一个人的背后，仲有一个家庭。」

梁志明点点头。他跟着师父跑了两年，见过太多打工仔出事之后一家陷入经济困难的例子。他说：「所以我哋成日话，保障唔系为你自己，系为咗屋企人。」

黄师傅说：「道理我懂，但我仲想知一件事——你哋成日话保险业几大几大，究竟有几多人真系做呢行？」

梁志明说：「我之前睇过一份统计，香港保险业请了十万人。」

「十万？」黄师傅吃惊，「成个观塘的工厂都冇咁多人。」

「十万人入面，大约七成六系做人寿保障，其余二成四做一般保障。

黄师傅沉默了一阵，然后说：「阿明，十万人唔系小数目。每一个数字后面，都系一个人，都有屋企人要养。」

这时候，周老板恰好路过，周老板在上环开海味舖，认识黄师傅多年，见面就问候：「黄师傅，近排点呀？」

黄师傅说：「周老板，我在了解保险业请了多少人。你知唔知，原来有成十万人食呢行饭？」

周老板想了想：「我间舖请了两个伙计，整个海味行有成百间舖，每间请两三个人，就已经系几百人。再加上你哋保险、运输、金融……」

他停了下来，说：「香港就系咁，每样嘢都关埋一齐。一行唔得，其他行都会受影响。」

梁志明点头。他抬头望着观塘工厂大厦的窗户，每一扇窗后面都是一个生计。夜色慢慢降临，工厂大厦的灯一间间亮起来，和维港对岸中环的办公大楼隔着海互相辉映。

黄师傅最后说：「好，我哋厂帮伙计每人买一份意外保障。你帮我计划吓。」

## 原文定义

截至2021年1月，香港保险业从业人员约102,288人。其中约76%主要从事人寿保险业务（人寿保险从业员中约84%为保险代理人或保险代理人的业务代表），其余约24%主要从事一般保险业务。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 黄师傅工厂的师傅人数 | 保险业直接创造大量就业岗位 |
| 「七成六做人寿，二成四做一般」 | 人寿与一般保险从业员比例约76%比24% |
| 梁志明介绍统计数字 | 人寿保险从业员中，代理及代理业务代表占约84% |
| 周老板谈及海味行与保险业的关系 | 保险业是香港经济体系的组成部分 |
