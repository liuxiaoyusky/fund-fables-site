# 5.1.1 保险的法定类别(Statutory Classification)

## 故事

上环海味铺周老板在整理一批新到的鲍鱼干货，忽然想起一件事——隔壁老陈去年买的那份保障，到底是保人的，还是保货的？他搞不清楚，只记得老陈说"什么都包"。周老板心里不踏实，坐在柜台上点了根烟，让伙计去把梁志明叫来。

梁志明是本地保险行的年轻伙计，踩着人字拖从楼梯间小跑上来。周老板招手让他坐下，把事情说了一遍。梁志明想了想，说："老陈那份我大概知道，但不是什么都包的。保险有分门别类，老陈买的是长期险，偏向保人的那一路。"

周老板皱眉："那我这个干货要是海上出了事，找谁赔？"

梁志明还没开口，门外停下一辆黑色轿车，Mr. Thompson从里面出来。Thompson是英资保险公司的部门主管，这天正好约了在梁志明这里碰面，讨论一份分保的文件。周老板认识他——去年他帮海味商会打听行价时见过。Thompson进门坐下，林若兰跟着进来，她是Thompson的中英打字员，手里抱着一沓英文文件。

Thompson听完周老板的问题，喝了一口茶，用带着口音的广东话说："这个问题问得好。许多人都以为保险是一种东西，其实不是。在香港，保险受《保险业条例》规管，条例把保险分成两大类：长期业务和一般业务。长期业务主要保人，一般业务主要保财产和责任。"

周老板说："那我要是想把干货和人都一起保，是不是要签两份？"

Thompson笑了笑："理论上是的。当然有些公司同时拿到两类业务的授权，可以一张单里包含不同内容。但授权是分开的，哪家公司能做哪类业务，要看它在政府那里拿到的牌照。"

林若兰翻开文件，指给梁志明看："你看，这里写明长期业务分了九类，每类用一个英文字母标记；一般业务分了十七类，每类用一个数字标记。授权的时候，政府是按这个分类来批的。"

梁志明凑过去看，纸张上是密密麻麻的字母和数字。他对周老板说："所以老陈买的那份，是长期业务里的一种，字母是L开头那种，保人的。我帮他看过那份建议书，里面有写。"

周老板点点头："那我这份干货呢？"

"一般业务，数字17那边，"梁志明说，"专门保货运的。"

Thompson把茶杯放下，对周老板说："你找哪家买都行，但首先要看那家公司有没有拿到针对这个业务的授权。没有授权就做是违法的。你可以去政府那里查，每一家获授权的保险公司都有一个名单。"

周老板站起来："明白了。多谢梁仔，也多谢Thompson先生。"

他走到门口又回头："对了，你们说的那个字母和数字的分类，普通人也要记住吗？"

Thompson摆摆手："不需要记住每一个类别。但你至少要知道两件事：你是要保人还是保财产责任，这是最大的分界；然后查清楚帮你做这单业务的公司，它有没有拿到对应业务的授权。"

周老板点点头，掀开门帘回了海味铺。Thompson转向梁志明说："你这位客人挺实在的。"梁志明说："上环做海味的都这样，问清楚才肯签字。"林若兰低头整理文件，嘴角微微动了一下，没说话。

## 原文定义

《保险业条例》把保险分为长期业务及一般业务两类作为主要分类，再分别进行次级分类：长期业务分为九类，以英文字母标记；一般业务分为十七类，以数字标记。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板的海味干货 vs 老陈的保障合同 | 长期业务（保人）vs 一般业务（保财产/责任） |
| Thompson出示的字母/数字分类文件 | 法定分类的具体标记方式（字母/数字） |
| Thompson说要看公司有没有"授权" | 保险公司须获授权才能经营对应业务 |
| 林若兰指出对应标记位置 | 法定分类用于监管授权范围 |
