# 5.5.3 协助索偿人或受害人的行业机构

## 故事

那年台风季节特别长。梁志明坐在保险行的办公室里，看着窗外被风吹歪的招牌，心里盘算着接下来几个月会有多少单理赔要处理。果然，麻烦的事情接踵而来，而且每单都牵扯到不同的机构。

第一单是周老板的海味仓库进了水，两百多箱干鲍鱼发霉了。周老板拿着保单去找保险公司，对方说他没有附加"淡水渍险"，只赔三成。周老板不服，又不知道怎么投诉。梁志明告诉他，可以找保险投诉局，让第三方机构来评判这件事该不该全赔。

第二单是冯船长朋友的事故——一辆货车在九龙撞伤了一名过马路的老人，货车司机是黑市司机，没有车保。老人家属一筹莫展。梁志明说，香港汽车保险局会管这件事。他们虽然不能让那个黑市司机赔钱，但可以从汽车保费的附加费里拿出一笔钱，补偿那个无辜受伤的老人。

第三单最棘手。黄师傅的油漆分包商的保险公司突然宣布清盘，那间公司欠着二十多个工人的雇主险赔款，分包商自己也赔不出来。梁志明叹了口气："这种情况，就轮到雇员补偿险无力偿债管理局出场了。这间机构由所有经营雇员补偿险的公司共同出资组成，专门处理这类事情——保险公司倒了，他们来接手，继续赔钱。"

周老板在旁边听完，沉默了很久，最后说："所以这张网，是一层叠一层的。我以为买了一份保险就够了，没想到后面还有这么多东西在兜底。"

"兜底的东西不用的时候看不见，"梁志明说，"但看不见，不代表不存在。"

## 原文定义

香港有三个协助索偿人或受害人的行业机构：保险投诉局（ICB）处理消费者与保险公司之间的纠纷；香港汽车保险局（MIB）在司机无保险或保险公司清盘时向交通意外受害人提供补偿；雇员补偿险无力偿债管理局（ECIIB）在会员保险公司无力偿债时承担其雇员补偿保单的法律责任，资金来源于保费附加费。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板的淡水渍险争议 | 保险投诉局（ICB）处理消费者与保险公司之间的纠纷 |
| 老人被黑市司机撞伤 | 香港汽车保险局（MIB）向无辜受害人提供补偿 |
| 清盘保险公司欠工人赔款 | 雇员补偿险无力偿债管理局（ECIIB）接手倒闭公司的赔付责任 |
| "一层叠一层"的兜底网络 | 三类机构分别覆盖不同场景下的消费者保护 |
