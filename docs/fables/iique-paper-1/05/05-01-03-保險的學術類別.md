# 5.1.3 保险的学术类别(Academic Classification of Insurance)

## 故事

何师奶的茶餐厅开了十二年，从来没有买过一张正式的保单。那天她的伙计在后巷搬货，不小心砸了一个木箱，货主找上门来要赔钱，她才第一次认真坐在梁志明对面。

"我不想听那些复杂的东西，"何师奶说，"你就告诉我，我开这间餐厅，到底要防备什么。"

梁志明想了想，把一张纸分成四格，在每一格里写了一个字。

第一格，他写"人"。何师奶的伙计在店里做事，出了意外谁管？自己和家人万一生病或者出事，家里怎么办？这一栏，是保人的。

第二格，他写"楼"。店里烧坏了怎么办，台风把招牌吹掉了，冰箱里的货全坏了怎么办？这栏是保东西的。

第三格，他写"责任"。伙计摔伤了，客人吃坏了肚子，装修工人把水管弄破了影响到隔壁——这些东西伤的不是你的身体，也不是你的货物，而是你要对别人负责的那部分风险。这栏是保责任的。

第四格，他写"生意"。何师奶听到这一格，眉毛抬了抬。"做生意还会亏？"

"不是亏，是停。"梁志明解释，"你的店要是被火烧了，铺面要重新装修，装修期间不能做生意，少了多少钱？有些保险可以赔这部分损失，叫营业中断保险。还有一种是你借钱做生意，借来的钱还不上，有一种保险专门保信贷风险。这些都不属于前面三格，所以另算一类。"

何师奶把那页纸看了很久。她忽然说："我以为买保险就是买一样东西，原来是在这四格里面都要想一遍。"

陈伯那天刚好来餐厅收账，看见了那张纸，愣了一下，说："你们在做分类啊？"

"不是我们发明的，"梁志明说，"这是学术界和考试用的分法，但是很有用。你看，你给客人解释保障的时候，先问他要保什么——保人、保东西、保责任、还是保生意上的钱——分清楚这四样，再去找对应的产品，就不会乱。"

何师奶把那页纸收起来，说："我现在知道我要买什么了。我要保我的人——伙计摔伤我要赔钱；保我的楼——起火了我要有钱重建；还要保我的责任——客人出事我脱不了身。至于最后一格，我暂时还不需要。"

梁志明送她出门，回头对陈伯说："这张纸，比很多产品手册都管用。"

陈伯点点头："因为它不是告诉你买什么产品，它是告诉你担心什么。这个思路变了，整个谈话就不一样了。"

## 原文定义

保险按标的或功能可分为四类：人身保险（以人为标的，包括人寿、健康及人身意外）；财产保险（承保实物损失和损害）；责任保险（承保对他人的法律责任）；经济权益保险（承保财务权益，如营业中断、信贷及租金保险）。从功能角度分类相当于手握一份有用的清单，帮助决定购买需求。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 何师奶担心伙计摔伤、家人出事 | 人身保险：保人的生命和健康 |
| 茶餐厅起火、冰箱货坏、招牌吹落 | 财产保险：保实物损失和损害 |
| 客人出事、工人弄破水管要赔钱 | 责任保险：保对他人的法律责任 |
| 店铺被烧、装修期间不能营业的损失 | 经济权益保险：营业中断、信贷、租金 |
| 把生意分成四格逐一核对需求 | 学术分类是帮助客户决定购买需求的专业工具 |
