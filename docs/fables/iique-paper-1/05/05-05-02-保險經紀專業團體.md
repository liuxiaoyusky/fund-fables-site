# 5.5.2 保险经纪人专业团体

## 故事

冯船长的"海利号"要在维港多停几天——不是因为货没卸完，是因为他找的那间经纪行，在等他那份新的综合货运险批下来。

"以前要等他们帮我递文件、等保险公司核完、再等他们把批文拿回来，"冯船长在码头边抽烟，跟梁志明抱怨，"整件事搞了快一个月。"

"因为你的货运量大，风险高，要看很多间公司才找到愿意接的那间，"梁志明说，"经纪行的好处就是他们认识多间公司，知道谁肯接高风险的船，谁有再保渠道可以分担。但2019年之后，这些都要经监管局直接批了，经纪行也不能替你包揽所有。"

林若兰正好路过，停下来插了一句："其实那两个经纪人团体——香港保险顾问联会和香港专业保险经纪人协会——以前也是自己管自己的会员，投诉处理、纪律守则什么的，都要靠他们。但现在监管局统一管了，他们就把这些权力交了出去。"

"那他们现在干什么？"冯船长问。

"还是代表你们这些经纪人，"林若兰说，"和监管局开会，讨论新政策对不对，给会员安排培训课——说到底，还是替你们说话、帮你们进步。"

冯船长把烟头掐灭："也就是说，我还是要找经纪行，但找的那间得在协会有名有姓，我才信得过。"

"对，"梁志明说，"协会有名有姓，代表他背后有人在看着。"

## 原文定义

香港有两个保险经纪人专业团体：香港保险顾问联会（HKCIB）和香港专业保险经纪人协会（PIBA）。2019年新监管制度生效后，两会不再执行自律监管职能，但继续在政策倡导和会员培训方面发挥重要作用。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 经纪行帮冯船长找愿意接高风险船的公司 | 经纪人具有宽阔的产品视野和独立专业性 |
| 2019年后经纪人也需监管局直接批牌 | 新的统一发牌制度取代了行业自律监管 |
| 协会从纪律机构转型为培训和倡导机构 | 协会功能从监管转为行业代表与专业提升 |
| "协会有名有姓，代表有人看着" | 协会会员身份是经纪人专业可信度的参考 |
