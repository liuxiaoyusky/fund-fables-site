# 5.5.4 雇员补偿联保计划管理局

## 故事

黄师傅从保险行走出来，脸色很难看。他在观塘做了十几年判头，手下二十多个工人，专门接制衣厂和货仓的维修工程。上个月他收到通知，说负责他工人保险的那间公司因为调整业务，不再接高风险工程判头的单了。

他跑了三间保险公司，每一间都说"风险太高"，要么不接，要么保费高得离谱。

梁志明在中环的英资保险公司门口碰到他，黄师傅正在抽烟解闷。"志明，我问你一件事，"他说，"如果所有保险公司都不肯接我的单，我是不是只能让工人裸着干活？"

梁志明想了想，说："不是。所有经营雇员补偿险的保险公司，有一个大家共同组成的'联保计划'。你是高风险工种，商业市场上没人愿意接单，你就可以走联保计划这条路——所有公司绑在一起，集体承担你这个风险。"

黄师傅把烟掐灭："所以他们不是不接，是个别接不了，大家绑在一起才接得了？"

梁志明点头："联保计划管理局就是看着这个计划怎么运作的。它不是一间公司，是一个行业组织，确保这个最后的渠道一直开着。"

黄师傅第二天就去了联保计划管理局的办公室，交了申请。保费确实比商业市场高一些，但至少不是没有人接。回来的路上，他想：原来保险这行也有"大家绑在一起"的时候，高风险的人，才最需要这种集体兜底的安排。

## 原文定义

雇员补偿联保计划（ECIRS）由所有在香港经营雇员补偿保险业务的保险公司共同组成，为从事高风险工种的雇主提供最终的投保渠道。该计划由雇员补偿联保计划管理局监察运作。
