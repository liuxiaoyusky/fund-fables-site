# 5.2.1 获授权保险人

## 故事

周老板的上环海味舖开了十二年，近年生意稳定，手头有了一点积蓄。他有个心事：再过几年自己就六十了，海味舖的事早晚要交出去，但退休金的事一直没想清楚。另外，舖子里的存货和装修也要找一份保障，万一失火或被偷，总要有个着落。

两件事混在一起，周老板不知道该找谁。

他先去了常买茶叶的那间茶餐厅，问茶餐厅老板阿强。阿强说自己买的是一般业务保险，赔火灾和盗窃的，但退休金的事要去问"那些人寿公司"。周老板又问了几个人，有的说"综合也行"，有的说"要找纯长期业务的"——周老板越听越乱。

后来阿强说："你不如去上环那间洋行找Thompson，他见多识广，什么都说得清楚。"

周老板那天上午去了Thompson的办公室。Thompson正在翻一份名单，见他进来，放下手里的文件，倒了杯茶。周老板把两件事一口气说了：舖子的保障，以及自己的退休金。

Thompson听完后，慢慢地说："周老板，这是两件事，而且可能是两间公司的事。"

周老板有些失望："不能一次过办吗？"

Thompson说，香港所有合法经营保险业务的公司，都必须拿到监管机构的授权。授权是分类的：一间公司如果只做长期承诺类的业务，比如人寿、退休金，常被称为做"纯长期业务"；如果只做短期保障类的业务，比如火险、水险、意外险，就是做"纯一般业务"；还有一种两类都做的，叫"综合业务"。三个类别各有各的要求，各有各的规矩。

Thompson说，香港目前有五十多间纯长期业务保险人拿到授权，三十多间纯一般业务的，综合的有将近二十间。

"所以，你是说我可能要找两间？"周老板问。

"不一定。但如果你想找一间什么都包的，那得确认它是综合业务的授权公司才行。"Thompson说完，从抽屉里拿出那份名单，指给他看。

周老板仔细看了一会儿："那些外地的公司呢？也能在香港卖保险？"

"可以，但要拿香港的授权。"Thompson说，"有些在伦敦，有些在百慕大，有些在新加坡，香港本地注册的有，本地的也有。只要拿到授权，就能做这三种业务里的任何一种。"

梁志明端茶进来，听到Thompson这句话，默默记下了这个数字对比。

周老板又问："那我怎么知道找的是不是合法授权的？"

Thompson说："你问有没有授权，合法公司会告诉你。不敢回答的，或者解释不清楚的，你就要小心。"

周老板点了点头。他终于把两件事分清楚了：舖子的事，找纯一般业务的授权保险人；退休金的事，找纯长期业务的授权保险人。两把锁，两把钥匙。他记下了Thompson给他的那几间公司名字，打算一家一家去问。

那天周老板离开时，梁志明送他到电梯口。周老板说："今天学了一课，原来保险不只是一张纸，背后还有一张'授权'的门槛。"

梁志明说是。周老板想了想，又补了一句："门槛高一点好，低了我还不放心呢。"

## 原文定义

获授权保险人按业务范围分三类：「纯」长期业务保险人（仅从事人寿、退休金等长期承诺）、「纯」一般业务保险人（仅从事火险、水险、意外险等短期保障）、综合保险人（同时从事长期及一般业务）。所有在香港经营保险业务的公司，均须取得相应类别的授权。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板分两次问：舖子保障 vs 退休金 | 长期业务 vs 一般业务的区分 |
| Thompson解释三类授权公司 | 纯长期、纯一般、综合三类授权 |
| 部分授权公司来自外地 | 授权保险公司可本地注册或外地注册 |
| 周老板问"是不是合法授权的" | 授权是合法经营保险业务的前提门槛 |
| Thompson拿出那份名单 | 获授权保险人有登记在册的记录 |
