# 5.2.4 保费收入

## 故事

周老板的上环海味舖已经做了三代。

这天午市过后，梁志明上门拜访。周老板坐在柜台后面，旁边放着算盘和一叠来往单据。他抬头见到梁志明，放下茶杯：「阿明，你又嚟了。」

「周老板，上次你话想了解吓究竟保费系点计，今日我带了资料过来。」

周老板站起来，带梁志明走进货仓。货仓里堆满了干鲍鱼、瑶柱、花胶，冬天的订单已经堆到天花板。周老板指着那批货：「这批货运去英国和欧洲，保费要几钱？」

梁志明打开文件：「周老板，我首先想问你一件事——你有冇谂过，保费究竟系乜嘢？」

「保费？」周老板不以为然，「咪就系每月要交嘅钱啰。」

「表面睇系咁。但保费的真正意思，系你将风险转移俾保险公司的代价。你话我知，这批货运去英国，海路风险大唔大？」

周老板想了想：「大。风浪、打劫、海关手续出问题，都有可能。」

「那如果这批货出了事，损失几十万，你一间舖能否自己孭飞？」

周老板摇头：「孭唔起。」

「所以，你付出保费，将这个风险转移俾保险公司。出事之后，佢哋负责赔偿。这就系保费的本质。」

周老板点起一支烟，慢慢消化这番话。他说：「那保险公司收了保费，钱去了哪里？」

「三个地方。」梁志明竖起三根手指，「第一，存入专门的储备金，用于支付日后的理赔；第二，用于公司的日常运作，包括员工薪酬、办公室租金；第三，如有盈利，会再投入业务。银行的监管要求他们必须把理赔的钱分开存放，唔可以乱使。」

「那我点解信佢哋？」

「因为保险公司唔靠拒绝理赔嚟赚钱。如果佢哋成日唔赔，人们就唔再投保，公司最终反而倒闭。佢哋的商业模式系靠精算师计算风险概率，向所有人收取保费，再整体用嚟赔偿少数人的损失。」

周老板把香烟揿灭：「我明白了。保费唔系一笔使费，系一份承诺的代价。」

「正是。」梁志明合上文件，「周老板，你决定点？」

周老板走到门口，望着上环的街巷。隔壁的鱼翅店正在收舖，对面的参茸行亮起了灯。他转身回来：「我决定投保。保费几多系几多，但出了事，我唔使一个人孭飞。」

梁志明站起来，准备告辞。周老板叫住佢：「阿明，今日我学识了一样嘢——保费唔系支出，系保障。你帮我record低，今天我买的不只是一张纸，系一份安心。」

梁志明离开时，天色已暗。上环的街灯亮了，唐楼的窗台传来饭香。他行过一间又一间商舖，心里想：这座城市，每一天都有人在计算风险，每一天都有人在为未来作准备。

## 原文定义

保费是风险转移的代价。保险公司收取保费后，须承担被保险人在保险合约范围内的赔偿责任的承诺。一般保险业务的毛保费总额反映了市场整体规模，长期业务的保费总额则体现保险业对香港经济的重要性。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 周老板的货运风险 | 保费是转移风险应付的代价 |
| 「出事之后，佢哋负责赔偿」 | 保险公司收取保费后承担赔偿责任 |
| 三个资金去向：储备金、运营、理赔 | 保险公司收取的保费用于赔付及运营 |
| 周老板的决定 | 保费换取的不是一张纸，是出事时的经济保障 |
