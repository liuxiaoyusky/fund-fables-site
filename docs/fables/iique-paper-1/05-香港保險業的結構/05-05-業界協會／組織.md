# 5.5 业界协会／组织

## 故事

那年年底，上环海味铺难得清闲，周老板难得休息一个下午。他泡了一壶铁观音，把梁志明和冯船长叫来聊天，话题不知怎么就转到了香港保险业这张看不见的网上。

"我卖了这么多年海味，"周老板说，"一直以为保险就是签个合同、交钱、等出事拿赔款。但这一年下来，我发现这张网比我想象的大得多。"

冯船长接话："何止是大。我跑船的，以前只知道货保和船保。这一年志明带我看了一圈，我才发现——我出事有人赔，老人被车撞有人赔，工人受伤也有人赔。这三件事，分属三个不同的机构在管。"

梁志明笑了笑："你们说的'三个机构'，背后其实是一整张网。保联代表所有正经的保险公司，他们定了规矩、设了标准；两个经纪人协会替投保人找合适的公司，确保有人帮你们挑选；三个消费者保护机构——投诉局、汽车保险局、无力偿债管理局——专门替出事但没人管的那一方说话。"

周老板放下茶杯："所以这张网有三个角：卖保险的、帮人选保险的、替出事的人兜底的。三个角撑在一起，才是一个完整的行业。"

"周老板说得对，"林若兰刚好送文件路过，听见了最后这句话，"Mr. Thompson也常说，没有规矩的行业是一盘散沙。但有了这些协会和组织，香港保险业才有了骨架。"

冯船长站起来，把茶杯里的茶喝完："骨架够不够硬，就看这三个角牢不牢了。"

## 原文定义

香港保险市场的主要行业组织包括：代表保险公司的香港保险业联会（HKFI）、代表保险经纪人的香港保险顾问联会（HKCIB）及香港专业保险经纪人协会（PIBA），以及协助消费者的保险投诉局（ICB）、香港汽车保险局（MIB）和雇员补偿险无力偿债管理局（ECIIB）。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 冯船长发现三类赔付分属不同机构 | 三类消费者保护机构分别覆盖不同场景 |
| 周老板总结"三个角撑在一起" | 行业组织生态由保险公司、经纪人、消费者保护三方构成 |
| 保联代表保险公司 | HKFI是获承认的保险公司行业代表 |
| 两个经纪人协会 | HKCIB和PIBA代表保险经纪人 |
| "骨架够不够硬" | 行业组织的健全程度影响整体保障水平 |
