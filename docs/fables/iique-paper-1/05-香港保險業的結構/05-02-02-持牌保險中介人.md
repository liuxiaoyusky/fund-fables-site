# 5.2.2 持牌保险中介人

## 故事

湾仔茶楼，早市最旺的时段。梁志明上个月升了主任，同事送了一条红纸，他请了黄师傅饮茶。黄师傅在观塘制衣厂做维修判头，手下有七八个师傅，个个身手好，但做起保险琐事就皱眉头。

黄师傅把叉烧夹进白饭，筷子一放，说：「阿明，你们保险业我真系搞唔清楚。上次我想问份单，问咗三间公司，个个口径唔同。一时话佢哋系『代理』，一时又话系『经纪』，究竟有咩分别？」

梁志明把糯米鸡分开，慢慢说：「黄师傅，你听我讲吓。香港保险业分两种人，一种系『代理人』，另一种系『经纪』。代理人是帮一间保险公司卖单的，这间公司叫他做边佢就做边，所以叫『代理』。经纪就唔同，佢可以同时帮几间公司，你想要咩，佢帮你揾边间最啱。」

「那岂不是经纪更公平？」

「睇你点睇啰。代理只帮一间公司，对那间公司的产品好熟，适合你乜嘢佢心里有数。经纪视角阔，但系中间人，份粮系保险公司出，严格嚟讲都系受监管的。」

黄师傅似懂非懂。梁志明接着说：「最重要的一点，无论系代理定系经纪，都要领牌。香港唔系边个都可以卖保险，个个人都要过监管机构那关，先可以正式执业。」

「原来系咁，」黄师傅拿起茶杯，「我以前以为保险就系一间大公司，请几个人卖单。原来入面仲分得咁细。」

这时候，茶楼门口停了一辆黑色轿车，一个穿着全套西装的外籍人走入来，操着一口流利英语和普通话夹杂的口音。梁志明站起来打招呼：「Mr. Thompson，早晨。」

Mr. Thompson系英资保险公司的代表，公司业务主要系航运和货物运输的保障，办公室设在中环。佢坐低，点了一杯普洱，向梁志明点点头：「梁先生，我哋公司的船舶保障部门上月完成了一个大案子。」

「你们做航运险，」梁志明向黄师傅解释，「Mr. Thompson的公司专门做船同埋船上面运的货。我哋公司做火险和意外险，两边业务唔同，但都系保险，只系范畴唔一样。」

Mr. Thompson笑说：「在伦敦总部眼里，每一间保险公司都系分担风险的节点。风险分散得越均匀，整个市场越稳健。」

「我明白了，」黄师傅说，「你哋唔系竞争，系分开嚟做。」

饮完茶，梁志明送黄师傅出门。湾仔的街道上人来人往，许多穿着西装的年轻人在各间保险公司的办公室之间穿梭递文件。梁志明望住那些背影，心想：这个行业的地图，比一间茶楼大得多。

## 原文定义

香港保险市场的持牌保险中介人分为两大类别：保险代理人（受雇于单一保险公司）和保险经纪（可同时代表多间保险公司）。所有中介人均须持牌并受监管机构的规管。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 梁志明所在公司 vs Mr. Thompson的公司 | 不同保险公司专营不同风险类别 |
| 「代理」与「经纪」的区别 | 代理人受雇于单一保险公司；经纪可同时代表多间公司 |
| 所有人均须持牌 | 持牌保险中介人须经监管机构审批方可执业 |
| 黄师傅在茶楼了解行业分工 | 保险市场由不同类别中介人各司其职 |
