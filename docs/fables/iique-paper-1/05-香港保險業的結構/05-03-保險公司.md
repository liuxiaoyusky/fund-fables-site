# 5.3 保险公司

## 故事

冯船长的货船"海利号"在维港卸完货，他上岸第一件事就是找周老板。周老板的上环海味铺从不做赊账生意，但冯船长是先到货后结钱的老伙计，可以例外。

"保费涨了三成，"冯船长把货运单递给周老板，"问了几间公司，报价差很远。有间说我船龄大了要加钱，有间说我跑的航线风险高——但有一间英资的，说我的船他们接，条件是得多保几项附加险。"

"哪间？"

"就是中环那间，Thompson做经理的那间。"冯船长压低声音，"他们和国际上很多船公司都有来往，背后还有再保公司在撑着。说是再保公司帮他们分担了大风险，他们才敢接我这单。"

周老板想了想，说："那我再问问我认识的本地保险公司。"

梁志明正好在旁边补货，插了一句："周老板，香港现在有一百六十五间获授权的保险公司，其中国际注册的占了六十八间。本地公司熟悉我们这边的情况，但英资公司往往和伦敦的总公司有再保渠道——风险大的时候，他们有地方分摊，我们这边小公司就吃力了。"

冯船长点点头："所以价格差那么远，不是因为他们欺负我，是因为他们承担风险的方式不一样。"

周老板忽然明白了："一般业务风险分散，大家都能做；长期业务——就是那些保人命的——才是真正集中的战场。英资公司两边都有渠道，所以他们进退自如。"

"对，"冯船长把货运单收好，"但不管哪边，最后赔钱的还是得有人站出来。"

## 原文定义

香港获授权保险公司共一百六十五名，其中九十七名在香港注册，其余在境外注册。一般业务前十名市场占有率约百分之四十二，长期业务前十名占百分之八十五点六。一般业务市场分布较为平均，长期业务市场则高度集中。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 英资公司有再保渠道 | 国际保险公司通过再保险分担大风险 |
| 本地公司熟悉街坊 | 本地公司熟悉本地风险特征 |
| 一般业务vs长期业务的集中度差异 | 一般业务市场竞争分散，长期业务高度集中 |
| 冯船长问"最后谁赔钱" | 无论哪类公司，最终都要有人承担赔付责任 |
