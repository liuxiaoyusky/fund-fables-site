# 5.2 行业规模

## 故事

梁志明在保险业监管局的接待处等周老板，手里捧着一杯还冒着热气的阿华田。茶餐厅的嘈杂声从外面传进来，但他满脑子都是冯船长昨天在码头边跟他说的话——"志明，你们这行到底有多大？出事了，我找谁才有用？"

他答不上来。

周老板从电梯里出来，手里攥着几张皱巴巴的保费收据。"问了，"他把收据往桌上一拍，"他们说我的海味货运单要加保，得先看看公司在那四个数里排第几。"

"四个数？"

"接待那个阿叔说的，"周老板坐下来，压低声音，"他说行里有多少持牌公司、有多少中介人、有多少在里面做事的人、每年收多少保费——这四样东西加在一起，才叫'规模'。我那间海味铺在这四样里面占多少？他也答不上来。"

梁志明接过收据，心里忽然明白了。冯船长的问题和周老板的问题其实是同一个问题：这张网有多大，大到什么程度，才能在真正出事的时候兜住人？

他带周老板去隔壁茶餐厅坐下，要了两杯奶茶。"我师父以前说，做保险的人，眼睛要装得下整片海，"梁志明把糖包撕开，"海有多宽、鱼有多少、船长有多少、货值多少——你看不见这些数字，就不知道该准备多少银子赔人家。"

"那你看见了吗？"周老板问。

"看见了，"他把糖搅进奶茶里，"四个数字合在一起，就是香港保险业的大小。大小不重要，重要是——它大到能兜住我们这种在码头边讨生活的人。"

周老板沉默了一会儿，把收据叠好放进口袋。"那下次我出海，"他说，"至少知道找谁了。"

## 原文定义

香港保险业规模可在以下四个标题下探讨：获授权的保险人数目、已注册或持牌保险中介人数目、行业中获聘用的人的数目、保费收入。（来源：保险业监管局）

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 四个数字（持牌公司/中介人/从业员/保费） | 行业规模的四个衡量维度 |
| 梁志明解释"整片海" | 从业者需要从宏观理解行业规模 |
| 周老板关心"出事找谁" | 行业规模影响消费者的实际保障感 |
