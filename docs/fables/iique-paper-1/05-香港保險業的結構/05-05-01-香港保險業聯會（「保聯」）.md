# 5.5.1 香港保险业联会（「保联」）

## 故事

中环英资保险公司的会议室里，Mr. Thompson把一份文件递给梁志明。文件是全英文的，上面印着"Hong Kong Federation of Insurers"的字样，角落里有一个缩写的"HKFI"。

"这是我们保险公司的'会'，"Mr. Thompson说，"香港所有正经做保险的公司，基本上都是这个会的会员。"

林若兰正好进来送文件，顺手把文件翻到第二页，指向其中一行字："他们的目标有两个——第一，向香港人推介保险；第二，鼓励会员达到最高道德标准。"

周老板在旁边听了半天，忍不住问："推介保险我知道，叫人多买保险嘛。但'道德标准'是什么？管用吗？"

Mr. Thompson想了想，从抽屉里拿出另一份文件。"以前有个保险代理人收了客人的钱，没有交给公司，自己用了。客人报赔的时候才发现保费根本没进保险公司的账。那个时候，就是这个'保联'下面的一个委员会在处理投诉、追查那个代理人。"

"那现在呢？"

"现在监管局直接管了，委员会撤了，但保联本身还在，"Mr. Thompson合上文件夹，"它代表所有会员公司和监管局打交道，制定行业守则，培训从业人员——说到底，就是让街坊相信：买保险这件事，是有人盯着、有规矩守着的。"

周老板沉默了一会儿，说："那我就放心了。我去告诉冯船长，他的那张保单，是有组织在看着的。"

## 原文定义

香港保险业联会（HKFI）获保险业监管局承认为香港保险人代表组织，宗旨是向香港人推介保险，并鼓励会员达到最高道德标准及专业水平，从而加强消费者对保险业的信心。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| HKFI会员覆盖所有正经保险公司 | HKFI是获承认的行业代表组织 |
| 道德标准和投诉处理机制 | HKFI曾设有委员会处理代理人与消费者之间的纠纷 |
| 监管局直接发牌后委员会撤销，但HKFI保留 | HKFI从监管职能转型为行业代表与培训职能 |
| 周老板说"有组织在看着" | HKFI的存在增强消费者对行业的信心 |
