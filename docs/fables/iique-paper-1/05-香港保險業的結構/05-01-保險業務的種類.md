# 5.1 保险业务的种类

## 故事

梁志明进保险行三年，跟着梁师傅学了不少，但始终有一个问题困扰着他：为什么同样一张保单，老家伙们一会儿说是"长期业务"，一会儿又说是"一般业务"，甚至还有"法定"两个字？

梁志明不是不懂术语，他是懂太多了，所以越学越乱。

那天午后，铺子里没什么客人，梁志明坐在柜台后翻一本旧保险杂志。周老板的海味舖在隔壁，周老板闲着没事，踱过来喝茶。周老板看了几眼那本杂志，说："你那本书里，写的都是什么？"

"保险分类。"梁志明放下杂志，"保险这东西，可以从不同角度来看。周老板你想听哪种？"

周老板是个直率人，说："你就告诉我，到底分几类。"

梁志明想了想。"三类……不对，是三种分法。"他被自己说糊涂了。

这时从楼梯上走下来一个人——是Mr. Thompson。Thompson是楼上那间英资保险公司的部门主管，偶尔会下来和本地保险行的人喝茶。梁志明赶紧站起来。

Thompson听了他的困惑，笑了一声。"这个题目，我刚进公司时也问过。你问的问题对，因为保险确实有很多分类方法，每一种都有道理。"

梁志明搬了两张凳子，三个人在柜台旁边坐下。

Thompson说，香港保险监管处对保险公司的授权，不是随便分的。他掰着手指：第一，为了监管机构的授权和监督，保险分成了"长期业务"和"一般业务"——这是"法定分类"，不符合要求就不准开业。第二，为了保险公司自己内部组织方便，同一套业务又可以按产品线、按部门来分——这是"实务分类"，各家公司做法可以不同。第三，为了专业人士进修和培训，还有学术上的分类——比如把保险分成水险、火险、人寿险，是为了方便学习和研究。

周老板听到这里，摸了摸下巴："所以同一样东西，可以同时是长期、也是一般？"

"不完全是。"Thompson说，"一份保单在同一时间只能属一种业务。但你从监管角度看是一套分法，从公司管理角度看又是一套分法，从学术教学角度看再是一套。目的不同，分法就不同。这不矛盾——分类本身就是工具，不是目的。"

梁志明把这些话记在笔记本上。他突然明白了自己之前为什么困惑：他不是在记分类，而是在找一个"唯一的正确答案"。但保险业务的分类本来就是多维度的，不同的目的自然就有不同的切法。

周老板问："那我买保险的时候，是不是也要搞清楚我那份属于哪一类？"

Thompson说："你不需要自己分，但找一个靠谱的保险人，他会帮你搞清楚。你负责说清楚你要什么，他负责告诉你属于哪一类、怎么保。"

梁志明把这句话也记了下来。那天傍晚Thompson走后，梁志明一个人在柜台后坐了很久，把笔记本上的三类分法重新整理了一遍。他现在知道了：不是保险本身有分歧，是人们看它的角度不同。而一个好的保险从业员，要能说清楚自己用的是哪一种角度。

## 原文定义

保险业务按不同目的可分为三大类：（a）法定的：为保险业监管机构授权及监察而设；（b）实务的：为公司内部组织而设；（c）学术的：为专业进修及培训而设。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 保险监管处对长期/一般业务的不同授权要求 | 法定分类：为监管机构授权而设 |
| 保险公司按产品线或部门划分业务 | 实务分类：为公司内部组织而设 |
| 梁志明在杂志上看到的分类框架 | 学术分类：为专业进修及培训而设 |
| Thompson解释"目的不同，分法就不同" | 分类本身是工具，不同目的产生不同分法 |
