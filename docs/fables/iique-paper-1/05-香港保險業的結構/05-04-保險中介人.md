# 5.4 保险中介人（Insurance Intermediaries）

## 故事

黄师傅在湾仔的工程判头行里坐立不安。前几天他的工人阿德从脚手架上摔下来，手臂骨折，医药费花了两千多块。他以为买过保险了，但找保险代理人报销的时候，对方说他买的是建筑商要的那种工程险，不是包工人受伤的雇主险。

"那我找你干什么？"黄师傅当时就火了。

他找梁志明诉苦。梁志明正在帮冯船长整理货运保单，抬头说："黄师傅，保险里有两路人，一路叫代理人，一路叫经纪人。你找的那个是代理人——他只替某一间保险公司卖东西，你买什么他就帮你办那间的单。但你要是找经纪人，经纪人不属于任何一间公司，他会帮你看几间公司的产品，然后告诉你哪一间适合你。"

"那为什么我那个朋友买的时候没人告诉我这些？"黄师傅闷闷地说。

"因为以前大多数人都是直接找代理人，"林若兰从打字机后抬起头，"但现在2019年了，所有中介人都要直接拿牌，由监管局统一管，不再是跟着保险公司或者经纪行吃饭了。"

黄师傅若有所思："所以以后不管找谁，都得先看他有没有牌？"

"对，"梁志明说，"有没有牌，代表他有没有被监管局审核过。代理人替一间公司办事，经纪人替客人办事——但两样都得有牌才做得成。"

"那阿德的医药费……"

"你得另外买一份雇主险，"梁志明说，"下次买之前，先想清楚你是要找人替你办事，还是找人帮你找一间合适的公司。两样都可以，只是要找到对的那个人。"

## 原文定义

保险中介人分保险代理人及保险经纪人两类。保险代理人代表保险公司行事，保险经纪人则被视为拥有较广阔经验和独立专业知识，可代表投保人选择合适产品。从2019年9月23日起，所有保险中介人均须获得保险业监管局的直接许可。

## 对应点

| 故事元素 | 概念对应 |
|---|---|
| 黄师傅找错代理人的经历 | 保险代理人代表特定保险公司，产品范围有限 |
| 经纪人不属于任何一间公司 | 保险经纪人具有独立性和广阔的产品比较视野 |
| 2019年后统一发牌制度 | 所有中介人均须直接获保险业监管局许可 |
| "先想清楚找谁办事" | 投保人需理解两类中介人的角色区别 |
