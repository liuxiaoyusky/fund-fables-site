# 三、债券投资的收益和风险

## ⚖️ 寓言故事 —— 《借条的收益与坑》

### 债券投资的三大收益来源

**1. 票息收入**

债券存续期间按票面利率定期支付的利息。票息收入受票面利率、付息频率和持有期限影响。付息频率越高（如季付 vs 年付），复利效应越大。

零息债券没有票息，收益来自面值与购买价的差额。

**2. 资本利得**

在二级市场买卖债券赚差价。债券价格与市场利率呈**反向关系**：
- 市场利率下行 → 债券价格上升 → 卖出赚资本利得
- 市场利率上行 → 债券价格下跌 → 可能产生资本损失

**3. 再投资收入**

把收到的利息和本金再投资产生的收益。零息债券没有中间现金流，不存在再投资收益；而抵押担保债券既有本金又有利息偿付，再投资风险更大。

跨境投资还有**汇兑损益**。

### 债券投资的五大风险

**1. 利率风险**

利率变动引起债券价格变化，是债券投资者面临的**主要风险**。固定利率债券的价格与市场利率呈反向变动：利率升→价格跌，利率降→价格升。浮动利率债券的利率风险较低。

**2. 通胀风险**

通胀使利息和本金的购买力下降。固定利率债券面临更大通胀风险。可买通胀联结债券（TIPS等）来对冲。

**3. 流动性风险**

债券变现的能力。买卖价差越小流动性越高。利率债流动性好于信用债，普通债好于次级债、高收益债。

**4. 提前赎回和再投资风险**

- **提前赎回风险**：发行人在利率下降时提前赎回债券，投资者只能以更低利率再投资
- **再投资风险**：市场利率下行时，收回的利息/本金只能以更低利率再投资

**5. 信用风险（违约风险）**

发行人未按契约支付本金和利息的风险。发行人财务状况越差，违约风险越大。

**信用评级**：
- 国际三大评级机构：穆迪（Moody's）、标准普尔（S&P）、惠誉（Fitch）
- 投资级债券：S&P/惠誉BBB-级以上，穆迪Baa3级以上
- 投机级债券（高收益/垃圾债券）：BB+级或以下
- "失落的天使"：从投资级降为投机级的债券

国内评级机构：大公国际、中诚信、联合资信、上海新世纪、中债资信等。

---

**📖 原文定义**

> 债券投资的收益来源主要有三个方面：票息收入、资本利得收入、再投资收入。

> 债券投资的风险可以划分为五大类：利率风险、通胀风险、流动性风险、提前赎回和再投资风险、信用风险。

> 利率风险是指利率变动引起债券价格的变化，它是债券投资者所面临的主要风险。固定利率债券的价格与市场利率呈反向变动关系。

> 信用风险又称"违约风险"，是指债券发行人未按照契约的规定支付债券的本金和利息，给债券投资者带来损失的可能性。

> 标准普尔和惠誉的BBB-级及穆迪的Baa3级以上等级的债券是投资级债券。投机级债券又被称为高收益债券或垃圾债券。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 定期利息 | 票息收入 |
| 低买高卖 | 资本利得 |
| 利息再投资 | 再投资收入 |
| 利率升→价格跌 | 利率风险，反向关系 |
| 通胀侵蚀购买力 | 通胀风险 |
| 变现快慢 | 流动性风险 |
| 发行人提前赎回 | 提前赎回风险 |

| 发行人还不起钱 | 信用风险/违约风险 |
| BBB-/Baa3以上 | 投资级债券 |
| BB+以下 | 投机级/垃圾债券 |
| 穆迪、标普、惠誉 | 国际三大评级机构 |
