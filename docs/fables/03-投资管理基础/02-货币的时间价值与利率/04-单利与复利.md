# 四、单利与复利

## 🏺 寓言故事 —— 《十亩与百亩》

阿朴和阿复是孪生兄弟，父亲临终前把家产平分：每人十亩麦田，一柄镰刀。

第一年秋收，每亩产麦一石。兄弟俩各收了十石。

阿朴把十石麦子全换成银子，锁进樟木箱。他盘算得清楚："父亲留给我十亩，每年十石，十年就是一百石。箱子里的银子只多不少。"

阿复只卖了五石，剩下的五石托人换了五亩荒地。他连夜开垦，赶在霜降前播下麦种。

阿朴摇头："你把到手的麦子换荒地，今年不就少吃五石？荒地产量还低，何苦？"

阿复擦着汗说："麦子吃了就没了，地才是活的。"

第二年，阿朴的十亩好田收十石。阿复的十亩好田收十石，五亩荒地收三石——共十三石。

阿朴照旧，十石全锁进箱子，累计二十石。阿复又卖了五石，用八石换了八亩荒地。

阿朴笑得更响了："你忙了一年，才比我多三石，却要多开垦八亩地。我躺在家里，箱子也在替我长银子。"

阿复没说话，只是继续开垦。

第三年，阿朴收十石，累计三十石。阿复二十三亩，收十八石。

第四年，阿朴收十石，累计四十石。阿复三十一亩，收二十五石。

第五年，阿朴收十石，累计五十石。阿复三十九亩，收三十三石。

阿朴打开箱子，五十两银子码得整整齐齐。他觉得自己赢了——阿复累死累活开垦了那么多地，五年下来也没比他多收多少麦子，还白白少吃那么多口粮。

但阿复的地已经连成一片。第六年起，他再也不用买地了——每年收成里拿出一部分，地自己会"生"出新的地：河滩淤积出沃野，边角垦出薄田，连田埂上都能种几垄豆子。

第八年，阿朴累计八十石。阿复的田已超过七十亩，一年收六十石。

第十年，阿朴打开樟木箱，一百两银子码了十排。他志得意满地去田里，十亩薄田在秋风里泛着熟悉的金黄，还是十石。

阿复没空接待他。一百多亩连片的麦田望不到边，雇工们正在收割。账房先生报数："今年收了一百一十石。"

阿朴愣在田埂上。他忽然明白：自己十年收了一百石，不过是把同一块地重复割了十次。阿复的每一块新地，都在替他割麦子。

阿复走过来，递给他一碗新麦粥："哥，你箱子里的一百两，是死的，今年用完就没了。我的地，明年还能收一百一十石，后年更多。"

阿朴捧着碗，热气熏红了眼。

**"钱锁在箱子里是死的，种进地里才是活的。死钱只生一次利，活钱年年都生利。"**

---

**📖 原文定义**

> 单利是计算利息的一种方法。按照这种方法，利息仅根据本金计算，无论经过多长时间，所产生的利息均不会加入本金继续计算新的利息。换句话说，利息只在最初的本金上累积，而不会形成"利滚利"的效果。

> 与单利不同，复利的特点是每经过一个计息期，所产生的利息会加入本金，形成新的本金，并在下一个计息期继续计算利息。这样，利息逐期累积，使得资金增长呈现加速趋势。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 父亲留下的十亩田 | 本金（PV） |
| 每年每亩产麦一石 | 利率（i） |
| 阿朴把收成锁进箱子 | 单利——利息不加入本金 |
| 阿复用收成换荒地 | 复利——利息加入本金继续生息 |
| 阿朴每年固定收十石 | 单利利息只按最初本金计算 |
| 阿复的地越来越多、收成加速增长 | 复利使资金增长呈加速趋势 |
| 第十年阿朴累计一百石 | 单利终值：本金 + 利息×期数 |
| 第十年阿复一年就收一百一十石 | 复利终值：本金×(1+利率)^期数 |
| "死钱只生一次利，活钱年年都生利" | 单利与复利的本质区别 |
