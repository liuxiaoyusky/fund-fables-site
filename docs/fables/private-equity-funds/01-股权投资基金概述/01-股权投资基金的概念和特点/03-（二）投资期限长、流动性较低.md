# （二）投资期限长、流动性较低

## 🌳 寓言故事 —— 《老槐的三年》

老槐是西边周家寨的庄稼人,种了二十年水稻,日子紧巴但还算稳当。这一年开春,村东头的胡大哥从外头回乡,手里攥着一把苗——是从湘西带回来的果苗,说三年挂果、五年丰产,挂果之后每年都能摘。苗子不要钱,可他没工夫伺候,想寻个踏实人接手。

老槐蹲下来看了半天,问:"苗归我,银子谁出?"胡大哥说:"苗、肥、工夫都是你的。挂果之后,头三年的收成,咱俩三七分;第四年起,果子全归你。"老槐心里盘算:三亩水田,一亩种稻稳当,两亩空着也是空着,种苗不亏。便把苗接了过来,挑了那两亩靠近山脚、朝阳又透水的地,一棵一棵栽下去。

头一年,苗子瘦弱,叶子稀稀拉拉。村里人路过都要指指点点:"老槐,你那两亩水田,一亩能收四百斤稻,他一棵树结几颗果子?糟蹋地。"老槐的婆娘更急,几次三番念叨:"娃儿明年要上学堂,束脩要钱,你把那苗刨了,种回稻子吧。"老槐摇头:"苗是苗,稻是稻,地里种什么,心里得有数。"

到了秋后,老槐手头真紧。村西的钱二叔找上门,说愿意出三两银子,把那两亩地上的苗连地带苗一起买走,第二天就付钱。老槐只要点了头,娃儿的束脩、当下的口粮、来年的种子,一下都有了。可他想起胡大哥的话——"三年挂果",还差两年呢。他摆摆手,把钱二叔送出了门。

第二年开春,胡大哥病了一场,银子看得紧,来了一趟,开口借二两。老槐的婆娘眼睛一瞪:"你倒好,苗在他地里,他来跟你借钱!"老槐也犹豫了一夜,可第二天还是从箱底摸出二两递过去。婆娘摔了碗:"这种苗,挂果不知哪年,钱倒先往外出!"老槐蹲在门槛上抽了一袋旱烟,没说话。

到了第三年春天,胡大哥的病好了,人也精神了。两亩地上的苗,枝干粗了一圈,叶子密得风都钻不进。这年谷雨刚过,老槐一早去地里,忽然看见——最低的那根枝条上,结了一颗小青果。他蹲在树下看了半天,没敢动。

消息传开,村里人都来瞧。钱二叔又来了,这回开价八两,要把树连果一起买。老槐还是不卖。到了秋后,那两亩地,大大小小摘了四百斤果子。头一年照约定三七分,胡大哥分了一百二十斤,老槐落二百八十斤。第二年五百斤,第三年七百斤——年年在涨。胡大哥的婆娘抱着一筐果子笑了:"老槐,你这一手,种出了真东西。"

老槐坐在自家门口,看着山上那片绿,跟来串门的赵大伯说:"胡大哥是懂苗的人,我只是种苗的人。苗是慢慢长的,中间缺钱、缺人、急事缠身,样样都要扛得住。扛不住,半路卖了,后面那些年的大收成,一样也落不到自己手里。"

赵大伯问:"你就不怕三年过去,树一棵都不结?"老槐笑了笑:"这事儿,种之前就得想明白。信,才种;种了,就不轻易刨。"

---

**📖 原文定义**

> 股权投资基金主要投资于未上市企业股权或上市企业的非公开交易股权,通常需要5年以上才能完成投资的全部流程实现退出,因此被称为"耐心资本"。股权投资基金的基金份额流动性较低,在基金清算前,基金份额的转让具有一定的难度。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老槐接管两亩地种下果苗 | 投资行为,需长期持有 |
| 胡大哥说"三年挂果、五年丰产" | 5年以上才能完成退出 |
| 苗子种下后头两年颗粒无收 | 投资期限长,早期无回报 |
| 婆娘反复劝老槐刨苗种稻 | 短期变现 vs 长期持有的两难 |
| 钱二叔开价三两要买走苗和地,老槐拒卖 | 在清算/退出前,转让具有一定难度 |
| 钱二叔第二年加价到八两,老槐还是不卖 | 基金份额流动性较低 |
| 胡大哥中途病重来借钱,老槐咬牙借出 | 长期持有过程中需承受中途资金压力 |
| 头三年收成按三七分,第四年起全归老槐 | 退出前后的不同阶段 |
| 老槐说"信,才种;种了,就不轻易刨" | "耐心资本"——愿意等、扛得住 |
| 老槐朋友最终认定"种出了真东西" | 长期持有才能换来的价值 |
