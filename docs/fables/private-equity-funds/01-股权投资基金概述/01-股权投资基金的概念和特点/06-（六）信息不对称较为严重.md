# （六）信息不对称较为严重

## 🔍 寓言故事 —— 《何老板的抽屉》

何老板在城北开着一间绸缎庄,这一年绸价忽上忽下,他心里总悬着。

老朋友老米来串门,说:"城西染坊的赵掌柜要退了,那几口缸、那张老招牌,折成银子不过八百两。我跟他打了十年交道,门道全知道。你出八百两,我替你接过来,后头我盯着;赚了,三七分;亏了,先亏我这一份。"

何老板动了心:老米在染布这行里混了十几年,哪家料好、哪家缸勤,闭着眼都摸得清。两人说定,各写一张字据,只写"八百两、分账",其余没细写。何老板把银子交到老米手上,老米拍拍胸脯:"老何,你只管等响。"

头三个月,老米每隔十来天来坐一坐,报一报:今天拉了什么料、明天请了哪个师傅。何老板心里便踏实一分。

可他不知道的是,老米这三个月里其实已与四个人交过手——赵掌柜的弟弟、染坊旧账房、缸匠张师傅、还有南边一爿布庄的沈老板。这四桩事老米桩桩都办了,桩桩都对何老板"管用"——他替何老板省下银两、换好新缸、谈妥销路;可桩桩的细节,老米只对何老板说"已经办妥",从不把底牌摊出来。

老米不是想坑何老板。在老米看来,这些事太琐碎,讲出来反而把何老板绕晕;自己已替他办好,何必再让他操心。何老板也没追问——他只知道账上确实多了银子,缸也换了一口,布也确实开始卖了。何老板觉得老米是个能托付的人。

第四个月,事情出了岔子。

一个穿着体面、自称"林掌柜"的中年人找上门,说手里有一批上好靛青料,价钱只要市价一半,印鉴齐全。何老板心想:老米那边染坊正缺好料,这事若办成稳赚。他没跟老米商量,也没找人核对契约,就从柜上另拿出二百两,签下了字据。

三天后再去寻"林掌柜",人已不见。何老板把契约拍给老米看,老米瞅一眼,脸色便变了:"这印鉴是仿的。这人,我听都没听过。"

何老板急了:"染坊的事你不都清楚?我以为这行的人你都认得!"

老米叹了口气:"染坊那摊子我认得清。可染坊之外的料商、脚夫、印鉴坊,这个江湖大着呢——里头哪些是真、哪些是套,我也不全认得。"

何老板愣住了。他回家坐在八仙桌前,看着抽屉里老米交来的那本账——账上确实多了银子,染坊确实在转;可抽屉里没有的,那些印鉴、契尾、缸底、销路、还有老米与四路人打过的交代,他从不知情。

他这才明白:自己这大半年,看见的只是老米"让他看见的";他以为自己在看账,其实账的背后还有账;他以为投的是银子,其实投的是对老米的信任。可信任这东西,光凭一张胸脯是不够的。

当晚,何老板把老米请来,头一次一字一字跟他提了三件事:办前要让他知道、办后把字据给他看一眼;另请账房每季过一遍账;赚了怎么分、亏了怎么担,白纸黑字写清楚。老米沉吟良久,点了点头,两人重立了一份细字据。何老板那把椅子底下,从此垫了一张实实在在的底。

何老板后来再没见过那位"林掌柜"。他只记得当时心里那一阵高兴——高兴得连老米都没问一句,就拍板了。这笔学费,何老板交得心疼。可他也清楚:若不是这笔学费,他大概还会坐在那张八仙桌前,看着抽屉里那本干净的账,以为一切尽在掌握。

---

**📖 原文定义**

> 在股权投资基金的运作架构中,基金管理人具有信息优势,投资者通常处于信息劣势地位。为防止基金管理人利用信息优势侵害投资者利益,可采取以下措施:一是基金管理人通过定期和临时信息披露,提高基金运作的透明度;二是投资者通过选择基金审计机构、审议关联交易等环节参与基金治理,防止管理人滥用其信息优势;三是通过设计合理的基金利益分配机制,减少管理人与投资者之间的利益冲突。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老米对染坊人脉、料行、缸匠如数家珍 | 基金管理人具备信息优势 |
| 何老板只看见账上数字和老米"让他看见的" | 投资者处于信息劣势地位 |
| 老米与赵掌柜弟弟、账房、张师傅、沈老板的暗中交涉 | 关联交易与多方博弈,投资者无从知晓 |
| 何老板从未追字据、追底牌,只凭老米一句话 | 信息不对称下,投资者易被信息筛选"喂养" |
| "林掌柜"登门,何老板未与老米商量便拍板 | 信息不对称时,投资者被套路的典型风险 |
| 何老板与老米立契:"办前告知、办后核字据" | 措施一:定期和临时信息披露,提高透明度 |
| 何老板另请账房先生每季过账 | 措施二:投资者借助第三方审计/账房参与治理 |
| 何老板与老米重订白纸黑字的利分规则 | 措施三:合理的利益分配机制,减少利益冲突 |
| 何老板最后交的"学费" | 信息不对称带来的实质损失 |
