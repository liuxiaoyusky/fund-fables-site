# （三）投后管理投入资源较多

## 🏺 寓言故事 —— 《三年三十趟》

清河镇东头有一爿老染坊,东家姓沈。三年前沈东家病倒,临终把染坊交给了老婆陈氏和十六岁的儿子小沈。孤儿寡母守着几口缸、五个伙计,外头还欠着印子钱,要账的登门不断,陈氏整夜合不上眼。

有一天,镇上的孙掌柜登门。他原是城里有名的布商,进门拍下一摞银票:"这银子你收着,染坊归我管账,年底赚头二八分。"陈氏捏着银票,心里七上八下——怕染坊从此不是自家的。孙掌柜像是看穿了:"我只管一件事——让它活下去,活到小沈能独当一面。"

头一年,孙掌柜往染坊跑了十二趟。头三趟替陈氏挡债——他让陈氏把欠债全列出来,挨家挨户上门,能谈的分期,不能谈的先垫。中间五趟是盯料——他带回三家新染料商,价更廉,残布率从两成掉到一成。后面四趟是教人——让小沈跟着学看账,又请了一位老染匠带他练上色。陈氏看着小沈在缸边忙到深夜,心里一阵发酸,又一阵踏实。

第二年,孙掌柜又跑了十二趟。城东新开两家大染坊,用西洋机子,出布又快又匀,镇上小布庄纷纷舍了沈家。陈氏急得哭了好几回。孙掌柜不急:"咱们不比快,比对。"他把活路改了——专做定色、专做老主顾回头单,又把布庄掌柜一个一个请到染坊里来看手艺。看过的,回去都续了约。

第三年开春,孙掌柜来得少了。清明,小沈独自去城里染料行走了一遍,带回一款新料;五月,陈氏自己理清去年账目,还跟孙掌柜商量起明年;七月,老染匠说:"少爷这手青蓝,能出师了。"

腊月二十三小年那天,孙掌柜照约来了,只提了一刀红纸、一壶酒。他等陈氏把今年红利数出来——二百四十两。孙掌柜接过分给自己那份,又把多出一锭推回去:"留着,给小沈明年娶媳妇用。"

陈氏红着眼收下。她想起三年前那个上门拍银票的人,那时候还怕他是来占便宜的。这三年她一笔笔数过来——挡债十二家,换商号三家,带新人两位,改活路一回,上门合计三十六趟。外人只看见今天这二百四十两,却看不见这三十六趟里,每一趟都是银子、人情、主意,还有一次次替人把心放回肚子里的功夫。

小沈端来一盘自家染的青布:"孙叔,这块您留着。下回再有哪家孤儿寡母撑不住,您就告诉他们——这布,是沈家染坊头一回,凭自己手艺挣来的。"

---

**📖 原文定义**

> （三）投后管理投入资源较多
>
> 股权投资是价值增值型投资。基金管理人通常在投后管理阶段投入大量资源，一方面，为被投资企业提供各种商业资源和管理支持，帮助被投资企业更好地发展；另一方面，也通过参加被投资企业股东会、董事会等形式，对被投资企业进行有效的投后管理，以应对被投资企业的信息不对称和企业管理层的道德风险。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 孙掌柜第一年十二趟,第二年又十二趟 | 投后管理阶段持续投入资源 |
| 帮陈氏挡债、挨家挨户谈分期、垫钱 | 为被投资企业提供商业资源 |
| 从城里带回三家染料商、调整活路 | 帮助被投资企业更好地发展 |
| 参加股东会/董事会类比——带小沈学看账、引入老染匠 | 加强对被投资企业的实际管理 |
| 改活路"专做定色、专做回头单"——孙掌柜拿主意 | 应对信息不对称,主动把握方向 |
| 三年三十六趟上门,陈氏一笔一笔数出来 | 大量资源投入的具象化 |
| 旁人只看见年底二百四十两,看不见三十六趟功夫 | 投后价值的隐蔽性,价值在过程中积累 |
| 小沈第三年自己上街看料、师傅说出师 | 资源投入带来被投资企业的成长 |
