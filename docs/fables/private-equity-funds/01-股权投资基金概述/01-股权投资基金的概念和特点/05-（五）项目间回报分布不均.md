# （五）项目间回报分布不均

## ⚖️ 寓言故事 —— 《百灶同街》

永兴街不长，从东头到西头，统共三百步。可就在这三百步的街上，开着整整一百家炒米铺——家家用的都是南陵产的三粒寸大米，家家架的都是同款厚底铁锅。街坊们都叫它"百灶同街"。同一批米、同一口锅，可五年下来，街面光景像是被人偷偷换过了一遍。

故事要打头的那年腊月讲起。

三家新铺子几乎同时开张。气派最大的是东头的孙大勺——四十出头，跑过十几年码头，炒米讲究"猛火重油"，开张头一个月就放话"三年要做街上最大"。街中间的是从南边来的哑叔，闷人一个，只守自己那口灶。西头是老周老吴两兄弟，在城里酒楼帮过厨，可家里底子薄，开张那阵连伙计都只请得起一个。

第五年开春，街坊们做了一回"算账酒"。

孙大勺头两年最风光，主顾最多。第三年城北新开大酒楼，把他的客人拉走一半。他咬牙借了一笔印子钱想扩成大酒楼，利滚利，第五年开春灶台冷了三天，米缸空着，外头还欠着账。

哑叔五年还是老样子——不换招牌，不添伙计，没人买就蹲门口看街。街坊一算，他没亏过，没借过钱，第五年反倒悄悄在街尾又盘下一间小铺。问他怎么撑下来的，他只笑了笑，比划一个"火候"——慢火，稳住气，米自然会一颗颗跳起来。

最让人吃惊的，是老周老吴。两兄弟五年里一直最不起眼。可算账酒上，老周掏出一本账本翻开——全是些零碎活儿：帮人代炒米、帮隔壁铺熬高汤、帮南边茶商代货。每一笔都小得可怜，五年拢一起，竟比街上大半铺子赚得都多。兄弟俩说："我们不挑生意。赚得多的少有，亏得惨的也没有，积少成多，倒成了这条街最厚实的家底。"

算账酒散席那晚，街坊们回头望那一百口灶台——同一袋米、同一口锅，五年下来，有的灶火烧得太旺最后熄了，有的火小一直烧着细水长流，有的看着不亮却在暗处一点点攒下一桶油。没人能说清这到底是为什么。

**📖 原文定义**

> （五）项目间回报分布不均
>
> 在整个金融资产类别中，股权投资基金属于高风险、高期望收益的资产类别。
>
> 高风险主要体现为不同投资项目的收益呈现出较大的差异性。创业投资基金通常投资于早中期的成长性企业，投资项目的收益波动性更大，有的投资项目会发生本金亏损，有的投资项目则可能带来巨大收益。并购基金通常投资于价值被低估但相对成熟的企业，投资项目的收益波动性相对小一些。
>
> 高期望收益主要体现在正常的市场环境中，股权投资基金作为一个整体，其能为投资者实现的投资回报率总体上处于较高的水平。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 永兴街上 100 家炒米铺用同一批米、同一口锅 | 同一类资产，进入门槛一致 |
| 孙大勺头两年最风光，最后灶冷缸空 | 高风险：单一项目可能本金亏损 |
| 老周老吴做散事积少成多成最厚实家底 | 高期望收益：整体回报水平仍较高 |
| 孙大勺"猛火重油"追风口、借印子钱扩规模 | 早中期成长性企业式高波动：可能巨赚也可能巨亏 |
| 哑叔不换方子不借债细水长流 | 价值被低估但相对成熟企业：波动相对小一些 |
| 三家新铺子开张条件差不多，五年后结局大不同 | 项目间回报分布不均：差异性是常态 |
| 没人能说清为什么，但每口灶都用了同一袋米 | 同类资产里，回报差异由选择决定 |
