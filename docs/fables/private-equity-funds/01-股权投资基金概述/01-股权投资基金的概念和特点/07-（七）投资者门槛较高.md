# （七）投资者门槛较高

## 🏺 寓言故事 —— 《码头边的凑本钱局》

民国年间,江南的临河镇有个做南北货生意的东家,叫沈伯年。他家祖上在码头边开了三十年的干货行,到了他这辈,手里攒下了不少本钱,可一个人毕竟做不大。沈伯年琢磨了半年,决定在镇上发起一桩"凑本钱做买卖"的局——不上市,不入号,本钱由几位东家凑出来,他出面操盘,赚了按事先说好的分。

消息一传出去,好些人都动了心。隔壁米行的少东家曹阿根头一个上门,揣着三百块大洋,兴冲冲地跟沈伯年说:"沈老板,这局我入一股,赚了我们二一添作五!"沈伯年接过茶碗,慢悠悠喝了一口,没点头,却反问:"阿根啊,我先问你一句——你这三百块,是闲钱,还是急钱?"

曹阿根一愣:"都是钱,有什么不一样?"沈伯年笑了笑,把他按回椅子上:"差得远。急钱,指的是下个月要进新米、月底要给伙计发工钱、明年要给儿子娶媳妇的钱。这种钱放进我的局里,三五个月不见回头,你就得跳脚;三五年不见回头,你家米行都得被人盘走。闲钱,才是我这局里要的钱——你得能扛得住,睡得着觉,听见隔壁王老板说'沈伯年这局黄了'也不动心。你若扛不住,我这局,你就别进来。"

曹阿根脸上挂不住,辩了一句:"我曹家还能扛不住?"沈伯年也不恼,伸出一根指头:"那你再答我第二桩——你看得懂账吗?我做的南北货行当,行情怎么走,货怎么囤,伙计怎么派,这里面有的是门道。你若只看见'赚'字,看不懂'险'字,莫说三五年,就是半年也熬不到头。我这局门槛高,高就高在这两桩——一要你能扛得住时间,二要你看得懂门道。差一样,我都不敢接你。"

曹阿根回去想了一夜,第二天把那三百块原封不动收回了箱底。他没再去找沈伯年,反倒去找了在镇上开布庄的何先生学看账。临走时他跟自家账房叹了句:"沈伯年那局,我先前只当是凑份子;如今才明白,那局是挑人。挑不挑得过,各人心里有数。"

---

**📖 原文定义**

> （七）投资者门槛较高
>
> 股权投资基金相对于其他投资形式,通常会设定较高的投资者准入门槛。首先,股权投资基金的投资策略和投资标的较为复杂,涉及企业的财务状况及行业前景等多方面的信息,因此,基金投资者需要具备一定的专业知识,以确保其具有一定的风险识别能力。其次,由于股权投资基金通常涉及高风险和长期投资,基金管理人通常会设定较高的单一投资者最低投资金额,以确保其具有一定的风险承担能力。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 沈伯年把"急钱"挡在门外 | 需具备一定的风险承担能力 |
| 沈伯年说"三五年不见回头,你也得不动心" | 长期投资属性对耐心的要求 |
| 沈伯年要求"看得懂行情、账、门道" | 需具备一定的专业知识与风险识别能力 |
| 沈伯年自己定规矩、挑人 | 基金管理人设定较高准入门槛 |
| 曹阿根带回三百块、改去学看账 | 进入门槛高,需自我达标 |
| 曹阿根感叹"那局是挑人" | 投资者门槛较高,缺一不可 |
