# 一、股权投资基金的概念

## 🌾 寓言故事 —— 《永宁镇与清平镇》

永宁镇和清平镇隔着三道山梁。

永宁镇热闹。镇口有布庄、粮行、钱庄、肉铺，招牌一块挨一块，每天太阳还没出来就有伙计在门口洒水扫地。镇上的读书人管这叫"明面市"。明面市做的是看得见的买卖——买进、卖出、当天结账、来回两讫。一桩生意从搭话到收尾，三五天见分晓。

清平镇安静。从永宁镇往山里再走三天，路就断了，再走三天才能到。镇上没有布庄、没有粮行，沿街只有几间不挂招牌的屋子。镇上人进进出出都不声张，做完了买卖就把门关上。读书人管这叫"闷声局"。闷声局做的买卖看不见——一块地底下埋着什么矿、一片山林里会长出什么木头、一个还没长成的后生将来能不能成事，都要等上好几年。

永宁镇的周老伯今年六十七，攒了一辈子钱，攒下八百两银子的棺材本。他原本只在明面市里转——买布、买粮、买点稳妥的田产。但这一年山外打仗，粮行收紧了铺面，布庄也压了货，银子攥在手里花不出去。

老周心里慌。

他的老朋友、住在清平镇远房表亲陈掌柜，听说这事，给他捎了一封口信。信上说："老周，你那八百两若是放在明面市里，三五年便要缩掉三成。倒不如拿出大半来，让我替你放进山里的几桩事。赚了，年底分你；亏了，也不至于全赔进去。"

老周心里疑惑——清平镇那几桩事，究竟是什么？

陈掌柜见他不解，便带他走了三天山路，来到清平镇。

清平镇的几间不挂招牌的屋子里，坐着的是管山、管林、管矿、管铺的几位把式。他们各自正在做几桩事——西边窑场在烧一种还没上市的细瓷，要等五年才能烧出一窑好货；东边药农在山里试种一种南方新引的草药，三年才能收；北边油坊在用一种新压法榨山茶子，第一年榨出来的油还不知道城里人买不买账。

陈掌柜对老周说："你把银两交给我，我替你买进这些窑场、药田、油坊的份子。赚了，我们分；亏了，先亏你这一份，但只亏这一份。买卖怎么做，几位把式自己定。你不掺和，也不让你掺和。"

老周把八百两银子压了六百两进清平镇，留下两百两过日子。

三年过去。

永宁镇的明面市起起伏伏——粮行又开了一家、布庄换了东家、钱庄差点倒了又被救回来。老周留下的两百两缩水到了九十五两。

清平镇那边，陈掌柜捎来消息：西边窑场的细瓷成了名品，烧一窑卖一窑，城里大户人家排着队等；东边药田的草药第一年试种失败、第二年补种、第三年终于收成，卖了个好价；北边油坊的山茶油被一位南来的行商一眼相中，整年包了下来。

陈掌柜算了算：老周那六百两，扣掉各项开支，三年共分得利银三百一十两。

老周坐在永宁镇的院子里，看着手里那张分账的纸条，想起三年前自己问陈掌柜的那句话——"清平镇那几桩事，究竟是什么？"

他终于明白了：

永宁镇的明面市，是"摆出来的买卖"——看得见、摸得着、当下有回报。

清平镇的闷声局，是"还没摆出来的买卖"——看不见、摸不着、要等几年才有回报。

陈掌柜替他做的事，就是把"摆出来的银两"换成了"还没摆出来的事"。

老周活到七十二那年，他孙子问他："爷爷，您这辈子做的最聪明的一件事是什么？"

老周想了想，说："是我把压箱底的银两，从看得见的铺面，换到了看不见的窑场、药田、油坊里——换来的是三桩还没长成的事。"

---

**📖 原文定义**

> 国内所称"股权投资基金"，是指主要投资于非公开交易企业股权的基金。这里的"非公开交易企业股权"，包括未上市企业的股权和上市企业非公开交易的普通股、可转换为普通股的优先股等。
>
> 在国际市场上，股权投资基金既有以非公开方式募集（私募）的，也有以公开方式募集（公募）的。在我国，目前股权投资基金只能以非公开方式募集。
>
> 从基金募集方式和基金投资标的两个维度来看，股权投资基金是投资基金家族中的一员，与证券投资基金有着明显区别。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 永宁镇明面市（布庄、粮行、钱庄、肉铺） | 证券投资基金的标的——已上市、可公开交易的证券 |
| 清平镇闷声局（窑场、药田、油坊） | 股权投资基金的标的——非公开交易的企业股权 |
| 陈掌柜替老周把六百两银两换成清平镇三桩事的份子 | 把资金换成对非公开交易企业股权的持有 |
| 老周赚或亏，都只以"他出的那一份"为限 | 基金财产的独立性与有限责任 |
| 老周不掺和窑场、药田、油坊怎么做 | 投资者不直接参与被投资企业的日常经营 |
| 老周远在永宁镇、陈掌柜在清平镇代他打理 | 资金的委托管理——基金"受人之托，代人理财" |
| 清平镇几间不挂招牌的屋子 | 非公开——区别于公开募集 / 公开交易 |
| 我国清平镇"闷声局"目前只能由熟人介绍、口头相托 | 我国股权投资基金目前只能以非公开方式募集 |
