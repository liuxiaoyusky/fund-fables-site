# 《十六铺桥头腊月廿三分红》

---

## 开场

一九五三年腊月，永丰镇十六铺桥头吴记茶馆里，老陆那桩窑药合股的生意到了收割季。

山后那桩窑厂整整烧了三年，腊月廿三这天窑匠世家陈师傅挑着扁担下山，一担白花花的银元外加两坛药行的陈酿药酒——合股八家头一次见着回头钱。

老陆、钱庄老吴、药行李掌柜、榨油师傅、刘账房、孙掌柜、周跑街、府城钱业公会孙先生，围着一张八仙桌。老陆把银元往桌面上一倒："先别急着数，听刘账房念契最后那页。"

## 冲突

刘账房翻到契纸最后一页，字字念道："合股分红，须先扣回本钱六千；再扣窑匠工钱、药行晒药人工、押车脚钱、契纸上的杂项开销；余下方为可分之银。八家按份子均分。"

钱二爷皱眉："八家均分？可这桩买卖，从挑窑址到烧砖到押货下山，跑前跑后都是我钱二爷与吴记茶馆一手张罗——没有带头钱，下回谁肯挑头？"

郑老大敲桌："合股契讲的是均分，多给带头的人银元，等于抽了旁人份子。"钱二爷冷笑："不抽我份子，谁替八家操心？"

孙掌柜指着契面："契上白纸黑字没写带头钱三个字。"钱二爷反问："没写便不许立？"窑匠陈师傅插嘴："我烧窑的手艺钱算不算？"

一时争执不下，八仙桌上茶盏咣咣响。钱庄老吴搁下算盘："诸位，今日若不分明白，往后再有合股，谁敢牵头？"

## 转折

老陆把茶盏一顿："诸位争的是分账的顺序规矩。我永丰镇合股契写到今日，靠的是四桩顺序——"

"一、先扣回本钱六千，柜上未回本之前不谈分账；二、再扣窑匠工钱、押车脚钱、契纸杂项，一笔一笔按账走；三、扣完开销，余下方为可分之银；四、可分之银里，先给八家一份保底——这就是契上的'领头水'。保底之外，再立一道'过线钱'：八家份子按'门槛'算，柜上银元若过这道门槛，再多出来的利钱，牵头的人才许抽。"

老陆顿了一顿，指着钱二爷："你牵头跑前跑后，本就该得'领头水'——这是牵头应得的，不是抽旁人份子。"又指陈师傅："窑匠手艺是工钱，按窑匠那一行单算，写进契面。"

钱庄老吴翻出契纸空白处："这些顺序、这些规矩，契面须写死。本钱、开销、可分之银、领头水、过线钱——一项一项写明白，免得明年再起争执。"孙先生点头："契约契面，字字落实。"

钱二爷眉头舒开："过线钱才是我该拿的——八家先过一道门槛，剩下的我抽。门槛之下，一文不取。"

## 结局

八家议定。老吴提笔，在契纸最后那页续写——

一、先扣回本钱六千；再扣窑匠工钱、药行晒药人工、押车脚钱、契纸杂项开销，一笔笔按账走。

二、扣完开销，余下为可分之银。八家按份子均分；分账日定腊月廿三，年前清账。

三、可分之银里，先给八家保底领头水；保底之上，再按份子均分。

四、立一道过线钱——八家份子算到一定数，过了这道线，牵头之人方从余下利钱中抽成；未过此线，牵头人一文不取。

五、若日后查出当年账有差错，允许多分的银元由牵头之人回拨柜上——这条回拨规矩，契面写明，跑不脱。

钱二爷按手印时笑道："过线钱该是我拿的，不是我白抽的——是八家先过门槛，剩下的我才抽。"郑老大点头："门槛之下不动你一文，过线之上归你抽成——这才公道。"

其余七位依次落印。刘账房把银元按账分好，八家各自捧回自家份子。钱二爷另外取了自己那一份过线钱，拱手道："明年再牵头。"

孙先生端起茶盏："合股契最后那页写满了分账的顺序与规矩，才是永丰镇合股的根。"陈师傅把契纸吹干，贴在十六铺桥头茶馆板壁上——来年再有合股，照此办理。

**📖 原文定义**

> （五）基金的收益分配。基金收益分配是基金运作中的重要环节，包括基金的可分配资金、分配时间、分配模式（按基金整体或按单一项目分配）、分配顺序、分配比例等。基金合同中应当就上述内容进行约定，避免产生歧义。基金分配环节，可设置回拨条款或类似条款，并采取有效措施来保障回拨条款的执行，回拨条款和保障措施应在基金合同中进行约定。为对基金管理人进行有效激励，并保证投资者获得合理的优先收益，鼓励设置恰当的管理人业绩报酬计提基准（门槛收益）和比例。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| "先扣回本钱六千，再扣工钱、脚钱、杂项" | 分配顺序 —— 先返还本金，再扣除费用 |
| "扣完开销，余下为可分之银" | 可分配资金 —— 扣除本金与费用后可供分配的资金 |
| "分账日定腊月廿三" | 分配时间 —— 基金合同约定的分配时点 |
| "按份子均分" / "按份子分账" | 分配模式（按基金整体分配）—— 按出资比例在全体出资人之间分配 |
| "八家一份保底领头水" | 分配比例与投资者优先收益 —— 保证投资者获得合理优先收益 |
| "过线钱"——牵头人过门槛后方抽成 | 管理人业绩报酬与门槛收益 —— 投资者收益率达到门槛后，管理人方可提取业绩报酬 |
| "过线钱抽成" | 分配比例 —— 管理人参与收益分配的比例 |
| "若查出当年账有差错，允许多分的银元回拨柜上" | 回拨条款 —— 在基金合同中约定并保障执行 |
| "回拨规矩，契面写明，跑不脱" | 保障回拨条款执行的措施 |
| "契约契面，字字落实" | 基金合同应就分配事项进行明确约定，避免歧义 |
