# （二）基金的存续期限

## 🕰️ 寓言故事 —— 《两季茶山》

皖南山里有个老周,做茶叶生意做了半辈子。这年开春,他把同村的老李、老陈、老孙叫到堂屋,把一张大纸往桌上一摊——是村后那片三百亩荒山的草图,打算合伙开茶园,五年内把茶树种满,第六年起年年采茶卖钱。

"钱的事,"老周敲了敲桌子,"一亩按一两银子算,一人先垫八十两。山上的活,头五年种树养苗,谁也不许动收成的念头;第六年起,茶青归公,卖了再分。五年期满,树没种满自己掏钱补;期满还没长好,大伙儿一起商量是接着等还是散伙。"四个人听了都点头,签字画押。

头两年,老周带着大家天天上山。种、剪、肥、灌,一环接一环。可到了第三年开春,三百亩地才种了二百二十亩——老陈的婆娘动了胎气,占去他大半精力;老李的腿摔了一跤,卧床三个月;老孙的孩子要进县学,人走不开。还有八十亩空着,老周坐在山脚的破庙里,把草图翻来覆去看了一夜。

第四年春天,有人找上门,劝老周把剩下八十亩转出去——"卖给山外的王家,出价公道。"老周本想点头,可他想起签的那张纸:五年期满再议。半路卖地,等于半路散伙,前面三年种下的二百二十亩,以后年年采茶,钱归谁?他咬了咬牙,没答应。回家跟三个老伙计一合计,大家说:"种不种满,先把剩下两年熬过去再说。"

到了第五年开春,茶树已经一片连一片,绿得发亮。老周知道,新苗不能再多种了——种下去也来不及长大,该把心思从"种"转到"管"上。他立了一条新规矩:从今年起,山上的活只做三件——剪枝、追肥、巡山防虫,谁也不许再提"再开新地"。老李的腿好了,老陈的婆娘也出了月子,大家天天在茶垄里弯腰,看着那一片片叶子由嫩转厚、由黄转深。

第六年开春,头一批春茶采了下来。四个老头站在山腰,看着背篓里青翠的茶青,谁也没说话。当年的茶青按约全归公账,卖了钱先补大伙这几年垫进去的工钱和肥钱,剩下的一分四份。第七年、第八年,年年如此,每年收成比头一年厚实一些。

到了第八年冬天,山上二百二十亩茶园,该卖的老树卖完,该移的苗子移完,能结的茶青都结了。四个人坐在堂屋,把账本一合——五年期满,又续了三年,前后八年,才把当年合伙时定下的事情,一件一件做完。老陈说:"头五年是真苦,光往里贴,一分钱看不见;后三年才见回头钱。"老孙说:"要是当年没忍住,第五年就散伙,后头那三年的茶,一棵也轮不到咱们。"

老周把草图卷起来,塞进柜子里头,慢慢说:"头五年是种地的命,后头是收粮的命。种地的时候想着收粮,人容易慌;收粮的时候还想着再多种几亩,粮也收不好。两段日子,各管各的事,心里才不乱。"

---

**📖 原文定义**

> 股权投资基金作为一种资产管理产品,一般会约定存续期限。通常情况下,基金的存续期限包括投资期和退出期两个阶段。在投资期内,基金主要进行对外投资活动;在退出期内,基金主要进行投后管理和项目退出活动,不再增加新的对外投资。当然,在投资期内,对于已经发生的对外投资,基金也会进行投后管理甚至是项目退出活动;在退出期内,对于投资期内已经完成投资决策及投资协议签署,但尚未完成对被投资企业实际出资的项目,也需要按投资协议约定完成实际出资等投资交割活动。
>
> 正常情况下,基金应在存续期内完成全部对外投资和项目退出活动,全部项目实现退出后,基金应及时进行清算。如果基金在约定的存续期内未能完成退出清算,可按基金合同约定的方式或经全体基金合同当事人协商一致延长基金存续期限。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周定的"五年种满,第六年起采茶" | 基金约定的存续期限 |
| 头五年种树养苗的阶段 | 投资期(基金主要进行对外投资活动) |
| 老陈婆娘养胎、老李摔腿,耽误种树进度 | 投资期内可能遇到的不确定性 |
| 第四年有人要买剩下八十亩,老周不卖 | 投资期内不能因短期诱惑中断既定安排 |
| 第五年起老周立规矩"不许再开新地,只做剪枝、追肥、巡山" | 退出期:不再增加新的对外投资,转为主要做投后管理 |
| 第六年起头一批春茶采下,头年茶青全归公账 | 退出期内基金主要进行项目退出活动 |
| 第五年剪枝、追肥、巡山的同时,老茶树继续有收成 | 投资期内对已投资项目也会进行投后管理和项目退出 |
| 头三年一直没见回头钱 | 投资期与退出期现金流时点不同 |
| 五年期满没做完,大伙一起商量续了三年 | 经全体基金合同当事人协商一致延长基金存续期限 |
| 第八年全部茶青结完、账本合上,准备散伙 | 基金在存续期内完成全部对外投资和项目退出后,应及时进行清算 |
| 老周最后一句"两段日子,各管各的事" | 投资期与退出期阶段分明、各有侧重 |
