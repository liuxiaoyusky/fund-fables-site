# （一）基金规模及出资方式

## 🪙 寓言故事 —— 《码头的合账》

江南临水码头上,做南北货的有十几号人,说话最算数的要数阿贵。开春,他把何大、赵嫂、孙七、林五、邱二、方大姐六位请到茶馆,关了门,摊开一张大纸:"咱码头要修三座仓,以后各家的大宗货就往仓里过,抽一笔公账养码头。这事我一个人做不下来,大伙儿一块儿凑,一块儿管。"

何大先开口:"凑多少合适?"阿贵伸出一只手:"每家出三百两,六家就是一千八百两。仓先修一座,余下的先放着,等生意做大再上第二座。"赵嫂倒吸一口凉气:"三百两,我两年才攒下八十两,仓还没修起来,我家娃娃的束脩先没了着落。"孙七也叫苦:"一样,银子压在货上,一时半会儿拿不出全数。"

阿贵把茶碗一推,半晌才说:"我也不是让大家今天就把三百两都拍在桌上。"他用指头在纸上画了两栏,一栏写"应出",一栏写"已出":"'应出'每家三百两,白纸黑字,签字画押,定了不能反悔。可银子不必一次搬来。仓要修到哪一步、要进多少货、要花多少,你们看着账本一分一分拿。我这边每动用一笔,就在'已出'那栏记下来,凑齐总数,再开第二座仓。"

赵嫂眼睛一亮:"就是先说个数,再慢慢掏?"阿贵点头:"先说个数,后掏银子。本来就是两件事。"

何大又问:"我那三百两能不能拿货顶?"他叫进伙计,抬进两箱徽墨,说这是年前压下的货,市价少说也值三百两。阿贵把箱子推回去:"何大,这事我只能认银子。徽墨我收了,过几年行情变了,值两百也值四百,账上没法跟大伙儿交代。咱码头上的事,只认白花花的银子——现银、银票、汇号通兑的,都行,旁的一律不收。"何大脸上挂不住,可回头一想,确实在理,便把墨重新抬回去了。

签约那天,六人按了手印。"应出"一栏六家各三百两,合计一千八百两;"已出"还是空的。阿贵把第一笔账摊开:第一座仓的地基、砖石、木料,合计一百二十两,六家按比例分摊,每家先付二十两。一个月后,仓动了工。再过三个月,第一座仓盖起来,阿贵又把账摊开:地基之外添了三十两的意外开销,每家再补五两。开张那天,"已出"合计一百五十两出头,离"应出"还差得远。何大心里却踏实了——大话是定过的,银子是一分分掏的,两边分开记,谁也不慌。

半年后,仓用得好,周转也顺,阿贵按约把第二座仓的图纸摊开,又把"应出"那栏重抄一遍——大伙儿看明白了,数还是一千八百两,可掏钱的方式没变,该几时掏、掏多少,看工程走,一笔一笔来。赵嫂这回没叫苦,反倒跟孙七商量:"咱先把下一笔的二十两备下,省得到时候抓瞎。"

老茶馆老板听完,问阿贵:"你这事,前前后后,不就一句话——先说大数,后掏小钱,数要清楚,钱要干净?"阿贵把茶一饮而尽,没答话,只把那张纸平平整整折好,塞进了袖口。

**📖 原文定义**

> 基金规模是全体投资者对基金出资的总和,分为认缴出资规模与实缴出资规模。认缴出资规模是指基金合同约定的投资者承诺的出资总额,实缴出资规模是指截至某一时点,基金投资者已经完成实际出资的资本规模。
>
> 股权投资基金通常要求全部用货币资金出资,一般不接受实物资产、无形资产等非货币资金出资。
>
> 根据基金合同约定,投资者可以一次性或分期履行其在股权投资基金中的出资义务。实践中,股权投资基金通常约定分期缴付,即投资者要承诺对基金出资的总规模,按合同约定的期限或条件分数次完成其出资行为。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 阿贵定下"每家出三百两,六家合计一千八百两" | 认缴出资规模:合同约定的全体投资者承诺的出资总额 |
| 签约时白纸黑字写下"应出"一栏,合计一千八百两 | 认缴规模以基金合同为依据,签字画押后即固定 |
| 纸上的"已出"一栏,随着地基、砖石、木料一笔笔填入 | 实缴出资规模:截至某一时点已经完成实际出资的资本规模 |
| 仓开张当天"已出"合计一百五十两出头,离"应出"还差得远 | 实缴规模动态累积,在认缴达成前始终小于认缴 |
| 阿贵把"应出"和"已出"分作两栏分开记账 | 认缴规模与实缴规模在账目上独立反映,二者既相关又分离 |
| 何大要拿两箱徽墨抵三百两,阿贵推回去 | 全部用货币资金出资,不接受实物资产出资 |
| 阿贵强调"只认白花花的银子,现银、银票、汇号通兑" | 货币资金出资的具体形式:可流通的现钞与银票等 |
| 阿贵说"先说个数,后掏银子,这是两件事" | 认缴与实缴可以分离:先确定承诺总额,后分期完成实际出资 |
| 修第一座仓时"每家先付二十两"对应地基、砖石、木料 | 分期缴付:按工程进度和资金需求分数次完成出资 |
| 阿贵说明"多就多放着修第二座,不够再开会重新定" | 实缴规模在认缴规模内可按需拨付,富余留存,不足再议 |
| 半年后第二座仓图纸摊开,"应出"那栏重抄一遍 | 在认缴规模内继续按工程节奏分期完成实缴 |
| 赵嫂主动跟孙七商量"先把下一笔二十两备下" | 实践中分期缴付已成常态,投资者按约履行的自觉 |
