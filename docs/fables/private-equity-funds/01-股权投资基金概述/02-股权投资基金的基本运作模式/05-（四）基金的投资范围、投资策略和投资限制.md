# （四）基金的投资范围、投资策略和投资限制

## 🎯 寓言故事 —— 《三个圈》

南城米行的老掌柜姓薛,做了三十年粮食买卖,手里攒下一笔银子,打算投到别处生利。邻里劝他自己上手做,薛掌柜摇头:"隔行如隔山,还是托给懂行的人。"他托人找了三拨掌柜来谈——城东的孟氏,城西的苏氏,还有本镇外号"活地图"的吴先生。三人各自说了一套,他却越听越糊涂。

头一位孟掌柜进门就铺开一张大纸:"从丝绸到盐铁,从钱庄到田亩,只要有赚头就下手。"薛掌柜听着,眉头越皱越紧——他这三十万两,要是全撒出去,怕是连本都看不住。第二位苏掌柜反而另一头:"我只收一种,年景好的米庄。旁的再便宜,也不动心。"薛掌柜又觉得太窄——他原本就干粮行,再去押米,等于没换。倒是第三位吴先生不同,没铺张纸,只问了一句话:"薛掌柜,你这钱,打算守住,还是打算长?"

这句话问住他了。薛掌柜原本是冲着"长"来的,可家里老母亲身体一年不如一遭,长子还要娶亲,总得留一笔"随时能拿回"的。一夜没睡,他心里慢慢有了一杆秤——他要"长",但得有底线。

第二天他把三人再请来,这回他自己先开口:"我有三个圈。"他用指头在桌上一圈一圈画:最里头一圈写"押米行"——他懂的,看得明白;中间一圈写"押临河的米行,运船三天能到的"——多一道地方限定,本乡本土他熟;最外面那一圈画得很虚,他不落字,只说:"这一圈里,我不要别人替我画。"

孟掌柜眼睛一亮:"薛掌柜,前两个圈我来守,中间那个画细图,临河几里、走哪条船,都写明白;最外面那圈,咱们三年一议,谁拍板也得您点头。"

薛掌柜这才点头,又提了一件事:"还有几样,我说在前头,谁来管账都不能碰——我不要把银子拿去放印子钱,那是要折寿的;我也不要替人担保,出了事我担不起;再有,最外面那一圈的事,要动可以,得十家人里有八家点头。"

孟掌柜没有二话,把桌子一拍:"就这么写。"

三年过去,薛掌柜那笔银子,稳稳当当长了两成,中间一年临河涨水,他看中的几家米庄硬是没亏。最让他心安的是——最外头那个虚圈,从头到尾没动过。

有人问他当初怎么想出来的,薛掌柜捋着胡子笑:"我哪懂什么门道,就是守着'我'这个字。'投到哪里'是门道,'挑哪一家'是门道,'什么不能碰'也是门道——可这三条,都得我自己的秤称过。别人的秤再准,称的不是我家的肉。"

**📖 原文定义**

> （四）基金的投资范围、投资策略和投资限制
>
> 基金的投资范围,是基金投资对象的集合。投资范围通常依据目标投资对象的行业、地域、发展阶段等属性来确定。
>
> 基金的投资策略,是指基金在选择具体投资对象时所使用的一系列规则、行为和程序的总和。基金投资策略的确定,通常依据基金的投资目标以及投资者对风险和期望收益的偏好来进行。
>
> 基金的投资限制,是指基金不得从事或必须依照一定程序得到投资者许可后才能从事的投资行为。投资限制的确定,通常服务于保护投资者利益这一目标。
>
> 通常情况下,基金的投资范围、投资策略和投资限制由基金管理人和投资者共同商定,并在基金合同中明确列示。

| 薛掌柜投三十万两给懂行的人,自己不直接上手 | 投资者与懂行的受托人共同商定框架 |
| 孟掌柜铺大纸"丝绸到盐铁都做" | 一类做法:投资范围过宽,风险敞口大 |
| 苏掌柜"只收米庄",薛掌柜觉得太窄 | 一类做法:投资范围过窄,失去分散意义 |
| 薛掌柜先想清楚"守住还是长" | 投资目标(风险/期望收益偏好)决定策略 |
| 最里头一圈"押米行"——他懂的 | 投资范围:依据行业属性确定投资对象集合 |
| 中间一圈"临河、运船三天" | 投资范围:依据地域属性进一步限定 |
| 孟掌柜承诺"中间那个画细图" | 投资策略:在范围内选择具体对象的规则和程序 |
| 最外圈"三年一议,八家点头才能动" | 投资限制:必须依照一定程序得到投资者许可才能从事 |
| 薛掌柜"不要放印子、不要替人担保" | 投资限制:基金不得从事的投资行为,保护投资者利益 |
| 三个圈最后白纸黑字定下来 | 投资范围、策略、限制共同在合同中明确列示 |
| 薛掌柜自己的秤称自家的肉 | 投资范围、策略、限制由投资者与受托人共同商定 |
