# （五）基金的收益分配

## ⚖️ 寓言故事 —— 《井水不犯河水》

南城外的榨油坊,东家姓钱,做了四十年,手艺人老实。三年前他盘下铺面扩大,借了同乡老范三十万两,约好"赚了再分"。老范是镇上的钱粮师爷,家里地多,银子散在几处用不上,便借着这一笔想"长出点活钱来"。两人找了油坊西头茶馆的孙先生当中人,立了字据,定下三七分账——赚的钱,东家七,老范三。

头一年行情稳,油坊赚回本钱之后又攒下三万两。两人欢欢喜喜把钱分了,老范拿九千,东家拿两万一。酒席上老范还拍着钱东家的肩说:"孙先生找得好,明年再多些,咱照办。"钱东家笑着点头,心里却留了个疙瘩——"照办"是怎么办?是把赚来的照办,还是把本钱也一并算进去?

第二年生意反倒淡了,到年底一算账,刨去油料、伙计工钱、修锅的花费,还倒贴了五千两。老范脸色难看了半天,孙先生出来打圆场:"赔了就不分,东家自个儿兜着,明年再说。"老范哼了一声,勉强点头。

到了第三年,油坊遇上大年。茶籽丰收,价又高,年底一盘,刨去花费、刨去老范那三十万两的本钱,还多出十二万两。老范眼睛亮起来,搓着手就往茶馆跑:"按三七,那十二万我分三万六!"

钱东家没接话,只说:"老范哥,先坐。"他让伙计把账本摊开,一笔一笔指给老范看。头一笔,先把今年的开支还清;第二笔,把你那三十万两的本钱原封不动退回去;第三笔,本钱退完之后剩下的那十二万两,才是"咱们一起赚的"。

"那三七分这三万六不正好?"老范急着接。

"不急。"钱东家又从柜上拿出一张纸,那是三年前立的字据,"当年咱们说的,是赚到'一个数'之后,你才拿三成;没赚到那个数,就是东家自个儿担着。"老范一愣——他记得这茬,只是从来没把那"一个数"当回事。孙先生当年写的是"年利十厘之上",也就是三万两。钱东家说:"今年赚了十二万,过了这坎了,那三成该你拿。"当场点出三万六,两清。

老范揣着银子,走到门口又回头:"东家,这十厘的'坎'是怎么定的?我回去也想跟我们家那几房立个规矩。"

钱东家笑了笑,说:"哪有什么玄虚,就是一句话——先把你借我的还清,再把我家铺子被人借的利还上,剩下才是大家分。你来借钱,图的是比放贷多的那一份;我来借你的力,图的是手艺放大。咱们先把各自的小账理清,再谈一起分大账——井水不犯河水,谁也不占谁便宜。"

孙先生在旁边听得直点头。茶馆外头,秋阳正暖,井边的辘轳咿呀转着,谁也没再提那本三年前的旧字据——账结了,字据也就没用了。

**📖 原文定义**

> （五）基金的收益分配
>
> 股权投资基金的收益分配主要在投资者与管理人之间进行。
>
> 股权投资基金的收入主要来源于被投资企业分配的股息红利以及实现项目退出后的股权转让所得。基金的收入扣除基金承担的各项费用和税收之后,由管理人按照基金合同约定进行收益分配。
>
> 由于股权投资基金具有专业性特点,股权投资基金管理人通常会参与基金投资收益的分配。通常情况下,根据股权投资基金与基金管理人的约定,基金投资者的收益率达到某一门槛收益率(Hurdle Rate)之后,管理人才可以获得业绩报酬。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老范借三十万两给钱东家扩铺面 | 投资者(出资人)向管理人(经营方)提供资金 |
| 赚了先还本——把老范三十万两原封退回 | 先返本:收入扣除费用税收后,先返还投资者本金 |
| 还本之后再分利——剩下十二万两才算"一起赚的" | 返本之后剩余部分才进行收益分配 |
| 字据写"年利十厘之上"老范才拿三成 | 门槛收益率(Hurdle Rate):投资者收益达到该水平后,管理人才可获业绩报酬 |
| 三年到期按字据清账,字据作废 | 基金合同约定的收益分配时点与方式 |
| 钱东家自己兜住五千两亏空,不分给老范 | 投资者以出资为限承担有限责任,亏损先由基金/管理人承担 |
| 第二年大年赚十二万两,过了"十厘"的坎老范才拿三成六 | 跨越门槛后管理人按约定比例(此处三七)提取业绩报酬 |
| 油坊的收入来自卖油所得——油料收成好、价高 | 收入主要来源于被投资企业业绩增长及退出所得 |
| 开支扣完、本钱退完、剩下的才"一起分" | 收入扣除基金承担的费用和税收之后才进行分配 |
