# 一、股权投资基金的基本架构

## 🌾 寓言故事 —— 《四个人的水渠》

山下大旱，井水见了底，三个村子凑在一起议水。

老张、老李、老赵都有一片祖传水田，一遇旱年稻子就蔫。三人合计：凑钱修一条渠，把山那边沁水河的水引过来。三家一共凑出三百两——老张一百二十两，老李一百两，老赵八十两。

这是头一桩安排：出钱的人。

钱交给谁去办？三人都不懂水脉。镇上林师傅干水利二十年，去岁还给县衙修过堤。三人把三百两交到林师傅手里，签了一张纸：钱由他管，事由他办，赚了归大伙分，亏了他自己也跟着一份。

这是第二桩安排：办事的人——自己也跟着出一点，跟出钱的人绑在一起。

可三人又嘀咕：林师傅兜里揣着三百两，进度怎么样、账目清不清，他自己说了算。老赵说："镇上钱庄的孙掌柜专管替人看钱记账，人称铁算盘，不如把银子搁他那儿，林师傅要用，按规矩支取。"林师傅想了想，也点了头。

这是第三桩安排：管钱的人——自己不办事，就守着那匣子银，按规矩看进看支。

开春动工。林师傅拿孙掌柜一支一支支出来的银两，请工匠、定石料、看水脉，三个月把水渠挖通。水哗哗地流进三个村子的田里。

第一年稻子长势喜人。第二年，沿途润到了邻村，邻村王秀才愿出二百两，买断水渠未来十年的浇水权。林师傅跟老张三人商议，把这笔买卖做了。银子又回到孙掌柜的钱匣。

林师傅把账拢了拢：浇田的收成分七成归老张三人，一份归自己；卖浇水权那二百两，扣掉工匠料钱零碎开支，余银按"先回本、再分余"算下来，老张三人得回一百二十两，林师傅得八两。

这是第四桩安排：水浇到了田里——出钱、办事、管钱，最后都得落到用水的人头上。

第二年冬天，林师傅跟老张坐在祠堂里喝茶。

"这桩事，咱们四个人谁也替不了谁。"林师傅说，"你不出钱，我再有本事也动不了工；我不出力，钱在匣子里发霉；孙掌柜不盯账，咱几个迟早起疑心；田里没人用水，渠修成了也成了摆设。"

老张笑笑："可不是么。一条水渠，四个角，少哪个都立不住。"

他顿了顿又说："这桩事成之前，我最怕的是钱交给你、你拿钱跑了；第二怕是钱交给你、你拿去乱花；最怕是渠修成了水来了，可田里没人要。"

林师傅点头："对。这三个怕，正好就是那三个角——把钱交对的人、把账看住的人、把水用起来的人。你出钱的人不亲自下田，可你离不开我们仨。"

---

**📖 原文定义**

> 股权投资基金是开展股权投资的载体，在管理人的运作和管理下，将从投资者募集的资金投向底层投资项目，通过被投资企业的业绩增长和发展获得资本增值，在投资项目实现投资退出后，再将相应的收益向投资者进行分配，从而实现为投资者获取投资收益的目标。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老张、老李、老赵凑出三百两 | 投资者——资金供给方，承担最终损益 |
| 林师傅修渠、办成事、自己也跟着出一点 | 管理人 / 共同出资人——受托运作资金 |
| 孙掌柜守着钱匣、按规矩支取 | 托管人——独立保管资金、按规则监督 |
| 三个村子的田亩、邻村王秀才用水 | 被投资企业——接收资金、创造价值 |
| 银两从三人匣子流向林师傅、再流向工匠石料 | 资金流：投资者→基金→被投资企业 |
| 林师傅卖浇水权得二百两 | 投资退出——将权益变现 |
| 分账时七成归老张三人、一份归林师傅 | 收益分配：投资者获本金与收益、管理人获报酬 |
| 林师傅自己也跟着一份、跟大伙绑在一起 | 管理人与投资者利益捆绑 |
| "四个角，少哪个都立不住" | 四方角色缺一不可——基本架构的本质 |
