# 《街口小铺子的借钱难》

## 开场

老吴把茶盏一搁："**上回讲陈师傅靠合股契烧新窑，今儿个换调门——唠唠街口小铺子借钱到底有多难。**"

孙先生掂着铜印："**难。大银号根本不瞅这些小买卖。**"

## 冲突

方掌柜叹气："**头一样——掌柜知道的事东家不知道。我米行好歹像样，跟大银号答账答得清。可街口小铺子柜上连成文的章程都没，进出多少装在掌柜肚子里，摸底摸不着。**"

刘账房拨珠："**第二样——小买卖抗不住大浪头。行情一变、官府一告示，说倒就倒。大银号最爱押稳当买卖，一翻就翻的小字号，给再多利息也不敢借。**"

陈师傅接话："**第三样难在我这行——新釉方子、新梭子，那叫没实物可押的本事。大银号摇头说'拿不出押物，借不成'。**"

周先生压低嗓门："**第四样——小铺子从摆摊到开张到做大，借钱花样不一样。摆摊要合股、开张要周转、做大要再压一两年大本钱。大银号只认一种借法，哪能都对得上？**"

郑老大皱眉："**那岂不是借不到钱、做不成买卖？**"

## 转折

老吴翻开花名册："**可这两年小铺子反倒越开越多，靠的就是合股契。解这四难，有四把钥匙。**"

刘账房点头："**头一把——翻家底。从寻买卖、挑买卖、查底细、估买卖、定契文、投了以后还管，一道道关都查，掌柜知道的事东家也得知道，糊弄不过去。**"

方掌柜接话："**第二把——几桩买卖一齐押。合股契不把大本钱压在一处，这桩赔了那桩赚，东家不慌，把大浪头摊开。**"

陈师傅拍腿："**第三把——合股契不要押物。我那新釉方子、药行新方子、机坊新梭子，全对路。**"

周先生接腔："**第四把——份子分等。头等份子先分红再管账；借据可换成份子的契，开张借钱、做大转成合股；普通份子长长久久跟着走。一道契套几样，分段对小铺子胃口。**"

许先生补一句："**合股契投了以后还管——介绍好主顾、把大银号搭把手帮着借钱的路子接上、帮着升窑样添新梭子。这种搭把手，大银号断不肯做。**"

孙先生掂印："**更难得是——合股契最爱早押小字号。摆摊那阵就敢押，一路陪着。谁都借得到钱的买卖，合股契真做到这一句了。**"

## 结局

老吴搁下茶盏："**合股契落到街口小铺子身上，扶的不是一家两家，是一整条街。早押小字号、借得到钱、借钱也不贵——这才是合股契落到根子上的好处。**"

郑老大环视一圈："**那合股契帮小铺子，到底帮的是啥？**"

众人齐声："**帮的是把大本钱压到小铺子里头，让掌柜敢放手做、敢试新、敢做长。一桩小买卖做起来，十桩百桩跟着起来，永丰镇的街口才热闹。**"

**📖 原文定义**

> **二、股权投资基金助力中小微企业融资**——中小微创业企业融资难的原因：一是信息不对称比较严重，许多中小微企业缺乏规范的公司治理，投资者获取其真实信息的难度和成本都比较高；二是中小微企业的抗风险能力较弱，容易受到政策或环境变化的冲击；三是科技型中小微企业的无形资产往往占比较高，易使抵押担保物不足；四是中小微企业在不同阶段的融资需求存在差异（早期需要股权投资，中后期需要优先股、可转换债等配套）。相对而言，股权投资基金能够更高效地满足中小微企业融资需求：从项目来源、初步筛选、尽职调查、项目估值、投资条款、投后管理等多个环节降低信息不对称；通过投资多个项目构建投资组合分散风险；股权投资无须企业提供担保；可灵活采取普通股、优先股、可转换债等方式匹配不同阶段融资需求。同时提供投后增值服务（介绍客户、对接资源、与银行合作开展助贷业务等）。股权投资基金践行"普惠金融"战略，加大"投早、投小"力度，解决中小微企业"融资难、融资贵"问题，成为中小微企业的重要直接融资渠道。
