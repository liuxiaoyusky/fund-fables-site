# （一）基金销售机构

## 🏺 寓言故事 —— 《两爿肉铺》

清朝年间，徽州老街上开着两爿肉铺。街东头的铺子老板叫阿贵，是本地人；街西头的铺子叫阿福，从外县迁来。两家挨得近，生意却天差地别。

阿贵自家养了二十头猪，每天清早现杀现卖。街坊们来买肉，他一边切肉一边说："这块五花三指厚，回家红烧最好。"街坊听惯了他的话，也信得过他——猪是他养的，秤是他称的，出了问题找他就是。

阿福不一样，他自己没有猪场，铺子里摆的肉，全是从临县屠户郑三那里进的货。他没有铺面之前，街坊们买肉只认阿贵，压根不知道他这号人。

开张头一个月，阿福的铺子冷冷清清。他急得在店里来回踱步，最后想出一个法子：让郑三替他在自己铺子门口挂出一块木牌，写着"临县郑三肉铺·徽州分号"。又让自己老婆每天早上在门口支一口大锅，熬骨头汤免费给过路人喝。街坊路过，闻见肉香总要停下脚步问一句。问的人一多，他便把郑三家的肉切开来一块块摆在案上，一边比划一边讲："郑师傅的猪是吃山芋长大的，膘肥肉香，跟阿贵家喂泔水的不一样。"听了这话，街坊渐渐松动，三文两文的肉也肯在他案上称。

可好景不长。半年后，街尾的李婶找上门来骂街——她在阿福铺上买的肉，回家发现夹着一层黄膘，第二天还拉了肚子。阿福一边赔礼，一边心里发虚：那块肉其实是郑三的徒弟小陈切的，他连那徒弟长什么样都没见过。街坊们议论纷纷，有人说阿福铺上的肉连他自己都说不清来路，劝大家还是回阿贵那儿买。阿福站在门口，懊悔地撕掉了那块木牌。

夜里，他坐了半晌，忽然想明白一件事：自己卖的不只是肉，是街坊的信任。郑三帮他张罗门脸、招揽客人，介绍的是郑三这块招牌；可端到街坊案板上的那块肉，是好是坏，街坊只认阿福。阿福不敢再图省事，第二天便把郑三那徒弟赶回了临县，自己跟着郑三亲自进猪场挑猪，又把每头猪的来路记在账本上，街坊买肉他就把那张纸亮出来让人看。

过了几月，阿福铺子前又排起了队。可这一回，他明白了一个道理：店是自己的，肉是别人的，出了问题担待的还是自己——若不想办法让街坊看清来路，再响的招牌也救不了这爿铺子。

---

**📖 原文定义**

> 股权投资基金的募集，可分为自行募集和委托募集。自行募集，是指基金管理人直接募集其管理的基金；委托募集，是指基金管理人委托基金销售机构代为募集基金。
>
> 受托募集的基金销售机构为基金管理人提供推介基金、发售基金份额、办理基金份额认缴及退出等募集服务。
>
> 基金销售机构在提供募集服务过程中，应将基金管理人的基金募集材料及时完整地提供给潜在基金投资者，全面如实地披露投资风险及可能的投资损失，不得隐瞒任何重要信息，不得对募集材料中的信息做出任何误导性陈述。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 阿贵自家养猪、自己卖肉 | 自行募集——管理方直接面对出资方 |
| 阿福没有肉源，请郑三供货并挂"分号"招牌 | 委托募集——委托他人代为办理 |
| 阿福在铺里介绍郑三家的猪、让街坊看账本 | 推介基金、发售份额、提供募集材料 |
| 阿福撕下木牌、把郑三徒弟赶走、自己跟着进猪场 | 全面如实披露、不得隐瞒重要信息 |
| 街坊在阿福铺上买肉出了问题只找阿福 | 受托方对出资方承担相应责任 |
| 阿福切肉比划、老婆熬骨头汤招徕 | 募集服务的具体方式（推介、发售） |
