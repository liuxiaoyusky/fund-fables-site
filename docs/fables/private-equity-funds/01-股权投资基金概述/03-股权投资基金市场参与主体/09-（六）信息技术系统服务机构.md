# （六）信息技术系统服务机构

## 🛠️ 寓言故事 —— 《老匠人的五把尺》

九十年代中，江南乌镇开了家"通汇号"票号。掌柜沈家骏精明强干，把生意做到了上海滩。可他最近愁得睡不着——票号越大，要管的事越多：进出的钱怎么记、来存银的人怎么登记、每笔账怎么算清、当天账目怎么拍平、大伙计下注买卖怎么盯……书房里堆满厚本子，每天对账点到后半夜。

他摆了一桌酒，请来本地最有名的老匠人秦一鸣。秦师傅一辈子造工具——算盘、账本夹、印鉴盒、锁头，闭着眼都做得出。沈家骏拱手道："秦师傅，我请您来，是想请您帮我做五样看家家伙。"他掰着指头数：要一套能记下每笔进出的账本，要一套给每位客户立名册的簿子，要一套能算清每一笔银两的算筹，要一套让柜上当天账目一摊平的流水盘，要一套让掌柜在后台盯着大伙计落牌的暗牌。"只要您造出这五样，票号里的活儿，我自己和伙计干得了。"

秦一鸣连连摇头："沈掌柜，这五样我能做。可有句老话我得说在前头——打铁的不能自己挑担子卖铁，造秤的不能上场做买卖。这五样家伙一旦交给您，我秦一鸣从此不进通汇号半步，更不能伸手去帮您敲算盘、替您登记名册、替您清点银两、替您按盘对账、替您翻牌下单。我只管做家伙、修家伙、保家伙不出毛病。"

沈家骏一愣，旋即明白过来。两人击掌为约，秦一鸣便领徒弟在通汇号后街租了间大屋，日夜赶工。三年下来，五样家伙抬进通汇号：红木大账本，每笔进出都留痕；铜面厚名册，每位来存银的客人都有自己的格子和编号；紫檀算筹，每颗珠子打磨得光滑；青铜流水盘，柜上每天进多少出多少，到打烊一推就对得平；最后是檀木暗牌，摆在掌柜后堂，前头伙计每一下落牌，后堂一一映出。

家伙造好那天，沈家骏要留秦一鸣喝庆功酒。秦一鸣摆摆手，提着工具箱就走了。往后三年，他一次也没踏进通汇号。柜上有个叫赵小满的伙计，算账算错了，银两差了一钱三厘。柜里人急得要派人去请秦师傅，沈家骏沉下脸："秦师傅造的是算筹，账目归咱们自己算。把算筹拿过来修一修是人家的事，让人替我算账不在约里。"赵小满红着脸，自己翻着算筹重新算，到掌灯时分终于把那一钱三厘找回来。

又有一次大伙计私下把一笔大买卖的暗牌记错了，沈家骏在后台一眼看出，拿着那副檀木暗牌去找秦一鸣。秦一鸣到柜上只做了一件事：把暗牌机括拆开擦了一遍，又重新装回去。擦完字迹比先前更清——至于大伙计该怎么罚、这笔买卖该怎么算，秦一鸣一字没提，背起工具箱就走。

票号生意越做越大，外头人提起通汇号，都说"那家号子的家伙最硬"。但凡懂行的人都知道，那五样家伙硬，离不开后街那位从不进通汇号大门的老匠人。
---

**📖 原文定义**

> 信息技术系统服务，是指为基金管理人、基金托管人和其他服务机构提供的基金业务核心应用系统、信息系统运营维护及安全保障等服务。基金业务核心应用系统包括销售系统、投资交易管理系统、份额登记系统、资金清算系统、估值核算系统等。信息技术系统服务机构提供基金业务核心应用系统时，不得从事与其所提供系统相对应的基金服务业务，不得直接进行相关业务操作，可以提供信息系统运营维护及安全保障等服务。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 沈家骏提出要造的五样看家家伙 | 基金业务核心应用系统的五项核心内容（销售、投资交易管理、份额登记、资金清算、估值核算） |
| 秦一鸣在通汇号后街大屋日夜赶工造家伙 | 信息技术系统服务机构提供基金业务核心应用系统 |
| 秦一鸣立约"从此不进通汇号半步" | 提供核心应用系统时不得从事相对应的业务 |
| 秦一鸣不替柜上敲算盘、不替人登记名册、不替人清点银两、不替人按盘对账、不替人翻牌下单 | 不得直接进行相关业务操作 |
| 赵小满算错账自己重新算，沈家骏拒绝请秦师傅代算 | 业务操作由管理人/服务机构自行完成 |
| 秦一鸣拆开暗牌擦机括后装回 | 信息系统运营维护及安全保障等服务 |
| 秦一鸣修完暗牌不评论大伙计的过错 | 维护方只对系统负责，不介入业务决策 |
| 通汇号的"家伙最硬" | 信息技术系统服务机构为基金运作提供基础设施保障 |
