# （二）基金管理人

## 🏺 寓言故事 —— 《老钱的账本》

江南梅雨刚停，镇上的染坊老板老钱请来一位叫沈川的掌柜。沈川三十出头，算盘打得精，听说专替人"打理闲钱"。老钱开门见山："钱生钱，省心。"沈川却慢慢放下茶盏："您把钱交给我，图个啥？咱得先定规矩——放到哪里去、做什么事、赚了怎么分、亏了怎么办，全写进契书里，签字画押。"老钱应了，两人在油灯下拟好文契，把本金、用途、分账法子、酬劳都写明白，老钱按了手印，把一袋银子交给沈川。这是沈川接下的第一桩代客打理的活。

没过多久，镇东头的王寡妇也托邻居找上门来，把亡夫留下的一百两银子托付给他。沈川对着两只钱袋发了愁——这是两户人家的钱，绝不能混作一堆花。他连夜支起一面账墙，左边贴着"老钱户"，右边贴着"王氏户"，进出账目用不同的墨笔分开记。妻子端夜宵进来，劝他"都是钱，何必分"，沈川摇头："同一桩事里头，只要钱是不同东家的，就得分得清清楚楚，混了就要出事。"他把这套分账的法子刻在心里——往后每一户托付来的银子，都单独开户、单独走账，盈亏各归各家。

秋去冬来，沈川替两家人各投了几桩小生意：王寡妇那笔钱买了一处铺面，等着长租；老钱的钱则借给一个想扩产的小染坊，月月收息。可到了年底，老钱的染坊却遇了场火，烧得精光，本钱打了水漂。老钱怒气冲冲找上门来拍桌子："钱是你管的，人是你挑的，赔了算谁的？"沈川没急，只把年初签下的契书摊开，一行一行指给他看：生意有赚有赔，托付之人只能按事先约定承担风险。"我能做的，是把每一步替您想清楚、把每一笔记明白、把能防的坑都替您挡住。可生意场上的赢亏，谁也不敢打包票。"老钱看着白纸黑字，闷了半晌，丢下一句"那明年怎么办"，摔门走了。

沈川没追。开春他做了两件事：一是把这一年的进出流水、每一笔去向，原原本本誊了一份给老钱，让他心里有数；二是按契书的约定，从赚来的收益里扣下一笔自己应得的酬劳，剩下的连本带利送到老钱家里。"受人之托，忠人之事"——这是沈川从小听父亲念叨的话。他想，自己拿的是那份操心费，担的却是整副担子：要把钱聚起来、要把钱管起来、要把赚来的钱分得公平、要让东家随时看得明白账目，少一样，这活儿就算没做透。

后来镇上找沈川托付银钱的人越来越多，他在自家堂屋里挂了块木牌，上头只写六个字："受托、勤勉、忠实"。他跟妻子说，前两个字是提醒自己——既然吃了这碗饭，就要对每一户东家尽心尽力、看得远、想得细；后两个字则是守住一条线：东家的钱就是东家的，绝不能挪作他用，更不能揣进自己腰包。

---

**📖 原文定义**

> 股权投资基金管理人是基金产品的募集者和管理者，其主要职责是按照基金合同的约定，负责基金财产的投资运作，在有效控制风险的基础上为基金投资者争取最大的投资收益。
>
> 基金管理人在基金运作中具有核心作用，基金产品的设计、基金份额的销售与备案、基金财产的管理等重要职能一般均由基金管理人或基金管理人选定的市场服务机构承担。
>
> 基金管理人受托管理基金财产，应勤勉、忠实并且审慎尽职地履行受托责任，并有权根据约定收取相应报酬。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 沈川与老钱签下文契后才收下银子 | 基金管理人按基金合同约定受托管理 |
| 沈川把钱"聚起来"的过程（两户人家先后托付） | 募集者职责 |
| 沈川为老钱挑染坊、为王寡妇选铺面 | 管理者职责：负责投资运作 |
| 沈川坚持"两户分账、不同墨笔" | 基金财产分别管理、分别记账 |
| 染坊失火后沈川按契书分摊赢亏 | 有效控制风险、按合同承担风险 |
| 沈川把账目誊抄给老钱、收益分账后送回 | 收益分配方案、向投资者分配收益 |
| 沈川从收益中扣下自己的酬劳 | 有权根据约定收取相应报酬 |
| 木牌上的"受托、勤勉、忠实" | 勤勉、忠实、审慎尽职的受托责任 |
