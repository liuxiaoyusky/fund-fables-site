# （一）基金投资者

## 🍵 寓言故事 —— 《出钱的那一位》

钱塘江边，三十出头的陈守拙守着父亲留下的绸缎铺，愁得厉害——行情连跌，伙计的工钱都快发不出。

半个月前，茶楼里他遇见了余敬堂。余先生说自己专替人寻好生意，正攒一桩：几家老茶号合股包下西湖边一片早春茶园，三年做下来可观的利。守拙听过"出钱与人做事、赚了分、亏了算"的路子，可在他看来，那是大户人家才玩得起的把戏。

第二回碰面，余先生带他去看山坳里的茶园，把厚厚的账册摊开。守拙一眼瞥见白纸黑字列着二十来位出资人的名字，每位名下注着几两到几十两不等的银数，有一栏专写着"分利之据"。

"我要出多少？"守拙问。

"看你。"余先生说，"出十两有十两的份额，出五十两有五十两的份额。日后茶卖出去，刨去开销，按你这个份额来分；若是茶价垮了，亏也按这个份额来摊。"

守拙还是没点头。他愁的是另一件事：钱一旦交出去，那箱银子就不再姓陈了。账归余先生管，事归余先生跑，连茶树今天抽了几根新芽他都看不见。

嫂子端来一碗莲子羹，听完他叹气，说："你愁的，是怕人拿了钱不办事。"

"对。"

"那这片茶园眼下做的人是谁？"

"余先生和他请的师傅。"

"你自己的手艺呢？"

守拙笑了："我只会看账。"

"那正好。"嫂子说，"你出钱，他出力；你不懂种茶就别去种茶；你只管一件事——这桩事是不是照说好的规矩办的。规矩写明白了，账查得到，茶价能打听得到，那就行了。"

第三日他去了，立了字据，递上三十两。余先生让账房把他的名字添在名册上，旁边注上份额，再递给他一张小纸——那是他的份额凭据。

头一年春茶卖得好，余先生把账理成册，挨家分到各人手中，守拙按份额稳稳收回了银钱。第二年雨水多、茶价低，余先生如实报账，守拙份额里那一部分也跟着薄了。他拿着账册一项一项对——肥料价、人工价、当季行价——都对得上。

第三年开春，余先生又来寻他。守拙把那张凭据推了推，问："这纸，我转给我表弟行不行？"

余先生说："凭据在你手里，份额就是你的。你愿让给谁，让给谁；不愿让，谁也不能动。"

守拙点了点头，又问："那万一哪天我跟你议不到一处，想把大家都叫到一起说理呢？"

"凭据上写明了，份额达到一定数的人，可提请召开大伙的会。大事小事，都得大家坐下来议。"

守拙没再说话，把那张凭据折好，揣进贴身的内袋里。

他忽然明白：他出的是银，担的是赚与亏；可这片茶园从头到尾都不是他的。茶园归这二十来位出资人按份共有，他只是其中一位——他出了钱，记下了份额，承担了盈亏，也享有了那张凭据上所写的一切。

---

**📖 原文定义**

> 股权投资基金投资者，也称为基金份额持有人，是股权投资基金的出资人、基金财产的所有者，按其所持有的基金份额享受收益并承担风险。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 陈守拙递出三十两 | 投资者作为出资人——将自有资金投入 |
| 余先生请师傅种茶、管账、跑销路 | 受托管理资金的办事人（与投资者角色区分） |
| 名册上"陈守拙"旁注的份额 | 基金份额——按出资金额确定持有比例 |
| 余先生递来的那张小纸凭据 | 基金份额持有人身份的凭证 |
| 头一年分利、第二年按同样份额承担薄收 | 按所持份额享受收益并承担风险 |
| 守拙把凭据转给表弟 | 依法转让所持份额 |
| "凑够份额可提请召开大伙的会" | 持有人的会议召集权与表决权 |
| 账册可查、茶价能打听、规矩事先写明 | 知情权与监督权——所有权人的根本保障 |
| 守拙"出钱不种茶、凭据替他说话" | 出资人身份的本质：财产所有者，而非经营者 |
