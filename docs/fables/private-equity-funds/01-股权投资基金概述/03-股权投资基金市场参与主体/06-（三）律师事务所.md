# （三）律师事务所

## ⚖️ 寓言故事 —— 《沈律的随行》

沈明远是城里一家小律所的合伙人，做了十年民商事的案子，小有名气，却从没碰过"基金"这种新鲜事物。这天，他接到一个电话，是老熟人赵哥打来的——赵哥去年下海做了一家管钱公司的老板，攒了一帮出资人，正打算把钱投向一家做新能源电池的工厂。电话那头，赵哥说得又急又乱："老沈，你得帮我，从头到尾都得帮我。"

沈明远刚想开口问"帮你什么"，电话已经被挂断了。第二天，他拎着公文包站到赵哥公司门口。会议室里坐了七八个陌生人，有管钱的、有出钱的人代表、有会计，还有两位西装革履的"金主"——一家是省里的母基金，一家是某大厂的战略投资部。赵哥把他拉到一边，压低声音："老沈，这帮人规矩多，咱们这边要立什么规矩、要审什么、要签什么、最后怎么把钱拿回来……你给我一条龙包圆了。"

沈明远心里咯噔一下。他过去接案子，从来都是"打一场官司"或"审一份合同"，时间短、任务单一。这一回，赵哥让他"从头帮到尾"——钱还没进来呢，就要开始干活；等钱进来了，要跟着走完每一笔钱投向哪里；投完了，要看着被投的公司别出事；等到要收账的那一天，还得帮着把账算清楚、把收益分明白。这不是一桩案子，这是一段很长的路。

他本想推辞。但赵哥是发小，这事又实在没人能做——那些做诉讼的同行只懂打官司，做上市的同行只懂报材料，公司里那些刚毕业的法务专员只够审审合同。一家从零起步的管钱公司，身边缺一个能跟着走完全程的人。沈明远咬咬牙，答应了。

接下来的几个月，沈明远像赵哥那帮管钱人的影子。第一步，他帮赵哥把这家管钱公司"立起来"——组织形式怎么选、内部谁说了算、出钱的人和管钱的人怎么分权，这些都要落到白纸黑字的章程里。钱还没进来，他已经熬了十几个通宵，草拟、修改、再草拟，把一份份枯燥的文件写到滴水不漏。赵哥那边出资人五花八门，有钱的有势的，谁都觉得自己该多拿点。沈明远一份一份地审他们的资质，把不合格的人悄悄挡在门外——他懂这行的规矩，知道一旦让不合格的人混进来，后患无穷。

第二步，钱进来了，要投出去。沈明远跟着赵哥的车，颠簸四个小时到了那家电池厂。厂子大、设备新、老板豪爽，但沈明远没心思看这些——他翻账本、查产权、核合同，一项一项地查这家厂子"干不干净"。原来这家厂的创始人私下有一笔说不清的借款，厂房的地还做过二次抵押。沈明远把这些查了个底朝天，写成厚厚的报告交给赵哥。赵哥看完脸色变了："老沈，要不是你查出来，我这就签字了。"沈明远没说话，只把一条条建议列在后面——哪条款要改、哪条要加、哪条要剔除。后来那份合同的定稿，是沈明远和对方律师在会议室里一条条磨出来的。

投完之后，沈明远没撤。电池厂开业那天他去了，半年后换股东他也去了。赵哥说："你怎么老跟着？"沈明远说："钱还在里头呢。"他帮赵哥盯着这家厂里的大事小情，股东变更、对外担保、关联交易、章程修改——每一件涉及那笔钱的事，他都要看一眼、出个意见。有一回电池厂要上新三板，沈明远又陪着熬了三个通宵，把历史沿革里的几个坑一一填平。

最后是退出那天。电池厂被一家上市公司收购，赵哥这边的钱要套现离场。沈明远翻开当初签下的合同，一条条核对退出方式、税务安排、款项结算。谈判桌上，他坐在赵哥身旁，一字一句地掰扯，对方律师说的每一句话他都认真听，认真记。三个月后，第一笔回款到账。

清算那天，赵哥请沈明远喝酒。几杯下肚，赵哥忽然问："老沈，你跟了我这几年，从立规矩到最后一笔账，全是你在帮。我付给你的那点律师费，够本吗？"沈明远想了想，说："赵哥，我这行不靠一锤子买卖。你这一单做下来，下回那个母基金还会找我，那个大厂也会介绍别人来。我不是卖一个案子给你，我是陪你走了一段路。"

窗外夜色沉沉，会议室的灯还亮着，下一场"从零开始"的会议正在等他。

**📖 原文定义**

> 股权投资基金在募集与设立、投资、投后管理、项目退出及基金清算等各个阶段，皆会涉及诸多法律问题，因此，基金管理人通常会为基金聘请律师事务所提供专业服务。鉴于基金投资的专业性，投资者亦可视情况聘请独立的法律顾问。
>
> 基金管理人聘请的律师应遵守诚实守信、勤勉尽责及审慎原则，根据相关法律法规，按照基金管理人的委托，在基金运作的各个阶段提供专业优质的法律服务。
>
> 在股权投资基金的募集与设立阶段，律师提供的服务包括：协助基金管理人设计基金的组织形式及内部治理结构；根据基金管理人与基金投资者的商务安排，起草基金法律文件；在基金管理人委托的范围内，对基金投资者的资质进行审核；协助完成基金管理人登记和基金备案工作，并根据需要出具法律意见书。
>
> 在股权投资基金的投资及投后管理阶段，律师提供的服务包括：就基金的投资领域、投资方向的限制向基金管理人提供咨询服务；在初步确定拟投资企业后，律师按照基金管理人的委托，勤勉审慎地对拟投资企业进行法律尽职调查，提交法律尽职调查报告或法律意见书，协助基金管理人分析投资涉及的法律问题和风险；协助基金管理人起草或审阅与基金投资有关的法律文件；在投后管理阶段，按照基金管理人的委托，根据投资法律文件的约定，保护基金在被投资企业中的合法权益。
>
> 在股权投资基金的项目退出阶段，律师提供的服务包括：按照基金管理人的委托，研究基金投资的退出结构及方式；根据不同退出方式的相关法律规定，起草法律文件，参与谈判，协助基金管理人最大限度地获取合法投资收益。
>
> 在股权投资基金的清算阶段，律师提供的服务包括：按照法律规定和基金合同的约定，协助确定清算主体；协助清算主体制订清算方案；协助实施清算方案，包括通知债权人、确认基金财产、分配基金财产；对清算人出具的清算报告进行合规性审核等。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 赵哥让沈明远"从头帮到尾"的电话 | 律师事务所在基金运作各阶段提供专业服务 |
| 沈明远把公司"立起来"、起草章程、审出资人资质 | 募集与设立阶段：设计组织形式与治理结构、起草法律文件、审核出资人资质 |
| 沈明远跟着去电池厂翻账本、查产权、核合同、写报告 | 投资阶段：对拟投资企业进行法律尽职调查、出具报告 |
| 沈明远帮赵哥盯着厂里的股东变更、担保、关联交易 | 投后管理阶段：保护钱在被投企业中的合法权益 |
| 沈明远坐在谈判桌旁核对退出方式、税务、结算 | 项目退出阶段：研究退出结构、起草法律文件、参与谈判 |
| 清算日沈明远仍陪着核对账目 | 清算阶段：协助制订和实施清算方案、分配财产 |
| 沈明远说"我不是卖一个案子，是陪你走了一段路" | 律师提供"全程一条龙"服务，而非单点零散服务 |
