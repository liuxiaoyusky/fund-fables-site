# （五）估值核算服务机构

## 🔍 寓言故事 —— 《老孙的账房》

民国二十三年冬，杭州城南绸缎庄老板钱守成急得团团转。年初他和江南几位布商合股做了一笔买卖：十二人凑钱，由钱守成操盘，到苏州收丝、湖州收缎。临近年关，存货还压着三成，合伙人来信催着"看看账"。钱守成自己算一遍，又让账房林叔算一遍——两本一对，差了四十块大洋。

北平孙老板连发三封电报："账对不上，年终怎么分？分红怎么定？"钱守成额头冒汗。他过去开绸缎庄，进货出货靠林叔一支笔、算盘一拨，那叫"流水账"。可如今十二个合伙人凑的钱，是一个"盘子"；货有涨有跌，钱有进有出，还要分清每个人占多少、赚了该怎么分。要把这本账算得让十二人都服气，他没这个本事。

钱守成想了三天，去上海请来陈秉文——洋行里做了十几年账的老手。饭桌上他说完苦处，陈秉文竖起三根指头："这账我能算，但有三条得先说清。"

"第一，"陈秉文说，"记账得照规矩来——账簿有账簿的样式，科目有科目的归类，存货怎么折钱、欠条怎么算、没卖出去的货按什么价入账，都得按通行的章法办。不能您说值多少就值多少。"

"第二，"他又伸出一根指头，"盘子里头的东西，我得按时点数清楚。一年点一次太粗，合伙人也等不及。每个月一小结，每季一大结，账目清清楚楚贴到柜上，让每个合伙人都看得见。"

"第三，"陈秉文放下酒杯，"年底您还得请一位先生来查账。他不是我，也不是林叔，是专门替人挑账毛病的。查账的人不是我请的，是您和十二位合伙人一起点头才请得动；查账的结果，也得让每个合伙人都晓得。"

钱守成想了想，点了头。从那天起，陈秉文在绸缎庄偏厢房支了一张大案，每月初一、十五按时盘货结账。库存的生丝按当月苏州牌价折算，缎子按湖州牌价折算；应收账款一笔笔核，坏账按规矩剔除。每月结完，他抄成十二份小册子寄给北平、天津、广州的合伙人。孙老板回电说"清楚了"，张老板也认了。

腊月二十那天，钱守成又把姓吴的老先生请来。戴圆框眼镜的老先生把陈秉文一年的账册抱到客房翻了五天，一笔一笔勾，一行一行算，最后在《账目核验单》上盖了红印："账目公允，并无舛误。"钱守成抄了十二份，连同年终分红一起寄出，合伙人都点了头。

年夜饭上，孙老板专门敬了陈秉文一碗黄酒。席散送客时，钱守成忽然问："陈先生，我过去开绸缎庄自己就能记账，如今不过多了几个合伙人，怎么就得专门请人算？"陈秉文笑了笑："钱老板，您过去算的是一家的账，进多少卖多少自己心里有数。如今这个盘子，钱是大家的，货是大家托您看着的，账就得让大家都认。这账不算清楚，年年都得吵架。"

街角的风灯晃了晃，钱守成站在门口，望了望偏厢房那盏还没熄的灯。

---

**📖 原文定义**

> 估值核算服务机构的基本职责包括：开展基金估值核算，配合基金管理人聘请的会计师事务所进行审计，以及法律法规及服务协议规定的其他职责。估值核算服务机构开展估值核算服务，应当遵守《企业会计准则》以及中国证券投资基金业协会的估值指引等相关法律法规与自律规则的规定。估值核算服务机构应当按照基金合同和服务协议规定的估值方法、估值频率、估值流程对基金财产进行估值核算。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 钱守成自己算账与林叔对不上、合伙人不认 | 自行核算难以满足多个出资方的信任与合规要求 |
| 钱守成专程去上海请陈秉文这位"洋行老手" | 引入估值核算服务机构 |
| 陈秉文强调"记账得按通行的章法办" | 遵守《企业会计准则》及协会的估值指引等法律法规与自律规则 |
| 陈秉文要求"每月一小结、每季一大结" | 按照服务协议规定的估值频率与估值流程开展核算 |
| 陈秉文按苏州、湖州牌价折算存货并剔除坏账 | 按照基金合同与服务协议规定的估值方法核算 |
| 陈秉文把账目抄成十二份寄给各地合伙人 | 估值核算结果需向相关方披露，保持透明可查 |
| 年底另请吴老先生翻账册、盖章"账目公允" | 配合基金管理人聘请的会计师事务所进行审计 |
| 吴老先生由钱守成与十二位合伙人共同点头才请得动 | 审计机构由相关方共同认可、保持独立性 |
