# （二）行业自律管理组织

## 🏛️ 寓言故事 —— 《沙湾那座会馆》

闽南沙湾港，至今还爱讲一桩老事。

那年港口扩建，海外货轮一艘接一艘涌进来，把这个原本只有几十条小渔船的小镇，撑成了一条金灿灿的海上门户。岸上陆陆续续冒出二十多家"船行"——有人出资、有人管事、有人装船发货，从南洋运香料回来，往北边运丝绸出去。生意越做越大，可码头秩序也一天比一天乱：有人抢别人家说好的货、有人以次充好糊弄出钱的东家、有人卷了银子跑路。出资人闹到码头哭天抢地，告状的、吵架的、卷铺盖走人的，把整个港口搅得乌烟瘴气。

事情闹到府衙，陈知府翻着案卷，每一桩都是鸡毛蒜皮——谁占了泊位、谁赖了货款——可桩桩都堆到衙门来。他把幕僚老吴叫来："你去码头走一趟，看看能不能让船行自己立个规矩出来。我这边管不过来，也不该什么都管。"

老吴换上便服，蹲在码头茶馆里听了三天。他找上了镇上最老资格、也最讲信义的船行东家林伯。林伯在这行沉浮四十年，从扛货小工一路做到东家，是公认的"老大"。老吴把知府的话一五一十转述："希望你们自己人管自己人，别什么都惊动官府。"

林伯听完沉默很久。他心里清楚，单靠官府的板子不够，可"自己人管自己人"——到底怎么管？管什么？由谁来管？他拿不准。

第二天他请各家管事喝茶，刚把"立会馆"的意思说出口，做丝绸出口的洪老板就冷笑一声："凭什么要我来守你的规矩？你得先说清楚——谁定的？定的什么？我犯了怎么办？"一桌子人七嘴八舌，吵得林伯脑仁疼。

老吴看出门道。他把"政府"和"会馆"掰开讲："官府管的是大规矩——犯法的事归他办。可同行之间怎么约账、怎么分钱、出了纠纷找谁说理，这是你们自家的事。官府是山上的灯塔，照大方向；你们自己定的规矩，是港湾里的航标，管船与船之间别撞上。"林伯听完眼睛亮了一下。

他重新把大家请回来，把丑话讲在前头：会馆由各家自愿加入，愿走愿留不强求；会馆只管行内的事，刑名钱粮仍归官府；规则由大家共同议定；账目公开。名字起得朴素——"沙湾船行公所"。

公所一立，麻烦跟着来。开张第三个月，做南洋线的小周撞了红线：他用劣质香料换上好货的包装，发给南洋老主顾。主家告到公所，林伯按事先议定的规矩，当着所有同行的面，让小周原路退回劣货、赔足主家损失，并把这件事记入"灰档"。若三年内再犯，扫地出门。小周低头认罚，再没犯过。

岸上的乱子一件件少下去。公所每年办几次"碰头会"，让各家管事坐一桌，把账本摆开说说，把分歧摆到台面聊；哪家有了好经验，公所请他上台讲一段；年轻管事想入行，公所办"夜课"教基础本事。日子久了，岸上的人开始说"沙湾港的船行，靠得住"。

这名声传到了府衙。岸上纠纷一年少了一半，陈知府终于从鸡毛蒜皮里抽出身来。他把公所当作朝廷和民间的"中间人"——朝廷新政策，先送到公所；岸上难处，公所再递到衙门。有一回朝廷下了新的出海登记规矩，衙门把条文递给公所，公所召集各家逐字讲解，再把大伙儿的疑问汇总回去——新规落得快，怨言也少。

几年下来，岸上的人都服了公所。出资的东家、跑船的管事、岸边的伙计，三教九流都认"有事找公所"。有个别不服的想去官府告公所"管太多"——可官府翻出当年的定规，明白写着"会馆只管行内事"，驳回了。

老吴再回沙湾港那天，林伯正坐在公所新修的厅堂里喝茶。窗外码头一派井然，老吴问："林兄，这公所现在还差什么？"林伯想了想说："什么都不差——岸上有人管，出钱的人安心，衙门也省心，这就是咱们这行自己的'家'。"

海风从窗口吹进来，厅堂里那块老匾在夕阳下泛着光——"沙湾船行公所"五个字，已经被海风磨得发亮。

**📖 原文定义**

> 股权投资基金行业自律，是指行业自律组织对股权投资基金市场、基金市场主体及其活动的监督和约束。股权投资基金行业自律组织一般采取行业协会形式，由基金管理人和其他基金服务机构自愿结成。
>
> 股权投资基金行业自律既是行业交易规则的自我制定过程，也是与政府监管相并行的一种市场治理手段。行业自律是在法律规定和政府监管的基础上，行业成员联合设定行业规则的活动。行业规则包括信息披露、利益冲突、内部治理和运营、行业行为守则等。行业自律组织不仅要创建行业规则，也应监督行业对规则的遵守和行业内部成员对规则的执行。
>
> 股权投资基金行业自律与政府监管既有联系又有区别，两者之间存在着相互补充、相互促进的关系。
>
> 第一，目的一致，都是为了确保国家有关股权投资基金的相关法律、法规、规章和政策得到贯彻执行，保护投资者的合法权益。
>
> 第二，行业自律是对政府监管的积极补充。
>
> 第三，行业自律组织在政府监管机构和市场主体之间起着桥梁和纽带作用。
>
> 性质不同。政府监管带有行政管理的性质，行业自律是行业的自我约束。
>
> 依据不同。政府监管依据国家的有关法律、法规、规章和政策；行业自律除了依据国家的有关法律、法规、规章和政策外，还依据行业自律组织制定的章程、业务规则和细则。
>
> 对违法违规者的处罚不同。政府监管机构可以对违法违规的市场主体采取责令改正、警告、罚款等行政监管措施以及市场禁入措施；行业自律组织可以对自律管理对象采取公开谴责、暂停会员资格、取消会员资格等自律措施。
>
> 股权投资基金行业自律的重要性，由政府有限监管和行业整体利益决定。
>
> 具体来说，股权投资基金行业自律组织可以在以下方面起到作用：第一，提供行业服务，促进行业交流和创新，提升行业职业素质，提高行业竞争力；第二，发挥行业与政府间桥梁与纽带作用，维护行业合法权益，促进公众对行业的理解，提升行业声誉；第三，履行行业自律管理，促进行业合规经营，维护行业的正当经营秩序；第四，促进会员忠实履行受托义务和社会责任，推动行业持续稳定健康发展。
>
> 中国证券投资基金业协会是我国股权投资基金行业的全国性自律组织。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 陈知府让老吴去码头"让船行自己立规矩" | 政府对市场的有限监管，需要行业组织分担治理 |
| 老吴把"灯塔"和"航标"分开讲 | 行业自律与政府监管性质不同：政府管大规矩，行业管同行之间的约定 |
| 林伯把"愿走愿留不强求"等铁规写进公所定规 | 行业自律组织由各方自愿结成，依据章程、业务规则、细则行事 |
| 小周以次充好被公所当众处罚、记灰档 | 行业自律组织监督规则遵守与执行，可采取内部惩戒措施 |
| 公所办"碰头会"、请人讲经验、办"夜课" | 行业自律的第一项作用：提供行业服务，促进行业交流和创新，提升职业素质与竞争力 |
| 朝廷新规由公所先收、再讲解、再把疑问递回衙门 | 行业自律组织的第二项作用：在政府与市场主体之间发挥桥梁和纽带作用 |
| 出资东家、跑船管事、岸边伙计"有事找公所" | 行业自律维护行业合法权益，提升行业声誉 |
| 有人去官府告公所"管太多"被驳回 | 行业自律与政府监管的边界：会馆只管行内事，刑名钱粮仍归官府 |
| 林伯说"岸上有人管、银子安稳、衙门省心" | 行业自律的第三、四项作用：促进行业合规经营、推动行业持续稳定健康发展 |
