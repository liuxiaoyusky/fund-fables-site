# （三）成熟企业投资基金的投后管理策略

## 🏺 寓言故事 —— 《老钱的两条路》

老钱是个做瓷器生意的老手，三十年前从一间小作坊起家，如今他的"青瓷坊"已是镇上最大的瓷器商号，有三十多个伙计，生意遍布三个省。去年，老钱把多年攒下的家底投进了两个老朋友的买卖里——一个是做茶叶批发的周掌柜，一个是做丝绸生意的孙东家。这两人都是老钱看着一步步做起来的，买卖早就上了轨道，账面上的流水比老钱自己的瓷器铺还稳当。

老钱投钱进去的时候，周掌柜拍着胸脯说："明年开春，我就去府衙挂牌，把茶叶行变成'公股'，到时候你的份子能翻三倍。"孙东家也点头："我的丝绸庄也一样，后年准挂牌。"老钱听了，心里踏实，回家把契纸收进铁盒，等着数日子。可日子一天天过去，周掌柜的茶叶行倒是越开越大，新开了三家分号，可每次老钱问起挂牌的事，周掌柜就摆手："再等等，再等等，今年茶税有变动，不是好时机。"孙东家的丝绸庄更麻烦——南边打仗，生丝断了来路，铺子关了半间，账房先生也跑了两个。

老钱坐不住了。他把两个老伙计分别请到家里喝酒，想听听实话。周掌柜那顿酒喝得很闷，三杯下肚才说实话："老钱，挂牌的事怕是悬了。衙门里换了新规矩，咱们茶叶行的账目有几笔早年不清不楚，查出来怕是要吃官司。"老钱听完，半宿没睡着。第二天孙东家来了，更是愁眉苦脸："生丝断了，铺子撑不过今年。老钱，你要是想撤，我把城南那两间仓库抵给你，值多少算多少。"

老钱在院子里转了一下午。他想，周掌柜那边，买卖本身没坏，就是挂牌的路走不通了。与其干等着，不如帮他另找出路——周掌柜的茶叶行口碑好、客源稳，镇西头做药材的刘大官人一直想 diversifying 买卖，若是撮合他们两家合并，自己那份子换成刘大官人商号里的股，虽说不挂牌了，但刘家根基深，未必不是条好路。至于孙东家，仓库抵账是杯水车薪，老钱索性请了个讼师，把早年签的契纸翻出来，里头有一条写着"若买卖连续三年亏损，原出资人有权按原价收回本钱"。孙东家起初不认，老钱也不急，让讼师一封一封地写状子，最终孙东家怕了官司，咬着牙把本钱退了回来。

三年后，周掌柜的茶叶行并进了刘大官人的药材商号，老钱那份子变成了刘家的"干股"，每年分红比当初预期的挂牌收益还稳。孙东家退了本钱之后，丝绸庄果然没撑过那年冬天，债主上门把他的仓库和宅子都收了去。老钱坐在自己的青瓷坊里，看着账本上两笔截然不同的结局，心里明白了一件事：把钱投给已经做大的买卖，不能光等着人家自己走到终点。该推的时候得推，该换路的时候得换，该拿法律说话的时候也不能手软。成熟的庄稼地，更需要好农夫。

---

**📖 原文定义**

> 成熟企业投资基金主要投资于扩张期、成熟期的企业，其投后管理模式最为丰富，内容上也更加全面。扩张期企业虽然已较为成熟，但在发展过程中存在各种问题，如何改善这些问题进一步优化企业治理模式也是提升企业实力的关键。因此，成熟企业投资基金倾向于配备更加专业的投后管理团队，在战略、财务、管理、运营、资源整合、资本运作等各个方面为被投资企业提供增值服务。需要重点关注的是，该类项目一般都是成熟的中大型企业，商业模式已得到市场验证，管理模式也相对来说较成熟，有较为明确的IPO时间表（预计一到两个会计年度内申报）。因此，基金管理人主要聚焦在企业顺利IPO的相关问题：对符合预期的被投资企业，主要是推动其上市，协助筹备上市相关活动，在资本市场领域给企业提供附加值；对不达预期的被投资企业，如IPO申报不顺利甚至申报无望，则需要考虑调整策略，重点加强投后管理力量，既可争取通过产业并购等方式实现转让退出，亦要考虑商业谈判或司法途径（比如诉讼仲裁）等方式争取回购退出。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老钱投资周掌柜、孙东家两个已做大的买卖 | 成熟企业投资基金主要投资于扩张期、成熟期的企业 |
| 老钱分别请两人喝酒、了解实情 | 配备更加专业的投后管理团队，在战略、财务、管理等各方面提供增值服务 |
| 周掌柜有明确的"挂牌"计划 | 成熟企业有较为明确的IPO时间表 |
| 老钱帮周掌柜撮合与刘大官人合并 | 对符合预期的企业，推动上市，或在资本市场领域提供附加值；不达预期时争取通过产业并购等方式实现转让退出 |
| 老钱请讼师翻出契纸、打官司收回本钱 | 对不达预期的企业，考虑商业谈判或司法途径（比如诉讼仲裁）等方式争取回购退出 |
| 老钱对两家采取完全不同的策略 | 投后管理模式最为丰富，内容上也更加全面 |
