# （五）母基金的投后管理策略

## 🏺 寓言故事 —— 《老周和十二个掌柜》

老周在江南一带经营钱庄已有三十年，是出了名的精明人。年过六旬后，他不再亲自去各地放贷，而是想了个新法子：把银子交给十二个年轻掌柜，让他们分头去不同行当经营。老周只管坐镇总号，定期查账，偶尔走动走动。这法子叫"放钱养人"——自己不亲力亲为，却让银子在十二个掌柜手里生钱。

起初两年，一切顺当。老周每季收一次账簿，年底聚十二个掌柜吃顿饭，听听各处的动静。可到了第三年，麻烦来了。两个掌柜接连出事：一个把钱投进了不靠谱的绸缎行，亏了大半；另一个更离谱，竟把银子挪去赌坊周转。老周气得拍桌子，但冷静下来一想，问题出在自己身上——他管得太松了。

老周决定改规矩。他不再只是"收账簿、吃顿饭"，而是做了三件实事。第一，每月派亲信去各掌柜处走动，不是查账，而是看人心——看掌柜气色如何、伙计是否齐心、门面是否景气。第二，他让各掌柜每月写一封亲笔信，不只报盈亏，还要写打算、写难处、写下一步想往哪走。第三，他把十二个掌柜按成色分了类：踏实肯干的，多给银子、多介绍好门路；心思活络但方向不明的，亲自点拨，甚至把自己认识的可靠买卖人引荐过去；那两个出过事的，不再追加银子，只盯着他们先把窟窿补上，必要时亲自出面替他们收拾残局。

又过了三年，老周的钱庄比从前更兴旺。那十二个掌柜里，有五个成了独当一面的大掌柜，有三人经老周引荐搭上了更好的买卖线，连那两个出过事的也翻了身。老周渐渐明白了一个道理：把钱撒出去只是开头，真正的本事在于——钱出去之后，眼睛不能离开，手不能闲着，心不能放下。要让人定期报信，要亲自走动看实情，要分门别类地对待不同的人，还要在关键时候伸手帮一把。银子不会自己长腿跑回来，得有人一路盯着、一路扶着、一路引着。

---

**📖 原文定义**

> 鉴于母基金投资策略包括 "子基金+二手份额+直接投资"，除直接投资对项目进行投后管理外，子基金和二手份额的投资均涉及母基金对已投资的多只子基金的投后管理，可分为日常性管理和帮助被投子基金增值两个部分。
> 母基金对子基金的日常性管理主要包含报告获取、沟通交流与运营监督三部分。
> 根据基金合同的约定，母基金可定期（季度、半年或年度）获取子基金运营报告，以跟踪其发展。同时，母基金通过参加年度合伙人会议、电话、邮件、定期走访等方式，与子基金保持日常沟通，掌握运营动态、团队变化和未来计划。在运营监督方面，母基金会密切关注子基金运营，一旦发现违规立即纠正，情节严重则依法处理。鉴于信息不对称易引发风险，母基金应督促子基金管理人履职，必要时可主张权利甚至诉讼。前述日常性管理的严格执行，有利于有效监督子基金运营，保障母基金的合法权益。
> 帮助被投子基金增值是近期越来越多的母基金管理人日益重视的部分，包括但不限于：推荐适配潜在投资者，推荐优质投资标的，提供政策解读等信息以协助合规运营，提供科学的退出建议和对接优质中介机构等。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周把钱交给十二个掌柜经营 | 母基金投资多只子基金 |
| 老周每季收账簿、年底聚会 | 报告获取、参加年度合伙人会议 |
| 每月派亲信走动、掌柜每月写亲笔信 | 定期走访、电话邮件沟通、掌握运营动态 |
| 两个掌柜出事，老周派人盯着、必要时收拾残局 | 运营监督、发现违规立即纠正、情节严重依法处理、主张权利 |
| 踏实掌柜多给银子、引荐好门路 | 帮助被投子基金增值、推荐优质投资标的 |
| 给方向不明掌柜引荐可靠买卖人 | 推荐适配潜在投资者、对接优质中介机构 |
| 按掌柜成色分类对待 | 日常性管理与增值服务的结合 |
| 老周明白钱出去后眼睛不能离开 | 日常性管理严格执行以保障母基金合法权益 |
