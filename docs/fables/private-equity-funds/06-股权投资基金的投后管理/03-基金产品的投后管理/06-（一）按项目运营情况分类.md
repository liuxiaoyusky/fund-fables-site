# （一）按项目运营情况分类

## 🔍 寓言故事 —— 《老周的菜园》


老周在城郊经营着一片菜园，二十年来，他种出的菜远近闻名。去年冬天，镇上的商会发起了一个新计划：由老周牵头，联合镇上七户人家，共同出资包下城郊五十亩荒地，按各家投入比例分地耕种，三年后按收成比例分红。老周被推选为总负责人，大家都叫他"周掌柜"。

开春后，五十亩地分成七块，由七户人家各自经营。老周的任务不是亲自下地，而是定期巡视，看哪块地出了问题，哪块地需要帮忙。起初他想："每块地都一样对待，公平。"可没过多久，他就发现这想法行不通。


三月里，老周巡到东头李家的三亩地。李家是种菜的老把式，地里的番茄苗整整齐齐，灌溉渠修得笔直，连杂草都看不见几根。老周站在田埂上看了半天，愣是没找出一个能插嘴的地方。李家的媳妇笑着递来一碗凉茶："周掌柜，您放心，我们心里有数，该施肥该打枝，一步不差。"老周喝完茶，点点头走了。他意识到，这样的地，自己要是硬凑上去指手画脚，反而是添乱。从那以后，老周去李家地里的次数最少，只是每月路过时远远望一眼，偶尔李家主动问起市场行情，他才搭几句话。

可到了西头王家的五亩地，情况完全不同。王家是新手，去年才从外地搬来，种的是老周从没见过的品种——一种据说能卖高价的紫色胡萝卜。地里的苗稀稀拉拉，灌溉也时断时续。老周第一次去看时，王家的年轻人蹲在田边，满脸愁容："周掌柜，这苗怎么越长越黄？"老周蹲下来翻了翻土，又看了看出水口，心里有了数。他回去后，把镇上种过类似作物的孙师傅请来，又帮王家重新规划了灌溉路线，还引荐了镇东头卖优质种子的老赵。那半个月，老周往王家地里跑了七八趟。


最让老周头疼的是北头赵家的四亩地。赵家原本也是老把式，可开春后赵老汉生了场大病，家里只剩一个二十出头的孙子小赵撑着。小赵心高气傲，觉得自己能行，硬是不听劝，把原本该种两季豆的地全改种了一种新引进的果树苗。结果到了五月，树苗因为水土不服，死了大半。小赵急红了眼，偷偷借了钱想补种，又被人骗了，买了假苗。老周知道后，连夜赶过去。小赵蹲在死苗堆里，头埋进膝盖，一声不吭。老周没有骂他，只是坐下来，跟他一起把死苗一棵棵拔出来，重新翻土。接下来的两个月，老周几乎住在了赵家地里，手把手教小赵怎么选苗、怎么施肥、怎么防虫。他还去镇上帮小赵谈了一笔周转钱，解了燃眉之急。


然而，南头张家的两亩地，却是老周最不愿意面对的。张家去年包地时 enthusiasm 最高，可实际上根本没打算好好种。开春后随便撒了些种子，人就进城打工去了，地荒了大半年。老周去了三趟，每次都只见野草长得比苗还高。他托人带话给张家，张家要么推脱说"忙完这阵就回来"，要么干脆不回应。到了秋天，那块地彻底废了，连野草都枯了。老周站在荒地上，风吹起干土，迷了他的眼。他知道，这块地已经没救了。再往里砸人力、砸资源，不过是把更多人的钱扔进无底洞。他回去后，在账本上把这亩地单独圈了出来，开始盘算怎么把地收回来、怎么清算张家欠下的份额，尽量减少大家的损失。

一年过去，老周渐渐摸索出了一套自己的办法。七块地，他不再一视同仁。有的地，他远远看着就行；有的地，他要常去搭把手；有的地，他得蹲下来一起干；还有的地，他得狠心做个了断。这不是偏心，而是他慢慢明白了一个道理：同样是菜园，有的已经能自己运转，有的正需要你拉一把，有的快不行了但你还能救，有的则已经烂到根里，越早止损越好。一个人只有两只手，时间有限，力气有限，只有把心思花在刀刃上，整片五十亩地才能有个好收成。

**📖 原文定义**

> 对于优秀企业，鉴于企业自身发展运转良好、风险较低，其商业模式已经较为成熟，团队较为完善，且具有优质的造血能力，可以自行有效地规划未来路径。处于该类的企业往往知道自己想要什么，也知道自己的问题该如何得以修正，无须投后管理介入日常管理活动。对于该类企业，投后管理人员往往仅需要定期回访了解情况收集数据，并尽可能地针对企业的需求提供资源帮助，投后管理的负责人更多承担着外部顾问角色。
>
> 对于符合预期、一般以及低于预期这三类项目，通过风险监测和增值服务等投后管理服务有机会去实现降低风险和保障投资收益的目标，在实践中是投后管理的重中之重，需要投入大部分的时间精力。若出现企业的商业模式尚未打磨成熟、盈利变现渠道未打通、不具有造血能力或者能力很差、团队结构不够完善稳定等问题，投后管理人员需要有针对性提供支持协助企业解决痛点问题，比如引荐优秀人员入职被投资企业、与企业负责人探讨商业模式、引荐优质的银行信贷机构以及专业服务咨询机构、就运营逻辑和融资规划与企业创始人充分探讨、协助开展新一轮融资活动等。对于低于预期状态持续时间较久的项目，也需要重点关注可能的退出方案安排。
>
> 对于最后一个级别困难类别的项目，鉴于项目发展陷入困境且已无其他有效措施，如继续进行投后管理服务效果甚微，做好针对性的投后监测是重点工作之一；另一重点是合规完成投资退出和项目清算的相关工作，基金管理人应要求投后管理人员工作重点为尽快退出和清算。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 李家的番茄地苗齐渠直，老周只每月远望、偶尔聊行情 | 优秀企业：自身运转良好，仅需定期回访，负责人承担外部顾问角色 |
| 王家紫色胡萝卜苗黄灌溉乱，老周请孙师傅、引老赵、重新规划 | 符合预期/一般/低于预期项目：投入大部分精力，针对性解决痛点 |
| 赵家树苗死、买假苗、小赵撑不住，老周蹲下来手把手教、帮谈周转钱 | 低于预期项目：引荐人员、探讨模式、协助融资，关注退出方案 |
| 张家地彻底荒废、人不回应，老周圈出账本、盘算收回清算 | 困难类别项目：投后监测+合规退出清算，尽快止损 |
| 老周不再一视同仁，而是按每块地的实际状况分配精力 | 按项目运营情况分级管理，差异化安排资源对接和关注跟进 |
