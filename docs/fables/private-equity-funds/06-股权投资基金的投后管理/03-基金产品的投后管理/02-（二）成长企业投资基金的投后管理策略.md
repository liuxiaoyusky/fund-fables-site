# （二）成长企业投资基金的投后管理策略

## 🌱 寓言故事 —— 《青苗与园丁》


江南有个叫周远行的木匠，手艺好，眼光也毒。二十年前，他攒下了一笔钱，不盖房不置地，专挑镇上那些有手艺但缺门路的年轻人，借钱给他们开铺子。他不要利息，只要铺子做大以后分他一份。镇上人笑他傻，说这些毛头小子十有八九要赔光。

周远行不理会。他看中的第一个年轻人叫林小树，在街角租了间漏雨的棚子做竹器。林小树手艺没得说，编的竹篮又轻又牢，可他不会算账，不会招揽客人，更不知道去哪买便宜的竹子。铺子开了半年，连房租都交不起。周远行没有撤资走人，反而三天两头往棚子里跑。他帮林小树联系上上游的竹农，砍掉了中间商；又教他在篮底压一道暗纹，做出自家招牌；还引荐了一个会算账的远房侄子来当掌柜。林小树的铺子活了下来，第三年换了正经门面，第五年雇了十二个伙计。

周远行并不满足。他陆续又投了六家铺子，有做酱菜的、有织布的、有打铁的。他给自己定了个规矩：投出去的每一笔钱，不能扔下不管，要像种树一样，浇水、修枝、除虫，看着它长。他专门雇了两个人，每天在各铺子之间走动，谁家缺人手、谁家得罪了主顾、谁家想开分号缺本钱，这两个人都要记下来，回来跟他商量。

有一年，镇上绸缎庄的老掌柜突然病故，他儿子仓促接手，把一批货色看走了眼，赔了一大笔。周远行投的那家织布铺子正给绸缎庄供货，货款收不回来，周转不开，眼看也要跟着倒。周远行连夜赶到织布铺子，没有撤资，而是把绸缎庄的债务接了过来，亲自出面跟老掌柜的儿子谈判，用分期三年、以货抵债的方式化解了危机。织布铺子缓过气来，后来成了镇上最大的布行。

又过了几年，周远行早年投的那些铺子，大多成了各行的头面。林小树的竹器铺子开到了省城，酱菜坊的牌子挂到了码头，布行更是养活了几十户人家。周远行每年从各家铺子分到的红利，早已超过当年本钱的几十倍。镇上的嘲笑早已变成了羡慕，有人问他的秘诀，他只是说："我不过是种树的人。树苗刚冒头的时候，风一吹就倒，你得扶着；等它长粗了，你还得修枝，不然长歪了；等到它自己能挡风遮雨了，你才能在树下乘凉。"他顿了顿，又补了一句："最难的，是树苗半大不大的时候。说它小吧，它已经扎了根；说它大吧，一阵急雨还能把它打趴下。这时候最费心思，也最值得费心思。"

---

**📖 原文定义**

> 成长企业投资基金主要投资于成长期企业，该类基金的投后管理体系较早期企业投资基金更加完善，风险监管和增值服务的工作内容都有了升级，投后管理整体工作开展有了更加明确的责任人和更完备的管理体系。从投后管理的内容来看，该类基金管理人一般聚焦在为企业赋能。因为所投资的企业尚处于成长阶段，基金管理人因在发展早期就接触和参与到被投资企业的发展，与企业创始人建立更亲密的信任关系，更易于接受基金管理人的支持和建议，因此在策略和资源方面都能较大程度帮助到企业，包括在技术升级、业务优化、后续融资协助、股权架构设计、管理层激励方案、并购支持以及内部管理等方面均能为被投资企业提供支持。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 周远行雇两个人每天在各铺子间走动记录 | 投后管理体系更加完善，有明确的责任人 |
| 帮林小树联系竹农、设计招牌、引荐掌柜 | 技术升级、业务优化、内部管理支持 |
| 织布铺子周转困难时，周远行接手债务并谈判化解 | 后续融资协助、并购支持 |
| 周远行说"树苗半大不大的时候最费心思" | 成长期企业需要更完备的投后管理体系 |
| 周远行早年接触林小树，建立信任后再提建议 | 发展早期就接触参与，建立亲密信任关系 |
| 周远行从策略和资源方面帮助各铺子 | 在策略和资源方面较大程度帮助企业 |
