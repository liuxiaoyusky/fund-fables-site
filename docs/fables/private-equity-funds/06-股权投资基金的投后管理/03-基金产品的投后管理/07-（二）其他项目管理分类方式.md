# （二）其他项目管理分类方式

## 📜 寓言故事 —— 《三把尺子量庄稼》

老周在县城经营一家粮食收购行，手下管着三个伙计，每年要从十里八乡收几十家农户的粮食。收得多了，他渐渐发现一个问题：每家农户的情况千差万别，要是用同一套办法去打交道，有的耽误了，有的又浪费了精力。

那年秋天，老周坐在柜台后面，账本上密密麻麻记着三十多户农户的名字。伙计阿福跑进来，满头大汗：“东家，东村李家今年的稻子又减产了，我去催了三回，他们总说再等等。可西村张家倒是丰收了，我反而没空去好好谈谈明年的收粮价。”老周揉了揉太阳穴，心想：这三十多户，到底该先顾谁？

他盯着账本发了三天愁。第一把尺子，是看自己在农户家里说话有多大分量。有些农户全靠收购行吃饭，老周说什么时候交粮、按什么价收，对方只能点头。这种人家，老周派个伙计定期去看看就行，不用天天盯着。可有些农户家大业大，除了卖给老周，还有别的销路，甚至自己开了小磨坊。对这种人，老周得亲自上门，陪笑脸、谈条件，不然人家明年就不跟他签了。阿福不解：“东家，为啥对大户反而更客气？”老周苦笑：“因为人家不指望咱，咱得指望人家。咱们在他们家占的份儿小，说话就不硬气，不多花心思，连门都进不去。”

第二把尺子，是看哪户粮食在自家仓库里占的堆头最大。老周翻出库存簿，发现北村王家今年交的麦子占了全库近两成。他立刻把阿福叫回来：“王家那边，以后你每旬去一趟，不是去催粮，是去问问他们田里有没有虫害、需不需要提前借种子钱。这户要是出了岔子，咱们整个冬天的买卖都要受影响。”阿福挠挠头：“东家，王家脾气倔，不好打交道。”老周把账本往桌上一拍：“正因为不好打交道，才要你去。这户分量重，容不得闪失。那些只占半袋仓的小户，出点问题不伤筋骨，王家要是黄了，咱们伤筋动骨。”

第三把尺子，最让老周夜里睡不着——看每份粮食合约还有多久到期。有些合约明年春天就到期，农户急着拿钱换种子，要是到时候老周手里没现钱，农户只能把粮食贱卖给隔壁县的行商。有些合约还有三四年，可以慢慢等个好价钱。老周在墙上挂了一张大纸，把三十多户的到期日子一字排开，红的、黄的、黑的标记分得清清楚楚。他对阿福说：“明年到期的这几户，今年冬天就得开始谈续签的事，该让利的让利，该提前给钱的提前给。不能等人家上门来要，那时候咱们就被动了。”

到了第二年，老周用这三把尺子重新分了工。说话分量轻的农户，伙计按月巡查；占堆头大的农户，阿福亲自盯着；快到期的合约，提前半年就开始走动。一年下来，没有一户粮食烂在手里，也没有一户因为价钱谈不拢而跑掉。年底算账，收购行的利润比往年高了两成。阿福问老周：“东家，您这法子有名字吗？”老周摆摆手：“没什么名字，就是别拿一把尺子量所有人。看人下菜碟，不是势利，是明白。”

**📖 原文定义**

> 1. 按照对被投资企业的影响力分类
> 基金在被投资企业的投资占比和对公司治理的参与程度，影响投后管理的安排和效果。如基金投资在被投资企业的占股比例较大，则在投资时可以争取到更高的股东会投票比例以及董事会席位，投后管理工作的风险监测和增值服务可以通过股东会和董事会决议影响被投资企业的重大决策。
> 2. 按照基金内部投资项目的权重分类
> 已投项目在基金的所占份额亦影响投后管理的策略和精力分配。鉴于该笔投资对于基金整体的投资回报影响较大，对于该类项目投后管理在风险等级排序中应给予优先级和关注度，重点跟进该类项目出现的相关风险问题的处置，进而实现整只基金投资回报的提升。
> 3. 按照已投项目的投资退出计划分类
> 已投项目的投资期限需要与投资基金的期限相匹配。若在投后管理的过程中未关注基金期限而使基金临期，部分投资者很可能由于资金流动性问题不同意续期，只能低价处置项目，无法取得更高的投资收益。因此，在投后管理的项目分级时，可以在企业发展维度分级标准之上增加基金期限这一维度，便于投后管理人员有效安排各项目的退出节奏，争取投资收益的最大化。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周在农户家说话的分量大小 | 对被投资企业的影响力（投资占比、治理参与程度） |
| 占堆头大的王家需要重点跟进 | 基金内部投资项目的权重（所占份额影响精力分配） |
| 合约到期日挂墙分类、提前续签 | 已投项目的投资退出计划（投资期限匹配、安排退出节奏） |
| 三把尺子分别量不同农户 | 其他项目管理分类方式的三种维度 |
| 大户不指望老周，老周得花更多心思 | 影响力小则需更多投后管理投入 |
| 王家占仓库近两成，容不得闪失 | 权重大的项目给予优先级和关注度 |
| 到期日临近的农户可能贱卖给隔壁行商 | 基金临期时投资者可能因流动性问题不同意续期、低价处置 |
