# （一）早期企业投资基金的投后管理策略

## 🌱 寓言故事 —— 《苗圃里的两种园丁》

西子湖畔有位老茶农周叔，膝下两个儿子——阿望和阿恒。哥俩子承父业，各自在山脚下包了一片苗圃，专帮外乡刚起步的小茶坊代养茶苗。

阿望的苗圃小，拢共就二十来株母株。他对每一株都熟得不能再熟：哪一株去年受过冻，今年得少剪一刀；哪一株根系浅，浇水要比别家多停半刻。每天清早，他搬把小竹椅坐在苗圃边上，一坐就是大半天，不动手，只看着。茶苗偶尔叶子发黄，他不急着上药，先蹲下看看根、看看土，再决定是松土还是追肥。方圆百里的小茶坊主都愿意把苗送到他这里来养，图的正是这份耐心。

阿恒的苗圃大，足有六七亩，密密麻麻排了上千株。阿恒忙不过来，便在苗圃边上立了几块大木牌，把每一类茶苗该浇多少水、施什么肥、几时打顶，全写得一清二楚。新来的伙计只要照着木牌上的步骤来，错也错不到哪里去。每隔几天，他请来镇上的老把式在苗圃里转一圈，挑出几株长势不好的打上红记号，单独堆到一处，统一处理。

这一年深秋，麻烦来了。城东小茶坊主沈秀娘，押着全部身家盘下三亩荒山开出茶园，苗子全是从阿望这里买的。入夏后连续大旱，沈秀娘的合伙人卷了银子跑了，账房也一哄而散。她一个人坐在茶园边上，望着东倒西歪的茶苗，眼泪都快哭干了。

消息传到阿望耳朵里，他放下手里的活，卷起铺盖就上了山。一住就是二十天。他没动剪子，先帮沈秀娘把剩下的账理清楚——工人欠薪要结，井水要重新凿深的，肥料要重新配比。忙完这些，他才下地，一株一株把茶苗扶正、补种、修剪。又过了一个月，茶园总算缓过气来。沈秀娘问他怎么谢，阿望摆摆手："你只管把茶种好。"

苗圃里另一头，阿恒的茶苗也出了岔子。一家远方客人的苗圃遭了虫灾，伙计按木牌上的法子喷了药，却没注意到这一批苗是从南方引的种，药量得减半。三天后半亩苗全蔫了。客人打上门来，阿恒赔了银子，又专门请了镇上最贵的老把式来善后。

冬至那天，两兄弟在山脚的茶馆里碰面。阿望端着热茶说："我那几株苗，认得清每一株的脸。哪一株今年长得好，明年得歇一歇；哪一株根系深，明年多留一茬杈。这些事，写不进木牌里，只能蹲在边上慢慢看。"阿恒点点头："我那一千多株认不过来，只能靠木牌、靠流程。木牌管不了的，就只能花钱请老把式来救场。"

阿望接着说："今天沈秀娘那边合伙人跑了，她一个人撑不住，我不去顶一阵子，那片茶园就全完了。等她站稳了，我再回我那半亩苗圃。"阿恒叹了口气："前几天我赔了客人那笔苗钱，数目不小。要是早多花点心思，跟客人多通几封信，兴许虫灾一开始就压得住。"

山风吹过茶馆，兄弟俩的茶都见了底。

---

**📖 原文定义**

> 早期企业投资基金专注于投资初创期和小型的成长期企业，对于该类基金的投后管理策略没有统一的标准，基金管理人可根据自身规模和投资风格选择不同的投后管理模式。如少而精的投资风格，可以匹配深耕细作的个性化投后管理；多而全的投资风格更适合宽泛的标准化投后管理。实践中，早期企业投资基金的投后管理较为简单，工作更多聚焦在关注和协助公司成长，甚至在企业管理层动荡时基金管理人会根据公司的需求变成管理者，协助承担起公司的部分管理职能。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 阿望苗圃小、只养二十来株 | 少而精的投资风格 |
| 阿望对一株一株熟悉习性、搬椅坐到地头 | 深耕细作的个性化管理 |
| 阿恒苗圃大、上千株苗、立木牌写流程 | 多而全的投资风格 |
| 阿恒的木牌、伙计照章办事、请老把式巡场 | 宽泛的标准化管理 |
| 沈秀娘茶园出事，阿望放下活上山住了二十天 | 关注和协助公司成长 |
| 阿望帮沈秀娘理账、凿井、扶苗、补种 | 协助承担起公司的部分管理职能 |
| 阿恒按木牌喷药疏忽，虫灾后赔钱请老把式善后 | 标准化管理的盲区与补救 |
