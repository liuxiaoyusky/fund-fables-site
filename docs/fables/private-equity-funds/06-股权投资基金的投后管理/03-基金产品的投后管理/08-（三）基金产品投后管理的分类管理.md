# （三）基金产品投后管理的分类管理

## 🏺 寓言故事 —— 《三味茶坊》

老周在古镇开了三家茶坊，分别叫"新芽""青枝""老根"。他每日在三家铺子之间来回走动，日子久了，发现一个头疼的问题：三家茶坊的客人不一样，茶叶不一样，连账本上数字的跳法都不一样。他若是用同一套法子管三家，总有一家要出问题。

一天，镇上来了位老茶客，姓吴，是个走南闯北的商人。吴老板在"新芽"订了一整年的雨前龙井，出手阔绰。可没过两个月，他又在"青枝"投了钱，说要试试新出的岩茶。老周起初没当回事，觉得有钱赚就好。可到了年底，吴老板突然派人来传话，说自己在"老根"也入了股，三家铺子加起来，他投的钱比老周自己的本金还多。老周这才慌了——吴老板若是同时从三家撤资，他的周转就断了。而那些在"新芽"只投了几两碎银的小茶客，反倒天天来问东问西，耗去他大半精力。

老周去找镇上的账房先生老孙讨主意。老孙翻了三本账，指着"新芽"说："这家客人杂，有散客、有常客、还有像吴老板这样的大主顾。你不能一视同仁。大主顾投得多，耐得起风浪，你只需逢年过节送盒好茶，稳住人心即可。散客投得少，风吹草动就心惊肉跳，你得常去走动，让他们安心。"老孙又翻到"青枝"，说："这家岩茶还在发酵，钱压在茶窖里，一时半会儿变不成现银。你得盯着茶窖的温度湿度，别让茶叶坏了。"最后指着"老根"："这家普洱年份久，越陈越香，账面上数字稳当，你反而最省心。"

老周恍然大悟。他回去把三家茶坊的客人按"投了多少""能扛多大风浪""钱什么时候能回来"分了类。"新芽"的大主顾归为一档，定期送礼问候；散客归为另一档，每月亲自上门泡茶聊天。"青枝"的岩茶窖里压着钱，他专门雇了人日夜看守。"老根"的普洱陈着，他只在每年清明前后盘点一次。更妙的是，他发现"新芽"里有个小茶客，姓林，是个走街串巷的货郎，虽然投得少，但消息灵通。老周每月请他吃顿茶，听他讲讲四乡八镇的行情，反而比自己闷头算账强得多。

三年后，镇上闹了一场茶市风波，几家茶坊接连倒闭。老周的"新芽"因为提前听林货郎说了风声，赶在跌价前把存货出手了大半；"青枝"的岩茶窖守得稳，没出一点差错；"老根"的普洱反倒因风波后陈茶涨价，赚了一笔。老周坐在"老根"的柜台后，看着三本分类清楚的账本，终于明白：管茶坊不是管茶，是管人——而人，从来不能一概而论。

**📖 原文定义**

> 基金管理人在募集阶段时就要对基金产品进行分类，《私募管理办法》要求基金管理人销售基金产品时应当进行风险评级，向风险识别能力和风险承担能力相匹配的投资者推介基金产品。《基金募集机构投资者适当性管理实施指引（试行）》明确要求，基金产品或者服务的风险等级要按照风险由低到高顺序至少划分为5个等级，且明确指出可以通过定量和定性相结合的方法进行风险分级。
> 基金管理人在投后管理过程中也会借鉴前述方法和分析因素，对基金产品及时进行评估，尤其是对于管理多只基金产品的情况。基金管理人开展投后管理的分类通常从以下几个方向进行综合考虑：
> 1. 基金产品主要要素情况
> 基金管理人可根据投资者类型（机构或自然人）、基金规模、基金期限、退出方式、是否托管、是否外包、管理方式（基金管理人自行管理或委托管理）、投资行业的宏观发展等主要要素，对基金产品进行综合定性分析。例如，基金投资者类型方面，机构投资者相对专业理性且风险承受能力相对较高，自然人投资者相对而言风险承受能力较弱；基金产品中处于投资期、退出期、清算期不同阶段，基金管理人面临的项目退出压力不同，基金投后管理的策略也不一样。
> 2. 基金产品底层资产情况
> 基金的底层资产情况重点是基金的估值，其中也包括资产的流动性评估。基金估值是基金运作中的重要内容，是指根据相关规则对基金的资产和负债的公允价值进行计算和评估，从而确定基金价值的过程。对于股权投资基金而言，基金估值的对象主要是基金所持有的非上市公司股权价值，也包括已上市被投资企业未退出股票价值。估值的日期和频率可以按照管理人和投资者的实际需要，大部分基金是每年或每半年进行一次估值更新。如基金估值高、资产流动性高，则退出预期较明确，投后管理工作也相对来说简单。反之，如估值水平不高、资产可变现的情况差，那么投后管理则应高度重视。
> 3. 其他特别事项
> 除了前述两方面外，如果基金存在一些特别事项对投后管理产生影响，也需要审慎对待将该类基金产品进行标记，在投后管理的工作重点关注。
> 例如：基金无法顺利进行工商延期预计将导致工商异常情形，该类基金可能面临基金证券账户被冻结无法及时进行分配，则需要在投后管理策略中做好计划，尽量在工商到期之前将存量已经上市的项目尽快在二级市场出售，以便顺利完成基金产品层面的分配。

|---------|---------|
| 老周按客人投钱多少、扛风险能力、回本时间分类管理 | 基金产品按风险由低到高至少划分为5个等级，定量与定性相结合进行风险分级 |
| 吴老板投得多、耐得起风浪，只需逢年过节送礼稳住 | 机构投资者相对专业理性、风险承受能力高，管理策略相对简单 |

| "青枝"岩茶还在发酵、钱压在茶窖里，雇人日夜看守 | 底层资产估值不高、可变现情况差，投后管理应高度重视 |
| "老根"普洱年份久、越陈越香，每年清明盘点一次 | 资产估值高、流动性高，退出预期明确，投后管理相对简单 |
