# 二、投后管理决策机制和组织架构

## 📜 寓言故事 —— 《老周的五家店》

老周在城东开了第一家钱庄，取名"周氏银号"。那时候店里就他一个掌柜，伙计三两个，每天什么生意该做、哪笔放款该收、哪家欠账该追，他往柜台后一站，脑子一转，嘴一张，事就定了。十年下来，银号稳稳当当，老周也攒下了口碑。

后来老周动了心思，在城南、城西、城北、城中连开四家分号。五家店齐头并进，老周却还想按老规矩来——所有分店的大事小事，统统送到他总号的书房里，由他一个人拍板。他心想：我一手带起来的生意，还能有谁比我更清楚？

起初倒也勉强应付。可到了第三年，麻烦像夏天的蚊子一样扑上来。一天早晨，城南分号的伙计阿福气喘吁吁跑进总号，说半年前放给绸缎庄的那笔款子，对方掌柜突然连夜搬走了大半货物，门脸虽还开着，内里怕是空了。阿福急得直跺脚："东家，再不去堵人，怕是要烂账！"

老周正埋头看城中分号送来的一份新生意文书——那是笔大买卖，对方是城中最大的粮商，若做成了，一整年的利钱都有着落。老周两头都想顾，两头都顾不过来。他挥挥手让阿福先回去"盯着"，说等自己审完这份文书就过去。阿福悻悻走了。等老周三天后终于腾出空来，绸缎庄的掌柜早已不知所踪，那笔款子成了一纸空文。

几乎同时，城西分号一个跟了老周多年的老客户——开茶山的林掌柜——来求追加一笔周转，说是春茶丰收，多收一仓就能多赚三成。这本该是好生意，可文书堆在老周案头，前面还压着城北分号的一堆旧账清理、总号年底的盘点章程，愣是拖了半个月。等老周批下来，林掌柜已经等不及，从别家借了钱，多赚的利钱白白送给了旁人。

老周坐在书房里，看着满桌没拆的文书，第一次觉得这五家店不是五棵摇钱树，而是五座山，全压在他一个人肩上。他约了几个老相识吃饭，席间说起这烦心事。对面坐着的老钱，在邻县管着八家分号，听了只是笑，给老周斟上一杯酒。

"老周，我问你，你开第一家店的时候，店里的事谁说了算？"

"当然是我。"

"那你现在开五家店，还想一个人说了算？"老钱放下酒杯，"你一家店时，你坐在柜台后一拍板，全店都听得见。现在五家店，城南的事传到总号，路上就得半日。你一个人再精明，能同时坐在五个柜台后面？"

老周闷头喝酒，不说话。

"我早年也跟你一样，八家店的文书全往我桌上堆，结果烂了三笔账，错过两桩好买卖。后来我想通了——"老钱伸出一根手指，"每个店，让他们自己凑个议事小组，掌柜、账房、跑外场的，三个人坐下来，把本店的大事小情先议出个眉目。不是啥事都往上送，只有真正跨店的、数额太大的、或者特别棘手的，才送到你这儿来。你管的是'总'，不是'全'。"

老周眼睛亮了："那……要是某个店自己拿不准，又怕耽误事呢？"

"让他们先拟个条陈，写明缘由，你定个规矩——几日之内必须批复，逾期默认放行。规矩写成册子，贴到各店墙上，大家都按册子办事，不比你一个人拍脑袋强？"

老周连连点头，又追问："那各店后面跟着的那些事——收账、盯账、看紧别让人骗了——这些活，是不是每个店都得专门养几个人？"

老钱摇头："不在于你有没有专门养人，而在于活有没有人干。我那八家店里，有三家小店，一个老伙计身兼数职，既跑新生意又盯旧账，干得挺好。有五家大店，人多事杂，我就设了专人专岗，一个人只管后续跟进，一个人只管账目核对，一个人只管外面风吹草动。店小店大，活法不一样，关键是别把活落在空地上。"

老周回去后，连夜改了章程。五家分号各自设了三人议事小组，日常大事本地议、本地定。总号老周只抓三件事：跨店联动的、数额过线的、以及各店议完还觉得心里没底的。他又把批复时限写成条文，贴到各店。至于后续跟进的活，城中、城西两家大店各添了专人；城南、城北、总号三家小店，则由老伙计兼着，谁熟哪摊谁管。

半年后，老周再请老钱吃饭。老钱问："还忙得过来吗？"老周笑呵呵地给他倒酒："现在各店自己的文书自己消化，送到我这儿的，一天不过三五份。我反倒有空去各店走动，看看人、听听事。那两家添了人的大店，旧账收得利索；三家小店兼着干的，也没落下什么。"他顿了顿，"以前我守着总号，五家店的事全往我脑子里灌，灌多了反而糊涂。现在脑袋轻了，眼睛反倒亮了。"

老钱举杯："这就是了。管一家店靠一个人，管五家店靠一套章法。章法不是'有没有这个衙门、那个衙门'，而是'活归谁干、事归谁定'。你把这两样理清楚了，店再多几家，也睡得着觉。"

窗外秋风起了，老周望着街对面自己那五家分号的灯笼，红彤彤亮成一片。他忽然觉得，这光不是压在他肩上的五座山，而是五盏灯，各有各的亮法，又都照着他这一片天。

**📖 原文定义**

> 实践中，股权投资基金的投委会是股权投资基金的核心决策机构，也是基金管理人对基金履行管理、投资、退出、监督等职能的核心方式。一般来讲，如果基金管理人在管产品较少，且对其管理的股权投资基金享有绝对的控制权，可直接在管理人层面设置投委会，无须在基金层面另行设置。当基金管理人在管股权投资基金较多，或者基金有专门约定时，可能涉及进行分层设置，即管理人和基金分别设置投委会，依据管理人公司章程（合伙协议）和基金合同等内部治理文件进行决策。
>
> 实践中，基金合同中大部分直接明确约定，应由基金管理人为基金所设立的投委会作为负责基金投资决策全部事项的议事机构，通常既包括对投资事项的决策，也包括对投后和退出事项的决策。投委会作为基金内设机构，在基金合同和相关协议文件约定的授权范围内行使职权，对投后管理作出决策。部分基金合同中约定特别事项除投委会决策外还需提交咨询委员会或合伙人会议决策（见图6-4）。
>
> 基金投委会的议事规则一般在基金合同或基金委托管理协议中加以明确。若未能明确，一般建议由基金管理人制作单独制度文件将规则书面化，并将相关规则提交给基金投资者审议或备案。如对规则进行修改，通常需经过基金投资者同意，或在经授权机制审批后及时向全体基金投资者备案，修改机制也应在规则中明确。
>
> 评价基金管理人是否重视投后管理并设有有效的组织架构，不在于是否设立专门的投后管理部门，而是需要观察投后管理职能的实际安排和资源配置。
>
> 根据基金管理人常见的组织架构、按照较为完整的部门设置，“设专门部门管”模式涉及的部门可能包括：投资部、投后部、风控部、财务部、运营部。各部门参与投后管理所承担的职能各不相同，具体包括：
>
> 1. 投后部
>
> 2. 投资部
>
> 3. 风控部
>
> 4. 财务部
>
> 5. 运营部
>
> 需要说明的是，上述投后各职能内容在不同的基金管理人可能设置各有不同。对于管理规模较小的基金管理人而言，由于项目不多，投资经理能够对所投项目进行跟踪赋能并取得较好成效，也能履行好向投资者和中国证券投资基金业协会的披露和报送，那么投资经理可以承担投资部、投后部以及运营部的相关职能，无须设置相关部门。另外一些情形下，有些管理规模较小的基金管理人将财务部、法律风控部合并设置（或统称为风险部门），统一对风险问题进行监测和管控，并不强制进行分部门设置。
>
> 实践中，随着基金产品管理规模不断发展壮大，投资项目数量不断增加，基金投资者的数量和类型不断丰富多元，基金管理人可对组织架构进行适应性调整。即使以往没有设置投后、运营等岗位或者部门的，也会进行增设以对被投资企业提供更为优质的投后服务，以及匹配监管报送等合规要求。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周早年一家店自己拍板 | 管理人层面设置投委会，产品少时无需在基金层面另行设置 |
| 五家店后各设三人议事小组 | 管理人和基金分别设置投委会的分层设置 |
| 老周只抓跨店、大额、棘手事 | 管理人层面投委会负责特别事项，基金层面投委会负责日常决策 |
| 批复时限写成条文贴到各店 | 议事规则书面化，提交投资者审议或备案 |
| 大店添专人、小店老伙计兼着 | 组织架构不在于是否设专门部门，而在于实际职能安排和资源配置 |
| 小店一人兼多职、大店专人专岗 | 小规模可合并部门，大规模需分部门设置 |
| 城中、城西大店增设人手 | 规模发展壮大后，对组织架构进行适应性调整 |
| 老周从"全管"到"总管" | 核心决策机构在授权范围内行使职权，对投后管理作出决策 |
