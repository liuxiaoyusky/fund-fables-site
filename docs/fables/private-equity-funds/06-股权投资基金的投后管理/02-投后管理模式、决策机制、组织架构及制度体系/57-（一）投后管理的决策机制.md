# （一）投后管理的决策机制

## 🏺 寓言故事 —— 《两间祠堂的议事厅》

徽州周氏宗族这几年日子越过越红火，祠堂里挂的匾额也越来越多。可也出了新麻烦——事情一多，原先那间小小的议事厅挤不下了。

周老太爷把几位管事的叫到厅里，说："今天咱们把祠堂里的规矩整一整。"

管事老吴翻开账册，指着上面一项项说："族里这一年光置田的事就有十几桩，每回都得几位长辈凑齐了才拍板。有时候凑不齐，事情就压着。"

周老太爷点头："我看分成两种情况。"

他伸出一根手指："要是族里就那么几位长辈看着，每回都能凑得齐，那就不必再另设一间屋。在咱们这间老厅堂里直接议就行。"

他又伸出第二根手指："要是族里田多事多，光靠老厅堂拍板来不及，就在老厅堂之外再设一间厅，专门议那些大事。两间厅各有各的规矩——一间议族里总的事，一间议单笔单条的事。两边各立各的账、各定各的规矩，可又都归族里管着。"

老吴担心："两边定出来的不一样怎么办？"

周老太爷笑："所以我才说要'各立规矩'。哪间厅议什么事、谁来议、议出来怎么定——写成文书。族里有自己的'族规'，单笔的田、单独的买卖立自己的'字据'，两边都对齐了，写下白纸黑字，后人照着办就是。"

果然，第二年周氏又置了一座茶山。这回没等几位长辈凑齐，周老太爷在另一间厅里和管事老吴单独议定——山价多少、何时交割、契书怎么立——按事先写好的规矩办完，前后只用了一个下午。

老吴事后感慨："从前事事都得在大厅堂里议完才能动，遇上人凑不齐就卡壳；今天按规矩分头办，反倒利索。"

周老太爷捋着胡子："不是因为事少了，是因为规矩分得清。哪件事归哪间厅议、谁来议、按什么字据办——写明白了，剩下的就是照章办事。"

---

**📖 原文定义**

> 实践中，股权类资金的投委会是股权类资金的核心决策机构，也是资金管理人对资金履行管理、投资、退出、监督等职能的核心方式。一般来讲，如果资金管理人在管产品较少，且对其管理的股权类资金享有绝对的控制权，可直接在管理人层面设置投委会，无须在资金层面另行设置。当资金管理人在管股权类资金较多，或者资金有专门约定时，可能涉及进行分层设置，即管理人和资金分别设置投委会，依据管理人公司章程（合伙协议）和资金合同等内部治理文件进行决策。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 周氏议事厅从前挤不下 | 投委会作为核心决策机构 |
| 在管产品少时直接在一间厅议 | 可直接在管理人层面设置投委会 |
| 事情多时另设一间厅 | 分层设置管理人和资金两套决策机构 |
| 老厅堂议族里总的事 | 管理人层面的决策 |
| 另一间厅议单笔单条的事 | 资金层面的决策 |
| "族规"和单笔"字据"各立各的 | 依据公司章程或资金合同等内部治理文件决策 |
| 老吴感慨"分头办反倒是正理" | 决策机制按规则分层运行 |
