# 二、投后管理决策机制和组织架构

## 🏺 寓言故事 —— 《老宅的账房与库房》

徽州李氏宗族在宅子后头盖了一间大大的院子，把祠堂、账房、库房、客屋都搬了进去。族长李老太爷坐在厅堂上，把管事的几位叫到跟前说："咱这院子，今儿起要订两条新规矩。"

管家老陈问："哪两条？"

李老太爷伸出两根手指："一条叫'谁说了算'，一条叫'谁干活'。"

族里众人一时没听明白。李老太爷让账房师傅端来一摞账本，翻开第一页，指着上面一条条买田、卖粮、佃户进出、借出去的银子说："这些事是这一族里的'大事'。哪家佃户要卖田，哪家媳妇要借银子，哪个孩子要送去县里读书——都得有个定下来的规矩，由谁来拍板。"

他又指了指后院："可光是定下来'谁说了算'还不行。账房管银钱进出、库房管物品存放、客屋管外客往来、田庄管地里收成——这些'谁干活'也得说清楚。不然规矩定得再大，没人去干，也是一句空话。"

管家老陈又问："那跟从前有什么不一样？"

李老太爷合上账本："从前咱们这个院子里，做什么都是族里几位长辈围着一张桌子说。今天凑不齐人，事情就耽误；明天大家意见不一，又吵成一锅粥。今天往后，'拍板的事'按新立的规矩办，'干活的事'按分好的屋舍办。两边各管各的，又连在一起。"

老陈似懂非懂："您是说，这院子里有两件事要分头安排？"

李老太爷点头："一是'谁说了算'，这是规矩；二是'谁干活'，这是人手。两件事分头安排妥当了，咱们这一族的事情才能办得长久。"

院子盖好之后，账房先生管银钱，库房管事管物品，几位执事按规矩议事。李老太爷常常拄着拐杖过来看一眼，半是玩笑半是叮嘱地说："这院子里，少了规矩库房就乱，少了人手账本就积灰。两边都得有，少哪一头都不成。"

---

**📖 原文定义**

> 投后管理决策机制和组织架构主要涉及投后管理的决策机制、投后管理的组织架构两个方面。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 李老太爷伸两根手指说两条规矩 | 投后管理涉及两个方面 |
| "谁说了算"指谁来拍板大事 | 投后管理的决策机制 |
| "谁干活"指分好部门各管各的 | 投后管理的组织架构 |
| 账房先生管银钱、库房管事管物品 | 部门分工 |
| 几位执事按规矩议事 | 决策机制按规则运行 |
| 李老太爷说"两边各管各的，又连在一起" | 决策机制与组织架构相互配合 |
| "少了规矩库房就乱，少了人手账本就积灰" | 决策机制与组织架构缺一不可 |
