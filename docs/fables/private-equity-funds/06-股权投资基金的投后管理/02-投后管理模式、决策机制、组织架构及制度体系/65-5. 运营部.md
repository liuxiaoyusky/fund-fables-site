# 5. 运营部

## 🏺 寓言故事 —— 《小周的茶山日记》

徽州吴记茶庄新请了一位管事，叫小周。这位小周与别的管事不太一样——他不挑山，不谈价，不翻账本，他专门做一件事：把茶园里每天发生的事记下来。

东家吴老爷头一回见他，问："小周，你来做什么差事？"

小周拱手："东家，我来'记'。"

吴老爷笑："记什么？"

小周答："记茶园里每天发生的事。哪一天哪位佃户上了山、哪一天摘了多少茶青、哪一天送进了哪一家茶行、哪一天苗补了多少、哪一天路上塌了一截——这些事，我都记。"

吴老爷皱眉："从前没人记？"

小周摇头："从前佃户自己记，可佃户记的事只有他自己看见的。我记的事，是把几座茶园放在一起看的。"

吴老爷点头："那你记吧。"

小周头一个月就把六座茶园的事排成了六本日记。每月月底，他拿六本日记一翻——咦，第四座茶园这个月摘的茶青比别家少了快两成。他跑去一查——是那片茶园的佃户老张家里有事，请了几天假。

第二个月，他又翻——第二座茶园这个月挑茶青上山花的脚夫银比别家多出将近一成。他跑去问——是那条上山的路今年雨季塌了一段，脚夫要绕远。

第三个月，他又翻——第五座茶园这个月补苗的银子和第三座一样多，可补的苗数少了一截。他跑去核实——是佃户老赵从邻村买的苗价比镇上贵了两成。

吴老爷看完小周递上来的三本小册子，眼睛亮了："你这三本册子一摆出来——哪座茶园的人手要加、哪座茶园的路要修、哪座茶园买苗的方式要换——全看清楚了。"

小周笑："东家，我一个人不动一锹土、不花一文钱，就靠记和比。"

又过了些日子，吴老爷又来问小周："今年茶园里最难定的事是哪一桩？"

小周答："是哪座茶园什么时候该转手。这件事我光记一年还不够，得连着记两三年——看哪一座茶园连着几年都比别家差，再来议。"

吴老爷拍他肩膀："你这差事——看着只是记和比，其实是把茶园的里里外外都'看'了一遍。"

小周点头："管日常事务这摊事，不是自己上阵挑山，是在旁边把'每天发生的事'看明白。"

---

**📖 原文定义**

> 运营部是资金管理人中负责运营管理的部门。其在投后管理中的主要职责包括：对被投资企业的运营情况进行跟踪和分析；对被投资企业的运营数据进行分析和报告；为被投资企业提供运营管理方面的支持；参与投后管理制度的制定等。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 小周专门"记"茶园里每天发生的事 | 对被投资企业的运营情况进行跟踪和分析 |
| 六座茶园的事排成六本日记 | 运营数据的日常记录 |
| 小周把六本日记放在一起"比" | 对被投资企业的运营数据进行分析和报告 |
| 发现第四座茶园茶青少两成 | 通过数据分析识别异常 |
| 提出人手、路、买苗三个建议 | 为被投资企业提供运营管理方面的支持 |
| 小周说要连着记两三年再议转手 | 持续跟踪运营情况 |
| "在旁边把每天发生的事看明白" | 参与运营分析与支持 |
