# （三）风险关注示例

## 🏺 寓言故事 —— 《挑刺老陈的四件心事》

徽州李记茶庄的挑刺管事老陈这两年是东家李老太爷身边最忙的人。年底盘账，老陈跟李老太爷说："东家，今年我心里头有四件放不下的事。"

李老太爷问："哪四件？"

老陈从袖子里掏出四张纸，往桌上一字排开："第一件——山头交接的事。挑山的把山头交给看园的，挑山的人怎么挑、当时怎么想、看园的人接过去怎么接着办，得有册子写明。"

"第二件——山头买下来之后每天的事。佃户按月交的茶青折多少银、补苗花多少、哪一天路塌了要修、哪一天有人偷茶青——这些事按月看、按季查，照章办事。"

"第三件——以后还要不要再添新山。这件事最难——添新山的银子从哪儿出、添进来之后谁管、添进来之后旧山头怎么平衡，得先议一定再动。"

"第四件——月报季报怎么写。哪一座山头这一季茶青收了多少、补苗花了多少、佃户稳不稳——按一季一报送给东家和族里长辈，长辈要查一查就翻得到。"

李老太爷看着桌上四张纸，半晌才说："老陈，这四件事——每一件挑出来都是事后才看得见，可事先能想到的才叫'挑刺'。"

老陈点头："东家，咱们茶庄里管茶园这件事，头绪本来就多——挑山的人归一拨，看园的人归另一拨，记账的归账房，挑刺的归我。再加上添新山、改规矩、月报季报——哪一头出岔子，整桩事都跟着乱。"

他又伸出一根手指："所以我今年花了整整一个月，把这四件事从头到尾捋了一遍——第一件'交接'心里清楚怎么盯，第二件'日常'按月按季查，第三件'添山'先议后动，第四件'月报季报'按章送到。这四件事盯住了，茶庄里管茶园这摊事就能稳住。"

李老太爷点头："从前我也只觉得管茶园是看园子、收茶青、补苗几件事——听你这一说才发现，管茶园这件事里藏着这么多要先盯的'刺'。"

老陈答："东家，'挑刺'二字看着不中听，可事先挑出来的才是小刺，事后才发现的就是窟窿。"

---

**📖 原文定义**

> 投后管理的工作周期长、工作事项繁杂、涉及部门及人员广泛，资金管理人在制定投后管理制度体系时需从风险关注的角度进行业务流程的全面梳理，保障制度体系运行的效率和效力。投后管理中重点风险关注示例如下：投后项目交接、投后日常管理、后续投资管理、投后管理报告。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老陈掏出四张纸 | 风险关注需全面梳理业务流程 |
| 第一张"山头交接" | 投后项目交接风险 |
| 第二张"山头买下来之后每天的事" | 投后日常管理风险 |
| 第三张"以后还要不要再添新山" | 后续投资管理风险 |
| 第四张"月报季报怎么写" | 投后管理报告风险 |
| 老陈按月按季查 | 制度体系运行的效率与效力 |
| "事先挑出来的是小刺，事后发现的就是窟窿" | 风险关注的前置性与系统性 |
