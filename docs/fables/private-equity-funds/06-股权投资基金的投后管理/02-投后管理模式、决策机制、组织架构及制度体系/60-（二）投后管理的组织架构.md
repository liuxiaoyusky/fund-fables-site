# （二）投后管理的组织架构

## 🏺 寓言故事 —— 《茶园里的五间屋子》

徽州城南的吴记茶庄这几年越做越大，光茶园就有五座。可东家吴老爷心里一直没底——他总觉得自己"管"不住这些茶园。

这一年开春，几位管事的坐在厅里议明年的事。账房先生老陈先开口："东家，咱们今年光算一算——每座茶园要不要补苗、什么时候摘、摘下来的茶青往哪儿送——就已经忙不过来了。您还一人说了算，可您不可能天天盯着每一座茶园。"

吴老爷问："那你觉得该咋办？"

老陈答："邻镇那家大茶庄，院子里分了五间屋子。一间屋专管已经买下的茶园，专门看着哪些苗要补、哪些茶要摘。一间屋专管挑茶园、谈价钱。一间屋专管看岔子，看哪片茶可能出事。一间屋专管银钱进出、账目对账。还有一间屋专管日常来客、外客接待、伙计调度。"

吴老爷皱眉："我从前觉得，只要我一人管得过来，不必再分屋子。"

老陈摇头："可您一人管不过来的时候，事情就压着；您不在的时候，整座茶庄就停摆。光看是不是专门分了一间屋子是次要的——得看事情是不是真有人盯、人手是不是真够。"

吴老爷听进去了，把五间屋子分别交给老陈、账房先生、库房管事等五人。他跟五人说："五间屋子各管各的，可都是为茶园办事。每间屋子有一摊子要负责，茶园的事才不至于没人盯。"

到了年底，五座茶园的茶青果然都顺利摘下、送进了城里的茶行。账房先生老陈笑着说："今年我没听见东家叹气。"

吴老爷也笑："不是我没叹气，是五间屋子替我叹了。"

---

**📖 原文定义**

> 评价资金管理人是否重视投后管理并设有有效的组织架构，不在于是否设立专门的投后管理部门，而是需要观察投后管理职能的实际安排和资源配置。
>
> 根据资金管理人常见的组织架构、按照较为完整的部门设置，"设专门部门管"模式涉及的部门可能包括：投资部、投后部、风控部、财务部、运营部。各部门参与投后管理所承担的职能各不相同。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老陈说"光看分了屋子是次要的" | 不在于是否设立专门部门 |
| "得看事情是不是真有人盯" | 需要观察投后管理职能的实际安排 |
| 五间屋子按专长分开 | 部门职能分工各不相同 |
| 第一间屋专管买下后的茶园 | 投后部 |
| 第二间屋专管挑茶园、谈价钱 | 投资部 |
| 第三间屋专管风险、第四间管账、第五间管来客 | 风控部、财务部、运营部 |
| "五间屋子都为茶园办事" | 各部门参与投后管理共同配合 |
