# （二）考核与激励机制

## 🏺 寓言故事 —— 《三把尺子》

老周在城东开了一家布庄，生意越做越大，后来干脆把本钱投给了三个伙计，让他们各自去外地开分号。这桩事说来轻巧，可真正的麻烦是从钱出去之后才开始的。

第一个伙计叫阿福，从布庄开张那天起就跟着老周。老周信得过他，把本钱交给他，连人带钱一起派去了苏州。阿福在苏州租了铺面、招了伙计，每月写信回来报账。老周起初还认真看，日子久了，信堆成山，他也懒得细翻。反正阿福是老人，不会出岔子。

第二个伙计叫阿贵，是阿福走后老周新招的。阿贵被派去杭州，老周对他不太放心，专门派了账房先生老吴跟着，每月查一次账，核对进出。阿贵起初挺不自在，觉得老周不信任自己，可日子久了，他也习惯了——老吴查他的同时，也帮他挡住了不少杭州本地地痞的敲诈，因为谁都知道，这铺子背后有人盯着。

第三个伙计叫阿顺，派去南京。阿顺是个精明人，老周喜欢他脑子活。可阿顺去了半年，信越写越短，账越报越粗。老周派去的人回来说，阿顺把铺子当成了自己的，进货挑便宜的卖，好料子都留着私下倒手。老周气得拍桌子，可钱已经投出去了，收不回来，只能想办法收拾烂摊子。

三年过去，三个分号的情况天差地别。阿福的苏州铺子稳当，可也仅仅是稳当，规模和老周当初投下去的时候差不多，没亏也没赚出花样。阿贵的杭州铺子倒是蒸蒸日上，老吴每月盯着，阿贵不敢偷懒，反倒琢磨出不少新门道，把本地丝绸和布庄生意搭上了线。最头疼的是阿顺的南京铺子，已经半死不活，老周不得不亲自去南京，把阿顺换下来，自己坐镇收拾。

老周坐在南京铺子的柜台后面，看着三本截然不同的账册，第一次认真想：我当初把钱撒出去的时候，只想过"谁能干"，从没想过"干得好怎么赏，干得差怎么罚，人跑了怎么办"。阿福没出错，可也没动力往前冲；阿贵被老吴盯着，反倒越干越起劲；阿顺没人管，直接把铺子当成了自己的私产。

更棘手的是阿福的事。阿福上个月来信说，家里老母病重，想辞了差事回乡。老周犯了难——苏州铺子投了三年，还没回本，阿福这一走，谁来接手？新来的伙计对铺子不熟，这账怎么算？算阿福的功劳，还是算他的过失？老周想了一整夜，第二天把三个铺子的管事都叫到南京，摆了一桌酒。

酒过三巡，老周开口了："从今往后，每个铺子定三条规矩。第一，每年年初，咱们坐下来，定好这一年的目标——卖多少布、开多少新客、压多少库存，白纸黑字写下来。年底对照，达标了，掌柜的分红利；不达标，扣俸禄。第二，掌柜的若中途要走，走之前必须把铺子的底细交代清楚——货在哪、钱在哪、哪些账还没收回来。接手的伙计按接手时的底子算功劳，以后赚了是他的，之前亏了的也不赖他。第三，南京这个铺子，我另派人来管，专门收拾烂摊子。这种收拾烂摊子的差事，不看赚了多少，只看收拾得干不干净、利索不利索，按进度算赏钱。"

三个伙计听了，各有心思。阿福松了口气——他正愁自己走了，这三年白干，新规矩下，交接清楚了，后来的伙计能接着干，自己的辛苦也算有个说法。阿贵眼睛亮了——他早就想扩大杭州的生意，可怕老周觉得他贪心，现在有目标、有赏格，他反倒可以放手去搏。阿顺低着头，知道自己那套私吞的把戏再也行不通了。

老周端起酒杯，望着三个分号的方向。他明白了，投钱出去只是开始，真正决定这三家铺子生死的，不是银子，而是他怎么让人干活、怎么量人干活、怎么换人干活。没有这三把尺子，钱撒出去就是撒进了无底洞。

**📖 原文定义**

> 投后管理持续时间长，工作量难以统计，工作成果缺乏量化指标来衡量。作为基金管理人，对于投后管理的工作成效如何考核、对投后管理的负责人员如何激励是一个较普遍的难题。
> 采用 "谁投谁管" 方式的，由于负责投资的投资经理全面负责投资和投后，投后的考核与激励可以与投资的考核与激励一并考虑。这种方式的难点主要在于负责投资的投资经理在项目未完成退出的情况下离职后，对后续接手的投资经理（未参与该项目投资）应如何考核与激励。可以考虑的方法是在交接时，由基金管理人分管投后的高管牵头组织人员对该项目的投资进行回顾、对目前项目的公允价值进行评价、列举实现退出的难点以及可行的措施，由此确定接手的投资经理的考核与激励指标（与退出时间和退出金额挂钩）。
> 采用 "设专门部门管" 方式的，基金管理人可以考虑每年度末制定下一年度的投后管理目标，例如退出金额目标、在管项目里程碑指标（例如后续融资、收入利润目标、上市节点目标等）等，经基金管理人决策机构通过后作为投后管理部门的部门指标，与投后管理部门的部门负责人的绩效直接相关，再由部门负责人分解到部门员工，作为个人的考核指标。考虑到股权投资基金投资期、退出期等运行期限的特点，应注意短期目标和中长期目标的结合，以充分激励员工，保障员工业绩目标和基金管理人业绩目标的一致性。
> 采用 "双线管理" 方式的，由于参与投后的人员多，基金管理人需要先梳理流程、设定权限规则，即投后管理的各项内容由谁发起，由谁审核，由谁决策。具体的考核和激励可以采用以上两种方法考核激励政策的结合。
> 采用 "问题分类管理" 方式的，由于投资部门和投后管理部门所负责的项目类别和发展情况不一样，因此基金管理人应当设置不同的绩效考核标准。尤其是针对投后管理部门负责的问题项目，可以预见的退出收益有限，单纯以退出金额来进行核算的考核激励方式显然无法与内部分工安排相匹配，对于具体投后活动的执行进度则可能是考核的要点，比如司法程序、不良资产处置等投后活动的开展进度以及裁决处置结果。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 阿福的苏州铺子稳当但无动力 | "谁投谁管"方式下，投后与投资考核一并考虑 |
| 阿福中途离职，老周发愁交接 | "谁投谁管"的难点：原负责人离职后，接手人未参与投资，如何考核与激励 |
| 老周要求交接时清点货、钱、未收账 | 交接时由高管牵头回顾投资、评价公允价值、列举退出难点和措施 |
| 阿贵被老吴盯着，反倒越干越起劲 | "设专门部门管"方式，专人监督、目标导向 |
| 每年年初定目标、年底对照、达标分红 | 年度投后管理目标（里程碑指标），经决策机构通过后作为部门和个人考核指标 |
| 南京铺子另派人收拾烂摊子，按进度算赏钱 | "问题分类管理"方式，对问题项目设置不同考核标准，以执行进度而非退出金额作为考核要点 |
| 老周同时用三种不同规矩管三个铺子 | 不同投后管理模式对应不同考核激励机制 |
