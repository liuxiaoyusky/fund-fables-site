# 三、投后管理对基金管理人的作用

## 🪞 寓言故事 —— 《钱庄掌柜的三件事》

江南有一间老字号钱庄，掌柜姓林，做了二十年生意，方圆百里的人都把钱存到他这儿。林掌柜有一句口头禅："钱庄靠钱吃饭，更靠'脸'吃饭。"他说的"脸"，不是他自己那张脸，而是钱庄在外头的名声。

这一日，林掌柜把儿子小林叫到跟前，要把柜上的事一桩一桩交代清楚。

"咱们钱庄这几年越做越大，"林掌柜搬出三只木匣子摆在桌上，"东家多了，生意多了，可是有一件事你要记牢——钱庄赚了银子是一回事，'钱庄在外头有没有脸面'是另一回事。这第二件事，比第一件更要紧。"

小林听得糊涂，问："爹，那要让钱庄有脸面，靠什么？"

林掌柜拍了拍三只匣子："第一只匣子里装的，叫'口碑'。咱们把钱贷给了街上的张屠户、李裁缝、王铁匠，银子出去了不是就不管了——得常常派人去问问生意好不好、账目清不清。要是张屠户的肉摊红火、李裁缝的铺子客人络绎不绝，东家们自然觉得'林家钱庄会挑生意'；可要是王铁匠的炉子三天两头熄火，借出去的银子打了水漂，再有新的东家想来投钱，心里就要掂量掂量了。"

"第二只匣子，"林掌柜揭开盖子，"装的叫'规矩'。东家把钱交给我们，是信得过我们。这份信任不是写在纸上的，是靠平日里一桩一桩做出来。今天报一次账、年底查一回、明明白白让东家看——这就是'规矩'。要是哪天东家来问，咱们支支吾吾说不出个所以然，那这钱庄的脸面就丢了一半。"

"第三只匣子最难讲，"林掌柜叹了口气，"装的叫'眼光'。咱们钱庄能挑出好生意、能一眼看出哪家铺子值得帮一把，靠的不是一时运气，而是日积月累练出来的本事。这本事怎么练？每借出去一笔银子，回来都要回头想想——这次为什么赚、那次为什么赔。一次两次不算本事，年年做、笔笔做，做上十年二十年，才是真本事。"

小林听完，看了看三只匣子，问："爹，这三件事和'钱庄挣银子'比起来，哪个更要紧？"

林掌柜沉默了一会儿，答道："银子挣得再多，脸面丢了，下次就没人愿意把银子交给我们。三件事看似和'赚钱'没关系，其实是钱庄能一直开下去的根本。"

小林点点头，把三只匣子的位置记在心里。

---

**📖 原文定义**

> 资金管理人积极开展投后管理，对其自身亦有重大影响，主要作用包括：一是对商誉品牌维护运营产生重要影响；二是对合规运营风险防范具有重要影响；三是投前投后联动机制进一步影响资金管理人投资能力。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 林掌柜说"钱庄靠钱吃饭，更靠'脸'吃饭" | 投后管理对管理人自身有重大影响 |
| 第一只匣子"口碑"——派人问张屠户、李裁缝、王铁匠的生意 | 一是对商誉品牌维护运营产生重要影响 |
| 第二只匣子"规矩"——账目清楚、随时让东家查阅 | 二是对合规运营风险防范具有重要影响 |
| 第三只匣子"眼光"——每笔借出去都回来回头想想 | 三是投前投后联动机制影响投资能力 |
| 林掌柜强调"脸面"比"赚银子"更要紧 | 三方面作用构成管理人持续发展的基础 |
| 小林把三只匣子的位置记在心里 | 三方面作用是管理人需长期坚持的工作 |
