# 1. 经营指标

## 🎯 寓言故事 —— 《老周的账本》

老周在县城东头开了间酱油坊，手艺是祖传的，酱油香能飘半条街。三年前，有个姓林的商人投了笔钱进来，说要帮老周把生意做大。林商人每月来一趟，问问销量、看看账本，从不插手做酱油的事，只坐在柜台后头喝茶。

那年秋天，林商人照例来访。老周递上账本，满脸堆笑："林老板，今年又比去年多卖了两成！"林商人翻了翻，点点头，喝了茶便走了。老周觉得这位投资人好打发，只要数字好看就行。

转过年来，林商人带来了个年轻人，说是自己新招的帮手，叫小郑。小郑不像林商人那样只看总账。他在酱油坊里转了三天，数了数每天来打酱油的客人，又跑去城西和城南的两家分店，记下了每家的客流量。他还蹲在杂货铺门口，看顾客买酱油时挑的是哪家牌子。回到老周这儿，小郑摊开一张自己画的表，上面密密麻麻记着：老主顾一个月来几趟、新客人占比多少、夏天和冬天的销量差、每缸酱油从酿到卖要花多少天。

老周有些不自在："小郑兄弟，你记这些做什么？我账本上不是写得明明白白？"小郑笑了笑："周掌柜，账本上的数字是结果。可我想知道的是，这结果是怎么来的。"

事情在一个月后起了变化。小郑发现，老周酱油坊的"回头客"比例连着三个月往下掉，可总销量账面上却年年涨。他多跑了几天，终于看出门道：老周为了冲数字，给杂货铺的账期越放越长，货是发出去了，钱却没收回来。杂货铺老板们乐意多进货，反正不用现钱。账面好看，可酱油坊的现钱越来越紧，连买黄豆都要赊账。

小郑把这事告诉了林商人。林商人再来时，没有发火，只是跟老周说："周掌柜，咱俩当初说好，我看的是你这酱油坊能不能长久地香下去。你账本上那个'总销量'，只是一个数。可我想看的，是客人还愿不愿意再来，是货出去以后多久能变成钱，是你的缸里酿着的和铺子里堆着的到底能不能卖得动。这些才是你这酱油坊真正的筋骨。"

老周愣了半晌。他这才发现，自己只顾着让账本上的总数好看，却从没想过那些数字背后藏着的窟窿。他照着林商人给的那张表，开始每天记下客人来的次数、货卖出去多久回款、缸里酿着的酱油多久能卖完。半年后，杂货铺的账期缩了回来，老主顾的比例慢慢回升，酱油坊的现钱也活泛了。

林商人再来时，老周主动把那张表递上去。林商人看了看，笑了："这回，我才是真正投了笔好买卖。"

---

**📖 原文定义**

> 在公司经营过程中，管理层为实现经营目标成果、考核员工，一般会设立经营指标。股权投资基金在投资企业时，一般也会有所关注的经营指标。在投后管理中，应对这些经营指标的变动实时关注、实时分析，合理地评估公司价值，及时发现经营风险。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 小郑记录回头客比例、客流量、账期、库存周转天数 | 投后管理中需要关注多维经营指标 |
| 老周只看总销量账面数字，忽视回款和复购 | 单一指标无法全面反映企业经营状况 |
| 小郑发现总销量上涨但回头客下降、账期拉长 | 通过经营指标变动及时发现经营风险 |
| 林商人要求关注"客人是否愿意再来、货多久变钱" | 经营指标用于合理评估企业真实价值 |
| 老周开始系统记录各项数据后经营状况改善 | 经营指标既是企业管理工具，也是投后管理工具 |
