# 6. 社会责任及绿色发展指标

## 🌱 寓言故事 —— 《林嫂的账本》

林嫂在县城开了三十年榨油坊，灶台还是那个灶台，炒锅还是那口大铁锅。她家用的是村里上百户人家种的山茶籽，榨出来的油香，半个县的人都来挑。城里来的陈先生去年冬天找上门，说想跟她合伙，把油坊扩成镇上最大的厂子，钱由他出，技术和销售他都包。林嫂一辈子没做过这么大的事，但她心里有杆秤——她这油坊，是村东头十几个大娘、大爷冬闲时帮着剥壳、晒籽的活路，她不能让这杆秤倒了。

【冲突】

厂子开起来头一年，订单雪片似地飞来，陈先生在城里跑销路，林嫂在镇上管生产。可林嫂心里渐渐发堵：乡里要把榨油剩下的茶枯饼当柴烧，省事；陈先生让她把炒锅从柴火换成煤气炉，说"又快又稳，产量至少翻一番"；镇上的几个年轻人也上门来求职，可岗位她一个都不敢开——新机器一上，剥壳、炒籽、压榨、清渣全是自动化，老姐妹们靠什么吃饭？林嫂把心事说给陈先生听，陈先生摆摆手："嫂子，生意归生意，账上好看才有人愿意投咱。你那些油烟、柴灰、剩饼的事，没人会管。"林嫂没说话，回家翻出那本跟了她二十年的老账本。

【转折】

第二年春天，省里下来一位戴眼镜的周先生，说要给"乡村振兴做得好的"作坊发一块牌子。周先生在厂里转了一圈，没看财务报表，倒先看了三样东西：油烟往哪里排、剥下来的茶壳有没有还田、雇的工人里有没有村里原本闲着的家庭妇女。林嫂把周先生请到堂屋，把自己那本老账本摊在桌上——上面密密麻麻记着每天的用工名字、进出茶籽的斤两、油渣的去向，还有每年冬天给村里五保户送油的明细。周先生看了半天，抬头笑了："嫂子，你这本子里记的，可比那张报表值钱。你记下的这些，是国家要看的、是县里要看的、是村里老少都要的——你只是用最笨的法子做对的事。"

林嫂愣了一下，扭头去看一直站在门口抽烟的陈先生。陈先生掐灭烟头走过来，指着周先生翻到的那页说："嫂子，从下个月起，我让城里来的小李专门跟你的本子，对一对你记的那些事：多少人在这干活、多少户的山茶籽进了厂、油烟怎么处理、茶壳还了多少田——这些以后得有据可查，每季度报一次。"林嫂听罢，长长地舒了一口气。

【结局】

那年年底，林嫂的油坊在镇口挂起一块新牌——"绿色作坊"。村里那几个冬天没活干的张大娘、李婶子、王家嫂子，全被请回来专门负责剥壳、晒籽、清渣，老姐妹们笑得眼睛都弯了。陈先生特意从省城赶回来陪林嫂喝米酒。席间他问："嫂子，那本老账本，能借我多看看吗？"林嫂笑："借什么借，你照着做一本新的就是——这事儿，本来就该我们这些人操心。"

---

**📖 原文定义**

> 在股权投资全过程中履行社会责任，考虑绿色投资因素已成为行业的共识，主要包括实现国家产业战略指标、财税指标、拉动就业指标、绿色指标、专项指标等。这既是部分投资者对基金管理人的刚性要求，也是基金管理人提升品牌力，实现基金管理人、社会、基金管理人的员工、基金投资者和被投资企业和谐发展的内在驱动力。投后管理过程中，投后管理人员应对相关指标的信息进行收集，并督促被投资企业开展相关活动。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 林嫂三十年用村里人剥壳、晒籽的榨油坊 | 履行社会责任（带动就业 / 产业根基） |
| 油烟、柴灰、茶枯饼等"看着没用"的细节 | 绿色发展指标的关注点 |
| 林嫂那本记着用工、斤两、油渣去向的老账本 | 对相关指标信息的持续记录与收集 |
| 陈先生起初只想"把账做漂亮" | 忽视社会责任与绿色指标的常见心态 |
| 周先生不看报表，看油烟、去向、雇工 | 对国家产业战略 / 就业 / 绿色指标的外部审视 |
| 陈先生让小李专门对着老账本核对并按季上报 | 投后管理中督促被投资企业开展相关活动 |

| 林嫂说"这事儿本来就该我们这些人操心" | 履行社会责任的内在驱动力 |
