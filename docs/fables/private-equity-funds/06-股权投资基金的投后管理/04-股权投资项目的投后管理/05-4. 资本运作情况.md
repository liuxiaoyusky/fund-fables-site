# 4. 资本运作情况

## 🏺 寓言故事 —— 《老周的第三条路》

老周在县城东头开了二十年的酱菜厂，手艺是祖传的，萝卜干、酱黄瓜在方圆百里都有口碑。三年前，省城来的老吴看中了他的手艺，投了一笔钱，帮他把小作坊扩成了像样的工厂。老吴不参与经营，只是每年年底来厂里看看账本，和老周喝顿酒。

三年过去，厂子年利润翻了两番，老周却越来越睡不着。当初老吴投钱时，两人约定五年为期，现在还剩两年。老周心里清楚，老吴投钱不是图每年那点分红，他迟早要全身而退。可怎么退？老周想过两条路：一是自己把老吴的股份买下来，可他手头没有那么多现钱；二是把厂子卖给外地的大企业，可那些企业出价压得太低，老周舍不得祖传的牌子。两条路都走不通，老周愁得头发白了一半。

一个秋日的下午，老吴又来了。酒过三巡，老周把心事倒了出来。老吴听完，放下酒杯，从包里掏出一份报纸，指着上面一则不起眼的消息：邻县一家做腊味的厂子，上个月挂牌进了省城的一个公开买卖平台，原来的老股东把股份分批卖了出去，自己既保留了经营权，又拿到了一大笔钱。老周眼睛亮了，可随即又暗下去："我这酱菜厂，能走这条路吗？"老吴点点头："你的厂子规模够了，账目也清楚，只差一个懂行的人帮你牵线搭桥。"老吴说他认识一个专门帮企业走这条路的老朋友，从梳理账目到对接平台，全程都能帮上忙。老周握着酒杯，手微微发抖。他忽然意识到，自己一直以为只有"买下来"和"卖掉"两条路，原来还有第三条路——让厂子自己"长"出一条腿来，堂堂正正地走进更大的天地。

第二年春天，老周在厂里专门辟出一间屋子，老吴介绍来的老朋友带着两个年轻人住了进来。他们帮老周把二十年的账目理得清清楚楚，又跑了几趟省城，和各路办事的人逐一对接。老周还是每天凌晨四点起床腌菜，但心里踏实了。他知道，那条第三条路正在脚下慢慢铺成。老吴再来喝酒时，两人不再只谈今年的利润，而是谈厂子挂牌的进度、下一步该见哪些人。老周忽然觉得，老吴不是那个"迟早要走"的外人，而是那个帮他找到第三条路的人。

---

**📖 原文定义**

> 股权投资基金投资企业最终需要完成项目的退出，无论是境内外上市还是并购，均是比较好的退出路径。了解被投资企业的资本运作情况既是基金管理人评估投后企业价值、提供增值服务的重要基础，也是基金投资者非常关注的内容。投后管理人员应充分了解被投资企业的资本运作情况。
>
> 投后管理人员应定期与企业内部负责资本运作的高管联系，例如董秘、上市办主任等人员。在股东会、董事会等会议上应与其他股东、董事及高管及时沟通企业资本运作的最新动态。在被投资企业达到一定的规模、符合上市辅导的基础条件时，应协助企业选定证券公司、会计师事务所、律师事务所等上市服务中介机构。投后管理人员应与上述中介机构的项目组成员保持联系，及时了解企业资本运作的动态及遇到的难题，定期参与中介机构协调会等会议。

| 故事元素 | 概念对应 |

| 老吴三年前的投资约定五年为期 | 投资最终需要完成项目退出 |
| 老周愁"老吴怎么全身而退" | 基金投资者非常关注退出路径 |
| 老吴带来报纸上的公开买卖平台消息 | 评估投后企业价值、提供增值服务 |
| 老吴介绍老朋友帮老周梳理账目、对接平台 | 协助企业选定上市服务中介机构 |
| 老周辟出屋子给老朋友的团队住下跟进 | 投后管理人员定期与企业负责资本运作的高管联系 |
| 老周和老吴不再只谈利润、而谈挂牌进度 | 在股东会等会议上及时沟通资本运作最新动态 |
| 老朋友跑省城对接各路办事的人 | 与中介机构项目组成员保持联系、定期参与协调会 |
