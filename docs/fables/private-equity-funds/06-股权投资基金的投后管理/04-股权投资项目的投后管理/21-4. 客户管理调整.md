# 4. 客户管理调整

## 🎣 寓言故事 —— 《老周的鱼摊》

老周在城南菜市场卖了十五年鱼，摊位不大，口碑却硬。每天凌晨三点去码头挑货，五点出摊，中午收工，下午补觉，日子像钟表一样准。他的鱼新鲜、秤足、不欺客，回头客多，老周觉得自己这门生意稳得很。

三年前，老周的儿子小周大学毕业回来，非要帮父亲"升级"。小周在城里读过书，见过世面，觉得父亲那套"老好人"做法太落伍。他偷偷给父亲接了几个大单子——给城里三家连锁饭店供货，每月签单额翻了两番。老周起初高兴，后来慢慢觉出不对劲。饭店要的鱼种类杂，今天鲈鱼明天石斑后天带鱼，每样只要几十斤，却要求早上七点准时送到。老周得凌晨一点就起床分拣，一个人根本忙不过来，只好雇了两个人帮忙。更头疼的是账期——饭店说行业规矩，月结六十天。老周以前卖菜都是现钱现货，现在每个月有一大半流水压在饭店的欠条里，手头紧得像勒着一根绳子。他去找小周商量，小周却不以为然："爸，大单子才是正经生意，你那菜市场的散客，一天才卖几个钱？"

那年冬天，变故来了。码头最大的供货商突然涨价，老周手里的现钱不够进货，想去银行贷款，银行一看他的流水——账面收入好看，可实际到账的没几笔，摇头说不行。老周急得嘴上起了泡，去找饭店催款，采购经理推三阻四，说年底财务忙，再等等。老周站在饭店后厨门口，手里攥着一沓欠条，忽然意识到自己这三年好像一直在"忙"，却越忙越穷。他回到菜市场，找隔壁卖豆腐的老刘喝酒。老刘听完，放下酒杯说："老周，你那些大饭店，看着风光，其实把你当免费的钱袋子用。你不如算算，这三年你真正落进口袋的钱，到底比从前多还是少？"

老周一夜没睡，把三年的账本翻出来，一笔一笔对。不算不知道，一算吓一跳：签单额确实翻了两番，可扣掉雇人成本、压货损耗、资金占用的利息，真正落进口袋的钱，比从前只多了不到两成。而那些大饭店越来越挑剔，要货杂、交期急，老周每天疲于奔命，连以前挑鱼的本事都生疏了。更让他心惊的是，菜市场里几个从前跟他买鱼的小餐馆老板，这几年慢慢做大了，有的已经开了三家分店——他们当年找老周谈合作，老周嫌单子小，让小周打发走了，如今他们都成了别人的铁杆客户。老周看着账本，手有些发抖。他想起老刘的话，又想起自己父亲说过的一句老话："鱼贩子的钱，在水里也在秤上。"他以前只懂"秤"，不懂"水"——那水就是流动的钱，水停了，鱼就死了。

第二天，老周做了三件事。第一，他去找那三家饭店，提出要么缩短账期到十五天，要么减少供货量。两家同意了，一家翻脸换了供应商，老周反而松了口气。第二，他把雇的人辞了一个，重新调整自己的作息，凌晨三点挑货，五点出摊，把精力放回菜市场。第三，他让小周去跑那些当年被他拒绝的小餐馆，一家一家谈，账期短、需求稳定，虽然单笔小，但积少成多。小周起初不愿意，觉得"掉价"，老周说："你当年接的大单子，看着体面，其实是把咱家的血抽干给别人用。这些小馆子，现在小，将来未必小。你眼光放长，别只盯着今天的秤。"

一年后，老周的生意变了模样。大饭店只剩一家，账期十五天，规矩清楚。小餐馆发展了十几家，有几家已经开了第二家分店，老周是他们最早的供货商，情分在、账期短、利润稳。老周重新雇了人，但这次是帮自己打杂，不是疲于奔命。他每天还能准时收摊，下午去河边钓鱼——这是他十五年来的老习惯，中断三年，终于捡回来了。

一个周末，小周带着一个新客户来见老周——那是一家刚起步的生鲜电商，创始人是当年菜市场里一个卖调料的小伙子，如今拿了别人的钱，要做线上鱼市。他点名要找老周合作，说："周叔，我当年在您旁边卖调料，看您挑鱼看了三年，知道您的货是什么成色。"老周愣了一下，随即笑了。他忽然明白，客户这东西，不是越大越好，也不是越小越好。重要的是——这个人能不能让你的"水"活起来，能不能让你的秤上，称出真正的分量。

**📖 原文定义**

> 企业要焕发活力，销售环节的不断攻城略地、实现增长以增加企业的"进项"是非常必要的。而要实现增长，需对客户管理不断优化调整。
>
> 客户管理调整首先是需要对现有客户进行分析。常见的需要调整的问题包括：原销售部门和生产部门、财务部门脱节；销售部门主要看重签单额、回款额，对客户给企业带来的实际利润分析不够；有些大客户实际的毛利不高，或者毛利尚可，但要货的类别杂、交期急，占用了大量的生产和供应链资源；销售部门对客户所在的行业发展及最终客户不了解不熟悉，对一些现在还是中小规模但将来可能变成行业巨头的客户支持力度不足，最终可能会将这些"未来之子"送给竞争对手。
>
> 其次，客户的账期非常重要，有的企业在对客户评价及内部对销售团队绩效评价时，存在对账期因素考虑不足甚至完全不考虑的情况。如果企业的收入增长需要占用大量的营运资金，可能会影响企业的收入快速增长进而延缓退出，也可能需要进一步增资，导致成本增加进而影响回报率。如果投资了存在上述问题的企业，应该对此进行调整，在销售端树立账期意识，对于一些账期长、回款慢的客户，可以考虑放弃。
>
> 最后，在客户调整的过程中，新客户的获取及原有客户新需求的开发非常重要，需要在销售部门的内部考核时予以考虑。原有的销售团队可能会存在一种倾向，即注重维护以及缓慢拓展原领域客户，对开拓新领域客户重视不够。可以考虑从市场上招聘成熟的销售团队加入以产生"鲶鱼效应"。
>
> 对于消费品行业来说，可以和企业一起重新调研市场、分析竞品，或者直接运用自己已有的行业经验，对被投资企业的产品结构进行调整，对销售渠道进行优化，激发经销商的动力以适应终端消费者新的喜好，或者拓展新的消费者群体。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 小周只看签单额，忽视实际利润和账期 | 销售部门看重签单额、回款额，对实际利润分析不够 |
| 大饭店要货杂、交期急，占用老周大量人力和时间 | 要货类别杂、交期急，占用大量生产和供应链资源 |
| 老周当年拒绝的小餐馆后来做大，成了别人的客户 | 对中小规模但将来可能变成行业巨头的客户支持力度不足，将"未来之子"送给竞争对手 |
| 饭店月结六十天，老周资金被大量占用，进货困难 | 账期因素考虑不足，收入增长占用大量营运资金 |
| 老周缩短账期、放弃回款慢的客户 | 树立账期意识，对账期长、回款慢的客户考虑放弃 |
| 老周让小周去开拓小餐馆和新客户 | 新客户获取及原有客户新需求开发 |
| 小周只愿维护大饭店，不愿开拓小客户 | 原有销售团队注重维护原领域客户，对新领域重视不够 |
| 老周重新调研账本、调整产品结构（大饭店转小餐馆） | 重新调研市场、分析竞品，调整产品结构、优化销售渠道 |
| 生鲜电商创始人是当年菜市场老熟人 | 拓展新的消费者群体，适应终端新喜好 |
