# 3. 人员变动情况

## 👥 寓言故事 —— 《换将的棋局》

老周在城南经营一家茶庄，二十年来风雨无阻。三年前，他把手里的积蓄交给了一个叫阿明的年轻人，让他去邻县开一家分店。阿明是茶庄的老伙计，从十七岁跟着老周学辨茶、学待客，手艺和人品都没得说。老周拍着他的肩膀说："钱我出，店你管，赚了咱们分，赔了算我的。"阿明眼眶一红，当天就收拾行李去了邻县。

头两年，分店的生意蒸蒸日上。阿明亲自跑山选茶，和茶农喝酒交心，订下的货色比别家高出一等。账本上每月都有盈余，老周看着数字，心里踏实，渐渐不再过问细节，只是逢年过节收收账。可到了第三年春天，事情开始变了味。先是阿明的副手老陈突然辞了工，说是回老家养病。没过两个月，管账的孙姐也走了，新来的年轻人连算盘都打不利索。老周隐约觉得不对劲，但分店每个月的账目还是好看的，他便安慰自己：人各有志，走几个伙计不碍事。

转折发生在初夏的一天。老周去邻县走亲戚，顺路想给分店送一批新到的春茶。他推门进去，只见柜台后站着一个陌生的面孔，问他要什么茶，连龙井和毛尖都分不清。老周心里一沉，径直走进后院，发现阿明不在，问伙计才知道，阿明两个月前就很少来店里了，说是身体抱恙，实际上天天在县城的酒楼里应酬。老周连夜翻了近半年的账本，越看心越凉——账目上数字漂亮，可细究起来，进货价比往年高了两成，卖出的茶却说是"薄利多销"，仓库里的存货也对不上数。他这才明白，老陈和孙姐的离开不是偶然，是阿明变了。老陈性子直，看不惯阿明虚报账目，走了；孙姐管账，发现数字对不上，也走了。走掉的是人，留下的却是一个被掏空的壳子。

老周没有声张，第二天一早就把阿明叫到跟前。他没有骂，只是把账本摊开，一页一页地指给他看。阿明起初还嘴硬，后来额头冒汗，终于瘫坐在椅子上，承认自己迷上了赌钱，挪用了货款，为了掩盖窟窿，只好虚报高价进货、低价出货，把账面做成平的。老周沉默良久，最后说："店我收回，债你慢慢还。但你要记住，我当年投的不是这家店，是你这个人。人变了，店也就不是店了。"阿明低着头，一句话也说不出来。老周当天就搬进了分店，亲自坐镇，重新聘了识茶的掌柜和精明的账房。三个月后，分店扭亏为盈。老周在总店的墙上挂了一幅字，写的是：看人，要看他身边的人走没走。

**📖 原文定义**

> 普遍的投资观点认为，投企业即是投人、投团队。只有人投对了，投资才能够成功；只要人投对了，即使企业经营暂时遇到波折，最终经营团队也能克服困难走向成功，或者避免较大的损失。因此，在投后管理中对企业人员变动情况的了解非常重要。在投资时基金管理人一般会对企业的高管团队进行充分的尽职调查，在投后管理中需要对这些高管变动高度关注。当被投资企业频繁出现高级管理人员异动或离职等情况时，往往不是企业正常发展的表现。投后管理人员需要及时发现以上变动，并了解其离职或者岗位调整的真实原因。除高级管理人员变动以外，对部分中层或者基层岗位的变动也需要投后管理人员有一定的洞察力，例如财务人员的频繁变动、负责主要大客户的销售人员的变动。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周把积蓄交给阿明开店 | 投资行为 |
| 老周说"投的不是店，是你这个人" | 投企业即是投人、投团队 |
| 副手老陈突然辞工 | 高级管理人员异动或离职 |
| 管账的孙姐也走了 | 财务人员频繁变动 |
| 老周发现账目异常、阿明挪用货款 | 人员变动背后往往有异常原因 |
| 老周重新聘掌柜和账房 | 投后管理中对人员变动的应对 |
| 老周挂的字"看人，要看他身边的人走没走" | 对人员变动保持高度关注 |
