# 5. 协议条款执行情况

## 📜 寓言故事 —— 《签字之后》

那年春天，老吴在自家院子里摆了一桌酒，请镇上几位老主顾吃红烧肉。席间他举起酒杯，跟做布匹生意的何掌柜碰了一下：“老何，你那批细棉布的尾款，我打算宽限半年——但咱们得把话说清楚。”他让小儿子取来纸笔，当场写了三件事：半年内货款分两次结清；何掌柜每季要给他看一眼铺子的进出账；半年后若布价涨过两成，何掌柜须多付一成的“补偿”。何掌柜急着要货，连看都没细看就按了手印。满桌人哄笑起来：“老何，你连字都不念就按？”何掌柜红着脸：“都是老交情，签字不过是走个过场嘛。”

签完之后头三个月，老吴忙着自己的田地，几乎把那张纸忘在抽屉里。何掌柜也照常进货出货，铺子门庭若市。两人偶尔在茶馆碰面，只顾着聊哪里的鱼新鲜，谁都没提那页纸的事。老吴心里琢磨：“既然是老交情，何必催得那么紧？让人说我小气。”何掌柜则暗自得意：“这老吴，估计早忘了。”

到了第四个月，镇上忽然来了一位从省城回来的钱庄掌柜，说手里有一笔大钱要放出去，利息给得比别家高两成。何掌柜听了眼睛发亮，连夜把铺子里压箱底的好布都拿出来作抵押，跟钱庄签了新约。这事他没跟老吴说一个字。等第五个月的结账日到了，何掌柜才急匆匆跑来老吴家：“老吴哥，宽限几日，布行最近钱紧——”“紧？”老吴愣了一下，“你不是铺子红红火火？再说咱那纸上写得清楚，半年分两笔，第一笔就在这个月！”何掌柜支支吾吾，把脸别过去。原来他账上大部分银子，已经被那笔新债抽走了。

老吴心头一沉。他回家把那张纸翻出来，对着烛光一行一行地念。这时候他才看清楚：纸上写着“每季查账一次”，可他这三个月一次都没登过门；写着“若布价涨过两成须多付”，可现在压根没人告诉他布价涨了多少；写着“宽限期内不得向他人作大额借款”，何掌柜这一笔显然踩了线。老吴咬着牙去找镇上有威望的周先生评理。周先生听他说完，把那张纸往桌上一拍：“老吴，规矩是你定的，可你没去查；何掌柜虽然违约，但你若早点提醒，这场官司他根本不会输。”周先生转头对何掌柜说：“新债的事你得赶紧收手，否则雪球越滚越大，连累你一家老小。”最后两人重新议了还款时辰，何掌柜咬牙把铺里最值钱的一批绸缎折价抵给了老吴。

老吴回到家，把那张纸小心贴在墙上最显眼的位置。媳妇问：“不就是一桩买卖吗，至于挂起来？”老吴摇摇头说：“从按手印那天起，这张纸就一直管着我和何掌柜。我不盯它，它就不替我说话——到时候真出了事，倒霉的是我自己。”

---

**📖 原文定义**

> 在投后管理阶段，投后管理人员需定期核查协议条款的执行情况，保护自身的合法权益。当发现协议执行中存在重大风险或出现不确定性的情况时，应当立即采取相应的补救措施。此外，有的投资协议条款可能会规定一些其他交割后的事项和被投资企业应继续履行的义务，基金管理人也需要对这些条款的履行情况进行持续跟踪。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老吴写下三条约定并让何掌柜按手印 | 协议条款的明确约定 |
| 老吴头三个月一次都没登门查账 | 协议签署后未持续跟踪 |
| 何掌柜私下向钱庄借款抵押 | 被投资方未履行约定义务 |
| 老吴回家逐字重读那张纸 | 定期核查协议条款执行情况 |
| 发现何掌柜踩了“不得大额借款”的红线 | 识别协议执行中的重大风险 |
| 立刻请周先生评理并要求何掌柜收手 | 立即采取相应补救措施 |
| 何掌柜折价抵绸缎，重新议还款时辰 | 对协议执行偏差进行持续跟踪与纠正 |
| 老吴把那张纸贴在墙上最显眼处 | 对交割后应继续履行的义务保持长期关注 |
