# 7. 整合完成后的日常管理

## 🏺 寓言故事 —— 《老周的放权》

老周在镇上经营一家药铺，三代传承，口碑极好。可他心里清楚，这铺子要是只守着老方子，迟早被城里的大药房挤垮。三年前，他咬牙把铺子六成股份卖给了一个叫"合兴堂"的大商号，换来了银子和资源。合兴堂派人来整顿了半年，换了账本、理顺了进货渠道，老周的药铺焕然一新。

整顿结束那天，合兴堂派来的管事老张找到老周，说以后的日子怎么过，得拿个章程。老周心想：这铺子毕竟是我一手拉扯大的，你们出了钱，可懂什么药材？他嘴上没说什么，心里却打定主意——大事小事，还得我老周点头。

头一个月，老周每天天不亮就到铺子里，站在柜台后面盯着伙计们抓药。新请来的掌柜姓林，三十出头，以前在城里大药房当过差，精明干练。老周看他调配一味当归，忍不住插嘴："这当归得挑甘肃的，你拿云南的凑合可不行。"林掌柜笑了笑，没争辩，换了药材。过了几天，林掌柜打算进一批新设备，把炮制工序机械化，老周一听就摇头："祖上传下来的手艺，机器能替代？"林掌柜的计划就这么搁下了。

又过半月，铺子里的气氛越来越怪。伙计们有事不敢找林掌柜，全跑来问老周。林掌柜坐在账房里，对着一堆报表发呆——他连批个采购单，老周都要过一眼。有一天，老周发现库房里堆了半年的陈皮发了霉，质问林掌柜。林掌柜苦着脸说："周老，我半个月前就写了进货单，您说再等等，看能不能从老渠道压点价……"老周一时语塞，心里却憋着一股气：我这是为铺子省钱，有错吗？

那天晚上，老周独自坐在后院，看着月光照在晒药架上。他忽然想起父亲临终前说的话："周家的药铺能传下来，不是因为咱们什么都管，而是因为每一代都找到了能托付的人。"他望着那排空荡荡的架子，第一次问自己：我是不是把缰绳攥得太紧了？

第二天一早，老周把林掌柜、几个老伙计和合兴堂派来的代表都叫到了后院。他泡了一壶浓茶，开口说："从今天起，铺子里的事，林掌柜做主。我只在合兴堂跟咱们商定的'议事会'上露面，听听大方向。进货、用人、换设备，林掌柜定了就定。"他顿了顿，又补了一句："但我有个条件——每个月的账本、每年的盘点，得请镇上有名望的先生们来核一核。铺子里还得留个专管传话的，有事随时能找我说。伙计们要是有委屈，也能直接来找我喝茶。"

林掌柜愣住了，半晌才郑重地点头。

三个月后，药铺进了那批新设备，炮制效率翻了一倍。半年后，林掌柜从邻县挖来一个懂新药的师傅，铺子开始卖丸散膏丹之外的成药。老周不再每天盯在铺子里，但他每月初一都会来，不是查账，是跟伙计们吃顿便饭，听他们说说话。有一回，一个老伙计悄悄告诉他，库房里新进的黄芪有点潮。老周没声张，只是下次议事会上，轻描淡写地提了一句"库房通风该修了"。林掌柜心领神会，三天后就换了新窗。

一年后的腊月，药铺的流水比往年多了四成。老周站在重新装修的门脸前，看着林掌柜带着伙计们忙进忙出，忽然觉得，这铺子好像比他在柜台后面盯着的时候，更像"他的"了。他明白了：真正的托付，不是把钥匙攥在手里，而是把规矩立好，把路铺平，然后让人走。

**📖 原文定义**

> 对整合完成后的日常管理来说，基金管理人应主要从完善治理结构、形成纠错机制考虑，要充分信任以总经理为首的高级管理团队，不能越俎代庖，应主要在企业董事会层面发挥作用。在有小股东存在的情况下，作为控股股东也要尊重小股东的意见，在股东会层面依法决策企业的相关事务。一般来说，被并购企业的日常管理机构分为以下层次（见表6-16）。
> 
> 作为控股股东的代表，基金管理人可以决定董事会全部或大多数的成员，一般可以考虑由基金管理人分管该项目的高层、基金管理人该项目的投资负责人、基金管理人该项目的投后管理负责人、企业的总经理和其他关键高管、外部专家、小股东代表（如有）构成。在被投资企业规模比较大的情况下，还可以考虑设立战略、薪酬考核、提名、审计等专业委员会负责相关事项。
> 
> 企业内部可以安排专门的联系人，负责和基金管理人的相关负责人沟通企业信息、组织会议等工作；可以设立内部审计机构，负责企业内部控制的督察等专项工作；可以通过职工监事、职工代表大会、工会等渠道，使作为控股股东的基金管理人能够有与员工直接沟通对话的渠道。
> 
> 基金管理人应对年度的绩效考核及外部审计工作充分重视，不断优化考核评价体系，谨慎选聘服务的会计师事务所。使用内部或外部律师保证企业运行的合规性，避免风险发生。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周把日常经营权交给林掌柜，自己只在"议事会"上参与 | 控股股东不越俎代庖，主要在董事会层面发挥作用 |
| 老周要求每月账本、每年盘点请镇上先生核一核 | 重视年度绩效考核及外部审计，谨慎选聘会计师事务所 |
| 铺子里留专管传话的人，伙计有委屈可直接找老周 | 企业内部安排专门联系人沟通信息；通过职工渠道保持与员工直接对话 |
| 老周在议事会上轻描淡写提库房通风，林掌柜心领神会整改 | 完善治理结构、形成纠错机制，而非直接插手日常管理 |
| 老周每月初一跟伙计们吃便饭、听说话 | 控股股东与被投企业保持信息沟通和监督渠道 |
| 药铺流水增长，老周觉得铺子"更像自己的了" | 充分信任管理团队，治理到位后企业价值提升 |
