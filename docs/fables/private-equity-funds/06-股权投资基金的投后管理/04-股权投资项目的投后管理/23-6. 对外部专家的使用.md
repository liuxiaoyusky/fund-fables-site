# 6. 对外部专家的使用

## 🔍 寓言故事 —— 《请先生》

宣府镇外有一家"永昌号"，掌柜林崇德做了三十多年皮货。三年前儿子林瑞接手，老林琢磨：光守着摊位迟早被挤垮，不如直接吃下一整间染坊，从皮到色到成衣全包在自己手里。

可老林越打听越心虚。染坊在杭州，账本怎么读、契书怎么签、作坊里几十号工人的手脚干不干净，他一无所知；皮货同行里这种大买卖的先例他听都没听过；再说就算吞下来，染坊常年给洋行供货，水路关务、汇率钱粮又是一大摊子。父子俩掰着手指头算：找齐这些行家自己养一支队伍，少说得七八个人，一年光嚼裹就几千两银子——何况生意做完人就闲下来，养不起也留不住。

老林揉了揉太阳穴，对儿子说："瑞儿，这桩买卖咱们自己硬撑怕是要栽。"林瑞问："那怎么办？"老林沉吟半晌："请先生。"

林瑞先去了趟省城，托人请来姓沈的"账房先生"。沈先生早年在大钱庄做账，又给两家布行清过盘，眼光毒得很。他花了六天把染坊近三年的进出账过了一遍，拍着桌子对林瑞说："林少东家，这账面花团锦簇，底下埋着三笔死账——给南洋商行的赊款两年没收回来、染缸里那批靛料掺了假没销、掌柜小舅子借了铺子二百两三年没补借条。按账面价吃进来，光这三笔窟窿就抵掉一年的利。"林瑞后背一层冷汗。当场让沈先生写了一份"账外账"的清单，谈判时凭这份清单硬生生把原价砍下来一成半。

账弄清了，契书更麻烦。永昌号自家的契纸从没超过三页，可染坊这事关涉七方：原东家、作坊师傅、洋行买办、衙门税契、伙计工契、邻铺优先购买权、还有一笔没还清的旧债。林瑞花重金从京城请来姓庄的"状师先生"。庄先生六十出头，给盐商打过几场大官司，一眼看出原契里两个坑：优先购买权写得不严密，将来邻铺反悔可能扯皮；旧债那一条，债权人若上门讨，永昌号很可能要替原东家背一半。他连夜改了六稿，又跑去邻铺和衙门逐一打招呼、补文押印。二十天，契书改到第七版，林瑞才敢落笔。

价定下、契签完，林瑞又先后请来三位"行内先生"：做了三十年蓝染的老师傅坐镇作坊把控成色；跑惯了南洋水路的船行老掌柜专管运价和关文；曾在洋行做过通译的周先生专管与洋人买办接话。三人各管一摊，每月来铺子一两趟，遇大事随叫随到。酬金远比养一个常年守在铺里的"自己人"贵，但算总账省下了大笔的人头开销。

半年后，染坊的蓝靛染得比从前更稳，洋行订单一期没误过。林瑞细细算了笔账：若硬要自己养一套人马，光头两年的嚼裹钱就够把几位先生的酬金翻着番地花出去。老林端起茶碗抿了一口："不是咱们聪明——这世上的行当太多，自己样样都精那不叫生意，那叫神话。"

**📖 原文定义**

> 对绝大多数基金管理人来说，并购投资是个低频次、大金额的交易，且每次并购可能发生在不同行业、不同区域，往往呈现明显的个性化特征，组建及运营一支全知全能的并购团队既不经济也不现实。基金管理人在并购交易执行和投后管理时往往采用大量的外部专家团队。对于跨国交易或者全球运营的被并购标的，可能还要在相关国家聘任相应的专家团队（见表6-15）。

> 一般来说，并购投资比创业投资的交易金额大，且基金管理人成为控股股东，要对企业的重大事务进行决策、日常管理进行监督，而非创业投资作为小股东从旁协助工作。在外部专家的聘请上也要更加慎重，要选用有实际经验和高水平的外部专家。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 林家父子盘算"养一支七八人的队伍要几千两银子" | 组建及运营全知全能的并购团队不经济也不现实 |
| 林瑞去省城请来沈先生查染坊账目 | 引入外部财务专家识别账外风险 |
| 沈先生挑出三笔死账，谈判压价一成半 | 外部专家在交易执行中发挥关键作用 |
| 从京城请来庄状师审改契书六稿 | 引入外部法律专家处理复杂多方协议 |
| 庄先生发现优先购买权与旧债两个坑 | 对大金额交易的法律风险进行专业把关 |
| 老林请来蓝染师傅、船行掌柜、洋行通译三位行内先生 | 在不同行业领域聘任相应的外部专家团队 |
| 三位先生每月来铺子一两趟，遇大事随叫随到 | 外部专家团队的常态支持模式 |
| 林瑞算总账：酬金远比自己养人头便宜 | 外部专家团队相比自建团队的成本优势 |
| 老林最后说"自己样样都精那不叫生意那叫神话" | 承认专业分工的边界，理性选择外部资源 |
| 林瑞挑选先生时反复掂量资历与经验 | 选用有实际经验和高水平的外部专家 |
