# 5. 外部拓展

## 🌉 寓言故事 —— 《老周的桥》

老周在江南开了二十年的纺织作坊，手艺好，工人踏实，但生意始终困在方圆三百里。他想过往外走，可一没门路，二缺本钱，三怕外面的水太深。每年秋天，他站在码头看着别人的货船南下北上，只能叹口气，转身回作坊。

那年开春，一个叫老钱的商人找到他，说想一起做事。老钱走南闯北多年，认识不少做布匹买卖的人，手头也有些活钱。老周起初犹豫——自己作坊虽小，却是祖业，怕外人插手坏了根基。但老钱不急着要答复，只隔三差五带些外地的样品来，让老周看看别人的花样和销路。老周越看越心惊：原来同样的料子，在别处能卖出翻倍的价格；原来有些作坊比他小一半，却因为搭上了大船队，一年能走他三年的量。他夜里睡不着，翻来覆去算一笔账：守着老地盘，稳是稳，可天花板就在头顶；往外走一步，摔了可能爬不起来，但不走，迟早被憋死。

梅雨季节，一场大水冲垮了老周作坊后院的仓库，刚收上来的生丝泡了汤。老周急得像热锅上的蚂蚁，想修仓库没钱，想赊货没人肯借——小作坊的信用，在灾年一文不值。老钱知道后，没提入股的事，直接牵线让老周从相熟的船队那里赊了一批货，又托自己在钱庄的朋友给老周转了一笔救急的钱。老周问老图什么，老钱只说："你先活下来，咱们再谈以后。"那场雨让老周明白，自己这些年不是不想走，是缺一座桥——缺一个能在关键时刻把外面的水引进来的人。

水退之后，老周主动找老钱喝了顿酒。他说，作坊还是他管，但老钱负责两件事：一是用他的人脉把货卖到原先够不着的地方，二是以后作坊要扩、要改、要过桥，老钱得帮忙想办法。老钱点头，又补了一句："我不光卖你的货，我认识的人里头，有做染坊的、有跑北方的船队、甚至有在海外开厂的——这些以后都能接上。"老周听着，想起多年前码头上那些远去的船帆。这一次，他觉得自己也终于有了一条船。

**📖 原文定义**

> 以上组织结构调整、战略调整、生产和供应链管理调整、客户管理调整更多是从被投资企业内部进行调整，从而促进企业内生增长和发展。但基金管理人更专业的，或者说被投资企业的员工对基金管理人更期待的应该是外部的拓展。
>
> 基金管理人自身拥有比较充足的资金或者比较丰富的金融资源，可以使用自有资金，或者通过银行借款、融资租赁、供应链金融、向其他基金管理人股权融资等金融手段为企业的发展提供持续的资金支持。
>
> 被投资企业可能只在局部区域经营，产品也可能只应用在局部领域，基金管理人通过自身产业链资源的嫁接，可以为企业跨区域经营、产品新增应用领域提供帮助。基金管理人现有投资组合中，可能就有适合与被投资企业整合的产业链上下游企业或者其他地区的同类型企业。有的基金管理人还具有协助企业海外并购的能力，从技术提升、专利获取、品牌收购、市场渠道开拓、供应链保证等角度为被投资企业提供全球的并购标的，并具有交易执行能力。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周的纺织作坊困在方圆三百里 | 被投资企业只在局部区域经营 |
| 老钱走南闯北的人脉和手头活钱 | 基金管理人拥有丰富的金融资源和产业链资源 |
| 大水后老钱牵线赊货、转贷救急 | 使用金融手段为企业提供持续的资金支持 |
| 老钱帮老周把货卖到够不着的地方 | 跨区域经营、开拓新市场渠道 |
| 老钱提到染坊、船队、海外开厂的资源可接上 | 产业链上下游资源整合、全球并购标的 |
| 老周负责作坊，老钱负责外部搭桥 | 内部管理与外部拓展的分工 |
