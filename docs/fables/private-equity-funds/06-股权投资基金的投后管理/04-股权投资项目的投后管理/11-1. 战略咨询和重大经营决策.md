# 1. 战略咨询和重大经营决策

## 🎯 寓言故事 —— 《老周的棋盘》

山城里有个做铁壶的老匠人叫周德山，做了三十年手工铁壶，手艺远近闻名。他有个徒弟叫阿明，年轻气盛，脑子活，三年前自立门户，在城里开了一家"明记铁器坊"，想把手艺做成大生意。周德山心疼徒弟，也看好这行当，便把自己攒下的十五两银子投了进去，换了三成股。两人约定：铺子的大事，得师徒俩都点头才能办。

阿明拿到银子，眼睛都亮了。他天天往茶馆跑，听来往客商说，邻县有个"王记铁器"正在到处开店，价格低、嗓门大，已经把半条街的生意抢走了。阿明急得睡不着，连夜写了一份大计划：把铁壶价钱压到成本线以下，再雇十个伙计挑着担子去四乡八镇叫卖，先让所有人都知道"明记"的名号，把对手挤垮再说。他算了算，这一仗打下来，至少再要三十两银子。

第二天一早，阿明兴冲冲地跑到师父家里，把计划往桌上一拍。周德山戴上老花镜，一行一行地看完，半晌没说话。阿明急了："师父，王记都快骑到咱脖子上了，再不动手，客人全跑光了！"周德山放下计划，端起茶碗抿了一口："阿明，我问你，咱的铁壶，和王记的比起来，好在哪儿？"阿明一愣："咱是手工打的，一锤一锤出来的，结实耐用。"周德山点点头："那王记的呢？"阿明撇撇嘴："机器压的，薄，用两年就漏水。"

周德山站起身，走到院子里，指着墙角一口旧铁壶："这口壶，是我师父传给我的，用了四十年，还在灶上蹲着。客人买壶，不是买牌子，是买一口能传家的壶。你现在去跟他拼价钱、拼嗓门，就算赢了，客人记住的是'便宜'，不是'好'。等银子烧完了，你拿什么撑？"

阿明脸涨得通红，觉得师父老了，胆子小。可当初白纸黑字写得清楚，大事要两个人都同意。周德山不点头，这计划就过不了。师徒俩僵在屋里，茶都凉了。

就在这时，隔壁镇传来消息：王记铁器为了抢客，把价钱压到了赔本的地步，还白送茶壶，说是"先占住灶头，以后靠修壶赚钱"。可买回去的人一用就发现，壶嘴漏水、壶底发黑，气得直接扔进了柴堆。王记收不到修壶的钱，白送的又收不回本，不到半年，库房里的银子见了底，伙计们工钱发不出，跑了一大半。掌柜的急得满嘴起泡，只能关了三家铺子，缩回老窝喘气。

阿明听到这消息，站在自家铺子门口，看着墙上挂着的"明记"招牌，出了一身冷汗。他想起师父那句话："客人记住的是'便宜'，不是'好'。"要是当初自己一意孤行，现在焦头烂额的，恐怕就是自己了。

那天晚上，阿明提着一壶酒去了师父家。酒过三巡，他红着脸说："师父，我想把计划改改。不拼价钱了，银子拿去请两个好师傅，把壶的样式再琢磨琢磨，再挑几把最好的壶，送给城里几个有头有脸的老茶客，让他们用，让人看。"周德山听完，第一次笑了，端起酒杯："这才像话。"

三年后，明记铁器坊的铁壶成了山城茶客口中的"一壶传三代"，价钱比王记贵一倍，照样供不应求。而王记铁器几经挣扎，终究没翻过身来，招牌都被人拆了当柴烧。阿明坐在铺子后院的梧桐树下，看着账本，忽然明白了：当年师父投进来的那十五两银子，不只是银子——那里面还藏着三十年看尽起落的眼光。有人替你按住冲动的手，不是拦你，是扶你。

**📖 原文定义**

> 基金管理人往往拥有更为宏观系统的行业认识与丰富的企业发展经验，识别和解决管理问题的经验相对丰富，基金管理人向被投资企业管理层提供一些有价值的经营管理思路和方法，有助于被投资企业进行更好的战略规划与规范管理、优化制度流程、提升市场影响力和品牌度等，也有助于被投资企业管理层较早觉察到企业未来可能出现的问题，降低潜在的运行风险。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 周德山三十年做铁壶的行业经验 | 宏观系统的行业认识与丰富的发展经验 |
| 周德山指出阿明计划的风险（烧钱抢市场后难持续） | 识别和解决管理问题的经验 |
| 周德山建议阿明先打磨产品、再建口碑 | 提供有价值的经营管理思路和方法 |
| 周德山以三成股身份参与重大决策，否决激进计划 | 协助进行战略规划与规范管理 |
| 阿明最终选择提升产品品质而非低价竞争 | 优化制度流程、提升市场影响力和品牌度 |
| 王记铁器因激进扩张而失败，明记因稳健而胜出 | 较早觉察未来可能出现的问题，降低潜在运行风险 |
