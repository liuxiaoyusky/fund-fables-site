# 5. 协助进行后续再融资、上市以及并购重组

## 🏺 寓言故事 —— 《老周的引路人》

老周在城东开了家酱菜铺，手艺是祖传的，腌的萝卜脆、泡的豆角酸，街坊邻居都爱买。十年下来，铺子从一间小平房扩成了三间门面，雇了七八个伙计。可老周心里清楚，酱菜这东西，出了城就没人认。他想把作坊搬到大码头边上，租个带地窖的大仓库，再招一批工人，把货卖到隔壁省去。

一算账，老周犯了难。搬仓库、买设备、招人手，少说也要再凑一大笔银子。他跑了几家钱庄，掌柜的都摇头："你这铺子是好，可没地契、没大仓，借多了我们心里没底。"老周坐在门槛上抽了半袋旱烟，烟圈一圈圈往上飘，他的眉头却越锁越紧。靠自己攒，怕是要等到胡子白；可这一步迈不出去，一辈子就困在城东这三间门面里。

这时候，老周想起了一个人——李掌柜。三年前，李掌柜看中了老周的铺子，出了一笔银子入了伙，平时不干涉做酱菜，只年底来对账。李掌柜在城里做了二十年买卖，认识的钱庄掌柜、码头把头、外乡客商不计其数。老周硬着头皮去找他，还没开口，李掌柜就笑了："我早猜到你会有这一天。"

李掌柜没直接掏银子。他先是带老周去见了一位做南北货的王老板，王老板正好想在酱菜行当里投一笔，李掌柜居中一撮合，老周拿到了第一笔扩充的银子。接着，李掌柜又引荐老周认识了码头边上最大的仓库东家，租金比市价低了两成。他还帮老周物色了一位从省城大酱园退下来的老师傅，专门教工人们标准化的腌制法。老周的作坊搬进了大仓库，第一批货发往邻省的时候，他站在码头上，看着船帆渐渐远去，忽然明白了李掌柜这三年来一直默默铺路的用意。

五年后，老周的酱菜铺子成了方圆百里最大的作坊，名字挂在城门口的石牌坊上。那年秋天，李掌柜又帮老周做了一件大事——他引荐了几位在京城做买卖的东家，把老周的作坊并入了更大的南北货商帮。老周不再是城东一间铺子的掌柜，而成了商帮里拿份子的东家之一。他偶尔还会回老铺子看看，站在当年坐过的门槛上，想起那个抽着旱烟犯愁的下午。没有李掌柜的那张关系网，他大概还在三间门面里腌萝卜。

---

**📖 原文定义**

> 一般而言，被投资企业擅长自身的业务发展，而对于如何进行资本规划、股票公开发行和上市、并购整合较为薄弱；与此相反，基金管理人对资本市场的熟悉程度较高、资本运作能力较强，在为被投资企业提供资本运作增值服务方面，基金管理人可以承担更多的责任。首先，在被投资企业的发展壮大过程中后续再融资非常重要，如果被投资企业需要引入新的股权资金或者债权资金，基金管理人往往会利用自己在资本市场和借贷市场上的关系网络，引荐其他基金管理人或商业银行。对于初创成长性企业来说，由于一般融资相关的专业人才不足，基金管理人的人员可以从旁提供协助和咨询的工作。同时，也正是由于基金管理人能够给被投资企业提供潜在的持续融资机会，被投资企业往往倾向于选择那些在投资行业内具有广泛关系网络的基金管理人，保证后续轮次的融资活动顺利进行。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周擅长做酱菜但不懂筹钱 | 被投资企业擅长自身业务发展，资本运作薄弱 |
| 李掌柜认识钱庄、客商、仓库东家 | 基金管理人拥有资本市场和借贷市场的关系网络 |
| 李掌柜引荐王老板投资、介绍仓库、物色老师傅 | 基金管理人利用关系网络引荐其他资金方，提供协助和咨询 |
| 老周最终并入南北货商帮 | 后续并购重组 |
| 老周当初选择李掌柜入伙 | 被投资企业倾向于选择有广泛关系网络的基金管理人 |
