# （一）信息获取的渠道与方式

## 🔍 寓言故事 —— 《老周的千里眼》

老周是城里一个做瓷器的匠人，手艺好，眼光也准。三年前，他把大半辈子的积蓄投给了邻县一家新开的瓷窑，占了三成股。对方掌柜姓马，人称马掌柜，是个精明人，烧出来的青花碗在市面上颇受欢迎。

起初，马掌柜每个月都派人送来账本，老周翻一翻，见数字好看，便安心做自己的活计。可日子久了，账本越来越薄，送来的间隔也越来越长。老周心里犯嘀咕，却不好意思总去问——毕竟自己只是个出钱的外人，瓷窑的日常他插不上手，马掌柜怎么烧、怎么卖、怎么管人，他一概不知。这种"知道一点、又不知道全部"的滋味，比完全不知道还难受。

转折发生在那年秋天。老周去邻县走亲戚，顺路想瞧瞧瓷窑，却在镇上的茶馆里听到几个同行闲聊，说马掌柜最近接了一笔大单子，急着赶工，把原先定好的釉料配方改了，换成了便宜的料。老周心头一紧，那配方是他当初最看重的，改了料，碗的成色必然下降。他连夜赶到瓷窑，马掌柜倒是笑脸相迎，领他看了新出的货，表面光鲜，老周却端起一只碗对着灯一照，釉下隐隐有气泡——这是偷工减料的痕迹。

老周没有当场发作。他回到城里，把自己关在屋里想了三天。他明白，自己不能继续当个"等账本的闲人"了。他要亲眼看见、亲耳听见，才能知道瓷窑真正的模样。

第二天，老周做了一件他以前从未做过的事——他主动给马掌柜写了一封信，说自己作为股东，想定期参加瓷窑的"议事会"。马掌柜不好拒绝，答应了。第一次议事会上，老周坐在角落里，听马掌柜讲下一季度的打算、听管账的先生报收支、听烧窑的老师傅谈火候的难处。他不多说话，只是听，但听得极认真。渐渐地，他从这些会议里摸清了瓷窑真正的脉络：哪些决策关乎长远，哪些开支藏着隐患，哪些人事变动可能影响手艺传承。

与此同时，老周不再只等那越来越薄的账本。他每隔两个月亲自去一趟瓷窑，不提前打招呼，到了就钻进作坊，跟老师傅们蹲在一起吃晌午饭，聊窑温、聊釉色、聊最近走货顺不顺。老师傅们起初拘谨，后来见老周是真懂行，话便多了。有一次，一个老烧工无意间提起，马掌柜为了赶那笔大单，把两个学徒的学徒期缩了一半，老周心里又记下一笔。

老周还养成了一个习惯：每次去邻县，必在茶馆里坐上半晌，听南来北往的客商聊行情。他也去拜访做釉料的老朋友，打听最近料价的涨跌和品质的变动。他甚至托人从外地捎来几本同业印行的册子，上面记着各地瓷窑的出货价和新花样。他把这些碎片拼在一起，渐渐拼出了一幅瓷窑的全景——不是马掌柜给他看的那一幅，而是他自己亲手拼出来的那一幅。

一年后，马掌柜又提出改配方，这次是为了省成本。老周在议事会上投了反对票，他手里握着三成股，马掌柜不得不认真考虑。老周没有说"你偷工减料"，而是把茶馆里听来的客商抱怨、老师傅们私下里的担忧、同业册子上别家瓷窑因降质而丢单的例子，一件一件摆出来。马掌柜沉默了半晌，最终收回了提议。

从那以后，马掌柜对老周多了几分敬重。不是因为老周有钱，而是因为老周"看得见"——不是隔着账本遥遥地望，而是走进作坊、坐在茶馆、翻开册子，用一双眼睛、两只耳朵，把瓷窑的里里外外都看在眼里。老周自己也明白了一个道理：当你把积蓄放进别人的窑火里，你就不能站在远处等烟散了才知道结果。你得走近些，再近些，从各个门缝里看进去，才能知道那火是烧得正旺，还是快要灭了。

**📖 原文定义**

> 基于相关法律法规对股东权利的保障，以及投资协议的约定，投后管理的负责人员一般通过如下几种渠道和方式获取被投资企业的信息：
>
> 1. 参加被投资企业股东会、董事会和监事会
>
> 基金管理人通过参与股东会、董事会和监事会，可以全面了解与公司发展相关的重要信息，并通过行使相应投票权促进被投资企业的良性发展。其中，股东会是公司的最高权力机构，由全体股东组成，负责修改公司章程，聘任和解聘公司董事，并对公司的上市、增资、减资、利润分配等重大事项进行决策。董事会负责批准公司发展战略、批准公司年度财务预算与决算、聘任和解聘公司高级管理人员以及决定公司高级管理人员的考核薪酬与激励制度等重要决策。监事会作为公司内部专门行使监督权的监督机构，对公司董事和高管的行为是否符合法律法规和公司章程行使监督权。
>
> 2. 关注被投资企业经营状况
>
> 在投资后直至投资退出的全过程中，通常情况下被投资企业有义务根据投资协议约定及时向股权投资基金提供与企业经营状况相关的报告，包括月度报告、季度报告、半年度报告、年度报告和有关专项报告等。基金管理人可以通过被投资企业提交的经营报告了解企业业务进展情况，从而对项目发展进行有效把握。
>
> 3. 日常联络与沟通工作
>
> 股权投资基金通常采取电话或会面、到企业实地考察等方式与被投资企业主要管理人员进行沟通接触，目的是了解企业的日常经营情况，并对其进行有针对性的咨询辅导和帮助。通常情况下，很多投后管理工作都是在与被投资企业定期或不定期的沟通中完成的。
>
> 4. 外部调研
>
> 股权投资基金可通过外部调研，例如拜访行业协会、科研机构，参加行业会议，阅读同行业企业的公开资料等方式侧面了解被投资企业的信息。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周参加瓷窑的"议事会"并听取各方汇报 | 参加被投资企业股东会、董事会和监事会 |
| 老周从会议中摸清瓷窑的决策、开支和人事脉络 | 通过参与会议全面了解公司发展相关重要信息 |
| 老周不再只等账本，而是定期亲自去瓷窑查看 | 关注被投资企业经营状况（经营报告） |
| 老周蹲在作坊跟老师傅吃晌午饭聊窑温走货 | 日常联络与沟通工作（电话/会面/实地考察） |
| 老周在茶馆听客商聊行情、拜访釉料老友、读同业册子 | 外部调研（行业协会/公开资料/行业会议） |
| 老周把各方碎片信息拼成瓷窑全景后，在议事会上有效反对改配方 | 多渠道信息获取后，有针对性地开展投后管理工作 |
