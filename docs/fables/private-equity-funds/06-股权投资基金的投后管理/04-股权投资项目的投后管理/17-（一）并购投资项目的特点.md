# （一）并购投资项目的特点

## 🎯 寓言故事 —— 《老周的船厂》

老周在江边经营一家修船厂，干了三十年，手艺好，口碑硬。厂子不大，二十几个工人，每年稳稳当当赚个温饱。老周今年五十六，儿子在国外读书，老伴身体不太好，他心里盘算着再干几年就退休，把厂子交给徒弟小赵。

可就在开春时，城里来了一位姓钱的商人，带着一队人马，提出要买下老周的船厂。老周起初不答应——这是他的命根子。但钱商人开出了他无法拒绝的条件：三倍于市价的钱，当场付清，而且老周可以继续留在厂里当总管，年薪比他自己经营时还高。老周心动了。他算了算，老伴的医药费、儿子的学费、自己养老的钱，一下子全有了着落。他签下了字，把船厂的大半股份卖了出去。

钱商人成了船厂的新东家。他做的第一件事，就是换掉老周的账房先生，换上自己从省城带来的财务班子。第二件事，把老周原来定下的"慢工出细活"规矩改成"按件计酬、限期交货"。老周站在车间里，看着工人们手忙脚乱赶工期，心里不是滋味。他去找钱商人商量，说这样赶出来的船质量不稳，会砸招牌。钱商人笑了笑，说招牌是他的事，老周只管把活干好。老周这才明白，自己虽然还叫"总管"，但船厂已经不是他说了算的地方。钱商人决定做什么船、接什么单、用哪家的木料，老周只有听命的份。他从一个说了算的人，变成了一个拿工钱的人。

一年后，钱商人把船厂转手卖给了一家更大的商会，赚了一笔可观的差价。老周站在江边，看着来来往往的新面孔，忽然想起自己当年在船厂门口挂的那块木牌子。牌子还在，只是上面的字已经换了。他摸了摸口袋里厚实的积蓄，说不清自己是赚了，还是丢了什么。

---

**📖 原文定义**

> 并购投资指的是股权投资基金收购被投资企业全部或者大部分股权，成为企业的控股股东。与创业投资不同的是，基金管理人在被投资企业经营中承担了主导角色，企业原先的大股东退出企业，或者成为小股东兼高管，或者成为无股权的"打工人"。对于基金管理人来说，基金成为控股股东后可以决定企业的经营和财务决策，可以掌握包括董事和高级管理人员的人事任命权。
>
> 由于并购企业需要准备大量的资金，有复杂烦琐的交易谈判过程，方案设计、尽职调查、投资交割需要充足的专业人员来落实，并购交易的投后管理需要有丰富企业管理经验的管理专家组织协同其他专业人员来进行，只有大型的基金管理人才会进行并购投资交易。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 钱商人买下老周船厂的大半股份 | 收购被投资企业全部或大部分股权，成为控股股东 |
| 钱商人换掉账房、改变规矩、决定经营方向 | 基金管理人在被投资企业经营中承担主导角色 |
| 老周从老板变成拿工钱的总管 | 企业原先的大股东退出，成为无股权的"打工人" |
| 钱商人一年后把船厂转卖给更大的商会 | 并购投资需要准备大量资金、复杂谈判，由大型机构进行 |
