# （二）收入分摊原则

## ⚖️ 寓言故事 —— 《四家酒坊的秋账》

江南水乡的沈家渡口，四家酒坊共用一口老井，按早年老族长立下的字据，井水四家平分。井边立着一块青石碑，碑上刻着四家名字和当时各家出的修井银子——钱家出得最多，林、孙、沈三家各出一半。每年秋后清账，由四家轮流当"司账"。

这一年秋后，轮到钱家少爷钱秉文当司账。他还没动笔，就听见隔壁林家嫂子在院子里嚷："上个月我酿了二十坛米酒，凭啥我只能分六坛的钱？"钱秉文翻开账本：四家虽然都从井里挑水，但出钱修井的"份数"不一样——钱家占四成，林、孙、沈三家各占两成。按碑上的比例，林家嫂子这月只该分到总收入的五分之一。

林家嫂子听了脸涨得通红："我干得最多，分得最少？"钱秉文正要答，孙家老爷子拄着拐杖慢慢走进来。他看着林家嫂子问："上月你酿的那二十坛米酒，第三天翻缸时发酸半坛，倒掉才压住味——这亏你记了没有？""记了。"老爷子又问："你家老二摔了腰那月，雇沈家小工帮工两日，赊了三百文工钱，秋后一起算，这你也记了吧？"林家嫂子点点头。

老爷子捻着胡子说："你家的酸酒、帮工钱，是你家当月的事，记在你自家小账上。可这口井是四家共有的，井里流出的水卖成了钱，得按当年出钱修井的份来分——这是老族长立字据时就定好的。出力是你的本事，记在你功劳簿上；卖酒的钱用的是大家的井水，得按碑上公家那份分。两笔账不能混。"

林家嫂子愣了半晌，低声说："秉文，你接着算吧，按碑上的份来。"顿了顿又补一句："只是我家老二摔腰那月、还有酸酒那月，我自己小账上的亏，你也得记上。"

钱秉文提笔在账本上写下："按碑上份额分；各家当月自记的盈亏，归各家。"夕阳把青石碑的影子斜斜地拉在井沿上。

---

**📖 原文定义**

> 项目投资相关收入的分摊原则与收益分配原则一致。对于基金仍持有的项目投资，在计算分摊金额时应假设在该时点进行分配，以计算每个投资者应得的金额。
>
> 除项目相关收入外，基金通过银行存款、购买结构性存款、货币市场基金等现金管理获得的超出本金部分的投资收入，根据基金合同的约定，通常按照投资者的实缴出资比例进行分摊。对于投资者未能按期履行实缴出资义务引发的违约金收入，通常应按照违约行为发生时点，在除违约投资者以外的其他守约投资者之间，按各自在上述时点的实缴出资额比例进行分摊。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 青石碑上刻的四家修井份数 | 合同约定的分摊比例 |
| 钱家占四成、其余三家各两成 | "实缴出资比例" |
| 卖酒的钱按修井份来分 | 现金管理收入按比例分摊 |
| 林家嫂子酿了二十坛却不能多拿 | 出力与分钱是两本账 |
| 各家当月酸酒、错放盐、帮工亏 | 各家当期自记的盈亏归属 |
| 林家嫂子最终同意按碑上份额分 | 收入分摊与收益分配原则一致 |
