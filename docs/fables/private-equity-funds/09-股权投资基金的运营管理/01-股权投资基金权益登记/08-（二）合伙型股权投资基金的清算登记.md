# （二）合伙型股权投资基金的清算登记

## 📜 寓言故事 —— 《散伙那天》

民国二十一年的秋天，青柳埠运河边那家"合记栈"，开了整整九年。

这一日，孙兆庭把周老板和赵老大请到老地方——当年的那间茶馆。三人坐定，茶过三巡，孙兆庭从袖子里取出一本账簿，轻轻放在桌上。账簿封皮上写着两个字：**结业**。

周老板翻了几页账，眉头舒展开来："九年，咱们从运河边那个绸缎小铺，做到如今四间分号、五条船。今年南边丝价一路走低，分号里最后一批绸也清了，再做下去没甚好做的了。"

赵老大抿了口茶，搁下杯子："我身子也一年不如一年，撑不动了。"

三人在茶馆里议了一夜，定下章程：账上还有多少银、欠谁的债、铺面家具怎么折、剩的银两怎么分——一笔一笔落在纸上。末了，周老板提笔写下散伙的日期，赵老大按上手印，孙兆庭签上名字。

第二天，孙兆庭揣着账簿去找师爷。师爷翻了几页，问："几位东家共同议定要散？"

"是。"孙兆庭答。

"可有立字据？"

孙兆庭取出昨夜那份写定的散伙章程——谁出面操持散伙、铺面怎么处置、欠的债怎么清、剩下的银两怎么分——一样一样白纸黑字。

师爷点点头，又问："散伙期间，这桩买卖还算不算活着？"

孙兆庭一愣。师爷说："算。但有一桩规矩——散伙的这段日子里，铺面虽然还挂'合记栈'的招牌，却不准再接新买卖、不能再进货出货。能做的，只有清账、分银、还债这一件事。"

回茶馆一议，三人公推孙兆庭为操持散伙事务的人。周老板有些迟疑："散伙归散伙，铺面里还有伙计跟着咱们吃饭，他们的工钱、欠的税、欠别人的货款，可得先结清，剩下的银两才能动。"

孙兆庭点点头，提笔重新排了一遍顺序：先付操持散伙的花费，再付伙计们的工钱，再补各项欠款，再清欠别人的货款——这些都结清之后，账上最后剩的银两，才轮到三位东家来分。

周老板在旁又补一句："清完账，咱们还要去一趟县衙——把咱们操持散伙这事的几个人是谁、怎么定下来的，向官面上交代明白。"

孙兆庭去县衙时，带着三样东西：操持散伙事务之人的名册、那份散伙的章程、从开铺到散伙的整本账册——由师爷当着三人的面核了人名、核了章程、核了账册。

县衙的红印落下。合记栈这块招牌，从今日起正式从县衙的册子上抹去。

孙兆庭走出县衙时，运河边的晚风已经起了凉意。他明白：这桩买卖散伙，不是把账簿往桌上一搁、银子往怀里一揣那么简单。得几位东家共同议定，得立下字据，得先清账后分银，得定下一个操持散伙事务的人，最后还得去县衙交代明白——桩桩件件落到了官面上的册子上，这场散伙才算真正散了。

合记栈九年，到此收梢。

---

**📖 原文定义**

> 一般地，合伙型股权投资基金出现以下情形的，应当解散：
>
> 合伙型股权投资基金解散，应当由清算人进行清算。清算期间，合伙型股权投资基金继续存续，但不得开展与清算无关的经营活动。清算退出的资产分配顺序上，合伙型股权投资基金财产在支付清算费用和职工工资、社会保险费用、法定补偿金以及缴纳所欠税款、清偿债务后的剩余财产，向合伙人进行分配。
>
> 合伙型股权投资基金清算，清算人应当自被确定之日起10日内，将清算人成员名单按规定进行公告，并提交清算登记申请文件，包括清算登记申请书、清算的决议、清算报告等。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 三人议定散伙日期并签章程 | 出现法定或约定的解散事由 |
| 公推孙兆庭为操持散伙事务之人 | 清算人 |
| 散伙期间铺面仍挂招牌但不准接新买卖 | 合伙企业继续存续，但不得开展与清算无关的经营活动 |
| 先付清账花费、伙计工钱、欠款、税款，再分剩余银两 | 支付清算费用、职工工资、社保费用、法定补偿金、缴纳税款、清偿债务后的剩余财产向合伙人分配 |
| 孙兆庭去县衙交代操持散伙之人名册 | 清算人自被确定之日起10日内公告清算人成员名单 |
| 散伙的章程（议定日期、签字画押） | 清算的决议 |
| 从开铺到散伙的整本账册 | 清算报告 |
| 递状子时带的三样东西 | 提交清算登记申请文件 |
| 县衙的红印落在合记栈的档上 | 清算登记（向市场监督管理部门注销） |
| "桩桩件件落到官面上的册子上，这场散伙才算真正散了" | 清算登记是解散的最终法律确认 |
