# （三）契约型股权投资基金的变更备案

## 🤝 寓言故事 —— 《一纸契约走天下》

老周是泉州码头的货栈掌柜，手里攥着一张泛黄的纸。

三年前，他没有开铺子、没有挂招牌，只邀了六个相熟的船老大，在这张纸上写了各自出多少份子、将来怎么分利、谁管账。六个人按了手印，钱凑在一起，买了三艘船跑南洋。没有衙门注册，没有合伙字号，就凭这一纸约定。

去年冬天，跑船的老刘家里急用钱，想把他的份子转给隔壁县来的药材商老郑。老周起初犯嘀咕：要不要把六个人全叫来商量？后来翻开那张纸，上面写得明白——谁想转手，只消卖方、买方、管账的三方点头，旁人不必过问。老周就是管账的。他当场叫来人，三方在纸背面添了一行字，老郑按了手印，老刘拿钱走人。从头到尾，没惊动其他五个人。

开春后，新来的绸缎商小陈想加入。老周同样翻开那张纸，小陈看完条款，写上进多少钱、占多少份，按了手印，钱一交，就成了第七人。没有换招牌、没有重新立号，只在原纸上多添了一个名字。

到了夏天，跑船的老吴年纪大了，不想漂了。他按纸上写的算法，该退多少、怎么算，老周三天内把银子点清。老吴拿钱退出，纸上的名字一划，从此这船生意与他无关。老周把这些变动一一在纸的夹页里记好，自己收着，不用上报给谁，也不去衙门换什么文书。

一张旧纸，进人、出人、转手，全在字里行间。老周守着这张纸，就像守着一扇活络的门——人来人去，门还是那扇门，规矩早就写在了门上。

---

**📖 原文定义**

> 契约型股权投资基金的变更登记包括申购、赎回及基金份额转让。
>
> 契约型股权投资基金的申购，是指基金投资者与基金管理人签订基金合同，承诺认缴金额，并按照基金合同的约定缴纳出资购买基金份额的行为。
>
> 契约型股权投资基金的赎回，是指在基金成立后，持有基金份额的基金投资者按照基金合同的约定将基金份额兑换为现金的行为。基金投资者持有的基金份额全部退出后，即不再是基金份额持有人和基金合同的当事人。
>
> 契约型股权投资基金的申购、赎回事项等具体规则及安排，由基金管理人、基金投资者及其他合同当事人（若有）在基金合同中约定。申购及赎回的价格，由基金管理人按照合同约定的规则进行计算。份额登记机构根据最终确定的价格计算基金投资者出资应得的基金份额以及退出应得的退出金额。
>
> 契约型股权投资基金的基金投资者可以依法转让其持有的基金份额。份额转让规则及流程按照基金合同的相关约定办理。一般地，基金份额的转让只涉及出让方、受让方及基金管理人，三方协商同意即可按照基金合同的约定办理转让手续，无须征得其他基金投资者的同意。
>
> 契约型股权投资基金的变更备案工作由基金管理人自行办理。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周和六个船老大按手印的"一张纸" | 基金合同——契约型基金的基础，不依赖公司或合伙注册 |
| 老刘把份子转给老郑，只需卖方、买方、管账的老周三方同意 | 基金份额转让：出让方、受让方、基金管理人三方协商，无须其他投资者同意 |
| 小陈写上进多少钱、占多少份，按手印交钱加入 | 申购：投资者签订基金合同，承诺认缴金额并缴纳出资 |
| 老吴按纸上算法拿钱退出，名字一划 | 赎回：投资者按合同约定将份额兑换为现金，全部退出后不再是合同当事人 |
| 老周自己把变动记在夹页里，不上报衙门 | 变更备案由基金管理人自行办理 |
