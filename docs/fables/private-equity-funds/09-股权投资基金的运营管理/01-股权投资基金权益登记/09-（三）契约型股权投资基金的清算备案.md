# （三）契约型股权投资基金的清算备案

## 📜 寓言故事 —— 《和春堂的最后一晚》

故事接上前头。和春堂开张十二年，沈远舟的客栈早已不是当年那一间——渡口扩建为港埠，他沿江三处又起了分号。扬州那批老主顾按份子分账，分到各家的银子早已翻了几番。

可这年入冬，老主顾一个接一个想抽身：杭州林老板的布行遭了火；徽州吴七哥跟着儿孙迁去武昌；苏北盐号陈二爷过世，他儿子来信料理后事。更要紧的是，当初签下的那张"合伙纸"上白纸黑字写明：和春堂以十二年为期，期满则账目清结、余银照份子分派、银箱交给"代管人"封存。算到这年腊月，十二年恰好届满。沈远舟站在江边，眉头紧锁。

事情落到了他头上——他是那张纸上写明"按纸办事"的人。可活着的钱怎么数、银箱怎么开、走的人钱怎么追给、留的人钱怎么分，他一个人说了不算。他想起当年县衙衙役问过的那句："合伙还是独资？"

这回，他想通了第二层。

到期散伙总归要把账面上一笔一笔列清楚——这是商场上千百年不变的理。可和春堂不是商号，没有"掌柜"这个名头，没有在官府里立过户，连一张"户帖"都没有。它只是那张纸。那张纸，到期了。

沈远舟回扬州，把"代管人"周先生请到渡口，又请来城里最老到的账房徐翰青做见证。三人关在和春堂总号花厅，议了一个下午。

议出来的章程，写在另一张新纸上：

——沈远舟牵头，把十二年的进出账目、各地分号的盈亏、各家份子的进出明细，一笔一笔列成清单，誊成册子。

——册子誊完之后，由沈远舟、周先生、徐翰青三人共同核章。

——分银按那张老纸上写明的份子比例，由周先生的银库直接划拨给各家。

——所有银两结清后，由沈远舟、徐翰青二人共同出具一份"结清册"——里头写明：账目核过、银子分毕、银箱已封、众人无异议。

这最后一份"结清册"，便是和春堂散伙的收据。

腊月廿三，小年夜。沈远舟把"结清册"和一张"散伙禀报"一起，按老纸上写好的步骤，递到当年他报开张的那一处登记处。报的内容是：和春堂所依的那张纸，期满已止；账目已清；分银已毕；老纸上注明的那位"代管人"，已对"结清册"核章。

登记处的管事翻了翻，抬眼问沈远舟："福兴号、合记栈散伙，得有官府告示、按印签押的公文。你这和春堂不是一户、不是几家合的铺面，是几张签了字的纸——凭什么报散？"

沈远舟把那张老纸递过去："就凭它。"

管事接过去看了一遍，点点头，把"结清册"和那张"散伙禀报"一并登了册。他在登记簿上写下："和春堂——期满已结。"

沈远舟拱手一揖。

走出登记处时，江面上飘着细雪。十二年的事，就此收在一份"结清册"里。林老板收到了从渡口汇来的银子，吴七哥的儿孙在武昌也收到了，陈二爷的儿子按老纸写明的方式，从"代管人"那里继承到了他父亲应得的那一份。

---

**📖 原文定义**

> 契约型股权投资基金的清算，是指当基金存续期限届满，或者全部投资项目退出，或出现基金合同约定的其他基金清算事由时，对基金财产进行处理和分配的行为。
>
> 契约型股权投资基金的清算由清算组负责。清算组一般由基金管理人负责组织，并由基金管理人、基金托管人以及相关中介服务机构组成。契约型股权投资基金的清算分配方案由基金合同进行约定。清算组应编制清算报告，并向基金投资者进行披露。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 和春堂的纸期满、老主顾散伙、各项分号到期 | 基金存续期限届满、全部投资项目退出、合同约定的其他清算事由 |
| 沈远舟作为纸上写明"按纸办事"的人牵头 | 契约型基金清算由清算组负责、由基金管理人负责组织 |
| 周先生（代管人）、徐翰青（账房）共同核章 | 清算组由基金管理人、基金托管人、相关中介服务机构组成 |
| 银两按老纸上的份子比例划拨 | 清算分配方案由基金合同进行约定 |
| 沈远舟与徐翰青共同出具"结清册" | 清算组应编制清算报告 |
| "结清册"和"散伙禀报"按老纸上的步骤递到登记处 | 清算报告应向基金投资者进行披露，并完成清算备案程序 |
