# （2）回购的义务主体

## 🤝 寓言故事 —— 《谁来接这个摊》

清河镇钱掌柜的染坊三年前拉了县城孙掌柜入伙。两人在契书上白纸黑字写明——遇上了"那一步"，孙掌柜可以把份额"要回"。

可"要回"这话好说，"谁来接"这事难讲。

这日两人又凑到一块儿，把这事儿掰开了想。

头一个主意——钱掌柜说："让染坊自己来接。"孙掌柜皱眉："染坊的钱在染缸里、料子上、柜上的货里，一时半刻拿不出现银来接我。再说染坊接回来，等于把份额收回去了，染坊自己的钱就更紧了。"

第二个主意——孙掌柜说："那让你老几位接。"钱掌柜也皱眉："我老几位手里就家里那点田产房契，接一份就是砸锅卖铁。要是接多了，倾家荡产也接不起。"

第三个主意——钱掌柜一拍大腿："这样，染坊和我老几位一起接，谁先顶得上来谁先顶。"孙掌柜点头："这就叫'绑一块儿'。可我得问一句——要是染坊一时半刻接不上,你老几位是不是还得照样接？"

钱掌柜想了想："我老几位可以'兜底'——染坊要是真接不动,我老几位就把染坊没接上的那一块补上。可不能反过来——染坊不能为我老几位的家事再补一刀。"

两人议到这里,孙掌柜又问："万一哪一天真让你老几位来接,你老几位就'砸锅卖铁'来接？"钱掌柜苦笑："那可不行——我老几位也得留条活路。接的'数'，最多到我老几位手里那点份额值多少钱,就到这个数为止。"

孙掌柜眼睛一亮："'数'有封顶——这就叫'封到份额'。你那三成多份额值多少银子,最多就接多少,家里的田产房契不动。"

钱掌柜长舒一口气："这下我老几位敢拍胸脯——'绑一块儿'也好、'兜底'也好,都是兜得起来的兜。"

孙掌柜也点头："这下我也敢拍胸脯——有人接、接得起、不会半路掉链子,我那三百两银子才有着落。"

两人提笔在契书最末那一页,把"谁来接"这事儿写得明明白白——染坊和我老几位一起接,接不上我老几位兜底,封到我老几位手里那份份额。

---

**📖 原文定义**

> 回购的义务主体通常可以分为以下几类：仅目标公司作为义务主体、仅创始团队作为义务主体、目标公司与创始团队作为连带的义务主体、创始团队对目标公司在回购不能的情况下承担补充责任。
>
> 在创始团队承担连带回购义务或为公司回购承担补充责任的情况下，为了避免创始团队以个人财产承担过大的责任，创始团队往往会要求对其回购义务设定责任上限。责任上限一般会以创始团队持有的目标公司股权的市场公允价值或变现价值为限。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 让染坊自己来接 | 仅目标公司作为义务主体 |
| 让老几位接 | 仅创始团队作为义务主体 |
| 染坊和老几位绑一块儿接 | 目标公司与创始团队作为连带的义务主体 |
| 染坊接不上老几位再补 | 创始团队对目标公司承担补充责任 |
| 最多封到老几位那份份额值多少钱 | 对创始团队的回购义务设定责任上限 |
| 责任上限以份额价值为限 | 责任上限以创始团队持有的目标公司股权价值为限 |
