# 4. 共同出售权

## 🏘️ 寓言故事 —— 《三兄弟的铺子》

青石镇上，周大、周二和周三是三兄弟，合伙开了一家绸缎铺。大哥周大管进货，二哥周二管账目，三弟周三最年轻，只出了少许银子，但三兄弟平分生意，铺子经营得红红火火。

那年秋天，城里来的钱掌柜看中了绸缎铺的地段，私下找到周大，说要买下他的那份生意。周大动了心，便召集两个弟弟，说钱掌柜出价公道，他想把自己的份额转手。

周二一听，忙说："大哥，这铺子咱们兄弟经营多年，你怎好卖给外人？你若真要卖，我按同样的价，买下你的份额。"

周三年轻，手头拮据，低头想了半晌，说："二哥要买，我买不起。但我也不想让铺子落进外人手里。"

周大见周二有诚意，便与周二谈妥了价钱。周二掏尽积蓄，把周大的份额接了过来。周大拿了银子，去城里另谋营生。绸缎铺成了周二和周三两人合伙，日子照旧。

过了两年，周二染了风寒，身体大不如前，也想歇手了。他想起当年钱掌柜的出价，便托人传话，说愿意按当年的条件，把自己的份额卖给钱掌柜。

周三一听慌了神。当年大哥走时，他穷，买不起；如今二哥也要走，若都卖给钱掌柜，那钱掌柜岂不成了铺子的大东家？自己一个年轻后生，独自守着这铺子，岂不是被外人捏在手心里？

周三连夜去找周二，说："二哥，你当年买了大哥的份额，我没拦你，因为我买不起。可如今你要卖给钱掌柜，我也拦不住你卖。但既然你要走，我也不能一个人留在这里。钱掌柜出多少价买你的，我也按同样的价，把我的那份卖给他。"

周二愣住了："三弟，你这是何意？"

周三红了眼眶："二哥，当年大哥走时，你有钱，能接得住；我没钱，只能眼睁睁看着。如今你要走，我照样接不住。但我不能眼睁睁看着外人进来，把我一个人晾在边上。你卖，我也卖。咱们兄弟一起进门，也该一起出门。"

周二沉默良久，叹了口气，收回了卖给钱掌柜的话。三兄弟重新坐下来，商量出一个规矩：日后谁若想转手自己的份额，必须先让其他兄弟按同样的价优先接手；若其他兄弟接不住，那想走的人可以把份额卖给外人，但接不住的兄弟也有权把自己的份额一并卖给那个外人，价钱和条件一模一样，谁也不能吃亏。

规矩定下，绸缎铺又安稳地开了许多年。周三后来攒足了本钱，也接住了二哥的份额。三兄弟的铺子，终究没有落进外人手里。

---

**📖 原文定义**

> 共同出售权（Co-sale Right/Tag-along Right）一般是指在创始团队转让其持有的目标公司股权时，未行使优先购买权的投资方可以按照约定的比例与创始团队共同向拟受让方转让其持有的目标公司股权的权利。部分情况下，还会进一步约定，如创始团队转让其持有的目标公司股权会导致目标公司控制权变更的，投资方有权优先于创始团队出售其持有的全部公司股权。共同出售权的目的在于保证投资方在创始团队在全部或部分退出的情况下有共同退出的权利。
>
> 共同出售权与转让限制、优先购买权一样，也是在股权投资中投资方普遍会要求的特殊权利。一般情况下，投资方行使共同出售权的前提是投资方未完全行使优先购买权，致使创始团队仍有权向拟受让方转让其持有的部分目标公司股权。
>
> 受制于本协议关于转让限制的约定，如创始股东拟转让目标公司股权的，在创始股东发出售股通知后，未行使优先购买权的投资方股东有权（即共同出售权）按照其共同出售权比例（见下文定义），并以售股通知中规定的价格与条件与售股股东一起向拟受让方出售其所持有的股权。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 三兄弟合伙开绸缎铺 | 创始团队与投资方共同持有目标公司股权 |
| 周大想卖份额给钱掌柜 | 创始团队拟向第三方转让股权 |
| 周二出钱买下周大的份额 | 投资方行使优先购买权（完全行使，共同出售权不触发） |
| 周二两年后想卖份额给钱掌柜 | 创始团队再次拟向第三方转让股权 |
| 周三当年买不起，如今也买不起 | 投资方未行使优先购买权（前提条件） |
| 周三要求按同样的价一起卖给钱掌柜 | 投资方行使共同出售权，按同等价格与条件共同转让 |
| 三兄弟定下规矩：接不住的兄弟有权一起卖 | 共同出售权条款的制度化安排 |
| 确保没有兄弟被单独留在铺子里面对外人 | 保证投资方在创始团队退出时有共同退出权利，避免被单独留下承担风险 |
