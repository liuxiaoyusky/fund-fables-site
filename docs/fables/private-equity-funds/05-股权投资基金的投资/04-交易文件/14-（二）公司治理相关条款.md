# （二）公司治理相关条款

## 🏛️ 寓言故事 —— 《三把椅子》

杭州城外的龙井山下，有一座老茶厂，主人叫周德山，人称"老周"。老周做茶三十年，手艺精湛，但茶厂一直做不大。那年春天，城里来了三位商人，说要帮老周把茶厂开到全国去。

第一位是钱掌柜，做绸缎生意起家，出手阔绰。第二位是孙先生，开钱庄的，精打细算。第三位是林姑娘，做海外茶叶买卖的，见多识广。三人各出一笔钱，换得了茶厂的一部分份额。老周虽然还是最大的东家，但已不是唯一的主人了。

钱掌柜最先开口："老周，我出了这么多钱，茶厂的大事我总得有个说法吧？"

老周想了想，在茶厂议事堂里摆下四把椅子。"钱掌柜，您坐这把。以后每月议事，您有一席之地。"

孙先生不甘落后："我也出了不少，难道只能站着听？"

老周又添了一把椅子。"孙先生，您请坐。"

林姑娘笑了笑："老周，我虽是个女流，但海外销路全靠我打通，议事堂里总该有我的位置。"

老周迟疑了。议事堂总共就那么大，椅子多了，说话的人多了，一件事议上三天也定不下来。可椅子少了，出钱的人寒了心，往后谁还愿意往里投钱？

老周最终摆了六把椅子。他自己和账房先生坐两把，三位商人各坐一把，还有一把空着，留给将来可能加入的新东家。但老周定了个规矩：椅子可以坐，但茶厂日常采茶、炒茶、卖茶这些事，由各作坊掌柜自行决定，不必事事上报议事堂。若是换作坊掌柜、买新地皮、跟别的茶厂合并、或者要把茶厂卖给外人——这些事，必须六把椅子上的东家都点头才行。任何一人摇头，这事就办不成。

钱掌柜皱眉："这也太束手束脚了吧？"

老周摇头："您各位出了钱，却不天天在茶厂盯着。万一我老周一时糊涂，把茶厂抵押出去换酒钱，或者跟个骗子合伙做买卖，各位的钱不就打了水漂？大事上各位把把关，是对大家都有好处。"

孙先生点头称是，却又补了一句："可要是鸡毛蒜皮的事也要六个人点头，茶厂还做不做生意了？"

老周早有准备，从袖中取出一张单子："哪些事要大家点头，我列得清清楚楚。换掌柜、买地卖地、借大钱、分红利、改规矩——就这些。别的事，我老周做主。"

三位商人传看单子，各自掂量。钱掌柜觉得单子太短，想再添几条；孙先生觉得已经够长，再添茶厂就动弹不得了。讨价还价了整整一个下午，单子上的条目终于定了下来。

椅子的事定了，老周又说："还有一件事。各位东家不常在茶厂，心里总该有个数。每月账房先生把收支簿子抄送各位一份。每年腊月，各位随时可来茶厂，查账、看库房、跟作坊掌柜聊天。"

林姑娘拍手叫好："正该如此。我在海外也有茶叶生意，知道同行怎么做事，正好给老周参谋参谋。"

老周却沉下脸："林姑娘，您做海外茶叶生意，我管不着。但茶厂的秘方、客人的名册、今年新试的炒茶法子——这些要是流到了您海外的同行手里，我老周可就对不起祖宗了。所以有一条：您若在外头还做着跟茶厂抢买卖的生意，茶厂的机密簿子，就不能送到您手上。"

林姑娘先是一愣，继而苦笑："老周，您这是信不过我？"

"不是信不过您，是规矩。"老周正色道，"出钱的人有知情的道理，做生意的人有保密的苦衷。两边都得顾全。"

林姑娘沉默良久，最终点头：" fair enough。海外的茶叶份额，我另找人打理。"

就这样，龙井茶厂在新规矩下运转起来。钱掌柜、孙先生、林姑娘各坐一把椅子，大事上有一票否决之权，每月看簿子，每年查库房。老周还是日常管事的人，但再也不能独断专行。茶厂反倒因此兴旺起来——三位东家各有人脉，钱掌柜介绍了绸缎庄的客户，孙先生帮茶厂从钱庄借到低息款项，林姑娘打通了海外销路。老周专心炒茶，手艺愈发精进。

五年后，茶厂从一家小作坊变成了江南数得着的大茶庄。议事堂里的椅子从六把变成了八把——又来了两位新东家。老周的份额被稀释了不少，但他仍然坐在主位上，因为他定的那些规矩让所有人都放心：椅子怎么坐，大事谁点头，簿子怎么看，机密怎么守——清清楚楚，明明白白。出钱的人不怕被蒙在鼓里，管事的人不怕被捆住手脚。

有一年，钱掌柜的绸缎庄亏了本，想偷偷把茶厂的份额卖给一个做瓷器的外乡人。那外乡人不懂茶叶，只想买了份额再转手倒卖。这事到了议事堂，孙先生摇头，林姑娘摇头，老周更是坚决反对。钱掌柜的算盘落了空，起初恼羞成怒，后来却想明白了：若是当年老周一个人说了算，这茶厂怕是早被人拆散卖光了，哪还有今日的气象？

那八把椅子，至今还在龙井山下的议事堂里摆着。老周常常跟新来的东家讲：椅子是给人坐的，但椅子也是给人拴缰绳的。出钱的人不拴缰绳，马就跑了。缰绳拴得太死，马就趴下了。好骑手知道松紧之间的分寸。

**📖 原文定义**

> 董事会席位条款约定目标企业董事会的席位数量、初始分配方案和后续调整规则，是目标企业控制权分配的重要条款。通常，持股达到某一最低比例的投资机构有权任命若干名董事。如果公司董事会席位增至一定数量，则投资方有权任命新的董事，从而参与目标企业董事会。董事会席位条款的实质是对被投资企业的控制权分配。在通常情况下，董事会席位条款会约定董事会席位总数及其分配规则。在创业融资中，创始人往往要求保持董事会的多数席位，以维持对公司的控制权。董事会席位条款通常约定董事会席位总数、分配规则、董事任命权、董事更换权、董事会会议规则等。董事会席位条款通常规定，每一轮融资的领投方有权任命一名董事，或者持股超过一定比例的投资方有权任命一名董事。如果投资方的持股比例因后续融资而被稀释至约定比例以下，则该投资方将失去董事任命权，但可能保留董事会观察员席位。观察员有权参加董事会会议，但没有投票权。持股比例较小的投资方可能获得董事会观察员席位。
>
> 保护性条款是指股权投资基金为保护自身利益而设置的要求目标企业在执行某些可能损害投资者利益或对投资者利益有重大影响的行为时，需取得投资者同意的条款。保护性条款通常针对的特定事项包括：修改公司章程、增加或减少注册资本、公司合并、分立、解散或变更公司形式、对外借款或担保、对外投资、关联交易、改变公司主营业务、发行新股或债券、IPO计划等。保护性条款通常约定，特定事项须经持有一定比例以上股权的投资者同意方可通过。保护性条款的谈判要点包括：保护性条款的适用范围（哪些事项需要投资者同意）、触发保护性条款的投资者持股比例门槛、以及保护性条款的行使方式（是在股东会层面还是董事会层面行使）。保护性条款过多会限制企业正常经营，过少则无法有效保护投资者利益。
>
> 信息权与检查权条款是指投资者有权获得公司经营和财务信息的权利，以及有权检查公司账簿和记录的权利。信息权通常包括获得公司定期财务报告、预算、业务计划等信息的权利。检查权通常包括检查公司设施、账簿、记录、访谈公司管理人员等权利。信息权与检查权条款的目的是使投资者能够了解公司的经营状况和财务状况，而不必参与公司的日常管理。信息权与检查权条款通常会约定信息提供的频率和形式，以及检查权的行使方式和时间安排。信息权与检查权条款通常会限制信息权的范围，例如，仅向持股超过一定比例（如5%）的投资者提供详细信息，而持股比例较低的投资者仅能获得基本信息。此外，如果投资者持有竞争对手的股权，则其信息权可能会被限制或取消。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 议事堂里的椅子 | 董事会席位条款——投资者参与企业决策的席位分配 |
| 老周定六把椅子、约定椅子数量与分配规则 | 董事会席位总数及其分配规则，领投方或持股超一定比例者有权任命董事 |
| 新东家加入后椅子增至八把 | 后续融资导致董事会席位增加，需重新分配 |
| 空椅子留给未来东家 | 董事会席位初始分配方案及后续调整规则 |
| 大事必须六把椅子上的东家都点头 | 保护性条款——特定事项须经投资者同意方可通过 |
| 老周列出的单子（换掌柜、买地卖地、借大钱等） | 保护性条款的适用范围/保留事项清单 |
| 钱掌柜想添条目、孙先生怕条目太多 | 保护性条款谈判要点——过多限制经营、过少无法保护投资者 |
| 每月账房先生抄送收支簿子 | 信息权——投资者获得公司定期财务报告和经营信息 |
| 每年腊月查账、看库房、跟掌柜聊天 | 检查权——投资者检查公司账簿、设施和访谈管理人员 |
| 林姑娘做海外茶叶生意则不得看机密簿子 | 信息权限制——持有竞争对手股权的投资者信息权可能被取消 |
| 老周说的"缰绳松紧之间的分寸" | 公司治理条款的核心平衡——既保护投资者利益，又不束缚企业日常经营 |
