# 2. 转让限制条款

## 🔒 寓言故事 —— 《哥哥的醋坊》

镇江城外有条老街，街尾有家"恒顺记"老醋坊，传到这辈是兄弟两个在撑——哥哥江慎言，五十出头，老实木讷；弟弟江慎行，四十出头，脑子活络，前些年跑外埠把醋卖到了扬州府。

两年前，醋坊要扩产，兄弟俩凑不出那笔银子。扬州的大商人周敬亭找上门来，愿出一笔厚资入股，条件是——三年之内，醋坊若没在扬州挂牌、也没被人整家买下，江家兄弟两人手里那点份额，没经过周家点头，一分都不准外流。弟弟拍着桌子说："签！咱要的是周转，又不是真要卖铺子。"哥哥叹了口气，提笔签了字。

头一年，醋坊借着周家那笔银子又开了三间分号。弟弟跑外埠，哥哥守着老作坊的几百口大缸，夏天翻醅，冬天捞冰。周敬亭逢年过节来镇上一次，翻翻账本，端一盏新醋尝一尝，点点头就走。

第二年开春，弟弟的小舅子在扬州亏了一笔，托人带话回来，问能不能把自己在醋坊那一份转出去救个急。弟弟一想，反正醋坊又不挂牌、也不卖——那条款不就没用了吗？便托了掮客陆半仙去找下家。半个月，陆半仙拉来一个盐商，愿按当初价码全盘接手。

没承想，签转让契书那天，周敬亭的管事赵四爷从扬州赶来。他坐在恒顺记堂屋里，不紧不慢把当初那份协议摊开："周老爷让我带句话——这份额，没他点头，动不了。"

弟弟的脸涨得通红："我卖自己的份额，关他周家什么事？"赵四爷摇头笑着说："江二爷，您当初可是画了押的。这醋坊一日没挂牌、没被人整家收走，您和江大爷的份额，一分不准外流。这是写进纸里的。"

哥哥在边上听着，没吭声。等赵四爷走了，他把弟弟拉到后院的醋缸边，从缸里舀了一勺新醋递过去："二弟，你尝尝。"弟弟喝了一口，酸得皱眉。哥哥说："这坛醋，从下料到出缸，要等三年。咱爹当年立字号时说过，最忌讳中间换人——换了人，味道就断了。周家银子进来时，咱们答应过，灶火三年不熄、人心三年不散。"

弟弟沉默了很久，最后把那勺醋喝完，涩得直咂嘴。哥哥又说："我去跟周家商量，看能不能给咱留条缝——家里真出了急事，经他点头，转一小份出去缴个税、救个急，也行。但这个口子，得周家点头才能开。"

第二年秋天，弟弟的小舅子没救回来，自己关了铺子去了苏州。弟弟反倒松了口气——他这才明白，当初签下的那张纸，不是绑他的绳子，是让那坛醋安安稳稳陈下去的坛子。要是真把份额转出去，那买主不是来酿醋的，是来捞一把就走的——这坛醋，三年未满就让人掀了盖，往后谁还敢喝"恒顺记"的醋？

到第三年开春，醋坊生意越做越稳。周敬亭又来了一趟，这回提了另一桩事——扬州有人愿出高价，把整家醋坊连缸带匾一起收走。兄弟俩在堂屋里对望一眼。这回，条款里写的那条路，敞开了。

---

**📖 原文定义**

> 转让限制条款（Transfer Restrictions）、优先购买权以及共同出售权是具有先后逻辑关系的三个条款，因此，我们也按照前后顺序，依次介绍这三个条款。
>
> 转让限制条款主要针对实际控制人和创始团队，一般是指在目标公司上市或被整体出售之前，未经投资方同意，实际控制人和创始团队不得直接或间接出售或处置其持有的目标公司股权。这一条款的主要目的是为了将目标公司和创始团队绑定，保障创始团队的稳定性。
>
> 依据不同的具体项目情况，转让限制条款可能会存在一些例外情形，比如：允许创始团队为了缴纳个税、回购股权等目的转让不超过一定比例的目标公司股权；为特定的财务规划目的，向配偶、子女转让股权等。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 周敬亭入股时立下"三年内未挂牌、未被整家收走则份额不准外流"的约定 | 转让限制条款的核心约定 |
| 醋坊"上市或被整体出售"对应原文的"目标公司上市或被整体出售" | 转让限制条款解除的两个触发条件 |
| 弟弟拟将份额私下转给盐商，被周家管事阻止 | 未经投资方同意，实际控制人/创始团队不得转让股权 |
| 哥哥提"留个口子"——急事下可经周家同意转一小份 | 转让限制条款的例外情形 |
| 醋坊"换人味道就断了"的祖训 | 条款目的：把创始团队与公司绑定，保障稳定性 |
| 第三年扬州有人愿整家收购，条款里的路敞开了 | 整体出售作为条款解除的另一路径 |
