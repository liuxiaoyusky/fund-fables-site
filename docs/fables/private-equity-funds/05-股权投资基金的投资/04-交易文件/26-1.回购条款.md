# 1. 回购条款

## 🛡️ 寓言故事 —— 《兜底的话》

清河镇上做染坊生意的钱掌柜，三年前拉了县城一位姓孙的商号入伙。孙掌柜掏了三百两银子,占了整间染坊三成份额。当时两人拍着胸脯签了一纸契书。

契书写得明白：染坊的事儿由钱掌柜一帮老兄弟打理，孙掌柜只管年底拿分红。可孙掌柜这人心细，签字之前拉住钱掌柜追问一句："钱掌柜，我这份银子掏得舒坦，可万一哪一天这染坊就是起不来,或者你老几位出点什么岔子——我这点份额能拿回去吗？"

钱掌柜是个厚道人，听了这一问，当场拍胸脯："孙掌柜，咱白纸黑字写清楚——真到了那一步，我按说好的价钱把你那三成接回来,绝不让你这三百两打了水漂。"

孙掌柜眼睛一亮："那什么叫'那一步'？价钱又怎么算？"

钱掌柜想了想："这样——咱把'什么时候接'和'接回来怎么算钱'分开说。'什么时候接'，就是到了某一天、某件事发生了,我就有这个接的义务。'接回来怎么算钱'，就是你当初掏的三百两，再加上这一两年里算的利息——总之,不会让你亏着走。"

孙掌柜点点头："成。那'接回来'这事,到底是我能'要'你接,还是你能'愿'接？"

钱掌柜笑了："是你要,我接——这是你的'要回'。你什么时候要,我什么时候接,这是规矩。"

两人把这事掰得明明白白，写进了契书的最末一条："合伙期间若遇事先约定之事由,孙掌柜有权要求钱掌柜一方按约把份额接回。"

孙掌柜揣着契书回县城,一路心里踏实——"三百两银子掏出去,就算生意没做大,也有个兜底的话。"这兜底的话,既是一句安慰,也是日后真到了那一步、钱掌柜必须照办的白纸黑字。

---

**📖 原文定义**

> 回购条款（Redemption Right）是指投资方与目标公司、创始团队约定，在发生特定事件时，投资方有权要求目标公司和/或创始团队按照事先约定的价格购买投资方持有的全部或部分目标公司股权的条款。投资方据此享有的权利称为"回购权"，回购权据以行使的特定事件称为"回购触发事件"。
>
> 投资方持有的目标公司股权在目标公司完成上市后，在公开市场上退出是理想状态，但现实中大部分的目标公司可能无法完成上市。回购权的目的就在于保障投资方在目标公司无法上市或无法在短期内上市情况下的退出渠道。
>
> 回购条款从以下四个方面详细解释：回购触发事件、回购的义务主体、回购的方式、回购价格。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 钱掌柜接回孙掌柜那三成份额 | 回购（按约购买投资方持有的股权） |
| 孙掌柜"有权要求"钱掌柜接 | 投资方享有的"回购权" |
| 事先约定的事由发生 | "回购触发事件" |
| 钱掌柜作为接回的义务人 | 回购的义务主体 |
| 当初的三百两加上约定利息 | 回购价格 |
| 写进契书最末一条 | 在交易文件中约定回购条款 |
