# 8. 估值调整条款

## 🎯 寓言故事 —— 《老马的赌约》

老马是镇上出了名的眼光准。那年春天，他相中了年轻人小周开的酱园。小周的酱园刚起步，但手艺好，口碑传得快。两人坐在酱园的老槐树下，老马端起一杯新酿的酱油，慢悠悠地说："小周，我给你二十万，换你酱园四成股。不过咱们得有个说法——你酱园现在值五十万，是觉得两年后能翻三倍，我才给这个价。"

小周拍着胸脯："马叔您放心，两年后我这酱园年利不少于十万，两年加起来不少于二十万。要是做不到，我认栽。"

两人当下立了字据，各执一份。老马的钱进了酱园的账，小周添了设备、雇了人手，酱园一时红红火火。

转眼两年过去。老马拿着账本子找上门，一算，两年利钱加起来才十二万，离当初说的二十万差了一大截。酱园还是那个酱园，只是没长到两人当初想的那么大。老马坐在槐树下，字据摊在石桌上，叹了口气："小周，按咱们当初的约定，这酱园现在值不了五十万，得按三十万算。你多拿了我的股，得还回来。"

小周脸涨得通红，额头冒汗。二十万已经变成新缸新灶，哪还有现钱退？他攥着字据，半晌说不出话。

老马也不逼他，指着字据上另一行小字："当初写了两条路。要么你把你手里多占的股转给我，咱们按新价重新分；要么你筹一笔钱给我，补上这个缺口。你选哪条？"

小周望着满院的酱缸，咬咬牙选了第一条——把自己手里的股转了一部分给老马，两人的比例重新按三十万的价码算过。酱园还是一起经营，只是老马占的多了，小周占的少了。

后来小周逢人就说："当初嘴快，把话说得太满。老马是好合伙人，但好合伙人也得把丑话说在前头。那字据看着像赌，其实是给双方都留了台阶。"

---

**📖 原文定义**

> 估值调整条款（Valuation Adjustment Mechanism）有时也称为"对赌条款"，是指投资方与目标公司或创始人事先约定一定的对赌条件，在对赌条件未成就时，目标公司或创始人向投资方按照一定的标准进行业绩补偿。
>
> 在估值调整条件触发时，目标公司或创始团队向投资方补偿的方式一般包括股份补偿和现金补偿两种。股份补偿主要是指目标公司以名义价格向投资方增发一定的股权或创始团队向投资方以最低价格转让一定的股权，以实现估值调整的目的。现金补偿主要是指目标公司或创始团队向投资方支付一定的现金以承担对赌失败的结果。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老马出资二十万换酱园四成股 | 投资方以一定估值入股 |
| 约定两年利润不少于二十万 | 事先约定的对赌条件 |
| 实际利润仅十二万，未达标 | 对赌条件未成就 |
| 酱园估值从五十万调至三十万 | 估值调整 |
| 小周转让部分股份给老马 | 股份补偿方式 |
| 或筹钱补上缺口 | 现金补偿方式 |
| 字据给双方留台阶 | 条款保护投资方利益，同时避免直接索赔 |
