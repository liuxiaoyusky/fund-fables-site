# 7. 优先清算权

## 🏚️ 寓言故事 —— 《老陈的客栈》

老陈在镇上开了三十年客栈，攒下三间瓦房和一口百年老井。儿子小陈接手后，想把客栈改成酒楼，却缺一笔钱翻修。邻镇的周老板听说后，带着三百两银子找上门来。

"这三百两算我入伙，"周老板说，"但咱们得说清楚——将来这买卖要是散了，我得先把我的三百两拿回去，剩下的才是你的。"

小陈想了想，答应了。两人立了字据，周老板的三百两换成了酒楼的三成份额。

三年后，酒楼生意红火，可镇上来了一个大商人，出价八百两要把整间酒楼连老井一起买走。小陈心动了，去找周老板商量。

周老板听完，从抽屉里取出当年的字据，指着其中一行小字说："你仔细看看，咱们当年写的是，不只是关门才算散伙——把整间店卖了、把老井卖了，都算。"

小陈愣住了。他原以为卖了酒楼，八百两按三成来算，周老板拿二百四十两，自己能得五百六十两。可周老板摇摇头，说字据上写的是另一种分法。

"我先拿回我的三百两，"周老板说，"剩下的五百两，咱俩再按三成七成来分。这样我拿三百加一百五，共四百五；你拿三百五。"

小陈急了："那我不是亏了吗？要是当初按份额平分，我该拿五百六啊！"

周老板又取出另一张纸，是后来补的一份约定。上面写着：如果按份额分比先拿回本钱更划算，那就按份额分。这叫"孰高"——哪个高，按哪个来。小陈一看，八百两的三成确实是二百四十两，不如先拿回三百两划算，只好认了。

又过了两年，一个外地客商出价一千五百两，不仅要买酒楼，还要买老陈传下来的酿酒方子。这次小陈学乖了，先算了一笔账：一千五的三成是四百五，可周老板的本钱加利息已经滚到了四百两。如果按老规矩，周老板先拿四百两，再分剩下的三成，那就是四百加三百三，共七百三；自己只能拿七百七。

可如果按那份"孰高"约定，周老板只能拿四百五，自己能拿一千零五十。小陈满心指望这次能按份额来分。

周老板却笑了笑，从怀里掏出第三张纸——那是去年新签的补充条款。上面写着：周老板先拿回本钱和利息，然后继续按份额分剩下的钱，但总共不能超过本钱的两倍。也就是说，周老板最多拿六百两，剩下的九百两才归小陈。

小陈瘫坐在老井边，看着三张越来越厚的字据，终于明白了一个道理：当初那三百两银子，不只是钱，更是一串排在最前面的钩子。酒楼散架的时候，这些钩子先把大块肉钩走，剩下的碎末才轮到自己。

老井里的水依旧清冽，可井沿上刻着的"陈记"两个字，已经模糊得快要认不出来了。

**📖 原文定义**

> 优先清算权（Liquidation Preference）是指在目标公司发生清算事件时，投资方有权优先于创始团队获得清算所得的权利。清算事件是一个交易文件中的常规定义，除了常规的清算、解散等情形外，还包括"视为清算事件"。
>
> "视为清算事件"有时也被称为"整体出售事件"，一般包括以下三种情形：目标公司股权被整体出售导致控制权变更，目标公司出售其全部或实质上的全部核心资产和业务，目标公司将其全部或实质上的全部知识产权出售和转让给第三方。
>
> 优先清算权一般分为参与型的优先清算和非参与型的优先清算，通常而言，参与型的优先清算更为常见，也是通常对投资方更为有利的条款。
>
> 参与型的优先清算，是指投资方按照事先约定的方式获得优先清算分配金额后，与其他股东一起按照各自持股比例参与剩余财产的分配。部分情况下，参与型的优先清算可能还会与投资方约定附分配上限的方案，例如在投资方按照参与型的分配方式获得的金额高于投资本金的一定倍数之后，投资方不再参与剩余财产的分配。
>
> 非参与型的优先清算，是指投资方在获得优先清算分配金额后，不再参与剩余财产的分配。非参与型的优先清算，特别是在"视同清算事件"发生时，可能会造成投资方未能很好地分享整体出售收益的情形。在这种情况下，投资方通常会要求适用"孰高原则"，即如果投资方按照各自持股比例所得到的分配金额高于优先清算分配金额时，自动适用按照持股比例获得的分配金额。
>
> 投资方的优先清算分配金额，常见的有两种计算方式：一种是投资成本的一定倍数（如1.2倍、1.5倍或2倍）；另一种是投资成本加上一定单利或复利计算的利息之和。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 周老板的三百两银子 | 投资方的投资本金 |
| 小陈的酒楼 | 目标公司/创始团队 |
| 把整间店卖了、把老井卖了 | 视为清算事件（整体出售事件） |
| 周老板先拿回三百两，再按三成份额分剩余 | 参与型优先清算 |
| 补充条款约定最多拿回本钱两倍 | 参与型优先清算的分配上限 |
| "孰高"约定——哪个高分哪个 | 非参与型优先清算的孰高原则 |
| 本钱加利息滚到四百两 | 优先清算分配金额的计算方式（本金+利息） |
| 三张越来越厚的字据 | 交易文件中的各项条款约定 |
