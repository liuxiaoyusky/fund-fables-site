# 一、交易文件概述

## 📝 寓言故事 —— 《钱庄的聘礼》

江南钱庄少东家周明远看中了一家百年酱园，想把它纳入自己的版图。这事急不得，得一步一步来。

第一步，周明远让账房先生写了一封"探路帖"。帖子上写明：钱庄有意与酱园联手，愿出多少银两、占几成股、酱园东家日后有何特权。这帖子不是卖身契，只是双方试探心意。但其中有两条是铁规矩：一是酱园东家不得把这事说给外人听，二是三个月内酱园不得与别人谈亲事。周明远要的是"独一份"的机会，好安心派人去酱园查账、验货、摸清底细。这三个月里，酱园东家心里七上八下，既盼着钱庄真心来投，又怕错过了别家的好亲事。可规矩就是规矩，探路帖上的墨字未干，酱园的大门便只能为钱庄敞开。酱园的老掌柜劝东家再等等，说城西还有一家绸缎庄也在打听，可东家摇了摇头，指着探路帖上那枚鲜红的印鉴说："字据签了，就得守。"

三个月后，周明远的人把酱园里里外外翻了个遍，觉得酱园底子还行，只是有几坛老醋生了霉、几间库房漏了雨。周明远决定：这门亲事可以结，但要先把丑话说在前头。

第二步，双方坐下来签了两份正经文书。第一份叫"入股契"，写明钱庄出多少银子、换多少股、银子分几批到账、酱园东家必须如实交代家底、哪些烂账必须清理干净才能收钱。第二份叫"共治约"，写明钱庄入股后，酱园的大事谁说了算、东家想卖自己的股得先问过谁、钱庄有什么特殊的护身符。这两份文书是绑在一起的，缺一不可。大多数时候，钱庄是往酱园"添银子扩股"；少数时候，是直接从老东家手里"买旧股"。但无论哪种方式，"共治约"都得签，因为钱庄要的不是一时买卖，而是长久的规矩。酱园老东家握着笔，手微微发抖——他明白，从落笔的那一刻起，酱园便不再是他说了算的家业了。可他也清楚，没有这份共治约，钱庄的银子便不会踏进酱园半步。

第三步，周明远又做了一件事。他把"共治约"里最要紧的几条——比如老东家的股不能随便卖给外人——抄了一份，送到衙门去备案。衙门的大印一盖，全城的买卖人都看得见。这样一来，将来若有外人想买酱园的股，先去衙门一查便知：这股有锁，得先问过钱庄。私下的约定变成了天下的告示，酱园的东家想赖也赖不掉。周明远深知，酱园老东家百年信誉，可人心会变，只有把最紧要的规矩写到天下人都看得见的地方，才能护住钱庄日后的安宁。老东家起初不乐意，觉得家丑外扬，可周明远只淡淡回了一句："写在天下人眼皮底下的规矩，才是撕不破的契约。"老东家沉默良久，最终在备案文书上按下了手印。

第四步，也是最后一步，叫"交割"。签完入股契，不等于银子已经过手。周明远列了一张"交清单"：酱园的烂账必须结清、老东家必须签完所有字据、衙门备案必须办完、酱园的大掌柜必须换成双方认可的人。每办完一件，就勾掉一件。等清单上的条目全部打勾，周明远才命人把银票抬进酱园大门，同时换回了股权的红契。那张打满勾的"交清单"，连同所有签过字的文书、酱园内部的决议、衙门回执、律师的保状，一并封进一只檀木箱里。这只箱子，就是钱庄真正成为酱园股东的凭证。交割那日，酱园门口鞭炮齐鸣，可周明远知道，真正值得庆贺的不是鞭炮，而是那只檀木箱里整整齐齐的四道文书。

酱园的老东家看着银子入库，长叹一声："原来结一门亲事，要过四道关。探路帖是定心意，入股契是定规矩，衙门备案是定天下，交割清单是定生死。少了一道，这亲事就不算数。"

周明远合上檀木箱，淡淡一笑："生意场上，白纸黑字是护身符，一步一步是保命诀。"

**📖 原文定义**

> 股权投资交易中，交易文件一般包括：
> 1. 投资意向书：在正式尽职调查之前签署，列明投资者、目标公司、估值、投资金额、特殊权利等要素。除保密条款和排他性条款外，其他条款通常不具有法律约束力。主要目的是确立投资意向，就估值和金额达成初步共识，并给予投资者排他期以锁定项目。
> 2. 增资协议/股权转让协议和股东协议：投资决策后，各方签署股份购买协议（SPA）和股东协议（SHA）。SPA规范股权的认购或转让（数量、价格、交割、陈述、先决条件、义务）。SHA规范投资后股东之间的权利义务——特殊权利、公司治理。多数投资采用增资方式；少数采用老股转让。即使采用转让方式，仍需签署SHA。
> 3. 章程：股东协议仅约束签署方。为保护投资者权利，SHA中的关键安排也写入公司章程（向工商登记的公示文件）。例如，对创始人股份转让的限制——若写入公示章程，潜在买方可以查阅，避免违反SHA。复杂章程设计与投资协议保持一致。
> 4. 交割文件：签署SPA不等于交易完成。完成以交割为标志——投资者支付资金并成为股东。付款前必须满足交割先决条件。交割文件册证明所有条件已满足：签署的交易文件、董事会/股东会决议、特殊条件证明、交割确认函、付款通知书、法律意见书。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 探路帖 | 投资意向书（Term Sheet）—— 确立意向、排他期、保密 |
| 入股契 | 增资协议/股权转让协议（SPA）—— 股权买卖的核心条款 |
| 共治约 | 股东协议（SHA）—— 投资后股东权利义务安排 |
| 衙门备案 | 公司章程（Articles）—— 公示效力，保护投资者 |
| 交清单 | 交割先决条件（Conditions Precedent） |
| 檀木箱 | 交割文件册（Closing Binder）—— 证明交易完成的完整凭证 |
| 四道关 | 交易文件的四个阶段：意向→协议→公示→交割 |
