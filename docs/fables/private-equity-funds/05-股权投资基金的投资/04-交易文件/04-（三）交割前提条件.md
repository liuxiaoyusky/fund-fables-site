# （三）交割前提条件

## 🏗️ 寓言故事 —— 《老周的验房清单》

老周在城东看中了一栋三层老宅，打算盘下来改做客栈。房主孙掌柜急着用钱，两人谈妥价钱，当场签了契约。老周握着那张纸，心里踏实了一半，以为这事就算成了。

孙掌柜催他三日内付清全款，老周却摆摆手，从怀里掏出一张折得方方正正的纸，上面密密麻麻写满了字。"孙掌柜，契约是签了，但钱还不能动。咱们得按这张单子，一条一条验过。"

孙掌柜凑过去一看，脸色顿时变了。单子上列着：房屋产权不得有纠纷，结构不得有隐患，水电线路须真实完好，邻里不得有未了的宿怨。更扎眼的是最后几条——后院那口枯井必须填埋，二楼漏雨的瓦片须全部更换，门前那棵歪脖槐树若遇大风倒了砸着路人，须由卖方担责。

"你这是信不过我？"孙掌柜把单子拍在桌上。

老周不急，给自己倒了杯茶。"不是信不过，是规矩。契约签了，不代表宅子就归我了。您想想，万一产权有官司缠身，我付了钱，衙门来人封了门，我找谁去？万一梁柱被白蚁蛀空，我住进去塌了，是您的宅子要我的命，还是我的钱买了您的棺材？"

孙掌柜哑口无言。他确实知道二楼瓦片漏雨，也晓得那棵槐树年久歪斜，只是没想到老周查得这么细。更让他心虚的是，前月邻居王铁匠还来闹过，说宅子西墙占了他半尺地，这事至今没扯清楚。

"给您半个月，"老周把茶喝完，站起身，"这半个月里，您该补的补，该填的填，该写的写。单子上的事全办妥了，我当场付钱，您当场交钥匙。办不妥，契约就只是一张纸，咱们各走各路，互不相欠。"

孙掌柜咬咬牙，接了这单差事。他先找王铁匠，赔了五两银子，立了地界文书；又雇瓦匠爬屋顶，三天换完整坡青瓦；再叫两个壮丁，把后院枯井填成平地，还铺了石板。最后，他请镇上的老秀才写了一份"俱已办妥"的保书，按了手印，一并交给老周。

老周带着泥瓦匠、木匠、邻居，逐条核对。瓦匠上屋顶敲了敲，说新瓦结实；木匠拿锤子敲了梁柱，说木质尚好；王铁匠当着众人的面，把地界文书念了一遍。老周这才从钱袋里掏出银票，双手递过去。

钥匙交接的那一刻，孙掌柜忽然想起一件事："老周，那棵歪脖槐树……我没砍。"

老周笑了笑："树不用砍。但您得写个字据，保证三年内若树倒了伤人，您仍担责。这是另一件事，钱我照付，宅子我照收，但这笔账，咱们另纸记着，日后兑现。"

孙掌柜愣了愣，终于明白：老周那张单子上的条目，有些是大门，不打开就别想进屋；有些是屋里的摆设，进了屋再慢慢收拾，不耽误住人。前者是门槛，后者是账本，各有各的用处。

半月后，客栈开张。老周坐在柜台后，常对新来的学徒说："签契约只是张口，迈门槛才是动身。门槛上的灰没扫干净，贸然进去，摔了跟头，怨不得别人。"

**📖 原文定义**

> 交割前提条件（Conditions Precedent to Closing，CP）是指交易各方履行交割义务的前提条件。通常我们所指的交割前提条件，是指投资方履行交割义务的前提条件。
>
> 在增资协议已经签署但交割前提条件尚未得到满足的情况下，除非投资方豁免其中未满足的交割条件，投资方无义务支付增资款，目标公司也无义务向投资方签发股东名册和出资证明书。因此，增资协议的签署并不当然意味着增资的完成，实践中存在着增资协议签署但交易无法完成的情况。需要说明的是，一旦交割条件得到完全的满足，则意味着投资方有义务进行交割，因此，交割前提条件的选择和设置需要充分考虑。
>
> 常规的交割前提条件包括：不存在禁止或限制交易发生的情形，无重大不利变化，陈述与保证真实准确完整，交易文件已经有效签署，批准融资的决议已经有效做出，目标公司出具交割前提条件已经得到全部满足的交割证明书。
>
> 除常规的交割前提条件之外，如果投资方在法律、财务或业务尽职调查过程中发现足以影响投资方决定是否投资的重大问题，一般也会选择将此类重大问题的解决作为交割前提条件。例如：目标公司已经完成新的员工持股计划（Employee Stock Ownership Plans，ESOP）的设立，创始人与其前雇主的知识产权纠纷已经得到妥善解决，目标公司已经回购某个特定股东的股权等。
>
> 过渡期是指从增资协议签署到交割完成的期间。在这一期间内，投资方虽然尚未支付增资款也尚未正式成为目标公司的股东，但因为增资协议已经签署，交割前提条件已经确定，所以投资方仍有必要对其在过渡期内权利以及目标公司的行为约束作出安排。
>
> 通常的过渡期安排包括：目标公司按照与过去惯例相符的正常营业方式开展业务的承诺，未经投资方同意不得做出的行为，投资方的信息查阅权利等。过渡期安排通常与交割前提条件中的"无重大不利变化"相关联，如果在过渡期内目标公司以非惯常的方式开展业务或未经投资方同意做出某些行为，则可能会导致"无重大不利变化"的交割前提条件得不到满足，进而使得交割无法正常进行。
>
> 交割后义务（Post-closing Obligations）是指目标公司在交割完成之后，仍然应当向投资方履行的义务。
>
> 交割后义务通常包含：交割完成后目标公司持续合规经营的承诺以及目标公司在交割完成之后，仍然需要履行的特定义务。在这两个方面之中，交割后义务的重点通常在于后者。
>
> 投资方在法律、财务或业务尽职调查过程中，可能会发现一些关于目标公司或创始人的风险或瑕疵事项，其中一些足以影响投资方决定是否投资的重大问题，通常会选择作为交割前提条件，而其他的一些风险或瑕疵，虽然不足以影响投资方决定是否投资，但可能会对目标公司的持续合规经营或未来的首次公开发行产生影响，在这种情况下，投资方往往倾向于将这些风险或瑕疵的妥善解决作为目标公司的交割后义务。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周与孙掌柜签契约 | 增资协议（SPA）的签署 |
| 老周不立即付钱，而是拿出验房清单 | 交割前提条件（CP）——投资方履行交割义务的前提 |
| 清单上的条目（产权无纠纷、结构无隐患等） | 常规交割前提条件（无禁止情形、无重大不利变化、陈述保证真实等） |
| 枯井填埋、瓦片更换、知识产权纠纷解决 | 非常规交割前提条件（DD中发现的重大问题） |
| 半个月的整改期，孙掌柜需维持宅子原状 | 过渡期安排——目标公司按惯常方式经营、不得擅自行动 |
| 清单全部验讫后老周才付钱交钥匙 | 交割条件全部满足后，投资方有义务进行交割 |
| 歪脖槐树不砍，但另纸保证日后担责 | 交割后义务（Post-closing Obligations）——不影响交易但需日后解决的事项 |
| 老周请第三方逐条核对 | 目标公司出具交割证明书，确认所有CP已满足 |
