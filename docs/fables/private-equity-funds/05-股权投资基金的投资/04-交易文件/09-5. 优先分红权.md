# 5. 优先分红权

## 🏺 寓言故事 —— 《账房先生的账本》

钱塘镇上有家老字号的酱园，做了快三十年，老板叫沈佑安，头发花白，做事稳当。镇东头新开了一家"陈记酱坊"，掌柜陈秉义三十出头，敢闯敢拼，第一年就把酱菜卖到了杭州府。镇上人都说，陈秉义日后怕是要把沈佑安压下去。

这一年秋天，陈秉义找到沈佑安，说要合伙。沈佑安手头正紧——今年雨水多，酱缸翻了一回，亏了不少银子；陈秉义那边却要扩产，缺钱。两人坐在茶馆里谈了一个下午，最后议定：陈秉义出一大笔银子入伙，占三成份额；往后每年酱园赚了钱，先把陈秉义这一份按约定分，剩下的再归沈佑安。

签字那天，陈秉义特意请来了镇上最老的钱秀才做中人。钱秀才问沈佑安："沈老板，账怎么个算法？"沈佑安想了想，说："每年赚了钱，先把陈掌柜应得的那份结清，多出来的是我老沈的。"钱秀才提笔写下这条，又追问一句："那陈掌柜分了这一份之后，剩下那七成里的，他还能不能再分一份？"沈佑安愣了一下，摇头："我出地、出人、出招牌，担的风险最大，剩下的就该我独享。"陈秉义在对面听着，没吭声，只点了点头。

头一年酱园生意不错，刨去开销还剩不少。年底盘账，陈秉义依约先拿走了属于他的那一份，沈佑安数着自己剩下的那叠银票，紧绷了一年的眉头才松下来。第二年赶上丰收，酱菜卖得更远，账上的结余比头年还厚。照旧，陈秉义先取走自己那一份，沈佑安看着账本，心里暗暗得意——陈秉义虽然钱多，但每一年他只能拿那一份固定的，剩下的全归自己这边。

到了第三年，行情突变。邻县也冒出了好几家酱坊，价格压得低，杭州府的客商来得少了。年底一算账，刨去陈秉义先拿的那一份之后，沈佑安手里剩下的反倒比头一年少了。他坐在账房里，对着那本账发了半晌呆。钱秀才路过，进门看他愁眉不展，问清缘由后笑了笑："沈老板，这账本上的规矩，当初可是你自己点头的。人家出钱在前，先拿他那一份，天经地义；剩下多少，才是你的。"

沈佑安合上账本，望着窗外灰蒙蒙的天，忽然想起签字那天的自己——那时候手头紧，想着"先拿人家的钱把厂子撑过去再说"，压根没细想：自己要让出去的，不只是那三成份额，更是一年里无论赚多赚少，都要让陈秉义先挑走属于他那一口的权利。剩下的，才轮到自己。

---

**📖 原文定义**

> 优先分红权（Dividend Preference）是指投资方优先于创始团队获得公司分红的权利。针对优先分红的金额，通常会有两种不同的安排，一种是参与型的优先分配，另一种是不参与型的优先分配。参与型的优先分配即投资方获得优先分配的金额后，仍要按照持股比例参与剩余金额的分配；不参与型的优先分配则是获得优先分配的金额后，不再参与剩余金额的分配。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 陈秉义先于沈佑安拿走自己应得的那一份 | 投资方优先于创始团队获得分红的权利 |
| 沈佑安拿剩下的那七成里的钱 | 优先分红之后由创始团队享有剩余分红 |
| 陈秉义每年只取自己那一份固定银子，不参与剩下金额的分配 | 不参与型的优先分配 |
| 沈佑安签字时只想着"先撑过去"，没细想让出去的权利 | 优先分红权对创始团队利益的让渡 |
