# 2. 若清算可分配财产足额扣除本轮优先清算额后仍有剩余，则该剩余可分配财产由公司全体股东（包括各投资方）按其届时各自的持股比例分配。

## 🏺 寓言故事 —— 《分余银》

江南有个老茶庄，掌柜周鹤年做茶大半辈子，铺子是他和女婿何守诚一手撑起来的——周鹤年管茶山和制茶，何守诚管客路和账目。

这年春上，南边来了位苏老板，想掺一股。三人坐在周家堂屋议定：苏老板出五百两入伙，字据上写明——"日后茶庄散伙，本金先还，剩下再分"。苏老板的银子入了账。

头两年茶庄顺当。第三年春，南边茶山遭了倒春寒，明前茶绝收；秋上一个客商又撤了单。周鹤年把账翻了几遍，叹气道："茶庄撑不下去了，得散伙。"

散伙那天，三人清点家底，存茶、应收、家伙什儿拢共折成八百两现银。周鹤年把账本摊开："苏老板的五百两本金，按字据先还。"五百两推到苏老板面前，剩下的三百两往桌中央一放，"这是账上所有的余头。"

苏老板原打算收了自己那五百两就走——字据上只写"本金先还"，并没说自己还能再分剩下的。可他抬头看了一眼周鹤年和何守诚，开口了："周老，这茶庄是您出茶山、何掌柜出客路一点一点挣下来的。我一个外人，借了五百两进来。剩下这三百两余头，本是你们攒下的家底，我一个人全拿，良心上过不去。"

周鹤年摆手："字据怎么写就怎么办。"何守诚也点头："您那五百两是本金，先还天经地义；剩下这三百两，您照理也是股东的一份。"苏老板沉默一会儿，提了个法子："要不这样，这三百两，咱三个人按当初字据里各自占的份额来分。"

周鹤年一愣："字据上可没这么写。"苏老板笑了一下："字据是字据，情分是情分。账上还有余头，股东就该按份额分账。再说，往后我还要在江南做买卖，传出去我不地道，谁还敢跟我合股？"

周鹤年想了想，也笑了："成。"于是三人按字据里的份额——苏老板两成，何守诚四成，周鹤年四成——把那三百两分了个干净。苏老板最后总共拿回五百六十两，本金完整，又额外得了六十两。他站起来冲两人作了一揖："承让，承让。"

苏老板走后，何守诚收拾茶盏，问岳父："爹，您怎么看？"周鹤年合上账本，望着门外的青石板路，慢慢说："我做了一辈子茶，见过散伙时分银子打得头破血流的。本金先还，是字据的规矩；还完本金还有余头，按份额再分一遍，那也是合伙做买卖的规矩。前一条讲的是先来后到，后一条讲的是剩下的一起担。"

---

**📖 原文定义**

> 2. 若清算可分配财产足额扣除本轮优先清算额后仍有剩余，则该剩余可分配财产由公司全体股东（包括各投资方）按其届时各自的持股比例分配。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 茶庄散伙清点，账上共有八百两现银 | 清算可分配财产（扣除法定债务后的剩余财产） |
| 苏老板的五百两本金先行如数归还 | 足额扣除本轮优先清算额 |
| 归还五百两本金后账上剩下的三百两 | 扣除优先清算额后仍有剩余的可分配财产 |
| 苏老板主动提出"按各自占的份额来分这三百两" | 全体股东按届时各自的持股比例参与剩余财产的分配 |
| 苏老板既是出钱的本轮股东，也参与余头分配 | 全体股东中"包括各投资方" |
| 周鹤年把份额定为苏两成、何四成、周四成 | 持股比例作为分配依据 |
| 周鹤年评价"按份额再分一遍是合伙做买卖的规矩" | 优先清算之后，剩余部分按比例共担共分 |
