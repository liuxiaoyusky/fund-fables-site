# 3. 投资方转让条款

## 🏮 寓言故事 —— 《茶楼的暗桩》

杭州城西湖边，有一座"听雨轩"茶楼，开了三代人。老掌柜周德山年事已高，膝下只有一个女儿周婉清。三年前，绸缎商林怀远拿出大半积蓄，入了茶楼的份子，成了二掌柜。两人约定，林怀远只管出钱，不管经营，每年按份子分红。

茶楼在林怀远的银钱注入后翻修一新，生意比从前红火了许多。周德山心里感激，却也藏着一丝不安。他常在夜里对女儿说："林掌柜是个好人，可他手里握着咱们三成的份子。万一哪天他变了心思，把这份子卖给咱们的对头，咱们周家三代的心血就完了。"

周婉清却不以为然："爹，林掌柜当初雪中送炭，咱们不能疑神疑鬼。他若想把份子转手，那是他的权利，咱们拦着，岂不是寒了合伙人的心？"

父女俩谁也说服不了谁。周德山便去找城里有名的讼师陈秀才，讨了个两全的法子。

这年清明，周德山备了一壶龙井，请林怀远到后院的樟树下喝茶。三巡过后，周德山从袖中取出一张纸，上面是他连夜写下的规矩。

"林掌柜，"周德山斟酌着词句，"您入份子这三年，茶楼能起死回生，全靠您的银钱。您这份子，想卖给谁，我周德山绝不拦着。只是——"

他顿了顿，指着纸上的一行字："唯独有一桩事，得请您担待。杭州城里，与咱们茶楼唱对台戏的只有三家：'松风阁'、'碧螺春舍'、'虎跑茶寮'。您若想把份子卖给这三家的东家，须得提前十日知会我一声。这十日里，我周家若凑得出银钱，便按您开的价，优先把这份子接过来。"

林怀远听完，先是一愣，继而哈哈大笑。他端起茶盏，一饮而尽："周掌柜，您这是怕我林怀远把茶楼卖给对头？"

周德山老脸微红，正要解释，林怀远却摆了摆手："您多虑了。我林怀远虽是个买卖人，却也知道信义二字。不过——"他收起笑容，认真地说，"您这条规矩，我认。只是我也有个条件：除了这三家对头，我若想把份子卖给旁人，您和婉清姑娘不得阻拦，也不得说什么'优先接过来'的话。"

周德山连连点头："这是自然。只要不是那三家，您卖给谁，我周德山拍手恭喜。"

两人当场在纸上画押。周德山将那三家对头的名字工工整整写在纸末，又添了一句："每年清明，这单子可更新一次，但最多不超过十家。"

林怀远看着这纸契约，心中五味杂陈。他本以为自己入了份子，这茶楼便也有他一份天地。如今才懂，合伙人的信任，从来不是一纸银钱能买断的。周德山防着他卖给对头，却也给了他向旁人自由转手的体面。这规矩看似束缚，实则是两人各退一步，给彼此都留了路。

三年后，林怀远的绸缎庄遇到难关，急需一笔银钱周转。他想把茶楼的份子转给一位做瓷器生意的旧相识。周德山闻讯，二话不说便帮着张罗过户，还备了酒席为林怀远饯行。

又过了两年，林怀远听说"松风阁"的东家暗中出价，想买他那份子。他想起当年樟树下的约定，提前十日给周德山写了信。周德山和周婉清连夜凑齐了银钱，把那份子接了回来。

交割那日，林怀远站在听雨轩门口，望着那块挂了几十年的匾额，轻声叹道："好规矩，救了三个人。"

**📖 原文定义**

> 投资方从目标公司的退出方式，除上市、回购、拖售之外，主要采取的是向第三方转让股权的形式。本节中，我们已经介绍了除上市之外的回购和拖售，以下我们将简要介绍投资方向第三方转让股权的条款（Share Transfer Right）。
>
> 根据《公司法》第84条，有限责任公司的股东向股东以外的人转让股权的，其他股东在同等条件下有优先购买权。公司章程对股权转让另有规定的，从其规定。通常情况下，投资方为了保障其能够自由转让其持有的股权，一般均会在交易文件中对股权转让做另行约定，以避免其他股东的优先购买权。
>
> 考虑到如果投资方向目标公司的竞争对手转让股权可能会对目标公司产生不利影响，创始团队可能会对投资方向竞争对手转股进行限制。常见的限制方法包括两种：一种是投资方向目标公司的竞争对手转让股权，需要事先获得创始团队的同意；另一种是创始团队就投资方向竞争对手转让股权享有优先购买权。
>
> 除本协议另有约定外，投资方持有的公司股权的处分不受任何限制，无须获得其他各股东的同意，且其他各股东放弃并承诺不行使其根据交易文件或根据法律所享有的优先购买权或其他优先权利，并承诺采取一切必要行动（包括但不限于及时签署批准转让的股东会决议）以配合投资方完成该等转让。
>
> 如投资方拟向公司的直接竞争对手（公司直接竞争对手清单见附件，该清单经投资方董事同意每年可更新一次，但清单应不超过10家）转让股权的，应当至少提前10个工作日向创始人发出书面通知，创始团队有权就该等转让行使优先购买权。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 林怀远入茶楼的"份子" | 投资方持有的股权 |
| 周德山与林怀远的合伙关系 | 投资方与创始团队的股东关系 |
| 林怀远可自由将份子卖给"瓷器生意的旧相识" | 投资方向非竞争对手自由转让股权，不受限制 |
| 周德山要求"卖给三家对头须提前十日知会" | 投资方向竞争对手转让股权需提前书面通知创始人 |
| 周德山享有"优先把份子接过来"的权利 | 创始团队对竞争对手受让股权享有优先购买权 |
| 纸末列明的三家对头，每年清明可更新，最多十家 | 直接竞争对手清单附件，每年可更新，不超过10家 |
| 周德山不得阻拦卖给非对头的人 | 其他股东放弃对非竞争对手转让的优先购买权 |
| 林怀远感慨"好规矩，救了三个人" | 平衡双方利益的条款设计，既保障投资方退出自由，又保护公司免受竞争对手威胁 |
