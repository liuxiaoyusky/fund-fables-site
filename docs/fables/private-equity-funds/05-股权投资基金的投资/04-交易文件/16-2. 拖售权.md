# 2. 拖售权

## 🏷️ 寓言故事 —— 《老钱的最后一刀》

老钱三年前和几个老乡合伙开了一家酱园，专门做陈年老酱。他投了三十万，占了两成份额，是几个合伙人里出钱最少的。大股东老周出了大头，占四成，还有几个小股东各占一成。酱园开起来后，老周管事，老钱只管年底分红，从不插手经营。

三年过去，酱园的生意越来越好。城南的赵员外是个做买卖的行家，看中了老酱园的招牌和手艺，想整个买下来，连人带牌子一起收。赵员外派人传话：我只买全部，不买零碎。要是你们七零八落地卖，我宁可不要。

老钱一听，心里动了念头。他这两成份额，单卖是卖不上价的。赵员外要的是整块招牌，不是他手里那点零碎。可老周和其他几个股东正干得热火朝天，谁也不想卖。老钱私下找老周谈了几回，老周每次都摆手："酱园才刚起步，以后日子长着呢，急什么？"

老钱不急不行。他儿子要进京赶考，家里等着用钱。他一个人说了不算，老周不卖，赵员外就不买，他的两成股份就烂在手里。

这时候，老钱想起当年入伙时签的那张契书。契书上写得明白：从开张算起，满三年之后，若是有人找到买家愿意出高价买下整间酱园，而持有份额过半数的合伙人都同意，那不同意的人也得跟着卖。但如果买家出的价钱不够高，这条规矩就不算数。

老钱把契书翻出来，又去找了另外两个占一成的小股东。三个人合计起来，正好超过了一半。他们一起去找老周，说赵员外出价不低，按酱园现在的家底算，比当初投进来时翻了好几倍，咱们谁都不会亏。

老周脸色铁青。他辛苦三年打下的江山，眼看就要被人连锅端走。他拍着桌子说："我不卖！你们能把我怎样？"

老钱把契书往桌上一拍："按契书办事。你要是不肯跟着卖，那就按赵员外出的价，把我们都买下来。"

老周愣住了。他哪有那么多银子把老钱他们都买下来？他手里攥着四成，可老钱和两个小的加起来超过了一半，契书的规矩已经触发。要么大家一起卖给赵员外，要么老周自己掏腰包把老钱他们全买走。两条路，老周都走不通。

僵持了几天，老周四处奔走，想找人借钱把老钱他们的份额接过来，可一时半会儿凑不出那么多银子。他又想拉别的买家来压价，可赵员外出的价已经是市面上最高的，没人肯出更多。

最后，老周长叹一声，在契书上按了手印。赵员外欢天喜地地接手了整间酱园，老钱拿到了钱，送儿子进京去了。老周虽然心有不甘，可也拿了一笔不小的银子，只是那间他一手做起来的酱园，从此姓了赵。

**📖 原文定义**

> 拖售权（Drag-along Right），也称"领售权"，是指在符合约定条件的情况下，投资方可以要求目标公司的其他股东与其共同向第三方出售目标公司的全部或大部分股权或资产的权利。
>
> 拖售权的目的在于保障投资方可以通过第三方并购的形式实现退出。投资方往往在目标公司中只持有少数股权，在第三方对目标公司有收购意图时，仅出售投资方持有的股权无法实现第三方并购的目的。因此，投资方可以通过强行要求其他股东与其共同出售的方式，来实现其通过第三方并购退出的诉求。
>
> （1）拖售权的行使主体。拖售权的行使主体一般为投资方，包括特定的几位投资方或合计持有一定股权比例的投资方。但实践中也有不少情况下，创始团队要求投资方行使拖售权，必须取得创始团队持有的一定股权比例的同意（即优先股的一定比例加上普通股的一定比例的同意）。另外，考虑到拖售权的行权结果是创始团队持有的目标公司股权会被强行售予第三方，创始团队为维持其对目标公司的控制，在无法争取到对于拖售的否决权时，也会考虑要求创始团队对于拖售权项下售出股权享有优先购买权。通过这种方式，创始团队有自筹资金或引入其他投资方来对抗拖售权的可能性。
>
> （2）拖售权的触发条件。拖售权的触发条件，是指仅在满足特定条件的情况下，拖售权人才能行使拖售权。常见的触发条件包括时间标准和估值标准。这两个标准可以单独设置也可以结合设置。
>
> 所谓时间标准，是指拖售权的行使，必须在交割后的一定期限经过之后，以保证目标公司有足够的时间可以运营。
>
> 所谓估值标准，是指行使拖售权时，对于目标公司的整体估值必须达到一定的标准，否则拖售权人无权行使。设定估值标准的目的在于使得拖售权人和被拖售的其他股东（特别是其他投资方）都可以获得满意的回报。一般而言，估值标准会按照最后一轮融资估值的一定倍数设置，以充分考虑和平衡各轮次投资方的利益。
>
> 如在C轮融资交割日起届满3年之后，A轮投资方、B轮投资方和C轮投资方所持的公司表决权合计总数中有2/3以上（合称"拖售方"）书面同意向善意第三方出售公司全部或大部分股权或资产（包括但不限于并购、合并、资产转让、控股权转让或其他导致公司控制权发生变更的交易形式），并且届时成交价格所依据的公司估值不低于20亿元人民币或等值美元（拖售交易），则除拖售方外的其他公司股东应同意该拖售交易，包括但不限于在公司股东会和/或促使其委派的董事在董事会上投赞成票通过出售公司股权/资产的决议、签署相关股权/资产转让合同、办理相关变更手续等。
>
> 如其他公司股东届时不同意该拖售交易的，则其有义务按照第三方提出的收购公司的条款和条件购买拖售方拟出售的全部公司股权。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老钱（两成份额的小股东） | 行使拖售权的投资方（少数股权） |
| 老周（四成份额的大股东） | 被拖售的创始团队/其他股东 |
| 赵员外只买整间酱园，不买零碎 | 第三方并购需要收购全部/大部分股权，仅买少数股权无法实现并购目的 |
| 契书约定"满三年之后"才能启动 | 拖售权的时间标准（交割后一定期限） |
| 赵员外出价"比当初翻了好几倍" | 拖售权的估值标准（达到一定估值门槛） |
| 老钱联合两个小股东过半数同意 | 拖售权的行使主体需合计持有一定比例股权/表决权 |
| 老周可以按同样价格买下老钱的份额 | 被拖售股东的优先购买权（对抗拖售权的机制） |
| 老周凑不出银子，最终被迫一起卖 | 被拖售股东无法行使优先购买权时，必须配合出售 |
