# 3. 优先购买权

## 🏠 寓言故事 —— 《老茶馆的分家》

城南有间老茶馆，是周掌柜和三个老伙计二十年前合伙开的。周掌柜掌勺泡茶，李师傅管账，王婶招呼客人，赵叔跑外采买。四人各占一份，茶馆的规矩都写在一张泛黄的契纸上，贴在柜台后头。

这年开春，赵叔的儿子在京城做了大买卖，来信催老父亲去享清福。赵叔心思活了，想把手里的份额换成盘缠，举家北迁。他私下里找了个外地客商，那人出价三百两，要接赵叔的位子。

赵叔收了客商定金，心里踏实，才想起柜台后头那张契纸。他展开一瞧，脸色变了——契纸上明明白白写着：谁想转让自己那份，得先告知其他三人，把价钱、条件、买主姓名一一说明，给其他三人十五天时间斟酌。其他三人若愿意照这个价接，买主便轮不到外人。

赵叔硬着头皮，把外地客商的姓名、出价三百两、一次付清的条件写成文书，送到周掌柜手里。周掌柜看完，递给李师傅和王婶。三人关在茶室里，一壶酽茶喝到凉透。

李师傅先开口："赵叔这份，我要接。我管账这些年，攒下些家底，三百两我拿得出。"王婶摇头："李师傅，你一个人吞了，我和掌柜的怎么办？契纸上写的是咱们三人都有份，不是谁先张口谁得。要我说，按咱们原有的比例来分，掌柜的占大头，他先挑，剩下的咱俩再按比例分，这才公道。"

周掌柜沉吟半晌，一拍桌子："王婶说得在理。赵叔要卖的是一份，不是三份。咱们三人按老规矩的比例来分，我出一百八十两接六成，李师傅出九十两接三成，王婶出三十两接一成。这样咱们谁也不用怕外人来搅局，茶馆还是老茶馆。"

李师傅起初想独占，听了这话，虽不甘心，也知王婶的提议于众人都好。三人议定，各自凑了银子，按三百两的总价、同样的条件，把赵叔的份额接了过来。外地客商得了退信和定金，虽懊恼，也无可奈何。

赵叔北上的那日，三人送到城门口。周掌柜塞给他一包新茶："老伙计，茶馆还是咱们的茶馆，你那份，我们替你守好。"

赵叔眼眶一热，拱手登车而去。马车扬尘，老茶馆的招牌在远处若隐若现，还是那三个熟悉的字。

**📖 原文定义**

> 优先购买权（Right of First Refusal），一般是指创始团队拟转让其持有的目标公司股权时，投资方在同等条件下优先于拟受让方购买的权利。优先购买权条款的主要目的在于限制创始团队的股权转让并赋予投资方增加在目标公司持股比例的一种路径。
>
> 在股东协议未设置转让限制条款，或者转让限制条款允许的情况下，如创始团队拟转让股权，则触发投资方的优先购买权。
>
> 与优先认购权类似，优先购买权的行权比例也有绝对比例和相对比例之分。绝对比例与相对比例的含义在优先认购权和优先购买权项下实质相同。不同的是，在优先认购权的情形下，以绝对比例更为常见，而在优先购买权的情形下，为最大限度地保证投资方可以购买创始团队拟转让的股权，以相对比例行使优先购买权更为常见。
>
> 受限于本协议关于转让限制的约定，如创始股东（即售股股东）拟转让其直接或间接持有的目标公司股权的，其应在与拟受让方（包括除售股股东外的其他目标公司股东）达成初步意向后、签订股权转让合同前，提前十五（15）个工作日给予投资方股东事先书面通知（即售股通知），载明该售股股东拟转让的股权（即待售股权）的数额、价格、其他重要交易条件以及拟受让方的名称等情况。投资方股东有权以售股通知中载明价格及其他重要交易条件，优先购买部分或全部待售股权的权利。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 赵叔想转让茶馆份额给外地客商 | 创始团队拟转让股权给拟受让方 |
| 柜台契纸要求先告知其他三人 | 股东协议中的优先购买权条款 |
| 十五天斟酌期限 | 提前十五个工作日书面通知（售股通知） |
| 写明价钱、条件、买主姓名 | 售股通知载明数额、价格、交易条件、拟受让方名称 |
| 李师傅想独占份额 | 绝对比例行权（一人购买全部） |
| 王婶提议按原有比例分配 | 相对比例行权（按现有持股比例分配） |
| 最终三人按原有比例分摊 | 优先购买权以相对比例行使更为常见，最大限度保证各方购买权 |
| 同等三百两、同样条件 | 同等条件下的优先购买权 |
| 阻止外地客商入伙 | 限制创始团队股权转让，保护现有合伙人/投资方利益 |
