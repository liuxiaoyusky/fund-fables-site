# 1. 优先认购权

## 🏪 寓言故事 —— 《陈老板的饼铺》

陈老板在城东开了家饼铺，手艺好、口碑佳，生意日渐红火。三年前，他找了两位合伙人一起干：出钱最多的赵掌柜占了四成股，出力最多的孙师傅占了两成股，陈老板自己留四成。三个人签了契约，白纸黑字写明了各自的份额。

去年秋天，饼铺对面开了家更大的面食店，抢走了不少客人。陈老板急在心里，想多开两家分店，可手头的钱不够。他召集赵掌柜和孙师傅商量，说想再找一位新合伙人，让对方出钱入股，一起把分店开起来。赵掌柜一听就皱了眉头，说："老陈，这饼铺当初是我投了大半的本钱，才有了今天的规模。现在你要让外人进来分一杯羹，我手里这四成岂不是要缩水？"

孙师傅也点头附和："是啊，我这三年来起早贪黑，手艺全搭在这铺子里了。新来的人没出过一分力，凭什么坐享其成？"

陈老板早有准备，从怀里掏出那份旧契约，指着其中一条说："两位别急，契约里写得清清楚楚——饼铺若要再找人合伙，你们俩有权按现在的份额，优先认购新来的那份。也就是说，新合伙人要出多少钱、占多少股，你们可以照同样的价钱先买，不让自己的份额被冲淡。"

赵掌柜眼睛一亮："这么说，我可以先买一部分，保住我这四成不变？"

"正是。"陈老板笑道，"而且按咱们当初约定的是绝对份额，不是你们俩之间重新分配。也就是说，你占四成，新出的股份你按四成比例认购，孙师傅按两成认购，这样你们各自的份额一点都不会少。"

孙师傅挠挠头："那要是我们俩不想出钱呢？"

"那就让新合伙人进来，你们俩的份额自然就被摊薄了。"陈老板如实答道，"但选择权在你们手里，不是被迫的。"

两人细细一想，觉得这条规矩公道。赵掌柜当场拍板，按原有四成比例认购了新股份；孙师傅手头紧，只认购了一小部分，剩下的才让给了一位姓钱的药材商。钱老板成了新合伙人，但赵掌柜的四成丝毫未动，孙师傅的两成也只少了一点点——那是他自己选择的结果。

不过，契约里也列了几条例外。比如去年年尾，陈老板为了留住店里做了五年的擀面师傅，送了对方一小份干股作为奖赏，这就不适用优先认购的规矩——那是奖励老伙计的，不是外人入股。再比如，陈老板曾提过，将来饼铺若做得足够大了，要到省城挂牌做公卖，那时发行的股份也不算在内——那是面向天下人的，不是私下找合伙人。还有一次，饼铺用股份换了对街一家倒闭酱园的老窖方子，这种用来换东西的发行股份，也不在优先认购之列。

年底分红时，赵掌柜看着自己账上纹丝不动的四成份额，心里踏实得很。他端起酒杯对陈老板说："当初签契约时我还嫌这条繁琐，如今才明白，这是给老合伙人系的一根安全带——车子要加速，老乘客有权利先坐稳。"

---

**📖 原文定义**

> 优先认购权（Preemptive Right），是指目标公司未来增资或发行新的股份时，投资方有权按其相对或绝对持股比例在同等条件下优先认购的权利。优先认购权的目的在于使投资方的股权比例不因目标公司未来融资而稀释。优先认购权是最常见最普遍的投资方特殊权利之一。
>
> 优先认购权的行权比例可以分为相对持股比例和绝对持股比例。相对持股比例是指各投资方按照投资方之间的相对持股比例计算其行使优先认购权可以认购的新增股份。按照相对持股比例行使优先购买权，投资方可以认购目标公司新发行的所有股份。绝对持股比例是指各投资方按照其在目标公司的持股比例计算其行使优先认购权可以认购的新增股份。按照绝对持股比例，投资方无法认购新发行的所有股份。由于优先认购权的目的在于保证投资方的股权比例不因新的融资而被稀释，所以按照绝对持股比例计算的优先认购权更为常见。
>
> 通常，优先认购权会有一些不适用的特殊情况，包括IPO时新发行的股份、为建立员工持股计划而新发行的股份、为收购其他公司而新发行的股份、资本公积转增注册资本等。
>
> 如果公司增加注册资本或发行新股，任何股东享有按照其在公司中的持股比例认购公司新增注册资本或新发股权的优先权（以下称优先认购权）。股东认购公司新增注册资本或新发股权的价格、条款和条件应与其他股东或潜在认购方的认购价格、条款和条件实质相同。
>
> 优先认购权不适用于公司发行的以下股权或股份：（a）按照董事会批准的员工期权计划发行的激励期权、股权或股份；（b）公司首次公开发行或上市时发行的股份；（c）因股票分拆、股息支付、资本重组及类似交易而发行的股权或股份；或者（d）因收购其他实体作为对价而发行的股份。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 陈老板的饼铺 | 目标公司 |
| 赵掌柜（四成）、孙师傅（二成） | 现有投资方/股东 |
| 钱老板（药材商） | 新投资方/潜在认购方 |
| 契约中"优先认购"条款 | 优先认购权条款 |
| 按原有份额认购新股份 | 按持股比例行使优先认购权 |
| 绝对份额（赵掌柜四成、孙师傅两成） | 绝对持股比例（保持原有比例不被稀释） |
| 赵掌柜保住四成、孙师傅只认购一小部分 | 投资方可以选择行使或放弃权利 |
| 同等价钱、同等条件 | 同等条件下的优先认购 |
| 送擀面师傅干股 | 员工持股计划（ESOP）例外 |
| 到省城挂牌做公卖 | 首次公开发行/上市（IPO）例外 |
| 用股份换酱园老窖方子 | 收购其他实体作为对价发行的股份例外 |
| "老乘客先坐稳" | 优先认购权的核心目的：防止股权被稀释 |
