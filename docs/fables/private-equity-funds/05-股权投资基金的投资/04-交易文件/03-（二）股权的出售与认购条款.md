# （二）股权的出售与认购条款

## 🏺 寓言故事 —— 《老钱入股》

老钱经营着一家瓷器作坊，三十年下来，作坊在镇上小有名气。账本上记着作坊的本钱为二百七十两银子，这是当年老钱和几位老伙计凑出来的家底。

这一年，老钱想做一批海外订单，缺一笔银子周转。镇上的周掌柜听说了，找上门来，说愿意出五万两银子，换作坊的一成份额。老钱心里盘算：周掌柜出五万两，只要一成，那我这作坊在他眼里，岂不是值五十万两？老钱既高兴又犹豫——高兴的是作坊被看重，犹豫的是让出一成后，自己说话还算不算数。

两人坐下来谈。周掌柜说："这五万两，我只拿三十两记入账本的本钱，剩下的四万九千九百七十两，算作我对你作坊多年心血的敬意，不记账本，但归作坊公用。"老钱一听明白了：周掌柜这是告诉自己，他出的银子分成了两块——一小块记入账本，算他在这个家里的名分；一大块不入账本，但实实在在进了作坊的库房，让作坊有了底气。

老钱翻出账本，指着那二百七十两的本钱问："周掌柜，你出五万两换三十两名分，那每一两本钱在你眼里值多少？"周掌柜笑道："老钱，你作坊三十年攒下的手艺、名声、主顾，哪是本钱账上写得清的？我按五万两换三十两来算，你账本上二百七十两的本钱，对应的就是四十五万两的实在家底。我五万两进来后，账本变成三百两，这作坊在我眼里，就是五十万两的家当。"

老钱提笔，在契书上写道：作坊本钱增三十两，由周掌柜按约认缴。周掌柜缴付五万两，占一成份额。三十两计入本钱，其余四万九千九百七十两归入公积。写罢，老钱心想，这契书上的每一个字，都决定着周掌柜出了多少、得了多少，也决定着外人眼里这作坊到底值几何。银子进了库房，名分记在账上，而作坊真正的分量，从来不在账本那几两本钱里，而在人心和契书的算法之间。

---

**📖 原文定义**

> 增资协议中的股权出售和认购条款，直接反映了投资方支付的投资款可以获得的目标公司股权数量，进而决定目标公司的估值。
>
> 股权的出售和认购条款的简略典型表述为：
>
> "公司应增加注册资本人民币30万元（'增资额'），由投资方按照约定的认购价格认购（'增资'）。增资完成后，公司的注册资本由人民币270万元增加为300万元。作为增资的对价，投资方应当向公司缴付共计人民币5000万元（'认购价格'）并在增资完成后，持有公司10%的股权。投资方增资额中的30万元计入注册资本，其余4970万元计入公司资本公积。"
>
> 根据这一条款的表述，投资方支付人民币5000万元获得目标公司的30万元注册资本，每一元注册资本的对价为5000/30=166.67（元）。目标公司在投资方投资之前的注册资本为人民币270万元，则目标公司的投前估值为2700000×166.67=4.5（亿元），目标公司的投后估值为3000000×166.67=5（亿元）。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老钱的瓷器作坊 | 目标公司 |
| 账本上二百七十两本钱 | 投资前注册资本270万元 |
| 周掌柜出五万两换一成份额 | 投资方出资5000万元获得10%股权 |
| 三十两记入账本本钱 | 30万元计入注册资本（增资额） |
| 四万九千九百七十两不入账本但归公用 | 4970万元计入资本公积（资本溢价） |
| 每一两本钱值多少 | 每元注册资本的认购价格 |
| 二百七十两对应四十五万两家底 | 投前估值 = 270万 × 166.67 = 4.5亿 |
| 三百两对应五十万两家当 | 投后估值 = 300万 × 166.67 = 5亿 |
| 契书上的字决定出多少、得多少、值几何 | 股权出售和认购条款直接反映股权数量并决定公司估值 |
