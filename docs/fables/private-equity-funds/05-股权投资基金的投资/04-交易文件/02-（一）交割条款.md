# （一）交割条款

## 🏠 寓言故事 —— 《老钱的钥匙》

老钱在镇上经营一家绸缎庄，生意红火。隔壁的布坊掌柜周老板年事已高，想把布坊交给可靠的人接手。两人谈了大半年，终于商定老钱出一笔钱入伙布坊，从此两家并一家。

谈妥那天，老钱高兴，在自家院子里摆了两桌酒，请来镇上的老账房李先生做见证。酒过三巡，周老板从怀里掏出一张写满字的纸，说："钱掌柜，咱们口说无凭，这纸上写得清楚——你什么时候把钱交到我指定的钱庄户头，什么时候布坊的账本和地契上就有你的名字。"老钱连连点头，把纸小心折好收进贴身的钱袋里。

日子一天天过去，老钱却犯了难。他绸缎庄的银子大多压在货上，一时凑不齐那笔入伙的钱。周老板派人来催了三次，老钱每次都赔笑脸说再宽限几日。到了第四回，周老板的管事板着脸送来一句话："周掌柜说了，月底之前钱不到账，这桩事就当从没提过。"老钱急得嘴上起了泡，连夜把库房里上好的绸缎低价贱卖，又向钱庄借了一笔，总算在期限的最后一天，把银子一分不少汇到了周老板指定的钱庄账户。

第二天一早，老钱揣着钱庄给的汇银回执，脚步匆匆赶到布坊。周老板早已在堂屋里等候，桌上摆着三样东西：一张盖着钱庄印章的回执，一本翻开的布坊伙计名册，上面工工整整添上了老钱的名字，还有一张盖着布坊大印的纸，写明老钱出了多少银子、占了多少份子。周老板把这三样东西一并推到老钱面前，又亲手从腰间解下一串钥匙，交到老钱手里。"钱掌柜，从今天起，布坊的东厢库房、西厢账房，你都有钥匙。这三样凭证你收好，日后万一有个说法，它们就是你的凭据。"

老钱接过钥匙，手指微微发颤。他知道，从这一刻起，自己不再是绸缎庄一个孤零零的掌柜，而是布坊名正言顺的当家人之一。那串钥匙能开门，那三样纸能立身。银子出去的那一刻只是开始，钥匙和凭证到手的那一刻，才算真正落定。

---

**📖 原文定义**

> 交割是股权投资和投融资交易文件中的核心概念之一，也是需要系统性理解的概念之一。简而言之，交割是投资方支付增资款，并正式成为目标公司股东的时点。
>
> 交割条款的简略典型表述为:
>
> "受限于本协议的条款和条件，本次交易的完成（'交割'）应于本协议所述之交割前提条件全部满足后的十五（15）个工作日内进行；交割发生之日为'交割日'。于交割日，本轮投资方应将增资款支付至公司书面指定的公司银行账户。"
>
> 交割条款中的另外一个重要概念是"交割交付物"（Closing Delivery）。交割交付物是佐证投资方支付增资款并正式成为目标公司股东的书面材料，通常由目标公司向投资方交付，包括收到增资款的银行回单、股东名册、股东的出资证明书等。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老钱与周老板谈妥入伙布坊 | 投资协议谈判与签署 |
| 老钱把钱汇到指定钱庄账户 | 投资方支付增资款至公司指定银行账户 |
| 月底之前钱不到账则交易作废 | 交割前提条件与期限约束 |
| 钱庄汇银回执 | 收到增资款的银行回单 |
| 布坊伙计名册添上老钱名字 | 股东名册 |
| 盖着布坊大印的出资占份文书 | 股东的出资证明书 |
| 三样凭证一并交给老钱 | 交割交付物（Closing Deliverables） |
| 周老板交出钥匙，老钱正式成为当家人 | 交割完成，投资方正式成为目标公司股东 |
