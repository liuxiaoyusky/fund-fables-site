# 1. 估计目标公司在股权投资基金退出时的股权价值

## 🔭 寓言故事 —— 《远眺南山的算账人》

清河镇上有个布商叫老邱，做了二十年买卖，稳得像镇上那口老井。可这一年他忽然犯了嘀咕——街坊们都说他闺女阿蘅和那个从南方回来的小子阿樵"八字不合"。

原来阿樵三年前辞了县里的差事，去南山脚下租了一片荒地，说要种一种南方才有的金丝楠木苗。一棵苗子眼下卖不出几个钱，要等十来年才能成材。可村里懂行的老把式说，这东西长成之后，一棵能抵城里一栋小院。

老邱背着媳妇，悄悄跑去看了一趟南山。他站在山脚下往上看，满坡都是拇指粗的小苗，风一吹，哗啦啦响成一片。他心里直打鼓——这要投钱，得投多少？十两？五十两？一百两？什么时候才能收回来？

回到家，老邱把账本摊开，又合上。他意识到，自己不能像看铺面那样算今天一年挣多少——这苗子今天一文不挣，明天也挣不了，要等十几年。他得换一种算法。

他跑去问镇上那个算命特别准、又懂点门道的赵半仙。赵半仙听罢，把茶杯往桌上一撂："你这就舍近求远了。你别算他今天值多少，你先算——你打算什么时候把这事儿脱手？"

老邱一愣："十来年吧。"赵半仙点头："好。十来年之后，这满山的金丝楠木长成了，按市价能卖多少？"老邱咬咬牙说："估摸能值一千两。"

赵半仙把这句话记在纸上："这就对了。你别看今天，你先把'脱手那一天'值多少算出来，今天这事儿才有个落脚点。"

老邱回家，把"脱手那一年值一千两"这行字端端正正写在账本第一页。然后他反着问自己：用什么法子估这一千两？他想了想——一是看镇上其他卖树苗的铺子，最近有人出过什么价，按那个倍数套；二是算一棵树未来十年能卖多少，按十年来折回今天——这两条路他都记下来了。

晚上媳妇问他去哪儿了，他说："去南山看了看。"媳妇又问："你到底投不投？"老邱没回答，只把那本账本压在枕头底下，闭上眼。他脑子里全是十来年后的南山——满坡金丝楠木成材，一棵能抵城里一栋小院，那就是一千两。这一千两，是他今晚闭上眼能看到的"远景"，也是他明天掏钱时心里那杆秤的"准星"。

第二天一早，他揣着五十两银子，上了南山。

---

**📖 原文定义**

> 预测投资退出的时点，然后估算该时点目标公司的股权价值。估值方法可使用市场乘数法或现金流折现法。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老邱先想"什么时候脱手" → 决定估值的时点 | 预测投资退出的时点 |
| 赵半仙让老邱"别看今天，先算脱手那一天值多少" | 估算该时点的股权价值 |
| 老邱写下"十来年之后能值一千两" | 退出时点的目标股权价值 |
| 看镇上其他铺子最近出过什么价、按倍数套 | 估值方法之一：市场乘数法 |
| 算一棵树未来十年能卖多少、按年折回今天 | 估值方法之二：现金流折现法 |
| 老邱把"一千两"压在枕头底下当准星 | 这一步是后续倒推当前价值的起点 |
