# 3. 企业价值 / 息税折旧摊销前利润

## 📜 寓言故事 —— 《折旧账》

清河镇老周商会的“江南织新”份子，账房先生阿福用“息税前利润”算锦绣织坊的企业价值时，赵掌柜又提了一个问题：“老周，锦绣织坊的织机，折旧算得慢——十年折一成；彩云织坊的织机，折旧算得快——六年折一成七。折旧快慢，直接影响息税前利润。彩云织坊折旧快，息税前利润就低；锦绣织坊折旧慢，息税前利润就高。这好比两个人比力气，一个穿厚棉袄，一个穿薄单衣——不公平。”

老周点头：“赵掌柜说得对。折旧和摊销，跟织坊本身赚不赚钱没关系，跟掌柜‘怎么算账’有关系。折旧快的织坊，账面利润低；折旧慢的织坊，账面利润高。可织机本身的价值损耗，两家是一样的——都是越用越旧。咱们要比的是‘织坊本身赚了多少’，不是‘掌柜怎么记账’。”

阿福问：“那怎么办？”老周说：“咱们再换一种算法——在息税前利润的基础上，把折旧和摊销也加回去。这叫‘息税折旧摊销前利润’。什么意思？就是织坊赚的钱，在还没扣利息、还没缴税、还没算折旧、还没算摊销之前，总共赚了多少。这笔钱，才是真正反映织坊‘造血能力’的。”

阿福翻开账本。锦绣织坊去年：净利润三百两，税五十两，利息十两，折旧四十两（织机按十年折一成，二十台织机总值一千六百两，每年折一百六十两——可锦绣织坊只折了四十两，因为两台老木机已经折完了）。摊销零两（锦绣织坊没有无形资产）。息税折旧摊销前利润 = 三百两 + 五十两 + 十两 + 四十两 + 零两 = 四百两。

彩云织坊去年：净利润四百两，税八十两，利息一百两，折旧一百两（织机按六年折一成七，二十台织机总值一千四百两，每年折约二百三十八两——可彩云织坊只折了一百两，因为部分织机已折完）。摊销零两。息税折旧摊销前利润 = 四百两 + 八十两 + 一百两 + 一百两 + 零两 = 六百八十两。

瑞祥织坊去年：净利润五百两，税一百两，利息两百两，折旧一百五十两（织机按五年折两成，二十台织机总值一千五百两，每年折三百两——可瑞祥织坊只折了一百五十两，因为部分已折完）。摊销二十两（瑞祥织坊有一项专利，每年摊二十两）。息税折旧摊销前利润 = 五百两 + 一百两 + 两百两 + 一百五十两 + 二十两 = 九百七十两。

老周说：“看，锦绣织坊息税前利润三百六十两，彩云织坊五百八十两，瑞祥织坊八百两——差距大。可加回折旧摊销后：锦绣织坊四百两，彩云织坊六百八十两，瑞祥织坊九百七十两——差距更大，但比例差不多。关键是，锦绣织坊折旧只算了四十两，瑞祥织坊折旧算了一百五十两，若把折旧统一按行业标准重新算，差距会更小。”

赵掌柜问：“那怎么用息税折旧摊销前利润算织坊值多少？”老周说：“市场上有一种倍数，叫‘企业价值 ÷ 息税折旧摊销前利润’。咱们找可比织坊，算它们的倍数，然后套到锦绣织坊上。”

阿福去打听。彩云织坊：企业价值四千两，息税折旧摊销前利润六百八十两，倍数 = 四千两 ÷ 六百八十两 ≈ 五点九倍。瑞祥织坊：企业价值五千两，息税折旧摊销前利润九百七十两，倍数 = 五千两 ÷ 九百七十两 ≈ 五点二倍。两家平均约五点五倍。

老周说：“锦绣织坊息税折旧摊销前利润四百两，按市场平均五点五倍算，企业价值 = 四百两 × 五点五 = 二千二百两。扣掉净债务二百两（欠债三百两减现金一百两），股权价值 = 二千二百两 - 二百两 = 二千两。这比用息税前利润算出来的二千一百七十六两略低，但差别不大——因为锦绣织坊折旧少，加回去也不多。”

赵掌柜又问：“那什么时候用息税折旧摊销前利润，什么时候用息税前利润？”老周说：“若两家织坊折旧摊销政策差很多——比如一家织机按三年折完，另一家按十年折完——用息税前利润比就不公平，得用息税折旧摊销前利润，把折旧摊销也剔出去。若两家折旧摊销差不多，用哪个都行。通常，咱们优先用息税折旧摊销前利润，因为它最‘干净’——利息、税、折旧、摊销，全部剔掉，只看织坊本身赚了多少钱。”

老周收起算盘，说：“息税折旧摊销前利润，是织坊的‘裸利润’——不穿衣服、不拄拐杖、不算损耗，只看它本身有多强壮。咱们投银子，投的是织坊的‘强壮程度’，不是它的‘记账技巧’。”

---

**📖 原文定义**

> 息税折旧摊销前利润（EBITDA）是指扣除利息费用、税、折旧和摊销之前的利润。息税折旧摊销前利润对应的价值也是企业价值。
>
> 息税折旧摊销前利润（EBITDA）= 息税前利润（EBIT）+ 折旧 + 摊销
>
> 企业价值/息税折旧摊销前利润乘数法的企业价值计算公式为：
>
> $$EV = EBITDA \times (EV / EBITDA \text{ 乘数})$$
>
> 企业价值/息税折旧摊销前利润乘数法不但考虑了资本结构的影响，还考虑了折旧摊销。企业的不同折旧摊销政策以及不同发展阶段的折旧摊销水平，都会影响企业的净利润和息税前利润，但息税折旧摊销前利润不受此影响。考虑到被投资企业可能存在不同的财务杠杆和资本结构，在企业价值/息税折旧摊销前利润乘数法适用的情况下，通常可考虑优先使用。
