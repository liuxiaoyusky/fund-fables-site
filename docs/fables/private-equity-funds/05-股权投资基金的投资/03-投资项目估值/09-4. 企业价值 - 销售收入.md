# 4. 企业价值 / 销售收入

## 📜 寓言故事 —— 《流水账》

清河镇老周商会的“江南织新”份子，账房先生阿福用“息税折旧摊销前利润”算锦绣织坊的企业价值时，又遇到了一个新难题。赵掌柜说：“老周，锦绣织坊去年净利三百两，可前年五百两，大前年八百两——一年不如一年。用利润算，今年值二千两，明年可能只值一千两。可它的‘流水’——每年卖出去的布——去年一千二百匹，前年一千一百匹，大前年一千匹。流水没怎么跌，利润却跌了一半。这怎么回事？”

老周说：“赵掌柜问得好。利润跌，可能是因为成本涨了、折旧多了、利息重了。可‘流水’——也就是销售收入——反映的是织坊的‘市场大小’。流水稳，说明市场还在；利润跌，说明内部管理或成本控制出了问题。若咱们只看利润，可能低估了一家‘市场还在、只是暂时没管好’的织坊。”

阿福问：“那怎么办？用流水算？”老周说：“对。市场上有一种算法，叫‘企业价值 ÷ 销售收入’。流水是所有出资人——股东和债权人——共同享有的，所以对应的是‘企业价值’，不是‘股权价值’。咱们找可比织坊，看它们‘整体值多少 ÷ 每年流水多少’的倍数，然后套到锦绣织坊上。”

阿福去打听。彩云织坊：去年流水一千五百匹，按每匹二两算，销售收入三千两。企业价值四千两。倍数 = 四千两 ÷ 三千两 ≈ 一点三三倍。瑞祥织坊：去年流水二千匹，销售收入四千两。企业价值五千两。倍数 = 五千两 ÷ 四千两 = 一点二五倍。两家平均约一点三倍。

老周说：“锦绣织坊去年流水一千二百匹，销售收入约二千四百两。按市场平均一点三倍算，企业价值 = 二千四百两 × 一点三 = 三千一百二十两。扣掉净债务二百两，股权价值 = 三千一百二十两 - 二百两 = 二千九百二十两。这比用利润算出来的二千两高多了。”

赵掌柜问：“为什么流水算出来高？”老周说：“因为锦绣织坊的利润跌，不是市场没了，而是成本失控、管理不善。流水还在，说明客户还在买它的布。若咱们投银子进去，帮它理顺成本、提升管理，利润可能回升。用流水算，算的是‘市场潜力’；用利润算，算的是‘当下表现’。两种算法，看的是不同的东西。”

可老周又皱起眉头：“但流水算法有个大毛病——它不管成本。锦绣织坊去年流水二千四百两，可成本花了二千一百两，只剩三百两利润。彩云织坊流水三千两，成本只花了二千两，剩一千两利润。两家流水差六百两，利润差七百两。若只看流水，锦绣织坊和彩云织坊差不多；可看利润，差远了。流水算法不能反映成本的影响，这是它的局限。”

赵掌柜说：“那什么时候用流水算法？”老周说：“两种情形。第一种：织坊的成本率很稳定——比如公共交通、商业服务、制药——每多卖一匹布，成本按固定比例涨。这种情形下，流水和利润成正比，用流水算方便。第二种：织坊还没赚钱，利润和现金流都是负的——比如初创的互联网公司，还在烧钱抢市场，没利润可言。这种情形下，没法用利润算，只能用流水算——流水反映的是‘市场大小’，是将来赚钱的基础。”

老周收起算盘，说：“流水算法像一把宽尺子——量的是‘市场有多大’，不是‘赚了多少’。宽尺子适合量还没赚钱的织坊，或者成本稳定的织坊。可若织坊成本失控、利润薄如纸，宽尺子会把它量得太大。用流水算，得心里记着——流水大，不等于赚钱多；流水稳，不等于将来一定赚钱。它只是‘市场还在’的信号。”

**📖 原文定义**

> 企业价值/销售收入（EV/S）反映公司的企业价值相对其销售收入的乘数。销售收入是由公司所有出资人共同享有的，所以与企业价值对应，而非股权价值。一般不采用市销率（P/Sales），除非可比公司与目标企业在财务杠杆和资本结构上非常接近。
>
> 企业价值/销售收入乘数法的企业价值计算公式为:
>
> $$EV = S \times (EV / S \text{ 乘数})$$
>
> 企业价值/销售收入乘数法主要适用于销售成本率较稳定的收入驱动型企业，如公共交通、商业服务、互联网、制药及通信设备制造公司，也可适用于当净利润和经营性现金流均为负数的初创企业，其局限是不能反映成本的影响。
