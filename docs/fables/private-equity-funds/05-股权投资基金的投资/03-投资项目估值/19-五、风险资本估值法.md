# 五、风险资本估值法

## 📜 寓言故事 —— 《赌局》

清河镇老周商会的“江南织新”份子，账房先生阿福用折现账、重估家底、散伙价算锦绣织坊的价值时，遇到了一个完全不同的难题。邻县来了一位年轻人，姓李，人称小李。小李手里有一项新技术——一种能让织机效率翻倍的“飞梭”图纸。他没有织坊、没有织工、没有客户，只有一张图纸和满腔热血。他来找老周：“周掌柜，我想建一座‘飞梭织坊’，需要五千两银子。五年后，我的织坊值十亿两。您投不投？”

赵掌柜听完，摇头：“小李，你去年亏了二百两（研发费），今年亏了一百两（差旅费），明年预计还要亏三百两（建厂费）。利润和现金流都是负数，折现账没法算——将来的银子折回今天，全是负数。重估家底也没法算——你只有一张图纸，值多少？散伙价更没法算——你还没建起来，散什么伙？这怎么投？”

老周却笑了：“赵掌柜，小李这种情形，不能用老办法。得换一种思路——‘赌局’法。”

“什么叫赌局法？咱们不问他‘今天值多少’，而是先问他‘五年后值多少’，然后倒推回今天。”

**第一步：赌五年后的价。**

老周问小李：“五年后，你的飞梭织坊若成功了，值多少？”小李说：“五年后，我的织坊有五十台飞梭织机，每台效率翻倍，一天能织二十匹。年产能三十六万匹，按每匹三两算，年销售收入一百零八万两。净利润率按两成算，年净利约二十一万六千两。按市场十倍市盈率，股权价值约二百一十六万两。若按企业价值/息税折旧摊销前利润乘数，倍数约八倍，企业价值约三百四十四万两。保守取低——股权价值约二百万两。”

老周摇头：“小李，你这是‘成了’的价。可飞梭技术可能失败——织机不稳定、织工学不会、客户不认新花样。咱们得打折。假设三种情形：

- 情形一（成功，概率三成）：五年后股权价值二百万两。
- 情形二（半成功，概率四成）：飞梭技术部分有效，效率只提升五成，五年后股权价值五十万两。
- 情形三（失败，概率三成）：技术完全失败，织坊倒闭，散伙价约五万两（织机当废铁卖、厂房当仓库租）。

加权平均 = 二百万 × 三成 + 五十万 × 四成 + 五万 × 三成 = 六十万 + 二十万 + 一万五千 = 八十一万五千两。这就是‘期望退出价’——考虑了成功、半成功、失败三种可能性。”

**第二步：赌今天的价。**

老周说：“咱们商会投小李，不是做慈善，是要赚银子。咱们定的目标收益率是八成——每年翻八成，五年后翻约十八点九倍（1.8⁵ ≈ 18.9）。若期望退出价是八十一万五千两，那今天的股权价值 = 八十一万五千两 ÷ 18.9 ≈ 四万三千两。”

赵掌柜瞪大眼睛：“四万三千两？小李要五千两，咱们投五千两，占五千两 ÷ 四万三千两 ≈ 一成一六？也就是说，咱们出五千两，只占他织坊的一成一六？”

老周点头：“对。这就是赌局法的核心——倒推。先定退出价，再定目标收益率，倒推今天的价值。目标收益率越高，今天的价值越低，咱们占的股份越高。若目标收益率降到五成（每年翻五成），五年后翻约七点六倍，今天价值 = 八十一万五千两 ÷ 七点六 ≈ 十万七千两，咱们投五千两只占约四点七成。收益率要求越高，咱们占的越多，因为风险越大。”

**第三步：赌后续稀释。**

老周又问小李：“你这五千两，是第一轮。五年中间，你可能还要再融几轮——比如明年再融一万两扩建，后年再融两万两扩市场。每融一轮，咱们的股份就被稀释。假设明年你融一万两，估值按今天四万三千两翻八成（因为过了一年），约七万七千两。新投资者占一万两 ÷ 七万七千两 ≈ 一成三。咱们原来的一成一六被稀释到一成一六 × (1 - 一成三) ≈ 一成零一。后年你再融两万两，估值再翻八成，约十三万八千两。新投资者占两万两 ÷ 十三万八千两 ≈ 一成四五。咱们被稀释到一成零一 × (1 - 一成四五) ≈ 零点八六五成。”

“所以，咱们今天投的时候，不能只占一成一六。得考虑后续稀释，倒推今天该占多少。若咱们想在退出时占一成，今天得占约一成一六 ÷ (1 - 一成三) ÷ (1 - 一成四五) ≈ 一成五五。也就是说，咱们今天投五千两，要占一成五五，而不是一成一六。小李，你同不同意？”

小李想了想：“周掌柜，一成五五意味着我的织坊今天值约三万二千两（五千两 ÷ 一成五五）。比您算的四万三千两还低。您是不是太保守了？”

老周说：“小李，这不是保守，是赌局。您现在没有利润、没有现金流、没有资产，只有一张图纸。咱们投的是‘赌您五年后成功’。赌局有赔率——咱们商会定的赔率是每年八成。您若觉得赔率太高，可以找别的庄家。但别的庄家也会算同样的账。市场上，投您这种‘早期、高风险、高回报’的织坊，赔率通常在五成到一倍之间。咱们八成，不算离谱。”

**第四步：赌结果。**

老周最后说：“赌局法不是算命，是‘定规矩’。咱们先定退出价（考虑了成功、半成功、失败三种情形），再定目标收益率（反映风险大小），再倒推今天价值，再考虑稀释，最后定持股比例。每一步都是假设，每一步都可能错。但赌局法的好处是——它把‘不确定’变成了‘可计算的不确定’。咱们不是拍脑袋说‘值五千两’，而是说‘若五年后值八十一万五千两，收益率八成，稀释两轮，今天值四万三千两，咱们投五千两占一成五五’。这些数字，将来可以验证——五年后看，退出价是不是八十一万五千两？收益率是不是八成？稀释是不是两轮？若都对，赌局法算得准；若不对，下次调参数。”

赵掌柜听完，叹道：“老周，这赌局法，像赌场里算赔率。赌场不怕输，怕的是赔率算错了。咱们投小李，不怕失败，怕的是‘以为成功率三成，实际只有一成’——那退出价、收益率、持股比例，全错了。”

老周点头：“对。赌局法的核心不是‘算得准’，而是‘算得清’——把每一步假设都摆出来，将来可以复盘、可以调整。小李的飞梭织坊，咱们投五千两占一成五五。五年后，若成功了，咱们赚十八倍；若半成功，赚四倍；若失败了，亏五千两。三种结果，咱们心里都有数。这叫‘愿赌服输’——不是瞎赌，是算清赔率再赌。”

小李最终同意了。老周在契约里写明：投资五千两，占一成五五；目标收益率八成；预计五年退出；若中途再融资，按比例稀释；若五年后未退出，可延长两年或启动回购条款。赵掌柜看着契约，说：“老周，这赌局法，比折现账、重估家底、散伙价都刺激。它不是算‘现在值多少’，是算‘将来可能值多少、今天该占多少’。赌的是未来，押的是现在。”

老周笑：“对。小李没有今天，只有将来。咱们不能用‘算今天’的方法算‘没有今天的人’。赌局法，就是给‘只有将来’的人定价的方法。”

**📖 原文定义**

> 如果股权投资基金拟投资的目标公司处于创业早期，利润和现金流均为负数，未来回报虽可能很高但存在高度不确定性，此时，往往会先评估目标公司退出时的股权价值，再基于目标回报倍数或收益率，倒推出目标公司的当前价值。这种方法被称为风险资本估值法。
>
> （二）风险资本估值法的步骤
>
> 1. 估计目标公司在股权投资基金退出时的股权价值
>
> 预测投资退出的时点，然后估算该时点目标公司的股权价值。估值方法可使用市场乘数法或现金流折现法。
>
> 2. 计算当前股权价值
>
> 使用目标回报倍数或者收益率将目标公司退出时的股权价值折算为当前股权价值。计算公式如下：
>
> 当前股权价值 = 退出时的股权价值 / 目标回报倍数 = 退出时的股权价值 / (1 + 目标收益率)^n
>
> 某股权投资基金拟投资从事智能机器人项目的E公司股权，预计5年退出时E公司的股权价值为100亿元，目标收益率为80%，则E公司的当前股权估值计算如下：
>
> E公司的股权价值 = 100 / (1 + 80%)^5 = 5.3（亿元）
>
> 3. 估计股权投资基金在退出时的要求持股比例
>
> 投资额除以当前股权价值，得到股权投资基金为获得目标回报倍数或收益率应有的持股比例。计算公式为：
>
> 要求持股比例 = 投资额 / 当前股权价值
>
> 4. 估计股权稀释情况，计算投资时的持股比例
>
> 若目标公司没有后续轮次的股权融资，则不会稀释股权投资基金的股权，投资时的持股比例即上一步计算出的要求持股比例。若目标公司有后续轮次的股权融资，需估计股权稀释情况，倒推出投资时的持股比例。
