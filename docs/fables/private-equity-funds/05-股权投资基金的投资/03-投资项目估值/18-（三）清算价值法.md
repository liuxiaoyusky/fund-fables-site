# （三）清算价值法

## 📜 寓言故事 —— 《散伙价》

清河镇老周商会的“江南织新”份子，账房先生阿福用折现账、重估家底法算锦绣织坊的价值时，赵掌柜又提了一个问题：“老周，折现账算的是‘将来能赚多少’，重估家底算的是‘今天重建值多少’。可万一锦绣织坊明天就关门了——张锦绣跑了、织工散了、客户没了——这时候它值多少？不是重建，是‘拆散卖零件’，能换多少银子？”

老周点头：“赵掌柜问到了最悲观的情况。有一种算法，不问将来、不问重建，只问‘若现在散伙，把锦绣织坊拆成一块一块卖，能卖多少’。这叫‘散伙价’。”

阿福问：“怎么拆？”老周说：“假设锦绣织坊明天破产，所有东西要拍卖、要变卖。这时候，买家不是买‘织坊’，而是买‘零件’——织机、厂房、仓库、存货、地皮，一样一样卖。每一样都得打折，因为急卖、散卖、没有整体价值。”

老周带着阿福、吴师傅，模拟了一场“散伙拍卖”。

**第一项：织机。** 二十台织机，十八台新水力织机、两台老木机。陈师傅说：“新水力织机，若整批卖给另一家织坊，每台能卖七十两。可散伙拍卖，买家知道卖家急，会压价。而且散卖不如整卖值钱——买家得自己运、自己装、自己调试。拍卖价按重置全价的六成算：每台九十两 × 六成 = 五十四两。十八台 = 九百七十二两。”

“两台老木机，更惨。新水力织机普及后，老木机没人要。只能当柴烧，或者卖给怀旧收藏家。每台最多卖五两。两台 = 十两。”

织机散伙价合计：九百七十二两 + 十两 = 九百八十二两。

**第二项：厂房和仓库。** 吴师傅说：“厂房三年前盖的，当时五百两，现在重置六百两。可散伙拍卖，厂房不能拆走，只能连地皮一起卖。地皮是宅基地，不能商用，买家受限。厂房加地皮，按重置全价的四成算：六百两 × 四成 = 二百四十两。”

“三间仓库，一间新的、两间旧的。新的那间，可以拆走材料卖给别的建商，按重置全价的三成算：二百两 × 三成 = 六十两。旧的两间，结构老化、漏雨，拆走材料不值钱，只能当废木料卖。两间合计约二十两。”

厂房仓库散伙价合计：二百四十两 + 六十两 + 二十两 = 三百二十两。

**第三项：存货。** 阿福说：“存货三百匹上等细布、四百匹中等粗布、二百匹次等布。上等细布，若慢慢卖给绸缎庄，每匹三两。可散伙拍卖，急卖，买家知道卖家没选择，会压价。按市价七成算：三百匹 × 三两 × 七成 = 六百三十两。中等粗布，市价每匹一两五，急卖按六成：四百匹 × 一两五 × 六成 = 三百六十两。次等布发霉了，几乎没人要，只能当抹布或回炉重纺。二百匹 × 五钱 × 三成 = 三十两。”

存货散伙价合计：六百三十两 + 三百六十两 + 三十两 = 一千零二十两。

**第四项：应收账款。** 阿福说：“账面上应收账款五百两——瑞祥绸缎庄三百两、邻县布商二百两。瑞祥绸缎庄信誉好，但散伙拍卖时，应收账款不能立刻变现。得打折卖给专门收债的人。按账面八成算：三百两 × 八成 = 二百四十两。邻县布商那二百两有争议，收债的人只肯出三成：二百两 × 三成 = 六十两。”

应收账款散伙价合计：二百四十两 + 六十两 = 三百两。

**第五项：现金和银行存款。** 阿福说：“账上现金一百两，银行存款五十两。这些不用打折，直接变现：一百五十两。”

老周汇总：织机九百八十二两 + 厂房仓库三百二十两 + 存货一千零二十两 + 应收账款三百两 + 现金一百五十两 = 二千七百七十二两。

然后扣掉必须还的债：欠钱庄三百两、欠供应商二百两、欠织工工钱五十两、散伙清算费用（拍卖费、讼师费、搬运费）约三十两。总负债 = 五百八十两。

散伙价净值 = 二千七百七十二两 - 五百八十两 = 二千一百九十二两。

赵掌柜问：“这散伙价比重估家底的三千七百二十九两低多了，为什么？”老周说：“重估家底算的是‘正常重置’——织机按重置全价、厂房按重置全价、存货按市价。散伙价算的是‘急卖、散卖、拆卖’——每一样都得打折，而且整体没了，零件不值钱。散伙价比重估家底低，是因为‘急’和‘散’。”

老周又说：“散伙价还有个特点——它假设锦绣织坊‘死了’，不是‘活着’。活着的织坊，织机、厂房、存货、客户、手艺、品牌，合在一起才有价值。死了的织坊，零件拆卖，价值大打折扣。所以，散伙价是最悲观的估值——它回答的是‘若一切最坏的情况发生，咱们最少能拿回多少’。”

赵掌柜问：“什么时候用散伙价？”老周说：“三种情形。第一：锦绣织坊真的破产了——比如张锦绣卷款跑了、债主追上门了——这时候只能算散伙价。第二：作为‘底线’——折现账算出来五千两、重估家底算出来三千七百两、散伙价算出来二千一百两。散伙价是最低底线，出价不能低于这个数，否则万一将来不行了，连本都保不住。第三：并购谈判中——若锦绣织坊经营不善，买家想压价，会拿散伙价说事：‘你织坊现在值多少？散伙也就二千一百两，我出二千五百两已经算高了。’”

老周顿了顿，又说：“但散伙价有个局限——它假设所有东西都能卖掉。实际上，有些资产根本卖不掉——比如锦绣织坊的‘锦绣’招牌，没注册、没名气，散伙时一文不值。比如张锦绣的祖传手艺，他跑了，手艺就没了，没法卖。比如客户关系，散了伙，客户立刻转投彩云织坊，客户关系瞬间归零。散伙价算的是‘有形资产’，无形资产在散伙时几乎全灭。所以，散伙价往往高估了实际能变现的价值——因为有些资产，死了就不值钱了。”

赵掌柜叹道：“老周，散伙价像‘棺材本’——人活着的时候不想算，可真到了那一天，才知道棺材本够不够厚。折现账算的是‘活着的上限’，重估家底算的是‘现在的中限’，散伙价算的是‘死了的下限’。三种价，三种活法。”

老周收起算盘，说：“对。咱们投银子，投的是‘活着的锦绣织坊’，所以折现账是主。但心里得装着‘重估家底’——万一经营不善，资产还能换多少。更得装着‘散伙价’——万一彻底完蛋，最少能拿回多少。三种价，三种保险。保险足了，银子才敢投。”

**📖 原文定义**

> 清算价值法假设企业破产或解散时，将企业拆分为可出售的几个业务或资产包，并分别估算这些业务或资产包的变现价值，加总后作为企业估值的参考标准。
>
> 清算价值法评估的步骤如下：
>
> 通常采用清算价值法估值时采用较低的折扣率。对股权投资基金而言，清算难以获得良好的投资回报，企业在正常可持续经营情况下，不会采用清算价值法。
>
> 某股权投资基金所投资的企业F公司由于经营不善，已经进入破产清算程序，该股权投资基金所占的企业股权比例为20%，企业账面资产总值10亿元，负债、利息和破产费用合计8.5亿元，实收资本4亿元，累计亏损2亿元。经法院拍卖资产，资产变现总值为10.8亿元，则该股权投资基金持有F公司的股权估值计算如下：
>
> 持有 F 公司的股权价值 = (10.8 - 8.5) × 20% = 0.46（亿元）
