# （三）投资前价值与投资后价值

## ⚖️ 寓言故事 —— 《分肉记》

村东头的老孙家养了三年的肥猪入了冬该出圈了。屠户老黄按规矩要先到孙家看猪定分量，再约个日子来宰。这一天，老黄背着杆大秤进了孙家院子，远远瞧见那猪正卧在墙根晒太阳，油光水滑，少说也有三百斤。老黄心里琢磨：这猪孙家养了三年，光猪崽、饲料、人工加起来怎么也得投进去两千块大洋的底子，再加上这三年的心血和功夫，这头猪眼下值多少呢？

老孙蹲在门槛上吧嗒吧嗒抽着旱烟，半天没说话。他心里有本账——猪圈、猪崽、过去三年喂进去的玉米红薯、每天起早贪黑的照料，这些全是已经花出去的，没法收回来。要是有个懂行的人愿意"进场"一起把这头猪养大、卖个好价钱，那他自己这三年的底子值多少？今天这头猪还没长成的分量又该值多少？他总觉得这两件事不是一码事，可又掰扯不清。

正犯难呢，村西卖豆腐的林嫂急匆匆跑进门来说，下个月镇上王屠户要嫁闺女办流水席，让她帮着张罗，其中一道硬菜就是红烧肉，足足要订二百斤上好的。她愿意先付一半定金，剩下的一手交肉一手交钱。老孙一听，脑子里"嗡"地一下亮了：林嫂这一进一出，他家的猪就身价不一样了。林嫂付进来的定金，加上他原来的底子，就是"接下来"这头猪的全部分量；而他孙家自己这三年的投入，那是"进"之前就摆在那里的，跟林嫂没关系。

老孙"啪"地把烟袋锅子一磕，站起来跟林嫂算了一笔明白账："嫂子，这头猪我原来这身家当，再加上你这一回投进来的定金，拢共就是这猪往后到出栏时的全部身价；这中间差的那一截，就是你这一注的功劳。我自己那老底子，是入伙前的；你这一注加进来之后整头猪的斤两，是入伙后的。两个数儿不是一个数儿，但中间就隔着你这笔钱。"

林嫂听完连连点头："这么一讲就清楚了——原来我那注钱进账前后，你家这门面'盘子'不一样大了，我进来的这一笔，正好就是中间那个差。"老黄在一旁听得直乐："老孙啊，你这抽了半袋烟的功夫，把'进门'前后的数儿分得门儿清，比账房先生算得还利索！"

---

**📖 原文定义**

> 投资前价值（Pre-money Valuation）是指在某轮融资之前目标公司的股权价值，投资后价值（Post-money Valuation）则是指完成该轮融资之后目标公司的股权价值。二者的差额等于该轮融资中以增资方式投入目标公司的金额，即：
>
> 投资后价值 $=$ 投资前价值 $+$ 本轮新增股权投资金额
>
> 从股权比例来看，本轮新增股权投资占被投资企业的股权比例，等于本轮投资金额占目标公司投资后价值的百分比，即：
>
> 本轮新增股权投资持股比例 $=$ 本轮新增股权投资金额/目标公司投资后价值

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老孙家三年攒下的猪圈、饲料、人工等"底子" | 投资前价值（Pre-money Valuation） |
| 林嫂付进来的订金 | 本轮新增投入金额 |
| 老孙原本的底子 + 林嫂这一注 = 整头猪出栏时的全部身价 | 投资后价值（Post-money Valuation）= 投资前价值 + 新增投入 |
| 两个数之间差的"那一截"，正好是林嫂的功劳 | 投资后价值与投资前价值的差额 = 本轮新增投入 |
| 林嫂出钱后她在这门生意里"占多少份儿" | 本轮新增投入占投资后价值的比例 |
