# （一）账面价值法

## 📜 寓言故事 —— 《老何的棉布账》

光绪十二年腊月，松江府七宝镇上开着三十年的"何记棉布庄"要易主。东家老何膝下三个闺女都出了阁，铺面、仓里压着的棉布、柜上的老伙计，连同街口那块招牌，要一起转出去。

接手的是同乡陈老爷。他把他账房的周先生从苏州叫来，两个人坐在何记二楼的小账房里，把何家三十年攒下的那本厚账从头翻起。

周先生先点资产那一头。仓里压着的棉布按进货价一笔笔算下来，总共值八百两。他拿朱笔在"应收"那栏里挑出两笔小字批注——"松江府衙李师爷赊的二十两"，李师爷去年底病故，账是死账；"青浦张记布行欠的十五两"，张记去年秋就关了铺。两笔各折一半，伤这一头十七两半。再翻到"器具"那页，老柜、量尺、库房铁皮柜，记着值六十两；可墙根那架老织机锈得咯吱响，铁皮柜门也合不严实，周先生又批了"折损十两"。最末一行写着"字号招牌"，何记在七宝镇开了三十年，那四个字是老秀才题的墨宝，账上挂的银数是一百两。周先生没动这一笔，倒在旁边贴了张小纸条："镇上有口皆碑的字号，行里人心里有数。"

翻到负债那一头，问题更多。"应交税银"那栏只记了六十两，可周先生听老伙计提起，前年布庄替府衙赶制冬衣垫过一笔料钱，至今没结。他提笔补上十二两。"未入账"那栏账上空着；可周先生翻了柜里老何留下的两封信——一封是老何答应过隔壁米铺王老板，替他在外乡担保进过一批米；另一封是布庄里一个跟了二十二年的老织工阿贵，前年伤了腰，老何答应每月贴三钱银养伤，这笔账上没记。周先生一气补上五十两。最末一项"未核定税金"那栏，老伙计说"前年丰年里多收过一季好布，那笔税金还没和衙门结清"，周先生又提笔加了八两。

周先生算完，把数报给陈老爷听。资产这一头：棉布八百两、器具六十两、字号一百两、应收款折完减二十七两半，是九百三十二两半。负债这一头：应交税银加未入账加未核定税金，拢共一百三十两。两下一抵，何记家当值八百零二两半。

陈老爷拿过算盘，拨了几粒珠子，抬头对老何说："镇上的人都说何记值一千两，我得说句实在话——柜上这本账翻完，能落到这个数。"老何捋着胡子点了一下头。陈老爷再加五十两，凑个八百五的整数，把何记接过来。

周先生合上账簿，跟陈老爷小声说一句："这本账上记的数，是何家三十年一笔笔记下的；可账是死的，镇上是活的。账上有的数，得照行里头的实情调一调；账上没记的事，得照柜上老人的嘴补一笔。这么一过一补，落下来的那个数，才是这本账真正值多少。"

陈老爷点头——柜上这本账，做的不是数，是数背后的实情。

---

**📖 原文定义**

> 账面价值法是指基于企业会计报表账面科目金额，调整估值计量与会计计量之间的差异，从而得出相关科目真实价值的方法。例如：
>
> (1) 对资产项目的调整, 应注意公司应收账款可能发生的坏账损失、外贸业务的汇兑损失、有价证券的市值是否低于账面价值、固定资产的折旧方式是否合理，以及无形资产（如专利权、商标权和商誉）的评估弹性。
>
> （2）对负债项目的调整，应注意是否有未入账的负债（如职工退休金、预提费用）、担保事项等或有负债以及尚未核定的税金等。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 何记柜上三十年攒下的厚账簿 | 企业会计报表的账面科目金额 |
| 应收账款中"李师爷病故欠的二十两""张记布行关铺欠的十五两"按一半折损 | 调整应收账款可能发生的坏账损失 |
| "丰年里多收好布"那笔税金在账上没结清 | 调整尚未核定的税金 |
| 隔壁米铺王老板的"担保事项"在账上没有 | 调整担保事项等或有负债 |
| 老织工阿贵的"每月三钱银养伤"在账上没记 | 调整未入账的负债（如职工退休金、预提费用） |
| 老织机锈得咯吱响、铁皮柜门合不严——器具折损十两 | 调整固定资产折旧方式是否合理 |
| 何记那四个字的老字号招牌账面挂一百两未动 | 评估无形资产（商标权、商誉）的弹性 |
| 周先生把账上"应收"和"器具"两栏批注、修改后报出的八百零二两半 | 账面价值法——调整估值计量与会计计量之间的差异，得出真实价值 |
