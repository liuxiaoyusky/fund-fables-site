# 5. 市净率

## 🏦 寓言故事 —— 《两家当铺》

老宋在城南开了二十年的当铺，柜上一杆老秤、一本厚账，把镇上每一件押进来的金镯子、玉佩、皮袄记得清清楚楚。年初商会换届，他被推举去给同行做账目核查。这一查，问题来了——同行老钱的铺子，账面写着"资产三十万两"，可老宋翻开明细一看：金条、古董字画、别人赊账借出去的借条，密密麻麻挤在一起。

老宋皱着眉问老钱："老哥，你这三十万两的家底，到底能立刻变成多少现银？"

老钱笑了笑："那可说不准。古董摆一年可能翻倍，借条嘛，遇上老赖就打了水漂。金条倒是实在，化成金子就能兑钱。"

老宋合上账本，心里有了数。同样的"三十万两"，开当铺的老钱和开钱庄的孙四郎，差得远了。老钱那些"家底"，大多压着不能动；孙四郎的家底，却是一箱箱现银和借据，加起来真的能兑出三十万两现银。

月底商会聚在一起，按各家铺子规模排座次、定会费。几个老掌柜各执一词：开粮仓的说"我存了十万袋粮"，开布庄的说"我架子上堆了八万匹布"——这些货色，账面价值是真的，但谁能立刻把它变成现银？谁也说不清。

轮到孙四郎报数，他往桌上一拍账本："我家底三十二万两现银加放贷，随时可兑。"老宋一怔。同样的账面，孙四郎的"三十万"是货真价实的三十万；老钱那些古董字画，至少要打个七折；老李那八万匹布，遇上行情不好，怕是连六折都卖不到。

散席后，老宋回家路上琢磨出一句话——看一家铺子值多少钱，光看账面上的"家底"还不够，得看这家底里**能立刻换成现银的部分**到底占多少。铺子里能立刻变出钱来的家底越多，那账面数字就越实在；越压着不能动的家底越多，账面数字就越虚。

他把这道理悄悄记在了自己的小本子上，从此每看一家同行的账本，先分一分——哪些是"硬家底"，哪些是"虚家底"。这一分，同行之间谁家殷实、谁家浮夸，就清清楚楚了。

**📖 原文定义**

> 市净率（P/B）反映公司的股权价值相对其净资产的乘数，其计算公式为：
>
> 市净率=股权价值÷净资产
>
> 或：
>
> 市净率=每股价值÷每股净资产
>
> 市净率乘数法估值与市盈率乘数法类似，先确定可比公司的市净率作为目标公司估值的市净率乘数，然后使用下述公式计算目标公司股权价值或每股价值：
>
> 股权价值 $=$ 净资产 $\times$ 市净率乘数
>
> 或：
>
> 每股价值 $=$ 每股净资产 $\times$ 市净率乘数
>
> 市净率乘数法比较适用于资产流动性较高的金融机构，因为这类企业的净资产账面价值更加接近市场价值。例如，银行业的估值通常会用市净率乘数法。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老钱当铺账面写着"资产三十万两" | 净资产的账面价值 |
| 老宋问"能立刻变成多少现银" | 资产的流动性判断 |
| 孙四郎的现银加放贷"随时可兑" | 流动性高的资产，账面价值≈市场价值 |
| 老钱的古董字画"至少打七折" | 流动性低的资产，账面价值偏离市场价值 |
| 老宋用"硬家底"和"虚家底"来区分 | 净资产质量差异决定估值方法的选择 |
| 当铺、钱庄、粮仓、布庄不同的家底构成 | 不同行业适用不同估值方法 |
