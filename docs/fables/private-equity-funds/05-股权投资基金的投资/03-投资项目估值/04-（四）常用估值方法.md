# （四）常用估值方法

## 🔍 寓言故事 —— 《四桩买卖》

光绪十二年腊月，绍兴城里有个掮客叫沈怀德，人称"沈半城"。他不卖货不囤货，专替人搭桥——把要卖铺子的东家介绍给要接手的买主，从中抽一笔中人钱。买卖做了大半辈子，老沈遇过的生意五花八门，可有一件事他琢磨了好多年没琢磨透：两家铺子摆在面前，凭什么东家开出的价和买主愿意出的钱，能差出好几倍？那本账，到底该按什么路子来算？

那年冬月，茶馆里一上午来了三拨人。头一拨是个穿长衫的中年人，把老沈拉到角落压低声音说：自家侄儿在省城开了个做洋火的小作坊，烧了三年窑账上还亏着本，最近有人想"入伙"一起干，可这作坊到底值多少，谁也说不清。第二拨是个戴瓜皮帽的胖掌柜，南街开糕饼铺的，开了二十多年，月月进账都差不离，每年稳稳当当赚二百两纹银，想把这铺子连牌子一块儿"出手"。第三拨是个姓胡的老木匠，镇外有座烧了十几年的砖窑，窑洞、砖架、存下的几万块青砖都是真金白银堆出来的，可年纪大了干不动了，想把砖窑连家底一起盘出去。

老沈关上门琢磨起来——洋火作坊账上一塌糊涂，明后赚不赚谁也说不清；糕饼铺倒是稳当，可去哪儿找一个"差不多"的同行做参照？砖窑更怪，一砖一瓦都摆在眼皮底下，可这砖窑的"本事"全在那眼窑、那座架子上，不在每年赚多少银子上。

第二天清早，老沈把三人请回茶馆。先跟洋火作坊的侄儿说：你要"入伙"，账上没东西可比，可你们前阵子不是刚拉了一个姓陈的合伙人进来？他那笔"入伙钱"就是你作坊眼下最贴近市面的一个"参照数"——就照着这个路子来，等来年账面好转了，再按那时的"行情"折回去修一修。侄儿眼睛一亮，连说"省了我半个月的闷头账"。

接着老沈又跟糕饼铺的胖掌柜讲：你这铺子稳当得很，年年赚得差不多。我只要去城西、城北找几家"开得年头、口碑、赚头都差不多"的同行，看看他们"一年赚的"和"整间铺子做下来要多少钱"之间是个什么比例，再把这个比例套到你头上，就八九不离十了。胖掌柜点点头——他原来对面茶馆也是这么估的。

最后轮到砖窑的胡木匠。老沈一开口就说：砖窑这种"家当"生意，别看每年进账多寡，要紧的是你那眼窑、那座架子、那一堆砖值多少钱——把这些"实打实的东西"一样一样加起来，再扣掉欠别人的债，剩下的就是你砖窑的眼下身价。胡木匠听完就笑了："我当年造这窑时一笔一笔的账都留着呢，回家翻账本去！"

老沈送走三人，在账簿上写下几行字——同样是"值多少"这件事，作坊要"看最近一笔买卖"，稳当铺子要"找同行比一比"，砖窑要"把家当堆一摞"；这几套路子都备齐了，摊主拿什么生意来，他心里都先有个底。

**📖 原文定义**

> 因企业所处阶段不同、行业不同、经营环境不同，对企业进行估值的估值方法也不尽相同。常用的估值方法包括市场法、收益法、成本法、风险资本估值法等（见表5-1）。市场法使用可比价值对目标公司进行价值评估，广泛应用于创业投资基金和并购基金。收益法主要为现金流折现法，适用于现金流稳定、未来可预测性较高的公司。成本法适用于价值主要来自其占有资产的企业，如重资产型企业或投资控股企业，也可用于经营情况不佳、可能面临清算的企业。风险资本估值法主要用于处于早期创业阶段的企业估值。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老沈遇上的三桩生意各有各的难处 | 不同企业阶段、行业、经营环境需要不同的估值方法 |
| 洋火作坊账上亏本、老沈让他照"陈姓合伙人刚入伙那笔钱"来估 | 市场法（参考最近一笔可比价格，应用于早期、尚未稳定盈利的企业） |
| 糕饼铺稳当，老沈去城里找"差不多"的同行作比照 | 市场法（市场乘数法，应用于现金流稳定、可比公司可得的情形） |
| 砖窑把窑、架子、存砖一样一样加起来再扣债 | 成本法（家当型/重资产型企业的估值方法） |
| 老沈写在账簿上的"路子得备齐" | 收益法、风险资本估值法等不同方法对应不同情形 |
