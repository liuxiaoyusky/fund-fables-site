# （二）红利折现模型

## 💰 寓言故事 —— 《茶庄的算账》

民国二十六年春天，杭州城涌金门外的老街上，林素云把父亲留下的"林记茶庄"接过手刚满半年。这茶庄从前年还能给伙计们分点红，到她接手时却已经停分了两年——父亲病中那阵子，钱都填进了新焙茶炉的窟窿。林素云想把这铺子卖掉，自己回苏州陪母亲。街坊给她介绍了两个人：一个是城西绸缎庄的吴老板，手里钱多；另一个是外埠来的郑掌柜，做南北货转手生意。

吴老板上门那天，林素云沏了杯龙井，把账本摊在桌上。她说："这铺子现在一年能分给东家的钱就这么多，街坊四邻都看着，往后两三年也大概就这个数。"吴老板点点头，没追着问。他要的是个稳当、能长出手的铺子——他盘算着，自己接过来，五年之后若再出手，按眼下城门外这条街的涨势，再加上这两年攒下的分账，值当得很。

郑掌柜来的时候，林素云说了几乎一样的话。郑掌柜却立刻皱起了眉："停分两年的铺子，我凭啥信你三年里头能分？"他掏出小算盘拨了一通，又问："若你肯把父亲那年新添的焙茶炉折给我，我现银多给一成。"林素云摇了摇头——那炉子是她想留下的念想。

送走两人，林素云坐在柜后头发怔。她问自己：我到底该让这铺子值多少？吴老板那种算法，是把"眼下到手的分账"一季一季地照着街上的息钱往回折，再把"五年后卖出铺子能得的那个数"也照同一种折法往回压——两笔数加起来，才是他今天肯掏的现银。郑掌柜那种算法，干脆把未来的分账估作零，只算"现下能拿到什么、铺子还剩哪些物件"。

夜里雨落下来，林素云翻出父亲的老账册，把两家出的价一笔一笔重新誊在一张宣纸上。她忽然明白：吴老板那套算法里头其实藏着一个假设——茶庄在这两三年里，年年分到东家手里的那份钱，得是稳稳的、看得见的、可数的；最后出手那年的卖价，也得是照着这个"稳"字来的。一旦中间哪一年不按这个数分了，或最后那年茶市大落卖不上价，他这套账就要重做。郑掌柜正是因为不信"稳"两个字，干脆不要那几年的分账，只算眼下能拿的。

她合上账册，对着窗外的雨丝轻轻叹了口气。一间铺子值多少，归根到底看两件事：你接它那几年里实实在在分到手的，加上你放手那年再脱手能换回的——但这两笔都得能算得"稳"，才摆到同一张桌上。

**📖 原文定义**

> 股权投资基金投资于目标公司的股权，预期获得两种现金流：一是持有股权期间的现金红利；二是持有期末卖出股权时的价格。二者的现值之和即目前的股权价值。计算公式为：
>
> $$
> V = \sum_{t = 1}^{n} \frac{DPS_{t}}{(1 + r)^{t}} + \frac{P_{n}}{(1 + r)^{n}}
> $$
>
> 其中，V 为股权价值， $DPS_{t}$ 为第 t 期的现金红利，n 为详细预测期数，r 为折现率， $P_{n}$ 为持有期末卖出股权时的预期价格。
>
> 红利折现模型要求目标公司的红利发放政策相对稳定，但很多企业的红利发放政策不稳定或无红利可发放，此时难以用红利折现模型进行估值。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 茶庄停分两年、林素云描述眼下年分账水平 | 现金红利 DPS——持有期间每年能分到东家手里的那份钱 |
| 吴老板把两三年分账照街面息钱往回折 | 红利现值 $\frac{DPS_t}{(1+r)^t}$——未来分账按折现率压回当下 |
| 吴老板把"五年后卖铺子能得的数"也往回压 | 期末卖出价现值 $\frac{P_n}{(1+r)^n}$——持有期末转手那笔钱的现值 |
| 林素云把两笔数（分账 + 卖价）摆在同一张宣纸上 | V 的构成——两笔现值之和即股权价值 |
| 茶庄停分两年、分账政策不再稳定 | 模型的适用前提：红利发放政策相对稳定 |
| 郑掌柜不信"稳"，把未来分账估作零 | 模型失效：发放政策不稳定时，难以用此方法估值 |
| 吴老板默认"两三年内分账年年同、五年后茶市稳" | r、n 的隐含假设——详细预测期与折现率所依赖的稳定预期 |
