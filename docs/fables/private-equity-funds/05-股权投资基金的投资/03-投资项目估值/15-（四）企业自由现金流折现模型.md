# （四）企业自由现金流折现模型

## 📜 寓言故事 —— 《织坊家底》

清河镇老周商会的“江南织新”份子，账房先生阿福用“股东银子”法算锦绣织坊的股权价值时，赵掌柜又提了一个问题：“老周，股东银子算的是‘股东能攥到手的’，可锦绣织坊还欠钱庄三百两银子。钱庄也是织坊的‘出资人’之一——股东出钱、钱庄也出钱。能不能算一种‘织坊整体能攥到手的银子’，既包括股东的，也包括钱庄的？这样算出来的是‘织坊整体家底’，再扣掉欠钱庄的，就是股东的。”

老周点头：“赵掌柜说得对。有一种算法，叫‘企业自由现金流’——算的是织坊在正常运营情况下，能向所有出资人（股东和钱庄）自由分配的银子。它不区分股东和钱庄，只看织坊本身‘造血’能力有多强。”

阿福问：“怎么算？”老周说：“从‘息税前利润’开始——就是织坊赚的钱，在还没扣利息、还没缴税之前。然后一步步调整。”

**第一步：息税前利润减去税。** 锦绣织坊去年息税前利润三百六十两。按税率一成四（五十两税 ÷ 三百六十两息税前利润 ≈ 一成四），调整后税 = 三百六十两 × 一成四 = 五十两。三百六十两 - 五十两 = 三百一十两。这叫“税后息前利润”——扣了税、但还没扣利息。

**第二步：加回折旧和摊销。** 锦绣织坊去年折旧四十两、摊销零两。加回去：三百一十两 + 四十两 + 零两 = 三百五十两。折旧摊销不是真花出去的银子，只是账面上的数字。

**第三步：减去营运银子增加。** 锦绣织坊去年为了接新订单，多囤了五十两丝、多赊了三十两给客户——营运银子增加了八十两。扣掉：三百五十两 - 八十两 = 二百七十两。这是真花出去的银子。

**第四步：加上长期经营负债增加。** 供应商同意延期三个月付款——长期经营负债增加了二十两。加回来：二百七十两 + 二十两 = 二百九十两。这是暂时不用付的银子，相当于多攥了二十两。

**第五步：减去长期经营资产增加。** 锦绣织坊去年买了一台新织机，花了二百两——长期经营资产增加。扣掉：二百九十两 - 二百两 = 九十两。这是真花出去的银子。

**第六步：减去资本性支出。** 修厂房、换水渠，花了五十两——资本性支出。扣掉：九十两 - 五十两 = 四十两。

阿福瞪大眼睛：“周掌柜，企业自由现金流只有四十两？比股东银子还少？”老周说：“对。股东银子是五十两，企业自由现金流是四十两。差在哪里？股东银子加了‘新增付息债务一百两’、减了‘债务本金偿还八十两’——净增二十两。企业自由现金流不考虑债务的来来去去，所以少了二十两。企业自由现金流是‘织坊本身造血’，不考虑外部借钱还钱；股东银子是‘股东实际能攥的’，考虑了借钱还钱。”

赵掌柜问：“那怎么用企业自由现金流算织坊值多少？”老周说：“跟折现账一样——预测未来每年企业自由现金流，折现加总，再加终值。但有一个关键区别：企业自由现金流用‘加权平均资本成本’做折现率，而不是普通折现率。”

“什么叫加权平均资本成本？锦绣织坊的资本有两部分：股东出的钱和钱庄借的钱。假设股东要求回报率一成五（因为股东担风险大），钱庄借款利率一成（因为钱庄风险小、有抵押），税率一成四。锦绣织坊的资本结构：债务三百两，股权按实际价值算约二千一百两（总资本约二千四百两）。债务占比 = 三百 ÷ 二千四百 = 一成二五；股权占比 = 二千一百 ÷ 二千四百 = 八成七五。”

“加权平均资本成本 = 债务占比 × 债务成本 × (1 - 税率) + 股权占比 × 股权成本 = 一成二五 × 一成 × (1 - 一成四) + 八成七五 × 一成五 = 一成二五 × 一成 × 八成六 + 八成七五 × 一成五 = 约一成零七 + 一成三一三 = 约一成四二。”

老周让阿福做预测表：

| 年份 | 预测企业自由现金流 |
|------|------------------|
| 第1年 | 四十两（今年实际） |
| 第2年 | 一百两（新织机投产，产能提升） |
| 第3年 | 一百五十两（管理改善，成本下降） |
| 第4年 | 二百两（新客户增加，流水涨） |
| 第5年 | 二百五十两（市场稳定，品牌建立） |

“五年之后，假设企业自由现金流以每年一成增长，永续下去。加权平均资本成本一成四二，增长率一成。终值 = 第二百五十两 × 1.1 ÷ (0.142 - 0.10) = 二百五十两 × 1.1 ÷ 0.042 = 约六千五百四十八两。”

阿福折现：

| 年份 | 企业自由现金流 | 折现后价值 |
|------|--------------|-----------|
| 第1年 | 四十两 | 约三十五两 |
| 第2年 | 一百两 | 约七十七两 |
| 第3年 | 一百五十两 | 约一百两 |
| 第4年 | 二百两 | 约一百一十七两 |
| 第5年 | 二百五十两 | 约一百二十八两 |
| 终值 | 六千五百四十八两 | 约三千三百四十一两 |

加总：三十五 + 七十七 + 一百 + 一百一十七 + 一百二十八 + 三千三百四十一 = 三千七百九十八两。

老周说：“这就是锦绣织坊的‘企业价值’——包括股东和债权人的。扣掉净债务二百两（欠债三百两减现金一百两），股权价值 = 三千七百九十八两 - 二百两 = 三千五百九十八两。这比用股东银子折现算出来的三千一百九十两高。为什么？因为加权平均资本成本一成四二，比股东银子折现率一成五低——企业自由现金流的风险由股东和债权人共同分担，所以折现率更低，算出来的价值更高。”

赵掌柜问：“什么时候用企业自由现金流，什么时候用股东银子？”老周说：“若你想算‘织坊整体值多少’——比如要卖整个织坊、或者要算债权人和股东加起来值多少——用企业自由现金流。它不考虑资本结构，只看织坊本身造血能力。若你只关心‘股东那一份值多少’——用股东银子更直接。两种算法结果应该一致，只是路径不同：企业自由现金流 → 企业价值 → 扣债务 → 股权价值；股东银子 → 直接股权价值。若两种算出来差很远，说明资本结构假设有问题，得重新检查。”

老周收起算盘，说：“企业自由现金流是‘织坊的家底’——不看谁出的钱、谁借的债，只看织坊本身在正常运营下能攥多少银子。用这个折现，算出来的是‘整体价值’。整体价值里，扣掉债权人的，剩下的才是股东的。这叫‘先算整体，再分家’。”

**📖 原文定义**

> 企业自由现金流（Free Cash Flow to Firm，FCFF）是公司在保持正常运营的情况下可以向所有股东和债权人进行自由分配的现金流。计算公式为：
>
> 企业自由现金流（FCFF）=息税前利润（EBIT）-调整的所得税+折旧+摊销-营运资金的增加+长期经营性负债的增加-长期经营性资产的增加-资本性支出
>
> 其中，调整的所得税有两种计算方法：一是直接用息税前利润乘以当期所得税税率；二是在利润表中当期所得税的基础上进行调整，加回财务费用的税盾，再扣除非经常损益对应的所得税。
>
> 企业自由现金流折现模型的估值公式为：
>
> $$EV = \sum_{t=1}^{n} \frac{FCFF_t}{(1 + WACC)^t} + \frac{TV}{(1 + WACC)^n}$$
>
> 其中，EV为企业价值，FCFF_t为第t期的企业自由现金流，WACC为加权平均资本成本，TV为企业自由现金流的终值。
>
> 加权平均资本成本（WACC）是企业各种融资来源的资本成本的加权平均值，计算公式为：
>
> $$WACC = \left[ \frac{D}{D + E} \times k_d \times (1 - t) \right] + \left(\frac{E}{D + E} \times k_e\right)$$
>
> 其中，D为付息债务的市场价值，E为股权的市场价值，k_d为税前债务成本，k_e为股权资本成本，t为所得税税率。
>
> 企业自由现金流折现模型得到的是企业价值，可以通过价值等式推出股权价值。
>
> 接上例，基于某股权投资基金拟投资的D公司的息税前利润（EBIT）增长情况，计算企业自由现金流。假设条件如下：
>
> （1）随着公司的增长，预计未来5年D公司的息税前利润将分别为1 000万元、1 200万元、1 400万元、1 600万元和1 800万元；未来永续增长率假设为3%；
>
> 企业第1年自由现金流为 $1000 - 1000 \times 25\% + 300 - 400 - 100 = 550$ 万元。以此类推，计算得出D公司未来5年企业自由现金流（见表5-3）：
>
> 根据第5年的现金流及永续增长率，计算第5年后的终值为：
>
> $$TV = 1150 \times (1 + 3\%) / (10\% - 3\%) = 1184.5 / 7\% = 16921 (\text{万元})$$
>
> 现值总和= $\sum_{t=1}^{n}\frac{FCFF_t}{(1+WACC)^t}+\frac{TV}{(1+WACC)^n}$
>
> = 550 / (1 + 10\%) + 700 / (1 + 10\%) ^ {2} + 850 / (1 + 10\%) ^ {3} + 1000 / (1 + 10\%) ^ {4} + 1150 / (1 + 10\%) ^{5} + 16921 / (1 + 10\%) ^{5}
>
> = 13621（万元）
