# 2. 企业价值 / 息税前利润

## 📜 寓言故事 —— 《息前账》

清河镇老周商会的“江南织新”份子，账房先生阿福用“市盈率”算锦绣织坊的股权价值时，遇到了一个难题。赵掌柜说：“老周，彩云织坊和瑞祥织坊都是借了很多银子扩张的，锦绣织坊却几乎不借钱。用市盈率算，锦绣织坊的净利因为没有利息支出而显得高，可彩云织坊和瑞祥织坊的净利因为扣了利息而显得低。这好比一个自己走路的人和一个拄拐杖的人比速度——不公平。”

老周点头：“赵掌柜说得对。市盈率算的是‘净利润’，净利润是扣了利息之后的——借钱多的织坊，利息扣得多，净利润就少；借钱少的织坊，利息扣得少，净利润就多。可利息多少，跟织坊本身赚不赚钱没关系，跟掌柜愿不愿意借钱有关系。咱们比的是织坊‘本身’值多少，不是比掌柜‘借钱本事’大小。”

阿福问：“那怎么办？”老周说：“咱们换一种算法——不算‘净利润’，算‘息税前利润’。什么叫息税前利润？就是织坊赚的钱，在还没扣利息、还没缴税之前，总共赚了多少。这笔钱，既是股东的，也是债权人的——股东分红利，债权人收利息。所以，息税前利润对应的是‘织坊整体’值多少，不是‘股东那一份’值多少。”

阿福翻开锦绣织坊的账本：去年净利润三百两，缴了税五十两，付了利息十两（锦绣织坊几乎不借钱，利息很少）。息税前利润 = 三百两 + 五十两 + 十两 = 三百六十两。

彩云织坊：去年净利润四百两，缴了税八十两，付了利息一百两（彩云织坊借了很多银子扩产能）。息税前利润 = 四百两 + 八十两 + 一百两 = 五百八十两。

瑞祥织坊：去年净利润五百两，缴了税一百两，付了利息两百两（瑞祥织坊借钱最多）。息税前利润 = 五百两 + 一百两 + 两百两 = 八百两。

老周说：“看，锦绣织坊净利润三百两，彩云织坊四百两，瑞祥织坊五百两——锦绣织坊最差。可算息税前利润：锦绣织坊三百六十两，彩云织坊五百八十两，瑞祥织坊八百两——锦绣织坊还是最差，但差距没那么大了。因为瑞祥织坊和彩云织坊的‘净利润’被利息‘吃掉’了一大块，而锦绣织坊几乎没有利息支出。”

赵掌柜问：“那怎么用息税前利润算织坊值多少？”老周说：“市场上有一种‘倍数’，叫‘企业价值 ÷ 息税前利润’。咱们找可比织坊，算它们的这个倍数，然后套到锦绣织坊上。”

阿福去打听。彩云织坊：整体卖了四千两（企业价值），息税前利润五百八十两，倍数 = 四千两 ÷ 五百八十两 ≈ 六点九倍。瑞祥织坊：整体卖了五千两（企业价值），息税前利润八百两，倍数 = 五千两 ÷ 八百两 = 六点二五倍。两家平均约六点六倍。

老周说：“锦绣织坊息税前利润三百六十两，按市场平均六点六倍算，企业价值 = 三百六十两 × 六点六 ≈ 两千三百七十六两。这是‘织坊整体’值多少——包括股东的和债权人的。可咱们投的是股权，不是债权。所以，得把债权人的那一份扣掉。”

锦绣织坊欠钱庄三百两（债务），账上现金一百两。净债务 = 三百两 - 一百两 = 二百两。股权价值 = 企业价值 - 净债务 = 两千三百七十六两 - 二百两 = 两千一百七十六两。

赵掌柜听完，叹道：“老周，这息税前利润的算法，比市盈率公平多了。它不管掌柜借不借钱、借多少，只看织坊本身赚不赚钱。借钱多的织坊，利息扣得多，净利润少；借钱少的织坊，利息扣得少，净利润多。可息税前利润一视同仁——把利息加回去，大家都站在同一条起跑线上。”

老周点头：“对。市盈率算的是‘股东那一份’，适合比股东那一份；息税前利润算的是‘织坊整体’，适合比织坊整体。若两家织坊借钱多少差很多，用市盈率比就不公平，得用息税前利润。算出来的是‘整体值多少’，再扣掉债权人的，才是‘股东能分到的那一份’。”

---

**📖 原文定义**

> 息税前利润（EBIT）是在扣除债权人利息之前的利润，所有股东和债权人对息税前利润都享有分配权，因此，息税前利润对应的价值是企业价值。
>
> 息税前利润（EBIT）= 净利润 + 所得税 + 利息
>
> 企业价值/息税前利润乘数法的企业价值计算公式为：
>
> $$EV = EBIT \times (EV / EBIT \text{ 乘数})$$
>
> 股权投资基金计算目标公司的股权价值，则可以通过价值公式由企业价值得出股权价值。
>
> 值得注意的是，市盈率乘数法是以利润指标作为估值基础的，而净利润仅归属于股东，无法反映债权人的求偿权，不适用目标公司与可比公司的资本结构存在较大差异时的情形。而息税前利润是向所有股东和债权人分配前的利润，因此不受股权和债务的比例即资本结构的影响。企业价值/息税前利润乘数法剔除了资本结构的影响，适用于资本结构差异较大的公司。
