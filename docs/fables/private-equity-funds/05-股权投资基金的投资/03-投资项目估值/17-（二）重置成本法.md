# （二）重置成本法

## 📜 寓言故事 —— 《重估家底》

清河镇老周商会的“江南织新”份子，账房先生阿福用折现账算锦绣织坊的价值时，赵掌柜提了一个问题：“老周，折现账算的是‘将来能赚多少’，可万一锦绣织坊将来不赚钱呢？比如张锦绣病了、织机全坏了、客户跑光了——这时候它值多少？能不能算一种‘最坏情况’下的价值？”

老周点头：“赵掌柜说得对。折现账算的是‘未来价值’，可未来谁说得准？有一种算法，不问将来，只问‘现在’——若锦绣织坊今天散了、卖了、或者重新建一个一模一样的，值多少？这叫‘重估家底’法。”

阿福问：“怎么重估？”老周说：“分两步。第一步：算‘重置全价’——就是今天重新建一个和锦绣织坊一模一样的织坊，需要花多少银子。第二步：算‘综合贬值’——锦绣织坊的织机、厂房、存货，用了这么多年，折旧了多少、技术落后了多少。重置全价减去综合贬值，就是锦绣织坊今天‘按家底算’的价值。”

老周带着阿福、陈师傅、吴师傅，再次来到锦绣织坊，逐项盘点。

**第一项：织机。** 锦绣织坊有二十台织机。陈师傅说：“十八台是新水力织机，三年前买的，每台当时花了八十两。现在重新买一模一样的，每台要九十两——因为丝价涨了、工匠工钱涨了。重置全价 = 十八台 × 九十两 = 一千六百二十两。可这三年来，织机天天运转，有磨损、有老化。按设计使用年限十五年，已用三年，剩余十二年。综合成新率 = 十二年 ÷ 十五年 = 八成。所以织机价值 = 一千六百二十两 × 八成 = 一千二百九十六两。”

“另外两台是老木机，十年前买的，每台当时花了二十两。现在重新买一模一样的老木机，没人做了，得请工匠定制，每台要五十两。重置全价 = 两台 × 五十两 = 一百两。可老木机已用十年，设计使用年限十二年，剩余两年。综合成新率 = 两年 ÷ 十二年 ≈ 一成七。但老木机技术落后了——新水力织机一天能织十匹，老木机一天只能织三匹。技术贬值严重。陈师傅综合判断：综合成新率按五成算。老木机价值 = 一百两 × 五成 = 五十两。”

**第二项：厂房。** 锦绣织坊有一座厂房、三间仓库。吴师傅说：“厂房三年前盖的，当时花了五百两。现在重新盖一模一样的，材料、人工都涨了，要六百两。重置全价 = 六百两。厂房设计使用年限二十年，已用三年，剩余十七年。综合成新率 = 十七年 ÷ 二十年 = 八成五。厂房价值 = 六百两 × 八成五 = 五百一十两。”

“三间仓库，一间是三年前盖的，和厂房同时，重置全价二百两，综合成新率八成五，价值 = 二百两 × 八成五 = 一百七十两。另外两间是十年前盖的，当时每间一百两，现在重置每间一百二十两，重置全价 = 二百四十两。设计使用年限二十年，已用十年，剩余十年，综合成新率五成。但十年前盖的仓库，结构不如现在的新式仓库牢固，有轻微裂缝、漏雨。吴师傅综合判断：综合成新率按四成五算。价值 = 二百四十两 × 四成五 = 一百零八两。”

**第三项：存货。** 锦绣织坊有上等细布三百匹、中等粗布四百匹、次等布二百匹。阿福说：“存货按市价算。上等细布每匹三两，中等粗布每匹一两五，次等布每匹五钱。但次等布堆在角落发霉，得打折。存货价值 = 三百匹 × 三两 + 四百匹 × 一两五 + 二百匹 × 五钱 × 七成 = 九百两 + 六百两 + 七十两 = 一千五百七十两。”

**第四项：土地。** 锦绣织坊的地皮是宅基地，县衙批的。吴师傅说：“宅基地不能商用，若锦绣织坊散了，地皮只能按宅基地卖。邻县类似宅基地，每亩约五十两。锦绣织坊占地半亩，价值约二十五两。可若县衙批准变更为商用，价值能翻三倍。但变更手续复杂，不确定性大。保守按宅基地算：二十五两。”

阿福把各项加起来：织机一千二百九十六两 + 五十两 = 一千三百四十六两；厂房五百一十两；仓库一百七十两 + 一百零八两 = 二百七十八两；存货一千五百七十两；土地二十五两。合计 = 三千七百二十九两。

老周说：“这就是锦绣织坊按‘重估家底’算的价值——三千七百二十九两。比折现账算出来的企业价值三千七百九十八两略低，但差不多。为什么？因为折现账算的是‘未来能赚多少’，重估家底算的是‘今天卖了能换多少’。两种算法，一个看未来，一个看现在。”

赵掌柜问：“重估家底法有什么用？”老周说：“三种情形。第一：锦绣织坊将来不赚钱的假设成立时——比如行业衰退、技术淘汰——折现账算出来很低甚至负数，但重估家底算出来还有个保底数。第二：锦绣织坊的资产很实在——织机、厂房、土地——但经营能力很差，折现账低估了其资产价值。第三：作为辅助验证——折现账和重估家底差太远，说明假设有问题，得重新检查。”

老周顿了顿，又说：“但重估家底法有个大毛病——主观因素太多。综合成新率怎么定？陈师傅说八成，吴师傅可能说七成五。技术贬值怎么量化？老木机比新水力织机慢三倍，贬值多少？五成还是三成？不同人判断不同，结果差很远。而且，历史成本和未来价值没有必然联系——三年前买的织机八十两，现在重置九十两，可三年后新技术出来了，水力织机可能只值四十两。重估家底只算‘现在重置多少钱’，不算‘将来值多少钱’。所以，它只能做辅助，不能做主。”

赵掌柜点头：“老周，重估家底像‘数家当’——织机几台、厂房几间、存货多少，一项一项数，一项一项折旧。数完了，加总，就是‘保底价’。保底价不是‘真价值’，是‘万一将来不行了，卖了还能换多少’。有保底，心里踏实。”

老周收起算盘，说：“对。折现账算的是‘上限’——锦绣织坊经营得好，将来值多少。重估家底算的是‘下限’——锦绣织坊经营得差，今天卖了值多少。上限和下限之间，就是咱们出价的区间。出价低于下限，稳赚；出价高于上限，稳亏；出价在区间内，赌的是将来。”

**📖 原文定义**

> 重置成本法通过计算待评估资产的完全重置成本（即重置全价）减去其各种贬值后的差额来评估该项资产价值。重置全价是指在现时条件下重新购置一项全新状态的资产所需的全部成本。计算公式如下：
>
> 待评估资产价值 = 重置全价 - 综合贬值
>
> 或：
>
> 待评估资产价值 = 重置全价 × 综合成新率
>
> 综合贬值包括有形损耗（物质的）和无形损耗（技术的）等。对某项设备综合成新率的判定，可以根据设计使用年限与剩余使用年限的关系，或是根据剩余折旧年限与残值规定，结合其他因素进行判断。
>
> 重置成本法的主观因素较大，且历史成本与未来价值并无必然联系，因此，重置成本法主要作为一种辅助方法使用。
