# （一）现金流折现法

## 📜 寓言故事 —— 《折现账》

清河镇老周商会的“江南织新”份子，账房先生阿福用市场乘数法算锦绣织坊的价值时，总觉得心里不踏实。赵掌柜问：“老周，咱们找彩云织坊、瑞祥织坊做可比，可锦绣织坊和它们终究不一样——彩云织坊在邻县，瑞祥织坊在县城，市场、客户、成本都不同。用别人的倍数套锦绣织坊，像用别人的鞋量自己的脚，总有点歪。”

老周点头：“赵掌柜说得对。市场乘数法是‘看邻居的价’，可邻居和咱们终究不是一家。有一种算法，不找邻居，而是‘自己算自己的’——把锦绣织坊将来每年能攥到手的银子，一笔一笔折回今天，加总，就是它值多少。这叫‘折现账’。”

阿福问：“怎么折？”老周说：“打个比方。锦绣织坊今年能赚三百两银子，可这三百两是‘今年的三百两’。若你问我‘明年的三百两’值今天多少？我得算一笔账——若你把今天的三百两存到钱庄，钱庄每年给一成利，那明年的三百两，今天只值约二百七十三两（三百两 ÷ 1.1）。后年的三百两，今天只值约二百四十八两（三百两 ÷ 1.1 ÷ 1.1）。越往后，同样的银子，今天越不值钱。这个‘一成利’，就是‘折现率’——反映的是‘银子的时间成本’和‘风险’。”

赵掌柜问：“为什么越往后越不值钱？”老周说：“两个原因。第一，时间成本——今天的银子可以存起来生利，明年的银子不能。第二，风险——锦绣织坊明年能不能赚到三百两？不一定。可能赚四百两，也可能只赚二百两，甚至亏本。折现率越高，说明风险越大，将来的银子今天越不值钱。”

老周让阿福做了一张“预测表”——预测锦绣织坊未来五年的现金流：

| 年份 | 预测现金流 |
|------|----------|
| 第1年 | 二百两（今年净利三百两，但需留一百两修织机） |
| 第2年 | 二百五十两（新客户增加，流水涨） |
| 第3年 | 三百两（管理改善，成本下降） |
| 第4年 | 三百五十两（新织机投产，产能提升） |
| 第5年 | 四百两（市场稳定，品牌建立） |

老周说：“五年之后呢？锦绣织坊不会关门，它会一直运转下去。可咱们没法预测永远，所以五年之后，咱们算一个‘终值’——假设第五年之后，锦绣织坊每年现金流以一成增长，永远增长下去。终值公式：终值 = 第五年现金流 × (1 + 增长率) ÷ (折现率 - 增长率)。若折现率一成五，增长率一成，终值 = 四百两 × 1.1 ÷ (0.15 - 0.1) = 四百两 × 1.1 ÷ 0.05 = 八千八百两。”

阿福瞪大眼睛：“八千八百两？第五年之后值这么多？”老周说：“这是永续价值的魔力——假设永远增长，哪怕每年只增一成，累积起来也是天文数字。可这个假设很脆弱——若增长率降到半成，终值 = 四百两 × 1.05 ÷ (0.15 - 0.05) = 四千二百两，腰斩。若折现率升到两成，终值 = 四百两 × 1.1 ÷ (0.20 - 0.10) = 四千四百两，也腰斩。所以终值对折现率和增长率的假设极其敏感，差之毫厘，谬以千里。”

阿福接着算折现：

| 年份 | 现金流 | 折现率1.15 | 折现后价值 |
|------|--------|-----------|-----------|
| 第1年 | 二百两 | ÷ 1.15 | 约一百七十四两 |
| 第2年 | 二百五十两 | ÷ 1.15² | 约一百八十九两 |
| 第3年 | 三百两 | ÷ 1.15³ | 约一百九十七两 |
| 第4年 | 三百五十两 | ÷ 1.15⁴ | 约二百两 |
| 第5年 | 四百两 | ÷ 1.15⁵ | 约一百九十九两 |
| 终值 | 八千八百两 | ÷ 1.15⁵ | 约四千三百七十四两 |

加总：一百七十四 + 一百八十九 + 一百九十七 + 二百 + 一百九十九 + 四千三百七十四 = 五千三百三十三两。

老周说：“这是锦绣织坊的‘企业价值’——包括股东和债权人的。扣掉净债务二百两，股权价值 = 五千三百三十三两 - 二百两 = 五千一百三十三两。这比用市场乘数法算出来的二千九百两高多了。为什么？因为折现账算的是‘未来’——假设锦绣织坊管理改善、产能提升、品牌建立，将来会越赚越多。而市场乘数法算的是‘现在’——看锦绣织坊当下表现和邻居的倍数。两种算法，一个看未来，一个看现在。”

赵掌柜问：“哪种对？”老周说：“没有绝对的对错。折现账的优点是‘内生’——不依赖邻居，只看锦绣织坊自己。它逼着咱们深入分析锦绣织坊的财务数据、经营模式、核心驱动因素，找出价值从哪里来、怎么提升。可它的缺点是‘假设多’——五年后的现金流、增长率、折现率，全是猜的。猜错一个，结果差一半。市场乘数法简单、直观、有市场依据，可它假设‘锦绣织坊和邻居一样’，这往往不成立。”

老周顿了顿，又说：“实务中，咱们两种都算。折现账算一个‘内在价值’，市场乘数法算一个‘市场价值’，两者对照。若折现账远高于市场乘数法，说明锦绣织坊被低估了，或者咱们的假设太乐观。若折现账远低于市场乘数法，说明锦绣织坊被高估了，或者市场太乐观。两种算法的差距，就是‘不确定性’——咱们得反复检验假设，直到差距缩小到能接受的范围。”

阿福把折现账的每一步写在帖子上，老周审过后，对赵掌柜说：“折现账不是算命，是‘把未来的银子搬回今天’。搬的时候，得算好‘时间成本’和‘风险折扣’。搬完了，加总，才是锦绣织坊真正的‘家底’。”

**📖 原文定义**

> 股权投资基金计算目标公司的股权价值时，通常使用的收益法为现金流折现法。现金流折现法的基本原理是将估值时点之后目标公司的未来现金流以合适的折现率进行折现，加总得到相应的价值。评估所得的价值，可以是股权价值，也可以是企业价值。现金流折现法的计算公式为：
>
> $$V = \sum_{t=1}^{n} \frac{CF_t}{(1 + r)^t}$$
>
> 其中，V为价值，t为时期，CF_t为第t期的现金流，r为未来所有时期的平均折现率。
>
> 使用现金流折现法对目标公司进行价值评估时，一般需要预测目标公司未来每期现金流。由于目标公司是永续的，不可能预测到永远，实际估值中会设定一个详细预测期，通过对此期间目标公司的收入与成本、资产与负债等进行详细预测，得出每期现金流。对于在预测期之后目标公司的价值，也就是终值（TV），可以采用终值倍数法或永续增长模型进行估算。此时，现金流折现法的计算公式可以表示为：
>
> $$V = \sum_{t=1}^{n} \frac{CF_t}{(1 + r)^t} + \frac{TV}{(1 + r)^n}$$
>
> 其中，n为详细预测期数，TV为终值。
>
> 另一种计算终值的方法为永续增长模型。该方法假设目标公司在详细预测期之后，现金流以一个稳定的增长率永续增长，将终值期所有现金流折现到详细预测期最后一年并加总，即可得到终值价值。
>
> 按照永续增长模型，目标公司第n年的终值TV可以使用以下公式计算得到：
>
> $$TV = \frac{CF_n \times (1 + g)}{r - g}$$
>
> 其中，CF_n为第n年目标公司自由现金流，g为永续增长期内的增长率。
>
> （6）对详细预测期现金流及终值进行折现并加总得到价值。将上述步骤得到的数据代入下式计算得到价值：
>
> $$V = \sum_{t=1}^{n} \frac{CF_t}{(1 + r)^t} + \frac{TV}{(1 + r)^n}$$
>
> 如果上式得到的是企业价值而不是股权价值，需利用价值等式推出股权价值。
>
> 现金流折现法的优点包括：评估得到的是内含价值，受市场短期变化和非经济因素的影响较小；深入分析目标公司的财务数据和经营模式，有助于发现目标公司价值的核心驱动因素，从而为企业未来发展战略和经营决策提供依据，有助于发现提升企业价值的方法。
