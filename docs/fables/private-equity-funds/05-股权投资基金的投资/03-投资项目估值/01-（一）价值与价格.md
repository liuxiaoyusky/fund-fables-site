# （一）价值与价格

## ⚖️ 寓言故事 —— 《染坊的买卖》

清光绪十九年，江南乌镇东栅的染坊街上，老周和小徐都看上了一间铺子。这铺子打烊三年，掌柜病故，后人想转手。街坊都传，位置好、水路通、屋梁结实，开染坊的师傅都想要。里正老方出面，请两边的东家坐到一张桌前谈。

小徐三十出头，湖州人，前年刚在乌镇开过一间小染坊，今年生意一般，账上没多少闲钱。一落座他就急着压价："这铺子三年没开过，梁上还漏过雨，墙根返过潮。我看顶多值两百两。"老方点头说"小徐出两百两"。

老周却慢悠悠放下茶碗。他五十多岁，乌镇本地人，从十六岁就在这条街上做事。他不急还价，先问里正：铺子坐东朝西还是坐西朝东？青石板地还是泥地？屋梁是松木还是杉木？里正一一答了：坐西朝东，门前街宽三丈二分；青石板地，半天就干；屋梁是黄山松木，七八年前换过，虫没蛀过。

老周又问了第四个——这铺子修缮加固、开成中等规模染坊，一年稳稳能得多少进账？里正按街上几家老染坊的算法细说了一番。老周端起茶碗喝了一口，抬眼对小徐说："我出三百八十两。"

小徐愣住了："凭什么你出三百八十两，我才出两百两？"老周没急，慢慢把账摆出来。这铺子坐西朝东，光照最足，染出来的布色最稳；青石板地半天就干，布挂得开、收得快；松木梁能再用二十年。一年光布匹一项，进账就比同街别家多出一截。里正老方这几年帮三家铺子谈过买卖，按街上年年实打实的成交算，这铺子"摊到一年能得多少稳进账"再按行当里常用的法子折下来，三百八十两是实数。

小徐听完，嘴张了张没说话。半晌他才答："老周叔，不是我故意压价。年初我染坊进的颜料赔了一笔，手头真紧。眼前能拿出来的就是这个数。"老周点了点头，没再抬价。三百八十两落下槌来。

成交那天晚上，老周一个人坐在新铺子后院的井沿上，望着刚升起的月亮，把白天这事从头想了一遍。同一样东西，两个人坐到一张桌前谈，一个出二百两，一个出三百八十两，差出近一半。要是换一个买家，可能喊出四百二十两；再换一个，可能只肯出一百八十两。这铺子还是这铺子，梁还是那根松木梁，井还是那口井，一夜之间没多一根柱子也没少一块砖。可今晚落定的"三百八十两"，只是老周和小徐这一对买卖人坐到这张桌前时，各自愿出和愿进的数。

他再往下想：这铺子"摊到一年能得多少稳进账"，是看青石板、松木梁、坐西朝东这三样实在东西算出来的，不会因为今晚谁坐在这就变，也不会因为小徐手头紧就变，也不会因为老周多喝了一盅酒就变。这个数，是从长处看稳的那份数。价格落在桌上，每一桩买卖都不同；那份稳的数，铺子开了门就有，关了门也在。

---

**📖 原文定义**

> 价值是事物的内在属性，是从长期来看合理的、内在的价值。而价格是交易双方在某次交易中认可的外在表现形式。价格有时等于价值，但更多情况下，价格受到多种因素的影响，会偏离价值。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周问朝向、石板、梁材和一年稳进账 | 价值——从长期看由铺子本身属性决定的内在数 |
| 小徐出二百两、老周出三百八十两 | 价格——同一桩交易中双方各自认可的外在数 |
| 小徐因年初赔钱手头紧压价 | 价格偏离价值的因素之一：买家当下资金状况 |
| 老周按"行当常用算法"算出的三百八十两 | 由铺子本身属性决定的内在价值 |
| 老周夜里想"铺子还是这铺子，价格却每桩不同" | 价值稳定，价格随交易变动 |
| 价格有时正好落在价值上，更多时候偏离 | 价值与价格"有时等于，更多情况下偏离" |
