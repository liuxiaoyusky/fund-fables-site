# （一）参考最近融资价格法

## 📜 寓言故事 —— 《邻居的价》

清河镇老周商会的“江南织新”份子，摸底队查完了锦绣织坊，老周算总账时遇到了一个难题：锦绣织坊去年刚完成一轮“凑份子”——邻县一位绸缎商投了五百两银子，占了一成股份。这意味着，当时锦绣织坊的整体估值是五千两。可今年，老周商会要投锦绣织坊，该按五千两算，还是重新算？

赵掌柜说：“老周，去年有人投了五百两占一成，说明去年有人认锦绣织坊值五千两。咱们今年投，直接按五千两的九成算——四千五百两，不就行了？这叫‘参考邻居的价’。”

老周摇头：“赵掌柜，邻居的价能参考，但不能照搬。得先看这‘邻居’是谁、怎么买的、为什么买。”

**第一步：看邻居是谁**

老周派人去打听去年投锦绣织坊的那位绸缎商——姓王，人称王掌柜。打听回来，王掌柜是邻县最大的绸缎商，和锦绣织坊合作了十年，对张锦绣的手艺知根知底。去年投五百两，不是纯为了赚利，而是想绑住锦绣织坊的货源——投了银子，以后锦绣织坊的好布优先供给他。

老周说：“王掌柜投银子，有‘战略目的’——他不是单纯看锦绣织坊值多少钱，而是看‘绑住货源’值多少钱。他的五百两，可能包含了‘货源优先权’的溢价。咱们商会投银子，没有货源需求，所以王掌柜的价，对咱们不一定公允。”

**第二步：看怎么买的**

老周又查：王掌柜的五百两，是“新发份子”——锦绣织坊新印了一成股份，卖给王掌柜。可与此同时，张锦绣的弟弟张锦华，把自己手里的一成股份，以四百两的价格卖给了一个外乡人。新发份子五百两，老股转让四百两——差了整整一百两。

老周说：“这里有两个价。新发份子五百两，是锦绣织坊‘新印’的股份，王掌柜买了之后，有完整的权利——分红、投票、查账。老股转让四百两，是张锦华把自己原有的股份卖给外乡人，外乡人买的时候，可能权利受限——比如不能投票、或者三年内不能转让。两个价不一样，因为权利不一样。咱们参考哪个？”

赵掌柜说：“咱们投的是新发份子，应该参考王掌柜的五百两。”老周点头：“对。但还得看——这五百两是不是‘有序交易’。”

**第三步：看是不是有序交易**

老周继续查：王掌柜投五百两的时候，锦绣织坊的经营状况如何？去年锦绣织坊接了一个大订单，赚了八百两，是近三年最好的一年。王掌柜投的时候，正是锦绣织坊最风光的时候。可今年，那个大订单没了，瑞祥绸缎庄的订单又砍了一半，锦绣织坊的净利跌到了三百两。

老周说：“王掌柜投的时候，锦绣织坊在‘山顶’；今年咱们投的时候，锦绣织坊在‘半山腰’。山顶的价，不能照搬给半山腰。而且，王掌柜投的时候，是锦绣织坊主动找的他——张锦绣想扩产能，急需银子。这算不算‘有序交易’？算，但交易时点特殊——是锦绣织坊‘最好的时候’，不是‘平常的时候’。”

**第四步：看时间隔了多久**

老周算了一下：王掌柜投银子是去年三月，现在是今年九月，隔了一年半。老周说：“邻居的价，隔得越久，参考意义越弱。一年半前，锦绣织坊的织机还是新的、客户还是稳定的、掌柜还是精力旺盛的。现在呢？织机旧了、客户跑了、张锦绣五十六了。时过境迁，邻居的价只能‘参考’，不能‘照搬’。”

**第五步：看业务指标变了多少**

老周让阿福列了一张表，对比去年三月和今年九月锦绣织坊的关键指标：

- 去年净利八百两，今年三百两——跌了六成二。
- 去年客户十二个，今年七个——少了近一半。
- 去年织机全速运转，今年两台坏了、三台闲置——产能利用率跌了四成。
- 去年存货周转一年四次，今年两次半——慢了将近一半。
- 去年应收账款三百两，今年五百两——多了六成七，回款更慢。

老周指着表说：“这些指标，是锦绣织坊‘值多少钱’的核心。王掌柜投的时候，这些指标都在高位；现在都在低位。若按王掌柜的价五千两算，今年这些指标跌了这么多，价也该跌。可跌多少？不是拍脑袋，得看这些指标和‘价’的关系。”

老周想了想，说：“咱们商会投锦绣织坊，不是为了‘参考邻居的价’，而是为了自己算清楚。邻居的价是一个‘锚’——去年有人认五千两，说明去年锦绣织坊至少值这个数。可今年，锚还在，船已经漂远了。咱们得根据今年的指标，重新调整。比如净利跌了六成二，若价和净利成正比，那今年的价大约是五千两 × 三成八 = 一千九百两。可实际不能这么简单算——因为客户少了、产能利用率跌了、回款慢了，这些不是简单的数字，是‘锦绣织坊还能不能活下去’的信号。”

赵掌柜听完，叹道：“老周，参考邻居的价，原来不是‘照搬’，而是‘借锚定船’。锚是去年的价，船是今年的锦绣织坊。锚在原地，船已经漂了。咱们得看船漂了多远、往哪个方向漂，才能定今年的价。”

老周点头：“对。参考最近融资价格法，最适用于那些还没稳定赚钱、但经常有人投银子的初创织坊——比如去年有人投、今年又有人投，每次投的价就是‘锚’。可锚多了，得挑最靠谱的锚——看是谁投的、怎么投的、是不是有序交易、隔了多久、指标变了多少。挑对了锚，才能定准船的位置。”

**📖 原文定义**

> 参考最近融资价格法是基于非上市目标公司最近一次融资的价格，作为对其股权进行估值的参考。该方法主要适用于尚未产生稳定收入或利润，但融资活动比较频繁的初创企业。在运用该方法时，应当判断最近融资价格的公允性。如果没有主要的新投资人参与最近融资，或最近融资金额对被投资企业而言并不重大，或最近交易被认为是非有序交易（如被迫出售股权或对被投资企业陷入危机后的拯救性投资），则该融资价格一般不作为被投资企业公允价值的最佳估计使用。
>
> 在进行公允性评估后，可根据被投资企业的主要业务指标的变化（如财务指标、技术发展阶段、市场份额等）对最近融资价格进行调整，以得出当前公司价值。值得注意的是，估值日距离融资完成的时间越久，最近融资价格的参考意义越弱。此外，在选择主要业务指标时，应重点考虑被投资企业所处行业特点及其自身的特点，选择最能反映被投资企业价值变化的业务指标。
