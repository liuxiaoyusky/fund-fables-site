# 5. 市净率

## ⚖️ 寓言故事 —— 《钱庄的账》

民国二十四年开春，上海汉口路上的"德和钱庄"要换东家。挂牌东家老魏年过六十，膝下无子，想把整间钱庄连铺面、连银库、连老存户一起转出去。钱庄开了十八年，账目干净，是上海滩少有的"实仓"。

想买的人里最惹眼的是两位：宁波来的"恒生号"少东家阿昌，三十出头，刚从英国念完商账回来；老魏多年的老主顾、本埠做丝绸生意的陈老板。

掮客老方头一个把阿昌请来谈。阿昌开口就问："您家一年进账多少？利钱多少？放出去的账收得回几成？"老魏一一答了。阿昌算到傍晚，皱眉说："按进账算，这价撑不过三万两。"老方又请来陈老板。陈老板一不看成色、二不数存银，只让老魏把柜上"总账簿"摊在桌上。账簿里一笔笔记着钱庄十八年来房子值多少、库存现银多少、放出账余额多少、柜上家具字画算几件。合上最后一行——房子、库存、放账、器具，加在一起两万两。老魏点头："柜上的实账，就这两万两。"

陈老板看了一眼钱庄前门——三开间门面、青砖外墙、楼下铁皮箱、一百二十多家老存户——心里另起一笔账：这套家当摆在汉口路十八年没挪过窝；管账的师傅老赵跟了老魏二十多年账上一笔不差；前头那四家洋行都把钱存在这；这"牌子"本身就是钱。陈老板抬头说："我出四万两。"

阿昌急了："陈老板，您家一年利钱再高，也撑不到四万两。"老方也犯难——按利钱算三万两到顶，按柜上家当算两万两打底；陈老板这四万两从哪来？正争不下，老魏把柜上老赵叫出来。

老赵捧出一叠"同业单子"——上海钱业公会每月汇齐的几家老钱庄"账面与售价"对照单。"城隍庙'永和'账面家当一万八，最后成交三万六；城北'同丰'账面两万五，最后成交四万七。'最后成交'与'账面家当'两行的比值，几年都稳在一点八到一点九之间。"老赵把账簿翻回陈老板眼前算："德和柜上实账两万两，按同业这条线来量——两万两乘一点八，是三万六；乘一点九，是三万八。陈老板出四万两，比这数略高，是看中十八年攒下的老存户、柜上老赵这块'活招牌'，再加他自个丝绸生意借钱的方便——这些不写在账上，行里都知道值点。这四万两，是从柜上那本账的两万两里推出来的，没脱开那个底。"

阿昌盯着同业单子看了半天，抬起头来。他那张英国学来的"按一年利钱算"的法子，在钱庄这一行里算不准——钱庄利钱薄、坏账藏着、账面看不见；钱庄值多少钱，关键看柜上那本"实账"折成多少现银，再按同业里大家都在用的那条线乘一下。柜上那本账的两万两，是这间钱庄的"根"；同业里一点八到一点九那条线，是钱庄这一行里大家伙儿都认的"尺"。陈老板用这两样东西撑出四万两，根在这儿，尺在那儿，撑出来的数就稳。

掮客老方当场落槌。德和钱庄连铺面带家当，归了陈老板。

**📖 原文定义**

> 市净率（P/B）反映公司的股权价值相对其净资产的乘数，其计算公式为：
>
> 市净率=股权价值÷净资产
>
> 或：
>
> 市净率=每股价值÷每股净资产
>
> 市净率乘数法估值与市盈率乘数法类似，先确定可比公司的市净率作为目标公司估值的市净率乘数，然后使用下述公式计算目标公司股权价值或每股价值：
>
> 股权价值 $=$ 净资产 $\times$ 市净率乘数
>
> 或：
>
> 每股价值 $=$ 每股净资产 $\times$ 市净率乘数
>
> 市净率乘数法比较适用于资产流动性较高的金融机构，因为这类企业的净资产账面价值更加接近市场价值。例如，银行业的估值通常会用市净率乘数法。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老魏摊开"总账簿"算得柜上家当两万两 | 净资产——钱庄账面上房子、银库、放账余额、器具等家当合计 |
| 城隍庙"永和"、城北"同丰"两家老钱庄的对照单 | 可比公司——同业里其他老钱庄 |
| 同业里"账面"与"成交"两行的比值落在一点八到一点九 | 市净率乘数——可比公司的市净率 |
| 老赵用两万两乘一点八到一点九算得三万六到三万八 | 股权价值 = 净资产 × 市净率乘数 |
| 陈老板出四万两（高于同业乘数结果） | 在乘数结果上叠加"老存户、老赵招牌、自身融资便利"等行里都认的溢价 |
| 阿昌用"按一年利钱算"的法子算不准 | 钱庄利钱薄、坏账藏——市盈率法对金融机构失真 |
| 德和钱庄十八年"实仓"、账目干净、存户稳当 | 净资产账面价值接近市场价值的资产——市净率法适用前提 |
| 掮客老方用同业这条线把"账面家当"折成"成交价" | 市净率乘数法的核心——用净资产乘以同业乘数得到股权价值 |
