# （三）股权自由现金流折现模型

## 📜 寓言故事 —— 《股东银子》

清河镇老周商会的“江南织新”份子，账房先生阿福用“企业自由现金流”算锦绣织坊的企业价值时，赵掌柜提了一个问题：“老周，企业自由现金流算的是‘织坊整体’能攥多少银子——包括给债权人的利息和还本。可咱们商会投银子，只关心‘股东能分到多少’。能不能直接算‘股东能攥的银子’，不用先算整体再扣债务？”

老周点头：“赵掌柜问到了点子上。有一种算法，直接算‘股东自由现金流’——就是织坊赚的钱，扣掉所有必须花的、必须还的之后，能自由分给股东的最多银子。这叫‘股东银子’法。”

阿福翻开锦绣织坊的账本，老周一条条教他算：

“第一步：从净利开始。锦绣织坊去年净利三百两——这是股东账面上的利润。”

“第二步：加回折旧和摊销。织机折旧四十两、专利摊销零两——这些不是真花出去的银子，是账面上的数字。加回去，因为折旧摊销不影响实际现金流。”

“第三步：减去营运银子增加。去年锦绣织坊为了接新订单，多囤了五十两丝、多赊了三十两给客户——营运银子增加了八十两。这八十两是真花出去的银子，得扣掉。”

“第四步：加上长期经营负债增加。锦绣织坊去年跟供应商签了长期协议，供应商同意延期三个月付款——长期经营负债增加了二十两。这二十两是‘欠着不用马上还’，相当于多攥了二十两银子。”

“第五步：减去长期经营资产增加。锦绣织坊去年买了一台新织机，花了二百两——长期经营资产增加了二百两。这二百两是真花出去的银子，得扣掉。”

“第六步：减去资本性支出。除了买织机，锦绣织坊还修了厂房、换了水渠，总共花了五十两——资本性支出。也得扣掉。”

“第七步：加上新增付息债务。锦绣织坊去年从钱庄新借了一百两——新增付息债务。这一百两是股东可以用的银子，加回来。”

“第八步：减去债务本金偿还。锦绣织坊去年还了钱庄八十两本金——债务本金偿还。这八十两是必须从股东现金流里扣掉的，因为借了要还。”

阿福按老周的指点，一笔一笔算：

股东银子 = 净利三百两 + 折旧四十两 + 摊销零两 - 营运银子增加八十两 + 长期经营负债增加二十两 - 长期经营资产增加二百两 - 资本性支出五十两 + 新增付息债务一百两 - 债务本金偿还八十两

= 三百 + 四十 + 零 - 八十 + 二十 - 二百 - 五十 + 一百 - 八十 = 五十两

阿福瞪大眼睛：“周掌柜，锦绣织坊账面上净利三百两，可股东实际能自由分的银子只有五十两？差这么多？”

老周说：“对。净利是账面上的，股东银子是实际能攥到手的。差在哪里？第一，买织机、修厂房花了二百五十两——资本性支出和长期经营资产增加，这些是为了将来赚钱，但今年真花出去了。第二，多囤丝、多赊给客户花了八十两——营运银子增加，也是为了将来接单，但今年真垫出去了。第三，还了钱庄八十两本金——借了要还。第四，新借了一百两——这是新增来源，加回来。第五，折旧四十两不是真花的，加回来。第六，供应商延期二十两——暂时不用付，加回来。七加八减，最后只剩五十两。”

赵掌柜问：“那怎么用这个股东银子算锦绣织坊值多少？”老周说：“跟企业自由现金流折现一样——预测未来每年股东银子，折现加总，再加终值。但这里有个关键区别：企业自由现金流折现，算出来的是‘企业价值’，还要扣掉净债务才是股权价值。股东银子折现，直接算出来就是‘股权价值’——因为股东银子已经扣掉了所有债务相关的影响（利息、新增借款、本金偿还），只剩股东能分的。”

老周让阿福做了一张预测表：

| 年份 | 预测股东银子 |
|------|------------|
| 第1年 | 五十两（今年实际） |
| 第2年 | 一百两（新织机投产，产能提升，净利涨） |
| 第3年 | 一百五十两（管理改善，成本下降） |
| 第4年 | 二百两（新客户增加，流水涨） |
| 第5年 | 二百五十两（市场稳定，品牌建立） |

“五年之后，假设股东银子以每年一成增长，永续下去。折现率一成五。终值 = 第二百五十两 × 1.1 ÷ (0.15 - 0.10) = 二百五十两 × 1.1 ÷ 0.05 = 五千五百两。”

阿福折现：

| 年份 | 股东银子 | 折现后价值 |
|------|---------|-----------|
| 第1年 | 五十两 | 约四十三两 |
| 第2年 | 一百两 | 约七十六两 |
| 第3年 | 一百五十两 | 约九十九两 |
| 第4年 | 二百两 | 约一百一十四两 |
| 第5年 | 二百五十两 | 约一百二十四两 |
| 终值 | 五千五百两 | 约二千七百三十四两 |

加总：四十三 + 七十六 + 九十九 + 一百一十四 + 一百二十四 + 二千七百三十四 = 三千一百九十两

老周说：“这就是锦绣织坊的‘股权价值’——直接算出来的，不用扣债务。比用企业自由现金流折现算出来的企业价值五千三百三十三两，扣掉净债务二百两，得股权价值五千一百三十三两，低了不少。为什么？因为股东银子已经考虑了债务的影响——锦绣织坊借钱多、还本多，股东实际能分的就少。企业自由现金流不考虑债务，算出来的是‘整体’价值；股东银子考虑债务，算出来的是‘股东那一份’的真实价值。”

赵掌柜问：“什么时候用企业自由现金流，什么时候用股东银子？”老周说：“若你想算‘织坊整体值多少’——比如要卖整个织坊、或者要算债权人和股东加起来值多少——用企业自由现金流。若你只关心‘股东那一份值多少’——比如咱们商会投银子只当股东、不当债权人——用股东银子更直接。两种算法结果应该一致，只是路径不同：企业自由现金流 → 企业价值 → 扣债务 → 股权价值；股东银子 → 直接股权价值。若两种算出来差很远，说明债务假设有问题，得重新检查。”

老周收起算盘，说：“股东银子是‘股东能攥到手的最大银子’。它从净利出发，把‘不是真花的’加回来，把‘真花出去的’扣掉，把‘债务的来来去去’算清楚。最后剩下的，才是股东真正能分的。用这个折现，直接就是股权价值——不用绕弯子。”

**📖 原文定义**

> 股权自由现金流（Free Cash Flow to Equity，FCFE）是可以自由分配给股东的最大化的现金流。计算公式为：
>
> 股权自由现金流（FCFE）=净利润（E）+折旧+摊销-营运资金的增加+长期经营性负债的增加-长期经营性资产的增加-资本性支出+新增付息债务-债务本金的偿还
>
> 股权自由现金流折现模型的估值公式为：
>
> $$V = \sum_{t=1}^{n} \frac{FCFE_t}{(1 + r)^t} + \frac{TV}{(1 + r)^n}$$
>
> 其中，V为股权价值，FCFE_t为第t期的股权自由现金流，r为折现率，TV为股权自由现金流的终值。
>
> 某股权投资基金拟投资的D公司是一家技术驱动型的初创企业，公司正处于成长阶段，预计未来5年将会有显著的收入增长，因而带动净利润的增长。假设条件如下：
>
> 基于上述假设，计算得出D公司第一年股权自由现金流为：500-400+300-100=300（万元）。以此类推，计算得出D公司未来五年股权自由现金流（见表5-2）：
>
> 根据第5年的现金流及永续增长率，计算第5年后的终值为：
>
> $$TV = \frac{CF_n \times (1 + g)}{r - g} = 700 \times (1 + 3\%) / (12\% - 3\%) = 721 / 9\% = 8011 (\text{万元})$$
>
> 现值总和：
>
> $$V = \sum_{t=1}^{n} \frac{FCFE_t}{(1 + r)^t} + \frac{TV}{(1 + r)^n}$$
>
> = 6267（万元）
>
> 因此，D公司的股权价值为6267万元。
