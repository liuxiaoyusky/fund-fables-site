# （二）企业价值与股权价值

## 🔍 寓言故事 —— 《算总账》

清河镇老周商会的“江南织新”份子，摸底队查完了锦绣织坊，阿福写了一份八十页的总帖。老周拿着帖子，对赵掌柜说：“帖子写完了，可还有一个关键问题没解决——锦绣织坊到底值多少银子？咱们该出多少价？”

赵掌柜说：“张锦绣自己说值三千两。可咱们查出来，实际只值两千二百两。这差八百两，怎么算出来的？”老周说：“这就是‘算总账’的学问。算总账不是数织机、数存货那么简单，得分清楚——这家织坊‘整体’值多少，和‘咱们能分到的那一份’值多少，是两回事。”

老周摊开一张纸，画了一个图。

“张锦绣的锦绣织坊，账面上写着：织机二十台、厂房一座、仓库三间、存货八百匹布、应收账款五百两、现金一百两。负债呢？欠钱庄三百两、欠供应商二百两。咱们先算‘整体值多少’——这叫‘织坊总价值’。”

“织坊总价值 = 所有能赚钱的东西 + 能自由花的银子。能赚钱的东西：织机、厂房、仓库、存货、应收账款——这些加起来，按咱们摸底后的实际估价，约两千八百两。能自由花的银子：账上现金一百两。所以织坊总价值 = 两千八百两 + 一百两 = 两千九百两。”

赵掌柜问：“那欠钱庄和供应商的五百两呢？”老周说：“欠债是别人的，不是织坊的。织坊总价值里，不包含欠债——因为欠债是要还给别人、不能算进织坊自己的。但欠债会影响‘咱们能分到的那一份’。”

老周继续画：“现在算‘咱们能分到的那一份’——这叫‘股权价值’。股权价值 = 织坊总价值 - 欠债。两千九百两 - 五百两 = 两千四百两。这就是锦绣织坊的全部股权价值。张锦绣现在一个人持百分之百，所以他的股权值两千四百两。咱们若投银子进去，占三成，那咱们的那一份值两千四百两 × 三成 = 七百二十两。”

赵掌柜点头：“明白了。织坊总价值两千九百两，是‘整体’；股权价值两千四百两，是‘张锦绣能卖的那一份’。中间差的五百两，是欠债。欠债越多，股权价值越少。”

老周又说：“可还有更复杂的情况。锦绣织坊账上，除了织坊本身，还有一样东西——张锦绣去年在省城买了一间铺面，花了二百两，租给别人开茶馆，每月收租五两。这间铺面，锦绣织坊的织机用不上、织工也用不上，跟织布的生意完全没关系。这叫‘非核心资产’。”

“非核心资产怎么处理？有两种算法。第一种：把它算进织坊总价值里。织坊总价值 = 织布的核心资产两千八百两 + 铺面二百两 + 现金一百两 = 三千一百两。股权价值 = 三千一百两 - 五百两欠债 = 两千六百两。第二种：把它从织坊总价值里剔出去。织坊总价值只算核心资产 = 两千八百两 + 现金一百两 = 两千九百两。股权价值 = 两千九百两 - 五百两 = 两千四百两。那铺面二百两，单独算——张锦绣若卖织坊，铺面可以单独卖，也可以不卖。”

赵掌柜问：“哪种算法对？”老周说：“看咱们投的是什么。若咱们投的是‘锦绣织坊’这个织布生意，铺面跟织布无关，应该剔出去。若咱们投的是‘张锦绣的全部家当’，铺面也算进去。通常，咱们投的是织布生意，所以用第二种算法——织坊总价值只算核心资产，非核心资产单独列。”

老周在纸上写下：

织坊总价值（核心资产） + 现金 + 非核心资产 = 欠债 + 股权价值

两千八百两 + 一百两 + 二百两 = 五百两 + 两千六百两

“这是‘一般算总账’的公式。左边是织坊拥有的：核心资产、能花的银子、非核心资产。右边是织坊欠的：别人的债、股东自己的股权。两边必须相等。”

赵掌柜又问：“若锦绣织坊还有一家子公司呢？比如它在邻县有一家小染坊，占七成股份，另外三成是别人的。”老周说：“那就更复杂了。织坊总价值里，要包含那家小染坊的七成价值。股权价值里，也要包含那七成。但另外三成，是‘别人的股权’，不能算进锦绣织坊的股权价值里。这叫‘少数股东权益’——得单独列出来。”

老周在纸上又加了一项：

织坊总价值（核心资产） + 现金 + 非核心资产 = 欠债 + 别人的股权（少数股东权益） + 咱们能分到的股权价值

赵掌柜看完，叹道：“老周，这算总账的学问，比织布还复杂。织坊总价值、股权价值、欠债、非核心资产、少数股东权益——五个数，互相牵扯，算错一个，全错。”

老周收起纸，说：“所以咱们投银子之前，必须先把这五个数算清楚。织坊总价值是‘整体’，股权价值是‘咱们能买到的那一份’。整体大，不一定能买到的那一份大——若欠债多、非核心资产多、别人的股权多，能买到的那一份就小。算总账，算的不是织坊有多大，而是咱们能分到多少。”

**📖 原文定义**

> 企业价值（EV）是指企业股东和债权人共同拥有的企业运营所产生的价值，包括企业的股东所拥有的股权价值以及企业的债权人所拥有的债权价值。股权投资基金关心的是股权价值，最终交易价格以股权价值为基础确定。但某些估值方法使用企业价值作为基数，这时需要运用价值等式，在企业价值和股权价值之间进行转换。
>
> 1. 简单价值等式
>
> 简单价值等式为:
>
> 企业价值+现金=股权价值+债务
>
> 其中，债务与现金的差额定义为净债务，等式可表示为：
>
> 企业价值 = 股权价值 + 净债务
>
> 需要注意的是，等式中的各项值都是市场价值。“债务”与资产负债表中的负债不同，是指要支付利息的负债（如银行贷款），而不包括无须支付利息的负债（如应付账款）。“现金”指企业持有的货币及现金等价物，即企业可自由支配的流动资金。
>
> 2. 一般价值等式
>
> 对拥有非核心资产的公司，并不适用简单价值等式。在实际估值中，很多公司都有非核心资产，需要对企业价值进行更为科学的定义：企业价值是指公司核心资产运营所产生的价值。核心资产对应的是主营业务，主营业务的价值即体现为企业价值；非核心资产对应非主营业务，如交易性金融资产或投资性房地产，其价值不应包含在企业价值当中。
>
> 核心资产与非核心资产的划分并不是绝对的，需依据具体公司情况具体分析。例如，制造业公司购买了房地产若不用于生产经营而仅作为投资，则该房地产属于非核心资产；但对房地产公司而言，则属于核心资产。
>
> 合并范围内的子公司中不属于母公司的权益部分在母公司的合并资产负债表中列为少数股东权益。股权价值为少数股东权益加上归属于母公司股东的股权价值。当股权投资基金的投资标的是母公司股权，应关注归属于母公司股东的股权价值。
>
> 综上，一般价值等式为：
>
> 企业价值+非核心资产价值+现金=债务+少数股东权益+归属于母公司股东的股权价值
