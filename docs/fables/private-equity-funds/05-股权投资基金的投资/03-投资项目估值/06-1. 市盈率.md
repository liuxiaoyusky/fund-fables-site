# 1. 市盈率

## 📜 寓言故事 —— 《倍数账》

清河镇老周商会的“江南织新”份子，摸底队查完了锦绣织坊，老周算总账时，遇到了一个关键问题：锦绣织坊到底值多少银子？

阿福说：“按账本，锦绣织坊去年净利三百两。若咱们按这个净利算，投三千两，占三成，那锦绣织坊整体值一万两。可去年净利三百两，前年是五百两，大前年是八百两——一年不如一年。按哪年算？”

老周说：“这就是‘倍数账’的学问。咱们不直接算‘值多少’，而是算‘净利乘以多少倍’。这个倍数，叫‘市盈率’——就是市场上别人买织坊时，愿意为每一两净利出多少两银子。”

赵掌柜问：“倍数从哪儿来？”老周说：“从‘可比织坊’来。咱们找三家和锦绣织坊规模相近、行当相同的织坊，看它们最近卖股份时，买家出了多少倍。”

老周派阿福去打听。三天后，阿福带回消息：

- 邻县“彩云织坊”：去年净利四百两，上个月一位绸缎商买了它一成股份，出了四百两。倍数 = 四百两 ÷ 四百两 = 一倍。可彩云织坊的掌柜年轻、织机新、客户稳定，买家愿意出一倍。
- 县城“瑞祥织坊”：去年净利五百两，去年一位外乡人买了它两成股份，出了一千两。倍数 = 一千两 ÷ 五百两 = 二倍。可瑞祥织坊有独家花样专利，买家愿意出二倍。
- 镇上“金线织坊”：去年净利二百两，前年一位布商买了它三成股份，出了六百两。倍数 = 六百两 ÷ 二百两 = 三倍。可金线织坊虽然净利少，但增长快，前年净利只有一百两，去年翻了一倍，买家赌的是未来。

老周听完，说：“三家织坊，倍数从一倍到三倍，差得远。为什么？因为彩云织坊稳但增长慢，瑞祥织坊有专利溢价，金线织坊增长快但基数小。锦绣织坊像哪一家？——像彩云织坊：稳但增长慢，净利逐年下滑。所以，锦绣织坊的倍数，应该接近彩云织坊的一倍，而不是金线织坊的三倍。”

赵掌柜问：“那咱们按一倍算？锦绣织坊去年净利三百两，一倍就是三百两。整体值三百两？那咱们投三千两，占三成，岂不是锦绣织坊整体值一万两？这差太远了。”

老周摇头：“赵掌柜，你混淆了。‘倍数’是‘股权价值 ÷ 净利’。彩云织坊去年净利四百两，买家出一成股份四百两，意味着彩云织坊整体股权价值 = 四百两 × 十倍 = 四千两。倍数是十倍，不是一倍。我算错了。”

阿福在一旁说：“周掌柜，我再算一遍。彩云织坊：买家出一成股份四百两，意味着整体股权价值 = 四百两 ÷ 一成 = 四千两。倍数 = 四千两 ÷ 四百两净利 = 十倍。瑞祥织坊：买家出两成股份一千两，整体股权价值 = 一千两 ÷ 两成 = 五千两。倍数 = 五千两 ÷ 五百两净利 = 十倍。金线织坊：买家出三成股份六百两，整体股权价值 = 六百两 ÷ 三成 = 两千两。倍数 = 两千两 ÷ 二百两净利 = 十倍。”

老周一愣：“三家都是十倍？那倍数不是由织坊本身决定的，而是由市场决定的——现在市场上买织坊，普遍愿意出净利的十倍。”

赵掌柜问：“那锦绣织坊也按十倍算？去年净利三百两，整体股权价值 = 三百两 × 十倍 = 三千两。咱们投三千两，占三成，那锦绣织坊投后整体值一万两？不对，咱们投三千两占三成，意味着投前锦绣织坊值七千两，投后值一万两。可去年净利只有三百两，十倍才三千两。这账对不上。”

老周笑了：“赵掌柜，你掉进坑里了。‘市盈率’算的是‘股权价值’，不是‘投后整体价值’。若锦绣织坊去年净利三百两，按市场十倍市盈率，它的股权价值 = 三百两 × 十倍 = 三千两。这是‘投前’的价值——不包含咱们要投的三千两。咱们投三千两后，锦绣织坊的整体价值 = 投前三千两 + 咱们投的三千两 = 六千两。咱们占三千两 ÷ 六千两 = 五成。可咱们只想占三成，这意味着咱们投的银子应该少于三千两。”

阿福在旁边飞速算：“若锦绣织坊投前股权价值三千两，咱们投后占三成，那投后整体价值 = 三千两 ÷ 七成 = 约四千二百八十六两。咱们投的银子 = 四千二百八十六两 - 三千两 = 一千二百八十六两。咱们出一千二百八十六两，占三成，张锦绣占七成。”

老周点头：“对。市盈率算的是‘投前股权价值’——基于锦绣织坊现有的净利。咱们投的银子，是‘新增价值’，要加进去。最终咱们占多少，取决于咱们出多少、投后整体值多少。”

可老周又皱起眉头：“但这里有个问题。锦绣织坊去年净利三百两，可前年是五百两，大前年是八百两。净利逐年下滑。若按去年三百两算，十倍市盈率，股权价值三千两。可若明年净利跌到二百两呢？股权价值就只有二千两了。咱们投一千二百八十六两，明年可能只值八百两。这风险太大。”

赵掌柜说：“那怎么办？按平均净利算？”老周说：“对。锦绣织坊近三年净利平均 = （八百 + 五百 + 三百）÷ 三年 = 五百三十三两。按十倍市盈率，股权价值 = 五百三十三两 × 十倍 = 五千三百三十两。可去年只有三百两，按平均算高估了；按去年算，又低估了未来的可能。这得看——锦绣织坊的净利下滑是暂时的，还是长期的？”

老周想了想说：“锦绣织坊净利下滑，是因为客户流失、产品老化、创新能力弱。这些是结构性问题，不是暂时的。所以，按去年净利三百两算，更保守、更安全。但十倍市盈率是‘市场平均’，锦绣织坊比平均差，倍数应该打折——比如七倍。股权价值 = 三百两 × 七倍 = 二千一百两。咱们投后占三成，投后整体值 = 二千一百两 ÷ 七成 = 三千两。咱们投的银子 = 三千两 - 二千一百两 = 九百两。咱们出九百两，占三成，张锦绣占七成。”

赵掌柜听完，叹道：“老周，这倍数账，算的是‘市场愿意出多少倍’，可最终决定倍数的，是织坊本身的好坏。好织坊，倍数高；差织坊，倍数低。倍数不是死的，是活的——由织坊的净利质量、增长前景、风险大小决定。”

老周收起算盘，说：“对。市盈率是‘市场的尺子’，可尺子的刻度，是织坊自己刻的。咱们用这把尺子量锦绣织坊，量出来的不是‘绝对值’，而是‘相对值’——相对市场上其他织坊，锦绣织坊值多少。量完了，还得打折扣——因为锦绣织坊比平均差。打完折扣，才是咱们愿意出的价。”

**📖 原文定义**

> 市盈率（P/E）反映公司的股权价值相对其净利润的乘数，其计算公式为：
>
> 市盈率 = 股权价值 ÷ 净利润
>
> 或：
>
> 市盈率 = 每股价值 ÷ 每股收益
>
> 在使用市盈率乘数法估值时，先确定可比公司的市盈率作为目标公司估值的市盈率乘数，然后使用下述公式计算目标公司股权价值或每股价值：
>
> 股权价值 = 净利润 × 市盈率乘数
>
> 或：
>
> 每股价值 = 每股收益 × 市盈率乘数
>
> 需要注意的是，市盈率中的盈利指标应为归属于母公司的净利润，而非全部净利润。
>
> 某股权投资基金拟投资A公司，A公司的管理层承诺投资当年的净利润为1.2亿元，按投资当年净利润的10倍市盈率进行投后估值，则A公司的估值计算如下：
>
> A 公司投资后价值 = 1.2 × 10 = 12（亿元）
>
> 如果A公司拟融资3亿元，则新进投资方所占股权比例 = 3/12 × 100% = 25%
>
> 另外，企业的净利润和市盈率均受经济周期影响较大，两种因素叠加会导致周期型企业的估值呈现大起大落的特征。股权投资基金应以平均市盈率为参考标准，而不是简单地跟随市场潮流。
