# 3. 估计股权投资基金在退出时的要求持股比例

## ⚖️ 寓言故事 —— 《五十两的份儿》

清河镇东街有个卖酱菜的老周，做人最公道，镇上谁家分家、合伙、退伙，都爱请他当中间人。最近他接了一桩新差事——给布商老邱和南山种金丝楠木苗的小子阿樵，划一划"份儿"。

老邱揣着五十两银子上了南山，在阿樵那间破屋里坐下。他把昨晚在孙伯那面铜镜里算出来的账摊开：十年后这满山金丝楠木成材，能值一千两；按他想赚的倍数倒推回来，今天这事儿只值约一百两。

阿樵听得眼睛发亮："那您投五十两，我给您一半？"老邱摆摆手："你急什么。我还没说我该占多少份儿呢。"

他把账本翻到新的一页，写下两行字：上头一行是"今天这事儿值多少"，下头一行是"我出多少"。然后他拿起笔，在两行字之间画了一道横线——上头那一百两是分母，下头那五十两是分子。

老邱给阿樵比划："你想——今天这事儿值一百两，我出五十两。这五十两是那一百两的一半。换句话说，今天这事儿切成一百份儿，我出五十份儿，就拿走一半。"他顿了顿，又补一句，"十年后那事儿值一千两的时候，这一百份儿也值一千两。我拿走一半，就是五百两。十年翻十倍——这事儿划得来。"

阿樵张大了嘴："那要是十年后只值五百两呢？"老邱说："那我就只拿二百五。"阿樵又问："那要是只值二百两呢？"老邱把笔搁下："那我就只拿一百两——还是当初我出的那一倍。不赚，可也不亏。"

他拍拍阿樵的肩："你看，这一笔账里我得守住一条线——我出多少，就拿回多少的份儿。这一条线，今儿写下来，十年后谁也别想赖。"

阿樵点点头，转身去拿笔墨，要把这条线白纸黑字写下来。老邱摆手："不急。你先拿这张纸，回去好好看看——你那一百份儿里头，给我切五十份儿，剩下五十份儿归你。这五十份儿里头，等你明年还要拉人、再要钱的时候，还得再细切——可无论怎么切，我这五十份儿是你最先答应下的，谁也别想从我手里头抹掉。"

阿樵接过那张纸，看着"一百份儿"和"五十份儿"那两行字，忽然明白了一件事——原来"份儿"不是嘴上说说，是照着今天这事儿值多少钱、自己出多少钱，一点一点切出来的。切多切少，全看分子和分母那条线。

老邱临走时回头看了一眼那张纸，笑了笑："份儿这东西，硬得很——出多少钱，就占多少份儿；今天值多少，就切多少份儿。这两条线拉直了，十年后才不会扯皮。"

---

**📖 原文定义**

> 投资额除以当前股权价值，得到股权投资基金为获得目标回报倍数或收益率应有的持股比例。计算公式如下：
>
> 要求持股比例 = 投资额 ÷ 当前股权价值

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 今天这事儿值一百两（分母） | 当前股权价值 |
| 老邱出五十两（分子） | 投资额 |
| 老邱画的那条横线："分子 ÷ 分母 = 份儿" | 投资额 ÷ 当前股权价值 = 要求持股比例 |
| 五十两 ÷ 一百两 → 拿走一半份儿 | 计算结果：要求持股比例为 50% |
| 十年后值一千两时拿五百两 / 五百两时拿二百五 / 二百两时拿一百两 | 持股比例固定，分享的是公司整体价值 |
| "出多少钱，就占多少份儿；今天值多少，就切多少份儿" | 公式的本质：份儿由今天值多少、出多少共同决定 |
