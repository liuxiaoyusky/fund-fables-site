# 六、股权投资基金的估值应用

## 🌱 寓言故事 —— 《三种土壤与两兄弟》


青山镇上有三片田地，分别住着三户人家。东头的老张家，刚开垦了一片荒地，只撒下种子，尚未见青苗。中村的老李家，麦苗已齐腰，但还需不断施肥浇水，才能盼来丰收。西尾的老王家，良田百亩，年年稳产，谷仓总是满的。

镇上来了一位远道而来的年轻人，名叫林远。他背着行囊，走遍了这三片田地，却用三种截然不同的眼光打量它们。

到了老张家的荒地，林远没有蹲下来数土里的种子，而是盯着老张父子看了一整天。他看老张的儿子阿诚，虽面黄肌瘦，却能在烈日下连续劳作八个时辰不歇；他看老张珍藏的一把铁犁，虽锈迹斑斑，形制却与邻县最先进的耕具如出一辙；他更看父子俩在油灯下比划的一张草图——那上面画的，是一种从未有人种过的作物。林远心中暗忖：这地眼下寸草不生，但只要人对了、器对了、法对了，三年之后或许就是一片金田。他掏出一袋银子，不是买地，而是押在了"人"这个字上。

离开东头，林远来到老李家。这里的麦子长势喜人，比镇上任何一家的都高出一截。可林远注意到，老李的牛棚里新添了三头壮牛，水渠也刚拓宽了半里，谷仓倒是空的——去年的收成全换成了这些家当。老李愁眉苦脸地说："今年怕是要借粮度日了。"林远却笑了。他走遍了方圆十里的田地，发现老李种的这种麦子，正是时下酒坊最急需的原料。他不再看老李家里有多少存粮，而是去酒坊打听这种麦子的市价，去邻县询问同样的田地能换多少银子。他在心里盘算的不是眼前这茬麦子值多少，而是这片地在一个"正在醒来的行当"里，占据着什么样的位置。他掏出了第二袋银子，比给老张的还多。

最后，林远来到老王的百亩良田。金黄的麦浪随风起伏，谷仓里堆着三年的陈粮，水车咿呀转动，一切都按部就班。林远没有问老王明年种什么新花样，而是坐下来，请老王翻开近十年的账本。他一笔一笔地算：每年收多少、花多少、能剩下多少。他算的是一条细水长流的河，而不是一场突如其来的雨。他算清了这条河的深浅，才掏出第三袋银子——这一袋最重，因为他知道，这片地不会给他惊喜，但也不会让他挨饿。

与此同时，青山镇还住着一对兄弟，哥哥叫赵铁，弟弟叫赵石。他们听说了林远的事，却嗤笑不已。

"那林远不过是小打小闹，"赵铁拍着胸脯说，"真正做大事的人，要像咱们这样。"

兄弟二人看中的是老王的百亩良田。他们自有本钱不多，却向镇上的钱庄借了一大笔银子，以老王的谷仓和水车作保，硬是把这片地买了下来。他们的算盘很简单：老王每年剩下的粮食，正好够还钱庄的利息；等债还清了，这百亩地就彻底姓赵了。他们不需要这片土地长出新奇的作物，不需要它突然丰收——他们只需要它像过去十年一样，稳稳地产出、稳稳地剩下。

三年过去，老张的荒地里果然长出了那种新奇的作物，远近闻名，林远的那袋银子翻了几番。老李的麦子成了酒坊的命脉，他的地价比当初涨了十倍。老王的地照旧每年产出，林远收着稳当的回报，不急不躁。

而赵家兄弟呢？他们守着那片良田，年年还着钱庄的债。有人问他们为何不也去看看荒地和新苗，赵铁只是摇头："我们的路不同。我们要的不是破土而出的惊喜，而是春种秋收的踏实。"

青山镇的人渐渐明白了：世间没有一种看地的眼光是万能的。有人看人看种看未来，有人看市看行看位置，有人看账看流看长远。而那一对兄弟，他们走另一条路——借别人的力，买稳当的田，收确定的粮。每一条路，都只适合一种土壤。

**📖 原文定义**

> 创业投资基金在评估目标公司时，会根据公司所处的发展阶段采取不同的估值方法。每个阶段的企业都有其独特的特点和估值需求，需要运用不同的分析工具和方法来准确评估其价值。
>
> 1. 初创期企业的估值：初创期企业通常处于概念验证或早期产品开发阶段，其收入少，现金流量和利润皆为负。这类企业通常面临高风险和高度不确定性，但也有可能带来高回报。创业投资基金在评估初创期企业时，主要从团队能力、技术潜力和商业模式来判断企业的投资价值，估值主要采用风险资本估值法。
>
> 2. 成长期企业的估值：成长期企业的特点是收入快速增长，但因大规模的资本性支出导致负现金流量。这类企业通常已经经过产品验证阶段，正在扩大市场份额和运营规模。创业投资基金在评估成长期企业时，主要从所处行业特有的价值驱动因素来判断企业的投资价值，估值主要采用市场法。
>
> 3. 成熟期企业的估值：成熟期企业的特点是收入和现金流量稳定增长，业务模式和市场地位已经相对稳定。这类企业通常在运营上更加稳健，风险较低，但增长速度也相对较慢。创业投资基金在评估成熟期企业时，可选择从现金流量来判断企业的投资价值，估值可适当采用收益法。
>
> 并购基金通常采用杠杆收购的方式进行投资。杠杆收购是一种利用高比例债务融资进行收购的策略，通过以目标公司的现金流和资产作为还款保障，使得股东可以用较少的自有资本实现较高的投资回报。因此，目标公司通常是成熟行业中具有稳定增长现金流量的公司，资本性支出较低，经营杠杆较低，可以偿付高额的并购债务利息。这类公司的稳健财务状况和稳定的现金流，使其适合作为杠杆收购的对象。并购基金对目标公司的估值通常采用基于现金流折现法的杠杆收购法。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老张家荒地（种子未发芽） | 初创期企业——收入少、现金流为负，估值看人看技术看模式 |
| 林远看老张父子、铁犁、草图 | 风险资本估值法——评估团队能力、技术潜力、商业模式 |
| 老李家麦苗（长势好但投入大、存粮空） | 成长期企业——收入快速增长但资本性支出大、现金流为负 |
| 林远打听酒坊市价、邻县地价 | 市场法——参考行业价值驱动因素、可比市场数据 |
| 老王百亩良田（稳产、有陈粮、账本清晰） | 成熟期企业——收入现金流稳定、业务模式成熟 |
| 林远翻十年账本算细账 | 收益法——基于现金流评估企业价值 |
| 赵家兄弟借钱买良田、以谷仓水车作保 | 杠杆收购（LBO）——以目标公司现金流和资产为还款保障 |
| 赵家只要稳稳产出、不求惊喜 | 并购基金偏好——成熟行业、稳定现金流、低资本支出、低经营杠杆 |
| 三种田地三种眼光 | 不同阶段企业适用不同估值方法 |
| 两兄弟走另一条路 | 并购基金与创业投资基金策略本质不同 |
