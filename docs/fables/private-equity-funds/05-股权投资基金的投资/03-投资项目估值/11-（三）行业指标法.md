# （三）行业指标法

## 🔍 寓言故事 —— 《米行的尺》

民国二十六年的秋天，无锡北门外新开了一家米行，掌柜姓沈，单名一个"信"字。沈信四十出头，做粮行二十多年。北门外米行少说十来家，沈信心里最记挂的，是隔壁那条街上的"聚成米行"——同行里头数它最大，账目最干净，沈信私下总拿它当镜子。

开业第一个月结束，沈信坐在后堂对账，越对心里越没底。本月共卖出去一千二百石米，刨去进货、运费、伙计工钱和店租，账面进账是多少？沈信提笔算了好几遍，怎么也没法和"聚成"比。"聚成"是开了十几年的老号，店面是自己的，没店租这一项；伙计按股分红，没有固定工钱；进货走运河大船，进价压得比沈信低一成。明面上看，沈信这月赚的数差出一截。

他连着三晚没睡好，第四天一早，叫上小伙计阿宝，去了"聚成"米行。聚成的老掌柜姓钱，叫钱守诚，六十开外，花白胡子，把沈信迎到后堂喝茶。沈信问："钱掌柜，我这号新开，第一月下来账面进账比您差出一截，我比照着您这行看自己，心里不踏实。"

钱守诚没接话，先给沈信续了茶，慢慢说道："沈老弟，你拿'聚成'这把尺子来量自己，量不准。"他从柜里翻出无锡米业公会的旧册子，摊在桌上说："你看，北门外这十来家米行，家家都和'聚成'比：比卖了多少石，比进账多少银两。可这十来家，卖的米有苏北粳米、本地晚稻、洋米，店面有自有有租赁，伙计有工钱有分红，挑夫费用差得远。一把'聚成'的尺子，哪里量得遍这十几家？"

钱守诚又指了指公会那本旧册子："同一行里比，得用同一把尺。你是苏北粳米、租店面、月结工钱、运河船进货，那就在公会册子里挑和你家情况差不多的几家米行，搬他们家今年来算账的几个常用数字出来——像'每月一石米摊多少进价''每两本钱一个月转几次'。你拿同行的小尺子，量到自己头上，量出来才是真数。"沈信听着，眉头渐渐松开。

沈信回店那晚，按"每石米摊多少进价""每两本钱一月转几次""店面租金占几成"这三条行当常用的算账法子，把这一个月重新排过。排到第三天晚上，他发现那笔差出来的进账，主要卡在店面租和进货船钱上。找到这个结，沈信立刻让阿宝去船行讲价，又和房东重新谈了租约。半月下来，账面便一天天好转。

三个月后的一个黄昏，沈信坐在后堂，面前摊着自家账本和公会旧册子。他不再拿"聚成"那把大尺子量自己了。他知道，米行这一行里头，要看得真切，得用行当里大家都在用的那几条小尺子——每月每石米摊多少、每两本钱转几次、店面租金占几成——拿这几把同行的小尺子量自家，量出来的才是真账。真要盘自家米行，还得回到自己一年的进出账，一笔一笔算实。沈信提起笔，在账本封面上写下四个字："同行校我。"

---

**📖 原文定义**

> 行业指标法是用与目标公司公允价值直接相关的特定行业指标作为估值参考依据。例如，对互联网公司估值时，会用企业价值/用户数等乘数。行业指标法适用于行业发展成熟且行业内企业差别较小的情况，一般用于检验其他估值方法的合理性，而不作为主要估值方法单独运用。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 沈信拿"聚成"的整本账来量自家 | 误把单一公司当行业指标 |
| 钱守诚指出米行各家情况差别大 | "行业内企业差别较小"这一适用前提 |
| 公会册子里挑情况相近的几家米行 | "特定行业指标"的来源——同行普遍使用的指标 |
| 沈信回店时按行当常用算法重排自家账 | 把行业指标作为参考依据落到自家 |
| 拿同行小尺子量出自家租金和船钱偏贵 | 行业指标法"检验其他方法合理性"的作用 |
| 沈信最终仍按自家一年进出账算实 | 行业指标法"不作为主要方法单独运用" |
