# 二、初步尽职调查

## 🔍 寓言故事 —— 《相人四问》

民国二十六年的夏天，浙东鹿鸣镇的茶商沈亦舟听说南边云和县有座三百亩的老茶园要出手，连着厂房、商标、老客户一起卖，开价三万大洋。这消息是县商会的老乡递过来的，递的时候还压低声音说："东家急着用钱还债。"

沈亦舟做茶十年，本钱还差一截。身边几个老相识都劝他赶紧下手："先把定金交了，把人留住，钱凑齐再去细看。"沈亦舟笑笑没吭声，当晚一个人坐在院里的石桌旁，铺开宣纸，提笔写下四个问号。

第二天一早，他没带钱，只带了本子和钢笔，搭骡车往云和去。镇上另外两位闻讯赶来的茶商——一个姓吴，一个姓钱——已经在茶园里转了一圈出来，正凑在茶棚里催沈亦舟："别磨叽了，我们准备签意向书了。"沈亦舟摆摆手："两位哥哥先别急，我这就进去转一圈。"

这一转就转了两天。第一天，他没看茶树，先去镇上把茶馆、茶栈、茶贩子挨个请来喝茶。他不问茶价高低，专问三件事：这东家是哪年接的盘？这些年卖出去的茶，多是散茶还是贴了牌子的？镇上从前有过几家茶园，如今还剩几家？问完出来，他在本子上写下四个字——"看人看势"。

第二天，他跟着东家李守仁一整天坐在厂里，听掌柜、账房、领班各自说话。中午吃饭时，他独自把镇上两个老茶农请到一棵大樟树下喝米酒。老茶农喝了半碗才开口："李老板人倒不坏，可他三个儿子没一个肯回来接这活儿，厂里管事的是他外甥，进出货都是这外甥一手经手。"沈亦舟又问："这三百亩的茶园，邻近几个县里还有几家这般大的？"老茶农伸出三根指头："云和、松阳、遂昌，三家，都说自己是正源。"

傍晚回到客栈，沈亦舟翻开本子，在头天晚上写的四个问号旁，一笔一笔地填上答案：第一笔，三个儿子都不愿回乡；第二笔，邻近三县各家茶园的大致情形；第三笔，"管事外甥"那几个字的来路；第四笔，他写不下去了，把笔搁下，望着窗外发了一会儿呆。

第二天清早，那姓吴的带着定金和一份事先印好的意向书要进茶园签。沈亦舟在门口拦下他，把本子递过去："吴兄，不是拦你发财，只是这桩买卖值不值三万大洋，我心里有四个问号，一个也答不出。"他指着自己写的几行字说："兄弟我这一趟不是去把脉开方，也不是去算命占卜。我只想用两天工夫，先掂一掂这茶园的分量。掂完了，能不能挣、能不能撑得起、能不能长久，才见分晓。"

姓吴的接过本子看了半晌，没说话，把定金和意向书揣回兜里。当天下午，三人一同搭车回乡，谁也没签那份意向书。

回镇那天夜里，沈亦舟在院里把那张宣纸取出来，在四个问号下面端端正正写了一行字——"两日功夫，看人看势，不下重手。"他把本子收进书箱，吹灭了灯。

---

**📖 原文定义**

> 初步尽职调查的核心是对目标公司进行价值判断，并非详尽调查或投资风险评估。
>
> 初步尽职调查阶段主要从以下方面对目标公司进行初步价值判断：管理团队，行业进入壁垒、行业集中度、市场占有率和主要竞争对手，商业模式、发展及盈利预期、政策与监管环境等。
>
> 初步尽职调查前，基金管理人（或股权投资基金）通常与目标公司签订保密协议（Non-Disclosure Agreement，NDA），双方承诺对在投融资业务沟通过程中接触到的对方商业秘密负有一定期限的保密义务。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 沈亦舟没带钱，只带本子和钢笔上云和 | 初步阶段核心是看，不是签 |
| 沈亦舟在宣纸上写下四个问号 | 初步价值判断聚焦少数关键问题 |
| 沈亦舟先在镇上请茶馆、茶栈、茶贩子喝茶 | 围绕行业进入壁垒、市场占有率和主要竞争对手做外围调查 |
| 一整天坐着听掌柜、账房、领班各自说话 | 对管理团队做基本判断 |
| 三个儿子都不愿回乡、邻近三县各有一家大茶园 | 接班人问题、地区行业集中度与主要竞争对手 |
| 老茶农点出"管事外甥"一手经手进出货 | 商业模式与盈利预期的初步线索 |
| 沈亦舟自言"不是去把脉开方，也不是算命占卜，只想掂一掂分量" | 初步尽职调查是"初步价值判断"，不是详尽调查或风险评估 |
| 两天后拦住姓吴的不签意向书 | 在价值判断没完成前，不推进到下一阶段 |
