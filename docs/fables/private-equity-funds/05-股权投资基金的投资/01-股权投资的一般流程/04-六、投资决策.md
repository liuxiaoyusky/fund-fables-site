# 六、投资决策

## ⚖️ 寓言故事 —— 《七人议事厅》

清光绪十二年腊月，天津卫老字号"义兴成"的东家贺敬亭病倒在床，临终前把九家分号的掌柜召到总号后院。九家分号九本账，贺家总账上压着二十万两白银的活钱，按祖训，这些钱要替族里办三件大事：修运河码头、设义仓、开新染坊。东家把这二十万两的最终去向，全交给了一个新设的"七人议事厅"——这是他头一回在族里立的规矩。

议事厅的七个人里，三位是跟了他二十年的老账房，三位是分号里最能干的掌柜，第七位是从上海洋行请回来的周慕桥。这周慕桥既不是贺家人，也不在九家分号任职，贺东家临终前特意吩咐长子贺伯谦："慕桥先生的眼睛是借来的，不带分号的私心。"

东家咽气第七天，贺伯谦把七人请进议事厅。桌上摊着两摞文册：一摞是九家分号递上来的用钱请求，一摞是请洋行先生做的"通盘账"。周慕桥先把那本通盘账翻开，指着末页："三件事加在一起二十六万两，账上只有二十万。码头若先动，义仓只够撑半年，新染坊再等一年。新染坊若先动，码头修到一半就得停工。这一笔，族议里没写。"

分号掌柜吴季同迟疑了一下："周先生，族议上头写得清清楚楚，是不是先依族议？码头一停，沿河八千亩地的粮运就断了。"

周慕桥没接他的话，把第一摞文册翻开，抽出一封夹在分号请求里的私信——是写给贺伯谦的。他把那封信推到桌面中央："这封信，按规矩，本来不该出现在议事厅里。我请大少爷先收回去。"贺伯谦愣了片刻，伸手把信收回袖中。三位老账房低头没说话。吴季同的手攥得指节发白。

屋里安静了一刻。老账房贺敬堂先开口："周先生说得对，这封信不到桌上来，才是贺家的体面。我赞成先开新染坊——回头钱养义仓，义仓有了底，码头后年再修也来得及。"吴季同涨红了脸："贺叔，码头一停——"贺敬堂抬手打断他："吴掌柜，今儿说的是二十万两怎么分，不是八千亩地怎么救。八千亩地的事，是县衙的差，不是贺家的账。"

七人依次开口。末了举手：四票开新染坊，两票修码头，一票弃权。贺伯谦抬眼看向周慕桥："先生，您那一票呢？"周慕桥把茶盏搁下："我的那一票，是请大少爷记一笔——今儿议事厅里撤下去的那封信。往后凡有私信夹在请求里递进来的，递信的人与拆信的掌柜，同罪出族。贺家二十万两银子的事，从今儿起，只在这张桌上定。这件事，比开哪间染坊更值钱。"

贺伯谦怔了半晌，提笔在那本族议册子的首页空白处，端端正正写了一行字："七人议事厅，凡议大事，依账、依理、不依人情。"

---

**📖 原文定义**

> 基金管理人可为基金独立设置投委会，也可在基金管理人内部设立集中投决机制。投委会委员由具备丰富投资管理经验与能力、有足够时间与精力履行相应职责的投资管理专业人士担任，通常由基金管理人的高级管理团队成员担任，有时也会聘请外部专家。
>
> 尽职调查完成后，投资管理小组向投委会提交、汇报尽职调查报告、投资建议书和其他文件资料，由投委会进行最终投资决策。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 七人议事厅是新设的最终拍板机构 | 独立设置的投委会 / 集中投决机制 |
| 周慕桥从洋行请来、不属于任何分号 | 投委会聘请外部专家，保持独立判断 |
| 贺伯谦、长子、各分号掌柜不能替议事厅做主 | 投委会成员由具备专业能力的人士担任，独立议事 |
| 三摞文册、九家分号请求、通盘账、族议 | 投资管理小组向投委会提交、汇报的"尽职调查报告、投资建议书和其他文件资料" |
| 七人举手、四票开染坊、两票修码头、一票弃权 | 投委会进行最终投资决策的集体议事 |
| 把夹带私信的请求当场撤下 | 议事规则排除人情干扰，确保独立判断 |
| 议事厅立下"依账、依理、不依人情"的规矩 | 投决机制独立于内部人情与外部压力 |
