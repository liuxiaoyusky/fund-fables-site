# 一、项目开发与筛选

## 🔍 寓言故事 —— 《满桌的请柬》

民国二十六年开春，杭州布商周鹤年在涌金门外自家老宅摆了一桌春酒。请的是生意场上的旧相识——做丝的张叔、做茶的吴二爷、洋行买办林叔。酒过一巡，周鹤年叹了口气："诸位老哥，我改行了，把手里布匹生意交给内人打理，自己做'替人挑买卖'的营生。"

张叔问："绸缎做得好好的，怎么忽然改行？"周鹤年从袖子里摸出一叠红封请柬，"啪"地摊在八仙桌上："缘故都在这儿。我从去年入冬到如今积了六十二张请柬——商会的、同乡会的、银行茶会的、皮革行会的、丝绸公会的。每张都说'某某某有一桩买卖要做，缺个识货的东家'。可这六十二张里头，能让我动心思的，至多四五张。"

吴二爷问："老周，你这是要做中人？还是要做东家？"周鹤年摆摆手："都不是。我做的是'挑买卖'——哪桩值当投点钱进去、哪桩不值当，这门判断就是我新买卖的本钱。"他随手把请柬分了三堆："头三十七张，是酒楼茶肆上的闲话——我连看都不看。再十九张，是熟人递上来的条子，写得花团锦簇，可连'这桩买卖卖的是什么'都说不清。剩六张——商会里打过照面的、银行掌柜私下透露的、老相识写了三页信纸细说缘由的——就这六张，我要亲自跑一趟。"

林叔插话道："老周，你这新买卖，第一步是'听风'，第二步是'筛沙'，第三步才是'看货'。可你做了十年绸缎，对茶、对银行都不内行，那六桩里若有一半是别的行当，你怎么筛？"

周鹤年从墙上取下一卷挂轴展开——是一张他自己画的表格，横着"行业、地域、阶段"三栏，竖着六十二张请柬的编号。他指着表格说："凡不在这三样里的，请柬写得再恳切，我也不接。我熟悉的是江南织造、印染；至于北边的皮货、上海的洋行买卖，不懂装懂去看也是白看。"顿了顿，他把那六张请柬翻开——湖州某织坊、绍兴某染坊、嘉兴某缫丝厂——"都是江南织造这一行的、本地人或邻近县份的、做到一定规模想往上走的。"

吴二爷笑道："老周这新买卖，做起来却是个苦差事——光'听风'要满杭州城转一圈，'筛沙'更要把六十二桩筛成两三桩。"周鹤年端起酒盏回敬："我从前做布匹是'挑布'，如今改行是'挑买卖'。挑布只看一桩事——这匹布织得好不好；挑买卖却要先把满城的请柬听一遍、筛一遍、再亲自跑一遍。听得多，筛得细，才能挑得准。"

春酒喝到月上柳梢才散。六十二张请柬被周鹤年收进一只樟木箱，搁在书房东墙根下，等着开春后一场一场地跑、一桩一桩地筛。

**📖 原文定义**

> 项目开发主要解决股权投资基金的项目来源问题。项目来源渠道多种多样，主要包括以下几方面：（1）行业研究：跟踪和研究国内外新技术的发展趋势以及资本市场的动态，通过资料调研、项目库推荐、访问企业等方式寻找项目信息；（2）中介机构推荐：包括财务顾问、律师、会计师、证券公司、商业银行等；（3）天使投资人或者同行推荐；（4）企业家联盟及各级商会组织推荐；（5）行业专家推荐；（6）政府机构推荐；（7）行业展会、创业计划大赛、创投论坛等。
>
> 股权投资基金管理人获得项目后，根据其在目标公司所处行业、地域、发展阶段等方面的投资偏好对项目进行筛选，选择符合其投资偏好的项目进行初步尽职调查。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 周鹤年摊开的六十二张请柬，来自商会、同乡会、银行茶会、各行业公会 | 项目来源渠道多种多样 |
| 第一堆三十七张闲话与第二堆十九张花团锦簇却说不清买卖本身的条子 | 项目开发要"解决来源问题"，来源鱼龙混杂，需先归类再筛选 |
| 周鹤年画的"行业、地域、阶段"三栏表格 | 基金管理人对项目的投资偏好——目标公司所处行业、地域、发展阶段 |
| 凡不在这三栏里的请柬，连看都不看 | 按投资偏好对项目进行筛选 |
| 第三堆六张——湖州织坊、绍兴染坊、嘉兴缫丝厂 | 筛选后留下的"符合其投资偏好的项目" |
| 周鹤年说"听得多，筛得细，才能挑得准" | 项目开发与筛选的核心——广泛汇集项目信息、按投资偏好层层过滤 |
