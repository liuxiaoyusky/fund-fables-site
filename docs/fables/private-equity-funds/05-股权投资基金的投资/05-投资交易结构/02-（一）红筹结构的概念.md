# （一）红筹结构的概念

## 🏮 寓言故事 —— 《陈老板的出海路》

陈老板在京城开了一家茶庄，名叫"玉叶轩"。他的茶叶来自云南深山，品质上乘，京城里的达官贵人都爱喝。几年下来，玉叶轩日进斗金，陈老板成了京城茶行里数一数二的人物。

可陈老板心里有一桩更大的买卖。他听说西洋人喜欢中国茶，若能将茶叶卖到海外，利润能比京城翻上几番。但西洋人买茶，只认"洋行"的牌子——他们习惯跟挂着洋名号的商号打交道，觉得那样才稳妥。陈老板一琢磨，决定在港岛注册一家"远东茶业有限公司"，用这块洋招牌去跟西洋人谈生意。

陈老板选了第一条路。他把自己在京城玉叶轩的股份，逐步转到了港岛公司的名下。港岛公司成了玉叶轩的东家，陈老板则通过持有港岛公司的股份，继续掌控着茶庄的一切。西洋商人看到港岛公司的洋招牌，纷纷愿意出高价入股。陈老板拿着这笔银子，在京城又开了三家分号，生意越做越大。这便是直来直去的一条路——洋招牌直接握着京城的买卖。

可陈老板的朋友林掌柜，就没这么顺了。林掌柜经营的是一家戏班，名叫"梨园春"。他的戏班在京城红极一时，名角儿辈出，连宫里都常请去唱堂会。林掌柜也想学陈老板，去港岛挂洋招牌、引西洋人的银子。可他刚要动手，就被人拦下了——戏班这行，京城有规矩，洋人是不能直接入股的。林掌柜急得团团转：不入股，怎么让洋招牌管住京城的戏班？

林掌柜去找陈老板讨主意。陈老板沉吟半晌，说："老林，入股不行，咱们换一条路。你让京城戏班还是你自个儿的，但跟港岛公司签几份文书——戏班的大事小情，由港岛公司说了算；戏班赚的银子，按约定分给港岛公司；戏班的名角儿，也由港岛公司帮着张罗。表面上，港岛公司不持有戏班的一股，可实际上，戏班唱什么戏、赚多少银子、怎么分账，全由港岛公司遥控。"

林掌柜照办了。他跟港岛公司签了一摞文书：一份管经营决策，一份管银子分账，一份管人事安排。西洋人看港岛公司有这般手段，能遥控京城最红的戏班，便纷纷掏钱入股港岛公司。林掌柜拿到了银子，在京城又养了三个新戏班，名角儿越出越多。

两条路，殊途同归。陈老板的路直，洋招牌直接握着京城的茶庄；林掌柜的路绕，洋招牌不持股，却靠一摞文书遥控着京城的戏班。可林掌柜心里始终不踏实——那些文书再细，也怕有朝一日戏班的人不认账，或者京城的规矩变了，文书成了一纸空文。他常对陈老板说："你这路走得踏实，我这路走得险，可谁让戏班这行，洋人入不得股呢？"

**📖 原文定义**

> 红筹结构是指中国境内（以下简称境内）企业股东在境外（以下简称境外）离岸法域（通常在香港、开曼群岛、英属维尔京群岛等）注册相应特殊目的公司（Special Purpose Vehicle，SPV），通过股权、协议等方式控制境内企业，以实现境内企业境外融资及上市目的。因该等离岸公司通常在香港上市，且由于中国背景，该类公司被称为红筹公司，其股票被称为红筹股。
>
> 搭建红筹架构时，需首先在境外离岸法域注册一个或多个SPV作为未来上市主体，然后由该等离岸公司收购或控制境内企业的股权、资产或业务，将境内企业的权益纳入到境外SPV的体系之下，最后由该等离岸公司在境外证券交易所上市融资。
>
> 根据离岸公司控制境内企业权益方式的不同，红筹结构可分为股权控制架构和协议控制架构（VIE结构）。
>
> 股权控制架构是较为传统的红筹架构，也是最为直接和简单的控制方式。在股权控制架构下，境内企业的实际控制人（创始人、股东等）在境外设立SPV，并将境内企业的股权或资产注入该SPV，由该离岸公司直接或通过其境外子公司间接持有境内企业的股权或资产，或者由该离岸公司在境内设立外商独资企业（WFOE）从事境内业务，然后由该SPV在境外证券交易所上市。
>
> VIE结构（Variable Interest Entity，可变利益实体），又称协议控制架构，是指境外上市实体与境内运营实体相分离，境外上市实体通过协议方式控制境内运营实体，该境内运营实体即为VIE。VIE的股东通常为境内运营实体的创始人或管理层，VIE持有开展业务所需的牌照、资质和资产，并实际开展业务。VIE的权益持有人（即境外上市实体）虽然并不直接持有VIE的股权，但根据相关协议安排，VIE的权益持有人能够实际支配VIE的行为，并从VIE的经营成果中获取经济利益，因此需要将VIE纳入合并报表范围。
>
> VIE结构自21世纪以来被广泛应用，极大地促进了中国互联网、传媒、教育、电信等受限制行业的企业的境外融资。通过VIE结构，境外SPV及其子公司可以在不直接持有境内企业股权的情况下，获得对境内企业的实际控制权，并享有境内企业的经营收益，从而实现境外融资和上市的目的。
>
> 在VIE结构下，境外上市实体通常通过一系列协议安排控制境内VIE的权益，包括控制VIE的股权、投票权、现金流等，从而实现与直接持股类似的控制效果。VIE结构的主要风险在于，其控制关系基于协议而非股权，存在协议被违反或不被执行的风险，以及因政策变化导致协议控制关系失效的风险。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 陈老板的茶庄在京城，想去海外卖茶 | 境内企业有境外融资上市需求 |
| 在港岛注册"远东茶业有限公司" | 在离岸法域注册SPV作为上市主体 |
| 陈老板把玉叶轩股份转到港岛公司名下 | 股权控制架构：离岸公司直接/间接持有境内企业股权 |
| 港岛公司成为玉叶轩东家，陈老板通过持股港岛公司继续掌控 | 境内实际控制人通过境外SPV控制境内企业 |
| 林掌柜的戏班，洋人不能直接入股 | 部分行业（传媒、电信等）限制外资直接持股 |
| 林掌柜与港岛公司签一摞文书管决策、分账、人事 | VIE协议控制：通过协议安排控制境内实体的股权、投票权、现金流 |
| 港岛公司不持股却能遥控戏班 | 境外上市实体不直接持有境内企业股权，但通过协议获得实际控制权 |
| 林掌柜担心文书不认账或规矩变了 | VIE结构风险：协议违约风险、政策变化导致协议失效风险 |
| 两条路殊途同归，都是洋招牌管京城买卖 | 股权控制架构和VIE架构都是红筹结构的两种实现方式 |
