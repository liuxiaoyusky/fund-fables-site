# 三、股债结合

## 🏗️ 寓言故事 —— 《老周的两扇门》


青石镇往东三十里，有个叫柳溪的小村子。村子依山傍水，家家户户以造纸为生，可造出来的纸粗糙发黄，只能包货垫底。村里有个年轻人叫阿禾，脑子活，手艺好，花了三年工夫琢磨出一种能把稻草变成优质草纸的法子。他造出来的纸又白又韧，写字不洇墨，消息传出去，邻县书院都派人来订。镇上的老周找上门来。

老周在镇上做了二十年买卖，专挑有出息的小作坊，出钱占一份，陪着一起长大。他经手的作坊有七八家，眼光毒，性子稳，从不做一锤子买卖。他亲自跑到柳溪，在阿禾的纸坊里蹲了三天，看阿禾配料、捣浆、抄纸、晒纸，又翻了他半年的出货单子，心里有了底。两人谈妥了：老周出一笔钱，换阿禾纸坊里的一份份子。可事情没那么快——老周得派两个老伙计去查阿禾的账本、清点家当、问遍他的主顾，这一套下来，少说也要两三个月。

偏偏这时，阿禾接了一笔急单。邻县书院要订三千刀纸，月底交货，可阿禾的料仓空了，没钱买稻草和石灰。阿禾急得满嘴起泡，在坊里团团转。老周也犯难：买卖还没谈定，现在就把大钱砸进去，万一查完账发现阿禾欠了一屁股债，或者主顾们的口碑是阿禾自己编的呢？这钱不就打了水漂？可不帮，阿禾这笔单子黄了，坊里工人的工钱发不出，人心散了，就算将来买卖谈成了，也是个烂摊子。

老周在阿禾的纸坊里转了三圈，忽然一拍大腿：“这样，我先借你一笔救急的钱，写清楚期限，按手印立字据。要是咱俩这买卖成了，这笔钱就算我入伙的本钱，不算利息；要是查完账我觉得不合适，你到时候还我本钱，再按市面上的规矩给点谢钱。”阿禾一听，眼泪都快下来了，当场按了手印。老周借的钱不多，刚好够买料，期限也只定了半年，利息低得几乎像是帮忙。阿禾拿着钱，连夜去邻村收稻草，石灰是从镇上赊的，月底交了货，赚了第一笔像样的钱。

纸坊开工了，阿禾按时交了货，赚了第一笔像样的钱。老周的伙计查完账，阿禾底子干净，主顾们的口碑也实打实，买卖成了，那笔救急的钱顺理成章变成了老周在纸坊里的份子。这是老周头一回用这招，后来他跟人说起，只说是“先递一根拐杖，再搀着手走路”，绝不敢把借钱当成正经营生，毕竟他的本分是入伙做买卖，不是开钱庄。他借出去的钱从来都是救急不救穷，小额、短期、低息，点到为止。

又过了两年，阿禾的纸坊变成了纸厂，雇了二十多个工人，可新问题来了。阿禾想添置一台水力捣浆机，价钱不菲，老周那份子钱已经用在扩建厂房上，手头也紧。阿禾去镇上的钱庄借，钱庄掌柜姓冯，戴着老花镜翻了翻阿禾的账本，摇摇头：“小本生意，风险大，利息得高。”阿禾一算，高利息吞掉利润，这机器买了也是白买，不买又赶不上订单，进退两难。

老周知道了，拉着阿禾去找冯掌柜。老周把自己的账本摊在桌上：这两年他在阿禾厂里投了多少钱，派了多少伙计盯着，阿禾每一笔进出、每一张出货单、每一个主顾的来历，他都门儿清。冯掌柜眯着眼看完，又瞅了瞅老周：“你的意思是？”

老周说：“我出份子，你出借款，咱俩一起扶这厂子。我的人在厂里盯着，你借出去的钱，风险我替你担一半眼力；真有个万一，你的钱先拿回来，我的份子后拿，甚至打水漂。可要是厂子做大了，你除了利息，再拿一张凭据，将来厂子出息了，你还能多分一杯。”

冯掌柜琢磨了三天，答应了。他借给阿禾的钱利息比市面上低两成，因为老周的眼力替他省了心；同时冯掌柜手里捏着一张凭据，万一阿禾将来发达了，他能按约定多拿一份。老周也高兴，阿禾有了冯掌柜的钱，机器买了，产量翻了三倍，老周那份子自然更值钱。阿禾最高兴，同样的机器，他花的本钱比隔壁厂子少了一大截，利润空间大了，还能再给工人涨点工钱。

青石镇的人后来管这叫“两扇门”——老周这扇是合伙的门，冯掌柜那扇是借钱的门，两扇门一起开，厂子才能既拿到长大的本钱，又不用被高利息勒住脖子。冯掌柜后来把这招用在了另外三家厂子，每次都先找一个像老周这样懂行、肯出力的人搭伙，钱庄的坏账少了，赚头反而多了。阿禾的纸厂五年后成了柳溪最大的作坊，老周和冯掌柜坐在他的新厅堂里喝茶，谁也不提当年那笔救急的借款，可心里都记着：有些事，一根拐杖比一份契约来得及时；有些人，两扇门比一扇门走得稳当。

**📖 原文定义**

> （一）股权投资基金对目标公司的债权投资
> 股权投资基金对目标公司进行债权投资，是指股权投资基金以债权形式对目标公司提供融资支持。一般情况下，股权投资基金不得从事名股实债、明基实贷等违规债权投资，但基于合理的商业理由，在以股权投资为目的的前提下，可以提供一定的债权融资，包括可转债等。
> 1. 以股权投资为目的，为被投资企业提供短期借款。在股权投资业务中，常见的情形有两种：一是被投资企业因生产经营需要，在股权投资款项尚未全部到账之前，由股权投资基金提供短期的流动资金支持；二是股权投资基金完成投资后，被投资企业由于经营发展需要，向股权投资基金提出借款需求，股权投资基金基于与被投资企业的合作关系，提供小额的股东借款。例如，某股权投资基金以10万元购买某被投资企业发行的6个月期可转债，约定在6个月内如果双方达成股权投资协议，则该笔可转债资金转化为股权投资款，否则被投资企业按照约定利率偿还本息。
> 2. 股权投资基金应当坚持股权投资为主的原则，债权投资为辅。股权投资基金对目标公司的债权投资应当遵循“小额、短期、低息”的原则，不得将债权投资作为主营业务，不得以利息收入为主要收入来源。
> （二）投贷联动
> 投贷联动是指股权投资基金与商业银行等金融机构合作，将股权投资与信贷融资相结合，为创新型企业提供综合金融服务的一种模式。在我国，社会融资仍然以商业银行贷款为主，但商业银行传统信贷模式在服务创新型企业方面面临信息不对称、风险与收益不匹配等挑战。投贷联动模式通过将股权投资与信贷融资相结合，有助于解决这些问题。
> 对于股权投资基金而言，投贷联动可以帮助被投资企业获得更多的资金支持，降低融资成本，提升企业价值和股东价值。对于商业银行而言，投贷联动有以下几个方面的好处：一是股权投资基金对目标公司进行了深入的尽职调查，商业银行可以借助这些信息降低信息不对称；二是商业银行作为债权人，在企业清算时具有优先受偿权，风险相对较低；三是商业银行通常可以通过认股权证等方式获得额外的收益。因此，投贷联动有助于商业银行在降低风险的同时提高预期收益。
> 投贷联动的主要模式包括：1. “银行+子公司”模式，即商业银行与其设立的具有投资功能的子公司合作；2. “银行+基金”模式，即商业银行与股权投资基金合作；3. 向股权投资基金发放贷款，由股权投资基金进行特定项目的股权投资。2016年，我国启动了投贷联动试点，选择了10家商业银行开展试点。一些地区还引入了担保、保险等机构，形成“投保贷联动”模式。
