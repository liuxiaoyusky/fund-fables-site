# （四）境内股权投资基金的投资方式

## 🏗️ 寓言故事 —— 《陈老板的过桥钱》

泉州城里有个做瓷器的老工匠陈德远，手艺精湛，烧出的青瓷远近闻名。可他有个心病——他的铺子虽然开在泉州，但早年间为了跑南洋生意，他在吕宋岛挂了个商号名头。如今南洋有个大商帮想正式入伙他的吕宋商号，按规矩得走一整套海路文书：要先去市舶司报备，再去商会登记，最后还得换得出海凭引。这一套下来，少说也要两三个月。

偏偏这时，陈德远的泉州窑口出了急事。他新研制的一批秘色瓷到了关键火候，需要立刻购进一批上等高岭土和松木柴，否则前功尽弃。可窑口账上的钱刚刚押在一批货上，周转不开。陈德远急得在窑口来回踱步，额头上全是汗。他派人去催了几家老客户结账，对方都说手头紧，要再宽限些日子。窑口里的工匠们看着渐渐熄下去的窑火，面面相觑，没人敢出声。

南洋大商帮的管事林世荣正好在泉州办事，听说了这事。林世荣是个老江湖，他想了想，对陈德远说："陈师傅，咱们入伙吕宋商号的文书已经在办了，迟早要成。但你的窑口等不了。不如这样——我先以个人名义借一笔钱给你救急，你拿去买土买柴，把瓷器烧出来。等海路文书办妥，我正式成为吕宋商号的合伙人，你再从窑口账上把这笔钱还我。我还你的这笔钱，正好就是我入伙吕宋商号该出的本钱。这样一来，窑口的难关解了，吕宋商号的合伙也顺了，两不耽误。"

陈德远一听，眼睛亮了。这法子妙啊！林世荣的钱先流到泉州窑口，解了燃眉之急；等文书下来，钱再流回林世荣手里，他拿这笔钱去完成吕宋商号的正式入伙。就像先在河上搭一座浮桥，让人先过去，等正式的大桥修好了，浮桥撤掉，人照样走大桥。

两人当即立下字据。林世荣把钱送到窑口，陈德远立刻派人去景德镇运高岭土，去武夷山买松木柴。窑火重新旺了起来，秘色瓷一炉一炉地出，色泽如玉，震惊了整个泉州港。

三个月后，海路文书终于齐备。林世荣正式在吕宋商号的合伙名册上落了名。陈德远按约从窑口账上把钱还给了林世荣。林世荣接过钱，转身完成了吕宋商号的正式出资。窑口的钱回到了林世荣手里，又流进了吕宋商号——钱还是那笔钱，路却绕了一个巧妙的弯。

后来泉州商帮都知道了林世荣这手"过桥钱"的妙计。但凡有人想入伙海外商号，而内地铺子又急着用钱，大家就都学这法子：先借钱救急，文书下来再还钱，还的钱正好用来完成正式入伙。这法子不违海路规矩，又解了燃眉之急，成了泉州商帮口耳相传的老智慧。有人说这法子绕了个弯，林世荣却笑而不语。他明白，做生意不是只看一条直路，有时候绕一个弯，反而能让两条船都顺顺当当地靠岸。

**📖 原文定义**

> 对于境内股权投资基金而言，在境内设立实体，必须按照相关法律规定办理境外直接投资（ODI）相关手续，通常包括发展改革委、商务部门的核准或备案，以及外汇管理部门的外汇登记。该过程通常需要数月时间。若涉及敏感项目或政策收紧，时间可能更长。
> 如果被投企业（OPCO）在ODI手续完成前急需资金，境内股权投资基金可以通过以下两种方式进行投资：
> 1. 境外认股权证+境内持股：境内股权投资基金与境外实体签署认购协议、股东协议、公司章程等文件，同时与OPCO签署增资协议，并与境外实体签署认股权证协议。基金向OPCO支付增资款，成为OPCO股东。ODI手续完成后，基金通过OPCO减资退出（减资金额等于增资金额），然后行使认股权证，使用减资所得资金完成对境外实体的投资。
> 2. 境外认股权证+境内借款：由于OPCO减资程序复杂且耗时较长，基金和被投企业更倾向于采用这种方式。基金与境外实体签署交易文件，同时与OPCO签署借款协议，并与境外实体签署认股权证协议。基金向OPCO提供借款。ODI手续完成后，OPCO偿还借款（一次性或分期），基金行使认股权证，使用偿还的借款资金完成对境外实体的投资。
> 例如：基金A拟投资1000万美元认购开曼公司B发行的100万股（每股10美元），预计ODI手续需要2-3个月。OPCO急需资金用于研发。基金A与B签署认购协议、股东协议、公司章程；与OPCO签署借款协议；与B签署认股权证协议。ODI手续完成后，基金A使用OPCO偿还的借款完成对B的投资。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 吕宋岛商号 | 境外实体（如开曼公司） |
| 泉州窑口 | 境内被投企业（OPCO） |
| 海路文书（出海凭引） | ODI手续（发改委、商务部门、外汇登记） |
| 林世荣想入伙吕宋商号 | 基金拟投资境外实体 |
| 窑口急需买土买柴 | OPCO急需资金 |
| 林世荣先借钱给窑口 | 基金向OPCO提供借款 |
| 文书办妥后窑口还钱 | ODI完成后OPCO偿还借款 |
| 还的钱用于正式入伙吕宋商号 | 基金用偿还的借款完成对境外实体的投资 |
| 认股权证协议 | 与境外实体签署的认股权证协议 |
| "过桥钱"的妙计 | 境外认股权证+境内借款的过渡性安排 |
