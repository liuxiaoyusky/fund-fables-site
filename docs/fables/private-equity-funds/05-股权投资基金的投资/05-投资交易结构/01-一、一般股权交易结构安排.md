# 一、一般股权交易结构安排

## 🏗️ 寓言故事 —— 《三兄弟入伙》

江南临安府有个做瓷器的老字号"青瓷坊"，东家姓周，膝下只有一个女儿周婉清。周婉清自幼跟着父亲学看窑火、辨釉色，手艺青出于蓝，可铺子要扩大，窑要添柴，账上却拿不出那么多银子。周婉清在账房里盘算了半个月，决定请外人来搭伙。

最先登门的是邻县的钱庄少东家钱满仓。钱满仓在临安府走了三圈，看过青瓷坊的窑口、库房和三十年的老账本，心里有了底。他开门见山对周婉清说："周姑娘，我要入伙，只走一条路——我把银子直接投进青瓷坊的账上。"周婉清问怎么个投法，钱满仓从袖中抽出一张纸，上面写着一个数。他说："这银子分作两半。一半记到青瓷坊的股本册子上，算是我的份子；另一半不记股本，算作我信得过你这门手艺的诚意，放进公账里留着周转。"周婉清一听便明白：钱满仓要的是青瓷坊从此姓"周钱"两家，而不是他个人借一笔债给铺子。她点了头，股本册子上添了钱满仓的名字，青瓷坊的底子厚了，新窑也砌了起来。

半年后，青瓷坊的老账房孙先生要告老还乡。他在铺子里干了二十年，手里握着一小股老股，想换成现银回乡买田养老。这时，城里又来了第二位客人——绸缎商李远行。李远行不爱往生意里添新钱，他只想把孙先生手里那点老股接过来。两人在茶楼谈妥，李远行把银票直接交到孙先生手上，青瓷坊的股本册子只换了一个名字，总数一文没增。周婉清看在眼里，心里清楚：李远行的银子没进铺子的公账，是孙先生个人得了养老钱，铺子本身没多出一分本钱。

又过了三个月，临安府来了个年轻人赵明远，说是从北边来的，专做南北货生意。赵明远比前两位都精明，他看中了青瓷坊的釉下彩新技法，却不想按钱满仓那么高的价码入伙。他在周婉清面前摆出一个三头六臂的方案：先出一份银子，从一位想抽身的旧伙计手里接下几股老股；再出一份更大的银子，像钱满仓那样直接投进铺子的公账，记上新的股本。周婉清起初皱眉，觉得这人把一桩买卖拆成两截，未免太绕。赵明远却笑着说："周姑娘，接老股的那笔银子，我按老价算，比新投的钱便宜些；新投的那笔银子，我按现在的价算，铺子得了活钱。两笔合在一处，我占的份子比单独走一条路都划算，您也一次清了旧人、进了新人，岂不是两全？"周婉清细想，旧伙计拿了现银走人，铺子公账又进了新钱，自己既没亏面子，也没少里子，便允了。

日子久了，青瓷坊里的三路人各走各的道。钱满仓每逢年底盘账，只关心公账上的流水和新窑的出息，从不问旁人手里老股转了几道手。李远行偶尔来铺子里坐坐，喝杯茶，看看股本册子上自己的名字，知道这铺子有他一份，却从不插手日常经营。赵明远倒是来得勤，南北两头跑，替青瓷坊拉来不少新主顾，可他心里始终有一本暗账：接老股那笔省下的银子，加上新投那笔占的份子，合起来比单打一的人都要合算。

周婉清有时在夜深人静时独自翻着股本册子，看着上面三个外姓的名字，忽然想起父亲说过的话："请人进门，银子怎么进门，比请谁进门更要紧。"她如今懂了——钱满仓把银子倒进铺子的锅，大家一起添柴；李远行从别人碗里舀了一勺，锅里的水没多也没少；赵明远两头都沾，锅里的水涨了，他自己碗里也满了。三条路，三种算法，端看你这铺子要的是什么。

**📖 原文定义**

> 1. 一般股权交易结构安排
>
> 在中国，除了一般意义上的境内直接股权交易外，股权交易有时还涉及境外架构以及股权和债权结合等安排。尽职调查、投资决策、谈判和签署投资协议后，进入交易交割环节。股权交易交割完成，股权交易方才算正式成为被投资企业的股东。
>
> 2. 股权交易有两种方式：一是增资，二是受让存量股权。创业投资基金通常主要采取增资方式进行投资，或者采取"增资+受让部分老股"的混合投资方式进行投资；并购基金通常以受让存量股权的方式对目标公司进行投资。
>
> 3. 增资：由于被投资企业的注册资本和估值通常不一致，投资金额的一部分计入注册资本，其余部分计入资本公积。例如，某股权投资基金拟投资2500万元于某目标公司，投资前目标公司的估值为10000万元，投资完成后基金持有目标公司20%的股权。假设2500万元中有25万元计入注册资本，则其余2475万元计入资本公积。
>
> 4. 受让存量股权：受让存量股权不会改变被投资企业的注册资本，投资金额全部支付给原股东。例如，某股权投资基金拟投资2500万元于某目标公司，投资前目标公司的估值为10000万元，投资完成后基金持有目标公司20%的股权。如果采取受让存量股权的方式，则2500万元全部支付给原股东。
>
> 5. "增资+受让部分老股"的混合投资方式：优点在于投资人可以部分老股转让较低的价格获得被投资企业的股权，降低投资成本，同时满足被投资企业原股东的变现需求。例如，某股权投资基金拟投资2800万元于某目标公司，其中800万元受让原股东持有的100万元注册资本（对应估值8000万元），2000万元进行增资（对应投前估值10000万元），获得200万元新增注册资本。投资完成后，基金持有目标公司300万元注册资本，占目标公司注册资本1200万元的25%，每1元注册资本的平均成本为93.33元。
>
> 6. 从资金来源看，创业投资基金通常主要使用自有资金进行投资，而并购基金通常使用较高比例的债务资金（杠杆收购）进行投资。
