# （三）VIE 结构的搭建与拆除

## 🏺 寓言故事 —— 《南北两爿店》

民国年间，江南水乡的沈寿庭做丝绸起家，靠一手"双面绣"的绝活，把小作坊做成了镇上数一数二的绸缎庄。绸缎庄要在本地开下去，得有丝绸专营的牌子；可这牌子官府管得严，轻易不发给外乡人。沈寿庭琢磨了半年，做出一个让人看不懂的安排。

他在南洋的槟城注册了一爿叫"南华绸业"的字号，名字挂在当地的华商名册上，自己做东家。南华绸业又派人回江南，在镇外租下铺面，挂出新招牌"华庭商行"。两边账房先生夜里悄悄碰头，签下厚厚一沓纸：沈寿庭在华庭商行挣的每一笔银子，先分一份给南华绸业；沈寿庭的进货单、卖货单、伙计的去留，都由南华绸业从槟城发话；镇上绸缎庄若出了事，华庭商行的招牌要顶在前面，南华绸业在槟城不露面；要是哪天华庭商行的伙计不认这沓纸了，南华绸业有权直接把人换掉。

南洋那边从此传出消息："槟城南华绸业，做的就是江南那爿绸缎庄的买卖。"外国买办一听，立刻拍出银票。可老主顾都在嘀咕：沈老板在镇上开铺，赚了钱却要先往南洋送一份，这账怎么算的？镇上同行也盯着华庭商行——一个本地铺子，凭什么听南洋一家空架子的号令？更让沈寿庭心里打鼓的是，那沓纸说到底只是一沓纸，伙计哪天翻脸不认，他连告状的地方都没有。

日子一晃就是五年。槟城南华绸业的钱越赚越多，可南洋那边风向也在变。当地商会传出一个说法：南华绸业若能"回家"——把槟城那层皮剥掉，让华庭商行用真名实姓在本地上市，镇上的老主顾便能名正言顺地认这门亲，绸缎庄往后的路也更稳。

消息传回江南那天晚上，沈寿庭在账房里坐了一整夜。他提笔给槟城写信，先开出一个价：愿把槟城南华绸业手里外头那些零散股东手里的份额，统统收回来。信末他又加一句：等把外头的人请走，再把那沓纸一张张烧掉，华庭商行从此就是华庭商行，再没有谁替它做主。

半年后，槟城那边的人散了，那沓纸也烧了。镇上的绸缎庄挂回了沈家自己的老招牌。老主顾再来买丝绸时，沈寿庭笑眯眯地说："这回，是我沈寿庭自己当家了。"

---

**📖 原文定义**

> 1. 典型的 VIE 结构搭建包括以下几个步骤：
>
> 上述VIE架构可以进行多方面的变化，来满足不同的商业和法律的诉求。例如在岸设立多个WFOE、多个OPCO甚至WFOE非100%协议控制OPCO。
>
> 基于一些商业考虑，此前已通过VIE结构完成境外融资甚至已在境外上市的企业，可能会选择拆除VIE结构并谋求境内上市。
>
> 2. 拆除 VIE 结构的一般步骤通常包括以下几点：
>
> （1）实施主体（通常是离岸上市主体的实际控制人股东，同时也是境内OPCO的主要股东）向离岸上市主体的公众股东和其他不愿意或不方便持有境内拟上市实体股权的其他股东发出收购要约，收购其持有的离岸上市主体股权；

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 沈寿庭在槟城注册南华绸业，又在江南租铺面开华庭商行 | VIE 结构中离岸上市主体与在岸业务运营实体相分离 |
| 双方签下的一沓厚厚的协议纸 | 离岸主体以"协议"方式控制境内业务实体的安排 |
| 南洋买办听说"南华绸业就是江南那爿绸缎庄的买卖"便拍出银票 | 离岸特殊目的载体在不持实际股权的情况下获得通常与所有权相关的权利和利益 |
| 伙计若翻脸不认那沓纸，南华绸业无处告状 | 通过协议实现控制存在合规不确定性及在岸主体违约的风险 |
| 沈寿庭写信要收走槟城外头散股东的份额 | 拆除 VIE 时实际控制人向公众股东发出收购要约 |
| 烧掉那沓纸、绸缎庄挂回老招牌 | 拆除 VIE 结构，谋求境内上市 |
