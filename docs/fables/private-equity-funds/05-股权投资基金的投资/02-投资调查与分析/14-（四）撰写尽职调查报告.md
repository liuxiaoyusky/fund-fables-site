# （四）撰写尽职调查报告

## 📜 寓言故事 —— 《绸庄的报告》

民国二十三年的春天，江南绸都盛泽镇的"锦源绸庄"想找一位大买家，把整座绸庄连货带人一起盘出去。消息传到上海，恒孚商行的陈慎之先生带了三个人，坐小火轮到盛泽看货。

陈先生是商行里出了名的眼睛毒，他一上手就安排了十五天的活：翻账房二十年的老账，走织造车间一间间看机杼响不响，把掌柜何仲德、账房老吴、车间领班和染坊主顾一一请到茶楼问话，又派人去下游布庄、裁缝铺打听这锦源绸子的口碑。十五天下来，三人各自揣了一摞厚本子，账页抄件、谈话条子、车间零碎、染坊收据，连茶楼小二无意说的话都记了一笔。

回上海那晚，陈先生把小周和阿郑叫到书房，往桌上"啪"地一撂那一摞本子："你们俩，三天之内，把这些全写成一份文卷，明天一早东家要过目定夺。"小周翻了两页就傻了："陈先生，这账目在第一本，谈话在第二本，车间在第三本，外头打听的在第四本，写给东家看，他怎么理得清？"阿郑也抓头："要不咱们就照时间排，从第一天写到第十五天？"陈先生摇头："写成流水账，东家看了只会更糊涂。"

陈先生不急，点起一锅烟，把三人领到自己那幅老挂轴前。挂轴是他早年在杭州拜师时老师留下的三个字——"纲、目、证"。他说："你们看这一摞本子，全是真东西，缺的不是料，是骨。写文卷的人，得给料搭一根骨。"小周问："什么骨？"陈先生用烟锅在桌上敲了三下："头一段，写你为什么去、怎么去、走了哪些路、见了哪些人——这叫'缘起'。中间那一大段，是全卷的肉，把账上的事、车间的事、人家嘴里说的事，分类别摆出来，再写下你和商行对这家绸庄的判断、愿出什么价、怕什么险、签合同时要卡哪些条款。末了一段，把所有撑腰的凭据——账页抄件、谈话条子、外头的口碑记录、染坊的收据——一样一样钉在后面，让东家随时能翻回去对。"

小周和阿郑各自闷头写了一夜。第二天清早，陈先生把两份稿摊在桌上。头一份果然三段分明：头一段写明此行是为商行拟收购锦源绸庄一事而做的摸底，列出十五天的行程和所见之人；中间一段先说生意好坏，再说账上疑点，再谈人和机器的稳不稳，最后落到商行的判断和出手条件；末一段把账本抄件、谈话纪要、外头打听来的口碑、染坊收据整整齐齐钉了二十多页。陈先生合上卷子，对小周点了点头："这卷子，东家看得懂，也信得过。"

半月之后，锦源绸庄的牌匾换了主人，何仲德领着一家老小回了乡下老宅。临走那天，他把陈先生请到茶馆，从怀里摸出一方旧印，说："陈先生，你们那卷文卷，比我这方印还稳。"陈先生笑："何掌柜，文卷靠的是人，靠的是把走过的路、听过的话、翻过的账，一样一样归到该归的地方去。"

**📖 原文定义**

> 撰写尽职调查报告是在对目标公司全面调查之后，对调查收集的资料进行客观全面的概括、判断与评价。
>
> 撰写尽职调查报告要遵循一定的格式与内容要求，不同的股权投资基金要求有所不同，但大体框架类似。详见本节"尽职调查报告"部分。
>
> 尽职调查报告主要分为前言、正文和附件三个部分。
>
> 前言部分主要阐述尽职调查的目标、方法、依据、程序和范围。需要阐述本次尽职调查针对公司业务、财务和法律进行了调研、核查和分析，对公司价值进行了系统和深度的挖掘，并已全部、完整披露在尽职调查报告中。此外，前言部分还对全文的主要概念进行阐述。
>
> 正文是尽职调查报告呈现内容的主体，是对被调查公司相关信息的全面、客观陈述。正文还包括结论和建议部分，报告尽职调查小组对目标公司的初步判断与评价，对投资框架、目标公司估值、投资风险识别及管理、投资协议要点以及投后管理重点等问题的建议。
>
> 附件主要附上尽职调查过程中的重要证据、第三方中介机构的报告、行业分析报告、访谈纪要等。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 陈先生一上手就安排翻账、走车间、请掌柜问话、派人去下游打听 | 全面调查（业务、财务、法律、口碑）后收集到的资料 |
| 桌上"啪"地一撂那一摞厚本子，账页抄件、谈话条子、车间零碎、染坊收据 | 调查收集的原始资料 |
| 小周和阿郑不知从何下笔，怕写成流水账 | 对调查资料"客观全面地概括、判断与评价"前需要搭好骨架 |
| 挂轴上的"纲、目、证"三个字 | 调查报告的大体框架 |
| 头一段：写为什么去、怎么去、走了哪些路、见了哪些人 | 前言——目标、方法、依据、程序、范围 |
| 中间一段：账上事、车间事、嘴里说的事分类别摆出来，落到商行的判断、出手条件、怕什么险、签合同时要卡哪些条款 | 正文——对被调查公司信息的全面、客观陈述，含结论与建议 |
| 末一段：把账页抄件、谈话条子、外头口碑、染坊收据一样样钉在后面 | 附件——重要证据、访谈纪要、第三方材料 |
| 何仲德临走说"那卷文卷，比我这方印还稳" | 报告是后续交易决策的重要参考和依据 |
