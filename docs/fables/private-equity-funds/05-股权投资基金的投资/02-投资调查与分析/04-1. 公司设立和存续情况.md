# 1. 公司设立和存续情况

## 📜 寓言故事 —— 《查底子》

清河镇老周商会的“江南织新”份子，账房先生阿福查完了锦绣织坊的银子，轮到讼师周先生上场——他要查锦绣织坊的“底子”，也就是这家织坊从生到今，是不是正经在县衙立了户、有没有合法的“身份”。

周先生带着两个书童，在县衙档案房里蹲了三天。他要查的东西，张锦绣自己可能都没想过会有人查得这么细。

**第一查：出生证**

周先生先查锦绣织坊的“出生证”——也就是县衙发的营业执照。他翻开县衙的册子，找到“锦绣织坊”的登记记录：光绪十五年三月设立，掌柜张锦绣，经营范围“织造业”，注册资本五百两。周先生逐字核对：名对、人对、经营范围对、资本额对。可他又翻到一本旧册子，发现锦绣织坊最初登记时，不叫“锦绣织坊”，叫“张家小织坊”，注册资本只有五十两。三年后，才改成“锦绣织坊”，资本增到五百两。

周先生问张锦绣：“这改名和增资，县衙批了吗？”张锦绣说：“批了，我交了文书。”周先生查了县衙的批文档案，确实有——改名批文、增资批文，手续齐全。他记下：出生合法，改名增资也合法。

**第二查：股东名册**

周先生再查股东名册。县衙册子上写着：锦绣织坊股东一人——张锦绣，出资五百两。可周先生又翻到一本“内部股东协议”——张锦绣和他弟弟张锦华签的，约定张锦华出资一百两，占两成股份，但不登记在县衙册子上。

周先生问张锦绣：“您弟弟这一百两，到账了吗？”张锦绣说：“到账了，可我嫌麻烦，没改县衙的册子。”周先生记下：股东名册与实际出资人不符，存在“隐名股东”。这在将来若有人查，或者张锦绣想卖织坊，会出麻烦——县衙只认册子上的人，不认私下协议。

**第三查：验资报告**

周先生查验资——也就是证明五百两资本确实到位的证据。县衙档案里有一份“验资报告”：光绪十五年，钱庄出具的证明，张锦绣存入五百两银子。周先生核对了钱庄的存根，确实——五百两，一日存入。可他又发现一个问题：张锦绣增资到五百两时，不是一次性存五百两，而是先存了三百两，两个月后存了两百两。验资报告只写了“总计五百两已到位”，没写是分两笔存的。周先生说：“分笔存没问题，但验资报告应该写明分笔的时间和金额，否则将来有人质疑——您是不是先存了、又抽走了、再存回来？”

张锦绣脸色变了：“我……我确实中间借出过一百两，一个月后还回来了。”周先生记下：存在“抽逃出资”嫌疑——资本到位后又借出，虽然还回来了，但验资报告没反映这一过程。

**第四查：非银子出资**

周先生最后查一项：张锦绣的五百两资本，全是银子吗？他翻到一本旧契约——张锦绣最初以“祖传织机图纸”作价五十两入股。可县衙档案里，没有这份图纸的“评估报告”，也没有“产权转移”记录。也就是说：张锦绣说图纸值五十两，可没人评估过；图纸本该从张锦绣个人名下转到织坊名下，可没办过户。

周先生对张锦绣说：“张掌柜，这图纸若将来有人质疑——到底值不值五十两？产权到底属您个人还是属织坊？——县衙没法认定。按规矩，非银子出资必须评估、必须过户。您这两条都没做，将来若织坊散了、或者您想卖股份，这五十两的窟窿，您得补上，而且织坊设立时的其他股东（如果有）还要连带担责。”

张锦绣听完，额头冒汗：“周先生，我这织坊开了二十年，头一回有人查得这么细。这些毛病，我自己都不知道。”

周先生把五天查到的所有问题，写成一份“底子报告”：出生合法、改名增资合法、但股东名册与实际不符、验资报告不完整、存在抽逃嫌疑、非银子出资未评估未过户。他在报告最后写了“解决建议”：第一，把弟弟张锦华正式登记为股东，或让他退股并签退股协议；第二，补一份完整的验资说明，解释分笔存款和中间借出情况；第三，请行家评估织机图纸价值，补办产权过户手续。

老周看完报告，对赵掌柜说：“查底子不是挑刺，是怕咱们投银子进去，将来发现这织坊的‘身份’有瑕疵——县衙不认、股东扯皮、出资有窟窿——那三千两就可能打了水漂。”

---

**📖 原文定义**

> 法律尽职调查中，需要注意目标公司的设立、注册资本、存续状况等是否符合法律规定。这一过程中，需要审阅的文件通常包括公司设立时的营业执照、股东名册、工商登记档案资料、验资报告和非货币资产出资的评估报告（如有）等。常见的问题包括目标公司股东没有足额出资、抽逃出资或目标公司缺少验资报告等。这些问题都可能在未来股权转让或IPO时受到监管部门的关注，应在法律尽职调查报告中记录这些问题，并尽量提供解决方案和建议。
>
> 根据《公司法》规定，以非货币财产出资的，对作为出资的非货币财产应当评估作价，还应当依法办理其财产权的转移手续。法律尽职调查过程中，应注意核查股东的非货币资产出资是否履行了评估手续，以及是否办理相应的过户手续，将财产权转移给公司。对于股东A而言，如果股东A实际出资的知识产权的实际价额显著低于所认缴的出资额，股东A有义务补足差额，目标公司设立时的其他股东还应当在股东A在出资不足的范围内承担连带责任。法律尽职调查报告应对目标公司存在的这一问题及相应的后果、解决建议予以提示和说明。
