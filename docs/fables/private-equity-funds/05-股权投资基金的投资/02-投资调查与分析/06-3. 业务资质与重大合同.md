# 3. 业务资质与重大合同

## 📜 寓言故事 —— 《查牌照》

清河镇老周商会的“江南织新”份子，账房先生阿福查完了锦绣织坊的银子，讼师周先生查完了底子，轮到另一位行家上场——老周派了商会里专管“行规”的孙先生，去查锦绣织坊有没有正经的“牌照”，以及它签过的“大单子”有没有坑。

孙先生到了锦绣织坊，开门见山：“张掌柜，您织坊开门做生意，县衙发的‘织造业执照’给我看看。”张锦绣从柜子里翻出一张泛黄的纸——光绪十五年县衙发的，经营范围写着“织造、销售布匹”。孙先生逐字核对：名对、经营范围对、发证机关对。可他翻到背面，发现执照右下角有一行小字：“有效期至光绪二十年三月”。

孙先生一惊：“张掌柜，这执照去年三月就到期了！您这一年多是无照经营？”张锦绣脸红了：“我……我去县衙续过，可县衙说新规矩，织坊要加设‘防火渠’才能续。我嫌麻烦，一直没办。”孙先生记下：核心牌照过期，织坊目前处于非法经营状态。若县衙来查，随时可能关门。他立刻让张锦绣去县衙补办，并要了一份“补办受理回执”——证明县衙已经受理，正在审核。

孙先生又问：“除了织造执照，您还有别的牌照吗？”张锦绣说：“水力织机要用河里的水，县衙发过‘用水许可’。”孙先生要来看——有效期还剩两年。他问：“两年后能续吗？”张锦绣说：“应该能，这条河的水权是镇上的，每年交租就行。”孙先生记下：用水许可有效，但需关注到期后续租风险。

**查完牌照，孙先生开始查“大单子”。**

张锦绣说：“我织坊最大的客户是县城‘瑞祥绸缎庄’，每年订三百匹上等细布，已经合作五年了。”孙先生要来看合同。合同上写着：每年三百匹，单价三两一匹，货到付款，合同期限“长期有效”。

孙先生皱眉：“‘长期有效’是什么意思？没写截止日期？也没写续签条件？”张锦绣说：“咱们口头约定，每年年底续签。”孙先生记下：合同期限不明确，存在随时终止的风险。他追问：“若瑞祥绸缎庄明年不续了，您这三百匹的销路怎么办？”张锦绣说：“还有别的客户……”可别的客户每年只订几十匹，远不及瑞祥。

孙先生继续翻合同，发现一条“隐藏条款”——合同背面用小字写着：“若锦绣织坊股权变更，需提前三个月通知瑞祥绸缎庄，瑞祥有权在通知后三十日内终止合同。”孙先生一惊：“张掌柜，您知道这条吗？”张锦绣摇头：“签的时候没注意，瑞祥的掌柜也没提。”

孙先生立刻明白：老周商会投三千两银子入股锦绣织坊，这本身就是“股权变更”。按这条款，张锦绣必须提前三个月通知瑞祥，而瑞祥有权在三十日内终止合同——也就是说，商会一入股，最大的客户可能就跑掉。

孙先生把这条指给张锦绣看，张锦绣脸色煞白：“这……这怎么办？”孙先生说：“两条路：第一，签补充协议，把这条删掉或改宽松；第二，若瑞祥不同意改，您得在入股前找好替代客户，或者降低对瑞祥的依赖。否则，商会一入股，您最大的单子就没了，织坊值三千两还是值一千两，难说。”

孙先生还查了另外两份“大单子”：一份是和邻县染坊的“独家供货协议”——锦绣织坊承诺三年内所有布匹只卖给这家染坊，染坊承诺以固定价格收购。可染坊去年换了掌柜，新掌柜说“固定价格太低，要重新谈”。这份合同正在扯皮中。另一份是和钱庄的“借款合同”——锦绣织坊借了三百两，以厂房抵押，若到期不还，钱庄有权收走厂房。下个月到期。

孙先生把三份合同的发现，写成“查牌照帖”：织造执照过期正在补办、用水许可有效但需关注续期、最大客户合同含“股权变更终止条款”且期限不明确、独家供货协议正在扯皮、借款合同即将到期可能丧失厂房。

老周看完，对赵掌柜说：“查牌照不是看张锦绣有没有证，是看他的证全不全、有没有到期、大单子背后有没有暗雷。一张过期的执照、一条隐藏的股权变更条款，都能让三千两银子变成废纸。”

---

**📖 原文定义**

> 应结合目标公司主营业务，审阅公司是否取得法律法规要求的经营资质，并关注相应业务资质的存续情况，例如有效期限、资质到期后能否续期的风险等。目标公司签订的重大合同对目标公司的价值通常具有较大的影响。在法律尽职调查过程中，应重点核查重大合同的金额、支付方式和主要条款，特别要关注合同履行过程中可能存在的不确定性。还应注意这些重大合同是否包含可能影响投资交易的条款。
