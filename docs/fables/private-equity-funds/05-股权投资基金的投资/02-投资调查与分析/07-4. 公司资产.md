# 4. 公司资产

## 📜 寓言故事 —— 《盘家产》

清河镇老周商会的“江南织新”份子，账房先生阿福查完了锦绣织坊的银子，讼师周先生查完了底子，孙先生查完了牌照和合同，最后轮到商会里专管“家产”的吴师傅——他要亲自到锦绣织坊，把张锦绣的家产一件一件盘清楚，看哪些是真金白银，哪些可能是“镜中花、水中月”。

吴师傅带着两个学徒，在锦绣织坊待了五天。他盘的不是账本上的数字，而是看得见、摸得着的“东西”。

**第一盘：地皮和房子**

锦绣织坊有一座厂房、三间仓库、一处住家院子。张锦绣说：“这厂房是我祖传的，地契在我手里。”吴师傅要来看地契——确实是张锦绣的名字，可翻到背面，有一行小字：“光绪十二年，抵押给县城钱庄，借款三百两，期限十年，到期未还则钱庄有权收房。”

吴师傅一惊：“张掌柜，这厂房抵押了？”张锦绣说：“是……是抵押了，可我每年还利息，钱庄没催。”吴师傅记下：厂房有抵押，若到期不还本金，钱庄有权收房。这厂房看似是张锦绣的，实际上“根”在钱庄手里。他问：“地契上写的土地用途是什么？”张锦绣说：“宅基地。”吴师傅摇头：“织坊是做生意的，宅基地不能商用。您这厂房盖在宅基地上，县衙若来查，可能算违建。”

他又查仓库——两间是张锦绣自己的，一间是租的。租约明年到期，房东已经口头说要涨租五成。吴师傅记下：自有仓库有抵押风险，租用仓库有涨价风险。

**第二盘：织机和工具**

账面上写着：二十台织机，每台估价八十两。陈师傅之前已经查过织机的新旧，吴师傅查的是“权属”——这些织机是不是锦绣织坊的？有没有抵押给别人？

张锦绣说：“织机全是我的，没抵押。”吴师傅要来看购买织机的文书和付款记录。十八台新水力织机，有购买契约和钱庄付款凭证，权属清楚。可两台老木机——张锦绣说“祖传”，但没有购买记录。吴师傅问：“这两台老木机，有没有可能是您父亲或兄弟的？”张锦绣说：“是我父亲的，他去世前传给了我，没立字据。”吴师傅记下：老木机权属有争议风险，若张锦绣的兄弟将来主张继承权，这两台织机可能不属于锦绣织坊。

**第三盘：招牌和手艺**

吴师傅问：“张掌柜，您织坊的招牌‘锦绣’，在县衙登记过吗？”张锦绣说：“没有，我就是用了几十年，大家都知道。”吴师傅记下：商标未注册，若邻县有人抢先注册“锦绣”招牌，张锦绣可能被迫改名，多年积累的商誉受损。

他又问：“您那台水力织机的图纸，是谁画的？”张锦绣说：“我请邻县一位老工匠画的，花了二十两。”吴师傅问：“图纸的著作权归谁？是您还是那位工匠？”张锦绣一愣：“我没想过……应该是我的吧？我付了钱。”吴师傅说：“付了钱不一定归您。若工匠当初没说清楚，他以后可能主张图纸是他的，您只是买了使用权。这水力织机是织坊的核心，图纸若权属不清，将来有人仿制，您没法告。”

**第四盘：域名和渠道**

吴师傅还查了一件张锦绣没想到的事：“张掌柜，您织坊的货，有没有通过‘运河商路’的线上渠道卖？”张锦绣说：“有，我托省城一家商号代卖，他们有个‘线上铺子’，挂着我的货。”吴师傅问：“这‘线上铺子’的域名是谁的？是您还是那家商号？”张锦绣说：“商号的吧……我没问过。”吴师傅记下：线上销售渠道依赖第三方，域名和平台归属不明，若商号不合作了，线上渠道可能瞬间消失。

五天盘完，吴师傅写了一份“盘家产帖”：厂房有抵押且土地用途不符、仓库一间租约到期将涨价、老木机权属有继承争议、招牌未注册、织机图纸著作权归属不清、线上渠道依赖第三方。他在帖子里加了一条建议：投资交割前，张锦绣必须完成以下手续——还清厂房抵押借款或取得钱庄同意不追索的书面承诺、将宅基地变更为商用土地、注册“锦绣”招牌、与老工匠补签图纸著作权转让协议、与省城商号确认线上渠道的合作期限和域名归属。

老周看完帖子，对赵掌柜说：“盘家产不是数银子，是看东西到底属不属于他、有没有被抵押、有没有被限制、将来会不会被人抢走。账面上值三千两的家产，若地皮有抵押、招牌没注册、图纸权属不清，实际可能只值一千五百两。盘清了，才知道银子该投多少。”

**📖 原文定义**

> 法律尽职调查还应对公司持有的资产及潜在法律风险予以关注。在不动产方面，应关注公司持有的土地、房产在建工程等情况，审阅相应的土地及房产权属证书是否齐全、土地用途是否符合要求、是否存在权利瑕疵等。对于主要通过租赁房屋开展业务运营的公司，还应关注公司的承租权是否存在瑕疵。在知识产权方面，应关注公司核心专利技术、商标权、著作权（包括计算机软件著作权）、域名等权属情况及许可情况，是否存在侵权风险、权利限制等问题。除此之外，对于公司主营业务有重要影响的其他资产，也应对其权属状况及法律风险予以关注。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 厂房地契背面有抵押记录，钱庄有权收房 | 不动产：审阅土地及房产权属证书是否齐全，是否存在权利瑕疵 |
| 土地用途是宅基地，织坊是商用，可能算违建 | 土地用途是否符合要求 |
| 仓库一间是租的，明年到期房东要涨租五成 | 租赁房屋：关注承租权是否存在瑕疵 |
| 老木机是祖传，无购买记录，权属有继承争议 | 资产权属状况及法律风险 |
| “锦绣”招牌用了几十年，未在县衙登记 | 知识产权：关注商标权权属情况 |
| 水力织机图纸付了钱但著作权归属不清 | 知识产权：关注著作权权属情况及许可情况 |
| 线上铺子域名归属不明，依赖第三方商号 | 域名等权属情况及许可情况，是否存在权利限制 |
| 建议交割前完成抵押清偿、土地变更、招牌注册、图纸转让、渠道确认 | 将取得权属证书作为投资协议的交割先决条件，化解潜在风险 |
