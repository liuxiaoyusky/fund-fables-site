# 6. 劳动人事

## 🧵 寓言故事 —— 《绣坊里的老绣娘》

江南的织锦街上，赵掌柜开了间绣坊，请了三十个绣娘，专给城里的绸缎庄做嫁衣。生意红火了七八年，赵掌柜却动了别的心思——他想把手里的绣坊盘出去，喊个高一点的价钱。

登报没几天，城西的钱老板带着账房先生找上了门，说是想接手。钱老板是个爽快人，开门见山："赵哥，我看了你柜上的活，确实漂亮。但我先问你一句：铺子里这三十个绣娘，是都签了长契，还是临时从外头喊来的？"

赵掌柜愣了一下，笑着说："咱们这行，靠的是手艺吃饭。签契这事儿，看缘分——几个老师傅跟了我十几年，自然走得近；去年活儿多，从隔壁镇临时招了几个，签的短契；还有几个，是从别的绣坊借过来的，按件给钱。"

钱老板点点头，又问："那官府那边的银两——你替她们交了没有？"赵掌柜脸上的笑有些挂不住了："老师傅们我都按月给的，新来的几个……这行人都缺，没那么讲究。"

钱老板喝了口茶，没接话。账房先生悄悄把几张写满字的纸塞到他手里，是赵掌柜没注意的时候，从绣娘嘴里打听来的——隔壁镇来的那几个绣娘，原本和另一家绣坊签了两年长契，那家绣坊给了她们东家一笔银子，说是把人"借"过来的。这事儿没经过官府备案，也没签正式的字据。

钱老板放下茶碗："赵哥，我再问一句——你这绣坊里，有没有人跟你扯过皮？"

赵掌柜叹了口气："不瞒你说，去年走的一个小绣娘，告到了衙门，说我少给了她两个月的工钱。我赔了银子才了结。这事儿我以为翻篇了，可她那老乡还惦记着，三天两头跑来要说法。"

账房先生又递过来一张纸——是那个走了的绣娘和她老乡的"讨字据"，上头按着指印，写明了赔偿的银子还没给齐。

钱老板站起来，走到窗边，看了看绣坊后院的井台。几个绣娘正围着看一封信，神色都有些慌。原来城西最有名的"针神"孙师傅刚收到别家绣坊的聘书，写信来问赵掌柜："我去还是不去？我那契约里有没有这一条？"

赵掌柜一时答不上来。他翻了翻柜子里的旧契书，要么是十年前草草写的几句话，要么压根没纸。钱老板轻轻摇了摇头，把椅子推回原位："赵哥，你这绣坊的手艺是真好。可你手里这三十个绣娘，有的契不清，有的银两没交齐，有的官司还没了结，有的老师傅想走你拦不住。我今天买下来，明天就得替你收拾这一摊子烂事。"

赵掌柜脸上的笑彻底没了。

账房先生合上纸，跟钱老板一起出了门。绣坊里的织机还在咔哒咔哒地响，可赵掌柜知道，那几匹没织完的锦，怕是要在自己手里停一停了。

---

**📖 原文定义**

> 在劳动人事方面，应关注公司职工的劳动合同签署情况及社保缴纳情况。具体而言，包括是否依法签订书面劳动合同，是否存在劳务派遣用工及其所占比例，是否及时足额缴纳社会保险费和住房公积金等，以及是否存在未妥善解决的劳动纠纷等，并提出适当的解决方案建议。还应对高级管理人员、核心技术骨干等人员的劳动合同中是否包括竞业禁止条款以及对应的补偿条款予以重点关注。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老师傅的口头长契与新绣娘的短契 | 劳动合同的签署情况（是否依法签订书面劳动合同） |
| 临时借来的几个绣娘，没签正式字据 | 劳务派遣用工及所占比例的关注 |
| 几个新绣娘没交的那份"官府银两" | 是否及时足额缴纳社会保险费和住房公积金 |
| 走了的小绣娘告到衙门、没赔齐的银子 | 未妥善解决的劳动纠纷 |
| 孙师傅拿着别家的聘书来问能不能走 | 高级管理人员、核心技术骨干的竞业禁止条款与补偿条款 |
| 钱老板看完后摇头起身离开 | 劳动人事风险对投资决策的影响 |
