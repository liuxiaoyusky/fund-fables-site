# （一）业务尽职调查

## 🔍 寓言故事 —— 《老陶买窑》

景德镇东郊的瓷商老陶做了二十年瓷器买卖，去年想自己烧窑。挑来挑去，相中半山腰的旧作坊——"周家窑"。窑主老周去年过世，他儿子小周接了半年没接住，想把整座窑盘出去，开价三千两。

老陶没急着拍钱。他跟自己说，不能只看板砖，得把这窑值不值得、能不能救，从里到外翻一遍。

老陶花了十五天，把周家窑翻了个底朝天。头五天，他在小周茶楼里跟几个老师傅一个一个聊。老郑干了四十年，老陶问他周家窑最拿手的是什么。老郑放下茶碗说："周老师傅在世时烧的是影青，走'薄胎厚釉'的细路子——别人家一窑出三百件粗碗，我们一窑出八十件薄胎小盏，单价贵三倍。"老陶心里一动，又问："后来呢？"老郑叹气："老师傅一走，小周把路子改了。他嫌细瓷出货慢、压钱多，改烧青花大罐——可窑膛形状、釉料配方、匣钵规格全是按细瓷来的，硬改粗瓷，十窑里废七窑，连师傅都走了两个。"

第二个五天，老陶去摸周家窑在整条链子里的位置。他连着三天起早去镇上赶集，看周家窑过去的老客户——杭州茶商、苏州绸庄、南京文具铺——还来不来拿货。茶商老林只拿了两件样品，说"周家窑改烧粗罐之后我们早换了别家"；绸庄李掌柜没来，派人带话："周老师傅那会儿盏底有一道'冰裂纹'，全江南只此一家，再没有了。"第三天，老陶跑去山下陈家窑串门，看人家青花大罐的价钱和路子。回来路上他心里有数了：陈家窑做粗瓷二十年，上下游稳，价钱压得比周家窑低两成——小周偏要跟陈家窑抢这门生意，怎么抢得过。

第三个五天，老陶扎进周家窑的窑场。他蹲在窑膛口看了一整天，看小周手下剩的两个年轻徒弟怎么装匣钵、怎么看火候、怎么配釉料。两个年轻人手脚生疏——釉料配比乱、匣钵留的空隙大小不一、看火候全凭感觉。

临走前一晚，老陶在屋里把十五天的见闻一笔一笔落下来：他看见了周家窑过去怎么挣钱——靠周老师傅那道独门的影青薄胎手艺；看见了现在怎么亏钱——小周丢了手艺去拼粗瓷大罐，又拼不过陈家窑；也看见了将来还能不能挣钱——能，办法只有一个：把影青薄胎的路子捡回来，把老郑那帮老师傅请回来，再把窑膛、釉料、匣钵调回细瓷的节奏。但救起来不容易：影青这条路少说得两三年才能把师傅、配方、老客一点点攒回来；老郑岁数大了；窑膛改造要停窑三个月。

老陶在桌前坐到天亮。他没去砍价，也没拍钱"明天就过户"。他出门前把小周请到茶馆，端起茶碗说："小周老弟，这窑值不值三千两，问我它有多少砖多少瓦，我答不上来。问我这门手艺还活不活、还能不能救回来——还活着，但救起来不容易。你要是愿意按'把影青捡回来'这条路走，咱们再谈。你要是还想烧粗罐拼陈家窑，那这三千两，我拿不出。"

小周端着茶碗的手，停在半空。

**📖 原文定义**

> 业务尽职调查是整个尽职调查工作的核心，目的是了解过去及现在企业创造价值的机制以及这种机制未来的变化趋势。

> 不同企业类型及所处发展阶段的目标公司调查内容的侧重点也不同。对于处于相对较早发展阶段的创业企业，业务尽职调查的重点是管理团队和产品、技术与服务部分；对于处于扩张期的企业，虽然管理团队也是尽职调查重点关注的事项，但对产品、技术与服务，发展战略及市场因素的关注程度相对早期企业要更高一些；对于成熟阶段的企业，尽职调查问题则更为广泛，管理团队、资产质量、融资运用、发展战略乃至风险分析都是需要考察的部分。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老陶没急着拍钱，先翻一遍窑 | 业务尽职调查是尽职调查的核心 |
| 老陶看的不只是砖瓦，而是"窑值不值得、能不能救" | 调查目的——了解企业创造价值的机制 |
| 老陶问老郑"周家窑过去最拿手的是什么" | 过去创造价值的机制（产品、技术与服务） |
| 老陶摸客户还来不来拿货 | 现在创造价值的机制——市场与客户 |
| 老陶跑去陈家窑串门看粗瓷价钱 | 竞争与市场因素 |
| 老陶蹲在窑膛口看徒弟装匣钵、看火候、配釉料 | 产品、技术与服务的实际执行能力 |
| 老陶写下"过去怎么挣 / 现在怎么亏 / 将来还能不能挣" | 机制过去、现在、未来变化趋势的三段式分析 |
| 老陶看出"影青手艺还在，但救起来不容易" | 业务尽职调查对投资风险与未来潜力的识别 |
| 老陶没拍钱，把判断摆给小周 | 业务尽职调查是投资决策的依据 |
