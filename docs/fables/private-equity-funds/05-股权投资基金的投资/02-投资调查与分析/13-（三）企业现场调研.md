# （三）企业现场调研

## 🔍 寓言故事 —— 《蹲点记》

清河镇老周商会的“江南织新”份子，摸底队五人已经查完了锦绣织坊的账本、底子、手艺、牌照和家产。可老周坐在商会正厅，把五份报告摊在桌上，摇了摇头。

“各位，你们查的都是纸上的东西——账本、契约、执照、地契。可锦绣织坊到底活不活、织工到底干不干、客户到底买不买，纸上写不出来。咱们得去蹲点——不是走马观花，是扎下去，看它的真面目。”

**第一天：内部访谈**

老周带着阿福、陈师傅、孙先生、吴师傅，再次来到锦绣织坊。张锦绣迎出来，脸色有些紧张：“周掌柜，该查的都查过了，还来？”老周笑：“张掌柜，这次不查纸，查人。您把织坊里上上下下的人，都叫来，咱们聊聊。”

老周先和张锦绣一对一谈了两个时辰。他问的不是账本数字，而是：“张掌柜，您每天几点到织坊？织工迟到您怎么管？上个月织坏了一批布，您怎么处理的？您儿子不愿接班，您打算怎么办？”张锦绣起初回答得流利，可问到“儿子不愿接班”时，他沉默了半晌，叹道：“我逼过他，可他宁愿去县城读书。我……我在培养徒弟，可徒弟才十七，手艺还嫩。”老周记下：接班人问题比账本上写的更严重，张锦绣的焦虑是真实的。

接着，老周让阿福去找织坊的账房先生——不是查账，是聊天。阿福问：“您每天记账忙到几时？张掌柜多久看一次账？”账房先生说：“忙到戌时，张掌柜每月初一看一次。”阿福问：“上个月有笔三百两的预收款，客户后来退订了，您怎么处理的？”账房先生一愣：“您……您怎么知道？”阿福笑：“账本上有痕迹。退了之后，银子还回去了吗？”账房先生低声：“退了，可张掌柜让我别记在明账上，怕您看出来。”阿福记下：有一笔隐性退款，张锦绣在刻意隐瞒。

陈师傅则去找织工头老李。老李在锦绣织坊干了十五年，手艺最好。陈师傅问：“李师傅，织坊二十台织机，每天实际开几台？”老李说：“十八台。两台老木机坏了，修不好，张掌柜舍不得扔，占着地方。”陈师傅问：“织工每天干几个时辰？”老李说：“春夏忙时八个时辰，秋冬闲时六个时辰。可去年秋冬，张掌柜让干七个时辰，说是赶订单，可订单是假的——他怕织工闲了跑掉，硬编的理由。”陈师傅记下：产能利用率有水分，织工管理靠“编订单”维系。

孙先生去找织坊的销售掌柜老王。老王管着所有客户。孙先生问：“最大的客户瑞祥绸缎庄，去年订了多少匹？”老王说：“三百匹上等细布。”孙先生问：“今年呢？”老王叹气：“今年只订了一百五十匹。瑞祥的掌柜说，邻县彩云织坊的布更便宜，他们转了一半订单过去。”孙先生记下：最大客户流失一半，账本上却写着“长期合同稳定”——与事实不符。

吴师傅则去仓库找管库的老赵。老赵带吴师傅看存货。账面上写着：上等细布五百匹、中等粗布三百匹、次等布一百匹。可实际一看：上等细布只有三百匹，而且其中一百匹边角有瑕疵；中等粗布四百匹，混杂着不少去年的陈货；次等布两百匹，堆在角落发霉。老赵低声说：“张掌柜让我把去年的陈货也算成今年的新货，账面好看。”吴师傅记下：存货质量和账面严重不符。

**第二天：外部访谈**

老周没有只问锦绣织坊内部的人。他派孙先生去邻县，找彩云织坊的掌柜喝茶。彩云掌柜说：“锦绣织坊？张锦绣手艺好，可他不会算账。去年他接了个大订单，赶工赶坏了三成布，赔了客户两百两。今年他学乖了，不敢接大单，可小单利润薄，织工又闲不住。”孙先生记下：张锦绣的经营能力有硬伤——接大单会砸，不接大单又养不起织工。

老周又派吴师傅去县城瑞祥绸缎庄。瑞祥掌柜说：“锦绣的布是好布，可张锦绣这人太固执。去年我们建议他改一种新花样，他死活不肯，说‘祖传手艺不能改’。结果彩云织坊改了，抢了我们的订单。张锦绣不是手艺不行，是脑子不转。”吴师傅记下：产品创新能力弱，管理层僵化。

老周还派阿福去县城钱庄，找给锦绣织坊做担保的掌柜。钱庄掌柜说：“张锦绣去年借了两百两，按时还利息，可本金到期后，他求我延期三个月。我延了，可他到现在还没还。我不是怕他赖账，是怕他的织坊现金流真的紧——若再来一笔大支出，他可能断链。”阿福记下：偿债能力比账面上显示的更脆弱。

**第三天：实地考察**

老周亲自带着陈师傅，在锦绣织坊蹲了一整天。早上卯时，织工陆续到岗——迟到了三个，张锦绣站在门口骂，可织工嬉皮笑脸，不当回事。上午，十八台织机全开着，织工手脚麻利，可陈师傅注意到：两台织机上的布，经纬线不齐，织工自己发现了，偷偷剪了重织——浪费了不少丝。中午，织工在院子里吃饭，三三两两聊天。老周凑过去听，听到一个年轻织工说：“李师傅手艺好，可张掌柜给的工钱比彩云低一成，明年我想去彩云。”另一个老织工说：“别去，彩云掌柜刻薄，锦绣虽然钱少，可人不坏。”老周记下：织工队伍有流失风险，但人心尚未散。

下午，老周跟着送货的牛车，去县城一家小布店。布店掌柜说：“锦绣的布我卖了五年，可今年客户嫌花样老，销量跌了四成。我劝张锦绣改花样，他说‘老主顾认这个’。可老主顾也在变老，年轻人不买账啊。”老周记下：终端销售下滑，产品老化。

晚上，老周回到商会，把三天的蹲点发现汇总：接班人危机、隐性退款、产能水分、客户流失、存货不实、经营能力硬伤、创新僵化、现金流紧、织工流失风险、终端销售下滑。

他对赵掌柜说：“蹲点三天，比查账七天看到的东西还多。纸上写的是锦绣织坊‘想让我们看到的样子’，蹲点看到的是它‘真实的样子’。两者一对比，才知道三千两银子到底该不该投、该投多少、该提什么条件。”

赵掌柜问：“那还投吗？”老周说：“投，但条件要改。第一，张锦绣必须两年内找到接班人或培养好徒弟；第二，商会派驻账房，每月查账，杜绝隐性操作；第三，投资金额从三千两降到两千五百两，估值按实际存货和产能重新算；第四，协议里加一条——若瑞祥绸缎庄订单再跌两成，商会有权启动回购条款。”

**📖 原文定义**

> 二手资料与项目信息的研读、行业前景的调查，都仅停留在“纸上谈兵”的阶段。要获得目标公司的第一手资料，尽职调查团队必须对目标公司进行现场调研。
>
> 尽职调查团队与目标公司管理层、员工的内部访谈可达到多个目的：第一，与管理层会谈可以全面了解目标公司的信息，尤其是某些比较关键、敏感的信息；第二，与管理层进行沟通可以间接考察管理层的素质；第三，与管理层进行沟通还可以增进了解、增强信任；第四，与各层级、各职能员工的访谈，可以获取企业各个环节的信息，有助于基金管理人从各个层面窥测企业全貌，并对信息的真实性进行考证。
>
> 对于高级管理人员、关键岗位人员要开展一对一访谈沟通，对于普通员工可以按照工作岗位展开小组座谈、随机访谈等非正式沟通，了解并验证相关业务、财务及法律问题。
>
> 此外，与企业的供应商与合作伙伴开展外部访谈同样重要。与目标公司外聘的法律顾问和审计机构的沟通会提供很多有用信息，尽职调查团队也应访问竞争对手及上下游企业负责人并获取他们对目标公司的评价。
>
> 实地考察的目的主要是将书面信息与现场调研信息进行相互印证，从而对企业生产经营情况有一定的感性认识。
>
> 对企业相关的场所进行现场调查，包括但不限于厂房、设备、土地、产品和存货等。譬如，走访企业的销售现场（如超市、门店或经销商）有助于评估真实的销售情况。根据目标公司所在行业与经营性质的差异，还应对目标公司的销售、采购、生产、物流、财务、人力资源等部门进行针对性的走访调研。
>
> 需要调查了解的目标公司现场信息包括：企业生产线是否运转正常；企业管理状况是否良好；企业客户数量是否稳定；客户对企业的产品与服务是否满意等。
