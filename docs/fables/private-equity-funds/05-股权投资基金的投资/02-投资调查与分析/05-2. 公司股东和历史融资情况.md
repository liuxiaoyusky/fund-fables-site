# 2. 公司股东和历史融资情况

## 🔍 寓言故事 —— 《老铺的旧账》

江南古镇上，林家酱园传了三代。小儿子林墨生两年前接了父亲的班，把酱园从两口缸扩成十八口，又新开一间门面。镇上的布商郑老板托人来问，愿出三万两银子入股占三成。林墨生又喜又忧——喜的是银子进账能再扩一条晒场，忧的是郑老板精明，签字前定要翻他的老底。

郑老板没亲自来，派了自家老账房钱四。钱四五十出头，提一只油布袋，袋里装着林家酱园三年的旧账、过契、单据。一进门便拱手："林少爷，您把家底给我晒晒。"

林墨生把账本捧出来。钱四没翻最新的那本，倒先抽最旧的那本。三年前，老股东周大胡子因病退股，把一成份额作价一千五百两卖给了林家。钱四指着过户契问："周大胡子那一千五百两，银钱是当面点清的吗？过户契盖的是镇公所的旧章，新章换过之后，有没有去补登？"林墨生一愣："那会儿我爹办的，我回头问问。"钱四不追问，本子上淡淡画了一笔——这一桩过户，还没去换新章。

钱四又翻到中间一摞："这一沓是什么？"是两年前镇上赵员外家入股的文书，投了五千两占两成，写明三年内若酱园再增新股，老赵有"优先跟投"的权利。钱四把文书摊平，问："赵员外这五千两，账上入了没有？出资凭证有没有？股东会决议、改过的章程，存不存？"林墨生脸红了："我爹走得急，这些……还没来得及拢齐。"

钱四合上账本，端起茶喝了一口："林少爷，我再多嘴一句——这三年缸数翻了一倍，又添了门面。添缸添门面用的银子，是从酱园账上走的，还是哪位新股东再投过？若有人投过，老章程里登记的份额、改过的名册，有没有一并改过来？"林墨生张了张嘴，答不上来——他想起开春那阵，找表弟挪了八百两垫缸钱，私下写了一张收条，连名目都没正经入账。

钱四起身告辞，把账本一本本码好还给林墨生："林少爷，我回去如实回郑老爷。入股这事，银子是一回事，您这家底里的旧账新账得先晒清。郑老爷的钱可以等，糊涂账不能等。"

林墨生送他到门口，看他提着油布袋走远。回到账房，他把周大胡子的旧契、赵员外的入股文书、自己那张皱收条，一样样摊在桌上。夕阳照在泛黄的纸页上，林墨生忽然明白——股东换了几茬、银子进进出出，账本上却没把这些脚印一一记牢。外人想踏进来，第一件要翻的就是这些旧页。

他坐下来研墨，铺开新纸写下"周大胡子旧契换章"六个字，又添一行："赵员外出资凭证、章程、名册补齐"。墨迹未干，他又翻出表弟那张收条，添最后一笔——"下月初二开股东会，议定表弟入股，先入账再议事"。

窗外暮色渐起，酱缸里的酱香飘进账房。林墨生知道，这三万两进不进来是一回事，但这些旧页若不先理清，下一个来翻账的人，看到的还是一笔糊涂账。

---

**📖 原文定义**

> 在目标公司出资方面，应关注公司股东及历史融资情况，具体包括股东出资金额、出资方式、出资期限、实缴情况等，并梳理目标公司的历次股权变更。如果目标公司经历过前轮融资，还应审阅相应的投资协议、公司章程、出资证明、股东会决议等相关文件，并判断交易是否存在瑕疵，例如：股权转让对价是否支付完毕，工商变更是否已经完成，目标公司是否存在未履行完毕的交割义务或尚未解决的与前轮投资人的争议等。
