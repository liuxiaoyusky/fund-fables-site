# 5. 资金借贷及对外担保

## 🔍 寓言故事 —— 《担保的影子》

江南水乡，织造坊一间连一间。老梁做了二十年丝绸生意，手下五十多个织工，去年刚翻新了染坊，银钱花得差不多了。今年开春，南边来的棉商陈四海找上门，要同他合伙把丝绸卖到南洋去，开口就要三万两入股。老梁心里盘算：这一笔若能成，明年的订单翻一番。

陈四海出手阔绰，请客吃饭、签字画押，一切都顺当得很。签约前夜，老梁叫来自己的小舅子、念过洋书的小柯，让他去把陈四海那本账册再翻一翻。小柯摊开账册，顺着"借款"那一栏看下去，越看眉头越紧。

"姐夫，"小柯指着账本说，"陈四海去年向孙记钱庄借了一万两，写的是一年为期。今年五月就该还的，眼下到期还有三个月，他账上银子不够。"老梁不以为意："我入的是三万两，又不是替他还债。"

小柯没应声，又往后翻了两页，把账本推到灯下："姐夫你看这两行——他给镇外赵家米铺、给县城周记绸庄、还给合伙的吴胖子，全做了担保人，加起来快有八千两了。哪一家出了岔子，他都得顶上。"

老梁手一抖，茶水泼在桌沿上："他怎么一个字也没跟我提？"小柯合上账本："我估摸着他自己心里也清楚，怕说出来您不入伙了。这债就像影子，白天看不见，落雨天就拖在地上一长条。"

老梁夜里翻来覆去睡不着，第二天一早便去找镇上的老账房许先生。许先生是见过世面的人，把账本接过去，一行一行看过，又问了些话，最后叹口气："陈四海的账面上还有六百两现银，可他要还钱庄的那一万两，加上三家担保的窟窿，最坏的情况全摊他身上，他这点家底连零头都填不上。您这三万两进去，等于给他垫了背。"

老梁把入股的事按下了。过了不到两个月，孙记钱庄派人上门催债，陈四海东挪西凑，把染坊的炉子都卖了；又过了半个月，赵家米铺出了事，他作为担保人，被债主堵在门口，一夜白了头。吴胖子来找老梁喝酒，席间苦笑道："幸亏你那天没进，进了咱们三个都得跟着翻船。"

老梁回家，给小柯倒了一杯酒："当初你翻那本账，我还嫌你多事。这账本背后看不见的债，比账面上的还吓人。"小柯摇头："我只是把他自己签过的字翻出来罢了。"

---

**📖 原文定义**

> 在法律尽职调查过程中，应审阅目标公司的重大借款合同，并梳理借款合同的当事方、借款金额、期限、利率等关键信息，关注目标公司是否存在逾期还款的风险。还应审阅目标公司对外担保相关文件，包括保证、抵押、质押等，并评估可能的影响。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 小柯翻看账册"借款"栏 | 审阅目标公司重大借款合同 |
| 陈四海向孙记钱庄借的一万两 | 借款合同的关键信息（金额、期限） |
| 小柯紧盯到期日与账上现银 | 关注逾期还款的风险 |
| 陈四海给赵家米铺、周记绸庄、吴胖子做担保 | 审阅对外担保相关文件（保证） |
| 担保金额合计近八千两 | 评估对外担保可能的影响 |
| 老梁叫小柯再翻一遍账本 | 法律尽职调查的核实动作 |
| 影子"白天看不见，落雨天拖地一长条" | 隐形负债不易察觉但真实存在 |
| 老梁按下不入伙的决定 | 识别风险后对投资交易的审慎判断 |
