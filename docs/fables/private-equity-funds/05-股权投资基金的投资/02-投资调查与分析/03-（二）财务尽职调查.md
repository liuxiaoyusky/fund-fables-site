# （二）财务尽职调查

## 🔍 寓言故事 —— 《查账帖》

清河镇老周商会的“江南织新”份子，决定投邻县“锦绣织坊”三千两银子。摸底小组的陈师傅和周先生已经查完了织坊的手艺和契约，轮到账房先生阿福上场——他要写一份“查账帖”，把锦绣织坊的银子来龙去脉查个底朝天。

阿福带着两个学徒，在锦绣织坊的账房里住了七天。他翻开锦绣织坊近三年的账本，逐页核对，逐笔勾稽。张锦绣站在一旁，起初还笑着，后来笑容渐渐僵了。

**第一查：记账规矩**

阿福先问：“张掌柜，您这账本是谁记的？”张锦绣说：“我侄子记的，他读过两年私塾。”阿福点头，翻开第一本——去年的账本。他发现：收入确认的方式，去年和前年不一样。前年，锦绣织坊是“货出门、钱到手”才算收入；去年改成了“货出门、客户签收”就算收入，不管钱有没有到账。阿福问：“为什么改？”张锦绣说：“我侄子说，这样显得生意好。”

阿福记下：记账规矩变了，但变的理由不充分。他按前年的规矩，把去年的收入重新算了一遍——去年账面收入一千二百两，按前年规矩，实际只有九百两。三百两的差额，是“货已出门、钱未到账”的虚增。阿福对学徒说：“记账规矩一变，数字就跟着变。咱们查账，先得把规矩统一，否则两年的数字没法比。”

他又发现：锦绣织坊把“织机折旧”算得很慢——一台织机用十年，每年只折一成。可行业惯例是六年折完，每年折一成七。阿福按行业惯例重新算，发现锦绣织坊的“净利”被虚增了约一百两。他记下：折旧规矩偏松，利润被高估。

**第二查：银子的真假**

阿福开始查“银子”——不是织坊里有多少现银，而是账面上各项资产的真假。

先看“应收账款”——别人欠锦绣织坊的银子。账面上写着：县城绸缎庄欠三百两、邻县布商欠二百两。阿福让学徒拿着账本，挨家去对。县城绸缎庄认账，但说“手头紧，年底才能还”；邻县布商却不认——说“那批货质量有问题，正在退货，这二百两不能算”。阿福回来对张锦绣说：“邻县那二百两，您得重新核实。若货确实退了，这二百两得从账上抹掉。”张锦绣额头冒汗：“我……我再问问。”

再看“存货”——织坊里堆的布。阿福带着陈师傅，逐匹查验。账面上写着：上等细布五百匹、中等粗布三百匹、次等布一百匹。可实际一查：上等细布只有三百匹，中等粗布四百匹，次等布两百匹。原来张锦绣为了账面好看，把一部分中等布算成了上等布，次等布算成了中等布。阿福按实际等级重新估价，存货总值比账面少了约一百五十两。

再看“固定资产”——织机。账面上二十台织机，每台估价八十两。陈师傅逐台查验：十八台新水力织机，确实值八十两；两台老木机，账面也写八十两，可实际只值二十两。阿福记下：固定资产高估一百二十两。

最后看“对外投资”——锦绣织坊在邻县一家染坊投了五十两。阿福一查，那家染坊去年已经倒闭，五十两血本无归。可账面上还写着“对外投资五十两”，没有计提损失。阿福记下：投资损失未确认，账面虚增五十两。

**第三查：银子的流向**

阿福翻开“现金流量”——不是账面上的利润，而是实际进出的银子。

去年账面净利五百两，可经营活动实际流出的银子却有三百两。阿福逐笔核对：买丝花了四百两、发工资三百两、交租一百两、还债二百两——合计一千两。收入呢？实际到账的货款只有七百两。五百两的账面利润，实际是“三百两的净流亏”。阿福问张锦绣：“您去年账面赚了五百两，可银子越花越少，这五百两去哪儿了？”张锦绣支吾：“有些货款还没收回来……”

阿福又查：锦绣织坊去年从钱庄借了两百两短期借款，今年到期。账面上“货币资金”只有一百两，可下个月要还两百两，缺口一百两。阿福记下：偿债能力紧张，若货款收不回来，可能资金链断裂。

**第四查：比一比**

阿福把锦绣织坊的“查账帖”数据，和镇上另一家“彩云织坊”对比——两家规模相近、都是投织坊的。彩云织坊的毛利率是三成五，锦绣织坊账面毛利率是四成——按阿福调整后的实际数据，毛利率只有二成八。彩云织坊的存货周转一年四次，锦绣织坊账面也是四次——实际只有两次半，次等布积压严重。彩云织坊的应收账款周转一年六次，锦绣织坊只有三次，回款慢。

阿福对老周说：“周掌柜，锦绣织坊的账面数字，和实际差得远。按账面算，值三千两；按实际算，最多值两千二百两。而且它的盈利能力、偿债能力、运营能力，都比同行差一截。”

**第五查：税和补贴**

阿福最后查了锦绣织坊的纳税记录。张锦绣说：“县衙去年免了我们一半税，说是扶持织坊。”阿福去县衙核实，确实——去年锦绣织坊享受税收优惠，少缴了约八十两税。可阿福问县衙文书：“这优惠能持续多久？”文书说：“只免去年一年，今年恢复全额。”阿福记下：去年净利里有八十两是“一次性优惠”，今年没了，净利要再减八十两。

阿福还查到：锦绣织坊去年收了一笔“县衙织坊补贴”五十两，账面上算成了收入。他去县衙问，文书说：“这补贴是专项的，只能用于买新织机，不能当利润分。”阿福记下：补贴有用途限制，不能算自由支配的利润。

七天查完，阿福的“查账帖”写了四十页。他对老周说：“周掌柜，锦绣织坊的账，像一件绣了一半的衣裳——正面看光鲜，翻过来全是线头。记账规矩不统一、资产估值偏高、回款慢、存货积压、偿债紧张、盈利能力比同行差、税收优惠和补贴不可持续。三千两银子投进去，按实际值，最多值两千二百两；而且它的持续经营能力有风险。”

老周拿着查账帖，对赵掌柜说：“阿福这七天，抵得上咱们三个月的打听。查账不是找茬，是让银子投出去之前，知道这件衣裳的里子是什么料子。”

**📖 原文定义**

> 财务尽职调查是尽职调查工作的重点环节，着重关注目标公司的历史财务业绩情况，并对企业未来财务状况进行合理预测。财务尽职调查团队应收集目标公司相关的财务报告及相关支持材料，了解其会计政策及相关会计假设，进行财务比率分析，重点考察企业的现金流、盈利及资产事项。财务尽职调查应涵盖企业的历史经营业绩、未来盈利预测、营运资金、融资结构、资本性开支以及财务风险敏感度分析等内容。财务尽职调查不同于审计，财务尽职调查强调发现企业的投资价值和潜在风险，注重对企业未来价值和成长性的合理预测，经常采用趋势分析和结构分析工具，在财务预测中往往会用到场景分析和敏感度分析等方法。
>
> 在股权投资基金投资运作流程中，财务尽职调查是开展投资方案设计、交易谈判、投资决策不可或缺的前提，是判断投资是否符合战略目标及投资原则的基础。它对了解目标公司资产负债、内部控制、经营管理的真实情况，充分揭示其财务风险，分析盈利能力、现金流，预测目标公司未来前景起到重大作用。
>
> 财务尽职调查需要重点关注如下几个方面：
>
> 1. 会计政策与会计估计：通过查阅公司财务资料，并与相关财务人员和会计师沟通，核查公司的会计政策和会计估计的合规性和稳健性。如公司报告期内存在会计政策或会计估计变更，重点核查变更内容、理由及对公司财务状况、经营成果的影响。会计政策与会计估计的选择会直接影响到后续财务相关数据计算的结果。在财务尽职调查过程中，应评估尽职调查期间的会计政策与会计估计是否发生重大变化，在进行各项比率分析前，应将尽职调查期间的会计政策与会计估计调整成相同的标准，从而规避会计政策与会计估计的影响，更准确地评估公司经营水平。
>
> 2. 财务报告及相关财务资料：财务报告核查及总体评价，包括对近年的资产负债表、利润表、现金流量表及相关财务资料的内容进行审慎核查。对于资产负债表，重点核查公司货币资金、应收款项、坏账计提、存货周转、对外投资、固定资产、无形资产、投资性房地产等资产项目的真实性，关注重要资产抵押情况、无形资产估值的合理性，核查对外担保等或有债务。对于利润表，重点核查收入确认、销售成本、期间费用核算的规范性，关注销售毛利率的波动性及非经常性损益对利润的影响。对于现金流量表，重点核查经营活动产生的现金流量及其变动情况，判断公司资产流动性、盈利能力及偿债能力。如果公司经营活动产生的现金流量净额持续为负或远低于同期净利润，应进行专项核查，并判断其真实盈利能力和持续经营能力。对于纳入合并报表范围的重要控股子公司的财务状况，应同样履行充分的审慎核查程序；对于公司披露的参股子公司，应获取最近一年的财务报告及审计报告。如公司最近收购兼并其他企业资产或股权，且被收购企业资产总额或营业收入或净利润超过收购前公司相应项目一定比例的，应获得被收购企业收购前一年的财务报表，核查其财务情况。财务尽职调查虽然与审计侧重点不同，但在尽职调查过程中，针对重要核查项目可以参照审计程序进行真实性和准确性的核查，以补充财务尽职调查在鉴证方面的不足。
>
> 3. 财务比率分析：盈利能力分析：计算公司各年度毛利率、总资产收益率、净资产收益率等，分析公司各年度盈利能力及其变动情况，分析和判断公司盈利能力的持续性。偿债能力分析：计算公司各年度资产负债率、流动比率、速动比率、利息保障倍数等，结合公司的现金流量状况、在银行的资信状况、可利用的融资渠道及授信额度、表内负债、表外融资及或有负债等情况，分析公司各年度偿债能力及其变动情况，判断公司的偿债能力和偿债风险。运营能力分析：计算公司各年度总资产周转率、存货周转率和应收账款周转率等，结合市场发展、行业竞争、公司生产模式及物流管理、销售模式及赊销政策等情况，分析公司各年度营运能力及其变动情况，判断公司经营风险和持续经营能力。综合评价：通过上述比率分析，与同行业可比公司的财务指标比较，综合分析公司的财务风险和经营风险，判断公司财务状况是否良好，是否存在持续经营问题。分析公司的各项财务指标不能脱离业务所处的市场环境与公司发展阶段，选择恰当的可比公司作为比较对象才具有意义。如果目标公司的某项财务比率大幅偏离行业平均水平时，应仔细核查该项比率的驱动因素，分析偏离原因。
>
> 4. 纳税分析：查阅公司的纳税资料，调查公司及其控股子公司所执行的税种、税基、税率是否符合现行法律、法规的要求及报告期内是否依法纳税。取得公司税收优惠或财政补贴资料，核查公司享有的税收优惠或财政补贴是否符合税收管理部门和财政管理部门的有关规定，调查税收优惠或财政补贴的来源、归属、用途及会计处理等情况，关注税收优惠期或补贴期及其未来影响。分析公司对税收政策的依赖程度和对未来经营业绩、财务状况的影响。
