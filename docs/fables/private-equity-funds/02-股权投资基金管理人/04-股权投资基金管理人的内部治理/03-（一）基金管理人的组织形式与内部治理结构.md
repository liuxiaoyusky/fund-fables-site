# （一）基金管理人的组织形式与内部治理结构

## 📜 寓言故事 —— 《两扇门》

江南古镇上，老周经营着一家钱庄，专门帮人打理银钱往来。几十年下来，生意越做越大，可老周心里始终悬着一块石头——钱庄里里外外就他一人说了算，管账的、跑腿的、看门的，全是听他吩咐。日子久了，账房先生老孙手脚不干净，挪了三百两银子去赌，老周竟半年后才发觉。

这年春天，老周把儿子小周从省城叫了回来。"爹想让你接手艺，可咱家这摊子，得改改规矩。"老周指着账本叹气，"一人独大，迟早出大事。"

小周在省城念过新书，见过大世面。他琢磨了三天，给爹出了两条路。第一条路，把钱庄改成"周氏银号"，按章程办事：老周当大东家，小周和几位老伙计当理事，再设一个监事专门盯着账目。大小事由理事们商议，监事有权查账，大东家也不能独断专行。第二条路，改成"周记合伙栈"，老周和小周做普通合伙人，对外跑生意、拿主意；另邀几位信得过的乡绅做有限合伙人，只出钱、不参与日常经营，但有权过问银钱去向。

老周听完犯了难。他问小周："就不能照旧么？"小周摇头："爹，镇上的规矩变了。如今要帮人管钱，只能是这两条路，别的法子行不通。"

老周带着小周去县城请教了相熟的师爷。师爷翻开律书说："周掌柜，您替人管钱，身份特殊。律法有定，只能走'公司'或'合伙'这两扇门，单枪匹马或者其他乱七八糟的架子，都不能开门做生意。进了哪扇门，就得守哪扇门的规矩。"

老周恍然大悟。原来不是自己想怎么改就怎么改，律法早就画好了两条道。他回到镇上，当众宣布：周氏银号走第一条路，设东家会、理事会、监事，三权分立，互相牵制。老周不再是说一不二的人，但他睡得踏实了。

三年后，周氏银号在邻县开了分号。老周庆幸自己推开了那扇对的门。

---

**📖 原文定义**

> 依据我国现行立法，基金管理人仅可采用公司型、合伙型两种法律组织形式，其他组织形式的机构以及自然人均不能受托作为基金管理人直接从事基金管理的活动。
>
> 与之对应，基金管理人因具体法律组织形式的不同而分别受到《公司法》《合伙企业法》及《中华人民共和国民法典》等法律的规制。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周的钱庄只能改"周氏银号"或"周记合伙栈" | 基金管理人仅可采用公司型、合伙型两种法律组织形式 |
| 自然人单枪匹马不能开门做生意 | 自然人均不能受托作为基金管理人直接从事基金管理的活动 |
| 进了哪扇门守哪扇门的规矩 | 因具体法律组织形式的不同而分别受到不同法律的规制 |
| 周氏银号设东家会、理事会、监事 | 公司型组织形式受《公司法》规制 |
| 周记合伙栈分普通合伙人和有限合伙人 | 合伙型组织形式受《合伙企业法》规制 |
