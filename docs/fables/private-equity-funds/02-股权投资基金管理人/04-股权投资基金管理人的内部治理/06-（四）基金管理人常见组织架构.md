# （四）基金管理人常见组织架构

## 🏛️ 寓言故事 —— 《老周的船队》

老周在运河边跑了二十年船，从一条小木船发展到十二条货船。生意大了，他一个人再也管不过来。去年冬天，三条船同时出事：一条装了瓷器却按粮食价收了运费，亏了大本；一条在暴风雨夜贸然出航，差点翻在江心；还有一条船的账，老周怎么也算不清，银子像漏进了暗河。

年三十晚上，老周独自坐在码头，看着江面上星星点点的船灯，心里发苦。他想起父亲临终前说的话："船大了，一个人掌舵，迟早要撞山。"

开春后，老周做了一个让同行惊讶的决定。他把十二条船分成几拨，每拨交给不同的人管，但谁也不许一个人说了算。

他找来了老赵。老赵跑船三十年，哪条航道有水匪、哪段江面有暗礁、什么季节该运什么货，门儿清。老周让他带着几个年轻后生专门"找买卖"——打听哪里缺米、哪里缺盐，谈价钱、定路线。但老赵谈成的每一单，必须拿给另一个人过目，不能自己拍板。

这个人是老孙。老孙年轻时在衙门里当过书办，为人死板，认死理。老周给他的差事就一件：盯着老赵谈成的买卖，看看有没有漏洞。比如货主靠不靠谱、价钱合不合理、路线安不安全。老孙有权否决任何一单，但老孙自己不能去谈买卖——他只能"挑刺"，不能"做主"。老周说："你俩一个往前冲，一个往后拉，船才走得稳。"

管账的是老周的外甥女小荷。她管着所有船的进出账目，每天记流水、算盈亏。但小荷只管记账，不经手银子。收来的运费交给码头上的老钱，老钱管着钱箱的钥匙，但他只管收付，不记账。月底小荷和老钱对账，差一文钱都要翻个底朝天。

还有阿贵，负责照应那些"搭船"的客人——有些商人把自己的货交给老周的船队运，他们最关心船到哪儿了、货安不安全。阿贵每天站在码头上，给这些客人报信，有问必答，从不嫌烦。老周说："人家把身家性命交给咱们的船，咱们得让人家睡得着觉。"

最后是老周自己。他不再跑船，每天坐在岸上的茶棚里，听着老赵汇报买卖、老孙汇报风险、小荷汇报账目、阿贵汇报客人的话。十二条船像十二条伸出去的手臂，但脑袋只有一个。老周不干涉具体的事，只干一件事：听。听完了，点头或者摇头。

有一回，老赵谈成了一笔大买卖，运一批丝绸去北方，利润丰厚。老孙审查后发现了问题：那批货要过一段江面，正值汛期，风险太大。老孙否决了。老赵气得拍桌子，骂老孙"胆小如鼠，挡人财路"。两人闹到老周面前。老周听完，只说了一句："老孙说得对。船沉了，利润再高也是别人的。"

老赵冷静下来，换了条路线，虽然少赚了两成，但货平平安安送到了。年底分红，老赵的赏钱一分没少——老周在账本里专门加了一条：挡灾的功劳，比赚钱的功劳大。

三年后，老周的船队从十二条变成了三十条，成了运河上最大的船帮。别的船帮羡慕他"命好"，老周只是笑笑。他知道，命好不如人好——让找买卖的人只管找买卖，让挑刺的人只管挑刺，让记账的不管钱、管钱的不管账，让搭船的客人有人照应，让自己坐在岸上听。每个人只做自己该做的事，谁也替不了谁，谁也管不了不该管的事。船队大了，不是船多了，是人对了。

**📖 原文定义**

> 基金管理人组织架构是其内部治理结构的直接体现。2016年2月，中国证券投资基金业协会颁布《私募投资基金管理人内部控制指引》，要求基金管理人组织结构应当体现职责明确、相互制约的原则，建立必要的防火墙制度和业务隔离制度，各部门有合理及明确的授权分工，操作相互独立。
>
> 对于基金管理人来说，实践中还应注意相关部门之间的衔接配合，使相关制度具有可操作性。一般情况下，规模以上的基金管理人的组织架构至少包括如下部门。
>
> 1. 投资部
>
> 较小规模的管理人通常只设置投资部，负责项目筛选、投资建议报告及投后跟踪管理等职能，并会将产业或行业研究工作纳入其中。而具备一定规模的管理人，可能会设置单独的研究部门。通常投资部主要职责包括：
>
> （5）组织对已通过审批项目的交易执行，执行投委会决议，落实投资审批决策意见；
>
> （6）对被投资企业开展投后管理及增值服务，定期提交被投资企业财务报表、投后管理情况报告，根据需要向被投资企业派出董事、监事等人选；
>
> (7) 发现风险后, 及时向风控合规部门提交重大问题报告等;
>
> （8）对拟退出项目，向投委会提交项目退出建议，按照批准的退出方案办理项目的退出手续；
>
> (9) 拟定项目管理的各项制度，经审批后执行；
>
> (10) 其他授权职能。
>
> 2. 风险控制与合规部（风控部）
>
> 风控岗是管理人必设岗位。具备一定规模的管理人通常会设置风险控制与合规部，其主要职责包括：
>
> 3. 投资者关系部
>
> 随着股权投资基金行业的发展以及信息披露要求的规范，管理人越来越重视与基金投资者的互动，并会设置专人或部门维护与基金投资者的关系。该部门主要职责包括：
>
> 4.运营管理部
>
> 基金管理人通常会设置后台支持部门，即运营管理部，主要职责包括行政事务管理、人力资源管理等职责。
>
> 5. 财务审计部
>
> 财务审计部的主要职责包括公司财务预决算规划、财务报告、记账、出纳、报税、资金管理、外部审计沟通等，以及基金产品对外投资的资金管理、财务核算等工作。
>
> 部分成熟或者大型的集团化管理人，其内部职能部门会更精细化、专业化，相关职责会进一步细分。以某大型基金管理人为例，前台投资方面，其根据管理基金产品类型（企业股权、并购基金、夹层基金、基建基金、美元基金等）区分工作单元（或部门），同时将行业研究、融资、业务拓展、投后增值服务等与投资直接相关的工作单元（或部门）一并纳入管理；中、后台运营及管理方面，区分为风险控制、合规、IT运营、综合管理、公共关系管理、投资者关系管理、行政支持等工作单元。

| 老赵找买卖、谈价钱、定路线 | 投资部负责项目筛选、交易执行、投后跟踪等职能 |

| 阿贵照应搭船客人、报信答疑 | 投资者关系部维护与投资者的关系、信息披露 |

| 记账的不管钱、管钱的不管账 | 业务隔离制度、各部门操作相互独立 |
