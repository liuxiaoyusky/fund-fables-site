# 二、基金管理人内部治理原则

## 🏺 寓言故事 —— 《六道门规》

清末汉口长街上有一爿"同德号"钱米铺，铺主陈老太爷是汉口数得上名号的大粮商。老太爷膝下三子皆无意经商，把铺子托给了远房侄孙陈守拙。

老太爷把守拙叫到后堂，推过来一面铜牌和一只小木匣："匣里有六张条子，是咱陈家三代人慢慢攒下来的门规。"

第一张写着："东家的事，永远摆在自家的事前头。"
第二张写着："朝廷的规矩，一行一字都不能越。"
第三张写着："大钱出手，先把最坏的可能想三遍。"
第四张写着："自家生意和铺子生意，不许在一张桌上谈。"
第五张写着："赚了多少、赔了多少、铺子里还剩什么，每季要给东家交一次底。"
第六张写着："我一个人说了不算——凡过千两的进出，都得过三个人才办得成。"

老太爷念完，把条子塞回匣子。"这匣子钥匙我收着，你只管照着做。做得好，铺子红利你拿三成。"

守拙接手头一年，铺子做得稳当。可到了第二年秋天，六桩麻烦接连上门。

头一桩，亲舅找上门借五百两周转。守拙没借——铺子是老太爷的，自家的事再大也是自家的事。

第二桩，对面"永丰号"压价抢客。守拙一查，永丰压过的是官府划定的底。他没跟："朝廷的线就在那，线下的钱赚一文都是祸。"

第三桩，有人兜售一批洞庭湖新稻，价格低三成。守拙差人去看货，到了渡口却停住——那条船没挂号旗，湖上风波莫测，遇上大水颗粒无收。他没下船："先想最坏，赔得起的钱才赚。"

第四桩最让人难受。守拙的连襟也在汉口做粮行，开春悄悄递话：把陈家几个老客户让给他，下半年分三成。守拙把茶碗往桌上一搁："这事咱俩下回再别谈。"他回去立了新规：自家人的铺子，一粒米也不许从同德号的账上过。

第五桩，年中盘账时老太爷的管家上门。守拙把放账簿、存货单、银钱流水一一摊开，连一笔三两的茶资都写在附页上。管家翻完赞一句："陈家三代人，没见过谁把账摆得这么清爽。"守拙答："东家把钱托给我，我藏着掖着对不住这把锁。"

第六桩，腊月同德号接到一笔三千两的大放账。守拙拿着契书，连夜跑了两家——先到老太爷府上，再拐到宋管家家，再拐到老伙计钱四爷家。三个人凑在堂屋里一笔一笔算，签字画押，翌日才办妥。

守拙跟伙计们说："东家把几千两银子搁在柜上，是信得过我。我一个人看得到的地方有限，得让几个人一起看，铺子才摔不了跟头。这六张条子不是给我立规矩，是给我搭台阶。"

老太爷八十岁那年，把钥匙交给了守拙。匣子里那六张条子已经翻得起了毛边，每一张边角都被人用朱砂小字批过——批的人是守拙自己。

"匣子交给你了，"老太爷说，"你再加两条新规，下一代用。"

守拙跪下接匣，叩了三个头。

---

**📖 原文定义**

> 不同组织类型基金管理人有着自身的特点和规律，这就要求所采取的治理方式、搭建的治理结构、构建的治理机制等应符合组织自身的治理逻辑和规律。
>
> 基金管理人作为股权投资基金的重要当事人，应遵循股权投资基金的一般治理原则；同时，结合其自身特性，进一步对基金管理人内部治理的原则要求进行说明（见表2-1）。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 陈老太爷把铺子托给守拙 | 治理机制要符合组织自身治理逻辑与规律 |
| 匣中六张条子作为"门规" | 内部治理原则是一系列正式的制度安排 |
| 第一张条：东家的事摆在自家事前头 | 投资者利益优先原则 |
| 第二张条：朝廷的规矩一行一字不越 | 守法合规原则 |
| 第三张条：大钱出手先想最坏 | 风险管理原则 |
| 第四张条：自家生意与铺子生意不在一桌谈 | 利益冲突防范原则 |
| 第五张条：每季向东家交一次底 | 信息披露透明原则 |
| 第六张条：大额进出过三个人才办 | 有效制衡原则 |
| 守拙把账摊开、三两茶资都写附页 | 信息披露要求全面、真实、及时 |
| 老太爷把钥匙交给守拙 | 良好的内部治理保障相关者利益最大化 |
