# （三）合伙型基金管理人的内部治理结构

## ⚖️ 寓言故事 —— 《两桌人》

光绪二十一年春，徽州绩溪县商人胡雪岩的远房族侄胡文清，到上海滩筹办一家做丝绸买卖的字号。比起那些"几兄弟凑钱、一个人说了算"的旧式商号，胡文清这一回摆了两桌。

第一桌坐着胡文清和三位族兄——四人凑了大本钱，管字号里外事务的是胡文清。第二桌坐着另外九位远亲近邻——他们银子出得不少，可都不懂丝绸生意，有的在外埠做茶叶，有的在老家教书，只盼着年底分些红。

开张那天，胡文清把"合伙契书"摊在桌上。九位出钱人里有个做茶叶的章老板，翻了翻契书问："文清，契上写你们四位是'做事的人'，我们是'只出钱的人'。万一你们背着我们把字号卖了，我们连坐的地方都没有？"

胡文清笑了笑："章兄讲得有理。可'规矩'怎么定，咱们十三个一起议。"

议了三日，议出三条新规。

第一条：九位"只出钱的人"里挑三位，组个"议事堂"，凡字号与自家人做的买卖、借出银子过三百两的——都得议事堂先议。议了不算，可议事堂若说"不妥"，胡文清就得停一停。

第二条：再请一位"老娘舅"——不是九位里头的，也不是胡文清这一桌的，是徽州商会里有名望的赵老掌柜，专管挑刺。每月翻一次账，遇着可能跟自家人相关的买卖、有拿不准的账目，赵老掌柜有权叫停，再让十三个合伙人一起议。

第三条：日后四人里头有人想退、新人想加入，得十三人里过三分之二的人头点头。

章老板合上契书："这三条，契书上可有？"

胡文清答："契书上没写。咱没旧例，那就一边做一边立，立下的就是咱的例。"

开张头一年，徽州府送来一批好蚕丝，要价不低。胡文清拍板：进。章老板在议事堂里一翻账——那卖丝的商号，恰好是胡文清连襟开的门脸。三位议事堂的兄弟把胡文清请来。胡文清脸一红，没辩，把这笔买卖停了，另找了苏州的丝行。

又一年，胡文清的族兄想借钱号三百两应急。这话传到了议事堂。议事堂议了一回，回话给胡文清：自家人的事，不能挂在字号的账上。胡文清把族兄劝了回去，自己掏腰包借他。

赵老掌柜每月底翻一次账，翻完提点：有两笔跟"自家人"擦边的进出，要写得更清楚。胡文清照办，账本上多了一行注脚。

两年下来，字号在租界小有名气。年底分账，章老板拿着一份分红，感叹："当初说咱是'只出钱的人'，怕是只能干坐着看。如今看下来——议事堂和老娘舅，比那四位做事的人看得还清。"

胡文清端起酒碗："两桌人凑在一起做买卖，靠的不是哪一个人，是那几条议事、挑刺、记账的规矩。规矩不是天上掉下来的，是十三个脑袋一次次吵出来的。"

章老板点头："吵得明明白白，分得也明明白白。"

那年冬月，胡文清把议事堂和老娘舅的规矩补进了"合伙契书"——十三个人的手印，一个不差。

---

**📖 原文定义**

> 不同于公司成熟的内部治理结构，《合伙企业法》并未明确内部组织结构如何设置，而是充分尊重商事主体之间的契约自由，允许创设类似公司的内部治理机构。
>
> 合伙型基金管理人可借鉴公司治理成熟经验，依据法律规定和自身情况设置合适的内部治理结构。例如，可设置有限合伙人委员会或顾问委员会审查内部利益冲突事项，可设置风险管理委员会负责内部合规和风控事项。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| "两桌人"——胡文清四位做事的人与九位只出钱的人 | 合伙型基金管理人中"管事者"与"出资人"两种角色的区分 |
| 契书未明文规定内部组织结构、靠众人商议立规 | 《合伙企业法》未硬性规定内部组织结构、充分尊重契约自由 |
| 章老板翻契书问"我们出钱的连个坐的地方都没有" | 出资人（即有限合伙人）对内部决策的知情与参与诉求 |
| 九位出资人推三位组成"议事堂" | 有限合伙人委员会（或顾问委员会），审查内部利益冲突事项 |
| 三百两以上大额对外签约须议事堂先议 | 议事堂对内部利益冲突事项的审查权 |
| 请徽州商会赵老掌柜任"老娘舅"、专管挑刺 | 风险管理委员会，负责内部合规和风控事项 |
| 议事堂叫停胡文清连襟开的丝行那笔买卖 | 议事堂审查内部利益冲突的具体运作 |
| 议事堂拒绝族兄借用字号三百两 | 防范"做事的人"将私事挂公账 |
| 议事规则按十三个人的"合伙契书"约定而非法律硬性规定 | 合伙型基金借鉴公司治理成熟经验、依据自身情况设置合适结构 |
| 冬月将议事堂和赵老掌柜的规矩补进合伙契书、十三个人的手印一个不差 | 治理结构依合伙协议约定并经全体合伙人确认的契约安排 |
