# （四）基金管理人常见组织架构

## 🏺 寓言故事 —— 《永丰号分柜》

永丰镇的郑老大，做了二十年的粮行生意，一口铺子打天下，从收粮、记账、跑客户到请客吃饭，全是他一个人带着两个伙计忙活。可这两年行情变了，镇外几家大商号找上门来，要把银子托给郑老大代为打理，做点田产、铺面的长线投资。生意一来，郑老大的眉头反而锁得更紧。

银子一多，麻烦就来了。账上短短半年功夫，就涌进了三万多两，加上原来自己粮行的进出，账房刘先生每天从天不亮拨算盘，拨到日头落山，账本堆得比人还高。有一回，刘账房核到一笔货款，发现和他三天前记的数差了一钱银子，急得一头汗，把当天的伙计叫来对了半宿，最后才发现是伙计把"两"记成了"钱"。郑老大坐在堂屋听了这事，半天没说话。

更让他坐不住的是一桩事。前些日子，镇东李家布庄的少东家想把自己新开的染坊折价入股永丰号，郑老大觉得这事能谈，可要命的是，前脚李家少东家刚走，后脚他就听说镇西孙记窑场的赵掌柜也来打听过，想用窑场抵一部分银子进来。要是两边谈成了，那永丰号就同时掺着布庄和窑场的账，刘账房一个人怎么分得清谁的钱到了哪里、谁该分多少红、谁的田产该归到哪一笔账下？更要命的是，万一哪天李家少东家问起"我那笔银子到底投到田产上还是铺面上"，刘账房连自己都答不上来，那麻烦就大了。

郑老大连着几夜没睡踏实。第四天清早，他把刘账房叫到堂屋，把算盘往桌上一放，说："老刘，咱这摊子不能再这么糊下去了。得拆开。"刘账房抬头看他，没吭声。郑老大站起来走到墙边，抄起一根炭条，在白墙上画了几道杠："我琢磨过了——管钱的、管投资的、管看风险的、管和外面东家打交道的、再加上管我们自家后院的，每一块都立起来一个人或者一个小班子，账分开算，事分开做。哪笔银子进来、出去、投到了哪块田、归到哪个东家名下，从进门那一笔到出门那一笔，全要分得清清楚楚，谁也碰不到谁的线。"

他回过头来对刘账房说："你不是只会拨算盘，往后专门管账、管税、管银子的进出，这块就叫账房部。我再请一个人，专门去外面看田产、看铺面、看哪桩买卖能做、哪桩不能做，这叫投资部。再请一个，专门替我把每桩买卖翻来覆去看一遍，看看里头有没有坑、有没有对不住外面东家的地方，这叫风控部。再立一个，专门跟外面那些把银子托给咱们的东家打交道，告诉人家钱投到了哪、现在赚了多少、明年打算怎么办，这叫跟东家接头的部。最后那些七零八碎的杂事——门口迎来送往、屋里茶米油盐、人手够不够使、招不招新人——也立一个人顶起来，这叫后院部。"

刘账房听完，把算盘往自己怀里一收："东家这主意成。早该这么干。"郑老大哈哈一笑："那往后，咱们这铺子就再不是从前那个一人包打天下的永丰号了。"

没过两年，永丰号果然又接了两笔大生意，镇上再没人说他们"账不清、责不明"。郑老大站在分好的柜面前，看着五个柜台上各自忙得井井有条，心里一块石头总算落了地。

---

## 📖 原文定义

> 基金管理人组织架构是其内部治理结构的直接体现。中国证券投资基金业协会颁布的《私募投资基金管理人内部控制指引》要求，基金管理人组织结构应当体现职责明确、相互制约的原则，建立必要的防火墙制度和业务隔离制度，各部门有合理及明确的授权分工，操作相互独立。规模以上的基金管理人的组织架构至少包括：投资部、风险控制与合规部、投资者关系部、运营管理部、财务审计部。

## 💡 对应点

| 故事元素 | 概念对应 |
|---------|---------|
| 郑老大决定"不能再这么糊下去" | 规模扩大后，原有"一人包打天下"模式难以为继，必须分部门 |
| 郑老大在墙上画几道杠、立五个班子 | 基金管理人常见组织架构的五个核心部门：投资部、风控部、投资者关系部、运营管理部、财务审计部 |
| 刘账房"专门管账、管税、管银子进出" | 财务审计部的职责——公司财务、记账、出纳、报税、资金管理 |
| "再请一个专门去看田产铺面、哪桩买卖能做" | 投资部的职责——项目筛选、投资建议、投后跟踪 |
| "专门替我把每桩买卖翻来覆去看一遍有没有坑" | 风险控制与合规部的职责——审查项目合规与风险，是必设岗位 |
| "专门跟外面把银子托给咱们的东家打交道" | 投资者关系部的职责——维护与基金投资者的关系、信息披露 |
| "七零八碎的杂事也立一个人顶起来" | 运营管理部的职责——行政事务、人力资源等后台支持 |
| "账分开算，事分开做，谁也碰不到谁的线" | 职责明确、相互制约、防火墙与业务隔离制度的通俗表达 |
| 镇东李家、镇西孙记同时入股时的混乱 | 投融资规模扩大、多方资金进入后，分部门的必要性 |
