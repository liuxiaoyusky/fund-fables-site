# （二）公司型基金管理人的内部治理结构

## 🏛️ 寓言故事 —— 《四方桌》

老宋从木器铺子退下来以后，闲不住，和三个老朋友凑了一笔钱，在城东开了家做扇子买卖的铺子。铺子要立规矩。四个人围着一张四方桌坐下，谁管钱、谁管事、谁盯账、谁拍板，得分清楚。

第一年还算顺当。第二年春天，麻烦来了。铺里进了一批上好的湘妃竹扇，正赶上端午前后，本该赚一笔。哪知掌柜老周一拍脑门，把价钱压到了成本价——他私下收了卖家一笔"茶水钱"。老宋知道时，扇子已卖出去一半。他去找老周问个明白，老周反倒拿出一张写着自己名字的"进货单据"来挡："这事儿我说了算。"

另两位股东坐不住了。开茶馆的钱大姐端着茶碗说："铺子是咱四个人凑的银子，怎么能由着他一个人胡来？"隔壁的陈老爹闷头抽着旱烟，半晌才说："我出钱的时候就说好了，按出钱多少说话。他出得最多，咱们吵不过他。"四个人你看看我，我看看你，谁也拿谁没辙。

老宋回家想了一夜。第二天他把四个人请到城隍庙旁边的茶楼上，把一张新画的图纸铺在桌上——图上画着一张四方桌，四个角各写了一个字：股东会、董事会、监事会、经理。

老宋指着图纸慢慢说："股东会是咱们四个出钱人坐的地方，大事在这里举手，按事先定好的规矩数人头，谁都不能一个人说了算。董事会呢，是从咱们中间推两三个懂行的人出来，专门拿主意、订计划、看过账。经理是真正站在柜台后面招呼客人的那一位，由董事会挑人，董事会说要换人就换人。监事会更简单，再推一个既不挑担子、也不站柜台的人，专门盯事——查账、问话、发现问题就喊停。"

他停了一下，看着老周："老周，你心细、懂行情，我推你进董事会。但进多少货、定什么价，得让董事会一起议，不能你一人点头。这不是你一个人的铺子。"

陈老爹的烟杆顿了一下："那他要是背着咱们拿回扣呢？"

"监事会盯着他。"老宋指了指第三个角，"钱大姐不爱站柜台，眼睛又尖，就请你当监事。进货出货的条子、账上的进出，每个月给她过一遍。她要是查出有来路不明的钱装进私人腰包，董事会就得叫停，老周要担责。"

钱大姐拍了下桌子："成。"老周沉默了一会儿，把玩着手里那把折扇，叹了口气："是我莽撞了。行。"

那年端午，湘妃竹扇没再降价。四个人按新规矩办了一次进货：股东会先议方向，董事会拿出具体方案，掌柜照方案去办，钱大姐月底翻账，账目清楚，分毫不差。到了年关，铺子头一回给四个人按约定分了红，老周也分到了一份——虽然没他私下拿的多，但是光明正大。

第二年端午，铺子门口挂了新匾，上面是老宋请人写的四个字：各司其职。

---

**📖 原文定义**

> 公司型基金管理人可区分为有限责任公司、股份有限公司两种组织形式。在公司传统的"三会一层"法定治理结构中，决策权、经营管理权、监督权分属于股东会、董事会、监事会。基金管理人可视需要设置其他内部机构，还可以通过公司章程设置其他约定职责，并赋予相应的权利。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老宋等四个出钱人围坐的"股东会" | 权力机构，重大事项由股东按规则表决 |
| 老宋提议设的"董事会"（推举懂行的人议事） | 经营决策机构，拿主意、订计划 |
| 站在柜台后面管买卖的"掌柜" | 经理层，执行董事会决议、负责日常经营 |
| 钱大姐专管查账的"监事会" | 监督机构，专门盯事、对抗内部人擅自做主 |
| 老周一人压价、收受"茶水钱"的冲突 | 出资人监督缺位下经理层的机会主义行为 |
| 老宋在城隍庙茶楼重新画图纸定规矩 | 通过章程明确分权与制衡的内部治理安排 |
| 第二次进货：议方向→出方案→执行→查账 | 决策机制与监督机制协同运行的流程 |
