# （二）基金管理人与相关主体的区分

## 📜 寓言故事 —— 《三顶帽子》

老周在城里做了二十年钱庄掌柜，经手的银钱流水数不清，口碑一直很好。去年秋天，他两个老主顾——绸缎商赵老板和茶叶商钱老板——找上门来，说想凑一笔银子做一桩大买卖：收购江南几家濒临倒闭的织坊，改造后转手卖出。两人各有五十万两，但都不懂织坊经营，更没工夫天天盯着，便请老周出面张罗。

老周想了想，答应了。三人约定：赵、钱二人只出钱，不参与日常经营；老周负责找织坊、谈价钱、改造后寻买家。赚了钱，三人按约定分账。这桩买卖，老周戴的是"掌柜"的帽子——替两位东家跑腿办事，凭的是经验和眼光。

事情一开始很顺利。老周跑遍江南，挑中三家织坊，谈好价钱，签了契。可到了要过户改契的时候，问题来了。按当地规矩，三人合伙做买卖，得去衙门登记一个"合伙字号"，字号里得有个"管事人"，专门负责对外签字、跑衙门、应付税赋。赵老板和钱老板都是甩手掌柜，这管事人只能是老周。

老周心里犯了嘀咕。他一边要忙着看织坊、盯改造，一边还得应付衙门里的琐碎事。更要命的是，他听说邻县有个掌柜因为只管事、不掌钱，被人钻了空子，最后里外不是人。老周越想越不安：自己现在既是替东家经营买卖的"掌柜"，又是合伙字号上登记的"管事人"。这两顶帽子，一顶靠的是本事，一顶靠的是名分。万一出了岔子，他到底该听谁的？

腊月里，老周去省城拜访一位退隐的老前辈，把这困惑说了。老前辈听完，捻着胡子笑了："小周啊，你这买卖里，其实有三顶帽子，不是两顶。"

老周一愣："三顶？"

"第一顶，"老前辈竖起一根手指，"是你替两位东家经营买卖的'掌柜'帽。这顶帽子看的是你的本事——你会不会挑织坊、懂不懂改造、能不能卖好价钱。两位东家信你，才把这买卖交给你。"

"第二顶，"他又竖起一根手指，"是合伙字号上登记的'管事人'帽。这顶帽子是衙门给的，对外签字、应付官差、税赋往来，都得这顶帽子出面。没有这顶帽子，字号就不合法。"

"第三顶呢？"老周追问。

"第三顶，"老前辈竖起第三根手指，"是字号里的'东家'帽。你们三人合伙，按规矩，管事人得同时是字号里的出资东家之一，对外承担无限责任。赵老板、钱老板只出钱、不管事，他们是'有限东家'；你既然管事，就得是'无限东家'——买卖亏了，你的家当也得赔进去。"

老周恍然大悟。原来这三顶帽子可以戴在一个人头上，也可以分给不同的人。比如，他可以只做"掌柜"，另找人做"管事人"和"无限东家"；或者他既做"掌柜"又做"管事人"，但让儿子来做"无限东家"。但老前辈话锋一转："要是'掌柜'和'管事人'不是同一个人，或者不是一家人，那就有麻烦了。管事人不懂买卖瞎签字，掌柜没名分说了不算。买卖做砸，东家找谁算账？"

老周回城后，把赵老板、钱老板请来，把这番道理摊开说了。三人商量后决定：老周三顶帽子一起戴——既是掌柜、又是管事人、也是无限东家。这样权责清楚，谁也挑不出毛病。年后，三家织坊改造完毕，转手卖了个好价钱。赵老板和钱老板数着银子，直夸老周办事利索。老周却心里清楚：这桩买卖能成，不是因为自己本事大，而是因为他从一开始就把三顶帽子分清楚、戴明白了。

**📖 原文定义**

> 《私募条例》规定，在中华人民共和国境内以非公开方式募集资金，设立投资基金或者以进行投资活动为目的依法设立公司、合伙企业，由私募基金管理人或者普通合伙人管理，为投资者的利益所进行的投资活动，均视为私募投资基金业务活动受规制；以合伙企业形式设立的私募基金，资产由普通合伙人管理的，普通合伙人也适用《私募条例》关于私募基金管理人的规定。
> 由于境外部分国家股权投资基金行业多采取有限合伙制，基金管理人经常会兼任合伙型基金的普通合伙人或执行事务合伙人，即使在不兼任情况下，两者通常也属主要管理团队出资设立的关联机构。因此，国外立法及行业实践更强调普通合伙人作为基金运营者的核心作用，而一定程度上淡化了基金管理人的作用，或者以普通合伙人指代基金管理人。实践中，基于对境内外监管规则的不同理解，业内经常将基金管理人与基金普通合伙人、执行事务合伙人混同使用，甚至以"GP团队"指代基金管理人团队，但是在法律意义上，基金管理人、普通合伙人、执行事务合伙人存在严格区分。
> 境内股权投资基金采用有限合伙企业形式设立的情形较为普遍。合伙型基金主要依据《中华人民共和国合伙企业法》（以下简称《合伙企业法》）规定设立，由基金管理人或其关联方作为普通合伙人负责执行合伙事务。根据《合伙企业法》规定，有限合伙企业中仅能由普通合伙人执行合伙事务。
> 根据中国证券投资基金业协会发布的相关自律规则，基金管理人设立合伙型基金，应当担任执行事务合伙人，或者与执行事务合伙人存在控制关系或者受同一控股股东、实际控制人控制。从法律上，基金管理人、普通合伙人、执行事务合伙人既可以三者竞合，也可能出现基金管理人与普通合伙人、执行事务合伙人分离的情形。为防止基金管理人通道化，基于前述规定，如果合伙型基金的基金管理人不担任执行事务合伙人，应与执行事务合伙人存在强关联关系。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周（负责找织坊、谈价钱、盯改造） | 基金管理人（负责投资管理、运营） |
| 合伙字号登记的"管事人"（对外签字、应付衙门） | 执行事务合伙人（执行合伙事务） |
| "无限东家"（承担无限责任） | 普通合伙人（GP，承担无限责任） |
| 老周三顶帽子一起戴 | 基金管理人、普通合伙人、执行事务合伙人三者竞合 |
| 老周只做掌柜，另找人做管事人和无限东家 | 三者分离的情形 |
| 老前辈强调"掌柜"和"管事人"若不是同一人或一家人，买卖会出乱子 | 基金管理人不担任执行事务合伙人时，应与执行事务合伙人存在强关联关系（防止通道化） |
| 赵老板、钱老板只出钱不管事 | 有限合伙人（LP） |
