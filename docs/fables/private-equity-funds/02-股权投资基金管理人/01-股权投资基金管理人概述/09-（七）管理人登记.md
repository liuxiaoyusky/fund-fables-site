# 《万事俱备 还得挂号》

## 开场

王老板把七本章程本本立在柜上，长出一口气——人手有了，本钱有了，七本章程也凑齐了。他拨通十六铺桥头吴记茶馆的电话："**老吴，人手、本钱、章程全齐了，我这便挂匾开张——**"

老吴把半个脑袋探进茶馆："**王老板——这桩事急不得。同业公会的告示上写得明白：字号万事俱备，最后一桩是到京城总舵挂号。号没挂上，匾不能挂、门不能开。**"

王老板一愣："**我这里万事俱备，还差什么？**"

老吴翻告示："**差一张挂号票。**"

## 冲突

王老板皱眉："**我在十六铺桥头立了字号，难道还得跑京城？**"

郑老大搁下茶盏："**不是跑京城——是到京城总舵那里挂号登记，把你字号的底子报上去，让总舵那边知道永丰镇十六铺桥头新开了一家做合股契买卖的字号。**"

王老板松了口气："**那我明日便动身。**"

孙先生敲铜印："**王老板——挂号之前，你得先把字号的本钱、东家、请的人、章程一样一样报上去。总舵那边的会规写得一清二楚：本钱够不够、东家干不干净、人手齐不齐、章程立没立，都要对。一项对不上，挂号就过不去。**"

王老板问："**若是对上了呢？**"

孙先生念："**对上了，总舵那边给你一张挂号票，字号才算正式立上号。**"

王老板又问："**若是我先挂了匾、开了门，再去挂号呢？**"

老吴摇头："**那便是坏了规矩——总舵那边的会规上写得明白：号没挂上，凑份子不能凑，合股契的家当不能管。**"

## 转折

王老板沉吟半晌："**也就是说——我得先把字号里里外外收拾齐了，再到总舵挂号，号挂上了，才能挂匾、开门、凑份子、管家当？**"

郑老大点头："**正是这个次序。前头所有的人手、本钱、章程，都是为了挂号这一日。这一日过不去，前头全白忙。**"

王老板又问："**那我自己定的字号的规矩，跟总舵那边的会规，哪个算数？**"

郑老大摆手："**两桩都得守——字号自个儿定的规矩守一份，总舵那边定的会规也得守一份。总舵那边的会规是立号的底线，字号自个儿定的规矩是字号自个儿怎么走。**"

王老板再问："**若是我立号之后，同业公会改了会规呢？**"

孙先生敲铜印："**改了你也得照新规矩走——会规是总舵那边定的，改了便照新的来。**"

王老板咬咬牙："**那我今夜把字号里里外外再过一遍：本钱够不够、东家干不干净、人手齐不齐、章程立没立、字号自个儿的规矩跟总舵那边的会规对一对——对完了，明日动身去京城总舵挂号。**"

郑老大点头："**这就对了——万事俱备，还得挂号。号挂上了，才能开门。**"

## 结局

王老板把七本章程、一本花名册、一本账本装进包袱："**那我这便动身。号挂上了，回十六铺桥头挂匾；号没挂上，回十六铺桥头再改。**"

孙先生把铜印递过来："**带上我的印——挂号时要验。**"

刘账房拨算盘："**带上我的账本——挂号时要查。**"

许先生从盘货铺子过来："**带上我的秤——挂号时要称。**"

郑老大起身："**万事俱备，还得挂号——号挂上了，匾能挂、门能开、份子能凑、家当能管；号没挂上，一桩都不能动。这就是同业公会立号的规矩。**"

老吴咂茶："**一句话——字号开张前最后一桩，是到京城总舵挂号登记。号挂上了，才是立上了号；号没挂上，凑份子不能凑，合股契的家当不能管。**"

---

**📖 原文定义**

> **（七）管理人登记**——基金管理人应当根据相关规定，向**中国证券投资基金业协会**申请管理人登记。基金管理人登记环节，应当满足相关**自律规则**要求的条件。在完成基金管理人登记**之前**，**不得开展**基金募集、基金管理等业务。

**💡 对应点**

| 故事元素 | 概念对应 |
|---|---|
| 字号到京城总舵挂号登记 | 基金管理人向中国证券投资基金业协会申请管理人登记 |
| 京城总舵 | 中国证券投资基金业协会 |
| 总舵那边定的会规 | 自律规则 |
| 字号的本钱、东家、人手、章程报上去 | 基金管理人登记应满足自律规则要求的条件 |
| 号挂上了才能挂匾、开门 | 完成管理人登记是开展业务的前提 |
| 号没挂上，凑份子不能凑，合股契的家当不能管 | 在完成管理人登记之前，不得开展基金募集、基金管理等业务 |
| 万事俱备还得挂号 | 管理人登记是基金管理人开展业务的最后一道关 |
