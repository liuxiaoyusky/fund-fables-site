# （一）管理人名称、经营范围和业务范围

## 🏺 寓言故事 —— 《老吴开张》

民国二十三年春天，老吴在天津卫劝业场后街租下一栋小洋楼，挂起牌匾要开张做买卖。

这十几年，他在租界里帮人挑铺子、做账房，把"谁家生意好、谁家底子厚"摸得门儿清。手里攒下几户老主顾的信任，又跟租界洋行学了几手看账的法子，便琢磨着自立门户，替那些手里有闲钱、想投实业却不懂行的老主顾们掌眼。

头一桩难题——牌匾上写什么？

老吴跑去商会问陈先生。陈先生戴着瓜皮帽，翻出薄册子念给他听："你若替人选铺子、帮人造园子，牌上只许写明'房产经纪'、'置业顾问'之类字样；你若替人选铺子、帮人造园子，还替人选股、做证券安排，那就另有一套说法。你是哪种？"

老吴愣了一下。他原本盘算：既替人选实业铺子，也替人选股票债券，反正都是"替人选产业"。可陈先生摇头："那不行。帮人选实业铺子，你得先把'管实业'四个字写明白；帮人选股票债券，那是另一种买卖——牌匾、账本、报备，全是另一套。你若两样都做，牌子上写不写？怎么写？将来出了事，老主顾看匾，是找你算'实业'的账，还是算'股票'的账？"

老吴回家琢磨了三天。他先把准备写"吴氏金融总行"那种大而笼统的牌匾撤下来——这种字眼唬人，可将来老主顾出了岔子，自己这牌子就成了"误导人"的把柄。陈先生私下递过来的册子他翻烂了，里面写得清楚：挂什么匾，做什么买卖，两头对得上；挂一行的匾、想两行的钱，迟早自己绊自己。

老吴把自己要做的事理了一遍：说到底，他只擅长"替人选实业铺子、帮实业铺子长大"这一桩。选股票、选债券的活儿他既不内行，也没那本事同时盯盘。他把这一行琢磨透、写明白，让所有来找他的人一看匾就明白"这人是干什么的"，就够了。

想明白这些，他重新拟了牌匾，正中四个字："**吴氏实业管顾**"——陈先生看过后点头，又在匾的右下角加了一行小字注脚，写明"专司实业铺子之代选、代管、代顾"。陈先生说："你这匾，将来就是你的边界。写的是什么，就只能做什么；没有写的，就不要伸手。伸手做了，匾上的字也保不住你。"

开张那天，老主顾冯四爷亲自来贺。冯四爷看了匾问："老吴，你这'吴氏金融总行'的牌匾呢？我记得你上回说要挂这五个字，怎么改了？"老吴指着新匾说："四爷，这五个字多好听，可它什么都是，也就什么都不是。我只管'实业'这一桩，写得明明白白，您看一眼就知道我是干什么的；将来我若做了什么对不起您的事，商会也拿这匾上的字跟我对账。"

冯四爷愣了一下，拍腿大笑："好小子！不挂那唬人的大牌子，反倒让老哥我放心了。走，进去喝茶，咱把第一桩买卖好好说。"

门帘掀开，老吴的新买卖，正式开张。

---

**📖 原文定义**

> 按照相关法律法规与自律规则要求，股权投资基金管理人作为私募基金管理人的一种，应当在名称中标明"私募基金""私募基金管理"或"创业投资"字样，并在经营范围中标明"私募投资基金管理""私募股权投资基金管理""创业投资基金管理"等体现行业特点的字样。未经批准，不得在名称中使用"金融控股""金融集团""中证"等字样，不得在名称中使用与国家重大发展战略、金融机构、知名私募基金管理人相同或者近似等可能误导投资者的字样，不得在名称中使用违背公序良俗或者造成不良社会影响的字样。
>
> 从业务范围来看，股权投资基金管理人应体现专业化经营特征，聚焦投资管理主业，可围绕股权投资基金管理开展资金募集、投资管理、为被投资企业提供管理咨询等业务，但不得从事任何与股权投资基金相冲突或无关的业务。此外，股权投资基金管理人不能同时从事证券投资基金管理业务。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老吴在牌匾上只能写明所经营的事 | 名称与经营范围须标明"私募基金""私募基金管理"等体现行业特点的字样 |
| "吴氏金融总行"这种笼统唬人的大牌子被撤下 | 不得使用"金融控股""金融集团"等可能误导的字样 |
| 牌匾右下角注明"专司实业铺子之代选、代管、代顾" | 经营范围须与实际业务一致，体现专业化经营 |
| 老吴主动放弃同时做"实业 + 股票债券"的打算 | 不得同时从事与主业相冲突或无关的业务 |
| 老吴只擅长"替人选实业铺子"，便只挂牌做这一行 | 聚焦主业、专一经营 |
| 冯四爷看匾一眼就明白老吴是干什么的 | 名称与经营范围须清晰反映真实业务，便于识别 |
| "匾上的字就是你的边界" | 名称与经营范围即机构的法律身份与业务边界 |
| 将来老主顾找上门对账，商会拿匾上的字说话 | 名称与经营范围作为合规依据约束实际经营行为 |
| 老吴重新拟匾、写明所司之事、放弃兼营股票债券 | 专业化经营的体现——有所为、有所不为 |
