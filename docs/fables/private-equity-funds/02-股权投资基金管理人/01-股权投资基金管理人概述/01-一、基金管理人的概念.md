# 一、基金管理人的概念

## 🏺 寓言故事 —— 《江南钱庄的老掌柜》

晚清年间，江南柳镇周家老爷子过世，留下一笔现银、几间铺面，分给三个儿子——周老大、周老二、周老三。分完家不到半年，三家都犯了难：周老大的孩子要娶亲，周老二要翻新屋舍养老母，周老三想南下进一批好料子。铺面被外地大作坊挤垮，三家各自的银子都办不成事。

那年正月，三兄弟围坐在老宅堂屋的八仙桌前，桌上摊着算盘和账本，谁也不先开口。

面前是两条路。

第一条：合在一起，由一房自管。可周老大说"我做过进货我熟"，周老二撇嘴"你花起钱来大手大脚"，周老三闷声不响，心里嘀咕"交给你们，我那份迟早花光"。第二条：合在一起，托给外人。可镇上做买卖的多如牛毛，谁也信不过谁。

沉默半晌，周老大先开口："镇东'孙记酱料行'的孙伯，做了四十年买卖，眼睛毒、人稳当，从不坑人。就托他。"

周老二眉头一挑："外人？万一他卷款跑了呢？"

周老大从袖里摸出一张契纸——原来他早有盘算："钱不放进他柜上，他人碰不着；三家共推一人每日记账；动用须三家画押。赚了按比例分，亏了也按比例分。"周老二、周老三翻来覆去看了一遍，对视一眼："成。"

孙掌柜是周老大三顾茅庐请来的。他只问清"托付、规则、分账"三件事，便从柜台底取出一本厚账簿，对三兄弟说："你们把钱交到我手里，是把身家托付给我。我替你们把这笔钱用起来，去找能让它长大的路。南北水路旱路，我做了四十年，熟。"

那一年，孙掌柜拿了合银南下苏州进好黄豆，又北上天津寻老作坊。秋天回来，原先的银子多了三成。三年下来，三兄弟各分到的，比自己守着铺面等客上门，翻了将近一番。

孙掌柜未多取一分。他拿的是契纸上写明的固定"辛苦钱"。他对三兄弟说："每一分钱的来去都记在这本账上，可查可问。"

第四年开春，三兄弟给老宅匾额重新上漆。匾额下那本合订账簿封面写着「周氏合业」四个字。镇上人都说："周家那三个笨兄弟，亏得找对了人。"

---

**📖 原文定义**

> 股权投资基金管理人通常指具有专门的知识与经验，依据法律规定及基金合同约定，受托管理基金财产，致力于通过股权投资等方式实现基金财产增值，并努力为基金投资者获取收益的专业机构。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 孙掌柜四十年买卖经验 | 专门的知识与经验 |
| 周家三兄弟托付银子、孙掌柜接受 | 受托管理基金财产 |
| 南北奔波寻货赚差价 | 通过合适方式实现财产增值 |
| 按比例给三兄弟分钱 | 为基金投资者获取收益 |
| 契纸与每日账簿 | 基金合同约定的受托关系 |
| 孙掌柜只拿固定"辛苦钱" | 专业机构的受托人定位 |
