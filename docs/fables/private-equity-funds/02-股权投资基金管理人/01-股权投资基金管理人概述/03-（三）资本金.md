# （三）资本金

## 🏺 寓言故事 —— 《老钱的铺子》

老钱在县城东街开铺子做了十来年买卖，攒下一笔家底。眼看闲钱越攒越多，他心里痒得很——想替别人打理钱财，自己也赚一份辛苦钱。街坊都知道他精明，老周便给他出主意：去码头那边新开的"账房街"租间门脸，挂个招牌，也做个管钱的主儿。

老钱乐颠颠跑去打听，一路上心里盘算：家里有几千两银子，铺子里几车布匹，城外良田二十亩也都是自家名下。拿这些撑撑门面，总能让街坊放心吧？

[冲突]

到了账房街筹备处，管事的小吏翻着册子问他："您打算拿什么来打底子？"老钱胸脯一拍："那可多了——布匹十二车、良田二十亩、城北一处带院子的老宅，还有刚花重金买下的几幅字画。"他一样样报出来，越说越得意。

小吏没点头，只是把册子翻到一页念给他听："这事儿要稳当，您得拿现银子出来，账上实实在在放着一笔。少一千两不行——哦，按新规矩，得一千万钱起步。而且得是真金白银摆在柜上，不能拿货物、字画、宅子抵。"老钱愣住了，脸上的笑意一点点僵住。

他扭头就去找老周诉苦："我这些东西样样值钱，加起来何止千万？凭什么不认？"老周没比他多懂多少，只是把茶杯一放，问他一句："你替人管钱，人家图个啥？"老钱没接上话。老周又说："人家把钱交给你，图的正是稳稳当当。你手里若是一堆布匹字画，万一行情变了、田地被收走，你拿什么赔给人家？"

[转折]

老钱从老周那儿回来，在自家院里坐了一整夜。月光从窗棂里漏进来，铺子里的布匹堆得整整齐齐，城北的租约躺在桌角，可他一想到要把这些折成"现银"，心里就发虚——哪一样能马上变成钱？哪一样变起来不打折扣？他越想越明白：自己手里值钱的东西不少，但能立刻"撑住场面、扛住风险"的东西，一样都没有。

第二天一早，老钱揣着地契去了钱庄。他忍痛把城外十亩良田折成现银入了账，又把铺子里压了半年的货款一笔笔收回来，前前后后忙了两个月。柜上终于稳稳当当摆着足够数目的现银，账目清清爽爽，他才敢再踏进筹备处。

小吏这回只看了一眼数目，点了点头："成。"老钱反而不放心，追问一句："往后我手里有了闲钱，再添几车布匹行不行？"小吏摇头："不行。进来时是什么，往后还得是什么——只认现银，不认别的。"

[结局]

老钱的门脸后来在账房街稳稳开了张。街坊愿意把闲钱交给他打理，一来是看他精明，二来也图个踏实。铺子里别的掌柜偶尔拿货物字画去做生意，老钱从不掺和——他心里始终记着那晚月色下的领悟：值钱的家当再多，也得有一笔"硬通货"压在柜台上，那才是替人办事的底气。后来有人问他开铺子的秘诀，他只笑笑说："手里得先有那笔让人放心的钱，别的都是锦上添花。"

---

**📖 原文定义**

> 基金管理人的出资人应以合法自有资金出资，注册资本及实缴资本均应当符合相关规定，确保有足够的资本金保证机构有效运转。依据相关规定，基金管理人应当财务状况良好，实缴货币资本不低于1000万元人民币或者等值可自由兑换货币，对专门管理创业投资基金的基金管理人另有规定的，从其规定。
>
> 基金管理人的资本金应当以货币出资，不得以实物、知识产权、土地使用权等非货币财产出资。境外出资人应当以可自由兑换的货币出资。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老钱想拿布匹、字画、良田去"撑门面" | 实物、知识产权、土地使用权等非货币财产 |
| 小吏只认"现银子摆在柜上" | 资本金应当以货币出资 |
| 一千万钱起步的硬杠杠 | 实缴货币资本不低于 1000 万元人民币 |
| 老钱连夜想通"货物再值钱，变不成现银也撑不住场面" | 资本金"保证机构有效运转、抵御风险"的作用 |
| 老钱忍痛卖田、催款，把账上补足 | 出资人应以合法自有资金出资、确保实缴到位 |
| 小吏说"进来时是现银，往后还得是现银" | 资本金形式（货币）的持续性要求 |
| 境外的另算 | 境外出资人应当以可自由兑换的货币出资 |
