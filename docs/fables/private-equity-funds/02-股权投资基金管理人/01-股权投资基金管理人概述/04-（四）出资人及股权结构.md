# （四）出资人及股权结构

## 🏺 寓言故事 —— 《三层套娃》

钱塘江边的老巷子里，赵三娘开的「诚信钱庄」要扩店面，街坊们都替她捏一把汗。她手里有一笔闲钱，可女儿阿芸要嫁到外地去陪嫁，镇上李裁缝的布料款还欠着，铺子要上新柜台又得花一笔。钱从哪里来？她翻来覆去想了一夜。

她先去找表姐孙婆。孙婆二话不说，从柜底摸出一袋银子：「拿去使，年后还我就行，利息好商量。」赵三娘刚要接，又停下了——她想起钱庄立下的规矩：「柜台上的钱，得是自家缸里舀出来的水，借来的不能倒进缸里。」表姐的钱虽是真心，可借来的水混进自家缸，缸里就说不清哪瓢是自家的了。她把银子推回去：「姐，欠了您的情我领，可这钱我不能这么使。」

她又去邻镇找大财主周半城。周半城一出手就是三十袋银子，但条件写得密密麻麻——赵三娘要在钱庄的招牌上加上「周记」二字，柜台、库房、伙计都归周半城管，赵三娘只管收钱时在旁边站一站。她看着那张契书，越看越糊涂：明明是我出钱庄，怎么反倒要听人吆喝？更要命的是，周半城还牵线另外两家钱庄，说大家绑在一起「互相帮衬」——这头出钱、那头管事、那头又托这家持股，绕来绕去像只三层的套娃，谁都说不清最里头到底是谁。

那天夜里，赵三娘坐在空荡荡的铺子里，听着屋檐的雨声发呆。婆婆临走时的话忽然浮上来：「钱庄的活招牌，是那个站在柜台后面、眼睛一眨不眨的人——别人眼里的钱庄，永远是开这间店的人的脸。你若借了别人的脸挂在自己招牌上，街坊日后记不住赵三娘，只记住那张脸了。」赵三娘鼻子一酸，把周半城那张契书慢慢撕碎。

第二天一早，她把自家的田契抵押给族里，换来一笔干干净净的银子；又把临街那间空屋租给布商三年，预收了租金。她站在柜台后面，把这笔记在自己那本旧账簿上：「此笔为家中旧产所换，自家缸中水。」街坊们来存钱、借钱，看见的都是她实实在在的家底和数得清的几间屋。镇上的茶馆里再聊起诚信钱庄，大伙儿都说：「这家啊，是赵三娘自己的——赔也是她，赚也是她，咱信得过。」

女儿阿芸临行前来辞别，看见娘亲一身轻装站在柜台后，笑了笑：「娘，您店里没那么多弯弯绕绕，挺好。」赵三娘也笑了，摸了摸招牌上的字：「是啊，水清，招牌才立得住。」

---

**📖 原文定义**

> 基于组织类型差异，基金管理人的出资人包括公司的股东、合伙企业的合伙人。依据相关规定，基金管理人的股权结构应当清晰，不得以委托资金、债务资金等非自有资金出资，不得违规通过委托他人或者接受他人委托方式持有股权、财产份额，不得存在循环出资、交叉持股、结构复杂等情形。基金管理人的实际控制人及主要出资人一般不应当是资产管理产品。
>
> 此外，基金管理人的股东、合伙人以及股东、合伙人的控股股东、实际控制人，控股或者实际控制其他基金管理人的，应当符合中国证监会的规定。前述规定主要包括相关主体财务状况、治理结构、信用记录、潜在利益冲突、经验匹配等方面的要求。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 赵三娘拒绝孙婆的借款 | 不得以债务资金等非自有资金出资 |
| 赵三娘撕掉周半城那张牵涉三家钱庄的契书 | 不得存在循环出资、交叉持股、结构复杂等情形 |
| 周半城在招牌上挂「周记」并管事 | 不得违规通过委托他人或者接受他人委托方式持有股权、财产份额 |
| 婆婆说「别人眼里的钱庄是开这间店的人的脸」 | 股权结构应当清晰（实际控制人与出资人可识别） |
| 赵三娘用自家田契和自家房屋租约换钱 | 出资人应以合法自有资金出资 |
| 街坊们认定「赔也是她，赚也是她」 | 实际控制人对机构承担清晰、可追溯的责任 |
| 阿芸说「店里没那么多弯弯绕绕」 | 禁止以资产管理产品作为实际控制人或主要出资人 |
