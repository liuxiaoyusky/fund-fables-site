# （一）基金管理人在基金中的角色

## 🏺 寓言故事 —— 《空宅的管家》

清末民初，杭州城西的青石板巷尾有一座三进老宅，门楣上挂着「陆氏宗业」四个褪金的字。街坊都管它叫"陆家空宅"——老宅账上堆着七十二家铺子的股份和两千亩水田的田契，可宅子里偏偏看不见一个正经管事的陆家人。陆家三代单传，老太爷过世后，少爷陆文清常年住在上海读书，宅里就剩一个看门的老仆。

有一年春天，陆文清回杭奔丧，远房堂叔陆仲勋找上门来。堂叔早年在洋行做账房，前年自立门户，开了一家叫"仲信"的小号，专门替大户打理家业。他坐下喝了一口茶，慢悠悠地说："文清，你家这门宗业，没人盯着收租，没人去铺子查账，更没人去衙门报契纳税。照这么下去，七十二家铺子的掌柜各自为政，佃户三年没换约，明年再涨一次厘金，连本带利都得漏光。"

陆文清皱眉："堂叔的意思是——"堂叔摆手："我不是来劝你卖祖产的。我只想问你：陆家这宅子，要继续当一座空宅，还是要让空宅里站进一个人？"

陆文清抬眼看向那三进深院。院子空得能听见风穿过回廊，账房落了灰。他忽然明白——这座"陆氏宗业"本身没有手脚、没有眼睛。它只是挂在祖辈名下的壳子，里面装着的铺子、田产、银子，要是没一个人替它去走、去问、去盯，就只是一堆慢慢朽掉的字据。

堂叔继续说："我可以替你担起这件事。我不是你的族人，不分你的家产，按月拿一份规银，每一笔进项都登册子，每年岁末把账本摊在你面前听你核。铺子掌柜听我调度，佃户找我换约，衙门里报契纳税的事我走。你只管做你的陆家少爷，不必困在这空宅里——但陆家的东西，陆家始终说了算。"

陆文清沉默良久，问："堂叔若是亏了、或是私吞了——"堂叔笑："白纸黑字立契约，每一笔有据可查；若亏了，我赔；若私了，公堂见。这事靠的不是情面，是规矩。"

那年秋天，堂叔领着三个账房先生搬进了老宅东厢。门房的老仆嘀咕：东厢这几位是亲戚还是租客？堂叔不解释，只在门楣下新挂一块小木牌：「仲信代管」。从此，这座空宅真正活了过来——秋收，堂叔亲自带人去佃户家过秤；每季末，他让账房把七十二家铺子的流水誊一份送到上海给陆文清过目；铺子要修灶、佃户要借粮，他先垫后报，从不拿本金放私债。

三年后，陆文清学成归杭。堂叔把三年大账摊开，分毫不差。陆文清举杯："堂叔这三年辛苦了。"堂叔摇头："辛苦的不是我，是这座空宅——它没长腿，却走完了七十二家铺子和两千亩田的每一寸路。我只是替它走了一趟。"他终于明白：空宅本身不会自己跑、不会自己收、不会自己查；可它依然是"陆氏宗业"——因为有一个不在他家谱上的人，按规矩替他把一切打理得井井有条。空宅还是那座空宅，里面却有了魂。

**📖 原文定义**

> 随着行业的发展以及基金管理人群体的崛起，股权投资基金更多地被视为"投资载体"或"投资工具"，除非法律有特别要求，基金内部并不配备常设人员负责基金经营管理活动。对于基金和基金投资者而言，基金管理人作为基金的运营者承担受托人的角色，其首要责任是为投资者的最佳利益行事。基金管理人将提供运营基金所需要的服务，或者组织其他的服务提供方来提供相应的服务。前述服务通常包括：
>
> 1. 投资管理服务
>
> 投资管理是基金管理人可提供的核心服务内容，是基金管理人赖以生存的基本技能。投资管理包括潜在投资标的的获取、调查、分析与评估，投资组合的构建，以及对被投资企业的投后管理和退出等。
>
> 2. 基金运营服务
>
> 基金合法有效的存续有赖于基金管理人提供必要的运营管理。基金运营服务包括投资者事务、工商及税务相关事项、信息披露与报送、基金估值与核算、其他日常行政管理等。
>
> 基金管理人可依法依规聘请符合条件的第三方服务机构提供部分基金运营服务。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| "陆氏宗业"老宅本身空着、却装着七十二家铺子和田产 | 基金作为"投资载体" / "投资工具"，本身不配备常设人员 |
| 陆文清常年不在家、老宅空置 | 基金内部不配备常设人员负责经营管理活动 |
| 堂叔陆仲勋带着仲信号搬进东厢、挂"仲信代管"木牌 | 基金管理人作为独立运营者承担受托人角色 |
| 堂叔不姓陆、按月拿规银、私吞要吃官司 | 基金管理人与基金相分离、独立运作 |
| 堂叔替陆家盯铺子、过秤收租、查佃户 | 投资管理服务（潜在标的获取、调查、投后管理） |
| 堂叔把流水誊送到上海给陆文清过目、每年摊大账 | 信息披露与报送 |
| 堂叔跑衙门报契纳税、垫款后报 | 基金运营服务（工商及税务、日常行政管理） |
| 堂叔可以请三个账房先生来帮忙 | 基金管理人可组织其他服务提供方 |
| "白纸黑字立契约，每一笔有据可查" | 为投资者的最佳利益行事的受托人责任 |
| 空宅还是空宅、却有了魂 | 基金管理人使"基金"这个载体真正运转起来 |
