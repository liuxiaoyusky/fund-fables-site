# （六）基本管理制度

## 📜 寓言故事 —— 《钱庄的墙》

清朝光绪年间，江南水乡的周巷镇上有一家老钱庄，掌柜叫沈万山。沈掌柜做了二十年钱庄生意，街坊邻居都信他。有一年开春，邻镇的赵大户主动找上门，要把自家三千两银子托付给沈掌柜打理，约定年底分三成利。

赵大户走后，沈掌柜把儿子沈砚叫到账房："砚儿，咱钱庄要做这门生意，得先立一整套墙。"沈砚不解："爹，咱是钱庄，做生意就是开张收钱，关门数钱，立什么墙？"沈掌柜摇头："钱庄的银子是别人的，别人的银子进了咱的门，就是一道大麻烦。"

麻烦很快就来了。开张头一个月，沈砚的表弟阿贵来借钱，说家里翻修瓦房差三十两。沈砚一抬手就从柜上支了。第二天，东家托付的银子里又有一笔要进项，沈砚和自家的钱混在一处记账，结果一时算不清哪些是赵大户的、哪些是钱庄自己的，他自己都差点绕进去。账房先生老吴黑着脸告诉他：照这么下去，年底分账时说不清，公堂上见。

沈砚慌了，连夜去敲沈掌柜的卧房门。沈掌柜披衣起来，坐在油灯下，提笔就写。他从柜上取出一摞空白账簿，一本一本地题字：第一本叫"进门簿"，谁的钱进、什么时候进、什么约定，白纸黑字。第二本叫"去处簿"，钱往哪走，投了哪桩生意，签的什么凭据，一项不落。第三本叫"自己账"，钱庄自家的柴米油盐、掌柜的工钱、伙计的开销，单独立账，绝不混进客账。第四本叫"规矩册"，家里谁管钱、谁审账、谁来盯风险、阿贵的借条属于什么情况怎么处置、白纸黑字写清楚。

写完四本，沈掌柜又把沈砚拉到院里，指着正堂和库房之间那道砖墙说："明日叫人把这道墙再加高三尺，开两个小窗，账房和库房从此不走同一个门。钱庄的归钱庄，客人的归客人，两边的钥匙分两个人管。"沈砚愣了半天："爹，您这是把家拆了？"沈掌柜笑："傻孩子，把家拆了，才能立得起家。"

半年后，镇上连降大雨，米价飞涨，隔壁镇有家钱庄的掌柜因为借钱给自家亲戚补窟窿，被债主堵了门，库房里的客人银子被自家借条啃得只剩一半。沈砚听着外面的传闻，再回头看看自己钱庄那道高墙，每一本账都清清楚楚，每一把钥匙都各归其主，那颗心才一点点放下来。年底结算，赵大户来分账，沈砚把账本摊开，进项去处一样样念完，分毫不差。赵大户点头离去时，回头看了一眼那道墙，说了一句话："沈家这门生意做得下去，是因为这道墙在。"

沈砚把账本合上，望向正堂方向。他没说什么，但从此再没让阿贵从柜上借过钱。

---

**📖 原文定义**

> 基金管理人应制定并执行一系列基本管理制度，制度内容应涵盖基金募集、投资管理、关联交易与利益冲突管理、基金信息披露、财务管理、内部控制、档案管理等。
>
> 基金管理人应保持自身与所管理基金的独立性。管理人与所管理基金之间、管理人管理的不同基金之间，不得存在财产混同现象。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 沈掌柜题字的"进门簿""去处簿" | 募集、投资管理制度 |
| 阿贵借钱被记入"自己账"并单独处置 | 关联交易与利益冲突管理 |
| 沈砚和赵大户的分账讲明进项去处 | 基金信息披露 |
| "自己账"与"客账"分立 | 财务管理 |
| 钱庄的钥匙分两个人管、墙加高 | 内部控制 |
| 四本账簿一摞空白纸上白纸黑字 | 档案管理 |
| 正堂与库房之间那道加高的砖墙、两把钥匙 | 管理人自身与所管理基金的独立性 |
| 隔壁钱庄被亲戚借条啃掉客人银子的下场 | 财产混同的反面警示 |
