# （二）基金管理人的勤勉义务

## ⚖️ 寓言故事 —— 《老周的船队》

老周在海边打了三十年鱼，是十里八乡公认的"船老大"。去年，村里二十多户凑钱买了艘新船，大家一致推举老周掌舵。老周拍着胸脯说："放心，这片海我闭着眼睛都能走。"

开春第一趟，老周按老规矩凌晨出海。新来的年轻人小林递上一叠纸，说是最近海流变化的数据，建议改走东线。老周摆摆手："我在这漂了三十年，哪条路线有鱼、哪片暗礁要避，全在脑子里。东线？绕远路，油钱谁出？"他坚持走西线老航道。结果那天西线渔获寥寥，回来的油钱都快抵不上收成。小林没吭声，但老周心里有点堵。

第二趟出海前，小林又拿来一份东西，说最近风向变了，东线水温升高，鱼群正在往那边迁移，还画了一张简单的海图。老周扫了一眼，把纸往旁边一推："年轻人，书本上的东西，海里不一定管用。"他还是走了西线。这次更糟——西线不光没鱼，还碰上了突如其来的大风。船在浪里颠了三个钟头，网具损毁了一半，有个老乡还扭了腰。老周浑身湿透地坐在船舱里，看着小林默默用备用绳修补网具，一句话也没说。

回港后，老周把自己关在屋里想了两天。第三趟出海前，他破天荒把大家叫到一块，让小林把海流图摊在桌上。老周指着图问："东线这片，你真去过？"小林点头："我跟我爹的小船去过三趟，鱼确实多，但暗礁也密，得有人专门瞭望。"老周沉吟半晌，说："行，这趟走东线。小林，你站船头瞭望；老张，你管轮机，随时听我口令调速度；我掌舵。"他又让另一个年轻人每天记录水温、风向，回来交给他看。

那趟东线之行，渔获是之前的两倍。更关键的是，船队安全回来，网具完好，人人有分工，没人闲着。从那以后，老周每次出海前都让小林先分析海况，自己再拍板路线。船队渐渐形成了规矩：出航前看数据、定路线、分岗位，归航后复盘记录。一年后，二十多户乡亲的分红比往年多了四成。有人私下问老周："您当年不是最烦年轻人指手画脚吗？"老周望着海平面，说："海是活的，人也得跟着活。光靠我一个人拍脑袋，对不起把家底交给我的人。"

---

**📖 原文定义**

> 勤勉义务要求基金管理人在行使投资决策和基金资产处置的自由裁量权过程中，应综合考虑基金目的、风险、期望收益等多个因素，秉持专业、审慎原则作出最符合基金投资者利益的决策。
>
> 在勤勉义务履行过程中，专业人士标准是最为重要的内容之一。专业人士标准，是指受托人应当具有与其受托目的和领域相适应的专业技能和专门知识，并使用这些专业技能和专门知识为受益人利益服务。股权投资基金管理人作为受托人，有义务按照基金运营目的和约定投资领域，配备专业的投资管理团队，从而为其勤勉义务的履行奠定专业基础。
>
> 勤勉义务，强调基金管理人应当为实现基金运营目的、维护投资者利益而不懈努力和发挥其专业能力。如果说团队专业性强调的是"能力"，那么"努力"就意味着基金管理人需要把这种"能力"付诸实践。一方面，在投资管理中努力寻找价值增值的机会；另一方面，要恪守注意义务，审慎进行投资风险管理。这就要求股权投资基金管理人建立专业化、市场化、规范化的投资管理体系，并确立制度化的内部激励和约束机制，以确保团队的专业性得到充分发挥并持续优化。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周受托带领乡亲出海捕鱼 | 受托人接受委托管理财产 |
| 老周凭个人经验拍脑袋走西线 | 缺乏专业审慎的决策方式 |
| 小林提供海流数据和海图 | 专业技能和专门知识的支持 |
| 老周让小林站船头瞭望、老张管轮机 | 配备专业团队，分工协作 |
| 老周综合分析海况后选择东线 | 综合考虑多个因素作出决策 |
| 出海前看数据定路线、归航后复盘记录 | 建立规范化的管理体系 |
| 老周从"我一个人说了算"到"让小林先分析、自己再拍板" | 将专业能力付诸实践，不懈努力和发挥专业能力 |
| 船队安全归来、渔获翻倍 | 维护投资者（乡亲）利益的结果 |
