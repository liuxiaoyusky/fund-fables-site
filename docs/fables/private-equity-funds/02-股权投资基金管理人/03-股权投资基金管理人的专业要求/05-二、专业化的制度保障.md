# 二、专业化的制度保障

## 📜 寓言故事 —— 《铁匠铺的规矩》

青石镇上有家铁匠铺，铺主姓周，人称老周。老周打了一辈子铁，手艺没得说，镇上锄头坏了、菜刀钝了，都找他。铺子开了三十年，口碑一直不错。

可老周有个毛病——全凭手感。徒弟问他："师傅，这刀要淬几遍火？"老周摆摆手："看颜色，红了就行。"问他镰刀弧度多少合适，老周说："凭眼瞧。"铺子里没有一本账，没有一张图，老周脑子里装着所有事。徒弟们跟着学，也是各凭各的感觉，有人淬三遍，有人淬五遍，打出来的东西时好时坏。老周不觉得这有问题，反正"以前不也这么过来了"。

去年冬天，事情变了。镇东头新开了一家铁器行，掌柜姓马，是从府城回来的。马掌柜的铺子一开张就接了县衙的大单子——给全县的衙役打制佩刀。老周不服气，托人去打听，回来的人说：马掌柜的铺子里有规矩，每一道工序都写在纸上，淬火的温度、捶打的次数、成品的重量，样样有数。徒弟进门先学规矩，不是先学手艺。老周听了嗤之以鼻："打铁的靠规矩？那还要老师傅干什么。"

转过年来，县衙又招标，给各乡的学堂打制课桌椅。老周信心满满地去应标，结果又输给了马掌柜。这一次，老周坐不住了，他揣着一壶酒，夜里去了马掌柜的铺子。

马掌柜正在灯下看一本厚厚的册子，见老周来了，笑着让座。老周开门见山："马掌柜，我打了三十年铁，手艺不比你差，为什么总输给你？"

马掌柜合上书册，给他倒了一杯酒："周师傅，您手艺好，我知道。可县衙选咱们，不是只看一把刀利不利。他们要看的是——您这铺子，能不能一百把刀都打得一样好。"他指着墙上的册子说，"我这铺子，从选料到交货，每一步都有人记、有人查。淬火不是看颜色，是计时。徒弟不是跟我学'感觉'，是学规矩。规矩定了，谁走都不怕，谁来都能上手。"

老周端着酒杯，半晌没说话。他想起自己铺子里，去年最能干的大徒弟去了邻县，带走了大半手艺，剩下的人打出来的东西质量就往下掉。他也想起，有回给张员外打的菜刀，徒弟淬少了火，用了半个月就崩了口，张员外找上门来，他赔了三倍的钱，却说不清到底是哪一步出了错。

酒喝到第三杯，老周放下杯子，长叹一声："马掌柜，我懂了。手艺是我的，可规矩是铺子的。没有规矩，手艺传不下去，人也管不住。"

三个月后，老周的铺子变了样。墙上挂起了工序册，每样活计从第几步到第几步，写得清清楚楚。他让二徒弟专门管查验，不让他碰锤子——"你只管看，别人打得对不对，你说了算。"老周自己还打铁，但不再凭感觉了，炉边放了沙漏，淬火按时间来。年底，县衙再招标，老周终于中了。签完文书回来，他摸着铺子里那本崭新的册子，对徒弟们说："以前我以为，规矩是捆人的绳子。现在才知道，规矩是撑船的篙，没有它，船再好看，也到不了对岸。"

**📖 原文定义**

> 股权投资基金管理人专业化的制度保障主要通过法律法规和自律规则以及管理人内部制度两个维度来实现。
>
> 法律法规和自律规则均明确要求基金管理人遵循专业化运营原则。从业人员应当恪守职业道德和行为规范，具备从事基金业务所需的专业能力。管理人的高级管理人员和员工应具备与其职位相匹配的专业胜任能力和相关工作经验，同时持有相应的从业资格证书，定期参加合规和专业能力培训。
>
> 基金管理人应依据法律法规和自律规则的要求，建立一套全面的风险管理和内部控制体系。该体系需涵盖资金募集、投资决策、运作管理、信息披露等关键环节，并制定相应的详细业务流程与操作规范。同时，管理人应加强对员工的合规和专业能力培训，以提高员工的专业能力和风险意识。在所有要素中，尤为重要的是建立一个有效的投资管理体系、考核评价与激励约束机制。
>
> 基金管理人需要建立起覆盖项目投资全流程的投资管理体系，包括行业研究与项目开发、项目筛选、尽职调查、投资决策、投后管理、项目退出等环节。完整的投资管理与投资决策流程以及完善的内部控制体系，对于保证基金管理人专业审慎具有重要的价值。例如，从内部控制角度来看，保持风险控制团队与投资团队的相对独立性至关重要，对项目风险的评估不应当完全由投资团队完成。否则，即使投资团队具有较高的专业水平，也可能因为利益关系导致其不能作出对基金投资者利益最佳的行为。
>
> 对基金管理团队实施有效的考核评价和激励约束机制，有利于激发团队的积极性，促使团队把专业能力更好地发挥出来。从考核评价机制来看，对股权投资基金管理团队的考核评价，应结合股权投资行业的特点进行。第一，股权投资基金的项目投资更多地表现为投资组合。由于股权投资市场风险相对较高，单个项目的投资损失并不必然表示投资管理能力或努力的不足，因此，在单个项目层面，应当建立起某种"容错"机制，考核评价的重点应当是投资组合即基金的整体表现，而非单个项目的业绩表现。第二，股权投资是长期投资，股权投资基金是"耐心资本"，基金投资收益的实现通常经历一段时间。因此，在投资收益考核中，应侧重对基金长期投资收益的考核。第三，在基金的不同阶段，基金管理活动的重点也不同。例如，在基金募集设立期，基金管理人的主要工作是完成基金的募集与交割；在基金投资期，基金管理人的主要工作是高质量完成对外投资；在基金退出期，基金管理人主要侧重于项目投后管理与退出、基金收益分配、基金清算等。不同阶段的考核评价，应当与各阶段的基金管理工作重点相适应。从激励约束的角度来看，对股权投资基金管理团队的激励和约束，主要通过团队持股、团队跟投、绩效奖励、业绩报酬分享、损失分担等方式实现。理论分析和行业实践均表明，科学有效的考核评价与激励约束机制对于提升股权投资基金管理团队的能力发挥具有显著的正向作用。

| 故事元素 | 概念对应 |
