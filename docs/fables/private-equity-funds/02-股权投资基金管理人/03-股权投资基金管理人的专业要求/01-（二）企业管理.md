# （二）企业管理

## 🏺 寓言故事 —— 《周掌柜的茶山》

民国二十三年的春天，浙江衢州的茶山遭了一场倒春寒。茶商周敬尧站在"敬记茶号"后院石阶上，盘算着一笔更远的账。

他打算把祖上传下来的茶山翻一翻新。手头三百两银子，够买下一座荒了两年、却坐落在好山场上的旧茶园——原来的东家去年过世，两个儿子一个教书、一个在省城做账房，都不接手。

周敬尧琢磨的是：买下来容易，把它"管"出样子来才难。

他请来两个老友一起掂量。一位是做过布行大掌柜的老方，一位是跑过洋行生意的何先生。三人坐在后院茶桌前，把事掰开了说。

老方先摇头："你周敬尧从小跟着父亲学做茶，认得好叶子、炒得好火候，论手艺，这方圆百里没几个人能比。可那座茶山为啥荒了两年？听说管事的老账房一走，伙计三个月没领到工钱，山上杂草比茶树还高。光定章程、排班次、查进出货这些事，就够你喝一壶的。"

何先生接口："我跑洋行多年，见过做手艺的人最后栽在'不会管'上。一个铺子能不能立住，靠的不只是手艺，是里头的人怎么安排、钱怎么走、事情怎么定先后。买山靠手艺，治山靠制度。"

周敬尧皱眉："我请人来帮我管，不就行了？"

老方说："请人来管，前提是你得先有'管'的章法。请来的师傅再能耐，若东家自己说不清一年要种几季、几月该施肥、收来的茶分几等、卖到哪里，他来了也只能东一榔头西一棒子。"何先生补一句："这章法，就是你自己的本事。"

周敬尧沉默半晌。

第二天，他做了一件让镇上人都看不懂的事——没急着去找周家兄弟谈买山，而是把自己关在后院账房里，写了整整七天。

他先把茶山一年四季要做的事，从翻土、追肥、修剪、采摘要分几批，列成一张总表；又把伙计分成三班，每班该干什么、听谁的话，写明白；再把进出银钱、卖出去的款子怎么走，定成死规矩；最后写一份三年小账目——头一年救活，第二年稳产，第三年做自己的字号。

何先生指着其中一行说："这条写得好——'师傅只管山上事，山下卖茶、对账、走人情，由东家自己担。'这就是规矩。掌柜的不必亲自翻土，但必须把分工画清楚。"

第七天头上，周敬尧去找周家兄弟谈买山。席间他没提银子，先把总表拿出来，从头到尾讲了一遍如何起死回生。周家老大教书先生看罢，问："照你这一套，山上得请几个师傅？"周敬尧答："至少要请一位懂修剪追肥的，再请一位会带班跑山的。"教书先生点头："好。山价让一成，算你这一份章法的见面礼。"

三年后，敬记茶号的新茶山在浙西一带小有名气。何先生托人带了一句话："你那天写的不是章法，是你自己。掌柜的本事长在身上，山上的事才能立住。"

周敬尧听完笑了笑，把那幅褪了色的总表重新裱了一层，挂到后院堂屋正中。

---

**📖 原文定义**

> 科学有效的企业管理（包括战略规划、组织结构设计、运营管理等）是企业经营成功的关键要素之一。在股权投资中，被投资企业的团队有可能缺乏专业企业管理的能力。例如，作为创业投资的对象，许多初创科技企业的创始团队成员往往是技术背景而非管理背景，创业投资基金对此类企业进行投资后，投后增值服务的重要内容之一就是需要帮助被投资企业提升其企业管理的能力，从而对投资团队企业管理的专业水平提出了要求。再如，在并购投资中，有时目标公司管理水平低下，导致价值被低估并成为并购基金的投资对象。在这种情况下，投后的重整将以提升被投资企业管理水平为重点，从而对投资团队的企业管理专业水平提出了较高的要求。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 荒废两年的茶山（"管事的老账房一走，伙计三个月没领到工钱"） | 目标公司管理水平低下，需要提升企业管理能力 |
| 周敬尧擅长做茶手艺，却对"管事"陌生 | 投资对象常是"技术背景而非管理背景"的创始团队 |
| 周敬尧花七天写下茶山全年总表、班次分工、银钱规矩、三年小账 | 战略规划、组织结构设计、运营管理三项企业管理的核心要素 |
| 老方指出"请师傅的前提是东家自己有管事的章法" | 投资团队需具备企业管理的专业水平，才能判断并指导外部资源 |
| 何先生强调"分工画清楚"——师傅管山上，东家管山下 | 组织结构设计中权责分工的明确 |
| 周家教书先生看到总表后主动让价一成 | 投后增值服务的逻辑：先识别"管"的缺口，再用专业水平去补 |
| 三年后山场在浙西立起字号 | 企业管理能力作为"投后重整"重点，对提升被投资企业价值的作用 |
| 何先生临别话："章法就是掌柜的本事" | 投资团队自身的企业管理专业水平是核心要求 |
