# （二）主要的并购投资主体

## ⚖️ 寓言故事 —— 《三家买栈》

民国十二年的春天，松江府棉纱行情连跌三月，淀河边上的「瑞兴永」栈房终于扛不住了。老板许伯年五十六岁，头发花白，半辈子把这栈房从八张仓板做到六十二张，眼下却要把它摆上桌面。消息一传，三个买家几乎同时登门。

头一个来的是沈砚舟，城西「同和祥」绸缎庄少东家，西装笔挺，进门翻开仓册，指着淀河对岸一处码头：「许伯，那块地我盯了三年。同和祥要走水路出货，借您这栈房中转，年省运费两千块大洋。咱们并成一家，仓还是您管，地契归我。」他要的是自家买卖顺当——买这栈房，是为把同和祥的生意补齐一块。

第二个来的是从南洋归来的老客钟衡，戴着礼帽，笑眯眯坐下，先要了一壶龙井。「许伯，栈房账面七万，我出七万五，全现金，一手交钱一手交契。三年后我若脱手，赚了咱们二八分，亏了算我钟某人的。」他不图棉纱，不图仓板，要的是一笔账算得过来的买卖。

第三个来的人有点怪。进门只递了张名片：「永合号·账房程敬之」。程先生不喝茶，只看账册，临走留一句：「我家东家这两年在淀河一带收栈房，已收了四家。您若愿意卖，永合号愿以六万八现银收下，并请您继续做掌柜，月俸另计。」许伯年听明白了——这桩买卖里，他既是卖家，也变成了替人干活的人。

许伯年关上门，把三人请过的客都回绝了一遍，独自在院里走了二十圈。他想：沈砚舟要的是自家买卖顺当，要这栈房是为让绸缎庄走水路；钟衡要的是一笔账算得过来的差价，无所谓里头是棉纱还是麻袋；永合号要的是淀河边上一整排栈房串成串，做成一门能反复倒手的生意。三家上门，不是同一回事。

夜里他把老友齐先生请来喝闷酒。齐先生听完三人提的条件，端起酒盅慢悠悠开了口：「砚舟那孩子出钱买的是同和祥自家要使的东西，他要的不是栈房本身，是这块地方能给他省下的运费——他图的是并进来之后两桩买卖并成一家。钟衡呢，他买的是一纸地契，转手赚多少赔多少全凭算账的本事，他跟你的栈房没什么情分，只认账。永合号那位程先生，是替别人来收楼台的，钱不是他自己的，也不打算自己来赶车——他要的是攒够一整排栈房，将来再一总出手。」许伯年把盏一放：「那我卖谁？」齐先生摇头：「那得看你要的是什么。要把一辈子的营生并进同和祥去，找砚舟；要现钱养老，找钟衡；要在这栈房里再干二十年但不愿独自扛行情，找永合号。」

第二天许伯年端着茶坐在院里，对儿子说：「爹想明白了——砚舟买我，是替他家补一块拼图；钟衡买我，是买一笔账；永合号买我，是攒一整排栈房。三家不一样，咱家只能走一条路。」

三天后，许伯年在「同和祥」沈砚舟带来的文书上落了印。

**📖 原文定义**

> 市场上的并购投资主体主要包括企业并购投资人和并购基金。
>
> 企业并购投资人一般是行业内的企业，其可能出于企业发展战略需要进行并购投资，也可能仅仅基于获取财务回报的目的进行并购投资。买方基于战略目标，发挥其与卖方企业之间的协同效应而进行的投资，被称为战略投资；买方主要基于获取阶段性财务回报目的而进行的投资，被称为财务投资。
>
> 企业并购投资人通常会利用其掌握的资源与被投资企业进行协同，因此，企业并购投资人会进行更多的战略投资。
>
> 并购基金通常主要基于获取财务回报的目的进行并购投资，因此，并购基金所进行的主要是财务投资。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 同和祥绸缎庄少东家沈砚舟 | 企业并购投资人（同行业内、有自家主营业务） |
| 沈砚舟想借瑞兴永码头走水路省运费、要「并成一家」 | 战略投资（基于战略目标、追求协同效应） |
| 老客钟衡出七万五买地契、三年后二八分账 | 财务投资（基于获取阶段性财务回报） |
| 永合号程先生替东家来、已收四家栈房、许伯年继续做掌柜 | 并购基金（专业攒盘、替出资人办事、谋的是整体出手的回报） |
| 许伯年想「并进同和祥」/「现钱养老」/「再干二十年」三种不同诉求 | 三类主体分别匹配企业不同需要 |
| 齐师傅一句「他图的是并进来之后两桩买卖并成一家」 | 战略投资追求协同效应的本质 |
| 齐师傅一句「他只认账」 | 财务投资只看账上算得过来的回报 |
| 齐师傅一句「替别人来收楼台的」 | 并购基金本身非自有资金、代人操盘的属性 |
