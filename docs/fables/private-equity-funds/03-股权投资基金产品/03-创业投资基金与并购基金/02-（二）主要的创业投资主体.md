# （二）主要的创业投资主体

## 🏺 寓言故事 —— 《三家凑钱》

宣统二年的腊月，杭州桃源巷的陈望舒辞了布庄的差事，要开一间做新式肥皂的小作坊。图纸画好了，锅炉订下了，独独缺钱——开张至少要三百两，他手里只有八十两。

开春后第一个找上门的，是巷尾茶馆里爱喝茶的周先生。他五十出头，早年炒过房产，手头有些闲钱。进门就开门见山：「我出三十两，不用写凭据，也不用谁来作保。钱你拿去用，等肥皂卖开了，我跟着分几成红利就行。」话没说完人已经走了，临走撂下一句：「孩子，做事小心点。」

第二个月，城里鼎鼎大号的瑞丰织造少东家林鹤鸣坐着青帷小轿来了。进门就亮明来意：「听说你在做肥皂，我那织造局正好缺这一味配浆洗剂的东西。我出五十两入股，将来肥皂卖给我家一勺不漏。」陈望舒心里琢磨：瑞丰要的是自家厂里能用的东西，他图的是自己买卖顺当。

再后来是南浔来的庞老板，差了个姓方的管事带封信。信里写得明白：「庞老板邀了十二位掌柜，每人各出十两，凑成一百二十两；另请城西钱庄代为看账，记进出、明分摊。钱交庞老板管，事请陈望舒做。账，按季公开。」陈望舒把信看了三遍，越看越觉得这一百二十两和前面两笔都不太一样——钱是十几个掌柜凑起来的，旁边还有钱庄的账房盯着。

三笔钱的来路不一样，性子也不一样。周先生的钱最轻，一人说了算，没人来过问怎么花；瑞丰的钱最重，是冲着自家买卖来的，将来做什么、卖给谁，得先问问林鹤鸣；庞老板那一笔最整，钱是凑的、账是明的，连自己花起来都得打起十二分精神。

陈望舒把三件事一说，齐师傅呷了一口酒，慢悠悠开腔：「你这愁的，不是哪笔钱先收，是这三笔钱从哪儿来，就注定了你往后怎么待它们。」他把酒盏往桌上一搁，「周先生那三十两，是他一人口袋里的钱，没人管他，他也不必让人管；瑞丰的五十两，是人家厂里掏出来的，自家要使，自然得听他安排；庞老板那一百二十两，十二个掌柜合在一处，钱庄又立了账——这一笔，分明是有人拿着规矩在管。」陈望舒歪着头想了一会儿：「那不是庞老板管我，是那十二位掌柜和钱庄管我？」齐师傅点头：「不止管你，也管庞老板自己。」

陈望舒送走齐师傅，回到桌前把三笔账重新写进簿子。周先生那笔，单独记一页，注上「不与外人闻」；瑞丰那笔，旁边写一行小字「购入以瑞丰为先」；庞老板那笔，用朱笔另起一格，把十二位掌柜的名字一一抄上，又把「按季对账」四字圈出来。簿子合上的时候，他忽然明白：这间小作坊虽还没开张，可账本上已经住进来三种不同的客人——一种散淡，一种贴紧，一种严整。客人本就从不同的路来，得照着各自的门道招待。

开春那天，桃源巷的鞭炮响了。

---

**📖 原文定义**

> 创业投资可以采取组织化形式和非组织化形式。组织化创业投资，主要包括创业投资基金和企业创业投资；非组织化创业投资，主要是指天使投资。
>
> 创业投资基金具有一定的外部性，因此多数国家对创业投资基金进行一定限度的行政监管。企业创业投资和天使投资不涉及对外募资，由投资者使用自有资金进行投资，不具有明显的外部性，因此通常不需要专门的行政监管。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 周先生一人出三十两，不写凭据、不需人作保 | 非组织化的天使投资（个人自有资金，无外部募资） |
| 周先生的钱陈望舒自由支配，无人过问 | 天使投资不具明显外部性，通常无需专门行政监管 |
| 瑞丰织造少东家林鹤鸣出五十两，要肥皂供应自家浆洗坊 | 组织化的企业创业投资（企业基于自身战略进行投入） |
| 陈望舒做肥皂卖给谁家得先问林鹤鸣 | 企业创业投资的目的与本业协同 |
| 庞老板邀十二位掌柜各出十两，凑成一百二十两 | 组织化的创业投资基金（向他人募集资金） |
| 城西钱庄代为看账、按季公开 | 创业投资基金需接受规范与监管 |
| 庞老板的合伙账本有十二位名字与按季对账 | 创业投资基金的「外部性」与相应监管要求 |
| 齐师傅一句「客人本就从不同的路来」 | 三类创业投资主体的本质区分：组织化 vs 非组织化 |
