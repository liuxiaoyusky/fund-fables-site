# （一）并购投资

## 🏗️ 寓言故事 —— 《收旧铺》

老周在镇上开了三十年绸缎庄，眼看着生意一天不如一天。不是他手艺不行，是时局变了——年轻人不爱穿绸缎，爱穿洋布。他的铺子，货堆满仓，伙计闲得打瞌睡，可债主天天上门催债。

---

老周想转手，可谁肯接这个烂摊子？绸缎庄的招牌虽然老，可里面的货、人、债，缠成一团麻。镇东头的钱庄掌柜老赵，却看中了这块地——不是看中绸缎，是看中铺子的位置。那地段，临街三间门面，后面连着仓库，换成什么买卖都值钱。

老赵找老周谈："铺子我接手，但有个条件——不是买你的货，是买你的招牌、你的地、你的人。你欠的债，我认；你的伙计，我留；你的货，我折价处理。往后这铺子姓赵，不姓周。"

老周愣了。他本想着把货清了、债还了、伙计散了，自己留块招牌养老。可老赵要的是全部——连根拔起，从头来过。老周咬咬牙，点了头。他知道，与其守着半死不活的铺子等死，不如让老赵拿去，换条活路。

老赵接手后，第一件事就是把绸缎全清了，换成洋布和成衣。原来的掌柜老周，老赵请他当顾问，每月给份俸禄，不让他管事。原来的伙计，能干的留下学新买卖，不能干的给笔遣散费。三个月不到，铺子焕然一新，客流如织。

老周坐在家里，看着街对面焕然一新的铺子，招牌换成了"赵记布庄"，心里五味杂陈。他忽然明白了：老赵买的不是他的绸缎，是他的"壳"——地段、门面、人手，这些都是现成的。绸缎死了，壳还在，换个芯，就能活。

一年后，老赵的布庄赚了钱，老周那份顾问俸禄也涨了一倍。有人问他后不后悔，老周说："我自己的手艺，救不了自己的铺子。老赵有眼光、有本钱、有魄力，他把我的壳拿去，换了新芯，这是他的本事。我守着那堆绸缎，只会一起烂掉。"

---

**📖 原文定义**

> （一）并购投资
>
> 并购即兼并与收购，并购投资是指通过投资取得另一企业的全部或部分资产或股权，从而对另一企业的产权及其经营管理实施部分或完全控制的行为。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老赵买下老周的全部铺子（招牌、地、人、债） | 取得另一企业的全部资产和股权 |
| 不是买货，是买"壳" | 并购的是企业产权而非具体资产 |
| 老赵接手后全面改造经营 | 对目标企业的经营管理实施完全控制 |
| 绸缎死了，壳还在，换个芯就能活 | 并购后通过重组改造实现价值提升 |
| 老周当顾问不管事 | 原股东/管理层退出经营控制 |
| 老周说"守着绸缎只会一起烂掉" | 并购使陷入困境的企业获得新生 |
