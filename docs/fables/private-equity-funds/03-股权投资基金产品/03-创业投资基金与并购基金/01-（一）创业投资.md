# （一）创业投资

## 🌱 寓言故事 —— 《育苗人》

老林在城郊有片苗圃，专育珍稀树苗。他不卖成树，只卖幼苗——那些刚发芽、枝干细弱、能不能活全看天意的苗。村里人都笑他傻："成树一棵卖十两，幼苗十棵才卖一两，你图什么？"

---

老林不恼。他在苗圃里搭了间草棚，守着那些幼苗，一守就是三年。浇水、施肥、除虫，比伺候月子还精心。有的苗活了，抽枝长叶；有的苗死了，烂在土里。老林把死的拔了，空地再补新种。十年下来，他育出的苗，活三成，死七成。

可活的那三成，后来都长成了参天大树。买他苗的人，有的是开茶园的，有的是建园林的，还有的是想种一片林子留给子孙的。那些苗在他们手里，又长了五年、十年，最后卖价是老林当初卖苗价的百倍。

老林自己不种树。他说："我只管育苗。苗活了，是买苗的人有眼光、有耐心；苗死了，是我没育好，或者天不饶人。我赚的是育苗的钱，不是成树的钱。"

有一年大旱，苗圃里死了大半苗。老林把剩下的好苗挑出来，连夜送到相熟的买家手里，说："这些苗我育了五年，根扎得深，旱不死。你们拿回去，好好种，十年后自有回报。"

买家问："你不怕我们种了不卖，自己留着？"

老林笑："苗是你们买的，树是你们种的，我管的是苗到你们手里那一刻。以后树大了，你们想卖想留，是你们的事。我只盼那些好苗，都能长成好树。"

老林一辈子没富过，可他的苗圃里，年年有新苗发芽。村里人后来才明白：他不是傻，是看得远。育苗的人，赚的是未来的钱，赌的是眼光和耐心。三成活，七成死，可活的那三成，抵得过所有死的。

---

**📖 原文定义**

> （一）创业投资
>
> 创业投资，也称风险投资，是指向处于各个创业阶段的未上市成长性企业的股权投资。具体来说，创业投资是指向处于创建或重建过程中的未上市成长性创业企业进行股权投资，以期所投资创业企业发育成熟或相对成熟后，主要通过股权转让获取资本增值收益的投资方式。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老林专育幼苗，不卖成树 | 创业投资投向处于创业阶段的未上市企业 |
| 幼苗细弱，能不能活看天意 | 创业企业的高不确定性 |
| 活三成，死七成 | 高风险特征 |
| 活的三成长成参天大树，卖价百倍 | 成功后的高资本增值回报 |
| 老林赚育苗钱，不赚成树钱 | 通过股权转让获取资本增值收益 |
| 大旱时把好苗连夜送给买家 | 投后管理，帮助企业渡过难关 |
| "苗到你们手里那一刻"后不管 | 投资后通过股权转让退出，不参与后续经营 |
