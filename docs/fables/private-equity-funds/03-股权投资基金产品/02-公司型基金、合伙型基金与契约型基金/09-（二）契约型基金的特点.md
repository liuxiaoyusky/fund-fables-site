# （二）契约型基金的特点

## 📜 寓言故事 —— 《一纸三签》

永丰镇上做丝绸生意的郑老大，这回遇上一桩麻烦。

他手头攒下了一笔闲银，原想自己南下苏州再开一间分号，可巧镇上的陈师傅来串门子，提起上海那边"洋行新花样"——几家东家凑钱，托一个懂行的人去投铺面，赚到钱了按份分，赔了也不连累各人。郑老大一听就动了心。可转念一想：钱交出去，那管钱的人拿着跑了怎么办？管钱的人去投，投赔了，是不是要找我补？投对了，赚的钱是不是要扒一层皮再给我？

左也不是，右也不是。

孙先生是镇上出了名的状师，专替人写契立据。郑老大寻到他书斋，铺开账本，把自己的顾虑一桩桩倒出来。孙先生听完，沉吟片刻，提起笔来，说："依律而论，你这桩事，不必去搭伙开铺子，也不必找谁当东家。只消你、我，再加一个替咱俩看着银子的，三方立一纸文书，银子的事就算妥了。"

郑老大听糊涂了："不搭伙、不开铺，那这钱搁谁手里？"

孙先生指着桌子说："就搁书桌上。你出钱，我替你管事。管事的人只管投，不管银子放在哪里；看银子的人，只管守钱，不能自己拿去投。三家分得清清楚楚，谁也碰不着谁的底。"

郑老大说："我怕你拿着跑了。"

孙先生说："依律，这钱虽在我手，却不是我孙某的私产，是替你这一档子事单立的。哪怕我自个儿欠了债，旁人要封我的账，也封不着这一笔。契约里写得明白。"

郑老大又说："我怕我自个儿欠了债，连累这一笔钱。"

孙先生答："依律同理。你这一份银子一旦入了契，便独立于你自家营生。你那边塌了天，这一份也稳当当搁着。"

周先生是镇上小票号的掌柜，平日专管替人看银子。孙先生便把周先生请来，三方坐定，写了三份契——一份出资契、一份管事契、一份看银契。三方各执一份，盖印为凭。钱从郑老大账上划到周先生票号单独开的一格里，钥匙在周先生手里，孙先生下注买卖需凭契书提银，周先生照契验明方才放行，赚了亏了都记在那一格里，季末按契书上的份子分回郑老大手里。

郑老大接过契书回去，心里一块石头落了地。后来那笔投苏州铺面的钱果然没出岔子。年终清算，他与孙先生、周先生坐在茶馆里把盏，郑老大感慨道："原先以为要把银子押到别人铺面上才踏实，这回明白了——不用搭伙、不用开铺、不用找东家，只要契上写明白，三家各守各的规矩，钱一样稳当。"

孙先生举杯一笑："契上白纸黑字，你的就是你的，谁也拿不走，谁也动不了。"

---

**📖 原文定义**

> 契约型基金主要有以下特点：
>
> 第一，财产独立。契约型基金属于信托性质，根据《信托法》《中华人民共和国证券投资基金法》（以下简称《证券投资基金法》）《私募条例》，基金财产独立于基金管理人、基金托管人的固有财产。因此，基金财产即使由管理人或托管人占有并运作，但不会被用于清偿管理人或托管人自身的债务。相较于一般合同关系，契约型基金财产的安全性更高。
>
> 第二，灵活度很高。契约型基金本质上是通过合同确立的一种契约关系，因此，合同各方不需要依据商事组织法设立商事实体，而可通过基金合同对基金的组织运作进行约定，相较新设公司或合伙等实体，运作更加灵活。
>
> 第三，避免双重纳税。契约型基金不是企业所得税的纳税主体，不需要缴纳所得税，只有基金投资者需要对取得的投资收益缴纳所得税。
>
> 第四，商事登记存在障碍。在我国目前的企业股权登记系统中，由市场监管部门负责的非公众公司股权登记系统不接受契约型基金登记为被投资企业股东。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 孙先生说"不搭伙、不开铺子，只消立一纸文书" | 契约型基金不设立单独实体，无独立法人地位 |
| 郑老大、孙先生、周先生三方各执一份契 | 三方关系：投资者-管理人-托管人 |
| 钱划到周先生票号单独开的一格 | 基金财产由托管人保管，独立运作 |
| 孙先生说"哪怕我欠了债，旁人封不着这一笔" | 基金财产独立于管理人的固有财产，不用于清偿管理人自身债务 |
| 郑老大说"我自个儿欠了债"也被挡住 | 基金财产独立于投资者，投资者债务不穿透 |
| 三方立契、不设商事实体 | 灵活度高，无需依《公司法》《合伙企业法》设实体 |
| 季末按契书的份子分回 | 收益分配由基金合同约定，避免双重纳税的安排 |
| 钱只能下注时凭契提取，平日不流通 | 资金隔离 + 托管人执行管理人指令的运作机制 |
