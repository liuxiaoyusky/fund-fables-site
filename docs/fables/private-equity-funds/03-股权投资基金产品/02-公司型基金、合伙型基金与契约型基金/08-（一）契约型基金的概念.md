# （一）契约型基金的概念

## 📜 寓言故事 —— 《一纸三方契》

那年开春，永丰号想把手头一笔闲银拢起来，去投南边茶园的股权。郑老大坐在账桌后头，把算盘拨了又拨，左想右想，找不出一个现成的法子。

照镇上老例，要做这件事，先得立个号。立号就得有铺面、有字号、有掌柜当东家。郑老大皱眉："我这银子是替一帮老主顾代管的，自己跑去当东家，账目搅在一起，将来分账说不清。"

他又想另一招，找几个信得过的人搭个伙。可合伙就得有人扛大头，万一下人把事办砸了，谁来兜底？再说了，搭伙要立字据，人一多，规矩就碎得快。

一筹莫展，郑老大让伙计去孙先生书斋递了张帖子。

孙先生翻完帖子，搁下茶盏，慢悠悠开口："依律而论，您这桩事，不必另立字号，也不必搭伙。您只要找三样人——管钱的、管账的、出份子的——大家坐到一起，把章程白纸黑字写清楚，就结了。"

郑老大皱眉："写清楚了，银子算谁的？"

孙先生答得干脆："银子名义上还是大伙儿的。您写的这张纸上只约定一件事——银子怎么凑、谁来管、谁来替大伙儿保管、分账怎么分。至于这张纸，本身不立铺面，不挂招牌，所以也用不着去镇公所另立门户。"

郑老大心里还是打鼓："那要是管钱的人卷了银子跑路呢？"

孙先生捻着胡须说："所以您得再请一个人，独立于管钱的人，专管银子的进出和保管。这人按管钱人开的条子照办，不碰钱的主意。管钱的只管拿主意、看行情，下单时也得由保管人在银子上盖戳子。"

郑老大来了精神："等于管钱的是大脑，保管的是手脚？"

孙先生点头："正是。一脑一手，加上出钱的东家，三方各守本分，全凭那一张纸连着。这纸，就是契。"

打这主意定了，郑老大请周先生当保管人。周先生的小票号在十六铺桥头开了二十年，街坊信得过。永丰号的刘账房把章程草拟出来，孙先生逐条过目，改了三遍，定稿。

那张纸上写得明明白白：出资人凭据出资多少、占几成；管钱人按约买卖南边茶园股权；保管人按管钱人指令进出银两，三方盖章为凭。纸上没立字号，没挂招牌，郑老大也不去镇公所申办铺面。

事成那天，郑老大站在十六铺桥头，对着永丰号的方向拱手："这就齐了。"

镇上有人嘀咕："这算个啥？既不是号，也不是伙，画一张纸就把事办了？"

孙先生路过听见了，摇着扇子说："依律，正是一张纸，才把事办得稳。三方不同坐一条船，靠这张纸把话说死；银子是大家的，不是哪一人的私产；谁也不能拍胸脯说'亏了我兜底'——大伙儿只就着这一张纸说话。"

周先生在桥头茶摊上喝茶，回头笑了一句："这纸啊，写得好，比铺面稳当。"

那张纸没名字，没人给它立户头。可三方各司其职，银两进出都有章可循。后来投茶园的收益真到了账，刘账房按约把分账一条条划给各位出资人。出资人拿的是按约分到的银子，不是哪家商号的分红。

镇上有人事后问孙先生："孙先生，这种没铺面、没字号的事，靠啥立得住？"

孙先生答："凭那张纸。纸上有三方，纸外没有铺面，没有招牌。它不是人，也不是字号——它就是大伙儿一起认的章法。"

---

## 📖 原文定义

> 契约型股权投资基金是指通过订立信托契约的形式设立的股权投资基金。契约型股权投资基金不设立单独的实体，也就当然不具有独立的法人地位。
>
> 契约型基金的参与主体主要为基金投资者、基金管理人及基金托管人。基金投资者通过购买基金份额，享有基金投资收益。基金管理人依据基金合同负责基金的经营和管理操作。基金托管人负责保管基金资产，执行管理人的有关指令，办理基金名下的资金往来。

## 💡 对应点

| 故事元素 | 概念对应 |
|---------|---------|
| 郑老大想拢银子投茶园 | 设立契约型基金的目的——集合出资做投资 |
| 不用另立字号、不用搭伙 | 契约型基金不设立单独实体，不具独立法人地位 |
| 管钱的、管账的（保管）、出资的三方 | 基金管理人、基金托管人、基金投资者三类参与主体 |
| 三方共立一张纸 | 通过订立信托契约（即基金合同）设立 |
| 纸上写明出资、买卖、保管、分账规则 | 基金合同对基金组织运作的约定 |
| 周先生按指令办事、不碰钱的主意 | 托管人负责保管基金资产，按管理人指令办理资金往来 |
| 刘账房按约划账给出资人 | 投资者通过购买基金份额享有基金投资收益 |
| 不去镇公所申办铺面 | 不需依商事组织法设立商事实体 |
| 孙先生说"这纸不是人也不是字号" | 契约型基金本身不是法人或实体，仅是契约关系 |
