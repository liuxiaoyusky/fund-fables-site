# （一）基本信息

## 📜 寓言故事 —— 《布告墙上那张纸》

民国初年的深秋，苏州城南的染坊街出了一件稀罕事。街口布告墙上贴出了一张崭新的纸，墨迹还带着清香，纸的右上角压着一方朱红印章。纸的下方围满了人，七嘴八舌地议论着。

纸的抬头写着：「城南织梦合伙」。看名字的人念出来，后面就有人接嘴：「这是合伙做生意的人家，把招牌挂出来给大家看呢。」老裁缝阿年拄着拐杖挤进人群，眯着眼把那张纸从头到尾看了三遍，回头跟徒弟阿贵说：「这张纸，件件都写着门道呢。你给我记牢了。」

阿贵是阿年的小徒弟，刚进染坊不到一年。他盯着那张纸，左看右看，眉头皱得能夹死一只蚊子。纸上一行一行写着：起头是「城南织梦合伙」——这是人家给自己起的大名；紧跟着注着「民国三年八月初九立」，阿年说这是它降生的日子。纸的中段写着合伙做事的年限，七年为期，阿贵心想，这跟染坊里染一匹布要几天几夜是同一个道理——做事总得有个起止。

再往下，纸上写了两行名字：第一行写着「陈守拙」，底下注着「主事之人」；第二行写着「和丰钱庄」，底下注着「代为看管账目、保管银钱」。阿贵指着第二行问阿年：「这钱庄是合伙人吗？」阿年摇摇头：「不是。钱庄就好比隔壁王大娘，做事的人把毛线交给她，她不帮你织，只管替你数清楚、看好、不让野猫叼了去。做事的人若换了一茬，她还在那里守着账本。」

阿贵点了点头，又问：「那为什么钱庄的名字写出来，陈掌柜的名字也写出来，可是人名底下又分两行呢？」阿年笑了一声：「因为这两路人担的不是一样的担子。一个动脑动手，一个只管盯钱袋子。担子不同，写法就该不同，让看的人一眼分得清。」

人群慢慢散了。布告墙前只剩阿贵一个人站着，把那张纸又念了一遍。他忽然觉得，这张纸虽然薄，可它让一个看不见摸不着的合伙，变得有了脸面、有了生日、有了起止、有了做事的人、有了看钱的人——这五样东西凑齐了，外头的人提起它，心里便有了一个实在的影子。哪怕它还没做成一匹布，人家提起「城南织梦合伙」，都知道说的是哪桩事、谁在里头、怎么个走法。

阿贵回到染坊，把今天学到的念叨给阿年听。阿年点点头，从染缸里捞起一匹刚上好色的布，对着光看了看，说：「做布是这样，挂招牌也是这样——样子齐了，里子才撑得起。」

**📖 原文定义**

> 基金产品应当具备自身的基本信息，包括：基金名称、成立日期、基金期限、管理人名称、托管人（如有）等。在我国，基金产品备案过程中还会申请产品编码，基金通过设立公司或合伙等实体作为运营载体的，还会有实体的统一社会信用代码、工商成立日期和营业期限。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 纸上写的「城南织梦合伙」 | 基金名称 |
| 「民国三年八月初九立」 | 成立日期 |
| 纸上注明的七年为期 | 基金期限 |
| 第一行写着的「陈守拙」及「主事之人」 | 管理人名称 |
| 第二行写着的「和丰钱庄」及「代为看管账目」 | 托管人（如有） |
| 纸上凑齐的「脸面、生日、起止、做事的人、看钱的人」 | 基本信息五要素的合集 |
