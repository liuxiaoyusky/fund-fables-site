# （二）投资阶段

## 🌱 寓言故事 —— 《三亩地的四季》

老林在城郊有三亩地，种了一辈子庄稼。他有个习惯：每年开春，三亩地分三批种，从不把鸡蛋放在一个篮子里。

---

第一亩，老林种的是新品种的果树苗。这些苗刚从农科院运来，枝干细弱，叶子还打着卷。村里人都摇头："这苗能不能活都是问题，你倒好，把最好的地给了它。"老林不说话，只是每天挑水施肥，比伺候月子还精心。他知道，这苗要是活了，三年后挂果，五年后丰产，那滋味是别的地给不了的。可要是死了呢？这亩地就白忙一年。这就是第一亩地的规矩——赌的是将来，眼下看不见收成。

第二亩，老林种的是已经长了两年的桃树。去年结了小半筐果子，今年枝条更壮，花苞密密麻麻。老林在这亩地上花的功夫少多了，偶尔修修枝、除除草。村里人又说："这桃树眼看就要出息了，你不多上点心？"老林笑笑："该做的前两年都做完了，现在它自己会长。"这亩地的收成比第一亩稳当，果子个头也大，但要说惊喜，也有限——好是好，可好不到天上去。

第三亩，老林从邻村老张家手里接了过来。那原本是一片老梨园，品种过时，虫子又多，老张头懒得管，任它荒着。老林却看中这块地位置好、离水近。他花了半个月，把老梨树全砍了，重新规划，挖渠、改土、搭架子，准备种葡萄。村里人这回彻底看不懂了："人家老张都放弃的地，你捡来当宝贝？"老林只回了一句："地还是好地，就是得换个种法。"

三年过去。第一亩的新果树苗，活了一半，死了一半。活的那几棵，今年第一次挂了果，虽然不多，但咬一口，甜得人眯眼。老林知道，再过两年，这亩地会给他最大的回报。死的那一半，他也没太心疼——本来就知道会这样，种的时候就备了补苗的钱。

第二亩的桃树，今年大丰收，收了整整二十筐。老林挑到集市上，卖了个好价钱。可他也知道，这桃树的黄金年头就这么几年，往后产量会慢慢降，得提前想着更新换代。

第三亩的葡萄园，经过两年的折腾，今年第一次有了收成。老张头路过，站在地头看了半天，最后叹口气："还是你有眼光。我当初要是舍得下那番功夫……"老林递给他一串葡萄："地不分好坏，看的是谁来种、怎么种。"

老林的三亩地，对应着庄稼人的三种活法：第一亩赌的是将来，眼下苦，将来甜；第二亩吃的是现成的，稳当，但也看得到头；第三亩是别人不要的东西，自己接过来，换个法子，照样能活。三种地，三种时节，三种心思，缺一不可。

**📖 原文定义**

> （二）投资阶段
>
> 依据底层被投资企业所处的不同发展阶段，可以对基金进行分类。股权投资基金产品的底层投资标的主要是非上市公司股权或上市公司非公开交易股权。公司从创立到发展，再到可能存在的重组或出售，整个生命周期可以大致划分为初创期、成长期、成熟期以及可能存在的转型重组期。基于对底层标的所处发展阶段的偏好，股权投资基金也相应分为几类，以适应不同阶段的资本需求和投资策略。
>
> 创业投资基金是指主要从事创业投资活动的基金。根据行业惯例，我们可以将这一广义上的创业投资基金概念进一步细分为狭义的创业投资基金和成长型股权基金。
>
> 狭义的创业投资基金主要投资于初创期和小型成长期企业，提供资本支持以帮助企业快速发展、成熟，并在过程中实现股权价值增值，最终通过首次公开发行股票（Initial Public Offering，IPO）或并购等方式转让股权以实现投资回报。通常情况下，初创企业的产品尚处于开发和探索的阶段，商业模式和盈利能力还未得到验证，距离上市也相对遥远，企业的未来发展具有较强的不确定性，因此公司股权价值较低，发展潜力大。基于这些特点，创业投资往往面临很大的不确定性和较长的投资周期，而一旦成功，投资回报也可能非常丰厚。创业投资基金是最早出现的一类股权投资基金，对于支持中小企业、推动科技创新和新兴行业发展发挥着不可替代的作用，是国家长期战略布局的重要助力。
>
> 成长型股权基金主要投资于成长期企业。与狭义的创业投资基金主要投资于初创期企业不同，成长型股权基金更关注那些已经具备一定规模并处于快速增长阶段的企业，帮助它们进一步扩展业务和提升价值。成长型股权基金的单项目投资规模通常更大，因为被投资企业的估值和资金需求都较高，但投资风险和预期回报均小于狭义的创业投资基金。在中国市场，企业往往以上市作为"成熟"的标志。有的成长型基金偏好投资于成长期即将结束，基本达到可上市状态的企业，被称为上市前投资基金。在核准制背景下，中国企业上市标准高，二级市场估值普遍高于一级市场，上市企业短时间跌破发行价的概率极低，基金投资上市前项目的风险低，收益大。但随着市场发展和注册制改革，目前类似套利空间已经缩小，风险变大，上市前投资基金的占比也在下滑。
>
> 并购基金主要投资于成熟期和转型重组期的企业。与初创期和成长期企业不同，资本通常不是限制成熟期或转型重组期企业发展的主要因素。相反，处于发展瓶颈的成熟期企业和陷入经营困境的转型重组期企业，迫切需要通过深度管理重构、战略调整和产业整合等全面的改革和重整来重振旗鼓。因此，基金必须通过并购等方式取得目标企业的控制权，这是启动任何重整计划的前提条件。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 第一亩种新品种果树苗 | 狭义创业投资（初创期） |
| 苗弱、可能死、三年后挂果 | 不确定性高、周期长、潜力大 |
| 活一半死一半，备了补苗钱 | 风险分散，成功则回报丰厚 |
| 第二亩种了两年已长壮的桃树 | 成长型股权基金（成长期） |
| 花功夫少，收成稳当但惊喜有限 | 风险较低，回报预期较确定 |
| 第三亩接手老张荒废的老梨园 | 并购基金（成熟期/转型重组期） |
| 砍老树、重新规划种葡萄 | 深度管理重构、战略调整 |
| 三亩地三种时节三种心思 | 不同阶段对应不同资本需求和策略 |
