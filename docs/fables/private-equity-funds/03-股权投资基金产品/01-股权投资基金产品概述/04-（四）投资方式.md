# （四）投资方式

## 🏺 寓言故事 —— 《老孟的十两银》

临河镇上有位布商老孟，做了二十年绸缎生意，攒下十两碎银。银子压在床底下，每晚翻来覆去睡不着——存钱庄利息太薄，存着不动又觉得亏。

镇上最近传出一桩事：东头茶庄的柳二娘，把银子交给了跑船的阿贵，跟着阿贵下江南贩丝绸。半年后，柳二娘连本带利拿回二十两，乐得请全巷子吃酒。老孟坐不住了，托人找到阿贵。

阿贵在自家后院摆了茶，请老孟坐下。

"孟哥想跟船？"

"想。十两都给你。"

阿贵摇头："不急。我先问你——你跟我的船，是要我去你家里挑货，还是我替你去寻别个信得过的船家？"

老孟一愣："这两桩有啥分别？"

阿贵笑："分别大了。头一桩，我自己挑货、自己押船，赚了赔了我担着——孟哥你就图个省心，十两丢进来，坐等数钱；第二桩，我不去挑货，我去寻几位老把势，把你的银子交到他们船上，让他们去挑货。他们赚了，我从里头抽一笔；他们赔了，你认。"

老孟端起茶碗，没喝："那第二种法子，你图啥？"

"我图个稳。"阿贵说，"江南水深的码头我熟，可今年丝价忽高忽低，我一个人押一条船，万一栽了呢？若我把你的银子分一分——三两跟张老大的船去苏州，三两跟陈四的船去湖州，三两跟赵七的船走杭州，剩一两我留着自用——十条船里赔了两条，剩下八条照常带银子回来。"

老孟皱眉："可那些个船家，你咋知道他们靠谱？"

"我跟张老大跑了八年，从没湿过鞋；陈四是我亲表弟；赵七是镇上赵屠户的闺女，从小看她长大。靠不靠得住，我心里有数。"

老孟又问："那你自己呢？你光坐着抽钱，不用干活？"

"我也挑货。"阿贵说，"只不过我把自己那份银子也只投一半在自家船上，另一半跟着陈四走。我吃肉，也让别人分我一口汤。"

老孟沉默许久，搁下茶碗：

"那万一我想分一分——五两跟着你的船去，五两跟着张老大的船去——行不行？"

阿贵一拍桌子："行！这就叫'两条腿走路'。一半跟我下江南，一半跟张老大跑湖州。我替你记账，到年底一张纸算清。"

老孟站起身，朝阿贵一拱手：

"那就照你说的——三两跟张老大去苏州，三两跟陈四去湖州，三两跟赵七走杭州，剩一两你自己挑货。"

阿贵笑着给他添了茶："孟哥，你这一两比他们的都精。"

十两碎银，一拆为四。第二天，老孟回去睡觉，这一回睡得踏实。

---

**📖 原文定义**

> 基金产品可以直接投资于被投资企业，即直接投资模式；也可以通过投资于子基金，再由子基金投向被投资企业，从而间接投资于底层被投资企业。基金还可以选择直接投资与投资子基金相结合的投资方式，即一部分资金用于直接投资，另一部分资金投向子基金，相应的比例安排也是基金投资策略的重要组成部分。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老孟把十两银子交给阿贵，让阿贵自己挑货押船 | 直接投资模式——资金直接投向底层被投资企业 |
| 阿贵不挑货，转去寻张老大、陈四、赵七的船，把老孟的银子交给他们 | 间接投资模式——通过子基金投向底层标的 |
| 张老大跑苏州、陈四跑湖州、赵七跑杭州，三条船分散风险 | 母基金特征——通过投资多个子基金扩大组合、分散风险 |
| 阿贵只抽一笔，不亲自挑货 | 母基金定位——主要资金投向其他基金产品 |
| 阿贵把自己的钱一半跟自家船、一半跟陈四 | 基金管理人自有资金的双线配置——直接投资与间接投资结合 |
| 老孟最终选择"三三三一"拆分：三条跟船 + 一条阿贵自营 | 组合投资方式——一部分资金直接投资，一部分投向子基金 |
| "三两跟张老大去苏州，三两跟陈四去湖州，三两跟赵七走杭州，剩一两你自己挑货" | 比例安排——基金投资策略的重要组成部分 |
| 阿贵给老孟记账、年底一张纸算清 | 投资策略的落实——比例安排作为策略核心 |
