# （三）投资方式

## 🎯 寓言故事 —— 《两条路》

老陈在镇上开了间杂货铺，攒了三十年的家底。年过五十，他不想守着铺子过一辈子，打算拿一笔银子出去"生钱"。可怎么个生法，他拿不定主意。

---

镇东头住着老周，是个走南闯北的货郎。老周的路子野，消息灵通，哪里的 timber 要涨价、哪里的瓷器要跌价，他门儿清。老陈找到老周，说想搭伙做买卖。老周拍着胸脯："你把钱给我，我亲自去挑货、押船、卖货，赚了咱们分，亏了我认一半。"老陈心动了——钱直接变成货，货再变成钱，中间没有旁人，利钱都是自己的。这就是老周说的"直来直去"，银子直接变成货，不绕弯子。

可老陈的婆娘不乐意。她说："老周一个人跑，万一他看走眼，咱们全部家当就打了水漂。你不如把钱交给镇上的钱庄，钱庄手底下养着十几个老周这样的人，东边不亮西边亮，总有一个能赚钱的。"

老陈犯了难。交给老周，赚得多，可也亏得多；交给钱庄，赚得少些，但稳当。他两头都放不下。

转折发生在那年秋天。老周押着一船丝绸去北方，半路遇上大雨，船翻了，一船货泡了汤。老陈投进去的五百两银子，血本无归。老周倒是认了账，可他也没钱赔，只能慢慢还，这一拖就是三年。

老陈坐在杂货铺里，看着账本上火红的赤字，忽然想起婆娘的话。他咬咬牙，把剩下的三百两银子分成了三份，一份交给钱庄，钱庄再分给底下三个跑货的人去经营；一份自己留着，跟着老周重新做，但只投小笔买卖；还有一份，他借给了隔壁开酒坊的老李，收固定的利钱。

三年后再看账本：跟着老周的那笔，又赚又亏，起伏不定，但总算把上次亏的补回来大半；借给老李的那笔，每年固定收三十两利钱，不多，可稳当；交给钱庄的那笔，钱庄底下三个跑货的人，一个赚了、一个平了、一个亏了，加起来居然还小赚了一笔。

老陈这才咂摸出滋味来。直接跟着老周跑，是"直来直去"，利大险也大；交给钱庄，是"间接转手"，钱庄替他挑人、替他看着，自己不用操心，虽然利钱薄了一层，但三家里面总有两家能赚钱，风险摊薄了。两条路，没有哪条更好，只看自己的胃口和胆量。

婆娘在一旁纳着鞋底，头也不抬："我早就说了，鸡蛋不能放在一个篮子里。"

老陈笑笑，没接话。他翻开账本，在第一页工工整整写下两行字：一行叫"直路"，一行叫"弯路"。直路走得快，弯路走得稳。两条路都走，才是长久之计。

---

**📖 原文定义**

> （三）投资方式
>
> 股权投资基金产品的底层标的主要是非上市公司股权或上市公司非公开交易股权。股权投资基金产品可以直接投资于该类标的，也可以通过投资其他股权投资基金产品，从而间接投资该类标的。将主要资金直接投资于底层标的的基金被称为直投基金，而将主要资金投资于其他股权投资基金产品的基金被称为股权投资母基金。
>
> 股权投资母基金的主要资金投向其他股权投资基金产品，而非直接投资于底层标的。这种投资策略扩大了投资组合的范围，从而有效分散了投资风险。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老陈直接跟着老周跑货买卖 | 直投基金（直接投资于底层标的） |
| 利大险大，船翻血本无归 | 直投风险集中 |
| 婆娘建议把钱交给钱庄 | 母基金（投资于其他基金产品） |
| 钱庄底下十几个老周这样的人 | 母基金扩大投资组合范围 |
| 三家跑货的人一赚一平一亏，总体小赚 | 母基金分散投资风险 |
| 老陈三条路并行 | 直投+母基金+固定收益的组合策略 |
| "直路"走得快，"弯路"走得稳 | 直投与母基金的风险收益特征对比 |
