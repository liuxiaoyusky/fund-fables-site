# （三）组织形式

## 🏺 寓言故事 —— 《三村合买山》

山脚下三个村——靠山村、临水村、柳家屯——祖辈都靠上山砍柴过活。柴越砍越少，山越伐越秃，眼看就要断活路。

老周是临水村的木匠，他琢磨出个法子：三家凑钱，把对面那座光头山包下来，整片种上杉木苗。杉木长成要十五年，砍下一根值五两银子；现在一棵苗才几文钱。他拍着胸脯说："这事能办！"

三家合计，凑出四百两。

凑完银子，问题来了：钱搁在哪儿？由谁出面去买山？由谁雇人种树？

靠山村的阿贵想了想，提了头一个法子："去县衙登个记，三家一起成立个'三村林业'。挂个匾，立个账房，雇掌柜管事。赚了钱，按出资分——正正经经做买卖。"

老周摇头："麻烦！县衙那套手续得跑三个月，账房一年工钱就吃掉二十两。再说哪家想抽身走人，整家'三村林业'都得点头，账才算得清。"

阿贵叹气："这法子，行是行，就是把人捆得死，进也难退也难。"

老周又提第二个法子："不立匾，按老规矩，几家凑成'伙'，推个德高望重的人当'头伙'，其他几家当'伙计'。头伙拍板办事，伙计只管掏银子。赚了钱，先把本还伙计，余下一半分给头伙。"

阿贵眼睛一亮："这法子利索！"

"可有一桩，"老周压低嗓门，"头伙办事办砸了，亏的钱，伙计也跟着摊——你担不担？"

阿贵想了想："那得看头伙是谁。信得过，我认。"

"信不过的呢？"阿贵不答了。

柳家屯的孙婆婆一直坐旁边没吭声，这时才慢悠悠开了口：

"你俩都要立一个'壳'——有的立大壳，有的立小壳。我倒想问：立不立壳，有那么要紧么？"

老周一怔："不立壳，钱交给谁？山登在谁名下？"

孙婆婆从怀里掏出一张纸："我找镇上陈先生写好了。三家签个名、按个手印，写清楚——出多少银、占多少份、谁来办事、赚了怎么分、亏了怎么担、白纸黑字据下来。银子搁在陈先生家匣子里，他要支一笔，得咱们三家点头。"

"这……"老周翻来覆去看那张纸，"这就是一张纸？"

"登谁的名都一样。"孙婆婆说，"这纸就是名，这匣就是号。纸上的字比匾上的匾牢靠。"

阿贵迟疑："可万一将来扯皮呢？"

"扯皮看纸。纸怎么写，就怎么判。"

老周没说话，把三种法子在脑子里过了一遍：

头一种——立个堂堂正正的"号"，进退有据，可手续烦、人难散、银钱先往里贴。

第二种——搭个小伙，推个头伙办事，办事的担大头，掏银的也跟着担，赚了多分一点，可万一头伙不靠谱，掏银的跟着吃挂落。

第三种——不立号不搭伙，一张纸一匣银就开干，最轻巧最快，可全凭那张纸说话，纸外的事没人管。

他望向对面那座光头山。山还是那座山，杉木苗还是几文钱一棵。

老周把那张纸接过去，折好，塞进怀里。

"成，"他说，"就照孙婆婆这法子办。先干起来。"

---

**📖 原文定义**

> 基金产品的投资运作需要依托一定的组织形式。基金产品可以采取不新设实体而仅以合同约定的方式运作，即契约型基金；也可以新设公司或合伙企业作为运作载体，即采用公司型或合伙型的组织形式。不同的组织形式，会对基金的具体运作和管理产生不同的影响。
>
> 通常而言，在组织形式上，股权投资基金产品可以考虑的方案包括：新设公司作为运营实体的公司型基金、新设有限合伙作为运营实体的合伙型基金以及不新设实体而仅以合同约束当事人的契约型基金。相应地，以此标准可将基金分为公司型基金、合伙型基金和契约型基金。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 三家凑出四百两 | 资金汇集——通过组织形式把分散的钱拢到一起 |
| 阿贵提议去县衙登记、挂匾、设账房 | 公司型——新设公司作为运营实体，按公司法运作 |
| 老周嫌手续烦、整家点头才走得脱 | 公司型特点——治理结构完整，但进退受公司框架约束 |
| 老周提议搭伙、立头伙与伙计、按"先回本再分余" | 合伙型——新设有限合伙作为运营实体 |
| 头伙担大头、伙计也跟着一份担 | 合伙型特点——头伙（执行事务合伙人）担主要责任，合伙人共担风险 |
| "万一头伙不靠谱，掏银的跟着吃挂落" | 合伙型核心——人合性强，依赖头伙信用 |
| 孙婆婆掏出写好的纸、一匣银、三家签字 | 契约型——不新设实体，以合同约束当事人 |
| "这纸就是名，这匣就是号" | 契约型本质——合同关系即组织本身 |
| 老周最后把纸接过去"先干起来" | 组织形式选择——基于实际业务需求与现实约束综合决定 |
