# （二）投资范围

## 🧭 寓言故事 —— 《阿禄的四张框》

光绪十九年开春，杭州城东「广盈号」绸缎庄里，少东家阿禄刚把父亲留下的账本合上。柜上老伙计赵叔端了壶热茶进来：「少爷，老掌柜走前说过——钱攥在手里不会下崽，得让它出去走一遭。可去哪儿、走多远，您心里得有张图。」

阿禄苦笑：「赵叔，我就是没图才愁。柜上拢共六千两银子，押错了地方连翻本都没了。」他摸出几张宣纸：「我打算画几张框，把这银子框住。」

第一张框，他写得最谨慎——「这银子不能进那些摆在明面上叫卖、随时有人接手的铺子，得投到那些没挂牌、没敲锣、人坐在屋里谈妥才成交的买卖里头去。」赵叔点头：「街对面王二麻子的布行天天开门，那是别人现成的货；您若把银子投进别人手底下还没长大的染坊，那染坊没挂牌，您也找不到地方像卖布一样转手——专找'私下签契'的营生。」

第二张框，他写得更仔细——「不光要找那种没挂牌的，还要找那种'换了契就能变成股东'的门道。」他念叨：「比方说隔壁汪家开的纸号，眼下缺周转，写了一张借据，约定三年后他若还得起，连本带息归还；若还不起，这张借据就变成纸号的份子，咱就成了纸号的人。」阿禄说：「框一装的是已能当家的，框二装的是'借出去是为了最后能变成主顾'的。两路合起来才是完整去处。」

第三张框，阿禄圈得最细——「我还要看，这买卖里的人，是刚点起灶火的，还是灶火已烧旺的，还是灶火将熄、正在改灶翻新的。」他指着本子说：「'灶火不同，火候便不同'。刚起灶的便宜，可也最容易灭；火烧正旺的稳，可价也高；正改灶的，前头有烂账要收拾，能捡漏，也最折腾。」他把三种灶火都圈进框里：「初起的、正在长的、已熟的、要翻新的——四种人家，都收得进来，看我自己吃哪一路。」

第四张框，阿禄停下笔，抬头问赵叔：「您说，咱杭绸这一路，最该把银子投到哪种营生上去？」赵叔想了想：「您平日翻的洋文书里常提到——制药的铁铺、织机上的巧家伙、能掐会算的算房、还有街上新冒出来的脂粉铺子。这些都算得上'杭绸人该往里凑一凑'的去处。」阿禄一笑，提笔写下：「药铺、机巧、算房、新脂粉」——四样营生，四个方向。他把第四张框圈好：「框四是定行当的。咱不能今天投药铺，明天投打铁；总得在自己看得到的几样营生里挑。」

四张框画完，阿禄把它们齐齐叠起：「框一是门道，框二是借据能变主顾的路，框三是看灶火，框四是定行当。这几张框合在一处，银子的去处才叫'看得见'——看不见的咱不去，看不清的咱不押，看不懂的咱不沾。」

赵叔端起茶碗，望着那几张墨迹未干的宣纸：「老掌柜要是还在，看您把这'去哪儿、走多远'画得这般清楚，怕是要开柜上那坛十八年的女儿红了。」

阿禄望向窗外早春的细雨：「等头一桩买卖落了地，再开不迟。」

---

**📖 原文定义**

> 基金产品应当明确投资范围。股权投资基金产品的投资范围应当为非公开交易的股权性质的投资或以获得股权为目的的投资，后者主要指可转换债等。从被投资企业发展阶段而言，可以约定主要投资于企业的初创期、成长期、成熟期或转型重组期。此外，基金产品也会在设立之初明确基金的投资方向，即投资侧重的行业领域，例如生物医药、高端制造、人工智能、新消费等。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 第一张框——「没挂牌、没敲锣、上门才谈」的买卖 | 非公开交易的股权性质的投资 |
| 汪家纸号那张「还不上就变成份子」的借据 | 以获得股权为目的的投资（如可转换债） |
| 第二张框把「借据能变主顾」也圈进去 | 投资范围同时涵盖「直接的股权」与「通向股权的债权」 |
| 第三张框画的「初起的、正在长的、已经熟的、要翻新的」 | 投资范围对企业阶段的覆盖——初创期/成长期/成熟期/转型重组期 |
| 阿禄把这四种灶火都圈进框里 | 「可以约定主要投资于」——具体阶段由基金自行确定 |
| 第四张框写的「药铺、机巧、算房、新脂粉」 | 投资方向/行业领域（对应原文的生物医药、高端制造、人工智能、新消费） |
| 阿禄说「总得在自己看得到的几样营生里挑」 | 投资范围在设立之初就要明确 |
| 「框一是门道，框二是借据变主顾，框三是看灶火，框四是定行当」 | 投资范围的四重要素：交易属性 + 取得方式 + 企业阶段 + 行业方向 |
