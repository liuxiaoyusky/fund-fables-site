# （五）现行产品备案分类

## 📜 寓言故事 —— 《四块牌子》

老赵在县城衙门里管了二十年文书，最擅长的事就是给各种买卖发牌子。没有牌子的买卖，不能开张；牌子不对，被人查了要罚。可这几年，来找他领牌子的人越来越多，卖什么的都有，老赵的抽屉里渐渐堆了十几种样式，自己都分不清。

---

一天，新来的县太爷把他叫去，指着满桌的牌子说："老赵，你这太乱了。卖布的、卖米的、卖药的，全混在一起。往后咱们就定四条规矩，凡是做'钱生钱'买卖的，只许领四种牌子，各归各类，不许串。"

老赵竖起耳朵听。县太爷说：

第一种牌子，叫"生钱一号"。领这牌子的，是拿自己的银子直接去买作坊、买铺面、买船队的。他们投的是实打实的产业，不是市面上流通的纸。老赵在簿子上记了一笔：这是"直投"，钱直接变成产业。

第二种牌子，叫"生钱二号"。领这牌子的，自己不直接买产业，而是把钱交给那些已经领了"生钱一号"的人，让他们去经营。老赵嘀咕："这不是多绕一道手吗？"县太爷说："绕是绕，可分散。你把钱分给十个'生钱一号'的人，东边亏了西边赚，总比全押在一家强。"老赵又记一笔：这是"间接"，钱先变成别人的本钱，再变成产业。

第三种牌子，叫"育苗牌"。领这牌子的，专门去找那些刚开张的小作坊、小铺子，给它们送钱、帮它们长大。县太爷特别强调："领'育苗牌'的，只能去帮小苗，不能去买大树。已经长成的铺子、已经上了市的大船队，不许碰。"老赵问："为啥？"县太爷说："小苗需要人扶，这是善政；大树已经有人抢了，再去凑热闹，就是投机。"

第四种牌子，叫"育苗二号"。和"生钱二号"一个道理，自己不直接去找小苗，而是把钱交给那些已经领了"育苗牌"的人，让他们去挑苗、去浇水。老赵恍然大悟：原来二号牌子都是"转手"，一号牌子都是"直投"，只不过一个投的是大树，一个投的是小苗。

老赵回去后，把抽屉里的旧牌子全翻出来，一一对照。那些买老船队、买大作坊的，归到"生钱一号"；那些买新船、新作坊的，归到"育苗牌"；那些自己不买东西、专把钱借给别人去买的，按对方买什么，分别归到"生钱二号"或"育苗二号"。

他还发现，领"生钱一号"的人，县太爷要求他们再细分：有人专门买快要倒闭的作坊，拆了重组，这叫"收旧"；有人专门买盖房子的地，这叫"置地"；有人专门修桥铺路，这叫"基建"；还有人专门买大船队额外放出来的股份，这叫"增发"。老赵在每种下面又画了小格子，一格一格，清清楚楚。

半年后，县城里做"钱生钱"买卖的，人人知道自己该领哪块牌子。牌子对了，衙门查起来快；牌子错了，轻则换牌，重则关门。老赵的簿子成了全县的规矩，四块大牌子，十二块小牌子，各有各的门槛，各有各的禁区。

有一天，一个外乡人来领牌，说自己既想投小苗，又想投大树。老赵把笔一放："规矩是死的，你只能选一块。想两样都沾，就领两块牌子，分两个账本子记。混在一起？不行。"

外乡人悻悻地走了。老赵看着他的背影，心想：这四块牌子，表面是分类，骨子里是分责。牌子清了，责任才清；责任清了，县太爷才能睡得着觉。

**📖 原文定义**

> （五）现行产品备案分类
>
> 根据我国现行法规，中国证券投资基金业协会是私募投资基金的登记备案机构，私募基金管理人应当在产品募集完毕后办理基金备案手续。目前产品备案通过资产管理业务综合报送平台线上化报送，在基金类型的选项中，有四种选项可供股权投资基金选择，分别为：私募股权投资基金、私募股权投资类FOF、创业投资基金和创业投资类FOF。
>
> 私募股权投资基金，指投资包括未上市企业和上市企业非公开发行和交易的普通股（含上市公司定向增发、大宗交易、协议转让等），可转换为普通股的优先股和可转换债等的私募基金。在系统上选择"私募股权投资基金"后，还需要选择进一步的细分类别，包括并购基金、房地产基金、基础设施基金、上市公司定增基金和其他类基金。并购基金，是指主要对处于重建期企业的存量股权展开收购的私募股权投资基金。房地产基金，是指从事房地产项目投资的私募基金。基础设施基金，是指投资于基础设施项目的私募基金。上市公司定增基金，是指主要投资于上市公司定向增发的私募股权投资基金。
>
> 创业投资基金，根据中国证券投资基金业协会的说明，指主要向处于创业各阶段的未上市成长性企业进行股权投资的基金。原则上，鼓励创业投资基金投资早期企业、中小企业和高新技术企业。除中国证监会和中国证券投资基金业协会另有规定外，创业投资基金不得直接或者间接投资于不动产（含基础设施）、首发企业股票和存托凭证、上市公司可转换债券和可交换债券、公开募集基础设施证券投资基金份额、资产支持证券及上市公司股票，但所投资公司上市后基金所持股份的未转让及其配售部分除外。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 县太爷定下四种牌子 | 四种备案类型：私募股权投资基金、私募股权投资类FOF、创业投资基金、创业投资类FOF |
| "生钱一号"直接买作坊铺面 | 私募股权投资基金（直投未上市/非公开发行股权） |
| "生钱二号"把钱交给一号的人 | 私募股权投资类FOF（投资于其他股权投资基金） |
| "育苗牌"专门帮小作坊 | 创业投资基金（投资早期/成长期未上市企业） |
| "育苗二号"交给育苗牌的人 | 创业投资类FOF（投资于创业投资基金） |
| "生钱一号"细分：收旧、置地、基建、增发 | 并购基金、房地产基金、基础设施基金、上市公司定增基金 |
| 育苗牌不许碰大树和上市大船队 | 创业投资基金的投资限制（不得投资不动产、上市公司股票等） |
| 牌子清了责任才清 | 备案分类的本质是明确投资范围和监管责任 |
