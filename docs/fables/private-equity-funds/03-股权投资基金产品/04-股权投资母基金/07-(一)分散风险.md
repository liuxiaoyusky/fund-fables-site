# (一) 分散风险

## 🏺 寓言故事 —— 《十只竹篮》

十六铺桥头的老茶馆里头，陈师傅那张八仙桌永远是最热闹的一张。来来往往的商人、脚夫、卖花姑娘，进了门都要先朝他点个头，问一句:"陈师傅，今儿个桥上有甚新闻？"

这一天是月底，几位东家坐在一处议一件事。

东家甲姓周，开了家棉纱号，账上攒了八百两现银;东家乙姓何，洋行里做了半辈子买办，兜里有四百两闲钱;东家丙是开染坊的，手里捏着一千两。三个人凑到一块，是听说城里新开了一家大票号，利息给得高——年息一分二厘，比桥头所有钱庄都阔气。三人都想把银子全搁进去，但到底先投谁、后投谁，犯了难。

"要我说，就投一家。"周东家把茶碗一搁，"利息最高那家，一次性塞进去，省事。"

"我看还是别。"何东家摇摇头，"票号新开，谁知根底?要是遇着个不靠谱的东家,八百两打水漂,我可担不起。"

陈师傅在旁边听了一会儿，磕了磕烟袋锅子，慢悠悠开了口。

"我讲个事给你们听。"他说,"前清末年，桥头王老板也是这样，手里攒了三千两,听人说苏州绸缎庄赚得多，一头全砸了进去。结果那年发大水,船沉货烂,三千两一根丝都没换回来。"

"那后来呢?"何东家问。

"后来王老板学乖了。"陈师傅伸出十根手指头,"他把下一回攒的钱,分成了十份。十只竹篮,每只里装一份。他不再只投一家绸缎庄——这一篮给东街的茶商,那一篮给南门的水客,再一份给镇外的米行,还留了一份给小本经营的染坊。"

"那他赚得少了吧?"周东家皱眉。

"你听我讲完。"陈师傅摆摆手,"头一年,绸缎庄那家果然又亏了,折了一整篮。但其余九只篮子,有的赚、有的平、有的小亏。九个里面,东街茶商那一年大赚了一笔,米行也稳当。最后算总账——三千两的本,反倒比从前全押一家时还多了二百两。"

"为什么?"染坊的丙东家接了话。

"因为鸡蛋不搁一只篮子里。"陈师傅敲了敲桌子,"一只篮子打了,还有九只;一个行当垮了,还有九个行当。十个东家十种本事,十个地方十种运气。押一个,赌的是天意;押十个,靠的是分散。"

三个东家沉默了片刻。

周东家先开口:"陈师傅的意思,是让我把钱拆成几份,别只投那家利息最高的?"

"不止。"陈师傅竖起一根指头,"是让你投到不同的票号里去。十个东家,有十个门道;十家票号,有十种投法。这家偏好做茶,那家偏好做布,另一家专做丝的生意。你把钱给十家,就等于把银子分给十个东家、十种行当、十个地方,还横跨好几年。"

何东家眼睛亮了:"东家多,就不会只靠一个东家;行当多,就不会只押一个行当——这就叫分散?——所以我担的风险,也变成十份里的一份了?"

"对了。"陈师傅点头,"这就是为啥桥头老话讲,银子不能押一个铺子。铺子多了,这铺倒了那铺还在;一个行当跌了,别的行当可能正好涨。"

"那利息少了吧?"周东家又嘀咕。

"利息是少了点。"陈师傅乐了,"但你睡得着觉啊。"

三个人一齐笑起来。

后来周东家果然照办,把八百两拆成四份,投到四家不同的票号;何东家把四百两拆成三份;染坊丙东家更小心,一千两拆成了五份,还特意挑了五家做不同行当的。一年后,四家票号里有一家出了岔子——可没伤着三人的筋骨,反倒让他们更信服陈师傅那句话。

打那以后,十六铺桥头再有人来问"我这一大笔银子往哪儿搁",陈师傅的烟袋锅子一磕,开口就一句:

"先想想——你打算分几只竹篮?"

---

## 📖 原文定义

> 母基金通常会将所募集的资金分散投资于多只股权投资基金,从而避免了单只股权投资基金投资中依赖于某一基金管理人的风险。通过分散投资于多只股权投资基金,母基金的投资得以实现多样性,如投资阶段、时间跨度、地域、行业、投资风格等,从而使投资者可以有效地实现风险分散。

## 💡 对应点

| 故事元素 | 概念对应 |
|---------|---------|
| 三位东家合计的现银 | 母基金所募集的资金 |
| 周东家提议"全押一家利息最高的票号" | 单只股权投资基金/单一 GP 的集中风险 |
| 陈师傅讲王老板把银子分成十只竹篮 | 母基金"分散投资于多只基金"的核心机制 |
| 头一年有一家亏了一整篮,但九只篮有赚有平 | 通过多基金/多 GP 抵消单只基金的风险 |
| 十家票号做十种行当(茶/布/丝/米/染) | 投资阶段、时间跨度、地域、行业、风格的多样性 |
| 押一个赌天意,押十个靠分散 | 集中投资 vs 分散投资的风险差异 |
| 三位东家照办——周四份、何三份、丙五份 | 母基金通过子基金组合实现底层资产多元化 |
| 一年后有一家出岔子,但未伤筋骨 | 分散投资在实践上"有效降低单点风险" |
| 银子不能押一个铺子 | "不要把鸡蛋放在一个篮子里"——分散风险的口诀 |
