# （三）直接投资

## 🎯 寓言故事 —— 《钱庄娘舅的「搭车」》

江南水乡有个开钱庄的老顾，做了二十年生意，街坊都喊他"顾半城"。他手里常年攥着一笔银子，自己不会种田也不会跑船，就把钱借给各路做买卖的人，坐收利钱。平日里他把钱分成一二十份，散给城里几个他信得过的"掌柜"——开染坊的、开茶号的、跑运河的——让他们各自去打理。

这一年开春，老顾照例把银子分了出去。可没过多久，他手下叫林三的"掌柜"来请他喝茶。林三在运河边做漕运转口，认识的人多，眼界也广。两人在临河茶馆坐下，林三喝了两杯黄酒，压低声音说起一件事。

"顾叔，城东外那座大染坊，姓陈的老掌柜家里出了事，要把他手里三成的份子让出来。这座染坊我盯了半年——染的料子专供京城绸缎庄，每年稳稳出几百匹好货，换了别家未必守得住。"林三搁下酒杯，"我想吃下这块肥肉，可我手头现银只够七成。"

老顾端着茶杯没吭声。林三有本事，但这事儿大，他一个人吃下来风险也大。可这机会实在好——他派人打听过，那座染坊光去年一年的进项，就抵得上他钱庄半年的利钱。

"顾叔，"林三又添一句，"您不用一个人押。您愿意出剩下那三成，咱俩一道把这块肥肉端下来。账目我管，人手我出，您只管按份子收红。"

老顾回家想了三天。三月里他亲自跑到那座染坊转了一圈，看院子、看染缸、看晾布的架子，又找林三身边几个老伙计闲聊。回来后他做了一个决定：拨出一笔单独的钱，绕过那些"掌柜"，直接跟着林三一起去拿下这座染坊的份子。

钱到位那天，老顾签的不是借据，是和染坊陈家、林三三人共立的一张"合股文书"。文书写得明白：林三出七成银，外加全部人手打理；老顾出三成银，不插手日常，只按份子年年分利。染坊易主之后，老顾每季派人去账房翻一眼账本，别的他一概不问。

头一年，染坊果然如林三所说，稳稳妥妥出了四百多匹好料，老顾分到的红比他往常给任何一个"掌柜"放出去的银子都多。第二年运河上出了匪乱，漕运转口的买卖一片惨淡，好几个"掌柜"都来求老顾缓一缓还钱。可老顾靠着这座染坊，稳稳当当把日子过了下去。

年底老顾在堂屋盘点全年的收成，翻到染坊那一笔时忽然笑出了声。原来这些年他一直把钱散给各路"掌柜"，等于是坐在岸上看别人下河摸鱼——摸到几条算几条。可这一回，他亲自跟着最有本事的那一个，下河去摸了一回大鱼。他出的钱不算最多，可分到的，却比平时给"掌柜"们多得多。

更妙的是，他没脱离自己"出银子"的老本行——只是这一次，他没把银子远远地撒出去，而是贴着最有把握的那桩买卖，亲手按了上去。钱还是那笔钱，可落点不同，滋味就全变了。

---

**📖 原文定义**

> 直接投资是指母基金直接对非公开交易的企业股权进行投资。
>
> 在实际操作中，母基金通常扮演跟投角色，和其所投资的股权投资基金联合投资，依托子基金管理人的资源和能力获取优质项目。
>
> 在直接投资业务中，母基金能够在股权投资基金的投资项目中，挑选最具吸引力、与其现有投资组合最佳匹配的项目，因此，直接投资也自然成为母基金的一项业务类型。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老顾把钱散给各路"掌柜"坐收利钱 | 母基金传统的间接出资（放权、远端）模式 |
| 林三盯上的那座城东外大染坊 | 子基金管理人发现/锁定的优质项目 |
| 林三找老顾"一道端下这块肥肉" | 母基金扮演跟投角色与子基金联合投资 |
| 老顾出三成银、林三出七成银和人手 | 母基金出小部分资金，依托子基金的能力与资源 |
| 三人共立的"合股文书"与按份子分利 | 直接持有非公开交易企业的股权 |
| 老顾从"掌柜"中单独挑出最有把握的那一笔 | 从子基金项目中挑选最具吸引力的项目 |
| 染坊年年稳出好料，分红比"掌柜"们多 | 直接投资中能与现有组合最佳匹配的项目 |
| 钱还是那笔钱，落点不同滋味全变 | 直接投资作为母基金业务类型的自然延伸 |
