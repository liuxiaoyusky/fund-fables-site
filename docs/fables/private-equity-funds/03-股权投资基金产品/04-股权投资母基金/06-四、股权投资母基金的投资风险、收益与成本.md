# 四、股权投资母基金的投资风险、收益与成本

## ⚖️ 寓言故事 —— 《中间人》

老陈在江边跑船，攒了三百两银子。他想扩大船队，可一个人造船风险太大。听说镇上的钱庄掌柜老赵专门帮人挑船队、分银子，老陈心动了。

---

老赵的规矩很简单：老陈把银子交给他，他替老陈挑十条船队，每条投三十两。东边不亮西边亮，总有几条能赚钱。老陈算了算，比自己投一条船稳当多了。

可老赵要收两份钱。第一份，每年按老陈银子的两成收"跑腿费"——不管赚不赚，这笔钱照收。第二份，等年底赚了，老赵再从利钱里抽两成当"谢礼"。老陈一算：要是赚了五十两，先给老赵十两跑腿费，再给八两谢礼，自己只剩三十二两。比自己直接投船队，少拿不少。

老陈犯了难。直接投船队，不用给老赵分钱，可自己不懂行，万一挑错了，三百两全赔。找老赵，稳当，但赚的要分出去一大块。

他去找老周商量。老周是老跑船的，见识广。老周说："你直接投一条船队，要么大赚，要么大赔，像掷骰子。找老赵，他替你投十条，赚的不一定最多，但赔的一定最少。你花的那两份钱，买的是'不翻船'。"

老陈又问："那值不值？"

老周算了笔账："假设你自己投，三年里可能有一年翻船赔光，两年赚翻。平均下来，看着多，可翻船那年你睡不着。找老赵，三年都稳当，每年赚的不多不少，加起来可能比你自己的平均数少一点，但你能夜夜安睡。这差价，就是买心安的价钱。"

老陈咬咬牙，把银子交给了老赵。第一年，十条船队里三条赚了，两条平了，五条亏了。老赵按规矩收了跑腿费和谢礼，老陈到手二十两。不多，可他睡得着。

第二年，江面上起了大风，东边三条船队翻了。老陈心想：要是自己投的是那三条之一，三百两就全没了。可老赵替他分散投了十条，翻了三条，还有七条撑着。这一年，老陈到手十五两，比头年少，可毕竟没亏。

第三年，风调雨顺，十条船队里八条赚了。老陈到手四十两，老赵收了跑腿费和谢礼后，老陈还剩二十八两。三年加起来，六十三两。老陈算了算：要是自己三年前选一条最好的船队投，可能赚一百两；可要是选错了，可能只剩五十两。找老赵，三年六十三两，不多不少，像老周说的——"买的是心安"。

他后来明白了：老赵收的那两份钱，一层是"挑船"的工钱，一层是"稳赚"的谢礼。直接投船队，省了这两份钱，可省了钱，也省了稳当。中间人不是白当的，他的本事和分散，都是要花钱买的。

---

**📖 原文定义**

> 四、股权投资母基金的投资风险、收益与成本
>
> (一) 风险
>
> 股权投资母基金通过分散投资降低了风险。研究表明，投资于单只股权投资基金的风险较高，极高内部收益率和极低内部收益率出现的可能性较大。而投资于母基金的风险小于投资于单只股权投资基金，且在很大程度上降低了极高内部收益率和极低内部收益率出现的可能性。
>
> (二) 收益
>
> 股权投资母基金能够在分散风险的同时，取得与风险相匹配的收益。
>
> （三）成本
>
> 股权投资母基金管理人要向母基金投资者收取管理费和业绩报酬，因此，相比直接投资股权投资基金，投资者通过母基金间接投资股权投资基金需额外承担成本。
>
> 投资者投资母基金时，应考虑投资母基金相对于直接投资股权投资基金的超额收益和成本。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老赵替老陈挑十条船队分散投 | 母基金通过分散投资降低风险 |
| 自己投一条可能大赚或大赔 | 单只基金风险高，收益率波动大 |
| 老赵投十条，赚的不多不少 | 母基金降低极高和极低收益率的可能性 |
| 三年六十三两，比最好情况少、比最坏情况多 | 风险与收益相匹配 |
| 老赵收两份钱：跑腿费和谢礼 | 管理费和业绩报酬（双重成本） |
| "买的是心安" | 投资者需权衡超额收益与额外成本 |
| 直接投省了钱，也省了稳当 | 直接投资 vs 间接投资的成本收益权衡 |
