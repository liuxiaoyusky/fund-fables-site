# （二）投资二手基金份额

## 🔄 寓言故事 —— 《买旧船》

老陈在江边跑船三十年，手里有三条货船，年年赚利钱。可他渐渐发现，跑船这行，最磨人的不是风浪，是等待——新船从造好到回本，少说五年。他年过五十，等不起了。

---

一天，他在码头听说，老周的船队要散伙。老周跑了八年船，五条船，三条已经回本，还有两条正在跑远线，再过两年就能赚。可老周家里出了事，急着用钱，想把船队里自己的份额卖掉。

老陈心动了。他去找老周谈。老周说："我这五条船，八年下来，该踩的坑都踩了，该熟的线都熟了。你买我的份额，不是买新船等回本，是买已经回本的船，直接收利钱。"

老陈算了笔账：要是自己造五条新船，从砍木头到跑第一趟货，至少三年，再到回本，又五年，前后八年。买老周的份额，老周已经跑了八年，他接手就能收利钱，等于把八年等成了零年。

可他也有顾虑："老周，你的船我都没上去看过，怎么知道里头有没有暗伤？"

老周笑了："你买新船，船厂给你看的只是木头和钉子，船跑起来怎么样，谁也不知道。我这五条船，八年跑了多少趟、赚了多少、修了多少回，账本都在这儿。你一条条看，比买新船心里踏实多了。"

老陈翻了三天的账本。五条船，三条跑近线，稳当但利薄；两条跑远线，风险大但利厚。八年下来，近线的船从没亏过，远线的船有一年遇上大风，差点翻了一条，可后来补上了。老陈心里有了数——这不是赌，是买明货。

更关键的是，老周急着用钱，卖价比自己造新船便宜了两成。老陈一算：便宜的两成，加上省去八年等待，这笔账怎么算都划算。

他掏了银子，买下老周份额。头一年，三条近线船稳稳当当送来利钱；第二年，两条远线船也回了本。老陈坐在船头，看着江面上的新船来来往往，心想：那些造新船的人，还在等回本呢，我已经在数钱了。

后来老陈专做这行——不买新船，专买旧份额。谁急着用钱，谁想散伙，他就去谈。价格折扣、省去等待、账本透明，这三样好处，让他成了江面上最稳当的船主。

---

**📖 原文定义**

> （二）投资二手基金份额
>
> 投资二手基金份额是指母基金在股权投资基金募集设立完成后，对存续基金或其投资组合进行投资。
>
> 根据投资标的不同，母基金的基金份额投资业务可以分为两种类型：一是购买存续基金份额及后续出资额；二是购买基金持有的投资组合的股权。
>
> 近年来，母基金投资基金份额业务的比例不断增加，主要基于以下原因：
>
> 第一，价格折扣。股权投资基金主要采取私募形式，缺乏流动性，因此，基金份额投资一般都有价格折扣，从而提高收益。
>
> 第二，加速投资回收。在基金份额投资业务中，母基金投资的是存续期的股权投资基金，缩短了投资等待期，从而能够更快地回收投资。例如，假设一只股权投资基金达到盈亏平衡点需要7年时间，实现收益需要8~12年时间，那么母基金购买存续5年的基金可以将盈亏平衡点缩短至2年，将实现收益的时间缩短至3~7年。
>
> 第三，底层资产透明度高。在投资子基金的业务中，母基金进行投资时，子基金尚未进行投资，因此还不知道基金的资产组合，只是基于对子基金管理人的信任而投资。但在二手基金份额投资业务中，母基金投资的是存续基金，基金已经进行了投资，因此母基金能够得知底层资产组合。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老陈买老周跑了八年的船队份额 | 购买存续基金份额 |
| 老周急着用钱，卖价便宜两成 | 价格折扣（缺乏流动性导致） |
| 省去八年等待，直接收利钱 | 加速投资回收，缩短盈亏平衡期 |
| 翻三天账本，五条船运行情况一目了然 | 底层资产透明度高 |
| 老周八年踩的坑都踩了 | 存续基金已有投资历史和业绩记录 |
| 老陈专买旧份额不买新船 | 二手份额投资成为母基金的重要策略 |
