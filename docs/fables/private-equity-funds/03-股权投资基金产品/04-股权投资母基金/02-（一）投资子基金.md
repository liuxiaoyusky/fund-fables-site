# （一）投资子基金

## 🌱 寓言故事 —— 《开荒那年》

江南有个做丝绸生意的老程，膝下一儿一女，家底殷实。前年行情好，他一口气在城里盘下了三间铺面，手里还攥着一笔闲钱，足够后半辈子不愁吃穿。可老程闲不住，总想再押一把大的，把这桩生意做成真正能传家的字号。

那年深秋，朋友老邱带他去郊外看一片荒地。荒地连着一条缓坡，土质松软，远处就是京杭运河的支汊。老邱蹲下来抓起一把土，搓了搓，说："这片地我盯了一年，准备开春种桑树，养成之后，养蚕、缫丝、织绸全在一个院子里干下来，从根上把成本压下去。我自己种过、也卖过，懂行。"他拍了拍老程的肩膀，"老程，你出钱，我出力。三年后第一茬桑叶下地，五年后蚕棚能见回头钱。"

老程心里七上八下。回家路上，他想起前阵子听人说的两句话——钱放在铺面上，稳是稳，可一年也就那几个点的进账；如果跟老邱一道去开这片荒地，最快也要等三五年，桑苗才能派上用场。可那笔闲钱若一直趴在账上睡大觉，他又不甘心。

他盘算了好几天。最让他犹豫的是：老邱这次要做的，是头一桩。圈里没人试过，他自己过去种桑、卖苗、办蚕房，能不能把整条链子跑通，谁都没底。万一哪一环卡住，投进去的钱就要跟着打了水漂。

老程没急着点头。他托人打听老邱这几年的做派，又去老邱原先的苗圃实地转了一圈，看他怎么选苗、怎么嫁接、怎么养护。他还约了老邱在茶馆里坐下来，把账摊开算——头一年的投入要多少，三年后还要不要再添，五年后能不能回本。老邱对答如流，连哪月哪日该施什么肥都说得清清楚楚，账本也对得上。那一晚老程回家，翻来覆去想了半宿。第二天一早，他差人把钱送到了老邱的账房。

开春，老邱领着一帮人在那片荒地上忙活起来。桑苗一垄一垄地落进土里，蚕房也照着图纸慢慢垒起来。老程偶尔去转转，看着地里冒出的嫩芽，心里踏实了不少——钱没白出，这事儿是真的有人在认真做。街坊有别的富户听说这事，后悔当初老程没喊他们一起，问老程为什么没多拉几个人。老程笑了笑说："人家才刚动土，我先把钱送过去，就是信他这个人。要是人多嘴杂、主意乱飞，这事儿反而办不成。"

五年后的秋天，第一批蚕茧出了棚，洁白一片。老程站在蚕房门口，看着老邱一边记账一边指挥人装车，心里忽然明白了一件事——自己这笔钱，最值的时候，不在五年后拿到回报的那一天，而在五年前老邱举起锄头开荒、自己把银子送过去的那一刻。错过了那个开头，再多银子也搭不上这条船。

---

**📖 原文定义**

> 投资子基金是指母基金在股权投资基金募集时对基金进行投资，成为基金投资者。母基金发展初期，主要从事子基金投资业务，这是母基金的本源业务。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老程手头有闲钱，想再押一把大的 | 母基金的资金属性（集合资金、再投入） |
| 老邱准备开春动土种桑树 | 股权投资基金处于募集设立阶段 |
| 老程在别人还没动土时就把钱送过去 | 母基金在基金募集时进行投资 |
| 老程亲自考察老邱的过往做事、盘问账目 | 投资子基金时对管理人的考察 |
| 街坊富户想加入，老程说"早过了那个开头" | 募集完成（关门）后难以再进入 |
| 五年后蚕茧出棚，老程回看当年那一笔 | 投资子基金作为母基金的本源业务 |
