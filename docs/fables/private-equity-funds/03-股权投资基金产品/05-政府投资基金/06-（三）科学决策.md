# （三）科学决策

## 🧠 寓言故事 —— 《挑掌柜》

老周在县里管了二十年钱粮，深知一个道理：衙门的银子，不能撒胡椒面。全县几十个行当，有的自己能活，有的离了衙门就死。银子该往哪撒，得看准了。

---

有一年，县里的瓷器行出了个怪事。老匠人手艺好，可年轻人不愿学，嫌来钱慢。几间老窑陆续熄火，眼看这门手艺要绝。县太爷急，老周更急——瓷器是县里的招牌，没了它，县里的名声就塌了一半。

老周想了个法子。他去找钱庄掌柜老赵，说："衙门出三百两，你出七百两，凑一千两。但这回有个规矩：只借给瓷器行的人，别的不借。"

老赵不解："织布的、种茶的也缺钱，为啥不借？"

老周说："织布的、种茶的，自己就能活。缺银子，他们找亲戚借、找钱庄贷，总有办法。瓷器行不一样——手艺难学、来钱慢、风险大，年轻人不愿入行，老匠人想改良没本钱。这是'市场不管'的地方，衙门不管，就没人管了。"

钱庄开张后，老周亲自拟了份名单：借给想开新窑的年轻人、想改良釉色的老匠人、想引进新烧法的外乡师傅。名单上的人，老赵一一去看，觉得靠谱才借。三年下来，五家新窑开了火，两家老窑改良了手艺，还有一家试出了新釉色，城里人抢着买。

可麻烦也来了。县太爷的侄子想开绸缎庄，来找老周要银子。老周说："绸缎庄自己能活，衙门不借。"侄子闹到县太爷面前，县太爷也不好驳面子。

老周拿出当初立下的字据，指着上面一行字："只借给瓷器行。这是铁规矩，破了规矩，往后谁都想来借，银子撒光了，瓷器行还是死。"

县太爷哑口无言。侄子悻悻走了，转头去民间钱庄借了银子，开了绸缎庄。三年后，绸缎庄赚了，可老周不眼红——本来绸缎庄就能自己活，衙门不借，它照样活。瓷器行要是衙门不借，就真死了。

老周后来把这法子推广到全县。他画了一张图：县里的行当，分成三类。第一类，自己能活的，衙门不碰；第二类，快死的，衙门救；第三类，刚冒头的、有出息的，衙门扶。银子只往第二、第三类撒，第一类让他们自己折腾。

他还定了个"反向清单"：开赌坊的、放高利贷的、炒地皮的，一律不借。这叫"脱虚向实"——银子要落到实实在在的手艺和作坊上，不能飘在空中炒来炒去。

十年后，县里的瓷器行成了远近闻名的招牌，连京城的人都来订货。老周退休时，把那张图交给了继任者，只说了一句话："衙门的手，只能伸到市场够不着的地方。伸长了，抢了百姓的饭碗；缩太短，该救的救不活。"

---

**📖 原文定义**

> （三）科学决策
>
> 政府投资基金运行中的科学决策既包括通过科学的程序筛选基金管理人，也包括基金管理人在投资具体项目或企业上的决策，还包括基金退出时机的决策。选择优秀的基金管理人是科学决策最重要的一环，原则上，基金管理人应采取市场化方式通过竞争性遴选程序产生，基金管理人应具备与基金运作相适应的投资能力和管理能力，具有股权投资或相关基金管理经验，具有符合要求的投资管理业绩。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周亲自拟名单，老赵一一去看 | 科学决策包括筛选管理人和投资项目 |
| 老赵觉得靠谱才借 | 基金管理人具备投资能力和管理能力 |
| 只借给瓷器行，不借绸缎庄 | 明确投资方向，科学决策项目选择 |
| 县太爷侄子被拒绝 | 坚持原则，不干预具体投资决策 |
| 老周画三类行当图，定反向清单 | 科学决策的系统化方法 |
| "衙门的手只能伸到市场够不着的地方" | 科学决策的边界和原则 |
| 选择老赵作为钱庄掌柜 | 通过竞争性遴选程序选择优秀管理人 |
| 老赵有二十年钱庄经验 | 基金管理人应具备相关管理经验和业绩 |
