# （二）市场运作

## ⚖️ 寓言故事 —— 《县衙的账房》

江南某县，新到任的周县令接下一笔烂账：府库里搁着一百五十万两闲钱，是十年里从田赋、盐税结余下来的。钱闲着发霉，可地方上纺织、造船、茶叶三条产业死活做不大。钱怎么花、花多少、花错了谁担责——朝廷没给章程。

他先找师爷老陈商量。老陈是跟了县衙二十年的老人，办事稳当："大人，照老法子，找殷实商户挨家发低息银，让他们扩工坊。"周县令皱眉："查过账，三年发出去四十万两，织坊多了两间，可大户拿了银子去镇上买田置地，没进机杝。钱出了库，没进产业。"

周县令又想到第二条路——自己拍板。哪坊子缺钱、缺多少、值不值得投，他来断。他出身翰林，读过《农政全书》《天工开物》，自认懂行。头一个月，他拍板投了一家造船坊，坊主卷了钱跑南洋；第二个月，又扶持一个茶商，茶运到北边销路不对路，亏得底掉。主簿颤巍巍递上账册："大人，库银下去十二万两，账面回款……零。"

周县令一夜没睡。半夜把老陈叫进签押房，把账册往桌上一摊："我错了。我一个读书人，哪懂哪条船值造、哪筐茶值收？可钱不能不花——花不出去，产业就死在县里。"

老陈反倒笑了："大人，我陪您走一趟苏州。"周县令一愣。老陈说："苏州城里有一家'协成号'，专替东家打理银钱。不是放贷的，不是做官的，就是替人选买卖、按月出明细的。府里几家大商号都把钱交给他们打理。"周县令眼睛一亮——随即又迟疑："钱是府库的，交给外人打理，怎么信得过？"老陈慢慢说："不用'信'，用契。咱们出钱，约定规则：钱怎么用、用多久、谁去管、管到什么程度、赚了怎么分、亏了谁担、账目怎么对、何时收回——白纸黑字签清楚。中间再找一家独立的账房，专管钱袋子，按月给县衙出明细。这样县衙只管盯大方向、查大账目；钱怎么花、怎么收、怎么赚，交给懂行的人按规矩办。大人，您别坐进买卖里去当掌柜，您就做东家。"

周县令沉默良久，点头："拟章程，分工定责，按契办事。"

半年后，"协成号"带了两笔账回来：一笔参股织机坊四成，坊子当年开了两条新线；一笔参股船坞做小股东，每季度分一次红。账目送到县衙，独立账房核过，笔笔对得上。老陈私下问："大人，这事要是办砸了呢？"周县令答："砸了按契赔、按契认，我担我那份；赚了按契分、按契收。中间谁来告状，咱把契约、账册、议事录摆出来，自有公论。"老陈长舒一口气："钱进了产业，库里的银子活着，规矩也立住了。"

年底述职，周县令把章程、账册、议事录一并呈上。巡抚翻了翻，点头："政府出的钱，仍照政府规矩管；可钱一旦进了买卖，就让买卖的规矩去管买卖。出钱的管方向不管掌柜，做事的管买卖不管官印。赏罚按契，账目公开，这才像个'东家'。"

---

**📖 原文定义**

> 从基金运作原理来看，政府投资基金虽然有政府财政出资参与，但仍应当遵循股权投资基金运作的一般规律，应尊重市场经济规律和基金管理流程，从募集、设立、投资、管理、退出等各主要环节，充分发挥市场配置资源的决策性作用，设立市场化基金运作实体，明确投资者、管理人和托管人等各方权责，强调"利益共享、风险共担"，信守契约精神。
>
> 从政府投资基金运作实践来看，委托专业基金管理人进行市场化运作是政府投资基金目前主流的管理方式。政府部门不干预基金的日常经营管理和投资决策，各投资者按照"利益共享、风险共担"的原则约定收益分配和亏损负担方式。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 周县令把府库银子直接发给商户、自己拍板投资 | 行政化干预、违背市场化运作规律的失败做法 |
| 老陈带县令去找"协成号"专业团队 | 委托专业管理人进行市场化运作 |
| 出钱的县衙、办事的协成号、独立的账房三方 | 投资者、管理人、托管人各方权责分明 |
| 白纸黑字写清"钱怎么用、赚了怎么分、亏了谁担" | 利益共享、风险共担的契约精神 |
| 县衙只盯大方向和大账目、不坐进买卖里当掌柜 | 政府部门不干预日常经营管理与具体项目决策 |
| 钱进了织坊、船坞，按产业规律去赚钱 | 充分发挥市场配置资源的决策性作用 |
| 巡抚查契约、查账册、查议事录 | 尊重基金行业规律、按市场化规则办事 |
