# 《库银出的一份》

永丰镇的河码头叫十六铺，从前是个鱼市，后来鱼市搬走了，铺面空了一半。镇上卖麻的、卖油的、跑船的，都盯着那几间空铺，却没人敢下手。

民国三十八年的秋天，府城来了位姓邱的师爷，带着县衙户房管库银的赵先生，找到了镇公所的吴镇长。

邱师爷在茶馆里摊开一张契纸，对吴镇长说："府城的令下来了，要把这码头重新旺起来。库银的账里拨一份出来，由镇公所出本钱，找街坊一同出，按街面的规矩来开合股契。"

吴镇长端着茶碗没动："我认得'合股契'三个字。可街上的合股契都是买卖人自己凑的本钱、求的是挣头，朝廷字号出本钱，那是头一遭。"

邱师爷笑了："吴镇长担心的，正是府城让户房赵先生来对您说的——这合股契，朝廷先垫的本钱，但街面自个儿转，拿主意有根有据，防漏防砸锅都要紧。这叫'朝廷出主意带路'。"

吴镇长这才点了头。

第二日开大会，街坊分作两拨。

一拨是开油坊的沈家、做麻绳的周家、撑船的何老大，凑了一桌。沈家老大说："我这一行，从榨油到卖油，再到用油渣喂鱼，这链子本就断了多年。朝廷合股契若肯押我们这一行里最要紧的那一环，把这一行的链子接上、接长、接牢，这一行砸不烂，我们便出。"

另一拨是巷尾新开木匠铺的柳小满、磨豆腐的寡妇冯嫂、染布的周寡妇，凑了一桌。柳小满说："我做的榫卯是新花样买卖的硬本事，可要早押小字号、压长本钱、押硬手艺，街上的财主不肯。朝廷合股契若肯押我们这一桩，便让冯嫂的豆渣喂我们的新木匠模子，算我两人合一股。"

吴镇长听完两桌话，回到镇公所，把茶碗往桌上一顿："邱师爷，这合股契怎么开？是朝廷合股契自个儿直接押买卖，还是母合股契投子合股契？两桩买卖可不一样。"

邱师爷请户房赵先生答。赵先生说："扶某一行买卖的合股契，可让镇公所的合股契自个儿直接押买卖，也可做母合股契投子合股契，让子合股契去押那几间油铺、几间麻铺。扶新买卖的合股契，多用母合股契投子合股契的法子，让街面上的字号领头去做。"

吴镇长沉吟片刻，又问："这合股契带朝廷的令，按朝廷的令来，挣头少些，要不要紧？"

赵先生正色道："这合股契原本就不是只为挣头。本意是朝廷要这一行砸不烂、要有新花样买卖的硬本事。挣头是次一头的事，扶买卖是头一头的事。"

吴镇长听罢，提笔在契纸上画了押。

半月后，两桩合股契在镇公所前头挂牌。沈家、柳小满等十一家街坊入股，库银的赵先生坐在上头看账。邱师爷临走时回头对吴镇长说："明年的今日，码头若再不旺，府城是要来查账的。"

吴镇长望着那两扇新漆的牌匾，只说了句："朝廷先垫的本钱，街坊自个儿转，按街面的规矩来开合股契。库银出的一份，我吴某人担着就是。"

## 📖 原文定义

**政府投资基金的含义**：依据《关于促进政府投资基金高质量发展的指导意见》，各级政府通过预算安排，单独出资或与社会资本共同出资设立，采用股权投资等市场化方式，引导各类社会资本支持相关产业和领域发展及创新创业的投资基金。政府出资主要包括通过预算直接出资、安排资金并委托国有企业出资等方式。财政部门通常根据本级政府授权或合同章程规定行使政府出资人职责。遵循"政府引导、市场运作、科学决策、防范风险"基本原则。

**政府投资基金的特点**：从投资标的看，可分为产业投资类基金（围绕特定目标产业开展投资，重点投资产业链关键环节和延链补链强链项目，推动提升产业链供应链韧性和安全水平）和创业投资类基金（围绕发展新质生产力开展投资，着力投早、投小、投长期、投硬科技）。从投资方式看，产业投资类可采取母子基金架构或直投基金架构，创业投资类多采用母子基金架构。母子基金架构指母基金投向子基金进而投资于项目企业股权；直投基金架构指直接投资于项目企业股权。不同于单纯追求经济收益的市场化基金，政府投资基金通常具有较强的政策属性。
