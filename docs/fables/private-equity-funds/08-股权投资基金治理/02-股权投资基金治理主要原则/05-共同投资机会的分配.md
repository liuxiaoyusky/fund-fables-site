# 老周的"一担米"难题

## 一、开场：双份差事与两份契约

1952年开春，青柳村的合作化运动正如火如荼。村里德高望重的老周，被社员们一致推选为"青柳合作社"的社长。这是全村头一份以土地入股的农业生产合作社，二十户乡亲把自家的好地、耕牛、农具折成股金投了进来，指望着老周带着大家把日子过得红火。

老周是个能干人。他当社长不到半年，邻村的"金穗合作社"也找上门来，说他们的老社长病退了，想请老周兼着管一管。金穗社的社员也是种水稻的，土质、气候和青柳村差不多，连选种、施肥、灌溉的路数都如出一辙。老周想着自己精力还够，推辞不过，就答应了下来。

两份契约摆在老周面前。青柳社的契约上写得明白：老周要把最好的心思用在青柳社，选种、买肥、卖粮，事事以青柳社的收成优先。金穗社的契约也写着类似的话。老周当时没多想，提笔就签了字。他心想，反正都是种水稻，自己经验多，两份差事一起办，岂不是两份功劳？

## 二、冲突：一担好米，两个社都要

转眼到了秋收后的冬闲时节。县供销社的老主任私下找到老周，说有一批上好的"珍珠米"良种，是省农科所刚培育出来的新种，产量高、抗病强，全县就分到这么一担。老主任知道老周懂行，特意先让他过目。

老周眼睛一亮。这良种要是落到自己手里，明年合作社的产量准能翻一番。可他刚要开口，心里却"咯噔"一下——他现在是两个社的社长，这良种该给哪个社？

青柳社是自己一手拉扯起来的，二十户乡亲眼巴巴盼着他拿主意。可金穗社的契约上也白纸黑字写着，自己得为金穗社谋最好的出路。更要命的是，两个社的章程里写的都是"以水稻种植为主，兼营副业"，投资策略一模一样。这良种给青柳社，金穗社的社员知道了，准要骂自己偏心；给金穗社，青柳社的人更饶不了他。

老周试着打马虎眼，说"先拿回去研究研究"。可县供销社的老主任是个急性子，说："老周，这良种抢手得很，你今个儿不拿定主意，明天我就给别的社了。"

老周急得额头冒汗。他想起上个月，自己给青柳社多批了两袋化肥，金穗社的会计就嘀咕了半个月；后来给金穗社换了一台新水泵，青柳社的社员又在背后说闲话。如今这良种是实打实的"投资机会"，分给谁都是一碗水端不平。

他咬咬牙，想了个"折中"的法子：把良种一分为二，两个社各一半。可老主任摇头："这良种讲究纯种繁育，拆开就糟蹋了。"

老周又提议："先给青柳社，明年收了种子再分给金穗社。"可转念一想，明年的事谁说得准？万一青柳社的社员把种子私藏了，或者金穗社等不及闹起来，自己两头不是人。

## 三、转折：张叔翻出旧账，规矩必须立在前头

正当老周左右为难之际，村里的老会计张叔来了。张叔在合作社的账房里干了半辈子，眼睛毒、心思细，一眼就看出了老周的窘境。

张叔没急着说话，先翻开青柳社的契约，指着其中一条念道："社长须以本社利益为最高准则，不得因兼任其他职务而损害本社权益。"又翻开金穗社的契约，也有一条差不多的话。

"老周啊，"张叔放下契约，慢悠悠地说，"你当初签这两份文书的时候，可没想着会有今天这局面吧？"

老周苦笑："张叔，我哪知道会有这'一担米'的难处？"

张叔正色道："问题就出在这儿。你兼两个社的社长，本来就该在入社的时候跟两边社员说清楚——要是遇到今天这种'好机会只有一个、两个社都想要'的情况，该怎么办？规矩没立在前头，事到临头才想补，谁服你？"

老周低头不语。

张叔接着说："我听说城里的大合作社，管这叫'机会分配'。社长要是同时管好几个社，必须在章程里写明：遇到这种'一担米'的事，怎么分、谁先谁后、按什么规矩来。而且这规矩不能社长一个人说了算，得在社员入社的时候就讲明白，让大家都点头同意，写进契约里。"

"那该按什么规矩分？"老周问。

张叔捋了捋胡子："常见的有几种。一种是'轮流优先'——今年青柳社先挑，明年金穗社先挑；一种是'按规模比例'——哪个社的股金多、社员多，就多分一些；还有一种是'竞价分配'——哪个社肯出更高的代价，机会就归谁。不管哪种，都得事先写在契约里，不能临时拍脑袋。"

老周恍然大悟："我这就召集两个社的社员，把这事摊开了说，重新立规矩！"

## 四、结局：白纸黑字定章程，一碗水端平

第二天，老周把青柳社和金穗社的社员代表请到一块儿，又把县供销社的老主任也请来做见证。他把"一担米"的难题原原本本讲了一遍，然后提出了一套分配规矩：

第一，两个社的社长可以由同一个人兼任，但在任何一个社的投资（比如买良种、购农具）还没完成七成之前，不得再兼管第三个种水稻的社。这是为了防止社长摊子铺太大，顾此失彼。

第二，遇到像"珍珠米良种"这种两个社都想要的好机会，一律按"轮流优先+规模比例"的复合规则分配：先按两个社的股金规模比例确定大致份额，再轮流决定哪一年哪个社享有优先选择权。具体细则写进两家合作社的章程附件里。

第三，这套分配规矩必须在每年春耕前的社员大会上重新宣读确认，有新社员入社时也要重点说明，确保人人知情、人人同意。

县供销社的老主任听了，连连点头："老周，你这做法比城里有些大社还规矩。机会分配最怕的就是'暗箱操作'，你把它摊在太阳底下，谁都没话说。"

果然，章程重新修订后，"一担米"的良种按规矩分给了当年轮到的青柳社。金穗社的社员虽然眼馋，但知道明年就轮到自己，又有比例份额保底，也就心平气和了。老周从此再遇到类似的好事，再也不犯愁——规矩在前头，照着办就是，谁也不得罪，谁也挑不出理。

年底，两个社的产量都比往年高出两成。老周在全县合作社表彰大会上发言，只说了一句话："当好社长，靠的不是本事大，而是规矩立得早、立得明。"

## 原文定义与对应点

**原文概念一：管理人同时管理策略相同基金**

> 在基金投资期内，管理人或其关联方管理或服务于与本基金的投资策略相同（或相交叉）的基金。基金合同中应当约定当基金尚未完成总认缴规模一定比例（如70%）或更高的投资额的，管理人或管理人关联方不得管理或服务于其他相同（或交叉）策略的基金，要管理或服务于其他不同策略的基金，应经全体投资者一致同意或经全体投资者认可的决策机制决策通过。

**原文概念二：共同投资机会的分配**

> 管理人及其关联方所获得的符合本基金的投资目标和投资策略的投资机会，同时也符合其所管理或服务的其他基金的投资目标和投资策略。管理人应制定明确的投资机会分配规则，并在募资时向投资者重点披露并征得投资者同意后约定到基金合同之中。
