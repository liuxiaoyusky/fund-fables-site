# （一）完善的管理人内部治理

## ⚖️ 寓言故事 —— 《三灶分炊》

民国二十五年，杭州城西"瑞丰号"大米行的陶掌柜过世，留下一句遗嘱："三兄弟共同执掌瑞丰号。"老大陶继祖稳重，老二陶继宗精明，老三陶继嗣莽撞。瑞丰号有水田、仓房、碾坊三摊子生意，账上原本由陶掌柜一人拍板，三个儿子从未分过家。

丧事办完的第二天，老二陶继宗就把账本抱到了堂屋正中。"大哥，进货出货归我管，三弟跑外勤，大哥你坐镇田亩，至于碾坊，每年交给三弟看管。"老大陶继祖听了，眉头拧成结："老二，你把持了米行的命脉，进货出货一人说了算，那往后咱三兄弟听你一人的？"陶继宗笑："亲兄弟还信不过？"老三倒是爽快："二哥让我跑外勤我就跑。"

头一年下来，米行账面上没出大漏子，可老二的院子翻新了两回，还添了辆黄包车。年底分账时，老三跳起来："二哥，你那'损耗'就报了一百八十块大洋，损耗哪里去了？"陶继宗把账本往桌上一摊："白纸黑字，爱查查。"老大把账本接过去翻了一夜，眉头越皱越紧。

开春后，老大把两个弟弟请到后院石桌前，摊开一张纸。"老二，咱爹那句'共同执掌'，可不是'一人独断'。我琢磨了半个月，瑞丰号得换章程。"

"头一桩，米行立三灶——账房、库房、碾坊三摊子事，三兄弟各管一摊，谁也碰不到谁的钱袋子。进货的钱不能过出货的账，碾坊的租子不能和米行的利混在一处。"

"第二桩，进出单笔过了五十块大洋的，三兄弟合议，少数服从多数，议完签字画押。可平时一摊子里的零碎小事，各自做主，不必事事过会，省得办事拖泥带水。"

"第三桩，年底算账改法子。各灶自负盈亏，兄弟三人按'一份底薪加两成活钱'分红，跑得好多拿，跑得差少拿，平时不混着算。"

"第四桩，碾坊不能由一人看管。三兄弟轮着盯，每人三个月，账月月清。"

陶继宗看了老大半天："大哥这章程，是冲着弟弟来的？"老大摇头："老二，你去年'损耗'多少，咱三兄弟心里都有一笔账。我这章程不是要削你的权，是要让你在阳光下掌勺——账清了，咱三兄弟的饭碗才都端得稳。"陶继宗又看了半晌，签了字，老三也跟着签了。

那年夏收，南乡佃租没拖；秋天临安米价上扬，老大做主提前出手，多赚了三百块大洋；第二年开春，碾坊的账清清楚楚，三兄弟把上一年的总账一摊开，陶继宗自己先乐了——"三灶分炊，原来锅里煮的饭一样多，谁也没少吃一口。"

---

**📖 原文定义**

> （一）完善的管理人内部治理
>
> 基金管理人应完善自身治理，设置权责清晰、相互制衡的治理结构。管理人股东（合伙人）通过股东会（合伙人会议）履行其职责，不应干涉管理人日常经营管理。若管理人股东（合伙人）兼任管理人高级管理人员，应具备相应的专业能力。
>
> 基金管理人可通过股权激励、业绩分成、员工跟投等措施，建立系统有效的内部约束激励机制。管理人建立的员工跟投制度，优先考虑在一定范围内采取强制跟投方式，要求投资项目组成员进行跟投且不得自愿进行选择性跟投，使执行和决策项目的团队成员的利益与项目成败挂钩。

| 故事元素 | 概念对应 |
|---------|---------|
| 陶掌柜遗嘱"三兄弟共同执掌" | 股东（合伙人）通过股东会（合伙人会议）履行其职责 |
| 老二想把进货出货收归自己一人 | 股东（合伙人）干涉管理人日常经营管理的反面 |
| 老大设立账房、库房、碾坊"三灶分炊" | 设置权责清晰、相互制衡的治理结构 |
| 三兄弟各管一摊、互不染指对方的钱袋 | 内部权责清晰、各司其职、相互制衡 |
| 单笔过五十块大洋的大事合议签字 | 重大事项集体决策，避免一人独断 |
| 平时零碎小事各自做主不事事过会 | 日常经营管理不干涉，保证运营效率 |
| 碾坊三兄弟轮值、账月月清 | 关键岗位轮岗制，防止一人长期把持 |
| "一份底薪加两成活钱"分红 | 业绩分成机制，把团队利益与经营成果挂钩 |
| 老大说"让你在阳光下掌勺" | 系统有效的内部约束激励机制 |
| 陶继宗最后感慨"三灶分炊谁也没少吃一口" | 治理结构完善后，所有人利益同向、激励相容 |
