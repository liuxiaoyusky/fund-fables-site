## 老周的"第二把算盘"

### 一

一九五三年春，柳树湾的"互助粮仓"已经运转了两年。老周是粮仓的"当家人"，负责把社员们凑起来的谷子借给村里有急用的乡亲，收回来的利息再按股分给大家。日子久了，老周在十里八乡有了名声，连邻村的"丰收仓"也请他去做"顾问"。

起初，老周两头跑，大家觉得是好事——见多识广嘛。可渐渐地，社员们发现不对劲了。

那年夏天，邻县闹旱灾，粮价飞涨。柳树湾粮仓凑的谷子，按理说该借给本村急需种子的农户，利息稳当。可老周却把一大半谷子拨给了"丰收仓"去放贷，说是"那边利息更高"。结果丰收仓赚了差价，柳树湾的社员们只分到薄薄一层。

更蹊跷的是，老周自己的外甥在邻村开了间磨坊，急需周转。老周没跟任何人商量，就从粮仓里借出一批谷子，利息定得比旁人低三成。社员们事后才知道，气得直跺脚，却又说不出个章程——粮仓的规矩里，压根没写"当家人做这种事该怎么办"。

### 二

矛盾在秋收时爆发了。

老周又做了一桩事：邻村有人想低价出兑一片河滩地，老周打算用粮仓的谷子垫资买下，再转手租给外乡人种莲藕。这买卖若成，老周自己能抽一笔"中人钱"。可他既没告诉社员们这片地的底细，也没说那外乡人是他远房表亲。

几个老社员凑在村口大槐树下，烟袋锅子敲得梆梆响："再这么下去，粮仓成了他老周自家的钱匣子！"

可光生气没用。有人提议"把老周撤了"，又有人摇头："撤了谁当？再说规矩里也没写能撤啊。"

正当众人一筹莫展时，村里的老会计张叔拄着拐杖来了。张叔在旧社会给镇上钱庄记过账，见过世面。他听完后，在树干上磕了磕烟袋："依我看，得立三条规矩。"

"头一条，"张叔竖起一根指头，"当家人再想做这种两头沾的买卖，不能他一个人拍板。得由几个信得过的社员凑在一起，专门盯着这些'犯嘀咕'的事。"

"第二条，"他竖起第二根指头，"若事情实在掰扯不清，得请外头不相干的人来看——比如镇上公家的账房先生，或者县里的文书，让他们说句公道话。"

"第三条，"张叔的声音沉下来，"若当家人真把粮仓的谷子糟蹋了、私吞了，得有个说法——该赔的赔，该担责的担责，不能一句'下不为例'就揭过去。"

### 三

张叔的话传开，社员们连夜开了大会。

会上，众人推举了五位公道正派的老社员，组成一个"议事小组"——专管那些"当家人自己拿不准、社员们心里犯嘀咕"的事。老周想再做河滩地那桩买卖，就得把来龙去脉写成条陈，交给议事小组审。小组里若有人觉得不妥，老周就得把外乡人的底细、地价怎么定的、转手租给谁，一桩桩说清。

规矩还写明：若事情牵扯到老周的亲戚、老周在外头兼的差事，或者老周自己也能捞着好处的买卖，议事小组可以直接请镇上公家的账房先生来查账。查账的钱从粮仓的公账里出，老周不得阻拦。

最要紧的一条写在最后：若老周瞒着社员做了亏待粮仓的事，查实之后，他得用自己的家产补上窟窿；若补不上，就报给区里的合作社联社，该处分处分，该法办法办。老周当初入伙时押在粮仓里的"信物钱"——那三亩水浇地的地契，也得先扣下作保。

老周起初不情愿，拍着桌子说"你们这是信不过我"。张叔慢悠悠回了一句："不是信不过你，是信不过'人'。旧社会钱庄的掌柜，哪个不是精明人？可没规矩盯着，精明人就变成贪心人。"

### 四

规矩立下的第二年，邻村又闹了旱灾。

这一次，老周照旧想拨谷子去"丰收仓"放贷。他写好条陈，递到议事小组。小组里五个老社员，三个投了反对票——"本村王寡妇等着种子下田，你倒把谷子借给外村人赚高息？"

老周不服，说"利息高了，社员们分红也多"。小组便请了镇上公家的账房先生来核账。账房先生翻了两天的账，回禀说："若借给外村，本村农户误了农时，来年粮仓收不回种子粮，反而要亏。"

老周哑口无言。议事小组的决议贴在了村口：谷子优先借给本村农户，利息按老规矩收，不许另开"高息外借"的口子。

老周起初觉得束手束脚，可日子久了，他发现粮仓的名声反而更好了。外村人来借粮，都说"柳树湾的规矩硬，当家人不敢胡来"。社员们分红虽没有"高息外借"时那么刺激，却年年稳当，没有一年落空。

又过了几年，老周年纪大了，把当家的位子交给了年轻人。交接那天，他对新当家的说了一番话："我这两把算盘，一把算公账，一把算私账。公账那把，得让议事小组看着；私账那把，得让自己管严。两把算盘不能混，混了，就是糊涂账。"

村口大槐树下的石碑上，刻着当年张叔立的三条规矩，字迹被风雨洗得发白，却没人敢动它一个字。

## 原文定义与对应点

**教材原文（第103行）**

> 基金合同应建立利益冲突管理机制。从基金治理角度，除非投资者一致同意，否则应当建立由投资者代表构成的咨询委员会（或类似机构），建立决策机制，对涉及其他利益冲突的事项进行独立决策。必要时引入第三方中介机构介入，同时建立问责机制。

> 同时，相关制度应当包括关联交易和其他利益冲突范围的界定、不同利益冲突情形的处理方式、制度披露和修订的程序以及**相应的损失赔偿规定**。

**故事对应点表**

| 教材要点 | 故事映射 |
|---------|---------|
| 基金合同应建立利益冲突管理机制 | 社员大会连夜立下的"三条规矩"，写入粮仓章程 |
| 由投资者代表构成的咨询委员会（或类似机构）进行独立决策 | "议事小组"——五位公道老社员，专审当家人"犯嘀咕"的事 |
| 必要时引入第三方中介机构介入 | 镇上公家账房先生被请来核账，查账钱从公账出 |
| 建立问责机制 | 查实亏待粮仓的事，老周须以家产补窟窿；地契扣下作保；报区里联社处分 |
| 损失赔偿规定 | "用自己的家产补上窟窿""三亩水浇地的地契扣下作保" |
| 除非投资者一致同意 | 议事小组投票机制，老周个人无法强行通过 |

**核心概念**

利益冲突管理机制的本质，是通过**制度化的制衡设计**（投资者代表机构 + 独立第三方 + 问责赔偿），防止管理人利用信息优势和职务便利损害投资者利益。它不是对管理人的"不信任投票"，而是对"人性弱点"的预先设防——正如张叔所说："不是信不过你，是信不过'人'。"
