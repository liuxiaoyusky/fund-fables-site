# 7. 境外投资者

## 📜 寓言故事 —— 《洋商过桥》

清河镇来了个金发碧眼的洋商，名叫约翰。他背着一箱银洋，想在镇上找几个靠谱的作坊投银子。可约翰不懂清河镇的话，也不认得衙门的路，手里的银洋还不能直接花——得先换成镇上的铜钱。

约翰去找县衙的通译官周明。周明说："洋商要在镇上投银子，以前规矩严得很——得先证明你家底厚、来历正，再一层层报上去批。等批下来，半年过去了。"约翰急了："我大老远来，等不了半年。"周明笑道："别急，近年县衙在几个大码头开了'洋商投银'的试点，规矩松多了。"

周明带约翰去了码头试点。那里的规矩果然不同：约翰只要填一张表，说明自己从哪儿来、带了多少银洋、想投什么作坊，三天就能批下来。银洋换铜钱也简便，在码头钱庄一次换清，不用跑三趟衙门。更妙的是，以前洋商只能投还没挂牌的作坊，现在试点放宽了——挂牌的铺子也能投，只要是通过私下协议买的份额；连那些别人欠账收不回来的"烂账"、空着的铺面，也能投。约翰喜出望外："这比我家那边还方便！"

周明还告诉他一个窍门："你若有十万两银洋，不必全投给一个作坊。码头允许你立一个'洋商投银会'，把十万两分成几份，分别投给不同的作坊。哪边赚了就多投，哪边亏了就少投，灵活调配。"约翰当即立了一个投银会，把银子分成三份：一份投给老周酒坊，一份投给新织坊，一份留着看机会。三年后，三份银子都涨了。约翰在码头酒馆里逢人就说："清河镇这'洋商投银'的桥，搭得好。我不用懂你们的话，也能跟着赚。"

---

**📖 原文定义**

> 境外投资者主要通过合格境外有限合伙人（Qualified Foreign Limited Partner, QFLP）试点渠道投资于中国境内的股权投资基金。QFLP政策通过逐步放宽的投资者条件、相对便利的结汇操作和简化的行政审批程序，使得境外投资者能够更便捷地参与境内股权投资市场。境外投资者能够将外汇资金兑换为人民币，进而直接投资于中国境内的非上市公司股权。近年来，一些试点地区陆续推出了更为宽松的政策，例如，将投资范围从非上市企业股权扩展至上市公司非公开发行和交易的股权、大宗交易、协议转让、不良资产投资以及非自用不动产等领域。此外，部分试点地区还允许QFLP试点机构在总量管理模式下，发起设立一只或多只试点基金，灵活调剂单只试点基金规模。QFLP为境外资本提供了便捷的投资渠道，促进了境外投资者对中国境内股权投资基金的投资。
>
> 境外投资者因各自不同的背景展现出多样化的投资偏好。一些投资者特别重视投资的安全性和收益性，倾向于选择能够提供稳定回报的投资机会。另一些则更关注投资与自身产业的协同效应。亦有一些投资者将目光投向特定产业领域，如科技创新、绿色能源、医疗健康等，这些领域通常具有强劲的增长潜力，或与全球可持续发展等趋势相契合。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 约翰不懂话、不认得衙门路 | 境外投资者参与境内市场存在障碍 |
| 码头试点规矩松、三天批下来 | QFLP试点：逐步放宽投资者条件、简化行政审批 |
| 银洋换铜钱一次换清 | 相对便利的结汇操作 |
| 以前只能投没挂牌的，现在挂牌的也能投 | 投资范围从非上市企业扩展至上市公司非公开发行等 |
| 烂账、空铺面也能投 | 扩展至不良资产投资、非自用不动产等 |
| 十万两分三份灵活调配 | 总量管理模式下发起设立多只试点基金，灵活调剂规模 |
| 三份银子分别投酒坊、织坊、留机会 | 境外投资者多样化投资偏好：稳定回报、产业协同、特定领域 |
