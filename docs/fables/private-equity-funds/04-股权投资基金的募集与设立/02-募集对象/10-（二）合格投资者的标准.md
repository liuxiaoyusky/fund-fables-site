# （二）合格投资者的标准

## 📜 寓言故事 —— 《门槛》

清河镇老周商会的门槛，向来不高。老周觉得：有钱就能进，投多投少随你便。可那年的"张寡妇事件"让县衙发了火，连夜出了新规。师爷老李带着告示来到商会，往柱子上一贴："从今日起，想进商会投银子，得过三道门槛。"

第一道门槛："你识不识得风险？"老李说，"投作坊不是存银子，可能赚也可能亏。你得证明自己懂这个理。"怎么证明？个人来说，要么你手里有值三百两的家当（房子、地契、存银都算），要么你近三年每年进项不少于五十两。对铺子来说，你的账上净资产得有一千两以上。达不到的，请回。老周嘀咕："这岂不是把穷人都挡在外面？"老李正色："不是嫌穷，是怕穷人把棺材本赔光。手里有三百两家当的，亏了三成还能活；手里只有十两的，亏了三成就得跳河。"

第二道门槛："你投多少？"老李说，"单只商会的一只'份子'，最少得投一百两。不能你拿五两银子进来，亏了哭天喊地。"老周点头：一百两是个整数，能投得起的人，自然家底不会太薄。这叫"以钱筛人"——不是信不过人，是让人自己掂量：我有没有资格坐这桌牌局？

第三道门槛最特别。老李说："有些人，不用过前两道门槛，直接进来。"老周问："谁？"老李数了数："镇上的养老会——他们管着全镇老人的棺材本，规矩比咱们还严；县衙批过文的正经'投银会'——他们专门帮人挑作坊，有县衙盯着；还有商会自己的掌柜和伙计——你们天天看账、挑作坊，比谁都懂行。这些人，县衙认他们是'当然的识货人'，不用查家底、不用验进项。"老周恍然大悟：养老会的钱有专人看管，投银会有县衙备案，自己人天天在一线——这些人天生就站在门槛里面。

告示贴出后，商会门前冷清了几天。张寡妇再来，老周客气地请她看告示："嫂子，您家底不够三百两，进项也不够五十两，这门槛您迈不过去。不是我不收，是县衙的规矩。您若有银子，存到钱庄去，那有县衙的印，稳当。"张寡妇悻悻离去，但老周心里踏实了——门槛之内，都是扛得住风浪的人。

半年后，钱庄赵掌柜来投银子。老周一看他的账：净资产两千两，近三年每年进项八十两，投一百两不在话下。更妙的是，赵掌柜自己就是钱庄的掌柜，看过百家账簿。老周连风险都没多解释，赵掌柜自己就问："你们投了几家作坊？各是什么行当？退出怎么安排？"老周一一作答，两人当场签了契约。赵掌柜走后，老周对伙计说："门槛不是挡人，是帮人。迈得进来的，自己心里有数；迈不进来的，县衙替咱们挡了麻烦。"

**📖 原文定义**

> 合格投资者制度起源于美国。经历了1929—1933年的经济“大萧条”后，美国对证券行业的监管体系进行了重构，并出台了一系列法律。这些法律要求基金管理人和基金必须向美国证券交易委员会申请注册。符合条件的非公开发行的股权投资基金可以豁免注册，其中的一个条件是必须向合格投资者募集。
>
> 西方国家的合格投资者标准经历了一些变化。现行的合格投资者标准通常具有以下四个特点：首先，原则性地要求投资者具备一定的风险识别和承担能力；其次，要求投资者认购/申购的基金份额达到某一最低标准；再次，根据投资者的资产规模或收入水平来评估其风险承担能力；最后，某些持有特定金融牌照（例如商业银行）或其他大型专业投资机构（例如养老基金）及其高级管理人员被认定为当然的合格投资者。
>
> 我国的合格投资者认定标准与国际上主要国家的标准基本一致。根据《私募条例》，合格投资者是指达到规定的资产规模或者收入水平，并且具备相应的风险识别能力和风险承担能力，其认购金额不低于规定限额的单位和个人。中国证监会在《私募投资基金监督管理暂行办法》（以下简称《私募管理办法》）中进一步明确，私募基金的合格投资者是指具备相应风险识别能力和风险承担能力，投资于单只私募基金的金额不低于100万元且符合下列相关标准的单位和个人：（1）净资产不低于1000万元的单位；（2）金融资产不低于300万元或者最近三年个人年均收入不低于50万元的个人。
>
> 在一般合格投资者界定的基础上，《私募管理办法》将以下四类投资者视为当然合格投资者：一是社会保障基金、企业年金等养老基金，慈善基金等社会公益基金；二是依法设立并在中国证券投资基金业协会备案的投资计划；三是投资于所管理私募基金的私募基金管理人及其从业人员；四是中国证监会规定的其他投资者。后续，中国证监会在《关于加强私募投资基金监管的若干规定》中进一步补充规定：国务院金融监督管理部门监管的机构依法发行的资产管理产品、合格境外机构投资者、人民币合格境外机构投资者视为《私募管理办法》规定的当然合格投资者，不再穿透核查最终投资者。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 县衙出三道门槛新规 | 合格投资者制度建立，源于大萧条后监管体系重构 |
| 第一道：识不识风险，个人三百两家当或年进项五十两 | 原则性要求风险识别和承担能力；根据资产规模或收入水平评估 |
| 铺子账上净资产一千两以上 | 单位净资产不低于1000万元 |
| 第二道：单只份子最少投一百两 | 认购/申购基金份额达到最低标准（100万元） |
| 第三道：养老会、县衙批文的投银会、商会自己人 | 当然合格投资者：养老基金、备案投资计划、管理人及其从业人员 |
| 钱庄赵掌柜自己看过百家账簿，不用多解释风险 | 持有特定金融牌照或高级管理人员被认定为当然合格投资者 |
| 门槛不是挡人是帮人 | 合格投资者制度保护投资者，促进健康合作关系 |
