# 1. 母基金

## 🏺 寓言故事 —— 《山中老药客》

秦岭深处的青崖镇，山民靠采药为生。每到秋天，镇上会来许多外乡药商，挨家挨户地收药。药商挑货有一套路子：看成色、捏干湿、闻药味。

镇上有个叫苏长河的老药客，做这行已经四十年。他自己不进山采药，也不去外地收药。他一年到头只做一件事——挑药商。

每年秋天他都坐在镇口那棵老槐树下，铺一张草席，摆一壶茶，把来收药的药商一个一个请到跟前聊。聊什么？聊他去年收的那批黄芪是从哪几户山民手里收的，怎么烘的；聊他前年赔过的一笔款，是怎么栽进去的；聊他带的伙计，懂不懂药性。

有一年秋天，苏长河照旧坐在槐树下。来了一个姓张的药商，要收镇上全年的药。苏长河把他请到跟前，问："张老板去年收了多少货？走的哪条道？"

那姓张的脸一红，含含糊糊地答不上来。原来他是个新贩子，第一年没挣到钱，心里急着翻本，逢人就许大价钱。山民们被他的高价吸引，一窝蜂把药都卖给了他。结果这批药在路上受了潮，到了汉口药店，掌柜的只肯给半价，姓张的赔得连回家的盘缠都凑不齐。

第二年春天，山民们聚在苏长河家的小院里议论纷纷。有人说以后得自己挑药商，不能再让人骗了。可山民们对药材行情两眼一抹黑，谈何挑人？

苏长河端着茶碗听大家说完，慢悠悠地说："你们不用自己挑。我替你们挑。"他指了指自己，"我四十年的功夫，不在认药上，在认人上。哪个药商靠谱、哪个药商会坑人，我心里有一本账。从今往后，你们把药先攒在我这儿，我挑几个最稳当的药商，替你们分批出手。哪家药商出了问题，我挡在前头。"

山民们将信将疑地点头。头一年，苏长河挑了三个老药商，又带了一个新人——就是去年栽了跟头的姓张的。"你去年赔的，不是因为眼力差，是因为心太急。"苏长河对他说，"今年跟我走一趟，看看我怎么做。"

到了秋天，苏长河亲自陪着姓张的上山，看他怎么分辨山民手里的货色，怎么在阴雨天保护药不受潮。姓张的越看越惭愧，自己以前收药就是闭着眼睛抓一把，谈不上半点章法。

三年下来，苏长河给山民们挑的药商队伍稳定到五个。这五个里有跑汉口的，有跑广州的，有跑成都的。山民们跟着他，再没吃过那年张老板那样的大亏。而苏长河从山民们卖出的每一笔药里抽一点点——不多，但五家药商加起来，一年下来也够他一家老小衣食无忧。

又一年秋天，外地来了个大药商，要独家包下青崖镇全年的药。山民们有些心动。苏长河摆摆手："独家就意味着一棵树吊死。他今年给你高价，明年要你降价，怎么办？五条道，五家商。这家跌了那家补，这条断了那条通。只要山民们跟着我，我就能一直把这条路踩稳。"

**📖 原文定义**

> 股权投资母基金的核心投资策略是将募集到的资金投向多个不同的股权投资基金。近年来，母基金在项目直投和跟投方面的趋势不断增强，提升了其在项目选择和资源配置中的影响力及市场竞争力。同时，面对行业普遍的退出压力，母基金也在积极探索投资基金二手份额等投资策略，以提高资本的流动性和退出效率。母基金通过多元化的投资方式，实现了投资组合的多样性和分散性，降低了投资的整体风险，不仅帮助其投资者获得稳定收益，也为市场提供长期、专业的资金来源。
>
> 母基金注重投资的灵活性和多样性，力求在风险可控的前提下，实现资本的有效配置和增值。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 山民们自己把药卖给药商 | 普通出资人直接面对众多基金 |
| 山民们不识药、不懂行情 | 普通出资人缺乏专业识别能力 |
| 苏长河四十年的认人功夫 | 母基金的核心专业能力——筛选与风险识别 |
| 苏长河不采药、专挑药商 | 母基金本身不直接做项目，而是投向其他基金 |
| 苏长河挑选 3-8 个药商组成稳定队伍 | 母基金将资金分散投向多个不同的基金 |
| 五个药商跑汉口、广州、成都 | 投资组合的多元化与分散性 |
| 姓张的新药商跟着苏长河学规矩 | 母基金在投后管理中提升所投基金的能力 |
| 苏长河挡在山民前面出问题时担责 | 母基金帮助其投资者降低整体风险 |
| 拒绝大药商独家包销 | 母基金注重灵活性与多样性，不把资金吊在单一渠道 |
