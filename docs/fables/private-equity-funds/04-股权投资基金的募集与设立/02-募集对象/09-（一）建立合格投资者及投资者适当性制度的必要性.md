# （一）建立合格投资者及投资者适当性制度的必要性

## 📜 寓言故事 —— 《暗室点灯》

清河镇的老周商会，专门帮有钱人挑作坊投银子。商会的规矩是：不贴告示、不敲锣打鼓，只私下找熟人谈。老周觉得这样省事——人少嘴杂，不用天天向全镇汇报投了哪家、赚了亏了。

可麻烦来了。镇上有个张寡妇，丈夫死后留了一百两银子。她听人说老周商会"稳赚"，便托人找上门来，要把全部身家投进去。老周起初不想收：张寡妇不懂生意，连账簿都看不明白，万一赔了怎么办？可张寡妇软磨硬泡，老周心软，收了她的银子，签了契约。一年后，商会投的一家织坊亏了本，张寡妇的银子少了三成。她跑到商会哭闹，说老周骗了她。老周有苦难言：契约上明明写着"可能亏损"，可张寡妇不识字，听人说是"稳赚"就信了。

这件事惊动了县衙。师爷老李查办后，发现更大的问题：商会不公开招人，老周知道每家作坊的底细——哪家的织机新、哪家的掌柜靠谱、哪家的债多——可投银子的人一概不知。老周说投哪家就投哪家，赚了老周抽成，亏了老周照抽管理费。有个别掌柜甚至故意把银子投给自家亲戚的作坊，亏了也是别人的，自己照样拿工钱。这种"暗室操作"，投银子的人完全蒙在鼓里，老周想怎么干就怎么干。

县衙于是出了新规：第一，老周商会只能找"识货的人"投银子——这些人得有家底、懂风险、看得懂账，不能是连字都不识的寡妇。这叫"挑对人"。第二，即便找对了人，也得把人分成两类：一类是像钱庄赵掌柜那样，投过十家作坊、看过百本账的老手；另一类是像绸缎庄孙老板那样，有钱但不太懂行的。对老手，可以少解释，多给自由；对生手，得把风险一条条讲清楚，不能让他投超过自己承受能力的银子。这叫"分对人"。第三，老周每季度得向所有投银子的人汇报：投了哪家、赚了多少、亏了多少，不能藏着掖着。这叫"说清账"。

老周起初觉得麻烦，可实行一年后，他发现商会反而更顺了。识货的人投银子前会自己查作坊的底，有问题直接问老周，老周不敢糊弄。钱庄赵掌柜甚至帮商会挑出了一家藏债的织坊，避免了一笔大亏。孙老板虽然不太懂行，但老周把风险讲透后，他投得少、心态稳，从不闹事。县衙师爷老李在年审时说："暗室里点灯，不是为了照亮老周你一个人，是为了让所有人都能看见路。"

---

**📖 原文定义**

> 合格投资者制度的建立，源于股权投资基金的基本特征：基金投资者和基金管理人之间属于委托受托关系，基金管理人负责进行基金的投资决策，但投资资金主要由基金投资者提供。股权投资基金管理人和投资者之间存在较为严重的信息不对称：基金管理人居于信息优势地位，掌握着关于基金投资项目质量、投资风险、投资业绩等关系到投资者利益的信息；而基金投资者通常处于信息劣势地位。在存在信息不对称的情况下，基金管理人有可能会利用其信息优势从事一些对自己有利而对基金投资者不利的“败德”行为，从而损害基金投资者利益。特别是当股权投资基金以非公开方式募集时，无法通过公开披露信息机制约束基金管理人，“败德”行为可能发生得更多。因此，不少国家规定，股权投资基金只能向合格投资者募集。不同于一般的投资者，合格投资者往往资本更为雄厚、风险识别与承受能力更强、专业知识更为丰富，对基金管理人及其发起设立的基金具有较强的筛选、判断及监督能力。在此基础上，辅以基金信息披露制度，才能较好地保护基金投资者。
>
> 在我国，投资者适当性制度中存在普通投资者和专业投资者的划分。专业投资者除了满足普通投资者的条件外还需具备更丰富的投资经验和专业知识，对基金合规运行可以进一步提高监督要求。同时，投资者适当性制度相关要求有助于更好地平衡投资者对相关基金产品的风险预期。合格投资者制度与投资者适当性制度共同发挥作用，可以促进基金管理人和投资者之间建立更加健康和透明的合作关系，这对于促进股权投资基金行业稳定发展和保护投资者权益具有重要意义。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周商会私下招人、不公开汇报 | 非公开方式募集，无法通过公开披露信息机制约束 |
| 老周知道作坊底细，投银子的人一概不知 | 信息不对称：管理人处于信息优势，投资者处于信息劣势 |
| 个别掌柜投给自家亲戚作坊，亏了照拿工钱 | 管理人利用信息优势从事“败德”行为，损害投资者利益 |
| 张寡妇不识字、听不懂风险，全部身家投入 | 一般投资者缺乏风险识别能力和专业知识 |
| 只能找“识货的人”——有家底、懂风险、看得懂账 | 合格投资者制度：资本雄厚、风险识别与承受能力强、专业知识丰富 |
| 钱庄赵掌柜能自己查底、帮商会避坑 | 合格投资者具有较强的筛选、判断及监督能力 |
| 分老手（赵掌柜）和生手（孙老板），区别对待 | 投资者适当性制度：普通投资者与专业投资者的划分 |
| 对生手把风险讲透、不让投超过承受能力的银子 | 适当性匹配：平衡投资者对产品的风险预期 |
| 每季度汇报投哪家、赚多少、亏多少 | 辅以信息披露制度，保护基金投资者 |
| 暗室里点灯，让所有人看见路 | 合格投资者制度与适当性制度共同促进健康透明合作，保护投资者权益 |
