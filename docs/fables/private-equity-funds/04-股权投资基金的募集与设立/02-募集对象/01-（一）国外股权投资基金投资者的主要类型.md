# （一）国外股权投资基金投资者的主要类型

## 📜 寓言故事 —— 《六家入伙》

清河镇的老周想把酒坊扩成商会，需要大笔银子。他本打算只找三五老友凑钱，没想到消息传出去，来问的人五花八门，各有各的打算。

最先上门的是镇上的"善堂"管事刘老先生。善堂是镇上大户捐资办的，专管救济孤寡、资助学子，手里攒着一笔代代相传的积蓄。刘老先生说："这笔银子放着也是放着，不如投到你酒坊里，赚些利钱回来，善堂以后帮人就更有底气。"老周有些意外：善堂的钱不求快回本，图的是长远。刘老先生又补了一句："我们不急，三年五年都等得，但得签个契约，写明每年分多少、怎么管。"老周点头，这第一位合伙人，性子稳、眼光长。

紧接着，钱庄的赵掌柜来了。赵掌柜手里管着十几家铺子的闲钱，他说："我不直接投你酒坊，我投给专门挑酒坊的人。"原来赵掌柜认识一位"挑坊人"周师傅，周师傅常年走南闯北，专看哪家酒坊有潜力，看了十家投一家。赵掌柜把钱交给周师傅，周师傅再去挑酒坊。老周明白了：赵掌柜自己不辨酒曲，但他信得过挑坊人的眼光，这是"借眼识珠"。

第三位是省城书院的山长陈先生。书院也收了不少捐赠，山长说："书院的钱不急用，学生读书是百年大计。这银子投出去，十年八年回本无妨，只要稳当。"老周肃然起敬：书院的钱和银子不同，它背后是一代代读书人的信任。

第四位是绸缎庄的孙老板。孙老板财大气粗，开门见山："我投你酒坊，一是为赚利，二是为将来我的绸缎能进你酒坊的礼盒，绑着卖。"老周一听就懂：孙老板不只是出钱，还想着生意上的勾连。这种人投钱，眼里不只盯着分红。

第五位是钱庄隔壁的"保生堂"大掌柜。保生堂专做银钱保管和借贷，掌柜说："我们手里银子多，自己也想学着挑酒坊。不如这样，我既投你一笔，又派个学徒跟着你学怎么管酒坊，将来我们自己也能挑。"老周笑了：保生堂这是"边投边学"，两头不落空。

最后一位是个穿着讲究的中年人，自称姓高，说是替"高家"来的。高家祖上做官，如今家底殷实，不事生产，专靠各路利钱过日子。高先生说："我家老爷不懂酿酒，也不打算懂。他就想把手里的闲钱散到几处可靠的买卖里，酒坊是其中之一。您只管酿好酒，分利的时候别短了我们就行。"

老周把这六家的条件一一记下，发现件有趣的事：善堂图长远、书院求稳当、赵掌柜借眼识珠、孙老板要勾连、保生堂边投边学、高家只管分利。六家银子凑到一处，酒坊的根基反而比只找一家更稳。老周在契约上写道："各取所需，各尽其能，酒坊方能长久。"

**📖 原文定义**

> 国外股权投资基金的投资者群体呈现专业化与多样化特征。各类投资者基于资金规模风险偏好、投资目标等差异，形成了各具特色的参与模式。
>
> 1. 养老基金：养老基金，包括公司养老基金和公共养老基金，是资本市场中稳定的机构投资者。养老基金能够提供长期资金来源，并基于投资风险和期望收益的平衡，将一定比例的资金配置到股权投资基金中。在美国和欧洲市场，养老基金因其庞大的规模，成为股权投资基金的主要资本供给者。
>
> 2. 母基金：作为股权投资行业专业化细分的产物，母基金专注于投资于其他股权投资基金。母基金在投资标的（即股权投资基金）的筛选、风险管理和投后管理等方面展现出较高的专业化水平，是国外股权投资基金的重要机构投资者。
>
> 3. 大学基金会：大学基金会是国外股权投资基金稳定的机构投资者，其资金主要来源于社会捐赠。作为非营利基金会，大学基金会为了维持学校的运营而设立，流动性需求较低，适合进行长期投资。
>
> 4. 大型企业：资金实力雄厚的大型企业通常通过两种方式参与股权投资：一是以自有资金投资，设立投资业务子公司开展股权投资或并购活动，通常聚焦于本企业主营业务相关或产业链上下游的企业；二是作为财务投资者，投资股权投资基金，以实现资本增值。
>
> 5. 金融机构：在西方国家，参与股权投资基金的金融机构主要包括商业银行、投资银行和保险公司。金融机构参与股权投资基金主要有两种方式：一是作为财务投资者投资于股权投资基金；二是作为基金管理人发起设立股权投资基金。
>
> 6. 高净值个人或家族办公室：高净值个人或家族办公室在进行资产配置时，通常也会考虑将一定比例的可投资资金配置到股权投资基金这一资产类别当中。高净值个人或家族办公室一般作为基金投资者而不是基金管理人参与到股权投资基金中。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 善堂管事刘老先生，银子代代相传、不急回本 | 养老基金：长期资金来源，稳定机构投资者 |
| 钱庄赵掌柜投"挑坊人"而非直接投酒坊 | 母基金：专注投资于其他股权投资基金 |
| 书院山长陈先生，钱不急用、图百年大计 | 大学基金会：非营利，流动性需求低，适合长期投资 |
| 绸缎庄孙老板投钱还想绑着卖绸缎 | 大型企业：财务投资+战略协同，产业链整合 |
| 保生堂既投钱又派学徒学管酒坊 | 金融机构：作为财务投资者投资，或作为管理人发起设立 |
| 高家只管分利、不参与经营 | 高净值个人/家族办公室：作为投资者而非管理人参与 |
| 六家各取所需、银子凑一处更稳 | 投资者群体专业化与多样化，各具特色参与模式 |
