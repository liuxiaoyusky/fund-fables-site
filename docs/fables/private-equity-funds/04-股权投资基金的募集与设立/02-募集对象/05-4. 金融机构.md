# 4. 金融机构

## 📜 寓言故事 —— 《钱庄变阵》

清河镇的钱庄联盟，向来只做两件事：收百姓的存款，放贷给商人赚利差。可这几年，规矩变了。县衙连发几道告示：不许钱庄暗地里把百姓的银子转手倒腾；不许钱庄拍胸脯保证"稳赚不赔"；银子投出去，得按市价算，亏了就是亏了，不能糊弄。

钱庄大掌柜赵德全看着告示发愁。老办法行不通了——以前钱庄把百姓的银子打包，转给外地的商会去投作坊，自己中间抽一层，出了事也不担责。现在县衙说：不许这么干。银子是百姓的，钱庄得自己挑、自己管，亏了得认。赵德全召集各分号掌柜商议："咱们得换个活法。"

第一个变阵的是"保生堂"——专做银钱保管和借贷的铺子。保生堂大掌柜孙立人说："咱们手里管着不少大铺子的养老份子，这钱期限长、不能急取。以前都存着买地契，现在地契利薄。不如咱们自己挑几家可靠的商会，把银子直接投进去。咱们有人、有眼，自己去看作坊、谈契约，不用经别人的手。"保生堂率先拨出一笔银子，投给了老周商会，派了两个懂行的学徒跟着学怎么挑作坊。三年下来，利钱比买地契高了两成。其他保生堂纷纷效仿。

第二个变阵的是赵德全自己的钱庄。他想得更远："咱们不只是投银子，还可以自己立一个'挑坊会'，专门帮百姓挑作坊。"他招了十几个懂酿酒、懂织机、懂船运的老手，成立了一个新部门。百姓把银子交给钱庄，钱庄用这批人的眼和手去挑作坊，赚了按契约分，亏了也按契约担。钱庄不再做"转手买卖"，而是做"管家买卖"。

三年后，清河镇的钱庄联盟焕然一新。以前钱庄只会收和放，现在各分号都有了挑作坊的眼力。县衙的告示反而逼出了一群新行家。赵德全在年会上说："规矩变了，咱们不能老想着倒手抽成。银子得自己挑、自己管，这才是正经生意。"

**📖 原文定义**

> 在我国，保险公司、商业银行、金融资产投资公司、证券公司、基金子公司等金融机构是股权投资基金市场中的重要投资者。随着2018年4月《关于规范金融机构资产管理业务的指导意见》的正式发布，2018—2019年关于银行、证券公司、信托公司等金融机构的监管政策密集落地，银行理财子公司、券商资管、信托等各类资管机构的配套规定陆续推出，资管行业呈现出"去通道、去杠杆、打破刚性兑付、净值化"的发展特征。在此背景下，相较于其他资管产品，股权投资基金行业凭借其主动管理的特性，更好地适应了新的监管环境和市场需求。包括证券期货经营机构发行的资产管理计划、信托计划、保险资产管理计划等在内的各类资管计划，亦成为我国股权投资基金投资者结构中的重要组成部分。
>
> 保险资金具有较长的投资期限，天然适合于股权投资基金的投资。自2010年我国允许保险资金进入股权投资市场以来，从各保险公司的保险资金运用情况来看，保险资金仍以债券、银行存款和其他金融产品为主，配置权益资产（含股票与股权）的比例远低于政策规定的可投上限。近年来，国家金融监督管理总局（原中国银保监会）逐步放宽了保险资金投资于股权投资基金的政策限制，促进了保险资金通过投资股权投资基金支持实体经济，特别是中小企业和科技企业的股权融资。
>
> 近年来，国家积极稳妥支持商业银行下属金融资产投资公司（AIC）业务拓展，推动商业银行通过AIC渠道开展股权投资业务。2018年6月，原银保监会发布《金融资产投资公司管理办法（试行）》，允许中国工商银行、中国农业银行、中国银行、中国建设银行、交通银行下属金融资产投资公司作为商业银行债转股的平台，以债转股为目的开展企业股权投资。2020年2月至3月，原银保监会陆续批准5家金融资产管理公司通过附属机构在上海开展不以债转股为目的的股权投资业务。2024年9月，国家金融监管总局将金融资产管理公司股权投资试点扩大至北京等18个城市。2025年3月，国家金融监管总局进一步将金融资产投资公司股权投资范围扩大至试点城市所在省份，支持符合条件的其他商业银行发起设立金融资产投资公司，并支持保险资金参与金融资产投资公司股权投资试点。
>
> 金融机构在投资股权投资基金时，追求长期稳定的投资回报，并注重投资的安全性与收益性，偏好投资于具有明确退出机制和高流动性的项目。以保险资金为例，一方面，保险公司对外配置资产需考虑偿付能力充足率的监管要求；另一方面，保险资金重视投资的收益性和回报，特别是在投后管理阶段，更关注项目的退出进度和已分配倍数（DPI）。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 县衙告示：不许转手倒腾、不许保证稳赚、按市价算 | 去通道、去杠杆、打破刚性兑付、净值化 |
| 保生堂自己挑商会、派人学管作坊 | 保险资金直接投资于股权投资基金，主动管理 |
| 钱庄成立"挑坊会"自己挑作坊 | 商业银行设立金融资产投资公司（AIC）开展股权投资 |
| 钱庄从"转手买卖"变"管家买卖" | 股权投资基金凭借主动管理特性适应新监管环境 |
| 保生堂银子期限长、不能急取 | 保险资金投资期限长，天然适合股权投资 |
| 赚了按契约分、亏了也按契约担 | 净值化：按实际收益分配，打破刚性兑付 |
| 各分号都有了挑作坊的眼力 | 各类资管计划成为股权投资基金投资者结构重要组成部分 |
| 关注退出进度和已分配倍数 | 投后管理阶段关注项目退出进度和DPI |
