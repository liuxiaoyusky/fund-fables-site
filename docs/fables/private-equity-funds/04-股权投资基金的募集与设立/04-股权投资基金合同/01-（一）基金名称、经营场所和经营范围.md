# （一）基金名称、经营场所和经营范围

## 📜 寓言故事 —— 《三块招牌》

清末民初，江南有个叫钱有根的教书先生，攒下些银两，想和同窗孔书桥一道做点生意。两人都看好当地新办的纺织业，便凑了一笔本钱，立了字据，准备正式开张。可到了衙门登记那天，师爷却把他们的文书打回来了。

"你这纸上写的啥？'钱孔商号'？这叫什么名字？"师爷皱着眉敲着柜台。钱有根陪笑："我们俩一个姓钱一个姓孔，便叫这名，好记。"师爷摇头："你们做的是布匹绸缎买卖，登记的纸上得把'买卖'两个字写明白——写成'钱孔绸缎商号'，让人一瞧这招牌，就知道是干嘛的。"

钱有根正要点头，孔书桥却插了一句："既如此，不如叫'钱孔布庄'，听着更气派。"师爷一听，索性把笔一搁："布也好，绸缎也罢，只挑一样写清楚——你那铺子里到底主营什么？旁的不管，你来我这里报的这桩买卖，叫啥就写啥，写了就认。"两人对视一眼，最后老老实实定下"钱孔绸缎庄"四个字。

师爷把名字填进册子，又问："铺子开在哪儿？开多大？做什么活计？"孔书桥忙答："城东柳巷第三家，前面是门面，后面是库房。"钱有根又抢着补："我们还打算顺便卖点笔墨纸砚——"师爷抬手止住："你方才说主营绸缎，那就只写绸缎。旁的营生，三个月后另立一张文书再来添。写一行就守一行，不能把没影的事也一并报上去，那叫谎报。"

两人被说得面红耳赤，回去的路上闷声不语。走到巷口，孔书桥忽然顿住脚："你瞧——"只见邻街新开了一家"周记笔墨铺"，门口围着七八个伙计正搬箱。再往前是"吴氏布行"，柜上码着成匹的苏杭丝绸。

孔书桥压低声音："每家铺子的招牌上头，都把'笔墨''布行'写得明明白白——人家干嘛的，一眼就看见，从不写些含含糊糊的字眼。"钱有根恍然大悟："咱们差点就把'绸缎'两字省了，挂个'钱孔商号'的破牌子，旁人瞧见还当是卖杂货的呢。"

回到住处，钱有根又把师爷的话细细琢磨了一遍：名字，得让人一眼看出你是做哪一行的；地方，得有个实实在在的落脚处；营生，得一项一项写清楚，写了就得认。他拿起笔，把字据上的名目逐条改过——"钱孔绸缎庄"，经营场所"城东柳巷第三号门脸"，经营范围"绸缎、布匹之买卖"。

三个月后，铺子顺顺当当开了张。孔书桥站在门口抬头望着那块新漆的招牌，"钱孔绸缎庄"五个字在日头底下亮堂堂的。他转头对钱有根笑道："现在旁人走过，不用问就知道我们是卖什么的。"

钱有根点点头，心里却多记了一笔账：当初若不是被师爷打回来，挂个"钱孔商号"的牌子，怕是开门一个月都没人晓得这家店是卖什么的——名字里藏着门道，经营范围里藏着边界，挂出去的招牌和报上去的字据，得是同一桩买卖。

**📖 原文定义**

> （一）基金名称、经营场所和经营范围
>
> 在我国，除契约型股权投资基金外，基金名称、经营场所和经营范围是基金提交市场监督管理部门登记的必备信息。根据法律法规和自律规则的要求，股权投资基金的名称应当标明"股权基金""股权投资"等字样，股权投资基金组织形式为契约型的，名称应当标明"私募股权基金"字样。创业投资基金名称应当包含"创业投资基金"，或者在公司型、合伙型创业投资基金的经营范围中包含"从事创业投资活动"等以体现创业投资策略的字样。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 钱孔商号被师爷打回 | 基金名称须标明"股权基金""股权投资"等字样 |
| 改名为"钱孔绸缎庄" | 名称需能让人一眼识别其主要业务 |
| 师爷要求"做啥写啥" | 经营范围须如实、明确登记 |
| 经营场所"城东柳巷第三号门脸" | 经营场所是登记必备信息 |
| 钱有根想"顺便卖笔墨"被驳回 | 经营范围之外的事项不得随意添加 |
| 三个月后另立文书再添新业务 | 经营范围变更需按规定重新登记 |
| 街上"周记笔墨铺""吴氏布行"招牌分明 | 名称、经营范围须与实际业务对应 |
| 钱有根领悟"挂出去的招牌和报上去的字据须一致" | 名称、场所、经营范围三要素的统一性 |
