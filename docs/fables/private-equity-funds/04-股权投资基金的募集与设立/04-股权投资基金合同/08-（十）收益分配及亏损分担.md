# （十）收益分配及亏损分担

## 📜 寓言故事 —— 《分利宴》

清河镇老周商会的“江南织新”份子运转了五年，投的三家织坊中，两家赚了，一家亏了。第五年秋天，老周把赵掌柜、李东家、吴老爷、小吴请到商会，摆了一桌“分利宴”。

老周先算了一笔总账：三家织坊，总共收回银子一万两千两。当初投出去的本金是一万两。多出来的两千两，是“可分的利”。可怎么分？老周说：“契约里写得清楚，分利有规矩，不是谁嗓门大谁多分。”

**第一步：先还本**

老周说：“各位先拿回自己的本金。赵掌柜认缴五千两，先拿回五千；李东家认缴三千两，先拿回三千；吴老爷认缴两千两，先拿回两千；小吴认缴一千两，先拿回一千。本金各归各，谁也不欠谁。”

**第二步：算门槛利**

还完本，还剩两千两。老周说：“契约里约定，每年的‘门槛利’是百分之八。也就是说，各位的银子放在商会五年，按每年百分之八算，这是各位该得的‘保底利’。赵掌柜五千两，五年门槛利约两千两；李东家三千两，五年门槛利约一千二百两；吴老爷两千两，五年门槛利约八百两；小吴一千两，五年门槛利约四百两。加起来，门槛利总共四千四百两。可咱们实际只多出来两千两，不够分门槛利。怎么办？”

赵掌柜说：“不够分，就说明实际收益没达到门槛。那这两千两，按各自认缴比例分——赵掌柜分一半，一千两；李东家分三成，六百两；吴老爷分二成，四百两；小吴分一成，二百两。老周，你作为管事的，这五年辛苦了，可收益没超门槛，你的‘业绩分成’一分没有。”

老周点头：“正该如此。业绩分成是‘超了门槛才有的赏’，没超门槛，我只拿固定的管理费，不多拿一分。”

**第三步：假设超了门槛**

老周又说：“可咱们假设一下——若第五年收回的不是一万两千两，而是一万八千两。那多出来的八千两，先分门槛利四千四百两，还剩三千六百两。这三千六百两，才叫‘超额利’。超额利怎么分？契约里写：老周作为管事的，分超额利的百分之二十，即七百二十两；剩下的百分之八十，按各位认缴比例分——赵掌柜分一千四百四十两，李东家分八百六十四两，吴老爷分五百七十六两，小吴分二百八十八两。”

李东家问：“若老周分了七百二十两业绩分成，可后来第六年发现，那家亏的织坊其实亏得更多，总收益算下来，实际没超门槛。那老周分走的七百二十两怎么办？”

老周说：“契约里写了‘回拨’条款——若清算时发现，各位实际拿到的收益低于门槛利，或者分利比例不符合契约约定，我已经分走的业绩分成，得按规矩退回来，重新分给各位。这叫‘多退少补’，不是一锤子买卖。”

**第四步：非银子的分法**

吴老爷问：“若织坊赚了，可织坊掌柜不肯买回去，银子变不成现钱。咱们能把织坊的份子直接分给各人吗？”

老周摇头：“契约里写了——非银子分配，必须全体合伙人一致同意。而且得说清楚：这织坊的份子值多少银子？怎么定价？各位愿不愿意接？通常大家都不愿意接——织坊的份子不是银子，接了还得自己管，麻烦。所以非银子分配是最后手段，能变现尽量变现。”

**第五步：亏了怎么办**

小吴问：“若总收益不到一万两，亏了怎么办？”

老周说：“亏了，按各位认缴比例分担。赵掌柜担五成，李东家担三成，吴老爷担二成，小吴担一成。契约里写得清楚——利益共享，风险共担。赚了按比例分，亏了也按比例担。这叫‘分级’也得守规矩——若咱们这商会分了‘优先’和‘劣后’两等，优先的先分、劣后的后分，但优先和劣后的比例不能超过一比一，优先的至少担百分之三十的亏，劣后的最多担百分之七十。不能优先的只赚不亏，那不成。”

赵掌柜举杯：“老周，你这分利宴摆得好。本金先还、门槛再算、超额才分、多退少补、非银要全体同意、亏了按比例担——每一条都写在契约里，咱们心里踏实。”

老周笑：“分利不是分赃，是按规矩办事。规矩立在前头，宴才能吃得安稳。”

**📖 原文定义**

> 收益分配是基金合同中投资者最为关注的经济利益条款之一。该条款通常涵盖以下内容：可分配收入的定义、分配时间、分配顺序、门槛收益率、业绩报酬，以及非现金分配等。
>
> 为确保管理人的利益与基金利益相一致，并提供正向激励，根据股权投资行业惯例，基金合同通常约定，在投资者的收益超过门槛收益后，基金管理人可以从收益中提取特定比例作为业绩报酬。
>
> 为保护投资者利益，基金通常会设置回拨机制。回拨机制是指在基金清算或其他约定时点，对已经分配的收益进行重新计算，如果基金投资者实际获得的收益率低于门槛收益率，或者收益分配比例不符合基金合同约定的，基金管理人需要将已经分得的业绩报酬按照合同约定的规则返还至基金资产，并分配给基金投资者。
>
> 非现金分配是指基金在无法将项目及时变现时，将股权或股票直接分配给投资者的机制。尽管非现金分配是确保基金到期后能够终结运作的重要手段，但投资者通常对此持保留态度，因为投资者对这些非现金资产了解有限，且没有能力或不愿承接和管理该等非现金资产。非现金分配方案（包括非现金资产的定价）应经全体投资者一致同意或一致认可的决策机制决策通过。
>
> 基金产生的亏损通常由所有投资者按照基金合同约定的方式进行分担。
>
> 一些特殊策略基金（例如不动产基金、特殊机会基金），可能会采分级安排，即先向优先级投资者进行分配，再向劣后级投资者进行分配。根据自律规则，分级股权投资基金应当符合利益共享、风险共担、风险与收益相匹配原则，优先级与劣后级的比例不得超过1：1，优先级份额投资者获取收益或者承担亏损的比例不得低于30%，劣后级份额投资者获取收益或者承担亏损的比例不得高于70%。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 先还本金，再算门槛利，再分超额利 | 分配顺序：先返还本金，再分配门槛收益，再分配超额收益 |
| 每年百分之八的门槛利 | 门槛收益率 |
| 超门槛后老周分超额利的百分之二十 | 业绩报酬：投资者收益超过门槛收益后，管理人提取特定比例 |
| 实际收益低于门槛，老周已分的要退回来 | 回拨机制：清算时若实际收益率低于门槛收益率，管理人返还已分业绩报酬 |
| 织坊份子直接分给合伙人，需全体同意 | 非现金分配：应经全体投资者一致同意或一致认可的决策机制通过 |
| 亏了按认缴比例分担 | 亏损由所有投资者按合同约定方式分担 |
| 优先先分、劣后后分，但比例不超一比一 | 分级安排：优先级与劣后级比例不得超过1:1 |
| 优先至少担百分之三十亏，劣后最多担百分之七十 | 优先级份额投资者承担亏损比例不得低于30%，劣后级不得高于70% |
| 利益共享、风险共担、风险与收益相匹配 | 分级基金应符合利益共享、风险共担、风险与收益相匹配原则 |
