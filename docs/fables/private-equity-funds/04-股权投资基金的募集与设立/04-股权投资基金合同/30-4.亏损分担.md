# 4. 亏损分担

## ⚖️ 寓言故事 —— 《蚀本账》

永丰号大掌柜郑老大坐在账房后头的小院里，面前摆着今年的总账。十六铺桥头茶馆里的陈师傅端着茶盅过来串门，进门就瞧见他脸色不对。

"郑东家，这是怎么了？"

郑老大叹了口气，把账本往桌上一推："城南那块地的买卖，亏了。买的时候听人说要起市集，左等右等，市集没等来，倒是等来个破落户在那儿搭了棚子，行情一落千丈。我这心里不痛快，倒不是为自己，是替那些投了钱进来的东家们发愁。"

陈师傅探头瞧了一眼账本上的数字，咂了咂嘴："亏了多少？"

"三百两。"

陈师傅吹了吹茶沫："那可不少。这钱是您一人的，还是——"

"我郑老大做事从不独断。"郑老大摇摇头，"早年间我办这号的时候，本钱不够，就找了几位老友入股。林家出二百两，何老客出一百两，我自己出一百两——我既是掌柜，也是东家。算下来共四百两本金。眼下亏三百两，账面上还剩一百两。这亏空怎么摊，章程里没写明，我心里没底。"

正说着，刘账房从里头套间抱着一摞旧账簿出来，抬眼看了看两人："东家，账我刚才又过了一遍。"

郑老大招呼他坐下："刘账房，你说说，这亏空该怎么担？"

刘账房把账本摊开，手指头点在"股本"那一栏，平静地说："依我看，这笔账得分三段讲。"

他竖起一根指头："其一，您和林家、何老客，三人当初说好了按出资多少各占份额。本钱四百两，林家二百两，何老客一百两，您一百两。出力办事是您，跑腿担责也是您——这是您当掌柜的活，本分。但蚀本这件事，是大家伙的钱在一起做生意赔进去的，不能让您一个人兜着。"

刘账房竖起第二根指头："其二，章程上没写明怎么分摊，那就得回头找当初那份老契。契上白纸黑字怎么说的，咱们就怎么办。林家出二百两，占一半，亏三百两，他担一半就是一百五十两；何老客出一百两，占四分之一，担七十五两；您出一百两，占四分之一，担七十五两。"

陈师傅插嘴："郑东家不是掌柜吗？掌柜的不要多担一点？"

刘账房没接陈师傅的话，竖起第三根指头："其三，掌柜担的是'办事'的责任，不是'赔钱'的责任。您管事管得好不好，赚多赚少，那是本事；可生意场上有赚有赔，赔的部分得按本钱的份子来摊。赚了按份子分，赔了按份子担，这才叫公道。"

郑老大听完，默了一阵。院子里头麻雀扑棱棱飞过墙头，街面上传来卖豆腐的吆喝声。

"我原想着，"他慢慢说，"这回是我拿主意买的地，没听大家的劝告，眼光也差了些，心里头过意不去，想自己多担些。"

刘账房摇头："东家，您要真把三百两都揽在自己名下，那才是乱了章法。往后有人会说——永丰号的规矩是赚了一起分、赔了一人担。那还叫合伙？那叫替人扛活。林家何老客往后出了钱，都得掂量掂量：是不是赚了归我、赔了归掌柜？这笔糊涂账，谁还敢跟您搭伙？"

郑老大眉头松了松："依你说，该怎么开口？"

刘账房把账本合上："把当初的老契拿出来，把这三段道理一五一十说清楚。分摊的数字当场算明白，林家一百五十两，何老客七十五两，您七十五两。白纸黑字落下去。亏是亏了，可规矩还在，下回再起事，大家心里都有数。"

陈师傅在一旁拍了下大腿："得嘞，这才像话。我去把林家老爷和何老客请来，咱们当面锣对面鼓，把这事说开。"

郑老大站起身，朝刘账房点了点头："账上说话。"

刘账房回了一句："账上可没小事。"

---

**📖 原文定义**

> 基金产生的亏损通常由所有投资者按照基金合同约定的方式进行分担。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 林家、何老客、郑老大三人共出四百两本金 | 基金由所有投资者共同出资的合伙结构 |
| 蚀本三百两无法由郑老大一人独担 | 亏损不能只由管理人一方承担 |
| 刘账房按"出资额多少"算各人份额 | 按基金合同约定的方式进行分担（以出资比例为依据） |
| 林家担一百五十两、何老客担七十五两、郑老大担七十五两 | 所有投资者按其出资比例共同分担亏损 |
| 郑老大身为掌柜只担七十五两而非更多 | 管理人作为出资人之一，按出资比例承担，不因其管理身份而额外多担 |
| 章程未明时回头查老契、按契办事 | 亏损分担须以基金合同约定为依据 |
| 郑老大原本想独揽亏空、刘账房劝阻 | 避免"赚了一起分、赔了一人担"破坏合伙根基，规则统一才有公信力 |
