# （四）基金费用

## 📜 寓言故事 —— 《算账》

清河镇老周商会的"江南织新"份子成立了，银子到位，织坊也开始挑了。可老周发现，银子还没投出去，账上先少了一截。他坐在账房里，把伙计阿福叫来："阿福，咱们得算一笔账——这商会的银子，哪些是该花的'公账'，哪些是我老周自己该担的。"

阿福摊开账本，一条条念。第一项："商会成立前，您跑衙门、请先生写章程、印文书、差人送帖子，前前后后花了三百两。"老周说："这是'开办费'——商会还没开张，先垫出去的钱。但得设个上限，比如五百两。超了，我自己掏腰包。"他提笔在契约里添了一句："开办费上限五百两，超出部分由老周自担。"

第二项："钱庄那边，咱们银子存在专用柜里，钱庄每月收五两保管费。"老周点头："这是'看管费'，公账出。"第三项："县衙每年年检、换文书、交税费，约二十两。"老周说："这也是公账。"第四项："商会每季度开一次合伙人大会，茶点、场地、文书，每次约十两。"老周记上："会议费，公账。"第五项："账房先生每月工钱、笔墨纸砚、租铺面的钱，约三十两。"老周说："日常运营费，公账。"第六项："挑织坊时，请行家评估织机值多少钱、织工手艺如何，每次评估费约五十两。"老周说："这是为了挑织坊花的，公账。"

阿福继续念："第七项——老周您自己的工钱。"老周放下笔，沉吟了。这是最关键的一条。他想了想，说："我的工钱，不能从公账里随便拿。得有个规矩。"

老周把赵掌柜、李东家、吴老爷请来，摊开账本说："各位，商会运转，有些费用是公账该出的——开办、看管、税费、开会、评估、日常运营。这些我都列出来了，大家看看有没有漏的、有没有不该出的。"赵掌柜看了说："老周，你列得清楚，我没意见。但你的工钱怎么算？"

老周说："我的工钱，叫'管理费'。不是从商会总银子里抽，而是按各位'认缴'的份额来算。比如赵掌柜认缴五千两，每年按五千两的百分之二算，就是一百两；李东家认缴三千两，每年六十两；吴老爷认缴两千两，每年四十两。合计每年二百两，作为我的固定工钱。"

李东家问："这百分之二怎么定的？"老周说："这是行规。我管商会，要养账房、跑衙门、看织坊、盯银子，这些成本我得覆盖。但我也不白拿——这二百两是固定的，不管商会赚不赚，我都拿这么多。赚了，我还有'分利'；亏了，这二百两照付，但我没别的额外好处。"

吴老爷说："可你挑织坊，评估费、差旅费，不是已经算在公账里了吗？"老周答："对。公账里的评估费、差旅费，是请外人、跑外地的直接花费。我的二百两管理费，是覆盖我自己的薪酬、养团队、租铺面这些日常开支。两者不重复。"

赵掌柜想了想，说："老周，这管理费按'认缴'算，可有些银子还没缴到位。比如我的五千两，目前只实缴了两千两。按五千两算管理费，是不是多了？"老周点头："赵掌柜说得对。咱们改一下——管理费按'认缴额'算，但每投出去一家织坊、退出一家织坊，重新核算一次。比如今年投了三家织坊，花了三千两，那管理费按认缴额算；明年退出了一家，收回了一千两，那尚未退出的投资成本变了，管理费重新算。每年年初，按截至上一年底还没退出的投资成本，重新核定。"

众人点头。老周在契约里写下："年度管理费按认缴出资额及尚未退出的投资项目的投资成本核算，每年首个工作日重新核定。"

可吴老爷又提了一个问题："老周，若明年商会投了第四家织坊，可这家织坊是咱们通过另一家'子商会'投的——子商会专门管这家织坊。那子商会的费用怎么算？"老周说："子商会的费用，也是公账出。但有一条——子商会的账，必须和主商会的账一起审，一起向各位汇报。不能子商会花了多少，各位都不知道。"

赵掌柜最后问："老周，若将来商会和别家起了纠纷，打官司的钱谁出？"老周说："为了商会利益打的官司，公账出。但若是有人告我老周个人——比如说我私吞银子——那我自己请律师、自己出钱。公账只出'为了商会'的官司费。"

契约定完，老周把费用清单贴在商会墙上。他常对新来的伙计说："算账不是小气，是让所有人知道——公账的每一文，花在了哪儿；我老周拿了多少，为什么拿。账算清了，人心才齐。"

---

**📖 原文定义**

> 基金合同通常会详细列举应由基金承担的费用，以区分于基金管理人以其收取的管理费自行承担的费用。
>
> 除支付给基金管理人的管理费以外，基金费用一般包含筹建费、募集专用账户募集监督费用、托管费、审计费、因项目投资及退出而发生的费用、会议费用、受偿人士费用（与免责保证条款相关）、解散清算相关费用、为基金利益提起的诉讼仲裁费用等。基金合同通常会对基金费用的范围进行明确约定。实务中，一些投资者可能会要求在基金合同或其他相关法律文件中设定基金费用总额或特定费用项目的最高金额或比例限制。
>
> （合伙型基金）合伙企业应负担所有与合伙企业之设立、运营、终止、解散、清算等相关的费用（"合伙费用"），包括：（1）开办费，包括为合伙企业的筹建设立而产生的前期代垫费用以及设立过程中的法律、财务、会计、税务、差旅、通讯、打印、备案登记和其他政府规费。开办费上限为 $\times$ 万元，超出部分由普通合伙人承担；（2）合伙企业发生的与投资交易或现金管理相关的所有费用和支出，包括但不限于聘请的法律、审计、评估、中介及其他第三方费用；（3）为投资而组建投资工具所发生的费用（如有）；（4）管理费；（5）合伙企业日常运营和管理的法律、会计和审计、税务、评估及其他第三方顾问费用；（6）合伙企业之财务报表及报告费用，包括制作、印刷和发送成本；（7）合伙人会议、投资决策委员会、顾问委员会会议费用以及与合伙企业运营有关的其他会议费用；（8）政府部门对合伙企业、合伙企业的收益或资产、合伙企业交易和运作所收取的税费及其他费用；（9）合伙企业为维持合法存续而发生的登记、备案、年检等工商、审计、税务相关的费用，注册/经营场所租赁和维护费等成本；（10）向募集结算资金专用账户监督机构、托管机构支付的费用；（11）由合伙企业发起或针对合伙企业的诉讼、仲裁或其他司法或行政程序的费用，与该等诉讼、仲裁或程序相关的律师费，以及行使由此产生的任何支出；（12）合伙企业向受补偿方直接或间接支付的补偿及其他相关的税费、政府收费、罚金、赔偿金或和解金，或合伙企业应付的保险费；（13）合伙企业清算、解散、注销相关的费用；（14）其他未明确列出，但合伙企业发生的与合伙企业的业务和运营有关的费用。
>
> 管理费是基金管理人收取的固定报酬，是基金费用的主要部分，主要用于覆盖其管理和运营基金的成本和开支，例如员工薪酬、差旅、办公场所等。基金合同通常会明确约定管理费的计算基数、费率水平以及支付周期等。作为基金合同的关键商业条款之一，管理费的设定通常体现了基金管理人与基金投资者博弈的结果。
>
> （合伙型基金）作为向合伙企业提供的投资咨询、运作、运营管理等服务的对价，合伙企业应向基金管理人支付管理费。合伙企业向基金管理人支付的管理费总额为全体有限合伙人按照本条应分摊的管理费的总额。自首次交割日起至合伙企业解散日，就每一有限合伙人，其应分摊的年度管理费按照如下方式计算：上述年度管理费适用的认缴出资额以及尚未退出的投资项目的投资成本每年度进行核算，以截至相应年度的首个工作日核算的金额为准。
>
> 关于平行基金、联接基金及特殊目的载体的条款也是投资事项中的重要组成部分。这些投资实体的设立通常是为了满足募资或投资端基于商业或法律原因的需求，其设立具有其合理性。然而，其结构的复杂性增加了对底层资产和相关投资实体权益进行识别的难度。因此，对基金管理人在信息披露、审计等方面的要求更为严格。为了遵循信息透明原则和公平对待投资者原则，投资者与基金管理人应就这些条款进行深入商讨，并达成明确的约定。
>
> 平行基金与主基金由同一基金管理人管理，二者具有相同的基金期限、投资领域与范围、投资决策程序，二者通常按固定比例同时投向相同的标的或从投资标的退出，即平行基金与主基金"同进同退"。联接基金与主基金由同一基金管理人管理，通常情况下，管理人不在联接基金层面收取管理费和业绩报酬，联接基金一般作为主基金的上层基金，其全部资金均投入主基金。特殊目的载体通常作为投资工具使用，即在特定情况下，股权投资基金先出资至特殊目的载体，再由特殊目的载体出资至实际的投资标的。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周区分公账该出的和自己该担的 | 基金合同详细列举应由基金承担的费用，区分于管理人自行承担的费用 |
| 开办费三百两，设上限五百两，超出老周自担 | 开办费（筹建费）上限，超出部分由普通合伙人承担 |
| 钱庄保管费、县衙税费、会议费、日常运营费、评估费 | 托管费、税费、会议费用、日常运营费用、第三方费用等 |
| 管理费按认缴份额的百分之二算 | 管理费计算基数为认缴出资额，约定费率水平 |
| 管理费覆盖老周薪酬、养团队、租铺面 | 管理费覆盖管理和运营基金的成本和开支：员工薪酬、差旅、办公场所等 |
| 按认缴额及尚未退出的投资成本每年重新核算 | 年度管理费适用的认缴出资额及尚未退出的投资成本每年度核算 |
| 子商会费用也是公账出，但账要和主商会一起审 | 特殊目的载体（投资工具）的费用，信息披露和审计要求更严格 |
| 为了商会利益打的官司公账出，告老周个人的自己出 | 为基金利益提起的诉讼仲裁费用由基金承担；管理人个人责任自行承担 |
| 契约里写明管理费计算方式，众人博弈后达成一致 | 管理费设定体现基金管理人与基金投资者博弈的结果 |
| 平行商会和主商会同进同退 | 平行基金与主基金同进同退 |
| 联接商会不另收管理费，全部投入主商会 | 联接基金不在联接基金层面收取管理费，全部资金均投入主基金 |
