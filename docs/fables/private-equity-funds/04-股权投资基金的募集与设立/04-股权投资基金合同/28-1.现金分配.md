# 1. 现金分配

## 🏺 寓言故事 —— 《十六铺分银》

永丰号这一回的买卖，赚到了一笔不小的银子。

是郑老大领着东家们一起投的绸缎庄——本来是外埠一家快揭不开锅的染坊，郑老大带着小满跑了三趟，看准了那几口老灶、那片现成的水源，押下重注。不到两年，外埠的行情翻了个身，染坊被对街的布行连铺面带存货一并吃下，银子哗哗地流回了永丰号。

银车进了镇，停在大院里。麻袋一解开，白花花的银锭堆了小半间屋子。东家们闻讯挤了进来，个个伸着脖子，有人已经在心里盘算自家能分几锭、拿回去置几亩地。

郑老大却没急着发钱。他把人拦在堂前，只说了一句："叫刘账房来。"

刘账房捧着一本靛蓝封皮的账册，踩着稳稳的步子走进来。他先不开口，把账册往桌上一摊，翻到最后一页，提起毛笔，在砚台上舔了舔，朝郑老大点了点头。

"东家们先坐。"刘账房开口了，声音不大，堂前却一下静下来，"这一笔银，按规矩分，今儿当着各位的面，一锭一锭地过。"

头一桩，是把那本钱还回去。东家们当时凑的本——张东家三百两，李东家两百两，何东家一百两，刘账房一笔一笔地念出来，念一个，便把对应份数的银锭从麻袋里挪到东家名字底下。这叫"先还本"。东家们本就是做小买卖的出身，本金回来，心里先踏实了一半。

可银子还剩下一大堆。本钱加起来才六百两，银车里足足有一千二百两。多出来的六百两，是利润。

这时堂前就有人嘀咕了。"这六百两怎么分？"一个瘦高的东家先开了腔，"按人头发下去，不就完了？我出三百两，分三百两的利；何东家出得少，就少分。"

话音未落，刘账房抬眼看了他一眼，那一眼不轻不重，却让那东家把后半句咽了回去。

"按出资比例分，也是一种法子。"刘账房慢慢说，"但永丰号立的约，不是这么写的。"

他翻开账册最前面那几页——那是当初各家按手印、画押时一并签下的约书。刘账房指着一行字念出来："利润分派，先保各东家一年八厘之息；不足八厘者，郑老大不取分文；超过八厘之数，方可议及掌柜之酬。"

堂前一下安静得落针可闻。

"啥意思？"张东家有点急，"八厘是多少？"

"打个比方，"刘账房说，"各东家本金加在一起算作一千两。一年八厘，就是八十两。咱们这笔买卖前后约一年半，先把这八十两的利，按各家本钱的多少，分到各位名下。这一步，叫'先过门槛'。"

他一边说，一边从剩下那堆银锭里拨出一份，对着账册上各家的份额，分给了在场的东家。东家们点着手里的银子，喜上眉梢——这一份，是稳稳的、旱涝保收的"过门槛"的利。

"那剩下的呢？"何东家忙问。

刘账房抬头看了郑老大一眼。郑老大站在堂后，袖着手，不说话，只轻轻点了一下头。

"剩下的，"刘账房拿笔在账册上一划，"咱们这回买卖总共多出六百两利润，先分八十两给各位当过门槛的利；剩下五百二十两——这五百二十两里，郑老大提走两成，作为掌柜操持这笔买卖的酬劳；其余八成，再按各家本钱多少，分给各位。"

他说完，提笔在账册上落下了最后一笔。

堂前又是一阵安静。可这一回的安静，跟刚才不一样。刚才是不放心，这回是算过账之后，心里有了数。

张东家低头一拨算盘，自己念出声来："本金三百两还回来了；过门槛分到二十四两；剩的五百二十两里先扣一百零四两给郑掌柜，剩下的四百一十六两里头，我三百两的本钱，占三成……"

他念到这里，抬头看了一眼郑老大，又看了一眼刘账房，咧嘴笑了："郑老大，您这回出力可不少。"

郑老大这才从堂后走出来，笑着摇头："分内的事，账上写着的。"

刘账房合上账册，把银子一锭一锭地递到各家手里。走出永丰号大门的时候，巷子里正飘着淡淡的桂香。东家们各自揣着银子散去——有的去还亲戚的旧账，有的去给儿子置办聘礼，有的去桥头陈师傅的茶馆，要一壶热茶，慢慢数这一回到底赚了多少。

至于郑老大拿的那一份"掌柜的酬劳"，按账上的规矩，得是各东家都过了那道"八厘"的门槛之后，才能从余利里提走。这一条写进约书里，白纸黑字，改不得。

---

**📖 原文定义**

> 为确保管理人的利益与基金利益相一致，并提供正向激励，根据股权投资行业惯例，基金合同通常约定，在投资者的收益超过门槛收益后，基金管理人可以从收益中提取特定比例作为业绩报酬。
>
> 对于合伙企业取得的可分配收入，应按以下顺序在所有合伙人之间分配：（一）向各合伙人支付其截至分配之日的实缴出资额；（二）向各合伙人支付门槛收益；（三）向普通合伙人支付业绩报酬；（四）剩余部分按合伙协议约定的方式分配。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 银车进院，先把东家们当初出的本金一份份还回去 | 第一步：返还各合伙人的实缴出资额 |
| 一千两百两里先拨出八十两，按本钱多少分给各东家当"过门槛的利" | 第二步：向各合伙人支付门槛收益（八厘之息） |
| 剩下的五百二十两里，郑老大按两成提走作为"掌柜的酬劳" | 第三步：向普通合伙人/管理人支付业绩报酬 |
| 余下的四百一十六两再按各家本钱比例分 | 第四步：剩余部分按合伙协议约定的方式分配 |
| 张东家、李东家、何东家按手印画押时定下的那本约书 | 基金合同对分配顺序的事先约定 |
| 刘账房在账册上一笔一笔地过、一锭一锭地递 | 分配执行须按合同约定的顺序进行 |
| 郑老大不先开口、袖手站在堂后等刘账房念完 | 管理人业绩报酬依附于投资者门槛收益的实现 |
