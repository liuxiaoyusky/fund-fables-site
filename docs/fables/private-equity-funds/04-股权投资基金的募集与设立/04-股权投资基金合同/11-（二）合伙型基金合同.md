# （二）合伙型基金合同

## 📜 寓言故事 —— 《合伙约》

清河镇老周商会的“江南织新”份子，最初是赵掌柜、李东家、吴老爷、小吴凑的银子，老周管事。一开始大家凭口头约定，可两年后，赵掌柜和老周吵了一架——赵掌柜说老周挑织坊太保守，老周说赵掌柜不懂行就别指手画脚。吵完，赵掌柜想撤银子，老周不让，两人闹到县衙。

县衙师爷老李翻了他们的契约，摇头：“你们这叫什么契约？谁管事、谁出钱、赚了怎么分、亏了怎么担、有人想走怎么办——一样没写清。合伙做生意，不能靠嘴，得靠纸。”

老周受了教训，回去重拟了一份《合伙约》。他把所有合伙人叫来，一条条念：

**第一条：名和地。** 商会叫“江南织新合伙商号”，主要经营场所在清河镇东街三号。

**第二条：目的。** 合伙只为一件事——投江南新织坊，不碰别的买卖。

**第三条：人分两种。** 老周是“管事合伙人”——负责挑织坊、管银子、跑衙门；赵掌柜、李东家、吴老爷、小吴是“出钱合伙人”——只管出钱、分利、监督，不插手日常经营。管事合伙人的名字、住所得写清楚；出钱合伙人的名字、住所得写清楚。

**第四条：出多少、怎么缴。** 赵掌柜出五千两，李东家出三千两，吴老爷出两千两，小吴出一千两。签约时缴三成，一年后缴三成，两年后缴四成。

**第五条：赚了怎么分、亏了怎么担。** 先还本，再算门槛利，超额部分老周分二成，其余按出资比例分。亏了，也按出资比例担。

**第六条：管事合伙人怎么选、怎么换。** 老周是首任管事合伙人，任期五年。若老周失职——比如擅自把银子投给自家亲戚、或者连续两次挑的织坊都亏了——出钱合伙人可以投票罢免他，选新的管事合伙人。新管事合伙人得通过县衙的资格审查。

**第七条：出钱合伙人怎么进、怎么出。** 新人想入伙，得全体合伙人同意，缴溢价。老人想退伙，提前六个月通知，其他合伙人有优先受让权。若老人违约——比如该缴的银子不缴——管事合伙人可以把他除名，其份额转给别人。

**第八条：管事合伙人和出钱合伙人能不能互换。** 原则上不能。但若某个出钱合伙人后来学会了挑织坊、通过了审查，经全体同意，可以转成管事合伙人；反之，管事合伙人若不想再管事，也可以转成出钱合伙人，但得提前一年通知。

**第九条：有争议怎么办。** 先协商，协商不成，请县衙调解，调解不成，按契约里写的仲裁条款办。

**第十条：怎么散伙、怎么清算。** 合伙期限十年，到期清算。若中途全体合伙人同意提前散伙，或者管事合伙人被罢免后无人接任，也进入清算。清算人由全体合伙人推选，先还债，再按出资比例分剩余银子。

**第十一条：违约责任。** 谁违约，谁赔。该缴的银子不缴，按日罚息；擅自把商会银子挪作私用，双倍赔偿。

赵掌柜听完，第一个签字：“老周，这《合伙约》比咱们当初的口头约定硬多了。但硬有硬的好处——谁该干什么、谁不该干什么，纸上写得清清楚楚，以后吵也有个凭据。”

老周把《合伙约》送到县衙备案，又抄了一份送到商会总会。他对伙计说：“合伙不是讲义气，是讲规矩。规矩写在纸上，县衙认、总会认、合伙人认，这商号才能长久。”

---

**📖 原文定义**

> 在合伙型基金中，普通合伙人、有限合伙人及基金管理人通过合伙协议、委托管理协议等一系列协议，约定相关的权利和责任，同时也对基金运作的相关事宜进行事先规范。
>
> 合伙型基金的基金合同一般是合伙协议。合伙协议需根据《合伙企业法》的规定制定，应当载明：合伙企业的名称和主要经营场所的地点；合伙目的和合伙经营范围；普通合伙人和有限合伙人的姓名或者名称、住所；合伙人的出资方式、数额和缴付期限；利润分配、亏损分担方式；执行事务合伙人应具备的条件和选择程序；执行事务合伙人权限与违约处理办法；执行事务合伙人的除名条件和更换程序；有限合伙人入伙、退伙的条件、程序以及相关责任；有限合伙人和普通合伙人相互转变程序；争议解决办法；合伙企业的解散与清算；违约责任。此外，合伙型基金的合伙协议应包含中国证券投资基金业协会在基金合同指引中规定的所有必备条款。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周是“管事合伙人”，赵掌柜等是“出钱合伙人” | 普通合伙人与有限合伙人的区分 |
| 管事合伙人负责挑织坊、管银子，出钱合伙人只出钱分利监督 | 执行事务合伙人权限与有限合伙人不执行合伙事务 |
| 管事合伙人失职可被罢免，选新的需通过资格审查 | 执行事务合伙人的除名条件和更换程序 |
| 新人入伙需全体同意缴溢价，老人退伙提前六个月通知 | 有限合伙人入伙、退伙的条件、程序以及相关责任 |
| 出钱合伙人通过审查可转管事合伙人，反之亦可 | 有限合伙人和普通合伙人相互转变程序 |
| 先协商、再调解、再仲裁 | 争议解决办法 |
| 合伙期限十年，到期清算或全体同意提前散伙 | 合伙企业的解散与清算 |
| 该缴银子不缴按日罚息，挪用私用双倍赔偿 | 违约责任 |
| 《合伙约》送县衙备案、抄送商会总会 | 合伙协议应包含中国证券投资基金业协会在基金合同指引中规定的所有必备条款 |
