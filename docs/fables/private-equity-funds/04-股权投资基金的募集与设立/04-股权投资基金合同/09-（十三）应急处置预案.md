# （十三）应急处置预案

## 📜 寓言故事 —— 《备灾册》

清河镇老周商会的“江南织新”份子运转到第六年，一切顺遂。可老周心里总有一根弦绷着——万一自己出了事，商会怎么办？

那年冬天，老周染了风寒，卧床半月。商会的事务堆积如山：一家织坊的掌柜突然跑了，银子没着落；另一家织坊的织工罢工，货交不出来；钱庄那边催着对账，账房先生阿福一个人忙得团团转。赵掌柜、李东家、吴老爷、小吴急得团团转，可契约里没写——老周若病了、跑了、甚至不在了，谁来接管？

赵掌柜拍桌子：“老周是管事的，他不在，咱们就散了？投出去的银子怎么办？”李东家说：“契约里只写了赚了怎么分、亏了怎么担，可没写‘老周不在怎么办’。”众人面面相觑，商会一度陷入停摆。

老周病愈后，第一件事就是召集合伙人，补写一份“备灾册”。他说：“各位，这次是我病了，下次若是我跑了、死了、或者被县衙查了，商会不能散。咱们得在契约里写明——‘管事人失能’时怎么办。”

老周一条条写：

**第一条：谁来顶？**
“若我老周失联、被注销管事资格、或者商会出现重大风险导致无法正常退出，以下各方可以启动应急：我指定的副手（若有）、看管银子的钱庄、持有一定份额以上的合伙人，或者全体合伙人大会。他们可以成立一个‘应急小组’，或者委托外头的账房先生、讼师先生来帮忙处置商会的财产，保护各位的权益。”

**第二条：怎么换管事人？**
“若我老周确实无法履职，商会可以换管事人。怎么换？由应急小组召集，全体合伙人表决。表决方式：书面签字或按手印。表决程序：提前七天通知，说明缘由。表决比例：过半数合伙人同意即可通过。新管事人上任后，原管事人的权责全部移交。”

**第三条：怎么清算？**
“若商会实在运转不下去，需要清算。清算人由全体合伙人推选，或者委托外头的账房先生。清算程序：先盘点所有织坊的份子、收回所有外债、卖掉能卖的资产；再按契约约定的顺序分银子——先还本、再算门槛利、再分超额利；最后注销商会的登记。清算期间，所有合伙人随时可查账。”

赵掌柜问：“老周，你这‘备灾册’写得这么细，是不是咒自己？”老周笑：“不是咒自己，是给各位一个定心丸。契约里写明了‘晴天’怎么过，也得写明‘雨天’怎么办。晴天时这页纸压在箱底，雨天时它就是救命符。”

众人逐条看过，一一签字。老周把“备灾册”和主契约锁在同一个铁箱里，钥匙由赵掌柜和吴老爷各持一把——两人同时在，才能开箱。

三年后，邻镇一家商会因管事人突然卷款跑路，合伙人闹到县衙，银子追不回来，商会散了。老周听说后，对伙计说：“他们没有‘备灾册’。咱们有。这不是多余，是底线。”

---

**📖 原文定义**

> 根据自律规则的要求，基金管理人因失联、注销基金管理人登记或者出现重大风险等情形无法履行或者怠于履行职责导致股权投资基金无法正常退出的，基金管理人、基金托管人、基金份额持有人大会或者持有一定份额比例以上的投资者，可以按照基金合同约定成立专项机构或者委托会计师事务所、律师事务所等中介服务机构，妥善处置基金财产，保护投资者合法权益。基金合同应当就基金管理人无法履行或者怠于履行管理职责等情况时，股权投资基金变更管理人、清算等相关决策机制、召集主体、表决方式、表决程序、表决比例等相关事项进行约定。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周病了，商会事务无人接管，陷入停摆 | 基金管理人因失联、注销登记或重大风险无法履行职责导致基金无法正常退出 |
| 补写“备灾册”——管事人失能时怎么办 | 基金合同应就基金管理人无法履行或怠于履行管理职责等情况进行约定 |
| 副手、钱庄、持有一定份额以上的合伙人、全体合伙人大会可启动应急 | 基金管理人、基金托管人、基金份额持有人大会或持有一定份额比例以上的投资者可以成立专项机构 |
| 委托外头账房先生、讼师先生帮忙处置 | 委托会计师事务所、律师事务所等中介服务机构妥善处置基金财产 |
| 全体合伙人表决，过半数同意即可换管事人 | 变更管理人的决策机制、召集主体、表决方式、表决程序、表决比例 |
| 清算人由全体合伙人推选，按顺序分银子，注销登记 | 清算相关决策机制、程序、剩余财产分配 |
| 备灾册和主契约锁在同一个铁箱，两人同时在才能开箱 | 应急处置预案与基金合同同等重要，需确保可执行性 |
| 邻镇商会因无备灾册，管事人跑路后散了 | 应急处置预案是保护投资者合法权益的底线安排 |
