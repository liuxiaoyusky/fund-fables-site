# （三）契约型基金合同

## 📜 寓言故事 —— 《三把锁的匣子》

清末年间，江南水乡有个叫沈望归的绸缎商，攒下一笔银子，想找个稳妥的营生。他听闻邻县有个叫顾青山的"掌眼先生"，眼光极准，投什么赚什么，便动了心。一日，沈望归登门拜访，顾青山正和一位面容严肃的中年男子说话。男子腰间挂着一枚铜牌，上刻一个"恒"字。

沈望归开门见山："顾先生，我想把钱放您这儿生利，您看行不行？"顾青山笑而不答，反问："你打算怎么放？"沈望归心想：从前跟人合伙做买卖，也不过两方画押——我与对方各执一份，凭良心办事。

顾青山摇摇头，取出一份空白文契，又指了指那位"恒"字铜牌的男子："这位是钱庄的方掌柜，专管银箱钥匙和出入账。我们这一行，跟合伙做买卖不同——你的银子、我的人、方掌柜的账本，要一并写进同一张文契里，三方各执一份，三方各担一份责。"

沈望归犯了嘀咕："两方画押的规矩我懂，三方同立一契，从哪里下笔？"顾青山让伙计端来一只小木匣，匣子装着三把不同的锁。"你瞧——你若只跟我和方掌柜各签一份单契，钱交给谁、谁来点算、谁来看管，全凭我们两个说了算。你没亲眼盯着，难保不会有人动你银子的脑筋。"

顾青山继续说："我跟你之间，不是合伙的兄弟，是替你'看铺子'的掌柜——钱怎么用、用多少、赚亏由谁担，写在契里。方掌柜呢，他不替你花一文钱，也不替你赚一文钱，他只做一件事：守账本、看银箱、查每笔进出是不是照契上写的办。你、我、方掌柜，三方同时按手印。一张文契，三方并立，少了哪一方，这桩事都不算数。"

沈望归又问："若是您看走了眼，亏了呢？"顾青山坦然道："契上白纸黑字写明，赚了怎么分、亏了怎么摊、谁先垫、谁后补；方掌柜若数错了银子、漏了账本，也要担他那份责。这跟两家合伙最大的不同——不是把银子扔进一口锅里大家搅一气，而是把银子、眼光、账本这三样东西，分头交给三个信得过的人，一张契锁住三方的手脚。"

方掌柜这才开口："契里写明，我每三日须把账目交你过目；你若觉得有疑，可随时拿契来对账。匣子上三把锁，三把钥匙分三人手中，谁也打不开谁的那把。"

沈望归终于点了头。当日三人同坐一桌，把份额、分成、止损、查账、退出等一桩桩写清，各执一份，按下手印，各收一把钥匙。

一个月后，沈望归翻出契来对账，发现一笔进出与契上所写有出入。方掌柜翻开账本核对，果然是顾青山一时疏忽记错了一笔。依契中条款，错账当场更正，顾青山按契上罚则扣了自己那季的分成。

沈望归看着那张三方文契，心里踏实得很——自己出的银子有人管、有人看、有人查，三方各有各的责，谁也糊弄不了谁。

---

**📖 原文定义**

> （三）契约型基金合同
>
> 契约型基金合同的签署方一般为基金管理人、基金投资者、托管人，契约型基金的基金合同应包含中国证券投资基金业协会在基金合同指引中规定的所有必备条款。
>
> 与前两种类型基金合同相比，契约型基金合同存在如下特殊性：

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 沈望归（出资人）登门托付 | 基金投资者的角色 |
| 顾青山（掌眼先生）替他看铺子 | 基金管理人的受托管理职责 |
| 方掌柜（"恒"字铜牌）守账本、看银箱 | 托管人的独立保管与监督职责 |
| 三人同坐一桌、同按一契、三方各执一份 | 契约型基金"三方合同"的本质 |
| 匣子三把锁、钥匙分持三人 | 三方独立、相互制衡的关系 |
| 契中写明赚亏分担、止损、查账、退出 | 基金合同指引中规定的必备条款 |
| 沈、顾、方各担各责，错账按契更正 | 三方分别承担相应责任 |
| 与"两家合伙"对比 | 与公司型、合伙型基金合同的区别 |
