# （二）备案要求及常见问题

## 📜 寓言故事 —— 《吴主簿的清单》

那年深秋，永丰号大掌柜郑老大领着一份新字号"永丰同兴号"的开办章程，去镇公所找吴主簿登记备案。郑老大这回办的是个新名目——和镇上几家布庄、米行一道凑份子做买卖，大伙儿出银子、委托他的人去操盘,事成按份分利。他本以为不过又是递一份文书、画一个押，便能在镇公所登个名号，开门做生意。

吴主簿接过章程，没急着下笔。他抬眼看了郑老大一下，搁下笔说："郑掌柜，这桩买卖和寻常开铺子不一样。您是替别人管银子的，那备案的手续，可就不是一张纸能说清的了。"

郑老大眉头一皱："吴先生，咱们镇上人做买卖，讲的就是个爽快。一份章程写清楚谁管事、谁出钱、怎么分账，不就成了？"

吴主簿没接话，只从柜上取出一本厚厚的登记册，翻开其中一页，缓缓念来："既然是替人管银子、镇里又有明文，那章程之外，还要一并交上：银子的募集方式是怎么做的、字号内部的架构是怎生搭起的、每一位出资人是否符合准许入伙的条件、出资人的银钱是从哪里来的、字号管事一方的职责是如何落定的、字号本身的合同文本、对外招徕用的说明文字、投资的方向与边界、银钱存放的托付安排、首期实缴银子的凭据、字号存续的年期、与关联方如何往来。这些项，缺一项，便要您补齐了再来。"

郑老大听得心里咯噔一下。他原以为递一份章程便够，没想到镇公所要求的是一整套文书——从账怎么收、钱怎么放、事怎么管、人怎么挑，每一条都有人要细看。他低声问："吴先生，这些可都是要备齐的？"

吴主簿点头，从柜里再取出一本薄册，翻到里面夹着的一张黄纸，纸上写满了"不予备案"的旧例。他指着其中三桩旧事，一桩桩说给郑老大听。

"头一桩，是镇上某布庄的东家，把店里伙计凑份子做的'同人股'拿来报作字号。那是他自家员工的分润法子，哪里是替外人管银子？镇里当年就没给登记。"

"第二桩，更出奇。有一回某商号想寻个没在镇里挂过号的人替他做主、寻项目、收管理银子——自己只挂个名。这事被镇里察觉，那人既没在镇公所挂过号，便没有资格替人管账做主；更何况本该担责的东家把主事的权都让了出去，自己成了个空壳，这号便不允登记。"

"第三桩，最叫人哭笑不得。有一家商号来登记时，三位出资人个个都报了几万两的份额，可账上真金白银只各凑了一百两。问起来才知道，那几位出资人自己也还在外头张罗银子，打算等登记批下来再去凑钱。镇里一听就摇头——您自个儿的银钱还没凑齐，拿什么来作数？这号也打回去了。"

郑老大听到这里，半晌没作声。他原本觉得自己这回章程写得有板有眼，可对照吴主簿念的清单和那三桩旧例，才发觉自己漏的、不清楚的、有疑问的，一抓一大把：那几位出资人的银钱究竟是从田产来的、还是从铺子周转里挪来的？字号里那几位一同管事的人，权责到底怎么分？托付存银的票号，签约了没有？这一项项摆到镇公所案头，能不能过得了他吴主簿这一关？

吴主簿看出了他的心思，把那本厚册和那本薄册一合，递到郑老大手边："郑掌柜，您回去，先把这一套文书记齐了，再来不迟。章程只是骨架，这一摞才是血肉。缺哪一样，登记簿上便落不下您的名字。"

郑老大接过那两本册子，拱手道了谢，出了镇公所便直奔永丰号。进了门，他把小满和刘账房都叫到账房，把吴主簿念的那一串要求原原本本复述了一遍，又把三桩旧例讲给两人听。小满先开了口："大伯，这事儿急不得。咱们一项项对着理，谁管钱、谁管账、谁挑项目、钱存在哪、出资人能拿出多少、合同写的和章程一不一样，拢共要交的文书不下十来样，少一样都过不了。"

刘账房也点头："账上说话。先把咱们这号里头每一笔进出的来路、每位出资人首期实缴的凭据、托付票号那边的协议都理清楚，再去递不迟。"

郑老大坐在太师椅上，叹口气道："原以为递一份章程就成，没想到镇公所要看的是这么一整套。罢了，照着吴主簿那份清单，一项项备齐了再来。这事走不得半步捷径。"

窗外，十六铺桥头的河水在秋风里静静流着。郑老大摊开一张新纸，提笔写下："备案所需文书——逐项备齐。"
