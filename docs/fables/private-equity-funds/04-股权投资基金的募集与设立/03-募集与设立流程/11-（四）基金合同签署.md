# （四）基金合同签署

## 📜 寓言故事 —— 《永丰号画押》

秋分过后第三日，镇西永丰号的厅堂里擦得发亮。郑老大穿一身新做的灰布长衫，端坐在太师椅上，身侧立着刘账房。堂下八仙桌一字排开三张，每张桌上压着一摞白纸，纸边角齐齐整整，红泥印也磨好了。

今日，是永丰合伙字号正式立契的日子。

小满头天从上海跑回来，鞋上还沾着码头的黄土，他凑到郑老大耳边低声问："爹，咱这字号，章程上是不是还得再议一议？"郑老大没接话，只用下巴往东墙一抬。小满顺着看过去，墙上挂着一副对联，墨迹是孙先生亲笔的——"落笔前想三遍，立字后守十年"。上头还压着孙先生的手书状师招牌。

日头刚过辰时，孙先生提着青布书箱到了。他把书箱放在主桌上，慢慢打开，取出一卷厚厚的宣纸，又掏出一方随身的小铜印。刘账房递上茶，孙先生摆摆手，从袖里抽出半尺长的狼毫，蘸了蘸墨，先在第一页右上角写下"立契人"三个字。

厅里一下子静了。

郑老大开口道："孙先生，烦你把条款给大伙念一遍。"

孙先生点点头，把宣纸摊平，朗声念起来：合伙字号的名号、出资的份额、谁来管事、谁来管账、赚了怎么分、亏了怎么担、何时散伙、若有争讼该去哪里断。这五条他念得极慢，每念完一条，便抬头扫一眼堂下几位东家。

念到第三条时，坐在左侧的赵二叔忽然插嘴："管事这一条，我得再问一句——字号里的钱，是谁花？怎么花？谁说了算？"厅里一阵轻咳。

郑老大没动，刘账房在账本上记了个"问"字。

孙先生把狼毫搁下，抬眼看向赵二叔，温声道："依律而论，契上白纸黑字，钱由管事一支笔批，但每笔都得登账，每季给各位东家看一次流水。管事只管花，不管揣。揣银子的另有其人。"

赵二叔脸色稍缓，又问："若我看账看出毛病呢？"

孙先生答："契里另立一款——任一东家皆可发起查账，账上若查出违规，管事须自掏腰包补齐。"

堂下几位东家互相看了一眼，纷纷点头。

这一议就议到了午后。茶水换了三遍，墨也研干了两回。最后一条念完，孙先生抬头："若再无疑义，便请各位东家依次画押。"

赵二叔第一个站起来，走到主桌前，伸出手指在红泥印上重重一按，再稳稳落在自己名字下方。小满第二个上，他的手心全是汗，孙先生在旁轻声提醒："莫抖，落笔即定。"小满深吸一口气，按下了印。郑老大最后上前，他没看契纸，只看了孙先生一眼，点了点头，抬手落下。

厅里响起一片低低的笑声。

孙先生把契纸收好，又从书箱里取出一份薄薄的小册子，递给郑老大："此乃监督契。依镇公所规矩，字号立起后，须另请一位与各位东家无亲无故的第三方，在镇上票号里专开一个'立契银匣'。日后各位投入的银两，须先落此匣，再按契上约日，由匣入字号正账。匣与账两本流水，每月对一次。"

郑老大接过册子，翻了两页，问："这位第三方，由谁请？"

孙先生答："由契上议定。今日便定，定了便写进监督契里。"

刘账房此时上前一步，低声道："东家，依我看，请十六铺桥头那家'周记小票号'的周先生便好。他与咱永丰号素无往来，做事也稳当。"

郑老大想了想，点头："便依你。"

周先生被请来之后，也细细看了契纸，问了几处银两划转的细节，最后也画了押。

待到夕阳西斜，契纸、章程、监督契三份文书都收进了孙先生书箱。孙先生抬头看了看天色，温声道："依律，契上自今日画押起算，各位东家须等一日一夜，方可把银子从自家匣中搬入周记的立契银匣。"

赵二叔一愣："这又是为何？"

孙先生正色道："这一日一夜，名唤'冷心期'。各人回家，关上门，独自想清楚——这笔钱投出去，便不是自家银子了。若冷心期内有东家反悔，只需写一封退字书，银子原路退回，契上便抹去他名字，余契照立。"

厅里又静了一息。赵二叔率先点头："好。"

次日傍晚日头偏西时，吴主簿带着镇公所的差役登门。差役手里捧着一本厚册。吴主簿笑眯眯对郑老大说："孙先生一早把契纸正本送到镇公所了。我看过，三份齐备，画押无误。从今日起，永丰合伙字号，登录在册，正式开张。"

郑老大拱手："有劳吴先生。"

吴主簿摆摆手："分内事。依镇公所规矩，字号开张后七日内，须将字号名号、出资份额、管事之人、票号监督之人在镇口告示栏贴出三日，让镇上知情。"

刘账房当日便把告示写好，第二日便贴了出去。

小满那天夜里睡不着，跑去账房找刘账房，问："刘叔，咱这字号，从今日起便是真的了？"

刘账房拨了拨算盘，答他："契上画押那刻起，字号就有了骨架。银子落匣那刻起，便有了血肉。吴主簿登记在册那刻起，便有了名号。三样齐全，方才算立住了。"

小满点点头，似懂非懂。

窗外，月色照在永丰号新漆的门楣上。门楣正中，是郑老大亲笔写的四个大字——

"永丰合号"。

---

**📖 原文定义**

> 基金合同是股权投资基金的核心法律文件，通常由基金管理人和基金投资者等相关主体共同签署。在基金投资者与基金管理人通过沟通和谈判就基金设立达成一致意见后，即可准备签署一系列基金文件。基金文件签署完毕后，基金管理人将启动基金设立和备案的相关程序。基金合同生效后，基金投资者将依据合同约定履行出资义务，基金将按照基金合同的约定进入投资期并开始投资运作。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 孙先生主持立契、画押仪式 | 基金合同由基金管理人和投资者共同签署的"核心法律文件"性质 |
| 赵二叔当堂发问、孙先生逐条答辩 | 基金合同是各方谈判达成一致意见后才签署的结果 |
| 契纸写明出资份额、谁来管事、如何分利、争讼管辖 | 基金合同所约定的核心条款（出资、治理、收益分配、争议解决） |
| 另请周记票号开"立契银匣"、专管银两划转 | 募集资金监督——监督机构、专用账户、连带责任 |
| "冷心期一日一夜"的规矩与退字书 | 投资冷静期安排——不少于24小时，期间不得主动联系投资者 |
| 吴主簿登门登记、告示贴出三日 | 基金设立后的备案/登记与信息披露程序 |
| 画押、银子入匣、镇公所登记——三步齐备 | 基金合同签署→资金到账→基金正式成立（备案）的完整闭环 |
