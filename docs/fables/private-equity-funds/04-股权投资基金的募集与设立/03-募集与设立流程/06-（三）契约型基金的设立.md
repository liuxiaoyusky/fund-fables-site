# （三）契约型基金的设立

## 📜 寓言故事 —— 《一纸三印》

清末年间，江南有个织锦重镇叫盛泽镇。镇上开染坊的周鹤年攒下了一笔闲银，听闻城里几位行家要凑在一起做一笔生意，便托人引荐，想把银子搭进去生利。引荐人是位老行尊，姓吴，单名一个"慎"字。

约定的那天，地点定在镇外一座老茶楼的三楼雅间。周鹤年到得不早不晚，推门进去，只见屋里已坐着两人。一位是年约四十的程仲谋，穿一身藏青长衫，目光锐利；另一位是位寡言的老者，自称柳守真，腰间挂着一枚刻有"信"字的铜牌。茶博士端上茶盏退下，屋里便只剩这三人。

周鹤年开门见山："吴老哥说今日签一份文契，我银子带来即可？"程仲谋笑着摇头："周老板莫急。签契之前，我得把话讲在前头——您若要同我们一道做这门生意，有一件事您得先明白。"周鹤年竖起耳朵。

程仲谋从袖中取出一份装订整齐的文册："我从前与人合伙开染坊，是去衙门备个案、领一张'合伙文书'回来，那纸文书便是合伙的凭证。您今日若同我搭伙做事，按老规矩，得先去衙门走一趟，备上名册、章程、合伙人的身契，少说也得耗上十天半月。"周鹤年点了点头："这规矩我懂。"

程仲谋话锋一转："但今天这桩事，不必去衙门。"周鹤年一愣。程仲谋指着柳守真："柳老是专管账本和银箱的，我们三人各担各责——我替您拿主意、找门路、谈价钱；您出银子；柳老守账、查数、看银箱。这些事，不靠衙门那纸批文，全靠这一份文契讲清楚、讲明白。"他顿了顿："我们三人一桌同坐、一份文契同立、一同按下三枚手印。从按下手印那一刻起，这桩事就算正式开张了——不必去衙门备案，也不必等那张官府的批文下来。"

周鹤年迟疑道："不去衙门，万一官府不认呢？"柳守真这才开腔："周老板，您和我们签的这份文契，不是写给官府看的批文，是写给我们三人自己看的字据。官府不管这桩事——您出钱、我和程先生出眼光和账本，三方合意、按印为凭，自那一刻起，您同我们就是一条船上的人，权利、义务、规矩，全写在这张纸上。"

周鹤年又问："若我回去反悔呢？"程仲谋笑答："那就照契上写的办——何时能退、能退多少、写没写在契里。契一立、事即成；之后便按契行事，不靠衙门那张批文。"

周鹤年终于点了头。那日下午，三人围坐一桌，将出资、分成、用钱规矩、查账、退出等一桩桩逐条过目，落于纸上。读罢，笔墨齐下，三枚朱红指印并排落在契尾。三人各执一份，程仲谋又当场取出账簿，当着三人的面把第一笔银子入了账。

茶凉人散。周鹤年走出茶楼，手心里还攥着那份刚按完印的文契。他抬头望了望天——天色未晚，可那桩事，在他按下手印的那一刻，已经悄悄地开张了。

---

**📖 原文定义**

> 根据相关法律法规，契约型基金的设立无须在市场监督管理部门登记。基金的成立通过签订基金合同来实现，合同中将明确约定基金投资者、管理人和托管人的权利、义务和职责，确保委托财产的安全并保护各方的合法权益。通常情况下，基金自基金合同生效之日起或于合同约定的成立日期即被视为正式成立。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 程仲谋说明"不必去衙门" | 契约型基金无须在市场监督管理部门登记 |
| 三人围坐一桌立契 | 通过签订基金合同实现基金成立 |
| 旧时合伙开染坊需去衙门备案的对比 | 与公司型、合伙型基金设立的区别 |
| 契中写明"我替您拿主意、您出银子、柳老守账" | 合同明确约定投资者、管理人、托管人的权利义务与职责 |
| 三枚朱红指印并排落在契尾 | 三方共同签署合同的行为 |
| "从按下手印那一刻起，这桩事就算正式开张了" | 基金自基金合同生效之日起被视为正式成立 |
| 不依赖官府批文 | 契约型基金的成立不依赖行政登记机关的批准 |
