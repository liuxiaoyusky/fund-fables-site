# （二）合伙型基金的设立

## ⚖️ 寓言故事 —— 《三敲县衙》

民国二十三年春，绍兴茶商沈亦舟攒下八千银元，想找几个信得过的人凑成大本钱，去收春茶、囤丝帛。牵线人是商界老前辈范定。

登门那天，范定开门见山："搭伙做大生意，光凭嘴不成——得先立一份合伙文书，把各人出多少银、分多少利、谁出面拿主意、谁替大家守账，全写在里头。再凭这份文书，去县衙备个案、登个记。"沈亦舟一愣："还要去县衙？"范定点头："按本朝律例，合伙人凑齐、章程有了，还得让官府登一笔、入了档，才算真正立起来。"

沈亦舟心里打鼓，原以为合伙买春茶这般小事，私下一张纸写明白就行。范定从柜里取出旧文册："这是去年城西四家茶号备案的存根。出银的、出面办事的、守账的，全写在这份文书里，再由衙门盖一颗红印。少了这颗印，外头打官司官府都不认。"沈亦舟点点头。

回家路上，沈亦舟拐进同乡秦奉初家。秦奉初是城里小有名气的账房先生，做账二十年。三人凑一处：沈亦舟出大头银子，范定出面拿主意找门路，秦奉初管账守银箱。

茶楼碰头，范定道："合伙文书先坐下来敲定。出银、退伙、谁来办事、谁来守账，全写在一份纸上。"沈亦舟皱眉："听说城南那桩茶号合伙，几个东家写了张纸按了印就算完，并未去衙门备案。"范定摇头："那种私下立约的合伙，没经官府入档。咱们做大生意，不能那样办。光写明白不够，还得让衙门登一笔。"

接下来半月，三人关在秦奉初书房，把出资、分成、退伙、查账、议事、违约等逐字敲定，写成一本文书。又请城里师爷过目润色，删去含糊之辞，补上违约处罚明文。

范定收好文书："明日去县衙备案。"沈亦舟紧张："衙门要查什么？"范定答："查三件事。一、合伙人齐不齐、有没有不合规矩的；二、文书条款是否齐备——东家、出资、分工、退出、议事，全要写明；三、登堂东家身份是否真实。驳回就照衙门意思改，改到衙门点头为止。衙门一旦盖了红印，咱这桩合伙才算入档、正式立起来。"

第二天一早，三人换了长衫，揣上合伙文书、身份凭据、住所证明，同去县衙。师爷翻了文书，问了几个章程里的细节，又让三人各报姓名、按下手印。折腾半日，红印落下，存根留底。师爷递回文书："自今日起，合伙立档。"

沈亦舟走出县衙，手里攥着那份盖了红印的文书，回头望一眼"登记备案"的木牌。他忽然明白：合伙这桩事，光三人屋里写好章程还不够，必须让衙门过目、入档、盖印。少那一道手续，外头办事、对外签契、打起官司都立不稳。而这枚红印的背后，是一份事先敲定的合伙文书——出银子、出主意、守账目，全写在里头。衙门登的不是空名，是三人白纸黑字立下的规矩。

>
> 合伙型基金与公司型基金均由市场监督管理部门进行登记管理，因而设立步骤相同，合伙型基金应依据《合伙企业法》《市场主体登记管理条例》、各地市场监督管理部门的要求等申请登记。
>
> 对于公司型和合伙型基金，除了遵循常规的登记设立程序外，还需注意不同地区可能对股权投资基金设立设置了特定的要求。这些要求包括但不限于需要获得当地金融监管部门的审批和复审等流程。如果基金涉及外资参与，还可能需要满足特定的资质或审批条件。

| 范定开门见山"先去县衙备个案、登个记" | 合伙型基金需向市场监督管理部门申请登记 |

| 范定说"驳回就照衙门意思改" | 申请材料不齐或不符要求时需补正 |
