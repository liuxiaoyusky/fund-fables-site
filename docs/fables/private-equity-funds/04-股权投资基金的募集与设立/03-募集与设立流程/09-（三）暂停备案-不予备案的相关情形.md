# （三）暂停备案/不予备案的相关情形

## 📜 寓言故事 —— 《红牌与黑牌》

清河镇商会总会，每年都要审查各家商会的"报册子"。管事周师傅手里有两块牌子：一块红的，一块黑的。红牌叫"暂停"——不是不许你报，是先停下，等你改好了再来；黑牌叫"不予"——不是改不改的问题，是你这报册子从根本上就不该报。

老周商会的"江南织新有限公司"报册子批下来后，老周得意了一阵，第二年又报了一只新"份子"——"江南船运会"。可这次周师傅没批，挂了红牌。老周急了，去问缘由。周师傅说："老周，你上只'织新'的报册子里，有一处数字写错了——合伙人赵掌柜的出资额，你写的是三千两，实际到账只有两千两。上回我们查出来，让你改了。可这次'船运会'的报册子里，你又把李东家认缴的四千两写成了五千两。一回是疏忽，两回就是惯犯。你报册子老有'错载、漏载'，我们没法信你。红牌挂你半年，这半年里你报什么我们都不看。回去把规矩学透了，把账房先生换个人，半年后再来。"

老周红着脸回去了。半年后，他重新培训了账房先生，报册子一字一句核对三遍，再递上去。周师傅这次看了，点点头："改好了，可以批。"

可镇上另一家商会"快钱庄"的掌柜钱大，就没这么幸运了。钱大报了一只"赌坊份子"——说是投作坊，实际上报册子里写的投资范围全是赌场、青楼、地下钱庄。周师傅一看，直接挂了黑牌。钱大不服："我银子够、合伙人齐、章程也写了，凭什么黑我？"周师傅说："钱大，你报的是'商会'，可你投的是赌场。商会总会的规矩第一条：只批正经生意。你这不是报册子有问题，是你这'份子'本身就不该存在。黑牌不是罚你，是告诉你——这东西，总会从来不认。"

钱大还想争辩，周师傅又说："你收了合伙人的银子，现在册子批不下来。按规矩，你得立刻告诉所有合伙人：'报册子没批，银子原路退回'。募集期间花的银子——印文书、跑衙门、请账房——这些费用怎么担，你当初契约里写了没有？若写了按契约办；若没写，你自己认。合伙人的银子，到账期间的利钱，退不退也得说清楚。钱大，黑牌不是儿戏，你得妥善善后，否则总会还要追究。"

钱大垂头丧气地走了。老周站在总会门口，看着钱大的背影，对伙计说："红牌是教训，黑牌是底线。报册子可以改，但'投什么'这根筋歪了，神仙也救不了。"

>
> 对于中国证券投资基金业协会不予备案的股权投资基金，基金管理人应当及时告知投资者，妥善处置相关财产，保护投资者的合法权益。在实务中，各相关方可能会在基金合同或认购文件中约定基金不能备案情况下的费用承担（如基金募集期间的费用承担方式）、资金返还（如是否返还实缴出资资金到账期间的利息）等事项。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 红牌"暂停"——先停下，改好了再来 | 暂停备案：管理人存在特定情形，暂停办理其基金备案 |
| 老周报册子老有错载、漏载，挂红牌半年 | 管理人存在违规情形（如虚假记载、误导性陈述或重大遗漏） |
| 黑牌"不予"——这东西本身就不该存在 | 不予办理备案：基金本身存在不符合适用法律规定的情形 |
| 钱大投赌场、青楼、地下钱庄，直接黑牌 | 基金投资范围不符合规定，不予备案 |
| 钱大得立刻告诉合伙人银子原路退回 | 基金管理人应当及时告知投资者，妥善处置相关财产 |
| 募集期间费用怎么担、利钱退不退，契约里要写清 | 基金合同或认购文件中约定不能备案情况下的费用承担、资金返还等事项 |
| 红牌是教训，黑牌是底线 | 暂停备案针对管理人问题，不予备案针对基金本身问题 |
