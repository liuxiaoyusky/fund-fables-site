# （二）备案要求及常见问题

## 📜 寓言故事 —— 《报册子》

清河镇老周商会的"江南织新有限公司"，牌子挂上了、章程立好了、银子到位了。老周以为万事大吉，可以开张了。可商会总会的管事周师傅拦住他："老周，牌子是县衙发的，但商会总会还有一本册子要报。不报，你这公司就不算总会认的正经成员，以后出了事总会不管。"

老周忙问怎么报。周师傅递来一张单子，上面列了十几项要交的材料：公司的章程、合伙人的名单、每人出了多少银子、银子怎么管的、谁负责挑作坊、挑作坊的规矩是什么、赚了怎么分、亏了怎么担、合伙人的银子从哪儿来的、是不是借的、有没有关联的人互相串通、公司打算开多久……老周一条一条准备，花了整整半个月。他这才明白：报册子不是走个过场，是要让总会把公司的里里外外都看清楚。

材料递上去第三天，周师傅把老周叫去，脸色不好看："老周，你这份报册子，有几个问题。"

第一个问题出在"合伙人银子从哪儿来"这条上。老周报的是：三个合伙人，赵掌柜、李东家、吴老爷，各出三千两。周师傅问："这三千两是他们自己的，还是借来的？"老周说："都是他们自己的。"周师傅摇头："你自己的报册子上写着——赵掌柜的三千两里，有两千两是从钱庄借的，借据还在。这不行。合伙人的银子必须是自己的，借来的银子不能算。你得让赵掌柜把借的两千两还清，或者换成他自己实有的银子，才能报。"

第二个问题更麻烦。老周报册子时，为了显得公司规模大，把三个合伙人的"认缴"银子都写成了"各认缴一万两"，可实际到账的只有每人三千两。周师傅指着册子说："你写的是认缴一万，实缴三千。那剩下的七千呢？赵掌柜、李东家、吴老爷，他们拿得出七千吗？"老周一愣："我……我没细问。我想着先报上去，等册子批了，再慢慢募。"周师傅正色："这不行。报册子必须以'银子都到位了'为前提。你写认缴一万，就得证明这一万都能实缴。若写三千，就报三千。'先报册子再募银子'，这叫'先备后募'，总会查出来要退回去整改的。"

老周汗颜，回去把三个合伙人叫来，重新核对：赵掌柜实有三千两，认缴就写三千；李东家实有四千两，认缴写四千；吴老爷实有五千两，认缴写五千。三人签字画押，确认"认缴的就是实缴的，不多写一分"。

第三个问题出在"谁管挑作坊"这条上。老周报的是：他自己管挑作坊，但另有一个"顾问"钱掌柜——不是商会的人，没在任何册子上登过记——也参与挑作坊，还拿一部分管理费。周师傅说："老周，挑作坊是商会的核心职责，只能由登过记的正经管事来挑。你让一个没登记的人参与挑作坊、拿管理费，等于把商会的职责转手给别人，自己当'通道'。这叫'出让管事权'，总会不予备案。而且，管理费只能给正经管事，不能给外人。"

老周恍然大悟，回去把章程改了：挑作坊的事，只由他自己和商会里登过记的账房先生负责。钱掌柜可以当"顾问"提建议，但不能参与决策，也不拿管理费。改完后重新报册子。

半个月后，册子批下来了。周师傅对老周说："报册子这关，不是为难你，是帮你把规矩立在前头。认缴和实缴要对得上、银子来源要清白、管事的人要正经登记——这三条守住了，公司才能走得远。"

老周把批下来的册子收好，贴在商会墙上。他常对新来的伙计说："牌子是面子，册子是里子。面子县衙给，里子自己守。"

**📖 原文定义**

> 基金备案以基金募集完毕为前提。“募集完毕”是指基金的已认缴投资者已签署基金合同，且首期实缴募集资金已进入托管账户等基金财产账户。单个投资者首期实缴出资除另有规定外，不得低于合格投资者的最低出资要求。
>
> 相关法律法规和自律规则明确了股权投资基金的备案要求，这些要求涵盖资金募集方式、基金架构、合格投资者及适当性管理、投资者资金来源、管理人职责、基金合同、募集推介材料、基金投资范围、托管机制、实缴出资金额、存续期、关联交易等方面。股权投资基金备案时，基金管理人需按照相关规定的要求在资产管理业务综合报送平台上传募集说明书、基金合同、委托管理协议（如涉及）、基金销售协议（如涉及）、募集结算资金专用账户监督协议、托管协议（如有）或者保障基金财产安全的制度措施相关文件、投资者适当性管理的相关文件、产品结构图、实缴出资证明等备案申请材料。
>
> 实务中，基金产品备案过程中会遇到一些常见问题。中国证券投资基金业协会通过定期发布案例，对基金备案中的常见问题进行梳理，并将筛选出的典型案例进行脱敏处理后公示。中国证券投资基金业协会通过对具体案例的直观描述和针对性分析，阐释自律规则的制定背景、考量因素和适用情形。
>
> 员工持股计划拟备案为私募基金：基金管理人A提交上市公司员工持股计划B备案申请，B作为上市公司员工持股计划载体，拟以非交易过户方式受让该上市公司股票。案例分析：员工持股计划一般是指公司根据员工意愿，通过合法方式使员工获得本公司股份并长期持有，股份权益按约定分配给员工的制度安排，其目的主要在于实现员工激励，建立和完善劳动者与所有者的利益共享机制。而私募基金是指以非公开方式向合格投资者募集资金设立的投资基金，其设立目的为“受人之托，代人理财”……因此，两者存在明显差异，故员工持股计划不属于私募基金备案范围，中国证券投资基金业协会亦不予备案。
>
> 管理人出让投资决策权成为“通道”：基金管理人A提交合伙型股权投资基金B备案申请，该基金设置双执行事务合伙人结构，分别为基金管理人A以及未登记机构C。根据合伙协议职责划分，机构C负责委任投资决策委员会委员、制定投资决策委员会议事规则、筛选投资项目并进行投后管理。同时，机构C将收取部分基金管理费。案例分析：首先，未经登记任何单位不得进行私募基金业务活动；基金管理人应当按照诚实信用、勤勉尽责原则切实履行受托管理职责，不得将投资管理职责委托他人行使。上述案例中，机构C并未登记为基金管理人，不具备管理私募基金的资质，但根据基金合伙协议约定，机构C控制基金投资决策委员会，负责筛选投资项目、进行投后管理，实质上管理基金；原基金管理人A出让投资决策权，将应当履行的受托责任转让他人，成为非登记机构开展私募业务的“通道”。其次，基金管理费为基金管理人从基金中收取的固定费用，用于覆盖基金管理的日常开支和基金管理人的基础运营成本，为基金管理人项下的专属费用科目，其他执行事务合伙人不得以“基金管理费”名义收取相关费用。上述案例中，机构C作为非基金管理人收取基金管理费，违反基金管理费仅可由基金管理人收取的要求。综上，中国证券投资基金业协会认为上述私募基金不符备案要求，已对其不予备案。
>
> 投资者不具备实缴出资能力：股权投资基金管理人A提交股权投资基金B备案申请，其投资者为3只集合资金信托计划，均为认缴上亿元规模、实缴仅100万元情形。经核实，3只集合资金信托计划均于近期设立且未募集完毕，信托计划账户内仅实缴100万元资金，不具备后续实缴出资能力，拟待股权投资基金B备案通过后，继续对外分批募集。案例分析：基金投资者对基金的认缴金额应与其实际出资能力相匹配，投资者认缴金额与实缴金额差异较大的，关注其是否具备后续实缴出资能力。上述案例中……此类行为涉嫌“先备后募”，中国证券投资基金业协会已退回要求基金管理人进行整改，其投资者信托计划应募集完毕并具备实缴出资能力后再提交基金备案申请。

| 故事元素 | 概念对应 |
|---------|---------|
| 报册子要以银子到位为前提 | 基金备案以基金募集完毕为前提：已认缴投资者签署合同，首期实缴资金进入托管账户 |

| 赵掌柜三千两里有两千是借的，不行 | 投资者资金来源需合法，借来的资金不能算 |
| 认缴一万实缴三千，先报册子再募银子 | 涉嫌"先备后募"：认缴金额应与实际出资能力匹配，需募集完毕并具备实缴出资能力后再备案 |
| 钱掌柜没登记却参与挑作坊、拿管理费 | 管理人出让投资决策权成为"通道"：未经登记不得进行私募基金业务，不得将投资管理职责委托他人 |

| 认缴和实缴要对得上、银子来源要清白、管事的人要正经登记 | 备案常见问题：实缴出资能力、资金来源、管理人资质与职责 |
