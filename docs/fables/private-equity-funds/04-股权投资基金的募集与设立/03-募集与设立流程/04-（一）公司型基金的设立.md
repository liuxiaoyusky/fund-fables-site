# （一）公司型基金的设立

## 📜 寓言故事 —— 《挂牌子》

清河镇老周商会的"新织坊份子"，字据签完、银子到位，老周却还不肯开张。他对赵掌柜说："银子是咱们的，但'织坊'这个名头，得让衙门认。不认，咱们就是私下凑钱，出了事没人管。"

老周带着契约和银子，去了县衙。第一步是"定名"——他给织坊起了个名字："江南织新有限公司"。县衙的文书查了册子，说："这名字没人用过，准了。"第二步是"立规矩"——老周把合伙契约改写成"公司章程"：谁出多少银子、占多少份子、谁管事、谁查账、赚了怎么分、亏了怎么担、有人想退出怎么办。章程写完后，所有合伙人签字画押，递上县衙备案。第三步是"安家"——老周在镇上租了一间铺面，作为织坊的"住所"，县衙派人来查验，确认确有其地。

县衙核准后，发了一块牌子："江南织新有限公司，依法设立，准予经营。"老周把牌子挂在铺面门口，对赵掌柜说："从现在起，咱们不是私下合伙了，是衙门认的正经公司。各位合伙人，按章程里写的份额认缴银子，认缴多少，将来亏了就担多少，赚了也分多少。这叫'有限责任'——不是无限兜底。"

牌子挂上那天，老周又做了一件事：他把公司的章程、合伙人的名单、出资的明细、管事的安排，全部抄了一份，送到商会总会备案。总会管事的说："备案之后，你们就是总会名册上的正经成员了。以后每年得报一次账，赚了亏了都要说清。"老周点头："正该如此。"

赵掌柜看着门口的牌子，对老周说："以前咱们合伙做生意，靠的是人情和口头。现在有了牌子、章程、备案，人情还在，但规矩硬了。这牌子一挂，咱们睡得着，外人也信得过。"

---

**📖 原文定义**

> 股权投资基金的设立流程主要包括基金合同的签署、基金在市场监督管理部门的注册（如涉及）等程序，并须按法律法规及自律规则要求在中国证券投资基金业协会资产管理业务综合报送平台完成基金产品备案。
>
> 根据《公司法》相关规定，设立有限责任公司，应当具备下列主要条件：股东共同制定公司章程；有公司名称（名称应当符合法律法规及自律规则的有关要求），设立符合有限责任公司要求的组织机构；有公司住所。
>
> 设立股份有限公司，应当具备下列主要条件：股份发行、筹办事项符合法律规定；由发起人共同制定公司章程；有公司名称（名称应当符合法律法规及自律规则的有关要求），设立符合股份有限公司要求的组织机构；有公司住所。
>
> 公司型基金依据《公司法》设立。有限责任公司的股东以其认缴的出资额为限对公司承担责任；股份有限公司的股东以其认购的股份为限对公司承担责任。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 去县衙定名、立规矩、安家 | 在市场监督管理部门注册 |
| 合伙契约改写成"公司章程" | 股东共同制定公司章程 |
| 名字查册子确认没人用过 | 有公司名称，符合法律法规及自律规则要求 |
| 租铺面作为住所，县衙查验 | 有公司住所 |
| 县衙发牌子准予经营 | 依法设立公司 |
| 按份额认缴银子，亏多少担多少 | 股东以认缴出资额为限/以认购股份为限承担责任 |
| 送到商会总会备案 | 在中国证券投资基金业协会完成基金产品备案 |
| 每年报一次账 | 按法律法规及自律规则要求报送 |
