# （四）基金合同签署

## 📜 寓言故事 —— 《立字据》

清河镇老周商会的"新织坊份子"，验货之后，赵掌柜、李东家、吴老爷三家都点了头。老周说："银子的事不急，先立字据。字据不立，银子不能动。"

老周拿出三份文书。第一份是"合伙契约"——写明这"份子"叫什么名、设在哪儿、做什么买卖、各家出多少银子、占多少份子、赚了怎么分、亏了怎么担、大事谁说了算、小事谁管。赵掌柜逐条看过，在"管账权"那条上停了笔："老周，你管账，我信。但每月账本得给我看一眼。"老周提笔添了一句："每月初五，账房公开账本，各合伙人可查阅。"李东家则在"退出"那条上加了条件："满三年后，若我想撤，得提前三个月告知，且找得到接手人。"老周点头："正该如此。"吴老爷最谨慎，他把"风险单子"上列的八条风险，又抄进了契约的附录里，说："这八条，契约里得有，不能光是口头。"

第二份文书是"银子监管约"。老周说："各位的银子，先统一存到钱庄的'专用柜'里。这柜子不归我管，归钱庄和监督人共管。银子从各位账上划到专用柜，再从专用柜划到织坊账上，每一笔都得有各位和监督人的印。"赵掌柜问："若我中途反悔，银子在专用柜里，能退吗？"老周说："能。但得按契约里写的规矩退——在专用柜里的银子，还是您的，不是织坊的。只有等字据正式生效、过了冷静期，银子才归织坊。"

第三份文书最特别。老周说："各位签字之后，二十四时辰之内，您随时可以反悔。这二十四时辰里，我不主动联系您，您若不想投了，写张条子给我，银子原路退回，分文不扣。这叫'想清期'。"吴老爷笑："老周，你这规矩比县衙还严。"老周正色："银子是各位的血汗，想清楚了再投，比投了再后悔强。"赵掌柜摆手："我这人想事快，不用二十四时辰。但规矩我认。"

三份文书签完，老周把契约、监管约、风险单子各收一份存档，另给各家一份副本。然后他做了一件让所有人都意外的事：他让商会里从不跑买卖的账房先生老陈，分别给赵掌柜、李东家、吴老爷各打了一个电话。老陈在电话里问："赵掌柜，您签的字据，是自己看清了才签的，还是老周催您签的？"赵掌柜答："我自己看清的，没人催。"老陈又问："契约里写的八条风险，老周有没有跟您说'放心，不会亏'？"赵掌柜答："没有，他说可能亏，让我自己想清。"老陈记下，挂了电话。这叫"回访"。

二十四时辰过去，没人反悔。老周宣布：字据生效，银子从专用柜转入织坊账上，新织坊份子正式成立。赵掌柜举杯："老周，你这'立字据'的规矩，我投了二十年银子，头一回见这么细的。"老周说："字据细，不是防人，是让所有人睡得着。"

**📖 原文定义**

> 在基金投资者与基金管理人通过沟通和谈判就基金设立达成一致意见后，即可准备签署一系列基金文件。基金合同是股权投资基金的核心法律文件，通常由基金管理人和基金投资者等相关主体共同签署，具体的签署主体可能会因基金的组织形式而有所不同。对于普通合伙人与基金管理人分离的合伙型基金以及受托管理的公司型基金，通常会由基金及/或普通合伙人与基金管理人签署委托管理协议。其他相关文件，例如募集结算资金专用账户监督协议、托管协议（如有），通常由基金、基金管理人及/或相关主体与募集监督机构、托管人（如有）等签署。
>
> 基金文件签署完毕后，基金管理人将启动基金设立和备案的相关程序。基金合同生效后，基金投资者将依据合同约定履行出资义务，基金将按照基金合同的约定进入投资期并开始投资运作。
>
> 这一阶段，募集合规事项主要包括募集资金监督、投资冷静期及回访确认等。
>
> 为保障基金募集阶段的投资者资金财产安全，股权投资基金的募集机构应当开立基金募集结算资金专用账户，用于统一归集基金募集结算资金、向投资者分配收益以及分配基金清算后的剩余基金财产等，确保资金原路返还。基金募集结算资金是指由募集机构归集的、在投资者资金账户与基金财产账户或托管资金账户之间划转的往来资金。从资金权属的角度来看，募集结算资金在从投资者的资金账户划出，尚未转入基金财产账户或托管资金账户的阶段，仍然属于投资者的合法财产。
>
> 募集机构应当与符合资质要求的监督机构签署账户监督协议，明确基金募集结算资金专用账户的控制权、责任划分及保障资金划转安全的条款。监督机构应当按照法律法规和账户监督协议的约定，对募集结算资金专用账户实施有效监督，并承担保障基金募集结算资金划转安全的连带责任。
>
> 在投资者签署基金合同之前，募集机构应当向投资者阐明相关法律法规，说明投资冷静期、回访确认等程序性安排。基金合同应当约定为投资者设置不少于24小时的投资冷静期，募集机构在投资冷静期内不得主动联系投资者。
>
> 在实务中，股权投资基金合同通常包含投资冷静期的条款，允许投资者在冷静期结束前通过书面通知基金管理人来行使解除基金合同的权利。在冷静期届满前，投资者实缴的出资不应从募集账户转移到基金财产账户或托管资金账户。同时，基金管理人不得利用这些资金进行任何投资活动。如果投资者选择在冷静期内解除基金合同，募集机构应依据合同约定及时退还投资者的全部已缴付的出资款项。
>
> 募集机构应当在投资冷静期满后，指令本机构从事基金销售推介业务以外的人员以录音电话、电邮、信函等适当方式进行投资回访。回访内容应当符合监管规定且回访过程不得出现诱导性陈述。
>
> 对于专业投资机构，可不适用关于投资冷静期以及回访确认程序；对于社会保障基金、企业年金等养老基金，慈善基金等社会公益基金，依法设立并已在中国证券投资基金业协会备案的私募基金产品，受国务院金融监督管理机构监管的金融产品，投资于所管理基金的基金管理人及其从业人员以及符合规定的其他投资者，可不适用特定对象确定、投资者适当性匹配、风险揭示书签署、投资冷静期以及回访确认等程序。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 三份文书：合伙契约、银子监管约、风险单子 | 签署一系列基金文件：基金合同、募集结算资金专用账户监督协议、托管协议 |
| 契约写明名、址、范围、出资、份额、分利、亏损、决策 | 基金合同是核心法律文件，约定核心条款 |
| 银子先存专用柜，不归老周管 | 开立募集结算资金专用账户，统一归集募集结算资金 |
| 专用柜里的银子还是各家的，不是织坊的 | 募集结算资金尚未转入基金财产账户前仍属于投资者合法财产 |
| 钱庄和监督人共管，每笔得有印 | 与监督机构签署账户监督协议，实施有效监督，承担连带责任 |
| 签字后二十四时辰内可反悔，原路退回 | 投资冷静期不少于24小时，冷静期内可解除合同，退还全部出资 |
| 冷静期内银子不转入织坊账，老周不能动 | 冷静期届满前出资不应从募集账户转移，基金管理人不得利用资金投资 |
| 老陈（不跑买卖的账房）打电话回访 | 冷静期满后由从事基金销售推介业务以外的人员进行回访 |
| 问是否自己看清、有没有被诱导 | 回访内容符合监管规定，不得出现诱导性陈述 |
| 赵掌柜想事快不用冷静期，但规矩认 | 专业投资机构可不适用冷静期及回访程序 |
