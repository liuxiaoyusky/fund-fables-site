# （一）备案方式、时间及前提

## 📜 寓言故事 —— 《牌匾前夜》

江南清河镇码头边，漕运商号"顺通行"的东家宋鹤年最近春风得意。他拉拢了三位老主顾，又说服了两位布商入伙，决定新开一条南货北运的商队。为了让这条线稳当，宋鹤年打算走商会的"明牌"——把商队名号挂在镇口那块大牌匾上，让全镇人都看得见。前辈冯掌柜劝他："牌匾可不是你想挂就能挂的，得先把账目和人员都交到商会的案头，等他们核完，发出公示贴出来，那才叫正经商队。"

宋鹤年本来以为挂个牌子不过是画个押的事。可商会陈文书一句话把他噎住了："你人是凑齐了，可你那两箱定金还堆在自个儿家里，账本也没送到公账柜上。这算什么'齐了'？挂牌的前提，是合伙人都画了押、第一笔真金白银进了公账柜的锁箱。你连锁箱都没碰，怎么敢来挂明牌？"宋鹤年这才意识到，他原本以为"我钱已经收来了"就算完事，可商会认的不是"收到"，是"入库"。他把六位合伙人的签押单据和定金连夜送进了公账柜，又让伙计在锁箱上加了三道封条。

接下来的路走得并不顺。宋鹤年把厚厚的文书卷宗抱进商会值房那天，正赶上梅雨。陈文书翻了几页，皱起眉头："缺东西。合伙人的印鉴不齐，公账柜的流水单少了一张，六位里头的马大嫂的出资凭据还是旧格式。"宋鹤年心里咯噔一下——他原以为材料凑成一摞就完事，没想到每一页都得对得上。回到商号，他连夜补齐印鉴，又去找马大嫂重换了凭据，再把每一份都核对了两遍。第二回递交时，陈文书翻了半个时辰，点了点头："这回齐了。我给你挂个回执，按规矩，二十个工日内商会会给你一个明确说法，过了关，牌匾的事才算定。"宋鹤年站在值房门口，雨打在油纸伞上，他忽然算起账来——其实从签押齐、入库毕那天起，二十个工日内必须把文书送进来，商会再花二十个工日核完。两个二十天，看似冗长，却是把"走过场"挡在外面的两道闸。

十四个工日后，镇口牌匾的位置贴出了一张黄纸公告，顺通行的南货北运商队名列其上。冯掌柜拍着宋鹤年的肩说："你瞧，这张黄纸可比那一摞契据值钱。有了它，你出远门，合伙人才真放心把身家押在你身上；你进货，货主才肯赊账；将来有了纠纷，商会才能替你说话。"宋鹤年望着那张黄纸，想起自己最初只想"挂个名"的小心思，忽然明白：明牌的真正意义，不是好看，是让所有人在出事前能相信你、在出事后能找到人。而那张黄纸背后，是签押、入库、按时上报、按时办结，一环扣一环，谁也省不得。

---

**📖 原文定义**

> 基金备案需要通过中国证券投资基金业协会资产管理业务综合报送平台提交相关材料或信息。
>
> 基金管理人应当自基金募集完毕之日起20个工作日内报送备案材料。中国证券投资基金业协会自备案材料齐备之日起20个工作日内为基金办结备案手续，并通过官方网站对已办理备案的基金相关信息进行公示。
>
> 基金备案以基金募集完毕为前提。"募集完毕"是指基金的已认缴投资者已签署基金合同，且首期实缴募集资金已进入托管账户等基金财产账户。单个投资者首期实缴出资除另有规定外，不得低于合格投资者的最低出资要求。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 宋鹤年把材料抱进商会值房 | 通过专门报送通道提交备案申请 |
| 陈文书用统一格式翻阅文书的程序 | 备案材料需在专门平台按规定提交 |
| 陈文书说"二十个工日内给你说法" | 协会自材料齐备之日起20个工作日内办结备案 |
| 宋鹤年从签押齐、入库毕到递交之间不能拖延 | 自募集完毕之日起20个工作日内须报送 |
| 陈文书以"我钱收来了不算完，得进公账柜锁箱"驳回 | 备案前提——首期实缴资金已进入托管账户 |
| 六位合伙人的签押单据必须齐全 | 备案前提——已认缴投资者已签署基金合同 |
| 马大嫂的旧格式出资凭据被退回 | 单个投资者首期实缴出资须符合出资要求并提交合规材料 |
| 镇口牌匾贴出黄纸公告 | 协会通过官方网站公示已备案基金相关信息 |
