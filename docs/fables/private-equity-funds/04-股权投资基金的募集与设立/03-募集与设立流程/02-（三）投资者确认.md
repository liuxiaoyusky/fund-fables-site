# （三）投资者确认

## 📜 寓言故事 —— 《验货》

清河镇老周商会的新织坊"份子"，茶会之后，有三家明确表示要投：钱庄赵掌柜、船行李东家、省城盐商吴老爷。老周心里高兴，却没急着收银子。他知道，茶会上是自己"卖"，现在轮到对方"验货"了。

赵掌柜第一个登门。他带了两个账房先生，在老周商会坐了整整两天。第一天，账房先生翻老周过去五年的账本：投过几家作坊、几家赚了、几家亏了、总体赚了多少。赵掌柜问得细："三年前你投的那家船厂，亏了四成，当时怎么判断的？后来怎么止损？"老周一一作答，连当时判断失误的缘由也毫不隐瞒。第二天，赵掌柜要见老周的团队——管账的、跑作坊的、盯银子的，一个个叫来问话。最后赵掌柜说："老周，你人我信得过，但你的班子我得看看齐不齐。"老周把三个核心伙计叫到跟前，赵掌柜和他们聊了半晌，才点头："班子齐，这买卖能做。"

李东家验货的方式不同。他不看账本，而是跑去看老周投过的三家织坊。他在第一家织坊蹲了三天，看织工怎么干活、掌柜怎么管人、出货怎么检验。回来对老周说："你挑的织坊，织工手艺好，但掌柜太抠，织工跳槽率高。这风险你得算进去。"老周苦笑："李东家，您这眼睛毒。那家我确实在考虑换掌柜。"李东家又说："我投你，不是只投银子，是投你的眼力。但你得让我知道，你换掌柜的规矩是什么、多久换一次、换不成怎么办。"老周当场写了一份"掌柜更替预案"，李东家才放心。

吴老爷最干脆。他不要看账本，也不要看作坊，只问老周一句话："若这新织坊三年后还亏着，我的银子怎么出来？"老周说："契约里写了，满三年后，可找其他合伙人接手您的份额，但得大家同意。若没人接手，再等两年，商会负责按市价回购。"吴老爷摇头："市价若跌了怎么办？"老周说："那您只能认亏。但契约里也写了——这五年里，商会每年给您看一次账，亏了您随时能撤，只是撤的时候按当时的市价算。"吴老爷沉默良久，说："老周，你实诚。我投。但有一条——你得给我写一份'风险单子'，把可能亏的情况一条条列出来，我签字画押之前，你得念给我听。"

老周照办了。风险单子上列了八条：织坊可能亏、丝价可能涨、掌柜可能跑、销路可能断、织机可能坏、合伙可能闹、五年可能回不了本、市价可能跌。念完，吴老爷逐条签字。赵掌柜和李东家也各签了一份。老周收起三份风险单子，对伙计说："验货验的是双向——他们验我，我也验他们。验完这关，才是真合伙人。"

---

**📖 原文定义**

> 对于股权投资基金，募集工作进展到这一阶段，基金潜在投资者的范围进一步缩小，基石投资者等重要投资者将被逐步锁定，此时，投资者也可开启反向尽职调查工作。股权投资基金对投资者来说属于期限较长、流动性较弱、估值难度较大的资产类别，因此，投资者有天然动力去主动、系统、深入地了解基金管理人，特别是业绩、团队、投资流程等，这些将成为反向尽职调查的重点。通过反向尽职调查，基金投资者可对基金和基金管理人有相对较为充分的了解，有助于其完成投资决策。在基金管理人与投资者互相了解、深度互动的基础上，通过进一步的合格投资者认定、投资者适当性匹配等流程，管理人可逐步确认特定的基金投资者。
>
> 在这个阶段，基金管理人可能会向有明确投资意向的潜在投资者提供基金合同，与其洽谈并初步确定基金合同条款。
>
> 在投资者确认阶段，基金管理人须关注募集合规的多个关键事项，包括基金风险揭示、合格投资者确认、投资者适当性匹配等。
>
> 关于基金风险揭示，基金管理人应当向投资者披露股权投资基金的投资风险、运营风险、流动性风险等风险情况，有下列情形之一的，基金管理人应当通过风险揭示书向投资者进行特别提示：基金存在分级安排或者其他复杂结构，或者涉及重大无先例事项；基金管理人的控股股东、实际控制人、普通合伙人发生变更，尚未在中国证券投资基金业协会完成变更手续；其他重大投资风险或者利益冲突风险。
>
> 基金投向单一标的、未进行组合投资的，基金管理人应当特别提示风险，对投资标的的基本情况、投资架构、因未进行组合投资而可能受到的损失、纠纷解决机制等进行书面揭示，并由投资者签署确认。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 茶会后三家明确表示要投，范围缩小 | 潜在投资者范围进一步缩小，基石投资者逐步锁定 |
| 赵掌柜带账房翻五年账本、问团队 | 投资者开启反向尽职调查，了解业绩、团队、投资流程 |
| 李东家蹲织坊三天看织工、掌柜 | 主动、系统、深入了解基金管理人 |
| 通过反向调查对基金有充分了解 | 有助于投资者完成投资决策 |
| 吴老爷问三年后亏着怎么出来 | 流动性风险：投资者关注退出机制 |
| 老周写风险单子、念给吴老爷听 | 基金管理人应当向投资者披露投资风险、运营风险、流动性风险 |
| 吴老爷逐条签字 | 风险揭示书由投资者签署确认 |
| 验完这关才是真合伙人 | 在互相了解、深度互动基础上，逐步确认特定基金投资者 |
