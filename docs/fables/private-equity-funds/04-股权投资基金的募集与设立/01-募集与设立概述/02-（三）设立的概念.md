# （三）设立的概念

## 📜 寓言故事 —— 《三友合股》

民国初年的冬天，苏州城外山塘街上，三个合伙人凑到一起，想做一笔绸缎生意。

牵头的是做过钱庄账房的冯伯庸，精明稳重；出大头银子的是开当铺的顾老板，性子急，最怕"夜长梦多、官府找茬"；还有一位从南洋回来的林宗越，见多识广，对洋行那一套公司、洋律都熟得很。三人聚在鹤鸣茶馆的二楼雅间，茶没喝两口，冯伯庸先把一摞账册摊到桌上，把进出算了个清清楚楚。算完账，问题摆到桌面上——三个人的钱凑到一起，怎么"成个事"？

顾老板最先发话："我出银子最多，最怕日后扯皮。不如去衙门备个案，领张'合股批文'回来，这样最稳当。"冯伯庸连连点头。林宗越端着茶盏没急着答，反问一句："咱到底要做哪种'事'？"顾老板一愣："这事还有种类？"

林宗越放下茶盏，说起他在南洋见过的三样做法。第一种，几位商人共同立一份章程，写明谁出多少银、谁管事、怎么分利，呈到衙门备案，衙门核准名号、立册登记，发一张盖了红印的"执照"，三人便凭这张执照做事。第二种，不必领执照，但要登一份"合伙文书"——合伙人姓名、份额、谁管账、谁拍板，都写明白，衙门盖戳，回来照样照章办事。第三种更简单——三人围坐一桌，立一份写清权利义务的文契，三方签字画押，文契一生效，"事"就成了，不必去衙门登记。

顾老板听完有些糊涂："那咱走哪条？"冯伯庸沉吟道："这三条路，开头不一样，后面做起来也不一样。领执照那条，规矩最严，每年要报账、摊在明面上，税也最重；登文书那条稍松，合伙人内部议定即可，但衙门那一关得过；只凭文契那条最简便，可也最考验信义——一旦有人反悔，全凭那张纸说话，官府是不管的。"

林宗越点头："三路各有利弊，得看咱这桩生意的脾性、咱三人的性情，再定夺。"顾老板又追问："定下来之后呢？"冯伯庸翻开账册："不管是领执照的、登文书的，还是只立文契的，'事'一旦定下，都得做最后一道登记——把这桩'事'的来龙去脉报到行会那里备个案，留个档。日后行会照册监管，外人查起来有据可查。这一步，少不了的。"

顾老板终于把眉头舒展开来："那今天先把利弊想透，过几日再定走哪条？"冯伯庸合上账册："正该如此。三条路摆在面前，挑哪条是各人的事；但挑完之后的每一步，都得照规矩来——这便是'成个事'的全部门道。"三人对望一眼，会心一笑，茶也添了第二回。

茶凉人散，山塘街的雪又下了一层。那桩绸缎生意究竟走哪条路，三人还没拍板；但他们心里都明白——"成事"二字，从来不是签一份契、立一句话就完的。挑哪一种"形"，过哪一道"门"，每一步都藏着各自的讲究，少一步都不成。

---

**📖 原文定义**

> 股权投资基金必须具备特定的组织形式。我国现行的股权投资基金组织形式包括公司型、合伙型和契约型。选择组织形式时需考虑多种因素，主要包括法律依据、监管要求、基金架构设计、投资业务开展与基金运营的需求以及税负等。不同组织形式的基金在设立时存在不同的要求。公司型基金和合伙型基金的设立通常需在市场监督管理部门完成名称预先核准、申请设立登记和领取营业执照的流程；由于契约型基金不具备工商实体，因此，其无须办理企业注册登记。按照现行法律法规，股权投资基金设立后，应向中国证券投资基金业协会提交相应的基金备案。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 林宗越讲述三种"成事"方式 | 股权投资基金的三种组织形式：公司型、合伙型、契约型 |
| "去衙门备案、领执照"的第一种 | 公司型/合伙型基金在市场监督管理部门完成名称预先核准、设立登记、领取营业执照的流程 |
| "不靠衙门、只凭文契"的第三种 | 契约型基金不具备工商实体，无须办理企业注册登记 |
| "立章程、写明份额、各人分利" | 选择组织形式需考虑基金架构设计、权利义务分配等 |
| "每年要报账、摊在明面上，税也最重" | 选择组织形式需考虑法律依据、监管要求与税负 |
| "事后还得去行会备案，留个档" | 基金设立后向中国证券投资基金业协会提交基金备案 |
| 三人议路、茶凉人散 | 设立是依法设立基金主体的过程，需综合考量多种因素 |
