# （二）募集方式

## 📜 寓言故事 —— 《老周的酒坊》

江南小镇上，老周经营着一家祖传酒坊。他的米酒醇厚，远近闻名，但作坊不大，每年只能酿三百坛。老周想扩大规模，需要买地、建窖、请师傅，少说也得五千两银子。他一个人拿不出这么多钱，便琢磨着找人合伙。

老周先想到了镇上的钱庄。钱庄掌柜说可以帮他"贴榜"，在镇上广场立一块牌子，写明"老周酒坊招合伙人，每股五十两，来者不拒"。老周一听就摇头：广场上三教九流什么人都有，万一引来地痞流氓、外乡赌徒，将来酒坊赚了钱，这些人闹起来怎么办？他想起镇上李掌柜的布庄，去年就是贴了这种"公开榜"，结果招来几十个不相干的人，后来分利不均，闹到县衙去了。老周觉得，酒坊是自己的命根子，合伙人必须知根知底，不能大街上随便拉人。

老周决定自己去找人。他先拜访了做茶叶生意的老吴，两人相识二十年，彼此信得过。老周说："我想扩酒坊，缺银子，你入一股，年底按份子分利。"老吴二话不说，投了五百两。接着老周又去找了开船行的孙老板、经营绸缎的赵夫人，都是镇上有头有脸、家底殷实的人。一个月下来，老周亲自跑遍了六位老友，凑了三千两。虽然辛苦，但每个人都知根知底，老周心里踏实。

可还差两千两。老周犯了愁，自己认识的人已经问遍了。这时，钱庄掌柜又找上门来："周老板，您不愿贴公开榜，这我明白。但我钱庄手里有不少体面客户，我帮您牵线，只引荐我信得过的人，您看如何？"老周想了想，钱庄掌柜确实认识不少正经商人，而且钱庄在镇上经营多年，招牌不能砸。但他也提了几个条件：第一，引荐的人必须是我这酒坊"合适"的合伙人，不能是来历不明的；第二，钱庄只是帮忙介绍，酒坊的大事小情、盈亏分利，最终还是我说了算，钱庄不能替我做主；第三，咱俩得立个字据，写清楚钱庄只负责引荐，出了岔子钱庄要担责，但酒坊的责任还是在我老周肩上。

钱庄掌柜点头应下。没过半月，又凑齐了两千两。老周看着银子到位，心里却不敢松懈。他私下对老吴说："不管是自己找的人，还是钱庄引荐的人，我都得一个个把关。酒坊是我的，选人这件事，谁也替不了我。"

新酒坊落成那年秋天，老周酿出了六百坛酒。分红宴上，老周举杯对众人说："咱们这酒坊，合伙人不多，但个个靠得住。这银子，来得稳，用得也稳。"

**📖 原文定义**

> 股权投资基金的募集，是指股权投资基金管理人或者受其委托的基金销售机构向投资者募集资金，用于设立股权投资基金的行为。募集行为包含推介私募基金，发售基金份额（权益），办理基金份额（权益）认/申购（认缴）、赎回（退出）等活动。
>
> 从募集机构的角度看，股权投资基金的募集可区分为自行募集和委托募集。自行募集是指由基金管理人自行寻找投资者并按照法律法规和自律规则的规定完成募集工作的募集方式，委托募集则是指基金管理人委托有资格的基金销售机构代为募集资金的募集方式。
>
> 无论采取自行募集还是委托募集方式，基金管理人均应设置有效机制，切实保障募集结算资金安全，确保股权投资基金仅向合格投资者募集，防止变相进行公开募集。基金管理人委托基金销售机构募集的，被委托机构应当获得中国证监会基金销售业务资格且成为中国证券投资基金业协会会员。基金管理人应制定募集机构遴选制度，与被委托机构签署基金销售协议。委托基金销售机构募集股权投资基金的，基金管理人不得因委托募集而免除其依法应承担的责任。
>
> 公开募集是指通过公开渠道向非特定对象推介基金产品；非公开募集则是指通过设置特定对象确定程序的渠道进行非公开推介基金产品的行为。
>
> 目前，我国不允许进行股权投资基金的公开募集。按照现行法律法规，股权投资基金只能以非公开方式募集，即必须同时满足两个条件：一是向符合合格投资者标准的特定对象募集；二是单只基金的投资者人数不得超过法律规定的上限。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周不愿贴"公开榜"招合伙人 | 不允许公开募集，只能非公开募集 |
| 老周亲自拜访六位老友筹钱 | 自行募集：管理人自己寻找投资者 |
| 钱庄掌柜引荐其信得过的客户 | 委托募集：委托有资格的机构代为募集 |
| 老周要求钱庄引荐的人必须"合适" | 仅向合格投资者募集 |
| 老周与钱庄立字据、明责任 | 签署基金销售协议，制定遴选制度 |
| 老周说"酒坊的责任还是在我肩上" | 基金管理人不得因委托募集而免除责任 |
| 合伙人人数有限、个个知根知底 | 向特定对象募集，人数有上限 |
