# （四）募集与设立的区别和联系

## 📜 寓言故事 —— 《老周开酒坊》

老周的米酒在镇上出了名，他想把作坊扩成正经的酒坊，需要银子、需要人、更需要一个名正言顺的"字号"。老周明白，这件事得分两步走：第一步是找人、谈钱、说规矩；第二步才是立契约、挂牌子、开门做生意。两步不一样，却连着一根线。

老周先挨家挨户拜访老友。每到一个地方，他都要讲清楚三件事：酒坊将来怎么管、赚了钱怎么分、亏了怎么办。老吴关心的是年底能拿多少利，孙老板在意的是酒坊大事谁说了算，赵夫人则反复确认自己的银子能不能随时撤走。老周一一应对，有时候为了一条分红规矩，能跟老吴在茶馆里坐到半夜。这一阶段，老周的心思全在"人"和"钱"上——谁愿意入伙、入多少、各自什么条件。至于酒坊将来挂什么招牌、去衙门办什么手续，那是后面的事。

三个月后，六位合伙人的条件都谈妥了。老周把众人请到家里，拿出一份写好的契约，上面记着每人出多少银子、占多少份子、酒坊怎么管、怎么分利。众人看过，一一签字画押。老周收起契约，第二天便去了县衙，递上文书，申请给酒坊立一个正经的字号。县衙核准后，酒坊才算在法律上有了身份。老周又拿着县衙的批文，去商会登了记，领了牌子。到了这一步，酒坊才算真正"立"住了。

老周选了个吉日，挂出"周氏酒坊"的招牌，当众宣布酒坊正式成立。从这一天起，合伙人们按契约把银子缴到账房，酒坊开始买粮、雇工、酿新酒。老周站在新挂的招牌下，心里清楚：前面那三个月找人谈钱的日子，叫"募"；后面签合同、跑衙门、挂牌子的日子，叫"立"。募是立的前提，没有募来的银子和谈好的规矩，立就是空架子；立是募的终点，牌子一挂，意味着前面所有的口头承诺都变成了白纸黑字的契约，酒坊从此有了名分，可以正经做生意了。

不过老周也留了个心眼。他在契约里写了一句话："酒坊成立后三年内，若老周觉得银子不够使，经现有合伙人同意，可再邀新人入伙。"老周知道，酿酒的生意周期长，万一中途需要扩窖，他不必从头再走一遍"募"的全过程，只需在已有的框架里补几个新人即可。募和立，不是一刀两断，而是立完之后，募还可以再来——但再来的时候，规矩已经立在那儿了。

**📖 原文定义**

> 基金的募集是基金管理人主导的募集资金的过程，主要关注募集方式、募集对象以及募集过程的合规性等问题。而基金的设立则是依法设立股权投资基金主体的过程，主要考量基金的组织形式、基金架构相关主体以及登记备案等事项。从程序上看，在基金募集阶段，基金投资者和基金管理人通过沟通谈判就基金成立达成一致意见，包括基金组织形式的选择、基金的核心要素和条款等；随后，进入合同签署、工商设立、基金备案等设立环节。基金于基金合同约定的条件成就后成立。实务中，常见的基金成立日是指基金合同签署且基金主体完成工商设立后，由基金管理人宣布的成立日期。有些基金合同会约定，基金管理人应在基金首期实缴出资全部到达托管账户后，宣布基金成立。
>
> 通常情况下，基金设立标志着基金募集工作已全部完成或阶段性完成。基金进入设立程序后，投资者签署基金合同，并按照基金合同中关于出资的条款向基金缴付出资。在完成中国证券投资基金业协会的备案后，股权投资基金即进入投资业务周期。在符合适用法律法规要求的前提下，结合基金的商业需求，部分基金会在合同中约定后续募集条款，以便于在基金设立后的约定时限内继续进行基金募集。后续募集主要是考虑到股权投资基金募集周期可能较长，为了适时启动投资、扩大基金规模等而进行的安排。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周先拜访老友谈钱谈条件 | 募集阶段：关注募集对象、方式、条件 |
| 老周讲清楚分红、管理、退出规矩 | 募集阶段沟通核心要素和条款 |
| 众人签字画押契约 | 合同签署：设立环节的开始 |
| 老周去县衙申请字号、商会登记 | 工商设立、登记备案：设立环节 |
| 挂牌宣布酒坊成立 | 基金成立日：合同签署且主体设立完成 |
| 合伙人按契约缴银子到账房 | 首期实缴出资到达后宣布成立 |
| 挂牌标志前面募的工作完成 | 设立标志募集全部或阶段性完成 |
| 契约里留三年后邀新人的口子 | 后续募集条款：设立后继续募资 |
