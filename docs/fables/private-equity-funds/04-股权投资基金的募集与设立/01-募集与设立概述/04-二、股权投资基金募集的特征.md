# 二、股权投资基金募集的特征

## 📜 寓言故事 —— 《挑掌柜》

江南运河边的清河镇，老周想开一间大酒坊，需要找一位能干的掌柜来主事。消息一传出去，来应聘的人络绎不绝。可老周发现，挑掌柜这件事，远比他想的复杂——不是他单方面选人，而是他和掌柜互相挑。

第一位来的是钱掌柜，在钱庄干了二十年，算盘打得噼啪响，账面上的功夫无人能及。老周起初很满意，可一聊到酒坊的经营，钱掌柜却皱起眉头："周老板，我只管账，酿酒的门道我不懂，也不想过问。"老周心里一沉：酒坊不是钱庄，掌柜不能只懂数字，还得懂酒、懂人、懂行情。钱掌柜 meanwhile 也在打量老周："周老板，您这酒坊八字还没一撇，银子凑齐了吗？地窖挖了吗？我若来了，万一您半年开不了张，我岂不是白忙？"两人客客气气地喝了杯茶，各自告辞。老周明白了：掌柜挑东家的眼光，和东家挑掌柜的眼光，一样苛刻。

第二位是孙掌柜，祖上三代酿酒，手艺没得说。老周和他聊到半夜，从糯米选料到发酵时辰，句句投机。可孙掌柜提了个条件："酒坊赚了钱，我不仅要工钱，还要占一成份子。"老周犹豫了。给份子意味着孙掌柜从此是半个东家，酒坊的大事他都要过问。老周想：我找他来是帮我管事，不是请个新东家。孙掌柜 meanwhile 也在盘算："周老板为人厚道，但酒坊能不能成，还得看他能不能凑够银子、能不能打通销路。我若把身家押进去，万一他那边出了岔子，我岂不是血本无归？"两人又是不欢而散。

第三位是李掌柜，四十出头，既在酒坊当过管事，又自己做过小买卖。老周和他谈了三次。第一次聊酒坊的规矩：怎么酿、怎么卖、赚了怎么分。第二次聊银子：老周凑了多少、还差多少、什么时候能到位。第三次聊人：老周认识哪些老主顾、哪些船帮能运酒、哪些酒楼肯收。李掌柜问得细，老周答得也细。末了，李掌柜说："周老板，我看好您的酒，也看好您这个人。但咱们得立个契约，写明您出银子、我出力，大事怎么定、小事怎么管、亏了怎么办。"老周点头："正该如此。"

契约立下的那天，老周在镇上的茶馆摆了一桌。他对李掌柜说："这三个月，我见了不少掌柜，也被人挑了不少回。挑掌柜和挑东家，原是双向的事。您信我能成事，我信您能管事，这酒坊才能开起来。"李掌柜举杯一笑："正是。您挑我的手艺，我挑您的根基。咱俩互相看得上，这买卖才算成了。"

酒坊开张那年，老周酿的米酒卖到了省城。他常对新来的伙计说："找掌柜别急着定，多谈几次、多看几眼。他挑你，你挑他，挑对了，酒才香。"

**📖 原文定义**

> 首先，股权投资基金的募集活动通常需要较长的时间周期，这一点在首次募集基金的基金管理人中尤为显著。鉴于募集活动是股权投资基金业务周期的起点和业务开展的先决条件，基金管理人需要对募集活动进行系统筹划，并合理安排募集流程。股权投资基金的募资环境会受到多重因素的叠加影响，诸如投资者数量和构成、退出市场发展状况以及宏观经济发展情况等。
>
> 其次，股权投资基金的募集是基金投资者和基金管理人之间的双向选择。基金管理人之间可能因发展阶段、管理规模、专注的行业领域、投资策略以及历史投资业绩等方面的不同展现出差异性和多样性，从而形成市场竞争。投资者群体也表现出多样化特征，部分投资者主要关注资本增值，而部分投资者旨在实现特定的功能或战略目标。在投资过程中，不同的制约因素可能影响投资者的决策，包括但不限于资金的流动性、个体的风险承受能力以及适用的监管要求。此外，投资者的投资频率、规模和集中度等同样呈现出个性化差异。例如，高净值个人投资者与大型金融机构虽然可能共同投资于同一只基金，但其在投资程序、期望的投资规模以及投资条件要求等方面可能存在显著差异。对于基金管理人而言，选择合适的基金投资者不仅是对其投资策略和市场定位的反映，也代表了对未来投资者关系工作的战略规划和合理预期。
>
> 最后，在股权投资基金的募集阶段，基金管理人与投资者之间需要进行频繁和深入的互动。具体而言，在启动任何一只基金的募集时，基金管理人需要提出所募集基金的架构、规模、存续期限、投资策略、基金费用、收益分配、决策机制等核心要素；而基金投资者需要与基金管理人进行充分的洽谈和协商，建立保障双方利益一致的机制，直至其认可基金的核心要素，并且信任基金的核心团队成员能够通过拟定的计划与安排实现基金投资目标。
>
> 综上所述，股权投资基金的募集过程具有其独特性，需要基金管理人与投资者之间的密切合作与共同努力。通过双向选择和深入互动，实现基金管理人与投资者之间的匹配，并为基金的未来运作奠定坚实基础。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 老周三个月见了不少掌柜 | 募集活动需要较长时间周期 |
| 钱掌柜、孙掌柜、李掌柜各有不同 | 管理人之间因能力、条件不同展现差异性 |
| 钱掌柜只懂账不懂酒，孙掌柜要份子 | 投资者群体有不同关注点和条件要求 |
| 老周挑掌柜，掌柜也挑老周 | 募集是投资者与管理人之间的双向选择 |
| 老周和李掌柜谈了三次，越聊越细 | 募集阶段需要频繁和深入的互动 |
| 第一次聊规矩、第二次聊银子、第三次聊人 | 管理人提出核心要素，投资者充分洽谈协商 |
| 契约写明银子、出力、大事小事、亏了怎么办 | 建立保障双方利益一致的机制 |
| 互相看得上，买卖才算成 | 双向选择和深入互动实现匹配 |
