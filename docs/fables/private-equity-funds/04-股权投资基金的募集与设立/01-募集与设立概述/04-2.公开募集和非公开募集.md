# 2. 公开募集和非公开募集

## 🏮 寓言故事 —— 《告示与请帖》

永丰镇的春天总是带着几分忙乱。郑老大坐在永丰号后院的太师椅上，手边搁着一盏凉透的茶，眉头皱得能夹死一只苍蝇。去年他在镇上做了一桩绸缎生意，账上结余了不少银子，眼看着十六铺桥头那边的作坊一个个开起来，他心里盘算着想把这些闲钱凑成一股，邀几个靠得住的人合伙做点买卖，赚的比存在票号里多些。

这事他琢磨了好些日子，今儿把侄子小满从上海叫回来，就是想听听这孩子的意思。

小满下了小火轮，风尘仆仆地进了永丰号后院，往椅子上一坐，先给郑老大斟了碗热茶，才开口："大伯，您这事儿，我在上海那边听洋行的人说过几回，他们管这叫'凑股本'，可不是简单把银子凑一凑就成。怎么个凑法，里头门道大着呢。"

郑老大端起茶碗抿了一口："你说说看。"

小满从怀里掏出一张皱巴巴的报纸，指着上面一则洋行的广告说："洋行那边有种做法，叫'公开发行'——就是在报纸上贴告示，把要做的买卖写得明明白白，谁看了愿意出钱都行，不认人、不挑人，只认钱。天南地北的陌生人，只要掏出银子来，就是合伙人。这种法子来钱快，人多力量大，可上海那边管得严，没拿到衙门批的文书，私自贴告示揽钱，那是要吃官司的。"

郑老大听得直摇头："咱永丰镇可不兴这套。告示一贴，三教九流都来，里头有几个真心做买卖的？万一混进几个拿了银子就跑的主儿，找谁说理去？再说，镇上这几百户人家，田产家底我心里大致有数，谁拿得出钱，谁家出了变故急需用钱，都瞒不过我。这种事，还是熟人间办才踏实。"

小满点了点头："大伯说得对。咱们永丰镇有永丰镇的规矩——不贴告示，只发请帖。请帖递给谁，那是要郑老大点头的。家里有几亩地、铺子开几年、账上有没有窟窿，这些都得心里有数才递。一桌子八个人、十个人，人少了好说话，账也好算。"

话说到这里，镇公所的吴主簿恰好路过永丰号串门。他听见这话，慢悠悠地走进来，把官帽往桌上一搁，端起茶就喝了一口，然后才慢条斯理地说："你们叔侄俩说到点子上了。衙门里有句老话——'告示招天下，请帖请熟客'。告示一招，来的人海了去了，叫'非特定'，谁都可以来；请帖一递，那叫'特定'，是挑过的。咱永丰镇上的买卖，只能用后一种法子。告示揽钱那条路，上头明文不许走，谁要是偷偷贴了告示，下个月的巡检就到他铺子里去了。"

郑老大被吴主簿说得一乐："吴兄，你这一来，倒省了我去镇上问。我正琢磨着要不要在外头贴张告示呢——听你这么一说，这心思就歇了吧。"

吴主簿竖起一根手指头，摇了摇："不止这些。请帖请来的人，还得是'够格'的。什么叫够格？家里得有闲钱，吃饭的钱不能动进来凑数；得能认字看得懂契书；得担得起亏本——万一生意黄了，不能哭到衙门来。这几条合在一块，叫'合格'。咱们私下凑的那一伙人，每个都得是合格的主儿，少一个都不行。"

小满接话："吴先生，那凑几个人封顶？"

吴主簿答得干脆："看你们选什么架势。要是几个老伙计凑一块儿立个门面，按铺子的规矩走，人不能太多。要是按合伙的规矩走，那就参照合伙那套。人一多，衙门就要过问；人一少，自己商量着来，没人管你。这条线划在那儿，是死的。"

郑老大听到这里，终于把茶碗放下了，眉头舒展开来。他拍了拍小满的肩膀："得，这事就照咱自家的法子办——你回去把镇上这几年的老主顾列个单子，咱们一家家去谈，谈得拢的递帖子，谈不拢的下回再说。告示万万贴不得，那不是省事的法子，那是惹事的口子。"

小满笑着点头："大伯放心，我明儿就开始写帖子。"

吴主簿端起茶碗，朝郑老大点了点头："郑老大，你这一辈子做事最让我放心的，就是'算盘打得精'四个字。这回也不例外——熟客上门，请帖先到，告示不贴，规矩不破。永丰号的招牌，才能立得久。"

后院里茶香又热了起来，几只麻雀从屋檐下扑棱棱飞过去。三个人谁也没再提"告示"两个字——那扇门，在永丰镇上，是关着的。

**📖 原文定义**

> 公开募集是指通过公开渠道向非特定对象推介基金产品；非公开募集则是指通过设置特定对象确定程序的渠道进行非公开推介基金产品的行为。
>
> 目前，我国不允许进行股权投资基金的公开募集。按照现行法律法规，股权投资基金只能以非公开方式募集，即必须同时满足两个条件：一是向符合合格投资者标准的特定对象募集；二是单只基金的投资者人数不得超过法律规定的上限。视基金选择的组织形式不同，投资者人数上限应分别符合《证券投资基金法》《公司法》《合伙企业法》等适用法律的规定。

**💡 对应点**

| 故事元素 | 概念对应 |
|---------|---------|
| 洋行在报纸上贴告示揽钱 | 公开募集——通过公开渠道向非特定对象推介 |
| 郑老大在镇上只发请帖 | 非公开募集——通过设置特定对象确定程序推介 |
| 请帖递给谁由郑老大挑 | 特定对象 + 合格投资者标准 |
| 小满列单子逐家去谈老主顾 | 不得向非特定对象公开推介 |
| 吴主簿说"告示不贴，规矩不破" | 我国现行法规不允许股权投资基金公开募集 |
| 吴主簿提"立门面"和"合伙"两套规矩 | 不同组织形式适用不同人数上限法律 |
| 镇公所对凑数人数有上限 | 单只基金投资者人数不得超过法定上限 |
| 三人在后院达成共识只用请帖 | 必须同时满足"合格投资者"和"人数上限"两个条件 |
